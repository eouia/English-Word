# Artificial Intelligence

- 주제 범위: 인공지능, 머신러닝, 데이터와 모델 학습, 생성형 AI, 자동화, 윤리와 위험

## 기본 개념

### artificial intelligence

- 핵심 의미: 인공지능
- 사용 맥락: `artificial intelligence systems`, `advances in artificial intelligence`
- 주의: 사람처럼 사고한다는 좁은 뜻보다, 기계가 인식·예측·판단·생성을 수행하는 기술 전반.

### machine learning

- 핵심 의미: 머신러닝, 기계학습
- 사용 맥락: `machine learning algorithm`, `machine learning model`
- 주의: 사람이 규칙을 일일이 쓰기보다, 데이터에서 패턴을 학습하는 방식.

### algorithm

- 발음: /ˈælgɚˌɪðəm/
- 핵심 의미: 알고리즘, 계산 절차
- 사용 맥락: `recommendation algorithm`, `algorithmic decision`
- 주의: 단순한 공식이 아니라, 입력을 처리해 결과를 내는 절차나 규칙 집합.

### [[mod#model|model]]

- 발음: /ˈmɑdəl/
- 핵심 의미: 모델, 학습된 예측·생성 시스템
- 사용 맥락: `train a model`, `language model`
- 주의: 일반 의미는 모형·본보기. AI에서는 데이터로 학습된 계산 구조.

### neural network

- 핵심 의미: 신경망
- 사용 맥락: `deep neural network`, `neural network architecture`
- 주의: 인간 뇌를 느슨하게 본뜬 구조지만, 실제 뇌와 같다는 뜻은 아님.

### deep learning

- 핵심 의미: 딥러닝, 심층학습
- 사용 맥락: `deep learning techniques`, `deep learning model`
- 주의: 여러 층의 신경망을 사용해 복잡한 패턴을 학습하는 머신러닝 방식.

### generative AI

- 핵심 의미: 생성형 AI
- 사용 맥락: `generative AI tools`, `generative AI models`
- 주의: 분류·예측만 하는 것이 아니라, 텍스트·이미지·코드 같은 새 출력을 생성하는 AI.

### natural language processing

- 핵심 의미: 자연어 처리
- 사용 맥락: `natural language processing tasks`, `NLP system`
- 주의: 사람이 쓰는 언어를 컴퓨터가 분석·이해·생성하게 하는 분야. 약어 `NLP`.

## 데이터와 학습

### dataset

- 발음: /ˈdeɪtəˌsɛt/
- 핵심 의미: 데이터셋, 자료 묶음
- 사용 맥락: `training dataset`, `large dataset`
- 주의: 단순한 파일 하나가 아니라, 학습·평가에 쓰는 구조화된 자료 묶음.

### training

- 발음: /ˈtreɪnɪŋ/
- 핵심 의미: 학습, 훈련
- 사용 맥락: `model training`, `training data`
- 주의: 사람 교육이 아니라, 모델이 데이터에서 패턴을 조정해 나가는 과정.

### train

- 발음: /ˈtreɪn/
- 핵심 의미: 학습시키다, 훈련하다
- 사용 맥락: `train a model`, `train on text data`
- 주의: AI 문맥에서는 모델의 내부 값을 데이터에 맞게 조정하는 것.

### parameter

- 발음: /pɚˈæmətɚ/
- 핵심 의미: 매개변수, 파라미터
- 사용 맥락: `model parameters`, `billions of parameters`
- 주의: 모델이 학습 과정에서 조정하는 내부 값. 일반 의미로는 조건·한계.

### [[fac-make#feature|feature]]

- 발음: /ˈfitʃɚ/
- 핵심 의미: 특징, 입력 특성
- 사용 맥락: `extract features`, `feature representation`
- 주의: 일반 의미는 특징. 머신러닝에서는 모델이 판단에 쓰는 입력 정보의 단위.

### label

- 발음: /ˈleɪbəl/
- 핵심 의미: 라벨, 정답 표시
- 사용 맥락: `labeled data`, `data labeling`
- 주의: 일반 의미는 이름표. 학습 데이터에서는 정답이나 분류 정보를 뜻함.

### [[not#annotation|annotation]]

- 발음: /ˌænəˈteɪʃən/
- 핵심 의미: 주석, 데이터 표기
- 사용 맥락: `human annotation`, `annotated dataset`
- 주의: 데이터에 의미, 분류, 경계, 정답 등을 사람이 붙이는 작업.

### [[corp#corpus|corpus]]

- 발음: /ˈkɔrpəs/
- 핵심 의미: 말뭉치, 대규모 텍스트 자료
- 사용 맥락: `text corpus`, `training corpus`
- 주의: 언어 연구나 NLP에서 분석·학습에 쓰는 텍스트 모음.

### pretraining

- 핵심 의미: 사전학습
- 사용 맥락: `pretraining stage`, `pretrained model`
- 주의: 특정 작업 전에 대규모 데이터로 일반 패턴을 먼저 배우는 단계.

### fine-tuning

- 핵심 의미: 미세조정, 파인튜닝
- 사용 맥락: `fine-tuning a model`, `fine-tuned model`
- 주의: 이미 학습된 모델을 특정 작업이나 데이터에 맞게 추가 조정하는 것.

## 추론과 성능

### inference

- 발음: /ˈɪnfɚəns/
- 핵심 의미: 추론, 모델 실행
- 사용 맥락: `model inference`, `inference time`
- 주의: 일반 의미는 논리적 추론. AI에서는 학습된 모델이 입력을 받아 출력을 내는 단계.

### [[dict#prediction|prediction]]

- 발음: /priˈdɪkʃən/
- 핵심 의미: 예측
- 사용 맥락: `make predictions`, `prediction accuracy`
- 주의: 미래만 예측한다는 뜻이 아니라, 입력에 대해 답이나 분류를 추정하는 것도 포함.

### [[class#classification|classification]]

- 발음: /ˌklæsəfəˈkeɪʃən/
- 핵심 의미: 분류
- 사용 맥락: `image classification`, `text classification`
- 주의: 입력을 미리 정해진 범주 중 하나로 나누는 작업.

### [[cogn#recognition|recognition]]

- 발음: /ˌrɛkəˈgnɪʃən/
- 핵심 의미: 인식, 알아봄
- 사용 맥락: `speech recognition`, `facial recognition`
- 주의: 외교에서는 승인, 일반 의미는 인정. AI에서는 소리·얼굴·패턴을 식별하는 것.

### [[cura#accuracy|accuracy]]

- 발음: /ˈækjɚəsi/
- 핵심 의미: 정확도
- 사용 맥락: `improve accuracy`, `high accuracy`
- 주의: 모든 성능을 대표하지 않음. 데이터가 불균형하면 정확도만으로는 부족할 수 있음.

### [[mark#benchmark|benchmark]]

- 발음: /ˈbɛˌntʃmɑrk/
- 핵심 의미: 기준 시험, 성능 기준
- 사용 맥락: `AI benchmark`, `benchmark results`
- 주의: 일반 의미는 기준점. AI에서는 모델 성능을 비교하는 테스트나 데이터셋.

### [[val#evaluation|evaluation]]

- 발음: /ɪˌvæljuˈeɪʃən/
- 핵심 의미: 평가
- 사용 맥락: `model evaluation`, `evaluation metrics`
- 주의: 모델이 실제로 잘 작동하는지 시험 데이터나 기준으로 확인하는 과정.

### [[lat#latency|latency]]

- 발음: /ˈleɪtənsi/
- 핵심 의미: 지연 시간
- 사용 맥락: `low latency`, `inference latency`
- 주의: 정확도와 별개로, 입력 후 결과가 나오기까지 걸리는 시간.

### scalability

- 발음: /ˈskeɪləˈbɪlɪti/
- 핵심 의미: 확장성
- 사용 맥락: `system scalability`, `scalable AI infrastructure`
- 주의: 사용자·데이터·연산량이 늘어도 시스템이 감당할 수 있는 능력.

## 생성형 AI와 자동화

### prompt

- 발음: /ˈprɑmpt/
- 핵심 의미: 프롬프트, 지시문
- 사용 맥락: `write a prompt`, `prompt engineering`
- 주의: 일반 의미는 촉구하다. 생성형 AI에서는 모델에 주는 입력 지시.

### output

- 발음: /ˈaʊˌtpʊt/
- 핵심 의미: 출력, 산출물
- 사용 맥락: `model output`, `generate output`
- 주의: AI가 만들어 낸 텍스트·이미지·분류 결과 등을 넓게 가리킴.

### hallucination

- 발음: /həˌlusəˈneɪʃən/
- 핵심 의미: 환각, 그럴듯한 오류 생성
- 사용 맥락: `AI hallucination`, `reduce hallucinations`
- 주의: AI가 사실이 아닌 내용을 자신 있게 생성하는 현상. 의학적 환각과 구분.

### automation

- 발음: /ɔtəˈmeɪʃən/
- 핵심 의미: 자동화
- 사용 맥락: `workflow automation`, `automation of routine tasks`
- 주의: AI와 같지는 않음. 단순 규칙 기반 자동화도 가능.

### autonomous

- 발음: /ɔˈtɑnəməs/
- 핵심 의미: 자율적인
- 사용 맥락: `autonomous vehicle`, `autonomous agent`
- 주의: 사람의 직접 조작 없이 스스로 판단·행동하는 정도를 나타냄.

### [[ag#agent|agent]]

- 발음: /ˈeɪdʒənt/
- 핵심 의미: 에이전트, 대리 수행 시스템
- 사용 맥락: `AI agent`, `autonomous agent`
- 주의: 일반 의미는 대리인. AI에서는 목표를 향해 도구를 쓰거나 여러 단계를 수행하는 시스템.

### multimodal

- 핵심 의미: 다중 양식의
- 사용 맥락: `multimodal model`, `multimodal input`
- 주의: 텍스트, 이미지, 음성, 영상 등 여러 종류의 입력·출력을 다루는 것.

### synthetic ([[sym#synthetic|sym]], [[thes#synthetic|thes]])

- 발음: /sɪˈnθɛtɪk/
- 핵심 의미: 합성의, 인공적인
- 사용 맥락: `synthetic data`, `synthetic media`
- 주의: 자연 발생 자료가 아니라 인공적으로 생성된 데이터나 콘텐츠.

## 윤리와 위험

### bias

- 발음: /ˈbaɪəs/
- 핵심 의미: 편향
- 사용 맥락: `algorithmic bias`, `data bias`
- 주의: 개인의 선입견뿐 아니라, 데이터나 모델이 특정 집단에 불리하게 작동하는 경향.

### fairness

- 발음: /ˈfɛrnəs/
- 핵심 의미: 공정성
- 사용 맥락: `AI fairness`, `fairness evaluation`
- 주의: 단순히 모두에게 같은 결과를 주는 것이 아니라, 집단별 불리함을 줄이는 기준과 관련.

### transparency

- 발음: /træˈnspɛrənsi/
- 핵심 의미: 투명성
- 사용 맥락: `algorithmic transparency`, `transparency requirements`
- 주의: 모델이나 의사결정 과정이 얼마나 설명 가능하고 검토 가능한지.

### [[count#accountability|accountability]]

- 발음: /əˈkaʊntəˌbɪlɪti/
- 핵심 의미: 책임성, 설명 책임
- 사용 맥락: `AI accountability`, `hold developers accountable`
- 주의: 문제가 생겼을 때 누가 설명하고 책임지는지에 관한 제도적 의미.

### alignment

- 발음: /əˈlaɪnmənt/
- 핵심 의미: 정렬, 목표 일치
- 사용 맥락: `AI alignment`, `alignment problem`
- 주의: 일반 의미는 줄 맞춤. AI에서는 모델 행동이 인간 의도와 가치에 맞도록 하는 문제.

### privacy

- 발음: /ˈpraɪvəsi/
- 핵심 의미: 사생활, 개인정보 보호
- 사용 맥락: `privacy concerns`, `data privacy`
- 주의: AI 학습과 서비스에서 개인 정보가 수집·노출·재식별될 위험과 관련.

### [[vid-watch#surveillance|surveillance]]

- 발음: /sɚˈveɪləns/
- 핵심 의미: 감시
- 사용 맥락: `AI surveillance`, `mass surveillance`
- 주의: 경찰 수사뿐 아니라, AI로 사람의 행동·얼굴·위치를 대규모 추적하는 문제.

### [[reg#regulation|regulation]]

- 발음: /ˌrɛgjəˈleɪʃən/
- 핵심 의미: 규제, 규정
- 사용 맥락: `AI regulation`, `regulatory framework`
- 주의: 기술 사용을 제한하거나 책임 기준을 세우는 법·행정 규칙.

## 주요 고유명사

### Turing Test

- 핵심 의미: 튜링 테스트
- 사용 맥락: `pass the Turing Test`
- 주의: 기계가 인간처럼 대화할 수 있는지를 묻는 고전적 사고실험.

### Alan Turing

- 핵심 의미: 앨런 튜링
- 사용 맥락: `Alan Turing proposed the imitation game`
- 주의: 계산 이론과 인공지능 논의의 초기 핵심 인물.

### Dartmouth Workshop

- 핵심 의미: 다트머스 워크숍
- 사용 맥락: `the Dartmouth Workshop of 1956`
- 주의: `artificial intelligence`라는 분야명이 공식적으로 자리 잡은 초기 사건으로 자주 언급됨.

### ImageNet

- 핵심 의미: 대규모 이미지 데이터셋
- 사용 맥락: `ImageNet competition`, `trained on ImageNet`
- 주의: 딥러닝 기반 컴퓨터 비전 발전을 설명할 때 자주 나오는 데이터셋.

### AlphaGo

- 핵심 의미: 알파고
- 사용 맥락: `AlphaGo defeated Lee Sedol`
- 주의: 강화학습과 딥러닝의 성과를 대중적으로 알린 AI 시스템.

### ChatGPT

- 핵심 의미: 대화형 생성 AI 서비스
- 사용 맥락: `ChatGPT popularized generative AI`
- 주의: 특정 제품명이지만, 생성형 AI 확산을 설명하는 고유명사로 자주 등장.

## 기억할 점

- `model`은 모형이 아니라 데이터로 학습된 AI 시스템.
- `training`은 모델을 학습시키는 단계, `inference`는 학습된 모델을 실행하는 단계.
- `parameter`, `feature`, `label`은 모두 데이터·모델 구조를 설명하는 핵심어.
- `recognition`은 AI에서는 얼굴·음성·패턴 인식, 외교에서는 승인.
- `automation`은 AI와 같지 않으며, AI 없이도 자동화는 가능.
- `hallucination`은 생성형 AI가 그럴듯하지만 틀린 내용을 만드는 현상.
- `alignment`는 AI 행동과 인간 의도·가치를 맞추는 문제.
