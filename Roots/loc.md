# loc

- 출발 단어: `allocate`
- 핵심 어원: Latin `locus` = 장소, 자리
- 동사형: Latin `locare` = 놓다, 배치하다, 자리 잡게 하다
- 핵심 의미: 장소, 위치, 제자리에 놓음
- 관련 문서: [[pos]], [[stat]], [[sed]], [[via]]
- 주의: `local`은 장소 계열. `colloquial`은 Latin `loqui` = 말하다 계열이라 `loc`와 어원 무관.

## 출발 단어 분석

`allocate` ← Medieval Latin `allocare`

- `ad-`: to
- `locare`: 놓다, 배치하다
- `locus`: 장소, 자리
- 원래 이미지: 어떤 목적에 맞는 자리를 정해 놓음
- 현대 의미: 배정하다, 할당하다

## 단어들

### allocate

- 발음: /ˈæləˌkeɪt/
- 구성: `ad-` = to + `loc` = place
- 뜻: 배정하다, 할당하다
- 어원적 연결: 특정 목적이나 사람에게 자리를 정해 놓음
- 사용 맥락: `allocate resources` = 자원을 배분하다

### allocation

- 발음: /ˌæləˈkeɪʃən/
- 구성: `allocate` + `-ion`
- 뜻: 배정, 할당, 배분
- 어원적 연결: 자리를 정해 나누어 둔 것
- 사용 맥락: `budget allocation` = 예산 배정

### allocated

- 발음: /ˈæləˌkeɪtɪd/
- 구성: `allocate` + `-ed`
- 뜻: 배정된, 할당된
- 어원적 연결: 특정 몫이나 장소가 정해진 상태
- 사용 맥락: `allocated funds` = 배정된 자금

### reallocate

- 발음: /riˈæləkeɪt/
- 구성: `re-` = again + `allocate`
- 뜻: 재배정하다
- 어원적 연결: 이미 놓은 몫이나 자리를 다시 정함
- 사용 맥락: `reallocate staff` = 인력을 재배치하다

### local

- 발음: /ˈloʊkəl/
- 구성: Late Latin `localis` ← `locus`
- 뜻: 지역의, 현지의
- 어원적 연결: 특정 장소에 속함
- 사용 맥락: `local government` = 지방 정부

### locality

- 발음: /loʊˈkæləti/
- 구성: `local` + `-ity`
- 뜻: 지역, 장소성
- 어원적 연결: 특정 장소나 그 성격
- 사용 맥락: `a rural locality` = 농촌 지역

### locale

- 발음: /loʊˈkæl/
- 구성: French `local` 계열
- 뜻: 장소, 현장, 배경
- 어원적 연결: 어떤 일이 벌어지는 특정 장소
- 사용 맥락: `the locale of the story` = 이야기의 배경 장소

### localize

- 발음: /ˈloʊkəˌlaɪz/
- 구성: `local` + `-ize`
- 뜻: 국소화하다, 특정 지역에 맞추다, 위치를 한정하다
- 어원적 연결: 어떤 현상이나 제품을 특정 장소·지역에 묶음
- 사용 맥락: `localize an app` = 앱을 현지화하다

### localized

- 발음: /ˈloʊkəˌlaɪzd/
- 구성: `localize` + `-ed`
- 뜻: 국소적인, 특정 지역에 한정된, 현지화된
- 어원적 연결: 영향이나 기능이 특정 장소에 제한된 상태
- 사용 맥락: `localized pain` = 국소 통증, `localized content` = 현지화된 콘텐츠

### localization

- 발음: /ˌloʊkələˈzeɪʃən/
- 구성: `localize` + `-ation`
- 뜻: 현지화, 국소화, 위치 한정
- 어원적 연결: 특정 장소나 시장에 맞게 놓고 조정하는 과정
- 사용 맥락: `software localization` = 소프트웨어 현지화

### locate

- 발음: /ˈloʊˌkeɪt/
- 구성: Latin `locatus` ← `locare`
- 뜻: 위치를 찾다, 위치시키다
- 어원적 연결: 장소를 정하거나 알아냄
- 사용 맥락: `locate the source` = 출처를 찾아내다

### located

- 발음: /ˈloʊˌkeɪtəd/
- 구성: `locate` + `-ed`
- 뜻: 위치한
- 어원적 연결: 특정 장소에 놓인 상태
- 사용 맥락: `located downtown` = 도심에 위치한

### location

- 발음: /loʊˈkeɪʃən/
- 구성: `locate` + `-ion`
- 뜻: 위치, 장소, 지점
- 어원적 연결: 놓인 자리
- 사용 맥락: `a central location` = 중심 위치

### relocation

- 발음: /ˌriˈloʊˈkeɪʃən/
- 구성: `re-` + `location`
- 뜻: 이전, 재배치
- 어원적 연결: 다른 장소로 다시 놓음
- 사용 맥락: `office relocation` = 사무실 이전

### relocate

- 발음: /ˌriˈloʊkeɪt/
- 구성: `re-` = again + `locate`
- 뜻: 이전하다, 재배치하다
- 어원적 연결: 새로운 장소에 다시 놓음
- 사용 맥락: `relocate to another city` = 다른 도시로 이전하다

### dislocate

- 발음: /ˈdɪsloʊkeɪt/
- 구성: `dis-` = away + `locate`
- 뜻: 탈구시키다, 위치를 어긋나게 하다
- 어원적 연결: 정상 위치에서 벗어나게 놓음
- 사용 맥락: `dislocate a shoulder` = 어깨가 탈구되다

### dislocation

- 발음: /dɪˈsloʊˈkeɪʃən/
- 구성: `dislocate` + `-ion`
- 뜻: 탈구, 위치 이탈, 혼란
- 어원적 연결: 제자리에서 벗어난 상태
- 사용 맥락: `economic dislocation` = 경제적 혼란

### locus

- 발음: /ˈloʊkəs/
- 구성: Latin `locus`
- 뜻: 장소, 위치; 궤적
- 어원적 연결: 어떤 일이 있거나 점들이 모이는 자리
- 사용 맥락: `the locus of control` = 통제 소재

### loci

- 발음: /ˈloʊki/
- 구성: Latin plural of `locus`
- 뜻: 여러 위치, 여러 궤적
- 어원적 연결: 여러 장소나 점들의 집합
- 사용 맥락: `genetic loci` = 유전적 위치들

### locomotive

- 발음: /ˌloʊkəˈmoʊtɪv/
- 구성: `loco-` = from place to place + `motive`
- 뜻: 기관차; 이동하는
- 어원적 연결: 장소에서 장소로 움직임
- 사용 맥락: `a steam locomotive` = 증기 기관차

### locomotion

- 발음: /ˌloʊkəˈmoʊʃən/
- 구성: `loco-` = place + `motion`
- 뜻: 이동, 운동
- 어원적 연결: 장소를 바꾸는 움직임
- 사용 맥락: `animal locomotion` = 동물의 이동

### collocate

- 구성: `com-` = together + `locate`
- 뜻: 함께 배치하다; 함께 자주 쓰이다
- 어원적 연결: 같은 장소나 문맥에 함께 놓임
- 사용 맥락: `words that collocate with each other` = 서로 잘 어울려 쓰이는 단어들

### collocation

- 발음: /kɑləˈkeɪʃən/
- 구성: `collocate` + `-ion`
- 뜻: 배열, 배치; 연어
- 어원적 연결: 단어들이 함께 놓이는 방식
- 사용 맥락: `a common collocation` = 흔한 연어

## 기억 포인트

- `loc`: 장소, 자리.
- `allocate`: 자리를 정해 배정함.
- `local`: 특정 장소에 속함.
- `localized`: 특정 장소에 한정됨.
- `locate`: 위치를 찾거나 놓음.
- `relocate`: 다시 다른 장소에 놓음.
- `dislocate`: 제자리에서 벗어나게 함.
- `locus`: 장소, 위치.

## 비교 단어

### pos

- [[pos]]도 놓다, 두다.
- `loc`는 “어느 장소에 놓는가”가 더 강함.
- `pos`는 놓는 행위와 배치 상태가 더 넓음.

### stat

- [[stat]]는 서 있는 상태와 공식적 위치.
- `loc`는 장소 자체나 위치 지정.

### sed

- [[sed]]는 앉다, 자리 잡다.
- `loc`는 앉는 동작보다 장소·위치를 정하는 말.

### allow

- [[allow]]는 Latin `allocare` = 배정하다 계열과 `allaudare` = 칭찬·승인하다 계열이 섞인 단어.
- `allowance`의 "배정된 몫" 의미는 `allocate`와 연결되지만, 현대 `allow`의 "허락하다" 의미는 혼합 어원으로 따로 보는 편이 좋음.
