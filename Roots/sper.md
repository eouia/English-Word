# sper

- 출발 단어: `desperate`
- 핵심 어원: 라틴어 `sperare` = 바라다, 희망하다
- 핵심 이미지: 희망을 품거나, 희망에서 멀어짐
- 관련해서 헷갈릴 수 있는 어근: [[spond]]의 `despond` = 약속/응답을 내려놓고 낙담함

## 출발 단어 분석

`desperate`는 라틴어 `desperare`에서 옴.

- `de-`: away, down
- `sperare`: 바라다, 희망하다

따라서 `desperate`는 원래 "희망에서 멀어진" → "절망적인, 필사적인"의 뜻.

## 단어들

### desperate

- 발음: /ˈdɛsprɪt/
- 구성: `de-` = away + `sperare`
- 뜻: 절망적인, 필사적인, 간절한
- 어원적 연결: 희망이 사라진 상태에서 마지막 수단까지 쓰려는 느낌
- 사용 맥락: `a desperate attempt` = 필사적인 시도

### desperately

- 발음: /ˈdɛspɚətli/
- 구성: `desperate` + `-ly`
- 뜻: 필사적으로, 몹시
- 어원적 연결: 절박한 마음으로 행동함
- 사용 맥락: `desperately need help` = 도움이 절실히 필요하다

### desperation

- 발음: /ˌdɛspɚˈeɪʃən/
- 구성: `desperate` + `-ion`
- 뜻: 절망, 자포자기, 절박함
- 어원적 연결: 희망이 거의 사라져 절박해진 상태
- 사용 맥락: `act out of desperation` = 절박함 때문에 행동하다

### despair

- 발음: /dɪˈspɛr/
- 구성: Old French `desesperer`, Latin `desperare`
- 뜻: 절망하다, 절망
- 어원적 연결: 희망에서 완전히 멀어짐
- 사용 맥락: `fall into despair` = 절망에 빠지다

### prosper

- 발음: /ˈprɑspɚ/
- 구성: Latin `prosperare`
- 뜻: 번영하다, 성공하다
- 어원적 연결: 전통적으로 `pro-` = according to, toward + `sperare` = hope와 연결. 기대한 대로 잘 풀리는 상태
- 사용 맥락: `businesses prosper` = 사업이 번창하다

### prosperity

- 발음: /prɑˈspɛrəti/
- 구성: `prosper` + `-ity`
- 뜻: 번영, 번창
- 어원적 연결: 일이 기대한 방향으로 잘 풀린 상태
- 사용 맥락: `economic prosperity` = 경제적 번영

### prosperous

- 발음: /ˈprɑspɚəs/
- 구성: `prosper` + `-ous`
- 뜻: 번영하는, 성공한
- 어원적 연결: 희망하고 기대한 대로 잘되는 상태
- 사용 맥락: `a prosperous country` = 번영하는 나라

## 기억 포인트

`sper` 계열은 "희망" 이미지.

- `desperate`: 희망에서 멀어진, 그래서 필사적인
- `desperation`: 희망이 사라진 절박함
- `despair`: 희망을 잃은 절망
- `prosper`: 기대한 대로 잘 풀림
- `prosperity`: 번영

## 비교 단어

### despond

- 구성: `de-` + `spond`
- 뜻: 낙담하다
- 구분 포인트: [[spond]]의 `despond`는 "약속/응답/기대에 대한 포기" 이미지. `desperate`, `despair`는 `sperare` = 희망에서 직접 출발
