# plic-display

- 출발 단어: `display`
- 핵심 어원: Latin `displicare` = scatter, unfold; Medieval Latin = unfold
- 핵심 이미지: 접힌 것, 말린 것, 닫힌 것을 밖으로 펴서 보이게 함
- 허브 문서: [[plic]]
- 관련 문서: [[plic-explain]]
- 주의: `display`와 `deploy`는 같은 Old French `desploiir/desployer` 계열의 가까운 친척. `display`는 보이게 펼침, `deploy`는 목적에 맞게 펼쳐 배치함.

## 단어들

### display

- 발음: /dɪˈspleɪ/
- 구성: `dis-` = apart + Latin `plicare` = fold
- 뜻: 전시하다, 보여 주다, 표시하다; 전시, 화면 표시
- 어원적 연결: 접힌 깃발이나 닫힌 것을 펼쳐 남이 볼 수 있게 함
- 사용 맥락: `display artwork` = 작품을 전시하다, `display information on a screen` = 화면에 정보를 표시하다

### displayable

- 구성: `display` + `-able`
- 뜻: 표시할 수 있는, 전시 가능한
- 어원적 연결: 펼쳐서 남에게 보여 줄 수 있음
- 사용 맥락: `displayable characters` = 표시 가능한 문자

### deploy

- 발음: /dɪˈplɔɪ/
- 구성: French `deployer` ← Old French `desploiier` = unfold
- 뜻: 배치하다, 전개하다, 사용하다
- 어원적 연결: 접힌 병력 대열이나 장비를 펼쳐 목적에 맞게 배치함
- 사용 맥락: `deploy troops` = 병력을 배치하다, `deploy software` = 소프트웨어를 배포·전개하다

### deployment

- 발음: /dɪˈplɔɪmənt/
- 구성: `deploy` + `-ment`
- 뜻: 배치, 전개, 배포
- 어원적 연결: 접혀 있던 병력·장비·시스템을 펼쳐 실제 위치나 환경에 놓음
- 사용 맥락: `troop deployment` = 병력 배치, `software deployment` = 소프트웨어 배포

### splay

- 구성: Middle English `splayen`, shortened from `displayen`
- 뜻: 벌리다, 바깥쪽으로 펼쳐지다
- 어원적 연결: `display`에서 앞부분이 줄어든 형태. 접힌 것이 부채처럼 벌어짐
- 사용 맥락: `splay one's fingers` = 손가락을 벌리다

### splayed

- 구성: `splay` + `-ed`
- 뜻: 벌어진, 밖으로 퍼진
- 어원적 연결: 펼쳐져 한쪽 방향이 아니라 바깥으로 퍼진 상태
- 사용 맥락: `splayed feet` = 벌어진 발

## 비교 단어

### explicit

- 뜻: 명시적인, 분명한
- 어원적 연결: [[plic-explain]] 계열. 접힌 뜻을 펴서 드러냄
- 구분 포인트: `explicit`은 말·의미가 펴져 분명함. `display`는 물건·정보·상태를 펼쳐 보임

### unfold

- 뜻: 펼치다, 드러나다
- 구분 포인트: `unfold`는 게르만계 기본어. `display/deploy`는 라틴계 `plicare`의 "접힌 것을 펴다" 이미지

## 기억 포인트

`plic-display`는 "펼쳐 보임, 펼쳐 배치함" 갈래.

- `display`: 펼쳐 보이다
- `deploy`: 펼쳐 배치하다
- `deployment`: 배치와 전개
- `splay`: 바깥쪽으로 벌리다
- `splayed`: 벌어진 상태
