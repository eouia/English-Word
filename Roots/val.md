# val

- 출발 단어: `value`
- 핵심 어원: 라틴어 `valere` = 힘이 있다, 건강하다, 가치가 있다
- 핵심 이미지: 힘과 가치가 있음
- 관련해서 헷갈릴 수 있는 어근: [[ambi]], [[equ]] = 같음, 균형, 공정함 계열, [[estim]] = 가치를 매기고 추산함

## 출발 단어 분석

`value`는 Latin `valere`에서 온 말.

- `valere`: be strong, be well, be worth
- Old French `value`: worth, price

따라서 `value`는 "힘이 있고 쓸모가 있음" → "가치, 값, 소중히 여기다"의 뜻.

## 단어들

### value

- 발음: /ˈvælju/
- 구성: Latin `valere`
- 뜻: 가치, 값, 소중히 여기다
- 어원적 연결: 힘이나 쓸모가 있어 값어치가 있음
- 사용 맥락: `the value of education` = 교육의 가치

### valuable

- 발음: /ˈvæljəbəl/
- 구성: `value` + `-able`
- 뜻: 가치 있는, 귀중한
- 어원적 연결: 값어치가 있을 만함
- 사용 맥락: `valuable advice` = 귀중한 조언

### valuation

- 발음: /væljuˈeɪʃən/
- 구성: `value` + `-ation`
- 뜻: 평가, 가치 산정
- 어원적 연결: 값어치를 따져 정함
- 사용 맥락: `company valuation` = 기업 가치 평가

### evaluate

- 발음: /ɪˈvæljuˌeɪt/
- 구성: `e- / ex-` = out + `value`
- 뜻: 평가하다
- 어원적 연결: 가치나 힘을 밖으로 드러내어 따짐
- 사용 맥락: `evaluate performance` = 성과를 평가하다

### evaluation

- 발음: /ɪˌvæljuˈeɪʃən/
- 구성: `evaluate` + `-ion`
- 뜻: 평가
- 어원적 연결: 가치를 따져 보는 과정
- 사용 맥락: `course evaluation` = 강의 평가

### valid

- 발음: /ˈvælɪd/
- 구성: Latin `validus` = strong
- 뜻: 유효한, 타당한
- 어원적 연결: 논리나 법적 효력이 강함
- 사용 맥락: `a valid argument` = 타당한 주장

### validity

- 발음: /vəˈlɪdəti/
- 구성: `valid` + `-ity`
- 뜻: 타당성, 유효성
- 어원적 연결: 힘 있게 성립하는 성질
- 사용 맥락: `question the validity` = 타당성에 의문을 제기하다

### validate

- 발음: /ˈvælədeɪt/
- 구성: `valid` + `-ate`
- 뜻: 검증하다, 확인하다, 유효하게 하다
- 어원적 연결: 어떤 주장·자료·절차가 힘 있게 성립함을 확인함
- 사용 맥락: `validate the results` = 결과를 검증하다, `validate a ticket` = 표를 유효하게 하다

### validation

- 발음: /ˌvæləˈdeɪʃən/
- 구성: `validate` + `-ion`
- 뜻: 검증, 확인, 유효화
- 어원적 연결: 타당성이나 효력이 있음을 확인하는 과정
- 사용 맥락: `data validation` = 데이터 검증

### invalidate

- 발음: /ˌɪˈnvælɪˌdeɪt/
- 구성: [[in]] = not + `valid` + `-ate`
- 뜻: 무효화하다
- 어원적 연결: 효력이나 타당성을 없앰
- 사용 맥락: `invalidate a contract` = 계약을 무효화하다

### equivalent

- 발음: /ɪˈkwɪvələnt/
- 구성: [[equ]] = equal + `valere`
- 뜻: 동등한, 상당하는 것
- 어원적 연결: 가치나 힘이 같음
- 사용 맥락: `an equivalent amount` = 동등한 양

### equivalence

- 발음: /ɪˈkwɪvələns/
- 구성: `equivalent` + `-ence`
- 뜻: 동등성, 등가
- 어원적 연결: 같은 가치나 힘을 가짐
- 사용 맥락: `logical equivalence` = 논리적 동등성

### prevail

- 발음: /prɪˈveɪl/
- 구성: `pre-` = before, superior + `valere`
- 뜻: 널리 퍼지다, 우세하다, 이기다
- 어원적 연결: 더 강한 힘으로 앞섬
- 사용 맥락: `justice will prevail` = 정의가 이길 것이다

### prevalent

- 발음: /ˈprɛvələnt/
- 구성: `prevail` + `-ent`
- 뜻: 널리 퍼진, 우세한
- 어원적 연결: 힘을 얻어 널리 자리 잡음
- 사용 맥락: `a prevalent belief` = 널리 퍼진 믿음

### valor

- 발음: /ˈvælɚ/
- 구성: Latin `valor` = worth, bravery
- 뜻: 용맹
- 어원적 연결: 힘과 가치가 있는 용기
- 사용 맥락: `acts of valor` = 용맹한 행동

## 기억 포인트

`val` 계열은 "힘 / 가치 / 효력" 이미지.

- `value`: 값어치
- `valuable`: 가치 있는
- `evaluate`: 가치를 따짐
- `valid`: 힘 있게 성립함
- `validate`: 성립함을 확인하거나 유효하게 함
- `invalidate`: 성립하는 힘을 없앰
- `equivalent`: 같은 가치
- `prevail`: 힘이 우세함
- `valor`: 용맹한 힘

## 비교 단어

### equ

- 뜻: 같음, 균형
- 구분 포인트: [[equ]]는 같게 맞춤. `val`은 가치나 힘이 있음. `equivalent`는 둘이 결합해 "같은 가치"의 뜻

### estim

- 뜻: 가치를 매기다, 어림해 판단하다
- 구분 포인트: [[estim]]은 가치나 수치를 판단하는 행위. `val`은 가치, 힘, 효력 자체

### prove

- 뜻: 증명하다
- 구분 포인트: [[prob]]는 시험해 보고 참됨을 보임. `validate`는 이미 제시된 결과·절차·주장의 유효성이나 타당성을 확인함.
