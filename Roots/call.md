# call

- 출발 단어: `call`
- 핵심 의미: 큰소리로 부르다, 이름 붙이다, 소집하다
- 관련 형태: Old Norse `kalla` = cry out, call
- 관련 문서: [[voc]], [[roll]]

## 핵심 이미지

`call` 계열의 바탕은 "소리 내어 부름".

사람을 부르면 `call`.
이름을 붙이면 `call`.
다시 부르면 `recall`.
부르는 사람은 `caller`.
명단을 부르면 `roll call`.
특별한 소명이나 직업적 부름은 `calling`.

## 단어들

### call

- 구성: Old Norse `kalla` = cry out, call
- 뜻: 부르다, 전화하다, 이름 붙이다; 요청, 부름
- 연결: 소리 내어 누군가를 부르거나 어떤 이름을 붙임
- 사용 맥락: `call a friend` = 친구에게 전화하다, `call it a success` = 그것을 성공이라고 부르다

### called

- 구성: `call` + `-ed`
- 뜻: 불린, 이름 붙여진
- 연결: 어떤 이름으로 불리게 됨
- 사용 맥락: `a book called Dune` = Dune이라는 책

### calling

- 구성: `call` + `-ing`
- 뜻: 부름, 직업적 소명
- 연결: 어떤 일을 하도록 부름을 받은 느낌
- 사용 맥락: `teaching is her calling` = 가르치는 일이 그녀의 소명이다

### caller

- 구성: `call` + `-er`
- 뜻: 전화한 사람, 방문자
- 연결: 부르거나 전화를 건 사람
- 사용 맥락: `unknown caller` = 알 수 없는 발신자

### recall

- 구성: `re-` = back/again + `call`
- 뜻: 기억해 내다, 회수하다
- 연결: 마음속으로 다시 불러옴
- 사용 맥락: `recall a memory` = 기억을 떠올리다

### roll call

- 구성: `roll` = list + `call`
- 뜻: 출석 확인, 점호
- 연결: 명단의 이름을 차례로 부름
- 사용 맥락: `take roll call` = 출석을 확인하다

### callback

- 구성: `call` + `back`
- 뜻: 회신 전화, 재호출, 콜백
- 연결: 다시 부르거나 다시 전화함
- 사용 맥락: `request a callback` = 회신 전화를 요청하다

### callout

- 구성: `call` + `out`
- 뜻: 지적, 공개 비판; 강조 표시
- 연결: 밖으로 소리 내어 부르거나 눈에 띄게 표시함
- 사용 맥락: `a callout box` = 강조 상자

## 비교 단어

`voc`는 [[voc]] 문서처럼 Latin `vocare` = 부르다, 목소리 계열.
`vocal`, `vocation`, `invoke` 같은 라틴계 시험어휘는 이쪽.

`claim`은 Latin `clamare` = 외치다 계열로 볼 수 있음.
뜻은 "주장하다, 요구하다"로 발전했으므로 `call`과 섞지 않음.

`calligraphy`는 Greek `kallos` = beauty + `graph` = writing.
철자에 `call`이 보여도 "부르다"가 아니라 "아름다운 글씨" 계열.
