# cre

- 출발 단어: `create`
- 핵심 어원: 라틴어 `creare` = 만들다, 낳다, 생기게 하다
- 핵심 의미: 새로 생기게 함, 만들어 냄
- 관련 문서: [[cresc]], [[fac]], [[gen]]
- 주의: [[cresc]]는 라틴어 `crescere` = 자라다 계열. 깊은 이미지는 가깝지만 영어 문서에서는 분리.

## 출발 단어 분석

`create` ← 라틴어 `creare`

- 원래 이미지: 낳다, 자라나게 하다, 존재하게 만들다
- 의미 확장: 무언가를 새로 만듦 → 창조함 → 결과를 발생시킴
- 현대 핵심: 없던 것을 생기게 함

## 단어들

### create

- 발음: /kriˈeɪt/
- 구성: 라틴어 `creare`
- 뜻: 만들다, 창조하다, 발생시키다
- 어원적 연결: 없던 것을 생기게 함
- 사용 맥락: `create an opportunity` = 기회를 만들다

### creation

- 발음: /kriˈeɪʃən/
- 구성: `create` + `-ion`
- 뜻: 창조, 창작물, 생성
- 어원적 연결: 생기게 만든 행위나 결과
- 사용 맥락: `the creation of jobs` = 일자리 창출

### creative

- 발음: /kriˈeɪtɪv/
- 구성: `create` + `-ive`
- 뜻: 창의적인, 창조적인
- 어원적 연결: 새것을 만들어 내는 성질
- 사용 맥락: `creative thinking` = 창의적 사고

### creativity

- 발음: /ˌkrieɪˈtɪvəti/
- 구성: `creative` + `-ity`
- 뜻: 창의성
- 어원적 연결: 새롭게 만들어 내는 능력
- 사용 맥락: `encourage creativity` = 창의성을 장려하다

### creator

- 발음: /kriˈeɪtɚ/
- 구성: `create` + `-or`
- 뜻: 창조자, 제작자
- 어원적 연결: 무언가를 생기게 만든 사람
- 사용 맥락: `content creator` = 콘텐츠 제작자

### creature

- 발음: /ˈkritʃɚ/
- 구성: 라틴어 `creatura` = 만들어진 것
- 뜻: 생물, 피조물
- 어원적 연결: 창조되어 생겨난 존재
- 사용 맥락: `a living creature` = 살아 있는 생물

### recreate

- 발음: /ˈrɛkriˌeɪt/
- 구성: `re-` = again + `create`
- 뜻: 재현하다, 다시 만들다
- 어원적 연결: 이미 있던 것을 다시 생기게 함
- 사용 맥락: `recreate the scene` = 장면을 재현하다

### recreation

- 발음: /ˌrɛkriˈeɪʃən/
- 구성: `re-` + `creation`
- 뜻: 오락, 휴양, 재충전
- 어원적 연결: 기운을 다시 만들어 냄
- 사용 맥락: `parks and recreation` = 공원과 여가 활동

### procreate

- 발음: /ˈproʊkriˈeɪt/
- 구성: `pro-` = forth + `create`
- 뜻: 낳다, 생식하다
- 어원적 연결: 생명을 앞으로 생겨나게 함
- 사용 맥락: `the ability to procreate` = 생식 능력

### procreation

- 발음: /ˈproʊkriˈeɪʃən/
- 구성: `procreate` + `-ion`
- 뜻: 생식, 출산
- 어원적 연결: 새 생명을 만들어 내는 행위
- 사용 맥락: `human procreation` = 인간 생식

## 기억 포인트

- `cre`는 “새로 생기게 함”.
- `create`는 예술 창작뿐 아니라 `create jobs`, `create problems`처럼 결과를 발생시키는 뜻도 강함.
- `creature`는 “만들어진 존재”에서 생물, 피조물.

## 비교 단어

### cresc

- [[cresc]]는 `crescere` = 자라다, 커지다.
- `increase`, `concrete`, `crescent`는 `creare`가 아니라 `crescere` 계열.

### fac

- [[fac]]는 만들다, 하다, 결과를 내다.
- `create`는 “새로 존재하게 함” 쪽이 더 강하고, `fac`는 “행위/제작/작용” 이미지가 강함.
