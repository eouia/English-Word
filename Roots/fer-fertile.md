# fer-fertile

- 출발 단어: `fertile`
- 핵심 어원: Latin `fertilis` = fruitful, bearing
- 핵심 이미지: 열매·자손·결과를 많이 지님
- 허브 문서: [[fer]]
- 주의: `fertile`은 넓게는 Latin `ferre` = bear, carry의 "지니다, 낳다" 이미지와 연결.

## 단어들

### fertile

- 발음: /ˈfɝtəl/
- 구성: Latin `fertilis`
- 뜻: 비옥한, 생산적인
- 어원적 연결: 열매나 결과를 많이 지니고 낳음
- 사용 맥락: `fertile soil` = 비옥한 토양

### fertility

- 발음: /fɚˈtɪləti/
- 구성: `fertile` + `-ity`
- 뜻: 비옥함, 생식력
- 어원적 연결: 생명이나 결과를 낳을 수 있는 힘
- 사용 맥락: `soil fertility` = 토양 비옥도

### fertilize

- 발음: /ˈfɝtəˌlaɪz/
- 구성: `fertile` + `-ize`
- 뜻: 비옥하게 하다, 수정시키다
- 어원적 연결: 낳거나 자라게 할 수 있는 상태로 만듦
- 사용 맥락: `fertilize the soil` = 토양에 비료를 주다

### fertilizer

- 발음: /ˈfɝtəˌlaɪzɚ/
- 구성: `fertilize` + `-er`
- 뜻: 비료
- 어원적 연결: 땅이 작물을 낳도록 돕는 것
- 사용 맥락: `organic fertilizer` = 유기 비료

### proliferate

- 발음: /proʊˈlɪfɚˌeɪt/
- 구성: Latin `proles` = offspring + `fer`
- 뜻: 급증하다, 증식하다
- 어원적 연결: 자손이나 결과를 많이 낳아 퍼뜨림
- 사용 맥락: `rumors proliferate online` = 소문이 온라인에서 급증하다

### proliferation

- 발음: /ˌproʊlɪfɚˈeɪʃən/
- 구성: `proliferate` + `-ion`
- 뜻: 급증, 확산
- 어원적 연결: 많이 생겨 퍼지는 과정
- 사용 맥락: `nuclear proliferation` = 핵 확산

## 기억 포인트

`fer-fertile`은 "지니고 낳음" 갈래.

- `fertile`: 많이 낳을 수 있음
- `fertility`: 생산력, 생식력
- `fertilize`: 낳을 수 있게 만듦
- `proliferate`: 많이 생겨 퍼짐
