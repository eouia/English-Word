# orat

- 출발 단어: `oration`
- 핵심 어원: Latin `orare` = 말하다, 기도하다, 간청하다
- 핵심 이미지: 공식적으로 말하거나 간청함
- 관련 문서: [[loqu]], [[dict]], [[voc]]
- 주의: `oral` = 입의 계열은 Latin `os` / `oris` = mouth에서 온 말로, 이 문서의 `orare`와 구분.

## 출발 단어 분석

`oration` ← Latin `oratio`

- `orare`: speak, pray, plead
- `oratio`: speech
- 원래 이미지: 공식적으로 말하거나 간청하는 말
- 현대 의미: 연설

## 단어들

### oration

- 구성: Latin `oratio`
- 뜻: 공식 연설
- 어원적 연결: 격식을 갖춰 말함
- 사용 맥락: `a funeral oration` = 추도 연설

### orator

- 구성: Latin `orator`
- 뜻: 연설가, 웅변가
- 어원적 연결: 공식적으로 말하는 사람
- 사용 맥락: `a gifted orator` = 타고난 연설가

### oratory

- 구성: Latin `oratorius`
- 뜻: 웅변술, 예배당
- 어원적 연결: 말하고 기도하는 장소나 기술
- 사용 맥락: `skills in oratory` = 웅변술

### peroration

- 구성: `per-` = through, to the end + `oration`
- 뜻: 연설의 결론부, 장황한 연설
- 어원적 연결: 연설을 끝까지 밀고 가는 마무리 말
- 사용 맥락: 수사학·격식 문맥에서 쓰임

### oracle

- 구성: Latin `oraculum` ← `orare`
- 뜻: 신탁, 예언자, 현명한 조언자
- 어원적 연결: 신이 말해 준다고 여겨진 말
- 사용 맥락: `the oracle of Delphi` = 델포이 신탁

### inexorable

- 구성: `in-` = not + Latin `exorare` = persuade by pleading
- 뜻: 막을 수 없는, 냉혹한
- 어원적 연결: 간청해도 마음을 돌릴 수 없음
- 사용 맥락: `inexorable decline` = 막을 수 없는 쇠퇴

## 기억 포인트

`orat` 계열은 "공식적으로 말하다, 간청하다" 이미지.

- `oration`: 공식 연설
- `orator`: 연설가
- `oratory`: 웅변술
- `oracle`: 신이 말해 준 말
- `inexorable`: 간청해도 바꿀 수 없음
