# nav

- 출발 단어: `navigation`
- 핵심 어원: Latin `navis` = 배, 선박
- 핵심 의미: 배, 항해, 이동 경로를 찾음
- 주요 형태: `nav`, `nau`
- 관련 문서: [[ag]], [[circum]], [[astr]]
- 주의: `navigate`는 Latin `navigare` = 배를 몰다, 항해하다. `navis` = 배 + `agere` = 몰다/움직이다가 결합한 형태.

## 출발 단어 분석

`navigation` = `navis` + `agere` + `-ation`

- `navis`: 배, 선박
- `agere`: 몰다, 움직이다
- `-ation`: 과정, 행위, 결과
- 원래 이미지: 배를 몰아 길을 찾아 감
- 의미 확장: 항해 → 길 찾기 → 웹·앱 화면 이동 구조

## 단어들

### navigation

- 발음: /ˈnævəˈgeɪʃən/
- 구성: Latin `navigatio` < `navigare`
- 뜻: 항해, 길 찾기, 내비게이션
- 어원적 연결: 배를 움직여 목적지까지 가는 과정
- 사용 맥락: `satellite navigation` = 위성 항법, `website navigation` = 웹사이트 이동 구조

### navigate

- 발음: /ˈnævəˌgeɪt/
- 구성: `navis` = ship + `agere` = drive
- 뜻: 항해하다, 길을 찾다, 헤쳐 나가다
- 어원적 연결: 배를 몰아 물길을 지나감
- 사용 맥락: `navigate through a crisis` = 위기를 헤쳐 나가다

### navigator

- 발음: /ˈnævəˌgeɪtɚ/
- 구성: `navigate` + `-or`
- 뜻: 항해사, 길을 찾는 사람
- 어원적 연결: 이동 경로를 판단하고 방향을 잡는 사람
- 사용 맥락: `a skilled navigator` = 숙련된 항해사

### navigable

- 발음: /ˈnævəgəbəl/
- 구성: `navigate` + `-able`
- 뜻: 항해할 수 있는, 통행 가능한
- 어원적 연결: 배나 사람이 길을 따라 지나갈 수 있음
- 사용 맥락: `a navigable river` = 배가 다닐 수 있는 강

### circumnavigate

- 발음: /ˌsɝkəˈmnævəˌgeɪt/
- 구성: [[circum]] = around + `navigate`
- 뜻: 배나 비행기로 일주하다
- 어원적 연결: 어떤 대상을 둘러 항해함
- 사용 맥락: `circumnavigate the globe` = 세계를 일주하다

### navy

- 발음: /ˈneɪvi/
- 구성: Old French `navie` < Latin `navis`
- 뜻: 해군, 함대
- 어원적 연결: 배들의 집단에서 국가의 해상 군사 조직으로 확장
- 사용 맥락: `join the navy` = 해군에 입대하다

### naval

- 발음: /ˈneɪvəl/
- 구성: Latin `navalis` < `navis`
- 뜻: 해군의, 선박의
- 어원적 연결: 배나 해군에 속함
- 사용 맥락: `a naval base` = 해군 기지

### nautical

- 발음: /ˈnɔtəkəl/
- 구성: Greek `nautikos` < `naus` = ship
- 뜻: 항해의, 선박의
- 어원적 연결: 그리스어의 배 계열. 라틴어 `navis`와 같은 오래된 뿌리로 연결됨
- 사용 맥락: `nautical miles` = 해리

### astronaut

- 발음: /ˈæstrəˌnɑt/
- 구성: [[astr]] = star + Greek `nautes` = sailor
- 뜻: 우주비행사
- 어원적 연결: 별 사이를 항해하는 사람이라는 이미지
- 사용 맥락: `train as an astronaut` = 우주비행사로 훈련하다

### aeronaut

- 구성: Greek `aero-` = air + `nautes` = sailor
- 뜻: 비행사, 항공 탐험가
- 어원적 연결: 공중을 항해하는 사람
- 사용 맥락: 역사적·문어적 표현. 현대 일반어로는 `aviator`, `pilot`이 더 흔함

### nausea

- 발음: /ˈnɔziə/
- 구성: Greek `nausia` < `naus` = ship
- 뜻: 메스꺼움, 구역질
- 어원적 연결: 원래 배멀미와 연결된 느낌에서 일반적인 메스꺼움으로 확장
- 사용 맥락: `feel nausea` = 메스꺼움을 느끼다

## 기억 포인트

`nav` 계열은 "배로 길을 찾아 감" 이미지.

- `navigate`: 배를 몰아 길을 감
- `navigation`: 길 찾기 과정
- `navy`, `naval`: 배를 기반으로 한 해군 계열
- `nautical`, `astronaut`: Greek `naus` 계열의 항해 이미지

## 비교 단어

### aviation

- 뜻: 항공, 비행
- 구분 포인트: `aviation`은 Latin `avis` = 새 계열. `navigation`의 `navis` = 배와 어원이 다름.
