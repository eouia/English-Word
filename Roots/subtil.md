# subtil

- 출발 단어: `subtle`
- 핵심 어원: Latin `subtilis` = fine, thin, delicate, precise
- 핵심 이미지: 너무 가늘고 섬세해 바로 드러나지 않음
- 관련 문서: [[nuance]], [[delic]]
- 주의: `subtle`의 `b`는 발음하지 않음. 현대 형태에서 `sub` + `tle`처럼 직접 분해하지 않음.

## 출발 단어 분석

`subtle` ← Old French `sotil/subtil` ← Latin `subtilis`

- `subtilis`: fine, thin, delicate, precise
- 전통적으로 `sub-` + `tela` = web, fabric와 연결해 "촘촘하고 가늘게 짜인" 이미지로 설명함
- 현대 의미: 미묘한, 포착하기 어려운, 섬세한; 교묘한

의미 변화는 "가늘고 정교함" → "감지하기 어려울 만큼 섬세함" → "미묘한, 정교한, 때로는 교묘한".

## 단어들

### subtle

- 구성: Latin `subtilis` = fine, thin, delicate
- 뜻: 미묘한, 포착하기 어려운, 섬세한; 교묘한
- 어원적 연결: 너무 가늘고 정교해 바로 눈에 띄지 않음
- 사용 맥락: `a subtle difference` = 미묘한 차이, `a subtle hint` = 은근한 암시

### subtly

- 구성: `subtle` + `-ly`
- 뜻: 미묘하게, 은근히, 교묘하게
- 어원적 연결: 겉으로 강하게 드러나지 않는 방식
- 사용 맥락: `subtly change the tone` = 어조를 미묘하게 바꾸다

### subtlety

- 구성: `subtle` + `-ty`
- 뜻: 미묘함, 섬세함; 교묘함
- 어원적 연결: 바로 알아차리기 어려운 세밀한 성질
- 사용 맥락: `miss the subtlety of the argument` = 논증의 미묘함을 놓치다

### subtleness

- 구성: `subtle` + `-ness`
- 뜻: 미묘함, 섬세함
- 어원적 연결: `subtlety`와 비슷하지만 덜 흔함
- 사용 맥락: 문어적·설명적 표현

### subtilize

- 구성: `subtil` + `-ize`
- 뜻: 지나치게 미세하게 구분하다, 세밀하게 논하다
- 어원적 연결: 너무 가늘게 나누어 따짐
- 사용 맥락: 드문 단어. 시험 핵심은 `subtle`, `subtly`, `subtlety`.

### subtile

- 구성: Latin/French `subtilis` 계열의 변형
- 뜻: 미묘한, 섬세한
- 어원적 연결: `subtle`과 같은 계열
- 사용 맥락: 드문 철자. 일반 영어에서는 `subtle`을 씀.

## 기억 포인트

`subtil` 계열은 "가늘고 섬세해 바로 드러나지 않음" 이미지.

- `subtle`: 미묘한, 섬세한
- `subtly`: 은근히, 미묘하게
- `subtlety`: 미묘함, 섬세함
- `subtilize`: 지나치게 세밀하게 구분하다

## 비교 단어

### nuance

- 뜻: 미묘한 차이, 뉘앙스
- 구분 포인트: [[nuance]]는 색조처럼 조금씩 다른 차이. `subtle`은 그 차이가 너무 섬세해 알아차리기 어렵다는 느낌.

### delicate

- 뜻: 섬세한, 연약한, 조심스러운
- 구분 포인트: [[delic]]은 즐거움을 주거나 조심히 다뤄야 할 섬세함. `subtle`은 눈치채기 어려운 미세함과 정교함.

### obvious

- 뜻: 명백한, 뚜렷한
- 구분 포인트: `obvious`는 바로 눈에 띔. `subtle`은 바로 드러나지 않아 주의 깊게 봐야 함.

### sly

- 뜻: 교활한, 은근한
- 구분 포인트: `sly`는 속임수나 은밀한 의도에 초점. `subtle`은 반드시 나쁜 뜻은 아니며, 미묘하고 정교한 표현에도 씀.
