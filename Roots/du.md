# du

- 출발 단어: `duet`
- 핵심 어원: Latin `duo` = two
- 깊은 어근: PIE `*dwo-` = 둘
- 핵심 이미지: 둘, 두 사람이 함께, 둘로 나뉜 상태, 두 마음 사이의 의심
- 관련 문서: [[bi]], [[di]]
- 주의: `bi-`는 접두사로 "둘"을 붙이는 경우가 많고, `du/duo`는 `duet`, `dual`, `duo`처럼 단어 자체에 둘의 이미지가 남은 경우가 많음.

## 출발 단어 분석

`duet` ← Italian `duetto` ← Latin `duo`

- `duo`: two
- Italian `duetto`: 두 사람이 하는 음악
- 원래 이미지: 두 사람이 함께 연주하거나 노래함
- 현대 의미: 2중주, 2중창, 듀엣

## 단어들

### duet

- 구성: Italian `duetto` ← Latin `duo` = two
- 뜻: 듀엣, 2중주, 2중창
- 어원적 연결: 두 사람이 함께 하는 음악
- 사용 맥락: `sing a duet` = 듀엣을 부르다

### duo

- 구성: Latin `duo` = two
- 뜻: 두 사람, 한 쌍, 듀오
- 어원적 연결: 둘로 이루어진 한 묶음
- 사용 맥락: `a comedy duo` = 코미디 듀오

### dual

- 구성: Latin `dualis` = containing two
- 뜻: 이중의, 두 부분으로 된
- 어원적 연결: 두 요소가 함께 있음
- 사용 맥락: `a dual role` = 이중 역할

### duality

- 구성: `dual` + `-ity`
- 뜻: 이중성
- 어원적 연결: 하나 안에 두 성질이나 두 면이 함께 있음
- 사용 맥락: `the duality of human nature` = 인간 본성의 이중성

### double

- 구성: Old French `doble` ← Latin `duplus` = twofold
- 뜻: 두 배의, 이중의, 두 배로 하다
- 어원적 연결: 하나가 두 겹이나 두 배가 됨
- 사용 맥락: `double the amount` = 양을 두 배로 늘리다

### double-check

- 구성: `double` + `check`
- 뜻: 재확인하다
- 어원적 연결: 한 번이 아니라 두 번 확인함
- 사용 맥락: `double-check the answer` = 답을 재확인하다

### duel

- 구성: Medieval Latin `duellum`, Latin `duo`와 연결되어 해석됨
- 뜻: 결투
- 어원적 연결: 두 사람이 맞서는 싸움
- 사용 맥락: `fight a duel` = 결투하다

### duplicate

- 구성: Latin `duplex` = twofold + `plicare` = fold
- 뜻: 복제하다, 사본, 중복의
- 어원적 연결: 둘로 접힌 것 → 하나가 두 벌이 됨
- 사용 맥락: `duplicate a file` = 파일을 복제하다

### duplicity

- 구성: Latin `duplex` = twofold
- 뜻: 이중성, 표리부동, 기만
- 어원적 연결: 겉과 속이 두 겹으로 다름
- 사용 맥락: `political duplicity` = 정치적 표리부동

### duplex

- 구성: Latin `duplex` = twofold
- 뜻: 복층 주택, 이중의
- 어원적 연결: 두 겹·두 부분으로 된 구조
- 사용 맥락: `a duplex apartment` = 복층 아파트

### doubt

- 구성: Old French `douter` ← Latin `dubitare` = waver in opinion
- 뜻: 의심하다, 의심
- 어원적 연결: 두 가능성 사이에서 마음이 흔들림
- 사용 맥락: `I doubt it.` = 나는 그것이 의심스럽다

### dubious

- 구성: Latin `dubius` = wavering, doubtful
- 뜻: 의심스러운, 수상한, 미심쩍은
- 어원적 연결: 두 방향 사이에서 확신하지 못함
- 사용 맥락: `a dubious claim` = 의심스러운 주장

## 기억 포인트

`du` 계열은 "둘, 두 사람이 함께, 두 겹, 두 마음" 이미지.

- `duet`: 두 사람이 하는 음악
- `duo`: 두 사람 한 쌍
- `dual`: 두 부분으로 된
- `double`: 두 배, 두 겹
- `duel`: 두 사람이 맞섬
- `duplicate`: 두 벌로 만듦
- `duplicity`: 두 얼굴, 표리부동
- `doubt`: 두 가능성 사이에서 흔들림

## 비교 단어

### bi

- 뜻: 둘, 두 번, 두 겹
- 구분 포인트: [[bi]]는 Latin 접두사로 `bilingual`, `bicycle`처럼 앞에 붙어 "둘"을 표시함. `du`는 `duo`, `duet`, `dual`처럼 독립 단어 계열에 가까움.

### di

- 뜻: 둘
- 구분 포인트: [[di]]는 Greek 계열. `du/bi`는 Latin 계열.

### plic-fold

- 뜻: 접다, 겹
- 구분 포인트: `duplicate`, `duplicity`는 [[plic-fold]]의 "접다/겹" 이미지도 함께 가짐. `du`는 그중 "둘" 부분을 설명함.
