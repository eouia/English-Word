# render

- 출발 단어: `surrender`
- 핵심 어원: Old French `rendre` = give back, hand over
- 더 깊은 연결: Latin `reddere` = give back, restore
- 핵심 이미지: 되돌려 주거나 내어 줌
- 관련 문서: [[yield]], [[give]], [[cumb]]
- 주의: `render`와 `rend` = 찢다는 철자가 비슷하지만 어원과 뜻이 다름.

## 출발 단어 분석

`surrender` = `sur-` + `render`

- `sur-`: over, above
- `render`: give back, hand over
- 원래 이미지: 권리나 물건을 상대 위로/상대에게 넘겨줌
- 현대 의미: 항복하다, 넘겨주다, 포기하다

`surrender to pressure`는 버티던 통제권이나 입장을 상대 힘에 넘기는 이미지.

## 단어들

### render

- 구성: Old French `rendre` = give back, hand over
- 뜻: 제공하다, 만들다, 표현하다, 번역하다
- 어원적 연결: 어떤 결과나 형태를 밖으로 내어 줌
- 사용 맥락: `render assistance` = 도움을 제공하다, `render a scene` = 장면을 렌더링하다

### rendering

- 구성: `render` + `-ing`
- 뜻: 번역, 표현, 묘사; 렌더링
- 어원적 연결: 내용을 다른 형태로 내어 놓음
- 사용 맥락: `an English rendering of the poem` = 그 시의 영어 번역

### rendition

- 구성: `render` 계열 명사형
- 뜻: 연주, 표현, 번역; 인도
- 어원적 연결: 어떤 작품이나 대상을 특정 방식으로 내어 놓음
- 사용 맥락: `a moving rendition of the song` = 감동적인 노래 연주

### surrender

- 구성: Old French `surrendre` = give up, deliver over
- 뜻: 항복하다, 넘겨주다, 포기하다; 항복
- 어원적 연결: 권리·통제권·자신을 상대에게 완전히 넘김
- 사용 맥락: `surrender the city` = 도시를 넘겨주다, `surrender to the enemy` = 적에게 항복하다

### surrendered

- 구성: `surrender` + `-ed`
- 뜻: 항복한, 넘겨준
- 어원적 연결: 더 이상 붙잡지 않고 상대에게 넘긴 상태
- 사용 맥락: `surrendered weapons` = 넘겨진 무기

### surrendering

- 구성: `surrender` + `-ing`
- 뜻: 항복하는, 포기하는
- 어원적 연결: 버티던 것을 넘겨주는 과정
- 사용 맥락: `surrendering control` = 통제권을 넘겨줌

## 기억 포인트

`render` 계열은 "내어 주거나 다른 형태로 내놓음" 이미지.

- `render`: 제공하다, 표현하다, 번역하다
- `rendering`: 표현, 번역, 렌더링
- `rendition`: 연주, 표현, 번역
- `surrender`: 완전히 넘겨주다, 항복하다

## 비교 단어

### yield

- 뜻: 내놓다, 양보하다, 굴복하다
- 구분 포인트: [[yield]]는 결과나 권리를 내놓는 넓은 말. `surrender`는 저항을 멈추고 통제권을 완전히 넘기는 느낌이 강함.

### give

- 뜻: 주다
- 구분 포인트: [[give]]는 가장 일반적인 "주다". `render`는 격식 있게 제공하거나 결과를 내놓는 느낌, `surrender`는 항복·포기의 느낌.

### succumb

- 뜻: 굴복하다, 병이나 유혹에 지다
- 구분 포인트: [[cumb]]의 `succumb`는 압력 아래 무너짐. `surrender`는 의식적으로 넘겨주거나 항복하는 행위에 초점.

### submit

- 뜻: 복종하다, 제출하다
- 구분 포인트: `submit`은 자신이나 문서를 권위 아래에 둠. `surrender`는 더 강하게 통제권·무기·저항을 포기함.
