# cid

- 출발 단어: `precise`
- 핵심 어원: 라틴어 `caedere` = 자르다, 치다
- 핵심 의미: 잘라 내다, 경계를 내다, 끊어 결정하다
- 주요 형태: `cid`, `cis`, `cise`, `cision`, `cide`
- 관련 문서: Latin `secare`, [[rupt]], [[pact]]
- 주의: `cad/cas` = 떨어지다 계열과 철자가 비슷하지만 다른 어원.

## 출발 단어 분석

`precise` ← Latin `praecisus` ← `praecidere`

- `prae-`: before, in front
- `caedere`: 자르다
- 원래 이미지: 앞부분이나 불필요한 부분을 잘라 냄
- 의미 확장: 군더더기를 잘라냄 → 경계가 정확함 → 정밀함

## 단어들

### precise

- 발음: /prɪˈsaɪs/
- 구성: `pre-` = before + `cis`
- 뜻: 정확한, 정밀한
- 어원적 연결: 불필요한 부분을 잘라 경계를 분명히 함
- 사용 맥락: `a precise definition` = 정확한 정의

### precision

- 발음: /priˈsɪʒən/
- 구성: `pre-` + `cision`
- 뜻: 정밀성, 정확도
- 어원적 연결: 핵심만 남기도록 잘라 낸 상태
- 사용 맥락: `measure with precision` = 정밀하게 측정하다

### precisely

- 발음: /prɪˈsaɪsli/
- 구성: `precise` + `-ly`
- 뜻: 정확히, 바로
- 어원적 연결: 잘라 낸 듯 경계가 분명하게 맞음
- 사용 맥락: `precisely at noon` = 정확히 정오에

### concise

- 발음: /kəˈnsaɪs/
- 구성: `con-` = completely + `cise`
- 뜻: 간결한
- 어원적 연결: 불필요한 말을 잘라 내어 짧고 분명함
- 사용 맥락: `a concise summary` = 간결한 요약

### concision

- 구성: `con-` + `cision`
- 뜻: 간결함
- 어원적 연결: 군더더기를 잘라 낸 표현
- 사용 맥락: `write with concision` = 간결하게 쓰다

### incise

- 발음: /ˌɪˈnsaɪz/
- 구성: `in-` = into + `cise`
- 뜻: 새기다, 절개하다
- 어원적 연결: 안으로 칼자국을 내며 자름
- 사용 맥락: `incise a pattern` = 무늬를 새기다

### incision

- 발음: /ˌɪˈnsɪʒən/
- 구성: `in-` + `cision`
- 뜻: 절개, 절개 부위
- 어원적 연결: 몸이나 표면 안으로 낸 칼자국
- 사용 맥락: `make an incision` = 절개하다

### incisive

- 발음: /ˌɪˈnsaɪsɪv/
- 구성: `incise` + `-ive`
- 뜻: 날카로운, 예리한
- 어원적 연결: 칼로 자르듯 핵심을 파고듦
- 사용 맥락: `incisive criticism` = 예리한 비판

### decision

- 발음: /dɪˈsɪʒən/
- 구성: `de-` = off + `cision`
- 뜻: 결정
- 어원적 연결: 여러 가능성에서 하나를 잘라 떼어 확정함
- 사용 맥락: `make a decision` = 결정을 내리다

### decide

- 발음: /ˌdɪˈsaɪd/
- 구성: `de-` = off + `cide`
- 뜻: 결정하다
- 어원적 연결: 망설임이나 다른 선택지를 잘라 내고 확정함
- 사용 맥락: `decide quickly` = 빨리 결정하다

### decisive

- 발음: /dɪˈsaɪsɪv/
- 구성: `decide` + `-ive`
- 뜻: 결정적인, 결단력 있는
- 어원적 연결: 애매함을 잘라 내고 결과를 확정함
- 사용 맥락: `a decisive victory` = 결정적 승리

### excise

- 발음: /ɛˈksaɪs/
- 구성: `ex-` = out + `cise`
- 뜻: 잘라 내다, 삭제하다
- 어원적 연결: 밖으로 도려내어 제거함
- 사용 맥락: `excise a passage` = 한 구절을 삭제하다

### excision

- 발음: /ɪˈksɪʒən/
- 구성: `ex-` + `cision`
- 뜻: 절제, 삭제
- 어원적 연결: 잘라 내어 없앤 행위
- 사용 맥락: `surgical excision` = 외과적 절제

### scissors

- 발음: /ˈsɪzɚz/
- 구성: Old French `cisoires` ← Latin `caedere`
- 뜻: 가위
- 어원적 연결: 자르는 도구
- 사용 맥락: `a pair of scissors` = 가위 한 자루

### -cide

- 구성: Latin `caedere`의 결합형
- 뜻: 죽이는 것, 살해
- 어원적 연결: 원래 “치다, 자르다”에서 “죽이다”로 확장
- 사용 맥락: `homicide`, `suicide`, `pesticide`

## 기억 포인트

- `cid/cis`는 “자르다”.
- `precise`: 잘라서 경계를 정확히 함.
- `concise`: 군더더기를 잘라 짧게 함.
- `decision`: 선택지를 잘라 내고 하나로 정함.
- `incision`: 안으로 낸 절개.
- `-cide`: 치거나 잘라 죽임.

## 비교 단어

### cad / cas

- `cadere` = 떨어지다 계열.
- `accident`, `occasion`, `cadence` 같은 단어와 관련될 수 있음.
- `caedere` = 자르다, 치다 와 다른 어원.

### sect

- `sect`는 라틴어 `secare` = 자르다 계열.
- `section`, `dissect` 같은 단어가 이쪽.
- `cid/cis`와 의미는 비슷하지만 어원은 다름.

### exact

- `exact`는 라틴어 `exigere` = 몰아내다, 요구하다, 측정하다 계열.
- `precise`와 뜻은 가깝지만 `caedere` 계열은 아님.
