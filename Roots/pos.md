# pos

- 출발 단어: `position`
- 핵심 어원: 라틴어 `ponere` / `posit-` = 놓다, 두다
- 핵심 이미지: 어떤 자리에 놓음
- 관련해서 헷갈릴 수 있는 어근: [[set]] = 놓다, 정하다 계열, [[present]] = 앞에 놓아 보이다 계열

## 출발 단어 분석

`position`은 Latin `positio`에서 온 말.

- `ponere`: 놓다, 두다
- `posit-`: 놓인

따라서 `position`은 "놓인 자리" → "위치, 입장, 직위"의 뜻.

## 단어들

### position

- 발음: /pəˈzɪʃən/
- 구성: `posit` + `-ion`
- 뜻: 위치, 입장, 직위
- 어원적 연결: 어떤 자리에 놓인 상태
- 사용 맥락: `take a position` = 입장을 취하다

### positive

- 발음: /ˈpɑzətɪv/
- 구성: Latin `positivus` = formally laid down
- 뜻: 긍정적인, 확실한, 양성의
- 어원적 연결: 확실히 놓아 정해진 것
- 사용 맥락: `a positive result` = 양성 결과

### compose

- 발음: /kəˈmpoʊz/
- 구성: `com-` = together + `pose`
- 뜻: 구성하다, 작곡하다, 진정시키다
- 어원적 연결: 함께 놓아 배열함
- 사용 맥락: `compose a poem` = 시를 짓다

### composition

- 발음: /ˌkɑmpəˈzɪʃən/
- 구성: `com-` + `posit` + `-ion`
- 뜻: 구성, 작문, 작품
- 어원적 연결: 여러 요소를 함께 놓아 만든 것
- 사용 맥락: `chemical composition` = 화학적 구성

### propose

- 발음: /prəˈpoʊz/
- 구성: `pro-` = forward + `pose`
- 뜻: 제안하다, 청혼하다
- 어원적 연결: 생각을 앞으로 내놓음
- 사용 맥락: `propose a solution` = 해결책을 제안하다

### proposal

- 발음: /prəˈpoʊzəl/
- 구성: `propose` + `-al`
- 뜻: 제안, 청혼
- 어원적 연결: 앞으로 놓인 생각
- 사용 맥락: `a research proposal` = 연구 제안서

### proponent

- 발음: /prəˈpoʊnənt/
- 구성: `pro-` = forward, in favor + Latin `ponent-` ← `ponere`
- 뜻: 지지자, 제안자
- 어원적 연결: 어떤 의견이나 방안을 앞에 내놓고 지지하는 사람
- 사용 맥락: `a proponent of renewable energy` = 재생에너지 지지자

### oppose

- 발음: /əˈpoʊz/
- 구성: `ob-` = against + `pose`
- 뜻: 반대하다
- 어원적 연결: 맞은편에 놓임
- 사용 맥락: `oppose the plan` = 계획에 반대하다

### opposition

- 발음: /ˌɑpəˈzɪʃən/
- 구성: `ob-` + `posit` + `-ion`
- 뜻: 반대, 야당
- 어원적 연결: 맞서 놓인 상태
- 사용 맥락: `political opposition` = 정치적 반대 세력

### expose

- 발음: /ɪˈkspoʊz/
- 구성: `ex-` = out + `pose`
- 뜻: 드러내다, 노출하다
- 어원적 연결: 밖에 놓아 보이게 함
- 사용 맥락: `expose corruption` = 부패를 폭로하다

### deposit

- 발음: /dəˈpɑzɪt/
- 구성: `de-` = down + `posit`
- 뜻: 예금하다, 보증금, 침전물
- 어원적 연결: 아래에 내려놓음
- 사용 맥락: `make a deposit` = 예금하다

### impose

- 발음: /ˌɪˈmpoʊz/
- 구성: `in-` = on + `pose`
- 뜻: 부과하다, 강요하다
- 어원적 연결: 위에 얹어 놓음
- 사용 맥락: `impose a tax` = 세금을 부과하다

### dispose

- 발음: /dɪˈspoʊz/
- 구성: `dis-` = apart + `pose`
- 뜻: 처리하다, 배치하다, ~하게 만들다
- 어원적 연결: 여기저기 나누어 놓음
- 사용 맥락: `dispose of waste` = 폐기물을 처리하다

### suppose

- 발음: /səˈpoʊz/
- 구성: `sub-` = under + `pose`
- 뜻: 가정하다, 생각하다
- 어원적 연결: 아래에 놓아 전제로 삼음
- 사용 맥락: `suppose it rains` = 비가 온다고 가정하자

### compound

- 발음: n./adj. /ˈkɑmpaʊnd/, v. /kəˈmpaʊnd/
- 구성: `com-` = together + `ponere`
- 뜻: 혼합물, 복합의, 악화시키다
- 어원적 연결: 함께 놓아 섞음
- 사용 맥락: `compound interest` = 복리

## 기억 포인트

`pos` 계열은 "놓다" 이미지.

- `position`: 놓인 자리
- `compose`: 함께 놓아 구성함
- `propose`: 앞으로 내놓음
- `proponent`: 의견을 앞에 내놓고 지지하는 사람
- `oppose`: 맞은편에 놓임
- `expose`: 밖에 놓아 드러냄
- `deposit`: 아래에 내려놓음
- `suppose`: 아래에 놓아 가정함

## 비교 단어

### set

- 뜻: 놓다, 정하다
- 구분 포인트: [[set]]은 게르만계 기본어. `pos`는 라틴계 단어에서 위치, 배치, 제안, 반대 같은 추상어로 많이 나타남

### present

- 뜻: 앞에 놓아 보이다
- 구분 포인트: [[present]]는 "앞에 있음/앞에 내놓음"에 초점. `propose`, `expose`는 생각이나 사실을 특정 위치에 놓는 느낌

### postulate

- 뜻: 가정하다, 전제로 삼다
- 구분 포인트: [[postul]] 계열. `posit`, `suppose`는 "전제로 놓다" 이미지가 강하고, `postulate`는 "받아들이라고 요구하다" 이미지가 강함.
