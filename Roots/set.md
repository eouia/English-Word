# set

- 출발 단어: `upset`, `setback`
- 핵심 어원: 고대 영어 `settan` = 놓다, 앉히다
- 핵심 이미지: 어떤 위치나 상태에 놓음
- 비교: [[sed]] = 앉다, 자리 잡다 계열

## 출발 단어 분석

`set`은 [[sed]]와 같은 큰 뿌리에서 나온 말.

- [[sed]]: 앉다, 자리 잡다
- `set`: 앉게 하다, 놓다, 고정하다

따라서 `set` 계열은 "어떤 위치나 상태에 놓음" 이미지.

## 단어들

### set

- 구성: 고대 영어 `settan`
- 뜻: 놓다, 정하다, 고정하다
- 어원적 연결: 어떤 것을 특정 위치나 상태에 둠
- 사용 맥락: `set a goal` = 목표를 정하다

### upset

- 구성: `up` + `set`
- 뜻: 속상하게 하다, 뒤엎다, 이변
- 어원적 연결: 원래 "위로 세움, 세워 놓음" 계열. 이후 "뒤집어 놓음" → "마음을 뒤흔듦"
- 사용 맥락: `feel upset` = 속상하다, `an upset victory` = 이변의 승리

### setback

- 구성: `set` + `back`
- 뜻: 차질, 좌절, 후퇴
- 어원적 연결: 진행하던 것을 뒤쪽 위치로 밀어 놓음
- 사용 맥락: `a major setback` = 큰 차질

### setup

- 구성: `set` + `up`
- 뜻: 설치, 구성, 상황, 함정
- 어원적 연결: 필요한 상태로 세팅해 둠
- 사용 맥락: `a simple setup` = 간단한 구성

### setting

- 구성: `set` + `-ing`
- 뜻: 환경, 배경, 설정
- 어원적 연결: 어떤 일이 놓인 자리나 조건
- 사용 맥락: `in a formal setting` = 공식적인 환경에서

### settle

- 구성: `set` 계열 + `-le`
- 뜻: 정착하다, 해결하다, 가라앉다
- 어원적 연결: 흔들리던 것이 한 자리에 놓임
- 사용 맥락: `settle the issue` = 문제를 해결하다

### settlement

- 구성: `settle` + `-ment`
- 뜻: 정착지, 합의, 해결
- 어원적 연결: 사람이 자리 잡은 곳 또는 문제가 안정된 상태
- 사용 맥락: `reach a settlement` = 합의에 도달하다

### outset

- 구성: `out` + `set`
- 뜻: 시작
- 어원적 연결: 밖으로 놓고 출발하는 지점
- 사용 맥락: `from the outset` = 처음부터

### onset

- 구성: `on` + `set`
- 뜻: 시작, 발병
- 어원적 연결: 어떤 일이 덮쳐 오듯 시작됨
- 사용 맥락: `the onset of symptoms` = 증상의 시작

### offset

- 구성: `off` + `set`
- 뜻: 상쇄하다, 벌충하다, 상쇄 효과
- 어원적 연결: 한쪽에 놓인 것을 반대쪽에 놓아 균형을 맞춤
- 사용 맥락: `offset the cost` = 비용을 상쇄하다

### reset

- 구성: `re-` + `set`
- 뜻: 다시 설정하다
- 어원적 연결: 원래 위치나 새 상태로 다시 놓음
- 사용 맥락: `reset a password` = 비밀번호를 재설정하다

## 기억 포인트

`set` 계열은 "위치나 상태에 놓음" 이미지.

- `set`: 놓음, 정함
- `upset`: 뒤집어 놓음, 마음을 흔듦
- `setback`: 뒤로 밀어 놓음
- `setup`: 세팅해 둠
- `setting`: 놓인 환경
- `settle`: 자리를 잡음
- `outset`: 시작점
- `onset`: 일이 시작됨
- `offset`: 반대쪽에 놓아 상쇄함
- `reset`: 다시 놓음
