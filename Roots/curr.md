# curr

- 출발 단어: `occur`
- 핵심 어원: 라틴어 `currere` / `curs-` = 달리다, 흐르다
- 핵심 이미지: 어떤 방향으로 달려가거나 흐름
- 관련해서 헷갈릴 수 있는 어근: [[cess]] = 가다, 물러나다 계열

## 출발 단어 분석

`occur`는 `ob-` + `currere`.

- `ob-`: toward, against, in the way
- `currere`: 달리다, 흐르다

따라서 `occur`는 원래 "마주 달려오다, 앞에 나타나다" → "일어나다, 떠오르다"의 뜻.

## 단어들

### occur

- 발음: /əˈkɝ/
- 구성: `ob-` = toward, in the way + `curr`
- 뜻: 일어나다, 발생하다, 떠오르다
- 어원적 연결: 어떤 일이 앞에 달려와 나타남. 생각이 마음에 떠오름
- 사용 맥락: `An accident occurred.` = 사고가 발생했다

### occurrence

- 발음: /əˈkɝəns/
- 구성: `occur` + `-ence`
- 뜻: 발생, 사건, 일어난 일
- 어원적 연결: 앞에 나타나 일어난 것
- 사용 맥락: `a rare occurrence` = 드문 일

### occasion

- 발음: /əˈkeɪʒən/
- 구성: Latin `occasio` = falling toward, opportunity
- 뜻: 경우, 때, 행사, 계기
- 어원적 연결: 직접 어근은 Latin `cadere` = 떨어지다 쪽이지만, 현대 의미는 `occur` / `occurrence`와 가까움. 앞에 생겨난 특정한 때나 기회
- 사용 맥락: `on this occasion` = 이번 경우에

### recur

- 발음: /rɪˈkɝ/
- 구성: `re-` = again, back + `curr`
- 뜻: 되풀이되다, 재발하다
- 어원적 연결: 다시 달려와 나타남
- 사용 맥락: `recurring problems` = 되풀이되는 문제

### recurrent

- 발음: /rɪˈkɝənt/
- 구성: `recur` + `-ent`
- 뜻: 반복되는, 재발하는
- 어원적 연결: 계속 다시 나타남
- 사용 맥락: `recurrent symptoms` = 반복되는 증상

### current

- 발음: /ˈkɝənt/
- 구성: Latin `currens`
- 뜻: 현재의, 흐름, 전류
- 어원적 연결: 지금 흐르고 있는 것
- 사용 맥락: `current events` = 시사, 현재 일어나는 사건들

### currency

- 발음: /ˈkɝənsi/
- 구성: `current` + `-cy`
- 뜻: 통화, 유통, 통용
- 어원적 연결: 사람들 사이를 흐르며 오가는 돈이나 말
- 사용 맥락: `foreign currency` = 외화

### curriculum

- 발음: /kɚˈɪkjələm/
- 구성: Latin `curriculum` = race course
- 뜻: 교육과정, 커리큘럼
- 어원적 연결: 학습자가 달려가는 과정이나 경로
- 사용 맥락: `school curriculum` = 학교 교육과정

### course

- 발음: /ˈkɔrs/
- 구성: Latin `cursus`
- 뜻: 과정, 강좌, 진로, 코스
- 어원적 연결: 달려가는 길이나 흐름
- 사용 맥락: `take a course` = 강좌를 듣다

### discourse

- 발음: /ˈdɪskɔrs/
- 구성: `dis-` = apart + `course`
- 뜻: 담론, 말하다
- 어원적 연결: 생각이나 말이 이리저리 달려 나감. 주제를 따라 전개되는 말
- 사용 맥락: `public discourse` = 공적 담론

### excursion

- 발음: /ɪˈkskɝʒən/
- 구성: `ex-` = out + `curs`
- 뜻: 짧은 여행, 소풍
- 어원적 연결: 밖으로 달려 나감
- 사용 맥락: `a weekend excursion` = 주말 소풍

### incur

- 발음: /ˌɪˈnkɝ/
- 구성: `in-` = into + `curr`
- 뜻: 초래하다, 부담하다
- 어원적 연결: 어떤 비용이나 위험 안으로 달려 들어감
- 사용 맥락: `incur costs` = 비용을 부담하다

### concurrent

- 발음: /kəˈnkɝənt/
- 구성: `con-` = together + `curr`
- 뜻: 동시에 일어나는, 병행하는
- 어원적 연결: 함께 달려감
- 사용 맥락: `concurrent events` = 동시에 일어나는 사건들

### concur

- 발음: /kəˈnkɝ/
- 구성: `con-` = together + `curr`
- 뜻: 동의하다, 동시에 일어나다
- 어원적 연결: 생각이 함께 달려 같은 방향으로 감
- 사용 맥락: `I concur with your opinion.` = 당신 의견에 동의한다

### precursor

- 발음: /priˈkɝsɚ/
- 구성: `pre-` = before + `cursor`
- 뜻: 선구자, 전조
- 어원적 연결: 앞서 달려가는 것
- 사용 맥락: `a precursor to modern computers` = 현대 컴퓨터의 선구

### cursor

- 발음: /ˈkɝsɚ/
- 구성: Latin `cursor` = runner
- 뜻: 커서, 지시자
- 어원적 연결: 화면 위를 달리듯 움직이는 표시
- 사용 맥락: `move the cursor` = 커서를 움직이다

## 기억 포인트

`curr` 계열은 "달리다 / 흐르다" 이미지.

- `occur`: 일이 앞에 달려와 나타남
- `occasion`: 앞에 생겨난 특정한 때나 계기
- `recur`: 다시 달려와 반복됨
- `current`: 지금 흐르는 것
- `currency`: 사람들 사이를 흐르는 돈
- `course`: 달려가는 길이나 과정
- `discourse`: 말이 주제를 따라 흘러감
- `incur`: 비용이나 위험 안으로 들어감
- `concur`: 함께 달려 같은 의견이 됨
- `precursor`: 앞서 달리는 것

## 비교 단어

### happen

- 뜻: 일어나다
- 구분 포인트: [[hap]] 계열. `happen`은 일상적인 "일이 생기다". `occur`는 더 격식 있고, "사건이 나타남" 또는 "생각이 떠오름"에 잘 씀

### cess

- 뜻: 가다, 물러나다
- 구분 포인트: [[cess]]는 이동/물러남 이미지. `curr`는 더 빠르게 달리거나 흐르는 이미지
