# morph

- 출발 단어: `metamorphosis`
- 핵심 어원: Greek `morphe` = 형태, 모양
- 핵심 이미지: 겉으로 드러나는 형태와 그 변화
- 관련 문서: [[form]], [[gram]]
- 주의: `form`은 라틴어 계열의 형태, `morph`는 그리스어 계열의 형태.

## 출발 단어 분석

`metamorphosis` ← Greek `meta-` + `morphe`

- `meta-`: change, beyond
- `morphe`: form
- 원래 이미지: 형태가 바뀜
- 현대 의미: 변태, 변형

## 단어들

### morphology

- 구성: `morpho-` = form + [[logos]] = study
- 뜻: 형태론, 형태학
- 어원적 연결: 형태와 구조를 연구함
- 사용 맥락: `linguistic morphology` = 언어 형태론

### metamorphosis

- 구성: `meta-` = change + `morphe` = form
- 뜻: 변태, 변형, 탈바꿈
- 어원적 연결: 형태가 크게 바뀜
- 사용 맥락: `the metamorphosis of a caterpillar` = 애벌레의 변태

### amorphous

- 구성: `a-` = without + `morphe`
- 뜻: 무정형의, 형태가 뚜렷하지 않은
- 어원적 연결: 분명한 형태가 없음
- 사용 맥락: `an amorphous mass` = 형태 없는 덩어리

### morpheme

- 구성: Greek `morphe`
- 뜻: 형태소
- 어원적 연결: 의미를 지닌 가장 작은 형태 단위
- 사용 맥락: 언어학에서 단어 구조를 설명할 때 쓰임

### anthropomorphic

- 구성: Greek `anthropos` = human + `morphe`
- 뜻: 의인화된, 인간 형태의
- 어원적 연결: 인간의 형태를 가짐
- 사용 맥락: `anthropomorphic animals` = 의인화된 동물

### polymorphic

- 구성: `poly-` = many + `morphe`
- 뜻: 다형의
- 어원적 연결: 여러 형태를 가짐
- 사용 맥락: 생물학·컴퓨터 과학 맥락에서 쓰임

## 기억 포인트

`morph` 계열은 "형태와 형태 변화" 이미지.

- `morphology`: 형태 연구
- `metamorphosis`: 형태 변화
- `amorphous`: 형태가 없음
- `morpheme`: 의미를 지닌 형태 단위
- `anthropomorphic`: 인간 형태의

## 비교 단어

### form

- 뜻: 모양, 형식
- 구분 포인트: [[form]]은 라틴어 형태 계열. `morph`는 그리스어 형태 계열.

### gram

- 뜻: 쓰인 것, 글자
- 구분 포인트: [[gram]]은 기록된 글자와 단위. `morph`는 형태와 구조.
