# ag

- 출발 단어: `ambiguous`
- 핵심 어원: 라틴어 `agere` = 몰다, 이끌다, 움직이다, 하다
- 핵심 의미: 움직이게 함, 몰고 감, 행함
- 주요 형태: `ag`, `act`, `ig`, `eg`
- 관련 문서: [[ambi]], [[ex]], [[in]], [[con]], [[cid]]
- 주의: `ambiguous`의 `-ig-`는 `agere`의 변형. 접두사 `ambi-`는 [[ambi]]에서 따로 정리. `ambivalent`는 `valere` 계열이라 본문 목록에 섞지 않음.

## 출발 단어 분석

`ambiguous` = `ambi-` + `agere`

- `ambi-`: both, around, on both sides
- `agere`: 몰다, 움직이다
- 원래 이미지: 양쪽으로 이끌려 왔다 갔다 함
- 의미 확장: 한쪽으로 정해지지 않음 → 두 가지 이상으로 해석 가능함

## 단어들

### ambiguous

- 구성: `ambi-` = both sides + `ig` < `agere`
- 뜻: 모호한, 여러 뜻으로 해석될 수 있는
- 어원적 연결: 의미가 한쪽으로 정해지지 않고 양쪽으로 움직임
- 사용 맥락: `an ambiguous statement` = 모호한 진술

### ambiguity

- 구성: `ambiguous` + `-ity`
- 뜻: 모호함, 중의성
- 어원적 연결: 의미가 여러 방향으로 끌리는 상태
- 사용 맥락: `avoid ambiguity` = 모호함을 피하다

### agent

- 구성: Latin `agens` < `agere`
- 뜻: 행위자, 대리인, 작용 물질
- 어원적 연결: 어떤 일을 실제로 움직여 수행하는 것
- 사용 맥락: `a travel agent` = 여행사 직원, `cleaning agent` = 세정제

### agency

- 구성: `agent` + `-cy`
- 뜻: 대리점, 기관, 행위 능력
- 어원적 연결: 일을 움직여 처리하는 조직이나 능력
- 사용 맥락: `a government agency` = 정부 기관

### agenda

- 구성: Latin `agenda` = things to be done
- 뜻: 의제, 해야 할 일 목록
- 어원적 연결: 앞으로 행해야 할 일들
- 사용 맥락: `set the agenda` = 의제를 정하다

### act

- 구성: Latin `actus` < `agere`
- 뜻: 행동하다, 행위, 법령
- 어원적 연결: 움직여 실제로 행함
- 사용 맥락: `act responsibly` = 책임 있게 행동하다

### action

- 구성: `act` + `-ion`
- 뜻: 행동, 조치
- 어원적 연결: 실제로 움직여 행한 것
- 사용 맥락: `take action` = 조치를 취하다

### active

- 구성: `act` + `-ive`
- 뜻: 활동적인, 능동적인
- 어원적 연결: 움직여 행하는 성질
- 사용 맥락: `an active role` = 적극적인 역할

### inactive

- 구성: `in-` = not + `active`
- 뜻: 활동하지 않는, 비활성의
- 어원적 연결: 움직여 행동하는 상태가 아님
- 사용 맥락: `an inactive account` = 비활성 계정, `remain inactive` = 활동하지 않은 채 남다

### inactivity

- 구성: `inactive` + `-ity`
- 뜻: 비활동, 무활동, 비활성 상태
- 어원적 연결: 움직임이나 행위가 없는 상태
- 사용 맥락: `a period of inactivity` = 비활동 기간

### activate

- 구성: `active` + `-ate`
- 뜻: 활성화하다, 작동시키다
- 어원적 연결: 움직여 작용하는 상태로 만듦
- 사용 맥락: `activate a device` = 장치를 작동시키다

### activation

- 구성: `activate` + `-ion`
- 뜻: 활성화, 작동 개시
- 어원적 연결: 움직여 작용하게 되는 과정
- 사용 맥락: `account activation` = 계정 활성화

### actor

- 구성: `act` + `-or`
- 뜻: 배우, 행위자
- 어원적 연결: 행동하거나 연기하는 사람
- 사용 맥락: `a lead actor` = 주연 배우

### react

- 구성: `re-` = back + `act`
- 뜻: 반응하다
- 어원적 연결: 자극에 되돌아 움직여 행동함
- 사용 맥락: `react quickly` = 빠르게 반응하다

### counteract

- 구성: `counter-` = against + `act`
- 뜻: 대응하여 약화시키다, 상쇄하다
- 어원적 연결: 어떤 작용에 맞서 반대 행동을 해 효과를 줄임
- 사용 맥락: `counteract the effects` = 효과를 상쇄하다

### counteraction

- 구성: `counteract` + `-ion`
- 뜻: 반작용, 상쇄 작용
- 어원적 연결: 반대쪽으로 움직여 기존 작용을 누름
- 사용 맥락: 문어적·과학적 맥락

### counteractive

- 구성: `counteract` + `-ive`
- 뜻: 상쇄하는, 반작용의
- 어원적 연결: 어떤 작용을 반대로 움직여 약화시키는 성질
- 사용 맥락: 드문 표현

### interact

- 구성: `inter-` = between + `act`
- 뜻: 상호작용하다
- 어원적 연결: 서로 사이에서 행동이 오감
- 사용 맥락: `interact with users` = 사용자와 상호작용하다

### transact

- 구성: `trans-` = across + `act`
- 뜻: 거래하다, 처리하다
- 어원적 연결: 일을 사이에서 움직여 끝까지 처리함
- 사용 맥락: `transact business` = 업무를 처리하다

### exact

- 구성: `ex-` = out + `act`
- 뜻: 정확한; 요구하다, 받아내다
- 어원적 연결: 원래는 밖으로 몰아내거나 끝까지 밀어붙임 → 엄밀하게 맞춤
- 사용 맥락: `an exact number` = 정확한 수

### exactitude

- 구성: `exact` + `-itude`
- 뜻: 정확함, 정밀함
- 어원적 연결: 끝까지 밀어붙여 빈틈없이 맞춘 상태
- 사용 맥락: `with scientific exactitude` = 과학적 정확성으로

### exacting

- 구성: `exact` + `-ing`
- 뜻: 엄격한, 까다로운, 많은 노력을 요구하는
- 어원적 연결: 높은 기준을 끝까지 요구함
- 사용 맥락: `an exacting standard` = 엄격한 기준

### inexact

- 구성: `in-` = not + `exact`
- 뜻: 부정확한, 엄밀하지 않은
- 어원적 연결: 끝까지 정확히 맞춘 상태가 아님
- 사용 맥락: `an inexact estimate` = 부정확한 추정치

### agile

- 구성: Latin `agilis` < `agere`
- 뜻: 민첩한, 기민한
- 어원적 연결: 빠르게 움직일 수 있음
- 사용 맥락: `an agile team` = 기민한 팀

### agitate

- 구성: Latin `agitare` = keep in motion
- 뜻: 동요시키다, 선동하다
- 어원적 연결: 계속 흔들고 움직이게 함
- 사용 맥락: `agitate the public` = 대중을 선동하다

### mitigate

- 구성: Latin `mitigare` = soften, make mild
- 뜻: 완화하다
- 어원적 연결: 현대 형태는 `-gate`처럼 보이지만 `agere` 직접 계열로 외우기보다 별도 주의
- 사용 맥락: `mitigate risk` = 위험을 완화하다

## 기억 포인트

- `ag/act`는 “움직이다, 하다”.
- `agent`: 일을 움직이는 사람/것.
- `agenda`: 해야 할 일.
- `ambiguous`: 의미가 양쪽으로 움직여 모호함.
- `act/action/active`: 실제 행동.
- `inactive`: 움직여 활동하지 않음.
- `agile`: 잘 움직임.

## 비교 단어

### ambivalent

- `ambivalent` = `ambi-` + [[val]].
- 두 가치나 감정이 동시에 있음.
- `ambiguous`와 같이 `ambi-`가 있지만 뒷어원이 다름.
- 접두사 정리: [[ambi]]

### ambition

- `ambition`은 Latin `ambire` = around + go 계열.
- 고대 로마에서 표를 얻으려고 돌아다니는 이미지.
- `agere` 직접 계열로 섞기보다 `ambi-` 관련 비교어로 기억.
- 접두사 정리: [[ambi]]

### cad / cid

- [[cid]]는 `caedere` = 자르다.
- `agere`와 철자·뜻이 다르지만 `exact`처럼 의미상 정밀함으로 만나는 단어가 있음.

### act vs fac

- [[fac]]는 만들다, 하다.
- `act`는 “움직여 행동함”, `fac`는 “만들어 결과를 냄”에 더 가까움.
