# sequ

- 출발 단어: `sequence`
- 핵심 어원: Latin `sequi` = 따라가다
- 관련 형태: Latin `secutus` = followed
- 핵심 이미지: 앞선 것을 따라 이어짐
- 관련 문서: [[cess]], [[duc]], [[curr]]
- 주의: `execute`, `prosecute`, `persecute`는 형태가 `secut`으로 바뀐 계열.

## 출발 단어 분석

`sequence` ← Latin `sequentia`

- `sequi`: follow
- `sequentia`: a following
- 원래 이미지: 하나가 다른 것 뒤를 따라 이어짐
- 현대 의미: 순서, 연속

## 단어들

### sequence

- 구성: Latin `sequi`
- 뜻: 순서, 연속
- 어원적 연결: 뒤따라 이어지는 배열
- 사용 맥락: `a sequence of events` = 일련의 사건

### sequential

- 구성: `sequence` + `-ial`
- 뜻: 순차적인
- 어원적 연결: 차례대로 뒤따라 이어짐
- 사용 맥락: `sequential order` = 순차적 순서

### consequence

- 구성: `con-` = together, after + `sequi`
- 뜻: 결과, 중요성
- 어원적 연결: 어떤 일 뒤를 따라오는 것
- 사용 맥락: `serious consequences` = 심각한 결과

### subsequent

- 구성: `sub-` = after, under + Latin `sequi`
- 뜻: 그다음의, 이후의
- 어원적 연결: 앞선 일 뒤를 따라옴
- 사용 맥락: `subsequent research` = 이후의 연구

### consecutive

- 구성: `con-` = together + Latin `secutus`
- 뜻: 연속적인
- 어원적 연결: 서로 뒤따라 끊기지 않고 이어짐
- 사용 맥락: `three consecutive days` = 사흘 연속

### execute

- 구성: `ex-` = out + Latin `sequi`
- 뜻: 실행하다, 처형하다
- 어원적 연결: 계획을 끝까지 따라가 밖으로 수행함
- 사용 맥락: `execute a plan` = 계획을 실행하다

### execution

- 구성: `execute` + `-ion`
- 뜻: 실행, 수행, 처형
- 어원적 연결: 일을 끝까지 따라가 실제로 해냄
- 사용 맥락: `flawless execution` = 완벽한 실행

### prosecute

- 구성: `pro-` = forward + Latin `sequi`
- 뜻: 기소하다, 수행하다
- 어원적 연결: 앞으로 계속 따라가며 법적으로 추적함
- 사용 맥락: `prosecute a case` = 사건을 기소하다

### persecute

- 구성: `per-` = thoroughly + Latin `sequi`
- 뜻: 박해하다
- 어원적 연결: 끝까지 따라다니며 괴롭힘
- 사용 맥락: `persecute minorities` = 소수자를 박해하다

## 기억 포인트

`sequ` 계열은 "따라가다" 이미지.

- `sequence`: 따라 이어지는 순서
- `consequence`: 뒤따라오는 결과
- `subsequent`: 그 뒤를 따르는
- `consecutive`: 연달아 따르는
- `execute`: 끝까지 따라가 실행함
- `persecute`: 끝까지 따라다니며 괴롭힘

## 비교 단어

### cess

- 뜻: 가다, 나아가다, 물러나다
- 구분 포인트: [[cess]]는 움직여 감. `sequ`는 앞선 것을 따라감.

### duc

- 뜻: 이끌다
- 구분 포인트: [[duc]]는 앞에서 끌고 감. `sequ`는 뒤에서 따라감.
