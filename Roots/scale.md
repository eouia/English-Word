# scale

- 출발 단어: `scale`
- 핵심 어원: 여러 갈래
  - Latin `scala` = ladder, stairs
  - Old Norse `skal` = bowl, cup
  - Germanic/French `scale` = shell, husk, fish scale
- 핵심 이미지: 단계, 저울, 껍질/비늘이 한 단어에 겹침
- 관련 어근: [[metr]], [[high]], [[angl]]

## 출발 단어 분석

`scale`은 한 어근으로 단순 처리하면 위험한 단어.

시험 어휘에서는 세 갈래를 구분하면 좋음.

1. "척도, 등급, 축척, 음계" = Latin `scala` = ladder, stairs 계열
2. "저울" = Old Norse `skal` = bowl, cup 계열
3. "비늘, 껍질, 제거하다" = shell/husk 계열

공통 암기 이미지는 "작은 단위들이 층층이 놓임" 정도로 잡되, 어원은 갈래가 다름.

## 갈래 1: 단계, 척도, 축척

### scale

- 발음: /ˈskeɪl/
- 구성: Latin `scala` = ladder, stairs
- 의미: 척도, 등급, 규모, 축척, 음계
- 어원적 연결: 사다리처럼 단계가 올라가거나 내려감
- 사용 맥락: `on a scale of 1 to 10` = 1부터 10까지의 척도에서

### musical scale

- 구성: `scale` = ladder, steps
- 의미: 음계
- 어원적 연결: 음이 계단처럼 단계적으로 올라감
- 사용 맥락: `major scale` = 장음계

### scale up

- 구성: `scale` + `up`
- 의미: 규모를 키우다, 확장하다
- 어원적 연결: 단계나 크기를 위로 올림
- 사용 맥락: `scale up production` = 생산 규모를 키우다

### scale down

- 구성: `scale` + `down`
- 의미: 규모를 줄이다, 축소하다
- 어원적 연결: 단계나 크기를 아래로 낮춤
- 사용 맥락: `scale down the project` = 프로젝트 규모를 줄이다

### scalable

- 발음: /ˈskeɪləbəl/
- 구성: `scale` + `-able`
- 의미: 확장 가능한
- 어원적 연결: 규모를 단계적으로 키울 수 있음
- 사용 맥락: `a scalable system` = 확장 가능한 시스템

### large-scale

- 발음: /ˈlɑˌrdʒskeɪl/
- 구성: `large` + `scale`
- 의미: 대규모의
- 어원적 연결: 척도나 범위가 큼
- 사용 맥락: `large-scale research` = 대규모 연구

### small-scale

- 구성: `small` + `scale`
- 의미: 소규모의
- 어원적 연결: 척도나 범위가 작음
- 사용 맥락: `small-scale farming` = 소규모 농업

### escalate

- 발음: /ˈɛskəˌleɪt/
- 구성: Latin `scala` = ladder
- 의미: 확대되다, 악화되다, 고조시키다
- 어원적 연결: 계단을 오르듯 단계가 올라감
- 사용 맥락: `escalate into conflict` = 갈등으로 확대되다

### escalator

- 발음: /ˈɛskəˌleɪtɚ/
- 구성: `escalate` 계열 + `-or`
- 의미: 에스컬레이터
- 어원적 연결: 계단처럼 사람을 위아래로 이동시킴
- 사용 맥락: `take the escalator` = 에스컬레이터를 타다

## 갈래 2: 저울

### scales

- 발음: /ˈskeɪlz/
- 구성: Old Norse `skal` = bowl, cup
- 의미: 저울
- 어원적 연결: 양쪽 그릇에 물건을 담아 무게를 잼
- 사용 맥락: `bathroom scales` = 체중계

### scale

- 발음: /ˈskeɪl/
- 구성: `scales`에서 단수화
- 의미: 저울, 체중계
- 어원적 연결: 원래는 저울 접시, 이후 전체 저울
- 사용 맥락: `step on the scale` = 체중계에 올라서다

### scale pan

- 구성: `scale` + `pan`
- 의미: 저울 접시
- 어원적 연결: 무게를 재려고 물건을 올리는 그릇
- 사용 맥락: `a balance scale pan` = 양팔저울 접시

## 갈래 3: 비늘, 껍질

### scale

- 발음: /ˈskeɪl/
- 구성: Germanic/French `scale` = shell, husk
- 의미: 비늘, 껍질
- 어원적 연결: 겹겹이 붙은 얇은 껍질
- 사용 맥락: `fish scales` = 물고기 비늘

### scaly

- 구성: `scale` + `-y`
- 의미: 비늘이 있는, 각질이 일어난
- 어원적 연결: 표면이 비늘처럼 덮임
- 사용 맥락: `scaly skin` = 비늘 같은 피부

### descale

- 구성: `de-` = remove + `scale`
- 의미: 비늘을 벗기다, 물때를 제거하다
- 어원적 연결: 표면의 비늘/석회질 껍질을 떼어 냄
- 사용 맥락: `descale a kettle` = 주전자 물때를 제거하다

### limescale

- 구성: `lime` + `scale`
- 의미: 석회질 물때
- 어원적 연결: 표면에 딱딱한 껍질처럼 붙은 침전물
- 사용 맥락: `remove limescale` = 석회질 물때를 제거하다

## 기억 포인트

- `scale` 척도: 사다리 계단처럼 단계
- `scale up/down`: 단계나 규모를 키우거나 줄임
- `escalate`: 계단을 오르듯 상황이 고조됨
- `scales`: 그릇 두 개로 재는 저울
- `fish scale`: 비늘, 껍질
- `descale`: 비늘/물때를 제거함

## 비교 단어

- [[metr]]
  - 핵심: 재다, 척도
  - 구분 포인트: `measure`는 재는 행위. `scale`은 단계가 표시된 척도나 규모

- [[high]]
  - 핵심: 높음
  - 구분 포인트: `escalate`는 단계가 올라감. `heighten`은 높이나 강도를 높임

- [[angl]]
  - 핵심: 각
  - 구분 포인트: 수학에서 `scale`은 축척/척도, `angle`은 두 선의 벌어진 정도
