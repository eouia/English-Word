# hamper

- 출발 단어: `hamper`
- 핵심 어원: Middle English `hamperen`, `hampren` = 방해하다, 묶다, 제한하다
- 핵심 이미지: 움직임이나 진행을 묶어 느리게 함
- 주의: 정확한 어원은 불확실함. `hamper` = 바구니, 빨래통은 Anglo-French `hanaper` 계열로 따로 봄
- 관련해서 헷갈릴 수 있는 어근: [[ped]]의 `impede` = 발을 막아 방해함, [[struct]]의 `obstruct` = 앞을 세워 막음

## 출발 단어 분석

`hamper`는 현대 영어에서 더 이상 작은 접두사/어근으로 잘게 나누어 외우기 어렵다.

- `hamper`: hinder, impede, restrict

따라서 `hamper`는 "묶거나 제한해 움직임을 어렵게 함"의 이미지로 기억. 완전히 막는 `prevent`보다, 진행을 느리게 하거나 어렵게 만드는 느낌이 강함.

## 단어들

### hamper

- 구성: Middle English `hamperen`, `hampren`
- 뜻: 방해하다, 저해하다, 제한하다
- 어원적 연결: 움직임이나 진행을 묶어 자유롭지 못하게 함
- 사용 맥락: `Heavy rain hampered rescue efforts.` = 폭우가 구조 작업을 방해했다

### hampered

- 구성: `hamper` + `-ed`
- 뜻: 방해받는, 제한된
- 어원적 연결: 움직임이나 진행이 묶인 상태
- 사용 맥락: `be hampered by a lack of funds` = 자금 부족으로 제약을 받다

### hampering

- 구성: `hamper` + `-ing`
- 뜻: 방해하는, 저해하는
- 어원적 연결: 진행을 계속 느리게 하거나 어렵게 만듦
- 사용 맥락: `hampering growth` = 성장을 저해하는

### hamper

- 구성: Anglo-French `hanaper` = goblet case, container
- 뜻: 바구니, 빨래통, 선물 바구니
- 어원적 연결: 물건을 담는 용기
- 사용 맥락: `a laundry hamper` = 빨래통, `a gift hamper` = 선물 바구니

## 기억 포인트

`hamper`는 두 뜻을 구분해서 기억.

- 동사 `hamper`: 진행을 묶어 방해함
- 명사 `hamper`: 물건을 담는 바구니
- 시험 어휘에서는 동사 `hamper` = `hinder`, `impede`, `obstruct` 쪽이 중요

## 비교 단어

### impede

- 뜻: 방해하다, 지연시키다
- 구분 포인트: [[ped]] 계열. 발을 막아 진행을 방해하는 이미지. `hamper`보다 더 격식적이고 시험 지문에서 자주 보임

### hinder

- 뜻: 방해하다, 저해하다
- 구분 포인트: `hamper`와 매우 가까운 일반 동사. `hamper`는 제약 조건이나 어려움 때문에 일이 느려지는 느낌이 좋음

### obstruct

- 뜻: 막다, 방해하다
- 구분 포인트: [[struct]] 계열. 앞에 장애물을 세워 물리적·제도적으로 막는 이미지가 강함
