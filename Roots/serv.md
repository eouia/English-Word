# serv

- 출발 단어: `observe`
- 핵심 어원: Latin `servare` = watch, keep safe, guard
- 핵심 이미지: 지켜보다, 지켜 보존하다
- 관련 어근: [[vid]], [[spect]], [[sumere]], [[servire]], [[astr]]

## 출발 단어 분석

`observe`는 Latin `observare`에서 온 말.

- `ob-`: before, in front of
- `servare`: watch, keep safe, guard

따라서 `observe`는 원래 "앞에서 지켜보다, 지키다"의 뜻. 여기서 "관찰하다, 규칙을 지키다"로 확장됨.

주의: `serve`, `service`, `deserve`는 겉모양은 비슷하지만 Latin `servire/servus` = serve, servant 계열이므로 [[servire]]에서 따로 정리.

### observe

- 발음: /əˈbzɝv/
- 구성: `ob-` = before + `servare`
- 의미: 관찰하다, 준수하다
- 어원적 연결: 앞에서 지켜보고, 규칙을 지킴
- 사용 맥락: `observe behavior` = 행동을 관찰하다

### observation

- 발음: /ˌɑbzɚˈveɪʃən/
- 구성: `observe` + `-ation`
- 의미: 관찰, 소견
- 어원적 연결: 지켜보아 알아낸 것
- 사용 맥락: `make an observation` = 관찰 소견을 말하다

### observer

- 발음: /əˈbzɝvɚ/
- 구성: `observe` + `-er`
- 의미: 관찰자
- 어원적 연결: 지켜보는 사람
- 사용 맥락: `an independent observer` = 독립 관찰자

### observant

- 발음: /əˈbzɝvənt/
- 구성: `observe` + `-ant`
- 의미: 관찰력이 예리한; 규칙을 잘 지키는
- 어원적 연결: 주의 깊게 지켜보거나 규칙을 지킴
- 사용 맥락: `an observant reader` = 관찰력이 예리한 독자

### observatory

- 발음: /əˈbzɝvəˌtɔri/
- 구성: `observe` + `-atory`
- 의미: 관측소, 천문대
- 어원적 연결: 하늘이나 자연 현상을 지켜보도록 만든 장소
- 사용 맥락: `an astronomical observatory` = 천문 관측소

### observance

- 발음: /əˈbzɝvəns/
- 구성: `observe` + `-ance`
- 의미: 준수, 의식
- 어원적 연결: 규칙이나 의식을 지킴
- 사용 맥락: `religious observance` = 종교 의식

### conserve

- 발음: /kəˈnsɝv/
- 구성: `con-` = together/completely + `servare`
- 의미: 보존하다, 아끼다
- 어원적 연결: 함께 지켜 보존함
- 사용 맥락: `conserve energy` = 에너지를 아끼다

### conservation

- 발음: /ˌkɑnsɚˈveɪʃən/
- 구성: `conserve` + `-ation`
- 의미: 보존, 보호
- 어원적 연결: 사라지지 않게 지킴
- 사용 맥락: `wildlife conservation` = 야생동물 보호

### preserve

- 발음: /prəˈzɝv/
- 구성: `pre-` = before + `servare`
- 의미: 보존하다, 보호하다
- 어원적 연결: 미리 지켜 손상되지 않게 함
- 사용 맥락: `preserve tradition` = 전통을 보존하다

### preservation

- 발음: /ˌprɛzɚˈveɪʃən/
- 구성: `preserve` + `-ation`
- 의미: 보존
- 어원적 연결: 미리 지켜 남기는 행위
- 사용 맥락: `historic preservation` = 역사적 보존

### reserve

- 발음: /rɪˈzɝv/
- 구성: `re-` = back + `servare`
- 의미: 남겨 두다, 예약하다, 비축
- 어원적 연결: 뒤로 따로 지켜 둠
- 사용 맥락: `reserve a seat` = 좌석을 예약하다

### reservation

- 발음: /ˌrɛzɚˈveɪʃən/
- 구성: `reserve` + `-ation`
- 의미: 예약, 유보
- 어원적 연결: 따로 지켜 둔 것
- 사용 맥락: `make a reservation` = 예약하다

## 기억 포인트

- `observe`: 지켜보다, 지키다
- `observatory`: 지켜보는 장소, 관측소
- `preserve`: 미리 지켜 보존함
- `conserve`: 함께/잘 지켜 아낌
- `reserve`: 따로 지켜 둠

## 비교 단어

- [[vid]]
  - 핵심: 보다
  - 구분 포인트: `observe`는 단순히 보는 것보다 "주의 깊게 지켜봄/준수함"이 강함

- [[spect]]
  - 핵심: 보다, 살피다
  - 구분 포인트: `inspect`는 안쪽까지 살펴봄. `observe`는 계속 지켜보거나 규칙을 지킴

- [[sumere]]
  - 핵심: 취하다
  - 구분 포인트: `reserve`는 따로 지켜 둠. `resume`은 다시 집어 들어 이어 감

- [[servire]]
  - 핵심: 섬기다, 봉사하다
  - 구분 포인트: `servare`는 지키고 보존함. `servire`는 섬기거나 서비스를 제공함
