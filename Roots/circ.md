# circ

- 출발 단어: `search`
- 핵심 어원: Latin `circus` = circle, ring
- 파생 동사: Late Latin `circare` = 둘러다니다, 이리저리 돌아 살피다
- 핵심 이미지: 원, 둘레, 주변을 빙 돎
- 관련 문서: [[fer]], [[dia]], [[tour]]
- 주의: `search`는 현대 철자만 보면 `se- + arch`처럼 보일 수 있으나, 역사적으로는 `circare` 계열.

## 출발 단어 분석

`search` = Old French `cerchier` ← Late Latin `circare`

- `circare`: go round, explore
- 원래 이미지: 어떤 대상을 빙 돌며 샅샅이 살핌
- 현대 의미: 찾다, 검색하다, 조사하다

## 단어들

### search

- 구성: Old French `cerchier` ← Late Latin `circare`
- 뜻: 찾다, 검색하다, 조사하다
- 어원적 연결: 주변을 돌며 자세히 살핌
- 사용 맥락: `search engine` = 검색 엔진

### research

- 구성: `re-` + `search`
- 뜻: 연구, 조사하다
- 어원적 연결: 다시 살피고 반복해서 찾음
- 사용 맥락: `scientific research` = 과학 연구

### circle

- 구성: Latin `circulus` = 작은 고리, 원
- 뜻: 원, 동그라미, 집단
- 어원적 연결: 둥근 형태 또는 둘러싼 무리
- 사용 맥락: `a circle of friends` = 친구 집단

### circular

- 구성: `circle` + `-ar`
- 뜻: 원형의, 순환하는
- 어원적 연결: 원처럼 둥글거나 되돌아옴
- 사용 맥락: `a circular argument` = 순환 논리

### circulate

- 구성: Latin `circulus` + 동사형
- 뜻: 순환하다, 유통되다, 퍼지다
- 어원적 연결: 원을 그리듯 돌고 퍼짐
- 사용 맥락: `blood circulates` = 피가 순환함

### circulation

- 구성: `circulate` + `-ion`
- 뜻: 순환, 유통, 발행 부수
- 어원적 연결: 돌고 도는 흐름
- 사용 맥락: `newspaper circulation` = 신문 발행 부수

### circuit

- 구성: Latin `circum` = around + `ire` = go
- 뜻: 회로, 순회, 한 바퀴 도는 길
- 어원적 연결: 둘레를 돌아가는 경로
- 사용 맥락: `an electrical circuit` = 전기 회로

### circumference

- 구성: Latin `circum` = around + [[fer]] = carry
- 뜻: 둘레, 원주
- 어원적 연결: 원 주위를 따라 가져간 선
- 사용 맥락: `the circumference of a circle` = 원의 둘레

### circumstantial

- 구성: `circumstance` + `-al`
- 뜻: 정황적인, 상황에 따른
- 어원적 연결: 중심 사실 주변의 조건에 근거함
- 사용 맥락: `circumstantial evidence` = 정황 증거

### circumvent

- 구성: Latin `circum` = around + `venire` = come
- 뜻: 우회하다, 피하다, 교묘히 회피하다
- 어원적 연결: 정면으로 가지 않고 빙 돌아감
- 사용 맥락: `circumvent the rules` = 규칙을 우회하다

### circumnavigate

- 구성: Latin `circum` = around + `navigate`
- 뜻: 일주하다, 둘러 항해하다
- 어원적 연결: 주변을 한 바퀴 돌아 항해함
- 사용 맥락: `circumnavigate the globe` = 세계를 일주하다

### circus

- 구성: Latin `circus` = ring, circular arena
- 뜻: 서커스, 원형 경기장
- 어원적 연결: 원형 공간에서 열리던 공연·경기
- 사용 맥락: `a traveling circus` = 순회 서커스

### circa

- 구성: Latin `circa` = around, about
- 뜻: 약, 대략
- 어원적 연결: 정확한 중심값 주변
- 사용 맥락: `circa 1900` = 1900년경

## 기억 포인트

`circ` 계열은 "원/둘레/주변" 이미지.

- `search`: 주변을 돌며 찾음
- `circle`: 원
- `circulate`: 돌며 퍼짐
- `circuit`: 한 바퀴 도는 길
- `circumference`: 원 주위를 따라간 선
- `circumvent`: 빙 돌아 피함

## 비교 단어

### cycle

- 뜻: 주기, 순환
- 구분 포인트: Greek `kyklos` = circle 계열. 의미는 가깝지만 Latin `circus`와 별도 경로.

### tour

- 뜻: 돌다, 순회하다
- 구분 포인트: [[tour]]는 돌려 깎는 도구·회전 이미지. `circ`는 원과 둘레 이미지가 더 직접적.

### stanc

- 뜻: 서 있는 경우, 자세, 실체, 거리
- 연결 포인트: `circumstance`는 `circum-` = around + `stare` = stand 복합어. 본문 항목은 [[stanc]]에서 정리.
