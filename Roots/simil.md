# simil

- 출발 단어: `assemble`
- 핵심 어원: Latin `similis` = like, similar; `simul` = together, at the same time
- 핵심 이미지: 같음, 비슷함, 함께 있음
- 관련 문서: [[sym]], [[uni]]
- 주의: [[sym]]은 Greek `syn-/sym-` = together. `simil`은 Latin `similis/simul` 계열.

## 출발 단어 분석

`assemble` = Old French `assembler` ← Latin/Vulgar Latin `assimulare`

- `ad-`: to
- `simul`: together
- `similis`: like, similar
- 원래 이미지: 비슷하거나 함께 둘 수 있는 것들을 한곳에 모음
- 현대 의미: 모으다, 조립하다

## 단어들

### assemble

- 구성: `ad-` + `simul/simil`
- 뜻: 모으다, 조립하다
- 어원적 연결: 함께 오게 하거나 한데 맞춰 둠
- 사용 맥락: `assemble a team` = 팀을 모으다

### assembly

- 구성: `assemble` 계열
- 뜻: 집회, 의회, 조립
- 어원적 연결: 함께 모인 사람이나 부품
- 사용 맥락: `assembly line` = 조립 라인

### assembled

- 구성: `assemble` + `-ed`
- 뜻: 모인, 조립된
- 어원적 연결: 함께 모여 하나가 된 상태
- 사용 맥락: `assembled guests` = 모인 손님들

### assembler

- 구성: `assemble` + `-er`
- 뜻: 조립공, 어셈블러
- 어원적 연결: 부품이나 명령을 한데 맞추는 것
- 사용 맥락: computer science 맥락

### ensemble

- 구성: French `ensemble` ← Late Latin `insimul` = at the same time
- 뜻: 앙상블, 전체 조합, 함께 입는 옷
- 어원적 연결: 여러 부분이 함께 있는 전체
- 사용 맥락: `a jazz ensemble` = 재즈 앙상블

### resemble

- 구성: `re-` + Old French `sembler` ← Latin `similis`
- 뜻: 닮다, 비슷하다
- 어원적 연결: 같은 모양처럼 보임
- 사용 맥락: `resemble her mother` = 어머니를 닮다

### resemblance

- 구성: `resemble` + `-ance`
- 뜻: 닮음, 유사성
- 어원적 연결: 서로 비슷하게 보이는 성질
- 사용 맥락: `family resemblance` = 가족 간 닮음

### semblance

- 구성: Old French `semblance` ← `sembler`
- 뜻: 겉모습, 외관, 조금이나마 있는 모습
- 어원적 연결: 실제와 같아 보이는 외형
- 사용 맥락: `a semblance of order` = 어느 정도의 질서

### dissemble

- 구성: `dis-` + `sembler`
- 뜻: 숨기다, 가장하다
- 어원적 연결: 실제와 다르게 보이게 함
- 사용 맥락: 문어적 표현

### simulate

- 구성: Latin `simulare` = make like, imitate
- 뜻: 흉내 내다, 모의 실험하다
- 어원적 연결: 실제와 비슷하게 만들어 보임
- 사용 맥락: `simulate an emergency` = 비상 상황을 모의하다

### simulation

- 구성: `simulate` + `-ion`
- 뜻: 모의 실험, 시뮬레이션
- 어원적 연결: 실제와 비슷하게 만든 과정
- 사용 맥락: `computer simulation` = 컴퓨터 시뮬레이션

### similar

- 구성: Latin `similis`
- 뜻: 비슷한, 유사한
- 어원적 연결: 같은 종류처럼 보임
- 사용 맥락: `similar results` = 비슷한 결과

### similarity

- 구성: `similar` + `-ity`
- 뜻: 유사성
- 어원적 연결: 비슷한 성질
- 사용 맥락: `notice a similarity` = 유사점을 알아차리다

### simultaneous

- 구성: Latin `simul` = at the same time
- 뜻: 동시에 일어나는
- 어원적 연결: 같은 시간에 함께 있음
- 사용 맥락: `simultaneous translation` = 동시통역

### assimilate

- 구성: `ad-` = to + `simil`
- 뜻: 동화되다, 흡수하다
- 어원적 연결: 비슷하게 만들어 한데 섞임
- 사용 맥락: `assimilate into a culture` = 문화에 동화되다

## 기억 포인트

`simil/simul` 계열은 "같음/비슷함/함께" 이미지.

- `assemble`: 함께 모음
- `ensemble`: 함께 있는 전체
- `resemble`: 비슷하게 보임
- `semblance`: 비슷해 보이는 외관
- `simulate`: 비슷하게 만들어 흉내 냄
- `similar`: 비슷한
- `simultaneous`: 같은 시간에 함께
- `assimilate`: 비슷하게 되어 섞임

## 비교 단어

### sym

- 뜻: 함께, 같이
- 구분 포인트: [[sym]]은 Greek `syn-/sym-`. `system`, `sympathy`, `symbol`은 이쪽.

### same

- 뜻: 같은
- 연결 포인트: 영어 `same`도 깊은 PIE `*sem-` 계열로 볼 수 있으나, 이 문서에서는 라틴계 시험 어휘 중심으로 정리.
