# hetero

- 출발 단어: `heterodox`
- 핵심 어원: Greek `heteros` = 다른, 이질적인
- 핵심 의미: 다름, 서로 다른 종류
- 관련 문서: [[homo]], [[dox]], [[onym]], [[gen]]
- 주의: `heresy`는 Greek `hairesis` = 선택한 사상 계열. 의미상 `heterodox`와 가깝지만 `hetero-` 자체에서 온 단어는 아님.

## 출발 단어 분석

`heterodox` = `hetero-` + `dox`

- `hetero-`: other, different
- `doxa`: opinion, belief
- 원래 이미지: 다른 의견을 가진 상태
- 의미 확장: 정통과 다른 의견 → 비정통의, 이단적인

## 단어들

### heterodox

- 구성: `hetero-` = different + [[dox]] = opinion
- 뜻: 비정통의, 이단적인, 주류와 다른
- 어원적 연결: 정통과 다른 의견을 가짐
- 사용 맥락: `heterodox economics` = 비주류 경제학

### heterodoxy

- 구성: `heterodox` + `-y`
- 뜻: 비정통, 이단적 견해
- 어원적 연결: 공식적으로 인정된 믿음과 다른 견해
- 사용 맥락: `religious heterodoxy` = 종교적 비정통

### heterogeneous

- 구성: `hetero-` = different + `genos` = kind
- 뜻: 이질적인, 여러 종류가 섞인
- 어원적 연결: 서로 다른 종류로 이루어짐
- 사용 맥락: `a heterogeneous society` = 이질적인 사회

### heterogeneity

- 구성: `heterogeneous` + `-ity`
- 뜻: 이질성, 다양하게 섞인 상태
- 어원적 연결: 같은 종류가 아니라 여러 종류가 함께 있음
- 사용 맥락: `cultural heterogeneity` = 문화적 이질성

### heterosexual

- 구성: `hetero-` = different + `sexual`
- 뜻: 이성애의
- 어원적 연결: 다른 성에 끌림
- 사용 맥락: 성적 지향을 설명하는 사회·심리 맥락

### heteronym

- 구성: `hetero-` = different + [[onym]] = name
- 뜻: 철자는 같지만 발음과 뜻이 다른 말
- 어원적 연결: 같은 글자 형태가 서로 다른 이름·발음으로 쓰임
- 사용 맥락: `lead` /liːd/와 `lead` /led/ 같은 예

### heterozygous

- 구성: `hetero-` = different + Greek `zygon` = yoke, pair
- 뜻: 이형접합의
- 어원적 연결: 한 쌍의 유전자가 서로 다른 상태
- 사용 맥락: genetics context

## 기억 포인트

`hetero-`는 "다른 종류" 이미지.

- `heterodox`: 다른 의견
- `heterogeneous`: 다른 종류가 섞임
- `heteronym`: 다른 이름/발음
- `heterozygous`: 한 쌍이 서로 다름

## 비교 단어

### homo

- 뜻: 같은
- 구분 포인트: [[homo]]는 `homogeneous`, `homonym`, `homophone`처럼 "같음" 이미지. `hetero-`는 반대로 "다름" 이미지.

### heresy

- 뜻: 이단, 정통에서 벗어난 믿음
- 구분 포인트: `heresy`는 `hairesis` = 선택한 사상 계열. `heterodox`와 의미상 가까우나 어원은 다름.
