# mir

- 출발 단어: `admire`
- 핵심 어원: Latin `mirari` = wonder at, marvel at
- 핵심 이미지: 놀라서 바라봄, 경이롭게 여김
- 관련 문서: 없음
- 주의: `mirror`는 "보는 것"이라는 의미 연결은 있지만, 형태·역사는 `mirari` 계열과 섞여 발전한 단어라 단순히 `mir + -or`로만 자르지 않음.

## 출발 단어 분석

`admire` = `ad-` + `mirari`

- `ad-`: to, toward
- `mirari`: wonder at, marvel at
- 원래 이미지: 어떤 대상을 향해 놀라 바라봄
- 현대 의미: 감탄하다, 존경하다

## 단어들

### admire

- 발음: /æˈdmaɪr/
- 구성: `ad-` = to + Latin `mirari` = wonder at
- 뜻: 감탄하다, 존경하다
- 어원적 연결: 대상을 경이롭게 바라봄
- 사용 맥락: `admire her courage` = 그녀의 용기에 감탄하다

### admiration

- 발음: /ˌædmɚˈeɪʃən/
- 구성: `admire` + `-ation`
- 뜻: 감탄, 존경
- 어원적 연결: 경이롭게 여기며 높이 평가함
- 사용 맥락: `express admiration` = 감탄을 표현하다

### admirable

- 발음: /ˈædmɚəbəl/
- 구성: `admire` + `-able`
- 뜻: 감탄할 만한, 훌륭한
- 어원적 연결: 존경과 감탄을 불러일으킬 만함
- 사용 맥락: `an admirable effort` = 훌륭한 노력

### admirer

- 발음: /æˈdmaɪrɚ/
- 구성: `admire` + `-er`
- 뜻: 찬미자, 존경하는 사람
- 어원적 연결: 어떤 대상을 높이 보고 좋아하는 사람
- 사용 맥락: `a secret admirer` = 비밀스러운 추종자

### miracle

- 발음: /ˈmɪrəkəl/
- 구성: Latin `miraculum` = object of wonder
- 뜻: 기적
- 어원적 연결: 사람을 놀라게 하는 경이로운 일
- 사용 맥락: `a medical miracle` = 의학적 기적

### miraculous

- 발음: /mɚˈækjələs/
- 구성: `miracle` + `-ous`
- 뜻: 기적적인
- 어원적 연결: 경이로울 정도로 믿기 어려움
- 사용 맥락: `a miraculous recovery` = 기적적인 회복

### mirage

- 발음: /mɚˈɑʒ/
- 구성: French `mirage` ← `mirer` = look at
- 뜻: 신기루, 환상
- 어원적 연결: 눈에 보이는 듯하지만 실제와 다른 경이로운 모습
- 사용 맥락: `a desert mirage` = 사막의 신기루

### mirror

- 발음: /ˈmɪrɚ/
- 구성: Old French `mirour`, `mireor` = looking glass
- 뜻: 거울, 반영하다
- 어원적 연결: 바라보는 도구, 모습을 비추어 보게 하는 것
- 사용 맥락: `look in the mirror` = 거울을 보다

## 기억 포인트

`mir` 계열은 "놀라서 바라봄, 경이" 이미지.

- `admire`: 감탄하며 바라봄
- `admiration`: 감탄, 존경
- `admirable`: 감탄할 만한
- `miracle`: 경이로운 일
- `mirage`: 눈앞에 보이는 신기루
- `mirror`: 바라보고 비춰 보는 도구

## 비교 단어

### spec

- 뜻: 보다, 바라보다, 살피다
- 구분 포인트: `spec`은 보는 행위와 관찰. `mir`는 놀라움과 감탄이 실린 바라봄.
