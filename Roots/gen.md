# gen

- 출발 단어: `generic`
- 핵심 어원: Greek/Latin `gen-` = birth, kind, race
- 핵심 이미지: 태어남, 종류, 혈통
- 관련 어근: [[nat]], [[cogn]], [[som]], [[dem]]

## 출발 단어 분석

`generic`은 Latin `genus` = kind, race에서 온 말.

핵심은 "태어난 종류, 부류"임.

따라서 `generic`은 "특정 브랜드나 개별성이 아니라 한 종류 전체에 속함" → "일반적인, 포괄적인"의 뜻.

## 단어들

### generic

- 구성: Latin `genus` = kind + `-ic`
- 의미: 일반적인, 포괄적인, 상표명이 아닌
- 어원적 연결: 특정 개체가 아니라 같은 종류 전체에 속함
- 사용 맥락: `generic medicine` = 복제약, 일반 의약품

### general

- 구성: Latin `generalis` < `genus`
- 의미: 일반적인, 전체적인
- 어원적 연결: 특정 하나가 아니라 종류 전체에 해당함
- 사용 맥락: `a general rule` = 일반 규칙

### generate

- 구성: Latin `generare` = beget, produce
- 의미: 만들어 내다, 발생시키다
- 어원적 연결: 새로 태어나게 함
- 사용 맥락: `generate electricity` = 전기를 생산하다

### generation

- 구성: `generate` + `-ion`
- 의미: 세대, 발생, 생성
- 어원적 연결: 태어나 이어지는 한 무리
- 사용 맥락: `the younger generation` = 젊은 세대

### generator

- 구성: `generate` + `-or`
- 의미: 발전기, 생성 장치
- 어원적 연결: 무언가를 만들어 내는 장치
- 사용 맥락: `a power generator` = 발전기

### genus

- 구성: Latin `genus`
- 의미: 속, 종류
- 어원적 연결: 같은 출생/종류로 묶인 분류
- 사용 맥락: `genus and species` = 속과 종

### genre

- 구성: French `genre` < Latin `genus`
- 의미: 장르, 유형
- 어원적 연결: 같은 종류의 작품 묶음
- 사용 맥락: `a literary genre` = 문학 장르

### gene

- 구성: Greek `genos` = race, kind 계열
- 의미: 유전자
- 어원적 연결: 태어나는 성질을 전달하는 단위
- 사용 맥락: `a dominant gene` = 우성 유전자

### genetic

- 구성: `gene` + `-tic`
- 의미: 유전의, 발생의
- 어원적 연결: 태어남과 유전 정보에 관련됨
- 사용 맥락: `genetic information` = 유전 정보

### genetics

- 구성: `genetic` + `-s`
- 의미: 유전학
- 어원적 연결: 유전과 발생을 다루는 학문
- 사용 맥락: `study genetics` = 유전학을 공부하다

### genuine

- 구성: Latin `genuinus` = innate, native
- 의미: 진짜의, 진정한
- 어원적 연결: 타고난 본래의 성질을 가진 것
- 사용 맥락: `genuine concern` = 진심 어린 걱정

### generous

- 구성: Latin `generosus` = of noble birth
- 의미: 관대한, 후한
- 어원적 연결: 원래 "좋은 혈통의"에서 고귀하고 후한 성품으로 확장
- 사용 맥락: `a generous offer` = 후한 제안

### gender

- 구성: Old French `gendre` < Latin `genus`
- 의미: 성, 젠더
- 어원적 연결: 종류, 부류에서 문법적 성/사회적 성으로 확장
- 사용 맥락: `gender equality` = 성평등

### indigenous

- 구성: Latin `indigena` = born in a place
- 의미: 토착의, 원주민의
- 어원적 연결: `in-` = in + `gen` = born. 그 땅 안에서 태어난 사람이나 것
- 사용 맥락: `indigenous people` = 원주민

### indigene

- 구성: French/Latin `indigene` ← Latin `indigena`
- 의미: 토착민, 원주민
- 어원적 연결: 특정 땅에서 태어난 사람
- 사용 맥락: academic / anthropological context

### indigeneity

- 구성: `indigene` + `-ity`
- 의미: 토착성, 원주민성
- 어원적 연결: 어떤 지역에 원래부터 태어나 속한 상태
- 사용 맥락: `questions of indigeneity` = 토착성 문제

### endogenous

- 구성: Greek `endon` = within + `gen` = born, produced
- 의미: 내부에서 생긴, 내생적인
- 어원적 연결: 바깥에서 온 것이 아니라 안에서 발생함
- 사용 맥락: `endogenous growth` = 내생적 성장

### engender

- 구성: `en-` = make/cause + `gender/gen`
- 의미: 일으키다, 낳다
- 어원적 연결: 어떤 결과가 태어나게 함
- 사용 맥락: `engender trust` = 신뢰를 낳다

## 가까운 갈래: 타고난 재능, 만들어 내는 장치

### genius

- 구성: Latin `genius` = inborn spirit
- 의미: 천재, 특별한 재능
- 어원적 연결: 타고난 정신이나 재능
- 사용 맥락: `a musical genius` = 음악 천재

### ingenious

- 구성: Latin `ingenium` = inborn talent
- 의미: 기발한, 독창적인
- 어원적 연결: 타고난 재능에서 나온 영리함
- 사용 맥락: `an ingenious solution` = 기발한 해결책

### engine

- 구성: Old French `engin` < Latin `ingenium`
- 의미: 엔진, 기계 장치
- 어원적 연결: 원래 재능, 고안, 장치의 뜻에서 기계로 확장
- 사용 맥락: `a search engine` = 검색 엔진

## 기억 포인트

- `generic`: 한 종류 전체에 속함
- `general`: 종류 전체에 해당함
- `generate`: 새로 생겨나게 함
- `generation`: 태어나 이어지는 세대
- `genre`: 작품의 종류
- `gene/genetic`: 태어나는 성질의 전달
- `genuine`: 본래 타고난 진짜
- `generous`: 좋은 태생에서 후한 성품으로 확장

## 비교 단어

- Latin `nat-`
  - 핵심: 태어나다
  - 구분 포인트: `native`, `nature`, `nation`은 [[nat]] 계열. `gen`은 종류, 혈통, 발생 쪽

- [[dem]]
  - 핵심: 사람들, 민중
  - 구분 포인트: `demographic`은 인구 집단. `generation`은 태어나 이어지는 세대

- [[gest]]
  - 핵심: 나르다, 처리하다
  - 구분 포인트: 철자는 비슷하지만 `generate/generic`의 `gen`과 어원 다름
