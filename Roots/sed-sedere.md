# sed-sedere

- 상위 문서: [[sed]]
- 핵심 어원: 라틴어 `sedere` = 앉다
- 핵심 이미지: 자리에 앉아 머묾 / 앞자리에 앉아 이끎 / 아래로 앉아 가라앉음
- 관련 갈래: [[sed-sit]], [[sed-sess]]

## 단어들

### sedentary

- 발음: /ˈsɛdəˌntɛri/
- 구성: 라틴어 `sedere` = 앉다
- 뜻: 앉아서 지내는, 활동량이 적은
- 어원적 연결: 오래 앉아 있는 생활 방식
- 사용 맥락: `a sedentary lifestyle` = 앉아서 지내는 생활 방식

### reside

- 발음: /rɪˈzaɪd/
- 구성: `re-` + `sedere`
- 뜻: 거주하다
- 어원적 연결: 한곳에 다시/계속 앉아 머묾
- 사용 맥락: `reside in Seoul` = 서울에 거주하다

### residence

- 발음: /ˈrɛzɪdəns/
- 구성: `reside` + `-ence`
- 뜻: 거주지, 주택
- 어원적 연결: 머물러 사는 장소
- 사용 맥락: `permanent residence` = 영주권, 영구 거주

### resident

- 발음: /ˈrɛzɪdənt/
- 구성: `reside` + `-ent`
- 뜻: 거주자, 거주하는
- 어원적 연결: 특정 장소에 머무는 사람
- 사용 맥락: `local residents` = 지역 주민들

### preside

- 발음: /prɪˈzaɪd/
- 구성: `pre-` = before + `sedere`
- 뜻: 주재하다, 사회를 보다
- 어원적 연결: 앞자리에 앉아 회의를 이끎
- 사용 맥락: `preside over a meeting` = 회의를 주재하다

### president

- 발음: /ˈprɛzəˌdɛnt/
- 구성: `preside` + `-ent`
- 뜻: 대통령, 회장
- 어원적 연결: 앞자리에 앉아 이끄는 사람
- 사용 맥락: `company president` = 회사 사장

### session

- 발음: /ˈsɛʃən/
- 구성: 라틴어 `sessio` = 앉음, 회의
- 뜻: 회의, 기간, 수업 시간
- 어원적 연결: 함께 앉아 진행하는 시간
- 사용 맥락: `a training session` = 교육 세션

### subside

- 발음: /səˈbsaɪd/
- 구성: `sub-` = under, down + `sidere`
- 뜻: 가라앉다, 진정되다
- 어원적 연결: 아래로 앉아 내려감. 이후 강도나 감정이 가라앉음
- 사용 맥락: `the pain subsided` = 통증이 가라앉았다

### subsidence

- 발음: /səˈbsaɪdəns/
- 구성: `subside` + `-ence`
- 뜻: 침하, 가라앉음
- 어원적 연결: 땅이나 표면이 아래로 내려앉음
- 사용 맥락: `land subsidence` = 지반 침하

### sediment

- 발음: /ˈsɛdəmənt/
- 구성: Latin `sedere` = sit, settle
- 뜻: 침전물, 퇴적물
- 어원적 연결: 물이나 공기 속 입자가 아래로 가라앉아 앉은 것
- 사용 맥락: `river sediment` = 강의 퇴적물

### sedimentary

- 발음: /ˌsɛdəˈmɛntɚi/
- 구성: `sediment` + `-ary`
- 뜻: 퇴적의, 퇴적암의
- 어원적 연결: 가라앉아 쌓인 물질로 이루어짐
- 사용 맥락: `sedimentary rock` = 퇴적암

### assiduous

- 발음: /əˈsɪdwəs/
- 구성: 라틴어 `assiduus` = 곁에 계속 앉아 있는
- 뜻: 근면한, 끈기 있는
- 어원적 연결: 한곳에 붙어 앉아 꾸준히 임함
- 사용 맥락: `assiduous effort` = 꾸준한 노력

### sessile

- 구성: 라틴어 `sessilis` = 앉아 있는
- 뜻: 고착된, 줄기 없이 붙어 있는
- 어원적 연결: 움직이지 않고 붙어 앉은 상태
- 사용 맥락: `sessile organism` = 고착 생물

## 기억 포인트

`sed-sedere`는 앉아 머무는 위치가 의미를 만듦.

- `reside`: 한곳에 머묾
- `preside`: 앞자리에 앉아 이끎
- `session`: 함께 앉아 진행하는 시간
- `subside`: 아래로 앉아 가라앉음
- `sediment`: 아래로 가라앉아 쌓인 물질
- `assiduous`: 붙어 앉아 꾸준함
