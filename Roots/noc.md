# noc

- 출발 단어: `innocuous`
- 핵심 어원: 라틴어 `nocere` = 해치다, 손상시키다
- 변형: `noc`, `nox`, `nuis`
- 핵심 이미지: 해를 끼침 / 해가 없음
- 관련해서 헷갈릴 수 있는 어근: [[culp]] = 죄, 책임 계열, [[tox]] = 독 계열

## 출발 단어 분석

`innocuous`는 `in-` + `noc` + `-uous`.

- `in-`: not
- `noc`: harm
- `-uous`: 형용사형 접미사

따라서 `innocuous`는 "해롭지 않은"의 뜻. 위험해 보이지 않거나 실제로 별 해가 없는 것을 가리킴.

## 단어들

### innocuous

- 발음: /ˌɪˈnɑkjuəs/
- 구성: `in-` = not + `noc` = harm + `-uous`
- 뜻: 해롭지 않은, 악의 없는
- 어원적 연결: 해를 끼치지 않음
- 사용 맥락: `an innocuous comment` = 악의 없는 말

### innocent

- 발음: /ˈɪnəsənt/
- 구성: `in-` = not + Latin `nocens` = harming
- 뜻: 무죄의, 순진한, 해가 없는
- 어원적 연결: 해를 끼치지 않았거나 죄가 없음
- 사용 맥락: `an innocent person` = 무고한 사람

### innocence

- 발음: /ˈɪnəsəns/
- 구성: `innocent` + `-ence`
- 뜻: 무죄, 순수함
- 어원적 연결: 해를 끼치지 않은 상태
- 사용 맥락: `prove one's innocence` = 결백을 입증하다

### noxious

- 발음: /ˈnɑkʃəs/
- 구성: Latin `noxa` = harm, injury + `-ious`
- 뜻: 해로운, 유독한
- 어원적 연결: 몸이나 환경에 해를 끼침
- 사용 맥락: `noxious fumes` = 유독 가스

### obnoxious

- 발음: /ɑˈbnɑkʃəs/
- 구성: `ob-` = toward, exposed to + `nox`
- 뜻: 불쾌한, 아주 싫은
- 어원적 연결: 원래는 해나 책임에 노출된 상태. 현대 영어에서는 남에게 해롭거나 거슬리는 태도
- 사용 맥락: `obnoxious behavior` = 불쾌한 행동

### nuisance

- 발음: /ˈnusəns/
- 구성: Old French `nuire` = harm, annoy < Latin `nocere`
- 뜻: 성가신 것, 골칫거리
- 어원적 연결: 큰 해는 아니어도 불편과 방해를 줌
- 사용 맥락: `a public nuisance` = 공공 골칫거리

### nocent

- 구성: Latin `nocens` = harming
- 뜻: 해로운, 유죄의
- 어원적 연결: 해를 끼치는 상태
- 사용 맥락: 드문 격식어. `innocent`의 반대 형태로 이해

## 기억 포인트

`noc/nox` 계열은 "해치다" 이미지.

- `innocuous`: 해롭지 않은
- `innocent`: 해를 끼치지 않은 → 무죄의, 순진한
- `noxious`: 해로운, 유독한
- `obnoxious`: 남에게 해롭게 거슬리는 → 불쾌한
- `nuisance`: 불편을 주는 골칫거리

## 비교 단어

### toxic

- 뜻: 독성의
- 구분 포인트: [[tox]]는 독 자체의 이미지. `noxious`는 더 넓게 해로운 것, 특히 유독한 기체나 영향에 자주 씀

### culpable

- 뜻: 책임이 있는, 비난받을 만한
- 구분 포인트: [[culp]]는 죄와 책임의 이미지. `innocent`는 해를 끼치지 않았다는 데서 무죄로 확장
