# grap

- 출발 단어: `grapple`
- 핵심 어원: Old French `grapil` / `grappelle` = hook
- 핵심 이미지: 갈고리처럼 걸어 붙잡고 맞붙음
- 관련 문서: [[prehend]]
- 주의: `grasp`, `grab`, `grip`은 의미가 가깝지만 어원 갈래가 완전히 같다고 단정하기보다 비교 단어로 둠.

## 출발 단어 분석

`grapple`은 현대 형태에서 `grap` + `-le`처럼 보이지만, 역사적으로는 "갈고리, 갈고리로 붙잡는 도구"에서 온 단어.

- 원래 의미: 갈고리, 갈고리로 붙잡다
- 의미 확장: 서로 붙잡고 몸싸움함 → 어려운 문제와 씨름함
- 현대 의미: 붙잡고 싸우다, 씨름하다; 어려운 문제를 해결하려 애쓰다

## 단어들

### grapple

- 구성: Middle English `grappel` ← Old French `grapil` = hook
- 뜻: 붙잡고 싸우다, 씨름하다; 어려운 문제와 맞붙다
- 어원적 연결: 갈고리로 걸어 붙잡듯 상대나 문제에 달라붙음
- 사용 맥락: `grapple with a problem` = 문제와 씨름하다, `grapple with an opponent` = 상대와 붙잡고 싸우다

### grappling

- 구성: `grapple` + `-ing`
- 뜻: 붙잡고 겨루기, 그래플링; 씨름하는
- 어원적 연결: 서로 걸고 붙잡아 몸으로 맞붙는 행위
- 사용 맥락: 격투기·레슬링·비유적 문제 해결 맥락

### grappled

- 구성: `grapple` + `-ed`
- 뜻: 붙잡고 싸운, 씨름한
- 어원적 연결: 상대나 문제에 실제로 달라붙어 맞선 상태
- 사용 맥락: `grappled with uncertainty` = 불확실성과 씨름했다

### grapnel

- 구성: Old French `grapil/grappin` 계열의 작은 갈고리
- 뜻: 작은 닻, 갈고리 닻
- 어원적 연결: 무언가를 걸어 붙잡는 갈고리 도구
- 사용 맥락: 해양·장비 문맥. `grappling hook`과 가까움.

### grappling hook

- 구성: `grappling` + `hook`
- 뜻: 갈고리 달린 도구, 그래플링 훅
- 어원적 연결: 대상에 걸어 붙잡기 위한 갈고리
- 사용 맥락: 등반·구조·게임 장비 문맥

## 기억 포인트

`grap`는 "갈고리로 걸어 붙잡음" 이미지.

- `grapple`: 붙잡고 싸우거나 문제와 씨름함
- `grappling`: 붙잡고 겨루는 행위
- `grapnel`: 작은 갈고리 닻
- `grappling hook`: 걸어 붙잡는 갈고리 도구

## 비교 단어

### grasp

- 뜻: 꽉 잡다; 이해하다
- 구분 포인트: [[prehend]]의 비교 단어로도 다룰 수 있음. `grapple`은 단순히 잡는 것보다 "맞붙어 씨름함"이 강함.

### grab

- 뜻: 움켜잡다
- 구분 포인트: 빠르게 낚아채듯 잡는 느낌. `grapple`은 붙잡은 뒤 계속 맞붙는 과정.

### grip

- 뜻: 꽉 잡다, 지배하다
- 구분 포인트: 잡은 힘이나 통제에 초점. `grapple`은 갈고리처럼 걸고 몸으로 맞서는 이미지.
