# mot

- 출발 단어: `motion`
- 핵심 어원: Latin `movere` = 움직이다
- 관련 형태: `mot-`, `mov-`, `mob-`
- 핵심 이미지: 움직이게 하거나 마음을 움직임
- 관련 문서: [[dynam]], [[anim]], [[path]]
- 주의: `emotion`은 마음이 밖으로 움직이는 이미지에서 감정의 뜻으로 확장.

## 출발 단어 분석

`motion` ← Latin `motio`

- `movere`: move
- `motio`: movement
- 원래 이미지: 움직임
- 현대 의미: 동작, 움직임, 제안

## 단어들

### motion

- 구성: Latin `motio`
- 뜻: 움직임, 동작, 제안
- 어원적 연결: 움직이는 상태나 움직이게 하는 제안
- 사용 맥락: `in motion` = 움직이는 중

### motive

- 구성: Latin `motivus` = moving
- 뜻: 동기, 이유
- 어원적 연결: 사람을 움직이게 하는 내적 힘
- 사용 맥락: `a hidden motive` = 숨은 동기

### motivate

- 구성: `motive` + `-ate`
- 뜻: 동기를 부여하다
- 어원적 연결: 사람을 움직이게 만듦
- 사용 맥락: `motivate students` = 학생들에게 동기를 부여하다

### motivation

- 구성: `motivate` + `-ion`
- 뜻: 동기, 동기 부여
- 어원적 연결: 행동을 움직이게 하는 힘
- 사용 맥락: `strong motivation` = 강한 동기

### emotion

- 구성: `e-` = out + Latin `movere`
- 뜻: 감정
- 어원적 연결: 마음이 밖으로 움직여 드러남
- 사용 맥락: `express emotion` = 감정을 표현하다

### move

- 구성: Old French `movoir` ← Latin `movere`
- 뜻: 움직이다, 이사하다, 감동시키다
- 어원적 연결: 물리적으로 또는 마음을 움직임
- 사용 맥락: `move quickly` = 빠르게 움직이다

### movement

- 구성: `move` + `-ment`
- 뜻: 움직임, 운동
- 어원적 연결: 움직이는 과정이나 집단적 변화
- 사용 맥락: `a social movement` = 사회 운동

### mobile

- 구성: Latin `mobilis` = movable
- 뜻: 움직일 수 있는, 이동식의
- 어원적 연결: 쉽게 움직일 수 있음
- 사용 맥락: `mobile devices` = 모바일 기기

### mobility

- 구성: `mobile` + `-ity`
- 뜻: 이동성
- 어원적 연결: 움직일 수 있는 성질
- 사용 맥락: `social mobility` = 사회적 이동성

### automobile

- 구성: Greek `auto-` = self + Latin `mobile`
- 뜻: 자동차
- 어원적 연결: 스스로 움직이는 것
- 사용 맥락: `automobile industry` = 자동차 산업

## 기억 포인트

`mot/mov/mob` 계열은 "움직이다" 이미지.

- `motion`: 움직임
- `motive`: 움직이게 하는 이유
- `motivate`: 움직이게 만들다
- `emotion`: 마음의 움직임
- `mobile`: 움직일 수 있음
- `automobile`: 스스로 움직이는 것

## 비교 단어

### dynam

- 뜻: 힘, 능력
- 구분 포인트: [[dynam]]은 움직임을 만드는 힘. `mot`는 실제 움직임과 동기.

### anim

- 뜻: 생명, 마음
- 구분 포인트: [[anim]]은 살아 움직이는 생명력. `mot`는 움직임 자체와 움직이게 하는 이유.
