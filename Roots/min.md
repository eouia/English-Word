# min

- 출발 단어: `diminish`
- 핵심 어원: Latin `minus` / `minor` / `minuere` = smaller, less, lessen
- 핵심 이미지: 작게 하거나 더 적은 쪽으로 줄임
- 관련 문서: [[mod]], [[part]], [[mag]]
- 주의: `minister`는 현대 의미상 "장관"이지만 어원은 "더 작은 사람, 시중드는 사람" 쪽에서 발전. `minute` = 시간 단위는 "작게 나눈 부분" 이미지.

## 출발 단어 분석

`diminish` ← Latin `diminuere` = lessen, break into smaller pieces

- `di-/de-`: apart, down
- `minuere`: make smaller, lessen
- 원래 이미지: 쪼개어 더 작게 만듦
- 의미 변화: 작게 만들다 → 줄이다, 약화시키다
- 현대 의미: 줄이다, 감소시키다; 줄어들다

## 단어들

### diminish

- 구성: Latin `di-/de-` = apart/down + `minuere` = lessen
- 뜻: 줄이다, 감소시키다; 줄어들다
- 어원적 연결: 크기나 힘을 더 작은 상태로 낮춤
- 사용 맥락: `diminish the risk` = 위험을 줄이다

### diminution

- 구성: Latin `diminutio` = lessening
- 뜻: 감소, 축소
- 어원적 연결: 작아지는 과정이나 결과
- 사용 맥락: `a diminution of influence` = 영향력의 감소

### diminutive

- 구성: Latin `diminutivus` = pertaining to lessening
- 뜻: 아주 작은; 지소사
- 어원적 연결: 작게 줄인 형태
- 사용 맥락: `a diminutive figure` = 작은 체구

### minor

- 구성: Latin `minor` = smaller, less
- 뜻: 작은, 덜 중요한; 미성년자; 부전공
- 어원적 연결: 더 작은 쪽, 더 낮은 중요도
- 사용 맥락: `a minor problem` = 사소한 문제

### minority

- 구성: `minor` + `-ity`
- 뜻: 소수, 소수 집단
- 어원적 연결: 수가 더 적은 쪽
- 사용 맥락: `minority rights` = 소수자 권리

### minus

- 구성: Latin `minus` = less
- 뜻: 빼기, 음의; ~이 없는
- 어원적 연결: 어떤 양에서 덜어 낸 상태
- 사용 맥락: `ten minus three` = 10 빼기 3

### minimum

- 구성: Latin `minimum` = smallest
- 뜻: 최저, 최소
- 어원적 연결: 가장 작은 한계
- 사용 맥락: `the minimum requirement` = 최소 요건

### minimal

- 구성: `minimum` 계열 + `-al`
- 뜻: 최소의, 아주 적은
- 어원적 연결: 필요한 양을 가장 작게 줄인 상태
- 사용 맥락: `minimal effort` = 최소한의 노력

### minimize

- 구성: `minimum` 계열 + `-ize`
- 뜻: 최소화하다, 축소하다
- 어원적 연결: 가능한 한 작게 만듦
- 사용 맥락: `minimize costs` = 비용을 최소화하다

### minute

- 구성: Latin `minuta` = small part
- 뜻: 분; 아주 작은
- 어원적 연결: 시간을 잘게 나눈 작은 단위, 또는 매우 작은 것
- 사용 맥락: `wait a minute` = 1분 기다리다, `minute details` = 미세한 세부 사항

### minutiae

- 구성: Latin `minutiae` = small details
- 뜻: 사소한 세부 사항들
- 어원적 연결: 아주 작게 나뉜 세부
- 사용 맥락: `focus on minutiae` = 사소한 세부에 집중하다

### minister

- 구성: Latin `minister` = servant, attendant; from `minus` = less
- 뜻: 장관, 목사; 돌보다, 봉사하다
- 어원적 연결: 원래 "더 낮은 위치에서 섬기는 사람" 이미지
- 사용 맥락: `the prime minister` = 총리, `minister to the sick` = 환자를 돌보다

### administer

- 구성: `ad-` = to + `minister`
- 뜻: 관리하다, 집행하다, 투여하다
- 어원적 연결: 맡은 일을 섬기듯 처리하고 운영함
- 사용 맥락: `administer a program` = 프로그램을 운영하다

## 기억 포인트

`min` 계열은 "작게, 더 적게" 이미지.

- `diminish`: 작게 줄임
- `minor`: 더 작은, 덜 중요한
- `minority`: 수가 더 적은 집단
- `minimum`: 가장 작은 한계
- `minute`: 작게 나눈 단위
- `minister`: 낮은 자리에서 섬기는 사람

## 비교 단어

### mag

- 뜻: 크다, 위대하다
- 구분 포인트: [[mag]]는 크게 함. `min`은 작게 하거나 더 적은 쪽.

### mod

- 뜻: 알맞은 정도, 조절
- 구분 포인트: [[mod]]는 적정 수준으로 맞춤. `min`은 양이나 크기 자체를 줄임.

### part

- 뜻: 나누어진 부분
- 구분 포인트: [[part]]는 나눔 자체. `min`은 나누어 작게 만들거나 작은 정도를 가리킴.
