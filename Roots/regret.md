# regret

- 출발 단어: `regret`
- 핵심 어원: Old French `regreter`
- 핵심 이미지: 지나간 일에 마음이 아프고 아쉬움
- 관련 문서: [[quer]]
- 주의: 깊은 어원은 불확실함. 현대 형태를 `re-` + `greet`처럼 분해하지 않음.

## 출발 단어 분석

`regret` ← Old French `regreter`

- Old French `regreter`: lament, bewail
- 더 깊은 기원은 확실하지 않음
- 원래 이미지: 잃거나 지나간 일을 슬퍼하고 한탄함
- 현대 의미: 후회하다, 유감으로 여기다; 후회, 유감

## 단어들

### regret

- 구성: Old French `regreter`
- 뜻: 후회하다, 유감으로 여기다; 후회, 유감
- 어원적 연결: 지나간 일이나 선택을 떠올리며 아쉬워하고 슬퍼함
- 사용 맥락: `regret the decision` = 그 결정을 후회하다

### regretful

- 구성: `regret` + `-ful`
- 뜻: 후회하는, 유감스러워하는
- 어원적 연결: 마음속에 후회와 아쉬움이 가득함
- 사용 맥락: `a regretful apology` = 후회가 담긴 사과

### regretfully

- 구성: `regretful` + `-ly`
- 뜻: 후회하며, 유감스럽게
- 어원적 연결: 후회나 아쉬움을 느끼는 방식
- 사용 맥락: `say regretfully` = 유감스럽게 말하다

### regrettable

- 구성: `regret` + `-able`
- 뜻: 유감스러운, 안타까운
- 어원적 연결: 후회나 유감을 느낄 만한 일
- 사용 맥락: `a regrettable mistake` = 유감스러운 실수

### regrettably

- 구성: `regrettable` + `-ly`
- 뜻: 유감스럽게도
- 어원적 연결: 안타깝게 여겨야 할 상황을 말할 때 사용
- 사용 맥락: `regrettably, we must cancel` = 유감스럽게도 취소해야 함

## 기억 포인트

`regret` 계열은 "지난 일에 대한 후회와 유감" 이미지.

- `regret`: 후회하다, 유감
- `regretful`: 후회하는
- `regretfully`: 후회하며
- `regrettable`: 유감스러운
- `regrettably`: 유감스럽게도

## 비교 단어

### rue

- 뜻: 후회하다
- 구분 포인트: `rue`는 문어적·간결한 표현. `regret`은 일상·공식 문맥 모두에서 넓게 씀.

### lament

- 뜻: 슬퍼하다, 한탄하다
- 구분 포인트: `lament`는 슬픔과 애도를 크게 드러냄. `regret`은 선택이나 결과에 대한 후회와 유감이 중심.

### quarrel

- 뜻: 말다툼
- 구분 포인트: [[quer]]는 불평을 말로 내놓는 계열. `regret`은 마음속 후회와 유감. 둘 다 부정적 감정 표현이지만 어원은 다름.
