# grad

- 출발 단어: `progress`
- 핵심 어원: Latin `gradus` = 걸음, 단계
- 관련 형태: Latin `gradi` = 걷다, 나아가다
- 핵심 이미지: 한 걸음씩 단계적으로 나아감
- 관련 문서: [[cess]], [[curr]]
- 주의: `grade`와 `graduate`는 "단계" 이미지, `progress`와 `regress`는 "걸어 나아감" 이미지.

## 출발 단어 분석

`progress` ← Latin `progredi`

- `pro-`: forward
- `gradi`: step, walk
- 원래 이미지: 앞으로 걸어 나아감
- 현대 의미: 진전, 발전

## 단어들

### grade

- 구성: Latin `gradus`
- 뜻: 등급, 학년, 성적
- 어원적 연결: 한 단계씩 나뉜 수준
- 사용 맥락: `get a good grade` = 좋은 성적을 받다

### gradual

- 구성: `grade` + `-ual`
- 뜻: 점진적인
- 어원적 연결: 단계별로 조금씩 진행됨
- 사용 맥락: `a gradual change` = 점진적 변화

### graduate

- 구성: Latin `gradus` + `-ate`
- 뜻: 졸업하다, 졸업생
- 어원적 연결: 한 교육 단계를 올라 마침
- 사용 맥락: `graduate from college` = 대학을 졸업하다

### graduation

- 구성: `graduate` + `-ion`
- 뜻: 졸업
- 어원적 연결: 한 단계의 완료
- 사용 맥락: `graduation ceremony` = 졸업식

### progress

- 구성: `pro-` = forward + Latin `gradi`
- 뜻: 진전, 발전하다
- 어원적 연결: 앞으로 걸어 나아감
- 사용 맥락: `make progress` = 진전을 이루다

### progressive

- 구성: `progress` + `-ive`
- 뜻: 진보적인, 점진적인
- 어원적 연결: 앞으로 나아가거나 단계적으로 발전함
- 사용 맥락: `progressive reform` = 진보적 개혁

### regress

- 구성: `re-` = back + Latin `gradi`
- 뜻: 퇴보하다, 되돌아가다
- 어원적 연결: 뒤로 걸어감
- 사용 맥락: `regress to old habits` = 예전 습관으로 퇴보하다

### aggressive

- 구성: `ad-` = toward + Latin `gradi`
- 뜻: 공격적인, 적극적인
- 어원적 연결: 상대를 향해 걸어 들어감
- 사용 맥락: `aggressive behavior` = 공격적 행동

### congress

- 구성: `con-` = together + Latin `gradi`
- 뜻: 의회, 회의
- 어원적 연결: 사람들이 함께 걸어 모임
- 사용 맥락: `the U.S. Congress` = 미국 의회

## 기억 포인트

`grad/gress` 계열은 "걸음과 단계" 이미지.

- `grade`: 단계
- `gradual`: 단계적으로
- `graduate`: 한 단계를 마침
- `progress`: 앞으로 나아감
- `regress`: 뒤로 돌아감
- `aggressive`: 상대를 향해 나아감

## 비교 단어

### cess

- 뜻: 가다, 물러나다, 양보하다
- 구분 포인트: [[cess]]는 이동 자체. `grad`는 걸음과 단계의 이미지가 강함.

### curr

- 뜻: 달리다, 흐르다
- 구분 포인트: [[curr]]는 빠르게 흐르거나 달림. `grad`는 한 걸음씩 나아감.
