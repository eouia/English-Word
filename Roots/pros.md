# pros

- 출발 단어: `prose`
- 핵심 어원: 라틴어 `prosa oratio` = 곧게 나아가는 말, 직접적인 말
- 핵심 의미: 운율에 묶이지 않고 곧게 이어지는 글
- 관련 문서: [[vers]], [[liter]], [[narr]]
- 주의: `prosody`는 그리스어 `prosodia` 계열. 철자는 비슷하지만 `prose`와 다른 어원.

## 출발 단어 분석

`prose` ← Latin `prosa oratio`

- `prosa`: 곧은, 앞으로 나아가는
- `oratio`: 말, 연설
- 원래 이미지: 운문처럼 줄 끝에서 되돌아가지 않고 곧게 이어지는 말
- 현대 의미: 산문, 보통 글말

## 단어들

### prose

- 구성: Latin `prosa oratio`
- 뜻: 산문, 보통 문장으로 쓴 글
- 어원적 연결: 운율이나 행 구분에 묶이지 않고 곧게 이어지는 말
- 사용 맥락: `write in prose` = 산문으로 쓰다

### prosaic

- 구성: `prose` + `-ic`
- 뜻: 산문적인, 평범한, 따분한
- 어원적 연결: 시적 장식 없이 평범하고 직접적인 성질
- 사용 맥락: `prosaic details` = 평범하고 지루한 세부 사항

### prosaically

- 구성: `prosaic` + `-ally`
- 뜻: 평범하게, 산문적으로
- 어원적 연결: 특별한 운율이나 장식 없이 직접적으로 말함
- 사용 맥락: `prosaically described` = 평범하게 묘사된

### prose writer

- 구성: `prose` + `writer`
- 뜻: 산문 작가
- 어원적 연결: 운문이 아닌 산문으로 글을 쓰는 사람
- 사용 맥락: `a nineteenth-century prose writer` = 19세기 산문 작가

### prose fiction

- 구성: `prose` + `fiction`
- 뜻: 산문 소설
- 어원적 연결: 운문이 아닌 산문 형식의 허구 서사
- 사용 맥락: `modern prose fiction` = 현대 산문 소설

### prose poem

- 구성: `prose` + `poem`
- 뜻: 산문시
- 어원적 연결: 겉형식은 산문이지만 시적 압축과 이미지가 강한 글
- 사용 맥락: `a short prose poem` = 짧은 산문시

## 기억 포인트

- `prose`는 “곧게 나아가는 말”.
- `verse`는 줄·운율이 있고, 원래 “돌아감” 이미지와 연결.
- `prosaic`은 “산문적인”에서 “평범한, 지루한”으로 확장.

## 비교 단어

### verse

- [[vers]]는 라틴어 `vertere` = 돌다, 돌리다 계열.
- `verse`는 원래 밭고랑이나 글줄이 끝에서 돌아가는 이미지.
- `prose`는 그런 행 구분 없이 곧게 이어지는 말.

### prosody

- `prosody`는 그리스어 `prosodia` = 노래에 맞춘 소리, 악센트.
- 운율·강세·리듬 연구를 뜻함.
- `prose`와 철자는 비슷하지만 다른 어원.
