# doom

- 출발 단어: `kingdom`
- 핵심 어원: Old English `dōm` = judgment, decree, law, authority, state
- 핵심 이미지: 판정된 지위나 권한, 그 상태에 속한 영역
- 관련 문서: [[dom]], [[reg]], [[comit]], [[dorm]]
- 주의: `kingdom`의 `-dom`은 Latin `domus/dominus` 계열이 아님. [[dom]]은 집·주인·지배자 계열, `doom/-dom`은 판정·권한·상태 계열.

## 출발 단어 분석

`kingdom` = `king` + `-dom`

- `king`: 왕
- `-dom`: rank, authority, domain, state
- 원래 이미지: 왕의 권한과 그 권한이 미치는 영역
- 의미 변화: 왕의 통치권·영역 → 왕국
- 현대 의미: 왕국, 왕정 국가; 분야·세계

## 단어들

### kingdom

- 발음: /ˈkɪŋdəm/
- 구성: `king` + Old English `-dōm` = authority, state, domain
- 뜻: 왕국, 왕정 국가; 영역
- 어원적 연결: 왕의 권한이 판정·인정된 상태와 그 영역
- 사용 맥락: `the United Kingdom` = 영국, `the animal kingdom` = 동물계

### freedom

- 발음: /ˈfridəm/
- 구성: `free` + `-dom`
- 뜻: 자유
- 어원적 연결: 자유로운 상태나 권리
- 사용 맥락: `freedom of speech` = 표현의 자유

### wisdom

- 발음: /ˈwɪzdəm/
- 구성: `wise` + `-dom`
- 뜻: 지혜
- 어원적 연결: 지혜로운 상태나 성질
- 사용 맥락: `practical wisdom` = 실천적 지혜

### boredom

- 발음: /ˈbɔrdəm/
- 구성: `bore` + `-dom`
- 뜻: 지루함
- 어원적 연결: 지루한 상태
- 사용 맥락: `relieve boredom` = 지루함을 덜다

### officialdom

- 발음: /əˈfɪʃəldəm/
- 구성: `official` + `-dom`
- 뜻: 관료 사회, 공무원 집단
- 어원적 연결: 특정 지위·집단에 속한 세계
- 사용 맥락: `local officialdom` = 지역 관료 사회

### earldom

- 발음: /ˈɝldəm/
- 구성: `earl` + `-dom`
- 뜻: 백작 지위, 백작령
- 어원적 연결: 백작의 지위와 그 권한이 미치는 영역
- 사용 맥락: `inherit an earldom` = 백작 작위를 상속하다

### Christendom

- 발음: /ˈkrɪsəndəm/
- 구성: `Christian/Christen` + `-dom`
- 뜻: 기독교 세계
- 어원적 연결: 기독교에 속한 사람들과 그 세계
- 사용 맥락: `medieval Christendom` = 중세 기독교 세계

### doom

- 발음: /ˈdum/
- 구성: Old English `dōm` = judgment, sentence
- 뜻: 운명, 파멸; 판결하다
- 어원적 연결: 내려진 판정이나 선고. 부정적 운명·파멸 의미로 좁아짐
- 사용 맥락: `meet one's doom` = 파멸을 맞다

## 기억 포인트

`doom/-dom` 계열은 "판정된 상태·권한·영역" 이미지.

- `kingdom`: 왕의 권한과 영역
- `freedom`: 자유로운 상태
- `wisdom`: 지혜로운 상태
- `officialdom`: 공무원 집단의 세계
- `earldom`: 백작 지위와 영역
- `doom`: 내려진 판정이 운명·파멸로 굳어짐

## 비교 단어

### dom

- 뜻: 집, 주인, 지배자
- 구분 포인트: [[dom]]은 Latin `domus/dominus` 계열. `kingdom`, `freedom`, `wisdom`의 `-dom`은 Old English `dōm` 계열.

### dormitory

- 뜻: 기숙사, 공동 침실
- 구분 포인트: [[dorm]]의 Latin `dormire` = sleep 계열. `domitory/domitary`처럼 보이더라도 `dom`이나 `-dom`과 분리.

### comit

- 뜻: 수행원에서 귀족 칭호와 관할지로 발전
- 구분 포인트: [[comit]]의 `county`는 백작 관할지. `kingdom`은 왕의 권한·영역을 나타내는 Germanic `-dom`.
