# pot

- 출발 단어: `omnipotent`
- 핵심 어원: Latin `potere`, `posse` = 할 수 있다, 힘이 있다
- 핵심 이미지: 어떤 일을 할 힘이나 가능성을 가짐
- 관련 문서: [[omni]], [[val]], [[virt]]

## 단어들

### omnipotent

- 구성: `omni-` = all + Latin `potens` = powerful
- 뜻: 전능한, 모든 힘을 가진
- 어원적 연결: 모든 것을 할 수 있는 힘
- 사용 맥락: `an omnipotent ruler` = 절대 권력을 가진 통치자

### omnipotence

- 구성: `omni-` + `potence`
- 뜻: 전능, 절대적 권능
- 어원적 연결: 모든 힘을 가진 상태
- 사용 맥락: 신의 능력이나 절대 권력을 말할 때 사용

### potent

- 구성: Latin `potens` = powerful
- 뜻: 강력한, 효력이 큰
- 어원적 연결: 실제 효과를 낼 힘이 있음
- 사용 맥락: `a potent argument` = 강력한 주장

### potency

- 구성: `potent` + `-cy`
- 뜻: 힘, 효력, 영향력
- 어원적 연결: 힘이 작동하는 정도
- 사용 맥락: `the potency of a medicine` = 약의 효능

### potential

- 구성: Latin `potentialis` = possible, powerful
- 뜻: 잠재적인; 가능성
- 어원적 연결: 아직 드러나지 않았지만 할 수 있는 힘
- 사용 맥락: `a potential problem` = 잠재적 문제

### potentially

- 구성: `potential` + `-ly`
- 뜻: 잠재적으로, 가능성 있게
- 어원적 연결: 어떤 일이 일어날 힘이나 가능성이 있음
- 사용 맥락: `potentially harmful` = 잠재적으로 해로운

### impotent

- 구성: `im-` = not + Latin `potens`
- 뜻: 무력한, 힘이 없는
- 어원적 연결: 일을 해낼 힘이 없음
- 사용 맥락: `feel impotent in a crisis` = 위기 속에서 무력감을 느낌

### possible

- 구성: Latin `possibilis` < `posse` = be able
- 뜻: 가능한
- 어원적 연결: 할 수 있는 상태
- 사용 맥락: `as soon as possible` = 가능한 한 빨리

### impossible

- 구성: `im-` = not + `possible`
- 뜻: 불가능한
- 어원적 연결: 할 수 있는 힘이나 조건이 없음
- 사용 맥락: `an impossible task` = 불가능한 일

### possibility

- 구성: `possible` + `-ity`
- 뜻: 가능성
- 어원적 연결: 일이 일어날 수 있는 상태
- 사용 맥락: `a real possibility` = 실제 가능성

### power

- 구성: Old French `poeir`, `povoir` < Latin `potere`
- 뜻: 힘, 권력, 능력
- 어원적 연결: 무엇을 할 수 있는 능력
- 사용 맥락: `political power` = 정치 권력

### potentate

- 구성: Latin `potentatus` = rulership, power
- 뜻: 권력자, 군주
- 어원적 연결: 힘과 권력을 가진 사람
- 사용 맥락: 격식 있는 문맥에서 강한 통치자를 가리킴

## 기억 포인트

`pot` 계열은 "할 수 있는 힘" 이미지.

- `omnipotent`: 모든 힘을 가진
- `potent`: 강력한
- `potential`: 아직 드러나지 않은 힘
- `impotent`: 힘이 없는
- `possible`: 할 수 있는
- `power`: 힘, 권력

## 비교 단어

### omni

- 뜻: 모두, 모든 것
- 구분 포인트: [[omni]]는 범위. `omnipotent`에서 `potent`가 힘의 핵심.

### val

- 뜻: 힘이 있다, 가치가 있다, 유효하다
- 구분 포인트: [[val]]은 가치나 효력이 "통함"에 가까움. `pot`은 할 수 있는 능력과 권력.

### virt

- 뜻: 힘, 뛰어남, 덕, 실제 효과
- 구분 포인트: [[virt]]는 뛰어남과 덕성으로 확장. `pot`은 가능성과 힘 자체.
