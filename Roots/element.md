# element

- 출발 단어: `element`
- 핵심 어원: Latin `elementum` = first principle, rudiment, basic constituent
- 핵심 이미지: 더 큰 것을 이루는 기본 구성 요소
- 관련 어근: [[part]], [[piece]], [[fract]]

## 출발 단어 분석

`element`는 Old French `element`를 거쳐 Latin `elementum`에서 온 말.

Latin `elementum`은 "첫 원리, 기초, 기본 구성 요소, 알파벳 글자"의 뜻.

다만 `elementum` 자체의 더 깊은 기원은 불확실함. 따라서 `el-`, `ment`처럼 현대 형태로 무리하게 쪼개지 않음.

## 단어들

### element

- 발음: /ˈɛləmənt/
- 구성: Latin `elementum`
- 의미: 요소, 원소, 기본 부분
- 어원적 연결: 전체를 이루는 가장 기본적인 구성 요소
- 사용 맥락: `a key element` = 핵심 요소

### elemental

- 발음: /ˌɛləˈmɛntəl/
- 구성: `element` + `-al`
- 의미: 기본적인, 원소의, 자연력의
- 어원적 연결: 기본 요소와 관련됨
- 사용 맥락: `elemental forces` = 자연의 기본 힘

### elementary

- 발음: /ˌɛləˈmɛntri/
- 구성: `element` + `-ary`
- 의미: 기초의, 초급의
- 어원적 연결: 가장 기본 요소부터 배우는 단계
- 사용 맥락: `elementary education` = 초등 교육

### chemical element

- 구성: `chemical` + `element`
- 의미: 화학 원소
- 어원적 연결: 화학적으로 더 단순한 물질로 나누기 어려운 기본 물질
- 사용 맥락: `oxygen is a chemical element` = 산소는 화학 원소다

### elementarily

- 구성: `elementary` + `-ly`
- 의미: 기초적으로, 초보적으로
- 어원적 연결: 기본 단계의 방식으로
- 사용 맥락: 시험 빈도는 낮아 참고만 함

## 기억 포인트

- `element`: 기본 요소, 원소
- `elemental`: 기본 요소의, 자연력의
- `elementary`: 기초의, 초급의
- `chemical element`: 화학 원소

## 비교 단어

- [[part]]
  - 핵심: 부분, 몫
  - 구분 포인트: `part`는 전체를 이루는 일반적 부분. `element`는 더 기본적인 구성 요소

- [[piece]]
  - 핵심: 조각, 한 단위
  - 구분 포인트: `piece`는 떨어져 나온 조각. `element`는 구조나 체계의 기본 성분

- [[fract]]
  - 핵심: 깨져 나뉨
  - 구분 포인트: `fragment`는 깨진 조각. `element`는 원래부터 기본 구성 요소로 보는 단위

## 주의

`element`의 깊은 어원은 불확실함.

`L-M-N` 글자 이름에서 왔다는 식의 설명은 전통적으로 언급되지만 확실한 어원으로 단정하지 않음.
