# comit

- 출발 단어: `county`
- 핵심 어원: Latin `comes` / `comitem` = companion, attendant
- 핵심 이미지: 권력자와 함께 가는 수행원에서 관직·귀족 칭호로 발전
- 관련 문서: [[count]], [[mark]], [[consul]]
- 주의: `count` = 세다, 계산하다는 [[count]]의 Latin `computare` 계열. `county`, 귀족 칭호 `count`는 Latin `comes/comitem` 계열로 분리.

## 출발 단어 분석

`county` ← Anglo-French `counté` = territory governed by a count

- `count`: 귀족 칭호, count
- `-y`: territory, domain 느낌의 명사형
- 원래 이미지: `count`가 맡은 관할 지역
- 의미 변화: 귀족의 영지 → 행정 구역
- 현대 의미: 군, 카운티; 영국·미국 등의 지방 행정 구역

## 단어들

### county

- 발음: /ˈkaʊnti/
- 구성: Anglo-French `counté` = region under a count
- 뜻: 군, 카운티, 지방 행정 구역
- 어원적 연결: 원래 `count`가 다스리던 지역
- 사용 맥락: `Los Angeles County` = 로스앤젤레스 카운티

### count

- 발음: /ˈkaʊnt/
- 구성: Old French `conte/comte` ← Latin `comitem` = companion, attendant
- 뜻: 백작, count
- 어원적 연결: 원래 권력자의 수행원·동료였던 사람이 관직·귀족 칭호로 발전
- 사용 맥락: `Count Dracula` = 드라큘라 백작

### countess

- 발음: /ˈkaʊntəs/
- 구성: `count` + `-ess`
- 뜻: 백작부인, 여성 백작
- 어원적 연결: `count`의 여성형 또는 배우자 칭호
- 사용 맥락: `a countess of noble birth` = 귀족 출신의 백작부인

### viscount

- 발음: /ˈvɪskaʊnt/
- 구성: Old French `visconte` ← Latin `vicecomes` = deputy count
- 뜻: 자작
- 어원적 연결: `vice-` = in place of + `count`; 원래 백작을 대신하던 관직
- 사용 맥락: `a British viscount` = 영국 자작

### viscountess

- 구성: `viscount` + `-ess`
- 뜻: 자작부인, 여성 자작
- 어원적 연결: `viscount`의 여성형 또는 배우자 칭호
- 사용 맥락: `the Viscountess Astor` = 애스터 자작부인

### comital

- 구성: Latin `comit-` = count + `-al`
- 뜻: 백작의, 백작령의
- 어원적 연결: `count`나 `county`에 속한 성질
- 사용 맥락: `comital authority` = 백작의 권한

### committee

- 발음: /kəˈmɪti/
- 구성: 다른 갈래. `commit` + `-ee`
- 뜻: 위원회
- 어원적 연결: Latin `committere` = entrust, join, send together 계열
- 사용 맥락: `form a committee` = 위원회를 구성하다
- 주의: 철자는 `comit-`처럼 보이지만 `comes/comitem` 계열이 아니라 [[mit]] 계열.

## 기억 포인트

`comit` 계열은 "함께 가는 수행원에서 귀족·관할지" 이미지.

- `count`: 원래 권력자의 수행원·동료 → 백작
- `county`: 백작이 다스리던 지역 → 행정 구역
- `countess`: 백작부인
- `viscount`: 백작 대리에서 온 자작
- `comital`: 백작의, 백작령의

## 비교 단어

### count

- 뜻: 세다, 계산하다
- 구분 포인트: [[count]]는 Latin `computare` = count, reckon 계열. `county`의 `count`는 Latin `comes/comitem` = companion 계열.

### mark

- 뜻: 경계, 변경 지대
- 구분 포인트: [[mark]]의 `margrave`는 변경백. `county`는 백작 관할지 일반.

### mit

- 뜻: 보내다, 맡기다
- 구분 포인트: [[mit]]는 `commit`, `committee` 계열. 철자가 비슷해도 `comit` = count/county 계열과 분리.
