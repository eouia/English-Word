# ject

- 출발 단어: `project`
- 핵심 어원: 라틴어 `jacere` / `ject-` = 던지다
- 핵심 이미지: 어떤 방향으로 던져 놓음
- 관련해서 헷갈릴 수 있는 어근: [[pos]] = 놓다, 두다 계열, [[mit]] = 보내다 계열

## 출발 단어 분석

`project`는 `pro-` + `ject`.

- `pro-`: forward
- `ject`: throw

따라서 `project`는 "앞으로 던지다"에서 "계획, 과제, 투사하다, 예상하다"로 확장됨.

## 단어들

### project

- 구성: `pro-` = forward + `ject`
- 뜻: 프로젝트, 계획, 투사하다, 예상하다
- 어원적 연결: 앞으로 던져 놓은 일이나 생각
- 사용 맥락: `launch a project` = 프로젝트를 시작하다

### projection

- 구성: `project` + `-ion`
- 뜻: 예측, 투사, 돌출
- 어원적 연결: 앞으로 던져 보이는 것
- 사용 맥락: `sales projections` = 매출 전망

### reject

- 구성: `re-` = back + `ject`
- 뜻: 거절하다, 거부하다
- 어원적 연결: 뒤로 던져 돌려보냄
- 사용 맥락: `reject an offer` = 제안을 거절하다

### rejection

- 구성: `reject` + `-ion`
- 뜻: 거절, 거부
- 어원적 연결: 받아들이지 않고 던져냄
- 사용 맥락: `fear of rejection` = 거절에 대한 두려움

### object

- 구성: `ob-` = against, before + `ject`
- 뜻: 물체, 반대하다
- 어원적 연결: 앞에 던져 놓인 것 / 맞서 던지는 말
- 사용 맥락: `object to the plan` = 계획에 반대하다

### objection

- 구성: `object` + `-ion`
- 뜻: 반대, 이의
- 어원적 연결: 앞에 던져 놓는 반대 의견
- 사용 맥락: `raise an objection` = 이의를 제기하다

### subject

- 구성: `sub-` = under + `ject`
- 뜻: 주제, 과목, 대상, 종속시키다
- 어원적 연결: 아래에 던져 놓인 것. 논의나 지배 아래 놓인 대상
- 사용 맥락: `the subject of the essay` = 글의 주제

### subjective

- 구성: `subject` + `-ive`
- 뜻: 주관적인
- 어원적 연결: 판단의 주체 안에 놓임
- 사용 맥락: `a subjective opinion` = 주관적 의견

### interject

- 구성: [[inter]] = between + `ject`
- 뜻: 끼어들어 말하다, 불쑥 말하다
- 어원적 연결: 대화 사이에 말을 던져 넣음
- 사용 맥락: `interject a comment` = 말을 끼워 넣다

### interjection

- 구성: `interject` + `-ion`
- 뜻: 감탄사, 끼어드는 말
- 어원적 연결: 문장이나 대화 사이에 던져 넣는 말
- 사용 맥락: `Oh!` and `Wow!` are interjections. = `Oh!`, `Wow!`는 감탄사

### inject

- 구성: [[in]] = into + `ject`
- 뜻: 주입하다
- 어원적 연결: 안으로 던져 넣음
- 사용 맥락: `inject a vaccine` = 백신을 주사하다

### eject

- 구성: [[ex]] = out + `ject`
- 뜻: 쫓아내다, 배출하다
- 어원적 연결: 밖으로 던져냄
- 사용 맥락: `eject a disk` = 디스크를 꺼내다

### dejected

- 구성: `de-` = down + `ject`
- 뜻: 낙담한
- 어원적 연결: 아래로 던져진 듯 기가 꺾임
- 사용 맥락: `look dejected` = 낙담해 보이다

## 기억 포인트

`ject` 계열은 "던지다" 이미지.

- `project`: 앞으로 던짐
- `reject`: 뒤로 던져 거절함
- `object`: 앞에 던져 놓인 것 / 반대함
- `subject`: 아래에 던져 놓인 대상
- `interject`: 사이에 던져 넣음
- `inject`: 안으로 던져 넣음
- `eject`: 밖으로 던져냄
