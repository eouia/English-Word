# anx

- 출발 단어: `anxiety`
- 핵심 어원: Latin `angere` = 조이다, 목을 조르다, 괴롭히다
- 관련 형태: Latin `anxius` = 불안한, 괴로운
- 깊은 어근: PIE `*angh-` = tight, painfully constricted
- 핵심 이미지: 좁게 조여 숨 막히는 압박감
- 관련 문서: [[strain]], [[press]]
- 주의: `angle`의 [[angl]]은 그리스어 `ankylos` 계열의 각/굽음. `anx`의 불안·고통 계열과 학습상 분리.

## 출발 단어 분석

`anxiety` ← Latin `anxietas` ← `anxius`

- `angere`: 조이다, 목을 조르다, 괴롭히다
- `anxius`: 마음이 조여 불안한
- 원래 이미지: 숨이 막힐 만큼 조이는 느낌
- 현대 의미: 불안, 걱정

## 단어들

### anxiety

- 구성: Latin `anxietas`
- 뜻: 불안, 걱정
- 어원적 연결: 마음이 조여 답답한 상태
- 사용 맥락: `test anxiety` = 시험 불안

### anxious

- 구성: Latin `anxius`
- 뜻: 불안한, 걱정하는, 열망하는
- 어원적 연결: 마음이 조여 초조함
- 사용 맥락: `anxious about the future` = 미래를 걱정하는

### anxiously

- 구성: `anxious` + `-ly`
- 뜻: 불안하게, 초조하게
- 어원적 연결: 마음이 조인 상태로
- 사용 맥락: `wait anxiously` = 초조하게 기다리다

### anguish

- 구성: Old French `angoisse` ← Latin `angustia`
- 뜻: 극심한 고통, 괴로움
- 어원적 연결: 좁게 조이는 압박이 고통으로 확장
- 사용 맥락: `mental anguish` = 정신적 고통

### angst

- 구성: German `Angst`
- 뜻: 불안, 존재론적 불안
- 어원적 연결: 같은 깊은 "조임, 답답함" 이미지
- 사용 맥락: `teenage angst` = 십대의 불안감

### anger

- 구성: Old Norse `angr` = grief, sorrow
- 뜻: 분노
- 어원적 연결: 원래는 괴로움과 슬픔의 압박감. 현대 영어에서 분노 중심으로 이동
- 사용 맥락: `express anger` = 분노를 표현하다

### angry

- 구성: `anger` + `-y`
- 뜻: 화난
- 어원적 연결: 압박된 괴로움이 밖으로 터지는 감정
- 사용 맥락: `angry customers` = 화난 고객들

### angina

- 구성: Latin `angina` ← `angere`
- 뜻: 협심증, 목 염증
- 어원적 연결: 조이는 통증이나 목이 막히는 느낌
- 사용 맥락: `angina pain` = 협심증 통증

## 기억 포인트

`anx` 계열은 "조여서 답답함" 이미지.

- `anxiety`: 마음이 조이는 불안
- `anxious`: 조여 초조한
- `anguish`: 극심하게 조이는 고통
- `angst`: 깊은 불안
- `anger`: 괴로움에서 분노로 이동

## 비교 단어

### angl

- 뜻: 각, 모서리
- 구분 포인트: [[angl]]은 각도와 꺾임. `anx`는 조임과 불안.
