# nunc

- 출발 단어: `announce`
- 핵심 어원: Latin `nuntiare` = announce, report
- 관련 형태: `nunci-`, `nounce`
- 핵심 이미지: 소식을 전하다, 공식적으로 말해 알리다
- 관련 문서: [[voc]], [[dict]]
- 주의: `nounce`는 독립 단어가 아니라 `announce`, `pronounce`, `denounce` 등에 남은 형태.

## 출발 단어 분석

`announce` = `an-` + `nounce`

- `an-`: to, toward
- `nounce`: announce, report 계열
- 원래 이미지: 어떤 소식을 사람들에게 전함
- 현대 의미: 발표하다, 알리다

## 단어들

### announce

- 발음: /əˈnaʊns/
- 구성: Latin `annuntiare` = report, announce
- 뜻: 발표하다, 알리다
- 어원적 연결: 소식을 공식적으로 전함
- 사용 맥락: `announce the results` = 결과를 발표하다

### announcement

- 발음: /əˈnaʊnsmənt/
- 구성: `announce` + `-ment`
- 뜻: 발표, 공지, 안내 방송
- 어원적 연결: 공식적으로 알린 내용
- 사용 맥락: `make an announcement` = 발표하다

### announcer

- 발음: /əˈnaʊnsɚ/
- 구성: `announce` + `-er`
- 뜻: 아나운서, 방송 진행자
- 어원적 연결: 소식이나 정보를 말로 전하는 사람
- 사용 맥락: `a radio announcer` = 라디오 아나운서

### pronounce

- 발음: /prəˈnaʊns/
- 구성: `pro-` = forth + Latin `nuntiare` 계열
- 뜻: 발음하다, 선언하다
- 어원적 연결: 소리나 판단을 밖으로 말해 냄
- 사용 맥락: `pronounce a word` = 단어를 발음하다

### pronunciation

- 발음: /proʊˌnʌnsiˈeɪʃən/
- 구성: `pronounce` + `-iation`
- 뜻: 발음
- 어원적 연결: 밖으로 말해 내는 방식
- 사용 맥락: `English pronunciation` = 영어 발음

### denounce

- 발음: /dɪˈnaʊns/
- 구성: `de-` = down, away + Latin `nuntiare`
- 뜻: 공개적으로 비난하다, 고발하다
- 어원적 연결: 나쁜 것으로 공식적으로 말해 알림
- 사용 맥락: `denounce corruption` = 부패를 공개 비난하다

### denunciation

- 발음: /dɪˌnʌnsiˈeɪʃən/
- 구성: `denounce` + `-iation`
- 뜻: 공개 비난, 고발
- 어원적 연결: 공식적으로 비난해 알리는 행위
- 사용 맥락: `a public denunciation` = 공개 비난

### renounce

- 발음: /rɪˈnaʊns/
- 구성: `re-` = back, away + Latin `nuntiare`
- 뜻: 포기하다, 단념하다, 관계를 끊겠다고 선언하다
- 어원적 연결: 더 이상 따르지 않겠다고 공식적으로 말함
- 사용 맥락: `renounce a claim` = 권리를 포기하다

### renunciation

- 발음: /rɪˌnʌnsiˈeɪʃən/
- 구성: `renounce` + `-iation`
- 뜻: 포기, 단념, 관계 단절 선언
- 어원적 연결: 공식적으로 버리겠다고 말하는 행위
- 사용 맥락: `renunciation of violence` = 폭력 포기 선언

## 기억 포인트

`nunc` 계열은 "말해 알림, 공식 선언" 이미지.

- `announce`: 발표하다
- `announcement`: 공지, 발표
- `pronounce`: 밖으로 말하다 → 발음하다, 선언하다
- `denounce`: 공개적으로 비난하다
- `renounce`: 포기한다고 선언하다

## 비교 단어

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 소리 내어 부름. `nunc`는 소식·판단·입장을 공식적으로 알림.

### dict

- 뜻: 말하다, 선언하다
- 구분 포인트: [[dict]]는 말로 정하거나 명령하는 이미지. `nunc`는 알림과 전달 이미지가 강함.
