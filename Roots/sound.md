# sound

- 출발 단어: `sound`
- 핵심 어원: 여러 갈래의 동형이의어
- 핵심 이미지: 소리 / 온전함 / 깊이를 재어 봄
- 관련 문서: [[cant]]
- 주의: `sound`는 뜻에 따라 어원이 다름. 한 어근으로 무리하게 합치지 않음.

## 출발 단어 분석

`sound`는 현대 영어에서 세 갈래가 중요함.

- `sound` = 소리: Old French `son` ← Latin `sonus`
- `sound` = 건전한, 온전한: Old English `gesund`
- `sound` = 깊이를 재다: 물의 깊이를 재는 항해 용어

따라서 문맥에 따라 "소리", "건전함", "측정"으로 나누어 읽어야 함.

## 갈래 1: 소리

### sound

- 구성: Old French `son` ← Latin `sonus`
- 뜻: 소리, 음향; 소리를 내다
- 어원적 연결: 귀에 들리는 진동이나 울림
- 사용 맥락: `the sound of rain` = 빗소리

### resound

- 구성: `re-` = back/again + `sound`
- 뜻: 울려 퍼지다, 반향하다
- 어원적 연결: 소리가 다시 울려 돌아옴
- 사용 맥락: `applause resounded through the hall` = 박수가 홀에 울려 퍼졌다

### sonic

- 구성: Latin `sonus` + `-ic`
- 뜻: 소리의, 음속의
- 어원적 연결: 소리나 음파와 관련됨
- 사용 맥락: `sonic waves` = 음파

### consonant

- 구성: `con-` = together + Latin `sonare` = sound
- 뜻: 자음; 조화되는
- 어원적 연결: 모음과 함께 소리나는 글자, 또는 함께 울려 어울림
- 사용 맥락: `a consonant sound` = 자음 소리

## 갈래 2: 온전함

### sound

- 구성: Old English `gesund`
- 뜻: 건강한, 온전한, 견실한, 타당한
- 어원적 연결: 손상되지 않고 제대로 기능하는 상태
- 사용 맥락: `sound advice` = 타당한 조언, `a sound body` = 건강한 몸

### soundly

- 구성: `sound` + `-ly`
- 뜻: 푹, 깊이; 타당하게
- 어원적 연결: 안정되고 온전한 방식
- 사용 맥락: `sleep soundly` = 푹 자다

### unsound

- 구성: `un-` = not + `sound`
- 뜻: 불건전한, 결함 있는, 타당하지 않은
- 어원적 연결: 온전하지 않거나 논리적으로 단단하지 않음
- 사용 맥락: `an unsound argument` = 타당하지 않은 주장

## 갈래 3: 깊이 측정

### sound

- 구성: 항해 용어. 물의 깊이를 재는 동사
- 뜻: 수심을 재다, 탐색하다
- 어원적 연결: 납줄 등을 내려 깊이를 확인함
- 사용 맥락: `sound the depth` = 수심을 재다

### sounding

- 구성: `sound` + `-ing`
- 뜻: 수심 측정; 탐색
- 어원적 연결: 깊이를 재거나 상태를 알아봄
- 사용 맥락: `take soundings` = 수심을 재다, 여론을 탐색하다

## 기억 포인트

`sound`는 세 갈래.

- 소리: `sound`, `resound`, `sonic`, `consonant`
- 온전함: `sound advice`, `soundly`, `unsound`
- 깊이 측정: `sound the depth`, `sounding`

## 비교 단어

### cant

- 뜻: 노래하듯 말하거나 외움
- 구분 포인트: [[cant]]는 성가·구호·주문처럼 형식 있는 목소리. `sound`는 소리 일반.

### whole

- 뜻: 전체의, 온전한
- 구분 포인트: `sound` "건전한"은 손상 없고 믿을 만한 상태. `whole`은 빠진 부분 없이 전체인 상태.
