# pand

- 출발 단어: `expand`
- 핵심 어원: 라틴어 `pandere` / `pass-` = 펴다, 펼치다
- 핵심 이미지: 접힌 것을 넓게 펼침
- 관련해서 헷갈릴 수 있는 어근: [[tend]] = 뻗다, 향하다 계열

## 출발 단어 분석

`expand`는 `ex-` + `pandere`.

- `ex-`: out
- `pandere`: spread, unfold

따라서 `expand`는 "밖으로 펼치다" → "확장하다, 팽창하다"의 뜻.

## 단어들

### expand

- 구성: `ex-` = out + `pand`
- 뜻: 확장하다, 팽창하다, 넓히다
- 어원적 연결: 밖으로 펼쳐 넓어짐
- 사용 맥락: `expand the business` = 사업을 확장하다

### expansion

- 구성: `ex-` + `pans` + `-ion`
- 뜻: 확장, 팽창
- 어원적 연결: 펼쳐져 커지는 과정
- 사용 맥락: `economic expansion` = 경제 확장

### expansive

- 구성: `expand` + `-ive`
- 뜻: 광대한, 포괄적인, 거리낌 없는
- 어원적 연결: 넓게 펼쳐진 상태
- 사용 맥락: `an expansive view` = 탁 트인 전망

### expanse

- 구성: Latin `expansum` = spread out
- 뜻: 넓게 펼쳐진 공간
- 어원적 연결: 넓게 펼쳐져 있는 것
- 사용 맥락: `a vast expanse of desert` = 광대한 사막

### compass

- 구성: `com-` = together + `pass`
- 뜻: 나침반, 범위, 둘러싸다
- 어원적 연결: 함께 둘러 펼쳐 범위를 정함
- 사용 맥락: `within the compass of the law` = 법의 범위 안에서

### encompass

- 구성: `en-` = in + `compass`
- 뜻: 포함하다, 둘러싸다
- 어원적 연결: 둘러 펼쳐 안에 넣음
- 사용 맥락: `encompass many topics` = 많은 주제를 포괄하다

### pace

- 구성: Latin `passus` = step, stretch of the leg
- 뜻: 걸음, 속도
- 어원적 연결: 다리를 펼쳐 내딛는 한 걸음
- 사용 맥락: `at a fast pace` = 빠른 속도로

### pass

- 구성: Latin `passus` = step
- 뜻: 지나가다, 통과하다
- 어원적 연결: 발을 펼쳐 내딛어 지나감
- 사용 맥락: `pass through the gate` = 문을 통과하다

### bypass

- 구성: `by` = beside, around + `pass`
- 뜻: 우회하다, 건너뛰다; 우회로
- 어원적 연결: 정해진 길이나 절차를 옆으로 지나감
- 사용 맥락: `bypass the usual procedure` = 일반 절차를 우회하다

### bypassed

- 구성: `bypass` + `-ed`
- 뜻: 우회된, 건너뛰어진
- 어원적 연결: 중심 경로나 절차를 지나치고 옆길로 간 상태
- 사용 맥락: `feel bypassed` = 배제되거나 건너뛰어진 느낌을 받다

### bypassing

- 구성: `bypass` + `-ing`
- 뜻: 우회, 건너뛰기
- 어원적 연결: 직접 통과하지 않고 옆으로 지나가는 과정
- 사용 맥락: `bypassing regulations` = 규정을 우회함

### past

- 구성: Middle English `passed/past`, from `pass`
- 뜻: 과거, 지나간; 지난
- 어원적 연결: 이미 지나가 버린 시간
- 사용 맥락: `in the past` = 과거에

### passed

- 구성: `pass` + `-ed`
- 뜻: 지나갔다, 통과했다
- 어원적 연결: 어떤 지점이나 시간을 넘어 지나감
- 사용 맥락: `years passed` = 세월이 흘렀다

### passing

- 구성: `pass` + `-ing`
- 뜻: 지나가는, 일시적인; 통과
- 어원적 연결: 머물지 않고 지나가는 상태
- 사용 맥락: `a passing thought` = 스쳐 지나가는 생각

### passage

- 구성: `pass` + `-age`
- 뜻: 통로, 지문, 통과
- 어원적 연결: 지나가는 길이나 과정
- 사용 맥락: `a reading passage` = 독해 지문

### passport

- 구성: `pass` + `port`
- 뜻: 여권
- 어원적 연결: 원래 항구나 국경을 통과하게 해 주는 문서
- 사용 맥락: `show your passport` = 여권을 보여 주다

### surpass

- 구성: `sur-` = over/beyond + `pass`
- 뜻: 능가하다, 뛰어넘다
- 어원적 연결: 기준을 지나 더 멀리 넘어감
- 사용 맥락: `surpass expectations` = 기대를 뛰어넘다

### trespass

- 구성: Old French `trespasser` = pass beyond
- 뜻: 무단 침입하다, 침해하다
- 어원적 연결: 허용된 경계를 넘어 지나감
- 사용 맥락: `No Trespassing` = 무단 출입 금지

### pastime

- 구성: `pass` + `time`
- 뜻: 오락, 취미
- 어원적 연결: 시간을 지나가게 하는 활동
- 사용 맥락: `a popular pastime` = 인기 있는 취미

### passion

- 구성: Latin `pati` / `passus` = suffer, endure
- 뜻: 열정, 격정, 고난
- 어원적 연결: 직접 `pandere`의 "펼치다"와는 갈래가 다르지만, 역사형 `pass-`가 겹침. 원래는 겪음, 고난 → 강한 감정
- 사용 맥락: `a passion for music` = 음악에 대한 열정

### compassion

- 구성: `com-` = with + `passion`
- 뜻: 연민, 동정
- 어원적 연결: 함께 고통을 겪는 마음
- 사용 맥락: `show compassion` = 연민을 보이다

## 기억 포인트

`pand` 계열은 "펼침" 이미지.

- `expand`: 밖으로 펼쳐 커짐
- `expansion`: 펼쳐져 커지는 과정
- `expanse`: 넓게 펼쳐진 공간
- `compass`: 둘러 펼친 범위
- `encompass`: 둘러싸 포함함
- `bypass`: 옆으로 지나 우회함

`passion` / `compassion`은 `pati` = 겪다 계열. 철자 `pass-` 때문에 함께 볼 수 있지만, `expand`와 직접 같은 의미선으로 외우지는 않음.

## 비교 단어

### tend

- 뜻: 뻗다, 향하다
- 구분 포인트: [[tend]]의 `extend`는 길게 뻗음. `expand`는 접힌 것이 넓게 펼쳐지며 커짐

### broad / spread

- 뜻: 넓은 / 펼치다
- 구분 포인트: 게르만계 기본어. `pand`는 `expand`, `expansion`, `expanse` 같은 라틴계 시험어휘에서 보임
