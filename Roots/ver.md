# ver

- 출발 단어: `verify`
- 핵심 어원: Latin `verus` = 참된, 진짜의
- 관련 형태: `ver-`, `veri-`, `verit-`
- 핵심 이미지: 참인지 확인하고 진실로 드러냄
- 관련 문서: [[fals]], [[cred]], [[dict]]
- 주의: `very`도 원래 "진실한"에서 "정말, 매우"로 의미가 강화된 말.

## 출발 단어 분석

`verify` ← Latin `verus` + `facere`

- `verus`: true
- `facere`: make
- 원래 이미지: 참으로 만들다, 참임을 확인하다
- 현대 의미: 확인하다, 검증하다

## 단어들

### verify

- 구성: `verus` = true + [[fac]] = make
- 뜻: 확인하다, 검증하다
- 어원적 연결: 참인지 확인해 사실로 세움
- 사용 맥락: `verify the information` = 정보를 확인하다

### verification

- 구성: `verify` + `-ication`
- 뜻: 확인, 검증
- 어원적 연결: 참임을 확인하는 과정
- 사용 맥락: `identity verification` = 신원 확인

### verity

- 구성: Latin `veritas` = truth
- 뜻: 진리, 참
- 어원적 연결: 참된 것
- 사용 맥락: 격식적·문학적 맥락에서 쓰임

### veritable

- 구성: Latin `veritas` + `-able`
- 뜻: 진정한, 그야말로
- 어원적 연결: 참이라고 할 만함
- 사용 맥락: `a veritable feast` = 그야말로 진수성찬

### verisimilitude

- 구성: `veri-` = true + `similis` = similar
- 뜻: 그럴듯함, 진실 같음
- 어원적 연결: 참과 비슷해 보임
- 사용 맥락: 문학·비평에서 현실감이나 그럴듯함을 말함

### aver

- 구성: Old French `averer` ← Latin `verus`
- 뜻: 단언하다, 주장하다
- 어원적 연결: 참이라고 강하게 말함
- 사용 맥락: 법률·격식 문맥에서 쓰임

### verdict

- 구성: `verus` = true + [[dict]] = say
- 뜻: 평결
- 어원적 연결: 배심원이 참이라고 말한 판단
- 사용 맥락: `reach a verdict` = 평결에 이르다

### very

- 구성: Old French `verai` ← Latin `verus`
- 뜻: 매우, 바로 그
- 어원적 연결: 참된 것에서 "정말로"의 강조 의미로 확장
- 사용 맥락: `the very first day` = 바로 첫날

## 기억 포인트

`ver` 계열은 "참, 진짜" 이미지.

- `verify`: 참인지 확인함
- `verification`: 검증
- `verity`: 진리
- `veritable`: 진정한
- `verisimilitude`: 진실 같음
- `verdict`: 참이라고 말한 판단
- `very`: 정말로, 매우

## 비교 단어

### fals

- 뜻: 거짓, 오류
- 구분 포인트: [[fals]]는 거짓과 속임. `ver`는 참과 검증.

### cred

- 뜻: 믿다
- 구분 포인트: [[cred]]는 믿음. `ver`는 믿을 만한지 참으로 확인함.
