# buttress

- 출발 단어: `buttress`
- 핵심 어원: Old French `boteret` / `buter` = thrust, support
- 핵심 이미지: 벽을 밖에서 밀어 받치는 구조물
- 관련 문서: [[port]], [[ten]]
- 주의: `buttress`는 현대 형태에서 `butt` + `-ress`처럼 생산적으로 나누기보다, 건축의 받침 구조에서 온 단어로 보는 편이 안전함.

## 출발 단어 분석

`buttress` ← Old French `boteret` / `buter`

- `buter`: to thrust, strike, support
- 원래 이미지: 벽이나 구조물이 무너지지 않도록 바깥에서 밀어 받침
- 의미 변화: 건축의 버팀벽 → 주장·제도·권위를 받쳐 주는 근거
- 현대 의미: 지지하다, 강화하다; 버팀벽

## 단어들

### buttress

- 발음: /ˈbʌtrəs/
- 구성: Old French `buter` = thrust, support 계열
- 뜻: 지지하다, 보강하다; 버팀벽
- 어원적 연결: 약한 벽을 밖에서 밀어 받쳐 무너지지 않게 함
- 사용 맥락: `buttress an argument` = 주장을 뒷받침하다

### buttressed

- 발음: /ˈbʌtrəst/
- 구성: `buttress` + `-ed`
- 뜻: 지지된, 보강된
- 어원적 연결: 근거나 구조물이 받쳐 준 상태
- 사용 맥락: `buttressed by evidence` = 증거로 뒷받침된

### buttressing

- 발음: /ˈbʌtrəsɪŋ/
- 구성: `buttress` + `-ing`
- 뜻: 지지, 보강; 버팀 구조
- 어원적 연결: 무너지지 않도록 받쳐 주는 과정이나 장치
- 사용 맥락: `external buttressing` = 외부 보강 구조

### flying buttress

- 구성: `flying` + `buttress`
- 뜻: 공중 부벽, 플라잉 버트레스
- 어원적 연결: 벽에서 떨어진 지지대가 아치처럼 힘을 전달해 받침
- 사용 맥락: 고딕 성당 건축 설명에서 자주 씀

## 기억 포인트

`buttress` 계열은 "밖에서 밀어 받쳐 무너지지 않게 함" 이미지.

- `buttress`: 지지하다, 버팀벽
- `buttressed`: 뒷받침된, 보강된
- `buttressing`: 지지, 보강
- `flying buttress`: 공중 부벽

## 비교 단어

### support

- 뜻: 지지하다, 지원하다
- 구분 포인트: [[port]] 계열로 `sub-` = under + `portare` = carry. 아래에서 받쳐 나르는 일반적 지지.

### sustain

- 뜻: 지탱하다, 지속하다
- 구분 포인트: [[ten]] 계열로 `sub-` = under + `tenere` = hold. 아래에서 붙잡아 오래 유지함.

### bolster

- 뜻: 강화하다, 북돋우다
- 구분 포인트: 원래 긴 베개나 받침 이미지. 주장·자신감·시장 등을 강화할 때 자주 씀. `buttress`는 건축적 버팀벽 이미지가 더 뚜렷함.

### prop up

- 뜻: 받쳐 세우다, 지탱하다
- 구분 포인트: 쓰러지려는 것을 임시로 받치는 느낌이 강함. `buttress`는 더 견고하고 구조적인 보강.
