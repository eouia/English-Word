# bi

- 출발 단어: `binary`
- 핵심 어원: Latin `bi-` = 둘, 두 번, 두 겹
- 깊은 어근: PIE `*dwo-` = 둘
- 핵심 이미지: 둘로 된 것, 두 부분 또는 두 번
- 관련 문서: [[di]], [[du]], [[ambi]], [[tri]]
- 주의: `bi-`는 Latin 계열, `di-`는 Greek 계열. 현대 영어에서는 혼합도 있지만 기본 구분은 유지.

## 출발 단어 분석

`binary` ← Latin `binarius`

- `bini`: 둘씩, 한 쌍씩
- 원래 이미지: 둘로 이루어진 체계
- 현대 의미: 이진의, 양자택일의

## 단어들

### binary

- 발음: /ˈbaɪnɚi/
- 구성: Latin `binarius`
- 뜻: 이진의, 둘로 된
- 어원적 연결: 두 값이나 두 부분으로 이루어짐
- 사용 맥락: `binary code` = 이진 코드

### bicycle

- 발음: /ˈbaɪsɪkəl/
- 구성: `bi-` = two + Greek `kyklos` = wheel
- 뜻: 자전거
- 어원적 연결: 바퀴가 두 개
- 사용 맥락: `ride a bicycle` = 자전거를 타다

### bilingual

- 발음: /baɪˈlɪŋgwəl/
- 구성: `bi-` = two + Latin `lingua` = language
- 뜻: 두 언어를 쓰는, 이중 언어의
- 어원적 연결: 두 언어에 속함
- 사용 맥락: `a bilingual speaker` = 이중 언어 화자

### bilateral

- 발음: /baɪˈlætɚəl/
- 구성: `bi-` = two + `lateral` = side
- 뜻: 양측의, 쌍방의
- 어원적 연결: 두 쪽에 모두 관련됨
- 사용 맥락: `bilateral relations` = 양국 관계

### biennial

- 발음: /baɪˈɛniəl/
- 구성: `bi-` = two + Latin `annus` = year
- 뜻: 2년마다의, 2년생의
- 어원적 연결: 두 해 단위로 반복됨
- 사용 맥락: `a biennial event` = 2년마다 열리는 행사

### bimonthly

- 발음: /baɪˈmʌnθli/
- 구성: `bi-` = two + `monthly`
- 뜻: 두 달마다의 또는 한 달에 두 번의
- 어원적 연결: 둘이라는 의미 때문에 실제 사용에서 애매함
- 사용 맥락: 일정에서는 `twice a month`나 `every two months`처럼 풀어 쓰는 편이 안전

### bipartisan

- 발음: /baɪˈpɑrtɪzən/
- 구성: `bi-` = two + `partisan`
- 뜻: 양당의, 초당적인
- 어원적 연결: 두 정당이 함께 관련됨
- 사용 맥락: `bipartisan support` = 초당적 지지

### binoculars

- 발음: /bəˈnɑkjəlɚz/
- 구성: `bi-` = two + Latin `oculus` = eye
- 뜻: 쌍안경
- 어원적 연결: 두 눈으로 보는 도구
- 사용 맥락: `a pair of binoculars` = 쌍안경

### biceps

- 발음: /ˈbaɪˌsɛps/
- 구성: Latin `biceps` = two-headed
- 뜻: 이두근
- 어원적 연결: 두 머리처럼 갈라진 근육
- 사용 맥락: `biceps exercise` = 이두근 운동

## 기억 포인트

`bi-`는 Latin "둘" 이미지.

- `binary`: 둘로 된 체계
- `bicycle`: 바퀴 둘
- `bilingual`: 언어 둘
- `bilateral`: 양쪽
- `biennial`: 두 해 단위

## 비교 단어

### di

- 뜻: 둘
- 구분 포인트: [[di]]는 Greek 계열. `dioxide`, `dichotomy`, `dilemma`처럼 보임.

### du

- 뜻: 둘, 두 사람이 함께, 두 겹
- 구분 포인트: [[du]]는 `duet`, `duo`, `dual`처럼 Latin `duo`의 독립 단어 계열. `bi-`는 접두사로 붙는 경우가 많음.

### ambi

- 뜻: 양쪽에, 둘레에
- 구분 포인트: [[ambi]]는 단순 숫자 2보다 "양쪽에 걸침" 이미지가 강함.
