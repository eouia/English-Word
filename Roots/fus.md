# fus

- 출발 단어: `fusion`
- 핵심 어원: Latin `fundere`, past participle `fusus` = 붓다, 쏟다, 녹이다
- 핵심 이미지: 액체처럼 붓기, 녹여 합치기, 퍼뜨리기
- 관련 문서: [[flu]], [[fibr]]
- 주의: `found` "세우다"와 `foundation`은 별도 계열. `foundry`의 `found`는 금속을 녹여 붓는 `fundere` 계열.

## 출발 단어 분석

`fusion` = Latin `fusio` ← `fusus` ← `fundere`

- `fundere`: pour, melt
- `fusus`: poured, melted
- 원래 이미지: 녹여 붓거나 흘려 합침
- 현대 의미: 융합, 결합, 핵융합

## 단어들

### fusion

- 구성: Latin `fusio` ← `fundere`
- 뜻: 융합, 결합, 핵융합
- 어원적 연결: 녹아 흘러 하나로 합쳐짐
- 사용 맥락: `nuclear fusion` = 핵융합

### fuse

- 구성: Latin `fusus` 계열
- 뜻: 녹이다, 융합하다; 퓨즈
- 어원적 연결: 녹아 하나가 되거나, 전류를 끊는 녹는 장치
- 사용 맥락: `fuse metal` = 금속을 녹여 붙이다

### fused

- 구성: `fuse` + `-ed`
- 뜻: 융합된, 붙어 하나가 된
- 어원적 연결: 녹아 합쳐진 상태
- 사용 맥락: `fused glass` = 융합 유리

### fusible

- 구성: `fuse` + `-ible`
- 뜻: 녹을 수 있는, 융해 가능한
- 어원적 연결: 열로 녹아 흐를 수 있음
- 사용 맥락: materials science 맥락

### confuse

- 구성: `con-` = together + `fuse`
- 뜻: 혼란스럽게 하다, 혼동하다
- 어원적 연결: 여러 것을 함께 부어 뒤섞음
- 사용 맥락: `confuse two words` = 두 단어를 혼동하다

### confusion

- 구성: `confuse` + `-ion`
- 뜻: 혼란, 혼동
- 어원적 연결: 뒤섞여 구분이 안 되는 상태
- 사용 맥락: `cause confusion` = 혼란을 일으키다

### diffuse

- 구성: `dis-` = apart, widely + `fuse`
- 뜻: 퍼뜨리다, 확산되다; 장황한
- 어원적 연결: 사방으로 쏟아져 퍼짐
- 사용 맥락: `diffuse light` = 빛을 확산시키다

### diffusion

- 구성: `diffuse` + `-ion`
- 뜻: 확산, 보급
- 어원적 연결: 넓게 퍼져 나가는 과정
- 사용 맥락: `cultural diffusion` = 문화 확산

### infuse

- 구성: `in-` = into + `fuse`
- 뜻: 주입하다, 스며들게 하다, 우리다
- 어원적 연결: 안으로 붓거나 스며들게 함
- 사용 맥락: `infuse confidence` = 자신감을 불어넣다

### infusion

- 구성: `infuse` + `-ion`
- 뜻: 주입, 우려낸 액체
- 어원적 연결: 액체나 성질을 안으로 붓는 일
- 사용 맥락: `herbal infusion` = 허브 우린 물

### effuse

- 구성: `ex-` = out + `fuse`
- 뜻: 쏟아내다, 발산하다
- 어원적 연결: 밖으로 흘려보냄
- 사용 맥락: 문어적 표현

### effusive

- 구성: `effuse` + `-ive`
- 뜻: 감정이 넘쳐흐르는, 과장되게 표현하는
- 어원적 연결: 감정이 밖으로 쏟아져 나옴
- 사용 맥락: `effusive praise` = 넘치는 칭찬

### profuse

- 구성: `pro-` = forth + `fuse`
- 뜻: 풍부한, 아낌없는, 넘치는
- 어원적 연결: 앞으로 쏟아져 나올 만큼 많음
- 사용 맥락: `profuse sweating` = 땀을 많이 흘림

### profusion

- 구성: `pro-` = forth + Latin `fusio` / `fusus`
- 뜻: 풍부함, 넘쳐남, 다량
- 어원적 연결: 밖으로 쏟아져 나와 가득한 상태
- 사용 맥락: `a profusion of flowers` = 꽃이 흐드러지게 많은 상태

### transfuse

- 구성: `trans-` = across + `fuse`
- 뜻: 수혈하다, 옮겨 붓다
- 어원적 연결: 한쪽에서 다른 쪽으로 부어 옮김
- 사용 맥락: medical 맥락

### transfusion

- 구성: `transfuse` + `-ion`
- 뜻: 수혈, 주입
- 어원적 연결: 액체를 건너 옮겨 붓는 일
- 사용 맥락: `blood transfusion` = 수혈

### suffuse

- 구성: `sub-` = under, throughout + `fuse`
- 뜻: 가득 퍼지다, 물들이다
- 어원적 연결: 아래에서부터 전체로 스며 퍼짐
- 사용 맥락: `suffused with light` = 빛으로 가득한

### refund

- 구성: `re-` = back + `fundere`
- 뜻: 환불하다, 돌려주다
- 어원적 연결: 돈을 다시 부어 돌려줌
- 사용 맥락: `refund a payment` = 결제를 환불하다

### refuse

- 구성: Old French `refuser`, 전통적으로 Latin `refundere`와 연결
- 뜻: 거절하다
- 어원적 연결: 되부어 돌려보냄 → 받아들이지 않음
- 사용 맥락: `refuse an offer` = 제안을 거절하다

## 기억 포인트

`fus` 계열은 "붓고 녹이고 퍼뜨림" 이미지.

- `fusion`: 녹아 합침
- `confuse`: 함께 부어 뒤섞음
- `diffuse`: 사방으로 퍼뜨림
- `infuse`: 안으로 붓고 스며들게 함
- `effusive`: 감정이 밖으로 넘쳐흐름
- `profuse`: 많이 쏟아져 나옴
- `profusion`: 쏟아져 나온 듯한 풍부함
- `transfusion`: 건너 옮겨 붓는 수혈
