# batt

- 출발 단어: `abate`
- 핵심 어원: Old French `batre`, `battre` = beat, strike
- 핵심 이미지: 치다, 두드리다, 쳐서 낮추거나 물러나게 함
- 관련 문서: 없음
- 주의: `abate`는 현대 형태만 보면 `a-` + `bate`처럼 보이지만, 역사적으로는 Old French `abatre` = beat down, cast down에서 옴.

## 출발 단어 분석

`abate` ← Old French `abatre`

- `a-`: down, away 쪽 의미
- `batre`: beat, strike
- 원래 이미지: 쳐서 낮추거나 떨어뜨림
- 현대 의미: 줄어들다, 누그러지다, 약해지다

## 단어들

### abate

- 발음: /əˈbeɪt/
- 구성: Old French `abatre` = beat down
- 뜻: 줄어들다, 누그러지다, 약해지다
- 어원적 연결: 강한 것을 쳐서 낮춤
- 사용 맥락: `The storm abated.` = 폭풍이 누그러졌다

### abatement

- 발음: /əˈbeɪtmənt/
- 구성: `abate` + `-ment`
- 뜻: 감소, 완화, 감면
- 어원적 연결: 세기나 부담이 낮아진 상태
- 사용 맥락: `tax abatement` = 세금 감면

### rebate

- 발음: /ˈriˌbeɪt/
- 구성: Old French `rebatre` = beat back
- 뜻: 환급, 할인액, 일부 돌려주다
- 어원적 연결: 가격이나 금액을 뒤로 쳐서 낮춤
- 사용 맥락: `a mail-in rebate` = 우편 환급 할인

### debate

- 발음: /dəˈbeɪt/
- 구성: Old French `debatre` = fight, contend, debate
- 뜻: 토론하다, 논쟁하다
- 어원적 연결: 말로 서로 치고받듯 다툼
- 사용 맥락: `debate the issue` = 그 문제를 토론하다

### combat

- 발음: /ˈkɑmbæt/
- 구성: `com-` = together + Old French `batre` = beat
- 뜻: 전투, 싸우다, 방지하다
- 어원적 연결: 서로 맞서 치고 싸움
- 사용 맥락: `combat disease` = 질병과 싸우다

### battle

- 발음: /ˈbætəl/
- 구성: Old French `bataille` 계열
- 뜻: 전투, 싸움
- 어원적 연결: 서로 치고 맞붙는 싸움
- 사용 맥락: `a fierce battle` = 치열한 전투

### battalion

- 발음: /bəˈtæljən/
- 구성: Italian/French `battaglia` 계열
- 뜻: 대대, 전투 부대
- 어원적 연결: 전투를 위해 조직된 병력
- 사용 맥락: `an infantry battalion` = 보병 대대

### batter

- 발음: /ˈbætɚ/
- 구성: Old French `batre` = beat
- 뜻: 세게 치다, 두들겨 패다
- 어원적 연결: 반복해서 두드림
- 사용 맥락: `batter the door` = 문을 세게 두드리다

### battery

- 발음: /ˈbætɚi/
- 구성: Old French `baterie` = beating, action of striking
- 뜻: 배터리, 포대, 폭행
- 어원적 연결: 원래 두드림·타격. 포대는 포를 쏘는 장비 집합, 전지는 여러 셀이 한 줄로 배열된 장치로 확장.
- 사용 맥락: `a phone battery` = 휴대폰 배터리, `assault and battery` = 폭행

## 기억 포인트

`batt` 계열은 "치다, 쳐서 낮추다, 맞붙다" 이미지.

- `abate`: 쳐서 낮춤 → 약해지다
- `rebate`: 뒤로 쳐서 낮춤 → 환급·할인
- `debate`: 말로 치고받음 → 토론
- `combat`: 함께 맞서 싸움
- `battle`: 전투
- `battery`: 타격, 포대, 전지

## 비교 단어

### beat

- 뜻: 치다, 두드리다
