# adipisc

- 출발 단어: `adept`
- 핵심 어원: Latin `adipisci` = attain, obtain, arrive at
- 핵심 이미지: 어떤 수준에 도달함, 기술을 얻어 숙련됨
- 관련 문서: [[apt]]
- 주의: `adept`는 `adapt`와 철자가 비슷하지만 직접 같은 단어가 아님. `adept`는 "도달하여 얻은 사람", `adapt`는 "알맞게 맞추다".

## 출발 단어 분석

`adept` ← Latin `adeptus`

- `adeptus`: having attained
- `adipisci`: attain, obtain
- 원래 이미지: 어떤 기술이나 비밀 지식에 도달해 얻음
- 현대 의미: 능숙한, 숙련자

## 단어들

### adept

- 발음: /əˈdɛpt/
- 구성: Latin `adeptus` = having attained
- 뜻: 능숙한, 숙련자
- 어원적 연결: 어떤 기술 수준에 도달해 익숙해진 사람
- 사용 맥락: `adept at negotiation` = 협상에 능숙한

### adeptly

- 구성: `adept` + `-ly`
- 뜻: 능숙하게
- 어원적 연결: 익힌 기술로 잘 해냄
- 사용 맥락: `handle the problem adeptly` = 문제를 능숙하게 처리하다

### adeptness

- 구성: `adept` + `-ness`
- 뜻: 능숙함, 숙련도
- 어원적 연결: 어떤 일을 잘 해낼 수준에 도달한 상태
- 사용 맥락: `technical adeptness` = 기술적 능숙함

## 기억 포인트

`adipisc` 계열은 "도달해 얻음 → 숙련됨" 이미지.

- `adept`: 기술을 얻어 능숙한
- `adeptly`: 능숙하게
- `adeptness`: 능숙함

## 비교 단어

### apt

- 뜻: 알맞은, ~하기 쉬운, 소질 있는
- 구분 포인트: [[apt]]는 Latin `aptus` = fitted, suited 계열. `adept`는 Latin `adeptus/adipisci` = attain 계열. 먼 관련 가능성은 있지만 현대 학습에서는 분리.

### adapt

- 뜻: 적응하다, 맞추다
- 구분 포인트: `adapt`는 [[apt]] 계열로 "맞추다". `adept`는 "도달해 얻은 숙련" 쪽.
