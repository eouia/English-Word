# mod

- 출발 단어: `commodity`
- 핵심 어원: 라틴어 `modus` = 방식, 정도, 알맞은 치수
- 핵심 이미지: 알맞은 방식이나 정도에 맞춤
- 관련해서 헷갈릴 수 있는 어근: `mood`는 별도 계열

## 출발 단어 분석

`commodity`는 라틴어 `commodus`에서 옴.

- `com-`: together, completely
- `modus`: 알맞은 치수, 방식
- `commodus`: 알맞은, 편리한, 유용한

따라서 `commodity`는 원래 "쓸모 있고 편리한 것" → "거래되는 상품, 원자재"의 뜻.

## 단어들

### commodity

- 발음: /kəˈmɑdəti/
- 구성: Latin `commodus` 계열
- 뜻: 상품, 원자재, 생활필수품
- 어원적 연결: 쓸모 있고 편리한 물건에서, 시장에서 거래되는 물건으로 의미 확대
- 사용 맥락: `agricultural commodities` = 농산물 상품

### commodious

- 발음: /kəˈmoʊdiəs/
- 구성: Latin `commodus`
- 뜻: 넓고 편리한
- 어원적 연결: 쓰기에 알맞고 편한 상태
- 사용 맥락: `a commodious room` = 넓고 편한 방

### accommodate

- 발음: /əˈkɑməˌdeɪt/
- 구성: `ad-` = to + `com-` + `mod`
- 뜻: 수용하다, 숙박시키다, 맞추다
- 어원적 연결: 어떤 조건이나 사람에게 알맞게 맞춤
- 사용 맥락: `accommodate guests` = 손님을 수용하다

### accommodation

- 발음: /əˌkɑməˈdeɪʃən/
- 구성: `accommodate` + `-ion`
- 뜻: 숙박 시설, 조정, 편의 제공
- 어원적 연결: 사람이나 상황에 맞게 마련한 것
- 사용 맥락: `student accommodation` = 학생 숙소

### mode

- 발음: /ˈmoʊd/
- 구성: Latin `modus`
- 뜻: 방식, 양식, 모드
- 어원적 연결: 일이 이루어지는 정해진 방식
- 사용 맥락: `sleep mode` = 절전 모드

### model

- 발음: /ˈmɑdəl/
- 구성: Italian `modello`, from Latin `modus`
- 뜻: 모형, 모델, 본보기
- 어원적 연결: 어떤 방식이나 치수에 맞춘 작은 기준
- 사용 맥락: `a business model` = 사업 모델

### modify

- 발음: /ˈmɑdəˌfaɪ/
- 구성: `mod` + `-fy`
- 뜻: 수정하다, 변경하다
- 어원적 연결: 알맞은 정도나 방식으로 바꿈
- 사용 맥락: `modify the plan` = 계획을 수정하다

### modification

- 발음: /ˌmɑdəfəˈkeɪʃən/
- 구성: `modify` + `-ication`
- 뜻: 수정, 변경
- 어원적 연결: 방식이나 정도를 조절해 바꾸는 일
- 사용 맥락: `minor modifications` = 작은 수정 사항

### moderate

- 발음: adj. /ˈmɑdɚət/, v. /ˈmɑdɚˌeɪt/
- 구성: Latin `moderari`
- 뜻: 적당한, 온건한, 조절하다
- 어원적 연결: 지나치지 않게 알맞은 정도로 맞춤
- 사용 맥락: `moderate exercise` = 적당한 운동

### moderation

- 발음: /ˌmɑdɚˈeɪʃən/
- 구성: `moderate` + `-ion`
- 뜻: 절제, 온건함, 조정
- 어원적 연결: 알맞은 정도를 넘지 않게 조절함
- 사용 맥락: `in moderation` = 적당히, 절제하여

### modest

- 발음: /ˈmɑdəst/
- 구성: Latin `modestus`
- 뜻: 겸손한, 적당한, 수수한
- 어원적 연결: 자신의 정도를 넘지 않고 알맞게 제한함
- 사용 맥락: `a modest increase` = 적당한 증가

### modesty

- 발음: /ˈmɑdəsti/
- 구성: `modest` + `-y`
- 뜻: 겸손, 수수함
- 어원적 연결: 지나치게 드러내지 않고 알맞은 정도를 지킴
- 사용 맥락: `speak with modesty` = 겸손하게 말하다

### modern

- 발음: /ˈmɑdɚn/
- 구성: Late Latin `modernus`, from `modo` = just now
- 뜻: 현대의
- 어원적 연결: 현재의 방식이나 지금의 시점과 관련
- 사용 맥락: `modern technology` = 현대 기술

## 기억 포인트

`mod` 계열은 "알맞은 방식과 정도" 이미지.

- `commodity`: 쓸모 있는 물건, 거래 상품
- `commodious`: 쓰기에 알맞고 넓은
- `accommodate`: 상황에 맞게 수용함
- `mode`: 방식
- `model`: 기준이 되는 형태
- `modify`: 알맞게 바꿈
- `moderate`: 알맞은 정도로 조절함
- `modest`: 정도를 넘지 않음

## 비교 단어

### common

- 뜻: 흔한, 공동의
- 구분 포인트: `commodity`의 `com-` 때문에 `common`과 닮아 보이지만, 핵심은 `modus` = 알맞은 방식/정도

### mood

- 뜻: 기분
- 구분 포인트: `mode`와 형태가 비슷하지만, `mood`는 게르만계로 별도 기억
