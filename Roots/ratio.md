# ratio

- 출발 단어: `rationalize`
- 핵심 어원: Latin `ratio` = reckoning, calculation, reason
- 핵심 이미지: 계산하고 따져서 이유를 세움
- 관련 문서: [[numer]], [[calc]], [[logos]]
- 주의: `rationalize`는 "이성적으로 설명하다"뿐 아니라 "그럴듯한 이유를 붙여 정당화하다"의 부정적 의미도 자주 씀.

## 출발 단어 분석

`rationalize` = `rational` + `-ize`

- `ratio`: 계산, 셈, 이유, 이성
- `rational`: 이성적인, 합리적인
- `-ize`: ~하게 만들다
- 원래 이미지: 어떤 일을 이유와 계산에 맞게 설명함
- 현대 의미: 합리화하다, 이성적으로 설명하다, 정당화하다

## 단어들

### rational

- 발음: /ˈræʃənəl/
- 구성: Latin `rationalis` ← `ratio`
- 뜻: 이성적인, 합리적인
- 어원적 연결: 감정보다 이유와 계산에 따라 판단함
- 사용 맥락: `a rational choice` = 합리적인 선택

### rationale

- 발음: /ˌræʃəˈnæl/
- 구성: Latin `rationale` = reasoned account
- 뜻: 근거, 이유, 논리적 설명
- 어원적 연결: 어떤 판단이나 행동 뒤에 세워진 이유
- 사용 맥락: `the rationale for the policy` = 그 정책의 근거

### rationalize

- 발음: /ˈræʃənəˌlaɪz/
- 구성: `rational` + `-ize`
- 뜻: 합리화하다, 이성적으로 설명하다, 정당화하다
- 어원적 연결: 어떤 행동이나 생각에 이유를 붙여 말이 되게 만듦
- 사용 맥락: `rationalize a bad decision` = 나쁜 결정을 그럴듯하게 합리화하다

### rationalization

- 발음: /ˌræʃənəlɪˈzeɪʃən/
- 구성: `rationalize` + `-ation`
- 뜻: 합리화, 정당화; 합리적 재편
- 어원적 연결: 이유를 붙여 설명하거나 체계를 더 효율적으로 정리함
- 사용 맥락: `a rationalization for failure` = 실패에 대한 합리화

### irrational

- 발음: /ˌɪˈræʃənəl/
- 구성: `ir-` = not + `rational`
- 뜻: 비이성적인, 불합리한
- 어원적 연결: 이유와 계산에 맞지 않음
- 사용 맥락: `an irrational fear` = 비이성적인 두려움

### irrationally

- 발음: /ˌɪˈræʃənəli/
- 구성: `irrational` + `-ly`
- 뜻: 비이성적으로, 불합리하게
- 어원적 연결: 이유에 맞지 않는 방식
- 사용 맥락: `act irrationally` = 비이성적으로 행동하다

### reason

- 발음: /ˈrizən/
- 구성: Old French `raison` ← Latin `ratio`
- 뜻: 이유, 이성; 추론하다
- 어원적 연결: 셈하고 따져 나온 근거
- 사용 맥락: `give a reason` = 이유를 대다, `reason carefully` = 신중히 추론하다

## 기억 포인트

`ratio` 계열은 "계산하고 따져 이유를 세움" 이미지.

- `rational`: 이성적인
- `rationale`: 근거, 이유
- `rationalize`: 이유를 붙여 합리화함
- `rationalization`: 합리화, 정당화
- `irrational`: 이유에 맞지 않는
- `reason`: 이유, 이성

## 비교 단어

### logic

- 뜻: 논리
- 구분 포인트: [[logos]]는 말과 논리의 구조에 초점. `ratio`는 계산, 비율, 이유, 이성의 이미지가 강함.

### calculate

- 뜻: 계산하다
- 구분 포인트: [[calc]]는 작은 돌로 세어 계산하는 이미지. `ratio`는 계산에서 더 나아가 이유와 합리성으로 확장됨.
