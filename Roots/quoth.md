# quoth

- 출발 단어: `bequeath`
- 핵심 어원: 고대 영어 `cweðan` = 말하다, 선언하다
- 핵심 이미지: 말로 선언함
- 비교: [[quote]]는 다른 어원

## 출발 단어 분석

`bequeath`는 `be-` + `cweðan` 계열.

- `be-`: about, by, around 계열 접두사
- `cweðan`: 말하다, 선언하다

따라서 `bequeath`는 "말로 남김" → "유언으로 남김"의 뜻.

## 단어들

### bequeath

- 발음: /bɪˈkwiθ/
- 구성: `be-` + `cweðan`
- 뜻: 유언으로 남기다, 물려주다
- 어원적 연결: 말이나 유언으로 남김
- 사용 맥락: `bequeath property to children` = 자녀에게 재산을 물려주다

### bequest

- 발음: /bɪˈkwɛst/
- 구성: `bequeath` 계열 명사
- 뜻: 유증, 유산으로 남긴 것
- 어원적 연결: 유언으로 남긴 재산이나 선물. [[quest]] 계열 아님
- 구분 포인트: `bequest` = 유언으로 남긴 것, `quest` = 찾거나 구하는 일
- 사용 맥락: `a charitable bequest` = 자선 단체에 남긴 유산

### quoth

- 발음: /ˈkwoʊθ/
- 구성: 고대 영어 `cweðan`의 과거형 계열
- 뜻: 말했다
- 어원적 연결: "말하다"의 오래된 과거형. 지금은 문학적/고어적 표현
- 사용 맥락: `quoth the speaker` = 말하는 이가 말했다

## 비교 단어

### quote

- 구성: 라틴어 `quotare` = 번호를 매기다, 표시하다
- 뜻: 인용하다
- 어원적 연결: `quoth`, `bequeath`와 어원 다름. [[quote]] 참고
- 구분 포인트: `quoth` = 말했다, `quote` = 인용하다
- 사용 맥락: `quote a sentence` = 문장을 인용하다

### quotation

- 구성: `quote` + `-ation`
- 뜻: 인용, 인용문
- 어원적 연결: 인용해 표시한 말
- 사용 맥락: `a direct quotation` = 직접 인용

### quota

- 구성: 라틴어 `quotus` = 몇 번째의, 얼마나 많은
- 뜻: 할당량
- 어원적 연결: 정해진 몫, 수량
- 사용 맥락: `meet a quota` = 할당량을 채우다

## 기억 포인트

`quoth` 계열은 "말로 선언함" 이미지.

- `bequeath`: 유언으로 말해 남김
- `bequest`: 유언으로 남긴 것
- `quoth`: 말했다

비교:

- `quote`: 인용하다. 라틴어 `quotare` 계열
- `quota`: 할당량. 라틴어 `quotus` 계열
