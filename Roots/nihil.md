# nihil

- 출발 단어: `annihilate`
- 핵심 어원: Latin `nihil` = 아무것도 아님, 무
- 핵심 이미지: 존재를 없애 무로 만듦
- 관련 문서: [[void]]

## 출발 단어 분석

`annihilate` ← Latin `ad-` + `nihil`

- `nihil`: nothing
- `ad nihil`: to nothing
- 원래 이미지: 아무것도 아닌 상태로 만듦
- 현대 의미: 완전히 파괴하다, 전멸시키다

## 단어들

### annihilate

- 구성: `ad-` = to + Latin `nihil` = nothing
- 뜻: 전멸시키다, 완전히 파괴하다
- 어원적 연결: 존재를 무로 돌림
- 사용 맥락: `annihilate the enemy` = 적을 전멸시키다

### annihilation

- 구성: `annihilate` + `-ion`
- 뜻: 전멸, 완전 파괴
- 어원적 연결: 무로 만드는 과정
- 사용 맥락: `nuclear annihilation` = 핵 전멸

### nihilism

- 구성: Latin `nihil` + `-ism`
- 뜻: 허무주의
- 어원적 연결: 가치나 의미를 무로 보는 사상
- 사용 맥락: `moral nihilism` = 도덕적 허무주의

### nihilist

- 구성: `nihilism` + `-ist`
- 뜻: 허무주의자
- 어원적 연결: 의미나 가치가 없다고 보는 사람
- 사용 맥락: `a political nihilist` = 정치적 허무주의자

## 기억 포인트

`nihil` 계열은 "무, 아무것도 없음" 이미지.

- `annihilate`: 무로 만들다
- `annihilation`: 전멸
- `nihilism`: 허무주의
- `nihilist`: 허무주의자
