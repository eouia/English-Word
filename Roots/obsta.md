# obsta

- 출발 단어: `oust`
- 핵심 어원: Latin `obstare` = 앞에 서다, 막아서다, 방해하다
- 핵심 이미지: 앞에 버티고 서서 막거나, 그 자리에서 밀어냄
- 관련 문서: [[stat]], [[stand]], [[pel]], [[ject]]

## 단어들

### oust

- 발음: /ˈaʊst/
- 구성: Anglo-French `ouster`, Old French `oster` < Latin `obstare`
- 뜻: 몰아내다, 축출하다, 자리에서 내쫓다
- 어원적 연결: 앞을 막고 서는 이미지에서 "제거하다, 쫓아내다"로 변화
- 사용 맥락: `oust a leader from power` = 지도자를 권좌에서 몰아내다

### ouster

- 발음: /ˈaʊstɚ/
- 구성: `oust` + 명사형 `-er`
- 뜻: 축출, 해임, 배제
- 어원적 연결: 어떤 사람을 자리나 권리에서 몰아내는 행위
- 사용 맥락: `the CEO's ouster` = CEO의 축출/해임

### ousted

- 발음: /ˈaʊstɪd/
- 구성: `oust` + `-ed`
- 뜻: 축출된, 쫓겨난
- 어원적 연결: 차지하던 자리에서 밀려난 상태
- 사용 맥락: `the ousted president` = 축출된 대통령

### obstacle

- 발음: /ˈɑbstəkəl/
- 구성: Latin `ob-` = against, in the way + `stare` = stand
- 뜻: 장애물, 방해 요인
- 어원적 연결: 앞에 서서 길을 막는 것
- 사용 맥락: `overcome an obstacle` = 장애물을 극복하다

### obstinate

- 발음: /ˈɑbstənət/
- 구성: Latin `obstinare` = persist, stand firm
- 뜻: 완고한, 고집 센
- 어원적 연결: 반대 앞에서도 버티고 서 있음
- 사용 맥락: `an obstinate refusal` = 완고한 거부

### obstinacy

- 발음: /ˈɑbstənəsi/
- 구성: `obstinate` + `-acy`
- 뜻: 완고함, 고집
- 어원적 연결: 자기 입장을 굽히지 않고 버티는 성질
- 사용 맥락: `political obstinacy` = 정치적 완고함

## 기억 포인트

`obsta` 계열은 "앞에 서서 막음 / 자리에서 밀어냄" 이미지.

- `oust`: 자리에서 몰아내다
- `ouster`: 축출
- `obstacle`: 앞에 서 있는 장애물
- `obstinate`: 고집스럽게 버티는

## 비교 단어

### out

- 뜻: 밖으로
- 구분 포인트: `oust`는 철자와 뜻 때문에 `out`처럼 보이지만, 표준 어원은 Old English `out`이 아니라 Latin `obstare` 계열.

### expel

- 뜻: 밀어내다, 쫓아내다
- 구분 포인트: [[pel]]은 "밀다" 이미지. `oust`는 권력, 직위, 점유하던 자리에서 몰아냄에 자주 쓰임.

### eject

- 뜻: 밖으로 던져 내다
- 구분 포인트: [[ject]]는 "던지다" 이미지. `oust`는 정치적, 조직적 축출 느낌이 강함.
