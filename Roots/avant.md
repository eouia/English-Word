# avant

- 출발 단어: `advantageous`
- 핵심 어원: Old French `avant` = before, in front
- 깊은 어원: Latin `ab ante` = from before
- 핵심 이미지: 앞에 있음, 앞선 위치, 유리한 자리
- 관련 문서: [[ante]]
- 주의: `advantage`의 `ad-`는 현대적으로 "to" 접두사처럼 보이지만, 역사적으로는 Old French `avantage` / `avant` 계열. `ad + vantage`로 단순 분해하지 않음.

## 출발 단어 분석

`advantageous` = `advantage` + `-ous`

- `advantage`: 유리함, 이점
- `-ous`: ~이 많은, ~의 성질을 가진
- 원래 이미지: 앞선 위치에 있어 더 유리함
- 현대 의미: 유리한, 이로운

## 단어들

### advantage

- 발음: /æˈdvæntɪdʒ/
- 구성: Old French `avantage` ← `avant` = before, in front
- 뜻: 이점, 유리함
- 어원적 연결: 남보다 앞선 위치에 있음
- 사용 맥락: `gain an advantage` = 이점을 얻다

### advantageous

- 발음: /ˌædvəˈnteɪdʒəs/
- 구성: `advantage` + `-ous`
- 뜻: 유리한, 이로운
- 어원적 연결: 앞선 위치나 조건을 제공함
- 사용 맥락: `an advantageous position` = 유리한 위치

### disadvantage

- 발음: /ˌdɪsəˈdvæntɪdʒ/
- 구성: `dis-` = opposite, lack + `advantage`
- 뜻: 불리함, 약점
- 어원적 연결: 유리한 위치가 없거나 반대로 불리함
- 사용 맥락: `at a disadvantage` = 불리한 입장에서

### vantage

- 발음: /ˈvæntədʒ/
- 구성: `advantage`에서 앞부분이 약해진 형태
- 뜻: 유리한 위치, 전망 좋은 지점
- 어원적 연결: 앞서 보거나 유리하게 볼 수 있는 자리
- 사용 맥락: `a vantage point` = 유리한 관찰 지점

### advance

- 발음: /əˈdvæns/
- 구성: Old French `avancer` ← `avant` = forward
- 뜻: 나아가다, 진전, 선불
- 어원적 연결: 앞쪽으로 움직임
- 사용 맥락: `advance rapidly` = 빠르게 전진하다, `pay in advance` = 선불로 내다

### advanced

- 발음: /əˈdvænst/
- 구성: `advance` + `-ed`
- 뜻: 고급의, 진보한, 앞선
- 어원적 연결: 앞 단계로 나아간 상태
- 사용 맥락: `advanced technology` = 첨단 기술

### advancement

- 발음: /əˈdvænsmənt/
- 구성: `advance` + `-ment`
- 뜻: 진보, 발전, 승진
- 어원적 연결: 앞쪽 단계로 나아감
- 사용 맥락: `career advancement` = 경력 발전

### avant-garde

- 발음: /əˈvɑˈntgɑrd/
- 구성: French `avant` = before, front + `garde` = guard
- 뜻: 전위적인, 아방가르드, 전위 예술
- 어원적 연결: 맨 앞에서 나아가는 선두 부대
- 사용 맥락: `avant-garde art` = 전위 예술

## 기억 포인트

`avant` 계열은 "앞에 있음, 앞선 위치" 이미지.

- `advantage`: 앞선 자리 → 이점
- `advantageous`: 유리한
- `disadvantage`: 앞선 자리가 없음 → 불리함
- `vantage`: 유리하게 보는 위치
- `advance`: 앞으로 나아감
- `advanced`: 앞선, 고급의
- `avant-garde`: 맨 앞의 선두

## 비교 단어

### ante

- 뜻: 앞에, 이전에
- 구분 포인트: [[ante]]는 Latin `ante` 계열의 기본 "앞/이전". `avant`는 French 경로로 들어와 `advance`, `advantage`, `vantage`에 남은 형태.
