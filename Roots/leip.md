# leip

- 출발 단어: `eclipse`
- 핵심 어원: Greek `leipein` = leave, fail, be missing; PIE `*leikw-` = leave
- 핵심 이미지: 있어야 할 자리에서 빠져나가거나 빠뜨려 남김
- 관련 문서: [[leave]], [[ex]], [[em]]
- 주의: English `leave`와 의미·깊은 뿌리는 가깝지만, `eclipse/ellipsis`는 Greek `leipein`을 거친 학술어 계열. `clip` = 자르다와는 분리.

## 출발 단어 분석

`eclipse` ← Greek `ekleipsis` = abandonment, failure to appear, eclipse

- `ek-/ex-`: out
- `leipein`: leave, fail, be missing
- 원래 이미지: 천체가 평소의 자리나 빛에서 빠져나간 듯 보임
- 의미 변화: 나타나지 않음, 빛이 사라짐 → 천체가 가려지는 현상 → 명성·중요성이 가려짐
- 현대 의미: 일식·월식, 가림; 가리다, 능가하다

## 단어들

### eclipse

- 발음: /ɪˈklɪps/
- 구성: Greek `ek-` = out + `leipein` = leave
- 뜻: 일식·월식; 가리다, 빛을 잃게 하다; 능가하다
- 어원적 연결: 해나 달이 있어야 할 빛을 "떠난" 듯 사라짐
- 사용 맥락: `a solar eclipse` = 일식, `eclipse the old record` = 이전 기록을 능가하다

### eclipsed

- 발음: /ɪˈklɪpst/
- 구성: `eclipse` + `-ed`
- 뜻: 가려진, 빛을 잃은, 능가된
- 어원적 연결: 다른 것이 앞에 와 빛이나 명성이 보이지 않게 됨
- 사용 맥락: `be eclipsed by a rival` = 경쟁자에게 가려지다

### ecliptic

- 발음: /ɪˈklɪptɪk/
- 구성: Greek `ekleiptikos` = of an eclipse
- 뜻: 황도, 태양의 겉보기 경로
- 어원적 연결: 일식·월식이 이 경로 근처에서 일어나므로 붙은 이름
- 사용 맥락: `the plane of the ecliptic` = 황도면

### ellipsis

- 구성: Greek `elleipsis` = omission, falling short
- 뜻: 생략; 말줄임표
- 어원적 연결: 있어야 할 말이나 요소를 남겨 두고 빠뜨림
- 사용 맥락: `use an ellipsis` = 말줄임표를 쓰다

### elliptical

- 발음: /ɪˈlɪptɪkəl/
- 구성: `ellipsis/ellipse` + `-ical`
- 뜻: 생략적인; 타원형의
- 어원적 연결: 말에서는 빠진 부분이 있는 표현. 기하에서는 원에서 부족하게 빠진 듯한 모양으로 이름 붙음
- 사용 맥락: `elliptical language` = 생략적인 표현

### relinquish

- 발음: /rɪˈlɪŋkwɪʃ/
- 구성: Latin `re-` = back + `linquere` = leave
- 뜻: 포기하다, 넘겨주다
- 어원적 연결: 가지고 있던 것을 뒤에 남기고 손에서 놓음
- 사용 맥락: `relinquish control` = 통제권을 포기하다

### derelict

- 발음: /ˈdɛrəˌlɪkt/
- 구성: Latin `de-` = away + `relinquere` = leave behind
- 뜻: 버려진; 직무 태만의; 유기된 사람
- 어원적 연결: 돌봄이나 책임에서 버려져 남겨짐
- 사용 맥락: `a derelict building` = 버려진 건물

### delinquent

- 발음: /dɪˈlɪŋkwənt/
- 구성: Latin `de-` = away + `linquere` = leave
- 뜻: 비행의, 의무를 이행하지 않는; 체납된
- 어원적 연결: 해야 할 의무를 남겨 두고 빠짐
- 사용 맥락: `juvenile delinquent` = 청소년 비행자, `delinquent taxes` = 체납 세금

## 기억 포인트

`leip/linqu` 계열은 "빠져나가거나 남겨 둠" 이미지.

- `eclipse`: 빛이 빠져나간 듯 가려짐
- `eclipsed`: 가려져 빛을 잃음
- `ecliptic`: eclipse가 일어나는 길
- `ellipsis`: 있어야 할 말을 빠뜨림
- `relinquish`: 손에서 놓고 포기함
- `derelict`: 버려져 남은
- `delinquent`: 의무에서 빠진

## 비교 단어

### leave

- 뜻: 남기다, 떠나다, 허락
- 구분 포인트: [[leave]]는 English/Germanic 계열. `leip`은 Greek/Latin 학술어 계열이지만 깊은 PIE 뿌리와 의미가 가까움.

### clip

- 뜻: 자르다, 잘라내다
- 구분 포인트: `eclipse`는 `clip`과 철자가 비슷하지만 "자르다"가 아니라 `ek- + leipein` = 밖으로 빠지다, 나타나지 않다 계열.

### ex

- 뜻: 밖으로
- 구분 포인트: [[ex]]는 방향 접두사. `eclipse`의 `ec-/ek-`가 같은 "밖으로" 역할을 함.
