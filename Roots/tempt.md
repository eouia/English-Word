# tempt

- 출발 단어: `attempt`
- 핵심 어원: 라틴어 `temptare`, `tentare` = 시험하다, 시도하다, 건드려 보다
- 핵심 이미지: 가능성을 시험해 봄 / 마음을 건드려 흔듦
- 관련해서 헷갈릴 수 있는 어근: [[test]] = 시험하다, 증명하다 계열, [[try]] = 시도하다, 시험하다 계열

## 출발 단어 분석

`attempt`는 `ad-` + `tempt`.

- `ad-`: to, toward
- `tempt`: try, test

따라서 `attempt`는 원래 "어떤 쪽으로 시험해 보다, 손을 대 보다"의 뜻. 여기서 "시도하다" → "시도, 노력"의 뜻.

## 단어들

### attempt

- 구성: `ad-` = toward + `tempt`
- 뜻: 시도하다, 시도
- 어원적 연결: 어떤 목표를 향해 시험 삼아 해 봄
- 사용 맥락: `attempt to solve the problem` = 문제를 풀려고 시도하다

### attempted

- 구성: `attempt` + `-ed`
- 뜻: 시도된, 미수의
- 어원적 연결: 어떤 행동이 실행되려 했지만 완성되지 않을 수 있음
- 사용 맥락: `an attempted escape` = 탈출 시도

### tempting

- 구성: `tempt` + `-ing`
- 뜻: 솔깃한, 유혹적인
- 어원적 연결: 마음을 건드려 반응을 시험함
- 사용 맥락: `a tempting offer` = 솔깃한 제안

### tempt

- 구성: Latin `temptare`
- 뜻: 유혹하다, 마음을 끌다
- 어원적 연결: 사람의 마음을 시험하듯 흔듦
- 사용 맥락: `tempt someone with money` = 돈으로 유혹하다

### temptation

- 구성: `tempt` + `-ation`
- 뜻: 유혹, 충동
- 어원적 연결: 마음을 시험하고 흔드는 힘
- 사용 맥락: `resist temptation` = 유혹을 이겨 내다

### tentative

- 구성: Latin `tentare` = try, test + `-ive`
- 뜻: 잠정적인, 시험적인
- 어원적 연결: 아직 확정하지 않고 시험 삼아 해 봄
- 사용 맥락: `a tentative plan` = 잠정 계획

## 기억 포인트

`tempt` 계열은 "시험해 보다"에서 출발.

- `attempt`: 목표를 향해 시험해 봄 → 시도하다
- `tempt`: 마음을 시험함 → 유혹하다
- `temptation`: 마음을 흔드는 시험 → 유혹
- `tentative`: 시험 삼아 둔 → 잠정적인

## 비교 단어

### contempt

- 뜻: 경멸
- 구분 포인트: Latin `contemnere` = 업신여기다 계열. `tempt` = 시험하다 계열과 섞지 않음

### test

- 뜻: 시험하다, 검사하다
- 구분 포인트: [[test]]는 검증과 증명의 이미지가 강함. `tempt`는 "시도해 보다" 또는 "마음을 시험하다"에 가까움
