# earnest

- 출발 단어: `earnest`
- 핵심 어원 1: Old English `eornost` = seriousness, serious intent
- 핵심 어원 2: Latin `arra` / Greek `arrhabon` = pledge, security
- 핵심 이미지: 진심으로 심각하게 임함; 약속을 보증하는 선불
- 관련 문서: [[gage]], [[trust]], [[auster]]
- 주의: `earn` = 벌다와 직접 연결하지 않음. `earnest`는 "진지함" 갈래와 "보증금" 갈래가 역사적으로 다름.

## 출발 단어 분석

`earnest`는 두 갈래가 영어에서 같은 철자로 합쳐져 보임.

1. 형용사·일반 명사: Old English `eornost` = seriousness, zeal
2. 보증금 명사: Middle English `ernes` ← Old French `erres` ← Latin `arra` ← Greek `arrhabon` = pledge

따라서 `earnest`는 "진지한 태도"와 "거래를 진지하게 하겠다는 보증금"을 함께 가짐.

## 단어들

### earnest

- 구성: Old English `eornost` = seriousness
- 뜻: 진지한, 성실한, 진심 어린
- 어원적 연결: 가볍게 장난하지 않고 마음을 다해 임함
- 사용 맥락: `an earnest request` = 진지한 요청

### in earnest

- 구성: `in` + `earnest`
- 뜻: 진지하게, 본격적으로
- 어원적 연결: 장난이나 말뿐이 아니라 진심의 상태로 들어감
- 사용 맥락: `work began in earnest` = 일이 본격적으로 시작되었다

### earnestly

- 구성: `earnest` + `-ly`
- 뜻: 진지하게, 간절히
- 어원적 연결: 진심을 담아 하는 방식
- 사용 맥락: `speak earnestly` = 진지하게 말하다

### earnestness

- 구성: `earnest` + `-ness`
- 뜻: 진지함, 성실함
- 어원적 연결: 마음이 가볍지 않고 진심인 상태
- 사용 맥락: `a tone of earnestness` = 진지한 어조

### earnest money

- 구성: `earnest` = pledge + `money`
- 뜻: 계약금, 보증금
- 어원적 연결: 거래를 실제로 이행하겠다는 진지한 약속의 표시로 먼저 거는 돈
- 사용 맥락: `pay earnest money` = 계약금을 내다

### earnest payment

- 구성: `earnest` = pledge + `payment`
- 뜻: 보증 성격의 선불금
- 어원적 연결: 뒤따를 이행을 보증하는 첫 지급
- 사용 맥락: real estate / procurement 문맥

## 기억 포인트

`earnest`는 두 갈래.

- `earnest`: 진지한, 진심 어린
- `in earnest`: 본격적으로, 진심으로
- `earnestly`: 진지하게
- `earnestness`: 진지함
- `earnest money`: 계약 이행을 보증하는 돈

## 비교 단어

### gage

- 뜻: 약속이나 담보로 묶음
- 구분 포인트: [[gage]]는 보증·담보 일반. `earnest money`는 거래 의사를 보이는 선불 보증금.

### trust

- 뜻: 믿고 맡김
- 구분 포인트: [[trust]]는 신뢰 자체. `earnest money`는 신뢰를 보이기 위해 실제 돈을 걸어 둔 것.

### serious

- 뜻: 진지한, 심각한
- 구분 포인트: `serious`는 무겁고 가볍지 않은 태도. `earnest`는 진심과 성실한 의도가 더 강함.
