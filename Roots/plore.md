# plore

- 출발 단어: `explore`
- 핵심 어원: 라틴어 `plorare` = 외치다, 울부짖다, 찾아 부르다
- 핵심 이미지: 소리 내어 찾고 살핌
- 관련해서 헷갈릴 수 있는 어근: [[quest]] = 찾다, 묻다 계열, [[spect]] = 보다, 살피다 계열

## 출발 단어 분석

`explore`는 `ex-` + `plorare`.

- `ex-`: out
- `plorare`: 외치다, 울부짖다, 찾아 부르다

따라서 `explore`는 원래 "밖으로 소리 내어 살피다" → "탐사하다, 조사하다, 모색하다"의 뜻.

## 단어들

### explore

- 발음: /ɪˈksplɔr/
- 구성: `ex-` = out + `plore`
- 뜻: 탐험하다, 탐구하다, 살펴보다
- 어원적 연결: 밖으로 나가 찾고 살핌
- 사용 맥락: `explore new ideas` = 새로운 생각을 탐구하다

### exploration

- 발음: /ˌɛksplɚˈeɪʃən/
- 구성: `explore` + `-ation`
- 뜻: 탐험, 탐사, 탐구
- 어원적 연결: 낯선 곳이나 가능성을 살피는 행위
- 사용 맥락: `space exploration` = 우주 탐사

### exploratory

- 발음: /ɪˈksplɔrəˌtɔri/
- 구성: `explore` + `-atory`
- 뜻: 탐색적인, 예비 조사의
- 어원적 연결: 본격적으로 들어가기 전 살펴봄
- 사용 맥락: `exploratory research` = 탐색적 연구

### explorer

- 발음: /ɪˈksplɔrɚ/
- 구성: `explore` + `-er`
- 뜻: 탐험가, 탐구자
- 어원적 연결: 낯선 곳이나 문제를 찾아 살피는 사람
- 사용 맥락: `polar explorers` = 극지 탐험가들

### implore

- 발음: /ˌɪˈmplɔr/
- 구성: `in-` = upon, toward + `plore`
- 뜻: 간청하다, 애원하다
- 어원적 연결: 상대를 향해 소리 내어 호소함
- 사용 맥락: `implore someone to help` = 도와 달라고 간청하다

### deplore

- 발음: /dɪˈplɔr/
- 구성: `de-` = down, strongly + `plore`
- 뜻: 개탄하다, 강하게 유감스러워하다
- 어원적 연결: 슬픔이나 반대를 소리 내어 드러냄
- 사용 맥락: `deplore violence` = 폭력을 개탄하다

### deplorable

- 발음: /dɪˈplɔrəbəl/
- 구성: `deplore` + `-able`
- 뜻: 개탄스러운, 한심한
- 어원적 연결: 강한 유감을 불러일으킬 만함
- 사용 맥락: `deplorable conditions` = 개탄스러운 상황

## 기억 포인트

`plore` 계열은 "소리 내어 찾거나 호소함" 이미지.

- `explore`: 밖으로 나가 살피고 찾음
- `implore`: 상대를 향해 간절히 호소함
- `deplore`: 슬픔이나 반대를 강하게 드러냄

## 비교 단어

### quest

- 뜻: 찾다, 묻다
- 구분 포인트: [[quest]]는 질문하고 추구함. `explore`는 낯선 영역을 살피며 가능성을 찾음

### spect

- 뜻: 보다, 살피다
- 구분 포인트: [[spect]]는 시선으로 봄. `plore`는 원래 소리 내어 찾고 살피는 이미지에서 출발
