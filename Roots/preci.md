# preci

- 출발 단어: `praise`
- 핵심 어원: Latin `pretium` = price, value, worth, reward
- 핵심 이미지: 가치를 매기고 귀하게 여김
- 관련 문서: [[val]], [[estim]], [[cost]]

## 단어들

### praise

- 발음: /ˈpreɪz/
- 구성: Old French `preisier`, `prisier` = value, esteem, praise
- 뜻: 칭찬하다, 찬양하다; 칭찬
- 어원적 연결: 어떤 대상의 가치를 높이 평가함
- 사용 맥락: `praise someone's effort` = 누군가의 노력을 칭찬하다

### praised

- 발음: /ˈpreɪzd/
- 구성: `praise` + `-ed`
- 뜻: 칭찬받은, 찬양받은
- 어원적 연결: 가치가 인정되어 높이 평가된 상태
- 사용 맥락: `a widely praised novel` = 널리 칭찬받은 소설

### praising

- 발음: /ˈpreɪzɪŋ/
- 구성: `praise` + `-ing`
- 뜻: 칭찬하는, 찬양하는
- 어원적 연결: 가치를 말로 높이 평가하는 행위
- 사용 맥락: `praising the team's work` = 팀의 작업을 칭찬함

### price

- 발음: /ˈpraɪs/
- 구성: Old French `pris` < Latin `pretium`
- 뜻: 가격, 대가
- 어원적 연결: 어떤 것에 매겨진 가치
- 사용 맥락: `a high price` = 높은 가격

### prize

- 발음: /ˈpraɪz/
- 구성: Old French `pris` = reward, value
- 뜻: 상, 상품; 귀하게 여기다
- 어원적 연결: 가치 있는 보상이나 귀하게 여기는 것
- 사용 맥락: `win a prize` = 상을 타다, `prize honesty` = 정직을 귀하게 여기다

### precious

- 발음: /ˈprɛʃəs/
- 구성: Latin `pretiosus` = valuable, costly
- 뜻: 귀중한, 소중한
- 어원적 연결: 값지고 가치가 높음
- 사용 맥락: `precious time` = 귀중한 시간

### appreciate

- 발음: /əˈpriʃiˌeɪt/
- 구성: `ad-` = to + Latin `pretium` = value
- 뜻: 진가를 알아보다, 감사하다; 가치가 오르다
- 어원적 연결: 대상의 가치를 향해 평가함
- 사용 맥락: `appreciate your help` = 당신의 도움에 감사하다

### appreciation

- 발음: /əˌpriʃiˈeɪʃən/
- 구성: `appreciate` + `-ion`
- 뜻: 감사, 감상, 진가 인식; 가치 상승
- 어원적 연결: 가치를 알아보고 인정하는 행위
- 사용 맥락: `show appreciation` = 감사를 표현하다

### appraise

- 발음: /əˈpreɪz/
- 구성: Old French `apreisier` = set a price on
- 뜻: 평가하다, 감정하다
- 어원적 연결: 가치를 매김
- 사용 맥락: `appraise a property` = 부동산을 감정하다

### appraisal

- 발음: /əˈpreɪzəl/
- 구성: `appraise` + `-al`
- 뜻: 평가, 감정
- 어원적 연결: 대상의 가치를 따져 매긴 결과
- 사용 맥락: `performance appraisal` = 성과 평가

## 기억 포인트

`preci` 계열은 "가치, 값, 귀하게 여김" 이미지.

- `praise`: 가치를 높이 평가해 칭찬함
- `price`: 매겨진 값
- `prize`: 가치 있는 보상, 귀하게 여기다
- `precious`: 값지고 소중한
- `appreciate`: 가치를 알아보다, 감사하다
- `appraise`: 값을 매겨 평가하다

## 비교 단어

### val

- 뜻: 힘이 있다, 가치가 있다, 유효하다
- 구분 포인트: [[val]]은 가치가 "통함/효력이 있음"에 가까움. `preci`는 가격·평가·귀중함에 가까움.

### estim

- 뜻: 가치를 매기다, 어림해 판단하다
- 구분 포인트: [[estim]]은 평가나 추정 행위. `preci`는 가격과 가치 자체의 이미지가 강함.

### laud

- 뜻: 칭찬하다, 찬양하다
- 구분 포인트: `laud`는 Latin `laudare` = praise 계열. `praise`는 `pretium`의 "가치 평가"에서 온 말.
