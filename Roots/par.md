# par

- 출발 단어: `disparage`
- 핵심 어원: Latin `par` = equal, matched
- 핵심 이미지: 같은 수준·지위·값에 맞음
- 관련 문서: [[equ]], [[simil]], [[comit]]
- 주의: `parboil`의 `par-`는 "부분적으로"로 재해석된 별도 형태. `disparage`, `pair`, `peer`, `parity`, `compare`의 `par`는 같음·동등함 계열.

## 출발 단어 분석

`disparage` ← Old French `desparagier` = reduce in rank, marry unequally

- `dis-/des-`: away, down, apart
- `parage`: equality of rank, lineage
- 원래 이미지: 동등한 지위·혈통의 수준에서 떨어뜨림
- 의미 변화: 낮은 신분과 연결해 명예를 떨어뜨림 → 가치를 낮게 말함 → 깎아내리다
- 현대 의미: 깎아내리다, 폄하하다

## 단어들

### disparage

- 발음: /dɪˈspɛrɪdʒ/
- 구성: `dis-` = away/down + `parage` = equality of rank, lineage
- 뜻: 깎아내리다, 폄하하다
- 어원적 연결: 동등한 수준에서 아래로 떨어뜨려 평가함
- 사용 맥락: `disparage someone's efforts` = 누군가의 노력을 깎아내리다

### disparaging

- 발음: /dɪˈspɛrɪdʒɪŋ/
- 구성: `disparage` + `-ing`
- 뜻: 폄하하는, 깎아내리는
- 어원적 연결: 상대의 가치나 지위를 낮춰 말하는 태도
- 사용 맥락: `a disparaging remark` = 폄하하는 발언

### disparagement

- 구성: `disparage` + `-ment`
- 뜻: 폄하, 비하
- 어원적 연결: 동등한 가치나 지위에서 낮춰 평가하는 행위
- 사용 맥락: `commercial disparagement` = 상업적 비방

### par

- 발음: /ˈpɑr/
- 구성: Latin `par` = equal
- 뜻: 동등한 수준, 기준치
- 어원적 연결: 어떤 기준과 같은 값이나 수준
- 사용 맥락: `on par with` = ~와 동등한 수준의

### pair

- 발음: /ˈpɛr/
- 구성: Old French `paire` ← Latin `paria` = equal things
- 뜻: 한 쌍, 짝
- 어원적 연결: 서로 맞고 동등하게 짝을 이루는 두 물건
- 사용 맥락: `a pair of shoes` = 신발 한 켤레

### peer

- 발음: /ˈpɪr/
- 구성: Old French `per` ← Latin `par` = equal
- 뜻: 동료, 또래; 귀족
- 어원적 연결: 지위나 나이가 같은 사람. 귀족 의미는 같은 등급의 고위층에서 발전
- 사용 맥락: `peer pressure` = 또래 압력, `peer review` = 동료 평가

### peerage

- 발음: /ˈpɪrədʒ/
- 구성: `peer` + `-age`
- 뜻: 귀족 계급, 귀족 신분
- 어원적 연결: 동등한 귀족들의 집단이나 지위
- 사용 맥락: `inherit a peerage` = 귀족 작위를 상속하다

### parity

- 발음: /ˈpɛrəti/
- 구성: Latin `paritas` = equality
- 뜻: 동등함, 균등, 동수
- 어원적 연결: 양쪽이 같은 수준이나 값에 놓임
- 사용 맥락: `gender parity` = 성평등

### disparity

- 발음: /dɪˈspɛrəti/
- 구성: `dis-` = apart/not + `parity`
- 뜻: 격차, 불균형, 차이
- 어원적 연결: 동등하지 않고 서로 벌어진 상태
- 사용 맥락: `income disparity` = 소득 격차

### compare

- 발음: /kəˈmpɛr/
- 구성: Latin `com-` = together + `par` = equal
- 뜻: 비교하다
- 어원적 연결: 둘을 함께 놓고 같은지 다른지 맞춰 봄
- 사용 맥락: `compare two options` = 두 선택지를 비교하다

### comparable

- 발음: /ˈkɑmpɚəbəl/
- 구성: `compare` + `-able`
- 뜻: 비교할 만한, 비슷한 수준의
- 어원적 연결: 함께 놓고 맞춰 볼 수 있을 만큼 관련되거나 동등함
- 사용 맥락: `comparable data` = 비교 가능한 자료

### peerless

- 발음: /ˈpɪrlɪs/
- 구성: `peer` + `-less`
- 뜻: 비길 데 없는
- 어원적 연결: 동등한 상대가 없음
- 사용 맥락: `peerless skill` = 비길 데 없는 솜씨

## 기억 포인트

`par` 계열은 "같은 수준" 이미지.

- `par`: 기준과 같은 수준
- `pair`: 서로 맞는 두 짝
- `peer`: 같은 지위·나이의 사람
- `parity`: 동등함
- `compare`: 함께 놓고 같은지 따짐
- `disparity`: 동등하지 않은 격차
- `disparage`: 같은 수준에서 낮춰 말함

## 비교 단어

### equ

- 뜻: 같음, 균등함
- 구분 포인트: [[equ]]는 균형 잡힌 같음. `par`는 수준·지위·값이 맞는 느낌.

### simil

- 뜻: 비슷함
- 구분 포인트: [[simil]]은 닮음. `par`는 비교 기준과 동등한 수준.

### comit

- 뜻: 수행원에서 귀족 칭호로 발전
- 구분 포인트: [[comit]]의 귀족 `count`는 수행원 계열. `peer/peerage`는 동등한 지위의 귀족 계열.
