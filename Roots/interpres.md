# interpres

- 출발 단어: `interpret`
- 핵심 어원: Latin `interpres` = agent, broker, translator, explainer
- 핵심 이미지: 사이에 서서 뜻을 풀어 전달함
- 관련 어근: [[medi]], [[dict]], [[logos]]

## 출발 단어 분석

`interpret`는 Latin `interpretari` = explain, expound, understand에서 온 말.

그 바탕의 `interpres`는 "대리인, 중개인, 번역가, 설명하는 사람"의 뜻.

`inter-` = between은 비교적 분명하지만, 뒤 요소는 어원적으로 불확실함. 전통적으로 "거래하다, 값을 매기다" 계열과 연결되기도 하나 단정하지 않음.

따라서 시험 어휘에서는 `interpres` 전체를 "사이에 서서 뜻을 풀어 주는 사람"으로 기억하는 편이 안전함.

## 주의할 형태

`interpretate`는 현대 표준 영어에서 일반적으로 쓰지 않음.

표준형은 다음과 같음.

- 동사: `interpret`
- 명사: `interpretation`
- 사람: `interpreter`
- 형용사: `interpretive`, `interpretative`

## 단어들

### interpret

- 발음: /ˌɪˈntɝprət/
- 구성: Latin `interpretari`
- 의미: 해석하다, 통역하다
- 어원적 연결: 뜻이 바로 통하지 않는 두 대상 사이에서 의미를 풀어 전달함
- 사용 맥락: `interpret the data` = 자료를 해석하다

### interpretation

- 발음: /ˌɪˌntɝprɪˈteɪʃən/
- 구성: `interpret` + `-ation`
- 의미: 해석, 통역, 설명
- 어원적 연결: 의미를 풀어 전달하는 행위나 결과
- 사용 맥락: `different interpretations` = 서로 다른 해석들

### interpreter

- 발음: /ˌɪˈntɝprətɚ/
- 구성: `interpret` + `-er`
- 의미: 통역사, 해석자
- 어원적 연결: 두 언어나 의미 사이에서 뜻을 옮겨 주는 사람
- 사용 맥락: `a Korean-English interpreter` = 한영 통역사

### interpretive

- 발음: /ˌɪˈntɝprətɪv/
- 구성: `interpret` + `-ive`
- 의미: 해석의, 해석적인
- 어원적 연결: 의미를 풀어 설명하는 성격
- 사용 맥락: `interpretive skills` = 해석 능력

### interpretative

- 구성: `interpret` + `-ative`
- 의미: 해석적인
- 어원적 연결: `interpretive`와 거의 같은 뜻. 더 길고 격식 있는 느낌
- 사용 맥락: `an interpretative approach` = 해석적 접근

### misinterpret

- 발음: /mɪsɪˈntɝprət/
- 구성: `mis-` = wrongly + `interpret`
- 의미: 잘못 해석하다
- 어원적 연결: 뜻을 잘못 풀어 전달함
- 사용 맥락: `misinterpret the message` = 메시지를 잘못 해석하다

### reinterpret

- 발음: /ˌriɪˈntɝprət/
- 구성: `re-` = again + `interpret`
- 의미: 재해석하다
- 어원적 연결: 뜻을 다시 풀어 봄
- 사용 맥락: `reinterpret history` = 역사를 재해석하다

### interpretable

- 구성: `interpret` + `-able`
- 의미: 해석 가능한
- 어원적 연결: 뜻을 풀어 설명할 수 있음
- 사용 맥락: `interpretable results` = 해석 가능한 결과

### interpretability

- 구성: `interpretable` + `-ity`
- 의미: 해석 가능성
- 어원적 연결: 결과나 모델을 사람이 이해할 수 있게 풀어 낼 수 있는 성질
- 사용 맥락: `model interpretability` = 모델 해석 가능성

## 기억 포인트

- `interpret`: 뜻을 사이에서 풀어 전달함
- `interpreter`: 사이에서 뜻을 옮기는 사람
- `interpretation`: 풀어 낸 해석
- `misinterpret`: 잘못 풀어 이해함
- `reinterpret`: 다시 풀어 해석함

## 비교 단어

  - 핵심: 사이, 중간
  - 구분 포인트: `mediate`는 갈등이나 관계 사이를 조정함. `interpret`는 의미나 언어 사이를 풀어 전달함

- [[dict]]

  - 구분 포인트: `dict`는 말로 정하거나 선언함. `interpret`는 이미 있는 말이나 자료의 뜻을 풀어 설명함

- [[logos]]
  - 핵심: 말, 설명, 논리
  - 구분 포인트: `logic`은 설명과 이성의 체계. `interpretation`은 특정 의미를 풀어 읽는 행위
