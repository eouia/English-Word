# fug

- 출발 단어: `fugitive`
- 핵심 어원: Latin `fugere` = 달아나다, 피하다
- 핵심 이미지: 위험이나 압박에서 달아남
- 관련 문서: [[scur]], [[tract]]

## 출발 단어 분석

`fugitive` ← Latin `fugere`

- `fugere`: flee
- `fugitivus`: fleeing, runaway
- 원래 이미지: 도망치는 사람
- 현대 의미: 도망자, 일시적인

## 단어들

### fugitive

- 발음: /ˈfjudʒətɪv/
- 구성: Latin `fugitivus`
- 뜻: 도망자; 덧없는, 일시적인
- 어원적 연결: 붙잡히지 않고 달아나는 존재
- 사용 맥락: `a fugitive from justice` = 사법망을 피해 도망친 사람

### refuge

- 발음: /ˈrɛfjudʒ/
- 구성: `re-` = back + Latin `fugere`
- 뜻: 피난처, 도피처
- 어원적 연결: 위험에서 달아나 숨는 곳
- 사용 맥락: `seek refuge` = 피난처를 찾다

### refugee

- 발음: /ˈrɛfjudʒi/
- 구성: `refuge` + `-ee`
- 뜻: 난민, 피난민
- 어원적 연결: 위험에서 피난한 사람
- 사용 맥락: `refugee crisis` = 난민 위기

### subterfuge

- 발음: /ˈsʌbtɚˌfjudʒ/
- 구성: `subter-` = secretly, under + Latin `fugere`
- 뜻: 속임수, 구실
- 어원적 연결: 직접 맞서지 않고 몰래 빠져나가는 수단
- 사용 맥락: `use subterfuge` = 속임수를 쓰다

### centrifugal

- 발음: /ˈsɛntrɪˌfjugəl/
- 구성: Latin `centrum` = center + `fugere` = flee
- 뜻: 원심의, 중심에서 멀어지는
- 어원적 연결: 중심에서 달아나는 방향
- 사용 맥락: `centrifugal force` = 원심력

## 기억 포인트

`fug` 계열은 "달아남, 피함" 이미지.

- `fugitive`: 도망자
- `refuge`: 피난처
- `refugee`: 피난한 사람
- `subterfuge`: 빠져나가기 위한 속임수
- `centrifugal`: 중심에서 달아나는
