# omni

- 출발 단어: `omnipotent`
- 핵심 어원: Latin `omnis` = all, every
- 핵심 이미지: 빠짐없이 모두, 전체 범위
- 관련 문서: [[pot]], [[vor]], [[present]]

## 단어들

### omnipotent

- 발음: /ɑˈmnɪpətənt/
- 구성: `omni-` = all + [[pot]] = power, be able
- 뜻: 전능한, 모든 힘을 가진
- 어원적 연결: 모든 것을 할 수 있는 힘
- 사용 맥락: `an omnipotent ruler` = 절대 권력을 가진 통치자

### omnipotence

- 발음: /ɑˈmnɪpətəns/
- 구성: `omni-` + `potence`
- 뜻: 전능, 절대적 권능
- 어원적 연결: 모든 힘을 가진 상태
- 사용 맥락: 신의 능력이나 절대 권력을 말할 때 사용

### omniscient

- 발음: /ɑˈmnɪʃənt/
- 구성: `omni-` = all + Latin `scire` = know
- 뜻: 전지한, 모든 것을 아는
- 어원적 연결: 모든 지식을 가진 상태
- 사용 맥락: `an omniscient narrator` = 전지적 작가 시점의 서술자

### omniscience

- 구성: `omni-` + `science`
- 뜻: 전지, 모든 것을 앎
- 어원적 연결: 지식의 범위가 전체로 확장됨
- 사용 맥락: 종교, 철학, 문학 비평에서 자주 사용

### omnipresent

- 발음: /ˌɑmnɪˈprɛzənt/
- 구성: `omni-` = all + `present` = being at hand
- 뜻: 어디에나 있는, 편재하는
- 어원적 연결: 모든 장소에 존재함
- 사용 맥락: `omnipresent advertising` = 어디에나 보이는 광고

### omnipresence

- 발음: /ˌɑmnəˈprɛzəns/
- 구성: `omni-` + `presence`
- 뜻: 편재, 어디에나 있음
- 어원적 연결: 존재의 범위가 모든 곳으로 확장됨
- 사용 맥락: 기술, 감시, 미디어처럼 늘 주변에 있는 것을 말할 때 사용

### omnivore

- 구성: `omni-` = all + [[vor]] = eat
- 뜻: 잡식동물
- 어원적 연결: 모든 종류의 음식을 먹음
- 사용 맥락: `humans are omnivores` = 인간은 잡식동물

### omnivorous

- 발음: /ɑˈmnɪvɚəs/
- 구성: `omnivore` + `-ous`
- 뜻: 잡식성의; 무엇이든 탐하는
- 어원적 연결: 음식이나 관심사가 한 종류에 제한되지 않음
- 사용 맥락: `an omnivorous reader` = 분야를 가리지 않고 읽는 사람

### omnibus

- 발음: /ˈɑmnəbəs/
- 구성: Latin `omnibus` = for all
- 뜻: 합본, 여러 작품을 묶은 것; 옴니버스
- 어원적 연결: 여러 대상을 모두 함께 실음
- 사용 맥락: `an omnibus edition` = 합본판

## 기억 포인트

`omni-`는 "모든, 전체" 범위를 붙이는 접두사.

- `omnipotent`: 모든 힘을 가진
- `omniscient`: 모든 것을 아는
- `omnipresent`: 어디에나 있는
- `omnivore`: 모든 것을 먹는
- `omnibus`: 모두를 위한 것, 여러 것을 묶은 것

## 비교 단어

### pan-

- 뜻: Greek `pan` = all
- 구분 포인트: `omni-`는 Latin 계열. `pan-`은 Greek 계열. 단, 기존 [[pan]] 문서는 "빵, 동료" 계열이라 어원이 다름.

### all

- 뜻: 모두, 전체
- 구분 포인트: 일상어에서는 `all-`이 자연스러움. `omni-`는 학술적, 종교적, 문학적 느낌이 강함.
