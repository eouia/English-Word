# recent

- 출발 단어: `recent`
- 핵심 어원: Latin `recens` / `recentis` = 새로운, 갓 생긴, 신선한
- 깊은 어근: PIE `*ken-` = fresh, new, young
- 핵심 이미지: 아직 오래되지 않은 새로움
- 관련 문서: [[nov]], [[nat]], [[ante]]
- 주의: `recent`의 `re-`는 현대 영어 접두사 `again/back`으로 분석하지 않는 편이 안전.

## 출발 단어 분석

`recent` ← Latin `recentem` ← `recens`

- `recens`: 최근의, 새로 생긴, 신선한
- 원래 이미지: 막 만들어져 아직 낡지 않은 상태
- 현대 의미: 최근의, 얼마 전의

## 단어들

### recent

- 구성: Latin `recens/recentis`
- 뜻: 최근의, 얼마 전의
- 어원적 연결: 새로 생겨 아직 오래되지 않음
- 사용 맥락: `recent research` = 최근 연구

### recently

- 구성: `recent` + `-ly`
- 뜻: 최근에
- 어원적 연결: 오래되지 않은 시점에
- 사용 맥락: `I recently moved` = 나는 최근에 이사했다

### recency

- 구성: `recent` + `-cy`
- 뜻: 최신성, 최근성
- 어원적 연결: 얼마나 최근에 생겼는가의 성질
- 사용 맥락: `recency effect` = 최신 효과

### recentness

- 구성: `recent` + `-ness`
- 뜻: 최근성
- 어원적 연결: 오래되지 않은 상태
- 사용 맥락: `the recentness of the data` = 자료의 최신성

## 기억 포인트

`recent` 계열은 "갓 생겨 오래되지 않음" 이미지.

- `recent`: 최근의
- `recently`: 최근에
- `recency`: 최신성, 최근성

## 비교 단어

### nov

- 뜻: 새로움, 새롭게 만들다
- 구분 포인트: [[nov]]는 Latin `novus` = new 계열. `innovate`, `novel`, `renovate`처럼 "새것/새 방식" 중심.

### new

- 뜻: 새로운
- 구분 포인트: 게르만계 기본어. `recent`는 시간적으로 오래되지 않았다는 느낌이 강함.

### ancient

- 뜻: 오래된, 고대의
- 구분 포인트: [[ante]] 계열의 `ancient`는 "앞선 시대의, 오래된" 쪽. `recent`와 반대 방향.
