# solv

- 출발 단어: `solution`
- 핵심 어원: 라틴어 `solvere` / `solut-` = 풀다, 느슨하게 하다
- 핵심 이미지: 묶인 것을 풀어 해결함
- 관련해서 헷갈릴 수 있는 어근: [[ject]] = 던지다 계열, [[rupt]] = 깨뜨리다 계열

## 출발 단어 분석

`solution`은 Latin `solutio`에서 온 말.

- `solvere`: loosen, release, solve
- `solutio`: 풀어 냄, 해소

따라서 `solution`은 "묶인 문제를 풀어 냄" → "해결책"의 뜻.  
화학에서는 물질이 풀려 섞인 것 → "용액"의 뜻.

## 단어들

### solve

- 발음: /ˈsɑlv/
- 구성: Latin `solvere`
- 뜻: 풀다, 해결하다
- 어원적 연결: 묶인 문제를 풀어냄
- 사용 맥락: `solve a problem` = 문제를 해결하다

### solution

- 발음: /səˈluʃən/
- 구성: `solut` + `-ion`
- 뜻: 해결책, 해법, 용액
- 어원적 연결: 풀어 낸 결과. 화학에서는 녹아 풀린 혼합물
- 사용 맥락: `find a solution` = 해결책을 찾다

### soluble

- 발음: /ˈsɑljəbəl/
- 구성: `solv` + `-ible`
- 뜻: 녹는, 해결 가능한
- 어원적 연결: 풀릴 수 있음
- 사용 맥락: `water-soluble vitamins` = 수용성 비타민

### dissolve

- 발음: /dɪˈzɑlv/
- 구성: `dis-` = apart + `solve`
- 뜻: 녹다, 해산하다, 사라지다
- 어원적 연결: 묶인 것이 풀려 흩어짐
- 사용 맥락: `dissolve sugar in water` = 물에 설탕을 녹이다

### dissolution

- 발음: /ˌdɪsəˈluʃən/
- 구성: `dissolve` + `-tion`
- 뜻: 해산, 해체, 용해
- 어원적 연결: 묶인 구조가 풀려 흩어짐
- 사용 맥락: `the dissolution of parliament` = 의회의 해산

### resolve

- 발음: /riˈzɑlv/
- 구성: `re-` = back, again + `solve`
- 뜻: 해결하다, 결심하다
- 어원적 연결: 다시 풀어 정리함. 마음속 갈등을 풀어 결심함
- 사용 맥락: `resolve a conflict` = 갈등을 해결하다

### resolution

- 발음: /ˌrɛzəˈluʃən/
- 구성: `resolve` + `-tion`
- 뜻: 해결, 결심, 결의안, 해상도
- 어원적 연결: 문제를 풀어 정한 결과
- 사용 맥락: `make a resolution` = 결심하다

### resolute

- 발음: /ˈrɛzəˌlut/
- 구성: Latin `resolutus`
- 뜻: 단호한, 결연한
- 어원적 연결: 마음속 망설임이 풀려 굳게 정해진 상태
- 사용 맥락: `a resolute leader` = 단호한 지도자

### absolute

- 발음: /ˈæbsəˌlut/
- 구성: `ab-` = away + `solut`
- 뜻: 절대적인, 완전한
- 어원적 연결: 다른 것에서 풀려 떨어져 독립된 상태
- 사용 맥락: `absolute power` = 절대 권력

### absolve

- 발음: /əˈbzɑlv/
- 구성: `ab-` = away + `solve`
- 뜻: 면제하다, 무죄로 하다
- 어원적 연결: 책임이나 죄에서 풀어 줌
- 사용 맥락: `absolve someone of blame` = 책임을 면제해 주다

### solvent

- 발음: /ˈsɑlvənt/
- 구성: `solv` + `-ent`
- 뜻: 용매, 지불 능력이 있는
- 어원적 연결: 녹여 푸는 물질. 빚을 풀 수 있을 만큼 지불 능력이 있음
- 사용 맥락: `organic solvent` = 유기 용매

### insolvent

- 발음: /ˌɪˈnsɑlvənt/
- 구성: [[in]] = not + `solvent`
- 뜻: 지급 불능의
- 어원적 연결: 빚을 풀어 갚을 능력이 없음
- 사용 맥락: `become insolvent` = 지급 불능이 되다

## 기억 포인트

`solv/solut` 계열은 "풀다 / 해결하다 / 녹이다" 이미지.

- `solve`: 문제를 풀다
- `solution`: 풀어 낸 결과, 해결책
- `dissolve`: 풀려 흩어지다
- `resolve`: 다시 풀어 해결하거나 결심하다
- `absolute`: 다른 것에서 풀려 독립됨
- `absolve`: 책임에서 풀어 줌
- `solvent`: 녹여 푸는 것 / 빚을 풀 능력

## 비교 단어

### rupt

- 뜻: 깨뜨리다, 터뜨리다
- 구분 포인트: [[rupt]]는 연결을 깨뜨림. `solv`는 묶임을 풀어 해소함
