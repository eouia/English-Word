# mun

- 출발 단어: `immune`
- 핵심 어원: Latin `munus/muneris` = duty, service, gift
- 핵심 이미지: 공동체에 지는 의무나 주고받는 봉사·선물
- 관련 문서: [[com]], [[civ]]
- 주의: `immune`의 원래 뜻은 "공적 의무에서 면제된". 현대 의학 의미는 "질병의 영향에서 면제·보호됨"으로 확장.

## 출발 단어 분석

`immune` = `in-` + `munis`

- `in-`: not
- `munis`: performing service, obligated
- 원래 이미지: 공동체에 지는 의무를 지지 않음
- 현대 의미: 면역의, 영향을 받지 않는, 면제된

## 단어들

### immune

- 발음: /ˌɪˈmjun/
- 구성: Latin `immunis` ← `in-` = not + `munis` = obligated, serving
- 뜻: 면역의, 영향을 받지 않는; 면제된
- 어원적 연결: 의무나 부담에서 벗어난 상태가 질병·공격의 영향에서 보호된 상태로 확장
- 사용 맥락: `immune to disease` = 질병에 면역이 있는, `immune from prosecution` = 기소에서 면제된

### immunity

- 발음: /ˌɪˈmjunəti/
- 구성: `immune` + `-ity`
- 뜻: 면역, 면책, 면제
- 어원적 연결: 질병·처벌·의무의 영향에서 벗어난 상태
- 사용 맥락: `diplomatic immunity` = 외교 면책, `natural immunity` = 자연 면역

### immunize

- 발음: /ˈɪmjuˌnaɪz/
- 구성: `immune` + `-ize`
- 뜻: 면역이 생기게 하다, 예방 접종하다
- 어원적 연결: 질병의 영향에서 보호된 상태로 만듦
- 사용 맥락: `immunize children against measles` = 아이들에게 홍역 예방 접종을 하다

### immunization

- 발음: /ˌɪmjunəˈzeɪʃən/
- 구성: `immunize` + `-ation`
- 뜻: 면역화, 예방 접종
- 어원적 연결: 질병의 영향에서 벗어나도록 만드는 과정
- 사용 맥락: `an immunization program` = 예방 접종 프로그램

### autoimmune

- 발음: /ˌɔˈtɔɪmjun/
- 구성: `auto-` = self + `immune`
- 뜻: 자가면역의
- 어원적 연결: 면역 체계가 자기 몸을 대상으로 반응함
- 사용 맥락: `an autoimmune disease` = 자가면역 질환

### commune

- 발음: /ˈkɑmjun/
- 구성: Latin `communis` ← `com-` = together + `munis`
- 뜻: 공동체; 친밀하게 교감하다
- 어원적 연결: 의무와 자원을 함께 나누는 집단
- 사용 맥락: `live in a commune` = 공동체에서 살다

### communal

- 발음: /kəˈmjunəl/
- 구성: `commune` + `-al`
- 뜻: 공동의, 공동체의
- 어원적 연결: 공동체가 함께 나누고 맡는 성질
- 사용 맥락: `communal life` = 공동체 생활

### municipal

- 발음: /mjuˈnɪsəpəl/
- 구성: Latin `municipium` ← `munus` = duty + `capere` = take
- 뜻: 지방 자치의, 시의
- 어원적 연결: 공적 의무를 맡은 시민 공동체와 관련됨
- 사용 맥락: `municipal government` = 지방 정부, `municipal services` = 시 공공 서비스

### remuneration

- 발음: /rɪˌmjunɚˈeɪʃən/
- 구성: Latin `remunerari` = reward, repay
- 뜻: 보수, 급료, 보상
- 어원적 연결: 봉사나 의무 수행에 대한 되돌려 주는 선물·보상
- 사용 맥락: `fair remuneration` = 공정한 보수

### munificent

- 구성: Latin `munificus` = generous in giving
- 뜻: 후한, 아낌없이 주는
- 어원적 연결: 선물이나 혜택을 크게 베푸는 성질
- 사용 맥락: `a munificent donation` = 후한 기부

## 기억 포인트

`mun`은 "의무·봉사·선물" 이미지.

- `immune`: 의무나 영향에서 면제됨
- `immunity`: 면제·면역 상태
- `immunize`: 면역 상태로 만듦
- `commune`: 의무와 자원을 함께 나누는 공동체
- `municipal`: 공적 의무를 맡은 도시 공동체의
- `remuneration`: 봉사에 대한 보상
- `munificent`: 선물을 후하게 베풂

## 비교 단어

### inoculate

- 뜻: 접종하다
- 구분 포인트: 원래 "눈접을 붙이다"에서 온 단어. 현대 의학에서는 `immunize`와 겹치지만 어원은 다름.

### exempt

- 뜻: 면제된
- 구분 포인트: Latin `eximere` = take out 계열. `immune`은 원래 공적 의무를 지지 않는 상태.

### community

- 뜻: 공동체
- 연결 포인트: `communis` 계열로 `mun`과 연결 가능. 다만 문서가 커지면 `commune/community` 갈래를 별도 분리할 수 있음.
