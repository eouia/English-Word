# anthrop

- 출발 단어: `misanthropic`
- 핵심 어원: Greek `anthropos` = human being, 인간
- 핵심 이미지: 인간, 사람 전체
- 관련해서 헷갈릴 수 있는 어근: [[phil]] = 사랑함, 좋아함 계열, [[morph]] = 형태, 모양 계열

## 출발 단어 분석

`misanthropic`은 `mis-` + `anthrop` + `-ic`.

- `mis-`: Greek `misein` = hate
- `anthrop`: human
- `-ic`: 형용사형 접미사

따라서 `misanthropic`은 "인간을 싫어하는"의 뜻. 사람 전체에 대한 불신이나 혐오를 보이는 태도를 가리킴.

## 단어들

### misanthropic

- 구성: `mis-` = hate + `anthrop` = human + `-ic`
- 뜻: 사람을 싫어하는, 인간 혐오적인
- 어원적 연결: 인간 전체를 싫어하거나 불신함
- 사용 맥락: `a misanthropic attitude` = 인간 혐오적인 태도

### misanthrope

- 구성: `mis-` = hate + `anthrop`
- 뜻: 인간 혐오자
- 어원적 연결: 사람들을 싫어하는 사람
- 사용 맥락: `He became a misanthrope.` = 그는 인간 혐오자가 되었다

### anthropology

- 구성: `anthrop` = human + `-logy` = study
- 뜻: 인류학
- 어원적 연결: 인간과 인간 문화를 연구함
- 사용 맥락: `cultural anthropology` = 문화인류학

### anthropologist

- 구성: `anthropology` + `-ist`
- 뜻: 인류학자
- 어원적 연결: 인간과 문화를 연구하는 사람
- 사용 맥락: `a field anthropologist` = 현장 인류학자

### anthropomorphic

- 구성: `anthrop` = human + [[morph]] = form + `-ic`
- 뜻: 의인화된, 사람 모양의
- 어원적 연결: 인간의 형태나 성질을 지닌 것처럼 표현함
- 사용 맥락: `anthropomorphic animals` = 의인화된 동물

### philanthropy

- 구성: [[phil]] = love + `anthrop` = human
- 뜻: 자선, 박애
- 어원적 연결: 인간을 사랑함
- 사용 맥락: `corporate philanthropy` = 기업 자선 활동

### philanthropist

- 구성: `philanthropy` + `-ist`
- 뜻: 자선가, 박애주의자
- 어원적 연결: 인간을 사랑해 돕는 사람
- 사용 맥락: `a wealthy philanthropist` = 부유한 자선가

## 기억 포인트

`anthrop` 계열은 "인간" 이미지.

- `misanthropic`: 인간을 싫어하는
- `misanthrope`: 인간 혐오자
- `anthropology`: 인간 연구 → 인류학
- `anthropomorphic`: 인간 형태의
- `philanthropy`: 인간 사랑 → 자선, 박애

## 비교 단어

### phil

- 뜻: 사랑함, 좋아함
- 구분 포인트: [[phil]]은 사랑이나 애호의 이미지. `philanthropy`는 `phil` + `anthrop`라서 두 문서 모두와 연결됨

### human

- 뜻: 인간의, 인간
- 구분 포인트: 영어 일반어. `anthrop`는 Greek 계열 학술·격식 어휘에서 주로 나타남
