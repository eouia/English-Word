# culp

- 출발 단어: `culpable`
- 핵심 어원: Latin `culpa` = 잘못, 죄, 책임
- 핵심 이미지: 잘못에 대한 책임
- 관련 문서: [[caus]], [[jud]]

## 출발 단어 분석

`culpable` ← Latin `culpa`

- `culpa`: fault, blame
- `culpabilis`: blameworthy
- 원래 이미지: 비난받을 만한 잘못이 있음
- 현대 의미: 책임이 있는, 과실이 있는

## 단어들

### culpable

- 발음: /ˈkʌlpəbəl/
- 구성: Latin `culpa` + `-able`
- 뜻: 책임이 있는, 비난받을 만한
- 어원적 연결: 잘못의 책임을 질 수 있음
- 사용 맥락: `culpable negligence` = 책임 있는 과실

### culpability

- 발음: /ˌkʌlpəˈbɪlɪti/
- 구성: `culpable` + `-ity`
- 뜻: 책임, 과실 책임
- 어원적 연결: 잘못에 대해 비난받을 수 있는 상태
- 사용 맥락: `establish culpability` = 책임을 입증하다

### culprit

- 발음: /ˈkʌlprɪt/
- 구성: 역사적으로 법정 표현에서 발전
- 뜻: 범인, 원인 제공자
- 어원적 연결: 잘못의 책임이 있는 사람이나 원인
- 사용 맥락: `the main culprit` = 주범, 주요 원인
- 주의: 사람뿐 아니라 문제의 원인에도 씀.

### exculpate

- 발음: /ˌɛˈkskʌlpeɪt/
- 구성: `ex-` = out + `culpa`
- 뜻: 무죄임을 밝히다, 책임에서 벗어나게 하다
- 어원적 연결: 잘못의 책임 밖으로 빼냄
- 사용 맥락: `exculpatory evidence` = 무죄를 뒷받침하는 증거

### inculpate

- 구성: `in-` = into + `culpa`
- 뜻: 죄를 지우다, 혐의를 씌우다
- 어원적 연결: 잘못의 책임 안에 넣음
- 사용 맥락: `evidence that inculpates the suspect` = 용의자에게 혐의를 두게 하는 증거

## 기억 포인트

`culp` 계열은 "잘못과 책임" 이미지.

- `culpable`: 책임이 있는
- `culpability`: 책임 있음
- `culprit`: 잘못의 주체나 원인
- `exculpate`: 책임에서 벗어나게 함
- `inculpate`: 책임 안에 넣음
