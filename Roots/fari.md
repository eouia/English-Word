# fari

- 출발 단어: `fable`
- 핵심 어원: Latin `fari` = speak, tell
- 핵심 이미지: 말해진 이야기, 말로 드러난 평판이나 운명
- 관련 어근: [[dict]], [[voc]], [[logos]]

## 출발 단어 분석

`fable`은 Latin `fabula`에서 온 말.

`fabula`는 "이야기, 말해진 것"의 뜻이며, Latin `fari` = speak, tell에서 나옴.

따라서 `fable`은 "말해진 이야기" → "우화, 교훈적 이야기"의 뜻.

## 단어들

### fable

- 구성: Latin `fabula` < `fari`
- 의미: 우화, 꾸며 낸 이야기
- 어원적 연결: 말로 전해지는 이야기
- 사용 맥락: `Aesop's fables` = 이솝 우화

### fabulous

- 구성: Latin `fabulosus` < `fabula`
- 의미: 굉장한, 전설적인, 믿기 어려운
- 어원적 연결: 이야기 속에 나올 만큼 놀라움
- 사용 맥락: `a fabulous view` = 멋진 전망

### fabulate

- 구성: Latin `fabulari` = tell stories
- 의미: 이야기를 꾸며 내다
- 어원적 연결: 말로 이야기를 만들어 냄
- 사용 맥락: 문학 비평에서 주로 쓰임

### confabulate

- 구성: `con-` = together + `fabulari`
- 의미: 담소하다, 꾸며 낸 기억을 말하다
- 어원적 연결: 함께 이야기함. 심리학에서는 기억의 빈틈을 이야기로 채움
- 사용 맥락: `confabulate details` = 세부 사항을 꾸며 말하다

### affable

- 구성: `ad-` = to + `fari`
- 의미: 상냥한, 말 붙이기 쉬운
- 어원적 연결: 말을 걸 수 있음
- 사용 맥락: `an affable host` = 상냥한 주인

### ineffable

- 구성: [[in]] = not + `effable`
- 의미: 말로 표현할 수 없는
- 어원적 연결: 말로 꺼내 표현할 수 없음
- 사용 맥락: `ineffable beauty` = 말로 표현할 수 없는 아름다움

### fame

- 구성: Latin `fama` = talk, rumor, reputation
- 의미: 명성
- 어원적 연결: 사람들 사이에 말로 퍼진 평판
- 사용 맥락: `gain fame` = 명성을 얻다

### famous

- 구성: `fame` + `-ous`
- 의미: 유명한
- 어원적 연결: 말로 널리 알려진
- 사용 맥락: `a famous actor` = 유명한 배우

### defame

- 구성: `de-` = down + `fame`
- 의미: 명예를 훼손하다
- 어원적 연결: 나쁜 말로 평판을 떨어뜨림
- 사용 맥락: `defame a public figure` = 공인을 명예훼손하다

### fate

- 구성: Latin `fatum` = thing spoken
- 의미: 운명
- 어원적 연결: 신들이 말해 정해진 것이라는 이미지
- 사용 맥락: `accept one's fate` = 운명을 받아들이다

### fatal

- 구성: Latin `fatalis` < `fatum`
- 의미: 치명적인, 운명적인
- 어원적 연결: 운명처럼 정해진 죽음이나 결과
- 사용 맥락: `a fatal accident` = 치명적인 사고

### infant

- 구성: [[in]] = not + `fans` = speaking
- 의미: 유아
- 어원적 연결: 아직 말하지 못하는 아이
- 사용 맥락: `infant care` = 유아 돌봄

## 기억 포인트

- `fable`: 말해진 이야기
- `fabulous`: 이야기 속처럼 굉장한
- `affable`: 말을 걸기 쉬운
- `ineffable`: 말로 표현할 수 없는
- `fame`: 말로 퍼진 평판
- `fate`: 말해져 정해진 운명
- `infant`: 말하지 못하는 아이

## 비교 단어

- [[dict]]
  - 핵심: 말하다, 선언하다
  - 구분 포인트: `dict`는 말로 정하거나 선언함. `fari`는 말해진 이야기, 평판, 운명 쪽으로 많이 남음

- [[voc]]
  - 핵심: 부르다, 목소리
  - 구분 포인트: `voc`는 소리 내어 부름. `fari`는 말하고 이야기하는 내용

- [[logos]]
  - 핵심: 말, 설명, 이성
  - 구분 포인트: `logos`는 말과 논리의 체계. `fari`는 라틴계 "말하다"에서 나온 이야기/평판 단어

- [[fac]]
  - 핵심: 만들다, 하다
  - 구분 포인트: `fable/fabulous`는 `fac`와 철자가 비슷하지만 어원 다름
