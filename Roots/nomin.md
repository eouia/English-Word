# nomin

- 출발 단어: `nominate`
- 핵심 어원: Latin `nomen` / `nomin-` = 이름
- 핵심 이미지: 이름을 부르고 공식적으로 지명함
- 관련 문서: [[onym]], [[voc]], [[dict]], [[numer]]
- 주의: `denominator`는 [[numer]] 문서에도 걸치는 수학 용어.

## 출발 단어 분석

`nominate` ← Latin `nominare`

- `nomen`: name
- `nominare`: name, call by name
- 원래 이미지: 이름을 불러 지명함
- 현대 의미: 지명하다, 후보로 추천하다

## 단어들

### nominate

- 발음: /ˈnɑmənət/
- 구성: Latin `nominare`
- 뜻: 지명하다, 후보로 추천하다
- 어원적 연결: 이름을 공식적으로 부름
- 사용 맥락: `nominate a candidate` = 후보를 지명하다

### nomination

- 발음: /ˌnɑməˈneɪʃən/
- 구성: `nominate` + `-ion`
- 뜻: 지명, 후보 추천
- 어원적 연결: 이름을 올려 후보로 삼는 행위
- 사용 맥락: `receive a nomination` = 후보 지명을 받다

### nominee

- 발음: /ˌnɑməˈni/
- 구성: `nominate` + `-ee`
- 뜻: 지명된 사람, 후보자
- 어원적 연결: 이름이 불려 후보가 된 사람
- 사용 맥락: `an Oscar nominee` = 오스카 후보자

### nominal

- 발음: /ˈnɑmənəl/
- 구성: Latin `nomen` + `-al`
- 뜻: 명목상의, 이름뿐인, 아주 적은
- 어원적 연결: 실제보다 이름상으로만 존재함
- 사용 맥락: `a nominal fee` = 명목상의 적은 요금

### nomenclature

- 발음: /ˈnoʊməˌnkleɪtʃɚ/
- 구성: Latin `nomen` = name + `calare` = call
- 뜻: 명명법, 전문 용어 체계
- 어원적 연결: 이름을 붙이고 부르는 체계
- 사용 맥락: `scientific nomenclature` = 과학 명명법

### denominator

- 발음: /dɪˈnɑməˌneɪtɚ/
- 구성: `de-` + Latin `nominare` = name
- 뜻: 분모, 공통 특성
- 어원적 연결: 분수에서 단위를 이름 붙여 구분하는 수
- 사용 맥락: `a common denominator` = 공통분모

### denomination

- 발음: /dɪˌnɔməˈneɪʃən/
- 구성: `de-` + `nominare`
- 뜻: 명칭, 종파, 화폐 단위
- 어원적 연결: 이름을 붙여 구분한 범주
- 사용 맥락: `a religious denomination` = 종파

## 기억 포인트

`nomin` 계열은 "이름을 붙이고 지명함" 이미지.

- `nominate`: 이름을 불러 후보로 세움
- `nominee`: 지명된 사람
- `nominal`: 이름뿐인
- `nomenclature`: 이름 붙이는 체계
- `denominator`: 이름 붙여 구분하는 수

## 비교 단어

### onym

- 뜻: 이름
- 구분 포인트: [[onym]]은 그리스어 이름 계열. `nomin`은 라틴어 이름 계열.

### voc

- 뜻: 부르다
- 구분 포인트: [[voc]]는 목소리로 부름. `nomin`은 이름을 불러 지명하거나 분류함.
