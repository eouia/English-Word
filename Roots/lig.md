# lig

- 출발 단어: `obligate`
- 핵심 어원: 라틴어 `ligare` = 묶다, 매다
- 핵심 이미지: 끈으로 묶음 / 의무나 약속으로 묶임
- 관련해서 헷갈릴 수 있는 어근: [[strict]] = 조이다, 묶다 계열, [[nect]] = 연결하다 계열

## 출발 단어 분석

`obligate`는 `ob-` + `lig` + `-ate`.

- `ob-`: toward, against, binding force
- `lig`: bind
- `-ate`: 동사형 접미사

따라서 `obligate`는 "묶어 두다"의 뜻에서 "의무를 지우다, 반드시 하게 하다"의 뜻.

## 단어들

### obligate

- 발음: /ˈɑbləˌgeɪt/
- 구성: `ob-` + `lig` + `-ate`
- 뜻: 의무를 지우다, 반드시 하게 하다; 필수적인
- 어원적 연결: 어떤 사람이나 상황을 의무로 묶어 둠
- 사용 맥락: `be obligated to pay` = 지불할 의무가 있다

### obligation

- 발음: /ˌɑbləˈgeɪʃən/
- 구성: `obligate/oblige` + `-ion`
- 뜻: 의무, 책임
- 어원적 연결: 법, 약속, 도덕으로 묶인 상태
- 사용 맥락: `legal obligation` = 법적 의무

### obligatory

- 발음: /əˈblɪgəˌtɔri/
- 구성: `obligate` 계열 + `-ory`
- 뜻: 의무적인, 필수의
- 어원적 연결: 반드시 해야 하도록 묶여 있음
- 사용 맥락: `an obligatory course` = 필수 과목

### oblige

- 발음: /əˈblaɪdʒ/
- 구성: Old French `obliger` < Latin `obligare`
- 뜻: 의무를 지우다; 고맙게도 해 주다
- 어원적 연결: 의무나 부탁으로 상대를 묶음
- 사용 맥락: `I would be obliged if...` = ...해 주시면 감사하겠습니다

### obliged

- 발음: /əˈblaɪdʒd/
- 구성: `oblige` + `-ed`
- 뜻: 어쩔 수 없이 해야 하는, 감사하는
- 어원적 연결: 의무나 호의의 빚에 묶인 상태
- 사용 맥락: `feel obliged to help` = 도와야 한다고 느끼다

### ligament

- 발음: /ˈlɪgəmənt/
- 구성: Latin `ligamentum` = band, tie
- 뜻: 인대
- 어원적 연결: 뼈와 뼈를 묶어 연결하는 조직
- 사용 맥락: `tear a ligament` = 인대를 찢다

### ligature

- 구성: Latin `ligare` 계열
- 뜻: 묶는 것, 결찰; 합자
- 어원적 연결: 묶거나 결합한 것
- 사용 맥락: `a surgical ligature` = 수술용 결찰

## 기억 포인트

`lig` 계열은 "묶다" 이미지.

- `obligate`: 의무로 묶다
- `obligation`: 의무라는 묶임
- `obligatory`: 반드시 해야 하게 묶인
- `oblige`: 의무나 부탁으로 묶다
- `ligament`: 몸을 묶어 연결하는 조직
- `ligature`: 묶는 것, 결합한 글자

## 비교 단어

### strict

- 뜻: 엄격한, 빡빡한
- 구분 포인트: [[strict]]은 조여 단단히 묶는 이미지. `lig`는 끈이나 의무로 묶어 연결하는 이미지

### connect

- 뜻: 연결하다
- 구분 포인트: [[nect]]는 잇고 연결하는 이미지. `lig`는 묶음의 압박과 의무감이 더 강함
