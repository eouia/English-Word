# sent

- 출발 단어: `sense`, `sentence`, `consent`
- 핵심 어원: 라틴어 `sentire` / `sens-` = 느끼다, 알아차리다, 판단하다
- 핵심 이미지: 마음이나 감각으로 느낌. 생각을 판단해 드러냄
- 관련 문서: [[path]], [[aesthes]], [[psych]], [[ment]]
- 관련해서 헷갈릴 수 있는 어근: [[present]]의 `present`는 `prae-` + `esse` = 앞에 있음

## 출발 단어 분석

`sent` 계열은 두 갈래로 기억하면 좋음.

- 감각으로 느낌: `sense`, `sensation`, `sensitive`
- 마음으로 판단함: `sentiment`, `consent`, `dissent`, `sentence`

`sentence`는 원래 "판단, 의견"에서 출발. 법정의 "판결"과 문법의 "문장"으로 갈라짐.

## 단어들

### sense

- 발음: /ˈsɛns/
- 구성: Latin `sensus`
- 뜻: 감각, 의미, 판단력
- 어원적 연결: 몸이나 마음으로 알아차리는 능력
- 사용 맥락: `a sense of humor` = 유머 감각

### sensation

- 발음: /sɛˈnseɪʃən/
- 구성: `sens-` + `-ation`
- 뜻: 감각, 느낌, 큰 반향
- 어원적 연결: 감각으로 강하게 느껴지는 것
- 사용 맥락: `a burning sensation` = 타는 듯한 느낌

### sensitive

- 발음: /ˈsɛnsətɪv/
- 구성: `sens-` + `-itive`
- 뜻: 민감한, 예민한
- 어원적 연결: 작은 자극도 잘 느끼는 상태
- 사용 맥락: `sensitive skin` = 민감한 피부

### sensible

- 발음: /ˈsɛnsəbəl/
- 구성: `sens-` + `-ible`
- 뜻: 분별 있는, 합리적인
- 어원적 연결: 잘 느끼고 판단할 수 있음
- 사용 맥락: `a sensible decision` = 분별 있는 결정

### insensible

- 발음: /ˌɪˈnsɛnsəbəl/
- 구성: `in-` = not + `sensible`
- 뜻: 의식이 없는, 감각이 없는; 알아차리지 못하는
- 어원적 연결: 느끼거나 판단하는 감각이 작동하지 않음
- 사용 맥락: `be insensible to danger` = 위험을 알아차리지 못하다

### insensitivity

- 발음: /ˌɪˌnsɛnsəˈtɪvəti/
- 구성: `in-` = not + `sensitive` + `-ity`
- 뜻: 무감각, 둔감함, 배려 없음
- 어원적 연결: 남의 감정이나 자극을 잘 느끼지 못함
- 사용 맥락: `cultural insensitivity` = 문화적 무감각

### senseless

- 발음: /ˈsɛnsləs/
- 구성: `sense` + `-less`
- 뜻: 무의미한, 분별없는
- 어원적 연결: 의미나 판단력이 없음
- 사용 맥락: `senseless violence` = 무의미한 폭력

### sentiment

- 발음: /ˈsɛntəmənt/
- 구성: Latin `sentire`
- 뜻: 감정, 정서, 의견
- 어원적 연결: 마음으로 느끼고 판단한 생각
- 사용 맥락: `public sentiment` = 대중 정서

### sentimental

- 발음: /ˌsɛntəˈmɛntəl/
- 구성: `sentiment` + `-al`
- 뜻: 감상적인
- 어원적 연결: 감정에 크게 기대어 느끼는 태도
- 사용 맥락: `a sentimental song` = 감상적인 노래

### consent

- 발음: /kəˈnsɛnt/
- 구성: `con-` = together + `sent`
- 뜻: 동의하다, 동의
- 어원적 연결: 함께 같은 생각을 느낌
- 사용 맥락: `give consent` = 동의하다

### consensus

- 발음: /kəˈnsɛnsəs/
- 구성: `con-` = together + `sensus`
- 뜻: 합의, 의견 일치
- 어원적 연결: 여러 사람이 함께 같은 판단을 느낌
- 사용 맥락: `reach a consensus` = 합의에 이르다

### dissent

- 발음: /dɪˈsɛnt/
- 구성: `dis-` = apart + `sent`
- 뜻: 반대하다, 이견
- 어원적 연결: 서로 다른 판단을 느낌
- 사용 맥락: `dissent from the majority` = 다수 의견에 반대하다

### resent

- 발음: /rɪˈzɛnt/
- 구성: `re-` = back, again + `sent`
- 뜻: 분개하다, 억울해하다
- 어원적 연결: 어떤 감정을 되풀이해 강하게 느낌
- 사용 맥락: `resent unfair treatment` = 부당한 대우에 분개하다

### sentence

- 발음: /ˈsɛntəns/
- 구성: Latin `sententia` = opinion, judgment
- 뜻: 문장, 판결, 형을 선고하다
- 어원적 연결: 마음속 판단이나 의견. 완결된 생각은 "문장", 법적 판단은 "판결"
- 사용 맥락: `write a sentence` = 문장을 쓰다

### sentencing

- 발음: /ˈsɛntənsɪŋ/
- 구성: `sentence` + `-ing`
- 뜻: 형 선고
- 어원적 연결: 법정이 판단을 내려 형을 정함
- 사용 맥락: `sentencing guidelines` = 양형 기준

## 기억 포인트

`sent` 계열은 "느끼고 판단함" 이미지.

- `sense`: 감각으로 알아차림
- `sensitive`: 잘 느낌
- `sensible`: 잘 판단함
- `insensible`: 느끼거나 알아차리지 못함
- `sentiment`: 마음으로 느낀 감정이나 의견
- `consent`: 함께 같은 생각을 느낌
- `dissent`: 다르게 느낌
- `sentence`: 판단이나 완결된 생각

## 비교 단어

### present

- 구성: Latin `praeesse` / `praesent-`
- 뜻: 현재의, 참석한, 제시하다
- 구분 포인트: [[present]]는 "앞에 있음". `sent`는 "느끼고 판단함"
