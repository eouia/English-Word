# agon

- 출발 단어: `antagonistic`
- 핵심 어원: Greek `agon` = contest, struggle
- 핵심 이미지: 경기장, 경쟁, 맞섬, 고통스러운 싸움
- 관련 문서: 없음
- 주의: `agony`는 단순한 통증보다 원래 "몸부림치는 싸움, 극심한 고통"의 이미지가 강함.

## 출발 단어 분석

`antagonistic` = `anti-` + `agon` + `-istic`

- `anti-`: against
- `agon`: contest, struggle
- `-istic`: 성질을 나타냄
- 원래 이미지: 상대와 맞서 싸우는 성질
- 현대 의미: 적대적인, 반대하는

## 단어들

### antagonist

- 발음: /æˈntægənəst/
- 구성: Greek `anti-` = against + `agonistes` = competitor
- 뜻: 적대자, 상대역, 길항제
- 어원적 연결: 맞서 싸우는 사람이나 힘
- 사용 맥락: `the antagonist in a novel` = 소설의 적대자

### antagonistic

- 발음: /æˌntægəˈnɪstɪk/
- 구성: `antagonist` + `-ic`
- 뜻: 적대적인, 반대하는
- 어원적 연결: 상대와 맞서 싸우는 태도
- 사용 맥락: `an antagonistic relationship` = 적대적 관계

### antagonism

- 발음: /æˈntægəˌnɪzəm/
- 구성: `antagonist` + `-ism`
- 뜻: 적대감, 대립
- 어원적 연결: 서로 맞서는 상태
- 사용 맥락: `political antagonism` = 정치적 대립

### protagonist

- 발음: /proʊˈtægənəst/
- 구성: Greek `protos` = first + `agonistes` = actor, competitor
- 뜻: 주인공, 주창자
- 어원적 연결: 무대나 싸움에서 첫째 역할을 맡은 사람
- 사용 맥락: `the protagonist of the story` = 이야기의 주인공

### agonize

- 발음: /ˈægəˌnaɪz/
- 구성: Greek `agon` = struggle + `-ize`
- 뜻: 괴로워하다, 고심하다
- 어원적 연결: 마음속에서 싸우듯 고통스러워함
- 사용 맥락: `agonize over a decision` = 결정을 두고 고심하다

### agonizing

- 발음: /ˈægənaɪzɪŋ/
- 구성: `agonize` + `-ing`
- 뜻: 고통스러운, 괴로운
- 어원적 연결: 싸움 같은 고통을 주는
- 사용 맥락: `an agonizing wait` = 괴로운 기다림

### agony

- 발음: /ˈægəni/
- 구성: Greek `agonia` = struggle, anguish
- 뜻: 극심한 고통, 고뇌
- 어원적 연결: 몸부림치는 싸움 같은 고통
- 사용 맥락: `in agony` = 극심한 고통 속에

## 기억 포인트

`agon` 계열은 "경쟁, 맞섬, 싸움 같은 고통" 이미지.

- `antagonist`: 맞서 싸우는 사람
- `antagonistic`: 적대적인
- `antagonism`: 대립, 적대감
- `protagonist`: 첫째 역할의 사람 → 주인공
- `agonize`: 고통스럽게 고심하다
- `agony`: 싸움 같은 극심한 고통

## 비교 단어

### anti

- 뜻: 반대하여, 맞서서
- 구분 포인트: `anti-`는 반대 방향의 접두사. `agon`은 경쟁·싸움 자체의 장면.

### athlete

- 뜻: 운동선수
- 구분 포인트: Greek 경기·경쟁 문화와 의미 영역은 가깝지만, `athlete`는 `athlon` = contest 계열로 따로 볼 수 있음.
