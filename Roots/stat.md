# stat

- 출발 단어: `statement`
- 핵심 어원: Latin `stare` / `status` = stand, state, condition
- 핵심 이미지: 세워진 상태, 공식적으로 세워 둔 말이나 기준
- 관련 어근: [[stand]], [[sist]], [[pos]], [[obsta]]

## 출발 단어 분석

`statement`는 `state` + `-ment`.

`state`는 Latin `status` = a standing, condition에서 온 말. 핵심은 "서 있는 상태"이고, 여기서 "상태, 국가, 공식적으로 말하다"로 확장됨.

따라서 `statement`는 "공식적으로 세워 둔 말" → "진술, 성명, 명세서"의 뜻.

## 단어들

### state

- 구성: Latin `status` = standing, condition
- 의미: 상태, 국가, 진술하다
- 어원적 연결: 어떤 방식으로 서 있는 상태
- 사용 맥락: `state the facts` = 사실을 진술하다

### statement

- 구성: `state` + `-ment`
- 의미: 진술, 성명, 명세서
- 어원적 연결: 말로 공식적으로 세워 둔 내용
- 사용 맥락: `make a statement` = 진술하다

### status

- 구성: Latin `status`
- 의미: 지위, 상태
- 어원적 연결: 사회나 상황 안에서 서 있는 자리
- 사용 맥락: `social status` = 사회적 지위

### statute

- 구성: Latin `statuere` = set up, establish
- 의미: 법령, 성문법
- 어원적 연결: 공식적으로 세워 둔 규칙
- 사용 맥락: `a federal statute` = 연방법

### statue

- 구성: Latin `statua` < `statuere`
- 의미: 조각상, 입상
- 어원적 연결: 세워 놓은 형상
- 사용 맥락: `a bronze statue` = 청동상

### station

- 구성: Latin `statio` = standing place
- 의미: 역, 위치, 배치하다
- 어원적 연결: 서 있는 자리, 정해진 위치
- 사용 맥락: `a train station` = 기차역

### stationary

- 구성: `station` + `-ary`
- 의미: 움직이지 않는, 고정된
- 어원적 연결: 제자리에 서 있음
- 사용 맥락: `a stationary object` = 정지한 물체

### stationery

- 구성: 역사적으로 `stationer` = 정해진 자리에 서서 파는 상인
- 의미: 문구류
- 어원적 연결: 이동 상인이 아니라 고정된 가게에서 팔던 물건
- 사용 맥락: `office stationery` = 사무용 문구류

### stable

- 구성: Latin `stabilis` < `stare`
- 의미: 안정된, 마구간
- 어원적 연결: 단단히 서 있어 흔들리지 않음
- 사용 맥락: `a stable economy` = 안정된 경제

### establish

- 구성: Old French `establiss-` < Latin `stabilis`
- 의미: 설립하다, 확립하다
- 어원적 연결: 안정적으로 서게 만듦
- 사용 맥락: `establish a company` = 회사를 설립하다

### established

- 구성: `establish` + `-ed`
- 의미: 확립된, 인정받은
- 어원적 연결: 이미 안정적으로 세워진 상태
- 사용 맥락: `an established theory` = 확립된 이론

### establishment

- 구성: `establish` + `-ment`
- 의미: 설립, 기관, 기성 체제
- 어원적 연결: 세워진 조직이나 제도
- 사용 맥락: `the political establishment` = 정치 기성 체제

### stability

- 구성: `stable` + `-ity`
- 의미: 안정성
- 어원적 연결: 흔들리지 않고 서 있는 성질
- 사용 맥락: `economic stability` = 경제 안정성

### stabilize

- 구성: `stable` + `-ize`
- 의미: 안정시키다, 안정되다
- 어원적 연결: 흔들리지 않고 서 있는 상태로 만들거나 그렇게 됨
- 사용 맥락: `stabilize prices` = 물가를 안정시키다

### stabilization

- 구성: `stabilize` + `-ation`
- 의미: 안정화
- 어원적 연결: 안정된 상태로 세우는 과정
- 사용 맥락: `economic stabilization` = 경제 안정화

### unstable

- 구성: `un-` = not + `stable`
- 의미: 불안정한
- 어원적 연결: 단단히 서 있지 못하고 흔들림
- 사용 맥락: `an unstable situation` = 불안정한 상황

### destabilize

- 구성: `de-` = reverse, remove + `stabilize`
- 의미: 불안정하게 만들다
- 어원적 연결: 서 있던 안정 상태를 무너뜨림
- 사용 맥락: `destabilize the region` = 지역을 불안정하게 만들다

### statistics

- 구성: German `Statistik` < state affairs
- 의미: 통계, 통계학
- 어원적 연결: 원래 국가의 상태를 수치로 파악하는 학문
- 사용 맥락: `official statistics` = 공식 통계

### estate

- 구성: Old French `estat` < Latin `status`
- 의미: 재산, 토지, 계층
- 어원적 연결: 사회적 상태나 지위에서 재산, 토지 보유 상태로 확장
- 사용 맥락: `real estate` = 부동산

## 기억 포인트

- `state`: 서 있는 상태, 공식적으로 말함
- `statement`: 세워 둔 말
- `status`: 사회 안에서 서 있는 자리
- `statute`: 세워 둔 법
- `station`: 서 있는 자리
- `stable`: 흔들리지 않고 섬
- `stabilize`: 흔들리지 않게 세움
- `statistics`: 국가/상태를 수치로 파악함

## 비교 단어

- [[stand]]
  - 핵심: 영어 기본어 "서다"
  - 구분 포인트: `stand`는 일상어와 합성어 중심. `stat`은 라틴계로 `state`, `status`, `statute`, `station`에 나타남

- [[sist]]
  - 핵심: 라틴어 `sistere` = 서다, 서게 하다
  - 구분 포인트: `sist`는 `exist`, `consist`, `resist`처럼 접두사와 결합해 "서 있음/버팀"이 드러남. `stat`은 "세워진 상태, 공식적 위치" 쪽

- [[stanc]]
  - 핵심: 라틴어 `stans/stant-` = 서 있는
  - 구분 포인트: `instance`, `instant`, `stance`, `substance`처럼 구체적 경우·자세·실체 쪽으로 나타남

- [[pos]]
  - 핵심: 놓다, 두다
  - 구분 포인트: `position`은 놓인 자리. `status`는 그 자리에 서 있는 상태나 지위
