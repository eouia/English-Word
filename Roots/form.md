# form

- 출발 단어: `transform`
- 핵심 어원: 라틴어 `forma` = 모양, 형태
- 핵심 이미지: 모양을 갖추거나 바꿈
- 관련해서 헷갈릴 수 있는 어근: [[fac]] = 만들다, 하다 계열, [[vers]] = 돌리다, 바꾸다 계열

## 출발 단어 분석

`transform`은 `trans-` + `form`.

- `trans-`: across, beyond
- `form`: shape

따라서 `transform`은 "형태를 넘어 바꾸다" → "변형하다, 완전히 바꾸다"의 뜻.

## 단어들

### form (noun)

- 구성: Latin `forma`
- 뜻: 형태, 양식, 형성하다
- 어원적 연결: 모양을 갖춘 것
- 사용 맥락: `fill out a form` = 양식을 작성하다

### formal

- 발음: /ˈfɔrməl/
- 구성: `form` + `-al`
- 뜻: 공식적인, 격식을 차린
- 어원적 연결: 정해진 형식을 갖춤
- 사용 맥락: `formal language` = 격식 있는 언어

### informal

- 발음: /ˌɪˈnfɔrməl/
- 구성: [[in]] = not + `formal`
- 뜻: 비공식적인, 격식 없는
- 어원적 연결: 정해진 형식이 덜함
- 사용 맥락: `informal conversation` = 비공식적 대화

### formality

- 발음: /fɔˈrmæləti/
- 구성: `formal` + `-ity`
- 뜻: 형식, 격식
- 어원적 연결: 일정한 형식을 갖춘 성질
- 사용 맥락: `legal formalities` = 법적 절차 형식

### form (verb)

- 구성: `form`
- 뜻: 형성하다, 만들다
- 어원적 연결: 모양을 갖게 함
- 사용 맥락: `form a habit` = 습관을 형성하다

### formation

- 발음: /fɔˈrmeɪʃən/
- 구성: `form` + `-ation`
- 뜻: 형성, 구성
- 어원적 연결: 형태가 만들어지는 과정
- 사용 맥락: `cloud formation` = 구름 형성

### reform

- 발음: /rəˈfɔrm/
- 구성: `re-` = again + `form`
- 뜻: 개혁하다, 개선하다
- 어원적 연결: 다시 형태를 잡음
- 사용 맥락: `reform the system` = 제도를 개혁하다

### transform

- 발음: /træˈnsfɔrm/
- 구성: `trans-` = across, beyond + `form`
- 뜻: 변형하다, 완전히 바꾸다
- 어원적 연결: 형태를 넘겨 다른 모습으로 바꿈
- 사용 맥락: `transform society` = 사회를 변화시키다

### transformation

- 발음: /ˌtrænsfɚˈmeɪʃən/
- 구성: `transform` + `-ation`
- 뜻: 변화, 변형
- 어원적 연결: 형태가 바뀌는 과정
- 사용 맥락: `digital transformation` = 디지털 전환

### uniform

- 발음: /ˈjunəˌfɔrm/
- 구성: [[uni]] = one + `form`
- 뜻: 똑같은, 제복
- 어원적 연결: 하나의 형태를 가짐
- 사용 맥락: `uniform standards` = 통일된 기준

### conform

- 발음: /kəˈnfɔrm/
- 구성: [[con]] = together + `form`
- 뜻: 따르다, 순응하다
- 어원적 연결: 함께 같은 형태를 이룸
- 사용 맥락: `conform to rules` = 규칙에 따르다

### perform

- 발음: /pɚˈfɔrm/
- 구성: `per-` = through, completely + `form`
- 뜻: 수행하다, 공연하다
- 어원적 연결: 끝까지 형태를 이루어 실행함
- 사용 맥락: `perform a task` = 일을 수행하다

### inform

- 발음: /ˌɪˈnfɔrm/
- 구성: [[in]] = into + `form`
- 뜻: 알리다
- 어원적 연결: 마음 안에 형태를 만들어 줌. 지식을 주입해 알게 함
- 사용 맥락: `inform the public` = 대중에게 알리다

### information

- 발음: /ˌɪnfɚˈmeɪʃən/
- 구성: `inform` + `-ation`
- 뜻: 정보
- 어원적 연결: 머릿속에 형태를 만들어 주는 내용
- 사용 맥락: `collect information` = 정보를 수집하다

## 기억 포인트

`form` 계열은 "모양 / 형식 / 형성" 이미지.

- `form`: 형태, 형성하다
- `formal`: 형식을 갖춘
- `reform`: 다시 형태를 잡음
- `transform`: 형태를 바꿈
- `uniform`: 하나의 형태
- `conform`: 함께 같은 형태를 이룸
- `perform`: 끝까지 형태로 실행함
- `inform`: 마음에 형태를 만들어 알림

## 비교 단어

- 뜻: 만들다, 하다
- 구분 포인트: [[fac]]는 만드는 행위와 결과. `form`은 만들어진 모양과 형식에 초점

### vers

- 구분 포인트: [[vers]]의 `convert`는 방향이나 상태 전환. `transform`은 형태 변화에 초점
