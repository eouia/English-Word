# tric

- 출발 단어: `intricate`
- 핵심 어원: 라틴어 `tricae` = 얽힌 것, 성가신 일, 복잡한 문제
- 핵심 이미지: 안으로 얽혀 복잡함 / 얽힌 데서 빼냄
- 관련해서 헷갈릴 수 있는 어근: [[plic]] = 접혀 얽힘 계열, [[plex]] = 얽히다 계열

## 출발 단어 분석

`intricate`는 `in-` + `tric` + `-ate`.

- `in-`: in, into
- `tric`: entanglement, perplexity
- `-ate`: 형용사형·동사형 접미사

따라서 `intricate`는 원래 "안으로 얽힌"의 뜻. 여기서 구조나 설명이 세밀하게 얽혀 "복잡한, 정교한"의 뜻.

## 단어들

### intricate

- 구성: `in-` = into + `tric` + `-ate`
- 뜻: 복잡한, 정교한
- 어원적 연결: 여러 요소가 안쪽으로 얽혀 있음
- 사용 맥락: `an intricate pattern` = 정교하게 얽힌 무늬

### intricacy

- 구성: `intricate` + `-cy`
- 뜻: 복잡함, 정교함, 복잡한 세부 사항
- 어원적 연결: 얽힌 구조의 복잡한 정도
- 사용 맥락: `the intricacies of law` = 법의 복잡한 세부 사항

### intrigue

- 구성: French `intriguer` ← Latin `intricare` = entangle
- 뜻: 흥미를 끌다; 음모, 책략
- 어원적 연결: 사람의 마음이나 사건을 얽히게 만들어 궁금하게 함. 정치적·비밀스러운 얽힘에서는 음모의 뜻
- 사용 맥락: `The mystery intrigued me.` = 그 미스터리가 내 흥미를 끌었다

### intriguing

- 구성: `intrigue` + `-ing`
- 뜻: 흥미로운, 호기심을 자극하는
- 어원적 연결: 마음을 얽어 붙잡아 더 알고 싶게 함
- 사용 맥락: `an intriguing question` = 흥미로운 질문

### intrigued

- 구성: `intrigue` + `-ed`
- 뜻: 흥미를 느끼는, 호기심이 생긴
- 어원적 연결: 마음이 어떤 대상에 얽혀 끌림
- 사용 맥락: `I was intrigued by the idea.` = 그 아이디어에 흥미가 생겼다

### extricate

- 구성: [[ex]] = out + `tric` + `-ate`
- 뜻: 빼내다, 구해 내다, 벗어나게 하다
- 어원적 연결: 얽힌 상태에서 밖으로 풀어냄
- 사용 맥락: `extricate someone from danger` = 누군가를 위험에서 빼내다

### extrication

- 구성: `extricate` + `-ion`
- 뜻: 구출, 탈출, 빼냄
- 어원적 연결: 얽힘에서 밖으로 풀려나는 과정
- 사용 맥락: `vehicle extrication` = 차량 구조

### inextricable

- 구성: `in-` = not + `extricable`
- 뜻: 떼어 낼 수 없는, 풀 수 없이 얽힌
- 어원적 연결: 얽힌 상태에서 밖으로 빼낼 수 없음
- 사용 맥락: `inextricably linked` = 떼려야 뗄 수 없이 연결된

## 기억 포인트

`tric` 계열은 "얽힘" 이미지.

- `intricate`: 안으로 얽힘 → 복잡한, 정교한
- `intricacy`: 얽힌 복잡함
- `intrigue`: 마음이나 사건을 얽음 → 흥미를 끌다, 음모
- `intriguing`: 마음을 얽어 붙잡는 → 흥미로운
- `extricate`: 얽힘에서 밖으로 빼냄
- `inextricable`: 빼낼 수 없을 만큼 얽힌

## 비교 단어

### complicated

- 뜻: 복잡한
- 구분 포인트: [[plic]] 계열. 여러 겹으로 접혀 얽힌 이미지. `intricate`는 세부 요소가 촘촘히 얽혀 정교한 느낌이 강함

### complex

- 뜻: 복잡한, 복합체
- 구분 포인트: 함께 얽힌 구조 전체를 가리키는 말. `intricate`는 미세한 세부 구조가 얽힌 느낌이 더 강함
