# grav

- 출발 단어: `gravity`
- 핵심 어원: Latin `gravis` = 무거운, 심각한
- 동사형: Latin `gravare` = 무겁게 하다, 부담을 주다
- 핵심 이미지: 무게가 있어 아래로 누르고 중요하게 느껴짐
- 관련 문서: [[mag]], [[press]]
- 주의: `grave` = 무덤은 Germanic 계열로 별도. `grave` = 심각한은 Latin `gravis` 계열.

## 출발 단어 분석

`gravity` ← Latin `gravitas`

- `gravis`: 무거운, 중대한
- `gravitas`: 무게, 진지함, 품위
- 원래 이미지: 실제 무게와 비유적 무게
- 현대 의미: 중력, 심각성, 엄숙함

## 단어들

### gravity

- 발음: /ˈgrævəti/
- 구성: Latin `gravitas`
- 뜻: 중력, 심각성, 엄숙함
- 어원적 연결: 무게가 아래로 끌고, 문제를 무겁게 만듦
- 사용 맥락: `the gravity of the situation` = 상황의 심각성

### grave

- 발음: /ˈgreɪv/
- 구성: Latin `gravis`
- 뜻: 심각한, 중대한, 엄숙한
- 어원적 연결: 무겁고 중요하게 느껴짐
- 사용 맥락: `a grave mistake` = 중대한 실수

### gravely

- 발음: /ˈgreɪvli/
- 구성: `grave` + `-ly`
- 뜻: 심각하게, 엄숙하게
- 어원적 연결: 무거운 태도로
- 사용 맥락: `gravely ill` = 위독한

### gravitas

- 발음: /ˈgrævətəs/
- 구성: Latin `gravitas`
- 뜻: 위엄, 중후함
- 어원적 연결: 사람에게 느껴지는 무게와 품격
- 사용 맥락: `speak with gravitas` = 중후하게 말하다

### gravitate

- 발음: /ˈgrævɪˌteɪt/
- 구성: `gravity` + `-ate`
- 뜻: 끌리다, 중력에 의해 움직이다
- 어원적 연결: 무게 중심 쪽으로 끌려감
- 사용 맥락: `gravitate toward cities` = 도시로 끌리다

### gravitation

- 발음: /ˌgrævɪˈteɪʃən/
- 구성: `gravitate` + `-ion`
- 뜻: 중력 작용, 끌림
- 어원적 연결: 무게가 서로 끌어당기는 작용
- 사용 맥락: `universal gravitation` = 만유인력

### gravitational

- 발음: /ˌgrævɪˈteɪʃənəl/
- 구성: `gravitation` + `-al`
- 뜻: 중력의
- 어원적 연결: 무게와 끌림에 관련됨
- 사용 맥락: `gravitational force` = 중력

### aggravate

- 발음: /ˈægrəˌveɪt/
- 구성: `ad-` = to + `gravare` = make heavy
- 뜻: 악화시키다, 더 짜증나게 하다
- 어원적 연결: 부담이나 문제를 더 무겁게 만듦
- 사용 맥락: `aggravate the problem` = 문제를 악화시키다

### aggravation

- 발음: /ˌægrəˈveɪʃən/
- 구성: `aggravate` + `-ion`
- 뜻: 악화, 짜증
- 어원적 연결: 더 무거워지고 힘들어짐
- 사용 맥락: `avoid further aggravation` = 추가 악화를 피하다

### grieve

- 발음: /ˈgriv/
- 구성: Old French `grever` ← Latin `gravare`
- 뜻: 슬퍼하다, 괴롭히다
- 어원적 연결: 마음에 무거운 부담을 줌
- 사용 맥락: `grieve for a friend` = 친구를 애도하다

### grief

- 발음: /ˈgrif/
- 구성: Old French `gref/grief` ← Latin `gravis`
- 뜻: 깊은 슬픔
- 어원적 연결: 마음을 무겁게 누르는 슬픔
- 사용 맥락: `deal with grief` = 슬픔을 감당하다

### grievance

- 발음: /ˈgrivəns/
- 구성: `grieve` 계열 + `-ance`
- 뜻: 불만, 고충
- 어원적 연결: 마음에 무겁게 남은 억울함
- 사용 맥락: `file a grievance` = 고충을 제기하다

## 기억 포인트

`grav` 계열은 "무거움" 이미지.

- `gravity`: 물리적 무게와 중력
- `grave`: 심각하고 중대한
- `gravitas`: 무게 있는 품격
- `aggravate`: 더 무겁게 만들어 악화시킴
- `grief`: 마음을 누르는 슬픔

## 비교 단어

### grave = 무덤

- 뜻: 무덤
- 구분 포인트: Old English `graef` 계열. Latin `gravis`의 `grave` = 심각한 과 어원 다름.
