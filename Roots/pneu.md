# pneu

- 출발 단어: `pneumonia`
- 핵심 어원: Greek `pneuma` = 숨, 바람, 공기, 영
- 핵심 이미지: 몸 안팎을 드나드는 숨과 공기
- 관련 문서: [[spir]], [[anim]], [[psych]], [[hal]], [[flat]]
- 주의: `pneumonia`처럼 `p`가 묵음인 경우가 많음.

## 출발 단어 분석

`pneumonia` ← Greek `pneumon` = lung

- `pneuma`: 숨, 바람, 공기
- `pneumon`: 폐
- 원래 이미지: 숨이 드나드는 기관
- 현대 의미: 폐렴

## 단어들

### pneumonia

- 구성: Greek `pneumon` = lung + `-ia`
- 뜻: 폐렴
- 어원적 연결: 숨을 담당하는 폐에 생긴 병
- 사용 맥락: `recover from pneumonia` = 폐렴에서 회복하다

### pneumatic

- 구성: Greek `pneuma` = air + `-atic`
- 뜻: 공기의, 압축 공기로 작동하는
- 어원적 연결: 공기나 바람의 힘으로 움직임
- 사용 맥락: `a pneumatic drill` = 공기압 드릴

### pneumatics

- 구성: `pneumatic` + `-s`
- 뜻: 공기압 기술, 공기역학의 한 분야
- 어원적 연결: 공기의 압력과 움직임을 다룸
- 사용 맥락: `hydraulics and pneumatics` = 유압과 공압

### apnea

- 구성: `a-` = without + Greek `pnoia` = breathing
- 뜻: 무호흡
- 어원적 연결: 숨이 없는 상태
- 사용 맥락: `sleep apnea` = 수면 무호흡증

### dyspnea

- 구성: `dys-` = bad, difficult + Greek `pnoia` = breathing
- 뜻: 호흡 곤란
- 어원적 연결: 숨쉬기가 어렵거나 나쁜 상태
- 사용 맥락: `shortness of breath and dyspnea` = 숨참과 호흡 곤란

### eupnea

- 구성: `eu-` = good, normal + Greek `pnoia` = breathing
- 뜻: 정상 호흡
- 어원적 연결: 숨이 정상적으로 잘 이어지는 상태
- 사용 맥락: 의학 맥락에서 정상 호흡 상태를 가리킴

## 기억 포인트

`pneu` 계열은 "숨과 공기" 이미지.

- `pneumonia`: 숨쉬는 기관인 폐의 병
- `pneumatic`: 공기의 힘으로 작동함
- `apnea`: 숨이 없음
- `dyspnea`: 숨쉬기 어려움
- `eupnea`: 숨이 정상적임

## 비교 단어

### spir

- 뜻: 숨쉬다, 숨
- 구분 포인트: [[spir]]는 Latin `spirare` = breathe 계열. `inspire`, `expire`, `respire`처럼 숨의 방향과 움직임에 초점.

### anim

- 뜻: 숨, 생명, 마음
- 구분 포인트: [[anim]]은 Latin `anima` = breath, life, soul 계열. 숨에서 생명과 마음의 이미지로 확장.
