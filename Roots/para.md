# para

- 출발 단어: `para-`
- 핵심 의미: 옆에, 곁에; 넘어서; 반대로, 비정상적으로
- 관련 형태: Greek `para` = beside, alongside, beyond, contrary to
- 관련 문서: [[dox]], [[med]], [[lys]], [[phras]], [[thes]]

## 핵심 이미지

`para-`의 바탕은 "바로 옆에 있음".

옆에 나란히 있으면 `parallel`.
의료인 옆에서 돕는 사람은 `paramedic`.
본문 옆에서 바꾸어 말하면 `paraphrase`.
통념 옆으로 벗어나 반대처럼 보이면 `paradox`.
정상 상태에서 벗어나면 `paralysis`.

## 단어들

### para

- 구성: Greek `para` = beside, alongside, beyond
- 뜻: 옆에, 곁에; 넘어서; 반대로
- 연결: 기준 바로 옆, 또는 기준에서 벗어난 위치
- 사용 맥락: Greek 계열 학술어와 전문어에서 자주 보임.

### parallel

- 구성: `para-` = beside + Greek `allelos` = one another
- 뜻: 평행한, 유사한; 평행선
- 연결: 서로 옆에 나란히 놓여 같은 방향으로 감
- 사용 맥락: `parallel lines` = 평행선

### paragraph

- 구성: Greek `para-` = beside + `graphein` = write
- 뜻: 문단
- 연결: 원래는 글 옆에 표시를 해 새 단락을 알린 것
- 사용 맥락: `write a paragraph` = 문단을 쓰다

### paraphrase

- 구성: `para-` = beside + `phrase`
- 뜻: 바꾸어 말하다, 의역하다
- 연결: 원문 옆에 놓을 수 있는 다른 표현
- 사용 맥락: `paraphrase a sentence` = 문장을 바꾸어 말하다

### paradox

- 구성: `para-` = contrary to + `doxa` = opinion
- 뜻: 역설
- 연결: 일반 의견 옆으로 벗어나 반대처럼 보이나 생각할 가치가 있는 말
- 사용 맥락: `a paradox of freedom` = 자유의 역설

### paramedic

- 구성: `para-` = alongside + `medic`
- 뜻: 응급구조사
- 연결: 의사나 의료 체계 곁에서 응급 처치를 돕는 사람
- 사용 맥락: `call the paramedics` = 응급구조사를 부르다

### parasite

- 구성: Greek `para-` = beside + `sitos` = food
- 뜻: 기생 생물, 기생충; 기생하는 사람
- 연결: 남의 식탁 옆에 붙어 먹는 이미지에서 출발
- 사용 맥락: `intestinal parasites` = 장내 기생충

### paralysis

- 구성: `para-` = beside/beyond + `lysis` = loosening
- 뜻: 마비
- 연결: 힘이 정상적으로 풀려 움직이지 못하는 상태
- 사용 맥락: `temporary paralysis` = 일시적 마비

### parenthesis

- 구성: `para-` = beside + `en-` = in + `thesis` = placing
- 뜻: 괄호, 삽입구
- 연결: 본문 옆이나 안에 덧붙여 놓은 말
- 사용 맥락: `in parentheses` = 괄호 안에

### paradigm

- 구성: Greek `para-` = beside + `deiknynai` = show
- 뜻: 전형, 모형, 사고의 틀
- 연결: 옆에 놓고 보여 주는 예시나 모델
- 사용 맥락: `a paradigm shift` = 패러다임 전환

## 비교 단어

`paradox`는 [[dox]] 문서에도 있음.
그쪽은 `doxa` = 의견 중심, 이 문서는 `para-` = 옆·반대 접두사 중심.

`paramedic`은 [[med]], `paralysis`는 [[lys]], `paraphrase`는 [[phras]], `parenthesis`는 [[thes]] 문서와 연결됨.
각 문서에서는 뒤쪽 어근을 중심으로 보고, 여기서는 `para-`가 만드는 방향 감각을 중심으로 봄.
