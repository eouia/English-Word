# neutr

- 출발 단어: `neutrality`
- 핵심 어원: 라틴어 `neuter` = neither one nor the other
- 핵심 이미지: 양쪽 어느 편도 아님 / 어느 성질도 띠지 않음
- 관련해서 헷갈릴 수 있는 어근: [[equ]] = 균형과 같음 계열, [[medi]] = 중간 계열

## 출발 단어 분석

`neutrality`는 `neutral` + `-ity`.

- `neutral`: 어느 편도 들지 않는
- `-ity`: 상태, 성질

따라서 `neutrality`는 "어느 편도 들지 않는 상태" → "중립, 중립성"의 뜻.

## 단어들

### neutrality

- 구성: `neutral` + `-ity`
- 뜻: 중립, 중립성
- 어원적 연결: 양쪽 어느 편도 아닌 상태
- 사용 맥락: `political neutrality` = 정치적 중립

### neutral

- 구성: Latin `neuter` = neither
- 뜻: 중립의, 중성의, 무색무취의
- 어원적 연결: 어느 한쪽 성질이나 편에 속하지 않음
- 사용 맥락: `a neutral position` = 중립적 입장

### neutralize

- 구성: `neutral` + `-ize`
- 뜻: 무력화하다, 중화하다
- 어원적 연결: 한쪽 성질이나 효과를 없애 중립 상태로 만듦
- 사용 맥락: `neutralize an acid` = 산을 중화하다

### neutralization

- 구성: `neutralize` + `-ation`
- 뜻: 중화, 무력화
- 어원적 연결: 강한 작용을 어느 쪽도 아닌 상태로 만드는 과정
- 사용 맥락: `chemical neutralization` = 화학적 중화

### neuter

- 구성: Latin `neuter` = neither
- 뜻: 중성의; 중성 명사; 거세하다
- 어원적 연결: 남성도 여성도 아닌 성질
- 사용 맥락: `a neuter noun` = 중성 명사

## 기억 포인트

`neutr` 계열은 "둘 중 어느 쪽도 아님" 이미지.

- `neutral`: 어느 편도 아님
- `neutrality`: 어느 편도 들지 않는 상태
- `neutralize`: 한쪽 효과를 없애 중립으로 만듦
- `neuter`: 남성도 여성도 아닌 중성

## 비교 단어

### equal

- 뜻: 같은, 평등한
- 구분 포인트: [[equ]]는 같음과 균형. `neutral`은 어느 쪽에도 속하지 않는 상태

### medium

- 뜻: 중간, 매체
- 구분 포인트: [[medi]]는 두 끝 사이의 가운데. `neutral`은 가운데라기보다 어느 편도 들지 않는 상태
