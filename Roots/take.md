# take

- 출발 단어: `unmistakable`
- 핵심 어원: Old English `tacan` = take, grasp
- 핵심 이미지: 잡다, 받아들이다, 어떤 것으로 여기다
- 관련 문서: [[cap]], [[prehend]]
- 주의: `mistake`는 `mis-` + `take`. 단순한 "실수"만이 아니라, 원래는 잘못 잡거나 잘못 이해하는 이미지.

## 출발 단어 분석

`unmistakable` = `un-` + `mistakable`

- `un-`: not
- `mistake`: 잘못 잡다, 잘못 이해하다; 실수
- `-able`: ~할 수 있는
- `unmistakable`: 잘못 알아볼 수 없는, 틀림없는, 명백한

`mistake`는 "잘못 잡다"에서 "잘못 이해하다" → "실수하다"로 발전. `unmistakable`은 잘못 잡거나 오해할 여지가 없을 만큼 뚜렷함.

## 단어들

### take

- 구성: Old English `tacan` = take, grasp
- 뜻: 잡다, 가져가다, 받아들이다
- 어원적 연결: 손이나 마음으로 어떤 것을 잡아 취함
- 사용 맥락: `take a seat` = 자리에 앉다, `take advice` = 조언을 받아들이다

### mistake

- 구성: `mis-` = wrongly + `take`
- 뜻: 실수, 오해; 잘못 알다
- 어원적 연결: 어떤 것을 잘못 잡거나 잘못 받아들임
- 사용 맥락: `make a mistake` = 실수하다, `mistake A for B` = A를 B로 착각하다

### mistaken

- 구성: `mistake`의 과거분사
- 뜻: 잘못된, 오해한
- 어원적 연결: 잘못 잡아 판단한 상태
- 사용 맥락: `a mistaken belief` = 잘못된 믿음

### mistakenly

- 구성: `mistaken` + `-ly`
- 뜻: 잘못하여, 착각하여
- 어원적 연결: 대상을 잘못 잡아 이해한 방식
- 사용 맥락: `mistakenly assume` = 잘못 가정하다

### mistakable

- 구성: `mistake` + `-able`
- 뜻: 오해할 수 있는, 착각할 수 있는
- 어원적 연결: 다른 것으로 잘못 잡을 가능성이 있음
- 사용 맥락: 드문 표현. 보통 `unmistakable`의 내부 구조를 이해할 때 유용.

### unmistakable

- 구성: `un-` = not + `mistakable`
- 뜻: 틀림없는, 명백한, 알아보기 쉬운
- 어원적 연결: 다른 것으로 잘못 잡을 수 없을 만큼 뚜렷함
- 사용 맥락: `an unmistakable sign` = 명백한 징후, `an unmistakable voice` = 확실히 알아볼 수 있는 목소리

### unmistakably

- 구성: `unmistakable` + `-ly`
- 뜻: 틀림없이, 명백히
- 어원적 연결: 오해할 여지 없이 드러나는 방식
- 사용 맥락: `unmistakably different` = 틀림없이 다른

### take for

- 구성: `take` + `for`
- 뜻: ~로 여기다, 착각하다
- 어원적 연결: 어떤 대상을 특정 의미로 잡아 받아들임
- 사용 맥락: `take someone for a fool` = 누군가를 바보로 여기다

## 기억 포인트

`take` 계열은 "잡아 받아들임" 이미지.

- `take`: 잡다, 받아들이다
- `mistake`: 잘못 잡음, 실수
- `mistaken`: 잘못 판단한
- `mistakenly`: 착각하여
- `unmistakable`: 잘못 알아볼 수 없는, 명백한
- `unmistakably`: 틀림없이

## 비교 단어

### capture

- 뜻: 붙잡다, 포획하다
- 구분 포인트: [[cap]]은 Latin `capere` = 잡다 계열. `take`는 Germanic 기본어로 더 일상적이고 넓음.

### prehend

- 뜻: 붙잡다, 이해하다
- 구분 포인트: [[prehend]]는 붙잡아 이해하는 Latin 계열. `mistake`는 잘못 잡아 오해하는 일상적 표현.

### obvious

- 뜻: 명백한, 뚜렷한
- 구분 포인트: `obvious`는 눈앞에 놓여 바로 보임. `unmistakable`은 다른 것과 혼동할 수 없을 만큼 뚜렷함.

### undeniable

- 뜻: 부인할 수 없는
- 구분 포인트: [[deny]]의 `undeniable`은 아니라고 말할 수 없을 만큼 명백함. `unmistakable`은 잘못 알아보거나 착각할 수 없을 만큼 명백함.
