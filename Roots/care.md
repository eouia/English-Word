# care

- 출발 단어: `carelessly`
- 핵심 어원: Old English `caru` / `carian` = sorrow, anxiety, be concerned
- 핵심 이미지: 마음을 쓰고 주의를 기울임
- 관련 문서: [[cura]]
- 주의: `care`는 English/Germanic 계열. Latin `cura`와 뜻은 비슷하지만 어원은 다름.

## 출발 단어 분석

`carelessly` = `care` + `-less` + `-ly`

- `care`: concern, attention
- `-less`: without
- `-ly`: 부사 어미
- 원래 이미지: 주의나 마음 씀이 없는 방식
- 현대 의미: 부주의하게, 경솔하게

## 단어들

### care

- 발음: /ˈkɛr/
- 구성: Old English `caru` / `carian` = sorrow, concern
- 뜻: 돌봄, 관심, 주의; 돌보다, 신경 쓰다
- 어원적 연결: 걱정하거나 마음을 써서 살핌
- 사용 맥락: `take care of someone` = 누군가를 돌보다

### careful

- 발음: /ˈkɛrfəl/
- 구성: `care` + `-ful`
- 뜻: 조심하는, 주의 깊은
- 어원적 연결: 주의와 마음 씀이 가득함
- 사용 맥락: `be careful with glass` = 유리를 조심히 다루다

### carefully

- 발음: /ˈkɛrfəli/
- 구성: `careful` + `-ly`
- 뜻: 조심스럽게, 주의 깊게
- 어원적 연결: 마음을 써서 실수하지 않도록 함
- 사용 맥락: `read carefully` = 주의 깊게 읽다

### carefulness

- 발음: /ˈkɛrfəˌlnɛs/
- 구성: `careful` + `-ness`
- 뜻: 조심성, 신중함
- 어원적 연결: 주의 깊은 성질
- 사용 맥락: `carefulness in planning` = 계획에서의 신중함

### careless

- 발음: /ˈkɛrləs/
- 구성: `care` + `-less`
- 뜻: 부주의한, 경솔한
- 어원적 연결: 주의나 마음 씀이 없음
- 사용 맥락: `a careless mistake` = 부주의한 실수

### carelessly

- 발음: /ˈkɛrləsli/
- 구성: `careless` + `-ly`
- 뜻: 부주의하게, 경솔하게
- 어원적 연결: 주의를 기울이지 않은 방식
- 사용 맥락: `drive carelessly` = 부주의하게 운전하다

### carelessness

- 발음: /ˈkɛrləsnəs/
- 구성: `careless` + `-ness`
- 뜻: 부주의, 경솔함
- 어원적 연결: 마음을 쓰지 않아 생기는 태도
- 사용 맥락: `carelessness caused the accident` = 부주의가 사고를 일으켰다

### caring

- 발음: /ˈkɛrɪŋ/
- 구성: `care` + `-ing`
- 뜻: 배려하는, 돌보는
- 어원적 연결: 마음을 쓰고 관심을 기울이는 성질
- 사용 맥락: `a caring teacher` = 배려심 있는 교사

### carefree

- 발음: /ˈkɛˌrfri/
- 구성: `care` + `free`
- 뜻: 걱정 없는, 근심 없는
- 어원적 연결: 마음을 무겁게 하는 걱정에서 자유로움
- 사용 맥락: `a carefree childhood` = 근심 없는 어린 시절

## 기억 포인트

`care` 계열은 "마음을 쓰고 주의를 기울임" 이미지.

- `care`: 돌봄, 관심, 주의
- `careful`: 주의 깊은
- `carefully`: 주의 깊게
- `careless`: 부주의한
- `carelessly`: 부주의하게
- `caring`: 배려하는
- `carefree`: 걱정 없는

## 비교 단어

### cura

- 뜻: 돌봄, 주의, 관리
- 구분 포인트: [[cura]]는 Latin 계열로 `cure`, `curate`, `accurate`, `secure` 등에 남음. `care`는 영어 기본어 계열.

### cautious

- 뜻: 조심스러운, 신중한
- 구분 포인트: 위험을 보고 조심하는 경계심. `careful`은 더 넓게 주의를 기울이는 태도.

### heedless

- 뜻: 부주의한, 개의치 않는
- 구분 포인트: `heed`는 주의해서 듣고 따름. `careless`는 마음 씀이나 주의가 부족한 일반 표현.
