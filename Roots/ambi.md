# ambi

- 출발 단어: `ambiguous`
- 핵심 어원: Latin `ambi-` = 양쪽에, 둘 다, 주위에
- 깊은 어근: PIE `*ambhi-` = around, on both sides
- 핵심 이미지: 양쪽을 둘러싸거나 두 방향에 걸침
- 관련 문서: [[bi]], [[amphi]], [[ag]], [[val]]
- 주의: `ambi-`는 Latin 계열. `amphi-`는 Greek 계열로 비슷하지만 어원 분리.

## 출발 단어 분석

`ambiguous` = `ambi-` + Latin `agere`

- `ambi-`: 양쪽에, 둘레에
- `agere`: 몰고 가다, 움직이다
- 원래 이미지: 의미가 양쪽으로 움직임
- 현대 의미: 모호한, 여러 해석이 가능한

## 단어들

### ambiguous

- 구성: `ambi-` = both sides + [[ag]] = move
- 뜻: 모호한, 여러 의미로 해석 가능한
- 어원적 연결: 의미가 한쪽으로 고정되지 않고 양쪽으로 움직임
- 사용 맥락: `an ambiguous statement` = 모호한 진술

### ambiguity

- 구성: `ambiguous` + `-ity`
- 뜻: 모호함, 애매함
- 어원적 연결: 의미가 두 방향 이상으로 갈 수 있는 상태
- 사용 맥락: `avoid ambiguity` = 모호함을 피하다

### ambivalent

- 구성: `ambi-` = both + [[val]] = strength, value
- 뜻: 양가적인, 상반된 감정을 함께 가진
- 어원적 연결: 두 방향의 힘이나 가치가 동시에 작용함
- 사용 맥락: `feel ambivalent about the plan` = 그 계획에 양가감정을 느끼다

### ambivalence

- 구성: `ambivalent` + `-ence`
- 뜻: 양가감정, 양면성
- 어원적 연결: 서로 다른 두 평가가 함께 있음
- 사용 맥락: `political ambivalence` = 정치적 양가성

### ambidextrous

- 구성: `ambi-` = both + Latin `dexter` = right hand
- 뜻: 양손잡이의, 양쪽을 능숙하게 쓰는
- 어원적 연결: 양쪽 모두 오른손처럼 능숙함
- 사용 맥락: `an ambidextrous athlete` = 양손을 잘 쓰는 선수

### ambidexterity

- 구성: `ambidextrous` 계열
- 뜻: 양손 사용 능력
- 어원적 연결: 양쪽 손을 모두 능숙하게 씀
- 사용 맥락: `develop ambidexterity` = 양손 사용 능력을 기르다

### ambient

- 구성: Latin `ambiens` ← `ambire` = go around
- 뜻: 주변의, 둘러싼
- 어원적 연결: 주위를 돌아 둘러싸고 있음
- 사용 맥락: `ambient temperature` = 주변 온도

### ambience

- 구성: French `ambiance` ← `ambient`
- 뜻: 분위기, 환경
- 어원적 연결: 사람을 둘러싼 주변 느낌
- 사용 맥락: `a relaxed ambience` = 편안한 분위기

### ambit

- 구성: Latin `ambitus` = a going around
- 뜻: 범위, 한계
- 어원적 연결: 둘러싼 테두리와 영향권
- 사용 맥락: `within the ambit of the law` = 법의 범위 안에서

### ambition

- 구성: Latin `ambire` = go around
- 뜻: 야망, 포부
- 어원적 연결: 원래는 지지를 얻으려고 주변을 돌아다님. 이후 높은 지위를 얻으려는 욕망
- 사용 맥락: `political ambition` = 정치적 야망

### ambitious

- 구성: `ambition` + `-ous`
- 뜻: 야심 있는, 목표가 큰
- 어원적 연결: 더 높은 것을 얻으려는 움직임
- 사용 맥락: `an ambitious project` = 야심 찬 프로젝트

### ambivert

- 구성: `ambi-` = both + `-vert` = turn
- 뜻: 양향성 성격, 외향성과 내향성을 모두 보이는 사람
- 어원적 연결: 두 방향으로 성향이 돌아감
- 사용 맥락: `an ambivert personality` = 양향성 성격

## 기억 포인트

`ambi-`는 "양쪽/둘레" 이미지.

- `ambiguous`: 의미가 양쪽으로 움직임
- `ambivalent`: 감정이나 평가가 양쪽으로 갈림
- `ambidextrous`: 양손을 모두 능숙하게 씀
- `ambient`: 주위를 둘러쌈
- `ambition`: 주변을 돌아다니며 지지를 구함

## 비교 단어

### amphi

- 뜻: 양쪽에, 둘레에
- 구분 포인트: [[amphi]]는 Greek 계열. `amphibian`, `amphitheater`처럼 Greek 단어에서 자주 보임. Latin `ambi-`와 의미는 비슷하지만 어원은 분리.

### bi

- 뜻: 둘
- 구분 포인트: `bi-`는 단순히 숫자 2. `ambi-`는 두 방향·양쪽·둘레에 걸친 움직임이 강함.
