# calibr

- 출발 단어: `calibrate`
- 핵심 어원: French `calibre` = bore, diameter, standard measure
- 핵심 이미지: 구멍의 지름을 기준으로 맞춤
- 관련 문서: [[metr]], [[scale]]
- 주의: `calibrate`는 `calculate`의 [[calc]] 계열과 철자가 비슷하지만 어원이 다름. `calibrate`는 기준 눈금·규격에 맞게 조정하는 쪽.

## 출발 단어 분석

`calibrate` ← `caliber` / `calibre`

- `calibre`: bore of a gun, diameter, standard measure
- 원래 이미지: 총신이나 관의 안쪽 지름을 재어 규격을 맞춤
- 의미 변화: 지름·규격을 잼 → 측정 장치를 기준값에 맞춤 → 판단이나 반응을 적절히 조정함
- 현대 의미: 보정하다, 눈금을 맞추다, 조정하다

## 단어들

### calibrate

- 발음: /ˈkæləˌbreɪt/
- 구성: `caliber` / `calibre` + `-ate`
- 뜻: 보정하다, 눈금을 맞추다, 조정하다
- 어원적 연결: 기준 규격에 맞도록 장비나 판단을 맞춤
- 사용 맥락: `calibrate the instrument` = 기기를 보정하다

### calibration

- 발음: /ˌkæləˈbreɪʃən/
- 구성: `calibrate` + `-ion`
- 뜻: 보정, 눈금 조정
- 어원적 연결: 기준값과 측정값이 맞도록 조정하는 과정
- 사용 맥락: `sensor calibration` = 센서 보정

### calibrated

- 발음: /ˈkæləˌbreɪtəd/
- 구성: `calibrate` + `-ed`
- 뜻: 보정된, 눈금이 맞춰진; 신중히 조절된
- 어원적 연결: 기준에 맞게 세밀하게 맞춰진 상태
- 사용 맥락: `a calibrated response` = 신중히 조절된 대응

### calibrator

- 구성: `calibrate` + `-or`
- 뜻: 보정기, 교정 장치
- 어원적 연결: 측정 장비를 기준값에 맞추는 도구
- 사용 맥락: `pressure calibrator` = 압력 교정기

### caliber

- 발음: /ˈkæləbɚ/
- 구성: French `calibre` = bore, diameter
- 뜻: 총구의 구경; 능력 수준, 품질
- 어원적 연결: 구멍의 지름이라는 규격 → 사람이나 물건의 수준
- 사용 맥락: `a writer of high caliber` = 수준 높은 작가

### calibre

- 구성: `caliber`의 British spelling
- 뜻: 구경; 능력 수준, 품질
- 어원적 연결: `caliber`와 같음
- 사용 맥락: `high-calibre candidates` = 수준 높은 후보자들

## 기억 포인트

`calibr` 계열은 "기준 규격에 맞게 조정" 이미지.

- `calibrate`: 보정하다, 조정하다
- `calibration`: 보정, 교정
- `calibrated`: 보정된, 신중히 조절된
- `calibrator`: 보정기
- `caliber`: 구경, 수준
- `calibre`: `caliber`의 영국식 철자

## 비교 단어

### measure

- 뜻: 재다, 측정하다
- 구분 포인트: [[metr]] 계열은 기준으로 값을 재는 일반적 측정. `calibrate`는 측정 장비 자체를 기준값에 맞추는 조정.

### scale

- 뜻: 척도, 눈금, 규모
- 구분 포인트: [[scale]]은 단계·척도·저울 이미지. `calibrate`는 그 눈금이나 장비가 정확히 맞도록 보정함.

### calculate

- 뜻: 계산하다
- 구분 포인트: [[calc]]는 작은 돌로 세어 계산함 계열. `calibrate`는 숫자를 계산하기보다 기준에 맞게 조정함.
