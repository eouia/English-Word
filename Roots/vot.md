# vot

- 출발 단어: `devote`
- 핵심 어원: 라틴어 `vovere` / `vot-` = 맹세하다, 바치다, 소원을 빌다
- 핵심 이미지: 마음이나 약속을 어떤 대상에 바침
- 관련해서 헷갈릴 수 있는 어근: [[voc]] = 부르다, 목소리

## 출발 단어 분석

`devote`는 라틴어 `devotare`에서 옴.

- `de-`: down, completely
- `vot-`: 맹세하다, 바치다

따라서 `devote`는 "완전히 맹세해 바치다" → "헌신하다, 전념하다"의 뜻.

## 단어들

### devote

- 구성: `de-` = down, completely + `vot`
- 뜻: 바치다, 전념하다, 헌신하다
- 어원적 연결: 시간, 노력, 마음을 어떤 대상에 맹세하듯 바침
- 사용 맥락: `devote time to study` = 공부에 시간을 바치다

### devoted

- 구성: `devote` + `-ed`
- 뜻: 헌신적인, 충실한
- 어원적 연결: 마음이 한 대상에 바쳐진 상태
- 사용 맥락: `a devoted teacher` = 헌신적인 교사

### devotion

- 구성: `devote` + `-ion`
- 뜻: 헌신, 전념, 깊은 애정
- 어원적 연결: 마음과 노력을 지속적으로 바치는 태도
- 사용 맥락: `devotion to family` = 가족에 대한 헌신

### devotee

- 구성: `devote` + `-ee`
- 뜻: 열성 지지자, 애호가
- 어원적 연결: 어떤 대상에 마음을 바친 사람
- 사용 맥락: `a jazz devotee` = 재즈 애호가

### vote

- 구성: Latin `votum`
- 뜻: 투표하다, 표
- 어원적 연결: 원래 "서약, 소원, 바람". 공동체 안에서 자신의 뜻을 공식적으로 드러내는 행위로 의미 확대
- 사용 맥락: `cast a vote` = 표를 던지다

### voter

- 구성: `vote` + `-er`
- 뜻: 유권자, 투표자
- 어원적 연결: 자신의 뜻을 공식적으로 표시하는 사람
- 사용 맥락: `young voters` = 젊은 유권자들

### voting

- 구성: `vote` + `-ing`
- 뜻: 투표
- 어원적 연결: 각자의 뜻을 공식적으로 표시하는 과정
- 사용 맥락: `voting rights` = 투표권

### vow

- 구성: Old French `vou`, Latin `votum`
- 뜻: 맹세, 서약하다
- 어원적 연결: 신이나 사람 앞에서 마음을 바치겠다고 약속함
- 사용 맥락: `make a vow` = 맹세하다

### avow

- 구성: Old French `avouer`
- 뜻: 공개적으로 인정하다, 공언하다
- 어원적 연결: 자신의 말이나 믿음을 공식적으로 내세움
- 사용 맥락: `avow one's beliefs` = 자신의 신념을 공언하다

### avowed

- 구성: `avow` + `-ed`
- 뜻: 공공연한, 자처하는
- 어원적 연결: 스스로 공식적으로 인정한 상태
- 사용 맥락: `an avowed supporter` = 공공연한 지지자

### votive

- 구성: Latin `votivus`
- 뜻: 서원의, 봉헌의
- 어원적 연결: 맹세나 소원을 담아 바치는 것
- 사용 맥락: `a votive candle` = 봉헌 초

## 기억 포인트

`vot` 계열은 "맹세하고 마음을 바침" 이미지.

- `devote`: 마음과 노력을 바침
- `devotion`: 헌신
- `devotee`: 마음을 바친 사람
- `vote`: 자신의 뜻을 공식적으로 표시함
- `vow`: 맹세
- `avow`: 공개적으로 인정함
- `votive`: 서원이나 봉헌과 관련됨

## 비교 단어

### voc

- 뜻: 부르다
- 구분 포인트: [[voc]]는 목소리로 부르는 이미지. `vot`는 맹세하고 바치는 이미지
