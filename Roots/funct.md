# funct

- 출발 단어: `defunct`
- 핵심 어원: Latin `fungi` / `functus` = perform, discharge, carry out
- 핵심 이미지: 맡은 일을 수행함, 임무를 끝냄
- 관련 문서: [[ag]], [[oper]], [[mal]]
- 주의: `defunct`는 현대 형태만 보면 `de- + funct`처럼 보이지만, 실제로는 Latin `defunctus` = finished, dead에서 옴. "기능이 없다"보다 "맡은 일을 다 끝내고 사라짐" 이미지.

## 출발 단어 분석

`defunct` ← Latin `defunctus` = discharged, finished, dead

- `de-`: down, away, completely
- `functus`: having performed, having discharged
- 원래 이미지: 맡은 일을 다 수행하고 끝난 상태
- 의미 변화: 임무를 마침 → 생의 역할을 끝냄 → 죽은, 더 이상 존재하지 않는
- 현대 의미: 더 이상 존재하지 않는, 폐지된, 작동하지 않는

## 단어들

### defunct

- 발음: /dɪˈfʌŋkt/
- 구성: Latin `defunctus` = finished, dead
- 뜻: 더 이상 존재하지 않는, 폐지된, 작동하지 않는
- 어원적 연결: 맡은 역할을 끝내고 사라진 상태
- 사용 맥락: `a defunct company` = 사라진 회사

### function

- 발음: /ˈfʌŋkʃən/
- 구성: Latin `functio` = performance, execution
- 뜻: 기능, 역할; 기능하다
- 어원적 연결: 어떤 것이 맡은 일을 수행함
- 사용 맥락: `serve a function` = 기능을 하다

### functional

- 발음: /ˈfʌŋkʃənəl/
- 구성: `function` + `-al`
- 뜻: 기능적인, 작동하는, 실용적인
- 어원적 연결: 실제로 맡은 일을 수행할 수 있음
- 사용 맥락: `a functional design` = 기능적인 설계

### functionary

- 발음: /ˈfʌŋkʃəˌnɛri/
- 구성: `function` + `-ary`
- 뜻: 하급 관리, 공무원
- 어원적 연결: 정해진 직무를 수행하는 사람
- 사용 맥락: `a government functionary` = 정부 관리

### dysfunction

- 발음: /dɪˈsfʌŋkʃən/
- 구성: `dys-` = bad, difficult + `function`
- 뜻: 기능 장애, 역기능
- 어원적 연결: 맡은 기능을 제대로 수행하지 못함
- 사용 맥락: `social dysfunction` = 사회적 역기능

### dysfunctional

- 발음: /dɪˈsfʌŋkʃənəl/
- 구성: `dysfunction` + `-al`
- 뜻: 제대로 기능하지 않는, 역기능적인
- 어원적 연결: 구조나 관계가 맡은 역할을 제대로 수행하지 못함
- 사용 맥락: `a dysfunctional system` = 제대로 돌아가지 않는 체계

### malfunction

- 발음: /mæˈlfʌŋkʃən/
- 구성: [[mal]] = badly + `function`
- 뜻: 오작동, 제대로 작동하지 않다
- 어원적 연결: 기계나 장치가 자기 기능을 나쁘게 수행함
- 사용 맥락: `equipment malfunction` = 장비 오작동

### perfunctory

- 발음: /pɚˈfʌŋktɚi/
- 구성: Latin `perfunctus` = done with, discharged
- 뜻: 형식적인, 건성의
- 어원적 연결: 해야 할 일을 그저 끝내기만 한 태도
- 사용 맥락: `a perfunctory apology` = 형식적인 사과

### fungible

- 발음: /ˈfʌndʒɪbəl/
- 구성: Medieval Latin `fungibilis`, from `fungi` = perform, use
- 뜻: 대체 가능한, 서로 교환 가능한
- 어원적 연결: 같은 기능을 수행할 수 있어 서로 바꿔 쓸 수 있음
- 사용 맥락: `fungible assets` = 대체 가능한 자산

## 기억 포인트

`funct` 계열은 "맡은 일을 수행함" 이미지.

- `function`: 수행하는 역할
- `functional`: 기능을 수행할 수 있음
- `dysfunction`: 기능 수행이 어긋남
- `malfunction`: 나쁘게 기능함
- `defunct`: 역할을 끝내고 사라짐
- `perfunctory`: 끝내기만 한 형식적 수행

## 비교 단어

### ag

- 뜻: 움직이다, 몰고 가다, 행하다
- 구분 포인트: [[ag]]는 행동 자체가 넓음. `funct`는 맡은 역할이나 직무를 수행하는 느낌.

### oper

- 뜻: 일하다, 작동하다
- 구분 포인트: [[oper]]가 실제 작업·작동 이미지라면, `funct`는 정해진 기능·역할 수행 이미지.

### mal

- 뜻: 나쁘게, 잘못
- 구분 포인트: [[mal]]은 접두사 문서. `malfunction`의 중심 동작은 `function`, 나쁘게 수행함은 `mal-`.
