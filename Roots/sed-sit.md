# sed-sit

- 상위 문서: [[sed]]
- 핵심 어원: PIE `*sed-` = 앉다
- 핵심 이미지: 몸을 자리에 두거나 한곳에 자리 잡음
- 관련 갈래: [[sed-sedere]], [[sed-sess]]

## 단어들

### sit

- 발음: /ˈsɪt/
- 구성: 고대 영어 `sittan`
- 뜻: 앉다
- 어원적 연결: 몸을 자리에 둠
- 사용 맥락: `sit down` = 앉다

### seat

- 발음: /ˈsit/
- 구성: 고대 노르드어 `saeti` 계열
- 뜻: 좌석, 자리, 앉히다
- 어원적 연결: 앉는 곳 또는 앉게 하는 행위
- 사용 맥락: `take a seat` = 자리에 앉다

### settle

- 발음: /ˈsɛtəl/
- 구성: `set` 계열 + `-le`
- 뜻: 정착하다, 해결하다
- 어원적 연결: 한곳에 앉듯 자리 잡음
- 사용 맥락: `settle in a city` = 도시에 정착하다

## 기억 포인트

`sed-sit`은 영어 기본어 쪽 갈래.

- `sit`: 앉음
- `seat`: 앉는 자리
- `settle`: 자리를 잡음

## 비교 단어

### set

- 뜻: 놓다, 정하다
- 구분 포인트: [[set]]은 같은 큰 뿌리에서 나온 "앉히다, 놓다" 계열. `sed-sit`은 앉는 동작과 자리 이미지에 초점
