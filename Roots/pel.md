# pel

- 출발 단어: `compel`
- 핵심 어원: 라틴어 `pellere` / `puls-` = 밀다, 몰아가다
- 핵심 이미지: 힘으로 밀어 움직이게 함
- 관련해서 헷갈릴 수 있는 어근: [[duc]] = 이끌다 계열, [[tract]] = 끌다 계열

## 출발 단어 분석

`compel`은 `com-` + `pellere`.

- `com-`: together, completely
- `pellere`: push, drive

따라서 `compel`은 "강하게 몰아가다" → "강요하다, 어쩔 수 없이 하게 하다"의 뜻.

## 단어들

### compel

- 구성: [[con]]의 `com-` = completely + `pel`
- 뜻: 강요하다, 억지로 하게 하다
- 어원적 연결: 힘으로 밀어 어떤 행동을 하게 함
- 사용 맥락: `compel obedience` = 복종을 강요하다

### compelling

- 구성: `compel` + `-ing`
- 뜻: 설득력 있는, 강렬한
- 어원적 연결: 마음을 밀어 움직일 만큼 강함
- 사용 맥락: `compelling evidence` = 설득력 있는 증거

### expel

- 구성: [[ex]] = out + `pel`
- 뜻: 쫓아내다, 퇴학시키다
- 어원적 연결: 밖으로 밀어냄
- 사용 맥락: `expel a student` = 학생을 퇴학시키다

### repel

- 구성: `re-` = back + `pel`
- 뜻: 물리치다, 불쾌하게 하다
- 어원적 연결: 뒤로 밀어냄
- 사용 맥락: `repel an attack` = 공격을 물리치다

### repellent

- 구성: `repel` + `-ent`
- 뜻: 방충제, 불쾌한
- 어원적 연결: 접근하지 못하게 밀어냄
- 사용 맥락: `insect repellent` = 방충제

### propel

- 구성: `pro-` = forward + `pel`
- 뜻: 추진하다, 나아가게 하다
- 어원적 연결: 앞으로 밀어 보냄
- 사용 맥락: `propel a boat` = 배를 나아가게 하다

### propulsion

- 구성: `pro-` + `puls`
- 뜻: 추진, 추진력
- 어원적 연결: 앞으로 밀어 움직이는 힘
- 사용 맥락: `jet propulsion` = 제트 추진

### impulse

- 구성: `in-` = into + `puls`
- 뜻: 충동, 자극
- 어원적 연결: 안에서 밀어 움직이게 하는 힘
- 사용 맥락: `act on impulse` = 충동적으로 행동하다

### impulsive

- 구성: `impulse` + `-ive`
- 뜻: 충동적인
- 어원적 연결: 안에서 미는 힘에 바로 움직임
- 사용 맥락: `an impulsive decision` = 충동적인 결정

### compulsory

- 구성: `com-` + `puls` + `-ory`
- 뜻: 의무적인, 강제적인
- 어원적 연결: 강하게 밀려 해야 하는 것
- 사용 맥락: `compulsory education` = 의무 교육

### pulse

- 구성: Latin `pulsus` = beating, pushing
- 뜻: 맥박, 고동
- 어원적 연결: 반복적으로 밀고 뛰는 움직임
- 사용 맥락: `check the pulse` = 맥박을 확인하다

## 기억 포인트

`pel/puls` 계열은 "밀다 / 몰아가다" 이미지.

- `compel`: 강하게 밀어 시킴
- `expel`: 밖으로 밀어냄
- `repel`: 뒤로 밀어냄
- `propel`: 앞으로 밀어 보냄
- `impulse`: 안에서 미는 충동
- `compulsory`: 강제로 밀려 해야 함

## 비교 단어

### duc

- 뜻: 이끌다
- 구분 포인트: [[duc]]는 방향을 잡아 이끎. `pel`은 힘으로 밀어 움직이게 함

### tract

- 뜻: 끌다
- 구분 포인트: [[tract]]는 당겨 움직임. `pel`은 밀어 움직임
