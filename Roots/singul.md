# singul

- 출발 단어: `single`
- 핵심 어원: Latin `singulus` = one each, one at a time
- 핵심 이미지: 하나씩 따로, 개별적인, 단 하나의
- 관련 문서: [[uni]], [[mono]], [[solus]]
- 주의: `single`은 "혼자"보다 "하나씩 따로 떨어진 개별 단위" 이미지가 강함. 외로움·고독은 [[solus]] 쪽이 더 가까움.

## 출발 단어 분석

`single` ← Old French `sengle` ← Latin `singulus`

- `singulus`: one each, individual
- 원래 이미지: 여러 개 중 하나씩 따로
- 현대 의미: 하나의, 단일의, 미혼의, 싱글

## 단어들

### single

- 발음: /ˈsɪŋgəl/
- 구성: Latin `singulus` = one each
- 뜻: 하나의, 단일의, 미혼의
- 어원적 연결: 한 개씩 따로 있는 상태
- 사용 맥락: `a single room` = 1인실, `single parents` = 한부모

### singular

- 발음: /ˈsɪŋgjəlɚ/
- 구성: Latin `singularis` = single, individual
- 뜻: 단수의, 특이한, 두드러진
- 어원적 연결: 하나로 따로 구별됨
- 사용 맥락: `a singular noun` = 단수 명사, `a singular achievement` = 특출한 업적

### singularity

- 발음: /ˈsɪŋgjəˈlɛrəti/
- 구성: `singular` + `-ity`
- 뜻: 특이점, 단일성, 특이함
- 어원적 연결: 다른 것과 구별되는 하나의 특별한 상태
- 사용 맥락: 과학·기술 맥락에서 `the technological singularity`

### singly

- 발음: /ˈsɪŋgli/
- 구성: `single` + `-ly`
- 뜻: 하나씩, 단독으로
- 어원적 연결: 여럿이 아니라 한 개씩 따로
- 사용 맥락: `work singly or in pairs` = 혼자 또는 둘씩 일하다

### single-handed

- 발음: /ˈsɪŋgəˈlhændɪd/
- 구성: `single` + `handed`
- 뜻: 혼자 힘으로 한, 단독의
- 어원적 연결: 한 사람의 손으로 해냄
- 사용 맥락: `single-handedly complete the task` = 혼자서 일을 끝내다

## 기억 포인트

`singul` 계열은 "하나씩 따로, 개별" 이미지.

- `single`: 하나의, 단일의
- `singular`: 단수의, 특이한
- `singularity`: 특이점, 특별히 하나로 구별되는 상태
- `singly`: 하나씩

## 비교 단어

### solus

- 뜻: 홀로, 혼자
- 구분 포인트: [[solus]]는 외부와 떨어져 홀로 있음. `singul`은 하나씩 개별로 셈.

### uni

- 뜻: 하나, 하나로 묶임
- 구분 포인트: [[uni]]는 여러 부분이 하나로 통합되는 이미지가 강함. `singul`은 하나씩 따로 구별되는 이미지.

### mono

- 뜻: 하나, 홀로
- 구분 포인트: [[mono]]는 Greek 계열 접두사. `singul`은 Latin `singulus` 계열.
