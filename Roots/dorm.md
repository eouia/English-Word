# dorm

- 출발 단어: `dormant`
- 핵심 어원: Latin `dormire` = sleep
- 핵심 이미지: 잠자거나 활동을 멈춘 상태
- 관련 문서: [[dom]], [[doom]]
- 주의: [[dom]]은 집·주인 계열, [[doom]]은 Old English `-dom` 계열. `dorm`은 Latin `dormire` = sleep 계열.

## 출발 단어 분석

`dormant` ← Old French `dormant` = sleeping

- `dorm-`: sleep
- `-ant`: ~하는 상태의
- 원래 이미지: 잠자고 있는 상태
- 의미 변화: 잠든 → 활동하지 않는 → 잠재해 있다가 나중에 깨어날 수 있는
- 현대 의미: 휴면 상태의, 잠복한, 활동하지 않는

## 단어들

### dormant

- 발음: /ˈdɔrmənt/
- 구성: Latin `dormire` = sleep + `-ant`
- 뜻: 휴면 상태의, 잠복한, 활동하지 않는
- 어원적 연결: 살아 있거나 존재하지만 잠자듯 활동하지 않음
- 사용 맥락: `a dormant volcano` = 휴화산, `dormant talent` = 잠재된 재능

### dormancy

- 발음: /ˈdɔrmənsi/
- 구성: `dormant` + `-cy`
- 뜻: 휴면, 비활동 상태
- 어원적 연결: 잠든 듯 활동을 멈춘 상태
- 사용 맥락: `seed dormancy` = 씨앗 휴면

### dormitory

- 발음: /ˈdɔrməˌtɔri/
- 구성: Latin `dormitorium` = sleeping place
- 뜻: 기숙사, 공동 침실
- 어원적 연결: 잠자는 장소
- 사용 맥락: `a college dormitory` = 대학 기숙사

### dorm

- 발음: /ˈdɔrm/
- 구성: short for `dormitory`
- 뜻: 기숙사
- 어원적 연결: `dormitory`의 줄임말
- 사용 맥락: `live in a dorm` = 기숙사에 살다

### dormer

- 발음: /ˈdɔrmɚ/
- 구성: Old French `dormoir` = sleeping room, later roof window
- 뜻: 지붕창, 도머 창
- 어원적 연결: 원래 지붕 아래 잠자는 방과 연결된 창
- 사용 맥락: `a dormer window` = 지붕창

## 기억 포인트

`dorm` 계열은 "잠" 이미지.

- `dormant`: 잠든 듯 활동하지 않음
- `dormancy`: 휴면 상태
- `dormitory`: 잠자는 장소
- `dorm`: 기숙사
- `dormer`: 지붕 아래 방의 창에서 온 말

## 비교 단어

### dom

- 뜻: 집, 주인, 지배자
- 구분 포인트: [[dom]]은 Latin `domus/dominus` 계열. `dormant`, `dormitory`는 Latin `dormire` = sleep 계열.

### doom

- 뜻: 판정된 상태·권한·영역
- 구분 포인트: [[doom]]은 Old English `dōm` 계열. `kingdom`, `freedom`, `wisdom`의 `-dom`과 연결됨.
