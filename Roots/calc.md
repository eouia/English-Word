# calc

- 출발 단어: `calculate`
- 핵심 어원: Latin `calculus` = 작은 돌, 계산용 조약돌
- 관련 형태: Latin `calx` = 석회석, 조약돌
- 핵심 이미지: 작은 돌을 놓고 세어 계산함
- 관련 문서: [[numer]], [[metr]], [[logos]]
- 주의: `calcium`, `chalk`는 석회·돌 이미지와 관련되지만 계산 계열과 의미가 갈라지므로 비교 단어로 분리.

## 출발 단어 분석

`calculate` ← Latin `calculare`

- `calculus`: 작은 돌, 계산용 조약돌
- `calculare`: 조약돌로 계산하다
- 원래 이미지: 돌을 놓아 수를 셈
- 현대 의미: 계산하다, 추정하다

## 단어들

### calculate

- 발음: /ˈkælkjəˌleɪt/
- 구성: Latin `calculare`
- 뜻: 계산하다, 추산하다
- 어원적 연결: 조약돌을 놓고 수를 세어 계산함
- 사용 맥락: `calculate the cost` = 비용을 계산하다

### calculation

- 발음: /ˌkælkjəˈleɪʃən/
- 구성: `calculate` + `-ion`
- 뜻: 계산, 산정
- 어원적 연결: 수를 세고 따져 결과를 냄
- 사용 맥락: `a quick calculation` = 빠른 계산

### calculator

- 발음: /ˈkælkjəˌleɪtɚ/
- 구성: `calculate` + `-or`
- 뜻: 계산기, 계산하는 사람
- 어원적 연결: 계산을 수행하는 도구나 사람
- 사용 맥락: `use a calculator` = 계산기를 사용하다

### calculus

- 발음: /ˈkælkjələs/
- 구성: Latin `calculus`
- 뜻: 미적분학, 계산법
- 어원적 연결: 원래는 계산용 작은 돌. 수학의 계산 체계로 확장
- 사용 맥락: `differential calculus` = 미분학

### miscalculate

- 발음: /mɪˈskælkjəˌleɪt/
- 구성: `mis-` = wrongly + `calculate`
- 뜻: 잘못 계산하다, 오판하다
- 어원적 연결: 수나 상황을 잘못 따짐
- 사용 맥락: `miscalculate the risk` = 위험을 오판하다

### incalculable

- 발음: /ˌɪˈnkælkjələbəl/
- 구성: `in-` = not + `calculate` + `-able`
- 뜻: 계산할 수 없는, 헤아릴 수 없이 큰
- 어원적 연결: 수로 따져 낼 수 없음
- 사용 맥락: `incalculable damage` = 헤아릴 수 없는 피해

### calculable

- 구성: `calculate` + `-able`
- 뜻: 계산 가능한, 예측 가능한
- 어원적 연결: 수나 근거로 따져 낼 수 있음
- 사용 맥락: `a calculable risk` = 계산 가능한 위험

## 기억 포인트

`calc` 계열은 "조약돌로 세어 계산함" 이미지.

- `calculate`: 계산하다
- `calculation`: 계산
- `calculator`: 계산 도구
- `calculus`: 계산법에서 발전한 수학 분야
- `miscalculate`: 잘못 계산함
- `incalculable`: 계산할 수 없음

## 비교 단어

### calcium

- 뜻: 칼슘
- 구분 포인트: Latin `calx` = lime, limestone 계열. 돌·석회 이미지와 관련되지만 현대 영어에서는 원소 이름으로 따로 기억.

### chalk

- 뜻: 분필, 백악
- 구분 포인트: 석회질 흰 돌 이미지. `calculate`의 계산 행위보다 물질 계열에 가까움.

### recalcitrant

- 뜻: 고집 센, 반항적인
- 구분 포인트: [[calcitr]] 계열. 철자는 `calc`와 닮았지만 "발길질하며 버팀" 이미지라 계산 계열과 분리.
