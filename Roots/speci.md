# speci

- 출발 단어: `species`
- 핵심 어원: Latin `species` = appearance, form, kind; from `specere` = look at
- 핵심 이미지: 보이는 모양에서 구분되는 종류, 특정한 형식
- 관련 문서: [[spect]], [[gen]], [[fac]]
- 주의: [[spect]]는 바라봄/살핌의 동작 중심. `speci`는 보이는 형상에서 나온 종류·특수성 중심.

## 출발 단어 분석

`species` = Latin `species`

- 원래 의미: 보이는 모양, 외관, 형태
- 확장 의미: 모습으로 구별되는 종류, 부류
- 현대 의미: 종, 종류

## 단어들

### species

- 구성: Latin `species` ← `specere`
- 뜻: 종, 종류
- 어원적 연결: 보이는 특징으로 구별되는 부류
- 사용 맥락: `endangered species` = 멸종 위기종

### specific

- 구성: Late Latin `specificus` = species-making
- 뜻: 특정한, 구체적인
- 어원적 연결: 한 종류나 경우를 분명히 구별함
- 사용 맥락: `specific details` = 구체적 세부 사항

### specifically

- 구성: `specific` + `-ally`
- 뜻: 구체적으로, 특별히
- 어원적 연결: 특정 종류나 항목을 집어 말함
- 사용 맥락: `specifically designed` = 특별히 설계된

### specify

- 구성: `species` + [[fac]] = make
- 뜻: 명시하다, 구체적으로 말하다
- 어원적 연결: 특정한 종류나 조건으로 만들어 구분함
- 사용 맥락: `specify a date` = 날짜를 명시하다

### specification

- 구성: `specify` + `-ation`
- 뜻: 명세, 사양
- 어원적 연결: 특정 조건을 분명히 적은 것
- 사용 맥락: `technical specifications` = 기술 사양

### special

- 구성: Latin `specialis` ← `species`
- 뜻: 특별한, 특수한
- 어원적 연결: 일반적인 것과 구별되는 특정 종류
- 사용 맥락: `special privilege` = 특별한 특권

### specialist

- 구성: `special` + `-ist`
- 뜻: 전문가
- 어원적 연결: 특정 분야에 집중한 사람
- 사용 맥락: `a medical specialist` = 전문의

### specialize

- 구성: `special` + `-ize`
- 뜻: 전문화하다, 전공하다
- 어원적 연결: 특정 분야로 좁혀 들어감
- 사용 맥락: `specialize in biology` = 생물학을 전공하다

### specialty

- 구성: `special` + `-ty`
- 뜻: 전문 분야, 특산품
- 어원적 연결: 특정하게 구별되는 분야나 물건
- 사용 맥락: `local specialty` = 지역 특산품

### specimen

- 구성: Latin `specimen` = mark, example, means of knowing
- 뜻: 표본, 견본
- 어원적 연결: 보고 그 종류를 알 수 있는 예
- 사용 맥락: `a blood specimen` = 혈액 표본

### specious

- 구성: Latin `speciosus` = beautiful, showy
- 뜻: 그럴듯하지만 거짓인, 외양만 그럴듯한
- 어원적 연결: 보이는 모습은 좋아 보이나 실제와 다름
- 사용 맥락: `a specious argument` = 그럴듯하지만 틀린 주장

### specie

- 구성: Latin `in specie` = in kind
- 뜻: 주화, 현금
- 어원적 연결: 추상적 가치가 아니라 특정한 물건의 형태
- 사용 맥락: `payment in specie` = 금속 화폐 지급

## 기억 포인트

`speci` 계열은 "보이는 모양으로 구별되는 종류" 이미지.

- `species`: 구별되는 종
- `specific`: 특정한
- `specify`: 구체적으로 정함
- `special`: 보통과 구별됨
- `specialist`: 특정 분야 전문가
- `specimen`: 보고 종류를 아는 표본
- `specious`: 겉보기만 그럴듯함

## 비교 단어

### spect

- 뜻: 보다, 살피다
- 구분 포인트: [[spect]]는 보는 동작과 방향. `speci`는 보이는 형상에서 종류와 특수성으로 확장.

### gen

- 뜻: 종류, 태어남, 혈통
- 구분 포인트: [[gen]]의 `genus/generic`은 태생·종류의 분류. `species`는 보이는 형상으로 구별되는 하위 종류 이미지.
