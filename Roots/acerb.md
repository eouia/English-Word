# acerb

- 출발 단어: `exacerbate`
- 핵심 어원: Latin `acerbus` = bitter, harsh, severe
- 핵심 이미지: 맛이나 말투가 쓰고 날카로워 더 견디기 어려움
- 관련 문서: [[acid]]
- 주의: `acrid`, `acrimony`는 Latin `acer/acris` = sharp 계열로 가깝지만, `exacerbate`는 직접적으로 `acerbus`에서 옴.

## 출발 단어 분석

`exacerbate` = `ex-` + `acerb` + `-ate`

- `ex-`: out, thoroughly, intensively
- `acerb`: bitter, harsh
- `-ate`: make, cause
- 원래 이미지: 이미 쓰라린 상태를 더 쓰고 거칠게 만듦
- 현대 의미: 악화시키다, 더 심하게 만들다

## 단어들

### exacerbate

- 발음: /ɪˈgzæsɚˌbeɪt/
- 구성: Latin `exacerbare` ← `ex-` + `acerbus`
- 뜻: 악화시키다, 더 심하게 만들다
- 어원적 연결: 쓰라린 상태를 더 쓰라리게 만듦
- 사용 맥락: `exacerbate a problem` = 문제를 악화시키다, `exacerbate tensions` = 긴장을 더 심하게 만들다

### exacerbation

- 발음: /ɪˌgzæsɚˈbeɪʃən/
- 구성: `exacerbate` + `-ion`
- 뜻: 악화, 심화; 병세의 급격한 악화
- 어원적 연결: 상태가 더 거칠고 견디기 어려워짐
- 사용 맥락: `an exacerbation of symptoms` = 증상의 악화

### acerbic

- 발음: /əˈsɛrbɪk/
- 구성: Latin `acerbus` + `-ic`
- 뜻: 신랄한, 날카롭게 비판적인
- 어원적 연결: 말맛이 쓰고 거칠어 상대를 찌름
- 사용 맥락: `an acerbic comment` = 신랄한 말

### acerbity

- 구성: `acerb` + `-ity`
- 뜻: 신랄함, 매서움; 쓴맛
- 어원적 연결: 맛이나 말투의 쓰고 거친 성질
- 사용 맥락: 문어적 표현. 말투·비평의 날카로움을 말할 때 사용.

### acerbate

- 구성: Latin `acerbare` = make bitter or harsh
- 뜻: 쓰게 하다, 거칠게 하다; 악화시키다
- 어원적 연결: 상태를 쓰고 harsh하게 만듦
- 사용 맥락: 드문 단어. 현대 영어에서는 보통 `exacerbate`를 사용.

## 기억 포인트

`acerb`는 "쓴맛 + 거친 말 + 악화" 이미지.

- `exacerbate`: 더 쓰라리게 만들어 악화함
- `exacerbation`: 악화된 상태
- `acerbic`: 말투가 쓰고 신랄함
- `acerbity`: 신랄함, 매서움
- `acerbate`: 쓰고 거칠게 만듦

## 비교 단어

### acrid

- 뜻: 톡 쏘는, 매캐한
- 구분 포인트: Latin `acer/acris` = sharp 계열. 냄새·맛이 날카롭게 찌르는 느낌.

### acrimony

- 뜻: 신랄함, 험악함
- 구분 포인트: Latin `acer/acris` = sharp 계열. 논쟁·관계가 날카롭고 거친 상태.

### acid

- 뜻: 산, 신맛의; 신랄한
- 구분 포인트: Latin `acidus` = sour 계열. 의미상 비슷하지만 `exacerbate`의 직접 어원은 `acerbus`.
