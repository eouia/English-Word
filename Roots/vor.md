# vor

- 출발 단어: `voracious`
- 핵심 어원: Latin `vorare` = 게걸스럽게 먹다, 삼키다
- 핵심 이미지: 무언가를 탐하듯 먹어 치움
- 관련 문서: [[carn]], [[omni]]

## 단어들

### voracious

- 구성: Latin `vorare` + `-acious`
- 뜻: 게걸스러운, 탐욕스러운
- 어원적 연결: 많이 먹거나 집어삼키려는 성향
- 사용 맥락: `a voracious reader` = 닥치는 대로 읽는 독서가

### devour

- 구성: `de-` = down, completely + Latin `vorare`
- 뜻: 게걸스럽게 먹다, 집어삼키다
- 어원적 연결: 완전히 먹어 치움
- 사용 맥락: `devour a meal` = 음식을 게걸스럽게 먹다

### carnivore

- 구성: [[carn]] = flesh + `vorare`
- 뜻: 육식동물
- 어원적 연결: 고기를 먹는 존재
- 사용 맥락: `large carnivores` = 대형 육식동물

### herbivore

- 구성: Latin `herba` = plant + `vorare`
- 뜻: 초식동물
- 어원적 연결: 풀과 식물을 먹는 존재
- 사용 맥락: `herbivores and carnivores` = 초식동물과 육식동물

### omnivore

- 구성: [[omni]] = all + `vorare`
- 뜻: 잡식동물
- 어원적 연결: 모든 종류의 음식을 먹음
- 사용 맥락: `humans are omnivores` = 인간은 잡식동물

## 기억 포인트

`vor` 계열은 "먹어 치우다" 이미지.

- `voracious`: 게걸스러운
- `devour`: 집어삼키다
- `carnivore`: 고기를 먹는 동물
- `herbivore`: 식물을 먹는 동물
- `omnivore`: 모두 먹는 동물

## 비교 단어

### carn

- 뜻: 살, 고기
- 구분 포인트: [[carn]]은 고기 자체. `vor`는 먹는 행위.
