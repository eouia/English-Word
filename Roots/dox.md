# dox

- 출발 단어: `paradox`
- 핵심 어원: Greek `doxa` = 의견, 믿음, 평판, 영광
- 더 깊은 어원: Greek `dokein` = 보이다, ~처럼 생각되다
- 핵심 의미: 보이는 바에 따른 의견, 받아들여진 믿음
- 관련 문서: [[logos]], [[dict]], [[leg]]
- 주의: `paradox`는 “완전한 모순”이 아니라 “통념과 어긋나 보이나 생각할 가치가 있는 말”.

## 출발 단어 분석

`paradox` = `para-` + `dox`

- `para-`: beside, contrary to
- `doxa`: opinion, belief
- 원래 이미지: 일반 의견이나 기대 옆으로 벗어난 말
- 현대 의미: 역설, 모순처럼 보이지만 의미가 있는 진술

## 단어들

### paradox

- 발음: /ˈpɛrəˌdɑks/
- 구성: `para-` = contrary to + `doxa` = opinion
- 뜻: 역설, 모순처럼 보이는 말
- 어원적 연결: 일반적인 의견이나 기대에 어긋남
- 사용 맥락: `a paradox of freedom` = 자유의 역설

### paradoxical

- 발음: /ˌpɛrəˈdɑksɪkəl/
- 구성: `paradox` + `-ical`
- 뜻: 역설적인
- 어원적 연결: 겉으로는 앞뒤가 맞지 않아 보이나 생각할 지점이 있음
- 사용 맥락: `a paradoxical result` = 역설적인 결과

### paradoxically

- 발음: /ˌpɛrəˈdɑksəkli/
- 구성: `paradoxical` + `-ly`
- 뜻: 역설적으로
- 어원적 연결: 예상과 반대로 보이는 방식
- 사용 맥락: `paradoxically, it worked` = 역설적으로 그것은 효과가 있었다

### orthodox

- 발음: /ˈɔrθəˌdɑks/
- 구성: Greek `orthos` = right, straight + `doxa` = opinion
- 뜻: 정통의, 정설의
- 어원적 연결: 바른 의견, 정통으로 인정된 믿음
- 사용 맥락: `orthodox medicine` = 정통 의학

### orthodoxy

- 발음: /ˈɔrθəˌdɑksi/
- 구성: `orthodox` + `-y`
- 뜻: 정통, 정설, 정통 교리
- 어원적 연결: 바른 믿음으로 인정된 체계
- 사용 맥락: `challenge orthodoxy` = 정설에 도전하다

### unorthodox

- 발음: /əˈnɔrθəˌdɑks/
- 구성: `un-` + `orthodox`
- 뜻: 비정통적인, 관습에서 벗어난
- 어원적 연결: 정통 의견과 맞지 않음
- 사용 맥락: `an unorthodox method` = 비정통적인 방법

### heterodox

- 발음: /ˈhɛtɚəˌdɑks/
- 구성: Greek `heteros` = other, different + `doxa` = opinion
- 뜻: 이단적인, 비정통의
- 어원적 연결: 다른 의견을 가짐
- 사용 맥락: `heterodox economics` = 비주류 경제학

### heterodoxy

- 발음: /ˈhɛtɚəˌdɑksi/
- 구성: `heterodox` + `-y`
- 뜻: 이단, 비정통 견해
- 어원적 연결: 정통과 다른 의견의 체계
- 사용 맥락: religious / academic 맥락

### doxa

- 구성: Greek `doxa`
- 뜻: 통념, 의견, 믿음; 평판, 영광
- 어원적 연결: 사람들에게 그렇게 보이고 받아들여진 것
- 사용 맥락: philosophy / theology 맥락

### doxology

- 발음: /dɑˈksɑlədʒi/
- 구성: `doxa` = glory + [[logos]]
- 뜻: 송영, 하나님께 영광을 돌리는 찬양문
- 어원적 연결: 영광을 말로 표현함
- 사용 맥락: Christian liturgy 맥락

### dogma

- 발음: /ˈdɑgmə/
- 구성: Greek `dogma` ← `dokein` = think, seem good
- 뜻: 교리, 독단적 신념
- 어원적 연결: 참으로 받아들여진 견해
- 사용 맥락: `religious dogma` = 종교 교리

### dogmatic

- 발음: /dɑˈgmætɪk/
- 구성: `dogma` + `-tic`
- 뜻: 독단적인, 교조적인
- 어원적 연결: 자기 견해를 확정된 진리처럼 고집함
- 사용 맥락: `a dogmatic attitude` = 독단적인 태도

### dogmatism

- 발음: /ˈdɑgməˌtɪzəm/
- 구성: `dogmatic` + `-ism`
- 뜻: 독단주의, 교조주의
- 어원적 연결: 의견을 검토 없이 확정된 교리처럼 붙듦
- 사용 맥락: `avoid dogmatism` = 독단주의를 피하다

### pseudodox

- 구성: `pseudo-` = false + `doxa` = opinion
- 뜻: 거짓 의견의, 잘못된 통념의
- 어원적 연결: 사실이 아닌데 널리 받아들여진 견해
- 사용 맥락: 드문 학술어

## 기억 포인트

- `dox`의 핵심은 “의견, 믿음, 그렇게 보이는 것”.
- `paradox`: 통념에 반대되는 말.
- `orthodox`: 바른 의견, 정통.
- `heterodox`: 다른 의견, 비정통.
- `dogma`: 확정된 의견, 교리.
- `doxology`: 영광을 말함.

## 비교 단어

### logos

- [[logos]]는 말, 설명, 이성.
- `dox`는 말 자체보다 사람들이 받아들이는 의견·믿음에 초점.

### dict

- [[dict]]는 말하다, 선언하다.
- `dogma`와 `orthodoxy`는 말로 선언된 내용보다 “받아들여진 믿음” 쪽.

### doctrine

- `doctrine`, `doctor`, `document`는 Latin `docere` = 가르치다 계열.
- 깊은 PIE 단계에서 가까울 수 있으나, 영어 학습에서는 `doxa/dogma`와 분리.
