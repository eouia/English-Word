# fix

- 출발 단어: `fix`
- 핵심 의미: 박다, 고정하다, 한곳에 붙이다
- 관련 형태: Latin `figere` = fasten, drive in, Latin `fixus` = fixed
- 주의: 현대 영어의 `fix` = 고치다는 "문제를 안정된 상태로 고정함"에서 확장된 의미.

## 핵심 이미지

`fix` 계열의 바탕은 "못을 박아 움직이지 않게 함".

물리적으로 붙이면 `affix`, 앞에 붙이면 `prefix`, 뒤에 붙이면 `suffix`.
상태나 약속을 고정하면 `fix`, `fixed`, `fixture`.
문제를 고치다는 뜻도 원래는 흔들리는 것을 안정된 상태로 만들어 놓는 그림.

## 단어들

### fix

- 구성: Latin `fixus` → English `fix`
- 뜻: 고정하다, 정하다, 수리하다
- 연결: 어떤 것을 한 상태에 박아 놓음
- 사용 맥락: `fix a date` = 날짜를 정하다, `fix a problem` = 문제를 고치다

### fixed

- 구성: `fix` + `-ed`
- 뜻: 고정된, 정해진, 변하지 않는
- 연결: 이미 한 위치나 상태에 박혀 있음
- 사용 맥락: `a fixed price` = 정가, `fixed ideas` = 굳어진 생각

### fixture

- 구성: `fix` + `-ture`
- 뜻: 고정 설비, 늘 그 자리에 있는 사람이나 물건
- 연결: 장소에 붙박이처럼 고정된 것
- 사용 맥락: `lighting fixtures` = 조명 설비

### affix

- 구성: `ad-` = to + `fix`
- 뜻: 붙이다; 접사
- 연결: 어떤 것에 덧붙여 고정함
- 사용 맥락: `affix a label` = 라벨을 붙이다

### prefix

- 구성: `pre-` = before + `fix`
- 뜻: 접두사
- 연결: 단어 앞에 붙어 고정된 요소
- 사용 맥락: `un-` in `unhappy` is a prefix.

### suffix

- 구성: `sub-` = under/after + `fix`
- 뜻: 접미사
- 연결: 단어 뒤쪽에 붙어 고정된 요소
- 사용 맥락: `-ness` in `happiness` is a suffix.

### fixation

- 구성: `fix` + `-ation`
- 뜻: 고정, 집착
- 연결: 생각이 한 대상에 고정됨
- 사용 맥락: `a fixation on success` = 성공에 대한 집착

### fixate

- 구성: `fix` + `-ate`
- 뜻: 고정시키다, 집착하다
- 연결: 시선이나 생각을 한곳에 박아 둠
- 사용 맥락: `Do not fixate on one mistake.` = 한 실수에 매달리지 말라

### infix

- 구성: `in-` = in + `fix`
- 뜻: 단어 안에 끼워 넣는 접사
- 연결: 단어 내부에 붙여 고정함
- 사용 맥락: 언어학 용어. 일반 시험 어휘에서는 `prefix`, `suffix`보다 덜 중요.

### transfix

- 구성: `trans-` = across/through + `fix`
- 뜻: 꼼짝 못 하게 하다, 꿰뚫다
- 연결: 무언가를 관통해 박아 움직이지 못하게 함
- 사용 맥락: `The audience was transfixed.` = 관객이 넋을 잃고 꼼짝 못 했다

### crucifix

- 구성: Latin `crux/cruc-` = cross + `fix`
- 뜻: 십자가상
- 연결: 십자가에 고정된 형상
- 사용 맥락: 문화·종교 문맥에서 주로 사용.

### crucifixion

- 구성: `crucifix` + `-ion`
- 뜻: 십자가형
- 연결: 십자가에 못 박아 고정하는 처형 방식
- 사용 맥락: 역사·종교 문맥.

## 비교 단어

`figure`, `fiction`, `feign`은 비슷하게 `fig-`가 보이지만, 주로 Latin `fingere` = shape, form, invent 계열이다.
`fix`의 `figere` = fasten, drive in 계열과 구분.
