# cresc

- 출발 단어: `concrete`
- 핵심 어원: 라틴어 `crescere` / `cret-` = 자라다, 커지다
- 핵심 이미지: 자라서 커짐, 함께 자라 굳음
- 비교: [[tract]]의 `abstract` = 떨어져 끌어냄

## 출발 단어 분석

`concrete`는 `con-` + `crescere`.

- `con-`: 함께
- `crescere`: 자라다, 커지다

따라서 `concrete`는 "함께 자라 굳은" → "구체적인, 콘크리트"의 뜻.

## 단어들

### concrete

- 구성: `con-` + `crescere`
- 뜻: 구체적인, 콘크리트
- 어원적 연결: 여러 요소가 함께 자라거나 굳어 한 덩어리가 됨
- 사용 맥락: `concrete evidence` = 구체적인 증거

### concretion

- 구성: `concrete` 계열 명사
- 뜻: 응결, 결합체
- 어원적 연결: 함께 굳어 덩어리가 된 것
- 사용 맥락: `mineral concretion` = 광물 응결체

### increase

- 구성: `in-` + `crescere`
- 뜻: 증가하다
- 어원적 연결: 안에서 자라 커짐
- 사용 맥락: `increase in value` = 가치가 증가하다

### decrease

- 구성: `de-` + `crescere`
- 뜻: 감소하다
- 어원적 연결: 성장이나 크기가 아래로 줄어듦
- 사용 맥락: `decrease costs` = 비용을 줄이다

### crescent

- 구성: 라틴어 `crescens` = 자라는
- 뜻: 초승달, 초승달 모양
- 어원적 연결: 점점 커지는 달
- 사용 맥락: `a crescent moon` = 초승달

### crescendo

- 구성: Italian `crescendo` = growing
- 뜻: 점점 커짐, 최고조
- 어원적 연결: 소리나 강도가 점점 자람
- 사용 맥락: `reach a crescendo` = 최고조에 이르다

### accrue

- 구성: Old French `acreue`, Latin `accrescere`
- 뜻: 쌓이다, 발생하다
- 어원적 연결: 시간이 지나며 더해져 자람
- 사용 맥락: `interest accrues` = 이자가 쌓이다

### accretion

- 구성: Latin `accrescere` 계열
- 뜻: 축적, 부착 성장
- 어원적 연결: 조금씩 붙어 자라 커짐
- 사용 맥락: `gradual accretion` = 점진적 축적

### recruit

- 구성: French `recrue`, Latin `recrescere`
- 뜻: 신입, 모집하다
- 어원적 연결: 다시 자라 보충됨
- 사용 맥락: `recruit new employees` = 신입 직원을 모집하다

## 비교 단어

### abstract

- 구성: `ab-` + `tract`
- 뜻: 추상적인, 요약
- 어원적 연결: [[tract]] 계열. 구체적 사례에서 핵심만 떨어져 끌어냄
- 구분 포인트: `concrete` = 함께 굳어 구체화, `abstract` = 떨어져 끌어내 추상화

## 기억 포인트

`cresc` 계열은 "자라서 커짐" 이미지.

- `concrete`: 함께 자라 굳음
- `increase`: 커짐
- `decrease`: 줄어듦
- `crescent`: 커지는 달
- `crescendo`: 점점 커지는 소리/강도
- `accrue`: 쌓여 자람
- `accretion`: 붙어서 커짐
- `recruit`: 다시 채워 자라게 함
