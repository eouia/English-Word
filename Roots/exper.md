# exper

- 출발 단어: `experiment`
- 핵심 어원: Latin `experiri` = try, test
- 핵심 이미지: 직접 시험해 보고 겪어서 알게 됨
- 관련 문서: [[ex]], [[per]]
- 주의: `empirical`은 Greek `empeiria` = experience 계열. Latin `experiri`와 직접 같은 형태는 아니지만 "시험·경험" 이미지가 가까움.

## 출발 단어 분석

`experiment` = `experi-` + `-ment`

- `experi-`: try, test, experience
- `-ment`: 결과·수단·상태를 나타내는 접미사
- 원래 이미지: 무엇이 맞는지 직접 시험해 봄
- 현대 의미: 실험, 시험; 실험하다

## 단어들

### experiment

- 구성: Latin `experimentum` ← `experiri` = try, test
- 뜻: 실험, 시험; 실험하다
- 어원적 연결: 직접 시도해 보고 결과로 판단함
- 사용 맥락: `conduct an experiment` = 실험을 수행하다

### experimental

- 구성: `experiment` + `-al`
- 뜻: 실험의, 실험적인
- 어원적 연결: 아직 확정하지 않고 시험해 보는 성질
- 사용 맥락: `experimental data` = 실험 자료, `an experimental method` = 실험적 방법

### experimentation

- 구성: `experiment` + `-ation`
- 뜻: 실험, 실험 과정
- 어원적 연결: 여러 방식으로 시험해 보는 과정
- 사용 맥락: `scientific experimentation` = 과학적 실험

### experience

- 구성: Old French `experience` ← Latin `experientia` < `experiri`
- 뜻: 경험, 체험; 경험하다
- 어원적 연결: 직접 시험하고 겪어 알게 된 것
- 사용 맥락: `gain experience` = 경험을 쌓다

### experienced

- 구성: `experience` + `-ed`
- 뜻: 경험 많은, 숙련된
- 어원적 연결: 여러 번 겪고 시험해 보아 능숙해진 상태
- 사용 맥락: `an experienced teacher` = 경험 많은 교사

### expert

- 구성: Latin `expertus` = tried, proved, experienced
- 뜻: 전문가, 숙련가; 전문적인
- 어원적 연결: 직접 시험과 경험을 거쳐 검증된 사람
- 사용 맥락: `an expert in economics` = 경제학 전문가

### expertise

- 구성: `expert` + `-ise`
- 뜻: 전문 지식, 전문 기술
- 어원적 연결: 경험으로 검증된 숙련 능력
- 사용 맥락: `technical expertise` = 기술 전문성

### inexpert

- 구성: `in-` = not + `expert`
- 뜻: 미숙한, 서투른
- 어원적 연결: 충분히 시험하고 겪어 숙련된 상태가 아님
- 사용 맥락: `an inexpert attempt` = 서투른 시도

### peril

- 구성: Latin `periculum` = trial, risk, danger
- 뜻: 위험, 위기
- 어원적 연결: 시험해 보는 상황이 곧 위험을 감수하는 일이 됨
- 사용 맥락: `in peril` = 위험에 처한

## 기억 포인트

`exper`는 "직접 시험하고 겪어서 앎" 이미지.

- `experiment`: 시험해 봄
- `experimental`: 실험적인
- `experience`: 겪어 봄
- `experienced`: 겪어서 능숙함
- `expert`: 시험과 경험으로 검증된 사람
- `expertise`: 검증된 전문 능력
- `peril`: 시험·시도가 위험으로 이어진 상태

## 비교 단어

### empirical

- 뜻: 경험적, 실증적인
- 구분 포인트: Greek `empeiria` = experience 계열. 관찰과 경험에 근거한다는 점에서 `experiment`와 함께 외우기 좋음.

### trial

- 뜻: 시도, 시험, 재판
- 구분 포인트: French/Latin `triare` 계열로, `experiri`와 직접 같은 어근은 아니지만 "검증을 위한 시험" 의미가 겹침.
