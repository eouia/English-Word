# dens

- 출발 단어: `dense`
- 핵심 어원: Latin `densus` = thick, crowded
- 핵심 이미지: 빽빽하게 모여 촘촘함
- 관련 문서: [[rar]], [[press]]
- 주의: `dense`는 물리적 밀도뿐 아니라 글·설명·사람의 이해가 "빽빽하거나 둔함"으로도 확장됨.

## 출발 단어 분석

`dense` ← Latin `densus`

- `densus`: 빽빽한, 두꺼운, 촘촘한
- 원래 이미지: 빈틈이 적게 모여 있음
- 현대 의미: 조밀한, 밀도가 높은, 이해하기 어려운

## 단어들

### dense

- 구성: Latin `densus`
- 뜻: 빽빽한, 조밀한, 밀도가 높은; 이해가 느린
- 어원적 연결: 빈틈 없이 촘촘히 모여 있음
- 사용 맥락: `dense forest` = 울창한 숲, `dense text` = 빽빽하고 어려운 글

### densely

- 구성: `dense` + `-ly`
- 뜻: 빽빽하게, 조밀하게
- 어원적 연결: 빈틈이 적게 모인 방식
- 사용 맥락: `densely populated` = 인구가 조밀한

### density

- 구성: `dense` + `-ity`
- 뜻: 밀도, 조밀함
- 어원적 연결: 일정 공간 안에 얼마나 빽빽하게 들어 있는지의 정도
- 사용 맥락: `population density` = 인구 밀도

### condense

- 구성: `con-` = together + `dense`
- 뜻: 응축하다, 압축하다, 요약하다
- 어원적 연결: 함께 빽빽하게 모아 더 작고 진하게 만듦
- 사용 맥락: `condense steam into water` = 수증기를 물로 응축하다, `condense a paragraph` = 문단을 요약하다

### condensed

- 구성: `condense` + `-ed`
- 뜻: 응축된, 압축된, 요약된
- 어원적 연결: 내용이나 물질이 빽빽하게 모아진 상태
- 사용 맥락: `condensed milk` = 연유, `a condensed version` = 요약판

### condensation

- 구성: `condense` + `-ation`
- 뜻: 응축, 결로; 요약
- 어원적 연결: 흩어진 것이 더 조밀한 상태로 모임
- 사용 맥락: `water condensation on a window` = 창문에 맺힌 결로

## 기억 포인트

`dens` 계열은 "빽빽하고 조밀함" 이미지.

- `dense`: 빽빽한, 조밀한
- `densely`: 빽빽하게
- `density`: 밀도
- `condense`: 응축하다, 요약하다
- `condensed`: 응축된, 요약된
- `condensation`: 응축, 결로

## 비교 단어

### rare

- 뜻: 드문, 희박한
- 구분 포인트: [[rar]]는 성기게 흩어져 드문 이미지. `dens`는 빈틈 없이 모여 빽빽한 이미지.

### compress

- 뜻: 압축하다
- 구분 포인트: [[press]]는 눌러서 작게 만드는 동작. `condense`는 흩어진 것의 밀도를 높여 조밀하게 만드는 결과 이미지가 강함.
