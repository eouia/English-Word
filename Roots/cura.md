# cura

- 출발 단어: `curing`
- 핵심 어원: Latin `cura` = 돌봄, 주의, 관심, 걱정
- 동사형: Latin `curare` = 돌보다, 주의를 기울이다, 치료하다
- 핵심 의미: 돌보고 주의하여 처리함
- 관련 문서: [[med]], [[manu]], [[ped]], [[curr]]
- 주의: `cura`는 care 계열. [[curr]] = 달리다, 흐르다 계열과 철자만 비슷함.

## 출발 단어 분석

`curing` = `cure` + `-ing`

- `cure`: 돌보다, 치료하다
- `curing`: 치료함; 식품을 말리거나 소금에 절여 보존 처리함; 재료를 굳힘
- 원래 이미지: 어떤 대상을 돌보고 처리하여 안정된 상태로 만듦

## 단어들

### curing

- 구성: `cure` + `-ing`
- 뜻: 치료, 치유; 보존 처리, 경화
- 어원적 연결: 돌보고 처리하여 상하거나 망가지지 않게 함
- 사용 맥락: `curing meat` = 고기를 염장·건조 처리함

### cure

- 구성: Latin `cura` / `curare`
- 뜻: 치료하다, 치유; 치료법
- 어원적 연결: 아픈 것을 돌보아 건강한 상태로 되돌림
- 사용 맥락: `cure a disease` = 병을 치료하다

### cured

- 구성: `cure` + `-ed`
- 뜻: 치료된; 보존 처리된
- 어원적 연결: 돌봄과 처리를 거쳐 안정된 상태
- 사용 맥락: `cured ham` = 염장·숙성 햄

### curable

- 구성: `cure` + `-able`
- 뜻: 치료 가능한
- 어원적 연결: 돌봄과 치료로 회복될 수 있음
- 사용 맥락: `a curable infection` = 치료 가능한 감염

### incurable

- 구성: `in-` = not + `curable`
- 뜻: 치료할 수 없는
- 어원적 연결: 돌봄과 치료로 고칠 수 없음
- 사용 맥락: `an incurable disease` = 불치병

### curative

- 구성: Latin `curativus` ← `curare`
- 뜻: 치료의, 치료 효과가 있는
- 어원적 연결: 병을 돌보고 낫게 하는 힘
- 사용 맥락: `curative treatment` = 치료 목적의 처치

### curate

- 구성: Medieval Latin `curatus` = one responsible for care
- 뜻: 큐레이션하다, 선별해 관리하다; 부목사
- 어원적 연결: 작품이나 사람들을 맡아 돌보고 관리함
- 사용 맥락: `curate an exhibition` = 전시를 기획·관리하다

### curator

- 구성: Latin `curator` = caretaker, manager
- 뜻: 큐레이터, 관리자
- 어원적 연결: 모음이나 기관을 돌보고 관리하는 사람
- 사용 맥락: `a museum curator` = 박물관 큐레이터

### curation

- 구성: `curate` + `-ion`
- 뜻: 큐레이션, 선별 관리
- 어원적 연결: 자료나 작품을 주의 깊게 돌보고 배열함
- 사용 맥락: `content curation` = 콘텐츠 큐레이션

### curious

- 구성: Latin `curiosus` = careful, inquisitive
- 뜻: 호기심 많은, 궁금한
- 어원적 연결: 어떤 것에 주의를 기울이고 알고 싶어함
- 사용 맥락: `a curious child` = 호기심 많은 아이

### curiosity

- 구성: `curious` + `-ity`
- 뜻: 호기심, 진기한 것
- 어원적 연결: 관심과 주의를 끄는 성질
- 사용 맥락: `satisfy curiosity` = 호기심을 충족하다

### accurate

- 구성: `ad-` = to + Latin `curare` = take care
- 뜻: 정확한
- 어원적 연결: 주의를 기울여 처리한 → 정확한
- 사용 맥락: `accurate measurement` = 정확한 측정

### accuracy

- 구성: `accurate` + `-cy`
- 뜻: 정확성
- 어원적 연결: 세심하게 처리된 정도
- 사용 맥락: `improve accuracy` = 정확성을 높이다

### secure

- 구성: Latin `se-` = without + `cura` = care, anxiety
- 뜻: 안전한, 확실한; 확보하다
- 어원적 연결: 걱정이 없는 상태
- 사용 맥락: `a secure system` = 안전한 시스템

### security

- 구성: `secure` + `-ity`
- 뜻: 안전, 보안, 확실성
- 어원적 연결: 걱정 없이 보호된 상태
- 사용 맥락: `national security` = 국가 안보

### procure

- 구성: `pro-` = for, forward + `curare` = care for
- 뜻: 구하다, 조달하다
- 어원적 연결: 어떤 것을 얻기 위해 돌보고 처리함
- 사용 맥락: `procure supplies` = 물자를 조달하다

### procurement

- 구성: `procure` + `-ment`
- 뜻: 조달, 구매
- 어원적 연결: 필요한 것을 얻기 위해 관리하고 처리하는 과정
- 사용 맥락: `public procurement` = 공공 조달

### manicure

- 구성: [[manu]] = hand + `cura` = care
- 뜻: 손톱 손질, 매니큐어
- 어원적 연결: 손과 손톱을 돌봄
- 사용 맥락: `get a manicure` = 매니큐어를 받다

### pedicure

- 구성: [[ped]] = foot + `cura` = care
- 뜻: 발 관리, 페디큐어
- 어원적 연결: 발과 발톱을 돌봄
- 사용 맥락: `a pedicure treatment` = 발 관리 시술

### sinecure

- 구성: Latin `sine` = without + `cura` = care
- 뜻: 일이 거의 없는 보수 있는 직책
- 어원적 연결: 원래 “영혼 돌봄이 없는 성직록” → 책임은 적고 수입은 있는 자리
- 사용 맥락: `a political sinecure` = 정치적 한직

## 기억 포인트

- `cura`: 돌봄, 주의, 관심.
- `cure`: 돌보아 낫게 함.
- `curing`: 치료 또는 보존 처리.
- `curate`: 맡아 돌보고 선별 관리함.
- `curious`: 관심을 기울여 알고 싶어함.
- `accurate`: 주의 깊게 처리되어 정확함.
- `secure`: 걱정이 없음.

## 비교 단어

### med

- [[med]]는 의학·치료 자체.
- `cure`는 원래 돌봄에서 치료 의미로 확장.

### curr

- [[curr]]는 달리다, 흐르다, 발생하다.
- `cura`는 돌봄과 주의. `current`, `occur`와 어원 무관.

### care

- `care`는 English/Germanic 계열.
- `cura`와 뜻은 비슷하지만 어원은 다름.
