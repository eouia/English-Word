# poly

- 출발 단어: `polychrome`
- 핵심 어원: Greek `polys` = many, much
- 핵심 이미지: 많음, 여러 개, 여러 종류
- 관련 문서: [[mono]], [[bi]], [[tri]], [[chrom]], [[theo]]
- 주의: `poly-`는 Greek 계열. Latin 계열의 "많은" 이미지는 `multi-`가 더 직접적.

## 출발 단어 분석

`polychrome` = `poly` + `chrome`

- `poly`: many
- `chrome`: color
- 원래 이미지: 여러 색이 함께 있음
- 현대 의미: 다색의, 여러 색으로 된

## 단어들

### polychrome

- 구성: Greek `poly` = many + [[chrom]] = color
- 뜻: 다색의, 여러 색의
- 어원적 연결: 여러 색을 함께 가짐
- 사용 맥락: `polychrome pottery` = 다색 도자기

### polygon

- 구성: Greek `poly` = many + `gonia` = angle
- 뜻: 다각형
- 어원적 연결: 각이 여러 개인 도형
- 사용 맥락: `a regular polygon` = 정다각형

### polysyllabic

- 구성: `poly` = many + `syllable`
- 뜻: 여러 음절의, 다음절의
- 어원적 연결: 음절이 여러 개 있음
- 사용 맥락: `a polysyllabic word` = 다음절 단어

### polyglot

- 구성: `poly` = many + Greek `glotta` = tongue, language
- 뜻: 여러 언어를 하는 사람, 다언어의
- 어원적 연결: 여러 언어의 혀를 가짐
- 사용 맥락: `a polyglot community` = 다언어 공동체

### polytheism

- 구성: `poly` = many + [[theo]] = god
- 뜻: 다신론
- 어원적 연결: 여러 신을 인정함
- 사용 맥락: `ancient polytheism` = 고대 다신론

### polygamy

- 구성: `poly` = many + Greek `gamos` = marriage
- 뜻: 다혼제, 일부다처·일처다부
- 어원적 연결: 배우자나 결혼 관계가 여럿임
- 사용 맥락: `monogamy and polygamy` = 일부일처제와 다혼제

### polymorphous

- 구성: `poly` = many + [[morph]] = form
- 뜻: 여러 형태를 가진
- 어원적 연결: 형태가 여러 가지로 나타남
- 사용 맥락: 과학·문학적 표현에서 쓰임

### polymer

- 구성: Greek `poly` = many + `meros` = part
- 뜻: 중합체, 고분자
- 어원적 연결: 많은 작은 부분이 연결된 큰 물질
- 사용 맥락: 화학·재료 맥락

## 기억 포인트

`poly-`는 "많음, 여러 개" 이미지.

- `polychrome`: 여러 색
- `polygon`: 여러 각
- `polyglot`: 여러 언어
- `polytheism`: 여러 신
- `polygamy`: 여러 결혼 관계
- `polymer`: 많은 부분이 연결됨

## 비교 단어

### mono

- 뜻: 하나, 홀로
- 구분 포인트: [[mono]]는 하나. `poly`는 많음.

### bi / tri

- 뜻: 둘 / 셋
- 구분 포인트: [[bi]], [[tri]]는 특정 숫자. `poly`는 정확한 수보다 "여러 개"라는 느낌.

### multi

- 뜻: 많은, 여러
- 구분 포인트: `multi-`는 Latin 계열의 "많은". `poly-`는 Greek 계열의 "많은".
