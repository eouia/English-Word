# solus

- 출발 단어: `solo`
- 핵심 어원: Latin `solus` = alone
- 핵심 이미지: 혼자, 홀로 떨어져 있음, 단독으로 함
- 관련 문서: [[singul]], [[mono]], [[uni]]
- 주의: `solar`의 `sol`은 Latin `sol` = sun 계열이라 `solo/sole/solitary`의 `solus`와 다름.

## 출발 단어 분석

`solo` ← Italian `solo` ← Latin `solus`

- `solus`: alone
- Italian `solo`: alone
- 원래 이미지: 다른 사람 없이 혼자 연주하거나 행동함
- 현대 의미: 독주, 독창, 혼자 하는

## 단어들

### solo

- 발음: /ˈsoʊˌloʊ/
- 구성: Italian `solo` ← Latin `solus` = alone
- 뜻: 독주, 독창, 혼자 하는
- 어원적 연결: 함께하지 않고 혼자 수행함
- 사용 맥락: `a guitar solo` = 기타 독주

### sole

- 발음: /ˈsoʊl/
- 구성: Old French `sol` ← Latin `solus` = alone
- 뜻: 유일한, 단독의
- 어원적 연결: 다른 것이 없이 혼자 하나만 있음
- 사용 맥락: `the sole survivor` = 유일한 생존자

### solely

- 발음: /ˈsoʊəli/
- 구성: `sole` + `-ly`
- 뜻: 오로지, 단독으로
- 어원적 연결: 다른 이유나 대상 없이 하나만
- 사용 맥락: `based solely on evidence` = 오로지 증거에만 근거한

### solitary

- 발음: /ˈsɑləˌtɛri/
- 구성: Latin `solitarius` = alone
- 뜻: 혼자의, 고독한, 외딴
- 어원적 연결: 다른 사람과 떨어져 홀로 있음
- 사용 맥락: `a solitary life` = 고독한 삶

### solitude

- 발음: /ˈsɑləˌtud/
- 구성: Latin `solitudo` = loneliness, being alone
- 뜻: 고독, 혼자 있음
- 어원적 연결: 홀로 있는 상태
- 사용 맥락: `enjoy solitude` = 혼자 있는 시간을 즐기다

### desolate

- 발음: /ˈdɛsələt/
- 구성: Latin `desolare` = leave alone, abandon
- 뜻: 황량한, 쓸쓸한, 버려진
- 어원적 연결: 홀로 남겨져 텅 빈 상태
- 사용 맥락: `a desolate landscape` = 황량한 풍경

### soliloquy

- 발음: /səˈlɪləkwi/
- 구성: Latin `solus` = alone + `loqui` = speak
- 뜻: 독백
- 어원적 연결: 혼자 말함
- 사용 맥락: 문학·연극에서 `Hamlet's soliloquy`

## 기억 포인트

`solus` 계열은 "혼자, 홀로" 이미지.

- `solo`: 혼자 연주함
- `sole`: 유일한, 단 하나뿐인
- `solitary`: 홀로 떨어진
- `solitude`: 혼자 있음
- `desolate`: 홀로 남겨져 황량한
- `soliloquy`: 혼자 말함

## 비교 단어

### singul

- 뜻: 하나씩 따로, 개별
- 구분 포인트: [[singul]]은 개별 단위로 하나씩 셈. `solus`는 함께할 대상 없이 홀로 있음.

### mono

- 뜻: 하나, 홀로
- 구분 포인트: [[mono]]는 Greek `monos` 계열. `monologue`처럼 `mono-` 접두사로 잘 보임.

### uni

- 뜻: 하나, 하나로 묶임
- 구분 포인트: [[uni]]는 Latin `unus` 계열의 "하나로 통합됨". `solus`는 혼자 떨어져 있음.
