# bell

- 출발 단어: `rebellion`
- 핵심 어원: Latin `bellum` = war
- 핵심 이미지: 전쟁, 싸움, 권위에 맞서는 충돌
- 관련 문서: [[batt]]
- 주의: `bell` "종"은 Germanic 계열로 별도 어원. 여기서는 Latin `bellum` 전쟁 계열만 다룸.

## 출발 단어 분석

`rebellion` ← Latin `rebellio` ← `rebellare`

- `re-`: again, back
- `bellum`: 전쟁
- 원래 이미지: 다시 전쟁을 벌임, 권위에 맞서 싸움
- 현대 의미: 반란, 저항, 반항

## 단어들

### rebellion

- 구성: `re-` = again/back + Latin `bellum`
- 뜻: 반란, 저항, 반항
- 어원적 연결: 지배 권위에 맞서 다시 싸움을 일으킴
- 사용 맥락: `a rebellion against authority` = 권위에 대한 반항

### rebel

- 구성: Latin `rebellis` = waging war again
- 뜻: 반란군, 반항자; 반항하다
- 어원적 연결: 권위나 지배에 맞서 싸우는 사람
- 사용 맥락: `rebel against the rules` = 규칙에 반항하다

### rebellious

- 구성: `rebel` + `-ious`
- 뜻: 반항적인, 반란을 일으키는
- 어원적 연결: 권위에 맞서 싸우려는 성향
- 사용 맥락: `a rebellious teenager` = 반항적인 십대

### belligerent

- 구성: Latin `bellum` = war + `gerere` = carry on
- 뜻: 호전적인, 교전 중인
- 어원적 연결: 전쟁을 수행하거나 싸움을 걸려는 태도
- 사용 맥락: `a belligerent attitude` = 호전적인 태도

### bellicose

- 구성: Latin `bellicosus` ← `bellum`
- 뜻: 호전적인, 싸움을 좋아하는
- 어원적 연결: 전쟁을 좋아하거나 싸움에 기울어진 성향
- 사용 맥락: `bellicose rhetoric` = 호전적인 수사

### antebellum

- 구성: `ante-` = before + `bellum`
- 뜻: 전쟁 전의, 특히 미국 남북전쟁 이전의
- 어원적 연결: 전쟁이 일어나기 전 시기
- 사용 맥락: `the antebellum South` = 남북전쟁 이전의 미국 남부

## 기억 포인트

`bell` 계열은 "전쟁과 맞섬" 이미지.

- `rebellion`: 반란, 저항
- `rebel`: 반항자, 반란군
- `rebellious`: 반항적인
- `belligerent`: 호전적인, 교전 중인
- `bellicose`: 싸움을 좋아하는
- `antebellum`: 전쟁 전의

## 비교 단어

### battle

- 뜻: 전투, 싸움
- 구분 포인트: [[batt]]는 치고 맞붙는 전투 이미지. `bell`은 Latin `bellum`에서 온 전쟁·반란의 문어적 계열.

### peace

- 뜻: 평화
- 구분 포인트: `peace`, `pacify`는 평화와 진정 이미지. `bell`은 그 반대인 전쟁과 충돌 이미지.
