# homo

- 출발 단어: `homogeneous`
- 핵심 어원: Greek `homos` = 같은, 동일한
- 핵심 의미: 같음, 같은 종류
- 관련 문서: [[hetero]], [[onym]], [[phone]], [[gen]]
- 주의: Latin `homo` = 사람과 철자가 같지만 어원 다름. `homo sapiens`의 `homo`는 "사람" 계열.

## 출발 단어 분석

`homogeneous` = `homo-` + `genos`

- `homo-`: same
- `genos`: kind, race, type
- 원래 이미지: 같은 종류로 이루어짐
- 의미 확장: 성질이 균일함, 한 종류로 섞임

## 단어들

### homogeneous

- 구성: `homo-` = same + `genos` = kind
- 뜻: 동질적인, 균질한
- 어원적 연결: 같은 종류나 성질로 이루어짐
- 사용 맥락: `a homogeneous population` = 동질적인 인구 집단

### homogeneity

- 구성: `homogeneous` + `-ity`
- 뜻: 동질성, 균질성
- 어원적 연결: 서로 같은 성질을 가진 상태
- 사용 맥락: `social homogeneity` = 사회적 동질성

### homogenize

- 구성: `homo-` + `genos` + `-ize`
- 뜻: 균질화하다, 동질화하다
- 어원적 연결: 서로 다른 것을 같은 상태로 만듦
- 사용 맥락: `homogenized milk` = 균질화 우유

### homonym

- 구성: `homo-` = same + [[onym]] = name
- 뜻: 동형이의어, 이름이나 형태가 같은 말
- 어원적 연결: 같은 이름을 가진 말
- 사용 맥락: `bank` as a homonym = `bank`의 동형이의어

### homophone

- 구성: `homo-` = same + [[phone]] = sound
- 뜻: 동음이의어
- 어원적 연결: 소리가 같은 말
- 사용 맥락: `to`, `too`, `two` are homophones.

### homologous

- 구성: `homo-` = same + Greek `logos` = relation, proportion
- 뜻: 상동의, 대응 관계에 있는
- 어원적 연결: 구조나 기원이 서로 대응됨
- 사용 맥락: `homologous structures` = 상동 기관

### homozygous

- 구성: `homo-` = same + Greek `zygon` = yoke, pair
- 뜻: 동형접합의
- 어원적 연결: 한 쌍의 유전자가 같은 상태
- 사용 맥락: genetics context

### homosexual

- 구성: `homo-` = same + `sexual`
- 뜻: 동성애의
- 어원적 연결: 같은 성에 끌림
- 사용 맥락: 성적 지향을 설명하는 사회·심리 맥락

### homophobia

- 구성: `homo-` = same, homosexual + [[phob]] = fear
- 뜻: 동성애 혐오
- 어원적 연결: 현대어에서는 `homosexual`에 대한 혐오·공포 의미
- 사용 맥락: 사회·인권 맥락

## 기억 포인트

`homo-`는 "같음" 이미지.

- `homogeneous`: 같은 종류로 이루어짐
- `homonym`: 이름이 같음
- `homophone`: 소리가 같음
- `homozygous`: 한 쌍이 같음

## 비교 단어

### hetero

- 뜻: 다른
- 구분 포인트: [[hetero]]는 `heterogeneous`, `heterodox`, `heteronym`처럼 "다름" 이미지.

### Homo sapiens

- 뜻: 현생 인류
- 구분 포인트: 여기서 `Homo`는 Latin `homo` = human. Greek `homo-` = same과 어원 다름.
