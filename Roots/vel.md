# vel

- 출발 단어: `reveal`
- 핵심 어원: 라틴어 `velum` = veil, covering, curtain
- 핵심 이미지: 베일로 덮음 / 베일을 걷어 드러냄
- 관련해서 헷갈릴 수 있는 어근: [[cover]] = 덮다, 가리다 계열, [[clud]] = 닫다, 막다 계열

## 출발 단어 분석

`reveal`은 `re-` + `veal`.

- `re-`: back, away
- `veal`: veil, covering

따라서 `reveal`은 원래 "베일을 뒤로 젖히다"의 뜻. 여기서 "가려진 것을 드러내다" → "밝히다, 공개하다"의 뜻.

## 단어들

### reveal

- 발음: /rɪˈvil/
- 구성: `re-` = back, away + `vel`
- 뜻: 드러내다, 밝히다, 공개하다
- 어원적 연결: 베일을 걷어 숨은 것을 보이게 함
- 사용 맥락: `reveal the truth` = 진실을 밝히다

### revelation

- 발음: /ˌrɛvəˈleɪʃən/
- 구성: `reveal` + `-ation`
- 뜻: 폭로, 드러남, 계시
- 어원적 연결: 가려져 있던 것이 밝혀지는 일
- 사용 맥락: `a shocking revelation` = 충격적인 폭로

### veil

- 발음: /ˈveɪl/
- 구성: Latin `velum` 계열
- 뜻: 베일, 면사포; 가리다
- 어원적 연결: 얼굴이나 대상을 덮어 감춤
- 사용 맥락: `wear a veil` = 베일을 쓰다

### unveil

- 발음: /əˈnveɪl/
- 구성: `un-` = reverse + `veil`
- 뜻: 베일을 벗기다, 공개하다
- 어원적 연결: 덮개를 벗겨 보이게 함
- 사용 맥락: `unveil a new product` = 신제품을 공개하다

### veiled

- 발음: /ˈveɪld/
- 구성: `veil` + `-ed`
- 뜻: 베일에 싸인, 숨겨진, 암시적인
- 어원적 연결: 겉으로 직접 드러나지 않게 가려짐
- 사용 맥락: `a veiled threat` = 은근한 협박

## 기억 포인트

`vel` 계열은 "베일" 이미지.

- `veil`: 덮개, 베일
- `reveal`: 베일을 뒤로 걷음 → 드러내다
- `revelation`: 드러남, 폭로, 계시
- `unveil`: 베일을 벗김 → 공개하다
- `veiled`: 베일에 가려진 → 숨은, 암시적인

## 비교 단어

### discover

- 뜻: 발견하다
- 구분 포인트: [[cover]] 계열. `discover`는 "덮개를 걷어냄", `reveal`은 "베일을 걷어 드러냄". 둘 다 감춰진 것을 보이게 하지만 대표 어근은 다름

### conceal

- 뜻: 숨기다, 감추다
- 구분 포인트: Latin `celare` = hide 계열. `veil`처럼 덮개 이미지는 아니고, 숨겨 보이지 않게 한다는 동작에 가까움
