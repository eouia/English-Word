# myst

- 출발 단어: `mysterious`
- 핵심 의미: 감춰진 것, 비밀, 신비
- 관련 형태: Greek `mysterion` = secret rite, secret thing; `myein` = close the eyes or lips

## 핵심 이미지

`myst` 계열의 바탕은 "닫혀 있어 드러나지 않음".

알 수 없이 감춰진 일은 `mystery`.
감춰져 있어 이해하기 어려우면 `mysterious`.
신비 체험이나 영적 비밀과 관련되면 `mystic`, `mystical`.
이해하기 어렵게 만들면 `mystify`.

## 단어들

### mysterious

- 발음: /mɪˈstɪriəs/
- 구성: `mystery` + `-ous`
- 뜻: 신비로운, 알 수 없는, 수수께끼 같은
- 연결: 비밀처럼 감춰져 분명히 알기 어려움
- 사용 맥락: `a mysterious disappearance` = 의문의 실종

### mystery

- 발음: /ˈmɪstɚi/
- 구성: Greek `mysterion` = secret rite, secret thing
- 뜻: 수수께끼, 미스터리, 비밀
- 연결: 닫혀 있어 아직 드러나지 않은 것
- 사용 맥락: `solve a mystery` = 수수께끼를 풀다

### mysteriously

- 발음: /ˌmɪˈstɪriəsli/
- 구성: `mysterious` + `-ly`
- 뜻: 신비롭게, 이상하게도
- 연결: 이유가 드러나지 않은 방식으로
- 사용 맥락: `mysteriously vanished` = 이상하게 사라졌다

### mystic

- 발음: /ˈmɪstɪk/
- 구성: Greek `mystikos` = secret, mystical
- 뜻: 신비주의자; 신비적인
- 연결: 감춰진 영적 진리나 체험과 관련됨
- 사용 맥락: `a mystic tradition` = 신비주의 전통

### mystical

- 발음: /ˈmɪstɪkəl/
- 구성: `mystic` + `-al`
- 뜻: 신비주의의, 신비로운
- 연결: 보통의 이성이나 감각으로 설명하기 어려운 영적 느낌
- 사용 맥락: `a mystical experience` = 신비 체험

### mysticism

- 발음: /ˈmɪstɪˌsɪzəm/
- 구성: `mystic` + `-ism`
- 뜻: 신비주의
- 연결: 감춰진 영적 진리와 직접 체험을 중시하는 사상
- 사용 맥락: religion, philosophy 문맥.

### mystify

- 발음: /ˈmɪstəˌfaɪ/
- 구성: `mystery/mystic` + `-fy`
- 뜻: 어리둥절하게 하다, 신비화하다
- 연결: 이해하기 어렵게 감춰 버림
- 사용 맥락: `The result mystified researchers.` = 그 결과는 연구자들을 어리둥절하게 했다

### mystification

- 구성: `mystify` + `-ication`
- 뜻: 혼란스럽게 함, 신비화
- 연결: 분명한 것을 감춰 이해하기 어렵게 만드는 일
- 사용 맥락: academic / critical writing에서 보임.

## 비교 단어

`secret`은 Latin `secernere` = separate 계열로, 따로 떼어 감춘 것이 중심.
`mystery`는 닫혀 있거나 비밀 의식처럼 감춰져 이해하기 어려운 느낌이 강함.

`obscure`는 어둡고 흐려서 안 보이는 이미지.
`mysterious`는 단순히 어두운 것보다 이유나 정체가 감춰진 느낌.
