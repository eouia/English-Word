# carn

- 출발 단어: `incarnate`
- 핵심 어원: 라틴어 `caro` / `carn-` = 살, 고기, 육체
- 핵심 이미지: 추상적인 것이 살과 몸을 얻음. 또는 살/고기를 먹음
- 관련해서 헷갈릴 수 있는 어근: [[body]], [[corp]]

## 출발 단어 분석

`incarnate`는 `in-` + `carn` + `-ate`.

- `in-`: 안으로, ~이 되게
- `carn`: 살, 육체
- `-ate`: 동사/형용사 형성

따라서 `incarnate`는 "살 안으로 들어가다" → "육체를 입다, 구체적 모습으로 나타나다"의 뜻.

## 단어들

### incarnate

- 구성: `in-` = into + `carn` + `-ate`
- 뜻: 육체를 입은, 구현된, 화신의
- 어원적 연결: 신성, 정신, 성질 같은 추상적인 것이 살과 몸을 얻어 나타남
- 사용 맥락: `evil incarnate` = 악의 화신

### incarnation

- 구성: `incarnate` + `-ion`
- 뜻: 화신, 구현, 육체화
- 어원적 연결: 추상적 존재나 성질이 육체를 얻어 나타난 상태
- 사용 맥락: `the latest incarnation of the product` = 그 제품의 최신 형태

### reincarnation

- 구성: `re-` = again + `incarnation`
- 뜻: 환생, 재탄생
- 어원적 연결: 다시 육체를 얻어 태어남
- 사용 맥락: `believe in reincarnation` = 환생을 믿다

### carnivore

- 구성: `carn` = flesh + `vorare` = eat
- 뜻: 육식동물
- 어원적 연결: 살/고기를 먹는 존재
- 사용 맥락: `large carnivores` = 대형 육식동물

### carnivorous

- 구성: `carnivore` + `-ous`
- 뜻: 육식성의
- 어원적 연결: 고기를 먹는 성질
- 사용 맥락: `carnivorous plants` = 식충식물

### carnal

- 구성: Latin `carnalis`
- 뜻: 육체의, 세속적인, 육욕적인
- 어원적 연결: 정신보다 살과 몸의 욕망에 가까움
- 사용 맥락: `carnal desires` = 육체적 욕망

### carnival

- 구성: Italian `carnevale`, 어원은 논쟁적이나 전통적으로 `carne` = meat와 연결됨
- 뜻: 축제, 카니발
- 어원적 연결: 전통적으로 금육 기간 전 고기와 즐거움을 누리는 축제와 연결
- 사용 맥락: `a street carnival` = 거리 축제

### chili con carne

- 구성: Spanish `con carne` = with meat
- 뜻: 고기 넣은 칠리 요리
- 어원적 연결: `carne`가 "고기"를 직접 뜻함
- 사용 맥락: `order chili con carne` = 칠리 콘 카르네를 주문하다

## 기억 포인트

`carn` 계열은 "살 / 고기 / 육체" 이미지.

- `incarnate`: 살과 몸을 얻어 나타남
- `incarnation`: 육체화, 구현
- `reincarnation`: 다시 육체를 얻음
- `carnivore`: 고기를 먹는 동물
- `carnivorous`: 육식성의
- `carnal`: 육체적, 세속적

## 비교 단어

### embody

- 구성: `em-` + `body`
- 뜻: 구현하다, 체현하다
- 구분 포인트: [[body]]의 `embody`는 영어 `body` 계열. `incarnate`는 라틴어 `carn` = 살, 육체 계열

### corporeal

- 구성: Latin `corpus` / `corpor-`
- 뜻: 육체의, 물질적인
- 구분 포인트: [[corp]]는 "몸 전체, 한 덩어리". `carn`은 더 직접적으로 "살, 고기" 이미지
