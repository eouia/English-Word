# tenu

- 출발 단어: `tenuous`
- 핵심 어원: Latin `tenuis` = thin, fine, slender
- 핵심 이미지: 가늘고 얇아 힘이 약함
- 관련 문서: [[subtil]]
- 주의: `tenuous`는 [[ten]] = 잡다, 유지하다 계열이나 [[tend]] = 뻗다 계열과 다름.

## 출발 단어 분석

`tenuous` ← Latin `tenuis`

- `tenuis`: thin, fine, slender
- `-ous`: 성질을 나타내는 형용사 접미사
- 현대 의미: 희박한, 빈약한, 근거가 약한, 연결이 약한

의미 변화는 "가늘고 얇음" → "밀도나 힘이 약함" → "근거·연결·관계가 빈약함".

## 단어들

### tenuous

- 발음: /ˈtɛnjəwəs/
- 구성: Latin `tenuis` = thin, fine, slender
- 뜻: 희박한, 빈약한, 근거가 약한
- 어원적 연결: 연결이나 근거가 가느다란 실처럼 약함
- 사용 맥락: `a tenuous link` = 약한 연결, `a tenuous argument` = 근거가 빈약한 논증

### tenuously

- 구성: `tenuous` + `-ly`
- 뜻: 희박하게, 약하게, 아슬아슬하게
- 어원적 연결: 가늘고 약하게 이어지는 방식
- 사용 맥락: `tenuously connected` = 약하게 연결된

### tenuousness

- 구성: `tenuous` + `-ness`
- 뜻: 희박함, 빈약함, 약한 연결성
- 어원적 연결: 얇고 약한 성질
- 사용 맥락: `the tenuousness of the evidence` = 증거의 빈약함

### attenuate

- 발음: /əˈtɛnjuˌeɪt/
- 구성: `ad-` = to + Latin `tenuis` = thin
- 뜻: 약화시키다, 줄이다, 묽게 하다
- 어원적 연결: 더 얇고 약하게 만듦
- 사용 맥락: `attenuate the signal` = 신호를 약화시키다

### attenuation

- 발음: /əˌtɛnjuˈeɪʃən/
- 구성: `attenuate` + `-ion`
- 뜻: 약화, 감쇠
- 어원적 연결: 힘이나 밀도가 얇아짐
- 사용 맥락: `signal attenuation` = 신호 감쇠

### extenuate

- 발음: /ɪˈkstɛnjuˌeɪt/
- 구성: `ex-` = out + Latin `tenuis` = thin
- 뜻: 죄나 잘못을 가볍게 보이게 하다, 참작하다
- 어원적 연결: 잘못의 무게를 얇게 줄여 보이게 함
- 사용 맥락: 보통 `extenuating circumstances` 형태로 많이 봄

### extenuating

- 발음: /ɪˈkstɛnjuˌeɪtɪŋ/
- 구성: `extenuate` + `-ing`
- 뜻: 정상 참작이 되는, 죄를 가볍게 하는
- 어원적 연결: 책임이나 잘못의 무게를 줄여 주는
- 사용 맥락: `extenuating circumstances` = 정상 참작 사유

## 기억 포인트

`tenu` 계열은 "얇고 가늘어 힘이 약함" 이미지.

- `tenuous`: 희박한, 근거가 약한
- `tenuously`: 약하게, 희박하게
- `attenuate`: 약화시키다
- `attenuation`: 약화, 감쇠
- `extenuating`: 정상 참작이 되는

## 비교 단어

### subtle

- 뜻: 미묘한, 섬세한
- 구분 포인트: [[subtil]]은 정교하고 미세해서 알아차리기 어려움. `tenuous`는 얇고 약해서 근거나 연결이 빈약함.

### weak

- 뜻: 약한
- 구분 포인트: `weak`는 가장 넓은 약함. `tenuous`는 특히 연결, 주장, 근거, 관계가 가느다랗고 불안정한 느낌.

### rarefied

- 뜻: 희박한, 고도로 세련된
- 구분 포인트: `rarefied`는 공기나 분위기가 얇고 희박함. `tenuous`는 물리적 희박함보다 논리적 연결·근거의 약함에 자주 씀.
