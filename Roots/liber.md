# liber

- 출발 단어: `liberty`
- 핵심 어원: Latin `liber` = 자유로운
- 핵심 이미지: 속박에서 풀려 자유로움
- 관련 문서: [[solv]], [[libr]]
- 주의: [[libr]]는 저울과 균형 계열. `liber`는 자유 계열.

## 출발 단어 분석

`liberty` ← Latin `liber`

- `liber`: free
- `libertas`: freedom
- 원래 이미지: 속박되거나 예속되지 않은 상태
- 현대 의미: 자유, 권리

## 단어들

### liberty

- 발음: /ˈlɪbɚˌti/
- 구성: Latin `libertas`
- 뜻: 자유, 자유권
- 어원적 연결: 구속에서 벗어난 상태
- 사용 맥락: `civil liberties` = 시민적 자유권

### liberal

- 발음: /ˈlɪˌbɝəl/
- 구성: Latin `liberalis` = befitting a free person
- 뜻: 자유주의의; 관대한
- 어원적 연결: 자유인에게 어울리는 태도에서 확장
- 사용 맥락: `liberal democracy` = 자유민주주의
- 주의: 정치 맥락의 의미는 국가와 시대에 따라 달라짐.

### liberate

- 발음: /ˈlɪˌbɝˌeɪt/
- 구성: `liber` + `-ate`
- 뜻: 해방하다, 자유롭게 하다
- 어원적 연결: 속박에서 풀어 줌
- 사용 맥락: `liberate a city` = 도시를 해방하다

### liberation

- 발음: /ˌlɪˌbɝˈeɪʃən/
- 구성: `liberate` + `-ion`
- 뜻: 해방, 자유화
- 어원적 연결: 묶인 상태에서 풀려나는 과정
- 사용 맥락: `national liberation` = 민족 해방

### libertarian

- 발음: /ˌlɪˌbɝˈtɛˌriən/
- 구성: `liberty` + `-arian`
- 뜻: 자유지상주의자; 자유의지를 믿는 사람
- 어원적 연결: 개인의 자유를 중심 가치로 둠
- 사용 맥락: `a libertarian view of government` = 정부에 대한 자유지상주의적 관점

## 기억 포인트

`liber` 계열은 "자유와 해방" 이미지.

- `liberty`: 자유
- `liberal`: 자유주의의, 관대한
- `liberate`: 해방하다
- `liberation`: 해방
- `libertarian`: 개인 자유를 중시하는 사람

## 비교 단어

### libr

- 뜻: 저울, 균형
- 구분 포인트: [[libr]]의 `deliberate`, `equilibrium`은 무게를 달아 균형을 맞춤. `liber`는 속박에서 풀린 자유.
