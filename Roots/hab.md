# hab

- 출발 단어: `exhibition`
- 핵심 어원: Latin `habere` = have, hold
- 변형: `hibit` = `habere`가 접두사 뒤에서 변한 결합 형태
- 핵심 이미지: 가지고 있음, 붙잡고 있음, 밖으로 내보임
- 관련 문서: [[cap]], [[ten]], [[em]]
- 주의: `habit`은 "가지고 있는 상태"에서 습관·거주로 확장. `exhibit/inhibit/prohibit`의 `-hibit`도 같은 `habere` 계열.

## 출발 단어 분석

`exhibition` = Latin `exhibitionem` ← `exhibere`

- `ex-`: out
- `habere`: hold
- `exhibere`: 밖으로 들고 내보이다, 제시하다
- 현대 의미: 전시, 전람회, 보여 줌

## 단어들

### exhibition

- 구성: `ex-` = out + `hibit` = hold
- 뜻: 전시, 전람회, 보여 줌
- 어원적 연결: 밖으로 들고 나와 보여 줌
- 사용 맥락: `art exhibition` = 미술 전시

### exhibit

- 구성: Latin `exhibere`
- 뜻: 전시하다, 보이다; 전시품
- 어원적 연결: 밖으로 내밀어 보여 줌
- 사용 맥락: `exhibit evidence` = 증거를 제시하다

### exhibitor

- 구성: `exhibit` + `-or`
- 뜻: 전시자, 출품자
- 어원적 연결: 무언가를 밖으로 내보이는 사람
- 사용 맥락: trade fair 맥락

### inhibit

- 구성: `in-` = in, on + `hibit`
- 뜻: 억제하다, 방해하다
- 어원적 연결: 안에 붙잡아 두어 나오지 못하게 함
- 사용 맥락: `inhibit growth` = 성장을 억제하다

### inhibition

- 구성: `inhibit` + `-ion`
- 뜻: 억제, 금지; 거리낌
- 어원적 연결: 안에 붙잡아 두는 작용
- 사용 맥락: `social inhibition` = 사회적 억제

### prohibitive

- 구성: `prohibit` + `-ive`
- 뜻: 금지하는; 너무 비싸서 못 하게 하는
- 어원적 연결: 앞에서 막아 못 하게 함
- 사용 맥락: `prohibitive cost` = 감당하기 어려운 비용

### prohibit

- 구성: `pro-` = before, forth + `hibit`
- 뜻: 금지하다
- 어원적 연결: 앞에서 붙잡아 막음
- 사용 맥락: `prohibit smoking` = 흡연을 금지하다

### prohibition

- 구성: `prohibit` + `-ion`
- 뜻: 금지, 금주법
- 어원적 연결: 공식적으로 막아 붙잡음
- 사용 맥락: `alcohol prohibition` = 금주법

### habit

- 구성: Latin `habitus` ← `habere`
- 뜻: 습관, 버릇
- 어원적 연결: 사람이 계속 가지고 있는 상태나 태도
- 사용 맥락: `form a habit` = 습관을 형성하다

### habitual

- 구성: `habit` + `-ual`
- 뜻: 습관적인, 상습적인
- 어원적 연결: 늘 가지고 있는 행동 방식
- 사용 맥락: `habitual behavior` = 습관적 행동

### habituate

- 구성: `habit` + `-uate`
- 뜻: 익숙하게 하다, 습관화하다
- 어원적 연결: 반복해서 가진 상태로 만듦
- 사용 맥락: psychology 맥락

### habitat

- 구성: Latin `habitare` = dwell
- 뜻: 서식지, 거주지
- 어원적 연결: 생물이 머물러 가지고 있는 장소
- 사용 맥락: `natural habitat` = 자연 서식지

### inhabit

- 구성: `in-` = in + `habitare`
- 뜻: 거주하다, 살다
- 어원적 연결: 안에 살며 장소를 차지함
- 사용 맥락: `inhabit an island` = 섬에 살다

### inhabitant

- 구성: `inhabit` + `-ant`
- 뜻: 거주자, 주민
- 어원적 연결: 그 장소 안에 사는 사람
- 사용 맥락: `city inhabitants` = 도시 주민

### habitable

- 구성: `habitare` + `-able`
- 뜻: 살 수 있는, 거주 가능한
- 어원적 연결: 머물러 살 수 있음
- 사용 맥락: `a habitable planet` = 거주 가능한 행성

### ability

- 구성: Latin `habilis` = easily handled, fit
- 뜻: 능력
- 어원적 연결: 손에 쥐고 다룰 수 있음 → 할 수 있음
- 사용 맥락: `cognitive ability` = 인지 능력

### able

- 구성: Old French `able/hable` ← Latin `habilis`
- 뜻: 할 수 있는, 유능한
- 어원적 연결: 잡고 다루기 적합함
- 사용 맥락: `be able to` = 할 수 있다

## 기억 포인트

`hab/hibit` 계열은 "가지고 붙잡음" 이미지.

- `exhibit`: 밖으로 들고 보여 줌
- `inhibit`: 안에 붙잡아 억제
- `prohibit`: 앞에서 붙잡아 금지
- `habit`: 계속 가지고 있는 행동 방식
- `habitat`: 머물러 사는 장소
- `ability`: 다룰 수 있는 힘

## 비교 단어

### cap

- 뜻: 잡다, 취하다
- 구분 포인트: [[cap]]은 잡아 취하는 동작. `hab`는 가지고 있거나 붙잡아 둔 상태.

### ten

- 뜻: 잡다, 유지하다
- 구분 포인트: [[ten]]은 붙잡아 유지함. `hab`는 소유·상태·거주로 확장됨.

### em

- 뜻: 안에 넣다, 어떤 상태로 만들다
- 연결 포인트: `enable`은 `en-` + `able` 복합어. 본문 항목은 [[em]]에서 정리.
