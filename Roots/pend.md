# pend

- 출발 단어: `suspend`
- 핵심 어원: 라틴어 `pendere` / `pens-` = 매달다, 매달리다, 무게를 달다, 지불하다
- 핵심 이미지: 위에 매달려 있거나, 저울에 달아 값을 정함
- 관련해서 헷갈릴 수 있는 어근: [[tend]] = 뻗다, 향하다 계열, [[yield]] = 내놓다, 지불하다 계열

## 출발 단어 분석

`suspend`는 `sub-` + `pendere`.

- `sub-`: under
- `pendere`: hang

따라서 `suspend`는 "아래에 매달다" → "일시 중지하다, 정직시키다, 매달다"의 뜻.

## 단어들

### suspend

- 구성: `sub-` = under + `pend`
- 뜻: 매달다, 중지하다, 정직시키다
- 어원적 연결: 어떤 것을 공중에 매달아 아직 내려놓지 않음
- 사용 맥락: `suspend the project` = 프로젝트를 중단하다

### suspension

- 구성: `sub-` + `pens` + `-ion`
- 뜻: 정지, 중단, 정직, 현가 장치
- 어원적 연결: 매달린 상태 또는 결정이 보류된 상태
- 사용 맥락: `temporary suspension` = 일시 정지

### depend

- 구성: `de-` = down + `pend`
- 뜻: 의존하다, 달려 있다
- 어원적 연결: 아래로 매달려 기대어 있음
- 사용 맥락: `depend on evidence` = 증거에 의존하다

### dependent

- 구성: `depend` + `-ent`
- 뜻: 의존하는, 부양가족
- 어원적 연결: 다른 것에 매달려 있음
- 사용 맥락: `financially dependent` = 경제적으로 의존하는

### independent

- 구성: `in-` = not + `dependent`
- 뜻: 독립적인
- 어원적 연결: 다른 것에 매달려 있지 않음
- 사용 맥락: `independent research` = 독립적 연구

### pending

- 구성: `pend` + `-ing`
- 뜻: 미결인, 보류 중인
- 어원적 연결: 결정이 아직 매달린 상태
- 사용 맥락: `a pending decision` = 보류 중인 결정

### impending

- 구성: `im-` = over, upon + `pend`
- 뜻: 임박한, 곧 닥칠
- 어원적 연결: 머리 위에 매달려 곧 떨어지거나 닥칠 듯함
- 사용 맥락: `an impending crisis` = 임박한 위기, `impending danger` = 다가오는 위험

### impend

- 구성: Latin `impendere` = hang over
- 뜻: 임박하다, 곧 닥치다
- 어원적 연결: 위에 매달려 드리운 상태에서 사건이 가까이 다가옴
- 사용 맥락: 드문 동사. 현대 영어에서는 보통 `impending` 형태가 더 자주 보임.

### pendulum

- 구성: Latin `pendulus` = hanging
- 뜻: 진자, 추
- 어원적 연결: 매달려 흔들리는 물체
- 사용 맥락: `a swinging pendulum` = 흔들리는 진자

### pendent

- 구성: Latin `pendens` = hanging
- 뜻: 매달린, 늘어진
- 어원적 연결: 아래로 매달려 있는 상태
- 사용 맥락: `pendent lights` = 매달린 조명

### perpendicular

- 구성: Latin `perpendiculum` = plumb line
- 뜻: 수직의, 직각을 이루는
- 어원적 연결: 추를 매달아 수직선을 재는 도구에서 온 말. "매달린 선"이 곧 수직 기준이 됨
- 사용 맥락: `a perpendicular line` = 수직선

### append

- 구성: `ad-` = to + `pend`
- 뜻: 덧붙이다, 첨부하다
- 어원적 연결: 끝에 매달아 붙임
- 사용 맥락: `append a note` = 주석을 덧붙이다

### appendix

- 구성: Latin `appendix` = something attached
- 뜻: 부록, 맹장
- 어원적 연결: 본문이나 장기에 매달려 붙은 것
- 사용 맥락: `an appendix to the report` = 보고서의 부록

### expend

- 구성: `ex-` = out + `pend`
- 뜻: 쓰다, 소비하다
- 어원적 연결: 저울에 달아 밖으로 지불함
- 사용 맥락: `expend energy` = 에너지를 쓰다

### spend

- 구성: Old French `despendre` / Latin `expendere`
- 뜻: 쓰다, 소비하다, 돈을 지출하다
- 어원적 연결: `expend`와 같은 "달아 보고 밖으로 지불함" 계열. 현대 형태에서는 앞부분이 줄어든 단어로 보면 좋음
- 사용 맥락: `spend money` = 돈을 쓰다

### expense

- 구성: `ex-` = out + `pens`
- 뜻: 비용, 지출
- 어원적 연결: 밖으로 달아 내준 값
- 사용 맥락: `travel expenses` = 여행 경비

### expensive

- 구성: `expense` + `-ive`
- 뜻: 비싼
- 어원적 연결: 많은 값을 지불해야 함
- 사용 맥락: `an expensive mistake` = 값비싼 실수

### inexpensive

- 구성: `in-` = not + `expensive`
- 뜻: 비싸지 않은, 저렴한
- 어원적 연결: 많이 지불해야 하는 상태가 아님
- 사용 맥락: `an inexpensive option` = 저렴한 선택지

### inexpensively

- 구성: `inexpensive` + `-ly`
- 뜻: 저렴하게
- 어원적 연결: 많은 비용을 들이지 않는 방식
- 사용 맥락: `travel inexpensively` = 저렴하게 여행하다

### dispense

- 구성: `dis-` = apart + `pens`
- 뜻: 나누어 주다, 분배하다; 면제하다
- 어원적 연결: 저울에 달아 몫을 나누어 지급함
- 사용 맥락: `dispense medicine` = 약을 조제해 주다, `dispense with formalities` = 형식을 생략하다

### dispensable

- 구성: `dispense` + `-able`
- 뜻: 없어도 되는, 생략 가능한
- 어원적 연결: 필요에서 떼어 내거나 면제할 수 있음
- 사용 맥락: `a dispensable luxury` = 없어도 되는 사치품

### indispensable

- 구성: `in-` = not + `dispensable`
- 뜻: 필수적인, 없어서는 안 되는
- 어원적 연결: 떼어 내거나 면제할 수 없을 만큼 필요함
- 사용 맥락: `an indispensable tool` = 없어서는 안 되는 도구, `indispensable to success` = 성공에 필수적인

### dispensation

- 구성: `dispense` + `-ation`
- 뜻: 분배, 배분; 특별 면제
- 어원적 연결: 권한에 따라 몫을 나누거나 의무를 면제함
- 사용 맥락: `a special dispensation` = 특별 면제

### compensate

- 구성: `com-` = together + `pens`
- 뜻: 보상하다, 보완하다
- 어원적 연결: 저울 양쪽의 무게를 맞추듯 값을 채워 줌
- 사용 맥락: `compensate for losses` = 손실을 보상하다

### compensation

- 구성: `compensate` + `-ion`
- 뜻: 보상, 보수
- 어원적 연결: 균형을 맞추기 위해 치르는 값
- 사용 맥락: `worker compensation` = 근로자 보상

### pension

- 구성: Latin `pensio` = payment
- 뜻: 연금
- 어원적 연결: 정기적으로 지불되는 돈
- 사용 맥락: `retirement pension` = 퇴직 연금

## 기억 포인트

`pend` 계열은 두 이미지가 함께 있음.

- 매달림: `suspend`, `depend`, `pending`, `pendulum`, `pendent`, `append`
- 임박함: `impending`, `impend`
- 수직 기준: `perpendicular`
- 무게를 달아 지불함: `expend`, `spend`, `expense`, `expensive`, `inexpensive`, `dispense`, `indispensable`, `compensate`, `pension`

## 비교 단어

### tend

- 뜻: 뻗다, 향하다
- 구분 포인트: [[tend]]의 `extend`, `tension`은 뻗고 팽팽해짐. `pend`는 매달림 또는 값을 달아 지불함

### yield

- 뜻: 내놓다, 산출하다, 양보하다
- 구분 포인트: [[yield]]는 값을 내놓거나 결과를 산출함. `pend/pens`의 `expense`, `compensate`는 저울에 달아 값을 치르는 이미지

### imminent

- 뜻: 임박한, 곧 닥칠
- 구분 포인트: [[minere]] 계열. `imminent`는 위로 드리워져 다가오는 이미지, `impending`은 위에 매달려 곧 닥칠 듯한 이미지.
