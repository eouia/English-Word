# via

- 출발 단어: `trivial`
- 핵심 어원: 라틴어 `via` = 길, 도로
- 핵심 이미지: 길을 따라 가거나, 길 위에 놓임
- 관련해서 헷갈릴 수 있는 어근: [[tri]] = 셋, [[trib]] = 부족/배정 계열

## 출발 단어 분석

`trivial`은 라틴어 `trivium`에서 옴.

- `tri-`: three
- `via`: road, way
- `trivium`: 세 길이 만나는 곳

사람들이 많이 오가는 길목의 흔한 이야기나 지식 → "흔한, 평범한" → "사소한"의 뜻.

## 단어들

### trivial

- 발음: /ˈtrɪviəl/
- 구성: `tri-` = three + `via` = road
- 뜻: 사소한, 하찮은
- 어원적 연결: 세 길이 만나는 흔한 장소의 이야기처럼 특별하지 않음
- 사용 맥락: `a trivial mistake` = 사소한 실수

### trivia

- 발음: /ˈtrɪviə/
- 구성: Latin `trivium`
- 뜻: 잡학, 사소한 정보
- 어원적 연결: 길목에서 흔히 접하는 자잘한 지식
- 사용 맥락: `movie trivia` = 영화 잡학

### trivium

- 구성: `tri-` + `via`
- 뜻: 세 길이 만나는 곳, 중세 기초 3학
- 어원적 연결: 원래 세 길이 만나는 곳. 이후 문법, 수사학, 논리학의 기초 3과목을 가리킴
- 사용 맥락: `the medieval trivium` = 중세의 3학

### via

- 발음: /ˈvaɪə/
- 구성: Latin `via`
- 뜻: ~을 경유하여, ~을 통해
- 어원적 연결: 어떤 길을 통해 감
- 사용 맥락: `travel via Paris` = 파리를 경유해 여행하다

### viaduct

- 발음: /ˈvaɪədəkt/
- 구성: `via` = road + `duct` = lead
- 뜻: 고가교, 육교
- 어원적 연결: 길을 이끌어 지나가게 하는 구조물
- 사용 맥락: `a railway viaduct` = 철도 고가교

### deviate

- 발음: /ˈdiviˌeɪt/
- 구성: `de-` = away + `via`
- 뜻: 벗어나다, 일탈하다
- 어원적 연결: 길에서 벗어남
- 사용 맥락: `deviate from the plan` = 계획에서 벗어나다

### deviation

- 발음: /ˌdiviˈeɪʃən/
- 구성: `deviate` + `-ion`
- 뜻: 일탈, 편차
- 어원적 연결: 정해진 길이나 기준에서 벗어난 정도
- 사용 맥락: `standard deviation` = 표준 편차

### devious

- 발음: /ˈdiviəs/
- 구성: `de-` = away + `via`
- 뜻: 교활한, 우회적인
- 어원적 연결: 곧은 길에서 벗어나 돌아가는 성질. 이후 속임수 있는 태도
- 사용 맥락: `a devious plan` = 교활한 계획

### obvious

- 발음: /ˈɑbviəs/
- 구성: `ob-` = toward, in front of + `via`
- 뜻: 명백한, 뻔한
- 어원적 연결: 길 앞에 놓여 누구나 볼 수 있음
- 사용 맥락: `an obvious answer` = 뻔한 답

### previous

- 발음: /ˈpriviəs/
- 구성: `pre-` = before + `via`
- 뜻: 이전의
- 어원적 연결: 길에서 앞서 간 것
- 사용 맥락: `the previous chapter` = 이전 장

### previously

- 발음: /ˈpriviəsli/
- 구성: `previous` + `-ly`
- 뜻: 이전에, 앞서
- 어원적 연결: 시간의 길에서 앞서 있었던 때
- 사용 맥락: `as previously mentioned` = 앞서 언급했듯이

### obviate

- 발음: /ˈɑbviˌeɪt/
- 구성: `ob-` = against, in the way + `via`
- 뜻: 없애다, 방지하다
- 어원적 연결: 길 앞에서 막아 필요 없게 만듦
- 사용 맥락: `obviate the need for repairs` = 수리 필요를 없애다

### voyage

- 발음: /ˈvɔɪədʒ/
- 구성: Old French `veiage`, Latin `viaticum` 계열
- 뜻: 여행, 항해
- 어원적 연결: 길을 따라 떠나는 여행
- 사용 맥락: `a long sea voyage` = 긴 항해

## 기억 포인트

`via` 계열은 "길" 이미지.

- `trivial`: 길목의 흔한 것처럼 사소함
- `trivia`: 자잘한 잡학
- `via`: ~을 통해, 경유하여
- `viaduct`: 길을 지나가게 하는 구조물
- `deviate`: 길에서 벗어남
- `obvious`: 길 앞에 놓여 분명함
- `previous`: 길에서 앞서 간
- `previously`: 시간상 앞서
- `voyage`: 길을 따라 떠나는 여행

## 비교 단어

### trib

- 뜻: 부족, 배정
- 구분 포인트: [[trib]]는 `tribus` / `tribuere` 계열. `trivial`의 `tri-via`와 직접 묶지 않음

### tri-

- 뜻: 셋
- 구분 포인트: `trivial`에는 `tri-`가 들어가지만, 핵심 의미 변화는 `via` = 길에서 나옴

### preview

- 뜻: 미리 봄
- 구분 포인트: [[vid]]의 `preview`는 `pre-` + `view`, "앞서 봄". `previous`는 `pre-` + `via`, "길에서 앞서 간"
