# astr

- 출발 단어: `astronomer`
- 핵심 의미: 별, 천체
- 관련 형태: Greek `astron`, `aster` = star
- 관련 문서: [[sider]], [[logos]]

## 핵심 이미지

`astr` 계열의 바탕은 "별".

별과 천체의 법칙을 연구하면 `astronomy`.
그 연구자는 `astronomer`.
별을 항해하듯 우주로 가는 사람은 `astronaut`.
별점을 보는 것은 `astrology`.
작은 별표는 `asterisk`.
별처럼 생긴 천체는 `asteroid`.

## 단어들

### astronomer

- 발음: /əˈstrɑnəmɚ/
- 구성: Greek `astron` = star + `nomos/nomia` = law, arrangement
- 뜻: 천문학자
- 연결: 별과 천체의 질서와 법칙을 연구하는 사람
- 사용 맥락: `an astronomer observes planets` = 천문학자가 행성을 관찰한다

### astronomy

- 발음: /əˈstrɑnəmi/
- 구성: `astro-` = star + Greek `nomia` = arrangement, law
- 뜻: 천문학
- 연결: 별과 천체의 질서를 연구하는 학문
- 사용 맥락: `study astronomy` = 천문학을 공부하다

### astronomical

- 발음: /ˌæstrəˈnɑmɪkəl/
- 구성: `astronomy` + `-ical`
- 뜻: 천문학의; 천문학적으로 큰
- 연결: 천체와 관련되거나, 우주 규모처럼 매우 큼
- 사용 맥락: `astronomical costs` = 엄청난 비용

### astronaut

- 발음: /ˈæstrəˌnɑt/
- 구성: `astro-` = star + Greek `nautes` = sailor
- 뜻: 우주비행사
- 연결: 별들 사이를 항해하는 사람
- 사용 맥락: `train as an astronaut` = 우주비행사로 훈련하다

### astrology

- 발음: /əˈstrɑlədʒi/
- 구성: `astro-` = star + `-logy` = study
- 뜻: 점성술
- 연결: 별의 위치가 인간사에 영향을 준다고 해석하는 체계
- 사용 맥락: `astrology is not astronomy` = 점성술은 천문학이 아니다

### astrologer

- 발음: /əˈstrɑlədʒɚ/
- 구성: `astrology` + `-er`
- 뜻: 점성술사
- 연결: 별자리와 운세를 해석하는 사람
- 사용 맥락: horoscope 문맥에서 보임.

### asterisk

- 발음: /ˈæstɚɪsk/
- 구성: Greek `asteriskos` = little star
- 뜻: 별표
- 연결: 작은 별 모양의 표시
- 사용 맥락: `mark it with an asterisk` = 별표로 표시하다

### asteroid

- 발음: /ˈæstɚˌɔɪd/
- 구성: Greek `aster` = star + `-oid` = like
- 뜻: 소행성
- 연결: 별처럼 보이는 작은 천체
- 사용 맥락: `an asteroid belt` = 소행성대

### disaster

- 발음: /dɪˈzæstɚ/
- 구성: Italian/French `dis-` = ill, bad + `astro` = star
- 뜻: 재난, 참사
- 연결: 나쁜 별의 영향이라는 옛 점성술적 생각에서 출발
- 사용 맥락: `a natural disaster` = 자연재해

### astral

- 발음: /ˈæstrəl/
- 구성: Latin/Greek `astr-` = star + `-al`
- 뜻: 별의, 천체의
- 연결: 별과 관련됨
- 사용 맥락: `astral light` = 별빛, 성광

## 비교 단어

`sider`는 [[sider]] 문서처럼 Latin `sidus/sider-` = 별, 별자리 계열.
`astr/aster`는 Greek 계열이라 `astronomy`, `asteroid`, `astronaut` 같은 학술어에 자주 보임.

`astrology`와 `astronomy`는 구분.
`astronomy`는 과학으로서 천문학, `astrology`는 별점을 해석하는 점성술.
