# count

- 출발 단어: `discount`
- 핵심 어원: Latin `computare` = count, reckon; `putare` = reckon, settle, originally prune
- 핵심 이미지: 잘라 정리하듯 수를 세고 계산함
- 관련 문서: [[counter]], [[calc]], [[comit]]
- 주의: `counter-` = against 접두사와 `count` = 계산하다 계열은 구분. `county`, 백작 `count`는 [[comit]] 계열.

## 출발 단어 분석

`discount` ← Medieval Latin `discomputare` = count off, deduct

- `dis-`: away, off
- `count`: count, reckon
- 원래 이미지: 전체 계산에서 따로 세어 빼냄
- 의미 변화: 계산에서 덜어 냄 → 가격을 깎음 → 중요성을 낮게 평가함
- 현대 의미: 할인; 할인하다; 무시하다, 평가절하하다

## 단어들

### discount

- 구성: `dis-` = off/away + `count`
- 뜻: 할인, 할인하다; 무시하다, 평가절하하다
- 어원적 연결: 전체 계산에서 일정 부분을 빼거나 덜어 냄
- 사용 맥락: `offer a discount` = 할인을 제공하다, `discount the evidence` = 증거를 대수롭지 않게 보다

### count

- 구성: Old French `conter` ← Latin `computare` = count, reckon
- 뜻: 세다, 계산하다; 중요하다
- 어원적 연결: 수를 세어 계산에 넣음. 계산에 들어가므로 중요하다는 뜻으로 확장
- 사용 맥락: `count the votes` = 표를 세다, `Every detail counts.` = 모든 세부가 중요하다

### account

- 구성: Old French `aconter` = count up, reckon
- 뜻: 계좌, 설명, 보고; 설명하다
- 어원적 연결: 계산해 정리한 내역에서 보고·설명으로 확장
- 사용 맥락: `bank account` = 은행 계좌, `give an account` = 설명하다

### accountant

- 구성: `account` + `-ant`
- 뜻: 회계사
- 어원적 연결: 계산 내역을 정리하고 책임지는 사람
- 사용 맥락: `hire an accountant` = 회계사를 고용하다

### accountability

- 구성: `accountable` + `-ity`
- 뜻: 책임, 설명 책임
- 어원적 연결: 자신의 행동을 계산 내역처럼 설명해야 하는 상태
- 사용 맥락: `public accountability` = 공적 책임성

### recount

- 구성: `re-` = again + `count`
- 뜻: 다시 세다; 자세히 이야기하다
- 어원적 연결: 수를 다시 세거나, 사건을 차례로 세듯 말함
- 사용 맥락: `recount the ballots` = 투표지를 다시 세다, `recount a story` = 이야기를 자세히 들려주다

### compute

- 구성: Latin `com-` = together + `putare` = reckon
- 뜻: 계산하다
- 어원적 연결: 여러 값을 함께 모아 따져 계산함
- 사용 맥락: `compute the average` = 평균을 계산하다

### computation

- 구성: `compute` + `-ation`
- 뜻: 계산, 연산
- 어원적 연결: 값을 따져 합산하고 처리하는 과정
- 사용 맥락: `complex computation` = 복잡한 연산

### computer

- 구성: `compute` + `-er`
- 뜻: 컴퓨터; 원래 계산하는 사람
- 어원적 연결: 계산을 수행하는 사람 또는 기계
- 사용 맥락: `a personal computer` = 개인용 컴퓨터

### putative

- 구성: Latin `putativus` ← `putare` = reckon, suppose
- 뜻: 추정상의, 소위의
- 어원적 연결: 계산하듯 따져 그렇게 여겨지는 것
- 사용 맥락: `the putative cause` = 추정 원인

### reputation

- 구성: Latin `reputare` = think over, reckon
- 뜻: 평판, 명성
- 어원적 연결: 사람들이 반복해서 따져 평가한 결과
- 사용 맥락: `a good reputation` = 좋은 평판

### impute

- 구성: Latin `in-` = in/to + `putare` = reckon
- 뜻: 탓으로 돌리다, 귀속시키다
- 어원적 연결: 어떤 원인이나 책임을 계산에 넣어 배정함
- 사용 맥락: `impute motives to someone` = 누군가에게 동기를 돌리다

### dispute

- 구성: Latin `dis-` = apart + `putare` = reckon, consider
- 뜻: 논쟁하다, 다투다; 분쟁
- 어원적 연결: 문제를 갈라 따져 보며 서로 다른 계산·판단을 내놓음
- 사용 맥락: `settle a dispute` = 분쟁을 해결하다

### disputed

- 구성: `dispute` + `-ed`
- 뜻: 논란이 있는, 다툼의 대상인
- 어원적 연결: 서로 다르게 따져 보며 아직 합의되지 않은 상태
- 사용 맥락: `a disputed territory` = 분쟁 지역, `a disputed claim` = 논란이 있는 주장

### disputable

- 구성: `dispute` + `-able`
- 뜻: 논쟁의 여지가 있는
- 어원적 연결: 서로 다르게 따져 볼 수 있어 확정되지 않음
- 사용 맥락: `a disputable claim` = 논쟁의 여지가 있는 주장

### indisputable

- 구성: `in-` = not + `disputable`
- 뜻: 반박할 수 없는, 명백한
- 어원적 연결: 더 따져 다툴 수 없을 만큼 확실함
- 사용 맥락: `indisputable evidence` = 반박할 수 없는 증거, `an indisputable fact` = 명백한 사실

### indisputably

- 구성: `indisputable` + `-ly`
- 뜻: 명백히, 반박의 여지 없이
- 어원적 연결: 따져 다툴 여지가 없는 방식
- 사용 맥락: `indisputably true` = 명백히 사실인

### disputation

- 구성: `dispute` + `-ation`
- 뜻: 논쟁, 토론
- 어원적 연결: 서로 다른 판단을 놓고 따지는 행위
- 사용 맥락: 격식적·학술적 표현

### undisputed

- 구성: `un-` = not + `disputed`
- 뜻: 논란의 여지가 없는, 명백한
- 어원적 연결: 더 이상 따져 다툴 여지가 없음
- 사용 맥락: `the undisputed champion` = 논란의 여지 없는 챔피언

## 기억 포인트

`count/pute` 계열은 "계산에 넣고 따짐" 이미지.

- `count`: 세어 계산함
- `account`: 계산 내역과 설명
- `discount`: 계산에서 빼거나 덜어 냄
- `compute`: 함께 계산함
- `recount`: 다시 세거나 차례로 이야기함
- `reputation`: 반복 평가로 생긴 평판
- `impute`: 책임이나 원인을 계산에 넣어 돌림
- `dispute`: 서로 갈라 따지며 다툼
- `indisputable`: 따져 다툴 수 없을 만큼 확실함

## 비교 단어

### counter

- 뜻: 반대로 맞서는 접두사
- 구분 포인트: [[counter]]는 `counter-` = against 중심. `count`는 계산·산정 중심.

### calc

- 뜻: 작은 돌로 세어 계산함
- 구분 포인트: [[calc]]는 `calculus` = 작은 돌에서 계산으로 확장. `count`는 수를 세고 정산하는 행위 자체.

### put

- 뜻: 두다, 놓다
- 구분 포인트: 영어 `put` = 놓다는 Germanic 계열. Latin `putare` = 계산하다, 정리하다와 구분.

### county / count

- 뜻: 군, 카운티; 백작
- 구분 포인트: [[comit]]의 Latin `comes/comitem` = companion 계열. `count` = 세다와 철자는 같지만 어원이 다름.
