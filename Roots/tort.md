# tort

- 출발 단어: `distort`
- 핵심 어원: Latin `torquere`, past participle `tortus` = 비틀다, 돌려 꼬다
- 핵심 이미지: 비틀림, 꼬임, 원래 모양에서 틀어짐
- 관련 문서: [[tour]], [[vers]], [[rupt]]
- 주의: [[tour]]는 돌다/회전하다 계열. `tort`는 비틀어 꼬거나 모양을 틀어 놓는 이미지.

## 출발 단어 분석

`distort` = Latin `dis-` + `torquere`

- `dis-`: apart, completely
- `torquere`: twist, wrench
- 원래 이미지: 여러 방향으로 비틀어 모양을 흐트러뜨림
- 현대 의미: 왜곡하다, 일그러뜨리다

## 단어들

### distort

- 구성: `dis-` = apart, completely + `tort`
- 뜻: 왜곡하다, 일그러뜨리다
- 어원적 연결: 원래 모양이나 의미를 비틀어 놓음
- 사용 맥락: `distort the facts` = 사실을 왜곡하다

### distortion

- 구성: `distort` + `-ion`
- 뜻: 왜곡, 일그러짐
- 어원적 연결: 비틀린 상태나 결과
- 사용 맥락: `sound distortion` = 음 왜곡

### contort

- 구성: `con-` = together, intensive + `tort`
- 뜻: 뒤틀다, 뒤틀리다
- 어원적 연결: 몸이나 얼굴을 심하게 비틀어 꼼
- 사용 맥락: `a contorted face` = 일그러진 얼굴

### contortion

- 구성: `contort` + `-ion`
- 뜻: 뒤틀림, 곡예적 몸 비틀기
- 어원적 연결: 몸을 비틀어 만든 형태
- 사용 맥락: `facial contortion` = 얼굴 뒤틀림

### extort

- 구성: `ex-` = out + `tort`
- 뜻: 강탈하다, 억지로 뜯어내다
- 어원적 연결: 비틀어 짜내듯 빼앗음
- 사용 맥락: `extort money` = 돈을 갈취하다

### extortion

- 구성: `extort` + `-ion`
- 뜻: 강탈, 갈취
- 어원적 연결: 압박해 비틀어 뜯어내는 행위
- 사용 맥락: `a charge of extortion` = 갈취 혐의

### retort

- 구성: `re-` = back + `tort`
- 뜻: 반박하다, 쏘아붙이다
- 어원적 연결: 상대 말에 다시 비틀어 돌려줌
- 사용 맥락: `she retorted angrily` = 그녀가 화난 듯 쏘아붙였다

### tort

- 구성: Old French `tort` ← Latin `tortus`
- 뜻: 불법행위, 민사상 손해
- 어원적 연결: 비틀린 것 → 잘못, 부정
- 사용 맥락: 법률 기본어 `tort law` = 불법행위법

### tortuous

- 구성: Latin `tortuosus` ← `tortus`
- 뜻: 구불구불한, 우회적인, 복잡한
- 어원적 연결: 길이나 논리가 꼬이고 비틀림
- 사용 맥락: `a tortuous path` = 구불구불한 길

### torturous

- 구성: `torture` + `-ous`
- 뜻: 고통스러운, 고문 같은
- 어원적 연결: 비틀어 괴롭히는 고통
- 사용 맥락: `a torturous wait` = 고통스러운 기다림

### torture

- 구성: Latin `tortura` ← `torquere`
- 뜻: 고문, 심한 고통
- 어원적 연결: 몸을 비틀어 괴롭힘
- 사용 맥락: `psychological torture` = 심리적 고문

### torment

- 구성: Latin `tormentum` ← `torquere`
- 뜻: 고통, 괴롭히다
- 어원적 연결: 비틀어 짜는 듯한 고통
- 사용 맥락: `mental torment` = 정신적 고통

### torsion

- 구성: Latin `torsio` ← `torquere`
- 뜻: 비틀림, 꼬임
- 어원적 연결: 물체가 축을 중심으로 비틀림
- 사용 맥락: 물리·공학 맥락

### torque

- 구성: Latin `torquere`
- 뜻: 토크, 회전력
- 어원적 연결: 물체를 비틀어 돌리는 힘
- 사용 맥락: `engine torque` = 엔진 토크

## 기억 포인트

`tort` 계열은 "비틀어 꼬기" 이미지.

- `distort`: 사실이나 모양을 비틀어 왜곡
- `contort`: 몸이나 얼굴을 뒤틀음
- `extort`: 비틀어 짜내듯 강탈
- `retort`: 말을 되비틀어 반박
- `tortuous`: 길이나 논리가 꼬임
- `torture`: 몸과 마음을 비틀 듯 괴롭힘
- `torque`: 비틀어 돌리는 힘
