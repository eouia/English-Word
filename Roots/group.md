# group

- 출발 단어: `group`
- 핵심 어원: 프랑스어 `groupe` ← 이탈리아어 `gruppo` = 무리, 매듭
- 핵심 의미: 한데 묶인 무리, 덩어리, 집합
- 관련 문서: [[uni]], [[part]], [[trib]]
- 주의: 더 깊은 기원은 게르만계 “둥근 덩어리, 뭉치”와 연결될 가능성이 크지만 단정은 피함.

## 출발 단어 분석

`group` ← French `groupe` ← Italian `gruppo`

- 원래 이미지: 매듭처럼 한데 뭉친 덩어리
- 초기 영어 의미: 그림 속 인물이나 사물이 조화롭게 모인 배열
- 현대 핵심: 사람·사물·개념을 한 묶음으로 본 것

## 단어들

### group

- 발음: /ˈgrup/
- 구성: French `groupe`
- 뜻: 그룹, 무리, 집단
- 어원적 연결: 한데 묶여 보이는 덩어리
- 사용 맥락: `a study group` = 스터디 그룹

### grouped

- 발음: /ˈgrupt/
- 구성: `group` + `-ed`
- 뜻: 묶인, 분류된
- 어원적 연결: 여러 항목이 한 덩어리로 묶인 상태
- 사용 맥락: `grouped by age` = 나이별로 묶인

### grouping

- 발음: /ˈgrupɪŋ/
- 구성: `group` + `-ing`
- 뜻: 분류, 묶음, 집단화
- 어원적 연결: 항목들을 한 묶음으로 배열함
- 사용 맥락: `data grouping` = 데이터 묶기

### regroup

- 발음: /riˈgrup/
- 구성: `re-` = again + `group`
- 뜻: 다시 모이다, 재정비하다
- 어원적 연결: 흩어진 것을 다시 한 무리로 묶음
- 사용 맥락: `regroup after a setback` = 좌절 뒤 다시 정비하다

### subgroup

- 발음: /ˈsʌˌbgrup/
- 구성: `sub-` = under + `group`
- 뜻: 하위 그룹, 부분 집단
- 어원적 연결: 큰 묶음 안의 작은 묶음
- 사용 맥락: `a subgroup of students` = 학생들의 하위 집단

### groupthink

- 구성: `group` + `think`
- 뜻: 집단사고
- 어원적 연결: 집단의 조화 압력이 개인 판단을 누르는 현상
- 사용 맥락: `avoid groupthink` = 집단사고를 피하다

### groupie

- 발음: /ˈgrupi/
- 구성: `group` + `-ie`
- 뜻: 열성 팬, 따라다니는 팬
- 어원적 연결: 음악 그룹 주변에 붙어 다니는 사람
- 사용 맥락: `a rock band groupie` = 록 밴드 열성 팬

## 기억 포인트

- `group`의 기본 이미지는 “한데 묶인 덩어리”.
- `regroup`은 흩어진 뒤 다시 묶임.
- `subgroup`은 큰 덩어리 안의 작은 덩어리.
- `groupthink`는 시험·사회과학 지문에서 자주 보이는 응용 단어.

## 비교 단어

### class

- `class`는 라틴어 `classis` = 부류, 등급, 소집된 집단 계열.
- 의미는 비슷하지만 `group`과 다른 어원.

### category

- `category`는 그리스어 `kategoria` = 진술, 범주 계열.
- 분류 의미는 겹치지만 `group` 계열은 아님.

### cluster

- `cluster`는 게르만계 “송이, 무리” 계열.
- `group`과 의미는 가깝지만 직접 같은 문서에 섞지 않음.
