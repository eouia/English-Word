# gard

- 출발 단어: `garden`
- 핵심 어원: Germanic `gard-` = 울타리 안의 땅, 둘러싸인 곳
- 관련 형태: Old English `geard` = fenced enclosure, yard
- 깊은 어근: PIE `*gher-` = 잡다, 둘러싸다, 에워싸다
- 핵심 이미지: 울타리나 경계 안에 들어온 땅
- 관련 문서: [[hort]], [[court]]
- 주의: Latin `hortus` = 정원 계열은 [[hort]]로 분리. 깊은 PIE 친족이어도 역사적 경로가 다름.

## 출발 단어 분석

`garden` ← Old North French `gardin/jardin` ← Germanic `gard-`

- `gard-`: 울타리 안의 땅, 둘러싸인 공간
- 원래 이미지: 경계로 둘러싸인 가꾸는 땅
- 현대 의미: 정원, 뜰

## 단어들

### garden

- 구성: Germanic `gard-` = enclosure
- 뜻: 정원
- 어원적 연결: 울타리로 둘러싸인 가꾸는 땅
- 사용 맥락: `a flower garden` = 꽃 정원

### gardener

- 구성: `garden` + `-er`
- 뜻: 정원사
- 어원적 연결: 둘러싸인 정원을 가꾸는 사람
- 사용 맥락: `a professional gardener` = 전문 정원사

### gardening

- 구성: `garden` + `-ing`
- 뜻: 정원 가꾸기
- 어원적 연결: 정원을 돌보고 가꾸는 일
- 사용 맥락: `enjoy gardening` = 정원 가꾸기를 즐기다

### yard

- 구성: Old English `geard` = fenced enclosure
- 뜻: 마당, 뜰
- 어원적 연결: 집 주변의 둘러싸인 땅
- 사용 맥락: `a backyard` = 뒷마당

### courtyard

- 구성: [[court]] + `yard`
- 뜻: 안뜰
- 어원적 연결: 건물이나 벽으로 둘러싸인 마당
- 사용 맥락: `a quiet courtyard` = 조용한 안뜰

### garth

- 구성: Old Norse `garthr` = yard, enclosure
- 뜻: 안뜰, 둘러싸인 땅
- 어원적 연결: 울타리나 벽 안의 공간
- 사용 맥락: 문학적·지역적 표현으로 제한적

### kindergarten

- 구성: German `Kinder` = children + `Garten` = garden
- 뜻: 유치원
- 어원적 연결: 아이들을 가꾸는 정원이라는 은유
- 사용 맥락: `kindergarten students` = 유치원생

### jardiniere

- 구성: French `jardin` = garden
- 뜻: 화분 받침, 장식용 화분
- 어원적 연결: 정원·식물을 담는 용기
- 사용 맥락: `a ceramic jardiniere` = 도자기 화분

### orchard

- 구성: Old English `ortgeard/wortgeard`
- 뜻: 과수원
- 어원적 연결: 식물·과일나무를 기르는 둘러싸인 땅. 첫 부분은 식물 뜻이고, 뒤의 `geard`가 울타리 안 땅
- 사용 맥락: `an apple orchard` = 사과 과수원

## 기억 포인트

`gard` 계열은 "울타리 안의 땅" 이미지.

- `garden`: 가꾸는 울타리 안 땅
- `yard`: 집 주변의 둘러싸인 땅
- `garth`: 둘러싸인 뜰
- `kindergarten`: 아이들의 정원
- `orchard`: 식물·과일나무가 있는 `geard`

## 비교 단어

### hort

- 뜻: Latin `hortus` = 정원
- 구분 포인트: [[hort]]는 `horticulture`, `horticultural`, `horticulturist`처럼 Latin `hortus` 경로. `gard`는 Germanic 경로.

### court

- 뜻: 둘러싸인 뜰, 궁정, 법정
- 구분 포인트: [[court]]는 Latin `cohors/cortis`에서 온 뜰·궁정·법정 계열.
