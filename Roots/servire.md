# servire

- 출발 단어: `serve`
- 핵심 어원: Latin `servire` / `servus` = serve, servant, slave
- 핵심 이미지: 섬기다, 봉사하다, 종으로 일하다
- 관련 어근: [[serv]], [[fac]], Latin `officium`

## 출발 단어 분석

`serve`는 Old French `servir`를 거쳐 Latin `servire`에서 온 말.

Latin `servire`는 "종이 되다, 섬기다"의 뜻이고, `servus` = servant, slave와 연결됨.

따라서 `serve`는 "누군가를 섬기며 일함"에서 "제공하다, 복무하다, 도움이 되다"로 확장됨.

주의: `observe`, `preserve`, `conserve`, `reserve`는 Latin `servare` = watch, keep safe 계열이므로 [[serv]]에서 따로 정리.

## 단어들

### serve

- 발음: /ˈsɝv/
- 구성: Latin `servire`
- 의미: 섬기다, 제공하다, 복무하다
- 어원적 연결: 누군가에게 봉사하거나 일을 해 줌
- 사용 맥락: `serve customers` = 고객을 응대하다

### service

- 발음: /ˈsɝvəs/
- 구성: Old French `servise` < Latin `servitium`
- 의미: 서비스, 봉사, 근무
- 어원적 연결: 섬기거나 도움을 제공하는 일
- 사용 맥락: `customer service` = 고객 서비스

### servant

- 발음: /ˈsɝvənt/
- 구성: `serve` + `-ant`
- 의미: 하인, 종
- 어원적 연결: 섬기는 사람
- 사용 맥락: `a household servant` = 가정의 하인

### servile

- 발음: /ˈsɝvəl/
- 구성: Latin `servilis`
- 의미: 비굴한, 노예적인
- 어원적 연결: 종처럼 굽신거리는 태도
- 사용 맥락: `servile obedience` = 비굴한 복종

### servitude

- 발음: /ˈsɝvəˌtud/
- 구성: Latin `servitudo`
- 의미: 노예 상태, 예속
- 어원적 연결: 섬기며 자유롭지 못한 상태
- 사용 맥락: `involuntary servitude` = 비자발적 예속

### deserve

- 발음: /dɪˈzɝv/
- 구성: `de-` = completely + `servire`
- 의미: ~을 받을 만하다
- 어원적 연결: 충분히 섬기거나 일해서 어떤 대우를 받을 자격을 얻음
- 사용 맥락: `deserve praise` = 칭찬받을 만하다

### dessert

- 발음: /dɪˈzɝt/
- 구성: Old French `desservir` = clear the table
- 의미: 디저트
- 어원적 연결: 식탁을 치운 뒤 나오는 마지막 음식
- 사용 맥락: `have dessert` = 디저트를 먹다

## 기억 포인트

- `serve`: 섬기다, 제공하다
- `service`: 봉사, 서비스
- `servant`: 섬기는 사람
- `servile`: 종처럼 비굴한
- `servitude`: 예속 상태
- `deserve`: 일한 만큼 받을 만함

## 비교 단어

- [[serv]]
  - 핵심: 지키다, 살피다, 보존하다
  - 구분 포인트: `servare` 계열은 `observe`, `preserve`, `reserve`. `servire` 계열은 `serve`, `service`, `deserve`

- [[fac]]
  - 핵심: 만들다, 하다
  - 구분 포인트: `service`는 남을 위해 하는 일. `fac`은 만들거나 실행하는 행위 일반

- `office`
  - 핵심: Latin `officium` = service, duty
  - 구분 포인트: 의미는 가깝지만 `office`는 [[fac]] 문서에서 정리한 `officium` 계열
