# sign

- 출발 단어: `sign`
- 핵심 의미: 표시, 신호, 표로 나타내다
- 관련 형태: Latin `signum` = mark, token, sign
- 관련 문서: [[Roots/index|index]], [[dict]]

## 핵심 이미지

`sign` 계열의 바탕은 "무언가를 알아보게 하는 표시".

눈에 보이는 표시는 `sign`.
뜻을 나타내는 신호는 `signal`.
이름을 표시로 남기면 `signature`.
표시가 의미를 가리키면 `signify`, `significant`, `insignificant`.
어떤 사람이나 대상에 표시를 붙여 정하면 `assign`, `designate`.

## 단어들

### sign

- 구성: Latin `signum` = mark, sign
- 뜻: 표시, 표지판, 징후; 서명하다
- 연결: 어떤 뜻이나 존재를 알아보게 하는 표시
- 사용 맥락: `a road sign` = 도로 표지판, `sign a contract` = 계약서에 서명하다

### signal

- 구성: `sign` + `-al`
- 뜻: 신호, 신호를 보내다
- 연결: 상대가 알아차리도록 보내는 표시
- 사용 맥락: `traffic signals` = 교통 신호

### signature

- 구성: Latin `signare` = mark, sign + `-ture`
- 뜻: 서명; 특징적인 표시
- 연결: 이름을 표시로 남김
- 사용 맥락: `put your signature here` = 여기에 서명하라

### signify

- 구성: Latin `significare` = make signs, mean
- 뜻: 의미하다, 나타내다
- 연결: 표시가 어떤 뜻을 가리킴
- 사용 맥락: `The color red can signify danger.` = 빨간색은 위험을 나타낼 수 있다

### significant

- 구성: `signify` 계열 + `-ant`
- 뜻: 중요한, 의미 있는, 상당한
- 연결: 그냥 표시가 아니라 의미를 강하게 가진 것
- 사용 맥락: `a significant change` = 중요한 변화

### significance

- 구성: `significant` + `-ce`
- 뜻: 중요성, 의미
- 연결: 표시가 담고 있는 의미의 무게
- 사용 맥락: `historical significance` = 역사적 중요성

### insignificant

- 구성: `in-` = not + `significant`
- 뜻: 중요하지 않은, 사소한, 의미가 작은
- 연결: 의미의 표시나 무게가 크지 않음
- 사용 맥락: `an insignificant detail` = 사소한 세부 사항

### insignificance

- 구성: `in-` = not + `significance`
- 뜻: 중요하지 않음, 사소함
- 연결: 의미나 영향의 무게가 작음
- 사용 맥락: `a sense of insignificance` = 자신이 하찮게 느껴지는 감각

### assign

- 구성: Latin `ad-` = to + `signare` = mark
- 뜻: 배정하다, 맡기다
- 연결: 사람이나 일에 표시를 붙여 정해 줌
- 사용 맥락: `assign homework` = 숙제를 내주다

### assignment

- 구성: `assign` + `-ment`
- 뜻: 과제, 임무, 배정
- 연결: 누구에게 표시되어 맡겨진 일
- 사용 맥락: `submit an assignment` = 과제를 제출하다

### design

- 구성: Latin `de-` = out/down + `signare` = mark
- 뜻: 설계하다, 디자인; 계획
- 연결: 만들 대상을 표시하고 윤곽을 정함
- 사용 맥락: `design a website` = 웹사이트를 설계하다

### designer

- 구성: `design` + `-er`
- 뜻: 디자이너, 설계자
- 연결: 형태와 기능을 표시해 계획하는 사람
- 사용 맥락: `a graphic designer` = 그래픽 디자이너

### designate

- 구성: Latin `de-` + `signare`
- 뜻: 지정하다, 지명하다
- 연결: 특정 대상에 표시를 붙여 따로 정함
- 사용 맥락: `designate a meeting place` = 만날 장소를 지정하다

### designation

- 구성: `designate` + `-ion`
- 뜻: 지정, 명칭, 직함
- 연결: 표시를 붙여 정한 이름이나 지위
- 사용 맥락: `official designation` = 공식 명칭

### resign

- 구성: Latin `re-` = back/again + `signare` = mark, seal
- 뜻: 사임하다, 물러나다
- 연결: 원래 "서명·도장을 되돌려 넘김" 이미지에서 직책을 내놓는 뜻으로 확장
- 사용 맥락: `resign from a position` = 직책에서 사임하다

### resignation

- 구성: `resign` + `-ation`
- 뜻: 사임, 체념
- 연결: 맡은 표시나 권한을 내려놓음
- 사용 맥락: `submit a resignation letter` = 사직서를 제출하다

### consign

- 구성: Latin `con-` = together/with + `signare`
- 뜻: 맡기다, 보내다, 위탁하다
- 연결: 표시를 붙여 어떤 곳이나 사람에게 넘김
- 사용 맥락: `consign goods to storage` = 물품을 창고에 맡기다

### insignia

- 구성: Latin `in-` + `signum`
- 뜻: 휘장, 배지, 계급장
- 연결: 소속·지위·권한을 알아보게 하는 표시
- 사용 맥락: 군대, 학교, 조직의 상징 표식.

## 비교 단어

`index`, `indicate`, `indicator`는 [[Roots/index|index]] 계열.
둘 다 "가리켜 알림" 이미지가 있지만, `sign`은 표시 자체와 그 표시가 담은 의미가 중심.

`dict`는 [[dict]] 계열로 "말하다, 선언하다" 중심.
`signify`는 말보다 표시가 의미를 가리키는 쪽에 가깝다.
