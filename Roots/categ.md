# categ

- 출발 단어: `category`
- 핵심 어원: 그리스어 `kategoria` = 고발, 단언, 술어, 범주
- 핵심 의미: 어떤 것에 대해 말해 붙인 범주
- 관련 문서: [[class]], [[group]], [[logos]], [[dict]]
- 주의: `category`는 `class`와 의미가 가깝지만 어원이 다름.

## 출발 단어 분석

`category` ← Greek `kategoria`

- `kata-`: 아래로, 향하여, 맞서
- `agoreuein`: 공개 장소에서 말하다
- 원래 이미지: 공개적으로 말해 어떤 것에 속성을 붙임
- 의미 확장: 단언, 술어 → 논리학의 최고 범주 → 일반적 분류 항목

## 단어들

### category

- 발음: /ˈkætəˌgɔri/
- 구성: Greek `kategoria`
- 뜻: 범주, 부류
- 어원적 연결: 어떤 대상에 대해 말해 붙인 분류
- 사용 맥락: `a product category` = 제품 범주

### categorical

- 발음: /ˌkætəˈgɑrɪkəl/
- 구성: `category` + `-al`
- 뜻: 범주의, 단정적인, 무조건적인
- 어원적 연결: 논리학에서 명확히 단언된 범주에 속함
- 사용 맥락: `a categorical denial` = 단호한 부인

### categorically

- 발음: /ˌkætəˈgɑrɪkli/
- 구성: `categorical` + `-ly`
- 뜻: 단호하게, 명백히
- 어원적 연결: 조건 없이 분명히 단언함
- 사용 맥락: `categorically reject the claim` = 그 주장을 단호히 거부하다

### categorize

- 발음: /ˈkætəgɚˌaɪz/
- 구성: `category` + `-ize`
- 뜻: 분류하다, 범주화하다
- 어원적 연결: 대상을 특정 범주에 넣음
- 사용 맥락: `categorize responses` = 응답을 분류하다

### categorization

- 발음: /ˌkætəgɚɪˈzeɪʃən/
- 구성: `categorize` + `-ation`
- 뜻: 분류, 범주화
- 어원적 연결: 항목을 범주에 따라 나눈 결과
- 사용 맥락: `data categorization` = 데이터 범주화

### categorized

- 발음: /ˈkætəgɚˌaɪzd/
- 구성: `categorize` + `-ed`
- 뜻: 분류된, 범주화된
- 어원적 연결: 특정 범주가 붙은 상태
- 사용 맥락: `categorized by topic` = 주제별로 분류된

### uncategorized

- 구성: `un-` = not + `categorized`
- 뜻: 분류되지 않은
- 어원적 연결: 아직 어떤 범주에도 넣지 않은 상태
- 사용 맥락: `uncategorized files` = 분류되지 않은 파일

### categorical imperative

- 구성: `categorical` + `imperative`
- 뜻: 정언명령
- 어원적 연결: 조건에 따라 달라지지 않는 절대적 명령
- 사용 맥락: `Kant's categorical imperative` = 칸트의 정언명령

## 기억 포인트

- `category`는 “대상에 대해 말해 붙인 범주”.
- `categorize`는 “범주에 넣다”.
- `categorical`은 두 갈래: “범주의”와 “단정적인”.
- `categorically`는 시험 지문에서 “단호히, 명백히”로 자주 나옴.

## 비교 단어

### class

- [[class]]는 라틴어 `classis` = 부류, 등급, 소집된 집단.
- `category`는 그리스어 말하기/술어 계열에서 온 범주.

### group

- [[group]]은 한데 묶인 무리, 덩어리.
- `category`는 개념적으로 이름 붙인 분류 항목에 가까움.

### predicate

- `predicate`는 라틴어 `praedicare` = 공개적으로 말하다, 선언하다 계열.
- 논리학에서 “술어”라는 의미가 있어 `category`와 개념상 가까움.
