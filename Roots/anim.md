# anim

- 출발 단어: `animal`
- 핵심 어원: Latin `anima` = 숨, 생명, 영혼
- 관련 형태: Latin `animus` = 마음, 정신, 기백
- 핵심 이미지: 숨이 있어 살아 움직이는 생명과 마음
- 관련 문서: [[spir]], [[pneu]], [[psych]], [[uni]], [[mag]]
- 주의: `unanimous`, `magnanimous`는 각각 [[uni]], [[mag]]에도 걸쳐 있는 합성어.

## 출발 단어 분석

`animal` ← Latin `animale`

- `anima`: 숨, 생명, 영혼
- `animalis`: 살아 있는, 숨쉬는
- 원래 이미지: 숨이 있어 살아 움직이는 존재
- 현대 의미: 동물

## 단어들

### animal

- 발음: /ˈænəməl/
- 구성: Latin `animale`
- 뜻: 동물
- 어원적 연결: 숨쉬고 살아 움직이는 존재
- 사용 맥락: `wild animals` = 야생 동물

### animate

- 발음: adj. /ˈænəmət/, v. /ˈænəˌmeɪt/
- 구성: Latin `animare` = give life to
- 뜻: 살아 있게 하다, 활기를 주다, 움직이게 하다
- 어원적 연결: 생명과 움직임을 불어넣음
- 사용 맥락: `animate a character` = 캐릭터를 움직이게 하다

### animated

- 발음: /ˈænəˌmeɪtɪd/
- 구성: `animate` + `-ed`
- 뜻: 활기찬, 애니메이션의
- 어원적 연결: 생기가 들어가 움직이는 상태
- 사용 맥락: `an animated discussion` = 활기찬 토론

### animation

- 발음: /ˌænəˈmeɪʃən/
- 구성: `animate` + `-ion`
- 뜻: 활기, 애니메이션
- 어원적 연결: 생명 없는 그림에 움직임을 줌
- 사용 맥락: `computer animation` = 컴퓨터 애니메이션

### inanimate

- 발음: /ˌɪˈnænəmət/
- 구성: `in-` = not + `animate`
- 뜻: 생명이 없는, 무생물의
- 어원적 연결: 숨과 생명이 들어 있지 않음
- 사용 맥락: `inanimate objects` = 무생물

### animism

- 발음: /ˈænəˌmɪzəm/
- 구성: `anima` + `-ism`
- 뜻: 애니미즘, 정령 신앙
- 어원적 연결: 자연물에도 영혼이나 생명이 있다고 보는 믿음
- 사용 맥락: `animism in traditional cultures` = 전통문화의 애니미즘

### animus

- 발음: /ˈænɪməs/
- 구성: Latin `animus`
- 뜻: 적의, 의도, 정신
- 어원적 연결: 마음속 기운이나 태도
- 사용 맥락: `personal animus` = 개인적 적의

### unanimous

- 발음: /juˈnænəməs/
- 구성: [[uni]] = one + `animus` = mind
- 뜻: 만장일치의
- 어원적 연결: 마음이 하나로 모임
- 사용 맥락: `a unanimous vote` = 만장일치 표결

### magnanimous

- 발음: /mæˈgnænəməs/
- 구성: [[mag]] = great + `animus` = mind
- 뜻: 도량이 큰, 관대한
- 어원적 연결: 마음이 큰 사람
- 사용 맥락: `a magnanimous response` = 도량 큰 대응

### equanimity

- 발음: /ˌikwəˈnɪmɪti/
- 구성: [[equ]] = even + `animus` = mind
- 뜻: 평정심, 침착
- 어원적 연결: 마음이 고르게 유지됨
- 사용 맥락: `maintain equanimity` = 평정심을 유지하다

### pusillanimous

- 발음: /pjusəˈlænəməs/
- 구성: Latin `pusillus` = very small + `animus` = mind
- 뜻: 겁 많은, 소심한
- 어원적 연결: 마음이 작고 약함
- 사용 맥락: `a pusillanimous reaction` = 소심한 반응

## 기억 포인트

`anim` 계열은 "숨과 생명, 마음" 이미지.

- `animal`: 숨쉬는 생명체
- `animate`: 생명과 움직임을 주다
- `inanimate`: 생명이 없음
- `unanimous`: 마음이 하나
- `magnanimous`: 마음이 큼

## 비교 단어

### spir

- 뜻: 숨쉬다, 숨
- 구분 포인트: [[spir]]는 `spirare` = 숨쉬는 행위에서 출발. `anim`은 숨이 깃든 생명과 마음의 상태에 초점.
