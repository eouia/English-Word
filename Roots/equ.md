# equ

- 출발 단어: `equation`
- 핵심 어원: 라틴어 `aequus` = 같은, 평평한, 공정한
- 핵심 이미지: 같은 높이로 맞춤, 균형
- 관련해서 헷갈릴 수 있는 어근: [[uni]] = 하나, 하나로 묶임 계열

## 출발 단어 분석

`equation`은 `equate` + `-ion`.

- `aequus`: equal, level, fair
- `equate`: 같게 놓다, 동일시하다
- `equation`: 같게 놓은 관계

따라서 `equation`은 "양쪽을 같게 놓은 식" → "방정식"의 뜻.

## 단어들

### equal

- 발음: /ˈikwəl/
- 구성: Latin `aequalis`
- 뜻: 같은, 동등한
- 어원적 연결: 높이나 양, 지위가 같음
- 사용 맥락: `equal rights` = 평등한 권리

### equality

- 발음: /ɪˈkwɑləti/
- 구성: `equal` + `-ity`
- 뜻: 평등, 동등성
- 어원적 연결: 같은 상태
- 사용 맥락: `gender equality` = 성평등

### equate

- 발음: /ɪˈkweɪt/
- 구성: `equ` + `-ate`
- 뜻: 동일시하다, 같다고 보다
- 어원적 연결: 서로 같은 높이에 놓음
- 사용 맥락: `equate wealth with success` = 부를 성공과 동일시하다

### equation

- 발음: /ɪˈkweɪʒən/
- 구성: `equate` + `-ion`
- 뜻: 방정식, 등식
- 어원적 연결: 양쪽을 같게 놓은 수학적 관계
- 사용 맥락: `solve an equation` = 방정식을 풀다. 이때 `solve`는 [[solv]] 계열

### equivalent

- 발음: /ɪˈkwɪvələnt/
- 구성: `equi-` = equal + [[val]]
- 뜻: 동등한, 상당하는 것
- 어원적 연결: 가치나 힘이 같음
- 사용 맥락: `an equivalent amount` = 동등한 양

### equivalence

- 발음: /ɪˈkwɪvələns/
- 구성: `equivalent` + `-ence`
- 뜻: 동등성, 등가
- 어원적 연결: 같은 가치나 의미를 가짐
- 사용 맥락: `moral equivalence` = 도덕적 동등성

### adequate

- 발음: /ˈædəkwət/
- 구성: `ad-` = to + `equate`
- 뜻: 충분한, 적절한
- 어원적 연결: 필요한 기준에 맞게 같아짐
- 사용 맥락: `adequate support` = 충분한 지원

### inadequate

- 발음: /ˌɪˈnædəkwət/
- 구성: [[in]] = not + `adequate`
- 뜻: 불충분한, 부적절한
- 어원적 연결: 기준에 맞을 만큼 같지 않음
- 사용 맥락: `inadequate evidence` = 불충분한 증거

### equity

- 발음: /ˈɛkwəti/
- 구성: Latin `aequitas`
- 뜻: 공정성, 형평, 자산 가치
- 어원적 연결: 같은 높이로 공정하게 맞춤
- 사용 맥락: `equity in education` = 교육의 형평성

### equitable

- 발음: /ˈɛkwətəbəl/
- 구성: `equity` + `-able`
- 뜻: 공정한, 공평한
- 어원적 연결: 균형 있게 맞춘 상태
- 사용 맥락: `an equitable solution` = 공정한 해결책

### equator

- 발음: /ɪˈkweɪtɚ/
- 구성: `equate` + `-or`
- 뜻: 적도
- 어원적 연결: 낮과 밤을 같게 나누는 선이라는 이미지
- 사용 맥락: `near the equator` = 적도 근처

### equivocal

- 발음: /ɪˈkwɪvəkəl/
- 구성: `equi-` = equal + [[voc]]
- 뜻: 애매한, 모호한
- 어원적 연결: 둘 이상의 목소리나 의미가 같게 들림
- 사용 맥락: `an equivocal answer` = 애매한 대답

### unequivocal

- 발음: /ˌʌnɪˈkwɪvəkəl/
- 구성: `un-` = not + `equivocal`
- 뜻: 명백한, 분명한
- 어원적 연결: 여러 의미로 같게 갈리지 않음
- 사용 맥락: `unequivocal evidence` = 명백한 증거

## 기억 포인트

`equ` 계열은 "같음 / 균형 / 공정함" 이미지.

- `equal`: 같음
- `equate`: 같다고 놓음
- `equation`: 같게 놓은 식
- `equivalent`: 같은 가치
- `adequate`: 기준에 맞을 만큼 충분함
- `equity`: 공정하게 맞춤
- `equivocal`: 여러 의미가 같게 들려 모호함

## 비교 단어

### uni

- 뜻: 하나
- 구분 포인트: [[uni]]는 하나로 묶임. `equ`는 둘 이상이 같은 높이나 가치로 맞음
