# nov

- 출발 단어: `innovative`
- 핵심 어원: 라틴어 `novus` = 새로운
- 핵심 의미: 새로움, 새롭게 만들거나 바꿈
- 관련 문서: [[recent]], [[mut]], [[cre]], [[narr]]
- 주의: `novel`은 “새로운” 뜻에서 출발해 “새로운 이야기 형식” → 소설로 확장.

## 출발 단어 분석

`innovative` = `in-` + `nov` + `-ative`

- `in-`: into, in
- `novus`: new
- `-ative`: ~하는 성질의
- 원래 이미지: 새것을 안으로 들여오거나 새롭게 바꾸는 성질

## 단어들

### innovative

- 발음: /ˈɪnəˌveɪtɪv/
- 구성: `in-` = into + `nov` + `-ative`
- 뜻: 혁신적인, 참신한
- 어원적 연결: 새 방식을 들여와 바꾸는 성질
- 사용 맥락: `an innovative solution` = 혁신적인 해결책

### innovate

- 발음: /ˈɪnəˌveɪt/
- 구성: `in-` + `nov` + `-ate`
- 뜻: 혁신하다, 새 방식을 도입하다
- 어원적 연결: 새것을 들여와 기존 방식을 바꿈
- 사용 맥락: `companies must innovate` = 기업은 혁신해야 한다

### innovation

- 발음: /ˌɪnəˈveɪʃən/
- 구성: `innovate` + `-ion`
- 뜻: 혁신, 새로운 발명이나 방식
- 어원적 연결: 새로 들여온 변화
- 사용 맥락: `technological innovation` = 기술 혁신

### innovator

- 발음: /ˈɪnəˌveɪtɚ/
- 구성: `innovate` + `-or`
- 뜻: 혁신가
- 어원적 연결: 새 방식을 들여오는 사람
- 사용 맥락: `a social innovator` = 사회 혁신가

### novel

- 발음: /ˈnɑvəl/
- 구성: Latin `novellus` = new, young
- 뜻: 새로운, 참신한; 소설
- 어원적 연결: 새롭고 이전과 다른 것. 문학에서는 새로운 이야기 형식
- 사용 맥락: `a novel approach` = 참신한 접근, `read a novel` = 소설을 읽다

### novelty

- 발음: /ˈnɑvəlti/
- 구성: `novel` + `-ty`
- 뜻: 새로움, 신기함, 새로운 물건
- 어원적 연결: 새롭기 때문에 눈에 띄는 성질
- 사용 맥락: `the novelty wore off` = 신기함이 사라졌다

### novice

- 발음: /ˈnɑvəs/
- 구성: Latin `novicius` = new, newly arrived
- 뜻: 초보자, 신참
- 어원적 연결: 어떤 분야에 새로 들어온 사람
- 사용 맥락: `a novice driver` = 초보 운전자

### renovate

- 발음: /ˈrɛnəˌveɪt/
- 구성: `re-` = again + `nov` + `-ate`
- 뜻: 개조하다, 새롭게 고치다
- 어원적 연결: 다시 새것처럼 만듦
- 사용 맥락: `renovate a house` = 집을 개조하다

### renovation

- 발음: /ˌrɛnəˈveɪʃən/
- 구성: `renovate` + `-ion`
- 뜻: 개조, 보수
- 어원적 연결: 낡은 것을 다시 새롭게 만드는 과정
- 사용 맥락: `building renovation` = 건물 개조

### nova

- 발음: /ˈnoʊvə/
- 구성: Latin `nova` = new
- 뜻: 신성, 갑자기 밝아지는 별
- 어원적 연결: 하늘에 새 별처럼 나타난 것으로 보임
- 사용 맥락: `observe a nova` = 신성을 관측하다

### supernova

- 발음: /ˌsupɚˈnoʊvə/
- 구성: `super-` + `nova`
- 뜻: 초신성
- 어원적 연결: 매우 밝게 새로 나타난 별처럼 보이는 폭발 현상
- 사용 맥락: `a supernova explosion` = 초신성 폭발

## 기억 포인트

- `nov`는 “새로움”.
- `innovate`: 새 방식을 들여옴.
- `renovate`: 다시 새롭게 만듦.
- `novice`: 새로 들어온 사람.
- `novel`: 새로운 것 → 참신한, 소설.

## 비교 단어

### new

- `new`는 게르만계 기본어.
- `nov`와 깊은 뿌리는 같은 계열로 볼 수 있지만, 영어 문서에서는 라틴어 `novus` 계열과 분리.

### recent

- [[recent]]는 Latin `recens` = 갓 생긴, 신선한 계열.
- `nov`는 "새것/새 방식" 자체, `recent`는 "시간상 오래되지 않음"에 초점.

### create

- [[cre]]는 새로 생기게 만들다.
- `nov`는 “새롭다/새롭게 바꾸다”에 초점.

### mutate

- [[mut]]는 바꾸다, 변화시키다.
- `innovate`는 변화 중에서도 “새 방식을 도입함”의 느낌이 강함.
