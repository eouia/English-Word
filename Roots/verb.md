# verb

- 출발 단어: `verb`
- 핵심 어원: Latin `verbum` = 말, 단어
- 핵심 이미지: 입으로 나온 말과 문법의 단어
- 관련 문서: [[dict]], [[voc]], [[logos]]
- 주의: `verb`는 문법 용어로는 "동사"지만, 어원적으로는 넓게 "말, 단어"에서 출발.

## 출발 단어 분석

`verb` ← Latin `verbum`

- `verbum`: word
- 원래 이미지: 말로 표현된 단어
- 현대 의미: 동사

## 단어들

### verb

- 발음: /ˈvɝb/
- 구성: Latin `verbum`
- 뜻: 동사
- 어원적 연결: 문장에서 행동이나 상태를 말하는 핵심 단어
- 사용 맥락: `an irregular verb` = 불규칙 동사

### verbal

- 발음: /ˈvɝbəl/
- 구성: `verb` + `-al`
- 뜻: 말의, 언어의, 구두의
- 어원적 연결: 말이나 단어로 표현됨
- 사용 맥락: `verbal communication` = 언어적 의사소통

### verbose

- 구성: Latin `verbosus` = wordy
- 뜻: 장황한, 말이 많은
- 어원적 연결: 단어가 너무 많음
- 사용 맥락: `a verbose explanation` = 장황한 설명

### verbosity

- 구성: `verbose` + `-ity`
- 뜻: 장황함
- 어원적 연결: 말이나 단어를 지나치게 많이 씀
- 사용 맥락: 글쓰기 평가에서 자주 쓰임

### proverb

- 발음: /ˈprɑvɚb/
- 구성: `pro-` = forth + `verbum` = word
- 뜻: 속담
- 어원적 연결: 널리 앞에 내놓아 전해지는 말
- 사용 맥락: `an old proverb` = 오래된 속담

### adverb

- 발음: /ˈædvɚb/
- 구성: `ad-` = to + `verb`
- 뜻: 부사
- 어원적 연결: 원래는 동사에 붙어 의미를 더하는 말
- 사용 맥락: `a frequency adverb` = 빈도 부사

### verbatim

- 발음: /vɚˈbeɪtəm/
- 구성: Latin `verbum` = word
- 뜻: 말 그대로, 축어적으로
- 어원적 연결: 단어 하나하나 그대로
- 사용 맥락: `quote verbatim` = 그대로 인용하다

### verbiage

- 발음: /ˈvɝbiɪdʒ/
- 구성: `verb` + `-age`
- 뜻: 장황한 말, 표현 방식
- 어원적 연결: 말이 많거나 표현이 길게 늘어짐
- 사용 맥락: `legal verbiage` = 법률 문구

## 기억 포인트

`verb` 계열은 "말과 단어" 이미지.

- `verb`: 문법의 동사
- `verbal`: 말의
- `verbose`: 말이 많은
- `proverb`: 전해지는 말
- `verbatim`: 단어 그대로

## 비교 단어

### dict

- 뜻: 말하다, 선언하다
- 구분 포인트: [[dict]]는 말로 정하거나 선언하는 행위. `verb`는 말·단어 자체.

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 목소리로 부르는 행위. `verb`는 말로 된 단어.
