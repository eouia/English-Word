# auctor

- 출발 단어: `author`
- 핵심 어원: Latin `auctor` = originator, creator, promoter, authority
- 핵심 이미지: 일을 일으킨 사람, 만든 사람, 보증하고 권위를 주는 사람
- 관련 문서: [[authent]]
- 주의: `author`, `authority`, `authorize`는 `authentic`과 철자가 닮았지만, 주로 Latin `auctor` 계열로 따로 봄.

## 출발 단어 분석

`author` ← Latin `auctor`

- `auctor`: originator, creator, promoter, authority
- 원래 이미지: 어떤 일을 시작하고 성립시키는 사람
- 현대 의미: 저자, 창작자, 원인 제공자

## 단어들

### author

- 발음: /ˈɔθɚ/
- 구성: Latin `auctor` = originator, creator
- 뜻: 저자, 작가, 창작자
- 어원적 연결: 글이나 생각을 처음 만들어 낸 사람
- 사용 맥락: `the author of a novel` = 소설의 저자

### authorship

- 발음: /ˈɔθɚˌʃɪp/
- 구성: `author` + `-ship`
- 뜻: 저자임, 저작권적·창작적 소유
- 어원적 연결: 어떤 글이나 작품을 만들어 낸 주체임
- 사용 맥락: `claim authorship` = 저자임을 주장하다

### coauthor

- 발음: /ˈkoʊˈɑθɚ/
- 구성: `co-` = together + `author`
- 뜻: 공동 저자
- 어원적 연결: 함께 작품이나 글을 만들어 낸 사람
- 사용 맥락: `coauthor a paper` = 논문을 공동 집필하다

### authority

- 발음: /əˈθɔrəti/
- 구성: Latin `auctoritas` = influence, command, judgment
- 뜻: 권위, 권한, 당국, 전문가
- 어원적 연결: 만든 사람·보증하는 사람에게서 나오는 힘과 신뢰
- 사용 맥락: `legal authority` = 법적 권한, `an authority on history` = 역사 권위자

### authoritative

- 발음: /əˈθɔrəˌteɪtɪv/
- 구성: `authority` + `-ative`
- 뜻: 권위 있는, 믿을 만한, 명령적인
- 어원적 연결: 권위 있는 근거에서 나온 것으로 받아들여짐
- 사용 맥락: `an authoritative source` = 권위 있는 출처

### authoritarian

- 발음: /əˌθɔrəˈtɛriən/
- 구성: `authority` + `-arian`
- 뜻: 권위주의적인, 독재적인
- 어원적 연결: 권위를 위에서 강하게 행사함
- 사용 맥락: `an authoritarian regime` = 권위주의 정권

### authorize

- 발음: /ˈɔθɚˌaɪz/
- 구성: `authority` + `-ize`
- 뜻: 권한을 부여하다, 허가하다
- 어원적 연결: 어떤 사람이 공식적으로 행동할 권위를 갖게 함
- 사용 맥락: `authorize payment` = 결제를 승인하다

### authorization

- 발음: /ˌɔθɚəˈzeɪʃən/
- 구성: `authorize` + `-ation`
- 뜻: 권한 부여, 승인, 허가
- 어원적 연결: 권위를 공식적으로 넘기거나 인정하는 절차
- 사용 맥락: `prior authorization` = 사전 승인

### unauthorized

- 발음: /əˈnɔθɚˌaɪzd/
- 구성: `un-` = not + `authorized`
- 뜻: 허가받지 않은, 무단의
- 어원적 연결: 권한이 부여되지 않음
- 사용 맥락: `unauthorized access` = 무단 접근

## 기억 포인트

`auctor` 계열은 "만든 사람, 보증자, 권위 부여" 이미지.

- `author`: 글을 만들어 낸 사람
- `authorship`: 저자임
- `authority`: 보증하고 명령할 수 있는 힘
- `authorize`: 권한을 부여함
- `unauthorized`: 권한을 받지 않음

## 비교 단어

### authent

- 뜻: 진짜임, 원본성, 인증
- 구분 포인트: [[authent]]는 Greek `authentikos` 계열. `auctor`는 Latin `auctor` 계열로, 저자·권위·허가의 의미가 중심.
