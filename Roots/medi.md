# medi

- 출발 단어: `mediate`
- 핵심 어원: 라틴어 `medius` = 가운데의, 중간의
- 핵심 이미지: 둘 사이의 가운데에 있음
- 관련해서 헷갈릴 수 있는 어근: [[med]] = 치료하다, 의학 계열

## 출발 단어 분석

`mediate`는 `medi` + `-ate`.

- `medi`: 가운데, 중간
- `-ate`: 동사 형성

따라서 `mediate`는 "중간에 서다" → "중재하다, 조정하다"의 뜻.

## 단어들

### mediate

- 구성: `medi` + `-ate`
- 뜻: 중재하다, 조정하다, 매개하다
- 어원적 연결: 양쪽 사이의 가운데에 서서 연결하거나 조정함
- 사용 맥락: `mediate a dispute` = 분쟁을 중재하다

### mediation

- 구성: `mediate` + `-ion`
- 뜻: 중재, 조정
- 어원적 연결: 양쪽 사이에서 해결점을 찾는 과정
- 사용 맥락: `enter mediation` = 중재 절차에 들어가다

### mediator

- 구성: `mediate` + `-or`
- 뜻: 중재자, 조정자
- 어원적 연결: 둘 사이의 가운데에 서는 사람
- 사용 맥락: `a neutral mediator` = 중립적인 중재자

### medium

- 구성: Latin `medium`
- 뜻: 매체, 수단, 중간
- 어원적 연결: 사이에서 전달을 가능하게 하는 것
- 사용 맥락: `a communication medium` = 의사소통 매체

### media

- 구성: Latin `medium`의 복수형
- 뜻: 매체, 언론
- 어원적 연결: 정보가 사람들 사이를 지나가게 하는 통로
- 사용 맥락: `social media` = 소셜 미디어

### median

- 구성: Latin `medianus`
- 뜻: 중앙의, 중간값
- 어원적 연결: 양쪽 사이의 가운데 위치
- 사용 맥락: `median income` = 중위 소득

### immediate

- 구성: `in-` = not + `mediate`
- 뜻: 즉각적인, 직접적인
- 어원적 연결: 중간에 끼는 것이 없음. 그래서 바로 일어남
- 사용 맥락: `an immediate response` = 즉각적인 응답

### intermediate

- 구성: `inter-` = between + `mediate`
- 뜻: 중간의, 중급의
- 어원적 연결: 둘 사이의 중간 단계에 있음
- 사용 맥락: `intermediate level` = 중급 수준

### medieval

- 구성: Latin `medium aevum` = middle age
- 뜻: 중세의
- 어원적 연결: 고대와 근대 사이의 중간 시대
- 사용 맥락: `medieval history` = 중세사

### Mediterranean

- 구성: Latin `medius` = middle + `terra` = land
- 뜻: 지중해의
- 어원적 연결: 땅들 사이 가운데에 있는 바다
- 사용 맥락: `Mediterranean climate` = 지중해성 기후

### mediocre

- 구성: Latin `mediocris`
- 뜻: 평범한, 보통밖에 안 되는
- 어원적 연결: 높지도 낮지도 않은 중간 정도. 현대 영어에서는 대개 부정적 의미
- 사용 맥락: `a mediocre performance` = 평범한 수준의 공연

### mediocrity

- 구성: `mediocre` + `-ity`
- 뜻: 평범함, 보통 수준
- 어원적 연결: 중간 정도에 머무는 상태
- 사용 맥락: `settle for mediocrity` = 평범함에 안주하다

## 기억 포인트

`medi` 계열은 "가운데 / 사이" 이미지.

- `mediate`: 가운데에서 중재함
- `medium`: 사이에서 전달하는 매체
- `median`: 가운데 위치
- `immediate`: 중간이 없음
- `intermediate`: 사이의 중간 단계
- `medieval`: 중간 시대
- `Mediterranean`: 땅들 사이 가운데의 바다
- `mediocre`: 중간 정도

## 비교 단어

### medical

- 뜻: 의학의, 치료의
- 구분 포인트: [[med]]의 `medical`, `medicine`은 치료하다 계열. `mediate`, `medium`의 `medi` = 가운데와 구분

### moderate

- 뜻: 적당한, 조절하다
- 구분 포인트: [[mod]]의 `moderate`는 알맞은 정도로 맞춤. `mediocre`는 중간 정도에 머묾
