# fer-carry

- 출발 단어: `transfer`
- 핵심 어원: Latin `ferre` / `lat-` = 나르다, 가져가다
- 핵심 이미지: 한 곳에서 다른 곳으로 들고 옮김
- 허브 문서: [[fer]]
- 관련 문서: [[mit]], [[duc]]

## 단어들

### transfer

- 구성: `trans-` = across + `fer`
- 뜻: 옮기다, 이전하다, 환승하다
- 어원적 연결: 한쪽에서 다른 쪽으로 옮겨 나름
- 사용 맥락: `transfer money` = 돈을 이체하다

### offer

- 구성: `ob-` = toward + `fer`
- 뜻: 제안하다, 제공하다
- 어원적 연결: 상대 앞에 가져다 놓음
- 사용 맥락: `offer help` = 도움을 제공하다

### infer

- 구성: `in-` = into + `fer`
- 뜻: 추론하다
- 어원적 연결: 증거를 마음 안으로 가져와 결론을 냄
- 사용 맥락: `infer meaning from context` = 문맥에서 의미를 추론하다

### confer

- 구성: `con-` = together + `fer`
- 뜻: 상의하다, 수여하다
- 어원적 연결: 생각을 함께 가져와 비교함. 권위나 학위를 가져다 줌
- 사용 맥락: `confer with colleagues` = 동료들과 상의하다

### conference

- 구성: `confer` + `-ence`
- 뜻: 회의, 학회
- 어원적 연결: 생각을 함께 가져와 놓는 자리
- 사용 맥락: `attend a conference` = 학회에 참석하다

### translate

- 구성: `trans-` = across + Latin `lat-` = carried
- 뜻: 번역하다, 옮기다
- 어원적 연결: 뜻을 한 언어에서 다른 언어로 건너 옮김
- 사용 맥락: `translate Korean into English` = 한국어를 영어로 번역하다

### translation

- 구성: `translate` + `-ion`
- 뜻: 번역, 옮김
- 어원적 연결: 뜻이 건너 옮겨진 결과
- 사용 맥락: `machine translation` = 기계 번역

### suffer

- 구성: `sub-` = under + `fer`
- 뜻: 고통을 겪다, 견디다
- 어원적 연결: 무거운 것을 아래에서 짊어짐
- 사용 맥락: `suffer from stress` = 스트레스로 고통받다

## 기억 포인트

`fer-carry`는 "들고 옮김" 갈래.

- `transfer`: 건너 옮김
- `offer`: 앞에 가져다 놓음
- `infer`: 안으로 가져와 결론을 냄
- `translate`: 뜻을 건너 옮김
- `suffer`: 아래에서 짊어짐
