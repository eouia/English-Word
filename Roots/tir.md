# tir

- 출발 단어: `retire`
- 핵심 어원: French `tirer` = 끌다, 당기다
- 핵심 이미지: 뒤로 끌어 물러나게 함
- 비교: [[tract]]의 `retreat`, `retract` = 라틴어 `trahere` 계열

## 출발 단어 분석

`retire`는 `re-` + French `tirer`.

- `re-`: back
- `tirer`: 끌다, 당기다

따라서 `retire`는 "뒤로 끌어 물러남" → "물러나다, 은퇴하다"의 뜻.

## 단어들

### retire

- 구성: `re-` + `tirer`
- 뜻: 은퇴하다, 물러나다, 잠자리에 들다
- 어원적 연결: 공적 활동이나 전투에서 뒤로 물러남
- 사용 맥락: `retire from work` = 직장에서 은퇴하다

### retired

- 구성: `retire` + `-ed`
- 뜻: 은퇴한, 물러난
- 어원적 연결: 활동에서 뒤로 물러난 상태
- 사용 맥락: `a retired teacher` = 은퇴한 교사

### retirement

- 구성: `retire` + `-ment`
- 뜻: 은퇴, 퇴직
- 어원적 연결: 직업이나 공적 역할에서 물러나는 행위
- 사용 맥락: `retirement plan` = 은퇴 계획

### retiree

- 구성: `retire` + `-ee`
- 뜻: 은퇴자
- 어원적 연결: 은퇴한 사람
- 사용 맥락: `retiree benefits` = 은퇴자 혜택

### tirade

- 구성: French `tirade`, Italian `tirata`
- 뜻: 장황한 비난, 긴 연설
- 어원적 연결: 길게 끌어낸 말
- 사용 맥락: `launch into a tirade` = 장황한 비난을 퍼붓다

## 비교 단어

### tired

- 구성: Old English `teorian`
- 뜻: 피곤한, 지친
- 어원적 연결: [[tir]] 계열 아님. "실패하다, 그치다, 지치다" 계열로 보되 기원은 불확실
- 구분 포인트: `retire` = 뒤로 끌려 물러남, `tired` = 힘이 빠져 지침

### retreat

- 구성: `re-` + `trahere`
- 뜻: 후퇴하다, 물러나다
- 어원적 연결: [[tract]] 계열. 뒤로 끌려 물러남
- 구분 포인트: `retreat` = 라틴어 계열, `retire` = French `tirer` 계열

### retract

- 구성: `re-` + `tract`
- 뜻: 철회하다, 집어넣다
- 어원적 연결: [[tract]] 계열. 뒤로 끌어당김
- 구분 포인트: `retract` = 말/주장/부분을 뒤로 끌어당김, `retire` = 사람이나 역할이 물러남

## 기억 포인트

`tir` 계열은 "끌어 물러나게 함" 이미지.

- `retire`: 물러남, 은퇴함
- `retired`: 물러난 상태
- `retirement`: 은퇴
- `retiree`: 은퇴자
- `tirade`: 길게 끌어낸 말

비교:

- `tired`: 어원 다름. 피곤한 상태
