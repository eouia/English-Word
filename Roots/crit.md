# crit

- 출발 단어: `critical`
- 핵심 어원: Greek `krinein` = separate, judge, decide
- 핵심 이미지: 갈라 판단함, 중대한 고비에서 판정함
- 관련 문서: [[cern]], [[jud]], [[cid]]
- 주의: [[cern]]은 Latin `cernere` 계열. `crit`, `crisis`, `criterion`은 Greek `krinein` 계열로 분리.

## 출발 단어 분석

`critical` ← Greek `kritikos` = able to judge

- `crit`: judge, decide
- `-ic`: ~에 관한
- `-al`: 형용사 접미사
- 원래 이미지: 판단하고 평가하는 성격
- 의미 변화: 판단하는 → 평가·비평의 → 판정이 갈리는 중대한 → 매우 중요한
- 현대 의미: 비판적인, 평가적인; 중대한, 결정적인

## 단어들

### critical

- 발음: /ˈkrɪtɪkəl/
- 구성: Greek `kritikos` = able to judge + `-al`
- 뜻: 비판적인, 평가적인; 중대한, 결정적인
- 어원적 연결: 어떤 대상을 판단하거나, 결과가 갈리는 지점을 가리킴
- 사용 맥락: `critical thinking` = 비판적 사고, `a critical moment` = 중대한 순간

### critic

- 발음: /ˈkrɪtɪk/
- 구성: Greek `kritikos` = judge, critic
- 뜻: 비평가, 평론가; 비판하는 사람
- 어원적 연결: 작품이나 행동을 살펴 판단하는 사람
- 사용 맥락: `a film critic` = 영화 평론가

### criticism

- 발음: /ˈkrɪtɪˌsɪzəm/
- 구성: `critic` + `-ism`
- 뜻: 비판, 비평
- 어원적 연결: 좋고 나쁨을 갈라 판단하는 말이나 글
- 사용 맥락: `constructive criticism` = 건설적인 비판

### criticize

- 발음: /ˈkrɪtɪˌsaɪz/
- 구성: `critic` + `-ize`
- 뜻: 비판하다, 비평하다
- 어원적 연결: 대상을 평가해 문제점이나 가치를 판단함
- 사용 맥락: `criticize the policy` = 정책을 비판하다

### critique

- 발음: /krəˈtik/
- 구성: French `critique` ← Greek `kritikos`
- 뜻: 비평, 평론; 비평하다
- 어원적 연결: 단순 비난보다 분석적 판단의 느낌
- 사용 맥락: `write a critique` = 비평문을 쓰다

### criterion

- 발음: /kraɪˈtɪriən/
- 구성: Greek `kriterion` = means for judging
- 뜻: 기준, 판단 기준
- 어원적 연결: 무엇을 판단할 때 쓰는 잣대
- 사용 맥락: `the main criterion` = 주요 기준

### criteria

- 발음: /kraɪˈtɪriə/
- 구성: plural of `criterion`
- 뜻: 기준들
- 어원적 연결: 여러 판단 기준
- 사용 맥락: `meet the criteria` = 기준을 충족하다

### crisis

- 발음: /ˈkraɪsəs/
- 구성: Greek `krisis` = decision, turning point
- 뜻: 위기, 중대한 고비
- 어원적 연결: 결과가 어느 쪽으로 갈릴지 판단·결정되는 지점
- 사용 맥락: `an economic crisis` = 경제 위기

### hypocrisy

- 발음: /hɪˈpɑkrəsi/
- 구성: Greek `hypokrisis` = answer, stage acting, pretense
- 뜻: 위선
- 어원적 연결: 원래는 응답·해석·연기와 연결. 현대 의미는 겉과 속이 다른 꾸민 태도
- 사용 맥락: `accuse someone of hypocrisy` = 누군가를 위선이라고 비난하다

## 기억 포인트

`crit` 계열은 "판단"이 중심.

- `critic`: 판단하는 사람
- `criticism`: 판단해서 평가하는 말
- `critical`: 판단하는, 또는 판정이 갈리는 만큼 중요한
- `criterion`: 판단 기준
- `crisis`: 결말이 갈리는 중대한 고비

## 비교 단어

### cern

- 뜻: 가려 구별하고 판단함
- 구분 포인트: [[cern]]은 Latin `cernere` 계열. `discern`, `certain`, `secret` 등에 보임.

### jud

- 뜻: 법이나 기준에 따라 판단함
- 구분 포인트: [[jud]]는 재판·판결 이미지가 강함. `crit`은 평가·비평·분기점 판단 이미지가 강함.

### cid

- 뜻: 자르다, 잘라 경계를 냄
- 구분 포인트: [[cid]]는 물리적·비유적 절단. `crit`은 갈라 보아 판단함.
