# come

- 출발 단어: `income`
- 핵심 어원: Old English `cuman` = come
- 핵심 이미지: 어떤 쪽으로 오거나, 어떤 상태에 이르게 됨
- 관련해서 헷갈릴 수 있는 어근: [[ven]] = 오다 계열 라틴어, [[curr]] = 달리다, 흐르다 계열, [[over]] = 위로/넘어서 접두사

## 출발 단어 분석

`income`은 `in` + `come`.

- `in`: 안으로
- `come`: 오다

따라서 `income`은 원래 "안으로 들어오는 것"의 뜻. 여기서 개인이나 조직으로 들어오는 돈 → "소득, 수입"의 뜻.

## 단어들

### income

- 구성: `in` + `come`
- 뜻: 소득, 수입
- 어원적 연결: 안으로 들어오는 돈
- 사용 맥락: `annual income` = 연간 소득

### incoming

- 구성: `in` + `coming`
- 뜻: 들어오는, 새로 부임하는
- 어원적 연결: 안쪽으로 오고 있는 상태
- 사용 맥락: `incoming calls` = 걸려 오는 전화, `the incoming president` = 새로 취임하는 대통령

### outcome

- 구성: `out` + `come`
- 뜻: 결과
- 어원적 연결: 어떤 과정에서 밖으로 나온 것
- 사용 맥락: `a positive outcome` = 긍정적인 결과

### become

- 구성: `be-` + `come`
- 뜻: ~이 되다
- 어원적 연결: 어떤 상태에 이르러 감
- 사용 맥락: `become famous` = 유명해지다

### newcomer

- 구성: `new` + `come` + `-er`
- 뜻: 새로 온 사람, 신참
- 어원적 연결: 새로 온 사람
- 사용 맥락: `welcome newcomers` = 새로 온 사람들을 환영하다

### comeback

- 구성: `come` + `back`
- 뜻: 복귀, 재기
- 어원적 연결: 다시 돌아옴
- 사용 맥락: `make a comeback` = 재기하다

## 기억 포인트

`come` 계열은 "오다, 이르다" 이미지.

- `income`: 안으로 들어오는 돈
- `incoming`: 들어오는
- `outcome`: 밖으로 나온 결과
- `become`: 어떤 상태에 이르다
- `newcomer`: 새로 온 사람
- `comeback`: 다시 돌아옴

## 비교 단어

### revenue

- 뜻: 수입, 세입
- 구분 포인트: [[ven]] 계열. 다시 돌아오는 돈의 이미지. `income`은 개인이나 조직 안으로 들어오는 돈이라는 기본어 이미지

### current

- 뜻: 흐름, 현재의
- 구분 포인트: [[curr]] 계열. 돈이나 흐름이 움직이는 이미지. `income`은 들어오는 방향에 초점
