# sovereign

- 출발 단어: `sovereignty`
- 핵심 어원: Old French `soverain` < Latin `superanus` = 위에 있는, 가장 높은
- 핵심 의미: 최고 권위, 스스로 다스리는 힘
- 관련 문서: [[super]], [[reg]], [[crat]], [[jur]]
- 주의: `reign`과 철자가 닮고 의미도 가깝지만, `sovereign`의 핵심은 "위에 있는 최고 권위" 이미지.

## 출발 단어 분석

`sovereignty` = `sovereign` + `-ty`

- `sovereign`: 최고 권력을 가진, 주권자
- `-ty`: 상태, 성질
- 원래 이미지: 다른 권위보다 위에 있는 상태
- 의미 확장: 군주의 최고권 → 국가가 스스로 다스리는 주권

## 단어들

### sovereignty

- 발음: /ˈsɑvrənti/
- 구성: `sovereign` + `-ty`
- 뜻: 주권, 자주권, 최고 권력
- 어원적 연결: 외부의 더 높은 권위 없이 스스로 다스리는 권리
- 사용 맥락: `national sovereignty` = 국가 주권

### sovereign

- 발음: /ˈsɑvrən/
- 구성: Old French `soverain`
- 뜻: 주권의, 자주적인; 군주, 통치자
- 어원적 연결: 가장 위에 있어 최종 권위를 가짐
- 사용 맥락: `a sovereign state` = 주권 국가

### sovereign state

- 구성: `sovereign` + `state`
- 뜻: 주권 국가
- 어원적 연결: 외부 지배가 아닌 독립된 최고 권위를 가진 정치 단위
- 사용 맥락: 국제법·외교 맥락

### sovereign power

- 구성: `sovereign` + `power`
- 뜻: 최고 통치권, 주권적 권한
- 어원적 연결: 법과 정치 질서에서 최종 결정권을 가짐
- 사용 맥락: `exercise sovereign power` = 주권적 권한을 행사하다

### suzerain

- 구성: Old French `suzerain`, `sus` = above
- 뜻: 종주국, 상위 지배자
- 어원적 연결: 다른 정치체 위에 있는 지배 권위
- 사용 맥락: 역사·국제관계 문어. `sovereign`보다 종속 관계가 뚜렷함

### supremacy

- 발음: /səˈprɛməsi/
- 구성: Latin `supremus` = highest
- 뜻: 우위, 최고권, 지상권
- 어원적 연결: 가장 위에 있음
- 사용 맥락: `constitutional supremacy` = 헌법 우위

### supreme

- 발음: /səˈprim/
- 구성: Latin `supremus`
- 뜻: 최고의, 최상위의
- 어원적 연결: 위계의 꼭대기에 있음
- 사용 맥락: `the Supreme Court` = 대법원

## 기억 포인트

`sovereign` 계열은 "가장 위의 권위" 이미지.

- `sovereignty`: 주권
- `sovereign`: 주권적, 군주
- `suzerain`: 위에서 지배하는 종주권자
- `supreme`: 최상위의

## 비교 단어

### reign

- 뜻: 통치, 재위
- 구분 포인트: [[reg]] 계열. 왕이 다스리는 행위. `sovereign`은 최고 권위의 상태.
