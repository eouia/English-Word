# chrom

- 출발 단어: `monochrome`
- 핵심 어원: Greek `chroma` = 색
- 핵심 이미지: 색, 색깔로 드러나는 특징, 염색되어 보이는 것
- 관련 문서: [[alb]], [[melan]], [[mono]], [[poly]]
- 주의: `chromosome`은 현대 의미가 "염색체"이지만, 이름은 현미경 관찰에서 염색되어 잘 보이는 구조에서 출발.

## 출발 단어 분석

`monochrome` = `mono` + `chrome`

- `mono`: 하나
- `chrome`: 색
- 원래 이미지: 하나의 색만 가진 상태
- 현대 의미: 단색의, 흑백의

## 단어들

### monochrome

- 발음: /ˈmɑnəˌkroʊm/
- 구성: Greek `mono` = one + `chroma` = color
- 뜻: 단색의, 흑백의
- 어원적 연결: 하나의 색조만 사용함
- 사용 맥락: `a monochrome image` = 단색 이미지, 흑백 이미지

### polychrome

- 발음: /ˈpɑˌliˌkroʊm/
- 구성: Greek `poly` = many + `chroma` = color
- 뜻: 여러 색의, 다색의
- 어원적 연결: 여러 색을 함께 사용함
- 사용 맥락: 미술·건축 장식에서 자주 보임

### chromatic

- 구성: Greek `chroma` = color + `-ic`
- 뜻: 색채의, 반음계의
- 어원적 연결: 색과 관련됨. 음악에서는 음 사이를 촘촘히 채우는 "색채감" 이미지로 확장.
- 사용 맥락: `chromatic scale` = 반음계

### achromatic

- 구성: `a-` = without + `chromatic`
- 뜻: 무색의, 색수차가 없는
- 어원적 연결: 색이 없거나 색 번짐을 없앤 상태
- 사용 맥락: 과학·광학 맥락에서 쓰임

### chromatography

- 발음: /kroʊməˈtɑgrəfi/
- 구성: `chroma` = color + `graphy` = writing, recording
- 뜻: 크로마토그래피, 색층 분석
- 어원적 연결: 성분이 색 띠처럼 분리되어 기록되는 실험법에서 출발
- 사용 맥락: 화학 실험에서 물질 분리·분석에 쓰임

### chromosome

- 발음: /ˈkroʊməˌsoʊm/
- 구성: `chroma` = color + `soma` = body
- 뜻: 염색체
- 어원적 연결: 염색약에 잘 물들어 보이는 세포 안의 "몸체"
- 사용 맥락: `human chromosomes` = 인간 염색체

### chrome

- 발음: /ˈkroʊm/
- 구성: Greek `chroma` 계열
- 뜻: 크롬, 크롬 도금, 금속성 광택
- 어원적 연결: 크롬 화합물이 강한 색을 띠는 데서 이름이 붙음
- 사용 맥락: `chrome plating` = 크롬 도금

### chroma

- 구성: Greek `chroma` = color
- 뜻: 색도, 채도
- 어원적 연결: 색의 선명함이나 순도
- 사용 맥락: 디자인·색채 이론에서 쓰임

## 기억 포인트

`chrom` 계열은 "색" 이미지.

- `monochrome`: 하나의 색
- `polychrome`: 여러 색
- `chromatic`: 색채의
- `chromosome`: 염색되어 보이는 몸체
- `chromatography`: 색 띠처럼 분리해 기록함

## 비교 단어

### color

- 뜻: 색, 물들이다
- 구분 포인트: `color`는 일상어 중심. `chrom`은 학술어·기술어에서 "색"을 나타내는 Greek 계열.

### melan

- 뜻: 검은색, 어두운 색
- 구분 포인트: [[melan]]은 색 전체가 아니라 검은색·어두운 색에 집중.
