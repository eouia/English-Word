# aqua

- 출발 단어: `aquatic`
- 핵심 어원: Latin `aqua` = 물
- 핵심 이미지: 물과 물속 환경
- 관련 문서: [[hydr]], [[duc]]

## 단어들

### aquatic

- 발음: /əˈkwɑtɪk/
- 구성: Latin `aqua` + `-tic`
- 뜻: 물의, 수생의
- 어원적 연결: 물속이나 물가에 속함
- 사용 맥락: `aquatic plants` = 수생 식물

### aquarium

- 발음: /əˈkwɛriəm/
- 구성: Latin `aqua` + `-arium`
- 뜻: 수족관, 어항
- 어원적 연결: 물을 담아 생물을 기르는 곳
- 사용 맥락: `visit an aquarium` = 수족관을 방문하다

### aqueduct

- 발음: /ˈækwəˌdʌkt/
- 구성: `aqua` = water + [[duc]] = lead
- 뜻: 수로, 송수로
- 어원적 연결: 물을 이끌어 가는 구조물
- 사용 맥락: `Roman aqueducts` = 로마 수로

### aqueous

- 발음: /ˈeɪkwiəs/
- 구성: Latin `aqua`
- 뜻: 물의, 수용성의
- 어원적 연결: 물과 같은 성질이나 물에 녹은 상태
- 사용 맥락: `aqueous solution` = 수용액

### aquifer

- 발음: /ˈækwəfɚ/
- 구성: `aqua` = water + [[fer]] = carry
- 뜻: 대수층
- 어원적 연결: 물을 지니고 운반하는 지층
- 사용 맥락: `groundwater aquifer` = 지하수 대수층

## 기억 포인트

`aqua` 계열은 "물" 이미지.

- `aquatic`: 물에 사는
- `aquarium`: 물을 담은 곳
- `aqueduct`: 물을 이끄는 수로
- `aqueous`: 물의
- `aquifer`: 물을 품은 지층

## 비교 단어

### hydr

- 뜻: 물
- 구분 포인트: [[hydr]]는 그리스어 물 계열. `aqua`는 라틴어 물 계열.
