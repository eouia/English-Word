# flect

- 출발 단어: `reflect`
- 핵심 어원: 라틴어 `flectere`, `flexus` = 굽히다, 휘다
- 핵심 이미지: 방향이 꺾임 / 되돌아 굽어 옴
- 관련해서 헷갈릴 수 있는 어근: [[plic]] = 접다, 접어 붙이다 계열, [[vers]] = 돌다, 방향을 바꾸다 계열

## 출발 단어 분석

`reflect`는 `re-` + `flect`.

- `re-`: back
- `flect`: bend

따라서 `reflect`는 원래 "뒤로 굽히다, 되돌려 보내다"의 뜻. 여기서 빛이나 소리가 되돌아옴 → "반사하다", 생각이 마음속으로 되돌아감 → "숙고하다", 어떤 성질을 되비쳐 보임 → "반영하다"의 뜻.

## 단어들

### reflect

- 발음: /rɪˈflɛkt/
- 구성: `re-` = back + `flect`
- 뜻: 반사하다, 반영하다, 숙고하다
- 어원적 연결: 빛·소리·생각이 뒤로 굽어 되돌아옴
- 사용 맥락: `reflect light` = 빛을 반사하다, `reflect on the past` = 과거를 숙고하다

### reflection

- 발음: /rɪˈflɛkʃən/
- 구성: `reflect` + `-ion`
- 뜻: 반사, 반영, 숙고
- 어원적 연결: 되돌아 굽어 온 것, 또는 마음속으로 되돌려 생각함
- 사용 맥락: `see your reflection` = 자신의 반사된 모습을 보다

### reflective

- 발음: /rɪˈflɛktɪv/
- 구성: `reflect` + `-ive`
- 뜻: 반사하는, 사색적인
- 어원적 연결: 빛을 되돌리거나 생각을 되돌려 보는 성질
- 사용 맥락: `reflective material` = 반사 소재, `a reflective mood` = 사색적인 기분

### deflect

- 발음: /dɪˈflɛkt/
- 구성: `de-` = away + `flect`
- 뜻: 빗나가게 하다, 방향을 바꾸다
- 어원적 연결: 원래 방향에서 꺾어 옆으로 보냄
- 사용 맥락: `deflect criticism` = 비판을 피하다

### deflection

- 발음: /dɪˈflɛkʃən/
- 구성: `deflect` + `-ion`
- 뜻: 편향, 굴절, 방향 전환
- 어원적 연결: 흐름이나 힘이 꺾여 다른 방향으로 감
- 사용 맥락: `the deflection of light` = 빛의 편향

### inflect

- 구성: `in-` = in, into + `flect`
- 뜻: 어형 변화시키다, 억양을 바꾸다
- 어원적 연결: 말의 형태나 소리를 안쪽으로 굽혀 변화시킴
- 사용 맥락: `inflect a verb` = 동사를 굴절시키다

### inflection

- 발음: /ˌɪˈnflɛkʃən/
- 구성: `inflect` + `-ion`
- 뜻: 어형 변화, 억양 변화
- 어원적 연결: 단어나 목소리의 흐름이 꺾여 변함
- 사용 맥락: `a rising inflection` = 올라가는 억양

### flexible

- 발음: /ˈflɛksəbəl/
- 구성: Latin `flexus` = bent + `-ible`
- 뜻: 유연한, 융통성 있는
- 어원적 연결: 잘 휘거나 방향을 바꿀 수 있음
- 사용 맥락: `a flexible schedule` = 유연한 일정

### flexibility

- 발음: /ˌflɛksəˈbɪləti/
- 구성: `flexible` + `-ity`
- 뜻: 유연성, 융통성
- 어원적 연결: 상황에 따라 휘어 적응할 수 있는 성질
- 사용 맥락: `workplace flexibility` = 직장 내 유연성

## 기억 포인트

`flect/flex` 계열은 "굽히다, 휘다" 이미지.

- `reflect`: 뒤로 굽어 되돌아옴 → 반사하다, 숙고하다
- `deflect`: 옆으로 굽힘 → 빗나가게 하다
- `inflect`: 형태나 소리를 굽힘 → 어형/억양을 바꾸다
- `flexible`: 잘 휨 → 유연한

## 비교 단어

### bend

- 뜻: 굽히다, 휘다
- 구분 포인트: 영어 기본어. `flect/flex`는 라틴계 시험 어휘에서 `reflect`, `deflect`, `flexible`처럼 나타남

### fold

- 뜻: 접다
- 구분 포인트: [[plic]]은 접어서 겹치거나 안에 넣는 이미지. `flect`는 방향이 꺾이거나 휘는 이미지

### reverse

- 뜻: 뒤집다, 반대로 하다
- 구분 포인트: [[vers]]는 방향을 돌리는 이미지. `reflect`는 굽어 되돌아오는 이미지가 더 강함
