# melior

- 출발 단어: `ameliorate`
- 핵심 어원: Latin `melior` = better
- 핵심 이미지: 더 나은 상태, 개선됨
- 관련 문서: 없음
- 주의: `melior`는 "더 나은" 뜻. [[melan]]의 `melan-` = 검은색 계열과 철자가 비슷하지만 무관.

## 출발 단어 분석

`ameliorate` = `a-` + `melior` + `-ate`

- `a-`: to, toward
- `melior`: better
- `-ate`: 동사화
- 원래 이미지: 더 나은 쪽으로 만들다
- 현대 의미: 개선하다, 완화하다

## 단어들

### ameliorate

- 발음: /əˈmiljɚˌeɪt/
- 구성: `a-` = to + Latin `melior` = better + `-ate`
- 뜻: 개선하다, 완화하다
- 어원적 연결: 나쁜 상태를 더 나은 상태로 만듦
- 사용 맥락: `ameliorate poverty` = 빈곤을 완화하다

### amelioration

- 발음: /əˌmiljɚˈeɪʃən/
- 구성: `ameliorate` + `-ion`
- 뜻: 개선, 완화
- 어원적 연결: 더 나은 상태가 되는 과정
- 사용 맥락: `social amelioration` = 사회적 개선

### ameliorative

- 구성: `ameliorate` + `-ive`
- 뜻: 개선하는, 완화하는
- 어원적 연결: 나쁜 상태를 낫게 만드는 성질
- 사용 맥락: `ameliorative policies` = 개선 정책

### meliorate

- 구성: Latin `melior` = better + `-ate`
- 뜻: 개선하다, 좋아지다
- 어원적 연결: 더 나은 상태로 바뀜
- 사용 맥락: `ameliorate`보다 드물게 쓰임

### meliorism

- 구성: `melior` = better + `-ism`
- 뜻: 개선주의, 세계는 노력으로 더 좋아질 수 있다는 관점
- 어원적 연결: 더 나아질 수 있다는 믿음
- 사용 맥락: 철학·사상 맥락

## 기억 포인트

`melior` 계열은 "더 나은 상태로 개선" 이미지.

- `ameliorate`: 나아지게 하다
- `amelioration`: 개선
- `ameliorative`: 개선하는
- `meliorate`: 개선하다
- `meliorism`: 개선 가능성을 믿는 관점

## 비교 단어

### better

- 뜻: 더 나은
- 구분 포인트: `better`는 영어 기본어. `ameliorate`는 공식적·학술적 문맥에서 "상황을 개선하다/완화하다"의 뜻으로 자주 씀.

### melan

- 뜻: 검은색, 어두운 색
- 구분 포인트: [[melan]]은 Greek `melan-` = black. `melior` = better와 무관.
