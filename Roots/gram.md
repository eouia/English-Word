# gram

- 출발 단어: `grammar`
- 핵심 어원: Greek `gramma` = 쓰인 것, 글자
- 핵심 이미지: 글자로 적힌 것과 기록된 단위
- 관련 문서: [[graph]], [[scrib]], [[liter]]
- 주의: 무게 단위 `gram`은 프랑스어 `gramme`에서 온 별도 단어지만, 궁극적으로 Greek `gramma`와 연결됨.

## 출발 단어 분석

`grammar` ← Greek `grammatike`

- `gramma`: letter, written mark
- `grammatike`: art of letters
- 원래 이미지: 글자를 다루는 기술
- 현대 의미: 문법

## 단어들

### grammar

- 구성: Greek `gramma`
- 뜻: 문법
- 어원적 연결: 글자와 문장을 다루는 규칙
- 사용 맥락: `English grammar` = 영어 문법

### grammatical

- 구성: `grammar` + `-ical`
- 뜻: 문법적인
- 어원적 연결: 문법 규칙에 맞음
- 사용 맥락: `a grammatical error` = 문법 오류

### diagram

- 구성: [[dia]] = through, across + `gramma`
- 뜻: 도표, 도식
- 어원적 연결: 선을 그어 설명한 기록
- 사용 맥락: `draw a diagram` = 도표를 그리다

### telegram

- 구성: [[tele]] = far + `gramma`
- 뜻: 전보
- 어원적 연결: 멀리 보낸 글
- 사용 맥락: 역사적 통신 맥락에서 쓰임

### program

- 구성: `pro-` = before + `gramma`
- 뜻: 프로그램, 계획표
- 어원적 연결: 미리 써 둔 일정이나 지시
- 사용 맥락: `a computer program` = 컴퓨터 프로그램

### anagram

- 구성: `ana-` = again, back + `gramma`
- 뜻: 철자 바꾸기
- 어원적 연결: 글자를 다시 배열함
- 사용 맥락: `listen` and `silent` as anagrams = listen과 silent의 애너그램

### monogram

- 구성: `mono-` = one + `gramma`
- 뜻: 모노그램, 이름 머리글자 문양
- 어원적 연결: 하나로 결합한 글자 표시
- 사용 맥락: `a monogrammed shirt` = 이니셜이 새겨진 셔츠

### epigram

- 구성: `epi-` = upon + `gramma`
- 뜻: 경구, 짧은 풍자시
- 어원적 연결: 새겨 적은 짧은 글
- 사용 맥락: 문학 맥락에서 쓰임

## 기억 포인트

`gram` 계열은 "쓰인 글자와 기록" 이미지.

- `grammar`: 글자의 기술 → 문법
- `diagram`: 그려 적은 설명
- `telegram`: 멀리 보낸 글
- `program`: 미리 써 둔 계획
- `anagram`: 글자를 다시 배열함

## 비교 단어

### graph

- 뜻: 쓰다, 그리다, 기록하다
- 구분 포인트: [[graph]]는 쓰고 그리는 행위. `gram`은 쓰인 글자와 기록물.

### liter

- 뜻: 글자, 글, 문학
- 구분 포인트: [[liter]]는 라틴어 글자 계열. `gram`은 그리스어 글자 계열.
