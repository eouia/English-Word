# tri

- 출발 단어: `triangle`
- 핵심 어원: Latin / Greek `tri-` = 셋, 세 번, 세 겹
- 깊은 어근: PIE `*trei-` = 셋
- 핵심 이미지: 셋으로 된 구조
- 관련 문서: [[bi]], [[di]], [[quad]], [[angl]]
- 주의: `trivial`은 [[via]]의 `trivium` = 세 길이 만나는 곳 계열로 `tri-` 의미도 보이지만, 본체는 길/교차로 이미지.

## 출발 단어 분석

`triangle` = `tri-` + `angle`

- `tri-`: 셋
- `angle`: 각
- 원래 이미지: 세 각을 가진 도형
- 현대 의미: 삼각형

## 단어들

### triangle

- 구성: `tri-` = three + [[angl]] = angle
- 뜻: 삼각형
- 어원적 연결: 세 각을 가진 도형
- 사용 맥락: `a right triangle` = 직각삼각형

### trilogy

- 구성: `tri-` = three + Greek `logos` = story, speech
- 뜻: 3부작
- 어원적 연결: 세 작품으로 이루어진 이야기 묶음
- 사용 맥락: `a film trilogy` = 영화 3부작

### tricycle

- 구성: `tri-` = three + Greek `kyklos` = wheel
- 뜻: 세발자전거
- 어원적 연결: 바퀴가 세 개
- 사용 맥락: `a child's tricycle` = 어린이 세발자전거

### tripod

- 구성: `tri-` = three + Greek `pod` = foot
- 뜻: 삼각대
- 어원적 연결: 발이 세 개
- 사용 맥락: `set up a tripod` = 삼각대를 세우다

### triple

- 구성: Latin `triplus`
- 뜻: 세 배의, 세 겹의
- 어원적 연결: 셋으로 늘어남
- 사용 맥락: `triple the amount` = 양을 세 배로 하다

### triplicate

- 구성: `tri-` = three + `plic` = fold
- 뜻: 세 부로 된, 세 통 작성하다
- 어원적 연결: 세 겹으로 접히거나 세 벌로 만듦
- 사용 맥락: `in triplicate` = 세 부로

### trilingual

- 구성: `tri-` = three + Latin `lingua` = language
- 뜻: 세 언어를 쓰는
- 어원적 연결: 세 언어에 속함
- 사용 맥락: `a trilingual child` = 세 언어를 쓰는 아이

### triennial

- 구성: `tri-` = three + Latin `annus` = year
- 뜻: 3년마다의
- 어원적 연결: 세 해 단위로 반복됨
- 사용 맥락: `a triennial conference` = 3년마다 열리는 회의

### triumvirate

- 구성: Latin `tres` = three + `vir` = man
- 뜻: 삼두 정치, 세 사람의 지배 집단
- 어원적 연결: 세 남자 또는 세 인물이 권력을 나눔
- 사용 맥락: `a political triumvirate` = 세 사람의 정치 권력 집단

### trinity

- 구성: Latin `trinitas`
- 뜻: 셋이 하나인 집합, 삼위일체
- 어원적 연결: 셋으로 이루어진 하나의 묶음
- 사용 맥락: `the Holy Trinity` = 성삼위일체

## 기억 포인트

`tri-`는 "셋" 이미지.

- `triangle`: 세 각
- `trilogy`: 세 이야기
- `tricycle`: 세 바퀴
- `tripod`: 세 발
- `triple`: 세 배
- `trilingual`: 세 언어

## 비교 단어

### quad

- 뜻: 넷
- 구분 포인트: [[quad]]는 `square`, `quarter`, `quadrant`처럼 네 개·네 부분.

### via

- 뜻: 길
- 구분 포인트: [[via]]의 `trivial`은 `tri-` + `via` = 세 길이 만나는 곳에서 출발. `tri-`도 보이지만 핵심 이미지는 교차로와 흔함.
