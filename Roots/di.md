# di

- 출발 단어: `dioxide`
- 핵심 어원: Greek `di-` = 둘, 두 번, 두 겹
- 깊은 어근: PIE `*dwo-` = 둘
- 핵심 이미지: 둘로 된 것, 두 번 반복됨
- 관련 문서: [[dia]], [[bi]], [[du]], [[tri]]
- 주의: `di-` = two와 `dia-` = through/across는 구분. `dialogue`, `diagnosis`, `diameter`는 보통 `dia-` 계열.

## 출발 단어 분석

`dioxide` = `di-` + `oxide`

- `di-`: 둘
- `oxide`: 산소와 결합한 화합물
- 원래 이미지: 산소 원자가 두 개 있음
- 현대 의미: 이산화물

## 단어들

### dioxide

- 구성: `di-` = two + `oxide`
- 뜻: 이산화물
- 어원적 연결: 산소 원자가 두 개 들어감
- 사용 맥락: `carbon dioxide` = 이산화탄소

### dichotomy

- 구성: Greek `dicha` = in two + `tome` = cutting
- 뜻: 양분, 이분법
- 어원적 연결: 둘로 잘라 나눔
- 사용 맥락: `a false dichotomy` = 잘못된 이분법

### dilemma

- 구성: Greek `di-` = two + `lemma` = premise
- 뜻: 딜레마, 난처한 양자택일
- 어원적 연결: 두 전제나 선택지 사이에 놓임
- 사용 맥락: `face a moral dilemma` = 도덕적 딜레마에 직면하다

### diphthong

- 구성: `di-` = two + Greek `phthongos` = sound
- 뜻: 이중모음
- 어원적 연결: 한 음절 안에서 두 모음 소리가 이어짐
- 사용 맥락: `English diphthongs` = 영어 이중모음

### diode

- 구성: `di-` = two + `-ode` = way
- 뜻: 다이오드
- 어원적 연결: 두 전극을 가진 전자 부품
- 사용 맥락: `a light-emitting diode` = 발광 다이오드

### diploid

- 구성: `di-` = two + Greek `ploos` = fold
- 뜻: 이배체의
- 어원적 연결: 염색체 세트가 두 벌 있음
- 사용 맥락: `diploid cells` = 이배체 세포

### digraph

- 구성: `di-` = two + Greek `graph` = write
- 뜻: 두 글자가 한 소리를 나타내는 철자 묶음
- 어원적 연결: 두 글자가 한 단위처럼 작용함
- 사용 맥락: `sh is a digraph` = sh는 이중문자다

### dimorphism

- 구성: `di-` = two + Greek `morphe` = form
- 뜻: 이형성
- 어원적 연결: 두 가지 형태가 있음
- 사용 맥락: `sexual dimorphism` = 성적 이형성

## 기억 포인트

`di-`는 Greek "둘" 이미지.

- `dioxide`: 산소 둘
- `dichotomy`: 둘로 자름
- `dilemma`: 두 선택지 사이
- `diphthong`: 두 모음 소리
- `digraph`: 두 글자

## 비교 단어

### bi

- 뜻: 둘
- 구분 포인트: [[bi]]는 Latin `bi-` 계열. `binary`, `bicycle`, `bilingual` 등.

### dia

- 뜻: 가로질러, 통과하여, 사이로
- 구분 포인트: [[dia]]는 `dialogue`, `diagnosis`, `diameter`처럼 가로지름·통과·사이의 이미지. `di-` = two와 분리.
