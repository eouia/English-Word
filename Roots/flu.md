# flu

- 출발 단어: `influence`
- 핵심 어원: 라틴어 `fluere` / `flux-` = 흐르다
- 핵심 이미지: 액체처럼 흘러 들어가거나 흘러나옴
- 관련해서 헷갈릴 수 있는 어근: [[curr]] = 흐르다, 달리다 계열

## 출발 단어 분석

`influence`는 Latin `influere`에서 온 말.

- `in-`: into
- `fluere`: flow

따라서 `influence`는 원래 "안으로 흘러 들어감" → "보이지 않게 작용함, 영향"의 뜻.

## 단어들

### influence

- 구성: `in-` = into + `flu`
- 뜻: 영향, 영향을 미치다
- 어원적 연결: 어떤 힘이나 생각이 안으로 흘러 들어와 작용함
- 사용 맥락: `influence public opinion` = 여론에 영향을 미치다

### influential

- 구성: `influence` + `-ial`
- 뜻: 영향력 있는
- 어원적 연결: 다른 사람이나 상황에 힘이 흘러 들어가 작용함
- 사용 맥락: `an influential leader` = 영향력 있는 지도자

### fluent

- 구성: Latin `fluens` = flowing
- 뜻: 유창한
- 어원적 연결: 말이 막히지 않고 흐름
- 사용 맥락: `fluent in Spanish` = 스페인어에 유창한

### fluency

- 구성: `fluent` + `-cy`
- 뜻: 유창함
- 어원적 연결: 말이나 표현이 자연스럽게 흐르는 상태
- 사용 맥락: `reading fluency` = 읽기 유창성

### fluid

- 구성: Latin `fluidus`
- 뜻: 액체, 유동적인
- 어원적 연결: 흐를 수 있는 것
- 사용 맥락: `fluid movement` = 유연한 움직임

### fluidity

- 구성: `fluid` + `-ity`
- 뜻: 유동성
- 어원적 연결: 고정되지 않고 흐르듯 바뀌는 성질
- 사용 맥락: `social fluidity` = 사회적 유동성

### fluctuate

- 구성: Latin `fluctus` = wave
- 뜻: 변동하다, 오르내리다
- 어원적 연결: 물결처럼 오르내리며 흐름
- 사용 맥락: `prices fluctuate` = 가격이 변동하다

### fluctuation

- 구성: `fluctuate` + `-ion`
- 뜻: 변동, 오르내림
- 어원적 연결: 흐름이 파도처럼 흔들리는 상태
- 사용 맥락: `market fluctuations` = 시장 변동

### influx

- 구성: `in-` = into + `flux`
- 뜻: 유입, 쇄도
- 어원적 연결: 안으로 흘러 들어옴
- 사용 맥락: `an influx of tourists` = 관광객의 유입

### reflux

- 구성: `re-` = back + `flux`
- 뜻: 역류
- 어원적 연결: 뒤로 흘러감
- 사용 맥락: `acid reflux` = 위산 역류

### superfluous

- 구성: `super-` = over + `flu`
- 뜻: 불필요한, 남아도는
- 어원적 연결: 필요 이상으로 넘쳐 흐름
- 사용 맥락: `superfluous details` = 불필요한 세부사항

### affluent

- 구성: `ad-` = to + `flu`
- 뜻: 부유한
- 어원적 연결: 재물이 흘러 들어옴
- 사용 맥락: `an affluent neighborhood` = 부유한 동네

### effluent

- 구성: `ex-` = out + `flu`
- 뜻: 폐수, 유출물
- 어원적 연결: 밖으로 흘러나가는 것
- 사용 맥락: `industrial effluent` = 산업 폐수

### flow

- 구성: Old English `flowan`
- 뜻: 흐르다, 흐름
- 어원적 연결: 게르만계 기본어. Latin `fluere`와 같은 "흐름" 이미지로 함께 기억하기 좋음
- 사용 맥락: `the flow of traffic` = 교통 흐름

### flood

- 구성: Old English `flod`
- 뜻: 홍수, 범람하다
- 어원적 연결: 게르만계 `flow`의 친척. Latin `fluere`의 직접 자식은 아니지만, 더 오래된 "흐르다" 이미지에서 의미상 가까움
- 사용 맥락: `a flash flood` = 갑작스러운 홍수

## 기억 포인트

`flu` 계열은 "흐름" 이미지.

- `influence`: 안으로 흘러 들어와 작용함
- `fluent`: 말이 흐름
- `fluid`: 흐를 수 있음
- `fluctuate`: 물결처럼 오르내림
- `influx`: 안으로 유입됨
- `superfluous`: 넘쳐 흘러 불필요함
- `affluent`: 재물이 흘러 들어와 부유함
- `flow` / `flood`: 영어 게르만계의 흐름 / 넘쳐흐름

## 비교 단어

### curr

- 뜻: 달리다, 흐르다
- 구분 포인트: [[curr]]는 물살이나 시간처럼 달려가는 흐름. `flu`는 액체처럼 스며들거나 흘러나오는 이미지

### affect

- 뜻: 영향을 미치다
- 구분 포인트: [[fac]] 계열. `affect`는 작용해 상태를 바꿈. `influence`는 보이지 않는 힘이 흘러 들어와 영향을 줌

### flood

- 뜻: 홍수, 범람
- 구분 포인트: `flood`는 Old English 계열이라 Latin `fluere`의 직접 파생어는 아님. 다만 `flow`와 한 가족이고, 학습상 `flu` 문서에서 함께 보면 좋음
