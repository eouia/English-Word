# pet

- 출발 단어: `competent`
- 핵심 어원: Latin `petere` = seek, go toward, ask
- 핵심 이미지: 어떤 목표를 향해 가거나 요구함
- 관련 문서: [[apt]], [[con]], [[manere]]
- 주의: `pet` = 애완동물은 별도 어원. 이 문서는 Latin `petere` 계열.

## 출발 단어 분석

`competent` ← Latin `competere`

- `com-`: together, with
- `petere`: seek, go toward, ask
- `competere`: come together, be suitable, be sufficient
- 원래 이미지: 여러 조건이 함께 맞아 들어감
- 의미 변화: 함께 맞음 → 알맞음, 충분함 → 능력이 있음
- 현대 의미: 유능한, 역량 있는, 적합한

## 단어들

### competent

- 발음: /ˈkɑmpətɪnt/
- 구성: Latin `com-` = together + `petere` = seek, go toward
- 뜻: 유능한, 역량 있는, 적합한
- 어원적 연결: 필요한 조건에 맞아 충분히 해낼 수 있음
- 사용 맥락: `a competent manager` = 유능한 관리자

### competence

- 발음: /ˈkɑmpətɪns/
- 구성: `competent`의 명사형
- 뜻: 능력, 역량, 적격성
- 어원적 연결: 일을 해낼 만큼 조건이 맞고 충분한 상태
- 사용 맥락: `professional competence` = 직무 역량

### competency

- 발음: /ˈkɑmpətɪnsi/
- 구성: `competent` + `-cy`
- 뜻: 역량, 능력 항목
- 어원적 연결: 특정 일을 수행할 수 있는 능력 단위
- 사용 맥락: `core competencies` = 핵심 역량

### compete

- 발음: /kəˈmpit/
- 구성: Latin `com-` = together + `petere` = seek
- 뜻: 경쟁하다
- 어원적 연결: 여러 사람이 같은 목표를 함께 추구함
- 사용 맥락: `compete for a prize` = 상을 두고 경쟁하다

### competition

- 발음: /ˌkɑmpəˈtɪʃən/
- 구성: `compete` + `-ition`
- 뜻: 경쟁, 대회
- 어원적 연결: 같은 목표를 얻기 위해 함께 추구하는 상황
- 사용 맥락: `fierce competition` = 치열한 경쟁

### competitive

- 발음: /kəˈmpɛtətɪv/
- 구성: `compete` + `-itive`
- 뜻: 경쟁적인, 경쟁력 있는
- 어원적 연결: 경쟁 속에서 목표를 얻으려는 성질
- 사용 맥락: `a competitive market` = 경쟁적인 시장

### petition

- 발음: /pəˈtɪʃən/
- 구성: Latin `petitio` = request, attack, seeking
- 뜻: 청원, 탄원; 청원하다
- 어원적 연결: 원하는 것을 공식적으로 요구함
- 사용 맥락: `sign a petition` = 청원서에 서명하다

### appetite

- 발음: /ˈæpəˌtaɪt/
- 구성: Latin `ad-` = toward + `petere` = seek
- 뜻: 식욕, 욕구
- 어원적 연결: 어떤 대상을 향해 강하게 찾고 원함
- 사용 맥락: `lose one's appetite` = 식욕을 잃다

### repetition

- 발음: /ˌrɛpəˈtɪʃən/
- 구성: Latin `re-` = again + `petere` = seek, go toward
- 뜻: 반복
- 어원적 연결: 같은 것을 다시 찾아가거나 다시 함
- 사용 맥락: `repetition helps memory` = 반복은 기억에 도움이 됨

### perpetual

- 발음: /pɚˈpɛtʃuəl/
- 구성: Latin `per-` = through + `petere` = go toward, seek
- 뜻: 끊임없는, 영구적인, 반복되는
- 어원적 연결: 계속 앞으로 이어져 끊기지 않음
- 사용 맥락: `perpetual motion` = 영구 운동, `perpetual conflict` = 끊임없는 갈등

### perpetually

- 발음: /pɚˈpɛtʃuəli/
- 구성: `perpetual` + `-ly`
- 뜻: 끊임없이, 계속해서
- 어원적 연결: 중단 없이 이어지는 방식
- 사용 맥락: `perpetually late` = 항상 늦는

### perpetuity

- 발음: /ˌpɝpɪˈtjuɪti/
- 구성: Latin `perpetuitas` < `perpetuus`
- 뜻: 영구, 영속; 무기한
- 어원적 연결: 끝없이 이어지는 상태
- 사용 맥락: `in perpetuity` = 영구히

### perpetuate

- 발음: /pɚˈpɛtʃəˌweɪt/
- 구성: `perpetual` 계열 + `-ate`
- 뜻: 영속시키다, 계속되게 하다
- 어원적 연결: 어떤 상태나 관행이 끊기지 않고 이어지게 함
- 사용 맥락: `perpetuate a myth` = 잘못된 믿음을 계속 퍼뜨리다

## 기억 포인트

`pet` 계열은 "목표를 향해 찾고 요구함" 이미지.

- `competent`: 조건에 맞아 충분히 해낼 수 있는
- `competence`: 역량
- `compete`: 같은 목표를 추구하며 경쟁하다
- `competition`: 경쟁
- `petition`: 공식적으로 요구함
- `appetite`: 향해 찾는 욕구
- `repetition`: 다시 함, 반복
- `perpetual`: 끊기지 않고 계속 이어짐

## 비교 단어

### apt

- 뜻: 알맞게 맞음, 소질
- 구분 포인트: [[apt]]는 적합하게 맞춰진 성질. `competent`는 필요한 기준에 충분히 맞아 실제 수행 능력이 있음.

### con

- 뜻: 함께, 같이, 완전히
- 구분 포인트: [[con]]은 `com-`의 일반 접두사 문서. `competent`, `compete`에서는 `com-`이 `petere`와 결합해 함께 목표를 향함.

### seek

- 뜻: 찾다, 추구하다
- 구분 포인트: 영어 기본어. `petition`, `competition`, `appetite`는 Latin `petere` 계열의 공식·학술 어휘.

### permanent

- 뜻: 영구적인, 지속적인
- 구분 포인트: [[manere]] 계열. `permanent`는 "계속 남아 있음", `perpetual`은 "끊임없이 이어짐/반복됨" 이미지.
