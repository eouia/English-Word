# vid-see

- 상위 문서: [[vid]]
- 핵심 어원: 라틴어 `videre` / `vis-` = 보다
- 핵심 이미지: 눈으로 봄 / 보이게 드러남 / 다시 봄
- 관련 갈래: [[vid-watch]], [[vid-provide]]

## 단어들

### video

- 발음: /ˈvɪdioʊ/
- 구성: Latin `video` = I see
- 뜻: 영상, 동영상
- 어원적 연결: 눈으로 보는 이미지
- 사용 맥락: `watch a video` = 영상을 보다

### vision

- 발음: /ˈvɪʒən/
- 구성: Latin `visio`
- 뜻: 시력, 시야, 비전
- 어원적 연결: 보는 능력. 나아가 머릿속으로 보는 미래상
- 사용 맥락: `a clear vision for the future` = 미래에 대한 분명한 비전

### visual

- 발음: /ˈvɪʒəwəl/
- 구성: `vis-` + `-ual`
- 뜻: 시각의, 눈에 보이는
- 어원적 연결: 눈으로 보는 것과 관련됨
- 사용 맥락: `visual information` = 시각 정보

### visible

- 발음: /ˈvɪzəbəl/
- 구성: `vis-` + `-ible`
- 뜻: 보이는, 눈에 띄는
- 어원적 연결: 볼 수 있는 상태
- 사용 맥락: `visible changes` = 눈에 보이는 변화

### invisible

- 발음: /ˌɪˈnvɪzəbəl/
- 구성: `in-` = not + `visible`
- 뜻: 보이지 않는
- 어원적 연결: 눈으로 볼 수 없음
- 사용 맥락: `invisible barriers` = 보이지 않는 장벽

### evident

- 발음: /ˈɛvədənt/
- 구성: `e-` = out + `vid`
- 뜻: 명백한, 분명한
- 어원적 연결: 밖으로 드러나 보여서 분명함
- 사용 맥락: `It is evident that...` = ~라는 것이 분명하다

비교: [[clar]]의 `clear`, `clarify`는 밝고 분명한 상태에 초점. `evident`는 보이게 드러남에 초점.

### evidence

- 발음: /ˈɛvədəns/
- 구성: `evident` 계열
- 뜻: 증거
- 어원적 연결: 보이게 드러나 사실을 뒷받침하는 것
- 사용 맥락: `strong evidence` = 강력한 증거

### view

- 발음: /ˈvju/
- 구성: Old French `veue`, Latin `videre`
- 뜻: 보다, 견해, 전망
- 어원적 연결: 눈으로 보는 것에서, 마음으로 보는 관점으로 의미 확대
- 사용 맥락: `in my view` = 내 견해로는

### review

- 발음: /ˌriˈvju/
- 구성: `re-` = again + `view`
- 뜻: 검토하다, 복습하다, 리뷰
- 어원적 연결: 다시 봄
- 사용 맥락: `review the report` = 보고서를 검토하다

### preview

- 발음: /ˈpriˌvju/
- 구성: `pre-` = before + `view`
- 뜻: 예고편, 미리 보기
- 어원적 연결: 본격적으로 보기 전에 먼저 봄
- 사용 맥락: `watch a preview` = 예고편을 보다

### interview

- 발음: /ˈɪntɚˌvju/
- 구성: `inter-` = between + `view`
- 뜻: 면접, 인터뷰
- 어원적 연결: 서로 마주 보며 질문하고 답함
- 사용 맥락: `job interview` = 취업 면접

### advise

- 발음: /æˈdvaɪz/
- 구성: Old French `aviser` ← Latin `ad-` = to + `visere` = look at
- 뜻: 조언하다, 충고하다, 알리다
- 어원적 연결: 살펴보고 판단한 내용을 말해 줌
- 사용 맥락: `advise students` = 학생들에게 조언하다

### advice

- 발음: /æˈdvaɪs/
- 구성: Old French `avis` = opinion, view
- 뜻: 조언, 충고
- 어원적 연결: 보고 판단한 견해
- 사용 맥락: `give advice` = 조언하다

### advisable

- 발음: /əˈdvaɪzəbəl/
- 구성: `advise` + `-able`
- 뜻: 바람직한, 권할 만한
- 어원적 연결: 살펴본 뒤 권할 만함
- 사용 맥락: `It is advisable to wait.` = 기다리는 것이 좋다

### visit

- 발음: /ˈvɪzɪt/
- 구성: Latin `visitare`, from `visere` = go to see
- 뜻: 방문하다
- 어원적 연결: 보러 감
- 사용 맥락: `visit a friend` = 친구를 방문하다

### visitor

- 발음: /ˈvɪzɪtɚ/
- 구성: `visit` + `-or`
- 뜻: 방문자
- 어원적 연결: 보러 온 사람
- 사용 맥락: `museum visitors` = 박물관 방문객

### revise

- 발음: /rɪˈvaɪz/
- 구성: `re-` = again + `vis`
- 뜻: 수정하다, 개정하다, 복습하다
- 어원적 연결: 다시 보고 고침
- 사용 맥락: `revise an essay` = 에세이를 수정하다

### revision

- 발음: /riˈvɪʒən/
- 구성: `revise` + `-ion`
- 뜻: 수정, 개정, 복습
- 어원적 연결: 다시 보며 고치는 과정
- 사용 맥락: `make revisions` = 수정을 하다

## 기억 포인트

`vid-see`는 "눈에 보임 / 다시 봄" 이미지.

- `video`: 보는 것, 영상
- `visible`: 볼 수 있는
- `evidence`: 보이게 드러난 증거
- `view`: 보기, 견해
- `review`: 다시 봄
- `preview`: 미리 봄
- `visit`: 보러 감
- `revise`: 다시 보고 고침
- `advise`: 보고 판단해 조언함
