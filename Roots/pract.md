# pract

- 출발 단어: `impractical`
- 핵심 어원: Greek `praktikos` = fit for action, active; `prassein/prattein` = do, act
- 핵심 이미지: 실제로 행동하고 적용해 봄
- 관련 문서: [[ag]], [[fac]]
- 주의: `practical`은 "실제 행동에 맞는" 이미지. `practicable`은 "실행 가능한" 뜻으로 더 좁음.

## 출발 단어 분석

`impractical` = `im-` + `practical`

- `im-`: not
- `practical`: 실제 사용이나 행동에 맞는
- 원래 이미지: 실제 행동으로 옮기기에 맞지 않음
- 현대 의미: 비현실적인, 실용적이지 않은

## 단어들

### impractical

- 구성: `im-` = not + `practical`
- 뜻: 비현실적인, 실용적이지 않은
- 어원적 연결: 실제 행동이나 사용으로 옮기기 어려움
- 사용 맥락: `an impractical plan` = 비현실적인 계획, `impractical for daily use` = 일상 사용에는 실용적이지 않은

### practical

- 구성: Greek `praktikos` = fit for action
- 뜻: 실제적인, 실용적인
- 어원적 연결: 이론보다 실제 행동과 적용에 맞음
- 사용 맥락: `practical advice` = 실용적인 조언, `practical skills` = 실무 능력

### practically

- 구성: `practical` + `-ly`
- 뜻: 실제로, 사실상; 거의
- 어원적 연결: 실제 상황에서 보면 그러함
- 사용 맥락: `practically impossible` = 사실상 불가능한

### practice

- 구성: Old French `practique` ← Greek `praktikos`
- 뜻: 연습, 실행, 관행; 연습하다, 실행하다
- 어원적 연결: 실제로 반복해 행함
- 사용 맥락: `practice speaking` = 말하기를 연습하다, `medical practice` = 의료 활동

### practiced

- 구성: `practice` + `-ed`
- 뜻: 숙련된, 연습을 많이 한
- 어원적 연결: 반복적으로 실행해 능숙해진 상태
- 사용 맥락: `a practiced speaker` = 숙련된 연설가

### practitioner

- 구성: `practice` + `-er`
- 뜻: 실무자, 전문직 종사자
- 어원적 연결: 지식이나 기술을 실제로 실행하는 사람
- 사용 맥락: `a medical practitioner` = 의료인, `a legal practitioner` = 법률 실무자

### practicable

- 구성: `practice` + `-able`
- 뜻: 실행 가능한, 실제로 할 수 있는
- 어원적 연결: 실제 행동으로 옮길 수 있음
- 사용 맥락: `a practicable solution` = 실행 가능한 해결책

### impracticable

- 구성: `im-` = not + `practicable`
- 뜻: 실행 불가능한
- 어원적 연결: 실제 행동으로 옮길 수 없음
- 사용 맥락: `an impracticable proposal` = 실행 불가능한 제안

### pragmatic

- 구성: Greek `pragmatikos` = skilled in affairs, practical
- 뜻: 실용적인, 현실적인
- 어원적 연결: 실제 일과 결과를 기준으로 판단함
- 사용 맥락: `a pragmatic approach` = 실용적인 접근

### praxis

- 구성: Greek `praxis` = action, practice
- 뜻: 실천, 실제 활동
- 어원적 연결: 이론이 실제 행동으로 나타남
- 사용 맥락: 철학·교육학·사회이론 문맥

## 기억 포인트

`pract`는 "실제로 해 봄" 이미지.

- `practice`: 실제로 반복해 행함
- `practical`: 실제 사용에 맞음
- `impractical`: 실제 사용에 맞지 않음
- `practicable`: 실행할 수 있음
- `pragmatic`: 실제 결과를 중시함
- `praxis`: 이론의 실천

## 비교 단어

### theoretical

- 뜻: 이론적인
- 구분 포인트: 실제 적용보다 원리·이론에 초점. `practical`과 자주 대비됨.

### practicable vs practical

- `practicable`: 실행 가능한지 여부
- `practical`: 쓸모 있고 현실적인지 여부

### malpractice

- 뜻: 전문직 과실, 부정 행위
- 연결 포인트: `mal-` + `practice`. 이미 [[mal]]에서 접두사 예로 다룸.
