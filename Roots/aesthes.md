# aesthes

- 출발 단어: `aesthetic`
- 핵심 어원: Greek `aisthesis` = 감각, 지각
- 핵심 이미지: 감각으로 느끼고 알아차림
- 관련 문서: [[sent]], [[path]], [[psych]]
- 주의: 미국식 `anesthesia`, 영국식 `anaesthesia`처럼 철자가 달라질 수 있음.

## 출발 단어 분석

`aesthetic` ← Greek `aisthesis`

- `aisthesis`: perception, sensation
- 원래 이미지: 감각으로 지각함
- 현대 의미: 미적인, 미학의

## 단어들

### aesthetic

- 구성: Greek `aisthesis` + `-ic`
- 뜻: 미적인, 미학의
- 어원적 연결: 감각으로 느끼는 아름다움과 관련됨
- 사용 맥락: `aesthetic value` = 미적 가치

### aesthetics

- 구성: `aesthetic` + `-s`
- 뜻: 미학, 미적 취향
- 어원적 연결: 감각과 지각을 통해 아름다움을 다룸
- 사용 맥락: `modern aesthetics` = 현대 미학

### anesthetic

- 구성: `an-` = without + `aesthetic`
- 뜻: 마취제, 마취의
- 어원적 연결: 감각이 없는 상태를 만듦
- 사용 맥락: `local anesthetic` = 국소 마취제

### anesthesia

- 구성: `an-` = without + Greek `aisthesis`
- 뜻: 마취, 감각 상실
- 어원적 연결: 느낌과 지각이 없는 상태
- 사용 맥락: `general anesthesia` = 전신 마취

### synesthesia

- 구성: [[sym]] = together + Greek `aisthesis`
- 뜻: 공감각
- 어원적 연결: 여러 감각이 함께 느껴짐
- 사용 맥락: `sound-color synesthesia` = 소리-색 공감각

### kinesthesia

- 구성: Greek `kinein` = move + `aisthesis`
- 뜻: 운동 감각
- 어원적 연결: 몸의 움직임을 감각으로 알아차림
- 사용 맥락: 심리학·운동학 맥락에서 쓰임

## 기억 포인트

`aesthes` 계열은 "감각과 지각" 이미지.

- `aesthetic`: 감각으로 느끼는 아름다움
- `anesthesia`: 감각이 없음
- `anesthetic`: 감각을 없애는 것
- `synesthesia`: 감각이 함께 느껴짐

## 비교 단어

### sent

- 뜻: 느끼다, 알아차리다
- 구분 포인트: [[sent]]는 라틴어 `sentire` 계열. `aesthes`는 그리스어 감각·지각 계열.

### path

- 뜻: 느낌, 고통
- 구분 포인트: [[path]]는 감정과 고통을 겪음. `aesthes`는 감각으로 지각함.
