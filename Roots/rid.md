# rid

- 출발 단어: `ridicule`
- 핵심 어원: Latin `ridere`, past participle `risus` = 웃다
- 관련 형태: `rid-`, `ris-`
- 핵심 이미지: 웃음, 비웃음
- 관련 문서: [[scorn]]

## 출발 단어 분석

`ridicule` ← Latin `ridiculum`

- `ridere`: laugh
- `ridiculus`: laughable
- 원래 이미지: 웃음거리가 되는 것
- 현대 의미: 조롱, 비웃음

## 단어들

### ridicule

- 발음: /ˈrɪdəˌkjul/
- 구성: Latin `ridiculum`
- 뜻: 조롱하다, 비웃다; 조롱
- 어원적 연결: 웃음거리로 만듦
- 사용 맥락: `ridicule an idea` = 생각을 조롱하다

### ridiculous

- 발음: /rɪˈdɪkjələs/
- 구성: `ridicule` 계열 + `-ous`
- 뜻: 터무니없는, 우스꽝스러운
- 어원적 연결: 비웃음을 살 만큼 말이 안 됨
- 사용 맥락: `a ridiculous claim` = 터무니없는 주장

### deride

- 발음: /dɪˈraɪd/
- 구성: `de-` = down + Latin `ridere`
- 뜻: 조롱하다, 비웃다
- 어원적 연결: 깔보며 웃음
- 사용 맥락: `deride the proposal` = 그 제안을 조롱하다

### derision

- 발음: /dɚˈɪʒən/
- 구성: `de-` + Latin `risus`
- 뜻: 조롱, 비웃음
- 어원적 연결: 아래로 내려다보며 웃는 태도
- 사용 맥락: `meet with derision` = 조롱을 받다

### risible

- 구성: Latin `risibilis` = laughable
- 뜻: 우스운, 웃음을 자아내는
- 어원적 연결: 웃을 만한 성질이 있음
- 사용 맥락: `a risible excuse` = 우스운 변명

## 기억 포인트

`rid/ris` 계열은 "웃음과 비웃음" 이미지.

- `ridicule`: 조롱하다
- `ridiculous`: 비웃을 만큼 터무니없는
- `deride`: 깔보며 비웃다
- `derision`: 조롱
- `risible`: 웃음을 자아내는
