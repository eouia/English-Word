# robor

- 출발 단어: `corroborate`
- 핵심 어원: Latin `robur` = oak, strength
- 핵심 이미지: 단단한 참나무처럼 힘을 보탬
- 관련 문서: [[ver]], [[cred]], [[dur]]
- 주의: `robot`과는 무관. `robot`은 Czech `robota` = forced labor 계열.

## 출발 단어 분석

`corroborate` = `cor-` + `robor` + `-ate`

- `cor-`: together, intensive
- `robor`: strength, hard oak
- `-ate`: 동사화
- 원래 이미지: 함께 힘을 보태 단단하게 함
- 현대 의미: 증거로 뒷받침하다, 확증하다

## 단어들

### corroborate

- 발음: /kɚˈɑbɚˌeɪt/
- 구성: Latin `cor-` = together/intensive + `roborare` = strengthen
- 뜻: 확증하다, 뒷받침하다
- 어원적 연결: 주장이나 진술에 힘을 더해 단단하게 만듦
- 사용 맥락: `corroborate the testimony` = 증언을 뒷받침하다

### corroboration

- 발음: /kɚˌɔbɚˈeɪʃən/
- 구성: `corroborate` + `-ion`
- 뜻: 확증, 뒷받침
- 어원적 연결: 다른 증거가 힘을 보태 신뢰성을 높임
- 사용 맥락: `independent corroboration` = 독립적 확증

### corroborative

- 발음: /kɚˈɑbɚəˌtɪv/
- 구성: `corroborate` + `-ive`
- 뜻: 확증하는, 뒷받침하는
- 어원적 연결: 주장에 힘을 더해 주는 성질
- 사용 맥락: `corroborative evidence` = 보강 증거

### robust

- 발음: /roʊˈbʌst/
- 구성: Latin `robustus` = strong, hard, oak-like
- 뜻: 튼튼한, 강건한, 견고한
- 어원적 연결: 참나무처럼 단단하고 힘 있음
- 사용 맥락: `a robust system` = 견고한 시스템

### robustly

- 발음: /ˌroʊˈbʌstli/
- 구성: `robust` + `-ly`
- 뜻: 튼튼하게, 강력하게
- 어원적 연결: 강하고 단단한 방식
- 사용 맥락: `respond robustly` = 강력하게 대응하다

### robustness

- 발음: /roʊˈbʌstnəs/
- 구성: `robust` + `-ness`
- 뜻: 견고함, 강건함
- 어원적 연결: 쉽게 무너지지 않는 힘과 단단함
- 사용 맥락: `model robustness` = 모델의 견고성

## 기억 포인트

`robor` 계열은 "단단한 힘을 보탬" 이미지.

- `corroborate`: 증거로 힘을 보태 확증하다
- `corroboration`: 확증, 뒷받침
- `corroborative`: 확증하는
- `robust`: 튼튼한, 견고한
- `robustness`: 견고함

## 비교 단어

### verify

- 뜻: 확인하다, 검증하다
- 구분 포인트: [[ver]]는 참인지 확인함. `corroborate`는 다른 증거가 힘을 보태 기존 주장이나 증언을 뒷받침함.

### confirm

- 뜻: 확인하다, 확정하다
- 구분 포인트: Latin `firmus` = firm 계열로 단단히 확정함. `corroborate`는 보강 증거로 단단하게 만드는 느낌.

### substantiate

- 뜻: 실증하다, 입증하다
- 구분 포인트: 주장에 실체 있는 근거를 대는 느낌. `corroborate`는 별도 증거가 서로 맞아 지지해 주는 느낌.
