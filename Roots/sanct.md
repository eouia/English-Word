# sanct

- 출발 단어: `sanction`
- 핵심 어원: Latin `sancire` = 신성하게 정하다, 법으로 확정하다, 승인하다
- 과거분사: `sanctus` = holy, consecrated
- 핵심 의미: 신성하게 정해 침범할 수 없게 함
- 관련 문서: [[leg]], [[dict]], [[fac]]
- 주의: `sanction`은 “공식 승인”과 “제재” 둘 다 가능. 둘 다 “법적·권위적으로 확정함”에서 갈라짐.

## 출발 단어 분석

`sanction` ← Latin `sanctio` = 법령, 명령, 제정 행위

- `sancire`: 신성하게 하다, 확정하다, 비준하다
- `sanctio`: 법으로 정한 것, 명령, 조례
- 원래 이미지: 어떤 규칙을 신성하고 침범 불가능한 것으로 세움
- 현대 의미: 승인, 인가; 제재, 처벌

## 단어들

### sanction

- 구성: Latin `sanctio` ← `sancire`
- 뜻: 승인, 인가; 제재, 처벌
- 어원적 연결: 권위로 어떤 행위를 확정함
- 사용 맥락: `official sanction` = 공식 승인, `economic sanctions` = 경제 제재

### sanctions

- 구성: `sanction`의 복수형
- 뜻: 제재, 특히 경제·외교 제재
- 어원적 연결: 법적 권위로 가하는 제한
- 사용 맥락: `impose sanctions` = 제재를 부과하다

### sanctionable

- 구성: `sanction` + `-able`
- 뜻: 제재할 수 있는, 처벌 가능한
- 어원적 연결: 규칙 위반으로 공식 처벌 대상이 됨
- 사용 맥락: legal / policy 맥락

### saint

- 구성: Old French `saint` ← Latin `sanctus`
- 뜻: 성인, 성자
- 어원적 연결: 신성하게 구별된 사람
- 사용 맥락: `Saint Paul` = 성 바울

### sanctity

- 구성: Latin `sanctitas` ← `sanctus`
- 뜻: 신성함, 존엄성
- 어원적 연결: 침범해서는 안 되는 신성한 성질
- 사용 맥락: `the sanctity of life` = 생명의 존엄성

### sanctify

- 구성: `sanct` = holy + [[fac]]
- 뜻: 신성하게 하다, 정당화하다
- 어원적 연결: 어떤 것을 신성한 용도나 정당한 상태로 만듦
- 사용 맥락: `sanctify a place` = 장소를 신성하게 하다

### sanctification

- 구성: `sanctify` + `-ation`
- 뜻: 성화, 신성화
- 어원적 연결: 신성하게 만들어지는 과정
- 사용 맥락: religious context

### sanctuary

- 구성: Latin `sanctuarium` ← `sanctus`
- 뜻: 성소, 보호구역, 피난처
- 어원적 연결: 신성하게 구별되어 보호받는 장소
- 사용 맥락: `a wildlife sanctuary` = 야생동물 보호구역

### sanctum

- 구성: Latin `sanctum` = holy place
- 뜻: 성소, 사적인 방
- 어원적 연결: 함부로 들어갈 수 없는 신성한 내부 공간
- 사용 맥락: `inner sanctum` = 내실, 핵심 내부

### sacrosanct

- 구성: Latin `sacro` = by sacred rite + `sanctus`
- 뜻: 지극히 신성한, 침범할 수 없는
- 어원적 연결: 신성한 의식으로 확정되어 건드릴 수 없음
- 사용 맥락: `a sacrosanct principle` = 침범할 수 없는 원칙

### sanctimonious

- 구성: `sanctimony` + `-ous`
- 뜻: 거룩한 척하는, 독선적인
- 어원적 연결: 신성함을 겉으로 과시하는 태도
- 사용 맥락: `a sanctimonious lecture` = 독선적인 설교

### sanctimony

- 구성: Latin `sanctimonia` = holiness
- 뜻: 거룩한 체함, 위선적 경건
- 어원적 연결: 신성함이 위선적 태도로 변한 말
- 사용 맥락: `moral sanctimony` = 도덕적 독선

## 기억 포인트

- `sanct`의 핵심은 “신성하게 정해 침범 불가로 만듦”.
- `sanction`: 권위로 확정함 → 승인 / 제재.
- `saint`: 신성하게 구별된 사람.
- `sanctuary`: 신성해서 보호받는 장소.
- `sacrosanct`: 건드릴 수 없을 만큼 신성한.

## sanction의 양면 의미

`sanction`은 문맥으로 판단.

- positive sanction: 공식 승인, 허가
- negative sanction: 제재, 처벌

둘 다 “권위가 어떤 행위에 법적 힘을 부여함”이라는 공통점.

## 비교 단어

### sacred / sacr

- `sacred`, `sacrifice`, `sacrament`, `consecrate`는 Latin `sacer` = sacred 계열.
- `sanct`와 깊은 단계에서는 가까운 말이지만, 라틴어 형태는 다르므로 별도 갈래로 봄.
- [[fac]]의 `sacrifice` = 신성하게 만드는 행위.

### leg

- [[leg]]는 법과 정해진 규칙.
- `sanction`은 법적 권위가 승인하거나 제재하는 행위.

### dictate

- [[dict]]의 `dictate`는 말로 명령함.
- `sanction`은 명령을 법적·신성한 효력으로 확정함.
