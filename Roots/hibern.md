# hibern

- 출발 단어: `hibernation`
- 핵심 어원: Latin `hibernus` = wintry, of winter
- 핵심 이미지: 겨울 동안 활동을 낮추고 지냄
- 관련 문서: [[vern]]
- 주의: `hibernate`는 동물의 겨울잠뿐 아니라 컴퓨터·조직 활동을 일시 중지하는 비유에도 씀.

## 출발 단어 분석

`hibernation` = `hibernate` + `-ion`

- `hibernate`: 겨울을 나다, 겨울잠을 자다
- `-ion`: 행위·상태를 나타내는 접미사
- 원래 이미지: 겨울 동안 숨어 지내며 활동을 줄이는 상태
- 현대 의미: 겨울잠, 동면; 활동 중지 상태

## 단어들

### hibernation

- 발음: /ˌhaɪbɚˈneɪʃən/
- 구성: `hibernate` + `-ion`
- 뜻: 겨울잠, 동면; 활동 중지 상태
- 어원적 연결: 겨울 동안 에너지를 아끼며 낮은 활동 상태로 지냄
- 사용 맥락: `bears in hibernation` = 동면 중인 곰, `come out of hibernation` = 활동을 재개하다

### hibernate

- 발음: /ˈhaɪbɚˌneɪt/
- 구성: Latin `hibernare` = pass the winter
- 뜻: 겨울잠을 자다, 동면하다; 일시 중지 상태에 들어가다
- 어원적 연결: 겨울을 견디기 위해 숨어 활동을 낮춤
- 사용 맥락: `animals hibernate in winter` = 동물들은 겨울에 동면한다

### hibernating

- 구성: `hibernate` + `-ing`
- 뜻: 동면 중인, 활동을 멈춘
- 어원적 연결: 겨울잠 상태에 들어가 있는 과정
- 사용 맥락: `a hibernating animal` = 동면 중인 동물

### hibernal

- 구성: Latin `hibernalis` = wintry
- 뜻: 겨울의, 겨울다운
- 어원적 연결: 겨울과 관련된 성질
- 사용 맥락: 문어적·시적 표현. 일반적으로는 `winter`나 `wintry`를 더 많이 씀.

### hibernaculum

- 구성: Latin `hibernaculum` = winter quarters
- 뜻: 동면 장소, 겨울 은신처
- 어원적 연결: 겨울을 나기 위해 머무는 장소
- 사용 맥락: 생물학·생태학 맥락

## 기억 포인트

`hibern`은 "겨울을 나며 활동을 낮춤" 이미지.

- `hibernate`: 겨울잠을 자다
- `hibernation`: 동면, 겨울잠 상태
- `hibernating`: 동면 중인
- `hibernal`: 겨울의
- `hibernaculum`: 겨울 은신처

## 비교 단어

### winter

- 뜻: 겨울; 겨울을 나다
- 구분 포인트: Germanic 기본어. `hibern`은 Latin 계열의 학술적·격식 있는 겨울 이미지.

### vernal

- 뜻: 봄의
- 구분 포인트: [[vern]]의 Latin `ver` = spring 계열. `hibernal`의 계절상 반대 이미지.

### dormancy

- 뜻: 휴면, 정지 상태
- 구분 포인트: 잠들어 있는 상태 자체에 초점. `hibernation`은 겨울을 나는 생물학적 휴면이 중심.
