# geo

- 출발 단어: `geography`
- 핵심 어원: Greek `ge` / `geo` = 땅, 지구
- 핵심 이미지: 지구와 땅을 그리고 설명함
- 관련 문서: [[terr]], [[graph]], [[metr]]

## 단어들

### geography

- 발음: /dʒiˈɑgrəfi/
- 구성: `geo` = earth + [[graph]] = write, describe
- 뜻: 지리학
- 어원적 연결: 지구와 땅을 기록하고 설명함
- 사용 맥락: `study geography` = 지리학을 공부하다

### geology

- 발음: /dʒiˈɑlədʒi/
- 구성: `geo` + [[logos]] = study
- 뜻: 지질학
- 어원적 연결: 지구의 물질과 구조를 연구함
- 사용 맥락: `marine geology` = 해양 지질학

### geometry

- 발음: /dʒiˈɑmətri/
- 구성: `geo` = earth + [[metr]] = measure
- 뜻: 기하학
- 어원적 연결: 원래는 땅을 재는 기술
- 사용 맥락: `Euclidean geometry` = 유클리드 기하학

### geopolitical

- 발음: /ˌdʒioʊpəˈlɪtɪkəl/
- 구성: `geo` + [[polis]]
- 뜻: 지정학적인
- 어원적 연결: 지리와 정치가 결합됨
- 사용 맥락: `geopolitical risk` = 지정학적 위험

### geocentric

- 발음: /ˌdʒioʊˈsɛntrɪk/
- 구성: `geo` = earth + `centric` = centered
- 뜻: 지구 중심의
- 어원적 연결: 지구를 중심으로 봄
- 사용 맥락: `geocentric model` = 지구 중심 모델

## 기억 포인트

`geo` 계열은 "지구와 땅" 이미지.

- `geography`: 땅을 기록함
- `geology`: 지구를 연구함
- `geometry`: 땅을 잼
- `geopolitical`: 지리와 정치의 결합

## 비교 단어

### terr

- 뜻: 땅, 영토
- 구분 포인트: [[terr]]는 라틴어 땅 계열. `geo`는 그리스어 땅 계열.
