# reg

- 출발 단어: `royal`
- 핵심 어원: Latin `rex` / `regis` = 왕
- 동사형: Latin `regere` = 곧게 이끌다, 지도하다, 다스리다
- 핵심 이미지: 방향을 바로잡아 이끌고 다스림
- 관련 문서: [[rect]], [[right]], [[govern]], [[leg]], [[crat]]
- 주의: `king`, `queen`은 게르만계 기본어. `royal`, `regal`, `reign`은 라틴계 왕권 어휘.

## 출발 단어 분석

`royal` ← Old French `roial/royal` ← Latin `regalis`

- `rex`: 왕
- `regis`: 왕의
- `regalis`: 왕의, 왕다운
- 원래 이미지: 왕에게 속한 것
- 현대 의미: 왕실의, 왕다운, 매우 훌륭한

## 단어들

### royal

- 발음: /ˈrɔɪəl/
- 구성: Old French `royal` ← Latin `regalis`
- 뜻: 왕실의, 왕다운
- 어원적 연결: 왕에게 속한 성질
- 사용 맥락: `the royal family` = 왕실

### royalty

- 발음: /ˈrɔɪəlti/
- 구성: `royal` + `-ty`
- 뜻: 왕족, 왕권, 사용료
- 어원적 연결: 왕에게 속한 권리와 지위
- 사용 맥락: `pay royalties` = 저작권 사용료를 지불하다

### regal

- 발음: /ˈrigəl/
- 구성: Latin `regalis`
- 뜻: 왕다운, 장엄한
- 어원적 연결: 왕에게 어울리는 품격
- 사용 맥락: `a regal appearance` = 위엄 있는 모습

### reign

- 발음: /ˈreɪn/
- 구성: Old French `reigne` ← Latin `regnum`
- 뜻: 통치, 재위 기간, 군림하다
- 어원적 연결: 왕권이 미치는 기간과 행위
- 사용 맥락: `during her reign` = 그녀의 재위 기간 동안

### realm

- 발음: /ˈrɛlm/
- 구성: Old French `realme/reaume` ← Latin `regalis/regimen` 계열
- 뜻: 왕국, 영역, 분야
- 어원적 연결: 왕권이 미치는 범위
- 사용 맥락: `the realm of science` = 과학의 영역

### regent

- 발음: /ˈridʒənt/
- 구성: Latin `regens` ← `regere`
- 뜻: 섭정, 대리 통치자
- 어원적 연결: 왕 대신 나라를 다스리는 사람
- 사용 맥락: `serve as regent` = 섭정으로 일하다

### regency

- 발음: /ˈridʒənsi/
- 구성: `regent` + `-cy`
- 뜻: 섭정 정치, 섭정 기간
- 어원적 연결: 대리로 다스리는 상태나 기간
- 사용 맥락: `the Regency period` = 섭정 시대

### regime

- 발음: /rəˈʒim/
- 구성: French `regime` ← Latin `regimen`
- 뜻: 정권, 체제
- 어원적 연결: 다스리는 방식과 질서
- 사용 맥락: `an authoritarian regime` = 권위주의 정권

### regimen

- 발음: /ˈrɛdʒəmən/
- 구성: Latin `regimen` = rule, guidance
- 뜻: 요법, 관리 방식
- 어원적 연결: 몸이나 생활을 규칙적으로 다스리는 방식
- 사용 맥락: `an exercise regimen` = 운동 요법

### regiment

- 발음: /ˈrɛdʒəmənt/
- 구성: Latin `regimentum` = rule, command
- 뜻: 연대, 엄격히 통제하다
- 어원적 연결: 명령 체계로 다스려지는 부대
- 사용 맥락: `a cavalry regiment` = 기병 연대

### region

- 발음: /ˈridʒən/
- 구성: Latin `regio` = direction, district
- 뜻: 지역, 지방, 영역
- 어원적 연결: 경계와 방향이 정해진 구역
- 사용 맥락: `a coastal region` = 해안 지역

### regular

- 발음: /ˈrɛgjəlɚ/
- 구성: Latin `regula` = rule, straight stick
- 뜻: 규칙적인, 정기적인, 보통의
- 어원적 연결: 정해진 기준과 규칙을 따름
- 사용 맥락: `regular exercise` = 규칙적인 운동

### regulate

- 발음: /ˈrɛgjəˌleɪt/
- 구성: Latin `regula` = rule + `-ate`
- 뜻: 규제하다, 조절하다
- 어원적 연결: 규칙에 맞게 다스리고 맞춤
- 사용 맥락: `regulate traffic` = 교통을 조절하다

### regulation

- 발음: /ˌrɛgjəˈleɪʃən/
- 구성: `regulate` + `-ion`
- 뜻: 규제, 규정
- 어원적 연결: 다스리기 위해 세운 규칙
- 사용 맥락: `safety regulations` = 안전 규정

### rule

- 발음: /ˈrul/
- 구성: Old French `reule` ← Latin `regula`
- 뜻: 규칙, 통치, 지배하다
- 어원적 연결: 곧은 막대기나 기준 → 따를 규칙 → 다스림
- 사용 맥락: `follow the rules` = 규칙을 따르다

### ruler

- 발음: /ˈrulɚ/
- 구성: `rule` + `-er`
- 뜻: 통치자, 자
- 어원적 연결: 다스리는 사람 / 곧게 재는 도구
- 사용 맥락: `a fair ruler` = 공정한 통치자

### regicide

- 구성: `regi-` = king + [[cid]] = kill
- 뜻: 국왕 시해, 국왕 시해자
- 어원적 연결: 왕을 죽임
- 사용 맥락: `the charge of regicide` = 국왕 시해 혐의

## 기억 포인트

`reg` 계열은 "왕, 규칙, 다스림" 이미지.

- `royal`, `regal`: 왕에게 속함
- `reign`, `realm`: 왕권이 작동하는 기간과 영역
- `regent`, `regime`: 다스리는 사람과 체제
- `regular`, `regulate`, `rule`: 기준을 세워 맞추고 다스림

## 비교 단어

### rect

- 뜻: 곧은, 바른
- 구분 포인트: [[rect]]는 같은 깊은 PIE `*reg-`에서 온 친족 계열이지만, 영어 학습에서는 `correct`, `direct`, `rectify`처럼 "곧게/바르게 함" 중심.

### king

- 뜻: 왕
- 구분 포인트: 게르만계 기본어. `royal`, `regal`, `reign`처럼 라틴·프랑스어계 왕권 어휘와 어원 분리.
