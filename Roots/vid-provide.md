# vid-provide

- 상위 문서: [[vid]]
- 핵심 어원: 라틴어 `providere` = 앞을 보다, 미리 마련하다
- 핵심 이미지: 앞을 보고 필요한 것을 준비함
- 관련 갈래: [[vid-see]], [[vid-watch]]

## 단어들

### provision

- 발음: /prəˈvɪʒən/
- 구성: `pro-` = before + `vis`
- 뜻: 공급, 준비, 조항
- 어원적 연결: 미리 보고 필요한 것을 마련함
- 사용 맥락: `make provisions for winter` = 겨울 준비를 하다

### provide

- 발음: /prəˈvaɪd/
- 구성: `pro-` = before + `vid`
- 뜻: 제공하다, 마련하다
- 어원적 연결: 미리 보고 필요한 것을 준비해 줌
- 사용 맥락: `provide support` = 지원을 제공하다

### provider

- 발음: /prəˈvaɪdɚ/
- 구성: `provide` + `-er`
- 뜻: 제공자, 공급자
- 어원적 연결: 필요한 것을 미리 마련해 주는 사람이나 기관
- 사용 맥락: `a healthcare provider` = 의료 서비스 제공자

### provided

- 발음: /prəˈvaɪdɪd/
- 구성: `provide` + `-ed`
- 뜻: 제공된; 만약 ~라면
- 어원적 연결: 조건이 미리 마련되어 있음을 전제로 함
- 사용 맥락: `provided that you agree` = 네가 동의한다면

### provident

- 발음: /ˈprɑvɪdənt/
- 구성: Latin `providens` = foreseeing, preparing
- 뜻: 앞날을 대비하는, 검소하고 신중한
- 어원적 연결: 앞을 보고 필요한 것을 미리 마련함
- 사용 맥락: `a provident habit` = 앞날을 대비하는 습관

### providence

- 발음: /ˈprɑvədəns/
- 구성: Latin `providentia` = foresight, divine care
- 뜻: 섭리, 신의 보살핌; 선견지명
- 어원적 연결: 앞을 내다보고 돌보거나 마련함
- 사용 맥락: `divine providence` = 신의 섭리

### prudent

- 발음: /ˈprudənt/
- 구성: Latin `prudens` ← `providens` = foreseeing
- 뜻: 신중한, 분별 있는, 앞일을 고려하는
- 어원적 연결: 앞을 미리 보고 위험과 결과를 따져 판단함
- 사용 맥락: `a prudent decision` = 신중한 결정

### prudently

- 발음: /ˈprudəntli/
- 구성: `prudent` + `-ly`
- 뜻: 신중하게, 현명하게
- 어원적 연결: 앞일을 보고 조심스럽게 판단하는 방식
- 사용 맥락: `spend money prudently` = 돈을 신중하게 쓰다

### prudence

- 발음: /ˈprudəns/
- 구성: Latin `prudentia` = foresight, practical judgment
- 뜻: 신중함, 분별, 조심성
- 어원적 연결: 앞을 내다보는 실제적 판단력
- 사용 맥락: `financial prudence` = 재정적 신중함

### improvident

- 구성: `im-` = not + `provident`
- 뜻: 앞날을 대비하지 않는, 낭비하는
- 어원적 연결: 앞을 보고 미리 마련하지 않음
- 사용 맥락: `an improvident decision` = 앞날을 고려하지 않은 결정

### improvidence

- 구성: `improvident` + `-ence`
- 뜻: 앞날을 대비하지 않음, 무분별함
- 어원적 연결: 필요한 것을 미리 보아 마련하지 않는 상태
- 사용 맥락: 문어적 표현. 재정·정책 판단의 부주의를 말할 때 사용.

### improvise

- 발음: /ˈɪmprəˌvaɪz/
- 구성: Latin `improvisus` = unforeseen ← `im-` = not + `provisus` = foreseen
- 뜻: 즉흥적으로 하다, 임기응변으로 처리하다
- 어원적 연결: 미리 보거나 준비하지 않은 상황에서 즉석으로 마련함
- 사용 맥락: `improvise a speech` = 즉석에서 연설하다, `improvise a solution` = 즉석에서 해결책을 만들다

### improvisation

- 발음: /ˌɪmprɑvɪˈzeɪʃən/
- 구성: `improvise` + `-ation`
- 뜻: 즉흥 연주, 즉흥적 처리, 임기응변
- 어원적 연결: 미리 준비된 것이 아니라 그 자리에서 만들어 냄
- 사용 맥락: `jazz improvisation` = 재즈 즉흥 연주

### improvised

- 발음: /ˈɪmprəˌvaɪzd/
- 구성: `improvise` + `-ed`
- 뜻: 즉석에서 만든, 임시변통의
- 어원적 연결: 미리 마련된 것이 없어 그 자리에서 만든 상태
- 사용 맥락: `an improvised shelter` = 즉석에서 만든 대피처

## 기억 포인트

`vid-provide`는 "미리 봄"에서 "마련함"으로 감.

- `provision`: 미리 마련한 것
- `provide`: 필요한 것을 마련해 줌
- `provider`: 제공자
- `provident`: 앞날을 보고 대비하는
- `providence`: 앞을 보고 돌봄, 섭리
- `prudent`: 앞일을 보고 신중하게 판단함
- `prudently`: 신중하고 분별 있게
- `improvident`: 앞날을 대비하지 않음
- `improvise`: 미리 보지 못한 상황에서 즉석으로 처리함
