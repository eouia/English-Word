# praed

- 출발 단어: `predatory`
- 핵심 어원: Latin `praeda` = plunder, booty, prey
- 핵심 이미지: 사냥감이나 전리품을 잡아 빼앗음
- 관련 문서: [[carn]], [[vor]]

## 단어들

### predatory

- 발음: /ˈprɛdəˌtɔri/
- 구성: Latin `praedatorius` < `praedator` = plunderer, predator
- 뜻: 약탈하는, 포식성의, 착취적인
- 어원적 연결: 먹잇감이나 약자를 노려 빼앗는 성질
- 사용 맥락: `predatory lending` = 약탈적 대출

### predator

- 발음: /ˈprɛdətɚ/
- 구성: Latin `praedator` = plunderer
- 뜻: 포식자, 약탈자, 착취자
- 어원적 연결: 먹잇감이나 전리품을 잡아 빼앗는 존재
- 사용 맥락: `apex predator` = 최상위 포식자

### predation

- 발음: /prəˈdeɪʃən/
- 구성: Latin `praedatio` = plundering
- 뜻: 포식, 약탈
- 어원적 연결: 먹잇감을 잡아먹거나 남의 것을 빼앗는 행위
- 사용 맥락: `predation pressure` = 포식 압력

### predatory behavior

- 구성: `predatory` + `behavior`
- 뜻: 포식적 행동, 착취적 행동
- 어원적 연결: 약한 대상을 노려 이익을 취하는 행동
- 사용 맥락: 동물 행동, 범죄, 금융 문맥에서 모두 사용

### prey

- 발음: /ˈpreɪ/
- 구성: Old French `preie` < Latin `praeda`
- 뜻: 먹잇감, 희생자; 잡아먹다
- 어원적 연결: 포식자나 약탈자의 대상
- 사용 맥락: `birds of prey` = 맹금류

### depredation

- 발음: /ˌdɛprəˈdeɪʃən/
- 구성: `de-` = down, thoroughly + Latin `praedari` = plunder
- 뜻: 약탈, 파괴, 피해
- 어원적 연결: 철저히 빼앗고 훼손함
- 사용 맥락: `environmental depredation` = 환경 파괴

### depredate

- 구성: `de-` + `praedari`
- 뜻: 약탈하다, 해치다
- 어원적 연결: 대상을 빼앗고 손상시킴
- 사용 맥락: 드문 단어이므로 `depredation` 중심으로 인식

### predacious

- 구성: Latin `praeda` + `-acious`
- 뜻: 포식성의, 탐욕스러운
- 어원적 연결: 먹잇감을 노리는 성질
- 사용 맥락: `predacious insects` = 포식성 곤충

## 기억 포인트

`praed` 계열은 "prey, plunder" 이미지.

- `predatory`: 포식성의, 착취적인
- `predator`: 포식자
- `predation`: 포식
- `prey`: 먹잇감
- `depredation`: 약탈, 파괴

## 비교 단어

### carn

- 뜻: 살, 고기
- 구분 포인트: [[carn]]은 먹는 대상인 고기. `praed`는 사냥감이나 약탈 대상.

### vor

- 뜻: 먹다, 삼키다
- 구분 포인트: [[vor]]는 먹는 행위. `praed`는 잡아 빼앗는 포식·약탈 이미지.

### rap

- 뜻: 붙잡아 빼앗다, 낚아채다
- 구분 포인트: `rap`은 강하게 낚아채는 행위. `praed`는 포식자와 먹잇감 관계에 가까움.

### predate

- 뜻: 포식하다; ~보다 앞서다
- 구분 포인트: `prey on` 의미의 `predate`는 `praeda` 계열. "~보다 앞서다" 의미의 `predate`는 `pre-` + `date`라 어원이 다름.
