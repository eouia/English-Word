# thought

- 출발 단어: `thoughtfulness`
- 핵심 어원: Old English `thencan` = think
- 핵심 이미지: 마음속으로 생각하고 살핌
- 관련 문서: [[care]]
- 주의: 표준 철자는 `thoughtfulness`. `thoughtfullness`처럼 `l`을 두 번 쓰지 않음.

## 출발 단어 분석

`thoughtfulness` = `thoughtful` + `-ness`

- `thought`: 생각
- `-ful`: 가득한, 성질을 지닌
- `-ness`: 상태나 성질을 나타내는 명사 접미사
- `thoughtfulness`: 사려 깊음, 배려심

`thoughtful`은 "생각이 있는"에서 출발해, 단순히 생각이 많다는 뜻뿐 아니라 남의 상황을 고려하는 "사려 깊은, 배려하는" 뜻으로 발전.

## 단어들

### thought

- 구성: Old English `thoht` ← `thencan` = think
- 뜻: 생각, 사고
- 어원적 연결: 마음속에서 무언가를 생각한 내용
- 사용 맥락: `a sudden thought` = 갑작스러운 생각

### thoughtful

- 구성: `thought` + `-ful`
- 뜻: 사려 깊은, 배려심 있는; 생각에 잠긴
- 어원적 연결: 생각을 충분히 들여 남이나 상황을 살핌
- 사용 맥락: `a thoughtful gesture` = 사려 깊은 행동

### thoughtfully

- 구성: `thoughtful` + `-ly`
- 뜻: 사려 깊게, 생각에 잠겨
- 어원적 연결: 생각을 들여 조심스럽게 하는 방식
- 사용 맥락: `answer thoughtfully` = 생각을 담아 답하다

### thoughtfulness

- 구성: `thoughtful` + `-ness`
- 뜻: 사려 깊음, 배려심
- 어원적 연결: 생각을 들여 다른 사람이나 상황을 고려하는 성질
- 사용 맥락: `appreciate her thoughtfulness` = 그녀의 배려심에 감사하다

### thoughtless

- 구성: `thought` + `-less`
- 뜻: 생각 없는, 배려 없는, 경솔한
- 어원적 연결: 생각이나 배려가 부족함
- 사용 맥락: `a thoughtless comment` = 생각 없는 말

### thoughtlessly

- 구성: `thoughtless` + `-ly`
- 뜻: 생각 없이, 경솔하게
- 어원적 연결: 결과나 타인을 고려하지 않는 방식
- 사용 맥락: `act thoughtlessly` = 생각 없이 행동하다

### thoughtlessness

- 구성: `thoughtless` + `-ness`
- 뜻: 경솔함, 배려 없음
- 어원적 연결: 생각과 배려가 빠진 성질
- 사용 맥락: `hurt someone through thoughtlessness` = 경솔함으로 누군가에게 상처를 주다

### thoughtful of

- 구성: `thoughtful` + `of`
- 뜻: ~에게 사려 깊은, ~을 배려하는
- 어원적 연결: 특정 사람을 마음에 두고 생각함
- 사용 맥락: `It was thoughtful of you to call.` = 전화해 주다니 사려 깊었다

## 기억 포인트

`thought` 계열은 "생각을 들여 살핌" 이미지.

- `thought`: 생각
- `thoughtful`: 사려 깊은, 배려심 있는
- `thoughtfulness`: 사려 깊음, 배려심
- `thoughtless`: 생각 없는, 배려 없는
- `thoughtlessness`: 경솔함, 배려 없음

## 비교 단어

### careful

- 뜻: 주의 깊은, 조심하는
- 구분 포인트: [[care]]는 실수나 위험을 피하려는 주의와 마음 씀. `thoughtful`은 남의 입장이나 의미를 생각해 배려하는 느낌이 강함.

### considerate

- 뜻: 배려심 있는, 사려 깊은
- 구분 포인트: `considerate`는 남의 감정과 필요를 고려하는 태도. `thoughtful`은 작은 행동이나 말에 생각이 담긴 느낌이 좋음.

### pensive

- 뜻: 생각에 잠긴
- 구분 포인트: `pensive`는 조용히 깊은 생각에 잠긴 상태. `thoughtful`은 "생각에 잠긴" 뜻도 있지만, 일상에서는 배려심 있는 행동에 더 자주 씀.
