# tend

- 출발 단어: `extend`
- 핵심 어원: 라틴어 `tendere` / `tens-` / `tent-` = 뻗다, 펼치다, 향하다
- 핵심 이미지: 팽팽하게 뻗거나 어떤 방향으로 향함
- 관련해서 헷갈릴 수 있는 어근: [[ten]] = 잡다, 유지하다 계열, [[strain]] = 조이다, 긴장시키다 계열

## 출발 단어 분석

`extend`는 `ex-` + `tendere`.

- `ex-`: out
- `tendere`: stretch

따라서 `extend`는 "밖으로 뻗다" → "연장하다, 확장하다, 내밀다"의 뜻.

## 단어들

### extend

- 발음: /ɪˈkstɛnd/
- 구성: `ex-` = out + `tend`
- 뜻: 연장하다, 확장하다, 내밀다
- 어원적 연결: 밖으로 길게 뻗음
- 사용 맥락: `extend the deadline` = 마감일을 연장하다

## 기억 포인트

`tend` 계열은 "뻗다 / 향하다 / 팽팽해지다" 이미지.

- `extend`: 밖으로 뻗음
- `intend`: 마음이 어떤 방향으로 향함
- `unintended`: 의도한 방향이 아니었음
- `intense`: 안으로 팽팽하게 몰림
- `tension`: 팽팽한 긴장
- `attention`: 마음을 그쪽으로 뻗음
- `attend`: 몸이나 마음을 그쪽으로 향함
- `pretend`: 앞에 모습을 내세움
- `contentious`: 맞서 팽팽해져 논쟁적인

## 비교 단어

### ten

- 뜻: 잡다, 유지하다
- 구분 포인트: [[ten]]은 Latin `tenere` = 잡고 유지하다. `tend`는 Latin `tendere` = 뻗다, 향하다. `tent-` 형태가 겹쳐도 갈라 외우는 편이 좋음

### strain

- 뜻: 조이다, 긴장시키다
- 구분 포인트: [[strain]]은 묶고 조이는 느낌. `tension`은 팽팽하게 뻗어 있는 상태
