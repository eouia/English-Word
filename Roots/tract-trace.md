# tract-trace

- 상위 문서: [[tract]]
- 핵심 어원: 라틴어 `trahere` / `tract-` = 끌다, 당기다
- 핵심 이미지: 끌고 지나간 선, 자국, 그려 드러낸 모습
- 관련 갈래: [[tract-pull]], [[tract-handle]]

## 단어들

### trace

- 구성: Old French `tracier`, Vulgar Latin `tractiare`
- 뜻: 흔적, 추적하다, 윤곽을 그리다
- 어원적 연결: 끌고 지나간 선이나 자국을 따라감
- 사용 맥락: `trace the origin` = 기원을 추적하다

### traceable

- 구성: `trace` + `-able`
- 뜻: 추적 가능한
- 어원적 연결: 자국이나 경로를 따라갈 수 있음
- 사용 맥락: `traceable records` = 추적 가능한 기록

### portray

- 구성: Old French `portraire`, Latin `protrahere`
- 뜻: 묘사하다, 그리다, 연기하다
- 어원적 연결: 어떤 모습을 앞으로 끌어내어 보이게 함. 선을 끌어 그리는 이미지에서 인물이나 성격을 드러내는 의미로 확대
- 사용 맥락: `portray a character` = 인물을 연기하다, 묘사하다

### portrayal

- 구성: `portray` + `-al`
- 뜻: 묘사, 표현, 연기
- 어원적 연결: 사람이나 상황을 밖으로 드러내 보이는 방식
- 사용 맥락: `a realistic portrayal` = 사실적인 묘사

### portrait

- 구성: Old French `portrait`, Latin `protrahere`
- 뜻: 초상화, 인물 묘사
- 어원적 연결: 사람의 모습을 선으로 끌어내어 나타낸 것
- 사용 맥락: `paint a portrait` = 초상화를 그리다

### trait

- 구성: Old French `trait`, Latin `tractus`
- 뜻: 특징, 특성
- 어원적 연결: 원래 "끌어 그은 선, 획". 얼굴이나 성격을 이루는 한 줄의 특징에서, 사람을 구별하는 특성으로 의미 확대
- 비교: `portrait`가 전체 인물 묘사라면, `trait`는 그 사람을 알아보게 하는 개별 특징
- 사용 맥락: `personality traits` = 성격적 특징, `inherited traits` = 유전 형질

### trail

- 구성: Old French `trailler`, Vulgar Latin `tragulare`
- 뜻: 자국, 길, 뒤따르다
- 어원적 연결: 끌고 지나간 뒤에 남은 자국
- 사용 맥락: `follow a trail` = 흔적을 따라가다

### trailer

- 구성: `trail` + `-er`
- 뜻: 트레일러, 예고편
- 어원적 연결: 뒤에 끌려가는 것. 영화 뒤에 붙는 홍보물에서 예고편 의미
- 사용 맥락: `movie trailer` = 영화 예고편
