# multi

- 출발 단어: `multifaceted`
- 핵심 의미: 많은, 여러
- 관련 형태: Latin `multus` = much, many
- 관련 문서: [[mono]], [[bi]], [[facet]]

## 핵심 이미지

`multi-`의 바탕은 "하나가 아니라 많음".

여러 면을 가지면 `multifaceted`.
여러 개이면 `multiple`.
여러 배로 늘리면 `multiply`.
많은 수나 무리는 `multitude`.
여러 국가에 걸치면 `multinational`.

## 단어들

### multifaceted

- 구성: `multi-` = many + `facet` = side, face
- 뜻: 다면적인, 여러 측면을 가진
- 연결: 여러 면을 가진 보석처럼, 문제나 사람이 여러 측면을 가짐
- 사용 맥락: `a multifaceted issue` = 다면적인 쟁점

### multiple

- 구성: Latin `multiplus` = manifold
- 뜻: 여러 개의, 다수의; 배수
- 연결: 하나보다 많은 수나 경우
- 사용 맥락: `multiple factors` = 여러 요인

### multiply

- 구성: `multi-` = many + `-ply` = fold
- 뜻: 곱하다, 증가하다
- 연결: 수를 여러 겹으로 늘림
- 사용 맥락: `multiply by three` = 3을 곱하다

### multiplication

- 구성: `multiply` + `-ation`
- 뜻: 곱셈, 증식
- 연결: 수나 개체가 여러 배로 늘어남
- 사용 맥락: `multiplication table` = 구구단

### multitude

- 구성: Latin `multitudo` = great number
- 뜻: 다수, 군중
- 연결: 많은 수가 모인 상태
- 사용 맥락: `a multitude of reasons` = 수많은 이유

### multinational

- 구성: `multi-` = many + `national`
- 뜻: 다국적의
- 연결: 여러 나라에 걸쳐 있음
- 사용 맥락: `a multinational company` = 다국적 기업

### multimedia

- 구성: `multi-` = many + `media`
- 뜻: 멀티미디어
- 연결: 여러 매체를 함께 사용함
- 사용 맥락: `multimedia content` = 멀티미디어 콘텐츠

## 비교 단어

`mono-`는 [[mono]] 문서처럼 하나, 홀로.
`bi-`는 [[bi]] 문서처럼 둘.
`multi-`는 셋 이상을 포함해 "많은, 여러"가 중심.
