# propri

- 출발 단어: `appropriate`
- 핵심 어원: Latin `proprius` = one's own, special, proper
- 핵심 이미지: 자기 것, 고유한 것, 목적에 알맞은 것
- 관련 문서: 없음
- 주의: `appropriate`는 형용사일 때 "적절한", 동사일 때 "전용하다, 충당하다, 부당하게 취하다"로 의미가 갈림. 둘 다 "자기 것/특정 목적에 맞게 배정함" 이미지에서 연결됨.

## 출발 단어 분석

`appropriate` = `ad-` + `propri` + `-ate`

- `ad-`: to, toward
- `proprius`: one's own, proper
- `-ate`: 동사·형용사 형성
- 원래 이미지: 어떤 것에 자기 것처럼 맞게 붙임
- 현대 의미: 적절한, 전용하다, 충당하다

## 단어들

### appropriate

- 구성: `ad-` = to + Latin `proprius` = proper, one's own
- 뜻: 적절한; 전용하다, 충당하다
- 어원적 연결: 상황이나 목적에 맞는 것. 또는 특정 목적의 것으로 배정함
- 사용 맥락: `appropriate behavior` = 적절한 행동, `appropriate funds` = 자금을 배정하다

### appropriately

- 구성: `appropriate` + `-ly`
- 뜻: 적절하게
- 어원적 연결: 상황에 맞게
- 사용 맥락: `respond appropriately` = 적절하게 대응하다

### appropriation

- 구성: `appropriate` + `-ion`
- 뜻: 전용, 충당, 세출 승인, 부당 사용
- 어원적 연결: 어떤 것을 특정 목적의 것으로 배정하거나 자기 것으로 삼음
- 사용 맥락: `cultural appropriation` = 문화적 전유, `budget appropriation` = 예산 배정

### proper

- 구성: Latin `proprius` = one's own, proper
- 뜻: 적절한, 제대로 된, 고유한
- 어원적 연결: 본래 자기 성격이나 기준에 맞음
- 사용 맥락: `proper procedure` = 적절한 절차

### properly

- 구성: `proper` + `-ly`
- 뜻: 제대로, 적절히
- 어원적 연결: 맞는 방식으로
- 사용 맥락: `function properly` = 제대로 작동하다

### property

- 구성: Latin `proprietas` = ownership, special quality
- 뜻: 재산, 소유물, 속성
- 어원적 연결: 자기에게 속한 것, 또는 그 대상의 고유한 성질
- 사용 맥락: `private property` = 사유 재산, `chemical properties` = 화학적 성질

### proprietor

- 구성: Latin `proprietarius` 계열
- 뜻: 소유자, 사업주
- 어원적 연결: 어떤 것을 자기 것으로 가진 사람
- 사용 맥락: `a hotel proprietor` = 호텔 주인

### proprietary

- 구성: `proprietor/property` 계열 + `-ary`
- 뜻: 독점 소유의, 전매의, 소유주의
- 어원적 연결: 특정 주체에게 고유하게 속함
- 사용 맥락: `proprietary software` = 독점 소프트웨어

### propriety

- 구성: Latin `proprietas` 계열
- 뜻: 예의바름, 적절성, 타당성
- 어원적 연결: 상황과 규범에 맞는 성질
- 사용 맥락: `questions of propriety` = 적절성 문제

## 기억 포인트

`propri` 계열은 "자기 것, 고유함, 알맞음" 이미지.

- `appropriate`: 상황에 맞는 / 특정 목적에 배정함
- `proper`: 제대로 맞는
- `property`: 자기 것, 고유 성질
- `proprietor`: 소유자
- `proprietary`: 독점 소유의
- `propriety`: 적절성, 예의바름

## 비교 단어

### apt

- 뜻: 알맞은, 목적에 맞는
- 구분 포인트: [[apt]]는 맞게 붙거나 적응함. `propri`는 자기 것·고유함에서 적절성으로 확장.

- 뜻: 기준에 맞음, 정당함
- 구분 포인트: [[just]]는 법·기준에 맞음. `proper/appropriate`는 상황·소유·목적에 맞음.
