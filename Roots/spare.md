# spare

- 출발 단어: `sparingly`
- 핵심 어원: Old English `sparian` = refrain from harming, save, use economically
- 핵심 이미지: 아껴 남겨 두거나 해치지 않고 봐줌
- 관련 문서: 없음
- 주의: `sparing`은 "아끼는, 절제하는"이고 `sparring`은 권투·논쟁의 "겨루기"라 어원이 다름.

## 출발 단어 분석

`sparingly` = `sparing` + `-ly`

- `sparing`: 아끼는, 절제하는
- `-ly`: 방식 부사 접미사
- `sparingly`: 아껴서, 조금씩, 절제하여

`spare`의 기본 이미지는 "다 쓰지 않고 남겨 둠" 또는 "해치지 않고 놓아 줌". 여기서 `sparingly`는 "많이 쓰지 않고 아껴서"의 뜻.

## 단어들

### spare

- 발음: /ˈspɛr/
- 구성: Old English `sparian`
- 뜻: 여분의; 아끼다; 살려 주다, 면하게 하다
- 어원적 연결: 다 쓰거나 해치지 않고 남겨 둠
- 사용 맥락: `a spare key` = 여분의 열쇠, `spare no effort` = 노력을 아끼지 않다

### sparing

- 발음: /ˈspɛrɪŋ/
- 구성: `spare` + `-ing`
- 뜻: 아끼는, 절제하는, 인색할 정도로 적은
- 어원적 연결: 많이 쓰지 않고 남기는 태도
- 사용 맥락: `be sparing with praise` = 칭찬을 아끼다

### sparingly

- 발음: /ˈspɛrɪŋli/
- 구성: `sparing` + `-ly`
- 뜻: 아껴서, 조금씩, 절제하여
- 어원적 연결: 필요한 만큼만 쓰고 남김
- 사용 맥락: `Use salt sparingly.` = 소금을 조금만 써라.

### unsparing

- 발음: /əˈnspɛrɪŋ/
- 구성: `un-` = not + `sparing`
- 뜻: 가차 없는, 아끼지 않는
- 어원적 연결: 봐주거나 남겨 두지 않음
- 사용 맥락: `unsparing criticism` = 가차 없는 비판

### sparely

- 구성: `spare` + `-ly`
- 뜻: 간소하게, 꾸밈없이; 드문드문
- 어원적 연결: 불필요한 것을 덜어 낸 방식
- 사용 맥락: `a sparely furnished room` = 간소하게 꾸민 방

### sparse

- 발음: /ˈspɑrs/
- 구성: Latin `sparsus` = scattered
- 뜻: 드문드문한, 희박한
- 어원적 연결: 어원은 다르지만 `spare`와 함께 "적고 듬성듬성함"으로 자주 헷갈림
- 사용 맥락: `sparse vegetation` = 드문드문 난 식생

### sparsely

- 발음: /ˈspɑrsli/
- 구성: `sparse` + `-ly`
- 뜻: 드문드문, 희박하게
- 어원적 연결: 넓은 곳에 적게 흩어진 방식
- 사용 맥락: `a sparsely populated area` = 인구가 희박한 지역

## 기억 포인트

`spare` 계열은 "다 쓰지 않고 남겨 둠" 이미지.

- `spare`: 여분의; 아끼다; 봐주다
- `sparing`: 아끼는, 절제하는
- `sparingly`: 아껴서, 조금씩
- `unsparing`: 가차 없는, 아끼지 않는
- `sparely`: 간소하게
- `sparse`: 드문드문한

## 비교 단어

### save

- 뜻: 구하다, 절약하다, 저장하다
- 구분 포인트: `save`는 위험에서 구하거나 나중을 위해 보존함. `spare`는 사용·공격·부담을 덜어 남겨 두는 느낌이 강함.

### scarce

- 뜻: 부족한, 드문
- 구분 포인트: `scarce`는 양이 부족한 상태. `sparingly`는 부족해서이든 절제해서이든 적게 쓰는 방식.

### sparse

- 뜻: 드문드문한
- 구분 포인트: `sparse`는 Latin `sparsus` = 흩어진 계열이라 엄밀히는 `spare`와 다른 어원. 학습상 "적음"의 느낌은 가깝지만, `sparingly`는 사용량을 아끼는 방식이고 `sparsely`는 분포가 듬성듬성한 방식.
