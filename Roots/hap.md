# hap

- 출발 단어: `happen`
- 핵심 어원: Middle English `hap` = 우연, 운, 닥친 일
- 핵심 이미지: 우연히 닥쳐 일어남
- 관련해서 헷갈릴 수 있는 어근: [[curr]] = 달리다, 흐르다, 발생하다 계열

## 출발 단어 분석

`happen`은 `hap` + `-en`.

- `hap`: chance, fortune, accident
- `-en`: 동사화 접미사

따라서 `happen`은 원래 "우연한 일이 닥치다" → "일어나다, 발생하다"의 뜻.

## 단어들

### happen

- 구성: `hap` + `-en`
- 뜻: 일어나다, 발생하다
- 어원적 연결: 어떤 일이 우연히 닥침
- 사용 맥락: `What happened?` = 무슨 일이 있었나?

### happening

- 구성: `happen` + `-ing`
- 뜻: 사건, 일어나는 일
- 어원적 연결: 지금 일어나고 있는 일
- 사용 맥락: `recent happenings` = 최근 일어난 일들

### happenstance

- 구성: `happen` + `stance`
- 뜻: 우연한 일, 우연
- 어원적 연결: 우연히 놓인 상황
- 사용 맥락: `by happenstance` = 우연히

### perhaps

- 구성: `per` = by + `haps`
- 뜻: 아마도, 어쩌면
- 어원적 연결: "우연에 따라, 경우에 따라"에서 가능성의 뜻
- 사용 맥락: `Perhaps he is right.` = 어쩌면 그가 맞을 수 있음

### mishap

- 구성: `mis-` = bad, wrong + `hap`
- 뜻: 작은 사고, 불운한 일
- 어원적 연결: 나쁜 우연이 닥친 일
- 사용 맥락: `a minor mishap` = 작은 사고

### happy

- 구성: `hap` + `-y`
- 뜻: 행복한, 기쁜
- 어원적 연결: 원래 "운이 좋은" → "행복한"으로 변화
- 사용 맥락: `a happy ending` = 행복한 결말

### happiness

- 구성: `happy` + `-ness`
- 뜻: 행복
- 어원적 연결: 좋은 운이 닥친 상태에서 일반적 행복으로 확장
- 사용 맥락: `pursue happiness` = 행복을 추구하다

### unhappy

- 구성: `un-` = not + `happy`
- 뜻: 불행한, 불만인
- 어원적 연결: 좋은 운이나 만족이 없는 상태
- 사용 맥락: `unhappy with the result` = 결과에 불만인

## 기억 포인트

`hap` 계열은 "우연히 닥친 일 / 운" 이미지.

- `happen`: 일이 닥쳐 일어남
- `perhaps`: 경우에 따라 그럴 수도 있음
- `mishap`: 나쁘게 닥친 일
- `happy`: 좋은 운을 얻은 상태에서 행복으로 확장

## 비교 단어

### occur

- 뜻: 일어나다, 발생하다, 떠오르다
- 구분 포인트: [[curr]] 계열. `occur`는 격식 있는 "발생하다"에 가깝고, `happen`은 일상적인 "일이 생기다"에 가까움

### haphazard

- 뜻: 되는대로의, 무계획적인
- 구분 포인트: `hap`이 보이지만 `hazard`와 결합한 단어라 [[hazard]]에서 따로 정리.
