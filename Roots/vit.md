# vit

- 출발 단어: `vitamin`
- 핵심 어원: Latin `vita` = 생명, 삶
- 관련 형태: Latin `vivere` = 살다, 살아 있다
- 핵심 이미지: 살아 있게 하는 생명력
- 관련 문서: [[anim]], [[gen]]
- 주의: `vitamin`은 원래 `vital amine`에서 만든 말. 뒤에 붙은 `amine`은 모든 비타민에 맞지는 않아 철자가 `vitamine`에서 `vitamin`으로 바뀜.

## 출발 단어 분석

`vitamin` ← `vital` + `amine`

- `vital`: 생명에 필요한
- `amine`: 질소를 포함한 화합물
- 원래 이미지: 생명에 꼭 필요한 아민 물질
- 현대 의미: 비타민

## 단어들

### vitamin

- 구성: `vital` + `amine`
- 뜻: 비타민
- 어원적 연결: 생명 유지에 필요한 영양 물질
- 사용 맥락: `vitamin D` = 비타민 D

### vital

- 구성: Latin `vita` = life + `-al`
- 뜻: 생명의, 필수적인, 중대한
- 어원적 연결: 생명과 직결될 만큼 중요함
- 사용 맥락: `a vital role` = 매우 중요한 역할

### vitality

- 구성: `vital` + `-ity`
- 뜻: 활력, 생명력
- 어원적 연결: 살아 움직이는 힘
- 사용 맥락: `full of vitality` = 활력이 넘치는

### vitalize

- 구성: `vital` + `-ize`
- 뜻: 생기를 주다, 활력을 불어넣다
- 어원적 연결: 생명력을 더함
- 사용 맥락: `vitalize the community` = 공동체에 활력을 주다

### revitalize

- 구성: `re-` = again + `vitalize`
- 뜻: 다시 활기를 주다, 재활성화하다
- 어원적 연결: 약해진 생명력이나 활력을 되살림
- 사용 맥락: `revitalize the economy` = 경제를 재활성화하다

### viable

- 구성: French `vie` = life + `-able`
- 뜻: 실행 가능한, 생존 가능한
- 어원적 연결: 살아남거나 실제로 작동할 수 있음
- 사용 맥락: `a viable option` = 실행 가능한 선택지

### vivid

- 구성: Latin `vivus` = alive
- 뜻: 생생한, 선명한
- 어원적 연결: 살아 있는 듯 뚜렷함
- 사용 맥락: `a vivid memory` = 생생한 기억

### vivacious

- 구성: Latin `vivax` = lively
- 뜻: 활발한, 생기 넘치는
- 어원적 연결: 살아 움직이는 기운이 강함
- 사용 맥락: `a vivacious personality` = 활발한 성격

### revive

- 구성: `re-` = again + Latin `vivere` = live
- 뜻: 되살아나다, 되살리다
- 어원적 연결: 다시 살아나게 함
- 사용 맥락: `revive interest` = 관심을 되살리다

### survive

- 구성: `sur-` = over, beyond + Latin `vivere`
- 뜻: 살아남다
- 어원적 연결: 위기나 시간을 넘어 계속 살아 있음
- 사용 맥락: `survive a crisis` = 위기를 넘기고 살아남다

### survival

- 구성: `survive` + `-al`
- 뜻: 생존
- 어원적 연결: 어려움을 넘어 살아남은 상태
- 사용 맥락: `survival skills` = 생존 기술

## 기억 포인트

`vit/viv` 계열은 "생명과 살아 있음" 이미지.

- `vitamin`: 생명에 필요한 영양 물질
- `vital`: 생명과 직결될 만큼 중요함
- `vitality`: 생명력
- `revitalize`: 활력을 되살림
- `viable`: 살아남거나 실행 가능함
- `vivid`: 살아 있는 듯 생생함
- `survive`: 계속 살아남음

## 비교 단어

### anim

- 뜻: 숨, 생명, 마음
- 구분 포인트: [[anim]]은 숨이 깃든 생명과 마음. `vit/viv`는 생명과 살아 있음 자체.

### gen

- 뜻: 태어남, 종류, 혈통
- 구분 포인트: [[gen]]은 생겨남과 출생. `vit/viv`는 태어난 뒤 살아 있는 상태와 생명력.
