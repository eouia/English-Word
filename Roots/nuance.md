# nuance

- 출발 단어: `nuanced`
- 핵심 어원: French `nuance` = shade, slight difference
- 더 깊은 연결: French `nue` = cloud, Latin `nubes` = cloud
- 핵심 이미지: 구름처럼 부드럽게 달라지는 색조 / 미묘한 차이
- 관련해서 헷갈릴 수 있는 어근: [[subtil]] = 섬세하고 미묘함 계열, [[vari]] = 여러 모습으로 달라짐 계열

## 출발 단어 분석

`nuanced`는 `nuance` + `-ed`.

- `nuance`: 미묘한 차이, 색조
- `-ed`: 형용사형 접미사

따라서 `nuanced`는 "미묘한 차이를 지닌"의 뜻. 단순히 좋다/나쁘다로 나누지 않고, 세부 차이를 섬세하게 담고 있음을 가리킴.

## 단어들

### nuanced

- 발음: /ˈnuɑnst/
- 구성: `nuance` + `-ed`
- 뜻: 미묘한 차이가 있는, 섬세한
- 어원적 연결: 색조가 조금씩 달라지듯 의미나 판단이 섬세하게 나뉨
- 사용 맥락: `a nuanced argument` = 미묘한 차이를 담은 논증

### nuance

- 발음: /ˈnuɑns/
- 구성: French `nuance`
- 뜻: 미묘한 차이, 뉘앙스
- 어원적 연결: 빛과 색의 그늘 차이에서 의미·감정의 작은 차이로 확장
- 사용 맥락: `understand the nuance` = 뉘앙스를 이해하다

## 기억 포인트

`nuance` 계열은 "색조의 작은 차이" 이미지.

- `nuance`: 미묘한 차이
- `nuanced`: 미묘한 차이를 담은

## 비교 단어

### subtle

- 뜻: 미묘한, 섬세한
- 구분 포인트: [[subtil]] 계열. 감지하기 어려울 만큼 섬세한 느낌. `nuanced`는 여러 작은 차이를 구분해 담고 있다는 느낌

### various

- 뜻: 다양한
- 구분 포인트: [[vari]]는 여러 종류가 있다는 일반 이미지. `nuance`는 종류의 많음보다 작은 차이의 섬세함에 초점
