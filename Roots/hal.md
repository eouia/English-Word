# hal

- 출발 단어: `exhale`
- 핵심 어원: Latin `halare` = 숨쉬다, 내뿜다
- 관련 형태: Latin `halitus` = 숨, 입김
- 핵심 이미지: 숨을 들이마시고 내쉼
- 관련 문서: [[spir]], [[pneu]]
- 주의: `inhale`, `exhale`에서는 `h`가 살아 있지만, Latin 계열 형태 변화로 `halitus`처럼 보이기도 함.

## 출발 단어 분석

`exhale` ← Latin `exhalare`

- `ex-`: out
- `halare`: breathe
- 원래 이미지: 숨을 밖으로 내보냄
- 현대 의미: 숨을 내쉬다

## 단어들

### exhale

- 구성: `ex-` = out + Latin `halare` = breathe
- 뜻: 숨을 내쉬다
- 어원적 연결: 숨을 밖으로 내보냄
- 사용 맥락: `exhale slowly` = 천천히 숨을 내쉬다

### exhalation

- 구성: `exhale` + `-ation`
- 뜻: 숨을 내쉼, 발산
- 어원적 연결: 숨이나 기체가 밖으로 나오는 과정
- 사용 맥락: `a deep exhalation` = 깊은 날숨

### inhale

- 구성: `in-` = into + Latin `halare`
- 뜻: 숨을 들이마시다
- 어원적 연결: 숨이나 공기를 안으로 들임
- 사용 맥락: `inhale deeply` = 깊이 숨을 들이마시다

### inhalation

- 구성: `inhale` + `-ation`
- 뜻: 흡입, 들이마심
- 어원적 연결: 공기나 물질을 안으로 들이는 과정
- 사용 맥락: `smoke inhalation` = 연기 흡입

### inhaler

- 구성: `inhale` + `-er`
- 뜻: 흡입기
- 어원적 연결: 약물을 숨과 함께 들이마시게 하는 도구
- 사용 맥락: `use an inhaler` = 흡입기를 사용하다

### halitosis

- 구성: Latin `halitus` = breath + `-osis` = condition
- 뜻: 구취
- 어원적 연결: 입에서 나는 숨의 상태
- 사용 맥락: 의학·건강 맥락에서 입 냄새를 가리킴

## 기억 포인트

`hal` 계열은 "숨을 들이마시고 내쉬다" 이미지.

- `inhale`: 안으로 숨을 들임
- `exhale`: 밖으로 숨을 내보냄
- `inhaler`: 숨으로 약을 들이는 도구
- `halitosis`: 숨 냄새의 문제

## 비교 단어

### spir

- 뜻: 숨쉬다
- 구분 포인트: [[spir]]는 `inspire`, `expire`, `respire`처럼 라틴어 `spirare` 계열. `hal`은 `inhale`, `exhale`처럼 실제 들숨과 날숨 표현에 직접 남음.

### pneu

- 뜻: 숨, 공기
- 구분 포인트: [[pneu]]는 그리스어 계열의 숨과 공기. `hal`은 라틴어 `halare`에서 온 들이마시고 내쉬는 동작.
