# fac-make

- 출발 단어: `factory`
- 핵심 어원: Latin `facere` / `fact-` / `fic-` = 만들다
- 핵심 이미지: 실제로 만들거나 만들어진 것
- 허브 문서: [[fac]]
- 관련 문서: [[manu]], [[art]]

## 단어들

### factory

- 구성: `fact` + `-ory`
- 뜻: 공장
- 어원적 연결: 물건을 만드는 장소
- 사용 맥락: `factory workers` = 공장 노동자

### factor

- 구성: Latin `factor` = maker, doer
- 뜻: 요인, 요소
- 어원적 연결: 결과를 만드는 원인
- 사용 맥락: `a key factor` = 핵심 요인

### fact

- 구성: Latin `factum` = thing done
- 뜻: 사실
- 어원적 연결: 실제로 행해지거나 만들어진 일
- 사용 맥락: `check the facts` = 사실을 확인하다

### manufacture

- 구성: [[manu]] = hand + `facture`
- 뜻: 제조하다
- 어원적 연결: 손으로 만듦
- 사용 맥락: `manufacture cars` = 자동차를 제조하다

### artificial

- 구성: [[art]] = skill + `fic`
- 뜻: 인공적인
- 어원적 연결: 기술로 만들어진 것
- 사용 맥락: `artificial intelligence` = 인공지능

### benefit

- 구성: Latin `benefactum` = good deed
- 뜻: 이익, 혜택
- 어원적 연결: 좋은 일이 만들어 낸 이로움
- 사용 맥락: `health benefits` = 건강상 이점

### sacrifice

- 구성: `sacer` = sacred + `fic`
- 뜻: 희생하다, 제물
- 어원적 연결: 신성하게 만드는 행위
- 사용 맥락: `make a sacrifice` = 희생하다

### feature

- 구성: Old French `faiture` = form, fashion, make
- 뜻: 특징, 특집, 기능
- 어원적 연결: 만들어진 모양이나 형성된 특징
- 사용 맥락: `a key feature` = 핵심 특징

## 기억 포인트

`fac-make`는 "만들어진 것" 갈래.

- `factory`: 만드는 장소
- `factor`: 결과를 만드는 요인
- `fact`: 실제로 만들어진 일
- `manufacture`: 손으로 만듦
- `artificial`: 기술로 만들어짐
- `feature`: 만들어진 특징
