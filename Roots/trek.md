# trek

- 출발 단어: `trek`
- 핵심 어원: Afrikaans/Dutch `trekken` = 끌다, 이동하다, 여행하다
- 핵심 이미지: 짐을 끌고 먼 길을 감
- 비교: [[tract]] = 라틴어 `trahere` = 끌다

## 출발 단어 분석

`trek`은 Afrikaans를 거쳐 영어에 들어온 말.

- Dutch `trekken`: 끌다, 행진하다, 여행하다
- Afrikaans `trek`: 여행, 이주, 이동

따라서 `trek`은 "수레를 끌고 이동함" → "길고 힘든 여행"의 뜻.

## 단어들

### trek

- 발음: /ˈtrɛk/
- 구성: Afrikaans/Dutch `trekken`
- 뜻: 길고 힘든 여행, 트레킹하다
- 어원적 연결: 짐이나 수레를 끌고 이동함
- 사용 맥락: `a mountain trek` = 산악 트레킹

### trekking

- 발음: /ˈtrɛkɪŋ/
- 구성: `trek` + `-ing`
- 뜻: 트레킹, 도보 여행
- 어원적 연결: 먼 길을 걸어 이동함
- 사용 맥락: `go trekking` = 트레킹을 가다

### trekker

- 구성: `trek` + `-er`
- 뜻: 트레킹하는 사람, 여행자, 당기는 장치
- 어원적 연결: 먼 길을 이동하는 사람. Dutch `trekker`는 "당기는 것"의 뜻도 가짐
- 사용 맥락: `experienced trekkers` = 경험 많은 트레커들

### trigger

- 발음: /ˈtrɪgɚ/
- 구성: Dutch `trekker`
- 뜻: 방아쇠, 유발하다
- 어원적 연결: 당기면 장치가 작동하는 부분
- 사용 맥락: `pull the trigger` = 방아쇠를 당기다, `trigger a reaction` = 반응을 유발하다

## 비교 단어

### tract

- 구성: 라틴어 `trahere`
- 뜻: 끌다 계열
- 어원적 연결: [[tract]] 참고
- 구분 포인트: `trek` = Dutch/Afrikaans 계열, `tract` = Latin 계열

### track

- 구성: Germanic 계열
- 뜻: 자국, 길, 추적하다
- 어원적 연결: `trek`과 관련 가능성이 있으나, 학습용으로는 별도 취급. [[tract]]의 `trace`와도 비교
- 구분 포인트: `track` = 지나간 자국/길, `trek` = 힘든 여정, `trace` = 자국을 따라감

## 기억 포인트

`trek` 계열은 "끌고 먼 길을 감" 이미지.

- `trek`: 힘든 여정
- `trekking`: 도보 장거리 여행
- `trekker`: 트레킹하는 사람
- `trigger`: 당겨서 작동시키는 장치

비교:

- [[tract]]: 라틴어 끌다 계열
- `track`: 자국, 길, 추적
