# rupt

- 출발 단어: `interrupt`
- 핵심 어원: 라틴어 `rumpere` / `rupt-` = 깨다, 터뜨리다
- 핵심 이미지: 이어진 것을 깨뜨림
- 관련해서 헷갈릴 수 있는 어근: [[fract]] = 부수다, 깨다 계열

## 출발 단어 분석

`interrupt`는 `inter-` + `rupt`.

- `inter-`: between
- `rupt`: break

따라서 `interrupt`는 "사이에 끼어 흐름을 깨뜨림" → "방해하다, 중단시키다"의 뜻.

## 단어들

### interrupt

- 구성: `inter-` = between + `rupt`
- 뜻: 방해하다, 중단시키다
- 어원적 연결: 사이에 들어와 흐름을 끊음
- 사용 맥락: `interrupt a conversation` = 대화를 끊다

### interruption

- 구성: `interrupt` + `-ion`
- 뜻: 방해, 중단
- 어원적 연결: 흐름이 깨진 상태
- 사용 맥락: `without interruption` = 중단 없이

### disrupt

- 구성: `dis-` = apart + `rupt`
- 뜻: 방해하다, 혼란시키다
- 어원적 연결: 질서나 흐름을 갈라 깨뜨림
- 사용 맥락: `disrupt the schedule` = 일정을 방해하다

### disruption

- 구성: `disrupt` + `-ion`
- 뜻: 혼란, 차질
- 어원적 연결: 정상 흐름이 깨짐
- 사용 맥락: `supply chain disruption` = 공급망 차질

### abrupt

- 구성: `ab-` = off, away + `rupt`
- 뜻: 갑작스러운, 퉁명스러운, 가파른
- 어원적 연결: 이어지던 흐름이 툭 끊어진 듯함
- 사용 맥락: `an abrupt change` = 갑작스러운 변화, `an abrupt tone` = 퉁명스러운 말투

### abruptly

- 구성: `abrupt` + `-ly`
- 뜻: 갑자기, 불쑥, 퉁명스럽게
- 어원적 연결: 자연스러운 연결 없이 끊기듯 일어남
- 사용 맥락: `The meeting ended abruptly.` = 회의가 갑자기 끝났다

### corrupt

- 구성: `cor-` = together, thoroughly + `rupt`
- 뜻: 부패한, 타락시키다
- 어원적 연결: 안쪽까지 완전히 망가뜨림
- 사용 맥락: `corrupt officials` = 부패한 공무원들

### corruption

- 구성: `corrupt` + `-ion`
- 뜻: 부패, 타락
- 어원적 연결: 제도나 사람이 망가진 상태
- 사용 맥락: `political corruption` = 정치 부패

### erupt

- 구성: `e- / ex-` = out + `rupt`
- 뜻: 분출하다, 폭발하다
- 어원적 연결: 밖으로 터져 나옴
- 사용 맥락: `a volcano erupts` = 화산이 분출하다

### eruption

- 구성: `erupt` + `-ion`
- 뜻: 분출, 폭발, 발진
- 어원적 연결: 안에서 밖으로 터져 나온 것
- 사용 맥락: `a volcanic eruption` = 화산 폭발

### bankrupt

- 구성: Italian `banca rotta` = broken bench
- 뜻: 파산한
- 어원적 연결: 장사판이 깨진 상태에서 파산의 뜻
- 사용 맥락: `go bankrupt` = 파산하다

### rupture

- 구성: Latin `ruptura`
- 뜻: 파열, 결렬, 파열되다
- 어원적 연결: 연결된 것이 찢어져 깨짐
- 사용 맥락: `a rupture in relations` = 관계의 결렬

## 기억 포인트

`rupt` 계열은 "깨뜨림 / 터짐" 이미지.

- `interrupt`: 사이에서 끊음
- `disrupt`: 질서를 깨뜨림
- `abrupt`: 흐름이 끊어진 듯 갑작스러움
- `corrupt`: 속까지 망가뜨림
- `erupt`: 밖으로 터짐
- `bankrupt`: 경제적으로 깨짐
- `rupture`: 파열됨
