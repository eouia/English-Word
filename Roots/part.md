# part

- 출발 단어: `participate`
- 핵심 어원: 라틴어 `pars` / `part-` = 부분, 몫
- 핵심 이미지: 전체에서 나뉜 부분이나 몫
- 관련해서 헷갈릴 수 있는 어근: [[trib]] = 나누어 배정하다 계열, [[piece]] = 조각, 한 단위 계열

## 출발 단어 분석

`participate`는 Latin `particeps`에서 온 말.

- `part`: share, part = 몫, 부분
- `capere`: take = 잡다, 취하다

따라서 `participate`는 "몫을 취하다" → "참여하다"의 뜻.

## 단어들

### part

- 발음: /ˈpɑrt/
- 구성: Latin `pars`
- 뜻: 부분, 역할, 나누다
- 어원적 연결: 전체에서 나뉜 몫
- 사용 맥락: `part of the problem` = 문제의 일부

### partial

- 발음: /ˈpɑrʃəl/
- 구성: `part` + `-ial`
- 뜻: 부분적인, 편파적인
- 어원적 연결: 전체가 아니라 한 부분에 치우침
- 사용 맥락: `a partial solution` = 부분적인 해결책

### partially

- 발음: /ˈpɑrʃəli/
- 구성: `partial` + `-ly`
- 뜻: 부분적으로
- 어원적 연결: 전체가 아니라 일부만
- 사용 맥락: `partially true` = 부분적으로 사실인

### impartial

- 발음: /ˌɪˈmpɑrʃəl/
- 구성: `im-` = not + `partial`
- 뜻: 공정한, 편파적이지 않은
- 어원적 연결: 어느 한쪽 부분이나 편에 치우치지 않음
- 사용 맥락: `an impartial judge` = 공정한 판사, `impartial advice` = 편견 없는 조언

### impartiality

- 발음: /ˌɪˌmpɑrʃiˈælɪti/
- 구성: `impartial` + `-ity`
- 뜻: 공정성, 중립성
- 어원적 연결: 한쪽 편에 기울지 않는 상태
- 사용 맥락: `maintain impartiality` = 공정성을 유지하다

### partiality

- 발음: /ˌpɑrʃiˈæləti/
- 구성: `partial` + `-ity`
- 뜻: 편파성, 편애; 특별히 좋아함
- 어원적 연결: 전체가 아니라 특정 부분이나 편에 마음이 기움
- 사용 맥락: `show partiality` = 편파성을 보이다, `a partiality for sweets` = 단것을 특히 좋아함

### particle

- 어원적 연결: 아주 작은 부분
- 사용 맥락: `dust particles` = 먼지 입자

### particular

- 발음: /pɚˈtɪkjəlɚ/
- 구성: Latin `particularis`
- 뜻: 특정한, 특별한
- 어원적 연결: 전체가 아니라 한 부분에 속한 것

### participate

### participant

- 어원적 연결: 몫을 갖고 참여하는 사람

### participation

- 발음: /pɑˌrtɪsəˈpeɪʃən/
- 구성: `participate` + `-ion`
- 뜻: 참여, 참가
- 어원적 연결: 몫을 갖고 함께하는 행위
- 사용 맥락: `active participation` = 적극적 참여

### partisan

- 발음: /ˈpɑrtəzən/
- 구성: Italian/French `partisan` ← `part` = side, faction
- 뜻: 당파적인 사람, 열성 지지자; 당파적인
- 어원적 연결: 어느 한 부분·편에 속해 그쪽을 지지함
- 사용 맥락: `partisan politics` = 당파 정치, `a partisan debate` = 당파적 논쟁

### portion

- 발음: /ˈpɔrʃən/
- 구성: Latin `portio` = share
- 뜻: 부분, 몫, 1인분
- 어원적 연결: 나누어진 몫
- 사용 맥락: `a large portion` = 큰 몫, 많은 양

### proportion

- 발음: /prəˈpɔrʃən/
- 구성: `pro-` = according to + `portion`
- 뜻: 비율, 균형
- 어원적 연결: 몫과 몫 사이의 알맞은 관계
- 사용 맥락: `a small proportion` = 작은 비율

### partition

- 발음: /pɑˈrtɪʃən/
- 구성: `part` + `-ition`
- 뜻: 분할, 칸막이
- 어원적 연결: 전체를 부분으로 나눔
- 사용 맥락: `partition a room` = 방을 나누다

### depart

- 발음: /dɪˈpɑrt/
- 구성: `de-` = apart + `part`
- 뜻: 떠나다
- 어원적 연결: 갈라져 떨어져 나감
- 사용 맥락: `depart from Seoul` = 서울에서 출발하다

### department

- 발음: /dɪˈpɑrtmənt/
- 구성: `depart` + `-ment`
- 뜻: 부서, 학과
- 어원적 연결: 전체 조직에서 나뉜 부분
- 사용 맥락: `the marketing department` = 마케팅 부서

## 기억 포인트

`part` 계열은 "부분 / 몫 / 나눔" 이미지.

- `part`: 부분
- `partial`: 한 부분에 치우친
- `impartial`: 어느 한 편에 치우치지 않음
- `particle`: 작은 부분
- `participate`: 몫을 갖고 참여함
- `partisan`: 한 편에 속해 지지함
- `portion`: 나뉜 몫
- `proportion`: 몫들의 비율
- `partition`: 부분으로 나눔
- `department`: 나뉜 부서
