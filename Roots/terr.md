# terr

- 출발 단어: `territory`
- 핵심 어원: Latin `terra` = 땅, 지구
- 핵심 이미지: 발 아래의 땅과 그 땅의 영역
- 관련 문서: [[geo]]

## 단어들

### terrain

- 발음: /tɚˈeɪn/
- 구성: Latin `terra`
- 뜻: 지형, 지세
- 어원적 연결: 땅의 모양과 상태
- 사용 맥락: `rough terrain` = 험한 지형

### territory

- 발음: /ˈtɛrɪˌtɔri/
- 구성: Latin `terra` + `-ory`
- 뜻: 영토, 영역
- 어원적 연결: 특정 땅의 범위
- 사용 맥락: `defend territory` = 영토를 지키다

### terrestrial

- 발음: /tɚˈɛstriəl/
- 구성: Latin `terra` + `-ial`
- 뜻: 지구의, 육지의
- 어원적 연결: 하늘이나 바다가 아니라 땅에 속함
- 사용 맥락: `terrestrial animals` = 육상 동물

### subterranean

- 발음: /səbtɚˈeɪniən/
- 구성: `sub-` = under + Latin `terra`
- 뜻: 지하의
- 어원적 연결: 땅 아래에 있음
- 사용 맥락: `subterranean tunnels` = 지하 터널

### Mediterranean

- 발음: /ˌmɛdətɚˈeɪniən/
- 구성: Latin `medius` = middle + `terra`
- 뜻: 지중해의
- 어원적 연결: 땅들 사이에 있는 바다
- 사용 맥락: `Mediterranean climate` = 지중해성 기후

## 기억 포인트

`terr` 계열은 "땅과 영토" 이미지.

- `terrain`: 땅의 모양
- `territory`: 땅의 영역
- `terrestrial`: 땅에 속한
- `subterranean`: 땅 아래의

## 비교 단어

### geo

- 뜻: 지구, 땅
- 구분 포인트: [[geo]]는 그리스어 땅 계열. `terr`는 라틴어 땅 계열.
