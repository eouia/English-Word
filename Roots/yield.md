# yield

- 출발 단어: `yield`
- 핵심 어원: Old English `gieldan` / `geldan` = pay, repay, give back
- 핵심 이미지: 값을 치르거나, 가진 것을 내줌
- 관련해서 헷갈릴 수 있는 어근: [[cess]] = 가다, 물러나다, 양보하다 계열

## 출발 단어 분석

`yield`는 고대 영어 `gieldan`에서 온 기본 동사.

- 원래 뜻: 값을 치르다, 갚다, 보답하다
- 의미 변화: 값을 내다 → 결과를 돌려주다 → 산출하다 / 넘겨주다 → 양보하다

따라서 현대 영어 `yield`는 "내놓다"라는 이미지가 중심.

## 단어들

### yield

- 발음: /ˈjild/
- 구성: Old English `gieldan`
- 뜻: 산출하다, 내주다, 양보하다, 굴복하다
- 어원적 연결: 어떤 값이나 결과를 돌려 내놓음
- 사용 맥락: `The farm yields wheat.` = 그 농장은 밀을 산출한다

### yield

- 발음: /ˈjild/
- 구성: 명사형 `yield`
- 뜻: 수확량, 산출량, 수익률
- 어원적 연결: 땅, 투자, 과정이 돌려주는 결과
- 사용 맥락: `crop yield` = 작물 수확량

### yielding

- 발음: /ˈjildɪŋ/
- 구성: `yield` + `-ing`
- 뜻: 잘 양보하는, 유연한, 쉽게 휘는
- 어원적 연결: 저항하지 않고 내줌
- 사용 맥락: `a yielding surface` = 눌리면 들어가는 표면

### unyielding

- 발음: /əˈnjildɪŋ/
- 구성: `un-` = not + `yielding`
- 뜻: 완고한, 굽히지 않는
- 어원적 연결: 자기 입장이나 형태를 내주지 않음
- 사용 맥락: `unyielding resistance` = 굽히지 않는 저항

### yield to

- 구성: `yield` + `to`
- 뜻: ~에 양보하다, ~에 굴복하다
- 어원적 연결: 권리, 자리, 주장, 힘을 상대에게 내줌
- 사용 맥락: `yield to pressure` = 압력에 굴복하다

### high-yield

- 구성: `high` + `yield`
- 뜻: 고수익의, 산출이 많은
- 어원적 연결: 많이 돌려주는 것
- 사용 맥락: `high-yield bonds` = 고수익 채권

## 기억 포인트

`yield`의 중심은 "내놓음".

- 밭이 곡식을 내놓음: `yield wheat`
- 투자가 수익을 내놓음: `yield a profit`
- 사람이 권리나 입장을 내놓음: `yield to others`
- 내놓지 않음: `unyielding`

## 비교 단어

### produce

- 뜻: 생산하다, 산출하다
- 구분 포인트: `produce`는 앞으로 끌어내어 만들어 냄. `yield`는 과정이나 대상이 결과를 돌려줌

### surrender

- 뜻: 항복하다, 넘겨주다
- 구분 포인트: [[render]]의 `surrender`는 포기하고 넘김이 더 강함. `yield`는 양보, 굴복, 산출까지 넓게 씀

### guild / geld

- 뜻: 조합, 회비 / 돈, 세금
- 구분 포인트: `guild`는 역사적으로 `gield` = 지불, 기여, 회비 쪽과 연결됨. `geld`도 "세금, 돈" 계열. 따라서 `yield`와 같은 "값을 치르다, 내놓다" 이미지로 묶을 수 있음

### gild / gold

- 뜻: 금을 입히다 / 금
- 구분 포인트: `gild`는 `gold`에서 나온 단어. `gold`는 "빛나다, 노랗다" 이미지. `yield` / `guild` / `geld`의 "지불하다, 내놓다" 계열과는 다름
