# idea

- 출발 단어: `ideals`
- 핵심 어원: Greek `idea` / `eidos` = form, look, visible shape
- 깊은 어근: PIE `*weid-` = 보다, 알다
- 핵심 이미지: 눈에 보이는 형태가 마음속의 모범·개념으로 잡힘
- 관련 형태: `idea`, `ideal`, `ideals`, `idealism`, `idealize`, `ideology`
- 관련 문서: [[vid]], [[theor]], [[logos]]
- 주의: `idea/ideal`은 Greek `idein` = 보다 계열. `idyllic`은 목가적 작은 그림·시를 뜻하는 Greek `eidyllion` 계열로 연결되지만, 현대 학습에서는 `ideal`과 섞어 외우지 않는 편이 안전함.

## 출발 단어 분석

`ideals`는 `ideal`의 복수형.

- `ideal`: Late Latin `idealis` = existing in idea
- Latin `idea`: Platonic idea, archetype
- Greek `idea`: 형태, 모습, 본질적 형상
- Greek `idein`: 보다

따라서 `ideal`은 원래 "마음속 형상이나 모범에 속한" → "완전한 기준, 이상적인"의 뜻. `ideals`는 사람이 따르려는 높은 가치나 기준.

## 단어들

### idea

- 발음: /aɪˈdiə/
- 구성: Greek `idea` = form, look
- 뜻: 생각, 발상, 개념
- 어원적 연결: 눈에 보이는 형태가 마음속에 잡힌 형상·개념으로 확장
- 사용 맥락: `a good idea` = 좋은 생각, `the idea of freedom` = 자유라는 개념

### ideal / ideals

- 발음: `ideal` /aɪˈdiəl/; `ideals` /aɪˈdiəlz/
- 구성: `idea` + `-al`
- 뜻: 이상적인; 이상, 이상적 기준
- 어원적 연결: 실제 사물보다 마음속의 완전한 형상·모범에 가까움
- 사용 맥락: `democratic ideals` = 민주주의적 이상, `an ideal solution` = 이상적인 해결책

### idealism

- 발음: /aɪˈdiəlɪzəm/
- 구성: `ideal` + `-ism`
- 뜻: 이상주의, 관념론
- 어원적 연결: 현실보다 이상적 기준이나 마음속 형상을 더 중요하게 봄
- 사용 맥락: `political idealism` = 정치적 이상주의

### idealist

- 발음: /aɪˈdiəlɪst/
- 구성: `ideal` + `-ist`
- 뜻: 이상주의자
- 어원적 연결: 현실적 이익보다 이상적 가치나 모범을 따르려는 사람
- 사용 맥락: `a young idealist` = 젊은 이상주의자

### idealize

- 발음: /aɪˈdiˌlaɪz/
- 구성: `ideal` + `-ize`
- 뜻: 이상화하다
- 어원적 연결: 실제 대상을 마음속 완전한 형상에 가깝게 봄
- 사용 맥락: `idealize the past` = 과거를 이상화하다

### ideological

- 발음: /ˌaɪdiəˈlɑdʒɪkəl/
- 구성: `ideology` + `-ical`
- 뜻: 이념적인
- 어원적 연결: 개별 생각이 아니라 사상 체계나 이념에 속함
- 사용 맥락: `ideological conflict` = 이념적 갈등

### ideology

- 발음: /ˌaɪdiˈɑlədʒi/
- 구성: `idea` + [[logos]] = study, discourse
- 뜻: 이념, 관념 체계
- 어원적 연결: 생각들을 말과 논리로 엮어 하나의 체계로 세움
- 사용 맥락: `political ideology` = 정치 이념
- 관련 문서: [[logos]]

## 비교 단어

### vision

- 뜻: 시각, 비전, 미래상
- 어원적 연결: [[vid]] 계열. Latin `videre` = 보다
- 구분 포인트: `vision`은 보는 능력이나 미래 그림. `ideal`은 추구할 완전한 기준이나 가치

### theory

- 뜻: 이론
- 어원적 연결: [[theor]] 계열. Greek `theorein` = 바라보다, 관찰하다
- 구분 포인트: `theory`는 관찰과 설명 체계. `idea`는 마음속 개념이나 발상

## 기억 포인트

`idea` 계열은 "보이는 형태가 마음속 형상·이상으로 잡힘" 이미지.

- `idea`: 마음속 개념
- `ideal`: 완전한 형상이나 기준
- `ideals`: 추구하는 가치와 이상
- `idealism`: 이상을 중시하는 태도
- `idealize`: 현실을 이상적 모습으로 봄
- `ideology`: 생각들이 체계가 된 이념
