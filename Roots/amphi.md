# amphi

- 출발 단어: `amphibian`
- 핵심 어원: Greek `amphi-` = 양쪽에, 둘레에, 두 종류의
- 깊은 어근: PIE `*ambhi-` = around, on both sides
- 핵심 이미지: 양쪽에 걸치거나 둘레를 둘러쌈
- 관련 문서: [[di]], [[bi]], [[ambi]]
- 주의: `amphi-`는 Greek 계열. Latin `ambi-`와 의미와 깊은 뿌리는 가깝지만 영어 문서에서는 분리.

## 출발 단어 분석

`amphibian` ← Greek `amphibios`

- `amphi-`: 양쪽에, 두 종류의
- `bios`: life
- 원래 이미지: 두 종류의 삶을 사는 것
- 현대 의미: 양서류, 수륙 양용의

## 단어들

### amphibian

- 발음: /æˈmfɪbiən/
- 구성: `amphi-` = both kinds + `bio` = life
- 뜻: 양서류
- 어원적 연결: 물과 땅, 두 환경에서 사는 생물
- 사용 맥락: `frogs are amphibians` = 개구리는 양서류다

### amphibious

- 발음: /æˈmfɪbiəs/
- 구성: Greek `amphibios`
- 뜻: 수륙 양용의, 양서의
- 어원적 연결: 물과 땅 양쪽에서 작동하거나 살아감
- 사용 맥락: `an amphibious vehicle` = 수륙 양용 차량

### amphitheater

- 발음: /ˈæmfəθiˈeɪtɚ/
- 구성: `amphi-` = around + `theater`
- 뜻: 원형극장, 계단식 관람석
- 어원적 연결: 관객이 둘레를 둘러싸고 보는 극장
- 사용 맥락: `a Roman amphitheater` = 로마 원형극장

### amphitheatre

- 발음: /ˈæmfəθiˈeɪtɚ/
- 구성: British spelling of `amphitheater`
- 뜻: 원형극장
- 어원적 연결: 같은 어원, 영국식 철자
- 사용 맥락: `an ancient amphitheatre` = 고대 원형극장

### amphora

- 발음: /ˈæmfɚə/
- 구성: Greek `amphoreus` = two-handled jar
- 뜻: 암포라, 양손잡이 항아리
- 어원적 연결: 양쪽에 손잡이가 달린 용기
- 사용 맥락: `a Greek amphora` = 그리스 암포라

### amphoteric

- 구성: Greek `amphoteros` = each of two
- 뜻: 양쪽성의, 산과 염기로 모두 작용하는
- 어원적 연결: 두 성질을 모두 가질 수 있음
- 사용 맥락: `amphoteric substances` = 양쪽성 물질

### amphipod

- 구성: `amphi-` = both sides + Greek `pod` = foot
- 뜻: 단각류
- 어원적 연결: 양쪽으로 난 발 모양에서 온 동물 분류명
- 사용 맥락: `marine amphipods` = 해양 단각류

### amphiphilic

- 구성: `amphi-` = both + Greek `philos` = loving
- 뜻: 양친매성의
- 어원적 연결: 물과 기름 양쪽 성질을 모두 좋아함
- 사용 맥락: `amphiphilic molecules` = 양친매성 분자

## 기억 포인트

`amphi-`는 "양쪽/둘레" 이미지.

- `amphibian`: 물과 땅 양쪽의 삶
- `amphibious`: 물과 땅 양쪽에서 작동
- `amphitheater`: 둘레로 둘러싼 극장
- `amphora`: 양쪽 손잡이 항아리
- `amphoteric`: 두 성질을 모두 가짐

## 비교 단어

### ambi

- 뜻: 양쪽에, 둘레에
- 구분 포인트: [[ambi]]는 Latin 계열. `ambiguous`, `ambivalent`, `ambient`처럼 라틴·프랑스어 경로 단어에 자주 보임.

### bi

- 뜻: 둘
- 구분 포인트: `bi-`는 숫자 2 자체. `amphi-`는 두 쪽에 걸치거나 주위를 둘러싼 배치 이미지.
