# honor

- 출발 단어: `honor`
- 핵심 어원: 라틴어 `honor`, `honos` = 명예, 존경, 공적 지위
- 핵심 이미지: 남에게 인정받는 가치 / 사회적으로 높이 여김
- 관련해서 헷갈릴 수 있는 어근: [[dign]] = 가치와 품위 계열, [[glori]] = 영광과 찬양 계열

## 출발 단어 분석

`honor`는 라틴어 `honor`에서 온 단어.

- `honor`: esteem, public respect, office

따라서 `honor`는 "존경받는 가치" → "명예, 영예" → "존경하다, 예우하다"의 뜻.

## 단어들

### honor

- 구성: 라틴어 `honor`
- 뜻: 명예, 영예; 존경하다
- 어원적 연결: 사회적으로 높이 인정받는 가치
- 사용 맥락: `win an honor` = 영예를 얻다, `honor a promise` = 약속을 지키다

### honorable

- 구성: `honor` + `-able`
- 뜻: 명예로운, 존경할 만한
- 어원적 연결: 존경받을 만한 가치가 있음
- 사용 맥락: `an honorable person` = 명예로운 사람

### honorary

- 구성: `honor` + `-ary`
- 뜻: 명예직의, 명예상의
- 어원적 연결: 실제 직무나 보수보다 예우와 존경의 의미가 큼
- 사용 맥락: `an honorary degree` = 명예 학위

### honored

- 구성: `honor` + `-ed`
- 뜻: 영광으로 여기는, 존경받는
- 어원적 연결: 명예를 받은 상태
- 사용 맥락: `I am honored to be here.` = 이 자리에 있어 영광임

### dishonor

- 구성: `dis-` = opposite, away + `honor`
- 뜻: 불명예, 명예를 더럽히다
- 어원적 연결: 존경받는 가치를 잃게 함
- 사용 맥락: `bring dishonor to the family` = 가족에게 불명예를 가져오다

### honest

- 구성: 라틴어 `honestus` = honorable, respectable
- 뜻: 정직한, 솔직한
- 어원적 연결: 원래는 "명예로운, 존경할 만한"의 뜻. 현대 영어에서는 도덕적으로 바르고 거짓이 없음
- 사용 맥락: `an honest answer` = 정직한 대답

### honesty

- 구성: `honest` + `-y`
- 뜻: 정직, 솔직함
- 어원적 연결: 명예로움이 도덕적 바름으로 좁아짐
- 사용 맥락: `value honesty` = 정직을 중시하다

## 기억 포인트

`honor` 계열은 "남에게 인정받는 가치"에서 출발.

- `honor`: 명예, 존경
- `honorable`: 존경할 만한
- `honorary`: 명예상의
- `dishonor`: 불명예
- `honest`: 명예로운 사람다운 → 정직한

## 비교 단어

### dignity

- 뜻: 존엄, 품위
- 구분 포인트: [[dign]]은 "가치 있음, 알맞은 자격"의 이미지. `honor`는 남에게 인정받고 존경받는 사회적 명예에 가까움

### glory

- 뜻: 영광, 찬란한 명성
- 구분 포인트: `glory`는 빛나는 명성과 찬양의 느낌이 강함. `honor`는 존경, 예우, 명예로운 책임에 가까움
