# precari

- 출발 단어: `precarious`
- 핵심 어원: Latin `prex`, `precari` = prayer, entreaty, ask earnestly
- 핵심 이미지: 간청에 의존함, 남의 허락이나 호의에 달림
- 관련 문서: [[postul]], [[pet]]

## 단어들

### precarious

- 발음: /priˈkɛriəs/
- 구성: Latin `precarius` = obtained by entreaty, dependent on favor
- 뜻: 불안정한, 위태로운
- 어원적 연결: 간청해서 얻은 것처럼 자기 권리로 안정된 것이 아니라 남의 호의에 달림
- 사용 맥락: `a precarious position` = 위태로운 처지

### precariously

- 발음: /prɪˈkɛriəsli/
- 구성: `precarious` + `-ly`
- 뜻: 위태롭게, 불안정하게
- 어원적 연결: 안정된 기반 없이 간신히 유지되는 방식
- 사용 맥락: `perched precariously` = 위태롭게 걸쳐 있는

### precariousness

- 구성: `precarious` + `-ness`
- 뜻: 불안정함, 위태로움
- 어원적 연결: 남의 호의나 외부 조건에 달려 안정되지 않은 상태
- 사용 맥락: `the precariousness of temporary work` = 임시직의 불안정성

### precarity

- 구성: French `precarite` < Latin `precarius`
- 뜻: 불안정성, 특히 노동·생활의 불안정성
- 어원적 연결: 생계나 권리가 안정적으로 보장되지 않음
- 사용 맥락: `economic precarity` = 경제적 불안정성

### pray

- 발음: /ˈpreɪ/
- 구성: Old French `preier` < Latin `precari`
- 뜻: 기도하다, 간청하다
- 어원적 연결: 간절히 요청함
- 사용 맥락: `pray for peace` = 평화를 위해 기도하다

### prayer

- 발음: /ˈprɛr/
- 구성: `pray` + `-er`
- 뜻: 기도, 간청
- 어원적 연결: 간절한 요청이나 빎
- 사용 맥락: `answer a prayer` = 기도에 응답하다

### deprecate

- 발음: /ˈdɛprəˌkeɪt/
- 구성: Latin `de-` = away + `precari` = pray, entreat
- 뜻: 반대하다, 비판하다; 낮추어 말하다
- 어원적 연결: 원래는 기도하듯 재앙을 물리쳐 달라고 빎. 이후 반대하거나 못마땅하게 여김
- 사용 맥락: `self-deprecating humor` = 자기를 낮추는 유머

### deprecation

- 구성: `deprecate` + `-ion`
- 뜻: 반대, 비판; 낮추어 봄
- 어원적 연결: 어떤 것을 좋지 않게 여기며 물리치려는 태도
- 사용 맥락: `self-deprecation` = 자기비하

### imprecation

- 구성: Latin `im-` = upon + `precari` = pray
- 뜻: 저주, 악담
- 어원적 연결: 누군가에게 나쁜 일이 일어나기를 기도함
- 사용 맥락: 문학적·격식 문맥에서 저주를 말할 때 사용

## 기억 포인트

`precari` 계열은 "기도하다, 간청하다" 이미지.

- `pray`: 기도하다, 간청하다
- `prayer`: 기도
- `precarious`: 남의 호의에 달려 불안정한
- `precarity`: 불안정성
- `deprecate`: 못마땅해하다, 낮추어 말하다
- `imprecation`: 저주

## 비교 단어

### postulate

- 뜻: 요구하다, 전제로 삼다
- 구분 포인트: [[postul]]은 요구하거나 전제로 받아들이게 하는 느낌. `precari`는 간절히 빌거나 호의에 기대는 느낌.

### petition

- 뜻: 청원하다, 청원
- 구분 포인트: [[pet]]은 목표를 향해 요구함. `precari`는 더 간청·기도에 가까움.

### dangerous

- 뜻: 위험한
- 구분 포인트: `dangerous`는 위험 자체. `precarious`는 안정된 기반이 없어 위험해질 수 있는 상태.
