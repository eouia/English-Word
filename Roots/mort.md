# mort

- 출발 단어: `mortal`
- 핵심 어원: 라틴어 `mors` / `mort-` = 죽음
- 핵심 이미지: 죽을 수 있음, 생명의 끝
- 관련해서 헷갈릴 수 있는 어근: [[carn]] = 살, 육체 계열

## 출발 단어 분석

`mortal`은 Latin `mortalis`에서 온 말.

- `mors`: death
- `mortalis`: subject to death

따라서 `mortal`은 "죽음에 속한" → "죽을 수 있는, 인간의" 뜻.

## 단어들

### mortal

- 발음: /ˈmɔrtəl/
- 구성: `mort` + `-al`
- 뜻: 죽을 수 있는, 치명적인, 인간
- 어원적 연결: 죽음의 조건을 가진 존재
- 사용 맥락: `mortal danger` = 치명적 위험

### mortality

- 발음: /mɔˈrtæləti/
- 구성: `mortal` + `-ity`
- 뜻: 사망률, 죽을 운명
- 어원적 연결: 죽음과 관련된 성질이나 비율
- 사용 맥락: `infant mortality` = 영아 사망률

### immortal

- 발음: /ˌɪˈmɔrtəl/
- 구성: [[in]] = not + `mortal`
- 뜻: 불멸의, 죽지 않는
- 어원적 연결: 죽음의 조건이 없음
- 사용 맥락: `immortal fame` = 불멸의 명성

### immortality

- 발음: /ˌɪmɔˈrtælɪti/
- 구성: `immortal` + `-ity`
- 뜻: 불멸
- 어원적 연결: 죽지 않는 상태
- 사용 맥락: `seek immortality` = 불멸을 추구하다

### mortify

- 발음: /ˈmɔrtɪˌfaɪ/
- 구성: `mort` + `-ify`
- 뜻: 굴욕감을 주다, 극도로 당황하게 하다
- 어원적 연결: 원래 "죽이다, 억누르다"에서 자존심을 죽이는 느낌으로 변화
- 사용 맥락: `be mortified by a mistake` = 실수로 몹시 창피해하다

### mortgage

- 발음: /ˈmɔrgədʒ/
- 구성: Old French `mort gage` = dead pledge
- 뜻: 주택 담보 대출
- 어원적 연결: 빚을 갚으면 담보가 "죽고", 못 갚으면 권리가 끝나는 약속
- 사용 맥락: `pay off a mortgage` = 주택 대출을 갚다

### postmortem

- 발음: /ˌpoʊˈstmɔrtɛm/
- 구성: `post-` = after + `mortem` = death
- 뜻: 사후의, 부검, 사후 분석
- 어원적 연결: 죽음 뒤에 이루어짐
- 사용 맥락: `a project postmortem` = 프로젝트 사후 분석

### mortuary

- 발음: /ˈmɔrtʃuˌɛri/
- 구성: `mort` + `-uary`
- 뜻: 영안실의, 장례의
- 어원적 연결: 죽은 사람과 관련된 장소나 일
- 사용 맥락: `mortuary services` = 장례 서비스

## 기억 포인트

`mort` 계열은 "죽음" 이미지.

- `mortal`: 죽을 수 있음
- `mortality`: 죽음의 비율이나 조건
- `immortal`: 죽지 않음
- `mortify`: 자존심을 죽이다
- `mortgage`: 죽은 담보 약속
- `postmortem`: 죽음 뒤

## 비교 단어

### carn

- 뜻: 살, 고기, 육체
- 구분 포인트: [[carn]]은 육체 자체. `mort`는 육체가 끝나는 죽음에 초점
