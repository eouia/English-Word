# ultra

- 출발 단어: `ultra`
- 핵심 어원: Latin `ultra` = beyond, on the far side of
- 핵심 이미지: 선을 넘어 더 멀리, 한계를 넘은 극단
- 관련 문서: [[beyond]], [[trans]], [[super]]
- 주의: 현대 영어의 `ultra-`는 단순한 위치의 "너머"보다 "극도로, 과도하게"의 강조 접두사로 많이 쓰임.

## 출발 단어 분석

`ultra` ← Latin `ultra`

- `ultra`: beyond, on the far side
- 원래 이미지: 기준선의 저쪽 너머
- 현대 의미: 극단적인, 초월한, 매우

## 단어들

### ultra

- 구성: Latin `ultra` = beyond
- 뜻: 극단주의자, 극단적인, 초고도의
- 어원적 연결: 보통 기준을 넘어선 상태
- 사용 맥락: `an ultra-conservative group` = 극보수 집단

### ultraviolet

- 구성: `ultra-` = beyond + `violet`
- 뜻: 자외선의
- 어원적 연결: 보라색보다 더 짧은 파장 쪽, 눈에 보이는 범위 너머
- 사용 맥락: `ultraviolet radiation` = 자외선 복사

### ultrasound

- 구성: `ultra-` = beyond + `sound`
- 뜻: 초음파, 초음파 검사
- 어원적 연결: 사람이 들을 수 있는 소리의 범위를 넘어선 주파수
- 사용 맥락: `an ultrasound scan` = 초음파 검사

### ultramodern

- 구성: `ultra-` = beyond, extremely + `modern`
- 뜻: 초현대적인
- 어원적 연결: 보통의 현대적 수준을 넘어섬
- 사용 맥락: `an ultramodern building` = 초현대식 건물

### ultraconservative

- 구성: `ultra-` = extremely + `conservative`
- 뜻: 극보수적인
- 어원적 연결: 보수적 성향이 보통 수준을 넘어 극단에 가까움
- 사용 맥락: 정치·사회 기사에서 자주 보임

### ultralight

- 구성: `ultra-` = extremely + `light`
- 뜻: 초경량의
- 어원적 연결: 매우 가벼운 상태
- 사용 맥락: `ultralight equipment` = 초경량 장비

### ultimate

- 구성: Latin `ultimus` = farthest, last
- 뜻: 최후의, 궁극적인, 최고의
- 어원적 연결: 가장 먼 끝, 마지막 단계
- 사용 맥락: `the ultimate goal` = 궁극적 목표

### ultimately

- 구성: `ultimate` + `-ly`
- 뜻: 궁극적으로, 결국
- 어원적 연결: 마지막 단계에 이르면
- 사용 맥락: `ultimately responsible` = 궁극적으로 책임이 있는

### ulterior

- 구성: Latin `ulterior` = farther, more distant
- 뜻: 숨은, 이면의, 더 먼
- 어원적 연결: 겉으로 보이는 것 너머에 있음
- 사용 맥락: `an ulterior motive` = 숨은 동기

## 기억 포인트

`ultra` 계열은 "너머, 한계를 넘은 극단" 이미지.

- `ultraviolet`: 보라색 너머의 빛
- `ultrasound`: 들을 수 있는 소리 너머
- `ultramodern`: 현대적인 수준을 넘어섬
- `ultimate`: 가장 먼 끝, 마지막 단계
- `ulterior`: 겉 너머의 숨은 것

## 비교 단어

### beyond

- 뜻: 저 너머에, 한계를 지나서
- 구분 포인트: [[beyond]]는 일상 부사·전치사. `ultra-`는 Latin 접두사로 결합어를 만들며 "극도로"의 강조가 강함.

### trans

- 뜻: 가로질러, 넘어서
- 구분 포인트: [[trans]]는 한쪽에서 다른 쪽으로 건너가는 움직임이 강함. `ultra`는 기준 너머의 극단 위치나 정도가 강함.

### super

- 뜻: 위에, 넘어서, 과도하게
- 구분 포인트: [[super]]는 위쪽·우월·초과 이미지. `ultra`는 선 너머의 극단 이미지.
