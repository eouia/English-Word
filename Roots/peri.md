# peri

- 출발 단어: `peripheral`
- 핵심 어원: Greek `peri` = around, about, enclosing
- 핵심 이미지: 둘레를 따라, 주변에, 주위를 감싸며
- 관련 문서: [[metr]], [[circ]], [[fer]]

## 단어들

### peripheral

- 발음: /pɚˈɪfɚəl/
- 구성: Greek `peri-` = around + `pherein` = carry + `-al`
- 뜻: 주변의, 부차적인; 말초의
- 어원적 연결: 중심이 아니라 둘레나 바깥쪽에 있는 것
- 사용 맥락: `peripheral vision` = 주변 시야, `a peripheral issue` = 부차적 문제

### periphery

- 발음: /pɚˈɪfɚi/
- 구성: Greek `periphereia` = circumference, outer surface
- 뜻: 주변부, 외곽, 말초
- 어원적 연결: 중심을 둘러싼 바깥 경계
- 사용 맥락: `on the periphery of the city` = 도시 외곽에

### peripherals

- 발음: /pɚˈɪfɚəlz/
- 구성: `peripheral`의 명사형 복수
- 뜻: 주변 기기
- 어원적 연결: 컴퓨터 본체의 중심 기능 바깥에 붙는 장치
- 사용 맥락: `computer peripherals` = 컴퓨터 주변 기기

### perimeter

- 발음: /pɚˈɪmətɚ/
- 구성: `peri-` = around + [[metr]] = measure
- 뜻: 둘레, 주변 경계선
- 어원적 연결: 둘레를 따라 잰 길이
- 사용 맥락: `secure the perimeter` = 주변 경계를 확보하다

### periscope

- 발음: /ˈpɛrəˌskoʊp/
- 구성: `peri-` = around + Greek `skopein` = look
- 뜻: 잠망경
- 어원적 연결: 주변을 둘러볼 수 있게 하는 장치
- 사용 맥락: 잠수함에서 수면 위를 관찰하는 장치

### periphrasis

- 구성: `peri-` = around + Greek `phrasis` = expression
- 뜻: 우회적 표현, 에둘러 말하기
- 어원적 연결: 직접 말하지 않고 주변을 돌아 표현함
- 사용 맥락: 문법, 수사학에서 쓰는 격식어

### peripatetic

- 발음: /ˌpɛrəpəˈtɛtɪk/
- 구성: Greek `peripatein` = walk about
- 뜻: 돌아다니는, 순회하는
- 어원적 연결: 이곳저곳 주변을 걸어 다님
- 사용 맥락: `a peripatetic teacher` = 순회 교사

### period

- 발음: /ˈpɪriəd/
- 구성: Greek `periodos` = going around, cycle
- 뜻: 기간, 시대; 마침표
- 어원적 연결: 한 바퀴 도는 시간의 주기
- 사용 맥락: `a difficult period` = 힘든 시기

## 기억 포인트

`peri-`는 "around" 이미지.

- `peripheral`: 주변의, 부차적인
- `periphery`: 주변부
- `perimeter`: 둘레를 잰 경계
- `periscope`: 주변을 보는 장치
- `periphrasis`: 둘러 말하기
- `period`: 한 바퀴 도는 주기

## 비교 단어

### circ

- 뜻: 원, 둘레, 주변을 돎
- 구분 포인트: [[circ]]는 Latin 계열. `peri-`는 Greek 계열.

### fer

- 뜻: 나르다, 가져가다
- 구분 포인트: [[fer]]는 Latin `ferre` 계열. `peripheral`의 `pherein`은 Greek "carry" 계열이라 어원이 다르지만 의미상 비교 가능.

### para

- 뜻: 옆에, 곁에, 넘어서, 반대로
- 구분 포인트: `para-`는 곁이나 빗겨남. `peri-`는 둘레를 감싸는 주변.
