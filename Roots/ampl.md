# ampl

- 출발 단어: `amplify`
- 핵심 어원: Latin `amplus` = large, spacious, abundant
- 핵심 이미지: 크고 넓음, 충분함, 더 크게 확장함
- 관련 문서: [[mag-magn]]
- 주의: `amplify`는 크기를 키우거나 내용을 확장함. `magnify`는 [[mag-magn]]처럼 크게 보이게 하거나 확대함의 이미지가 강함.

## 출발 단어 분석

`amplify` = `ampl` + `-ify`

- `amplus`: large, spacious, abundant
- `-ify`: make
- 원래 이미지: 더 크고 넓게 만들다
- 현대 의미: 증폭하다, 확대하다, 자세히 설명하다

## 단어들

### amplify

- 발음: /ˈæmpləˌfaɪ/
- 구성: Latin `amplus` = large, spacious + `-ify`
- 뜻: 증폭하다, 확대하다, 자세히 설명하다
- 어원적 연결: 소리·효과·설명을 더 크고 넓게 만듦
- 사용 맥락: `amplify the signal` = 신호를 증폭하다

### amplification

- 발음: /ˌæmpləfəˈkeɪʃən/
- 구성: `amplify` + `-ation`
- 뜻: 증폭, 확대, 부연 설명
- 어원적 연결: 더 크게 만들거나 더 넓게 설명하는 과정
- 사용 맥락: `sound amplification` = 음향 증폭

### amplifier

- 발음: /ˈæmpləˌfaɪɚ/
- 구성: `amplify` + `-er`
- 뜻: 증폭기, 앰프
- 어원적 연결: 소리나 신호를 크게 만드는 장치
- 사용 맥락: `a guitar amplifier` = 기타 앰프

### ample

- 발음: /ˈæmpəl/
- 구성: Latin `amplus` = large, abundant
- 뜻: 충분한, 넉넉한, 넓은
- 어원적 연결: 크고 넓어 부족하지 않음
- 사용 맥락: `ample evidence` = 충분한 증거

### amply

- 발음: /ˈæmpli/
- 구성: `ample` + `-ly`
- 뜻: 충분히, 넉넉히
- 어원적 연결: 부족하지 않을 만큼 넓고 많게
- 사용 맥락: `amply demonstrated` = 충분히 입증된

### amplitude

- 발음: /ˈæmpləˌtud/
- 구성: Latin `amplitudo` = largeness, breadth
- 뜻: 진폭, 크기, 넓이
- 어원적 연결: 움직임이나 파동의 크기
- 사용 맥락: `wave amplitude` = 파동의 진폭

## 기억 포인트

`ampl` 계열은 "크고 넓음, 더 크게 확장" 이미지.

- `amplify`: 더 크게 만들다
- `amplification`: 증폭, 확대
- `amplifier`: 증폭기
- `ample`: 충분하고 넉넉한
- `amplitude`: 크기, 진폭

## 비교 단어

### mag-magn

- 뜻: 크다, 크게 만들다
- 구분 포인트: [[mag-magn]]은 크고 위대한 이미지. `ampl`은 넓고 충분하며, 신호·소리·설명을 확장하는 느낌이 강함.

### expand

- 뜻: 펼쳐 넓히다
- 구분 포인트: `expand`는 물리적·범위적 확장. `amplify`는 소리·신호·효과·설명을 더 크게 만드는 공식적 표현.
