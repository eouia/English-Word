# trib

- 출발 단어: `tribe`
- 핵심 어원: 라틴어 `tribus` = 부족, 로마의 행정 구역 / `tribuere` = 나누어 주다, 배정하다
- 핵심 이미지: 사람들을 갈래로 나누고, 각 몫을 배정함
- 관련해서 헷갈릴 수 있는 어근: [[part]] = 부분, 몫 계열, `triple`, `tri-` = 셋 계열과 구분 필요

## 출발 단어 분석

`tribe`는 라틴어 `tribus`에서 옴.

- `tribus`: 부족, 씨족, 로마의 행정 구역
- 원래 로마 시민을 나눈 집단 단위와 관련

따라서 `tribe`는 "나누어진 사람들의 집단" → "부족, 집단"의 뜻.

`contribute`, `distribute`, `attribute`는 라틴어 `tribuere` = "나누어 주다, 배정하다"에서 온 단어들.

## 단어들

### tribe

- 발음: /ˈtraɪb/
- 구성: Latin `tribus`
- 뜻: 부족, 집단
- 어원적 연결: 사람들을 나눈 사회적 집단
- 사용 맥락: `a nomadic tribe` = 유목 부족

### tribal

- 발음: /ˈtraɪbəl/
- 구성: `tribe` + `-al`
- 뜻: 부족의, 집단적인
- 어원적 연결: 한 부족이나 집단에 속함
- 사용 맥락: `tribal customs` = 부족 관습

### tribute

- 발음: /ˈtrɪbjut/
- 구성: Latin `tributum`
- 뜻: 조공, 헌사, 찬사
- 어원적 연결: 각 집단에 배정된 몫이나 세금. 이후 존경을 담아 바치는 말이나 행위
- 사용 맥락: `pay tribute to a hero` = 영웅에게 경의를 표하다

### contribute

- 발음: /kəˈntrɪbjut/
- 구성: `con-` = together + `tribute`
- 뜻: 기여하다, 기부하다
- 어원적 연결: 함께 각자의 몫을 내놓음
- 사용 맥락: `contribute to the project` = 프로젝트에 기여하다

### contribution

- 발음: /ˌkɑntrəˈbjuʃən/
- 구성: `contribute` + `-ion`
- 뜻: 기여, 기부금
- 어원적 연결: 함께 내놓은 몫
- 사용 맥락: `make a contribution` = 기여하다, 기부하다

### contributor

- 발음: /kəˈntrɪbjətɚ/
- 구성: `contribute` + `-or`
- 뜻: 기여자, 기부자
- 어원적 연결: 자기 몫을 내놓는 사람
- 사용 맥락: `a major contributor` = 주요 기여자

### distribute

- 발음: /dɪˈstrɪbjut/
- 구성: `dis-` = apart + `tribute`
- 뜻: 분배하다, 배포하다
- 어원적 연결: 여러 곳으로 몫을 나누어 줌
- 사용 맥락: `distribute food` = 음식을 배급하다

### distribution

- 발음: /ˌdɪstrəˈbjuʃən/
- 구성: `distribute` + `-ion`
- 뜻: 분배, 유통, 분포
- 어원적 연결: 여러 곳에 나뉘어 퍼진 상태
- 사용 맥락: `income distribution` = 소득 분배

### attribute

- 발음: n. /ˈætrəˌbjut/, v. /əˈtrɪˌbjut/
- 구성: `ad-` = to + `tribute`
- 뜻: ~의 탓으로 돌리다, 속성
- 어원적 연결: 원인이나 성질을 어떤 대상에게 배정함
- 사용 맥락: `attribute success to hard work` = 성공을 노력 덕분으로 돌리다

### attribution

- 발음: /ˌætrɪˈbjuʃən/
- 구성: `attribute` + `-ion`
- 뜻: 귀속, 원인 설명, 출처 표시
- 어원적 연결: 어떤 말, 원인, 성질을 특정 대상에게 배정함
- 사용 맥락: `proper attribution` = 적절한 출처 표시

### retribution

- 발음: /ˌrɛtrəˈbjuʃən/
- 구성: `re-` = back + `tribution`
- 뜻: 응보, 보복, 징벌
- 어원적 연결: 행위에 맞는 대가를 되돌려 배정함
- 사용 맥락: `divine retribution` = 신의 응징

### tribunal

- 발음: /trəˈbjunəl/
- 구성: Latin `tribunal`
- 뜻: 재판소, 법정
- 어원적 연결: 로마의 `tribune`이 앉던 자리에서 나온 말. 부족/민중 단위의 행정·재판 이미지와 연결
- 사용 맥락: `an international tribunal` = 국제 재판소

### tribune

- 발음: /ˈtrɪbjun/
- 구성: Latin `tribunus`
- 뜻: 호민관, 대변자
- 어원적 연결: 로마의 `tribe` 또는 민중을 대표하는 관리
- 사용 맥락: `tribune of the people` = 민중의 대변자

## 기억 포인트

`trib` 계열은 "나누어진 집단 / 몫을 배정함" 이미지.

- `tribe`: 나누어진 사람들의 집단
- `tribal`: 부족의
- `tribute`: 배정된 몫, 바치는 것
- `contribute`: 함께 몫을 냄
- `distribute`: 몫을 나누어 줌
- `attribute`: 원인이나 성질을 어떤 대상에게 배정함
- `retribution`: 대가를 되돌려 배정함
- `tribunal`: 대표자/관리의 재판 자리

## 비교 단어

### tri-

- 뜻: 셋
- 구분 포인트: `tribe`가 고대 로마의 세 부족과 연결되었다는 설명도 있지만, 현대 영어 학습에서는 `tri-` = 셋과 `trib` = 부족/배정 계열을 구분해 두는 편이 좋음

### trial

- 뜻: 재판, 시도
- 구분 포인트: `trial`은 직접 `trib` 계열로 묶기보다 Old French `trier` = 고르다, 시험하다 계열로 따로 보는 편이 안전함

### part

- 뜻: 부분, 몫
- 구분 포인트: [[part]]는 나뉜 몫 자체. `trib`는 그 몫을 나누어 배정하는 행위에 초점
