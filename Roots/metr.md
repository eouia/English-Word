# metr

- 출발 단어: `measure`
- 핵심 어원: Latin `metiri` / `mensura` = measure, Greek `metron` = measure
- 핵심 이미지: 기준으로 재고, 정도를 정함
- 관련 어근: [[estim]], [[equ]], [[mod]]

## 출발 단어 분석

`measure`는 Old French `mesure`를 거쳐 Latin `mensura` = measuring, measurement에서 온 말.

더 깊게는 Latin `metiri` = measure와 연결됨.

`meter`, `metric`, `geometry`의 `-metry`는 Greek `metron` = measure 계열. 라틴계와 그리스계가 형태는 다르지만 시험 어휘에서는 "재다, 척도" 이미지로 함께 묶어도 좋음.

## 단어들

### measure

- 구성: Latin `mensura` = measurement
- 의미: 재다, 측정하다, 조치, 기준
- 어원적 연결: 어떤 기준으로 크기나 정도를 잼
- 사용 맥락: `measure the distance` = 거리를 재다

### measurement

- 구성: `measure` + `-ment`
- 의미: 측정, 측정값
- 어원적 연결: 재서 얻은 값
- 사용 맥락: `accurate measurement` = 정확한 측정

### measurable

- 구성: `measure` + `-able`
- 의미: 측정 가능한
- 어원적 연결: 기준을 대어 잴 수 있음
- 사용 맥락: `measurable progress` = 측정 가능한 진전

### immeasurable

- 구성: [[in]] = not + `measurable`
- 의미: 헤아릴 수 없는, 막대한
- 어원적 연결: 너무 커서 기준으로 잴 수 없음
- 사용 맥락: `immeasurable value` = 헤아릴 수 없는 가치

### meter

- 구성: Greek `metron` = measure
- 의미: 미터, 계량기
- 어원적 연결: 길이나 양을 재는 기준 또는 장치
- 사용 맥락: `a parking meter` = 주차 계량기

### metric

- 구성: Greek `metron` + `-ic`
- 의미: 미터법의, 측정 기준의
- 어원적 연결: 측정 체계나 기준과 관련됨
- 사용 맥락: `the metric system` = 미터법

### thermometer

- 구성: Greek `thermo-` = heat + `meter`
- 의미: 온도계
- 어원적 연결: 열의 정도를 재는 기구
- 사용 맥락: `use a thermometer` = 온도계를 쓰다

### diameter

- 구성: Greek [[dia]] = across + `meter`
- 의미: 지름
- 어원적 연결: 원을 가로질러 잰 길이
- 사용 맥락: `the diameter of a circle` = 원의 지름

### perimeter

- 구성: Greek `peri-` = around + `meter`
- 의미: 둘레, 주위 경계
- 어원적 연결: 도형의 주변을 돌아가며 잰 길이
- 사용 맥락: `the perimeter of a rectangle` = 직사각형의 둘레

### geometry

- 구성: Greek `geo-` = earth + `-metry` = measuring
- 의미: 기하학
- 어원적 연결: 원래는 땅을 재는 학문
- 사용 맥락: `study geometry` = 기하학을 공부하다

### symmetry

- 구성: [[sym]] = together + `metry`
- 의미: 대칭, 균형
- 어원적 연결: 함께 맞게 재어진 비례
- 사용 맥락: `facial symmetry` = 얼굴의 대칭

## 기억 포인트

- `measure`: 재다, 기준
- `measurement`: 재서 얻은 값
- `meter`: 재는 단위나 장치
- `metric`: 측정 기준
- `perimeter`: 둘레를 돌아가며 잰 길이
- `geometry`: 땅을 재는 학문
- `symmetry`: 함께 맞게 잰 균형

## 비교 단어

- [[estim]]
  - 핵심: 가치를 매기고 어림함
  - 구분 포인트: `estimate`는 대략 판단. `measure`는 기준을 대고 잼

- [[equ]]
  - 핵심: 같음, 균형
  - 구분 포인트: `equ`는 같게 맞춤. `metr`은 맞는지 재는 기준

- [[mod]]
  - 핵심: 알맞은 방식, 정도
  - 구분 포인트: `moderate`는 정도를 적절히 조절함. `measure`는 정도를 잼
