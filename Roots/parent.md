# parent

- 출발 단어: `parent`
- 핵심 의미: 낳는 사람, 근원이 되는 존재
- 관련 형태: Latin `parere` = bring forth, give birth to, produce
- 관련 문서: [[nat]], [[gen]], [[para]]

## 핵심 이미지

`parent` 계열의 바탕은 "낳아 생기게 함".

아이를 낳거나 기르는 사람은 `parent`.
부모와 관련되면 `parental`.
부모가 된 상태는 `parenthood`.
넓게는 어떤 것의 근원이나 모체도 `parent`.

## 단어들

### parent

- 발음: /ˈpɛrənt/
- 구성: Latin `parens`, from `parere` = give birth to
- 뜻: 부모; 근원이 되는 것, 모체
- 연결: 아이나 결과를 생기게 한 존재
- 사용 맥락: `parent company` = 모회사, `single parent` = 한부모

### parental

- 발음: /pɚˈɛntəl/
- 구성: `parent` + `-al`
- 뜻: 부모의, 부모다운
- 연결: 부모와 관련된 성질이나 책임
- 사용 맥락: `parental responsibility` = 부모의 책임

### parenthood

- 발음: /ˈpɛrəˌnthʊd/
- 구성: `parent` + `-hood`
- 뜻: 부모 됨, 부모 역할
- 연결: 아이를 낳거나 길러 부모가 된 상태
- 사용 맥락: `the challenges of parenthood` = 부모 역할의 어려움

### parenting

- 발음: /ˈpɛrəntɪŋ/
- 구성: `parent` + `-ing`
- 뜻: 양육, 부모 노릇
- 연결: 아이를 기르고 돌보는 실제 행위
- 사용 맥락: `parenting skills` = 양육 기술

### parentage

- 발음: /ˈpɛrəntədʒ/
- 구성: `parent` + `-age`
- 뜻: 혈통, 출신, 부모 관계
- 연결: 누구에게서 태어났는지의 근원
- 사용 맥락: `of unknown parentage` = 부모를 알 수 없는

### grandparent

- 발음: /ˈgræˌndpɛrənt/
- 구성: `grand-` + `parent`
- 뜻: 조부모
- 연결: 부모의 부모
- 사용 맥락: `visit my grandparents` = 조부모를 방문하다

### stepparent

- 구성: `step-` + `parent`
- 뜻: 의붓부모
- 연결: 혈연 부모는 아니지만 부모 역할을 맡은 사람
- 사용 맥락: family vocabulary에서 자주 사용.

## 비교 단어

`parenthesis`는 [[thes]]와 [[para]] 계열.
`para-` = beside + `thesis` = placing에서 온 말로, `parent` = 낳는 사람과 어원이 다름.

`apparent`, `transparent`는 Latin `parere` = appear, be visible 계열로 설명되는 별도 갈래.
철자는 비슷하지만 `parent`의 "낳다" 계열과 직접 섞지 않는 편이 좋음.

`nat`은 [[nat]] 문서처럼 태어남 자체, `gen`은 [[gen]] 문서처럼 낳음·종류·혈통이 중심.
`parent`는 그중에서도 "낳거나 기르는 존재"가 중심.
