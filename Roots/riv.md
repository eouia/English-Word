# riv

- 출발 단어: `derived`
- 핵심 어원: Latin `rivus` = stream, brook, watercourse
- 핵심 이미지: 물길에서 흘러나오거나 갈라져 나옴
- 관련 문서: [[flu]], [[duct]], [[ori]]
- 주의: `rive` = 찢다, 쪼개다는 별도 Germanic 계열. `derive`의 `riv`는 물길 `rivus` 계열.

## 출발 단어 분석

`derived` ← Latin `derivare` = draw off water, lead away, derive

- `de-`: away, down, from
- `rivus`: stream, watercourse
- `-ed`: 과거분사·형용사 접미사
- 원래 이미지: 본래 물길에서 물을 다른 쪽으로 끌어냄
- 의미 변화: 물길을 돌림 → 어떤 근원에서 끌어냄 → 유래하다, 파생되다
- 현대 의미: ~에서 나온, 유래한, 파생된

## 단어들

### derived

- 구성: `derive` + `-ed`
- 뜻: 유래한, 파생된
- 어원적 연결: 한 근원에서 흘러나와 생긴 상태
- 사용 맥락: `a derived form` = 파생 형태

### derive

- 구성: Latin `de-` = from/away + `rivus` = stream
- 뜻: 얻다, 끌어내다; 유래하다
- 어원적 연결: 원래 물길에서 물을 끌어내듯 근원에서 의미나 결과를 끌어냄
- 사용 맥락: `derive meaning from context` = 문맥에서 의미를 끌어내다

### derivation

- 구성: `derive` + `-ation`
- 뜻: 유래, 파생, 어원적 도출
- 어원적 연결: 한 근원에서 갈라져 나온 과정
- 사용 맥락: `the derivation of a word` = 단어의 유래

### derivative

- 구성: `derive` + `-ative`
- 뜻: 파생된 것; 파생적인; 미분계수
- 어원적 연결: 본래 근원에서 갈라져 나온 것
- 사용 맥락: `a derivative work` = 파생 작품

### river

- 구성: Latin `riparia` = riverbank, shore; 관련 배경은 물길 이미지
- 뜻: 강
- 어원적 연결: 물이 흐르는 길
- 사용 맥락: `cross a river` = 강을 건너다

### rivulet

- 구성: Latin `rivulus` = small stream
- 뜻: 작은 시내, 개울
- 어원적 연결: 작은 물길
- 사용 맥락: `a mountain rivulet` = 산속 개울

### rival

- 구성: Latin `rivalis` = one using the same stream
- 뜻: 경쟁자
- 어원적 연결: 같은 물길을 함께 쓰는 사람 → 같은 자원을 두고 다투는 사람
- 사용 맥락: `a political rival` = 정치적 경쟁자

### rivalry

- 구성: `rival` + `-ry`
- 뜻: 경쟁, rival 관계
- 어원적 연결: 같은 자원이나 목표를 두고 맞서는 관계
- 사용 맥락: `a fierce rivalry` = 치열한 경쟁 관계

## 기억 포인트

`riv` 계열은 "물길에서 흘러나옴" 이미지.

- `derive`: 근원에서 끌어냄
- `derived`: 근원에서 나온
- `derivation`: 유래·파생 과정
- `derivative`: 파생된 것
- `river`: 물길
- `rival`: 같은 물길을 두고 다투는 사람

## 비교 단어

### flu

- 뜻: 흐르다
- 구분 포인트: [[flu]]는 흐름 자체. `riv`는 물길·수로, 그리고 그 물길에서 갈라져 나온다는 이미지.

### duct

- 뜻: 이끌다, 데려가다
- 구분 포인트: [[duct]]는 이끄는 행위. `derive`는 물길을 돌려 어떤 근원에서 끌어내는 느낌.

### rive

- 뜻: 찢다, 쪼개다
- 구분 포인트: `rive`, `riven`은 Germanic 계열. `derive`, `rival`, `rivulet`의 `riv`와 어원이 다름.
