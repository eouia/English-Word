# hes

- 출발 단어: `cohesion`
- 핵심 어원: 라틴어 `haerere` / `haes-` = 붙다, 달라붙다, 머물다
- 핵심 이미지: 떨어지지 않고 붙어 있음
- 관련해서 헷갈릴 수 있는 어근: `hold` 계열은 게르만계로 별도

## 출발 단어 분석

`cohesion`은 `co-` + `hes` + `-ion`.

- `co-`: together
- `hes`: 붙다
- `-ion`: 행위, 상태

따라서 `cohesion`은 "함께 붙어 있는 상태" → "응집력, 결속"의 뜻.

## 단어들

### cohesion

- 발음: /koʊˈhiʒən/
- 구성: `co-` = together + `hes` + `-ion`
- 뜻: 응집, 결속, 일관성
- 어원적 연결: 여러 부분이 서로 붙어 하나처럼 유지됨
- 사용 맥락: `social cohesion` = 사회적 결속

### cohesive

- 발음: /koʊˈhisɪv/
- 구성: `cohesion` 계열 + `-ive`
- 뜻: 응집력 있는, 결속된, 일관된
- 어원적 연결: 여러 요소가 잘 붙어 하나로 이어짐
- 사용 맥락: `a cohesive team` = 결속력 있는 팀

### cohere

- 발음: /koʊˈhɪr/
- 구성: `co-` = together + `haerere`
- 뜻: 결합하다, 앞뒤가 맞다
- 어원적 연결: 부분들이 서로 붙어 논리나 구조가 유지됨
- 사용 맥락: `The argument does not cohere.` = 그 주장은 앞뒤가 맞지 않음

### coherent

- 발음: /koʊˈhɪrənt/
- 구성: `cohere` + `-ent`
- 뜻: 일관성 있는, 조리 있는
- 어원적 연결: 생각이나 말의 부분들이 서로 잘 붙어 있음
- 사용 맥락: `a coherent explanation` = 조리 있는 설명

### coherence

- 발음: /koʊˈhɪrəns/
- 구성: `coherent` + `-ence`
- 뜻: 일관성, 응집성
- 어원적 연결: 부분들이 서로 붙어 하나의 흐름을 이룸
- 사용 맥락: `logical coherence` = 논리적 일관성

### adhere

- 발음: /əˈdhɪr/
- 구성: `ad-` = to + `haerere`
- 뜻: 달라붙다, 고수하다
- 어원적 연결: 어떤 표면이나 원칙에 붙어 있음
- 사용 맥락: `adhere to the rules` = 규칙을 고수하다

### adhesion

- 발음: /æˈdhiʒən/
- 구성: `adhere` + `-ion`
- 뜻: 접착, 부착
- 어원적 연결: 한 대상이 다른 대상에 달라붙음
- 사용 맥락: `cell adhesion` = 세포 부착

### adhesive

- 발음: /æˈdhisɪv/
- 구성: `adhere` 계열 + `-ive`
- 뜻: 접착성의, 접착제
- 어원적 연결: 붙게 하는 성질을 가짐
- 사용 맥락: `adhesive tape` = 접착테이프

### inherent

- 발음: /ɪˈnhɪrənt/
- 구성: `in-` = in + `haerere`
- 뜻: 내재하는, 본래 갖추어진
- 어원적 연결: 어떤 것 안에 붙어 떨어지지 않는 성질
- 사용 맥락: `inherent risks` = 내재적 위험

### inhere

- 구성: `in-` = in + `haerere`
- 뜻: 내재하다
- 어원적 연결: 성질이 대상 안에 붙어 있음
- 사용 맥락: `rights that inhere in all people` = 모든 사람에게 내재하는 권리

### hesitate

- 발음: /ˈhɛzəˌteɪt/
- 구성: Latin `haesitare`
- 뜻: 망설이다, 주저하다
- 어원적 연결: 원래 "걸리다, 붙들리다"의 반복 느낌. 마음이나 행동이 앞으로 나아가지 못하고 걸려 있음
- 사용 맥락: `hesitate to answer` = 대답하기를 망설이다

### hesitation

- 발음: /ˌhɛzəˈteɪʃən/
- 구성: `hesitate` + `-ion`
- 뜻: 망설임, 주저
- 어원적 연결: 마음이나 행동이 붙들려 멈칫하는 상태
- 사용 맥락: `without hesitation` = 망설임 없이

## 기억 포인트

`hes` 계열은 "붙다 / 붙들리다" 이미지.

- `cohesion`: 함께 붙어 있음
- `cohesive`: 잘 붙어 하나로 이어짐
- `coherent`: 생각이 서로 붙어 일관됨
- `adhere`: 대상에 달라붙음, 원칙을 고수함
- `adhesive`: 붙게 하는 것
- `inherent`: 안에 붙어 떨어지지 않는
- `hesitate`: 마음이 붙들려 멈칫함

## 비교 단어

### unity

- 구성: Latin `unus`
- 뜻: 하나 됨, 통일
- 구분 포인트: [[uni]]의 `unity`는 "하나" 이미지. `cohesion`은 여러 부분이 "서로 붙어 있음" 이미지

### consistent

- 뜻: 일관된
- 구분 포인트: `coherent`와 뜻이 비슷하지만, [[sist]] 계열. "함께 서 있음"에서 일관성의 뜻
