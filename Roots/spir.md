# spir

- 출발 단어: `spirit`
- 핵심 어원: Latin `spirare` = 숨쉬다
- 명사형: Latin `spiritus` = 숨, 생기, 정신
- 핵심 이미지: 숨이 들어와 생명과 정신이 움직임
- 관련 문서: [[anim]], [[pneu]], [[psych]], [[hal]], [[flat]], [[enth]], [[fid]], [[voc]]
- 주의: `spiral`은 Greek `speira` = coil 계열. `spirare`와 분리.

## 출발 단어 분석

`spirit` ← Latin `spiritus`

- `spirare`: 숨쉬다
- `spiritus`: 숨, 생기, 정신
- 원래 이미지: 살아 있음을 드러내는 숨
- 현대 의미: 정신, 영혼, 태도, 기운

## 단어들

### spirit

- 발음: /ˈspɪrət/
- 구성: Latin `spiritus`
- 뜻: 정신, 영혼, 기운
- 어원적 연결: 숨에서 생명과 정신의 이미지로 확장
- 사용 맥락: `team spirit` = 팀 정신

### spiritual

- 발음: /ˈspɪrɪˌtʃuəl/
- 구성: `spirit` + `-ual`
- 뜻: 정신적인, 영적인
- 어원적 연결: 물질보다 숨·정신의 영역에 속함
- 사용 맥락: `spiritual growth` = 정신적 성장

### inspire

- 발음: /ˌɪˈnspaɪr/
- 구성: `in-` = into + `spirare` = breathe
- 뜻: 영감을 주다, 고무하다
- 어원적 연결: 숨이나 정신을 안으로 불어넣음
- 사용 맥락: `inspire confidence` = 자신감을 불어넣다

### inspiration

- 발음: /ˌɪnspɚˈeɪʃən/
- 구성: `inspire` + `-ation`
- 뜻: 영감, 고무
- 어원적 연결: 안으로 불어넣어진 생기와 생각
- 사용 맥락: `a source of inspiration` = 영감의 원천

### aspire

- 발음: /əˈspaɪr/
- 구성: `ad-` = toward + `spirare`
- 뜻: 열망하다, 지향하다
- 어원적 연결: 어떤 목표를 향해 숨을 내쉬듯 마음을 뻗음
- 사용 맥락: `aspire to leadership` = 지도자가 되기를 열망하다

### aspiration

- 발음: /ˌæspɚˈeɪʃən/
- 구성: `aspire` + `-ation`
- 뜻: 열망, 포부
- 어원적 연결: 목표를 향해 뻗는 마음의 숨
- 사용 맥락: `career aspirations` = 직업적 포부

### expire

- 발음: /ɪˈkspaɪr/
- 구성: `ex-` = out + `spirare`
- 뜻: 만료되다, 숨을 거두다
- 어원적 연결: 숨이 밖으로 나가 끝남
- 사용 맥락: `the contract expires` = 계약이 만료된다

### expiration

- 발음: /ˌɛkspɚˈeɪʃən/
- 구성: `expire` + `-ation`
- 뜻: 만료, 숨을 내쉼
- 어원적 연결: 밖으로 숨이 나가 끝나는 과정
- 사용 맥락: `expiration date` = 만료일

### respire

- 구성: `re-` = again/back + `spirare`
- 뜻: 호흡하다
- 어원적 연결: 숨을 되풀이해 쉼
- 사용 맥락: `cells respire` = 세포가 호흡하다

### respiration

- 발음: /ˌrɛspɚˈeɪʃən/
- 구성: `respire` + `-ation`
- 뜻: 호흡
- 어원적 연결: 숨을 들이쉬고 내쉬는 과정
- 사용 맥락: `cellular respiration` = 세포 호흡

### conspire

- 발음: /kəˈnspaɪɚ/
- 구성: `con-` = together + `spirare`
- 뜻: 공모하다
- 어원적 연결: 함께 숨을 맞추듯 은밀히 계획함
- 사용 맥락: `conspire against the king` = 왕에 맞서 공모하다

### conspiracy

- 발음: /kəˈnspɪrəsi/
- 구성: `conspire` + `-acy`
- 뜻: 음모, 공모
- 어원적 연결: 함께 숨을 맞춘 은밀한 계획
- 사용 맥락: `a conspiracy theory` = 음모론

### perspire

- 발음: /pɚˈspaɪr/
- 구성: `per-` = through + `spirare`
- 뜻: 땀을 흘리다
- 어원적 연결: 피부를 통해 숨·수분이 배어 나옴
- 사용 맥락: `perspire heavily` = 땀을 많이 흘리다

## 기억 포인트

`spir` 계열은 "숨" 이미지.

- `spirit`: 숨 → 생기와 정신
- `inspire`: 안으로 숨을 불어넣음
- `aspire`: 목표를 향해 마음을 뻗음
- `expire`: 숨이 밖으로 나가 끝남
- `conspire`: 함께 숨을 맞춤
