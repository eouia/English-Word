# sphere

- 출발 단어: `sphere`
- 핵심 어원: Greek `sphaira` = ball, globe
- 핵심 이미지: 둥근 공, 구, 둘러싼 영역
- 관련 문서: [[geo]], [[astr]], [[terr]]
- 주의: `sphere`는 물리적 "구"에서 출발하지만, 현대 영어에서는 "활동 영역, 분야"라는 추상 의미도 중요함.

## 출발 단어 분석

`sphere`는 현대 형태에서 더 작게 의미 있게 분해하기 어려움.

- Greek `sphaira`: ball, globe
- Latin `sphaera`: sphere, celestial globe
- Old French `sphere`: 천구, 구
- 현대 의미: 구, 구체; 영역, 분야

## 단어들

### sphere

- 발음: /ˈsfɪr/
- 구성: Greek `sphaira` = ball, globe
- 뜻: 구, 구체; 영역, 분야
- 어원적 연결: 둥근 공간이나 어떤 영향력이 미치는 범위
- 사용 맥락: `the public sphere` = 공적 영역

### spherical

- 발음: /ˈsfɛrɪkəl/
- 구성: `sphere` + `-ical`
- 뜻: 구형의, 둥근
- 어원적 연결: 공처럼 둥근 형태
- 사용 맥락: `a spherical object` = 구형 물체

### hemisphere

- 발음: /ˈhɛmɪˌsfɪr/
- 구성: Greek `hemi-` = half + `sphere`
- 뜻: 반구; 지구의 반쪽 영역
- 어원적 연결: 구를 반으로 나눈 부분
- 사용 맥락: `the Northern Hemisphere` = 북반구

### atmosphere

- 발음: /ˈætməˌsfɪr/
- 구성: Greek `atmos` = vapor, steam + `sphere`
- 뜻: 대기; 분위기
- 어원적 연결: 지구를 둘러싼 증기·공기의 둥근 층
- 사용 맥락: `Earth's atmosphere` = 지구 대기
- 주의: 추상적으로는 장소나 상황의 "분위기"도 뜻함.

### atmospheric

- 발음: /ˌætməˈsfɛrɪk/
- 구성: `atmosphere` + `-ic`
- 뜻: 대기의; 분위기 있는
- 어원적 연결: 공기층이나 분위기에 속함
- 사용 맥락: `atmospheric pressure` = 대기압

### biosphere

- 발음: /ˈbaɪoʊˌsfɪr/
- 구성: Greek `bio-` = life + `sphere`
- 뜻: 생물권
- 어원적 연결: 생명체가 살아가는 지구의 영역
- 사용 맥락: `protect the biosphere` = 생물권을 보호하다

### lithosphere

- 구성: Greek `lithos` = stone + `sphere`
- 뜻: 암석권, 지각과 상부 맨틀의 단단한 층
- 어원적 연결: 지구의 돌·암석으로 된 둥근 외층
- 사용 맥락: `lithospheric plates` = 암석권 판

### hydrosphere

- 구성: Greek `hydro-` = water + `sphere`
- 뜻: 수권, 지구의 물 전체
- 어원적 연결: 지구에서 물이 차지하는 영역
- 사용 맥락: `the hydrosphere includes oceans and rivers`

### stratosphere

- 발음: /ˈstrætəˌsfɪr/
- 구성: Latin `stratum` = layer + `sphere`
- 뜻: 성층권
- 어원적 연결: 층으로 나뉜 대기권의 한 부분
- 사용 맥락: `ozone in the stratosphere` = 성층권의 오존

### troposphere

- 구성: Greek `tropos` = turn, change + `sphere`
- 뜻: 대류권
- 어원적 연결: 날씨 변화와 대류가 일어나는 대기층
- 사용 맥락: `weather occurs in the troposphere`

### exosphere

- 구성: Greek `exo-` = outside + `sphere`
- 뜻: 외기권, 대기의 가장 바깥층
- 어원적 연결: 지구 대기권의 바깥쪽 영역
- 사용 맥락: `the outer exosphere`

### geosphere

- 구성: Greek `geo-` = earth + `sphere`
- 뜻: 지권, 지구의 고체 부분
- 어원적 연결: 지구를 이루는 땅과 암석의 영역
- 사용 맥락: `interactions between the atmosphere and geosphere`
- 관련 문서: [[geo]]

### ecosphere

- 구성: `eco-` = environment, habitat + `sphere`
- 뜻: 생태권, 생태계 전체 영역
- 어원적 연결: 생명체와 환경이 함께 작동하는 영역
- 사용 맥락: `the global ecosphere`

## 기억 포인트

`sphere` 계열은 "둥근 공간과 영향 범위" 이미지.

- `sphere`: 구, 영역
- `hemisphere`: 반구
- `atmosphere`: 지구를 둘러싼 공기층
- `biosphere`: 생명이 사는 영역
- `stratosphere`: 층을 이룬 대기권
- `public sphere`: 사회적 논의가 이루어지는 영역

## 비교 단어

- `circle`: 평면의 원. `sphere`는 3차원의 구.
- `globe`: 지구본이나 둥근 물체. `sphere`보다 구체적 사물 느낌이 강함.
- `orbit`: 천체가 도는 길. `sphere`는 공간이나 영역 자체.
