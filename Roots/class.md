# class

- 출발 단어: `class`
- 핵심 어원: 라틴어 `classis` = 부류, 구분, 군대, 함대
- 핵심 의미: 기준에 따라 나눈 집단, 등급
- 관련 문서: [[group]], [[part]], [[ord]]
- 주의: 더 깊게는 “소집하다, 부르다” 계열과 연결될 가능성이 큼. 영어 학습에서는 “분류된 집단/등급”으로 기억.

## 출발 단어 분석

`class` ← French `classe` ← Latin `classis`

- 원래 이미지: 사람들을 기준에 따라 나누어 소집한 집단
- 의미 확장: 부류, 등급 → 학교의 반 → 사회 계급 → 분류 단위
- 현대 핵심: 같은 기준으로 묶인 등급 또는 집단

## 단어들

### class

- 발음: /ˈklæs/
- 구성: Latin `classis`
- 뜻: 반, 수업, 계급, 부류
- 어원적 연결: 기준에 따라 나뉜 집단
- 사용 맥락: `take a class` = 수업을 듣다, `social class` = 사회 계급

### classic

- 발음: /ˈklæsɪk/
- 구성: Latin `classicus` = 최고 계급의
- 뜻: 고전의, 전형적인, 명작
- 어원적 연결: 가장 높은 등급에 속한다고 인정된 것
- 사용 맥락: `a classic novel` = 고전 소설

### classical

- 발음: /ˈklæsɪkəl/
- 구성: `classic` + `-al`
- 뜻: 고전적인, 고전주의의
- 어원적 연결: 오래 인정된 높은 등급의 전통과 관련
- 사용 맥락: `classical music` = 클래식 음악

### classify

- 발음: /ˈklæsəˌfaɪ/
- 구성: `class` + `-ify`
- 뜻: 분류하다
- 어원적 연결: 대상을 특정 부류에 넣음
- 사용 맥락: `classify animals` = 동물을 분류하다

### classification

- 발음: /ˌklæsəfəˈkeɪʃən/
- 구성: `classify` + `-ation`
- 뜻: 분류, 등급
- 어원적 연결: 기준에 따라 부류를 나눈 결과
- 사용 맥락: `scientific classification` = 과학적 분류

### classified

- 발음: /ˈklæsəˌfaɪd/
- 구성: `classify` + `-ed`
- 뜻: 분류된, 기밀의
- 어원적 연결: 특정 등급으로 나뉜 상태. 정보에서는 접근 등급이 매겨진 상태
- 사용 맥락: `classified information` = 기밀 정보

### declassify

- 발음: /dɪˈklæsəˌfaɪ/
- 구성: `de-` = remove + `classify`
- 뜻: 기밀 해제하다
- 어원적 연결: 정보에 붙은 비밀 등급을 제거함
- 사용 맥락: `declassify documents` = 문서를 기밀 해제하다

### classmate

- 발음: /ˈklæˌsmeɪt/
- 구성: `class` + `mate`
- 뜻: 반 친구, 동급생
- 어원적 연결: 같은 반 또는 같은 수업 집단에 속한 사람
- 사용 맥락: `a former classmate` = 옛 동급생

### classroom

- 발음: /ˈklæˌsrum/
- 구성: `class` + `room`
- 뜻: 교실
- 어원적 연결: 한 반이 모여 수업하는 공간
- 사용 맥락: `classroom discussion` = 교실 토론

### classy

- 발음: /ˈklæsi/
- 구성: `class` + `-y`
- 뜻: 세련된, 고급스러운
- 어원적 연결: 높은 등급에 속한 듯한 느낌
- 사용 맥락: `a classy restaurant` = 고급스러운 식당

### middle-class

- 발음: /ˈmɪdəˈlklæs/
- 구성: `middle` + `class`
- 뜻: 중산층의
- 어원적 연결: 사회적 등급 가운데에 위치한 집단
- 사용 맥락: `middle-class families` = 중산층 가정

## 기억 포인트

- `class`의 핵심은 “기준으로 나눈 집단/등급”.
- `classic`은 “최고 등급으로 인정된 것”에서 고전, 명작.
- `classify`는 “class로 만들다” → 분류하다.
- `classified information`은 “등급이 매겨진 정보” → 기밀 정보.

## 비교 단어

### group

- [[group]]은 한데 묶인 무리, 덩어리.
- `class`는 기준에 따라 나뉜 부류/등급 느낌이 더 강함.

### category

- `category`는 그리스어 `kategoria` = 진술, 범주 계열.
- 분류 의미는 비슷하지만 `classis` 계열은 아님.
