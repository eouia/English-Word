# ment

- 출발 단어: `mental`
- 핵심 어원: Latin `mens` / `mentis` = 마음, 정신
- 관련 뿌리: PIE `men-` = think
- 핵심 이미지: 마음으로 생각하고 기억함
- 관련 문서: [[psych]], [[mem]], [[sent]]
- 주의: 영어 접미사 `-ment`는 별도 형태. `government`, `movement`의 `-ment`와 이 문서의 `ment`를 구분.

## 출발 단어 분석

`mental` ← Latin `mens`

- `mens`: mind
- `mentis`: mind의 속격형
- 원래 이미지: 생각하고 판단하는 마음
- 현대 의미: 정신의, 마음의

## 단어들

### mental

- 구성: Latin `mens` / `mentis` + `-al`
- 뜻: 정신의, 마음의
- 어원적 연결: 마음과 생각에 관련됨
- 사용 맥락: `mental health` = 정신 건강

### mentality

- 구성: `mental` + `-ity`
- 뜻: 사고방식, 정신 상태
- 어원적 연결: 마음이 작동하는 방식
- 사용 맥락: `a growth mentality` = 성장형 사고방식

### dementia

- 구성: `de-` = away + Latin `mens` = mind
- 뜻: 치매
- 어원적 연결: 마음이나 정신 기능이 벗어나 약해진 상태
- 사용 맥락: `patients with dementia` = 치매 환자

### demented

- 구성: `de-` + `mens` + `-ed`
- 뜻: 정신이 이상해진, 미친 듯한
- 어원적 연결: 제정신에서 벗어난 상태
- 사용 맥락: `a demented laugh` = 미친 듯한 웃음

### mention

- 구성: Latin `mentio` = a calling to mind
- 뜻: 언급하다, 언급
- 어원적 연결: 어떤 것을 마음에 떠올려 말함
- 사용 맥락: `mention a problem` = 문제를 언급하다

### comment

- 구성: Latin `commentari` = think over, explain
- 뜻: 논평하다, 의견
- 어원적 연결: 마음으로 생각해 설명을 덧붙임
- 사용 맥락: `make a comment` = 의견을 말하다

### commentary

- 구성: `comment` + `-ary`
- 뜻: 해설, 논평
- 어원적 연결: 생각한 내용을 설명으로 덧붙임
- 사용 맥락: `political commentary` = 정치 논평

## 기억 포인트

`ment` 계열은 "마음과 생각" 이미지.

- `mental`: 마음의
- `mentality`: 마음의 방식
- `dementia`: 마음 기능이 약해짐
- `mention`: 마음에 떠올려 말함
- `comment`: 생각을 덧붙임

## 비교 단어

### psych

- 뜻: 마음, 정신
- 구분 포인트: [[psych]]는 그리스어 계열. `ment`는 라틴어 `mens`에서 온 마음·생각 계열.

### mem

- 뜻: 기억
- 구분 포인트: [[mem]]은 마음에 남아 다시 떠올림. `ment`는 마음과 사고 자체.
