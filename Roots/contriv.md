# contriv

- 출발 단어: `contrived`
- 핵심 어원: Old French `controver` = find, devise
- 핵심 이미지: 찾아내어 꾸며 만들기
- 관련 문서: [[ven]], [[cover]]
- 주의: `contrived`는 현대 영어에서 "인위적인, 억지로 만든" 부정적 느낌이 강함. 단순한 `created`보다 자연스럽지 않게 꾸며낸 느낌.

## 출발 단어 분석

`contrived` = `contrive` + `-ed`

- `contrive`: devise, invent, plan
- `-ed`: 과거분사·형용사 어미
- 원래 이미지: 방법을 찾아내어 고안함
- 의미 변화: 고안하다 → 일부러 꾸며 만들다 → 억지스럽고 인위적인
- 현대 의미: 인위적인, 억지로 꾸민; 고안된

## 단어들

### contrive

- 발음: /kəˈntraɪv/
- 구성: Old French `controver` = find, devise
- 뜻: 고안하다, 꾸며내다, 어떻게든 해내다
- 어원적 연결: 방법이나 장치를 찾아내어 만듦
- 사용 맥락: `contrive a solution` = 해결책을 고안하다

### contrived

- 발음: /kəˈntraɪvd/
- 구성: `contrive` + `-ed`
- 뜻: 인위적인, 억지로 꾸민
- 어원적 연결: 자연스럽게 생긴 것이 아니라 일부러 꾸며 만들어짐
- 사용 맥락: `a contrived plot` = 억지스러운 줄거리

### contrivance

- 발음: /kəˈntraɪvəns/
- 구성: `contrive` + `-ance`
- 뜻: 고안물, 장치; 억지스러운 설정
- 어원적 연결: 생각해 내어 만든 물건이나 장치
- 사용 맥락: `a clever contrivance` = 영리한 고안물

### contriver

- 구성: `contrive` + `-er`
- 뜻: 고안자, 꾸며내는 사람
- 어원적 연결: 방법이나 장치를 찾아 만들어 내는 사람
- 사용 맥락: 드물고 문어적인 표현

## 기억 포인트

`contriv` 계열은 "찾아내어 꾸며 만들기" 이미지.

- `contrive`: 고안하다, 꾸며내다
- `contrived`: 인위적인, 억지스러운
- `contrivance`: 고안물, 장치
- `contriver`: 고안자

## 비교 단어

### invent

- 뜻: 발명하다, 지어내다
- 구분 포인트: [[ven]]의 `invent`는 원래 "찾아오다/발견하다"에서 발명으로 확장. `contrive`는 더 계획적이고 때로 억지로 꾸며낸 느낌.

### discover

- 뜻: 발견하다
- 구분 포인트: [[cover]]는 덮개를 걷어 드러냄. `contrive`는 발견보다 방법을 짜내어 만들어 냄.

### devise

- 뜻: 고안하다, 계획하다
- 구분 포인트: `devise`는 계획을 생각해 냄. `contrive`는 어려운 상황에서 억지로 방법을 만들어내거나 꾸며낸 뉘앙스가 더 강할 수 있음.
