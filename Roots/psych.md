# psych

- 출발 단어: `psychology`
- 핵심 어원: Greek `psyche` = 숨, 생명, 영혼, 마음
- 핵심 이미지: 살아 있는 숨에서 마음과 정신으로 확장
- 관련 문서: [[pneu]], [[spir]], [[anim]], [[ment]], [[path]], [[som]], [[iatr]], [[phren]], [[logos]]
- 주의: `psychology`는 [[logos]] 계열에도 걸친 합성어.

## 출발 단어 분석

`psychology` ← Greek `psyche` + `-logy`

- `psyche`: 숨, 생명, 영혼, 마음
- `-logy`: 학문, 체계적 설명
- 원래 이미지: 마음과 정신에 대한 설명
- 현대 의미: 심리학

## 단어들

### psyche

- 구성: Greek `psyche`
- 뜻: 마음, 정신, 영혼
- 어원적 연결: 숨과 생명에서 마음의 영역으로 확장
- 사용 맥락: `the human psyche` = 인간의 정신

### psychology

- 구성: `psycho-` = mind + [[logos]] = study
- 뜻: 심리학
- 어원적 연결: 마음을 체계적으로 설명하는 학문
- 사용 맥락: `clinical psychology` = 임상심리학

### psychological

- 구성: `psychology` + `-ical`
- 뜻: 심리적인
- 어원적 연결: 마음과 정신 상태에 관련됨
- 사용 맥락: `psychological pressure` = 심리적 압박

### psychiatrist

- 구성: `psyche` = mind + Greek `iatros` = doctor
- 뜻: 정신과 의사
- 어원적 연결: 마음의 병을 다루는 의사
- 사용 맥락: `see a psychiatrist` = 정신과 의사를 만나다

### psychiatry

- 구성: `psyche` + Greek `iatreia` = healing
- 뜻: 정신의학
- 어원적 연결: 마음의 병을 치료하는 의학 분야
- 사용 맥락: `modern psychiatry` = 현대 정신의학

### psychic

- 구성: `psyche` + `-ic`
- 뜻: 정신의, 초능력의
- 어원적 연결: 마음이나 정신의 힘과 관련됨
- 사용 맥락: `psychic powers` = 초능력

### psychosis

- 구성: `psych-` + `-osis` = condition
- 뜻: 정신증
- 어원적 연결: 정신 기능에 심각한 이상이 생긴 상태
- 사용 맥락: 의학·심리학 맥락에서 쓰임

### psychosomatic

- 구성: `psycho-` = mind + Greek `soma` = body
- 뜻: 심신의, 정신 신체의
- 어원적 연결: 마음의 상태가 몸의 증상과 연결됨
- 사용 맥락: `psychosomatic symptoms` = 심신성 증상

## 기억 포인트

`psych` 계열은 "숨에서 마음으로" 이어지는 이미지.

- `psyche`: 마음, 정신
- `psychology`: 마음에 대한 학문
- `psychiatrist`: 마음의 병을 치료하는 의사
- `psychosomatic`: 마음과 몸이 연결됨

## 비교 단어

### pneu

- 뜻: 숨, 바람, 공기
- 구분 포인트: [[pneu]]는 숨과 공기의 물리적 이미지가 강함. `psych`는 숨에서 마음·정신으로 확장된 이미지.

### anim

- 뜻: 숨, 생명, 마음
- 구분 포인트: [[anim]]은 생명과 마음의 활기. `psych`는 심리와 정신 영역에 더 직접 연결.
