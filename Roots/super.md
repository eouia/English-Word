# super

- 출발 단어: `superficial`
- 핵심 의미: 위에, 위쪽에서, 넘어서
- 관련 형태: Latin `super` = over, above, beyond
- 관련 문서: [[high]], [[vid]], [[flu]], [[facet]]

## 핵심 이미지

`super-`의 바탕은 "위에 있음".

위에서 보면 `supervise`.
위에 있거나 더 뛰어나면 `superior`.
넘쳐 흐르면 `superfluous`.
자연을 넘어선 것은 `supernatural`.
겉면 위에만 머무르면 `superficial`.

## 단어들

### super

- 발음: /ˈsupɚ/
- 구성: Latin `super` = over, above
- 뜻: 위에, 넘어서; 대단한
- 연결: 기준보다 위에 있거나 더 큰 상태
- 사용 맥락: 접두사와 일상 형용사로 모두 쓰임.

### supervise

- 발음: /ˈsupɚˌvaɪz/
- 구성: `super-` = over + `vis` = see
- 뜻: 감독하다, 관리하다
- 연결: 위에서 내려다보며 살핌
- 사용 맥락: `supervise a project` = 프로젝트를 감독하다

### supervisor

- 발음: /ˈsupɚˌvaɪzɚ/
- 구성: `supervise` + `-or`
- 뜻: 감독자, 상사
- 연결: 위에서 보고 관리하는 사람
- 사용 맥락: `ask your supervisor` = 상사에게 물어보다

### superior

- 발음: /suˈpɪriɚ/
- 구성: Latin `superior` = higher
- 뜻: 더 나은, 우월한; 상급자
- 연결: 위치나 등급이 위에 있음
- 사용 맥락: `superior quality` = 우수한 품질

### superfluous

- 발음: /ˈsupɚˌflwʌs/
- 구성: `super-` = over + `flu` = flow
- 뜻: 불필요한, 남아도는
- 연결: 필요한 양을 넘어 흘러넘침
- 사용 맥락: `superfluous details` = 불필요한 세부 사항

### supernatural

- 발음: /ˌsupɚˈnætʃɚəl/
- 구성: `super-` = beyond + `natural`
- 뜻: 초자연적인
- 연결: 자연의 범위를 넘어선 것
- 사용 맥락: `supernatural powers` = 초자연적 힘

### superficial

- 발음: /ˌsupɚˈfɪʃəl/
- 구성: Latin `superficies` = surface, from `super-` + `facies`
- 뜻: 표면적인, 피상적인
- 연결: 깊은 곳이 아니라 겉면 위에만 머무름
- 사용 맥락: `a superficial analysis` = 피상적인 분석

## 비교 단어

`high`는 [[high]] 문서처럼 독립 형용사 "높음"이 중심.
`super-`는 단어 앞에서 "위에, 넘어서"라는 방향과 정도를 더하는 접두사.
