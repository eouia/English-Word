# apt

- 출발 단어: `adapt`
- 핵심 어원: Latin `aptus` = fitted, suited, fastened
- 핵심 이미지: 알맞게 맞음, 목적에 맞게 조정됨, ~하기 쉬움
- 관련 문서: [[fit]]
- 주의: `adapt`, `adopt`, `adept`는 철자가 비슷하지만 구분. `adapt`는 맞추어 조정함, `adopt`는 선택해 받아들임, `adept`는 [[adipisc]] 계열의 숙련됨.

## 출발 단어 분석

`adapt` = `ad-` + `apt`

- `ad-`: to, toward
- `aptus`: fitted, suited
- 원래 이미지: 어떤 대상이나 환경에 맞게 붙임
- 현대 의미: 적응하다, 맞추다, 각색하다

## 단어들

### adapt

- 발음: /əˈdæpt/
- 구성: `ad-` = to + Latin `aptus` = fitted, suited
- 뜻: 적응하다, 맞추다, 각색하다
- 어원적 연결: 새로운 조건에 알맞게 맞춤
- 사용 맥락: `adapt to change` = 변화에 적응하다, `adapt a novel into a film` = 소설을 영화로 각색하다

### adaptation

- 발음: /ˌædəˈpteɪʃən/
- 구성: `adapt` + `-ation`
- 뜻: 적응, 각색, 개작
- 어원적 연결: 환경이나 목적에 맞게 바뀐 것
- 사용 맥락: `a film adaptation` = 영화 각색작

### adaptable

- 발음: /əˈdæptəbəl/
- 구성: `adapt` + `-able`
- 뜻: 적응력이 있는, 조정 가능한
- 어원적 연결: 여러 조건에 맞게 바뀔 수 있음
- 사용 맥락: `an adaptable worker` = 적응력이 있는 직원

### adaptive

- 발음: /əˈdæptɪv/
- 구성: `adapt` + `-ive`
- 뜻: 적응의, 적응적인
- 어원적 연결: 변화하는 조건에 맞게 반응함
- 사용 맥락: `adaptive learning` = 적응형 학습

### apt

- 발음: /ˈæpt/
- 구성: Latin `aptus` = fitted, suited
- 뜻: 적절한, ~하기 쉬운, 재능 있는
- 어원적 연결: 어떤 상황에 잘 맞거나 어떤 행동으로 이어지기 쉬움
- 사용 맥락: `an apt example` = 적절한 예, `be apt to forget` = 잊어버리기 쉽다

### aptly

- 발음: /ˈæptli/
- 구성: `apt` + `-ly`
- 뜻: 적절하게
- 어원적 연결: 상황에 알맞게
- 사용 맥락: `aptly named` = 이름이 적절하게 붙은

### aptitude

- 발음: /ˈæptəˌtud/
- 구성: Latin `aptitudo` = fitness, suitability
- 뜻: 적성, 소질
- 어원적 연결: 어떤 일을 하기에 알맞은 능력
- 사용 맥락: `a language aptitude test` = 언어 적성 검사

### inapt

- 구성: `in-` = not + `apt`
- 뜻: 부적절한, 서투른
- 어원적 연결: 상황이나 목적에 맞지 않음
- 사용 맥락: `an inapt comparison` = 부적절한 비교

## 기억 포인트

`apt` 계열은 "알맞게 맞음, 적응, 소질" 이미지.

- `adapt`: 환경에 맞추다
- `adaptation`: 맞춘 결과, 각색
- `adaptable`: 맞춰 바뀔 수 있음
- `apt`: 알맞은, ~하기 쉬운
- `aptitude`: 어떤 일에 맞는 소질

## 비교 단어

### fit

- 뜻: 알맞게 들어맞다
- 구분 포인트: [[fit]]은 영어 기본어의 "맞다". `apt`는 Latin `aptus` 계열로 `adapt`, `aptitude`처럼 공식·학습 어휘에 많이 보임.

### adopt

- 뜻: 채택하다, 입양하다
- 구분 포인트: `adopt`는 Latin `optare` = choose 계열. `adapt` = 맞추다와 다름.

### adept

- 뜻: 능숙한, 숙련자
- 구분 포인트: [[adipisc]]는 Latin `adipisci` = attain 계열. `adapt`와 철자가 비슷하지만 "맞추다"가 아니라 "도달해 얻다"에서 숙련 의미가 나옴.
