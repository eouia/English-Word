# arch

- 출발 단어: `monarch`
- 핵심 어원: 그리스어 `arkhein` / `arkhos` = 시작하다, 지배하다, 우두머리
- 핵심 이미지: 맨 처음에 있거나 위에서 이끎
- 관련해서 헷갈릴 수 있는 어근: [[crat]] = 권력, 지배 체제 계열, [[head]] = 머리, 우두머리 계열

## 출발 단어 분석

`arch-`는 Greek `arkhein`에서 온 계열.

- `arkhein`: begin, rule
- `arkhos`: ruler, chief

따라서 `arch` 계열은 "처음/원형"과 "지배자/우두머리" 두 갈래가 함께 있음.

## 갈래 1: 지배자, 우두머리

### monarch

- 발음: /ˈmɑˌnɑrk/
- 구성: `mono-` = one + `arch`
- 뜻: 군주
- 어원적 연결: 한 사람이 지배함
- 사용 맥락: `a constitutional monarch` = 입헌 군주

### monarchy

- 발음: /ˈmɑnɑrki/
- 구성: `mono-` = one + `archy`
- 뜻: 군주제
- 어원적 연결: 한 사람이 지배하는 체제
- 사용 맥락: `absolute monarchy` = 절대군주제

### anarchy

- 발음: /ˈænɚˌki/
- 구성: `an-` = without + `archy`
- 뜻: 무정부 상태, 무질서
- 어원적 연결: 지배자나 통치 질서가 없음
- 사용 맥락: `political anarchy` = 정치적 무정부 상태

### hierarchy

- 발음: /ˈhaɪɚˌɑrki/
- 구성: Greek `hieros` = sacred + `archy`
- 뜻: 위계, 계층 구조
- 어원적 연결: 원래 신성한 지배 질서. 현대에는 위아래 등급 구조
- 사용 맥락: `corporate hierarchy` = 회사 위계 구조

### patriarch

- 발음: /ˈpeɪtriˌɑrk/
- 구성: `patri-` = father + `arch`
- 뜻: 가장, 남성 원로
- 어원적 연결: 아버지 우두머리
- 사용 맥락: `family patriarch` = 집안의 가장

### matriarch

- 발음: /ˈmeɪtriˌɑrk/
- 구성: `matri-` = mother + `arch`
- 뜻: 여성 가장, 여족장
- 어원적 연결: 어머니 우두머리
- 사용 맥락: `the matriarch of the family` = 집안의 여성 가장

## 갈래 2: 시작, 원형, 오래된 것

### archaic

- 발음: /ɑˈrkeɪɪk/
- 구성: Greek `arkhaikos`
- 뜻: 고대의, 낡은
- 어원적 연결: 처음 시기의 것
- 사용 맥락: `archaic language` = 고어체

### archaeology

- 발음: /ˌɑrkiˈɑlədʒi/
- 구성: `archaeo-` = ancient + [[logos]]
- 뜻: 고고학
- 어원적 연결: 오래된 것에 대한 학문
- 사용 맥락: `study archaeology` = 고고학을 공부하다

### archetype

- 발음: /ˈɑrkɪˌtaɪp/
- 구성: `arch-` = original + `type`
- 뜻: 원형, 전형
- 어원적 연결: 처음 되는 본보기
- 사용 맥락: `a hero archetype` = 영웅 원형

### archive

- 발음: /ˈɑˌrkaɪv/
- 구성: Greek `arkheion` = public records office
- 뜻: 기록 보관소, 기록하다
- 어원적 연결: 통치 관청의 기록 보관소에서 온 말
- 사용 맥락: `digital archive` = 디지털 기록 보관소

### architect

- 발음: /ˈɑrkəˌtɛkt/
- 구성: `archi-` = chief + `tecton` = builder
- 뜻: 건축가, 설계자
- 어원적 연결: 우두머리 건축자
- 사용 맥락: `software architect` = 소프트웨어 설계자

### architecture

- 발음: /ˈɑrkəˌtɛktʃɚ/
- 구성: `architect` + `-ure`
- 뜻: 건축, 구조, 설계
- 어원적 연결: 우두머리 건축자가 만드는 설계 체계
- 사용 맥락: `system architecture` = 시스템 구조

## 기억 포인트

`arch` 계열은 "처음 / 우두머리" 이미지.

- `monarch`: 한 지배자
- `anarchy`: 지배자가 없음
- `hierarchy`: 위계적 지배 구조
- `patriarch`: 아버지 우두머리
- `archaic`: 처음 시기의, 오래된
- `archetype`: 원형
- `architect`: 우두머리 건축자

## 비교 단어

- 뜻: 권력, 지배 체제

### head

- 뜻: 머리, 우두머리
- 구분 포인트: [[head]]는 영어 기본어. `arch`는 Greek계 고급어휘에서 나타남
