# cit

- 출발 단어: `incite`
- 핵심 어원: Latin `ciere` / `citare` = set in motion, summon, rouse
- 핵심 이미지: 불러내거나 자극해 움직이게 함
- 관련 문서: [[voc]], [[ag]]
- 주의: `cite`는 원래 "공식적으로 불러내다, 소환하다"에서 "인용하다"로 확장. `incite`는 움직이도록 부추기는 쪽.

## 출발 단어 분석

`incite` = `in-` + `citare`

- `in-`: into, on, upon
- `citare`: move, summon, rouse
- 원래 이미지: 안쪽으로 자극을 넣어 움직이게 함
- 현대 의미: 선동하다, 부추기다, 자극하다

## 단어들

### incite

- 구성: Latin `incitare` ← `in-` + `citare`
- 뜻: 선동하다, 부추기다, 자극하다
- 어원적 연결: 사람이나 감정을 움직이도록 강하게 자극함
- 사용 맥락: `incite violence` = 폭력을 선동하다, `incite anger` = 분노를 자극하다

### incitement

- 구성: `incite` + `-ment`
- 뜻: 선동, 자극, 부추김
- 어원적 연결: 행동이나 감정을 움직이게 만드는 행위
- 사용 맥락: `incitement to violence` = 폭력 선동

### inciter

- 구성: `incite` + `-er`
- 뜻: 선동자, 부추기는 사람
- 어원적 연결: 다른 사람을 움직이게 자극하는 사람
- 사용 맥락: 정치·법률·사회 갈등 맥락

### excite

- 구성: Latin `excitare` ← `ex-` = out + `citare`
- 뜻: 흥분시키다, 자극하다
- 어원적 연결: 안에 있던 반응이나 감정을 밖으로 일으켜 움직이게 함
- 사용 맥락: `excite curiosity` = 호기심을 자극하다

### excitement

- 구성: `excite` + `-ment`
- 뜻: 흥분, 들뜸, 자극
- 어원적 연결: 감정이 일어나 움직이는 상태
- 사용 맥락: `a sense of excitement` = 들뜬 느낌

### exciting

- 구성: `excite` + `-ing`
- 뜻: 신나는, 흥미진진한
- 어원적 연결: 감정이나 관심을 움직이게 함
- 사용 맥락: `an exciting opportunity` = 흥미로운 기회

### cite

- 구성: Old French `citer` ← Latin `citare`
- 뜻: 인용하다; 소환하다; 언급하다
- 어원적 연결: 원래는 사람을 공식적으로 불러냄. 글에서는 근거를 불러와 제시함.
- 사용 맥락: `cite a source` = 출처를 인용하다

### citation

- 구성: `cite` + `-ation`
- 뜻: 인용, 출처 표기; 소환장
- 어원적 연결: 근거나 사람을 불러내는 행위
- 사용 맥락: `academic citation` = 학술 인용

### recite

- 구성: `re-` = again, back + `cite`
- 뜻: 암송하다, 낭독하다
- 어원적 연결: 배운 내용을 다시 불러내어 말함
- 사용 맥락: `recite a poem` = 시를 암송하다

### recitation

- 구성: `recite` + `-ation`
- 뜻: 암송, 낭독
- 어원적 연결: 내용을 다시 불러내어 말하는 행위
- 사용 맥락: 학교·문학·종교 문맥

## 기억 포인트

`cit`는 "불러내어 움직이게 함" 이미지.

- `incite`: 행동하게 부추김
- `incitement`: 선동, 부추김
- `excite`: 감정이나 반응을 일으킴
- `cite`: 근거나 사람을 불러냄
- `citation`: 인용, 소환
- `recite`: 내용을 다시 불러내 말함

## 비교 단어

### solicit

- 뜻: 요청하다, 간청하다
- 연결 포인트: Latin `sollicitare`가 `citare`와 관련됨. 다만 현대 형태와 의미가 달라 별도 항목으로 확장 가능.

### provoke

- 뜻: 유발하다, 도발하다
- 구분 포인트: [[voc]]의 `vocare` = call 계열. `provoke`는 앞으로 불러내어 반응을 일으키는 이미지.

### agitate

- 뜻: 동요시키다, 선동하다
- 구분 포인트: [[ag]] 계열. 계속 흔들고 움직이게 하는 이미지. `incite`는 특정 행동이나 감정을 부추김.
