# odd

- 출발 단어: `oddity`
- 핵심 어원: Old Norse `oddi` = point, triangle, odd number
- 핵심 이미지: 짝이 맞지 않음 / 남는 것 / 보통과 다름
- 관련해서 헷갈릴 수 있는 어근: [[extra]] = 범위 밖, 외부 계열, [[anomal]] = 규칙에서 벗어남 계열

## 출발 단어 분석

`oddity`는 `odd` + `-ity`.

- `odd`: strange, not even, unmatched
- `-ity`: 상태, 성질

따라서 `oddity`는 "짝이 맞지 않고 보통과 다른 성질" → "이상한 점, 특이한 것"의 뜻.

## 단어들

### oddity

- 구성: `odd` + `-ity`
- 뜻: 이상한 점, 특이한 것
- 어원적 연결: 보통의 짝이나 규칙에서 벗어난 것
- 사용 맥락: `a historical oddity` = 역사적 특이점

### odd

- 구성: Old Norse `oddi`
- 뜻: 이상한, 홀수의, 짝이 맞지 않는
- 어원적 연결: 둘씩 맞아떨어지지 않고 하나가 남거나 다름
- 사용 맥락: `an odd habit` = 이상한 습관, `odd numbers` = 홀수

### oddly

- 구성: `odd` + `-ly`
- 뜻: 이상하게, 묘하게
- 어원적 연결: 보통과 다르게 보이는 방식
- 사용 맥락: `oddly familiar` = 이상하게 익숙한

### odds

- 구성: plural of `odd`
- 뜻: 가능성, 확률; 불리한 조건
- 어원적 연결: 짝이 맞지 않는 차이, 한쪽으로 기운 차이에서 확률의 뜻
- 사용 맥락: `against all odds` = 온갖 역경에도 불구하고

## 기억 포인트

`odd` 계열은 "짝이 맞지 않음" 이미지.

- `odd`: 이상한, 홀수의
- `oddity`: 이상한 점, 특이한 것
- `oddly`: 이상하게
- `odds`: 가능성, 확률, 불리한 조건

## 비교 단어

### strange

- 뜻: 이상한, 낯선
- 구분 포인트: [[extra]] 계열의 `strange`는 외부에서 온 낯섦의 이미지. `odd`는 짝이나 규칙에서 어긋난 느낌

### anomaly

- 뜻: 변칙, 이례
- 구분 포인트: [[anomal]]은 규칙에서 벗어난 현상. `oddity`는 더 일상적이고 가벼운 "특이한 점"에도 씀
