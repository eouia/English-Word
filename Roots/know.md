# know

- 출발 단어: `acknowledge`
- 핵심 어원: Old English `cnawan` = know, recognize
- 핵심 이미지: 알고 있음, 알아봄, 알고 있음을 인정함
- 관련 문서: [[cogn]]
- 주의: `acknowledge`는 라틴어 `cogn` 계열이 아니라 English `know/knowledge`에 `ac-`가 붙은 형태로 보는 것이 실용적.

## 출발 단어 분석

`acknowledge` = `ac-` + `knowledge`

- `ac-`: to, toward, 또는 강조
- `knowledge`: 앎, 지식
- 원래 이미지: 어떤 사실을 알고 있음을 받아들임
- 현대 의미: 인정하다, 확인하다, 감사를 표하다

## 단어들

### know

- 발음: /ˈnoʊ/
- 구성: Old English `cnawan`
- 뜻: 알다, 알고 있다
- 어원적 연결: 사실이나 사람을 지식으로 파악함
- 사용 맥락: `know the answer` = 답을 알다

### known

- 발음: /ˈnoʊn/
- 구성: `know`의 과거분사
- 뜻: 알려진, 알고 있는
- 어원적 연결: 이미 지식으로 드러난 상태
- 사용 맥락: `a well-known writer` = 잘 알려진 작가

### unknown

- 발음: /əˈnnoʊn/
- 구성: `un-` = not + `known`
- 뜻: 알려지지 않은, 미지의
- 어원적 연결: 아직 지식으로 파악되지 않음
- 사용 맥락: `unknown causes` = 알려지지 않은 원인

### knowledge

- 발음: /ˈnɑlədʒ/
- 구성: `know` + `-ledge`
- 뜻: 지식, 앎
- 어원적 연결: 알고 있는 내용이나 상태
- 사용 맥락: `background knowledge` = 배경지식

### knowledgeable

- 발음: /ˈnɑlədʒəbəl/
- 구성: `knowledge` + `-able`
- 뜻: 지식이 풍부한
- 어원적 연결: 많은 것을 알고 있음
- 사용 맥락: `a knowledgeable guide` = 박식한 안내자

### acknowledge

- 발음: /æˈknɑlɪdʒ/
- 구성: `ac-` + `knowledge`
- 뜻: 인정하다, 확인하다, 감사를 표하다
- 어원적 연결: 알고 있음을 받아들여 드러냄
- 사용 맥락: `acknowledge a mistake` = 실수를 인정하다

### acknowledgment

- 발음: /æˈknɑlɪdʒmənt/
- 구성: `acknowledge` + `-ment`
- 뜻: 인정, 확인, 감사의 표시
- 어원적 연결: 알고 있음을 공식적으로 드러내는 행위
- 사용 맥락: `an acknowledgment of receipt` = 수령 확인

### knowingly

- 발음: /ˈnoʊɪŋli/
- 구성: `knowing` + `-ly`
- 뜻: 알면서, 고의로
- 어원적 연결: 알고 있는 상태에서 행동함
- 사용 맥락: `knowingly violate the rule` = 규칙을 알면서 위반하다

## 기억 포인트

`know` 계열은 "앎, 알아봄, 인정" 이미지.

- `know`: 알다
- `known`: 알려진
- `unknown`: 알려지지 않은
- `knowledge`: 지식
- `acknowledge`: 알고 있음을 인정함
- `acknowledgment`: 인정·확인

## 비교 단어

### cogn

- 뜻: 알다, 알아보다, 인식하다
- 구분 포인트: [[cogn]]은 Latin `cognoscere` 계열. `know`는 Germanic 기본어. `recognize`는 라틴계, `acknowledge`는 영어 내부 형성.
