# sincer

- 출발 단어: `sincerity`
- 핵심 어원: Latin `sincerus` = whole, clean, pure
- 핵심 이미지: 섞이거나 손상되지 않은 온전함
- 관련 문서: [[cand]], [[honor]], [[fid]]
- 주의: `sincere`를 `sin` + `cere`처럼 현대 형태로 쪼개지 않음. `without wax` 설은 유명하지만 확실한 어원으로 보지 않음.

## 출발 단어 분석

`sincerity` = `sincere` + `-ity`

- `sincere`: 진심의, 거짓 없는
- `-ity`: 성질, 상태를 나타내는 명사 접미사
- `sincerity`: 진심, 성실함, 거짓 없음

라틴어 `sincerus`는 "온전한, 깨끗한, 순수한"의 뜻. 의미 변화는 "섞이지 않고 깨끗함" → "속임이 없음" → "진심 어린 태도".

## 단어들

### sincere

- 구성: Latin `sincerus` = whole, clean, pure
- 뜻: 진심의, 성실한, 거짓 없는
- 어원적 연결: 섞이거나 꾸며지지 않은 마음
- 사용 맥락: `a sincere apology` = 진심 어린 사과

### sincerely

- 구성: `sincere` + `-ly`
- 뜻: 진심으로, 성실하게
- 어원적 연결: 꾸밈없이 마음에서 나온 방식
- 사용 맥락: `I sincerely hope so.` = 진심으로 그렇게 바라다

### sincerity

- 구성: `sincere` + `-ity`
- 뜻: 진심, 성실함
- 어원적 연결: 마음이나 말이 섞임 없이 온전한 상태
- 사용 맥락: `question someone's sincerity` = 진심을 의심하다

### insincere

- 구성: `in-` = not + `sincere`
- 뜻: 진심이 아닌, 가식적인
- 어원적 연결: 말과 마음이 온전히 맞지 않음
- 사용 맥락: `an insincere compliment` = 진심 없는 칭찬

### insincerity

- 구성: `in-` = not + `sincerity`
- 뜻: 불성실함, 가식
- 어원적 연결: 온전한 진심이 빠진 상태
- 사용 맥락: `detect insincerity` = 가식을 알아차리다

## 기억 포인트

`sincer` 계열은 "섞이지 않은 온전함"에서 출발.

- `sincere`: 마음이 온전한 → 진심의
- `sincerely`: 진심으로
- `sincerity`: 진심, 성실함
- `insincere`: 진심이 아닌
- `insincerity`: 가식, 불성실함

## 비교 단어

### candid

- 뜻: 솔직한, 숨김없는
- 구분 포인트: [[cand]]는 "희고 밝게 드러남" 이미지. `sincere`는 말보다 마음의 온전함과 진심에 초점.

### honest

- 뜻: 정직한
- 구분 포인트: [[honor]] 계열의 `honest`는 "명예로운 사람다운"에서 거짓 없음으로 발전. `sincere`는 속마음이 꾸며지지 않았다는 느낌이 강함.

### faithful

- 뜻: 충실한, 믿음을 지키는
- 구분 포인트: [[fid]]는 믿음과 신뢰 관계. `sincere`는 믿음을 지키는 행동보다 마음의 진정성에 가까움.

## 주의할 어원 이야기

`sincere`를 "밀랍 없이"라는 뜻의 Latin `sine cera`에서 왔다고 설명하는 이야기가 널리 퍼져 있음. 하지만 일반적으로 확실한 어원으로 보지 않음.

학습용으로는 `sincerus`의 "온전한, 깨끗한, 순수한" 이미지에서 `sincere`와 `sincerity`를 연결하는 편이 안전함.
