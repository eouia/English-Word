# hoard

- 출발 단어: `hoard`
- 핵심 어원: Old English `hord` = treasure, hidden store
- 핵심 이미지: 가치 있는 것을 숨겨 모아 둠
- 관련 문서: [[store]]
- 주의: `horde` = 큰 무리, 군중은 별도 어원. 철자와 발음이 비슷해도 `hoard`와 섞지 않음.

## 출발 단어 분석

`hoard`는 현대 형태에서 더 작게 쪼개기보다 Old English `hord` 전체로 보는 단어.

- 원래 의미: 보물, 숨겨 둔 저장물
- 의미 확장: 귀중품을 저장함 → 필요 이상으로 쌓아 둠
- 현대 의미: 비축물; 몰래 모아 두다, 사재기하다

## 단어들

### hoard

- 발음: /ˈhɔrd/
- 구성: Old English `hord` = treasure, hidden store
- 뜻: 비축물, 저장물; 몰래 모아 두다, 사재기하다
- 어원적 연결: 가치 있는 것을 숨겨 두고 꺼내 쓰지 않음
- 사용 맥락: `hoard food` = 음식을 사재기하다, `a hoard of coins` = 동전 더미

### hoarding

- 발음: /ˈhɔrdɪŋ/
- 구성: `hoard` + `-ing`
- 뜻: 사재기, 저장 강박; 몰래 쌓아 두는 행위
- 어원적 연결: 물건을 계속 모아 숨겨 두는 과정
- 사용 맥락: `panic hoarding` = 공황성 사재기, `hoarding disorder` = 저장 강박 장애

### hoarded

- 발음: /ˈhɔrdɪd/
- 구성: `hoard` + `-ed`
- 뜻: 비축된, 몰래 모아 둔
- 어원적 연결: 쓰이지 않고 숨겨 저장된 상태
- 사용 맥락: `hoarded supplies` = 비축된 물자

### hoarder

- 구성: `hoard` + `-er`
- 뜻: 사재기하는 사람, 저장 강박이 있는 사람
- 어원적 연결: 물건을 계속 쌓아 두는 사람
- 사용 맥락: `a compulsive hoarder` = 강박적으로 물건을 모으는 사람

## 기억 포인트

`hoard`는 "숨겨 모아 둔 저장물" 이미지.

- `hoard`: 몰래 모아 두다; 비축물
- `hoarding`: 사재기, 계속 쌓아 둠
- `hoarded`: 비축된
- `hoarder`: 쌓아 두는 사람

## 비교 단어

### horde

- 뜻: 큰 무리, 군중
- 구분 포인트: Turkic/Old French 계열로 보는 별도 어원. `a horde of people` = 사람들의 큰 무리. `a hoard of gold` = 금의 비축물.

### cache

- 뜻: 은닉처, 숨겨 둔 저장물
- 구분 포인트: 숨겨 둔 장소나 저장 공간에 초점. `hoard`는 모아 둔 물건 자체와 모으는 행위에 초점.

### store

- 뜻: 저장하다, 저장물, 가게
- 구분 포인트: 일반적인 저장. `hoard`는 몰래, 과도하게, 쓰지 않고 쌓아 두는 느낌이 강함.
