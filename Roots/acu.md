# acu

- 출발 단어: `acute`
- 핵심 어원: Latin `acus` = 바늘, Latin `acuere` = 날카롭게 하다
- 핵심 이미지: 뾰족하고 날카로움
- 관련 문서: [[cid]], [[acerb]]

## 출발 단어 분석

`acute` ← Latin `acutus`

- `acuere`: sharpen
- `acutus`: sharp, pointed
- 원래 이미지: 끝이 날카롭게 선 상태
- 현대 의미: 날카로운, 예리한; 심각한; 급성의

## 단어들

### acute

- 구성: Latin `acutus` = sharp, pointed
- 뜻: 날카로운, 예리한; 심각한; 급성의
- 어원적 연결: 감각·지성·상태가 날카롭게 선 느낌
- 사용 맥락: `acute pain` = 급성 통증, `acute awareness` = 예리한 인식
- 주의: 의학에서는 `acute`가 "급성의", `chronic`은 "만성의".

### acutely

- 구성: `acute` + `-ly`
- 뜻: 예리하게, 강렬하게, 절실히
- 어원적 연결: 감각이나 문제 인식이 날카롭게 작동함
- 사용 맥락: `acutely aware of the risk` = 위험을 절실히 인식하는

### acumen

- 구성: Latin `acumen` = sharp point, keenness
- 뜻: 통찰력, 판단력
- 어원적 연결: 사물의 핵심을 날카롭게 찌르는 지적 능력
- 사용 맥락: `business acumen` = 사업 감각, 사업적 판단력

### acuity

- 구성: Latin `acuitas` = sharpness
- 뜻: 예민함, 예리함; 시력·청력의 선명도
- 어원적 연결: 감각이나 지각이 날카롭게 구분함
- 사용 맥락: `visual acuity` = 시력

### acupuncture

- 구성: Latin `acus` = needle + `puncture` = 찌름
- 뜻: 침술
- 어원적 연결: 바늘로 찌르는 치료법
- 사용 맥락: `receive acupuncture treatment` = 침술 치료를 받다

### acrid

- 구성: Latin `acer/acris` = sharp, pungent
- 뜻: 매캐한, 톡 쏘는; 신랄한
- 어원적 연결: 냄새나 말투가 날카롭게 찌름
- 사용 맥락: `an acrid smell` = 매캐한 냄새

## 기억 포인트

`acu` 계열은 "날카로움" 이미지.

- `acute`: 날카롭고 예리함
- `acumen`: 날카로운 판단력
- `acuity`: 감각의 예리함
- `acupuncture`: 바늘로 찌르는 치료
- `acrid`: 톡 쏘듯 날카로운 냄새나 말투

## 비교 단어

### cid

- 뜻: 자르다, 끊다
- 구분 포인트: [[cid]]는 잘라 결정하거나 분리함. `acu`는 끝이 날카롭거나 감각이 예리함.

### acerb

- 뜻: 쓰고 거칠며 악화시킴
- 구분 포인트: [[acerb]]는 맛·말투·상태의 거친 씀. `acu`는 뾰족하고 예리한 날카로움.
