# indic

- 출발 단어: `index`
- 핵심 어원: Latin `index/indicis` = pointer, sign; from `indicare` = point out, show
- 깊은 어근: `in-` + `dicare` = proclaim, point out; PIE `*deik-` = show, point
- 핵심 이미지: 손가락이나 표시로 가리켜 알려 줌
- 관련 문서: [[dict]], [[digit]], [[speci]]
- 주의: 웹사이트의 폴더 인덱스와 충돌을 피하려고 대표 문서명은 `indic`.

## 출발 단어 분석

`index` = Latin `index`

- `index`: 가리키는 사람, 표시, 인덱스, 검지
- `indicare`: 가리키다, 알려 주다
- 원래 이미지: 무엇을 가리켜 보이게 하는 것
- 현대 의미: 색인, 지표, 지수

## 단어들

### index

- 발음: /ˈɪndɛks/
- 구성: Latin `index` = pointer, sign
- 뜻: 색인, 지표, 지수; 색인을 만들다
- 어원적 연결: 찾을 곳이나 상태를 가리켜 주는 표시
- 사용 맥락: `search the index` = 색인에서 찾다

### indices

- 발음: /ˈɪndɪˌsiz/
- 구성: Latin plural of `index`
- 뜻: 지표들, 지수들, 색인들
- 어원적 연결: 여러 개의 가리키는 표시
- 사용 맥락: `economic indices` = 경제 지표들

### indexes

- 발음: /ˈɪndɛksɪz/
- 구성: English plural of `index`
- 뜻: 색인들, 지표들
- 어원적 연결: `indices`보다 일반적·실용적 복수형
- 사용 맥락: database `indexes` = 데이터베이스 색인들

### indicate

- 발음: /ˈɪndəˌkeɪt/
- 구성: `in-` + `dicare`
- 뜻: 나타내다, 가리키다
- 어원적 연결: 말이나 표시로 가리켜 알려 줌
- 사용 맥락: `indicate a problem` = 문제를 나타내다

### indication

- 발음: /ˌɪndəˈkeɪʃən/
- 구성: `indicate` + `-ion`
- 뜻: 표시, 징후, 암시
- 어원적 연결: 어떤 사실을 가리키는 신호
- 사용 맥락: `an indication of progress` = 진전의 표시

### indicator

- 발음: /ˈɪndəˌkeɪtɚ/
- 구성: `indicate` + `-or`
- 뜻: 지표, 표시기
- 어원적 연결: 상태나 방향을 가리키는 것
- 사용 맥락: `economic indicator` = 경제 지표

### indicative

- 발음: /ˌɪˈndɪkətɪv/
- 구성: `indicate` + `-ive`
- 뜻: 나타내는, 암시하는; 직설법의
- 어원적 연결: 어떤 사실을 가리켜 보임
- 사용 맥락: `indicative of change` = 변화의 징후인

### indicium

- 구성: Latin `indicium` = notice, evidence
- 뜻: 표시, 징표, 증거
- 어원적 연결: 사실을 가리키는 단서
- 사용 맥락: 법률·학술 맥락

### indict

- 발음: /ˌɪˈndaɪt/
- 구성: Vulgar Latin `indictare` = declare, accuse in writing
- 뜻: 기소하다
- 어원적 연결: 공식적으로 말해 혐의를 가리킴
- 사용 맥락: `indict a suspect` = 용의자를 기소하다

### indictment

- 발음: /ˌɪˈndaɪtmənt/
- 구성: `indict` + `-ment`
- 뜻: 기소, 기소장
- 어원적 연결: 혐의를 공식적으로 적어 가리킨 문서
- 사용 맥락: `face an indictment` = 기소를 당하다

### indite

- 발음: /ˌɪˈndaɪt/
- 구성: same source as `indict`
- 뜻: 글을 짓다, 쓰다
- 어원적 연결: 말로 불러 쓰게 하거나 글로 선언함
- 사용 맥락: 문어적·고어적 표현

## 기억 포인트

`index` 계열은 "가리켜 알려 줌" 이미지.

- `index`: 찾을 곳을 가리키는 색인
- `indicate`: 가리켜 보임
- `indicator`: 상태를 가리키는 지표
- `indicative`: 무엇을 암시·표시함
- `indict`: 혐의를 공식적으로 가리켜 말함

## 비교 단어

### dict

- 뜻: 말하다, 선언하다
- 연결 포인트: [[dict]]와 같은 깊은 `dicere/dicare` 계열. `dict`는 말로 밝히는 행위, `index`는 가리키고 표시하는 기능이 강함.

### digit

- 뜻: 손가락, 숫자
- 연결 포인트: Latin `digitus` = finger도 깊은 PIE `*deik-` 계열로 연결됨. 손가락으로 가리키는 이미지.
