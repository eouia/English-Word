# oper

- 출발 단어: `operate`
- 핵심 어원: Latin `opus` / `operari` = 일, 작업하다
- 핵심 이미지: 실제로 작동하거나 일을 수행함
- 관련 문서: [[labor]], [[ag]], [[art]]

## 출발 단어 분석

`operate` ← Latin `operari`

- `opus`: work
- `operari`: work, perform
- 원래 이미지: 일을 수행함
- 현대 의미: 작동하다, 운영하다, 수술하다

## 단어들

### operate

- 구성: Latin `operari`
- 뜻: 작동하다, 운영하다, 수술하다
- 어원적 연결: 어떤 일이 실제로 수행됨
- 사용 맥락: `operate a machine` = 기계를 작동하다

### operation

- 구성: `operate` + `-ion`
- 뜻: 작동, 운영, 수술, 작전
- 어원적 연결: 일을 수행하는 과정
- 사용 맥락: `business operations` = 사업 운영

### operator

- 구성: `operate` + `-or`
- 뜻: 조작자, 운영자
- 어원적 연결: 장치나 작업을 실제로 수행하는 사람
- 사용 맥락: `a machine operator` = 기계 조작자

### cooperative

- 구성: [[con]] = together + `operate`
- 뜻: 협력적인
- 어원적 연결: 함께 일함
- 사용 맥락: `a cooperative attitude` = 협력적인 태도

### cooperate

- 구성: [[con]] = together + `operate`
- 뜻: 협력하다
- 어원적 연결: 함께 일을 수행함
- 사용 맥락: `cooperate with others` = 다른 사람들과 협력하다

### operational

- 구성: `operation` + `-al`
- 뜻: 운영상의, 작동 가능한
- 어원적 연결: 실제 작동이나 운영에 관련됨
- 사용 맥락: `operational efficiency` = 운영 효율

## 기억 포인트

`oper` 계열은 "일이 실제로 작동함" 이미지.

- `operate`: 작동하다, 운영하다
- `operation`: 작동과 운영
- `operator`: 조작하는 사람
- `cooperate`: 함께 일하다

## 비교 단어

### labor

- 뜻: 노동, 수고
- 구분 포인트: [[labor]]는 힘들여 일함. `oper`는 일이 실제로 수행되고 작동함.
