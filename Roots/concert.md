# concert

- 출발 단어: `disconcerting`
- 핵심 어원: French/Italian `concert` / `concerto` = agreement, harmony; Latin `concertare` = contend, debate, strive together
- 핵심 이미지: 서로 맞추어 조율함, 조율이 깨지면 당황함
- 관련 문서: [[cern]], [[consul]]
- 주의: `concert`의 역사적 어원은 다소 복잡함. Latin `concertare`는 원래 "다투다, 겨루다"였으나 Italian/French에서 "합의시키다, 조화시키다"로 발전. 음악 의미는 "함께 맞춘 조화" 이미지로 기억.

## 출발 단어 분석

`disconcerting` ← `disconcert` + `-ing`

- `dis-`: apart, away, undo
- `concert`: agreement, harmony, mutual arrangement
- `-ing`: 형용사 접미사
- 원래 이미지: 함께 맞춰진 조율을 흐트러뜨림
- 의미 변화: 조화·계획을 깨뜨림 → 마음의 평정을 깨뜨림 → 당황스럽고 불안하게 함
- 현대 의미: 당황하게 하는, 불안하게 하는, 마음을 흐트러뜨리는

## 단어들

### disconcerting

- 발음: /ˌdɪskəˈnsɝtɪŋ/
- 구성: `disconcert` + `-ing`
- 뜻: 당황스럽게 하는, 불안하게 하는
- 어원적 연결: 마음속의 조율과 안정감을 깨뜨리는 성질
- 사용 맥락: `a disconcerting silence` = 당황스러운 침묵

### disconcert

- 발음: /ˌdɪskəˈnsɝt/
- 구성: `dis-` = apart/undo + `concert`
- 뜻: 당황하게 하다, 불안하게 하다
- 어원적 연결: 맞춰진 계획이나 마음의 균형을 흐트러뜨림
- 사용 맥락: `The question disconcerted him.` = 그 질문은 그를 당황하게 했다.

### disconcerted

- 구성: `disconcert` + `-ed`
- 뜻: 당황한, 불안해진
- 어원적 연결: 마음의 조율이 깨진 상태
- 사용 맥락: `look disconcerted` = 당황한 표정을 짓다

### concert

- 발음: /ˈkɑnsɚt/
- 구성: French `concert` ← Italian `concerto`
- 뜻: 음악회; 조화, 협력
- 어원적 연결: 여러 소리나 사람이 함께 맞추어 조화를 이룸
- 사용 맥락: `in concert with others` = 다른 사람들과 협력하여

### concerted

- 발음: /kəˈnsɝtəd/
- 구성: `concert` + `-ed`
- 뜻: 협력한, 합동의, 일치된
- 어원적 연결: 여러 사람이 뜻과 행동을 맞춘 상태
- 사용 맥락: `a concerted effort` = 합심한 노력

### concerto

- 발음: /kəˈntʃɛrtoʊ/
- 구성: Italian `concerto` = concert, harmony
- 뜻: 협주곡
- 어원적 연결: 독주 악기와 오케스트라가 서로 맞서고 맞추며 조화를 이룸
- 사용 맥락: `a piano concerto` = 피아노 협주곡

### concertmaster

- 발음: /ˈkɑnsɚˌtmæstɚ/
- 구성: `concert` + `master`
- 뜻: 오케스트라의 악장
- 어원적 연결: 연주의 조율과 질서를 이끄는 수석 연주자
- 사용 맥락: `the concertmaster tuned the orchestra` = 악장이 오케스트라의 조율을 이끌었다

### concertina

- 발음: /kɑnsɚˈtinə/
- 구성: `concert` + Italian-style diminutive `-ina`
- 뜻: 콘서티나, 작은 아코디언류 악기
- 어원적 연결: 음악 연주와 관련된 작은 악기 이름
- 사용 맥락: `play the concertina` = 콘서티나를 연주하다

## 기억 포인트

`concert` 계열은 "함께 맞춤" 이미지.

- `concert`: 함께 맞춘 음악·조화
- `concerted`: 함께 맞춘, 합동의
- `concerto`: 독주와 합주의 조화
- `disconcert`: 조율을 깨 당황하게 함
- `disconcerting`: 마음의 평정을 깨는

## 비교 단어

### cern

- 뜻: 가려 구별하고 판단함
- 구분 포인트: [[cern]]은 Latin `cernere` 계열의 구별·판단. `concert`는 그 뒤의 `certare` "겨루다/논의하다"를 거쳐 조율·합의 이미지로 발전한 갈래.

### consul

- 뜻: 함께 숙고하고 조언을 구함
- 구분 포인트: [[consul]]은 상담·협의 과정. `concerted`는 협의를 거쳐 행동이 맞춰진 상태에 가까움.

### can

- 뜻: 노래하다
- 구분 포인트: `can/cant` 계열은 노래 자체. `concert`의 음악 의미는 여러 연주가 함께 조율된 공연.
