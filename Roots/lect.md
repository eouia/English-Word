# lect

- 출발 단어: `college`, `colleague`
- 핵심 어원: 라틴어 `legere` / `lect-` = 고르다, 모으다, 읽다
- 핵심 이미지: 여럿 중에서 "골라 모음" / 글을 골라 읽어 줌
- 비교: [[leg]] = 법, 정해진 규칙 계열, [[voc]] = 부르다 / 목소리 계열, [[logos]] = 말 / 설명 / 이성 계열

## 출발 단어 분석

`college`, `colleague`는 [[leg]]의 "법" 이미지보다 `legere / lect-`의 "고르다, 모으다" 이미지로 보는 편이 좋음.

- `col-`: 함께, 같이
- `legere / legare`: 고르다, 모으다, 임명하다
- `legere / lect-`: 읽다. 원래는 글자를 "골라 모아 읽음" 이미지

따라서 `colleague`는 "함께 선택된 사람" → "동료"의 뜻.  
`college`는 "함께 선택된 사람들의 모임" → "단체, 학회, 학교"의 뜻.

`lecture` 쪽에서는 "읽다"가 "글을 읽어 주다" → "강의하다, 설명하다"로 확장됨.  
다만 `lect` 자체가 일반적인 "말하다" 어근은 아님. 목소리로 부르는 말은 [[voc]], 논리적 말/설명은 [[logos]], 선언하듯 말함은 [[quoth]] 쪽과 비교.

## 단어들

### college

- 발음: /ˈkɑlɪdʒ/
- 구성: `col-` + `legere`
- 뜻: 대학, 단체
- 어원적 연결: 함께 선택되거나 모인 사람들의 집단
- 사용 맥락: `college students` = 대학생들

### colleague

- 발음: /ˈkɑlig/
- 구성: `col-` + `legare / legere`
- 뜻: 동료
- 어원적 연결: 같은 목적이나 직무를 위해 함께 선택된 사람
- 사용 맥락: `a former colleague` = 전 직장 동료

### collect

- 발음: /kəˈlɛkt/
- 구성: `col-` + `lect`
- 뜻: 모으다, 수집하다
- 어원적 연결: 여러 대상을 함께 골라 모음
- 사용 맥락: `collect data` = 자료를 수집하다

### collection

- 발음: /kəˈlɛkʃən/
- 구성: `collect` + `-ion`
- 뜻: 수집, 수집품
- 어원적 연결: 골라 모은 것들의 묶음
- 사용 맥락: `a collection of essays` = 에세이 모음집

### select

- 발음: /səˈlɛkt/
- 구성: `se-` = apart + `lect`
- 뜻: 선택하다
- 어원적 연결: 여럿 중에서 따로 골라냄
- 사용 맥락: `select a candidate` = 후보자를 선택하다

### elect

- 발음: /ɪˈlɛkt/
- 구성: `e- / ex-` = out + `lect`
- 뜻: 선출하다
- 어원적 연결: 사람들 중에서 밖으로 골라 뽑음
- 사용 맥락: `elect a president` = 대통령을 선출하다

### election

- 발음: /ɪˈlɛkʃən/
- 구성: `elect` + `-ion`
- 뜻: 선거
- 어원적 연결: 대표자를 골라 뽑는 과정
- 사용 맥락: `a national election` = 전국 선거

### neglect

- 발음: /nəˈglɛkt/
- 구성: `neg-` = not + `lect` = choose, gather
- 뜻: 방치하다, 소홀히 하다
- 어원적 연결: 골라 챙기지 않고 내버려 둠
- 사용 맥락: `neglect one's duties` = 의무를 소홀히 하다

### neglectful

- 발음: /nɪˈglɛktfəl/
- 구성: `neglect` + `-ful`
- 뜻: 소홀한, 태만한
- 어원적 연결: 챙겨야 할 것을 고르지 않고 지나치는 태도
- 사용 맥락: `neglectful parents` = 방임적인 부모

### negligent

- 발음: /ˈnɛglədʒənt/
- 구성: Latin `negligere` = disregard, neglect + `-ent`
- 뜻: 부주의한, 태만한
- 어원적 연결: 마땅히 살펴 골라야 할 것을 소홀히 함
- 사용 맥락: `negligent behavior` = 부주의한 행동

### delegate

- 발음: n. /ˈdɛləgət/, v. /ˈdɛləˌgeɪt/
- 구성: `de-` + `legare`
- 뜻: 대표자, 위임하다
- 어원적 연결: 어떤 임무를 맡겨 보내는 사람 또는 행위
- 사용 맥락: `delegate authority` = 권한을 위임하다

### allege

- 발음: /əˈlɛdʒ/
- 구성: `al- / ad-` = toward + `legare`
- 뜻: 주장하다, 혐의를 제기하다
- 어원적 연결: 어떤 말을 근거로 "가져다 붙임" 또는 "내세움"
- 사용 맥락: `allege misconduct` = 부정행위 혐의를 제기하다

### allegation

- 발음: /ˌæləˈgeɪʃən/
- 구성: `allege` + `-ation`
- 뜻: 주장, 혐의
- 어원적 연결: 사실로 확정되지 않은 주장이나 제기된 혐의
- 사용 맥락: `serious allegations` = 중대한 혐의들

### legacy

- 발음: /ˈlɛgəsi/
- 구성: 라틴어 `legare` = 보내다, 맡기다, 유언으로 남기다
- 뜻: 유산, 물려받은 것
- 어원적 연결: 누군가에게 맡겨 남긴 것. 이후 돈, 재산, 영향, 전통까지 확장
- 사용 맥락: `a lasting legacy` = 오래 남는 유산

### eligible

- 발음: /ˈɛlədʒəbəl/
- 구성: 라틴어 `eligere` = 골라낼 수 있는
- 뜻: 자격이 있는
- 어원적 연결: 선택될 수 있는 조건을 갖춤
- 사용 맥락: `eligible for admission` = 입학 자격이 있는

### elite

- 발음: /ɪˈlit/
- 구성: 프랑스어 `elite`, 라틴어 `eligere` 계열
- 뜻: 엘리트, 정예
- 어원적 연결: 선택된 사람들
- 사용 맥락: `an elite group` = 정예 집단

### lecture

- 발음: /ˈlɛktʃɚ/
- 구성: 라틴어 `lectura` = 읽기
- 뜻: 강의, 강의하다, 훈계하다
- 어원적 연결: 골라 모은 글을 읽어 주는 행위에서 강의와 말로 설명하는 행위로 확장
- 사용 맥락: `attend a lecture` = 강의를 듣다

### lecturer

- 발음: /ˈlɛktʃɚɚ/
- 구성: `lecture` + `-er`
- 뜻: 강사, 강연자
- 어원적 연결: 글을 읽어 주고 설명하는 사람
- 사용 맥락: `a university lecturer` = 대학 강사

### legible

- 발음: /ˈlɛdʒəbəl/
- 구성: `legere` = 읽다 + `-ible`
- 뜻: 읽기 쉬운
- 어원적 연결: 읽을 수 있는 상태
- 사용 맥락: `legible handwriting` = 읽기 쉬운 필체

### legend

- 발음: /ˈlɛdʒənd/
- 구성: 라틴어 `legenda` = 읽혀야 할 것
- 뜻: 전설
- 어원적 연결: 원래 "읽을 거리"의 뜻. 이후 전해 내려오는 이야기
- 사용 맥락: `urban legend` = 도시 전설

## 비교 단어

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 소리 내어 부르거나 목소리로 표현하는 이미지. `vocal`, `vocabulary`, `advocate`처럼 나타남. `lecture`의 말하기는 "읽어 주기"에서 온 확장 의미

### logos

- 뜻: 말, 설명, 이성, 학문
- 구분 포인트: [[logos]]는 말의 내용, 논리, 학문적 설명 쪽. `lect`는 글자를 골라 읽는 행위에서 출발

### bequeath

- 구성: 고대 영어 `be-` + `cweðan` = 말하다
- 뜻: 유언으로 남기다, 물려주다
- 어원적 연결: [[lect]] 계열 아님. [[quoth]] 계열. 의미상 `legacy`와 가까움
- 구분 포인트: `legacy` = 남겨진 유산, `bequeath` = 유산을 남기는 행위
- 사용 맥락: `bequeath property to children` = 자녀에게 재산을 물려주다

## 기억 포인트

`lect` 계열은 "고르고 모아 읽음" 이미지.

- `collect`: 함께 모음
- `select`: 따로 골라냄
- `elect`: 밖으로 뽑아냄
- `neglect`: 골라 챙기지 않고 내버려 둠
- `delegate`: 임무를 맡겨 보냄
- `allege`: 근거로 내세움
- `legacy`: 맡겨 남긴 것
- `eligible`: 선택될 자격 있음
- `college`: 함께 모인 집단
- `colleague`: 함께 선택된 사람
- `lecture`: 읽기에서 강의로 확장
- `lecturer`: 읽어 주고 설명하는 사람
- `legible`: 읽을 수 있음

비교:

- `voc`: 목소리로 부름
- `logos`: 말의 내용과 논리
- `bequeath`: 어원은 다름. 의미상 `legacy`와 함께 기억
