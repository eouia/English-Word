# tact

- 출발 단어: `tangible`
- 핵심 어원: Latin `tangere` = 만지다, 닿다
- 관련 형태: `tact-`, `tang-`, `tag-`
- 핵심 이미지: 손이나 감각이 닿음
- 관련 문서: [[manu]], [[sent]]

## 출발 단어 분석

`tangible` ← Latin `tangere`

- `tangere`: touch
- `tangibilis`: touchable
- 원래 이미지: 만질 수 있음
- 현대 의미: 유형의, 실제적인

## 단어들

### tangible

- 구성: Latin `tangere` + `-ible`
- 뜻: 만질 수 있는, 실질적인
- 어원적 연결: 손에 닿을 만큼 실제로 존재함
- 사용 맥락: `tangible evidence` = 실질적 증거

### intangible

- 구성: `in-` = not + `tangible`
- 뜻: 무형의, 손에 잡히지 않는
- 어원적 연결: 만질 수 없지만 존재하거나 가치가 있음
- 사용 맥락: `intangible assets` = 무형 자산

### contact

- 구성: `con-` = together + Latin `tangere`
- 뜻: 접촉, 연락
- 어원적 연결: 서로 닿음
- 사용 맥락: `make contact` = 연락하다

### tactile

- 구성: Latin `tactus` = touch
- 뜻: 촉각의
- 어원적 연결: 만지는 감각과 관련됨
- 사용 맥락: `tactile feedback` = 촉각 피드백

### tangent

- 구성: Latin `tangens` = touching
- 뜻: 접선, 탄젠트, 본론에서 벗어난 말
- 어원적 연결: 곡선에 한 점에서 "닿는" 선
- 사용 맥락: `go off on a tangent` = 이야기가 옆길로 새다

### tangential

- 구성: `tangent` + `-ial`
- 뜻: 접선의, 부차적인, 간접적인
- 어원적 연결: 중심을 통과하지 않고 옆에서 닿기만 함
- 사용 맥락: `a tangential issue` = 부차적인 문제

### tact

- 구성: Latin `tactus` = touch
- 뜻: 재치, 눈치, 요령
- 어원적 연결: 상황을 섬세하게 "건드리는" 감각
- 사용 맥락: `handle it with tact` = 재치 있게 처리하다

### intact

- 구성: `in-` = not + Latin `tactus`
- 뜻: 손상되지 않은, 온전한
- 어원적 연결: 건드려지지 않음
- 사용 맥락: `remain intact` = 온전하게 남다

### contagious

- 구성: `con-` = together + Latin `tangere`
- 뜻: 전염성의
- 어원적 연결: 접촉을 통해 옮겨 감
- 사용 맥락: `a contagious disease` = 전염병

### contagion

- 구성: `con-` = together + Latin `tangere`
- 뜻: 전염, 감염의 확산; 나쁜 영향의 확산
- 어원적 연결: 서로 닿아 옮겨 가는 상태
- 사용 맥락: `financial contagion` = 금융 위기의 전염성 확산
- 주의: 의학뿐 아니라 공포, 폭력, 금융 불안처럼 퍼져 나가는 현상에도 씀.

### contiguous

- 구성: Latin `contiguus` ← `con-` = together + `tangere`
- 뜻: 맞닿은, 인접한
- 어원적 연결: 경계나 면이 서로 닿아 있음
- 사용 맥락: `contiguous states` = 서로 이어져 있는 주들
- 주의: `continuous`는 끊김 없이 이어짐. `contiguous`는 서로 맞닿아 있음.

### contiguity

- 구성: `contiguous` + `-ity`
- 뜻: 인접성, 맞닿아 있음
- 어원적 연결: 둘 이상의 대상이 서로 닿아 있는 상태
- 사용 맥락: `spatial contiguity` = 공간적 인접성

### contingent

- 구성: Latin `con-` = together + `tangere` = touch
- 뜻: 조건부의, 우발적인; 대표단
- 어원적 연결: 어떤 조건이나 사건에 닿아 함께 일어남
- 사용 맥락: `contingent on approval` = 승인에 달린

### contingency

- 구성: `contingent` + `-cy`
- 뜻: 우발 사태, 만일의 경우; 조건부 상황
- 어원적 연결: 어떤 일이 닿아 발생할 수 있는 가능성
- 사용 맥락: `a contingency plan` = 비상 계획

### contingently

- 구성: `contingent` + `-ly`
- 뜻: 조건부로, 우발적으로
- 어원적 연결: 특정 조건에 닿아 달라지는 방식
- 사용 맥락: 철학·논리 문맥에서 자주 씀

## 기억 포인트

`tact/tang` 계열은 "닿다, 만지다" 이미지.

- `tangible`: 만질 수 있음
- `intangible`: 만질 수 없음
- `contact`: 서로 닿음
- `tactile`: 촉각의
- `tangent`: 한 점에서 닿는 선
- `intact`: 건드려지지 않음
- `contagion`: 접촉처럼 퍼져 나감
- `contiguous`: 서로 맞닿아 있음
- `contingent`: 조건에 닿아 달린

## 비교 단어

### manu

- 뜻: 손
- 구분 포인트: [[manu]]는 손 자체와 손으로 다룸. `tact`는 닿고 만지는 행위.

### cad

- 뜻: 떨어지다, 우연히 일어나다
- 구분 포인트: [[cad]]의 `accident`, `incident`는 일이 우연히 닥치거나 발생함. `contingent`는 어떤 조건에 닿아 달려 있거나 그 경우에 따라 일어남.
