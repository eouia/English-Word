# seize

- 출발 단어: `seize`
- 핵심 어원: Old French `saisir` = 소유하게 하다, 붙잡다
- 핵심 이미지: 갑자기 붙잡아 자기 통제 아래 둠
- 비교: [[sed]]의 `possess` = 차지하고 앉음
- 비교: [[prehend]] = 붙잡다, 마음으로 이해하다 계열

## 출발 단어 분석

`seize`는 "붙잡다, 장악하다"의 뜻.

- Old French `saisir`: 소유하게 하다, 붙잡다
- Medieval Latin `sacire`: 법적으로 차지하게 하다 계열
- Germanic `set` 계열과 연결 가능성도 있으나, 학습용으로는 별도 취급

따라서 `seize`는 "붙잡아 소유/통제함" → "장악하다, 압수하다"의 뜻.

## 단어들

### seize

- 구성: Old French `saisir`
- 뜻: 붙잡다, 장악하다, 압수하다
- 어원적 연결: 갑자기 잡아 자기 통제 아래 둠
- 사용 맥락: `seize the opportunity` = 기회를 붙잡다

### seizure

- 구성: `seize` + `-ure`
- 뜻: 압수, 장악, 발작
- 어원적 연결: 무언가가 갑자기 몸이나 물건을 장악함
- 사용 맥락: `a seizure of assets` = 자산 압류

### seisin

- 구성: Anglo-French `seisine`
- 뜻: 법적 점유
- 어원적 연결: 토지나 재산을 법적으로 차지한 상태
- 사용 맥락: 법률 용어. 시험 어휘로는 참고 수준

## 비교 단어

### possess

- 구성: 라틴어 `possidere` = 차지하고 앉다
- 뜻: 소유하다
- 어원적 연결: [[sed]] 계열. 어떤 자리를 차지하고 있음
- 구분 포인트: `possess` = 가지고 있음, `seize` = 갑자기 붙잡아 장악함

### hunt

- 구성: Old English `huntian`
- 뜻: 사냥하다
- 어원적 연결: 붙잡기 위해 쫓음. [[haunt]]와는 별도
- 구분 포인트: `hunt` = 쫓는 과정, `seize` = 붙잡아 장악하는 순간

### apprehend

- 구성: 라틴어 `ad-` + `prehendere`
- 뜻: 체포하다, 이해하다
- 어원적 연결: [[prehend]] 계열. 몸이나 마음으로 붙잡음
- 구분 포인트: `seize` = 장악함, `apprehend` = 체포하거나 의미를 붙잡음

## 기억 포인트

`seize` 계열은 "붙잡아 장악함" 이미지.

- `seize`: 붙잡아 장악함
- `seizure`: 갑작스러운 장악, 압수, 발작
- `seisin`: 법적 점유

비교:

- `possess`: 이미 차지하고 있음
- `seize`: 강하게 붙잡아 차지함
