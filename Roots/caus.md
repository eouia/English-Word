# caus

- 출발 단어: `cause`
- 핵심 어원: Latin `causa` = cause, reason, legal case
- 핵심 이미지: 어떤 일이 일어난 이유, 책임을 묻는 사유
- 관련 문서: [[cad]], [[argu]]
- 주의: `cause`는 "원인"과 "대의/명분"을 모두 가짐. Latin `causa`가 이유뿐 아니라 법적 사건·소송 사유도 뜻했기 때문.

## 출발 단어 분석

`cause` ← Latin `causa`

- `causa`: cause, reason, lawsuit, case
- 원래 이미지: 어떤 일의 이유나 법적으로 따져야 할 사유
- 의미 변화: 이유·사유 → 원인 → 주장할 명분·대의
- 현대 의미: 원인, 이유; 대의, 목적; 일으키다

## 단어들

### cause

- 구성: Latin `causa` = cause, reason
- 뜻: 원인, 이유; 대의; 일으키다
- 어원적 연결: 어떤 일이 왜 생겼는지 설명하는 사유
- 사용 맥락: `cause a problem` = 문제를 일으키다

### because

- 구성: `by` + `cause`
- 뜻: 왜냐하면, 때문에
- 어원적 연결: 어떤 이유에 의해
- 사용 맥락: `because of the rain` = 비 때문에

### causal

- 구성: `cause` + `-al`
- 뜻: 원인의, 인과 관계의
- 어원적 연결: 한 일이 다른 일을 일으키는 관계
- 사용 맥락: `a causal link` = 인과 관계

### causality

- 구성: `causal` + `-ity`
- 뜻: 인과성, 인과 관계
- 어원적 연결: 원인과 결과가 이어지는 성질
- 사용 맥락: `correlation does not imply causality` = 상관관계가 인과관계를 뜻하지는 않음

### causation

- 구성: `cause` + `-ation`
- 뜻: 원인 작용, 인과 관계
- 어원적 연결: 어떤 결과를 일으키는 작용
- 사용 맥락: 법률·철학·연구 방법론에서 자주 씀

### causative

- 구성: `cause` + `-ative`
- 뜻: 원인이 되는, 사역의
- 어원적 연결: 어떤 결과가 일어나게 만드는 성질
- 사용 맥락: `a causative factor` = 원인 요인

### accuse

- 구성: Latin `ad-` = to + `causa` = legal case, charge
- 뜻: 고발하다, 비난하다
- 어원적 연결: 어떤 사람에게 책임 사유를 붙임
- 사용 맥락: `accuse someone of lying` = 누군가를 거짓말로 비난하다

### accusation

- 구성: `accuse` + `-ation`
- 뜻: 고발, 비난, 혐의
- 어원적 연결: 책임을 묻는 사유를 제기함
- 사용 맥락: `a serious accusation` = 심각한 혐의

### excuse

- 구성: Latin `ex-` = out + `causa` = charge, legal case
- 뜻: 변명하다, 면제하다; 변명
- 어원적 연결: 책임 사유에서 빼내어 벗어나게 함
- 사용 맥락: `make an excuse` = 변명하다

### recuse

- 구성: Latin `recusare` = object to, refuse, from `causa` 계열
- 뜻: 법관·배심원을 기피하다; 스스로 사건에서 빠지다
- 어원적 연결: 특정 사유 때문에 사건 판단에서 물러나게 함
- 사용 맥락: `recuse oneself from a case` = 사건에서 스스로 물러나다

## 기억 포인트

`caus` 계열은 "이유와 책임 사유" 이미지.

- `cause`: 원인, 일으키다
- `because`: 어떤 이유 때문에
- `causal`: 인과 관계의
- `causality`: 인과성
- `accuse`: 책임 사유를 붙여 고발하다
- `excuse`: 책임 사유에서 빼내다, 변명하다
- `recuse`: 사유가 있어 사건에서 빠지다

## 비교 단어

### cad

- 뜻: 떨어지다, 우연히 일어나다
- 구분 포인트: [[cad]]의 `case`는 Latin `casus` = fall, event 계열. `cause`의 `causa`와 철자·법률 의미가 가까워 보여도 어원은 구분.

### reason

- 뜻: 이유, 이성
- 구분 포인트: `reason`은 설명·논리 쪽이 강함. `cause`는 결과를 실제로 일으킨 원인 쪽이 강함.

### argue

- 뜻: 주장하다, 논증하다
- 구분 포인트: [[argu]]는 이유를 들어 밝히고 증명하는 행위. `caus`는 그 이유·원인 자체.
