# cap

- 출발 단어: `capture`
- 핵심 어원: 라틴어 `capere` / `capt-` / `cept-` / `cip-` = 잡다, 받다, 취하다
- 핵심 이미지: 손에 잡거나 마음으로 받아들임
- 관련해서 헷갈릴 수 있는 어근: [[prehend]] = 붙잡다, 이해하다 계열, [[capit]] = 머리, 우두머리 계열

## 출발 단어 분석

`capture`는 Latin `captura`에서 온 말.

- `capere`: 잡다, 취하다
- `capt-`: 잡힌, 붙잡은 형태

따라서 `capture`는 "붙잡아 자기 쪽으로 가져옴" → "포획하다, 사로잡다"의 뜻.

## 단어들

### capture

- 발음: /ˈkæptʃɚ/
- 구성: `capt` + `-ure`
- 뜻: 붙잡다, 포획하다, 포착하다
- 어원적 연결: 물리적으로 잡거나 이미지, 관심을 붙잡음
- 사용 맥락: `capture attention` = 관심을 사로잡다

### captive

- 발음: /ˈkæptɪv/
- 구성: `capt` + `-ive`
- 뜻: 포로의, 사로잡힌
- 어원적 연결: 잡힌 상태
- 사용 맥락: `a captive audience` = 어쩔 수 없이 듣는 청중

### capacity

- 발음: /kəˈpæsəti/
- 구성: Latin `capax` = able to hold
- 뜻: 수용력, 능력
- 어원적 연결: 안에 받아 담을 수 있는 힘
- 사용 맥락: `limited capacity` = 제한된 수용력

### capable

- 발음: /ˈkeɪpəbəl/
- 구성: `cap` + `-able`
- 뜻: 능력 있는, ~할 수 있는
- 어원적 연결: 일을 받아 처리할 수 있음
- 사용 맥락: `capable of change` = 변화할 수 있는

### accept

- 발음: /æˈksɛpt/
- 구성: `ad-` = to + `cept`
- 뜻: 받아들이다, 수락하다
- 어원적 연결: 자기 쪽으로 잡아 받음
- 사용 맥락: `accept an offer` = 제안을 받아들이다

### except

- 발음: /ɪˈksɛpt/
- 구성: `ex-` = out + `cept`
- 뜻: 제외하다, ~을 제외하고
- 어원적 연결: 전체에서 밖으로 잡아 뺌
- 사용 맥락: `everyone except me` = 나를 제외한 모두

### concept

- 발음: /ˈkɑnsɛpt/
- 구성: `con-` = together + `cept`
- 뜻: 개념
- 어원적 연결: 여러 생각을 함께 잡아 마음속에 형성한 것
- 사용 맥락: `a difficult concept` = 어려운 개념

### conception

- 발음: /kəˈnsɛpʃən/
- 구성: `concept/conceive` 계열 + `-ion`
- 뜻: 개념 형성, 생각; 임신
- 어원적 연결: 마음속이나 몸 안에 받아 품어 생기게 함
- 사용 맥락: `the conception of justice` = 정의에 대한 개념

### conceptual

- 발음: /kəˈnsɛptʃuəl/
- 구성: `concept` + `-ual`
- 뜻: 개념적인, 관념적인
- 어원적 연결: 구체적 사물보다 마음속에 잡은 생각과 관련됨
- 사용 맥락: `a conceptual framework` = 개념적 틀

### conceive

- 발음: /kəˈnsiv/
- 구성: `con-` = together + `cip`
- 뜻: 생각해 내다, 품다, 임신하다
- 어원적 연결: 마음이나 몸 안에 받아 품음
- 사용 맥락: `conceive an idea` = 생각을 품다

### misconception

- 발음: /mɪskəˈnsɛpʃən/
- 구성: `mis-` = wrongly + `conception`
- 뜻: 오해, 잘못된 생각
- 어원적 연결: 마음속에 잘못 잡힌 개념
- 사용 맥락: `a common misconception` = 흔한 오해

### inception

- 발음: /ˌɪˈnsɛpʃən/
- 구성: `in-` = in + `cept`
- 뜻: 시작, 개시
- 어원적 연결: 어떤 일이 안으로 잡혀 처음 생겨남
- 사용 맥락: `from its inception` = 그것이 시작된 때부터

### receive

- 발음: /rəˈsiv/
- 구성: `re-` = back + `ceive`
- 뜻: 받다, 수신하다
- 어원적 연결: 자기 쪽으로 돌아온 것을 받음
- 사용 맥락: `receive a message` = 메시지를 받다

### reception

- 발음: /rɪˈsɛpʃən/
- 구성: Latin `receptio` ← `recipere`
- 뜻: 접수, 수신, 환영회, 반응
- 어원적 연결: 사람·정보·신호를 받아들임
- 사용 맥락: `a warm reception` = 따뜻한 환영, `phone reception` = 전화 수신 상태

### receptive

- 발음: /rɪˈsɛptɪv/
- 구성: Latin `receptus` + `-ive`
- 뜻: 잘 받아들이는, 수용적인
- 어원적 연결: 생각이나 자극을 받아들일 준비가 되어 있음
- 사용 맥락: `be receptive to feedback` = 피드백을 잘 받아들이다

### receptivity

- 발음: /ˈriˌsɛˈptɪvɪti/
- 구성: `receptive` + `-ity`
- 뜻: 수용성, 받아들이는 성질
- 어원적 연결: 외부의 생각이나 자극을 받아들이는 능력이나 태도
- 사용 맥락: `receptivity to new ideas` = 새 생각에 대한 수용성

### susceptible

- 발음: /səˈsɛptəbəl/
- 구성: Latin `suscipere` = take up, receive from below + `-ible`
- 뜻: 영향을 받기 쉬운, 감염되기 쉬운, 취약한
- 어원적 연결: 외부 힘이나 병을 받아들일 수 있는 상태
- 사용 맥락: `susceptible to infection` = 감염에 취약한, `susceptible to pressure` = 압력에 흔들리기 쉬운

### susceptibility

- 발음: /səˌsɛptəˈbɪləti/
- 구성: `susceptible` + `-ity`
- 뜻: 감수성, 취약성, 영향을 받기 쉬움
- 어원적 연결: 외부 자극이나 위험을 받아들이는 성질
- 사용 맥락: `genetic susceptibility` = 유전적 취약성

### susceptive

- 구성: Latin `susceptus` = taken up, received + `-ive`
- 뜻: 받아들이기 쉬운, 감수성 있는
- 어원적 연결: 외부 영향을 받을 수 있는 성질
- 사용 맥락: 드문 단어. 시험 핵심은 `susceptible`, `susceptibility`.

### perceive

- 발음: /pɚˈsiv/

### perception

- 사용 맥락: `public perception` = 대중의 인식, `visual perception` = 시각 지각

### perceptible

- 구성: `percept` + `-ible`
- 뜻: 감지할 수 있는, 알아차릴 수 있는
- 어원적 연결: 감각을 통해 받아 잡을 수 있음
- 사용 맥락: `a perceptible change` = 감지할 수 있는 변화

### imperceptible

- 구성: `im-` = not + `perceptible`
- 뜻: 감지하기 어려운, 거의 알아차릴 수 없는
- 어원적 연결: 감각으로 받아 잡을 수 없을 만큼 작거나 미묘함
- 사용 맥락: `an imperceptible movement` = 거의 알아차릴 수 없는 움직임

### perceptive

- 발음: /pɚˈsɛptɪv/

- 뜻: 통찰력 있는, 지각이 예민한
- 어원적 연결: 미묘한 정보까지 잘 받아 잡아 이해함
- 사용 맥락: `a perceptive comment` = 통찰력 있는 말

### deceive

- 발음: /dɪˈsiv/
- 구성: `de-` = away + `ceive`
- 뜻: 속이다
- 어원적 연결: 상대의 마음을 잘못된 방향으로 잡아 끌어감
- 사용 맥락: `deceive the public` = 대중을 속이다

### deceptive

- 발음: /dɪˈsɛptɪv/
- 구성: `deceive` + `-ive`
- 뜻: 속이는, 기만적인; 겉보기와 다른
- 어원적 연결: 상대가 잘못 받아들이도록 마음을 엉뚱한 방향으로 잡아 끄는 성질
- 사용 맥락: `a deceptive appearance` = 속기 쉬운 겉모습

### intercept

- 발음: /ˌɪntɚˈsɛpt/
- 구성: `inter-` = between + `cept`
- 뜻: 가로채다, 차단하다
- 어원적 연결: 중간에서 잡음
- 사용 맥락: `intercept a pass` = 패스를 가로채다

### participate

- 발음: /pɑˈrtɪsəˌpeɪt/
- 구성: [[part]] + Latin `capere` = take
- 뜻: 참여하다
- 어원적 연결: 한 몫을 잡아 가짐
- 사용 맥락: `participate in a debate` = 토론에 참여하다

### occupy

- 발음: /ˈɑkjəˌpaɪ/

- 뜻: 차지하다, 점유하다
- 어원적 연결: 어떤 자리를 붙잡아 차지함
- 사용 맥락: `occupy a seat` = 자리를 차지하다

## 기억 포인트

`cap` 계열은 "잡고 받음" 이미지.

- `capture`: 밖의 대상을 잡음
- `accept`: 자기 쪽으로 받아들임
- `except`: 밖으로 잡아 뺌
- `concept`: 생각을 함께 잡아 개념화함
- `receive`: 자기 쪽으로 받아들임
- `receptive`: 잘 받아들이는
- `susceptible`: 영향이나 병을 받기 쉬움
- `perceive`: 감각으로 받아 알아차림
- `imperceptible`: 감각으로 받아 잡기 어려움
- `occupy`: 자리를 잡아 차지함

## 비교 단어

### prehend

- 뜻: 붙잡다, 이해하다
- 구분 포인트: [[prehend]]는 손으로 붙잡는 이미지가 강함. `cap`은 잡기와 받기, 수용까지 더 넓게 감

### capit

- 뜻: 머리, 꼭대기, 우두머리
- 구분 포인트: [[capit]]은 Latin `caput` = 머리 계열. `capital`, `captain`, `chapter`는 이쪽. `capture`, `accept`, `receive`의 `cap`은 Latin `capere` = 잡다 계열
