# tract-pull

- 상위 문서: [[tract]]
- 핵심 어원: 라틴어 `trahere` / `tract-` = 끌다, 당기다
- 핵심 이미지: 방향을 잡아 끌어당기거나 빼냄
- 관련 갈래: [[tract-trace]], [[tract-handle]]

## 단어들

### attract

- 구성: `ad-` = toward + `tract`
- 뜻: 끌어당기다, 매력을 느끼게 하다
- 어원적 연결: 자기 쪽으로 끌어옴
- 사용 맥락: `attract attention` = 관심을 끌다

### attraction

- 구성: `attract` + `-ion`
- 뜻: 끌림, 매력, 명소
- 어원적 연결: 사람이나 관심을 끌어당기는 힘
- 사용 맥락: `tourist attraction` = 관광 명소

### distract

- 구성: `dis-` = apart + `tract`
- 뜻: 산만하게 하다, 주의를 돌리다
- 어원적 연결: 주의를 여러 방향으로 끌어 흩뜨림
- 사용 맥락: `distract students` = 학생들의 주의를 산만하게 하다

### distraction

- 구성: `distract` + `-ion`
- 뜻: 산만하게 하는 것, 방해
- 어원적 연결: 주의를 다른 곳으로 끌어가는 것
- 사용 맥락: `avoid distractions` = 방해 요소를 피하다

### subtract

- 구성: `sub-` = under, away + `tract`
- 뜻: 빼다
- 어원적 연결: 전체에서 어떤 양을 끌어 빼냄
- 사용 맥락: `subtract five from ten` = 10에서 5를 빼다

### subtraction

- 구성: `subtract` + `-ion`
- 뜻: 뺄셈
- 어원적 연결: 수량을 끌어 빼는 계산
- 사용 맥락: `learn subtraction` = 뺄셈을 배우다

### extract

- 구성: `ex-` = out + `tract`
- 뜻: 끌어내다, 추출하다, 발췌
- 어원적 연결: 안에서 밖으로 끌어냄
- 사용 맥락: `extract information` = 정보를 뽑아내다

### extraction

- 구성: `extract` + `-ion`
- 뜻: 추출, 발췌, 발치
- 어원적 연결: 안에 있는 것을 밖으로 끌어내는 행위
- 사용 맥락: `tooth extraction` = 발치

### abstract

- 구성: `ab-` = away + `tract`
- 뜻: 추상적인, 요약, 추출하다
- 어원적 연결: 구체적 사례에서 핵심만 떨어져 끌어냄
- 비교: [[cresc]]의 `concrete` = 함께 자라 굳어 구체화
- 사용 맥락: `abstract ideas` = 추상적 개념

### abstraction

- 구성: `abstract` + `-ion`
- 뜻: 추상화
- 어원적 연결: 구체적인 것에서 핵심만 끌어내는 과정
- 사용 맥락: `a high level of abstraction` = 높은 수준의 추상화

### contract

- 구성: `con-` = together + `tract`
- 뜻: 계약, 수축하다
- 어원적 연결: 함께 끌어 묶음. 물체가 안쪽으로 끌려 줄어듦
- 비교: [[pact]] = 약속으로 묶은 합의
- 사용 맥락: `sign a contract` = 계약서에 서명하다

### contractor

- 구성: `contract` + `-or`
- 뜻: 계약자, 도급업자
- 어원적 연결: 계약으로 묶인 사람이나 회사
- 사용 맥락: `a building contractor` = 건설 도급업자

### retract

- 구성: `re-` = back + `tract`
- 뜻: 철회하다, 집어넣다
- 어원적 연결: 뒤로 끌어당김
- 사용 맥락: `retract a statement` = 발언을 철회하다

### retreat

- 구성: `re-` + `trahere`
- 뜻: 후퇴하다, 물러나다, 은거지
- 어원적 연결: 뒤로 끌려 물러남
- 사용 맥락: `retreat from battle` = 전투에서 후퇴하다

### retire

`retire`는 [[tract]] 계열 아님. [[tir]] 참고.

- 구성: `re-` + French `tirer`
- 뜻: 은퇴하다, 물러나다
- 구분 포인트: `retreat` = 라틴어 `trahere`, `retire` = French `tirer`

### protract

- 구성: `pro-` = forward + `tract`
- 뜻: 오래 끌다, 연장하다
- 어원적 연결: 앞으로 길게 끌고 감
- 사용 맥락: `protracted negotiations` = 장기화된 협상

### detract

- 구성: `de-` = down, away + `tract`
- 뜻: 가치를 떨어뜨리다
- 어원적 연결: 좋은 점에서 무언가를 끌어내려 빼앗음
- 사용 맥락: `detract from the value` = 가치를 떨어뜨리다

### traction

- 구성: `tract` + `-ion`
- 뜻: 견인력, 지지/관심 확보
- 어원적 연결: 끌어당기는 힘
- 사용 맥락: `gain traction` = 힘을 얻다, 주목받다

### tractor

- 구성: `tract` + `-or`
- 뜻: 트랙터
- 어원적 연결: 무거운 것을 끄는 기계
- 사용 맥락: `farm tractor` = 농장 트랙터
