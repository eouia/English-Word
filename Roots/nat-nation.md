# nat-nation

- 출발 단어: `nation`
- 핵심 어원: Latin `natio` = birth, origin, people
- 핵심 이미지: 같은 출생·출신으로 묶인 사람들의 집단
- 허브 문서: [[nat]]
- 관련 문서: [[popul]], [[dem]], [[polis]]

## 단어들

### nation

- 발음: /ˈneɪʃən/
- 구성: Latin `natio` = birth, people
- 뜻: 국가, 국민, 민족
- 어원적 연결: 같은 출신으로 태어난 사람들의 집단
- 사용 맥락: `a sovereign nation` = 주권 국가

### national

- 발음: /ˈnæʃənəl/
- 구성: `nation` + `-al`
- 뜻: 국가의, 국민의, 전국적인
- 어원적 연결: 한 nation 전체에 속함
- 사용 맥락: `a national election` = 전국 선거

### nationality

- 발음: /ˌnæʃəˈnæləti/
- 구성: `national` + `-ity`
- 뜻: 국적, 민족성
- 어원적 연결: 어떤 nation에 속한 상태
- 사용 맥락: `dual nationality` = 이중 국적

### nationalism

- 발음: /ˈnæʃənəˌlɪzəm/
- 구성: `national` + `-ism`
- 뜻: 민족주의, 국가주의
- 어원적 연결: 같은 출신·국가 집단에 대한 강한 소속 의식
- 사용 맥락: `rising nationalism` = 커지는 민족주의

### nationalist

- 발음: /ˈnæʃənələst/
- 구성: `nationalism` + `-ist`
- 뜻: 민족주의자, 국가주의자
- 어원적 연결: nation 중심의 정체성을 강조하는 사람
- 사용 맥락: `a nationalist movement` = 민족주의 운동

### international

- 발음: /ˌɪntɚˈnæʃənəl/
- 구성: `inter-` = between + `national`
- 뜻: 국제적인
- 어원적 연결: 여러 nation 사이에 있음
- 사용 맥락: `international trade` = 국제 무역

### multinational

- 발음: /ˌmʌˌltaɪˈnæʃənəl/
- 구성: `multi-` = many + `national`
- 뜻: 다국적의
- 어원적 연결: 여러 nation에 걸침
- 사용 맥락: `a multinational corporation` = 다국적 기업

## 기억 포인트

`nat-nation`은 "같이 태어난 집단" 갈래.

- `nation`: 출신 집단 → 국가
- `national`: 국가 전체에 속함
- `nationality`: 소속 국적
- `nationalism`: 국가·민족 중심 의식
- `international`: 국가들 사이
