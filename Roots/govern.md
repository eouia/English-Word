# govern

- 출발 단어: `government`
- 핵심 어원: 그리스어 `kybernan` = 배를 조종하다, 이끌다
- 중간 형태: 라틴어 `gubernare` = 조종하다, 다스리다
- 핵심 의미: 방향을 잡아 이끌고 통제함
- 관련 문서: [[reg]], [[crat]], [[duc]], [[rect]], modern `cyber-`
- 주의: `government`의 깊은 그림은 “국가라는 배를 조종함”.

## 출발 단어 분석

`government` = `govern` + `-ment`

- `govern`: 다스리다, 통제하다
- `-ment`: 행위나 결과
- 원래 이미지: 배의 키를 잡고 방향을 이끔
- 현대 의미: 정부, 통치, 지배 체계

## 단어들

### government

- 발음: /ˈgʌvɚmənt/
- 구성: `govern` + `-ment`
- 뜻: 정부, 통치, 지배 체계
- 어원적 연결: 사회나 국가의 방향을 잡아 이끄는 체계
- 사용 맥락: `local government` = 지방 정부

### govern

- 발음: /ˈgʌvɚn/
- 구성: Old French `governer` ← Latin `gubernare`
- 뜻: 다스리다, 통제하다, 지배하다
- 어원적 연결: 배를 조종하듯 방향을 잡아 이끎
- 사용 맥락: `govern a country` = 나라를 다스리다

### governor

- 발음: /ˈgʌvɚnɚ/
- 구성: `govern` + `-or`
- 뜻: 주지사, 통치자, 조절 장치
- 어원적 연결: 방향과 질서를 관리하는 사람이나 장치
- 사용 맥락: `state governor` = 주지사

### governance

- 발음: /ˈgʌvɚnəns/
- 구성: `govern` + `-ance`
- 뜻: 통치 방식, 거버넌스
- 어원적 연결: 조직이나 사회를 이끄는 방식
- 사용 맥락: `corporate governance` = 기업 지배 구조

### governmental

- 발음: /ˈgʌvɚˌmɛntəl/
- 구성: `government` + `-al`
- 뜻: 정부의, 통치상의
- 어원적 연결: 정부라는 통치 체계와 관련됨
- 사용 맥락: `governmental policy` = 정부 정책

### self-government

- 발음: /ˈsɛˈlfgʌvɚnmənt/
- 구성: `self` + `government`
- 뜻: 자치, 자기 통치
- 어원적 연결: 스스로 방향을 잡고 다스림
- 사용 맥락: `local self-government` = 지방 자치

### gubernatorial

- 발음: /guˌbɝnəˈtɔˌriəl/
- 구성: Latin `gubernator` = governor + `-ial`
- 뜻: 주지사의
- 어원적 연결: `governor`와 같은 라틴어 형태를 보존한 단어
- 사용 맥락: `a gubernatorial election` = 주지사 선거

### cybernetics

- 구성: Greek `kybernetes` = steersman, governor + `-ics`
- 뜻: 사이버네틱스, 제어·통신 이론
- 어원적 연결: 배를 조종하듯 시스템을 통제하고 피드백으로 이끎
- 사용 맥락: `the study of cybernetics` = 사이버네틱스 연구

### cybernetic

- 구성: `cybernetics`의 형용사형
- 뜻: 사이버네틱스의, 제어 시스템의
- 어원적 연결: 조종과 피드백 제어에 관련됨
- 사용 맥락: `a cybernetic model` = 사이버네틱스 모델

## 기억 포인트

- `govern`의 원래 그림은 “배를 조종하다”.
- `government`: 사회나 국가의 방향을 조종하는 체계.
- `governor`: 조종자, 통치자, 조절 장치.
- `gubernatorial`: 라틴어 `gubernator` 형태가 남은 단어.
- `cybernetics`: 같은 그리스어 뿌리에서 나온 “제어와 통신” 학문.

## 비교 단어

### crat

- [[crat]]는 권력, 지배 체제.
- `govern`은 권력 자체보다 방향을 잡고 운영하는 행위에 초점.

### duc

- [[duc]]는 이끌다, 끌고 가다.
- `govern`도 이끎이 있지만, 조종·통제·관리의 뉘앙스가 더 강함.

### cyber

- `cyber-`는 현대에는 컴퓨터·네트워크 공간을 뜻함.
- 깊은 뿌리는 `kybernan` = 조종하다와 연결되지만, 별도 현대 접두사 문서 후보.
