# rest

- 출발 단어: `rest`
- 핵심 어원 1: Old English `ræst` = 쉼, 휴식
- 핵심 어원 2: Old French `rester`, Latin `restare` = 남다, 뒤에 서 있다
- 핵심 이미지: 멈춰 있음 / 남아 있음
- 비교: [[sed]] = 앉다, 자리 잡다 계열

## 출발 단어 분석

`rest`는 두 갈래로 나눠 보기.

1. `rest` = 휴식  
   - Germanic 계열
   - "움직임을 멈춤, 쉼"

2. `rest` = 나머지  
   - Latin `re-` + `stare`
   - "뒤에 서 있음, 남아 있음"

따라서 `take a rest`와 `the rest of them`은 현대 철자는 같지만 어원은 다름.

## 단어들

### rest

- 구성: Old English `ræst`
- 뜻: 휴식, 쉬다
- 어원적 연결: 움직임이나 일을 멈추고 쉼
- 사용 맥락: `take a rest` = 휴식을 취하다

### restful

- 구성: `rest` + `-ful`
- 뜻: 편안한, 휴식을 주는
- 어원적 연결: 쉬는 느낌을 주는 상태
- 사용 맥락: `a restful place` = 편안한 장소

### restless

- 구성: `rest` + `-less`
- 뜻: 불안한, 가만히 있지 못하는
- 어원적 연결: 쉬거나 안정되지 못함
- 사용 맥락: `feel restless` = 안절부절못하다

### rest

- 구성: Old French `rester`, Latin `restare`
- 뜻: 나머지, 남다
- 어원적 연결: 다른 것이 사라진 뒤에도 남아 있음
- 사용 맥락: `the rest of the students` = 나머지 학생들

### restaurant

- 구성: French `restaurer` = 회복시키다
- 뜻: 식당
- 어원적 연결: 원래 "기력을 회복시키는 음식"에서 식당으로 확장
- 사용 맥락: `a local restaurant` = 동네 식당

### restore

- 구성: `re-` + Latin `staurare / restaurare`
- 뜻: 회복시키다, 복원하다
- 어원적 연결: 원래 상태로 되돌려 세움
- 사용 맥락: `restore confidence` = 신뢰를 회복하다

### restoration

- 구성: `restore` + `-ation`
- 뜻: 회복, 복원
- 어원적 연결: 원래 상태로 되돌리는 과정
- 사용 맥락: `building restoration` = 건물 복원

## 비교 단어

### reside

- 구성: `re-` + `sedere`
- 뜻: 거주하다
- 어원적 연결: [[sed]] 계열. 한곳에 앉아 머묾
- 구분 포인트: `reside` = 자리 잡고 삶, `rest` = 쉼 또는 남음

### remain

- 구성: `re-` + Latin `manere`
- 뜻: 남다, 머무르다
- 어원적 연결: [[manere]] 계열. `rest` = 나머지와 의미는 가깝지만 어원 다름
- 구분 포인트: `remain` = 남아 있음, `rest` = 남은 부분

## 기억 포인트

`rest`는 두 갈래.

- `rest`: 쉼
- `restful`: 쉬게 해 주는
- `restless`: 쉬지 못하는
- `the rest`: 남은 부분

주의:

- `take a rest`: Germanic 계열
- `the rest of them`: Latin `restare` 계열
