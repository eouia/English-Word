# plic-fold

- 출발 단어: `complicated`
- 핵심 어원: Latin `plicare` = 접다
- 핵심 이미지: 여러 겹으로 접히거나 같은 접힘을 다시 만듦
- 허브 문서: [[plic]]

## 단어들

### complicated

- 구성: `com-` = together + `plic` = fold
- 뜻: 복잡한
- 어원적 연결: 여러 겹으로 함께 접혀 얽힘
- 사용 맥락: `a complicated issue` = 복잡한 문제

### complicate

- 구성: `com-` = together + `plic`
- 뜻: 복잡하게 만들다
- 어원적 연결: 일을 여러 겹으로 접어 얽히게 함
- 사용 맥락: `complicate the process` = 과정을 복잡하게 만들다

### complication

- 구성: `complicate` + `-ion`
- 뜻: 복잡한 문제, 합병증
- 어원적 연결: 단순한 흐름에 여러 겹이 얽혀 생긴 문제
- 사용 맥락: `medical complications` = 의학적 합병증

### duplicate

- 구성: Latin `duplex` = twofold
- 뜻: 복제하다, 사본; 이중의
- 어원적 연결: 두 겹으로 접히거나 둘로 겹침
- 사용 맥락: `duplicate a file` = 파일을 복제하다

### duplicity

- 구성: Latin `duplex` = double
- 뜻: 이중성, 표리부동
- 어원적 연결: 겉과 속이 두 겹으로 나뉨
- 사용 맥락: `political duplicity` = 정치적 표리부동

### replicate

- 구성: `re-` = again + `plic`
- 뜻: 복제하다, 반복하다
- 어원적 연결: 같은 접힘·형태를 다시 만듦
- 사용 맥락: `replicate the results` = 결과를 재현하다

### reply

- 구성: Old French `replier` ← Latin `replicare`
- 뜻: 대답하다, 답장하다
- 어원적 연결: 말이나 주장에 다시 접어 맞받아침
- 사용 맥락: `reply to an email` = 이메일에 답장하다

### pliable

- 구성: Old French `plier` ← Latin `plicare`
- 뜻: 잘 휘는, 유연한
- 어원적 연결: 쉽게 접히거나 구부러짐
- 사용 맥락: `pliable material` = 잘 휘는 재료

### pliant

- 구성: `ply/pli` 계열
- 뜻: 잘 휘는, 순응적인
- 어원적 연결: 물리적으로나 태도상 쉽게 굽힘
- 사용 맥락: `a pliant attitude` = 순응적인 태도

## 기억 포인트

`plic-fold`는 "접힘과 겹침" 갈래.

- `complicated`: 여러 겹으로 얽힘
- `duplicate`: 두 겹으로 만듦
- `replicate`: 다시 만듦
- `reply`: 다시 접어 응답함
- `pliable`: 잘 접힘
