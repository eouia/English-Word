# spers

- 출발 단어: `disperse`
- 핵심 의미: 뿌리다, 흩다, 흩어지다
- 관련 형태: Latin `spargere` = scatter, sprinkle; `spersus` = scattered
- 관련 문서: [[dis]]

## 핵심 이미지

`spers` 계열의 바탕은 "작은 것들을 여기저기 뿌려 흩음".

여러 방향으로 흩어지면 `disperse`.
흩어진 상태나 분포는 `dispersion`.
드문드문 흩어져 있으면 `sparse`.
사이사이에 흩어 넣으면 `intersperse`.

## 단어들

### disperse

- 구성: Latin `dis-` = apart + `spargere/spers-` = scatter
- 뜻: 흩어지다, 흩어지게 하다, 분산시키다
- 연결: 한곳에 모인 것을 여러 방향으로 뿌려 보냄
- 사용 맥락: `disperse the crowd` = 군중을 해산시키다

### dispersed

- 구성: `disperse` + `-ed`
- 뜻: 흩어진, 분산된
- 연결: 이미 여러 곳으로 퍼져 있음
- 사용 맥락: `a dispersed population` = 흩어져 사는 인구

### dispersion

- 구성: `disperse` + `-ion`
- 뜻: 분산, 확산, 흩어짐
- 연결: 퍼져 흩어지는 과정이나 상태
- 사용 맥락: `the dispersion of seeds` = 씨앗의 산포

### dispersal

- 구성: `disperse` + `-al`
- 뜻: 분산, 확산, 해산
- 연결: 흩어져 나가는 행위나 결과
- 사용 맥락: `seed dispersal` = 씨앗 퍼뜨림

### sparse

- 구성: Latin `sparsus` = scattered
- 뜻: 드문드문한, 희박한
- 연결: 여기저기 흩어져 있어 밀도가 낮음
- 사용 맥락: `sparse vegetation` = 드문드문한 식생

### sparsely

- 구성: `sparse` + `-ly`
- 뜻: 드문드문, 희박하게
- 연결: 서로 멀리 떨어져 흩어진 방식
- 사용 맥락: `a sparsely populated area` = 인구가 희박한 지역

### intersperse

- 구성: `inter-` = between + `sperse`
- 뜻: 사이사이에 흩어 넣다, 섞어 넣다
- 연결: 어떤 것들 사이에 다른 것을 뿌리듯 배치함
- 사용 맥락: `intersperse examples throughout the essay` = 글 전체에 예시를 사이사이에 넣다

### aspersion

- 구성: Latin `ad-` = to + `spargere` = sprinkle
- 뜻: 비방, 중상
- 연결: 원래는 뿌림의 뜻. 현대 영어에서는 나쁜 말을 뿌려 명예를 더럽히는 이미지.
- 사용 맥락: `cast aspersions on someone` = 누군가를 비방하다

## 비교 단어

`spread`는 Germanic 계열의 일반 동사.
의미는 비슷하지만 `disperse`, `sparse`, `intersperse`는 Latin `spargere/spers-` 계열로 묶는 편이 좋음.

`sprinkle`은 "뿌리다"라는 뜻은 비슷하지만 별도 영어 계열.
학습상 `disperse`는 "여러 방향으로 흩어짐", `sparse`는 "흩어져 밀도가 낮음"으로 구분.
