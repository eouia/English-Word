# matr

- 출발 단어: `maternal`
- 핵심 어원: Latin `mater` = 어머니
- 관련 형태: `matr-`, `mater-`
- 핵심 이미지: 어머니와 어머니 쪽 계통, 무언가를 낳는 바탕
- 관련 문서: [[patr]], [[frater]], [[nat]]
- 주의: `matrix`는 `mater`에서 온 "자궁, 모체" 갈래라 함께 둠. `matriculate`는 `matricula` = 명부, 등록부를 거친 행정어라 핵심 단어가 아니라 주의할 형태로만 기억.

## 단어들

### maternal

- 발음: /məˈtɝnəl/
- 구성: Latin `mater` + `-nal`
- 뜻: 어머니의, 모계의
- 어원적 연결: 어머니 쪽에 속함
- 사용 맥락: `maternal care` = 모성적 돌봄

### maternity

- 발음: /məˈtɝnɪti/
- 구성: `mater` + `-ity`
- 뜻: 모성, 출산
- 어원적 연결: 어머니가 되는 상태
- 사용 맥락: `maternity leave` = 출산 휴가

### matriarch

- 발음: /ˈmeɪtriˌɑrk/
- 구성: `matri-` = mother + Greek `archein` = rule
- 뜻: 여가장, 모계 지도자
- 어원적 연결: 어머니처럼 집단을 이끄는 사람
- 사용 맥락: `family matriarch` = 집안의 여가장

### matriarchy

- 발음: /ˈmeɪtriˌɑrki/
- 구성: `matriarch` + `-y`
- 뜻: 모계 사회, 모권제
- 어원적 연결: 어머니·여성 중심의 사회 구조
- 사용 맥락: 인류학·사회학 맥락에서 쓰임

### matrix

- 발음: /ˈmeɪtrɪks/
- 구성: Latin `matrix` = womb
- 뜻: 모체, 행렬, 기반
- 어원적 연결: 무언가가 생겨나는 어머니 같은 바탕
- 사용 맥락: `a matrix of data` = 데이터 행렬

## 기억 포인트

`matr` 계열은 "어머니와 생성의 바탕" 이미지.

- `maternal`: 어머니의
- `maternity`: 모성, 출산
- `matriarch`: 여가장
- `matrix`: 무언가가 생겨나는 바탕

## 주의할 형태

### matriculate

- 발음: /məˈtrɪkjəleɪt/
- 뜻: 입학하다, 등록하다
- 구분 포인트: Latin `matricula` = register에서 온 행정어. `matrix`를 거쳐 `mater`와 역사적으로 연결되지만, 현대 학습에서는 "어머니" 이미지보다 "명부에 등록함"으로 기억하는 편이 안전.

## 비교 단어

### patr

- 뜻: 아버지
- 구분 포인트: [[patr]]는 아버지와 부계. `matr`는 어머니와 모계.
