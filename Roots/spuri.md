# spuri

- 출발 단어: `spurious`
- 핵심 어원: Latin `spurius` = illegitimate, false
- 핵심 이미지: 출처나 근거가 진짜가 아닌 것
- 관련 문서: 없음
- 주의: `spurious`는 `spur` = 박차, 자극하다와 어원이 다름.

## 출발 단어 분석

`spurious` ← Latin `spurius`

- `spurius`: illegitimate, false
- 현대 의미: 가짜의, 허위의, 겉보기만 그럴듯한

원래는 "정당한 출생이 아닌"이라는 의미에서 출발. 이후 "진짜 출처나 근거가 없음" → "가짜의, 허위의, 타당하지 않은" 뜻으로 넓어짐.

## 단어들

### spurious

- 구성: Latin `spurius` = illegitimate, false
- 뜻: 가짜의, 허위의, 타당하지 않은
- 어원적 연결: 진짜 출처나 근거를 갖지 못함
- 사용 맥락: `a spurious claim` = 근거 없는 주장, `a spurious correlation` = 가짜 상관관계

### spuriously

- 구성: `spurious` + `-ly`
- 뜻: 허위로, 겉보기만 그럴듯하게
- 어원적 연결: 진짜 근거 없이 그렇게 보이게 함
- 사용 맥락: `spuriously linked to the disease` = 그 병과 허위로 연결됨

### spuriousness

- 구성: `spurious` + `-ness`
- 뜻: 허위성, 가짜임
- 어원적 연결: 진짜 근거가 없는 성질
- 사용 맥락: `the spuriousness of the evidence` = 증거의 허위성

### spurious correlation

- 구성: `spurious` + `correlation`
- 뜻: 허위 상관관계, 겉보기 상관
- 어원적 연결: 두 현상이 함께 움직이는 듯 보이지만 실제 인과나 타당한 관련이 없음
- 사용 맥락: statistics, research 문맥에서 자주 보임.

## 기억 포인트

`spuri` 계열은 "진짜 출처나 근거가 없음" 이미지.

- `spurious`: 가짜의, 허위의
- `spuriously`: 허위로
- `spuriousness`: 허위성
- `spurious correlation`: 겉보기만 있는 상관관계

## 비교 단어

### false

- 뜻: 거짓의, 틀린
- 구분 포인트: `false`는 가장 넓은 "참이 아님". `spurious`는 출처·근거·논리적 연결이 가짜라는 느낌이 강함.

### fake

- 뜻: 가짜의, 위조의
- 구분 포인트: `fake`는 사람이 일부러 만든 모조품의 느낌이 강함. `spurious`는 주장, 증거, 상관관계처럼 추상적인 것에도 자주 씀.

### pseudo-

- 뜻: 가짜의, 유사한
- 구분 포인트: `pseudo-`는 접두사로 "겉만 그런"의 뜻. `spurious`는 형용사로 주장·이유·관계가 타당하지 않음을 말함.
