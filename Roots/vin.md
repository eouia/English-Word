# vin

- 출발 단어: `vine`
- 핵심 의미: 포도나무, 포도주
- 관련 형태: Latin `vinea` = vineyard, vine; Latin `vinum` = wine
- 관련 문서: [[gard]]

## 핵심 이미지

`vin` 계열의 바탕은 "포도나무와 포도주".

포도 덩굴은 `vine`.
포도나무를 기르는 밭은 `vineyard`.
포도로 만든 술은 `wine`.
특정 해의 포도 수확과 와인은 `vintage`.
시어진 포도주는 `vinegar`.

## 단어들

### vine

- 구성: Latin `vinea` = vine, vineyard
- 뜻: 덩굴식물, 포도나무
- 연결: 포도나무처럼 감고 올라가는 식물
- 사용 맥락: `grape vines` = 포도나무 덩굴

### vineyard

- 구성: `vine` + `yard`
- 뜻: 포도밭
- 연결: 포도나무를 기르는 울타리 친 땅
- 사용 맥락: `a vineyard in California` = 캘리포니아의 포도밭

### wine

- 구성: Latin `vinum` → Old English `win`
- 뜻: 포도주, 와인
- 연결: 포도로 만든 술
- 사용 맥락: `red wine` = 레드 와인

### vintage

- 구성: Old French `vendange` = grape harvest, from Latin `vindemia`
- 뜻: 특정 연도산 와인; 빈티지의, 오래되어 가치 있는
- 연결: 원래는 포도 수확, 특히 어느 해에 수확한 포도로 만든 와인
- 사용 맥락: `a vintage car` = 오래되어 가치 있는 클래식 자동차

### viniculture

- 구성: Latin `vinum` = wine + `culture`
- 뜻: 포도 재배, 와인 생산
- 연결: 포도나무를 길러 포도주를 만드는 일
- 사용 맥락: 전문성이 조금 있는 농업·와인 문맥.

### viticulture

- 구성: Latin `vitis` = vine + `culture`
- 뜻: 포도 재배
- 연결: 포도나무를 기르는 농업 기술
- 사용 맥락: `viticulture and winemaking` = 포도 재배와 와인 양조

### vintner

- 구성: Old French `vintner`, from `vin` = wine
- 뜻: 와인 상인, 와인 제조업자
- 연결: 와인을 만들거나 파는 사람
- 사용 맥락: 와인 산업 문맥.

### vinegar

- 구성: Old French `vin aigre` = sour wine
- 뜻: 식초
- 연결: 시어진 포도주에서 나온 말
- 사용 맥락: `apple cider vinegar` = 사과 식초

## 비교 단어

`divine`은 `deus/divus` = 신 계열로, `vine`과 어원이 다름.
철자 안에 `vine`이 보이지만 포도나무와 연결하지 않음.

`vinyl`은 화학 명명에서 쓰이는 말.
역사적으로 Latin `vinum`과 이름이 닿아 있지만, 현대 어휘 학습에서는 포도나무·와인 계열 핵심 단어로 보지 않는 편이 좋음.
