# fig

- 출발 단어: `figure`
- 핵심 어원: Latin `figura` / `fingere` = shape, form, devise
- 핵심 이미지: 빚어 만든 모양, 머릿속에 그린 형상
- 관련 어근: [[form]], [[fac]], [[res]]

## 출발 단어 분석

`figure`는 Old French `figure`를 거쳐 Latin `figura`에서 온 말.

`figura`는 Latin `fingere` = shape, form, devise에서 나옴.

핵심은 "빚어 만든 모양"이며, 여기서 "형태, 인물, 숫자, 도형, 생각해 내다"까지 확장됨.

## 단어들

### figure

- 구성: Latin `figura`
- 의미: 모습, 인물, 숫자, 도형, 이해하다
- 어원적 연결: 눈앞이나 머릿속에 잡히는 모양
- 사용 맥락: `a public figure` = 공적 인물, `figure out the answer` = 답을 알아내다

### figurative

- 구성: `figure` + `-ative`
- 의미: 비유적인, 상징적인
- 어원적 연결: 직접 뜻이 아니라 머릿속에 그린 형상으로 말함
- 사용 맥락: `figurative language` = 비유적 표현

### figuratively

- 구성: `figurative` + `-ly`
- 의미: 비유적으로
- 어원적 연결: 문자 그대로가 아니라 형상으로 빗대어 말함
- 사용 맥락: `figuratively speaking` = 비유적으로 말하자면

### configure

- 구성: `con-` = together + `figure`
- 의미: 구성하다, 설정하다
- 어원적 연결: 여러 요소를 함께 모양 잡음
- 사용 맥락: `configure the system` = 시스템을 설정하다

### configuration

- 구성: `configure` + `-ation`
- 의미: 구성, 배열, 설정
- 어원적 연결: 함께 짜여 만들어진 형태
- 사용 맥락: `network configuration` = 네트워크 설정

### fiction

- 구성: Latin `fictio` < `fingere`
- 의미: 소설, 허구
- 어원적 연결: 실제가 아니라 지어 만든 이야기
- 사용 맥락: `science fiction` = 과학 소설

### fictitious

- 구성: Latin `ficticius` < `fingere`
- 의미: 허구의, 가공의
- 어원적 연결: 실제가 아니라 만들어 낸 것
- 사용 맥락: `a fictitious name` = 가명

### feign

- 구성: Old French `feindre` < Latin `fingere`
- 의미: 가장하다, 꾸미다
- 어원적 연결: 실제가 아닌 모습을 만들어 보임
- 사용 맥락: `feign ignorance` = 모르는 척하다

### figment

- 구성: Latin `figmentum` < `fingere`
- 의미: 꾸며 낸 것, 허구
- 어원적 연결: 마음이 빚어 만든 상상
- 사용 맥락: `a figment of imagination` = 상상의 산물

### effigy

- 구성: Latin `effigies` < `ex-` + `fingere`
- 의미: 인물상, 형상
- 어원적 연결: 밖으로 빚어 낸 모습
- 사용 맥락: `burn someone in effigy` = 누군가의 인형을 태우다

## 기억 포인트

- `figure`: 모양, 인물, 숫자
- `figurative`: 형상으로 빗댄
- `configure`: 함께 모양 잡음
- `fiction`: 지어 만든 이야기
- `feign`: 꾸며 보임
- `figment`: 마음이 빚어 낸 허구
- `effigy`: 빚어 낸 형상

## 비교 단어

- [[form]]
  - 핵심: 모양, 형식
  - 구분 포인트: `form`은 형태 일반. `fig`는 빚어 만들거나 머릿속에 그린 형상

- [[fac]]
  - 핵심: 만들다, 하다
  - 구분 포인트: `facere`는 만들고 행함. `fingere/figura`는 빚어 모양을 내거나 꾸며 냄

- [[res]]
  - 핵심: 실제 사물, 일
  - 구분 포인트: `fiction/figment`는 지어 낸 것. `real`은 실제 사물로 있는 것
