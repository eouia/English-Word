# mantle

- 출발 단어: `dismantle`
- 핵심 어원: Old French `mantel` = 망토, 덮개
- 핵심 의미: 덮개를 씌우거나 벗김, 방어물을 제거함
- 관련 문서: [[struct]], [[ex]], [[em]]
- 주의: `dismantle`은 영어에서 `dis-` + `mantle`로 새로 만든 말이 아니라 French `desmanteler`에서 들어온 말.

## 출발 단어 분석

`dismantle` ← French `desmanteler`

- `des-`: away, off
- `manteler`: 망토를 씌우다, 덮다
- `mantel`: 망토, 덮개
- 원래 이미지: 요새의 방어벽·장비를 벗겨 냄
- 현대 의미: 해체하다, 분해하다, 철거하다

## 단어들

### dismantle

- 구성: French `des-` = off + `manteler` = cloak
- 뜻: 해체하다, 분해하다, 철거하다
- 어원적 연결: 덮개나 방어물을 벗겨 내어 기능을 없앰
- 사용 맥락: `dismantle a machine` = 기계를 분해하다

### dismantling

- 구성: `dismantle` + `-ing`
- 뜻: 해체, 철거, 분해
- 어원적 연결: 장비나 제도를 하나씩 벗겨 내는 과정
- 사용 맥락: `the dismantling of old systems` = 낡은 제도의 해체

### dismantled

- 구성: `dismantle` + `-ed`
- 뜻: 해체된, 분해된
- 어원적 연결: 덮개와 구조물이 벗겨져 기능을 잃은 상태
- 사용 맥락: `a dismantled factory` = 해체된 공장

### mantle

- 구성: Old French `mantel` = cloak
- 뜻: 망토, 덮개; 지위나 책임
- 어원적 연결: 몸이나 사물을 덮는 것 → 맡아 걸치는 역할
- 사용 맥락: `take up the mantle` = 역할이나 책임을 이어받다

### mantel

- 구성: Middle English `mantel` = mantle
- 뜻: 벽난로 선반
- 어원적 연결: 원래 벽난로 위를 덮거나 받치는 구조물
- 사용 맥락: `a clock on the mantel` = 벽난로 선반 위의 시계

### mantelpiece

- 구성: `mantel` + `piece`
- 뜻: 벽난로 선반
- 어원적 연결: 벽난로 위를 덮거나 장식하는 부분
- 사용 맥락: `photos on the mantelpiece` = 벽난로 선반 위의 사진들

### mantling

- 구성: `mantle` + `-ing`
- 뜻: 덮음, 덮개; 문장학의 장식 천
- 어원적 연결: 망토처럼 덮는 것
- 사용 맥락: heraldry context

## 기억 포인트

- `mantle`: 망토, 덮개.
- `dismantle`: 덮개·방어물을 벗겨 해체함.
- `mantel`: 벽난로 위 선반.
- `take up the mantle`: 망토를 걸치듯 역할을 이어받음.

## 비교 단어

### disassemble

- `disassemble`은 조립된 부품을 풀어 분해함.
- `dismantle`은 원래 방어물·장비를 벗겨 기능을 없애는 느낌이 강함.

### struct

- [[struct]]는 쌓고 세운 구조.
- `dismantle`은 세워진 구조를 벗기고 해체함.

### mantle vs mantel

- `mantle`: 망토, 덮개, 책임.
- `mantel`: 벽난로 선반.
- 둘은 역사적으로 가까우나 현대 철자와 용법은 구분.
