# junct

- 출발 단어: `join`
- 핵심 어원: Latin `iungere` / `iunctus` = 잇다, 결합하다, 멍에를 씌우다
- 깊은 어근: PIE `*yeug-` = join, yoke
- 핵심 이미지: 따로 있던 것을 하나로 이어 묶음
- 관련 문서: [[uni]], [[con]], [[sym]]
- 주의: `joint`의 신체 관절, `conjunction`의 문법 접속사는 모두 "이어 붙임" 이미지.

## 출발 단어 분석

`join` ← Old French `joindre` ← Latin `iungere`

- `iungere`: 잇다, 연결하다
- `iunctus`: 이어진, 결합된
- 원래 이미지: 둘을 멍에나 연결부로 묶음
- 현대 의미: 합류하다, 연결하다

## 단어들

### join

- 발음: /ˈdʒɔɪn/
- 구성: Old French `joindre`
- 뜻: 합류하다, 연결하다
- 어원적 연결: 따로 있던 것을 이어 붙임
- 사용 맥락: `join a club` = 동아리에 가입하다

### joint

- 발음: /ˈdʒɔɪnt/
- 구성: Old French `joint` ← Latin `iunctus`
- 뜻: 관절, 공동의, 연결부
- 어원적 연결: 두 부분이 만나는 연결 지점
- 사용 맥락: `joint effort` = 공동 노력

### junction

- 발음: /ˈdʒʌŋkʃən/
- 구성: Latin `iunctio`
- 뜻: 교차점, 접합점
- 어원적 연결: 길이나 선이 만나는 지점
- 사용 맥락: `a road junction` = 도로 교차점

### juncture

- 발음: /ˈdʒʌŋktʃɚ/
- 구성: Latin `iunctura`
- 뜻: 접합, 시점, 국면
- 어원적 연결: 일이 서로 맞물리는 지점
- 사용 맥락: `at this juncture` = 이 시점에서

### conjunction

- 발음: /kəˈndʒʌŋkʃən/
- 구성: `con-` = together + `junct`
- 뜻: 결합, 접속사
- 어원적 연결: 단어·구·절을 함께 이어 줌
- 사용 맥락: `and is a conjunction` = and는 접속사다

### conjunctive

- 구성: `conjunction` 계열
- 뜻: 결합하는, 접속의
- 어원적 연결: 둘 이상을 이어 주는 성질
- 사용 맥락: `conjunctive adverbs` = 접속 부사

### adjunct

- 발음: /ˈæˌdʒʌŋkt/
- 구성: `ad-` = to + `junct`
- 뜻: 부속물, 부가어
- 어원적 연결: 본체에 붙어 더해진 것
- 사용 맥락: `an adjunct professor` = 겸임 교수

### adjoin

- 발음: /əˈdʒɔɪn/
- 구성: `ad-` = to + `join`
- 뜻: 인접하다, 붙어 있다
- 어원적 연결: 옆에 이어 붙음
- 사용 맥락: `adjoining rooms` = 붙어 있는 방들

### conjoin

- 발음: /ˌkɑˈndʒɔɪn/
- 구성: `con-` = together + `join`
- 뜻: 결합하다
- 어원적 연결: 함께 이어 붙임
- 사용 맥락: `conjoined ideas` = 결합된 생각들

### disjointed

- 발음: /dɪˈsdʒɔɪntɪd/
- 구성: `dis-` = apart + `joint`
- 뜻: 앞뒤가 안 맞는, 분리된
- 어원적 연결: 이어져야 할 연결부가 떨어짐
- 사용 맥락: `a disjointed argument` = 앞뒤가 안 맞는 주장

### rejoin

- 발음: /riˈdʒɔɪn/
- 구성: `re-` = again/back + `join`
- 뜻: 다시 합류하다, 대답하다
- 어원적 연결: 다시 이어 붙거나 말로 되받아 연결함
- 사용 맥락: `rejoin the group` = 무리에 다시 합류하다

### rejoinder

- 발음: /rɪˈdʒɔɪndɚ/
- 구성: `rejoin` + `-er`
- 뜻: 반박, 응답
- 어원적 연결: 앞말에 다시 이어 붙인 대답
- 사용 맥락: `a sharp rejoinder` = 날카로운 반박

### subjunctive

- 구성: `sub-` = under + `junct`
- 뜻: 접속법
- 어원적 연결: 원래 "아래에 붙여진, 종속되어 이어진" 문법 형태
- 사용 맥락: `the subjunctive mood` = 접속법

## 기억 포인트

`junct` 계열은 "이어 붙임" 이미지.

- `join`: 잇다
- `joint`: 연결부
- `junction`: 만나는 지점
- `conjunction`: 함께 이어 줌
- `adjunct`: 붙어 더해진 것
- `subjunctive`: 종속적으로 붙은 문법 형태

## 비교 단어

### uni

- 뜻: 하나
- 구분 포인트: [[uni]]는 하나인 상태. `junct`는 따로 있던 것을 이어 하나로 만드는 동작과 지점.
