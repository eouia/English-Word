# chron

- 출발 단어: `chronology`
- 핵심 어원: Greek `chronos` = 시간
- 핵심 이미지: 시간의 흐름과 순서
- 관련 문서: [[tempor]], [[sequ]]

## 단어들

### chronology

- 구성: `chronos` = time + [[logos]] = study
- 뜻: 연대기, 연대순 배열
- 어원적 연결: 시간 순서를 따져 설명함
- 사용 맥락: `establish a chronology` = 연대순을 세우다

### chronological

- 구성: `chronology` + `-ical`
- 뜻: 연대순의
- 어원적 연결: 시간 순서대로 배열됨
- 사용 맥락: `chronological order` = 시간순

### chronic

- 구성: Greek `chronos`
- 뜻: 만성적인, 오래 지속되는
- 어원적 연결: 오랜 시간 계속됨
- 사용 맥락: `chronic pain` = 만성 통증

### chronicle

- 구성: Greek `chronos`
- 뜻: 연대기, 기록하다
- 어원적 연결: 시간 순서에 따라 사건을 기록함
- 사용 맥락: `chronicle events` = 사건을 기록하다

### synchronize

- 구성: `syn-` = together + `chronos`
- 뜻: 동시에 맞추다, 동기화하다
- 어원적 연결: 시간을 함께 맞춤
- 사용 맥락: `synchronize watches` = 시계를 맞추다

### anachronism

- 구성: `ana-` = back, against + `chronos`
- 뜻: 시대착오
- 어원적 연결: 시간 위치가 맞지 않음
- 사용 맥락: `a historical anachronism` = 역사적 시대착오

## 기억 포인트

`chron` 계열은 "시간" 이미지.

- `chronology`: 시간 순서
- `chronic`: 오래 지속되는
- `chronicle`: 시간순 기록
- `synchronize`: 시간을 맞춤
- `anachronism`: 시간에 맞지 않음

## 비교 단어

### tempor

- 뜻: 시간, 기간
- 구분 포인트: [[tempor]]는 라틴어 시간 계열. `chron`은 그리스어 시간 계열.
