# sci

- 출발 단어: `science`
- 핵심 어원: Latin `scire` = know
- 핵심 이미지: 알고 구별하여 체계적 지식으로 만듦
- 관련 문서: [[cogn]], [[know]], [[soph]]
- 주의: `sci`는 현대 영어에서 보통 `science`, `conscious`, `conscience`, `omniscient`처럼 형태가 변해 나타남.

## 출발 단어 분석

`science` ← Latin `scientia`

- `scire`: to know
- `scientia`: knowledge, knowing
- `science`: 지식 → 체계적 지식 → 과학

`sci` 계열은 단순히 "안다"보다, 사실을 알고 구별하고 체계화하는 이미지가 강함.

## 단어들

### science

- 발음: /ˈsaɪəns/
- 구성: Latin `scientia` = knowledge
- 뜻: 과학, 체계적 지식
- 어원적 연결: 알고 있는 내용을 체계로 정리한 지식
- 사용 맥락: `natural science` = 자연과학

### scientific

- 발음: /ˌsaɪəˈntɪfɪk/
- 구성: `science` + `-ific`
- 뜻: 과학의, 과학적인
- 어원적 연결: 지식을 검증하고 체계화하는 방식
- 사용 맥락: `scientific evidence` = 과학적 증거

### scientist

- 발음: /ˈsaɪəntɪst/
- 구성: `science` + `-ist`
- 뜻: 과학자
- 어원적 연결: 체계적 지식을 탐구하는 사람
- 사용 맥락: `a research scientist` = 연구 과학자

### scientifically

- 발음: /ˌsaɪəˈntɪfɪkəli/
- 구성: `scientific` + `-ally`
- 뜻: 과학적으로
- 어원적 연결: 검증 가능한 지식의 방식으로
- 사용 맥락: `scientifically proven` = 과학적으로 입증된

### conscious

- 발음: /ˈkɑnʃəs/
- 구성: Latin `con-` = with, together + `scire` = know
- 뜻: 의식하고 있는, 의식이 있는
- 어원적 연결: 스스로 또는 함께 알고 있는 상태
- 사용 맥락: `be conscious of danger` = 위험을 의식하다

### consciousness

- 발음: /ˈkɑnʃəsnəs/
- 구성: `conscious` + `-ness`
- 뜻: 의식, 자각
- 어원적 연결: 알고 있음을 느끼는 정신 상태
- 사용 맥락: `lose consciousness` = 의식을 잃다

### subconscious

- 발음: /səˈbkɑnʃəs/
- 구성: `sub-` = under + `conscious`
- 뜻: 잠재의식의, 어렴풋이 의식하는
- 어원적 연결: 뚜렷한 의식 아래에 있는 앎
- 사용 맥락: `a subconscious fear` = 잠재의식 속 두려움

### conscience

- 발음: /ˈkɑnʃəns/
- 구성: Latin `conscientia` = knowledge within oneself
- 뜻: 양심
- 어원적 연결: 자기 안에서 옳고 그름을 알고 있는 감각
- 사용 맥락: `a guilty conscience` = 죄책감을 느끼는 양심

### conscientious

- 발음: /ˌkɑnʃiˈɛnʃəs/
- 구성: `conscience` + `-ious`
- 뜻: 양심적인, 성실한, 꼼꼼한
- 어원적 연결: 자기 안의 기준을 알고 책임 있게 행동함
- 사용 맥락: `a conscientious worker` = 성실하고 꼼꼼한 직원

### omniscient

- 발음: /ɑˈmnɪʃənt/
- 구성: `omni-` = all + Latin `scient-` = knowing
- 뜻: 전지의, 모든 것을 아는
- 어원적 연결: 모든 것을 알고 있음
- 사용 맥락: `an omniscient narrator` = 전지적 화자

### prescient

- 발음: /ˈprɛsiənt/
- 구성: `pre-` = before + Latin `scient-` = knowing
- 뜻: 예지력 있는, 앞일을 아는 듯한
- 어원적 연결: 일이 일어나기 전에 알고 있음
- 사용 맥락: `a prescient warning` = 앞일을 내다본 경고

### nescience

- 구성: Latin `ne-` = not + `scire` = know
- 뜻: 무지, 알지 못함
- 어원적 연결: 지식이 없는 상태
- 사용 맥락: 시험 빈도는 낮지만 `omniscient`, `prescient`와 구조 비교에 유용.

## 기억 포인트

`sci` 계열은 "앎을 체계화하거나 의식함" 이미지.

- `science`: 체계적 지식, 과학
- `scientific`: 과학적인
- `conscious`: 의식하는
- `conscience`: 자기 안에서 아는 기준 → 양심
- `conscientious`: 양심적이고 꼼꼼한
- `omniscient`: 모든 것을 아는
- `prescient`: 미리 아는 듯한

## 비교 단어

### cogn

- 뜻: 알다, 알아보다, 인식하다
- 구분 포인트: [[cogn]]은 어떤 대상을 알아보거나 인식하는 과정. `sci`는 알고 있는 내용이 지식·의식·판단 체계로 굳어진 느낌.

### know

- 뜻: 알다
- 구분 포인트: [[know]]는 Germanic 기본어. `sci`는 Latin 계열이라 학문·의식·양심 같은 추상어에 많이 남음.

### soph

- 뜻: 지혜, 기술
- 구분 포인트: [[soph]]는 지혜와 기교, 판단의 세련됨. `sci`는 사실을 알고 체계화한 지식.
