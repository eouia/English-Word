# medit

- 출발 단어: `premeditated`
- 핵심 어원: Latin `meditari` = think over, reflect, consider
- 핵심 이미지: 마음속으로 되풀이해 생각하고 준비함
- 관련 문서: [[med]], [[medi]]

## 단어들

### premeditated

- 발음: /priˈmɛdəˌteɪtɪd/
- 구성: `pre-` = before + `meditate` + `-ed`
- 뜻: 사전에 계획된, 계획적인
- 어원적 연결: 행동하기 전에 미리 깊이 생각하고 준비함
- 사용 맥락: `premeditated murder` = 계획 살인

### premeditate

- 발음: /priˈmɛdəˌteɪt/
- 구성: Latin `prae-` = before + `meditari` = think over
- 뜻: 미리 계획하다, 사전에 숙고하다
- 어원적 연결: 실행 전에 먼저 생각해 둠
- 사용 맥락: 범죄나 의도적 행동을 말할 때 자주 사용

### premeditation

- 발음: /priˌmɛdəˈteɪʃən/
- 구성: `premeditate` + `-ion`
- 뜻: 사전 계획, 예모
- 어원적 연결: 행동 전에 마음속으로 계획하고 검토하는 행위
- 사용 맥락: `evidence of premeditation` = 계획성의 증거

### meditate

- 발음: /ˈmɛdəˌteɪt/
- 구성: Latin `meditari` = think over, reflect
- 뜻: 명상하다, 깊이 생각하다
- 어원적 연결: 마음속으로 되풀이해 숙고함
- 사용 맥락: `meditate on a problem` = 문제를 깊이 생각하다

### meditation

- 발음: /ˌmɛdəˈteɪʃən/
- 구성: `meditate` + `-ion`
- 뜻: 명상, 숙고
- 어원적 연결: 마음을 모아 생각하거나 성찰하는 행위
- 사용 맥락: `daily meditation` = 매일 하는 명상

### meditative

- 발음: /ˈmɛdəˌteɪtɪv/
- 구성: `meditate` + `-ive`
- 뜻: 명상적인, 사색적인
- 어원적 연결: 깊이 생각하는 성질
- 사용 맥락: `a meditative mood` = 사색적인 분위기

### meditated

- 구성: `meditate` + `-ed`
- 뜻: 숙고한, 명상한
- 어원적 연결: 마음속으로 생각해 본 상태
- 사용 맥락: `a meditated response` = 숙고한 반응

## 기억 포인트

`medit` 계열은 "생각해 보다, 숙고하다" 이미지.

- `meditate`: 깊이 생각하다, 명상하다
- `meditation`: 명상, 숙고
- `premeditate`: 미리 생각하고 계획하다
- `premeditated`: 사전에 계획된

## 비교 단어

### med

- 뜻: 치료하다, 의학
- 구분 포인트: [[med]]는 medicine 계열. `medit`는 생각·숙고 계열. 형태가 비슷하지만 문서에서 분리.

### medi

- 뜻: 가운데, 사이, 중간
- 구분 포인트: [[medi]]는 middle 계열. `meditate`와 `mediate`는 철자가 비슷하지만 `mediate`는 사이에서 중재함.

### deliberate

- 뜻: 신중히 생각하다; 의도적인
- 구분 포인트: `deliberate`는 저울질해 판단하는 이미지. `premeditated`는 행동 전에 미리 계획했다는 법적·도덕적 뉘앙스가 강함.
