# authent

- 출발 단어: `authentic`
- 핵심 어원: Greek `authentikos` = original, genuine, principal
- 핵심 이미지: 진짜인, 스스로 근거를 가진, 원본에서 나온
- 관련 문서: [[auto]], [[auctor]]
- 주의: `authentic`은 현대 철자상 `author`, `authority`와 비슷하지만 같은 어근으로 단순 병합하지 않음. `author`, `authority`는 주로 Latin `auctor` 계열.

## 출발 단어 분석

`authentic` ← Greek `authentikos`

- `authentikos`: original, genuine, principal
- `authentes`: one acting on one's own authority
- 원래 이미지: 남이 대신한 것이 아니라, 스스로 근거와 권위를 가진 것
- 현대 의미: 진짜의, 진정성 있는, 믿을 수 있는

## 단어들

### authentic

- 구성: Greek `authentikos` = genuine, original + `-ic`
- 뜻: 진짜의, 진정한, 진정성 있는
- 어원적 연결: 원본에서 나온 것으로 믿을 수 있음
- 사용 맥락: `an authentic document` = 진짜 문서, `authentic cuisine` = 현지식에 가까운 음식

### authenticity

- 구성: `authentic` + `-ity`
- 뜻: 진짜임, 진정성
- 어원적 연결: 원본성·진짜성의 상태
- 사용 맥락: `verify the authenticity` = 진위 여부를 확인하다

### authentically

- 구성: `authentic` + `-ally`
- 뜻: 진정성 있게, 진짜답게
- 어원적 연결: 꾸민 것이 아니라 자기 본래 모습에 맞게
- 사용 맥락: `live authentically` = 진정성 있게 살다

### authenticate

- 구성: `authentic` + `-ate`
- 뜻: 진짜임을 입증하다, 인증하다
- 어원적 연결: 어떤 것이 원본·진짜임을 확인함
- 사용 맥락: `authenticate a signature` = 서명의 진위를 확인하다

### authentication

- 구성: `authenticate` + `-ion`
- 뜻: 인증, 진위 확인
- 어원적 연결: 진짜임을 확인하는 절차
- 사용 맥락: `two-factor authentication` = 이중 인증

### authenticator

- 구성: `authenticate` + `-or`
- 뜻: 인증자, 인증 장치, 인증 앱
- 어원적 연결: 진짜 사용자나 진짜 대상을 확인하는 것
- 사용 맥락: `an authenticator app` = 인증 앱

### inauthentic

- 구성: `in-` = not + `authentic`
- 뜻: 진짜가 아닌, 진정성 없는
- 어원적 연결: 원본성이나 진정성이 부족함
- 사용 맥락: `an inauthentic apology` = 진정성 없는 사과

### unauthenticated

- 구성: `un-` = not + `authenticated`
- 뜻: 인증되지 않은, 진위가 확인되지 않은
- 어원적 연결: 진짜임이 아직 확인되지 않음
- 사용 맥락: `unauthenticated access` = 인증되지 않은 접근

## 기억 포인트

`authent` 계열은 "진짜임, 원본성, 인증" 이미지.

- `authentic`: 진짜의
- `authenticity`: 진짜임, 진정성
- `authenticate`: 진짜임을 확인하다
- `authentication`: 인증 절차
- `inauthentic`: 진짜답지 않은

## 비교 단어

### author / authority

- 뜻: 저자, 권위
- 구분 포인트: [[auctor]]는 Latin `auctor` = 창시자, 보증자 계열. `authentic`과 의미상 "근거·권위"가 닿지만, 시험 어휘 정리에서는 별도 계열로 보는 편이 안전.

### auto

- 뜻: 스스로, 자기 자신
- 구분 포인트: [[auto]]는 Greek `autos` = self 계열. `authentic`의 더 오래된 분석에 `self` 이미지가 닿지만, 현대 학습에서는 `authent` = 진짜·인증으로 묶는 것이 실용적.
