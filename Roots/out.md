# out

- 출발 단어: `outstrip`
- 핵심 어원: Old English `ut`, `ute` = out, outside
- 핵심 이미지: 밖으로 나감, 범위를 넘어섬, 더 앞섬
- 관련 문서: [[ex]], [[ultra]], [[cess]]

## 단어들

### outstrip

- 발음: /aʊˈtstrɪp/
- 구성: `out-` = beyond, surpassing + Middle English `strip` = move quickly로 추정
- 뜻: 앞지르다, 능가하다
- 어원적 연결: 달리기에서 남보다 더 앞으로 나아감
- 사용 맥락: `demand outstrips supply` = 수요가 공급을 앞지름

### outdo

- 발음: /ˌaʊˈtdu/
- 구성: `out-` + `do`
- 뜻: 능가하다, 더 잘하다
- 어원적 연결: 남보다 더 뛰어나게 해냄
- 사용 맥락: `outdo competitors` = 경쟁자들을 능가하다

### outperform

- 발음: /ˈaʊtpɚˌfɔrm/
- 구성: `out-` + `perform`
- 뜻: 더 좋은 성과를 내다, 능가하다
- 어원적 연결: 수행 결과가 남보다 앞섬
- 사용 맥락: `outperform the market` = 시장 평균보다 좋은 성과를 내다

### outrun

- 발음: /aʊˈtrʌn/
- 구성: `out-` + `run`
- 뜻: 더 빨리 달리다, 앞질러 달아나다
- 어원적 연결: 달려서 상대보다 밖/앞으로 벗어남
- 사용 맥락: `outrun danger` = 위험에서 달아나다

### outnumber

- 발음: /aʊˈtnʌmbɚ/
- 구성: `out-` + `number`
- 뜻: 수적으로 우세하다
- 어원적 연결: 수가 상대보다 더 많음
- 사용 맥락: `visitors outnumber residents` = 방문객 수가 주민 수보다 많다

### outweigh

- 발음: /ˈaʊˌtweɪ/
- 구성: `out-` + `weigh`
- 뜻: 더 중요하다, 더 무겁다
- 어원적 연결: 무게나 중요성이 상대를 넘어섬
- 사용 맥락: `benefits outweigh risks` = 이익이 위험보다 크다

### outlast

- 발음: /ˈaʊˌtlæst/
- 구성: `out-` + `last`
- 뜻: 더 오래가다, 끝까지 버티다
- 어원적 연결: 시간상 상대보다 더 오래 남음
- 사용 맥락: `outlast the competition` = 경쟁에서 끝까지 살아남다

### outgrow

- 발음: /ˌaʊˈtgroʊ/
- 구성: `out-` + `grow`
- 뜻: 너무 커져 맞지 않게 되다; 벗어나다
- 어원적 연결: 성장해서 기존 범위를 넘어섬
- 사용 맥락: `outgrow old habits` = 오래된 습관에서 벗어나다

### outlive

- 발음: /ˌaʊˈtlɪv/
- 구성: `out-` + `live`
- 뜻: 더 오래 살다, 살아남다
- 어원적 연결: 생존 기간이 상대보다 더 김
- 사용 맥락: `outlive one's usefulness` = 더 이상 쓸모가 없어지다

### outspoken

- 발음: /ˈaʊˈtspoʊkən/
- 구성: `out-` + `spoken`
- 뜻: 솔직한, 거리낌 없이 말하는
- 어원적 연결: 생각을 밖으로 드러내 말함
- 사용 맥락: `an outspoken critic` = 거침없이 말하는 비판자

## 기억 포인트

`out-`은 두 갈래로 기억.

- 밖으로 드러남: `outspoken`
- 상대를 넘어섬: `outstrip`, `outdo`, `outperform`, `outnumber`, `outweigh`

## 비교 단어

### ex

- 뜻: 밖으로, 밖에
- 구분 포인트: [[ex]]는 Latin 계열 접두사. `out-`은 영어 기본어 계열이라 일상 합성어에 많이 붙음.

### surpass

- 뜻: 넘어서다, 능가하다
- 구분 포인트: `surpass`는 French/Latin 계열 단어. `outdo`, `outperform`, `outstrip`은 영어식 합성어 느낌이 강함.

### outstrip의 strip

- 뜻: 빠르게 움직이다로 추정되는 오래된 `strip`
- 구분 포인트: 현대의 `strip` = 벗기다, 떼어내다와 바로 연결해 외우기보다, `outstrip`은 통째로 "앞질러 넘어서다"로 기억.
