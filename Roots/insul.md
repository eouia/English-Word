# insul

- 출발 단어: `isolate`
- 핵심 어원: 라틴어 `insula` = island, 섬
- 변형: `insul`, `isol`, `isle`
- 핵심 이미지: 섬처럼 따로 떨어짐 / 둘러싸여 분리됨
- 관련해서 헷갈릴 수 있는 어근: [[separ]] = 나누어 떼다 계열, [[sol]] = 홀로 있음 계열

## 출발 단어 분석

`isolate`는 Italian `isola` = island에서 온 말에 `-ate`가 붙은 형태.

- `isola`: island
- `-ate`: 동사형 접미사

따라서 `isolate`는 원래 "섬처럼 따로 만들다"의 뜻. 여기서 "고립시키다, 분리하다"의 뜻.

## 단어들

### isolate

- 발음: /ˈaɪsəˌleɪt/
- 구성: `isol` = island + `-ate`
- 뜻: 고립시키다, 분리하다
- 어원적 연결: 섬처럼 주변과 떨어뜨림
- 사용 맥락: `isolate the variable` = 변수를 분리하다

### isolated

- 발음: /ˈaɪsəˌleɪtəd/
- 구성: `isolate` + `-ed`
- 뜻: 고립된, 외딴
- 어원적 연결: 섬처럼 따로 떨어져 있음
- 사용 맥락: `an isolated village` = 외딴 마을

### isolation

- 발음: /ˌaɪsəˈleɪʃən/
- 구성: `isolate` + `-ion`
- 뜻: 고립, 격리, 분리
- 어원적 연결: 주변과 떨어져 섬처럼 된 상태
- 사용 맥락: `social isolation` = 사회적 고립

### insulate

- 발음: /ˈɪnsəˌleɪt/
- 구성: Latin `insula` = island + `-ate`
- 뜻: 단열하다, 절연하다, 보호해 분리하다
- 어원적 연결: 섬처럼 둘러싸 외부와 떨어뜨림
- 사용 맥락: `insulate a house` = 집을 단열하다

### insulation

- 발음: /ˌɪnsəˈleɪʃən/
- 구성: `insulate` + `-ion`
- 뜻: 단열, 절연, 단열재
- 어원적 연결: 열이나 전기 흐름과 분리하는 층
- 사용 맥락: `thermal insulation` = 단열

### insular

- 발음: /ˈɪnsəlɚ/
- 구성: Latin `insularis` = of an island
- 뜻: 섬의; 편협한, 고립적인
- 어원적 연결: 섬처럼 외부와 떨어져 좁은 세계에 머묾
- 사용 맥락: `an insular attitude` = 편협한 태도

### island

- 발음: /ˈaɪlənd/
- 구성: Old English `iegland`; 철자 `s`는 `isle` 영향
- 뜻: 섬
- 어원적 연결: 물로 둘러싸인 땅
- 사용 맥락: `a remote island` = 외딴 섬

### isle

- 발음: /ˈaɪl/
- 구성: Old French `ile` < Latin `insula`
- 뜻: 섬
- 어원적 연결: 라틴어 `insula` 계열의 문학적 형태
- 사용 맥락: `a tropical isle` = 열대 섬

### peninsula

- 발음: /pəˈnɪnsələ/
- 구성: Latin `paene` = almost + `insula` = island
- 뜻: 반도
- 어원적 연결: 거의 섬처럼 물에 둘러싸인 땅
- 사용 맥락: `the Korean Peninsula` = 한반도

## 기억 포인트

`insul/isol` 계열은 "섬" 이미지.

- `isolate`: 섬처럼 따로 만들다
- `isolation`: 섬처럼 떨어진 상태
- `insulate`: 외부와 분리되게 둘러싸다
- `insular`: 섬처럼 고립된, 편협한
- `peninsula`: 거의 섬인 땅

## 비교 단어

### separate

- 뜻: 분리하다
- 구분 포인트: [[separ]]는 둘을 떼어 나누는 동작에 초점. `isolate`는 섬처럼 떨어진 상태에 초점

### solitary

- 뜻: 혼자의, 고독한
- 구분 포인트: [[sol]]은 홀로 있음의 이미지. `isolate`는 주변과 차단되어 떨어지는 이미지
