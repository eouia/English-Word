# don

- 출발 단어: `donate`
- 핵심 어원: Latin `donum` = gift, `donare` = give
- 핵심 이미지: 선물로 주다, 공식적으로 내어 주다
- 관련 어근: [[give]], [[vot]]

## 출발 단어 분석

`donate`는 Latin `donare`에서 온 말.

핵심은 "선물로 주다"이며, 현대 영어에서는 특히 돈, 물품, 장기, 시간 등을 공식적으로 기부하는 느낌이 강함.

## 단어들

### donate

- 구성: Latin `donare` = give
- 의미: 기부하다, 기증하다
- 어원적 연결: 선물로 내어 줌
- 사용 맥락: `donate money` = 돈을 기부하다

### donation

- 구성: `donate` + `-ion`
- 의미: 기부, 기증품
- 어원적 연결: 주는 행위 또는 주어진 것
- 사용 맥락: `make a donation` = 기부하다

### donor

- 구성: Latin `donator` 계열
- 의미: 기증자, 기부자
- 어원적 연결: 주는 사람
- 사용 맥락: `an organ donor` = 장기 기증자

### pardon

- 구성: Old French `pardoner` < `per-` + Latin `donare`
- 의미: 용서하다, 사면하다
- 어원적 연결: 잘못에 대한 처벌을 완전히 내어 주거나 놓아 줌
- 사용 맥락: `pardon the mistake` = 실수를 용서하다

### condone

- 구성: Latin `con-` + `donare`
- 의미: 묵인하다, 용납하다
- 어원적 연결: 잘못을 함께/완전히 놓아 줌
- 사용 맥락: `condone violence` = 폭력을 용인하다

## 기억 포인트

- `donate`: 선물로 줌, 기부함
- `donor`: 주는 사람
- `pardon`: 잘못을 놓아 줌
- `condone`: 잘못을 그냥 넘어가 줌

## 비교 단어

- [[give]]
  - 핵심: 일반적인 "주다"
  - 구분 포인트: `give`는 게르만계 기본 동사. `donate`는 라틴계로 더 공식적이고 기부의 느낌

- [[vot]]
  - 핵심: 맹세하다, 바치다
  - 구분 포인트: `devote`는 마음, 시간, 노력을 바침. `donate`는 물건이나 돈을 기부함

- `data`
  - 핵심: Latin `dare` = give 계열에서 "주어진 것"
  - 구분 포인트: `data`는 현대 영어에서 "자료, 데이터"로 굳어져 `donate`와 직접 묶어 외우기보다는 참고만 함
