# virt

- 출발 단어: `virtue`
- 핵심 어원: Latin `virtus` = 힘, 용기, 뛰어남, 덕
- 깊은 어원: Latin `vir` = man, adult male
- 핵심 의미: 힘과 뛰어남에서 나온 덕, 능력, 효력
- 관련 문서: [[val]], [[fac]], [[res]]
- 주의: 현대 영어의 `virtue`는 성별 의미 없이 “덕, 미덕” 중심. 라틴어의 `vir` 배경은 역사적 설명으로만 기억.

## 출발 단어 분석

`virtue` ← Old French `vertu/virtu` ← Latin `virtus`

- `vir`: 성인 남성
- `virtus`: 남성적 힘, 용기, 뛰어남, 도덕적 훌륭함
- 의미 변화: 힘과 용기 → 뛰어남 → 도덕적 덕
- 현대 의미: 미덕, 덕성, 장점

## 단어들

### virtue

- 발음: /ˈvɝtʃu/
- 구성: Latin `virtus` = excellence, moral strength
- 뜻: 미덕, 덕성, 장점
- 어원적 연결: 뛰어난 힘과 품성이 도덕적 훌륭함으로 확장
- 사용 맥락: `patience is a virtue` = 인내는 미덕

### virtuous

- 발음: /ˈvɝtʃuəs/
- 구성: `virtue` + `-ous`
- 뜻: 덕 있는, 고결한
- 어원적 연결: 덕을 지닌 상태
- 사용 맥락: `a virtuous person` = 덕 있는 사람

### virtuously

- 구성: `virtuous` + `-ly`
- 뜻: 덕 있게, 고결하게
- 어원적 연결: 도덕적으로 훌륭한 방식
- 사용 맥락: `act virtuously` = 덕 있게 행동하다

### virtuousness

- 구성: `virtuous` + `-ness`
- 뜻: 고결함, 덕성
- 어원적 연결: 덕 있는 성질
- 사용 맥락: moral philosophy 맥락

### by virtue of

- 구성: `virtue` = power, efficacy
- 뜻: ~의 힘으로, ~덕분에
- 어원적 연결: 어떤 것의 내재적 힘이나 자격에 의해
- 사용 맥락: `by virtue of experience` = 경험 덕분에

### virtual

- 발음: /ˈvɝtʃuəl/
- 구성: Medieval Latin `virtualis` ← `virtus`
- 뜻: 사실상의, 가상의
- 어원적 연결: 겉모습은 아니어도 효과를 내는 힘을 가짐
- 사용 맥락: `a virtual meeting` = 온라인 회의, 가상 회의

### virtually

- 발음: /ˈvɝtʃuəli/
- 구성: `virtual` + `-ly`
- 뜻: 사실상, 거의
- 어원적 연결: 실제와 같은 효과를 낼 만큼
- 사용 맥락: `virtually impossible` = 사실상 불가능한

### virtuality

- 구성: `virtual` + `-ity`
- 뜻: 가상성, 사실상 존재함
- 어원적 연결: 실제처럼 작용하는 성질
- 사용 맥락: philosophy / media studies 맥락

### virtuoso

- 발음: /vɚtʃuˈoʊsoʊ/
- 구성: Italian `virtuoso` ← Late Latin `virtuosus`
- 뜻: 거장, 명연주자
- 어원적 연결: 뛰어난 기술과 예술적 우수성을 지닌 사람
- 사용 맥락: `a piano virtuoso` = 피아노 거장

### virtuosity

- 발음: /vɚtʃuˈɑsɪti/
- 구성: `virtuoso` 계열 + `-ity`
- 뜻: 탁월한 기교, 명인적 솜씨
- 어원적 연결: 뛰어난 능력과 기술
- 사용 맥락: `technical virtuosity` = 기술적 탁월함

### virtu

- 구성: Italian `virtu` ← Latin `virtus`
- 뜻: 예술품에 대한 감식력, 예술적 가치
- 어원적 연결: 뛰어남과 가치가 예술 감식의 의미로 이동
- 사용 맥락: 드문 격식어

### virtus

- 구성: Latin `virtus`
- 뜻: 덕, 용기, 힘, 뛰어남
- 어원적 연결: 로마적 미덕과 용기의 핵심어
- 사용 맥락: 고전·철학 맥락

### virile

- 발음: /ˈvɪrəl/
- 구성: Latin `virilis` ← `vir`
- 뜻: 남성적인, 힘찬
- 어원적 연결: Latin `vir` = man에서 직접 연결
- 사용 맥락: `virile energy` = 힘찬 에너지

### virility

- 발음: /vɚˈɪləti/
- 구성: `virile` + `-ity`
- 뜻: 남성다움, 활력
- 어원적 연결: 성인 남성의 힘이라는 오래된 이미지
- 사용 맥락: biological / cultural 맥락

### triumvirate

- 발음: /traɪˈʌmvɚət/
- 구성: Latin `tri-` = three + `vir` = man
- 뜻: 3인 통치 체제, 3인조
- 어원적 연결: 세 명의 남성이 함께 권력을 나눔
- 사용 맥락: `a political triumvirate` = 정치적 3인 체제

## 기억 포인트

- `virtue`: 힘과 뛰어남 → 덕.
- `virtuous`: 덕 있는.
- `virtual`: 실제 형태는 아니어도 효과를 내는 힘이 있는 → 사실상의, 가상의.
- `virtuoso`: 뛰어난 기술을 가진 거장.
- `virile`: `vir` = man 쪽이 직접 남은 단어.

## 비교 단어

### val

- [[val]]은 힘이 있다, 가치가 있다, 유효하다.
- `virt`도 힘과 가치가 있지만, 덕·우수성·능력의 색이 강함.

### moral

- `moral`은 Latin `mos/mores` = 관습, 풍습 계열.
- `virtue`는 내적 힘과 뛰어남이 도덕성으로 확장된 단어.

### physical

- `virtual`은 “가상의”라서 `physical`과 자주 대비됨.
- 어원적으로는 `virtual` = 효과를 내는 힘, `physical` = 자연·몸의 성질.
