# mag-magn

- 상위 문서: [[mag]]
- 핵심 어원: Latin `magnus` = great, large
- 핵심 이미지: 크다, 크게 만들다, 큰 정도
- 관련 갈래: [[mag-master]], [[mag-major]]

## 단어들

### magnificent

- 발음: /mæˈgnɪfəsənt/
- 구성: Latin `magnus` = great + `facere` = make/do
- 뜻: 장엄한, 훌륭한
- 어원적 연결: 크게 만들어진 듯 뛰어나고 인상적임
- 사용 맥락: `a magnificent view` = 장엄한 경치

### magnificence

- 구성: `magnificent` 계열
- 뜻: 장엄함, 화려함
- 어원적 연결: 큰 규모와 뛰어남이 주는 인상
- 사용 맥락: `the magnificence of the palace` = 궁전의 장엄함

### magnify

- 발음: /ˈmægnəˌfaɪ/
- 구성: `magnus` = great + `-fy` = make
- 뜻: 확대하다, 과장하다
- 어원적 연결: 크게 만듦
- 사용 맥락: `magnify the image` = 이미지를 확대하다

### magnitude

- 발음: /ˈmægnəˌtud/
- 구성: Latin `magnitudo` ← `magnus`
- 뜻: 크기, 규모, 중요도
- 어원적 연결: 큰 정도
- 사용 맥락: `the magnitude of the problem` = 문제의 규모

### magnate

- 발음: /ˈmægnət/
- 구성: Latin `magnates` = great men
- 뜻: 거물, 유력자
- 어원적 연결: 큰 권력이나 부를 가진 사람
- 사용 맥락: `a media magnate` = 미디어 거물

### magnanimous

- 발음: /mæˈgnænəməs/
- 구성: Latin `magnus` = great + `animus` = mind, spirit
- 뜻: 도량이 큰, 관대한
- 어원적 연결: 마음이 큰 사람
- 사용 맥락: `a magnanimous gesture` = 도량 큰 행동
- 관련 문서: [[anim]]

### Magna Carta

- 구성: Latin `magna` = great + `carta` = charter
- 뜻: 대헌장
- 어원적 연결: "큰 헌장"이라는 뜻의 역사적 문서
- 사용 맥락: English constitutional history

## 기억 포인트

`mag-magn`은 "큰 크기와 큰 마음" 이미지.

- `magnificent`: 크게 뛰어난
- `magnify`: 크게 만들다
- `magnitude`: 큰 정도, 규모
- `magnate`: 큰 권력자
- `magnanimous`: 마음이 큰
