# stand

- 출발 단어: `stand`
- 핵심 어원: Old English `standan` = 서다
- 핵심 이미지: 서 있음, 버팀, 입장을 취함
- 관련해서 헷갈릴 수 있는 어근: [[sist]] = 라틴어 `sistere`, 서다 / 서게 하다 계열, [[stat]] = 라틴어 `stare/status`, 서 있는 상태 계열

## 출발 단어 분석

`stand`는 게르만계 영어 기본어.

- Old English `standan`: stand
- 핵심 의미: 몸을 세우다, 한 자리에 버티다

따라서 `stand`는 물리적 "서다"에서 "견디다, 입장을 취하다, 상태에 있다"로 넓어짐.

## 단어들

### stand

- 구성: Old English `standan`
- 뜻: 서다, 견디다, 입장
- 어원적 연결: 몸이나 입장을 세우고 버팀
- 사용 맥락: `stand still` = 가만히 서 있다

### standing

- 구성: `stand` + `-ing`
- 뜻: 지위, 평판, 서 있는
- 어원적 연결: 사회 안에서 서 있는 자리
- 사용 맥락: `social standing` = 사회적 지위

### standpoint

- 구성: `stand` + `point`
- 뜻: 관점, 입장
- 어원적 연결: 어디에 서서 보느냐의 자리
- 사용 맥락: `from an economic standpoint` = 경제적 관점에서

### standard

- 구성: Old French `estandart`
- 뜻: 기준, 표준
- 어원적 연결: 원래 깃발/군기 이미지. 모두가 바라보고 따르는 세워진 기준
- 사용 맥락: `high standards` = 높은 기준

### outstanding

- 구성: `out` + `standing`
- 뜻: 뛰어난, 미해결의
- 어원적 연결: 밖으로 두드러져 서 있음 / 아직 남아 서 있음
- 사용 맥락: `an outstanding achievement` = 뛰어난 성취

### withstand

- 구성: `with-` = against + `stand`
- 뜻: 견디다, 버티다
- 어원적 연결: 맞서 서서 버팀
- 사용 맥락: `withstand pressure` = 압력을 견디다

### understand

- 구성: `under` + `stand`
- 뜻: 이해하다
- 어원적 연결: 역사적으로 "가까이 서다, 사이에 서서 파악하다" 쪽으로 설명됨. 현대 영어에서는 직접 분해보다 통째로 기억
- 사용 맥락: `understand the meaning` = 의미를 이해하다

### understanding

- 구성: `understand` + `-ing`
- 뜻: 이해, 이해심 있는
- 어원적 연결: 어떤 뜻을 파악하고 받아들임
- 사용 맥락: `a deep understanding` = 깊은 이해

### misunderstanding

- 구성: `mis-` = wrong + `understanding`
- 뜻: 오해
- 어원적 연결: 잘못 이해함
- 사용 맥락: `clear up a misunderstanding` = 오해를 풀다

### standstill

- 구성: `stand` + `still`
- 뜻: 정지, 멈춤
- 어원적 연결: 움직이지 않고 서 있음
- 사용 맥락: `traffic came to a standstill` = 교통이 멈췄다

### bystander

- 구성: `by` + `stander`
- 뜻: 구경꾼, 방관자
- 어원적 연결: 옆에 서 있는 사람
- 사용 맥락: `innocent bystanders` = 무고한 방관자들

## 기억 포인트

`stand` 계열은 "서 있음 / 입장 / 버팀" 이미지.

- `stand`: 서다, 버티다
- `standing`: 사회에서 서 있는 자리
- `standpoint`: 서서 보는 자리
- `withstand`: 맞서 서서 견딤
- `outstanding`: 밖으로 두드러져 서 있음
- `understand`: 이해하다. 현대에서는 통째로 기억

## 비교 단어

### sist

- 뜻: 서다, 서게 하다, 버티다
- 구분 포인트: [[sist]]는 라틴계 시험 어휘. `exist`, `consist`, `persist`, `resist`처럼 나타남. `stand`는 영어 기본어로 합성어와 관용 표현에 많이 나타남

### stat

- 뜻: 서 있는 상태, 공식적 위치
- 구분 포인트: [[stat]]은 `state`, `status`, `statement`, `station`처럼 "세워진 상태나 자리"를 나타냄. `stand`는 영어 기본 동사
