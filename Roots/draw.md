# draw

- 출발 단어: `withdraw`
- 핵심 어원: Old English `dragan` = draw, drag
- 핵심 이미지: 끌어당기거나 끌어냄
- 관련 문서: [[tract]]
- 주의: `withdraw`의 `with-`는 여기서 "함께"가 아니라 "뒤로, 떨어져, 반대 방향으로"의 옛 의미.

## 출발 단어 분석

`withdraw` = `with-` + `draw`

- `with-`: back, away
- `draw`: pull, drag
- 원래 이미지: 뒤로 끌어내다
- 현대 의미: 철수하다, 물러나다, 인출하다, 철회하다

`withdraw money`는 돈을 계좌에서 "밖으로 끌어냄". `withdraw a statement`는 이미 낸 말을 "뒤로 거둬들임".

## 단어들

### draw

- 발음: /ˈdrɔ/
- 구성: Old English `dragan` = draw, drag
- 뜻: 끌다, 끌어내다; 그리다; 추첨하다
- 어원적 연결: 선이나 대상을 끌어 움직임
- 사용 맥락: `draw a line` = 선을 긋다, `draw attention` = 관심을 끌다

### withdraw

- 발음: /wɪˈðdrɔ/
- 구성: `with-` = back, away + `draw`
- 뜻: 철수하다, 물러나다, 인출하다, 철회하다
- 어원적 연결: 뒤로 또는 밖으로 끌어냄
- 사용 맥락: `withdraw money` = 돈을 인출하다, `withdraw from the race` = 경주에서 물러나다

### withdrawal

- 발음: /wɪˈðdrɔəl/
- 구성: `withdraw` + `-al`
- 뜻: 철수, 회수, 인출; 금단 증상
- 어원적 연결: 어떤 상태나 장소에서 뒤로 빠져나오는 과정
- 사용 맥락: `troop withdrawal` = 병력 철수, `cash withdrawal` = 현금 인출

### withdrawn

- 발음: /wɪˈθdrɔn/
- 구성: `withdraw`의 과거분사
- 뜻: 철회된; 내성적인, 말수가 적은
- 어원적 연결: 밖으로 나서지 않고 뒤로 물러난 상태
- 사용 맥락: `a withdrawn child` = 말수가 적고 내성적인 아이

### withdrawable

- 구성: `withdraw` + `-able`
- 뜻: 인출 가능한, 철회 가능한
- 어원적 연결: 뒤로 거두거나 밖으로 끌어낼 수 있음
- 사용 맥락: `withdrawable funds` = 인출 가능한 자금

### drawback

- 발음: /ˈdrɔˌbæk/
- 구성: `draw` + `back`
- 뜻: 단점, 결점
- 어원적 연결: 뒤로 잡아당겨 진행을 불리하게 만드는 것
- 사용 맥락: `a major drawback` = 큰 단점

### drawbridge

- 발음: /ˈdrɔˌbrɪdʒ/
- 구성: `draw` + `bridge`
- 뜻: 도개교
- 어원적 연결: 위로 끌어올릴 수 있는 다리
- 사용 맥락: castle, canal 문맥에서 자주 보임.

## 기억 포인트

`draw` 계열은 "끌어당기거나 끌어냄" 이미지.

- `draw`: 끌다, 끌어내다, 그리다
- `withdraw`: 뒤로 끌어내다, 철수하다, 인출하다
- `withdrawal`: 철수, 인출, 금단
- `withdrawn`: 물러난, 내성적인
- `drawback`: 뒤로 잡아끄는 단점

## 비교 단어

### tract

- 뜻: 끌다, 당기다
- 구분 포인트: [[tract]]는 Latin `trahere` 계열. `draw`는 Germanic 기본어로 일상적이고, 그림을 그리거나 관심을 끄는 뜻까지 넓음.

### retreat

- 뜻: 후퇴하다, 물러나다
- 구분 포인트: `retreat`는 군사·전략적으로 뒤로 물러남. `withdraw`는 군대, 돈, 발언, 참여 등에서 넓게 "뒤로 거둠".

### revoke

- 뜻: 취소하다, 철회하다
- 구분 포인트: `revoke`는 권한·허가·법적 효력을 공식적으로 취소함. `withdraw`는 말·지원·돈·참여를 뒤로 거두는 넓은 표현.
