# arma

- 출발 단어: `army`
- 핵심 어원: Latin `arma` = weapons, arms
- 핵심 이미지: 무기, 무장, 무기를 갖춘 집단
- 관련 문서: 없음
- 주의: 몸의 `arm` = 팔과 무기 `arms`는 역사적으로 서로 다른 계열로 보는 것이 안전함. 현대 영어에서는 철자가 같아 의미가 섞여 보임.

## 출발 단어 분석

`army` ← Old French `armee` ← Latin `armata`

- `arma`: weapons, arms
- `armata`: armed force
- 원래 이미지: 무장한 사람들의 집단
- 현대 의미: 군대, 육군, 대군

## 단어들

### army

- 발음: /ˈɑrmi/
- 구성: Old French `armee` ← Latin `armata` = armed force
- 뜻: 군대, 육군, 대군
- 어원적 연결: 무기를 갖춘 집단
- 사용 맥락: `join the army` = 군대에 입대하다

### arm

- 발음: /ˈɑrm/
- 구성: Latin/French `armer` 계열
- 뜻: 무장시키다, 무기
- 어원적 연결: 무기를 갖추게 함
- 사용 맥락: `arm the guards` = 경비원을 무장시키다

### arms

- 발음: /ˈɑrmz/
- 구성: Latin `arma` = weapons
- 뜻: 무기, 병기
- 어원적 연결: 전투에 쓰는 장비
- 사용 맥락: `arms control` = 군비 통제

### armed

- 발음: /ˈɑrmd/
- 구성: `arm` + `-ed`
- 뜻: 무장한
- 어원적 연결: 무기를 갖춘 상태
- 사용 맥락: `armed forces` = 군대, `armed robbery` = 무장 강도

### armor

- 발음: /ˈɑrmɚ/
- 구성: Old French `armure` ← Latin `armatura`
- 뜻: 갑옷, 장갑, 방호 장비
- 어원적 연결: 무장과 방어 장비
- 사용 맥락: `body armor` = 방탄복

### armory

- 발음: /ˈɑrmɚi/
- 구성: `armor/arms` 계열 + `-y`
- 뜻: 무기고
- 어원적 연결: 무기를 보관하는 곳
- 사용 맥락: `a military armory` = 군 무기고

### armament

- 발음: /ˈɑrməmənt/
- 구성: `arm` + `-ament`
- 뜻: 군비, 무장, 무기 체계
- 어원적 연결: 무장을 위한 장비 전체
- 사용 맥락: `nuclear armaments` = 핵무기

### disarm

- 발음: /dɪˈsɑrm/
- 구성: `dis-` = remove, away + `arm`
- 뜻: 무장을 해제하다, 경계심을 풀게 하다
- 어원적 연결: 무기를 빼앗거나 내려놓게 함
- 사용 맥락: `disarm the suspect` = 용의자의 무장을 해제하다

### disarmament

- 발음: /dɪˈsɑrməmənt/
- 구성: `disarm` + `-ament`
- 뜻: 군축, 무장 해제
- 어원적 연결: 무기 체계를 줄이거나 없앰
- 사용 맥락: `nuclear disarmament` = 핵군축

### armistice

- 발음: /ˈɑrməstəs/
- 구성: Latin `arma` = weapons + `-stitium` = stopping
- 뜻: 휴전
- 어원적 연결: 무기를 멈춤
- 사용 맥락: `sign an armistice` = 휴전 협정에 서명하다

## 기억 포인트

`arma` 계열은 "무기, 무장" 이미지.

- `army`: 무장한 집단
- `arms`: 무기
- `armed`: 무장한
- `armor`: 방어용 무장
- `disarm`: 무장을 해제함
- `armistice`: 무기를 멈춤

## 비교 단어

### arm

- 뜻: 팔
- 구분 포인트: 몸의 `arm`은 Germanic 계열. `army`, `arms`, `armor`의 `arma`는 Latin 무기 계열. 현대 영어에서 철자가 같지만 어원 학습에서는 구분.
