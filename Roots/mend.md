# mend

- 출발 단어: `amend`
- 핵심 어원: 라틴어 `menda` = fault, blemish
- 핵심 이미지: 흠이나 잘못을 고쳐 더 낫게 함
- 관련해서 헷갈릴 수 있는 어근: [[rect]] = 곧게 하다, 바로잡다 계열, [[alter]] = 바꾸다 계열

## 출발 단어 분석

`amend`는 `a-` + `mend`.

- `a-`: away, off
- `mend`: fault, defect와 관련

따라서 `amend`는 원래 "잘못이나 흠을 제거하다"의 뜻. 여기서 "수정하다, 개정하다, 개선하다"의 뜻.

## 단어들

### amend

- 발음: /əˈmɛnd/
- 구성: `a-` = away + `mend`
- 뜻: 수정하다, 개정하다, 개선하다
- 어원적 연결: 잘못이나 흠을 없애 더 낫게 함
- 사용 맥락: `amend a law` = 법을 개정하다

### amendment

- 발음: /əˈmɛndmənt/
- 구성: `amend` + `-ment`
- 뜻: 수정, 개정; 수정 조항
- 어원적 연결: 잘못을 고치거나 내용을 더 낫게 바꾼 결과
- 사용 맥락: `a constitutional amendment` = 헌법 수정 조항

### mending

- 발음: /ˈmɛndɪŋ/
- 구성: `mend` + `-ing`
- 뜻: 수선, 고침
- 어원적 연결: 망가진 곳이나 흠을 고치는 행위
- 사용 맥락: `clothes mending` = 옷 수선

### mend

- 발음: /ˈmɛnd/
- 구성: shortening of `amend`
- 뜻: 고치다, 수선하다, 회복되다
- 어원적 연결: 잘못되거나 망가진 것을 더 나은 상태로 고침
- 사용 맥락: `mend a fence` = 울타리를 고치다, `mend a relationship` = 관계를 회복하다

### emend

- 구성: `e-` = out + `mend`
- 뜻: 교정하다, 수정하다
- 어원적 연결: 글의 오류를 밖으로 빼내 고침
- 사용 맥락: `emend a manuscript` = 원고를 교정하다

### emendation

- 구성: `emend` + `-ation`
- 뜻: 교정, 수정
- 어원적 연결: 글이나 문헌의 오류를 고친 결과
- 사용 맥락: `textual emendation` = 본문 교정

## 기억 포인트

`mend` 계열은 "흠을 고쳐 더 낫게 함" 이미지.

- `amend`: 법, 문서, 상황을 수정하다
- `amendment`: 수정, 개정 조항
- `mend`: 물건이나 관계를 고치다
- `emend`: 글의 오류를 교정하다

## 비교 단어

### correct

- 뜻: 고치다, 바로잡다
- 구분 포인트: [[rect]] 계열. 곧게 바로잡는 이미지. `amend`는 흠을 제거해 개선하는 느낌

### alter

- 뜻: 바꾸다
- 구분 포인트: [[alter]]는 단순히 다른 상태로 바꾸는 것. `amend`는 보통 더 낫게 고치는 방향
