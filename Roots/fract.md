# fract

- 출발 단어: `fragment`
- 핵심 어원: Latin `frangere` / `fract-` = break
- 핵심 이미지: 깨져 나뉨, 부러짐
- 관련 어근: [[rupt]], [[piece]], [[part]]

## 출발 단어 분석

`fragment`는 Latin `fragmentum` = a broken piece에서 온 말.

바탕은 Latin `frangere` = break, 그리고 과거분사형 `fract-` = broken.

핵심은 "깨져서 조각으로 나뉨".

## 단어들

### fragment

- 발음: /ˈfrægmənt/
- 구성: Latin `fragmentum` < `frangere`
- 의미: 조각, 파편
- 어원적 연결: 깨져 나온 작은 부분
- 사용 맥락: `sentence fragment` = 문장 조각

### fraction

- 발음: /ˈfrækʃən/
- 구성: Latin `fractio` = breaking
- 의미: 분수, 일부
- 어원적 연결: 전체를 깨듯 나누어 놓은 부분
- 사용 맥락: `a fraction of the cost` = 비용의 일부

### fracture

- 발음: /ˈfræktʃɚ/
- 구성: Latin `fractura`
- 의미: 골절, 균열, 부러뜨리다
- 어원적 연결: 뼈나 물체가 깨져 갈라짐
- 사용 맥락: `a bone fracture` = 골절

### fragile

- 발음: /ˈfrædʒəl/
- 구성: Latin `fragilis` < `frangere`
- 의미: 깨지기 쉬운, 취약한
- 어원적 연결: 쉽게 부서질 수 있음
- 사용 맥락: `fragile glass` = 깨지기 쉬운 유리

### refract

- 구성: `re-` = back + `fract`
- 의미: 굴절시키다
- 어원적 연결: 빛의 진행이 꺾여 "부러진" 듯 방향이 바뀜
- 사용 맥락: `refract light` = 빛을 굴절시키다

### infraction

- 발음: /ˌɪˈnfrækʃən/
- 구성: `in-` = into/against + `fract`
- 의미: 위반, 규칙 위반
- 어원적 연결: 규칙을 깨뜨림
- 사용 맥락: `a traffic infraction` = 교통 법규 위반

## 기억 포인트

- `fragment`: 깨진 조각
- `fraction`: 나뉜 일부
- `fracture`: 깨짐, 골절
- `fragile`: 깨지기 쉬움
- `infraction`: 규칙을 깨뜨림

## 비교 단어

- [[piece]]
  - 핵심: 조각, 한 단위
  - 구분 포인트: `piece`는 반드시 깨진 조각일 필요는 없음. `fract`는 깨짐, 부러짐이 핵심

- [[rupt]]
  - 핵심: 터뜨리다, 흐름을 끊다
  - 구분 포인트: `rupt`는 터져 끊기는 느낌. `fract`는 단단한 것이 깨져 갈라지는 느낌

- [[part]]
  - 핵심: 부분, 몫
  - 구분 포인트: `part`는 전체를 이루는 부분. `fraction/fragment`는 나뉘거나 깨져 나온 부분
