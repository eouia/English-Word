# mut

- 출발 단어: `commute`
- 핵심 어원: 라틴어 `mutare` = 바꾸다, 변화시키다
- 핵심 의미: `change`, `exchange`, `shift`
- 관련 문서: [[con]], [[mod]]
- 주의: `mute` = 말 없는, 소리 없는. 라틴어 `mutus` 계열로 `mutare`와 다른 어원.

## 출발 단어 분석

`commute` = `com-` + `mutare`

- `com-`: 함께, 서로
- `mutare`: 바꾸다
- 원래 이미지: 서로 바꿈, 교환함
- 현대 의미: 통근하다, 형을 감형하다, 수학에서 순서를 바꾸어도 결과가 같음

## 단어들

### commute

- 구성: `com-` = together + `mut`
- 뜻: 통근하다, 대체하다, 감형하다
- 어원적 연결: 한 상태를 다른 상태로 바꿈. 집과 직장 사이를 정기적으로 오가는 의미로 확장
- 사용 맥락: `commute to work` = 직장에 통근하다

### commuter

- 구성: `commute` + `-er`
- 뜻: 통근자
- 어원적 연결: 두 장소 사이를 정기적으로 오가는 사람
- 사용 맥락: `commuter train` = 통근 열차

### commutation

- 구성: `commute` + `-ation`
- 뜻: 교환, 감형, 교환 법칙
- 어원적 연결: 하나를 다른 것으로 바꾸는 행위
- 사용 맥락: `commutation of a sentence` = 형의 감형

### mutate

- 구성: `mut` + `-ate`
- 뜻: 변이하다, 변화하다
- 어원적 연결: 형태나 성질이 바뀜
- 사용 맥락: `the virus may mutate` = 바이러스가 변이할 수 있다

### mutation

- 구성: `mutate` + `-ion`
- 뜻: 변이, 돌연변이
- 어원적 연결: 바뀐 결과 또는 변화 과정
- 사용 맥락: `genetic mutation` = 유전적 돌연변이

### mutable

- 구성: `mut` + `-able`
- 뜻: 변할 수 있는, 가변의
- 어원적 연결: 바뀔 수 있음
- 사용 맥락: `mutable data` = 변경 가능한 데이터

### immutable

- 구성: `in-` = not + `mutable`
- 뜻: 변하지 않는, 불변의
- 어원적 연결: 바뀔 수 없음
- 사용 맥락: `immutable rules` = 불변의 규칙

### transmute

- 구성: `trans-` = across, beyond + `mut`
- 뜻: 변형시키다, 완전히 바꾸다
- 어원적 연결: 한 상태를 넘어 다른 상태로 바꿈
- 사용 맥락: `transmute fear into courage` = 두려움을 용기로 바꾸다

### permutation

- 구성: `per-` = through, thoroughly + `mut`
- 뜻: 순열, 배열 변경
- 어원적 연결: 요소들의 자리를 바꾸어 만든 배열
- 사용 맥락: `all possible permutations` = 가능한 모든 순열

### mutual

- 구성: 라틴어 `mutuus` = 서로 주고받는, 상호의
- 뜻: 상호의, 서로의
- 어원적 연결: 직접 형태는 `mutuus`지만, “서로 바꿈/교환” 이미지와 가까움
- 사용 맥락: `mutual respect` = 상호 존중

## 기억 포인트

- `mut`는 “바뀜”이 핵심.
- `commute`는 단순히 “통근”으로 외우기보다, 원래 “교환/대체” 이미지에서 출발.
- `mutation`, `mutable`, `permutation`은 시험·학술 지문에서 자주 보이는 핵심 단어.

## 비교 단어

### mute

- `mute`, `muted`는 “말 없는, 소리가 약한” 뜻.
- 라틴어 `mutus` = silent 계열.
- 철자는 비슷하지만 `mutare` = change 계열과 분리.

### reciprocal

- 뜻: 상호의, 서로 되돌려 주는
- 구분 포인트: [[reciproc]]는 오가며 되돌려 주는 대응성에 초점. `mutual`은 서로 공유하거나 서로에게 해당하는 관계를 넓게 말함.
