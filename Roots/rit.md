# rit

- 출발 단어: `ritual`
- 핵심 어원: Latin `ritus` = 종교 의식, 관례
- 핵심 의미: 정해진 방식의 의식, 관례
- 관련 비교: `religion`, `culture`, `custom`

## 출발 단어 분석

`ritual` = `rite` + `-ual`

- `ritus`: religious ceremony, custom
- `rite`: 의식, 예식
- `-ual`: 관련된 성질
- 원래 이미지: 정해진 순서와 형식에 따라 행하는 의식
- 의미 확장: 종교 의식 → 사회적 관례 → 반복 습관

## 단어들

### ritual

- 발음: /ˈrɪtʃuəl/
- 구성: Latin `ritus`
- 뜻: 의식, 의례; 반복되는 관습
- 어원적 연결: 정해진 방식으로 되풀이되는 행위
- 사용 맥락: `religious ritual` = 종교 의식, `morning ritual` = 아침 루틴

### rite

- 발음: /ˈraɪt/
- 구성: Latin `ritus`
- 뜻: 의식, 예식
- 어원적 연결: 공동체가 정한 형식에 따라 치르는 절차
- 사용 맥락: `funeral rites` = 장례 의식

### rites

- 발음: /ˈraɪts/
- 구성: `rite`의 복수형
- 뜻: 의례, 의식 절차
- 어원적 연결: 여러 단계나 종류의 공식 의식
- 사용 맥락: `religious rites` = 종교 의례

### rite of passage

- 구성: `rite` + `passage`
- 뜻: 통과 의례
- 어원적 연결: 한 지위나 생애 단계에서 다른 단계로 넘어가는 의식
- 사용 맥락: 성인식, 졸업, 입문 의례

### ritualize

- 구성: `ritual` + `-ize`
- 뜻: 의례화하다, 정해진 절차로 만들다
- 어원적 연결: 반복 행위를 공식 의식처럼 만듦
- 사용 맥락: `ritualized behavior` = 의례화된 행동

### ritualistic

- 발음: /ˌrɪtʃuəˈlɪstɪk/
- 구성: `ritual` + `-istic`
- 뜻: 의식적인, 의례를 중시하는
- 어원적 연결: 형식과 절차를 강조함
- 사용 맥락: `ritualistic practices` = 의례적 관행

### ritualism

- 구성: `ritual` + `-ism`
- 뜻: 의례주의, 형식주의적 의식 중시
- 어원적 연결: 내용보다 의식 형식을 강조하는 태도
- 사용 맥락: 종교·사회학 문맥

## 기억 포인트

`rit` 계열은 "정해진 절차로 행하는 의식" 이미지.

- `rite`: 의식
- `ritual`: 의례, 반복 관습
- `rite of passage`: 통과 의례
- `ritualize`: 의례화하다

## 비교 단어

### custom

- 뜻: 관습, 풍습
- 구분 포인트: `custom`은 사회적으로 익숙해진 관행. `ritual`은 절차와 상징성이 더 강함.
