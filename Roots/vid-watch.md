# vid-watch

- 상위 문서: [[vid]]
- 핵심 어원: 라틴어 `videre` / `vis-` = 보다
- 핵심 이미지: 위에서 내려다보거나 전체를 살핌
- 관련 갈래: [[vid-see]], [[vid-provide]]

## 단어들

### supervise

- 구성: `super-` = over + `vis`
- 뜻: 감독하다, 관리하다
- 어원적 연결: 위에서 내려다보며 살핌
- 사용 맥락: `supervise students` = 학생들을 감독하다

### supervisor

- 구성: `supervise` + `-or`
- 뜻: 감독자, 상사
- 어원적 연결: 위에서 보고 관리하는 사람
- 사용 맥락: `ask your supervisor` = 상사에게 물어보다

### survey

- 구성: `sur-` = over + `vey/view`
- 뜻: 조사하다, 둘러보다, 설문 조사
- 어원적 연결: 위에서 전체를 내려다보듯 살핌
- 사용 맥락: `conduct a survey` = 설문 조사를 실시하다

### surveillance

- 구성: French `surveiller` = watch over
- 뜻: 감시, 감독
- 어원적 연결: 위에서 계속 지켜봄
- 사용 맥락: `video surveillance` = 영상 감시

### overview

- 구성: `over` + `view`
- 뜻: 개요, 전반적 검토
- 어원적 연결: 위에서 전체를 한눈에 봄
- 사용 맥락: `give an overview` = 개요를 설명하다

### oversee

- 구성: `over` + `see`
- 뜻: 감독하다, 관리하다
- 어원적 연결: 위에서 내려다보며 살핌
- 사용 맥락: `oversee the project` = 프로젝트를 감독하다

## 기억 포인트

`vid-watch`는 "위에서 봄" 이미지.

- `supervise`: 위에서 감독함
- `survey`: 전체를 내려다보며 살핌
- `surveillance`: 계속 지켜봄
- `overview`: 위에서 전체를 봄
- `oversee`: 위에서 보고 관리함
