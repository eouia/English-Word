# plex

- 출발 단어: `complex`
- 핵심 어원: Latin `plectere` / `complecti` = 엮다, 얽다, 감싸다
- 핵심 이미지: 여러 요소가 함께 얽혀 하나의 구조를 이룸
- 관련 문서: [[plic]], [[junct]]
- 주의: [[plic]] = 접다 계열과 의미가 가깝지만, `plex`는 "얽히고 엮인 구조" 이미지가 강함.

## 출발 단어 분석

`complex` ← Latin `complexus`

- `com-`: together
- `plectere`: braid, weave
- `complecti`: embrace, comprise
- 원래 이미지: 함께 엮여 감싸인 상태
- 현대 의미: 복잡한, 복합체

## 단어들

### complex

- 발음: /ˈkɑmplɛks/
- 구성: `com-` = together + Latin `plectere`
- 뜻: 복잡한, 복합체
- 어원적 연결: 여러 부분이 함께 얽혀 있음
- 사용 맥락: `a complex problem` = 복잡한 문제

### complexity

- 발음: /kəˈmplɛksəti/
- 구성: `complex` + `-ity`
- 뜻: 복잡성
- 어원적 연결: 여러 요소가 얽힌 정도
- 사용 맥락: `the complexity of the system` = 시스템의 복잡성

### complexion

- 발음: /kəˈmplɛkʃən/
- 구성: Latin `complexio` = combination
- 뜻: 안색, 양상, 성격
- 어원적 연결: 여러 요소가 함께 얽혀 드러난 상태
- 사용 맥락: `a fair complexion` = 밝은 안색

### perplex

- 발음: /pɚˈplɛks/
- 구성: `per-` = thoroughly + Latin `plectere`
- 뜻: 당혹하게 하다, 혼란스럽게 하다
- 어원적 연결: 생각이 완전히 얽혀 헷갈림
- 사용 맥락: `perplex the audience` = 청중을 당혹하게 하다

### perplexed

- 발음: /pɚˈplɛkst/
- 구성: `perplex` + `-ed`
- 뜻: 당혹한, 어리둥절한
- 어원적 연결: 생각이 얽혀 풀리지 않는 상태
- 사용 맥락: `a perplexed look` = 당혹한 표정

### perplexing

- 발음: /pɚˈplɛksɪŋ/
- 구성: `perplex` + `-ing`
- 뜻: 당혹스러운, 난해한
- 어원적 연결: 생각을 얽히게 만들어 이해하기 어려움
- 사용 맥락: `a perplexing question` = 난해한 질문

### plexus

- 발음: /ˈplɛksəs/
- 구성: Latin `plexus` = braid, network
- 뜻: 신경총, 망상 조직
- 어원적 연결: 신경이나 혈관이 그물처럼 얽힘
- 사용 맥락: `solar plexus` = 태양신경총

## 기억 포인트

`plex` 계열은 "얽히고 엮인 구조" 이미지.

- `complex`: 함께 얽힌
- `complexity`: 얽힌 정도
- `complexion`: 여러 요소가 드러난 상태
- `perplex`: 생각을 얽히게 함
- `plexus`: 그물처럼 얽힌 조직

## 비교 단어

### plic

- 뜻: 접다, 접어 넣다
- 구분 포인트: [[plic]]는 접힘과 펼침. `plex`는 여러 요소가 엮여 복잡해진 구조.

### junct

- 뜻: 묶다, 연결하다
- 구분 포인트: [[junct]]는 이어 붙이는 연결점. `plex`는 연결된 것들이 얽혀 구조를 이룸.
