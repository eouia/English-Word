# cogn

- 출발 단어: `recognize`
- 핵심 어원: Latin `cognoscere` / `noscere` = know, learn, become acquainted with
- 핵심 이미지: 다시 알아봄, 지식으로 파악함
- 관련 어근: [[vid]], [[cred]], [[know]], Greek `gnos`

## 출발 단어 분석

`recognize`는 Latin `recognoscere`에서 온 말.

- `re-`: again
- `cognoscere`: know, learn

따라서 `recognize`는 "다시 알아보다" → "인식하다, 인정하다"의 뜻.

## 단어들

### recognize

- 구성: `re-` = again + `cognoscere`
- 의미: 알아보다, 인식하다, 인정하다
- 어원적 연결: 전에 알던 것을 다시 알아봄
- 사용 맥락: `recognize a face` = 얼굴을 알아보다

### recognition

- 구성: `recognize` + `-ion`
- 의미: 인식, 인정
- 어원적 연결: 알아보거나 공식적으로 인정하는 행위
- 사용 맥락: `gain recognition` = 인정을 받다

### recognizable

- 구성: `recognize` + `-able`
- 의미: 알아볼 수 있는
- 어원적 연결: 다시 보고 알 수 있음
- 사용 맥락: `a recognizable voice` = 알아들을 수 있는 목소리

### cognitive

- 구성: Latin `cognit-` = known + `-ive`
- 의미: 인지의, 사고의
- 어원적 연결: 알고 이해하는 정신 작용
- 사용 맥락: `cognitive ability` = 인지 능력

### cognition

- 구성: Latin `cognitio`
- 의미: 인지, 지식 작용
- 어원적 연결: 무언가를 알아 가는 과정
- 사용 맥락: `human cognition` = 인간 인지

### cognizant

- 구성: `cognize` 계열 + `-ant`
- 의미: 알고 있는, 인식하고 있는
- 어원적 연결: 어떤 사실을 지식으로 갖고 있음
- 사용 맥락: `be cognizant of risk` = 위험을 인식하고 있다

### incognito

- 구성: [[in]] = not + Latin `cognitus` = known
- 의미: 신분을 숨긴, 익명의
- 어원적 연결: 알려지지 않은 상태
- 사용 맥락: `travel incognito` = 신분을 숨기고 여행하다

### ignorant

- 구성: Latin `ignorare` = not know
- 의미: 무지한, 모르는
- 어원적 연결: 알지 못하는 상태
- 사용 맥락: `ignorant of the facts` = 사실을 모르는

### ignore

- 구성: Latin `ignorare` = not know, disregard
- 의미: 무시하다
- 어원적 연결: 알지 않으려 하거나 모르는 체함
- 사용 맥락: `ignore the warning` = 경고를 무시하다

## 기억 포인트

- `recognize`: 다시 알아봄
- `recognition`: 알아봄, 인정
- `cognitive`: 아는 정신 작용
- `cognition`: 인지
- `incognito`: 알려지지 않은 상태
- `ignorant`: 알지 못함
- `ignore`: 모르는 체함

## 비교 단어

- [[vid]]
  - 핵심: 보다, 보아서 판단하다
  - 구분 포인트: `recognize`는 보고 "알아봄". `vid`는 보는 행위와 시각 판단

- [[cred]]
  - 핵심: 믿다
  - 구분 포인트: `recognize`는 알아보거나 인정함. `credit`은 믿거나 공로를 인정함

- Greek `gnos`
  - 핵심: 알다
  - 구분 포인트: `diagnosis`, `prognosis`, `agnostic`은 Greek `gignoskein/gnosis` 계열이라 `cogn` 문서에 섞지 않음

- [[know]]
  - 핵심: 알다
  - 구분 포인트: Germanic 기본어. 멀리 같은 인도유럽계 뿌리와 닿지만 문서상 `cogn`과 분리
