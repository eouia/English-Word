# limin

- 출발 단어: `preliminary`
- 핵심 어원: Latin `limen`, `liminis` = 문턱, 경계
- 핵심 이미지: 들어가기 전의 문턱과 경계
- 관련 문서: [[term]], [[fin]]

## 출발 단어 분석

`preliminary` ← Latin `prae-` + `limen`

- `prae-`: before
- `limen`: threshold
- 원래 이미지: 본격적으로 들어가기 전 문턱에 있는 것
- 현대 의미: 예비의, 사전의

## 단어들

### preliminary

- 발음: /prɪˈlɪməˌnɛri/
- 구성: `pre-` = before + Latin `limen`
- 뜻: 예비의, 사전의; 예선
- 어원적 연결: 본 단계의 문턱 앞에 있음
- 사용 맥락: `preliminary results` = 예비 결과

### liminal

- 구성: Latin `limen` + `-al`
- 뜻: 경계의, 전환기의
- 어원적 연결: 한 상태에서 다른 상태로 넘어가는 문턱에 있음
- 사용 맥락: `a liminal space` = 경계적 공간

### subliminal

- 발음: /səˈblɪmɪnəl/
- 구성: `sub-` = below + Latin `limen`
- 뜻: 잠재의식의, 식역하의
- 어원적 연결: 의식의 문턱 아래에 있음
- 사용 맥락: `subliminal messages` = 잠재의식에 작용하는 메시지

### eliminate

- 발음: /ɪˈlɪməˌneɪt/
- 구성: `e-/ex-` = out + Latin `limen`
- 뜻: 제거하다, 탈락시키다
- 어원적 연결: 문턱 밖으로 내보냄
- 사용 맥락: `eliminate errors` = 오류를 제거하다

### elimination

- 발음: /ɪˌlɪməˈneɪʃən/
- 구성: `eliminate` + `-ion`
- 뜻: 제거, 탈락
- 어원적 연결: 경계 밖으로 밀어내는 과정
- 사용 맥락: `single-elimination tournament` = 단판 탈락 토너먼트

## 기억 포인트

`limin` 계열은 "문턱과 경계" 이미지.

- `preliminary`: 문턱 앞의 예비 단계
- `liminal`: 경계에 있는
- `subliminal`: 의식의 문턱 아래 있는
- `eliminate`: 문턱 밖으로 내보냄
