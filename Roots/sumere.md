# sumere

- 출발 단어: `assume`
- 핵심 어원: Latin `sumere` / `sumpt-` = take, take up
- 핵심 이미지: 집어 들다, 자기 쪽으로 취하다
- 관련 어근: [[cap]], [[prehend]], [[sum]]

## 출발 단어 분석

`assume`은 Latin `assumere`에서 온 말.

- `ad-`: to
- `sumere`: take, take up

따라서 `assume`은 "자기 쪽으로 취함"에서 "가정하다, 역할을 맡다"의 뜻.

`sumere`는 `summa` = 합계, 최고점 계열과 겉모양이 비슷하지만 문서상 분리해서 외움.

## 단어들

### assume

- 구성: `ad-` = to + `sumere`
- 의미: 가정하다, 맡다
- 어원적 연결: 어떤 생각이나 역할을 자기 쪽으로 취함
- 사용 맥락: `assume responsibility` = 책임을 맡다

### assumption

- 구성: `ad-` + `sumpt-` + `-ion`
- 의미: 가정, 떠맡음
- 어원적 연결: 어떤 생각이나 역할을 취함
- 사용 맥락: `a false assumption` = 잘못된 가정

### consume

- 구성: `con-` = completely + `sumere`
- 의미: 소비하다, 다 써버리다
- 어원적 연결: 완전히 가져가 써 없앰
- 사용 맥락: `consume energy` = 에너지를 소비하다

### consumer

- 구성: `consume` + `-er`
- 의미: 소비자
- 어원적 연결: 물건이나 서비스를 가져다 쓰는 사람
- 사용 맥락: `consumer behavior` = 소비자 행동

### consumption

- 구성: `con-` + `sumpt-` + `-ion`
- 의미: 소비, 소모
- 어원적 연결: 완전히 가져다 써버림
- 사용 맥락: `energy consumption` = 에너지 소비

### presume

- 구성: `pre-` = before + `sumere`
- 의미: 추정하다, 감히 하다
- 어원적 연결: 미리 어떤 생각을 취함
- 사용 맥락: `presume innocence` = 무죄로 추정하다

### presumption

- 구성: `pre-` + `sumpt-` + `-ion`
- 의미: 추정, 주제넘음
- 어원적 연결: 미리 취한 판단
- 사용 맥락: `the presumption of innocence` = 무죄 추정

### resume

- 구성: `re-` = again + `sumere`
- 의미: 다시 시작하다, 재개하다
- 어원적 연결: 다시 집어 들고 이어 감
- 사용 맥락: `resume work` = 일을 재개하다

### resumption

- 구성: `re-` + `sumpt-` + `-ion`
- 의미: 재개
- 어원적 연결: 다시 집어 들어 이어 가는 행위
- 사용 맥락: `the resumption of talks` = 회담 재개

## 기억 포인트

- `assume`: 생각이나 역할을 취함
- `assumption`: 취해 둔 가정
- `consume`: 완전히 가져다 씀
- `presume`: 미리 취해 판단함
- `resume`: 다시 집어 들어 이어 감

## 비교 단어

- [[sum]]
  - 핵심: 합계, 최고점, 요약
  - 구분 포인트: `sum`은 Latin `summa`. `sumere`는 취하다, 집어 들다

- [[cap]]
  - 핵심: 잡다, 취하다
  - 구분 포인트: `capere` 계열은 `capture`, `receive`, `accept`처럼 잡거나 받는 느낌. `sumere`는 자기 쪽으로 집어 들어 취하는 느낌

- [[prehend]]
  - 핵심: 붙잡다, 이해하다
  - 구분 포인트: `prehend`는 손으로 붙잡는 이미지가 강함. `sumere`는 취해서 맡거나 써버리는 방향
