# pack

- 출발 단어: `package`
- 핵심 어원: Middle Dutch / Middle Low German `pak` = 꾸러미, 짐, 묶음
- 핵심 이미지: 물건을 싸서 한 덩어리로 묶음
- 관련 형태: `pack`, `package`, `packet`, `packing`, `packaging`
- 관련 문서: [[pact]]
- 주의: `pack`은 의미상 "묶다" 이미지가 있지만 보통 게르만계 `pak` 계열. `pact/compact/impact`의 Latin `pangere/pacisci` 계열과 구분.

## 출발 단어 분석

`package`는 `pack` + `-age`로 분석.

- `pack`: 꾸러미, 짐을 싸다
- `-age`: 결과물, 상태, 집합을 만드는 접미사

따라서 `package`는 "싸서 묶은 것" → "소포, 포장물, 패키지"의 뜻.

## 단어들

### pack

- 구성: Middle English `pak`
- 뜻: 꾸러미, 묶음, 짐을 싸다
- 어원적 연결: 천이나 상품을 싸서 한 덩어리로 만든 물건
- 사용 맥락: `pack a suitcase` = 여행가방을 싸다, `a pack of cards` = 카드 한 벌

### package

- 구성: `pack` + `-age`
- 뜻: 소포, 포장물, 패키지, 일괄 상품
- 어원적 연결: 싸서 하나로 묶인 물건에서 상품·서비스 묶음으로 확장
- 사용 맥락: `a software package` = 소프트웨어 패키지, `a package deal` = 일괄 거래

### packaging

- 구성: `package` + `-ing`
- 뜻: 포장, 포장재, 상품을 제시하는 방식
- 어원적 연결: 물건을 싸는 행위에서 상품의 외형·제시 방식으로 확장
- 사용 맥락: `eco-friendly packaging` = 친환경 포장재

### packet

- 구성: `pack` + diminutive `-et`
- 뜻: 작은 꾸러미, 작은 봉지, 정보 묶음
- 어원적 연결: 작은 `pack`. 우편물 묶음에서 작은 포장 단위, 데이터 단위로 확장
- 사용 맥락: `a packet of sugar` = 설탕 한 봉지, `a data packet` = 데이터 패킷

### backpack

- 구성: `back` + `pack`
- 뜻: 배낭, 배낭을 메고 여행하다
- 어원적 연결: 등에 메는 꾸러미
- 사용 맥락: `go backpacking across Europe` = 유럽을 배낭여행하다

### backpacker

- 구성: `backpack` + `-er`
- 뜻: 배낭여행자
- 어원적 연결: 배낭을 메고 이동하는 사람
- 사용 맥락: `budget backpackers` = 저예산 배낭여행자들

### unpack

- 구성: `un-` + `pack`
- 뜻: 짐을 풀다, 내용을 자세히 설명하다
- 어원적 연결: 싸 둔 묶음을 풀어 안의 것을 드러냄
- 사용 맥락: `unpack the argument` = 논지를 하나씩 풀어 설명하다

### repack

- 구성: `re-` + `pack`
- 뜻: 다시 싸다, 다시 포장하다
- 어원적 연결: 이미 싸인 것을 새 방식으로 다시 묶음
- 사용 맥락: `repack the boxes` = 상자들을 다시 포장하다

### packed

- 구성: `pack` + `-ed`
- 뜻: 꽉 찬, 빽빽한
- 어원적 연결: 물건을 빈틈없이 눌러 담은 상태
- 사용 맥락: `a packed schedule` = 빽빽한 일정, `a packed train` = 꽉 찬 기차

## 비교 단어

### pact

- 구성: Latin `pactum` ← `pacisci`
- 뜻: 협정, 조약
- 어원적 연결: [[pact]] 계열. 약속으로 서로를 묶음
- 구분 포인트: `pack`은 물건을 싸서 묶는 게르만계 이미지, `pact`는 합의로 묶는 라틴계 이미지

### compact

- 구성: `com-` + Latin `pangere`
- 뜻: 조밀한, 소형의, 협정
- 어원적 연결: [[pact]] 계열. 함께 단단히 고정하거나 묶음
- 구분 포인트: `compact`는 철자상 `pack`처럼 보이지만 `pack`에서 나온 단어가 아님

## 주의할 의미

### pack the cards / pack the jury

- 뜻: 카드를 조작하다, 배심원을 자기편으로 채우다
- 주의: 이 `pack`의 조작·담합 의미는 Elizabethan 시대에 `pact`를 잘못 발음한 영향으로 설명됨
- 학습 포인트: 일반 `pack` = 싸다, 채우다. 조작 의미는 [[pact]] 쪽 의미가 섞인 특수 갈래로 기억.

## 기억 포인트

`pack`은 "싸서 한 묶음으로 만들다"의 이미지.

- `pack`: 묶음, 짐을 싸다
- `package`: 싸서 묶은 포장물
- `packet`: 작은 꾸러미
- `backpack`: 등에 메는 꾸러미
- `unpack`: 묶음을 풀어 내용 드러냄
- `packed`: 빈틈없이 채워진 상태
