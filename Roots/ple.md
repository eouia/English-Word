# ple

- 출발 단어: `complete`
- 핵심 어원: 라틴어 `plere` = 채우다
- 핵심 의미: 빈 곳을 채워 완전하게 만듦
- 관련 문서: [[con]], [[fac]]
- 주의: `full`, `fill`은 게르만계 기본어. 깊은 뿌리는 가깝지만 라틴어 `plere` 직접 파생어와 분리.

## 출발 단어 분석

`complete` = `com-` + `plere`

- `com-`: 함께, 완전히
- `plere`: 채우다
- 원래 이미지: 빠진 부분 없이 모두 채움
- 현대 의미: 완전한, 완료하다

## 단어들

### complete

- 발음: /kəˈmplit/
- 구성: `com-` = completely + `ple`
- 뜻: 완전한, 완료하다
- 어원적 연결: 빠진 부분 없이 전부 채움
- 사용 맥락: `complete the task` = 일을 완료하다

### completion

- 발음: /kəˈmpliʃən/
- 구성: `complete` + `-ion`
- 뜻: 완료, 완성
- 어원적 연결: 채워서 끝낸 상태나 행위
- 사용 맥락: `project completion` = 프로젝트 완료

### completely

- 발음: /kəˈmplitli/
- 구성: `complete` + `-ly`
- 뜻: 완전히
- 어원적 연결: 빠짐없이 다 채워진 정도
- 사용 맥락: `completely different` = 완전히 다른

### incomplete

- 발음: /ˌɪnkəˈmplit/
- 구성: `in-` = not + `complete`
- 뜻: 불완전한, 미완성의
- 어원적 연결: 아직 채워지지 않은 부분이 있음
- 사용 맥락: `incomplete data` = 불완전한 데이터

### complement

- 발음: /ˈkɑmpləmənt/
- 구성: `com-` + `ple`
- 뜻: 보완물, 보완하다
- 어원적 연결: 부족한 부분을 채워 전체를 완성함
- 사용 맥락: `skills that complement each other` = 서로 보완하는 기술

### complementary

- 발음: /ˌkɑmpləˈmɛntri/
- 구성: `complement` + `-ary`
- 뜻: 상호 보완적인
- 어원적 연결: 서로 빠진 부분을 채워 완전하게 만듦
- 사용 맥락: `complementary roles` = 상호 보완적 역할

### supplement

- 발음: /ˈsʌpləmənt/
- 구성: `sub-` = under, in addition + `ple`
- 뜻: 보충하다, 보충물
- 어원적 연결: 부족한 곳에 더 넣어 채움
- 사용 맥락: `supplement the diet` = 식단을 보충하다

### supplementary

- 발음: /ˌsʌpləˈmɛntɚi/
- 구성: `supplement` + `-ary`
- 뜻: 보충의, 추가의
- 어원적 연결: 본문이나 기본 부분을 더 채움
- 사용 맥락: `supplementary materials` = 보충 자료

### deplete

- 발음: /dɪˈplit/
- 구성: `de-` = down, away + `ple`
- 뜻: 고갈시키다, 줄이다
- 어원적 연결: 채워진 것을 비워 냄
- 사용 맥락: `deplete resources` = 자원을 고갈시키다

### depletion

- 발음: /dɪˈpliʃən/
- 구성: `deplete` + `-ion`
- 뜻: 고갈, 감소
- 어원적 연결: 채워진 양이 빠져나간 상태
- 사용 맥락: `ozone depletion` = 오존층 고갈

### replete

- 발음: /riˈplit/
- 구성: `re-` = again, fully + `ple`
- 뜻: 가득 찬, 풍부한
- 어원적 연결: 다시 또는 충분히 채워진 상태
- 사용 맥락: `a report replete with errors` = 오류로 가득한 보고서

### replenish

- 발음: /riˈplɛnɪʃ/
- 구성: `re-` = again + Old French `pleniss-` ← Latin `plenus`
- 뜻: 다시 채우다, 보충하다
- 어원적 연결: 비거나 줄어든 것을 다시 가득하게 채움
- 사용 맥락: `replenish supplies` = 물자를 보충하다

### replenishment

- 발음: /rɪˈplɛnɪʃmənt/
- 구성: `replenish` + `-ment`
- 뜻: 보충, 재공급
- 어원적 연결: 줄어든 양을 다시 채우는 행위
- 사용 맥락: `inventory replenishment` = 재고 보충

### plenty

- 발음: /ˈplɛnti/
- 구성: Old French `plente` ← Latin `plenitas`
- 뜻: 충분함, 풍부함
- 어원적 연결: 가득 찬 상태
- 사용 맥락: `plenty of time` = 충분한 시간

### plentiful

- 발음: /ˈplɛntəfəl/
- 구성: `plenty` + `-ful`
- 뜻: 풍부한
- 어원적 연결: 채워져 많음
- 사용 맥락: `plentiful resources` = 풍부한 자원

### plenary

- 발음: /ˈplɛnɚi/
- 구성: Latin `plenus` = full
- 뜻: 전체의, 전원 참석의
- 어원적 연결: 빠진 사람이나 부분 없이 가득 찬 전체
- 사용 맥락: `a plenary session` = 전체 회의

### implement

- 발음: /ˈɪmpləmənt/
- 구성: Latin `implere` = fill up
- 뜻: 실행하다, 도구
- 어원적 연결: 필요한 것을 채워 실제로 이루어지게 함
- 사용 맥락: `implement a policy` = 정책을 실행하다

### accomplish

- 발음: /əˈkɑmplɪʃ/
- 구성: Old French `acomplir` ← Latin `complere`
- 뜻: 성취하다, 완수하다
- 어원적 연결: 해야 할 일을 채워 끝까지 완성함
- 사용 맥락: `accomplish a goal` = 목표를 성취하다

### accomplishment

- 발음: /əˈkɑmplɪʃmənt/
- 구성: `accomplish` + `-ment`
- 뜻: 성취, 업적, 완수
- 어원적 연결: 일을 끝까지 채워 완성한 결과
- 사용 맥락: `a major accomplishment` = 중대한 성취

## 기억 포인트

- `ple`의 기본 그림은 “채움”.
- `complete`: 다 채워서 끝냄.
- `complement`: 부족한 부분을 채워 전체를 완성함.
- `supplement`: 추가로 채움.
- `deplete`: 채워진 것을 비워 고갈시킴.
- `plenty/plenary`: 가득 참, 충분함.
- `replenish`: 줄어든 것을 다시 채움.
- `accomplish`: 끝까지 채워 성취함.

## 비교 단어

### full / fill

- `full`, `fill`은 게르만계 기본어.
- 깊은 PIE 층에서는 `plere`와 같은 “채우다/가득하다” 이미지와 연결될 수 있음.
- 영어 문서에서는 라틴어 `ple` 직접 파생어와 분리.

### perfect

- [[fac]]의 `perfect`는 `per-` + `facere` = 완전히 만들다.
- `complete`와 뜻은 가깝지만 어원은 다름.

### comply

- `comply`는 역사적으로 `complete/compliment` 계열과 접촉했지만 현대 뜻은 “따르다, 응하다”.
- 시험 학습에서는 `complete`와 바로 같은 목록에 넣지 않는 편이 안전.
