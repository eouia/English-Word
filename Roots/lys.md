# lys

- 출발 단어: `analysis`
- 핵심 어원: 그리스어 `lysis` / `lyein` = 풀다, 느슨하게 하다, 해체하다
- 핵심 의미: 묶인 것을 풀어 부분으로 나눔
- 관련 문서: [[solv]], [[sym]], [[cid]]
- 주의: `solve/solution`은 라틴어 `solvere` 계열. 뜻은 가깝지만 `lysis`와 어원 분리.

## 출발 단어 분석

`analysis` = `ana-` + `lysis`

- `ana-`: up, back, throughout
- `lysis`: 풀기, 해체
- 원래 이미지: 복잡하게 묶인 것을 풀어 구성 요소로 나눔
- 현대 의미: 분석, 해석, 검토

## 단어들

### analysis

- 구성: `ana-` + `lysis`
- 뜻: 분석, 해석
- 어원적 연결: 복잡한 것을 풀어 부분과 구조를 봄
- 사용 맥락: `data analysis` = 데이터 분석

### analyze

- 구성: `ana-` + `lyze`
- 뜻: 분석하다
- 어원적 연결: 문제나 자료를 풀어 작은 요소로 나눔
- 사용 맥락: `analyze the results` = 결과를 분석하다

### analyst

- 구성: `analysis` + `-ist`
- 뜻: 분석가
- 어원적 연결: 복잡한 자료를 풀어 해석하는 사람
- 사용 맥락: `a financial analyst` = 금융 분석가

### analytic

- 구성: `analysis` 계열 형용사
- 뜻: 분석적인, 해석적인
- 어원적 연결: 전체를 풀어 요소별로 따지는 성질
- 사용 맥락: `analytic philosophy` = 분석철학

### analytical

- 구성: `analytic` + `-al`
- 뜻: 분석적인
- 어원적 연결: 문제를 쪼개어 따지는 방식
- 사용 맥락: `analytical thinking` = 분석적 사고

### analytics

- 구성: `analytic` + `-s`
- 뜻: 분석학, 데이터 분석
- 어원적 연결: 자료를 풀어 패턴과 의미를 찾는 체계
- 사용 맥락: `web analytics` = 웹 분석

### catalysis

- 구성: `cata-` = down, completely + `lysis`
- 뜻: 촉매 작용
- 어원적 연결: 반응을 풀어 진행하게 만드는 과정
- 사용 맥락: `enzyme catalysis` = 효소 촉매 작용

### catalyst

- 구성: `cata-` + `lys` + `-t`
- 뜻: 촉매, 변화를 일으키는 것
- 어원적 연결: 묶인 반응을 풀어 변화가 일어나게 함
- 사용 맥락: `a catalyst for change` = 변화의 촉매

### dialysis

- 구성: [[dia]] = through, apart + `lysis`
- 뜻: 투석
- 어원적 연결: 막을 통해 분리하고 풀어 냄
- 사용 맥락: `kidney dialysis` = 신장 투석

### electrolysis

- 구성: `electro-` + `lysis`
- 뜻: 전기분해
- 어원적 연결: 전기로 물질을 분해함
- 사용 맥락: `water electrolysis` = 물 전기분해

### hydrolysis

- 구성: `hydro-` = water + `lysis`
- 뜻: 가수분해
- 어원적 연결: 물을 이용해 결합을 풀어 나눔
- 사용 맥락: `protein hydrolysis` = 단백질 가수분해

### paralysis

- 구성: `para-` = beside, beyond + `lysis`
- 뜻: 마비
- 어원적 연결: 몸의 힘이 풀려 움직이지 못하는 상태
- 사용 맥락: `temporary paralysis` = 일시적 마비

### paralyze

- 구성: `para-` + `lyze`
- 뜻: 마비시키다
- 어원적 연결: 힘이나 기능을 풀어 멈추게 함
- 사용 맥락: `paralyze the system` = 시스템을 마비시키다

## 기억 포인트

- `lys`는 “풀다, 해체하다”.
- `analysis`: 복잡한 것을 풀어 부분으로 봄.
- `catalyst`: 반응이 풀려 진행되게 하는 것.
- `dialysis`: 막을 통해 분리해 풀어 냄.
- `paralysis`: 힘이 풀려 움직이지 못함.

## 비교 단어

### synthesis

- [[sym]]의 `synthesis`는 요소를 함께 놓아 종합함.
- `analysis`는 반대로 전체를 풀어 요소로 나눔.

### solve / solution

- [[solv]]는 라틴어 `solvere` = 풀다, 해결하다.
- `analysis`와 뜻은 가깝지만 그리스어 `lysis` 계열은 따로 관리.

### loose / lose

- `loose`, `lose`는 게르만계 단어.
- 더 깊은 PIE 층에서는 “느슨하게 하다/풀다” 이미지가 닿지만, 영어 학습 문서에서는 `lys`와 분리.
