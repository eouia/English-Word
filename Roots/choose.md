# choose

- 출발 단어: `choice`
- 핵심 어원: Old English `ceosan` = 고르다, 시험하다, 맛보다
- 핵심 의미: 여러 가능성 중 시험해 보고 고름
- 관련 문서: [[lect]], [[cap]], [[val]]
- 주의: `choice`는 Old French `chois`를 거쳐 들어왔지만, `choose`와 같은 게르만계 뿌리로 연결됨.

## 출발 단어 분석

`choice` ← Old French `chois` ← `choisir`

- `choisir`: 고르다, 구별하다, 알아보다
- 게르만계 `choose`와 관련
- 더 깊은 뿌리: PIE `geus-` = 맛보다, 선택하다
- 원래 이미지: 맛보고 시험해 본 뒤 고름

## 단어들

### choice

- 구성: Old French `chois`
- 뜻: 선택, 선택권, 선택된 것
- 어원적 연결: 여러 가능성 중 골라낸 것
- 사용 맥락: `make a choice` = 선택하다

### choose

- 구성: Old English `ceosan`
- 뜻: 고르다, 선택하다
- 어원적 연결: 시험해 보고 더 알맞은 것을 고름
- 사용 맥락: `choose a topic` = 주제를 고르다

### chose

- 구성: `choose`의 과거형
- 뜻: 골랐다
- 어원적 연결: 선택 행위가 과거에 완료됨
- 사용 맥락: `She chose wisely.` = 그녀는 현명하게 골랐다

### chosen

- 구성: `choose`의 과거분사
- 뜻: 선택된
- 어원적 연결: 여러 것 중 골라진 상태
- 사용 맥락: `the chosen candidate` = 선택된 후보

### choosing

- 구성: `choose` + `-ing`
- 뜻: 선택하기, 고르는 중
- 어원적 연결: 가능성들을 비교해 고르는 과정
- 사용 맥락: `choosing between options` = 선택지 사이에서 고르기

### chooser

- 구성: `choose` + `-er`
- 뜻: 고르는 사람
- 어원적 연결: 선택 행위를 하는 사람
- 사용 맥락: `the chooser of the prize` = 상을 고르는 사람

### choosy

- 구성: `choose` + `-y`
- 뜻: 까다로운, 가리는
- 어원적 연결: 아무거나 고르지 않고 세심하게 고름
- 사용 맥락: `choosy customers` = 까다로운 고객

### choice adjective

- 구성: `choice`의 형용사화
- 뜻: 최상의, 엄선된
- 어원적 연결: 골라 뽑을 만큼 좋은 것
- 사용 맥락: `choice cuts of meat` = 엄선된 고기 부위

## 기억 포인트

- `choose/choice`는 “맛보고 시험해 보고 고름”.
- `choice`는 명사로 “선택”, 형용사로 “엄선된, 좋은”.
- `choosy`는 선택 기준이 까다로운 사람.

## 비교 단어

### select

- [[lect]]의 `select`는 라틴어 `seligere` = 따로 골라내다 계열.
- `choose`는 게르만계 기본어, `select`는 더 격식 있는 라틴계 단어.

### elect

- [[lect]]의 `elect`도 라틴어 `eligere` = 골라내다 계열.
- 선거·공식 선택에서 자주 쓰임.

### option

- `option`은 라틴어 `optare` = 바라다, 선택하다 계열.
- 의미는 가깝지만 `choose/choice`와 어원이 다름.
