# scrut

- 출발 단어: `scrutinize`
- 핵심 어원: Latin `scrutari` = search, examine
- 핵심 이미지: 구석구석 뒤져 세밀하게 살핌
- 관련 문서: [[scope]], [[quest]]
- 주의: `scrib` = 쓰다 계열과 철자가 비슷하지만 어원·의미가 다름.

## 출발 단어 분석

`scrutinize` ← `scrutiny` ← Latin `scrutinium`

- `scrutari`: 뒤지다, 살피다, 조사하다
- `scrutinium`: 세밀한 조사, 검토
- 원래 이미지: 작은 것까지 뒤져 살핌
- 현대 의미: 면밀히 조사하다, 자세히 검토하다

## 단어들

### scrutinize

- 구성: `scrutiny` + `-ize`
- 뜻: 면밀히 조사하다, 자세히 검토하다
- 어원적 연결: 숨은 부분이나 작은 점까지 뒤져 살핌
- 사용 맥락: `scrutinize the evidence` = 증거를 면밀히 검토하다

### scrutiny

- 구성: Latin `scrutinium`
- 뜻: 면밀한 조사, 정밀 검토, 감시
- 어원적 연결: 세부까지 파고들어 살펴보는 행위
- 사용 맥락: `under public scrutiny` = 대중의 감시 아래

### scrutinized

- 구성: `scrutinize` + `-ed`
- 뜻: 면밀히 검토된, 조사받은
- 어원적 연결: 자세히 들여다보아 평가된 상태
- 사용 맥락: `closely scrutinized data` = 면밀히 검토된 자료

### scrutinizing

- 구성: `scrutinize` + `-ing`
- 뜻: 면밀히 살피는
- 어원적 연결: 세부를 놓치지 않고 들여다보는 동작
- 사용 맥락: `a scrutinizing gaze` = 세밀히 살피는 시선

## 기억 포인트

`scrut` 계열은 "뒤져서 세밀히 살핌" 이미지.

- `scrutinize`: 면밀히 조사하다
- `scrutiny`: 정밀 검토, 감시
- `scrutinized`: 면밀히 검토된
- `scrutinizing`: 세밀히 살피는

## 비교 단어

### scope

- 뜻: 보다, 살피다, 보는 도구
- 구분 포인트: [[scope]]는 보는 도구나 범위의 이미지. `scrut`는 작은 것까지 뒤져 검사하는 면밀함.

### search

- 뜻: 찾다, 수색하다
- 구분 포인트: `search`는 대상을 찾는 넓은 행위. `scrutinize`는 이미 있는 대상의 세부를 꼼꼼히 검토함.

### scrib

- 뜻: 쓰다
- 구분 포인트: [[scrib]]는 쓰기와 기록. `scrut`는 조사와 검토.
