# shift

- 출발 단어: `shift`
- 핵심 의미: 옮기다, 바꾸다, 위치나 상태를 전환하다
- 관련 형태: Old English `sciftan` = arrange, divide, assign
- 관련 문서: [[mut]]

## 핵심 이미지

`shift` 계열의 바탕은 "나누어 배치하고, 다른 자리로 옮김".

위치가 바뀌면 `shift`.
일정한 근무 구간도 나누어 배치한 것이므로 `shift`.
임시로 어떻게든 맞춰 만든 것은 `makeshift`.
생각이나 책임을 다른 쪽으로 옮기는 것도 `shift`.

## 단어들

### shift

- 구성: Old English `sciftan` = arrange, divide, assign
- 뜻: 옮기다, 바꾸다; 변화, 전환; 교대 근무
- 연결: 어떤 것을 다른 자리나 상태로 배치함
- 사용 맥락: `shift the focus` = 초점을 옮기다, `work the night shift` = 야간 근무를 하다

### shifted

- 구성: `shift` + `-ed`
- 뜻: 옮겨진, 바뀐
- 연결: 이미 다른 위치나 상태로 이동함
- 사용 맥락: `shifted priorities` = 바뀐 우선순위

### shifting

- 구성: `shift` + `-ing`
- 뜻: 변하는, 이동하는
- 연결: 위치나 조건이 계속 바뀜
- 사용 맥락: `shifting patterns` = 변화하는 패턴

### shiftless

- 구성: `shift` + `-less`
- 뜻: 무기력한, 게으른, 대책 없는
- 연결: 원래 `shift`에는 "방책, 수단"의 뜻도 있었음. `shiftless`는 수단이나 능력이 없는 상태.
- 사용 맥락: 문학적·비판적 표현으로 자주 보임.

### makeshift

- 구성: `make` + `shift`
- 뜻: 임시변통의, 임시방편
- 연결: 있는 수단을 써서 어떻게든 맞춰 배치함
- 사용 맥락: `a makeshift shelter` = 임시 대피소

### gearshift

- 구성: `gear` + `shift`
- 뜻: 변속 장치
- 연결: 기어를 다른 위치로 옮겨 동력 전달 상태를 바꿈
- 사용 맥락: 자동차나 기계 문맥.

### paradigm shift

- 구성: `paradigm` + `shift`
- 뜻: 패러다임 전환
- 연결: 생각의 틀이나 기준이 다른 자리로 옮겨 감
- 사용 맥락: `a paradigm shift in science` = 과학의 패러다임 전환

### shift key

- 구성: `shift` + `key`
- 뜻: 시프트 키
- 연결: 키 입력 상태를 다른 모드로 전환함
- 사용 맥락: `Press the Shift key.` = 시프트 키를 누르라

## 비교 단어

`mutate`, `commute`, `mutable`은 [[mut]] 계열.
둘 다 "바뀜"을 말하지만, `mut`는 라틴어 기반의 변화·교환 이미지가 강하고, `shift`는 자리·방향·상태를 옮겨 배치하는 느낌이 강함.

`shape`와 `shaft`는 철자가 비슷해 보여도 `shift` 계열로 묶지 않음.
