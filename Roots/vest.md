# vest

- 출발 단어: `invest`
- 핵심 어원: 라틴어 `vestire` = 옷을 입히다, 덮다
- 핵심 이미지: 옷을 입힘 / 권한을 입힘 / 돈을 어떤 곳에 씌워 맡김
- 관련해서 헷갈릴 수 있는 어근: [[vestig]] = 발자국, 흔적 계열, [[cover]] = 덮다, 가리다 계열

## 출발 단어 분석

`invest`는 `in-` + `vest`.

- `in-`: in, into
- `vest`: clothe, dress

따라서 `invest`는 원래 "옷을 입히다"의 뜻. 여기서 "공식 옷이나 상징을 입혀 직위에 앉히다" → "권한을 부여하다" → "돈을 어떤 대상에 넣어 맡기다"의 뜻으로 확장.

## 단어들

### invest

- 구성: `in-` = into + `vest`
- 뜻: 투자하다; 권한을 부여하다
- 어원적 연결: 어떤 대상에 옷을 입히듯 자원이나 권한을 씌움
- 사용 맥락: `invest in education` = 교육에 투자하다

### investment

- 구성: `invest` + `-ment`
- 뜻: 투자, 투자금
- 어원적 연결: 돈이나 노력을 어떤 대상에 넣어 맡긴 것
- 사용 맥락: `a long-term investment` = 장기 투자

### investor

- 구성: `invest` + `-or`
- 뜻: 투자자
- 어원적 연결: 돈이나 자원을 특정 대상에 맡기는 사람
- 사용 맥락: `foreign investors` = 외국인 투자자

### vested

- 구성: `vest` + `-ed`
- 뜻: 기득의, 확정된, 권리가 부여된
- 어원적 연결: 권한이나 권리가 옷처럼 입혀져 있음
- 사용 맥락: `vested interests` = 기득권

### divest

- 구성: `di-`, `dis-` = away + `vest`
- 뜻: 처분하다, 빼앗다, 벗기다
- 어원적 연결: 입혀진 권리나 자산을 벗겨 냄
- 사용 맥락: `divest assets` = 자산을 처분하다

### vest

- 구성: Latin `vestis` = garment
- 뜻: 조끼, 속옷
- 어원적 연결: 몸에 입는 옷
- 사용 맥락: `a bulletproof vest` = 방탄조끼

### vestment

- 구성: `vest` + `-ment`
- 뜻: 예복, 제의
- 어원적 연결: 공식적 의미를 가진 옷
- 사용 맥락: `ceremonial vestments` = 의식용 예복

## 기억 포인트

`vest` 계열은 "옷을 입히다"에서 출발.

- `vest`: 옷, 조끼
- `invest`: 안에 옷을 입힘 → 권한/돈을 부여함 → 투자하다
- `investment`: 맡겨 넣은 돈이나 노력
- `vested`: 권리가 입혀져 확정된
- `divest`: 입혀진 것을 벗김 → 처분하다, 박탈하다

## 비교 단어

### investigate

- 뜻: 조사하다, 수사하다
- 구분 포인트: [[vestig]] 계열. `investigate`는 Latin `vestigium` = footprint, trace에서 온 단어. `invest`의 `vestire` = clothe와 어원이 다름

### cover

- 뜻: 덮다, 가리다
- 구분 포인트: [[cover]]는 덮어 감추는 이미지. `vest`는 옷을 입히거나 권한을 입히는 이미지
