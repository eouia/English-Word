# calcitr

- 출발 단어: `recalcitrant`
- 핵심 어원: Latin `calcitrare` = kick with the heel
- 핵심 이미지: 뒤로 발길질하며 버티고 따르지 않음
- 관련 문서: [[calc]], [[bell]]
- 주의: `calculate`의 [[calc]]와 철자가 닮았지만 어원·의미가 다름. `calcitr`는 발길질과 저항 이미지.

## 출발 단어 분석

`recalcitrant` ← Latin `recalcitrans` ← `recalcitrare`

- `re-`: back, against
- `calcitrare`: 발꿈치로 차다, 발길질하다
- 원래 이미지: 뒤로 발길질하며 끌려가지 않으려 버팀
- 현대 의미: 말을 듣지 않는, 고집 센, 반항적인

## 단어들

### recalcitrant

- 구성: `re-` = back/against + Latin `calcitrare`
- 뜻: 고집 센, 반항적인, 말을 안 듣는
- 어원적 연결: 발길질하며 뒤로 버티는 사람이나 태도
- 사용 맥락: `a recalcitrant student` = 말을 잘 듣지 않는 학생

### recalcitrance

- 구성: `recalcitrant` + `-ance`
- 뜻: 완강한 저항, 고집스러운 반항
- 어원적 연결: 지시나 통제에 맞서 버티는 상태
- 사용 맥락: `bureaucratic recalcitrance` = 관료 조직의 완강한 저항

### recalcitrantly

- 구성: `recalcitrant` + `-ly`
- 뜻: 고집스럽게, 반항적으로
- 어원적 연결: 발길질하듯 따르지 않는 방식
- 사용 맥락: 문어적 표현. 태도나 대응 방식을 묘사할 때 사용.

### recalcitrate

- 구성: Latin `recalcitrare`
- 뜻: 반항하다, 완강히 저항하다
- 어원적 연결: 뒤로 발길질하며 거부함
- 사용 맥락: 드문 단어. 보통 `resist`, `defy`, `be recalcitrant`를 더 많이 씀.

## 기억 포인트

`calcitr` 계열은 "발길질하며 버팀" 이미지.

- `recalcitrant`: 고집 세고 반항적인
- `recalcitrance`: 완강한 저항
- `recalcitrantly`: 반항적으로
- `recalcitrate`: 완강히 저항하다

## 비교 단어

### calculate

- 뜻: 계산하다
- 구분 포인트: [[calc]]는 조약돌로 세어 계산함. `calcitr`는 발꿈치로 차며 버티는 저항 이미지.

### rebellious

- 뜻: 반항적인
- 구분 포인트: [[bell]]의 `rebellious`는 권위에 맞서 싸우는 반란 이미지. `recalcitrant`는 지시나 통제에 완강히 따르지 않는 고집 이미지가 강함.
