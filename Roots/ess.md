# ess

- 출발 단어: `interest`
- 핵심 의미: 있다, 존재하다, 본질
- 관련 형태: Latin `esse` = to be
- 관련 문서: [[present]], [[inter]]

## 핵심 이미지

`ess` 계열의 바탕은 "있음".

무엇의 가장 깊은 있음은 `essence`.
꼭 있어야 하는 것은 `essential`.
스스로 존재하는 것은 `entity`.
무언가 사이에 있으면 `interest`.
눈앞에 있으면 `present`, 자리에 없으면 `absent`.

## 단어들

### interest

- 발음: /ˈɪntrəst/
- 구성: Latin `inter-` = between + `esse` = be
- 뜻: 관심, 흥미; 이익; 이자
- 연결: 어떤 일 "사이에 있어" 관련되거나 걸려 있음
- 사용 맥락: `show interest in science` = 과학에 관심을 보이다, `compound interest` = 복리

### interested

- 발음: /ˈɪntrəstɪd/
- 구성: `interest` + `-ed`
- 뜻: 관심 있는; 이해관계가 있는
- 연결: 어떤 일과 관계 속에 들어가 있음
- 사용 맥락: `interested in history` = 역사에 관심 있는

### interesting

- 발음: /ˈɪntrəstɪŋ/
- 구성: `interest` + `-ing`
- 뜻: 흥미로운
- 연결: 사람의 관심을 그 일 안으로 끌어들임
- 사용 맥락: `an interesting question` = 흥미로운 질문

### disinterested

- 발음: /dɪˈsɪntrɪstɪd/
- 구성: `dis-` = apart/not + `interested`
- 뜻: 사심 없는, 이해관계가 없는
- 연결: 개인적 이익 사이에 걸려 있지 않음
- 사용 맥락: `a disinterested judge` = 사심 없는 판단자

### uninterested

- 발음: /əˈnɪntrəstɪd/
- 구성: `un-` = not + `interested`
- 뜻: 관심 없는
- 연결: 관심이 그 일 안에 들어가 있지 않음
- 사용 맥락: `uninterested in politics` = 정치에 관심 없는

### essence

- 발음: /ˈɛsəns/
- 구성: Latin `essentia` = being, essence
- 뜻: 본질, 정수
- 연결: 어떤 것이 그것으로 존재하게 하는 핵심
- 사용 맥락: `the essence of democracy` = 민주주의의 본질

### essential

- 발음: /ɛˈsɛnʃəl/
- 구성: `essence` + `-ial`
- 뜻: 필수적인, 본질적인
- 연결: 그것이 존재하려면 꼭 있어야 하는 것
- 사용 맥락: `essential skills` = 필수 기술

### entity

- 발음: /ˈɛntəti/
- 구성: Latin `ens/ent-` = being
- 뜻: 실체, 존재, 독립체
- 연결: 하나의 존재로 인정되는 것
- 사용 맥락: `a legal entity` = 법적 실체

## 비교 단어

`present`, `presence`, `absent`, `absence`는 [[present]] 문서에 정리.
이들도 Latin `esse`와 연결되지만, 그 문서는 "앞에 있음/없음" 이미지가 중심.

`inter-`는 "사이에"를 뜻하는 접두사.
`interest`는 `inter-` + `esse`라 "사이에 있음, 관계되어 있음"에서 관심·이익으로 확장.
