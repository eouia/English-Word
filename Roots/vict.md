# vict

- 출발 표현: `veni, vidi, vici`
- 핵심 어원: 라틴어 `vincere` / `vict-` = 이기다, 정복하다
- 핵심 이미지: 상대를 이기거나, 마음을 이겨 납득시킴
- 관련해서 헷갈릴 수 있는 어근: [[ven]] = 오다, [[vid]] = 보다

## 출발 표현 분석

`veni, vidi, vici`에서 `vici`는 라틴어 `vincere` = "이기다, 정복하다"의 1인칭 과거형.

- `veni`: I came = 나는 왔다
- `vidi`: I saw = 나는 보았다
- `vici`: I conquered = 나는 이겼다

영어에서는 주로 `vict-`, `vinc-` 형태로 남음.

## 단어들

### victory

- 발음: /ˈvɪktɚi/
- 구성: Latin `victoria`
- 뜻: 승리
- 어원적 연결: 상대를 이긴 결과
- 사용 맥락: `win a victory` = 승리를 거두다

### victorious

- 발음: /vɪˈktɔriəs/
- 구성: `victory` + `-ous`
- 뜻: 승리한, 의기양양한
- 어원적 연결: 이김의 상태를 가진
- 사용 맥락: `a victorious army` = 승리한 군대

### victor

- 발음: /ˈvɪktɚ/
- 구성: Latin `victor`
- 뜻: 승자
- 어원적 연결: 이긴 사람
- 사용 맥락: `the victor of the match` = 경기의 승자

### vanquish

- 발음: /ˈvæŋkwɪʃ/
- 구성: Old French `vainquir`, Latin `vincere`
- 뜻: 완전히 무찌르다, 정복하다
- 어원적 연결: 상대를 이겨 굴복시킴
- 사용 맥락: `vanquish an enemy` = 적을 무찌르다

### invincible

- 발음: /ˌɪˈnvɪnsəbəl/
- 구성: `in-` = not + `vincible`
- 뜻: 무적의, 이길 수 없는
- 어원적 연결: 이길 수 없음
- 사용 맥락: `an invincible team` = 무적의 팀

### vincible

- 구성: Latin `vincibilis`
- 뜻: 이길 수 있는
- 어원적 연결: 패배시킬 수 있음
- 사용 맥락: 드문 단어. `invincible`을 이해하기 위한 형태

### convince

- 발음: /kəˈnvɪns/
- 구성: `con-` = completely + `vincere`
- 뜻: 확신시키다, 납득시키다
- 어원적 연결: 상대의 의심이나 반대를 완전히 이김. 마음을 이겨 납득시킴
- 사용 맥락: `convince someone to agree` = 누군가를 설득해 동의하게 하다

### convincing

- 발음: /kəˈnvɪnsɪŋ/
- 구성: `convince` + `-ing`
- 뜻: 설득력 있는
- 어원적 연결: 상대의 마음을 이길 만큼 강함
- 사용 맥락: `a convincing argument` = 설득력 있는 주장

### convict

- 발음: n. /ˈkɑnvɪkt/, v. /kəˈnvɪkt/
- 구성: `con-` = completely + `vict`
- 뜻: 유죄를 선고하다, 죄수
- 어원적 연결: 법정에서 증거로 상대를 완전히 이김. 혐의를 입증함
- 사용 맥락: `convict him of fraud` = 그에게 사기죄를 선고하다

### conviction

- 발음: /kəˈnvɪkʃən/
- 구성: `convict` + `-ion`
- 뜻: 유죄 판결, 확신, 신념
- 어원적 연결: 법적으로는 혐의를 이겨 입증한 결과. 마음속으로는 의심을 이긴 확신
- 사용 맥락: `a strong conviction` = 강한 신념

### evict

- 발음: /ɪˈvɪkt/
- 구성: `e-` = out + `vict`
- 뜻: 퇴거시키다, 쫓아내다
- 어원적 연결: 법적으로 이겨 밖으로 몰아냄
- 사용 맥락: `evict a tenant` = 세입자를 퇴거시키다

### eviction

- 발음: /ɪˈvɪkʃən/
- 구성: `evict` + `-ion`
- 뜻: 퇴거, 축출
- 어원적 연결: 법적 절차로 이겨 내보내는 행위
- 사용 맥락: `face eviction` = 퇴거 위기에 놓이다

## 기억 포인트

`vict` 계열은 "이기다" 이미지.

- `victory`: 이긴 결과
- `victor`: 이긴 사람
- `vanquish`: 완전히 무찌름
- `invincible`: 이길 수 없음
- `convince`: 마음의 반대를 이김
- `convict`: 법정에서 혐의를 입증해 이김
- `evict`: 법적으로 이겨 밖으로 내보냄

## 비교 단어

### victim

- 뜻: 희생자, 피해자
- 구분 포인트: `victim`은 라틴어 `victima` = 제물, 희생물 계열. `vincere` = 이기다 계열과 직접 묶지 않음

### victory / victim

- `victory`: 이김
- `victim`: 피해를 입은 사람
- 구분 포인트: 형태가 비슷하지만 어원 이미지는 다름
