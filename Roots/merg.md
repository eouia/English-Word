# merg

- 출발 단어: `emerge`
- 핵심 의미: 잠기다, 담그다; 밖으로 떠오르다
- 관련 형태: Latin `mergere` = dip, sink, plunge
- 관련 문서: [[ex]], [[in]]

## 핵심 이미지

`merg` 계열의 바탕은 "물속에 잠기거나, 잠긴 곳에서 떠오름".

안으로 담그면 `immerse`.
아래로 잠기면 `submerge`.
밖으로 떠오르면 `emerge`.
서로 경계가 잠기듯 합쳐지면 `merge`.
갑자기 드러난 상황은 `emergency`.

## 단어들

### emerge

- 구성: Latin `e-/ex-` = out + `mergere` = sink, dip
- 뜻: 나타나다, 드러나다, 부상하다
- 연결: 물속이나 보이지 않는 곳에서 밖으로 떠오름
- 사용 맥락: `new evidence emerged` = 새로운 증거가 드러났다

### emergence

- 구성: `emerge` + `-ence`
- 뜻: 출현, 발생, 부상
- 연결: 숨어 있던 것이 밖으로 드러나는 과정
- 사용 맥락: `the emergence of new technology` = 신기술의 출현

### emergent

- 구성: `emerge` + `-ent`
- 뜻: 새로 나타나는, 발생 중인
- 연결: 막 떠오르거나 드러나는 상태
- 사용 맥락: `emergent trends` = 새로 나타나는 경향

### emergency

- 구성: `emerge` 계열 + `-ency`
- 뜻: 비상사태, 긴급 상황
- 연결: 갑자기 드러나 즉시 대응해야 하는 상황
- 사용 맥락: `in an emergency` = 비상시에

### merge

- 구성: Latin `mergere` = sink, plunge
- 뜻: 합치다, 병합하다
- 연결: 둘의 경계가 한곳에 잠기듯 사라져 합쳐짐
- 사용 맥락: `merge two companies` = 두 회사를 합병하다

### merger

- 구성: `merge` + `-er`
- 뜻: 합병, 병합
- 연결: 따로 있던 둘이 하나로 합쳐지는 일
- 사용 맥락: `a corporate merger` = 기업 합병

### submerge

- 구성: `sub-` = under + `merge`
- 뜻: 물속에 잠기게 하다, 잠수하다
- 연결: 아래로 들어가 물에 덮임
- 사용 맥락: `submerge the vegetables in water` = 채소를 물에 담그다

### submerged

- 구성: `submerge` + `-ed`
- 뜻: 물속에 잠긴, 숨겨진
- 연결: 표면 아래에 가려져 보이지 않음
- 사용 맥락: `submerged rocks` = 물속에 잠긴 바위

### immerse

- 구성: Latin `in-` = in + `mergere`
- 뜻: 담그다, 몰입시키다
- 연결: 어떤 것 안에 완전히 잠기게 함
- 사용 맥락: `immerse yourself in a language` = 언어에 몰입하다

### immersion

- 구성: `immerse` + `-ion`
- 뜻: 몰입, 담금, 몰입식 학습
- 연결: 한 환경 안에 완전히 잠긴 상태
- 사용 맥락: `language immersion` = 언어 몰입 학습

## 비교 단어

`emerge`는 [[ex]] 계열의 "밖으로" 이미지가 강하게 붙은 단어.
`merge`만 보면 "합쳐짐"으로 보이지만, 역사적으로는 `mergere` = 잠기다에서 온 말.

`immerse`는 [[in]] 계열의 "안으로" 이미지와 연결.
학습할 때는 `emerge` = 밖으로 떠오름, `immerse` = 안으로 잠김으로 대비하면 좋음.
