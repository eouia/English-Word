# spond

- 출발 단어: `spouse`
- 핵심 어원: 라틴어 `spondere` / `spons-` = 약속하다, 보증하다
- 핵심 이미지: 말로 약속하고 그 약속에 책임짐
- 관련해서 헷갈릴 수 있는 어근: [[pact]] = 묶어 고정한 합의

## 출발 단어 분석

`spouse`는 라틴어 `sponsus`에서 옴.

- `sponsus`: 약속한 사람, 약혼자
- `spondere`: 엄숙하게 약속하다

따라서 `spouse`는 "결혼 약속으로 맺어진 사람" → "배우자"의 뜻.

## 단어들

### spouse

- 구성: Latin `sponsus`
- 뜻: 배우자
- 어원적 연결: 결혼을 약속해 맺어진 사람
- 사용 맥락: `a spouse visa` = 배우자 비자

### espouse

- 구성: Old French `espouser`, Latin `sponsare`
- 뜻: 지지하다, 받아들이다, 아내/남편으로 맞다
- 어원적 연결: 어떤 사람이나 생각과 약속으로 결합함
- 사용 맥락: `espouse a cause` = 어떤 대의를 지지하다

### sponsor

- 구성: Latin `sponsor`
- 뜻: 후원자, 보증인, 후원하다
- 어원적 연결: 누군가를 위해 약속하고 책임지는 사람
- 사용 맥락: `sponsor an event` = 행사를 후원하다

### sponsorship

- 구성: `sponsor` + `-ship`
- 뜻: 후원, 후원 관계
- 어원적 연결: 책임지고 지원하겠다는 약속의 관계
- 사용 맥락: `corporate sponsorship` = 기업 후원

### respond

- 구성: `re-` = back + `spond`
- 뜻: 대답하다, 반응하다
- 어원적 연결: 상대의 말이나 요구에 다시 약속하듯 답함
- 사용 맥락: `respond to a question` = 질문에 대답하다

### response

- 구성: `respond`의 명사형
- 뜻: 응답, 반응
- 어원적 연결: 되돌아오는 답이나 반응
- 사용 맥락: `an immediate response` = 즉각적인 응답

### responsive

- 구성: `response` + `-ive`
- 뜻: 즉각 반응하는, 호응하는
- 어원적 연결: 요청이나 자극에 잘 답함
- 사용 맥락: `responsive customer service` = 반응이 빠른 고객 서비스

### responsible

- 구성: `response` 계열 + `-ible`
- 뜻: 책임 있는, 책임져야 하는
- 어원적 연결: 물음이나 요구에 답할 수 있고 책임질 수 있음
- 사용 맥락: `be responsible for the result` = 결과에 책임이 있다

### responsibility

- 구성: `responsible` + `-ity`
- 뜻: 책임
- 어원적 연결: 답하고 감당해야 하는 의무
- 사용 맥락: `take responsibility` = 책임지다

### correspond

- 구성: `cor-` = together + `respond`
- 뜻: 일치하다, 서신을 주고받다
- 어원적 연결: 서로 답이 맞아떨어짐. 편지를 주고받으며 서로 응답함
- 사용 맥락: `correspond to the data` = 자료와 일치하다

### correspondence

- 구성: `correspond` + `-ence`
- 뜻: 일치, 서신 왕래
- 어원적 연결: 서로 맞게 응답하거나 주고받는 관계
- 사용 맥락: `email correspondence` = 이메일 서신

### correspondent

- 구성: `correspond` + `-ent`
- 뜻: 통신원, 특파원
- 어원적 연결: 현장에서 소식을 보내 응답하는 사람
- 사용 맥락: `a foreign correspondent` = 해외 특파원

### despond

- 구성: `de-` = down, away + `spond`
- 뜻: 낙담하다, 풀이 죽다
- 어원적 연결: 원래 "약속을 포기하다, 단념하다". 기대나 희망에 더는 응답하지 못하고 내려놓음
- 사용 맥락: `do not despond` = 낙담하지 마라

### despondent

- 구성: Latin `despondere`
- 뜻: 낙담한, 풀이 죽은
- 어원적 연결: 원래 "약속을 포기하다, 단념하다"에서 희망을 내려놓은 상태로 의미 확대
- 사용 맥락: `feel despondent` = 낙담하다

### despondency

- 구성: `despondent` + `-cy`
- 뜻: 낙담, 실의
- 어원적 연결: 기대나 희망을 포기한 마음 상태
- 사용 맥락: `fall into despondency` = 실의에 빠지다

## 기억 포인트

`spond` 계열은 "약속하고 답하며 책임짐" 이미지.

- `spouse`: 결혼 약속으로 맺어진 사람
- `espouse`: 어떤 사람이나 생각을 받아들여 지지함
- `sponsor`: 책임지고 후원하는 사람
- `respond`: 되돌아 답함
- `responsible`: 답하고 책임질 수 있음
- `correspond`: 서로 답이 맞음
- `despond`: 기대나 희망을 내려놓음
- `despondent`: 희망을 내려놓고 낙담함
