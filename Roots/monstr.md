# monstr

- 출발 단어: `demonstration`
- 핵심 어원: Latin `monstrare` = show, point out; `monstrum` = omen, warning, strange sign
- 핵심 이미지: 눈앞에 보여 주고, 징조처럼 드러남
- 관련 문서: [[indic]], [[dem]]
- 주의: `demonstrate`의 `de-`는 Greek `demos` = people 계열의 [[dem]]과 무관함.

## 출발 단어 분석

`demonstration` = `de-` + `monstr` + `-ation`

- `de-`: down, fully, clearly
- `monstrare`: show, point out
- `-ation`: 과정·행위 명사
- 원래 이미지: 분명히 보여 주어 가리킴
- 현대 의미: 시위, 설명·증명, 시연

## 단어들

### demonstrate

- 발음: /ˈdɛməˌnstreɪt/
- 구성: `de-` + Latin `monstrare` = show
- 뜻: 보여 주다, 입증하다, 시위하다
- 어원적 연결: 눈앞에 분명히 드러내 보임
- 사용 맥락: `demonstrate the method` = 방법을 시연하다

### demonstration

- 발음: /ˌdɛməˈnstreɪʃən/
- 구성: `demonstrate` + `-ion`
- 뜻: 시위, 시연, 입증
- 어원적 연결: 생각·요구·방법을 공개적으로 보여 주는 행위
- 사용 맥락: `mass demonstration` = 대규모 시위
- 주의: 정치 맥락에서는 시위, 과학·교육 맥락에서는 시연·입증.

### demonstrative

- 발음: /dɪˈmɑnstrətɪv/
- 구성: `demonstrate` + `-ive`
- 뜻: 드러내 보이는, 감정을 잘 표현하는; 지시형의
- 어원적 연결: 감정이나 대상을 밖으로 뚜렷이 보여 줌
- 사용 맥락: `a demonstrative gesture` = 감정을 드러내는 몸짓

### demonstrable

- 발음: /ˈdɛmənstrəbəl/
- 구성: `demonstrate` + `-able`
- 뜻: 입증 가능한
- 어원적 연결: 증거로 보여 줄 수 있음
- 사용 맥락: `demonstrable evidence` = 입증 가능한 증거

### remonstrate

- 구성: `re-` = back, again + `monstrare`
- 뜻: 항의하다, 이의를 제기하다
- 어원적 연결: 상대에게 잘못이나 반대 이유를 다시 가리켜 보임
- 사용 맥락: `remonstrate against unfair treatment` = 부당한 대우에 항의하다
- 주의: 격식 있는 단어. 단순히 불평하다보다 논리적으로 이의를 제기하는 느낌.

### remonstrance

- 구성: `remonstrate` + `-ance`
- 뜻: 항의, 이의 제기
- 어원적 연결: 반대 이유를 드러내 보이는 행위
- 사용 맥락: formal writing

### monster

- 발음: /ˈmɑnstɚ/
- 구성: Latin `monstrum` = omen, warning, strange sign
- 뜻: 괴물, 잔혹한 사람
- 어원적 연결: 자연 질서에서 벗어나 경고나 징조처럼 보이는 존재
- 사용 맥락: `a mythical monster` = 신화 속 괴물

### monstrous

- 발음: /ˈmɑnstrəs/
- 구성: `monster` + `-ous`
- 뜻: 괴물 같은, 터무니없는, 잔혹한
- 어원적 연결: 정상 범위를 벗어나 두드러지게 이상하거나 끔찍함
- 사용 맥락: `a monstrous crime` = 끔찍한 범죄

### monstrosity

- 발음: /mɑˈnstrɑsəti/
- 구성: `monstrous` + `-ity`
- 뜻: 기괴한 것, 괴물 같은 것, 잔혹 행위
- 어원적 연결: 이상하게 두드러져 경고처럼 보이는 존재나 행위
- 사용 맥락: `an architectural monstrosity` = 기괴한 건축물

### monstrance

- 구성: Medieval Latin `monstrantia` = showing
- 뜻: 성체 현시대
- 어원적 연결: 성체를 신자들에게 보여 주기 위한 용기
- 사용 맥락: Catholic liturgy
- 주의: 전문 종교 어휘라 일반 시험 어휘로는 우선순위가 낮음.

## 기억 포인트

`monstr` 계열은 "보여 주어 드러냄"과 "징조처럼 이상하게 드러남"이 함께 있음.

- `demonstrate`: 분명히 보여 주다
- `demonstration`: 시위, 시연, 입증
- `remonstrate`: 반대 이유를 보여 주며 항의하다
- `monster`: 이상한 징조처럼 보이는 괴물
- `monstrous`: 괴물처럼 끔찍한

## 비교 단어

- `dem`: [[dem]]은 Greek `demos` = people. `demonstrate`의 `dem-`처럼 보이는 부분과 무관함.
- `indic`: [[indic]]는 말이나 표시로 가리킴. `monstr`는 직접 보여 주어 드러내는 느낌.
- `show`: 일반적으로 보여 주다. `demonstrate`는 설명·증거·행동으로 분명히 보여 주는 격식어.
