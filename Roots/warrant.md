# warrant

- 출발 단어: `warranted`
- 핵심 어원: Old North French `warant` = protector, defender, authorization
- 핵심 이미지: 보증이나 권한이 뒤에서 받쳐 줌
- 관련 문서: [[just]], [[val]]
- 주의: `warrant`는 명사로 "영장, 보증, 정당한 근거", 동사로 "정당화하다, 보증하다"의 뜻.

## 출발 단어 분석

`warranted` = `warrant` + `-ed`

- `warrant`: 보증, 권한, 정당한 근거; 정당화하다
- `-ed`: 상태를 나타내는 과거분사형
- `warranted`: 정당한 근거가 있는, 타당한

`warrant`는 원래 보호·보증·권한과 연결. 어떤 판단이나 행동이 `warranted`라면, 그것을 뒷받침하는 충분한 이유나 권한이 있다는 뜻.

## 단어들

### warrant

- 발음: /ˈwɔrənt/
- 구성: Old North French `warant` = protector, authorization
- 뜻: 영장, 보증, 정당한 근거; 정당화하다
- 어원적 연결: 어떤 행동이나 주장을 뒷받침하는 권한·보증
- 사용 맥락: `a search warrant` = 수색 영장, `warrant further study` = 추가 연구가 필요할 만큼 근거가 있다

### warranted

- 발음: /ˈwɔrəntɪd/
- 구성: `warrant` + `-ed`
- 뜻: 정당한, 근거 있는, 타당한
- 어원적 연결: 보증이나 이유가 뒤에서 받쳐 주는 상태
- 사용 맥락: `concern is warranted` = 우려할 만한 근거가 있다

### unwarranted

- 발음: /əˈnwɔrəntɪd/
- 구성: `un-` = not + `warranted`
- 뜻: 정당한 근거가 없는, 부당한
- 어원적 연결: 뒷받침하는 보증이나 이유가 없음
- 사용 맥락: `unwarranted criticism` = 근거 없는 비판

### warranty

- 발음: /ˈwɔrənti/
- 구성: `warrant` 계열 명사형
- 뜻: 보증, 품질 보증서
- 어원적 연결: 제품이나 약속을 책임지고 보증함
- 사용 맥락: `under warranty` = 보증 기간 중인

### warrantable

- 구성: `warrant` + `-able`
- 뜻: 정당화될 수 있는, 보증 가능한
- 어원적 연결: 이유나 권한으로 뒷받침할 수 있음
- 사용 맥락: 드문 문어 표현. 시험 핵심은 `warranted`, `unwarranted`, `warranty`.

## 기억 포인트

`warrant` 계열은 "보증이나 근거가 받쳐 줌" 이미지.

- `warrant`: 영장, 보증, 정당한 근거
- `warranted`: 근거 있는, 타당한
- `unwarranted`: 근거 없는, 부당한
- `warranty`: 품질 보증

## 비교 단어

### justify

- 뜻: 정당화하다
- 구분 포인트: [[just]]는 기준이나 정의에 맞는 것으로 설명함. `warrant`는 그 설명이나 행동을 뒷받침하는 권한·보증·근거에 초점.

### valid

- 뜻: 유효한, 타당한
- 구분 포인트: [[val]]은 효력이나 타당성이 힘 있게 성립함. `warranted`는 그 성립을 뒷받침할 만한 이유가 있다는 느낌.

### guarantee

- 뜻: 보증하다, 보장
- 구분 포인트: `guarantee`는 결과나 품질을 책임지고 보장함. `warrant`는 법적 권한, 정당한 근거, 제품 보증까지 넓게 씀.

### authorize

- 뜻: 권한을 부여하다
- 구분 포인트: `authorize`는 공식 권한을 줌. `warrant`는 어떤 행동을 가능하게 하는 권한이나 근거 자체를 가리킬 수 있음.
