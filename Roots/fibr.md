# fibr

- 출발 단어: `fiber`
- 핵심 어원: Latin `fibra` = fiber, filament
- 핵심 이미지: 실, 섬유, 가느다란 줄기
- 관련 문서: [[fil]], [[tex]]
- 주의: `fabric`은 Latin `faber/fabrica` = 장인, 제작, 구조 계열로 `fiber`와 다른 어원.

## 출발 단어 분석

`fiber` = Latin `fibra`

- `fibra`: 섬유, 가는 실, 장기 조직의 실 같은 부분
- 원래 이미지: 가늘고 길게 이어진 조직
- 현대 의미: 섬유, 섬유질, 성질·기질

## 단어들

### fiber

- 발음: /ˈfaɪbɚ/
- 구성: Latin `fibra`
- 뜻: 섬유, 섬유질
- 어원적 연결: 가늘고 긴 실 같은 조직
- 사용 맥락: `dietary fiber` = 식이섬유

### fibre

- 발음: /ˈfaɪbɚ/
- 구성: British spelling of `fiber`
- 뜻: 섬유, 섬유질
- 어원적 연결: `fiber`와 같은 단어의 영국식 철자
- 사용 맥락: `natural fibre` = 천연 섬유

### fibrous

- 발음: /ˈfaɪbrəs/
- 구성: `fiber` + `-ous`
- 뜻: 섬유질의, 섬유 같은
- 어원적 연결: 실 같은 조직이 많음
- 사용 맥락: `fibrous tissue` = 섬유 조직

### fibril

- 구성: Modern Latin `fibrilla` = 작은 섬유
- 뜻: 작은 섬유, 미세 섬유
- 어원적 연결: `fibra`의 축소형
- 사용 맥락: 생물·의학 맥락

### fibrillation

- 발음: /ˌfɪbrɪˈleɪʃən/
- 구성: `fibril` + `-ation`
- 뜻: 세동, 잔떨림
- 어원적 연결: 근육 섬유가 미세하게 떨림
- 사용 맥락: `atrial fibrillation` = 심방세동

### microfiber

- 구성: `micro-` = very small + `fiber`
- 뜻: 극세사, 미세 섬유
- 어원적 연결: 매우 작은 섬유
- 사용 맥락: `microfiber cloth` = 극세사 천

### fiberglass

- 발음: /ˈfaɪbɚˌglæs/
- 구성: `fiber` + `glass`
- 뜻: 유리섬유
- 어원적 연결: 유리를 섬유처럼 가늘게 만든 재료
- 사용 맥락: `fiberglass insulation` = 유리섬유 단열재

### fiber-optic

- 구성: `fiber` + `optic`
- 뜻: 광섬유의
- 어원적 연결: 빛을 보내는 가는 섬유
- 사용 맥락: `fiber-optic cable` = 광섬유 케이블

### fibrosis

- 발음: /faɪˈbroʊsəs/
- 구성: `fibra` + `-osis`
- 뜻: 섬유증
- 어원적 연결: 조직이 섬유처럼 굳거나 늘어남
- 사용 맥락: `pulmonary fibrosis` = 폐섬유증

### fibroid

- 발음: /ˈfaɪbrɔɪd/
- 구성: `fibra` + `-oid` = like
- 뜻: 섬유종, 섬유 같은
- 어원적 연결: 섬유 조직과 비슷한 덩어리
- 사용 맥락: 의학 맥락

## 기억 포인트

`fibr` 계열은 "가느다란 실/섬유" 이미지.

- `fiber`: 섬유
- `fibrous`: 섬유질의
- `fibril`: 작은 섬유
- `fibrillation`: 섬유의 잔떨림
- `fiber-optic`: 빛을 보내는 섬유

## 비교 단어

### fabric

- 뜻: 직물, 구조, 만들다 계열
- 구분 포인트: Latin `faber/fabrica` = 장인, 제작, 구조. `fiber`와 철자는 비슷하지만 어원이 다름.

### filament

- 뜻: 실, 필라멘트
- 연결 포인트: Latin `filum` = thread 계열. `fibra`와 관련 가능성이 제기되지만 확실하지 않으므로 별도 문서로 둠.
