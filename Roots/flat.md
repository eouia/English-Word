# flat

- 출발 단어: `inflate`
- 핵심 어원: Latin `flare` = 불다, 바람을 불어넣다
- 관련 형태: Latin `flatus` = 숨, 바람, 불기
- 핵심 이미지: 공기를 불어 넣거나 빼서 상태가 변함
- 관련 문서: [[spir]], [[pneu]]
- 주의: 영어 형용사 `flat` = 평평한 말은 별도 어원. 여기서는 `inflate`, `deflate`의 `flat` 계열.

## 출발 단어 분석

`inflate` ← Latin `inflare`

- `in-`: into
- `flare`: blow
- 원래 이미지: 안으로 바람을 불어넣음
- 현대 의미: 부풀리다, 상승시키다

## 단어들

### inflate

- 발음: /ɪˈnfleɪt/
- 구성: `in-` = into + Latin `flare` = blow
- 뜻: 부풀리다, 과장하다, 물가가 오르다
- 어원적 연결: 안으로 공기를 불어넣어 크게 만듦
- 사용 맥락: `inflate a balloon` = 풍선을 부풀리다

### inflation

- 발음: /ˌɪˈnfleɪʃən/
- 구성: `inflate` + `-ion`
- 뜻: 팽창, 인플레이션
- 어원적 연결: 가격이나 규모가 부풀어 오름
- 사용 맥락: `high inflation` = 높은 물가 상승

### deflate

- 발음: /dɪˈfleɪt/
- 구성: `de-` = down, away + Latin `flare`
- 뜻: 바람을 빼다, 기를 꺾다, 축소시키다
- 어원적 연결: 들어 있던 공기나 과장을 빼냄
- 사용 맥락: `deflate expectations` = 기대를 낮추다

### deflation

- 발음: /dɪˈfleɪʃən/
- 구성: `deflate` + `-ion`
- 뜻: 수축, 디플레이션
- 어원적 연결: 부풀어 있던 것이 줄어듦
- 사용 맥락: `economic deflation` = 경제적 디플레이션

### conflate

- 발음: /kəˈnfleɪt/
- 구성: `con-` = together + Latin `flare`
- 뜻: 뒤섞다, 하나로 합치다
- 어원적 연결: 여러 것을 함께 불어 넣어 하나로 녹여 합침
- 사용 맥락: `conflate two issues` = 두 문제를 혼동해 합치다

### conflation

- 발음: /kəˈnfleɪʃən/
- 구성: `conflate` + `-ion`
- 뜻: 혼합, 융합, 혼동
- 어원적 연결: 서로 다른 것을 하나로 합친 상태
- 사용 맥락: `a conflation of fact and opinion` = 사실과 의견의 혼동

### flatulence

- 구성: Latin `flatus` = blowing, wind + `-ence`
- 뜻: 장내 가스, 헛배부름
- 어원적 연결: 몸 안의 바람이나 가스
- 사용 맥락: 의학·일상 맥락에서 위장 가스를 가리킴

### flatus

- 구성: Latin `flatus`
- 뜻: 방귀, 장내 가스
- 어원적 연결: 몸 밖으로 나오는 바람
- 사용 맥락: 의학 맥락에서 쓰임

### afflatus

- 구성: `ad-` = toward + Latin `flatus` = breath
- 뜻: 영감, 신적 계시
- 어원적 연결: 바깥에서 숨이나 영감이 불어 들어옴
- 사용 맥락: 문학적·종교적 맥락에서 쓰임

## 기억 포인트

`flat` 계열은 "불어 넣다, 바람" 이미지.

- `inflate`: 안으로 불어 넣어 부풀림
- `deflate`: 바람을 빼서 줄임
- `conflate`: 함께 불어 넣어 섞음
- `flatulence`: 몸 안의 바람

## 비교 단어

### spir

- 뜻: 숨쉬다
- 구분 포인트: [[spir]]는 숨의 흐름과 생기. `flat`은 바람을 불어 넣어 상태를 바꾸는 작용에 초점.

### pneu

- 뜻: 숨, 공기
- 구분 포인트: [[pneu]]는 공기와 호흡 자체. `flat`은 공기를 넣고 빼는 동작과 결과.
