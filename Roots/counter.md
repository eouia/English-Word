# counter

- 출발 단어: `counteract`
- 핵심 어원: Latin `contra` = against, opposite
- 핵심 이미지: 맞은편에서 반대로 맞섬
- 관련 문서: [[ag]], [[argu]], [[comit]], [[count]]
- 주의: 계산대 `counter`는 원래 계산판·탁자 쪽 의미도 있음. `county`는 [[comit]] 계열. `country`는 `contra`에서 온 "마주 놓인 땅" 계열.

## 출발 단어 분석

`counteract` = `counter-` + `act`

- `counter-`: against, opposite
- `act`: do, act
- 원래 이미지: 어떤 작용에 맞서 반대로 행동함
- 현대 의미: 상쇄하다, 약화시키다

## 단어들

### counteract

- 발음: /ˈkaʊntɚˌækt/
- 구성: `counter-` = against + `act`
- 뜻: 대응하여 약화시키다, 상쇄하다
- 어원적 연결: 반대 행동으로 기존 작용의 힘을 줄임
- 사용 맥락: `counteract the effects of stress` = 스트레스의 영향을 상쇄하다
- 관련: [[ag]]

### counteraction

- 구성: `counteract` + `-ion`
- 뜻: 반작용, 상쇄 작용
- 어원적 연결: 반대쪽 작용이 기존 작용에 맞섬
- 사용 맥락: 과학·논리 문맥
- 관련: [[ag]]

### counteractive

- 구성: `counteract` + `-ive`
- 뜻: 상쇄하는, 반작용의
- 어원적 연결: 반대 방향으로 작용하는 성질
- 사용 맥락: 드문 표현
- 관련: [[ag]]

### counterargument

- 구성: `counter-` = against + `argument`
- 뜻: 반론, 반대 논거
- 어원적 연결: 앞선 주장에 맞서 제시한 이유
- 사용 맥락: `raise a counterargument` = 반론을 제기하다
- 관련: [[argu]]

### counterbalance

- 발음: /ˈkaʊntɚˌbæləns/
- 구성: `counter-` = opposite + `balance`
- 뜻: 균형을 맞추다, 상쇄하다; 평형추
- 어원적 연결: 반대쪽 무게로 균형을 맞춤
- 사용 맥락: `counterbalance risk` = 위험을 상쇄하다

### counterproductive

- 발음: /ˈkaʊntɚprəˌdʌktɪv/
- 구성: `counter-` = against + `productive`
- 뜻: 역효과를 내는
- 어원적 연결: 의도한 생산적 결과와 반대로 작용함
- 사용 맥락: `a counterproductive policy` = 역효과를 내는 정책

### encounter

- 발음: /ɪˈnkaʊntɚ/
- 구성: Old French `encontrer` ← `en-` + `contre` = against, opposite
- 뜻: 우연히 만나다, 맞닥뜨리다; 만남, 조우
- 어원적 연결: 맞은편에 있는 대상과 서로 마주침
- 사용 맥락: `encounter a problem` = 문제에 맞닥뜨리다, `a chance encounter` = 우연한 만남

### country

- 발음: /ˈkʌntri/
- 구성: Old French `contree` ← Medieval Latin `contrata` = land lying opposite
- 뜻: 나라, 국토; 시골, 지방
- 어원적 연결: 눈앞에 마주 펼쳐진 땅, 한 지역
- 사용 맥락: `govern a country` = 나라를 다스리다, `live in the country` = 시골에 살다
- 주의: `county`와 철자가 비슷하지만 어원이 다름.

### countryside

- 발음: /ˈkʌntriˌsaɪd/
- 구성: `country` + `side`
- 뜻: 시골 지역, 전원
- 어원적 연결: 도시가 아니라 넓게 펼쳐진 지방의 땅
- 사용 맥락: `walk through the countryside` = 시골길을 걷다

### contrary

- 발음: /ˈkɑntrɛri/
- 구성: Latin `contrarius` = opposite, opposed
- 뜻: 반대의, 상반되는
- 어원적 연결: 맞은편에 놓여 서로 반대됨
- 사용 맥락: `contrary evidence` = 반대 증거

### contrast

- 발음: n. /ˈkɑntræst/, v. /kəˈntræst/
- 구성: Latin `contra-` = against + `stare` = stand
- 뜻: 대조, 대비; 대조하다
- 어원적 연결: 서로 맞은편에 세워 차이가 드러남
- 사용 맥락: `contrast two ideas` = 두 생각을 대조하다

### contra

- 발음: /ˈkɑntrə/
- 구성: Latin `contra` = against, opposite
- 뜻: 반대하여, 맞서
- 어원적 연결: 맞은편에 놓인 위치나 반대 방향
- 사용 맥락: `pro and contra` = 찬반

## 기억 포인트

`counter-`는 "맞은편에서 반대로 맞섬" 이미지.

- `counteract`: 반대로 행동해 상쇄함
- `counterargument`: 반대 논거
- `counterbalance`: 반대 무게로 균형을 맞춤
- `counterproductive`: 의도와 반대로 작용함
- `encounter`: 맞은편 대상과 마주침
- `country`: 눈앞에 마주 펼쳐진 지역
- `contrary`: 반대쪽에 놓인
- `contrast`: 맞세워 차이를 드러냄

## 비교 단어

### county

- 뜻: 군, 카운티
- 구분 포인트: [[comit]]의 Latin `comes/comitem` = companion 계열. `country`는 Latin `contra` = opposite 계열.

### count

- 뜻: 세다, 계산하다
- 구분 포인트: [[count]]는 Latin `computare` = count, reckon 계열. `country`와 `county` 모두와 분리.

### re

- 뜻: 다시, 뒤로
- 구분 포인트: `react`는 되돌아 반응함. `counteract`는 기존 작용에 맞서 효과를 줄이는 반대 행동.
