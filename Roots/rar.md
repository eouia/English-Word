# rar

- 출발 단어: `rarely`
- 핵심 어원: Latin `rarus` = loose, thin, sparse, uncommon
- 핵심 이미지: 성기게 흩어져 있어 자주 보이지 않음
- 관련 문서: [[frequ]]
- 주의: `rare` "드문"은 Latin `rarus` 계열. `rare steak`의 `rare`는 보통 별도 발달로 보며, 여기서는 "희귀한, 드문" 뜻을 중심으로 정리.

## 출발 단어 분석

`rarely` = `rare` + `-ly`

- `rare`: 드문, 희귀한
- `-ly`: 방식이나 빈도를 나타내는 부사 접미사
- 원래 이미지: 성기게 떨어져 있어 자주 나타나지 않음
- 현대 의미: 드물게, 좀처럼 ~하지 않게

## 단어들

### rare

- 발음: /ˈrɛr/
- 구성: Latin `rarus`
- 뜻: 드문, 희귀한, 진귀한
- 어원적 연결: 촘촘하지 않고 성기게 나타남
- 사용 맥락: `a rare disease` = 희귀 질환

### rarely

- 발음: /ˈrɛrli/
- 구성: `rare` + `-ly`
- 뜻: 드물게, 좀처럼 ~하지 않게
- 어원적 연결: 어떤 일이 성기게 나타나는 방식
- 사용 맥락: `rarely happens` = 드물게 일어나다

### rarity

- 발음: /ˈrɛrəti/
- 구성: `rare` + `-ity`
- 뜻: 희귀함, 보기 드문 것
- 어원적 연결: 드물게 존재하는 성질이나 대상
- 사용 맥락: `a rarity in modern life` = 현대 생활에서 보기 드문 것

### rarefy

- 발음: /ˈrɛrəfaɪ/
- 구성: `rare` + `-fy`
- 뜻: 희박하게 하다, 묽게 하다
- 어원적 연결: 촘촘한 상태를 성기고 얇게 만듦
- 사용 맥락: 과학·문어 맥락. `rarefied air` = 희박한 공기

### rarefied

- 발음: /ˈrɛrəfaɪd/
- 구성: `rarefy` + `-ed`
- 뜻: 희박한; 고상하고 배타적인
- 어원적 연결: 공기나 분위기가 얇고 성긴 상태. 비유적으로 일반 대중과 떨어진 세련된 영역
- 사용 맥락: `a rarefied atmosphere` = 희박한 공기, 고상한 분위기

## 기억 포인트

`rar` 계열은 "성기고 드묾" 이미지.

- `rare`: 드문
- `rarely`: 드물게
- `rarity`: 희귀함, 보기 드문 것
- `rarefy`: 희박하게 하다
- `rarefied`: 희박한, 고상하고 배타적인

## 비교 단어

### frequent

- 뜻: 빈번한, 자주 일어나는
- 구분 포인트: [[frequ]]는 많이 모이고 자주 반복되는 이미지. `rar`는 성기게 나타나 드문 이미지.

### dense

- 뜻: 빽빽한, 밀도가 높은
- 구분 포인트: [[dens]]는 촘촘하고 빽빽함. `rar`는 그 반대인 성김과 희박함.
