# mand

- 출발 단어: `mandatory`
- 핵심 어원: 라틴어 `mandare` = 맡기다, 명령하다
- 구성 배경: `manus` = hand + `dare` = give
- 핵심 이미지: 손에 맡겨 지시함 / 명령으로 부여함
- 관련해서 헷갈릴 수 있는 어근: [[manu]] = 손 계열, [[dict]] = 말로 정하다 계열

## 출발 단어 분석

`mandatory`는 `mandate` 계열 + `-ory`.

- `mandare`: entrust, order
- `-ory`: 관련된, 성질을 가진

따라서 `mandatory`는 "명령으로 부여된" → "의무적인, 필수의" 뜻.

## 단어들

### mandatory

- 구성: `mandate` 계열 + `-ory`
- 뜻: 의무적인, 필수의
- 어원적 연결: 명령이나 권한으로 반드시 해야 하게 맡겨짐
- 사용 맥락: `mandatory training` = 의무 교육

### mandate

- 구성: Latin `mandatum` = command, commission
- 뜻: 명령, 위임 권한; 명령하다
- 어원적 연결: 어떤 일을 손에 맡기고 공식적으로 지시함
- 사용 맥락: `a government mandate` = 정부 명령

### command

- 구성: `com-` = together, fully + `mand`
- 뜻: 명령하다, 지휘하다
- 어원적 연결: 권한을 가지고 지시를 맡김
- 사용 맥락: `command an army` = 군대를 지휘하다

### commander

- 구성: `command` + `-er`
- 뜻: 지휘관
- 어원적 연결: 명령을 내리고 지휘하는 사람
- 사용 맥락: `a military commander` = 군 지휘관

### demand

- 구성: `de-` = fully, formally + `mand`
- 뜻: 요구하다, 수요
- 어원적 연결: 공식적으로 맡겨 달라거나 내놓으라고 요구함
- 사용 맥락: `demand an explanation` = 설명을 요구하다

### recommend

- 구성: `re-` = again, back + `commend`
- 뜻: 추천하다, 권하다
- 어원적 연결: 어떤 사람이나 물건을 남의 손에 맡기듯 권함
- 사용 맥락: `recommend a book` = 책을 추천하다

### recommendation

- 구성: `recommend` + `-ation`
- 뜻: 추천, 권고
- 어원적 연결: 맡겨도 좋다고 권하는 판단
- 사용 맥락: `a letter of recommendation` = 추천서

### remand

- 구성: `re-` = back + `mand`
- 뜻: 돌려보내다, 구금 환송하다
- 어원적 연결: 명령으로 다시 보내거나 돌려 맡김
- 사용 맥락: `remand a case` = 사건을 환송하다

## 기억 포인트

`mand` 계열은 "손에 맡겨 명령함" 이미지.

- `mandate`: 공식 명령, 위임 권한
- `mandatory`: 명령으로 부여된 → 의무적인
- `command`: 지휘하며 명령하다
- `demand`: 강하게 요구하다
- `recommend`: 맡겨도 좋다고 권하다

## 비교 단어

### manual

- 뜻: 손의, 수동의; 설명서
- 구분 포인트: [[manu]]는 손 자체의 이미지. `mand`는 `manus` + `dare` 배경을 가지지만 현대 영어에서는 맡김과 명령의 이미지가 중심

### dictate

- 뜻: 지시하다, 받아쓰게 하다
- 구분 포인트: [[dict]]는 말로 정하고 선언하는 이미지. `mand`는 권한을 맡기거나 명령으로 부여하는 이미지
