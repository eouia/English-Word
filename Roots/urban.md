# urban

- 출발 단어: `urbanization`
- 핵심 어원: Latin `urbs` = 도시
- 핵심 의미: 도시, 도시화, 도시적 세련됨
- 관련 문서: [[polis]]
- 관련 비교: `civil`, `suburb`

## 출발 단어 분석

`urbanization` = `urban` + `-ize` + `-ation`

- `urbs`: city
- `urban`: 도시의
- `-ize`: 되게 하다
- `-ation`: 과정
- 원래 이미지: 도시와 관련된 상태
- 의미 확장: 도시의 → 도시처럼 됨 → 도시 인구와 생활 방식의 확산

## 단어들

### urbanization

- 발음: /ˌɝbənəˈzeɪʃən/
- 구성: `urbanize` + `-ation`
- 뜻: 도시화
- 어원적 연결: 사람과 경제 활동이 도시 중심으로 옮겨 가는 과정
- 사용 맥락: `rapid urbanization` = 급속한 도시화

### urban

- 발음: /ˈɝbən/
- 구성: Latin `urbanus` < `urbs`
- 뜻: 도시의, 도시적인
- 어원적 연결: 도시와 관련됨
- 사용 맥락: `urban areas` = 도시 지역

### urbanize

- 발음: /ˈɝbəˌnaɪz/
- 구성: `urban` + `-ize`
- 뜻: 도시화하다, 도시화되다
- 어원적 연결: 도시적 환경이나 생활 방식으로 바뀜
- 사용 맥락: `urbanize rural areas` = 농촌 지역을 도시화하다

### urbanized

- 발음: /ˈɝbəˌnaɪzd/
- 구성: `urbanize` + `-ed`
- 뜻: 도시화된
- 어원적 연결: 도시적 구조와 생활 방식이 자리 잡은 상태
- 사용 맥락: `highly urbanized regions` = 고도로 도시화된 지역

### urbanism

- 발음: /ˈɝbənɪzm/
- 구성: `urban` + `-ism`
- 뜻: 도시 생활 양식, 도시 계획 사상
- 어원적 연결: 도시를 중심으로 한 삶과 설계 방식
- 사용 맥락: `new urbanism` = 뉴어버니즘

### urbane

- 발음: /ɚˈbeɪn/
- 구성: Latin `urbanus`
- 뜻: 세련된, 점잖은
- 어원적 연결: 도시 사람다운 세련됨이라는 이미지
- 사용 맥락: `an urbane manner` = 세련된 태도

### urbanity

- 구성: `urbane` + `-ity`
- 뜻: 세련됨, 도시적 예의
- 어원적 연결: 도시적 교양과 매너
- 사용 맥락: 문어적 표현

### suburban

- 발음: /səˈbɝbən/
- 구성: `sub-` = under/near + `urban`
- 뜻: 교외의
- 어원적 연결: 도시 가까이에 있는 지역
- 사용 맥락: `suburban neighborhoods` = 교외 주거지

### suburb

- 발음: /ˈsʌbɚb/
- 구성: `sub-` = near + `urb`
- 뜻: 교외, 근교
- 어원적 연결: 도시 주변 지역
- 사용 맥락: `live in the suburbs` = 교외에 살다

## 기억 포인트

`urban` 계열은 "도시" 이미지.

- `urban`: 도시의
- `urbanization`: 도시화
- `suburban`: 교외의
- `urbane`: 도시적 세련됨

## 비교 단어

### polis

- 뜻: 도시국가, 정치 공동체
- 구분 포인트: [[polis]]는 Greek 도시국가와 정치 계열. `urban`은 Latin `urbs`의 도시 환경 계열.
