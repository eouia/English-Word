# migr

- 출발 단어: `migration`
- 핵심 어원: Latin `migrare` = 옮겨 가다, 이주하다
- 핵심 의미: 장소를 옮김, 이주
- 관련 문서: [[ex]], [[in]], [[trans]]

## 출발 단어 분석

`migration` = `migr` + `-ation`

- `migrare`: move from one place to another
- `-ation`: 과정, 행위
- 원래 이미지: 한 장소에서 다른 장소로 옮겨 감
- 의미 확장: 사람의 이주 → 동물의 이동 → 데이터·세포·개념의 이동

## 단어들

### migration

- 구성: Latin `migrare` + `-ation`
- 뜻: 이주, 이동
- 어원적 연결: 거처나 위치를 옮기는 과정
- 사용 맥락: `human migration` = 인류 이주, `bird migration` = 새의 이동

### migrate

- 구성: Latin `migrare`
- 뜻: 이주하다, 이동하다
- 어원적 연결: 한 장소에서 다른 장소로 옮겨 감
- 사용 맥락: `migrate to cities` = 도시로 이주하다

### migrant

- 구성: `migrate` + `-ant`
- 뜻: 이주자, 이동하는
- 어원적 연결: 장소를 옮기는 사람이나 생물
- 사용 맥락: `migrant workers` = 이주 노동자

### immigrant

- 구성: `in-` = into + `migr`
- 뜻: 이민자, 들어온 이주자
- 어원적 연결: 어떤 나라나 지역 안으로 옮겨 들어옴
- 사용 맥락: `immigrant communities` = 이민자 공동체

### immigrate

- 구성: `in-` = into + `migrate`
- 뜻: 이민 오다
- 어원적 연결: 특정 국가 안으로 이동함
- 사용 맥락: `immigrate to Canada` = 캐나다로 이민 오다

### emigrate

- 구성: `e-`/`ex-` = out + `migrate`
- 뜻: 이민 가다, 국외로 나가다
- 어원적 연결: 자기 나라 밖으로 옮겨 감
- 사용 맥락: `emigrate from Ireland` = 아일랜드에서 이민 가다

### emigrant

- 구성: `emigrate` + `-ant`
- 뜻: 이민 가는 사람, 국외 이주자
- 어원적 연결: 밖으로 떠나는 이주자
- 사용 맥락: 출발지 관점에서 쓰임

### migratory

- 구성: `migrate` + `-ory`
- 뜻: 이동성의, 철새의
- 어원적 연결: 주기적으로 장소를 옮기는 성질
- 사용 맥락: `migratory birds` = 철새

### transmigration

- 구성: [[trans]] = across + `migration`
- 뜻: 이주, 영혼의 전생·윤회
- 어원적 연결: 경계를 넘어 옮겨 감
- 사용 맥락: 종교·철학 맥락에서는 영혼의 이동

## 기억 포인트

`migr` 계열은 "장소를 옮겨 감" 이미지.

- `migration`: 이동 과정
- `immigrant`: 들어온 사람
- `emigrant`: 떠나간 사람
- `migratory`: 주기적으로 이동하는

## 비교 단어

### immigrate / emigrate

- `immigrate`: 도착지 안으로 들어옴
- `emigrate`: 출발지 밖으로 나감
