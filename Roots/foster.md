# foster

- 출발 단어: `foster`
- 핵심 어원: Old English `fostrian` = feed, nourish, bring up
- 핵심 이미지: 먹이고 돌보아 자라게 함
- 관련 문서: [[food]], [[nutri]]
- 주의: Latin 계열 접사가 아니라 Germanic 기본어 계열. 현대 영어에서는 비유적으로 "발전·감정·관계를 키우다"에 자주 씀.

## 출발 단어 분석

`foster`는 현대 형태에서 접두사·접미사로 나누기보다 Old English `fostrian` 전체로 보는 단어.

- 원래 의미: 먹이다, 양육하다
- 의미 확장: 아이를 돌봄 → 어떤 성질·관계·발전을 키움
- 현대 의미: 조성하다, 촉진하다; 위탁 양육하다

## 단어들

### foster

- 구성: Old English `fostrian` = feed, nourish, bring up
- 뜻: 조성하다, 촉진하다; 위탁 양육하다
- 어원적 연결: 먹이고 돌보아 자라게 하듯 관계·능력·환경을 키움
- 사용 맥락: `foster innovation` = 혁신을 촉진하다, `foster a child` = 아이를 위탁 양육하다

### fostering

- 구성: `foster` + `-ing`
- 뜻: 양육, 조성; 촉진하는
- 어원적 연결: 자라도록 계속 돌보고 키우는 과정
- 사용 맥락: `the fostering of trust` = 신뢰의 조성

### fostered

- 구성: `foster` + `-ed`
- 뜻: 조성된, 길러진; 위탁 양육된
- 어원적 연결: 돌봄과 환경에 의해 자라난 상태
- 사용 맥락: `a culture fostered by leadership` = 리더십에 의해 조성된 문화

### fosterage

- 구성: `foster` + `-age`
- 뜻: 위탁 양육, 양육 상태
- 어원적 연결: 아이를 맡아 먹이고 기르는 제도나 상태
- 사용 맥락: 문어적·역사적 표현

### foster parent

- 구성: `foster` + `parent`
- 뜻: 위탁 부모, 양부모
- 어원적 연결: 친부모가 아니지만 아이를 돌보고 기르는 사람
- 사용 맥락: `live with foster parents` = 위탁 부모와 살다

### foster care

- 구성: `foster` + `care`
- 뜻: 위탁 보호, 위탁 양육 제도
- 어원적 연결: 아이를 안전하게 돌보아 기르는 제도
- 사용 맥락: `children in foster care` = 위탁 보호를 받는 아이들

### foster home

- 구성: `foster` + `home`
- 뜻: 위탁 가정
- 어원적 연결: 아이가 맡겨져 돌봄을 받는 집
- 사용 맥락: 사회복지·가정법 맥락

## 기억 포인트

`foster`는 "먹이고 돌보아 자라게 함" 이미지.

- `foster a child`: 아이를 맡아 기름
- `foster trust`: 신뢰를 키움
- `foster innovation`: 혁신이 자라날 환경을 만듦
- `foster care`: 위탁 양육 제도

## 비교 단어

### nurture

- 뜻: 양육하다, 키우다
- 구분 포인트: Latin `nutrire` = nourish 계열. `foster`보다 일반적인 돌봄·성장 지원 느낌.

### nourish

- 뜻: 영양분을 주다, 기르다
- 구분 포인트: 음식·영양으로 살게 하는 이미지가 강함. 비유적으로 생각·관계를 키울 때도 사용.

### promote

- 뜻: 촉진하다, 승진시키다
- 구분 포인트: 앞으로 밀어 올리는 이미지. `foster`는 환경을 조성해 자라게 하는 느낌이 더 강함.
