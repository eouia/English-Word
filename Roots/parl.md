# parl

- 출발 단어: `parlour`
- 핵심 어원: Old French `parler` = 말하다
- 더 깊은 어원: Late Latin `parabolare` = 말하다, 담화하다
- 핵심 의미: 말하다, 대화하다, 말로 협의하다
- 관련 문서: [[logos]], [[dict]], [[phras]], [[spond]]
- 주의: `parlour`는 British spelling, `parlor`는 American spelling. 둘 다 “말하는 방”에서 출발.

## 출발 단어 분석

`parlour` ← Old French `parleor/parloir` = 말하는 곳

- `parler`: 말하다
- 원래 이미지: 수도원에서 외부 사람과 대화하는 방
- 현대 의미: 응접실, 영업용 방

## 단어들

### parlour

- 발음: /ˈpɑrlɚ/
- 구성: Old French `parleor` = speaking place
- 뜻: 응접실, 영업용 방
- 어원적 연결: 사람을 맞아 대화하는 방
- 사용 맥락: `an ice-cream parlour` = 아이스크림 가게

### parlor

- 발음: /ˈpɑrlɚ/
- 구성: `parlour`의 American spelling
- 뜻: 응접실, 영업용 방
- 어원적 연결: 말하고 손님을 맞는 공간
- 사용 맥락: `a beauty parlor` = 미용실

### parley

- 발음: /ˈpɑrli/
- 구성: Old French `parlee` ← `parler`
- 뜻: 협상, 회담; 협상하다
- 어원적 연결: 특히 적이나 상대와 말로 협의함
- 사용 맥락: `hold a parley` = 회담을 열다

### parliament

- 발음: /ˈpɑrləmənt/
- 구성: Old French `parlement` ← `parler`
- 뜻: 의회
- 어원적 연결: 말하고 토론하는 공식 회의체
- 사용 맥락: `the British Parliament` = 영국 의회

### parliamentary

- 발음: /ˌpɑrləˈmɛntɚi/
- 구성: `parliament` + `-ary`
- 뜻: 의회의, 의회식의
- 어원적 연결: 토론과 회의로 운영되는 의회에 속함
- 사용 맥락: `parliamentary democracy` = 의회 민주주의

### parliamentarian

- 발음: /ˌpɑrləməˈntɛriən/
- 구성: `parliamentary` + `-ian`
- 뜻: 의원, 의회 절차 전문가
- 어원적 연결: 의회 토론과 절차에 참여하는 사람
- 사용 맥락: `a veteran parliamentarian` = 노련한 의원

### parlance

- 발음: /ˈpɑrləns/
- 구성: Old French `parlaunce` ← `parler`
- 뜻: 말투, 표현 방식, 용어
- 어원적 연결: 사람들이 말하는 방식
- 사용 맥락: `in common parlance` = 일반적인 말로는

### parole

- 발음: /pɚˈoʊl/
- 구성: French `parole` = word, speech
- 뜻: 가석방; 약속의 말
- 어원적 연결: 원래 “명예의 말, 약속” → 조건을 지키겠다는 약속 아래 풀려남
- 사용 맥락: `be released on parole` = 가석방되다

### parol

- 구성: French `parole`의 법률형
- 뜻: 구두의, 말로 된; 구두 약속
- 어원적 연결: 글보다 말로 한 약속
- 사용 맥락: legal context

### palaver

- 구성: Portuguese `palavra` = word ← Latin `parabola`
- 뜻: 장황한 말, 실없는 이야기, 협상
- 어원적 연결: 말이 길게 오가는 상황
- 사용 맥락: `cut the palaver` = 장황한 말은 그만하다

### parler

- 발음: /ˈpɑrlɚ/
- 구성: French `parler`
- 뜻: 말하다
- 어원적 연결: `parl-` 계열의 직접 출발점
- 사용 맥락: French loan / etymology context

## 깊은 연결: parabola / parable

`parler`는 더 깊게 Late Latin `parabolare`와 연결됨.

- `parabola`: Greek `parabole` = beside-throwing, comparison
- `parable`: 비유 이야기
- 의미 변화: 비유·비교 이야기 → 말, 담화 → 말하다

이 연결은 깊은 어원 이해용. 시험 암기에서는 `parl-` = 말하다, 협의하다로 잡으면 충분.

## 기억 포인트

- `parlour/parlor`: 말하고 손님을 맞는 방.
- `parley`: 말로 협상함.
- `parliament`: 말하고 토론하는 의회.
- `parlance`: 말투, 표현 방식.
- `parole`: 말로 한 약속 → 가석방 조건.

## 비교 단어

### dict

- [[dict]]는 말하다, 선언하다.
- `parl`은 대화·토론·협의의 말하기에 가까움.

### logos

- [[logos]]는 말, 설명, 이성, 논리.
- `parl`은 French `parler`에서 온 실제 대화 행위 쪽.

### phras

- [[phras]]는 표현의 묶음, 말의 방식.
- `parlance`와 의미가 가까우나, `parlance`는 특정 집단의 말투나 용법에 초점.

### spond

- [[spond]]는 약속하다, 보증하다.
- `parole`은 “말로 한 약속”에서 출발한 법률·감시 용어.
