# bold

- 출발 단어: `bold`
- 핵심 어원: Old English `beald` = brave, confident, strong
- 핵심 이미지: 겁내지 않고 앞으로 나서는 힘
- 관련 문서: [[cord]]
- 주의: `bold`는 현대 영어에서 더 작은 접두사·어근으로 나누기보다 Old English 기본 형용사로 보는 편이 안전함.

## 출발 단어 분석

`bold` ← Old English `beald`

- `beald`: brave, confident, strong
- 원래 이미지: 강하고 용감하여 물러서지 않음
- 의미 변화: 용감한 태도 → 자신감 있고 대담한 행동 → 눈에 띄게 굵고 강한 형태
- 현대 의미: 대담한, 용감한; 선명한, 굵은

## 단어들

### bold

- 구성: Old English `beald` = brave, confident, strong
- 뜻: 대담한, 용감한; 굵은, 선명한
- 어원적 연결: 겁내지 않고 드러나게 나서는 성질
- 사용 맥락: `a bold decision` = 대담한 결정

### boldly

- 구성: `bold` + `-ly`
- 뜻: 대담하게, 과감하게
- 어원적 연결: 두려움 없이 앞에 나서는 방식
- 사용 맥락: `speak boldly` = 대담하게 말하다

### boldness

- 구성: `bold` + `-ness`
- 뜻: 대담함, 과감함; 두드러짐
- 어원적 연결: 물러서지 않고 강하게 드러나는 성질
- 사용 맥락: `the boldness of the plan` = 계획의 대담함

### embolden

- 구성: `em-` = make + `bold` + `-en`
- 뜻: 용기를 북돋우다, 대담하게 만들다
- 어원적 연결: 누군가를 더 `bold`한 상태로 만듦
- 사용 맥락: `embolden protesters` = 시위자들에게 용기를 주다

### boldface

- 구성: `bold` + `face`
- 뜻: 굵은 활자, 볼드체
- 어원적 연결: 글자가 강하고 눈에 띄게 드러남
- 사용 맥락: `use boldface for headings` = 제목에 볼드체를 쓰다

## 기억 포인트

`bold` 계열은 "겁내지 않고 강하게 드러남" 이미지.

- `bold`: 대담한, 굵은
- `boldly`: 대담하게
- `boldness`: 대담함
- `embolden`: 대담하게 만들다
- `boldface`: 볼드체

## 비교 단어

### brave

- 뜻: 용감한
- 구분 포인트: 위험이나 두려움 앞에서 견디는 용기. `bold`는 더 적극적으로 나서거나 눈에 띄게 드러나는 느낌.

### courage

- 뜻: 용기
- 구분 포인트: [[cord]] 계열로 마음·심장에서 나온 용기. `bold`는 태도나 표현이 겉으로 강하게 드러나는 쪽.

### audacious

- 뜻: 대담한, 무모할 만큼 과감한
- 구분 포인트: 칭찬과 비판이 모두 가능하지만, `audacious`는 규범을 넘거나 위험을 감수하는 뉘앙스가 더 강함.

### blatant

- 뜻: 노골적인, 뻔뻔한
- 구분 포인트: `bold`가 긍정적 용기일 수 있는 반면, `blatant`는 나쁜 행동이 숨김없이 드러나는 부정적 표현.
