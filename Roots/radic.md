# radic

- 출발 단어: `radical`
- 핵심 어원: Latin `radix`, `radicis` = 뿌리
- 핵심 이미지: 뿌리와 근본
- 관련 문서: [[origin]], [[plant]]

## 출발 단어 분석

`radical` ← Latin `radix`

- `radix`: root
- 원래 이미지: 뿌리에 속한, 근본적인
- 현대 의미: 근본적인; 급진적인

## 단어들

### radical

- 발음: /ˈrædəkəl/
- 구성: Latin `radix` + `-al`
- 뜻: 근본적인; 급진적인
- 어원적 연결: 문제의 뿌리까지 건드림
- 사용 맥락: `radical change` = 근본적 변화
- 주의: 정치 맥락에서는 기존 체제를 뿌리부터 바꾸려는 "급진적" 의미.

### radically

- 발음: /ˈrædɪkli/
- 구성: `radical` + `-ly`
- 뜻: 근본적으로, 급진적으로
- 어원적 연결: 표면이 아니라 뿌리 수준에서 달라짐
- 사용 맥락: `radically different` = 근본적으로 다른

### eradicate

- 발음: /ɪˈrædəˌkeɪt/
- 구성: `e-/ex-` = out + Latin `radix`
- 뜻: 뿌리 뽑다, 근절하다
- 어원적 연결: 뿌리째 뽑아 없앰
- 사용 맥락: `eradicate poverty` = 빈곤을 근절하다

### eradication

- 발음: /ɪˌrædəˈkeɪʃən/
- 구성: `eradicate` + `-ion`
- 뜻: 근절, 박멸
- 어원적 연결: 뿌리째 제거하는 과정
- 사용 맥락: `disease eradication` = 질병 박멸

### radish

- 발음: /ˈrædɪʃ/
- 구성: Latin `radix` 계열
- 뜻: 무, 래디시
- 어원적 연결: 먹는 뿌리 식물
- 사용 맥락: `radish salad` = 무 샐러드

### radicle

- 구성: Latin `radicula` = little root
- 뜻: 어린뿌리
- 어원적 연결: 식물의 작은 뿌리
- 사용 맥락: 생물학·식물학 문맥

## 기억 포인트

`radic` 계열은 "뿌리와 근본" 이미지.

- `radical`: 뿌리 수준의, 급진적인
- `eradicate`: 뿌리째 없애다
- `eradication`: 근절
- `radish`: 뿌리 식물
- `radicle`: 어린뿌리
