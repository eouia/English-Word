# enth

- 출발 단어: `enthusiasm`
- 핵심 어원: Greek `entheos` = 신이 안에 들어온
- 구성 이미지: `en-` = in + `theos` = god
- 핵심 이미지: 마음 안에 강한 영감과 열기가 들어옴
- 관련 문서: [[spir]], [[theo]]
- 주의: 현대 영어에서는 종교적 의미보다 열정, 몰입, 흥미의 뜻이 중심.

## 출발 단어 분석

`enthusiasm` ← Greek `enthousiasmos`

- `en-`: 안에
- `theos`: 신
- 원래 이미지: 신적 기운이 안에 들어와 사로잡힌 상태
- 현대 의미: 열정, 열의

## 단어들

### enthusiasm

- 발음: /ɪˈnθuziˌæzəm/
- 구성: Greek `entheos` + `-asm`
- 뜻: 열정, 열의
- 어원적 연결: 마음속에 강한 영감과 기운이 들어온 상태
- 사용 맥락: `show enthusiasm for the project` = 프로젝트에 열의를 보이다

### enthusiastic

- 발음: /ɪˌnθuziˈæstɪk/
- 구성: `enthusiasm` + `-astic`
- 뜻: 열정적인
- 어원적 연결: 안에서 솟는 강한 기운을 보임
- 사용 맥락: `an enthusiastic response` = 열정적인 반응

### enthusiast

- 발음: /ɛˈnθuziˌæst/
- 구성: `enthusiasm` + `-ist`
- 뜻: 열성가, 애호가
- 어원적 연결: 어떤 대상에 강하게 사로잡힌 사람
- 사용 맥락: `a film enthusiast` = 영화 애호가

### enthuse

- 발음: /ɪˈnθuz/
- 구성: `enthusiasm`에서 역성된 동사
- 뜻: 열광하다, 열광하게 하다
- 어원적 연결: 열정의 상태를 말이나 행동으로 드러냄
- 사용 맥락: `critics enthused about the novel` = 비평가들이 그 소설에 열광함

### entheogen

- 구성: `entheos` = inspired by god + `-gen` = produce
- 뜻: 신성 체험을 일으키는 물질
- 어원적 연결: 안에 신적 경험을 생기게 한다는 뜻
- 사용 맥락: 종교·인류학 맥락에서 제한적으로 쓰임

## 기억 포인트

`enth` 계열은 "안에 들어온 영감과 열정" 이미지.

- `enthusiasm`: 안에서 솟는 열정
- `enthusiastic`: 열정이 드러나는
- `enthusiast`: 무언가에 열정을 가진 사람
- `enthuse`: 열광하다

## 비교 단어

### spir

- 뜻: 숨쉬다, 영감을 불어넣다
- 구분 포인트: [[spir]]의 `inspire`는 숨이나 정신을 불어넣는 이미지. `enth`는 신적 기운이 안에 들어와 열정이 생기는 이미지.

### theo

- 뜻: 신
- 구분 포인트: [[theo]]는 신 자체와 신학 계열. `enth`는 `theos`가 `en-`과 결합해 열정·몰입으로 굳어진 계열.
