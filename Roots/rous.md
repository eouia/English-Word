# rous

- 출발 단어: `rousing`
- 핵심 어원: Old Norse 계열로 추정되는 `rouse`
- 핵심 이미지: 잠든 상태나 가라앉은 마음을 깨워 일으킴
- 관련 문서: 없음
- 주의: 깊은 어원은 불확실함. Latin/Greek 어근처럼 잘게 분해하지 않고 `rouse` 계열로 묶어 기억.

## 출발 단어 분석

`rousing` = `rouse` + `-ing`

- `rouse`: 깨우다, 일으키다, 자극하다
- `-ing`: 성질이나 진행을 나타내는 접미사
- 원래 이미지: 잠이나 무기력에서 깨워 일어나게 함
- 현대 의미: 고무적인, 열렬한, 사람을 일으키는

## 단어들

### rouse

- 구성: Old Norse 계열로 추정. 세부 어원은 불확실함
- 뜻: 깨우다, 일으키다, 자극하다
- 어원적 연결: 잠든 몸이나 가라앉은 마음을 움직이게 함
- 사용 맥락: `rouse public anger` = 대중의 분노를 일으키다

### rousing

- 구성: `rouse` + `-ing`
- 뜻: 고무적인, 열렬한, 사람을 일으키는
- 어원적 연결: 감정과 의지를 깨워 강하게 움직이게 함
- 사용 맥락: `a rousing speech` = 사람들을 고무하는 연설

### arouse

- 구성: `a-` + `rouse`
- 뜻: 깨우다, 불러일으키다, 자극하다
- 어원적 연결: 감정·관심·욕구를 일어나게 함
- 사용 맥락: `arouse curiosity` = 호기심을 불러일으키다

### arousal

- 구성: `arouse` + `-al`
- 뜻: 각성, 흥분, 자극
- 어원적 연결: 몸이나 마음이 깨어 활성화되는 상태
- 사용 맥락: psychology / biology context

### roused

- 구성: `rouse` + `-ed`
- 뜻: 깨어난, 자극받은, 분기한
- 어원적 연결: 잠잠하던 상태에서 일어나 움직이게 됨
- 사용 맥락: `roused by the noise` = 소리에 깨어난

## 기억 포인트

`rous` 계열은 "깨워 일으킴" 이미지.

- `rouse`: 깨우다, 일으키다
- `rousing`: 고무적인, 열렬한
- `arouse`: 감정·관심을 불러일으키다
- `arousal`: 각성, 흥분
- `roused`: 깨어난, 자극받은

## 비교 단어

### rise

- 뜻: 일어나다, 오르다
- 구분 포인트: `rise`는 스스로 올라가거나 일어남. `rouse`는 누군가·무언가가 깨워 일으킴.

### wake

- 뜻: 깨다, 깨우다
- 구분 포인트: `wake`는 잠에서 깨는 기본어. `rouse`는 감정·의지·관심을 강하게 자극하는 의미로도 넓게 씀.
