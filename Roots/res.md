# res

- 출발 단어: `real`
- 핵심 어원: Latin `res` = thing, matter, affair, property
- 핵심 이미지: 실제로 있는 것, 사물, 일
- 관련 어근: [[sist]], [[stat]], [[popul]]

## 출발 단어 분석

`real`은 Old French `reel`을 거쳐 Late Latin `realis`에서 온 말.

`realis`는 Latin `res` = thing, matter, affair에서 나온 말로, 원래 "사물 자체에 속한, 실제의"라는 뜻.

따라서 `real`은 "생각이나 상상이 아니라 실제 사물로 있는" → "진짜의, 실제의" 뜻.

## 단어들

### real

- 발음: /ˈril/
- 구성: Latin `res` + `-alis`
- 의미: 실제의, 진짜의
- 어원적 연결: 사물 자체에 속함
- 사용 맥락: `a real problem` = 실제 문제

### reality

- 발음: /ˌriˈæləˌti/
- 구성: `real` + `-ity`
- 의미: 현실, 실제
- 어원적 연결: 실제 사물이나 사실로 존재하는 상태
- 사용 맥락: `face reality` = 현실을 직시하다

### realistic

- 발음: /ˌriəˈlɪstɪk/
- 구성: `real` + `-istic`
- 의미: 현실적인, 사실적인
- 어원적 연결: 실제와 맞닿아 있음
- 사용 맥락: `a realistic plan` = 현실적인 계획

### realize

- 발음: /ˈriəˌlaɪz/
- 구성: `real` + `-ize`
- 의미: 깨닫다, 실현하다
- 어원적 연결: 어떤 것을 현실로 만들거나 실제로 알아차림
- 사용 맥락: `realize the danger` = 위험을 깨닫다

### realization

- 발음: /ˈriləˈzeɪʃən/
- 구성: `realize` + `-ation`
- 의미: 깨달음, 실현
- 어원적 연결: 실제가 되거나 실제로 알아차리는 과정
- 사용 맥락: `the realization of a dream` = 꿈의 실현

### realty

- 발음: /ˈriəˌlti/
- 구성: `real` + `-ty`
- 의미: 부동산
- 어원적 연결: 법에서 "실물 재산, 토지 재산"을 가리킴
- 사용 맥락: `realty market` = 부동산 시장

### real estate

- 구성: `real` + [[stat]]의 `estate`
- 의미: 부동산
- 어원적 연결: 움직이는 개인 물건이 아니라 실제 토지와 건물로 된 재산
- 사용 맥락: `real estate prices` = 부동산 가격

### republic

- 발음: /riˈpʌblək/
- 구성: Latin `res publica` = public thing
- 의미: 공화국
- 어원적 연결: 국가를 왕 개인의 소유가 아니라 공적인 일로 봄
- 사용 맥락: `the Republic of Korea` = 대한민국

## 기억 포인트

- `real`: 실제 사물로 있는
- `reality`: 실제 상태
- `realistic`: 현실에 맞는
- `realize`: 실제로 알아차리거나 실현함
- `real estate`: 실제 재산, 부동산
- `republic`: 공적인 일

## 비교 단어

- [[sist]]
  - 핵심: 서다, 존재하다
  - 구분 포인트: `exist`는 존재해 서 있음. `real`은 실제 사물/일로 있음

- [[stat]]
  - 핵심: 서 있는 상태
  - 구분 포인트: `estate`는 상태, 지위, 재산 계열. `real estate`의 `real`은 Latin `res` = thing에서 온 말

- [[popul]]
  - 핵심: 사람들, 공공
  - 구분 포인트: `republic`은 `res publica` = 공적인 일. `public`은 사람들/공공의 뜻

- `realm`
  - 핵심: kingdom, royal power
  - 구분 포인트: `realm`은 Old French `realme/royaume` 계열로 왕권/왕국과 관련. `real` = 실제의 와 어원 다름
