# sangu

- 출발 단어: `sanguine`
- 핵심 어원: Latin `sanguis` = blood
- 핵심 이미지: 피, 혈색, 피와 연결된 기질
- 관련 문서: 없음
- 주의: `sanguine`의 "낙관적인" 뜻은 옛 기질론에서 혈색 좋은 사람이 쾌활하다고 본 데서 확장됨.

## 출발 단어 분석

`sanguine` ← Latin `sanguineus` ← `sanguis`

- `sanguis`: 피
- `sanguineus`: 피의, 피 같은, 혈색 좋은
- 의미 변화: 피의 색 → 혈색 좋은 → 쾌활한 기질 → 낙관적인
- 현대 의미: 낙관적인, 자신 있는; 혈색의

## 단어들

### sanguine

- 구성: Latin `sanguineus` ← `sanguis`
- 뜻: 낙관적인, 자신 있는; 혈색의
- 어원적 연결: 혈색 좋고 쾌활한 기질에서 낙관성으로 확장
- 사용 맥락: `remain sanguine about the future` = 미래에 대해 낙관적이다

### sanguinary

- 구성: Latin `sanguinarius` ← `sanguis`
- 뜻: 피비린내 나는, 잔혹한, 유혈의
- 어원적 연결: 피가 많이 흐르는 폭력이나 전쟁
- 사용 맥락: `a sanguinary battle` = 유혈이 낭자한 전투

### sanguineous

- 구성: Latin `sanguineus`
- 뜻: 피의, 피 같은, 혈액을 포함한
- 어원적 연결: 피와 직접 관련됨
- 사용 맥락: medical / formal context

### consanguinity

- 구성: `con-` = together + Latin `sanguis`
- 뜻: 혈족 관계, 혈연
- 어원적 연결: 같은 피를 나눈 관계
- 사용 맥락: legal / anthropology context

### sangria

- 구성: Spanish `sangria` ← `sangre` = blood
- 뜻: 상그리아
- 어원적 연결: 붉은 색이 피를 연상시킴
- 사용 맥락: drink name

## 기억 포인트

`sangu` 계열은 "피와 혈색" 이미지.

- `sanguine`: 혈색 좋은 → 쾌활한 → 낙관적인
- `sanguinary`: 피비린내 나는, 잔혹한
- `sanguineous`: 피의
- `consanguinity`: 같은 피, 혈연
- `sangria`: 피 같은 붉은 음료 이름

## 비교 단어

### hemat / hemo

- 뜻: 피
- 구분 포인트: Greek 계열. `hematology`, `hemoglobin`, `hemorrhage`처럼 의학·과학 단어에 많음. `sangu`는 Latin 계열이라 `sanguine`, `sanguinary`, `consanguinity`에 나타남.

### optimistic

- 뜻: 낙관적인
- 구분 포인트: `optimistic`은 좋은 결과를 기대한다는 일반어. `sanguine`은 문어적이고, 혈색 좋은 쾌활한 기질에서 온 낙관성의 뉘앙스가 있음.
