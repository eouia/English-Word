# libr

- 출발 단어: `deliberation`
- 핵심 어원: Latin `libra` = balance, scales
- 핵심 이미지: 저울에 올려 무게를 재고 균형을 맞춤
- 관련 문서: [[metr]], [[calibr]]
- 주의: `liberty`, `liberate`의 Latin `liber` = free 계열과 다름. `deliberation`은 "자유"가 아니라 "저울질하며 판단함" 이미지.

## 출발 단어 분석

`deliberation` ← Latin `deliberare` = weigh well, consider

- `de-`: down, thoroughly
- `libra`: balance, scales
- `-tion`: 명사 접미사
- 원래 이미지: 저울에 올려 충분히 무게를 재 봄
- 의미 변화: 무게를 재다 → 여러 선택지를 신중히 따져 보다 → 숙고, 심의
- 현대 의미: 숙고, 신중한 논의, 심의

## 단어들

### deliberation

- 구성: Latin `deliberare` = weigh well, consider + `-tion`
- 뜻: 숙고, 심의, 신중한 논의
- 어원적 연결: 선택지의 무게를 하나씩 달아 보고 판단함
- 사용 맥락: `after careful deliberation` = 신중한 숙고 끝에

### deliberate

- 구성: Latin `deliberatus` = considered, resolved
- 뜻: 신중한; 의도적인; 숙고하다
- 어원적 연결: 충분히 저울질한 뒤 결정함
- 사용 맥락: `a deliberate decision` = 신중한 결정, `deliberate on the issue` = 그 문제를 심의하다

### deliberately

- 구성: `deliberate` + `-ly`
- 뜻: 신중하게; 의도적으로
- 어원적 연결: 생각 없이 한 행동이 아니라 따져 보고 선택한 방식
- 사용 맥락: `act deliberately` = 의도적으로 행동하다

### deliberative

- 구성: `deliberate` + `-ive`
- 뜻: 심의하는, 숙의적인
- 어원적 연결: 결정을 내리기 전에 여러 의견과 근거를 저울질함
- 사용 맥락: `a deliberative body` = 심의 기관

### equilibrium

- 구성: Latin `aequi-` = equal + `libra` = balance
- 뜻: 균형, 평형
- 어원적 연결: 저울 양쪽이 같은 무게로 맞춰진 상태
- 사용 맥락: `economic equilibrium` = 경제적 균형

### equilibrate

- 구성: `equilibrium` 계열 + `-ate`
- 뜻: 평형을 이루게 하다, 균형을 맞추다
- 어원적 연결: 저울이 한쪽으로 기울지 않도록 맞춤
- 사용 맥락: `equilibrate the system` = 체계의 균형을 맞추다

### Libra

- 구성: Latin `libra` = scales
- 뜻: 천칭자리, 천칭궁
- 어원적 연결: 별자리 이름 자체가 "저울"을 가리킴
- 사용 맥락: `the constellation Libra` = 천칭자리

## 기억 포인트

`libr` 계열은 "저울질" 이미지.

- `deliberation`: 선택지를 저울질하며 숙고함
- `deliberate`: 충분히 따져 본, 의도적인
- `deliberative`: 함께 저울질하며 심의하는
- `equilibrium`: 저울 양쪽이 맞은 균형
- `Libra`: 저울 모양의 천칭자리

## 비교 단어

### liber

- 뜻: 자유로운, 풀어 줌
- 구분 포인트: `liberty`, `liberate`, `liberal`은 Latin `liber` = free 계열. `deliberation`의 `libr`는 Latin `libra` = balance, scales 계열.

### ponder

- 뜻: 무게를 달다, 깊이 생각하다
- 구분 포인트: `ponder`가 "무게" 자체에서 생각으로 확장된다면, `libr`는 "저울"로 균형을 맞추며 판단하는 느낌.

### calibr

- 뜻: 기준에 맞게 조정함
- 구분 포인트: [[calibr]]는 장비나 판단을 기준값에 맞추는 보정. `libr`는 선택지의 무게를 달아 균형 있게 판단함.
