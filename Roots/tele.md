# tele

- 출발 단어: `telescope`
- 핵심 의미: 멀리, 먼 거리에서
- 관련 형태: Greek `tele` = far off, at a distance
- 관련 문서: [[scope]], [[phone]], [[graph]], [[vid]]

## 핵심 이미지

`tele-`의 바탕은 "멀리 떨어져 있음".

멀리 보는 도구는 `telescope`.
멀리 소리를 보내면 `telephone`.
멀리 영상을 보내면 `television`.
멀리 글을 보내면 `telegraph`.
멀리 있는 장면을 찍으면 `telephoto`.

## 단어들

### tele

- 발음: /ˈtɛli/
- 구성: Greek `tele` = far, distant
- 뜻: 멀리, 원격으로
- 연결: 공간적으로 떨어진 곳까지 닿음
- 사용 맥락: 기술·통신 단어에 자주 보임.

### telescope

- 발음: /ˈtɛləˌskoʊp/
- 구성: `tele-` = far + `scope` = look
- 뜻: 망원경
- 연결: 멀리 있는 것을 보게 해 주는 도구
- 사용 맥락: `a radio telescope` = 전파 망원경

### telephone

- 발음: /ˈtɛləˌfoʊn/
- 구성: `tele-` = far + Greek `phone` = voice, sound
- 뜻: 전화
- 연결: 먼 거리로 소리를 보냄
- 사용 맥락: `telephone call` = 전화 통화

### television

- 발음: /ˈtɛləˌvɪʒən/
- 구성: `tele-` = far + `vision` = seeing
- 뜻: 텔레비전, TV
- 연결: 먼 곳의 영상을 보게 함
- 사용 맥락: `watch television` = TV를 보다

### telegraph

- 발음: /ˈtɛləˌgræf/
- 구성: `tele-` = far + `graph` = write
- 뜻: 전신, 전보
- 연결: 먼 거리로 글이나 신호를 보냄
- 사용 맥락: 역사·통신 문맥.

### telephoto

- 발음: /ˈtɛləˌfoʊtoʊ/
- 구성: `tele-` = far + `photo` = light/image
- 뜻: 망원 사진의, 망원 렌즈의
- 연결: 멀리 있는 장면을 가까이 찍음
- 사용 맥락: `a telephoto lens` = 망원 렌즈

### telecommute

- 발음: /ˈtɛləkəˌmjut/
- 구성: `tele-` = far + `commute`
- 뜻: 재택근무하다, 원격근무하다
- 연결: 멀리 떨어진 곳에서 일함
- 사용 맥락: work / technology 문맥.

## 비교 단어

`scope`는 [[scope]] 문서처럼 보다, 살피다 계열.
`telescope`는 `tele-`와 `scope`가 결합한 말.

`graph`는 [[graph]] 문서처럼 쓰다, 기록하다 계열.
`telegraph`는 먼 거리로 글이나 신호를 보낸다는 뜻.
