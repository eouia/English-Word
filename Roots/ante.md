# ante

- 출발 단어: `antiquity`
- 핵심 어원: Latin `ante` = before, in front of
- 깊은 어원: PIE `ant-` = 앞, 이마, 앞쪽
- 핵심 의미: 앞에 있음, 이전에 있음, 먼저 옴
- 관련 문서: [[cess]], [[cap]], [[arch]], [[vid]]
- 주의: `antique`, `antiquity`의 `anti-`는 “반대” 접두사가 아님. Latin `ante` = before 계열.

## 출발 단어 분석

`antiquity` ← Latin `antiquitas` ← `antiquus`

- `antiquus`: 오래된, 옛날의, 이전 시대의
- `ante`: 앞에, 이전에
- 원래 이미지: 지금보다 앞선 시대에 속함
- 현대 의미: 고대, 고대성, 오래됨

## 단어들

### antiquity

- 구성: Latin `antiquitas` ← `antiquus`
- 뜻: 고대, 오래됨, 고대 유물
- 어원적 연결: 지금보다 앞선 먼 시대
- 사용 맥락: `classical antiquity` = 고전 고대

### antique

- 구성: Latin `antiquus` = ancient, old
- 뜻: 골동품, 오래된; 고풍의
- 어원적 연결: 이전 시대에 속한 물건이나 양식
- 사용 맥락: `an antique chair` = 골동품 의자

### antiquated

- 구성: `antiquate` + `-ed`
- 뜻: 구식의, 낡은
- 어원적 연결: 너무 옛날 방식이라 지금에는 맞지 않음
- 사용 맥락: `antiquated laws` = 낡은 법

### antiquarian

- 구성: Latin `antiquarius` ← `antiquus`
- 뜻: 골동품 수집가, 고물 연구가; 고대 유물의
- 어원적 연결: 옛 물건과 고대에 관심 있는 사람
- 사용 맥락: `an antiquarian bookshop` = 고서점

### ancient

- 구성: Old French `ancien` ← Vulgar Latin `anteanus`
- 뜻: 고대의, 아주 오래된
- 어원적 연결: 이전 시대에 속함
- 사용 맥락: `ancient history` = 고대사

### ancestor

- 구성: Late Latin `antecessor` = one who goes before
- 뜻: 조상, 선조
- 어원적 연결: 나보다 앞서 간 사람
- 사용 맥락: `our ancestors` = 우리의 조상들

### ancestral

- 구성: `ancestor` + `-al`
- 뜻: 조상의, 대대로 내려온
- 어원적 연결: 앞선 세대에서 이어짐
- 사용 맥락: `ancestral land` = 조상의 땅

### antecedent

- 구성: `ante-` = before + [[cess]] = go
- 뜻: 선행사, 앞선 사건, 전례
- 어원적 연결: 앞에 가 있는 것
- 사용 맥락: `a pronoun's antecedent` = 대명사의 선행사

### antecede

- 구성: `ante-` = before + [[cess]]
- 뜻: 앞서다, 선행하다
- 어원적 연결: 시간이나 순서상 먼저 감
- 사용 맥락: `events that antecede the war` = 전쟁보다 앞선 사건들

### antedate

- 구성: `ante-` = before + `date`
- 뜻: ~보다 앞서다, 날짜를 앞당겨 적다
- 어원적 연결: 특정 날짜보다 이전에 둠
- 사용 맥락: `antedate the discovery` = 그 발견보다 앞서다

### antediluvian

- 구성: `ante-` = before + Latin `diluvium` = flood
- 뜻: 대홍수 이전의; 아주 구식의
- 어원적 연결: 성서의 홍수보다 앞선 시대
- 사용 맥락: `antediluvian ideas` = 아주 낡은 생각

### antebellum

- 구성: `ante-` = before + Latin `bellum` = war
- 뜻: 전쟁 전의, 특히 미국 남북전쟁 전의
- 어원적 연결: 전쟁보다 앞선 시대
- 사용 맥락: `the antebellum South` = 남북전쟁 전 미국 남부

### anterior

- 구성: Latin `anterior` = more before, in front
- 뜻: 앞쪽의, 이전의
- 어원적 연결: 위치나 시간에서 더 앞에 있음
- 사용 맥락: `anterior muscles` = 앞쪽 근육

### anteroom

- 구성: `ante-` = before + `room`
- 뜻: 대기실, 앞방
- 어원적 연결: 본방 앞에 있는 방
- 사용 맥락: `wait in the anteroom` = 대기실에서 기다리다

### anticipate

- 구성: `ante-` = before + [[cap]] = take
- 뜻: 예상하다, 기대하다, 앞서 처리하다
- 어원적 연결: 일이 오기 전에 미리 잡음
- 사용 맥락: `anticipate problems` = 문제를 예상하다

### anticipation

- 구성: `anticipate` + `-ion`
- 뜻: 예상, 기대
- 어원적 연결: 앞서 마음속으로 잡아 둠
- 사용 맥락: `in anticipation of change` = 변화에 대비하여

## 기억 포인트

- `ante`: 앞에, 이전에.
- `antiquity`: 앞선 먼 시대, 고대.
- `antique`: 이전 시대 물건.
- `ancient`: 오래전의.
- `ancestor`: 앞서 간 사람.
- `antecedent`: 앞서 간 것.
- `anticipate`: 앞서 잡다, 예상하다.

## 비교 단어

### anti

- Greek `anti-`는 반대, 맞섬.
- `antiquity`, `antique`, `ancient`의 `anti/anci`는 Latin `ante` = before 계열이므로 분리.

### arch

- [[arch]]는 시작, 원형, 우두머리.
- `archaic`은 오래된 느낌이 있지만 Greek `arkhaios` 계열.
- `antiquity`는 Latin `ante/antiquus` 계열.

### previous

- [[via]]의 `previous`는 앞길에 있는, 이전의.
- `ante`는 더 직접적인 “앞/이전” 접두사.

### precede

- [[cess]]의 `precede`는 앞서 가다.
- `antecede`도 매우 비슷하지만 `ante-`가 “앞에” 의미를 더 노골적으로 드러냄.
