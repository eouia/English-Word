# inter

- 출발 단어: `interest`
- 핵심 의미: 사이에, 서로 사이에서
- 관련 형태: Latin `inter` = between, among
- 관련 문서: [[ess]], [[ag]], [[cap]], [[vid]], [[medi]]

## 핵심 이미지

`inter-`의 바탕은 "둘 이상 사이에 있음".

사이에 있어 관계되면 `interest`.
서로 사이에서 행동하면 `interact`.
나라들 사이이면 `international`.
사람 사이에서 보고 묻고 답하면 `interview`.
중간에서 잡으면 `intercept`.
두 면 사이의 접점은 `interface`.

## 단어들

### inter

- 발음: /ˌɪˈntɝ/
- 구성: Latin `inter` = between, among
- 뜻: 사이에, 서로 사이에서
- 연결: 둘 이상의 대상 사이에 놓임
- 사용 맥락: 접두사로 매우 자주 쓰임.

### interest

- 발음: /ˈɪntrəst/
- 구성: `inter-` = between + `esse` = be
- 뜻: 관심, 흥미; 이익; 이자
- 연결: 어떤 일 사이에 걸려 있어 관련됨
- 사용 맥락: `public interest` = 공익, `interest rate` = 이자율

### interact

- 발음: /ˌɪntɚˈækt/
- 구성: `inter-` = between + `act`
- 뜻: 상호작용하다
- 연결: 서로 사이에서 행동이 오감
- 사용 맥락: `interact with users` = 사용자와 상호작용하다

### international

- 발음: /ˌɪntɚˈnæʃənəl/
- 구성: `inter-` = between + `national`
- 뜻: 국제적인
- 연결: 나라들 사이에 걸침
- 사용 맥락: `international trade` = 국제 무역

### interview

- 발음: /ˈɪntɚˌvju/
- 구성: `inter-` = between + `view`
- 뜻: 면접, 인터뷰
- 연결: 서로 마주 보고 질문과 답을 주고받음
- 사용 맥락: `job interview` = 취업 면접

### intercept

- 발음: /ˌɪntɚˈsɛpt/
- 구성: `inter-` = between + `cept` = catch
- 뜻: 가로채다, 차단하다
- 연결: 중간에서 잡음
- 사용 맥락: `intercept a pass` = 패스를 가로채다

### interface

- 발음: /ˈɪntɚˌfeɪs/
- 구성: `inter-` = between + `face`
- 뜻: 접점, 인터페이스
- 연결: 두 면이나 시스템 사이에서 만나는 지점
- 사용 맥락: `user interface` = 사용자 인터페이스

### intermediate

- 발음: /ˌɪntɚˈmidiɪt/
- 구성: `inter-` = between + `medi` = middle
- 뜻: 중간의, 중급의
- 연결: 두 단계 사이에 있음
- 사용 맥락: `intermediate level` = 중급 수준

## 비교 단어

`intra-`는 "안쪽에서, 내부의"라는 뜻.
`inter-`는 둘 이상 사이의 관계를 가리킴.

`interest`는 [[ess]] 문서에도 있음.
그쪽은 `esse` = 있다 중심, 이 문서는 `inter-` = 사이에 접두사 중심.
