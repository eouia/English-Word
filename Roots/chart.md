# chart

- 출발 단어: `chart`
- 핵심 어원: Greek `khartes` = papyrus leaf, paper → Latin `charta/carta` = paper, document
- 핵심 이미지: 종이에 적은 문서, 지도, 표
- 관련 문서: [[graph]]
- 주의: `cart` = 수레 계열은 별도 어원. `chart`와 철자가 비슷해도 연결하지 않음.

## 출발 단어 분석

`chart`는 현대 형태에서 `ch` + `art`처럼 쪼개지지 않음.

- Greek `khartes`: 파피루스 종이, 종이 한 장
- Latin `charta/carta`: 종이, 문서
- Old French `charte`: 문서, 헌장
- 현대 의미: 도표, 차트, 해도, 지도

## 단어들

### chart

- 발음: /ˈtʃɑrt/
- 구성: Latin `charta/carta` = paper, document
- 뜻: 차트, 도표; 해도, 지도; 기록하다
- 어원적 연결: 종이에 정보를 표시한 것
- 사용 맥락: `a growth chart` = 성장 그래프/도표

### charter

- 발음: /ˈtʃɑrtɚ/
- 구성: Old French `charte` = document + `-er`
- 뜻: 헌장, 인가장; 전세 내다
- 어원적 연결: 권리나 허가를 적어 둔 공식 문서
- 사용 맥락: `the UN Charter` = 유엔 헌장

### Magna Carta

- 구성: Latin `magna` = great + `carta` = charter
- 뜻: 마그나 카르타, 대헌장
- 어원적 연결: "큰 헌장"이라는 뜻의 역사적 문서
- 사용 맥락: English constitutional history
- 관련 문서: [[mag-magn]]

### cartography

- 구성: `carto-` = map + Greek `graphia` = writing, drawing
- 뜻: 지도 제작, 지도학
- 어원적 연결: 종이에 지도를 그리고 기록함
- 사용 맥락: `modern cartography` = 현대 지도 제작
- 관련 문서: [[graph]]

### cartographer

- 구성: `cartography` + `-er`
- 뜻: 지도 제작자
- 어원적 연결: 지도를 그려 기록하는 사람
- 사용 맥락: `a skilled cartographer` = 숙련된 지도 제작자

### cartographic

- 구성: `cartography` + `-ic`
- 뜻: 지도 제작의, 지도학의
- 어원적 연결: 지도에 정보를 표시하는 방식
- 사용 맥락: `cartographic techniques` = 지도 제작 기법

### card

- 발음: /ˈkɑrd/
- 구성: Old French `carte` = card, map, document
- 뜻: 카드, 표, 명함
- 어원적 연결: 작은 종이 조각에 정보나 표시를 담음
- 사용 맥락: `an ID card` = 신분증

### postcard

- 발음: /ˈpoʊˌstkɑrd/
- 구성: `post` + `card`
- 뜻: 엽서
- 어원적 연결: 우편으로 보내는 작은 종이 카드
- 사용 맥락: `send a postcard` = 엽서를 보내다

### discard

- 발음: /dɪˈskɑrd/
- 구성: `dis-` + `card`
- 뜻: 버리다, 폐기하다
- 어원적 연결: 카드놀이에서 필요 없는 카드를 버리는 데서 확장
- 사용 맥락: `discard old assumptions` = 낡은 가정을 버리다

### carton

- 발음: /ˈkɑrtən/
- 구성: French `carton` ← Italian `cartone` = pasteboard
- 뜻: 종이상자, 판지 상자
- 어원적 연결: 두꺼운 종이로 만든 용기
- 사용 맥락: `a carton of milk` = 우유 한 팩/상자

### cartoon

- 발음: /kɑˈrtun/
- 구성: Italian `cartone` = large sheet of paper
- 뜻: 만화; 풍자 그림
- 어원적 연결: 원래는 큰 종이에 그린 밑그림
- 사용 맥락: `a political cartoon` = 정치 풍자 만화

## 기억 포인트

`chart` 계열은 "종이 위에 적은 정보" 이미지.

- `chart`: 정보를 종이에 표시한 도표·지도
- `charter`: 권리와 허가를 적은 공식 문서
- `cartography`: 지도를 그려 기록하는 일
- `card`: 작은 종이 조각
- `carton`: 두꺼운 종이 상자
- `cartoon`: 큰 종이 밑그림에서 만화로 확장

## 비교 단어

- `graph`: 쓰다, 그리다, 기록하다. [[graph]]는 쓰고 그리는 행위. `chart`는 그 결과물인 종이 문서·지도·도표 쪽.
- `map`: 지도, 지도로 나타내다. 지리적 위치를 나타내는 그림. `chart`는 항해용 지도나 데이터 도표까지 넓게 사용.
- `cart`: 수레, 운반하다. Germanic 계열의 수레 단어. `cartography`, `carton`의 `cart-`와 다름.
