# dur

- 출발 단어: `endure`
- 핵심 어원: Latin `durus` = hard, firm; `durare` = harden, last
- 핵심 이미지: 단단해서 오래 버팀, 깨지지 않고 견딤
- 관련 문서: [[tol]], [[pand]], [[ten]], [[long]]
- 주의: `during`은 시간 동안이라는 현대 기능어지만, 역사적으로는 "지속되는 동안" 이미지에서 연결.

## 출발 단어 분석

`endure` = Old French `endurer` ← Latin `indurare`

- `in-`: in, into
- `durare`: harden, last
- `durus`: hard
- 원래 이미지: 단단해져 버팀
- 현대 의미: 견디다, 지속되다

## 단어들

### endure

- 구성: `en-`/`in-` + `dur`
- 뜻: 견디다, 지속되다
- 어원적 연결: 단단해져 깨지지 않고 버팀
- 사용 맥락: `endure hardship` = 고난을 견디다

### endurance

- 구성: `endure` + `-ance`
- 뜻: 인내, 지구력, 지속력
- 어원적 연결: 오래 버티는 힘
- 사용 맥락: `endurance training` = 지구력 훈련

### enduring

- 구성: `endure` + `-ing`
- 뜻: 오래 지속되는, 견디는
- 어원적 연결: 시간이 지나도 버티고 남음
- 사용 맥락: `enduring value` = 지속적 가치

### durable

- 구성: Latin `durabilis` ← `durare`
- 뜻: 내구성 있는, 오래가는
- 어원적 연결: 단단해서 오래 지속됨
- 사용 맥락: `durable goods` = 내구재

### durability

- 구성: `durable` + `-ity`
- 뜻: 내구성
- 어원적 연결: 오래 버티는 성질
- 사용 맥락: `product durability` = 제품 내구성

### duration

- 구성: Latin `duratio` ← `durare`
- 뜻: 지속 기간
- 어원적 연결: 어떤 상태가 계속 버티는 시간
- 사용 맥락: `for the duration of the course` = 강의 기간 동안

### during

- 구성: Middle English, from `dure` = last, continue
- 뜻: ~동안
- 어원적 연결: 어떤 일이 지속되는 시간 안에서
- 사용 맥락: `during the meeting` = 회의 동안

### duress

- 구성: Old French `duresce` ← Latin `duritia` = hardness
- 뜻: 강압, 협박, 압박
- 어원적 연결: 단단하고 가혹한 압력
- 사용 맥락: `under duress` = 강압을 받아

### durance

- 구성: Old French `durance`
- 뜻: 감금, 지속
- 어원적 연결: 오래 붙들어 두는 상태
- 사용 맥락: 문어적·법률적 표현

### obdurate

- 구성: Latin `ob-` = against + `durare`
- 뜻: 완고한, 고집 센
- 어원적 연결: 마음이 단단하게 굳어 바뀌지 않음
- 사용 맥락: `an obdurate refusal` = 완강한 거절

### indurate

- 구성: Latin `in-` + `durare`
- 뜻: 단단하게 하다, 굳히다
- 어원적 연결: 안으로 단단하게 굳음
- 사용 맥락: 의학·문어 맥락

### perdure

- 구성: Latin `per-` = through + `durare`
- 뜻: 오래 지속되다
- 어원적 연결: 끝까지 버티며 지속됨
- 사용 맥락: 철학·문어 맥락

### perdurable

- 구성: `perdure` + `-able`
- 뜻: 오래 지속되는, 영속적인
- 어원적 연결: 오래 버틸 수 있음
- 사용 맥락: 문어적 표현

## 기억 포인트

`dur` 계열은 "단단해서 오래 버팀" 이미지.

- `endure`: 깨지지 않고 견딤
- `endurance`: 오래 버티는 힘
- `durable`: 오래가는
- `duration`: 지속되는 시간
- `duress`: 가혹하게 누르는 압박
- `obdurate`: 마음이 단단히 굳음

## 비교 단어

### tol

- 뜻: 견디다, 참다
- 구분 포인트: [[tol]]은 감정적·사회적으로 참아 받아들임. `dur`는 단단히 버티고 오래 지속됨.

### pand

- 뜻: 고통을 겪다
- 구분 포인트: [[pand]]의 `patience`, `passion`은 겪고 감내하는 쪽. `dur`는 단단함과 지속성.
