# ram

- 출발 단어: `ramify`
- 핵심 어원: Latin `ramus` = branch
- 핵심 이미지: 나뭇가지처럼 갈라져 퍼짐
- 관련 문서: [[riv]], [[sect]]
- 주의: `ram` "숫양, 들이받다"와는 별도 어원. 여기서는 `ramus` 가지 계열만 다룸.

## 출발 단어 분석

`ramify` = Latin `ramus` + `-ify`

- `ramus`: 가지
- `-ify`: ~하게 만들다, ~가 되다
- 원래 이미지: 가지를 내며 갈라짐
- 현대 의미: 분기하다, 갈라져 퍼지다, 파생되다

## 단어들

### ramify

- 발음: /ˈræməˌfaɪ/
- 구성: Latin `ramus` = branch + `-ify`
- 뜻: 분기하다, 갈라져 퍼지다
- 어원적 연결: 나무가 가지를 내듯 여러 방향으로 나뉨
- 사용 맥락: `the issue may ramify into other areas` = 문제가 다른 영역으로 번질 수 있음

### ramification

- 발음: /ˌræməfəˈkeɪʃən/
- 구성: `ramify` + `-ication`
- 뜻: 파급 효과, 복잡한 결과; 가지처럼 갈라짐
- 어원적 연결: 한 원인에서 여러 결과가 가지처럼 뻗어 나감
- 사용 맥락: `political ramifications` = 정치적 파급 효과

### ramified

- 구성: `ramify` + `-ed`
- 뜻: 가지처럼 갈라진, 복잡하게 분기된
- 어원적 연결: 여러 갈래로 뻗어 나간 상태
- 사용 맥락: `a ramified structure` = 가지처럼 갈라진 구조

### ramose

- 구성: Latin `ramosus` = full of branches
- 뜻: 가지가 많은, 분지한
- 어원적 연결: 실제 가지가 많거나 가지 모양으로 갈라짐
- 사용 맥락: 과학·식물학 맥락에서 주로 사용. 일반 시험 어휘에서는 드묾.

## 기억 포인트

`ram` 계열은 "가지처럼 갈라짐" 이미지.

- `ramify`: 가지처럼 분기하다
- `ramification`: 가지처럼 뻗은 결과, 파급 효과
- `ramified`: 갈라진, 분기된
- `ramose`: 가지가 많은

## 비교 단어

### branch

- 뜻: 가지, 지점, 분기하다
- 구분 포인트: 기본 영어 단어. `ramify`와 `ramification`은 라틴계 문어·학술 표현으로, 결과가 여러 방향으로 복잡하게 퍼지는 느낌이 강함.

### river

- 뜻: 강
- 구분 포인트: [[riv]]는 물길이 갈라지거나 흘러나오는 이미지. `ram`은 나뭇가지처럼 구조가 갈라지는 이미지.
