# loqu

- 출발 단어: `eloquent`
- 핵심 어원: Latin `loqui` = 말하다
- 핵심 이미지: 말을 어떻게 흘려 내보내고 주고받는가
- 관련 문서: [[dict]], [[verb]], [[voc]]
- 주의: `locate`의 `loc` = place 계열과 구분.

## 출발 단어 분석

`eloquent` ← Latin `eloqui`

- `e-` / `ex-`: out
- `loqui`: speak
- 원래 이미지: 말을 밖으로 잘 풀어냄
- 현대 의미: 유창한, 설득력 있는

## 단어들

### eloquent

- 구성: `e-` = out + Latin `loqui` = speak
- 뜻: 유창한, 설득력 있는
- 어원적 연결: 말을 잘 풀어 밖으로 냄
- 사용 맥락: `an eloquent speech` = 유창한 연설

### eloquence

- 구성: `eloquent` + `-ence`
- 뜻: 웅변, 유창함
- 어원적 연결: 말이 힘 있게 잘 흘러나오는 능력
- 사용 맥락: `speak with eloquence` = 유창하게 말하다

### loquacious

- 구성: Latin `loquax` = talkative
- 뜻: 말이 많은, 수다스러운
- 어원적 연결: 말을 많이 하는 성향
- 사용 맥락: `a loquacious host` = 말이 많은 진행자

### colloquial

- 구성: `col-` = together + Latin `loqui`
- 뜻: 구어체의, 일상 회화의
- 어원적 연결: 사람들이 함께 말할 때 쓰는 표현
- 사용 맥락: `colloquial English` = 구어체 영어

### colloquium

- 구성: `col-` = together + `loqui`
- 뜻: 학술 토론회, 콜로키움
- 어원적 연결: 사람들이 함께 말하고 토론함
- 사용 맥락: `a research colloquium` = 연구 콜로키움

### soliloquy

- 구성: Latin `solus` = alone + `loqui`
- 뜻: 독백
- 어원적 연결: 혼자 말함
- 사용 맥락: `Hamlet's soliloquy` = 햄릿의 독백

### circumlocution

- 구성: `circum-` = around + Latin `loqui`
- 뜻: 완곡한 표현, 에둘러 말하기
- 어원적 연결: 직접 말하지 않고 둘러서 말함
- 사용 맥락: `avoid circumlocution` = 에둘러 말하기를 피하다

### interlocutor

- 구성: `inter-` = between + Latin `loqui`
- 뜻: 대화 상대
- 어원적 연결: 서로 사이에서 말을 주고받는 사람
- 사용 맥락: `a skilled interlocutor` = 능숙한 대화 상대

## 기억 포인트

`loqu` 계열은 "말하다" 이미지.

- `eloquent`: 말을 잘 풀어냄
- `loquacious`: 말이 많음
- `colloquial`: 함께 말하는 일상 구어체
- `soliloquy`: 혼자 말함
- `circumlocution`: 둘러 말함

- 구분 포인트: [[verb]]는 말로 된 단어. `loqu`는 말하는 행위.
