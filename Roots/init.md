# init

- 출발 단어: `initial`
- 핵심 어원: Latin `inire` = go into, enter upon, begin; `initium` = beginning
- 핵심 이미지: 안으로 들어가 시작함, 첫 단계
- 관련 문서: [[in]], [[via]], [[ven]]
- 주의: `init`는 Latin `ire` = go의 `in-` 결합형. `itinerary`, `transit`, `exit` 같은 넓은 `ire` 계열과 연결되지만, 이 문서는 시작 계열 중심.

## 출발 단어 분석

`initial` = Latin `initialis` ← `initium` ← `inire`

- `in-`: into
- `ire`: go
- `inire`: 들어가다, 착수하다, 시작하다
- `initium`: 시작, 입구, 첫 단계
- 현대 의미: 처음의, 머리글자

## 단어들

### initial

- 구성: Latin `initialis` ← `initium`
- 뜻: 처음의, 초기의; 머리글자
- 어원적 연결: 어떤 일로 들어가는 첫 지점
- 사용 맥락: `initial response` = 초기 반응

### initially

- 구성: `initial` + `-ly`
- 뜻: 처음에는, 초기에
- 어원적 연결: 시작 지점에서
- 사용 맥락: `initially unclear` = 처음에는 불분명한

### initiate

- 구성: Latin `initiare` ← `initium`
- 뜻: 시작하다, 개시하다; 입문시키다
- 어원적 연결: 어떤 과정 안으로 들어가게 함
- 사용 맥락: `initiate a project` = 프로젝트를 시작하다

### initiation

- 구성: `initiate` + `-ion`
- 뜻: 시작, 개시; 입문
- 어원적 연결: 새로운 단계로 들어가는 의식이나 과정
- 사용 맥락: `initiation ceremony` = 입문식

### initiative

- 구성: French `initiative` ← Latin `initium`
- 뜻: 주도권, 계획, 진취성
- 어원적 연결: 먼저 시작하는 힘
- 사용 맥락: `take the initiative` = 주도권을 잡다

### initiator

- 구성: `initiate` + `-or`
- 뜻: 시작한 사람, 발의자
- 어원적 연결: 처음으로 일을 안으로 밀어 넣는 사람
- 사용 맥락: `the initiator of reform` = 개혁의 발의자

### initiatory

- 구성: `initiate` + `-ory`
- 뜻: 입문의, 시작의
- 어원적 연결: 새로운 단계에 들어가게 하는 성격
- 사용 맥락: 문어적 표현

## 기억 포인트

`init` 계열은 "안으로 들어가 시작함" 이미지.

- `initial`: 시작 지점의
- `initially`: 처음에는
- `initiate`: 시작하게 함
- `initiation`: 새로운 단계로 들어감
- `initiative`: 먼저 시작하는 힘
- `initiator`: 시작한 사람

## 비교 단어

### ire

- 뜻: 가다
- 연결 포인트: Latin `ire` = go. `initial`은 `in-` + `ire`에서 온 "안으로 들어가 시작함" 계열.

### itinerary

- 뜻: 여행 일정, 경로
- 구분 포인트: `itinerary`는 Latin `iter/itineris` = journey 계열로, `ire`와 연결되지만 "시작"보다 "길·여정" 이미지.

### exit

- 뜻: 나가다, 출구
- 구분 포인트: `exit`는 `ex-` + `ire` = 밖으로 가다. `initial`은 안으로 들어가 시작함.
