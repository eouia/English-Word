# repudi

- 출발 단어: `repudiate`
- 핵심 어원: Latin `repudiare` = reject, divorce, put away
- 핵심 이미지: 관계·주장·책임을 공식적으로 끊고 거부함
- 관련 문서: [[refut]]
- 주의: Latin `repudium`의 더 깊은 기원은 불확실함. 현대 형태를 `re-` + 확실한 어근으로 무리하게 분해하지 않음.

## 출발 단어 분석

`repudiate` ← Latin `repudiare`

- `repudiare`: 거절하다, 물리치다, 이혼하다
- `repudium`: 거절, 이혼, 관계 단절
- 원래 이미지: 관계나 의무를 끊어 버림
- 현대 의미: 부인하다, 거부하다, 절연하다

## 단어들

### repudiate

- 발음: /riˈpjudiˌeɪt/
- 구성: Latin `repudiare`
- 뜻: 부인하다, 거부하다, 절연하다
- 어원적 연결: 주장·책임·관계가 자기 것임을 인정하지 않고 끊어 냄
- 사용 맥락: `repudiate the accusation` = 혐의를 부인하다, `repudiate a treaty` = 조약을 파기하다

### repudiation

- 발음: /rɪˌpjudiˈeɪʃən/
- 구성: `repudiate` + `-ion`
- 뜻: 부인, 거부, 파기, 절연
- 어원적 연결: 어떤 주장이나 관계를 공식적으로 받아들이지 않는 행위
- 사용 맥락: `a repudiation of violence` = 폭력에 대한 거부

### repudiated

- 발음: /riˈpjudiˌeɪtɪd/
- 구성: `repudiate` + `-ed`
- 뜻: 부인된, 거부된, 파기된
- 어원적 연결: 더 이상 인정되거나 받아들여지지 않는 상태
- 사용 맥락: `a repudiated claim` = 부인된 주장

### repudiable

- 구성: `repudiate` + `-able`
- 뜻: 부인할 수 있는, 거부할 수 있는
- 어원적 연결: 받아들이지 않고 끊어 낼 수 있음
- 사용 맥락: 드문 단어. 보통 `can be repudiated`처럼 풀어 씀.

## 기억 포인트

`repudi` 계열은 "공식적으로 끊고 거부함" 이미지.

- `repudiate`: 부인하다, 거부하다
- `repudiation`: 부인, 거부, 파기
- `repudiated`: 부인된, 파기된
- `repudiable`: 부인 가능한

## 비교 단어

### refute

- 뜻: 논박하다
- 구분 포인트: [[refut]]는 근거로 틀렸음을 입증함. `repudiate`는 받아들이거나 인정하기를 공식적으로 거부함.

### renounce

- 뜻: 포기하다, 단념하다, 공식적으로 버리다
- 구분 포인트: `renounce`는 말로 선언해 포기함이 핵심. `repudiate`는 관계·책임·주장을 끊어 거부하는 결이 강함.

### reject

- 뜻: 거절하다, 거부하다
- 구분 포인트: `reject`는 넓은 일반어. `repudiate`는 더 공식적이고 강하며, 주장·책임·조약·신념을 부인할 때 자주 씀.
