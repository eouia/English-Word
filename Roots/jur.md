# jur

- 출발 단어: `jury`
- 핵심 어원: Latin `ius` / `iuris` = 법, 권리
- 동사형: Latin `iurare` = 맹세하다
- 핵심 이미지: 법적 권리와 맹세로 묶인 판단
- 관련 문서: [[court]], [[jud]], [[leg]], [[dict]], [[rect]], [[just]]
- 주의: [[jud]]는 판단·판결 행위. `jur`는 법·권리·맹세의 기반.

## 출발 단어 분석

`jury` ← Anglo-French `juree` ← Latin `iurare`

- `ius/iuris`: 법, 권리
- `iurare`: 맹세하다
- 원래 이미지: 법 앞에서 맹세한 사람들의 집단
- 현대 의미: 배심원단

## 단어들

### jury

- 구성: Anglo-French `juree`
- 뜻: 배심원단
- 어원적 연결: 맹세하고 사실 판단을 맡은 사람들
- 사용 맥락: `a jury verdict` = 배심원 평결

### juror

- 구성: `jury` 계열 + `-or`
- 뜻: 배심원
- 어원적 연결: 맹세하고 재판 판단에 참여하는 사람
- 사용 맥락: `serve as a juror` = 배심원으로 참여하다

### jurisdiction

- 구성: `juris` = law + [[dict]] = speak
- 뜻: 관할권, 사법권
- 어원적 연결: 법을 말하고 적용할 수 있는 권한과 범위
- 사용 맥락: `within the court's jurisdiction` = 법원의 관할 안에서

### jurisprudence

- 구성: `juris` = law + Latin `prudentia` = knowledge
- 뜻: 법학, 법철학
- 어원적 연결: 법에 대한 지식과 이론
- 사용 맥락: `constitutional jurisprudence` = 헌법 법리

### jurist

- 구성: Latin `iurista`
- 뜻: 법학자, 법률가
- 어원적 연결: 법을 전문적으로 아는 사람
- 사용 맥락: `a respected jurist` = 존경받는 법학자

### juridical

- 구성: Latin `iuridicus` = legal
- 뜻: 법률상의, 사법의
- 어원적 연결: 법에 속하거나 법으로 말해지는 것
- 사용 맥락: `juridical status` = 법적 지위

### justice

- 구성: Latin `iustitia` ← `ius`
- 뜻: 정의, 사법, 공정
- 어원적 연결: 법과 권리에 맞는 상태
- 사용 맥락: `social justice` = 사회 정의

### just

- 구성: Latin `iustus`
- 뜻: 정의로운, 정당한, 딱 맞는
- 어원적 연결: 법과 권리에 맞음
- 사용 맥락: `a just decision` = 정당한 결정
- 관련 문서: [[just]]

### justify

- 구성: `just` + `-fy`
- 뜻: 정당화하다
- 어원적 연결: 법과 정의에 맞는 것으로 보이게 함
- 사용 맥락: `justify the cost` = 비용을 정당화하다

### injustice

- 구성: `in-` = not + `justice`
- 뜻: 부정의, 불공정
- 어원적 연결: 법과 권리에 맞지 않음
- 사용 맥락: `fight injustice` = 부정의에 맞서다

### injure

- 구성: Latin `iniuria` = injustice, wrong
- 뜻: 다치게 하다, 해치다
- 어원적 연결: 법적 권리를 침해함 → 해를 끼침
- 사용 맥락: `injure a player` = 선수를 다치게 하다

### injury

- 구성: Latin `iniuria`
- 뜻: 부상, 손해
- 어원적 연결: 권리 침해나 해로움
- 사용 맥락: `a serious injury` = 심각한 부상

## 기억 포인트

`jur` 계열은 "법과 권리, 맹세" 이미지.

- `jury`: 맹세한 판단 집단
- `jurisdiction`: 법을 말할 권한
- `jurist`: 법 전문가
- `justice`: 법과 권리에 맞음
- `injury`: 권리 침해에서 해로움으로 확장

### jud

- 뜻: 판단하다, 판결하다
- 구분 포인트: [[jud]]는 `judge`, `judgment`, `prejudice`처럼 판단 행위 중심.
