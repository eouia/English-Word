# cover

- 출발 단어: `discover`
- 핵심 어원: 라틴어 `operire` = 덮다, 가리다
- 핵심 이미지: 덮어 감춤 / 덮개를 걷어 드러냄
- 관련해서 헷갈릴 수 있는 어근: [[clud]] = 닫다, 막다 계열, [[vel]] = 덮개, 베일 계열

## 출발 단어 분석

`discover`는 `dis-` + `cover`.

- `dis-`: apart, away
- `cover`: 덮다, 가리다

따라서 `discover`는 원래 "덮인 것을 걷어내다"의 뜻. 여기서 "숨겨진 것을 드러내다" → "발견하다"의 뜻.

## 단어들

### discover

- 구성: `dis-` = away + `cover`
- 뜻: 발견하다, 알아내다
- 어원적 연결: 덮개를 걷어 숨은 것을 드러냄
- 사용 맥락: `discover a new method` = 새로운 방법을 발견하다

### discovery

- 구성: `discover` + `-y`
- 뜻: 발견, 발견물
- 어원적 연결: 가려져 있던 것이 드러난 결과
- 사용 맥락: `a scientific discovery` = 과학적 발견

### cover

- 구성: Old French `covrir` < Latin `cooperire`
- 뜻: 덮다, 가리다; 다루다, 보도하다
- 어원적 연결: 위에서 덮어 보이지 않게 함
- 사용 맥락: `cover the table` = 탁자를 덮다, `cover the story` = 그 사건을 보도하다

### coverage

- 구성: `cover` + `-age`
- 뜻: 보도, 적용 범위, 보장 범위
- 어원적 연결: 어떤 영역을 덮거나 포함하는 범위
- 사용 맥락: `media coverage` = 언론 보도, `insurance coverage` = 보험 보장 범위

### uncover

- 구성: `un-` = reverse + `cover`
- 뜻: 덮개를 벗기다, 밝혀내다
- 어원적 연결: 덮인 것을 벗겨 드러냄
- 사용 맥락: `uncover evidence` = 증거를 밝혀내다

### undercover

- 구성: `under` + `cover`
- 뜻: 비밀의, 위장한
- 어원적 연결: 덮개 아래 숨어 정체를 감춤
- 사용 맥락: `an undercover agent` = 비밀 요원

### covert

- 구성: Old French `covert` = covered
- 뜻: 은밀한, 숨겨진
- 어원적 연결: 덮여서 겉으로 드러나지 않음
- 사용 맥락: `covert operations` = 비밀 작전

### covertly

- 구성: `covert` + `-ly`
- 뜻: 은밀히, 몰래
- 어원적 연결: 덮인 상태처럼 겉으로 드러나지 않게
- 사용 맥락: `covertly monitor activity` = 활동을 은밀히 감시하다

### overt

- 구성: Old French `overt` = open, uncovered
- 뜻: 명백한, 공개적인
- 어원적 연결: 덮이지 않고 열려 있음
- 사용 맥락: `overt discrimination` = 노골적인 차별

### overtly

- 구성: `overt` + `-ly`
- 뜻: 명백히, 공개적으로, 노골적으로
- 어원적 연결: 덮이지 않고 드러난 방식으로
- 사용 맥락: `overtly hostile behavior` = 노골적으로 적대적인 행동

## 기억 포인트

`cover` 계열은 "덮다"에서 출발.

- `cover`: 덮다
- `discover`: 덮개를 걷어냄 → 발견하다
- `uncover`: 덮인 것을 벗김 → 밝혀내다
- `covert`: 덮여 있음 → 은밀한
- `covertly`: 은밀히
- `overt`: 덮이지 않음 → 명백한
- `overtly`: 명백히, 노골적으로
- `coverage`: 덮는 범위 → 보도/보장 범위

## 비교 단어

### recover

- 뜻: 회복하다, 되찾다
- 구분 포인트: 일반적인 `recover` = regain, get back은 라틴어 `recuperare` 계열. `cover` = 덮다 계열과 본문 단어로 섞지 않음. 다만 `re-cover` = 다시 덮다라는 별도 뜻은 현재 형태 기준으로 가능

### cloak

- 뜻: 망토, 감추다
- 구분 포인트: 의미는 "덮어 감춤"으로 가깝지만 `cover`와 어원 계열은 다름
