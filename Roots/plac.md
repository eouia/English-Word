# plac

- 출발 단어: `please`
- 핵심 어원: Latin `placere` = 기쁘게 하다, 만족시키다
- 관련 형태: `pleas-`, `plac-`
- 핵심 이미지: 마음을 편하게 하거나 만족시킴
- 관련 문서: [[path]]

## 출발 단어 분석

`please` ← Old French `plaisir` ← Latin `placere`

- `placere`: please
- 원래 이미지: 만족스럽고 기분 좋음
- 현대 의미: 기쁘게 하다, 제발

## 단어들

### please

- 구성: Latin `placere`
- 뜻: 기쁘게 하다, 제발
- 어원적 연결: 상대의 마음을 만족시킴
- 사용 맥락: `please the audience` = 관객을 기쁘게 하다

### pleasant

- 구성: Old French `plaisant`
- 뜻: 즐거운, 쾌적한
- 어원적 연결: 마음에 들어 기분 좋음
- 사용 맥락: `a pleasant surprise` = 기분 좋은 놀라움

### pleasure

- 구성: Old French `plaisir`
- 뜻: 즐거움, 기쁨
- 어원적 연결: 마음이 만족한 상태
- 사용 맥락: `take pleasure in reading` = 독서에서 즐거움을 얻다

### complacent

- 구성: `com-` = very, together + Latin `placere`
- 뜻: 자기만족적인, 안일한
- 어원적 연결: 자기 상태에 지나치게 만족함
- 사용 맥락: `become complacent` = 안일해지다

### placate

- 구성: Latin `placare` = calm, soothe
- 뜻: 달래다, 진정시키다
- 어원적 연결: 불만을 누그러뜨려 마음을 편하게 함
- 사용 맥락: `placate critics` = 비판자들을 달래다

### placebo

- 구성: Latin `placebo` = I shall please
- 뜻: 위약
- 어원적 연결: 실제 약효보다 만족감·기대 효과를 줌
- 사용 맥락: `placebo effect` = 위약 효과

## 기억 포인트

`plac/pleas` 계열은 "기쁘게 하다, 만족시키다" 이미지.

- `please`: 기쁘게 하다
- `pleasant`: 기분 좋은
- `pleasure`: 즐거움
- `complacent`: 지나치게 만족한
- `placate`: 달래다
- `placebo`: 만족감을 주는 위약
