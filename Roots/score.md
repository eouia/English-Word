# score

- 출발 단어: `underscore`
- 핵심 어원: Old Norse `skor` = notch, tally, mark cut in wood
- 핵심 이미지: 긁거나 그어 표시를 남김
- 관련 문서: [[mark]]
- 주의: `underscore`는 코딩의 `_` 기호와 "강조하다" 뜻을 모두 가짐.

## 출발 단어 분석

`underscore` = `under` + `score`

- `under`: 아래에
- `score`: 선을 긋다, 표시하다
- 원래 이미지: 글자 아래에 선을 그음
- 현대 의미: 밑줄을 긋다; 강조하다; 밑줄 기호 `_`

`score`는 원래 나무 등에 칼로 자국을 내어 수를 세던 표시와 연결. 여기서 점수, 악보, 선을 긋는 행위로 의미가 넓어짐.

## 단어들

### score

- 구성: Old Norse `skor` = notch, tally
- 뜻: 점수, 득점하다; 악보; 선을 긋다
- 어원적 연결: 자국을 새겨 수나 정보를 표시함
- 사용 맥락: `score a goal` = 골을 넣다, `a musical score` = 악보

### underscore

- 구성: `under` + `score`
- 뜻: 밑줄을 긋다; 강조하다; 밑줄 기호 `_`
- 어원적 연결: 아래에 선을 그어 눈에 띄게 함
- 사용 맥락: `underscore the importance` = 중요성을 강조하다

### underscored

- 구성: `underscore` + `-ed`
- 뜻: 밑줄 친; 강조된
- 어원적 연결: 아래에 선이 그어져 두드러진 상태
- 사용 맥락: `the report underscored the risk` = 보고서가 위험을 강조했다

### underscoring

- 구성: `underscore` + `-ing`
- 뜻: 밑줄 긋기; 강조
- 어원적 연결: 선을 그어 두드러지게 만드는 행위
- 사용 맥락: `an underscoring of the problem` = 문제의 강조

### scoreboard

- 구성: `score` + `board`
- 뜻: 점수판
- 어원적 연결: 점수를 표시하는 판
- 사용 맥락: `check the scoreboard` = 점수판을 확인하다

### scorecard

- 구성: `score` + `card`
- 뜻: 점수표, 평가표
- 어원적 연결: 점수나 평가를 적어 표시한 카드
- 사용 맥락: `a performance scorecard` = 성과 평가표

### scoring

- 구성: `score` + `-ing`
- 뜻: 득점; 채점; 악보 작성
- 어원적 연결: 점수나 음악 정보를 표시하는 과정
- 사용 맥락: `scoring criteria` = 채점 기준

## 기억 포인트

`score` 계열은 "선을 그어 표시함" 이미지.

- `score`: 점수, 악보, 표시하다
- `underscore`: 아래에 선을 그어 강조함
- `scoreboard`: 점수판
- `scorecard`: 점수표, 평가표
- `scoring`: 득점, 채점

## 비교 단어

### mark

- 뜻: 표시하다, 자국, 표식
- 구분 포인트: [[mark]]는 알아보게 남긴 표시 전체. `score`는 긁거나 선을 그어 남긴 표시, 그리고 그 표시로 센 점수에 초점.

### emphasize

- 뜻: 강조하다
- 구분 포인트: `emphasize`는 말이나 글에서 중요성을 부각함. `underscore`는 원래 밑줄을 긋는 물리적 이미지에서 비유적 강조로 확장.

### underline

- 뜻: 밑줄을 긋다, 강조하다
- 구분 포인트: `underline`은 일상적 표현. `underscore`는 글쓰기·공식 문맥에서 "강조하다" 뜻으로 자주 쓰고, `_` 기호 이름이기도 함.
