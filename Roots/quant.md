# quant

- 출발 단어: `quantity`
- 핵심 어원: Latin `quantus` = how much, how great
- 핵심 이미지: 얼마나 많은가, 어느 정도의 양인가
- 관련 문서: [[quote]], [[quad]], [[qual]]
- 주의: `quantity`는 양과 수량. `quality`는 성질과 특성이라 별도 계열.

## 출발 단어 분석

`quantity` ← Latin `quantitas` ← `quantus`

- `quantus`: 얼마나 큰, 얼마나 많은
- `-ity`: 상태나 성질을 나타내는 접미사
- 원래 이미지: 어느 정도의 크기나 양
- 현대 의미: 양, 수량, 분량

## 단어들

### quantity

- 발음: /ˈkwɑntəti/
- 구성: Latin `quantitas` ← `quantus`
- 뜻: 양, 수량, 분량
- 어원적 연결: 얼마나 많은지 나타내는 값이나 규모
- 사용 맥락: `a large quantity of data` = 많은 양의 데이터

### quantitative

- 발음: /ˈkwɑntɪˌteɪtɪv/
- 구성: `quantity` + `-ative`
- 뜻: 양적인, 수량에 관한
- 어원적 연결: 성질보다 수치와 양으로 따짐
- 사용 맥락: `quantitative analysis` = 정량 분석

### quantify

- 발음: /ˈkwɑntɪˌfaɪ/
- 구성: `quant` + `-ify`
- 뜻: 수량화하다, 양으로 나타내다
- 어원적 연결: 추상적인 것을 숫자나 양으로 표시함
- 사용 맥락: `quantify the risk` = 위험을 수량화하다

### quantifiable

- 발음: /ˌkwɑntɪˈfaɪəbɛl/
- 구성: `quantify` + `-able`
- 뜻: 수량화할 수 있는
- 어원적 연결: 숫자나 양으로 측정 가능함
- 사용 맥락: `quantifiable results` = 수량화 가능한 결과

### quantum

- 발음: /ˈkwɑntəm/
- 구성: Latin `quantum` = how much, amount
- 뜻: 양, 몫; 양자
- 어원적 연결: 물리학에서는 더 이상 나눌 수 없는 에너지의 최소량 이미지
- 사용 맥락: `quantum physics` = 양자 물리학

### quota

- 발음: /ˈkwoʊtə/
- 구성: Medieval Latin `quota` ← Latin `quotus` / `quantus` 계열
- 뜻: 할당량, 몫
- 어원적 연결: 정해진 양이나 비율만큼 나누어 맡김
- 사용 맥락: `meet a sales quota` = 판매 할당량을 채우다

## 기억 포인트

`quant` 계열은 "얼마나 많은가" 이미지.

- `quantity`: 양, 수량
- `quantitative`: 양으로 따지는
- `quantify`: 수량화하다
- `quantifiable`: 수량화 가능한
- `quantum`: 정해진 양, 최소 단위
- `quota`: 할당된 양

## 비교 단어

### quality

- 뜻: 질, 성질, 특성
- 구분 포인트: [[qual]]은 "어떤 종류인가"에 초점. `quantity`는 "얼마나 많은가"에 초점.

### quote

- 뜻: 인용하다, 가격을 부르다
- 구분 포인트: [[quote]]는 "표시하다, 순서나 수를 정하다" 이미지. `quota`처럼 양을 나누는 단어에서 일부 겹쳐 보일 수 있으나, `quantity` 학습은 `quantus` 중심으로 보는 편이 좋음.
