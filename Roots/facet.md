# facet

- 출발 단어: `multifaceted`
- 핵심 의미: 얼굴, 겉모습, 표면, 한 면
- 관련 형태: Latin `facies` = face, appearance, form
- 관련 문서: [[multi]], [[super]]

## 핵심 이미지

`facet` 계열의 바탕은 "겉으로 보이는 면".

사람의 앞면은 `face`.
얼굴과 관련되면 `facial`.
보석이나 사물의 한 면은 `facet`.
여러 면을 가지면 `multifaceted`.
겉면은 `surface`.
서로 맞닿는 면은 `interface`.

## 단어들

### multifaceted

- 구성: `multi-` = many + `facet` = small face, side
- 뜻: 다면적인, 여러 측면을 가진
- 연결: 보석처럼 여러 면을 가진 상태에서, 문제나 사람이 여러 측면을 지님으로 확장
- 사용 맥락: `a multifaceted problem` = 다면적인 문제

### facet

- 구성: French `facette`, from Latin `facies` = face
- 뜻: 한 면, 측면
- 연결: 보석이나 사물의 작은 얼굴 같은 면
- 사용 맥락: `one facet of the issue` = 그 문제의 한 측면

### faceted

- 구성: `facet` + `-ed`
- 뜻: 면이 있는, 깎인 면을 가진
- 연결: 여러 평평한 면으로 나뉘어 있음
- 사용 맥락: `a faceted gemstone` = 깎인 면이 있는 보석

### face

- 구성: Latin `facies` = face, appearance
- 뜻: 얼굴, 표면; 마주하다
- 연결: 겉으로 보이는 앞면
- 사용 맥락: `face a challenge` = 도전에 직면하다

### facial

- 구성: `face` + `-ial`
- 뜻: 얼굴의
- 연결: 얼굴과 관련됨
- 사용 맥락: `facial expression` = 얼굴 표정

### surface

- 구성: `sur-` = over + `face`
- 뜻: 표면, 겉면; 드러나다
- 연결: 위쪽에 드러난 얼굴 같은 면
- 사용 맥락: `on the surface` = 표면적으로는

### interface

- 구성: `inter-` = between + `face`
- 뜻: 접점, 인터페이스
- 연결: 두 시스템이나 물체가 서로 마주 보는 면
- 사용 맥락: `user interface` = 사용자 인터페이스

### superficial

- 구성: Latin `superficies` = surface, from `super-` + `facies`
- 뜻: 표면적인, 피상적인
- 연결: 깊이 들어가지 않고 겉면에만 머무름
- 사용 맥락: `a superficial understanding` = 피상적인 이해

### efface

- 구성: `ex-` = out/away + `face`
- 뜻: 지우다, 없애다
- 연결: 표면의 얼굴 같은 표시를 지워 없앰
- 사용 맥락: `efface the marks` = 흔적을 지우다

## 비교 단어

`fac` = 만들다, 하다 계열은 [[fac]] 문서.
`facet/face/facial`의 `facies` = 얼굴, 겉모습과 철자가 비슷하므로 구분.

`multi-`는 "많은"을 뜻하는 접두사.
아직 독립 문서가 없다면 나중에 `multiple`, `multiply`, `multitude`와 함께 따로 만들 가치가 있음.
