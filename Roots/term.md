# term

- 출발 단어: `term`
- 핵심 어원: Latin `terminus` = 끝, 경계, 한계
- 핵심 이미지: 경계를 그어 끝을 정함
- 관련 문서: [[fin]], [[reg]]
- 주의: [[fin]]도 끝·경계 계열이지만 Latin `finis`. `term`은 `terminus`의 경계표지 이미지가 강함.

## 출발 단어 분석

`term` ← Old French `terme` ← Latin `terminus`

- `terminus`: 경계표, 끝, 한계
- 원래 이미지: 여기까지라고 표시한 끝점
- 현대 의미: 기간, 용어, 조건, 학기

## 단어들

### term

- 발음: /ˈtɝm/
- 구성: Latin `terminus`
- 뜻: 기간, 용어, 조건, 학기
- 어원적 연결: 의미나 기간의 경계를 정한 것
- 사용 맥락: `legal terms` = 법적 조건

### terminal

- 발음: /ˈtɝmənəl/
- 구성: `term` + `-al`
- 뜻: 종착역, 말기의, 단말기의
- 어원적 연결: 끝점에 있는 것
- 사용 맥락: `airport terminal` = 공항 터미널

### terminate

- 발음: /ˈtɝməˌneɪt/
- 구성: `terminus` + `-ate`
- 뜻: 끝내다, 종료하다
- 어원적 연결: 경계를 찍어 끝에 이르게 함
- 사용 맥락: `terminate a contract` = 계약을 종료하다

### termination

- 발음: /tɚməˈneɪʃən/
- 구성: `terminate` + `-ion`
- 뜻: 종료, 종결
- 어원적 연결: 끝을 정하는 행위
- 사용 맥락: `employment termination` = 고용 종료

### determine

- 발음: /dəˈtɝmən/
- 구성: `de-` = completely + `term` = boundary
- 뜻: 결정하다, 알아내다, 규정하다
- 어원적 연결: 경계를 분명히 정해 결론을 냄
- 사용 맥락: `determine the cause` = 원인을 알아내다

### determination

- 발음: /dɪˌtɝməˈneɪʃən/
- 구성: `determine` + `-ation`
- 뜻: 결정, 판단, 결심
- 어원적 연결: 마음이나 판단의 경계를 확정함
- 사용 맥락: `strong determination` = 강한 결심

### determined

- 발음: /dɪˈtɝmənd/
- 구성: `determine` + `-ed`
- 뜻: 결정된; 단호한
- 어원적 연결: 경계나 마음이 확정되어 흔들리지 않음
- 사용 맥락: `a determined effort` = 단호한 노력

### predetermine

- 발음: /ˌpridɪˈtɝmən/
- 구성: `pre-` = before + `determine`
- 뜻: 미리 결정하다, 예정하다
- 어원적 연결: 일이 일어나기 전에 경계나 결과를 정해 둠
- 사용 맥락: `predetermine the outcome` = 결과를 미리 정하다

### predetermined

- 발음: /ˌpridiˈtɝmɪnd/
- 구성: `predetermine` + `-ed`
- 뜻: 미리 정해진, 예정된
- 어원적 연결: 이미 경계나 결과가 정해져 있는 상태
- 사용 맥락: `a predetermined schedule` = 미리 정해진 일정

### predetermination

- 구성: `pre-` + `determination`
- 뜻: 사전 결정, 예정
- 어원적 연결: 판단이나 결과가 미리 확정됨
- 사용 맥락: `genetic predetermination` = 유전적 예정

### determinate

- 발음: /dɪˈtɝməˌneɪt/
- 구성: `de-` + `terminate`
- 뜻: 확정된, 한정된
- 어원적 연결: 끝과 경계가 정해져 있음
- 사용 맥락: `a determinate sentence` = 확정형

### indeterminate

- 발음: /ˌɪndɪˈtɝmɪnɪt/
- 구성: `in-` = not + `determinate`
- 뜻: 불확정의, 애매한
- 어원적 연결: 경계나 끝이 정해지지 않음
- 사용 맥락: `an indeterminate result` = 불확정 결과

### terminology

- 발음: /ˌtɝmɪˈnɑlədʒi/
- 구성: `term` + Greek `-logy`
- 뜻: 전문 용어 체계
- 어원적 연결: 의미 경계가 정해진 말들의 체계
- 사용 맥락: `medical terminology` = 의학 용어

### interminable

- 발음: /ˌɪˈntɝmənəbəl/
- 구성: `in-` = not + `terminable`
- 뜻: 끝없는 듯한, 지루하게 긴
- 어원적 연결: 끝이 보이지 않음
- 사용 맥락: `an interminable meeting` = 끝없이 긴 회의

### exterminate

- 발음: /ɪˈkstɝməˌneɪt/
- 구성: `ex-` = out + `terminus`
- 뜻: 몰살하다, 박멸하다
- 어원적 연결: 경계 밖으로 몰아내어 없앰
- 사용 맥락: `exterminate pests` = 해충을 박멸하다

## 기억 포인트

`term` 계열은 "끝점과 경계" 이미지.

- `term`: 기간·의미·조건의 경계
- `terminal`: 끝점
- `terminate`: 끝내다
- `determine`: 경계를 정해 판단하다
- `predetermined`: 미리 경계나 결과가 정해진
- `interminable`: 끝이 없는 듯함

## 비교 단어

### fin

- 뜻: 끝, 경계, 한계
- 구분 포인트: [[fin]]은 `final`, `finite`, `define`, `confine`처럼 끝·한계를 정하는 일반 이미지. `term`은 경계표지나 기간의 끝점 느낌이 강함.
