# over

- 출발 단어: `overcome`
- 핵심 어원: Old English `ofer` = above, beyond, across
- 핵심 이미지: 위로, 건너서, 한계를 넘어서, 지나치게
- 관련 문서: [[come]], [[ag]], [[out]], [[ex]], [[ultra]]

## 단어들

### overcome

- 구성: `over-` = across, beyond + [[come]] = come
- 뜻: 극복하다, 이겨내다; 압도되다
- 어원적 연결: 어려움이나 반대를 넘어 지나감
- 사용 맥락: `overcome difficulties` = 어려움을 극복하다

### overcame

- 구성: `overcome`의 과거형
- 뜻: 극복했다, 이겨냈다
- 어원적 연결: 이미 장애를 넘어선 상태
- 사용 맥락: `she overcame fear` = 그녀는 두려움을 극복했다

### overtake

- 구성: `over-` + `take`
- 뜻: 따라잡다, 앞지르다; 갑자기 닥치다
- 어원적 연결: 뒤에서 와서 상대를 넘어 잡음
- 사용 맥락: `overtake a car` = 차를 추월하다

### overthrow

- 구성: `over-` + `throw`
- 뜻: 전복하다, 타도하다
- 어원적 연결: 위에서 뒤집어 던져 권력이나 체제를 무너뜨림
- 사용 맥락: `overthrow a regime` = 정권을 전복하다

### overwhelm

- 구성: `over-` + Old English `whelmen` = overturn, cover
- 뜻: 압도하다, 감당하지 못하게 하다
- 어원적 연결: 위에서 덮어 눌러 버림
- 사용 맥락: `feel overwhelmed` = 압도당한 느낌을 받다

### overshadow

- 구성: `over-` + `shadow`
- 뜻: 그늘지게 하다; 빛을 잃게 하다, 가리다
- 어원적 연결: 위에서 그림자를 드리워 덮음. 여기서 다른 것보다 더 두드러져 상대를 가림
- 사용 맥락: `one scandal overshadowed the campaign` = 한 스캔들이 선거 운동을 가려 버렸다

### overshadowed

- 구성: `overshadow` + `-ed`
- 뜻: 가려진, 빛을 잃은
- 어원적 연결: 더 큰 존재나 사건의 그림자 아래 들어간 상태
- 사용 맥락: `an overshadowed achievement` = 가려진 성취

### overshadowing

- 구성: `overshadow` + `-ing`
- 뜻: 가리고 있는, 압도적으로 두드러지는
- 어원적 연결: 현재 그림자를 드리우듯 다른 것을 덮어 가림
- 사용 맥락: `an overshadowing concern` = 다른 문제들을 압도하는 걱정거리

### overlook

- 구성: `over-` + `look`
- 뜻: 내려다보다; 못 보고 넘어가다; 눈감아 주다
- 어원적 연결: 위에서 바라봄 / 시선이 위로 지나가 세부를 놓침 / 일부러 문제 삼지 않고 넘어감
- 사용 맥락: `overlook a mistake` = 실수를 못 보고 넘어가다, 눈감아 주다

### overlooked

- 구성: `overlook` + `-ed`
- 뜻: 간과된, 무시된, 눈에 띄지 않은
- 어원적 연결: 시선이나 관심이 지나쳐 버린 상태
- 사용 맥락: `an overlooked detail` = 간과된 세부 사항

### overlooking

- 구성: `overlook` + `-ing`
- 뜻: 내려다보는; 간과하는
- 어원적 연결: 위에서 바라보거나, 주의가 세부를 지나쳐 감
- 사용 맥락: `a room overlooking the sea` = 바다가 내려다보이는 방

### oversee

- 구성: `over-` + `see`
- 뜻: 감독하다, 살피다
- 어원적 연결: 위에서 전체를 내려다보며 봄
- 사용 맥락: `oversee a project` = 프로젝트를 감독하다

### overlap

- 구성: `over-` + `lap`
- 뜻: 겹치다, 중복되다
- 어원적 연결: 한 부분이 다른 부분 위로 덮임
- 사용 맥락: `overlapping schedules` = 겹치는 일정

### overestimate

- 구성: `over-` + [[estim]] = value, estimate
- 뜻: 과대평가하다
- 어원적 연결: 실제보다 지나치게 높게 평가함
- 사용 맥락: `overestimate the cost` = 비용을 과대평가하다

### overstate

- 구성: `over-` + `state`
- 뜻: 과장해서 말하다
- 어원적 연결: 실제보다 지나치게 크게 말함
- 사용 맥락: `overstate the benefits` = 장점을 과장하다

### overact

- 구성: `over-` + [[ag]]의 `act`
- 뜻: 과장되게 연기하다, 지나치게 행동하다
- 어원적 연결: 행동이나 연기가 적정선을 넘어 과해짐
- 사용 맥락: `overact in a scene` = 한 장면에서 과장되게 연기하다

### overacting

- 구성: `overact` + `-ing`
- 뜻: 과장된 연기
- 어원적 연결: 감정 표현이나 몸짓이 지나치게 큼
- 사용 맥락: `the film suffers from overacting` = 그 영화는 과장된 연기가 문제다

### overacted

- 구성: `overact` + `-ed`
- 뜻: 과장되게 연기한, 과장된
- 어원적 연결: 행동이나 연기가 이미 적정선을 넘은 상태
- 사용 맥락: `an overacted performance` = 과장된 연기

### overtime

- 구성: `over-` + `time`
- 뜻: 초과 근무; 연장전
- 어원적 연결: 정해진 시간을 넘어섬
- 사용 맥락: `work overtime` = 초과 근무하다

## 기억 포인트

`over-`는 네 갈래로 기억.

- 위에서: `overlook`, `oversee`
- 넘어서: `overcome`, `overtake`
- 뒤집거나 덮어 압도: `overthrow`, `overwhelm`
- 지나치게: `overestimate`, `overstate`, `overtime`

## 비교 단어

### out

- 뜻: 밖으로, 상대를 넘어섬
- 구분 포인트: [[out]]은 경쟁에서 "더 앞섬" 느낌이 강함. `over-`는 위, 건너감, 한계 초과 이미지가 함께 있음.

### ex

- 뜻: 밖으로
- 구분 포인트: [[ex]]는 Latin 계열 접두사. `over-`는 영어 기본어 계열이라 일상 합성어에 많이 붙음.

### ultra

- 뜻: 넘어서, 극단적으로
- 구분 포인트: [[ultra]]는 Latin 계열로 "범위 밖, 극단" 느낌이 강함. `over-`는 더 일상적인 "위/넘음/과도함".
