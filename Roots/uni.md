# uni

- 출발 단어: `unity`
- 핵심 어원: 라틴어 `unus` = 하나
- 핵심 이미지: 여러 것이 하나로 묶이거나, 하나뿐임
- 관련해서 헷갈릴 수 있는 어근: [[anim]], [[mono]], [[singul]], [[solus]], `un-` = not 부정 접두사와 구분 필요

## 출발 단어 분석

`unity`는 `unit` + `-y`.

- `unit`: 하나의 단위
- `-y`: 상태, 성질

따라서 `unity`는 "하나인 상태" → "통일, 단결, 일체감"의 뜻.

## 단어들

### unity

- 구성: `unit` + `-y`
- 뜻: 통일, 단결, 일체감, 통일성
- 어원적 연결: 여러 부분이 하나의 전체처럼 맞물린 상태
- 사용 맥락: `national unity` = 국민적 단결, `unity of design` = 디자인의 통일성

### unite

- 구성: Latin `unire`
- 뜻: 결합하다, 통합하다
- 어원적 연결: 여러 대상을 하나로 만듦
- 사용 맥락: `unite the team` = 팀을 하나로 단결시키다

### united

- 구성: `unite` + `-ed`
- 뜻: 연합한, 단결한
- 어원적 연결: 하나로 묶인 상태
- 사용 맥락: `the United Nations` = 국제 연합

### union

- 구성: Latin `unio`
- 뜻: 연합, 조합, 결합
- 어원적 연결: 여러 사람이 하나로 묶인 조직이나 상태
- 사용 맥락: `labor union` = 노동조합

### unit

- 구성: Latin `unus`
- 뜻: 단위, 한 개
- 어원적 연결: 하나로 셀 수 있는 기본 덩어리
- 사용 맥락: `a basic unit` = 기본 단위

### unique

- 구성: Latin `unicus`
- 뜻: 유일한, 독특한
- 어원적 연결: 하나뿐인 성질
- 사용 맥락: `a unique opportunity` = 유일한 기회

### uniform

- 구성: `uni-` = one + [[form]]
- 뜻: 똑같은, 제복
- 어원적 연결: 하나의 형태를 가짐
- 사용 맥락: `uniform standards` = 통일된 기준

### unify

- 구성: `uni-` + `-fy`
- 뜻: 통일하다, 하나로 만들다
- 어원적 연결: 여러 부분을 하나가 되게 함
- 사용 맥락: `unify the system` = 시스템을 통합하다

### unified

- 구성: `unify` + `-ed`
- 뜻: 통일된, 일원화된
- 어원적 연결: 하나의 체계로 묶인 상태
- 사용 맥락: `a unified approach` = 통일된 접근법

### universe

- 구성: Latin `universum`
- 뜻: 우주, 세계, 전체
- 어원적 연결: 전통적으로 `uni-` = one + `vers-` = turn. 하나로 돌아 묶인 전체
- 사용 맥락: `the observable universe` = 관측 가능한 우주

### universal

- 구성: `universe` + `-al`
- 뜻: 보편적인, 전 세계적인
- 어원적 연결: 전체에 하나같이 적용됨
- 사용 맥락: `universal human rights` = 보편적 인권

### university

- 구성: Latin `universitas`
- 뜻: 대학교
- 어원적 연결: 교사와 학생들이 하나의 공동체로 묶인 조직
- 사용 맥락: `attend university` = 대학에 다니다

### unanimous

- 구성: `un-` = one + `animus` = mind
- 뜻: 만장일치의
- 어원적 연결: 마음이 하나로 모임
- 사용 맥락: `a unanimous decision` = 만장일치 결정
- 관련 문서: [[anim]]

### unicorn

- 구성: `uni-` = one + `corn` = horn
- 뜻: 유니콘
- 어원적 연결: 뿔이 하나인 존재
- 사용 맥락: `a unicorn startup` = 유니콘 스타트업

## 기억 포인트

`uni` 계열은 "하나" 이미지.

- `unity`: 하나인 상태
- `unite`: 하나로 묶다
- `union`: 하나로 묶인 조직
- `unit`: 하나의 단위
- `unique`: 하나뿐인
- `uniform`: 하나의 형태
- `unify`: 하나로 만들다
- `universe`: 하나로 묶인 전체
- `unanimous`: 마음이 하나인

## 비교 단어

### un-

- 뜻: not
- 구분 포인트: `unhappy`, `unclear`의 `un-`은 부정 접두사. `unity`, `unique`, `universe`의 `uni-`는 "하나" 계열
