# tol

- 출발 단어: `tolerance`
- 핵심 어원: 라틴어 `tolerare` = 견디다, 참다, 떠받치다
- 핵심 이미지: 무게나 불편함을 견디고 받아들임
- 관련해서 헷갈릴 수 있는 어근: `toll` = 요금, 종소리 계열과 구분 필요

## 출발 단어 분석

`tolerance`는 `tolerate` + `-ance`.

- `tolerate`: 참고 견디다, 받아들이다
- `-ance`: 상태, 성질, 정도

따라서 `tolerance`는 "견디고 받아들이는 성질이나 정도" → "관용, 허용 오차, 내성"의 뜻.

## 단어들

### tolerate

- 구성: Latin `tolerare`
- 뜻: 참다, 견디다, 용인하다
- 어원적 연결: 불편하거나 마음에 들지 않는 것을 감당함
- 사용 맥락: `tolerate criticism` = 비판을 견디다

### tolerance

- 구성: `tolerate` + `-ance`
- 뜻: 관용, 참을성, 허용 오차, 내성
- 어원적 연결: 어느 정도까지 견디거나 받아들일 수 있는 범위
- 사용 맥락: `religious tolerance` = 종교적 관용, `drug tolerance` = 약물 내성

### tolerant

- 구성: `tolerate` + `-ant`
- 뜻: 관대한, 잘 참는
- 어원적 연결: 다른 생각이나 불편함을 받아들일 수 있음
- 사용 맥락: `a tolerant society` = 관용적인 사회

### intolerant

- 구성: `in-` = not + `tolerant`
- 뜻: 편협한, 참지 못하는
- 어원적 연결: 차이나 불편함을 받아들이지 못함
- 사용 맥락: `intolerant of criticism` = 비판을 참지 못하는

### tolerable

- 구성: `tolerate` + `-able`
- 뜻: 견딜 만한, 참을 수 있는
- 어원적 연결: 완전히 좋지는 않지만 감당 가능함
- 사용 맥락: `a tolerable level of pain` = 견딜 만한 통증 수준

### intolerable

- 구성: `in-` = not + `tolerable`
- 뜻: 참을 수 없는
- 어원적 연결: 감당할 수 있는 범위를 넘어섬
- 사용 맥락: `intolerable noise` = 참을 수 없는 소음

### toleration

- 구성: `tolerate` + `-ion`
- 뜻: 묵인, 용인, 관용
- 어원적 연결: 어떤 차이나 행동을 막지 않고 받아들임
- 사용 맥락: `religious toleration` = 종교적 관용

## 기억 포인트

`tol` 계열은 "무게나 불편함을 견딤" 이미지.

- `tolerate`: 참고 견딤
- `tolerance`: 견디고 받아들이는 정도
- `tolerant`: 잘 받아들이는
- `intolerant`: 받아들이지 못하는
- `tolerable`: 견딜 만한
- `intolerable`: 견딜 수 없는

## 비교 단어

### extol

- 구성: Latin `ex-` = out, up + `tollere` = 들어 올리다
- 뜻: 극찬하다, 칭송하다
- 어원적 연결: 누군가를 높이 들어 올려 칭찬함
- 구분 포인트: `tolerate`는 "견디다", `extol`은 "들어 올리다". 라틴어 계열상 닿는 부분이 있지만 현대 학습에서는 의미 갈래를 분리해 기억하는 편이 좋음

### toll

- 뜻: 통행료, 희생자 수, 종을 울리다
- 구분 포인트: `toll`은 `tolerance`의 `tol`과 직접 묶어 외우지 않는 편이 좋음
