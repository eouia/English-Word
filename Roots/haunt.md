# haunt

- 출발 단어: `haunt`
- 핵심 어원: Old French `hanter` = 자주 방문하다, 드나들다
- 핵심 이미지: 어떤 장소나 기억에 반복해서 찾아옴
- 비교: [[sed]]의 `obsess` = 생각이 마음을 포위함

## 출발 단어 분석

`haunt`는 원래 "자주 가다, 드나들다"의 뜻.

- Old French `hanter`: 자주 방문하다, 익숙하게 드나들다
- 가능성 있는 더 먼 뿌리: Germanic `home` 계열

따라서 `haunt`는 "자주 찾아옴" → "귀신이 나타남" → "기억이나 감정이 계속 따라다님"의 뜻.

## 단어들

### haunt

- 구성: Old French `hanter`
- 뜻: 자주 나타나다, 괴롭히다, 머릿속에 남다
- 어원적 연결: 어떤 장소나 기억을 반복해서 찾아옴
- 사용 맥락: `memories haunt him` = 기억이 그를 괴롭히다

### haunted

- 구성: `haunt` + `-ed`
- 뜻: 귀신 들린, 괴로운 기억에 시달리는
- 어원적 연결: 무언가가 반복해서 찾아오는 상태
- 사용 맥락: `a haunted house` = 유령이 나오는 집

### haunting

- 구성: `haunt` + `-ing`
- 뜻: 잊히지 않는, 마음에 남는
- 어원적 연결: 감정이나 장면이 계속 되돌아옴
- 사용 맥락: `a haunting melody` = 잊히지 않는 선율

### haunt

- 구성: 명사형 `haunt`
- 뜻: 자주 가는 장소
- 어원적 연결: 반복해서 방문하는 곳
- 사용 맥락: `an old haunt` = 예전에 자주 가던 곳

## 비교 단어

### obsess

- 구성: `ob-` + `sedere`
- 뜻: 사로잡다, 집착하게 하다
- 어원적 연결: [[sed]] 계열. 생각이 마음 앞에 앉아 포위함
- 구분 포인트: `haunt` = 반복해서 찾아옴, `obsess` = 마음을 포위해 사로잡음

### hunt

- 구성: Old English `huntian`
- 뜻: 사냥하다, 찾다
- 어원적 연결: `haunt`와 어원 다름. `hentan` = 붙잡다, seize 계열과 관련 가능
- 구분 포인트: `hunt` = 쫓아가 잡음, `haunt` = 반복해서 찾아옴

### hunter

- 구성: `hunt` + `-er`
- 뜻: 사냥꾼, 찾는 사람
- 어원적 연결: 잡기 위해 추적하는 사람
- 구분 포인트: `hunter`는 대상을 쫓음. [[seize]]는 붙잡아 장악함. `haunting memory`는 기억이 나를 찾아옴

## 기억 포인트

`haunt` 계열은 "반복해서 찾아옴" 이미지.

- `haunt`: 계속 찾아와 괴롭힘
- `haunted`: 반복적으로 시달리는 상태
- `haunting`: 잊히지 않고 되돌아오는 느낌
- `haunt`: 자주 가던 장소

비교:

- `hunt`: 내가 대상을 쫓음
- `haunt`: 대상이나 기억이 나를 찾아옴
