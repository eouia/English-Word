# delic

- 출발 단어: `delicate`
- 핵심 어원: Latin `deliciae` = delight, pleasure; `delicatus` = alluring, delightful, dainty
- 핵심 이미지: 기분 좋게 끌리는 섬세함, 조심히 다뤄야 할 고급스러움
- 관련 문서: [[lux]]
- 주의: `license`, `illicit`의 Latin `licere` = be allowed 계열과 다름. `delicate`는 허락이 아니라 즐거움·섬세함 계열.

## 출발 단어 분석

`delicate` ← Latin `delicatus` = alluring, delightful, dainty

- `delic-`: delight, pleasure, daintiness
- `-ate`: 형용사 형태
- 원래 이미지: 즐거움을 주는, 고급스럽고 섬세한
- 의미 변화: 즐겁게 끄는 → 맛있고 고급스러운 → 섬세한, 연약한 → 조심스러운
- 현대 의미: 섬세한, 연약한, 미묘한, 조심스러운

## 단어들

### delicate

- 발음: /ˈdɛləkət/
- 구성: Latin `delicatus` = delightful, dainty
- 뜻: 섬세한, 연약한, 미묘한; 조심스러운
- 어원적 연결: 기분 좋게 끌리지만 쉽게 상할 만큼 섬세함
- 사용 맥락: `a delicate issue` = 조심스러운 문제, `delicate skin` = 민감한 피부

### delicacy

- 발음: /ˈdɛləkəsi/
- 구성: `delicate` + `-cy`
- 뜻: 섬세함, 민감함; 진미
- 어원적 연결: 섬세한 성질 또는 특별히 즐거움을 주는 음식
- 사용 맥락: `handle the matter with delicacy` = 그 일을 섬세하게 다루다

### delicious

- 발음: /dɪˈlɪʃəs/
- 구성: Latin `deliciosus` = delightful, pleasing
- 뜻: 맛있는, 매우 즐거운
- 어원적 연결: 감각에 즐거움을 주는 맛
- 사용 맥락: `a delicious meal` = 맛있는 식사

### deliciously

- 발음: /dɪˈlɪʃəʃli/
- 구성: `delicious` + `-ly`
- 뜻: 맛있게, 기분 좋게
- 어원적 연결: 즐거움을 주는 방식
- 사용 맥락: `deliciously sweet` = 기분 좋게 달콤한

### delicatessen

- 발음: /ˌdɛlɪkəˈtɛsən/
- 구성: German `Delikatessen` ← French/Latin `delicate` 계열
- 뜻: 고급 식료품점, 조리 식품점; 조리 식품
- 어원적 연결: 섬세하고 맛있는 음식, 진미를 파는 곳
- 사용 맥락: `a neighborhood delicatessen` = 동네 델리 식품점

### delectable

- 발음: /dɪˈlɛktəbəl/
- 구성: Latin `delectare` = delight, charm + `-able`
- 뜻: 맛있는, 즐거운
- 어원적 연결: 감각이나 마음을 즐겁게 하는 성질
- 사용 맥락: `a delectable dessert` = 맛있는 디저트

### delectation

- 구성: Latin `delectatio` = delight, enjoyment
- 뜻: 즐거움, 기쁨
- 어원적 연결: 마음이 끌려 즐거워하는 상태
- 사용 맥락: `for your delectation` = 당신의 즐거움을 위해

## 기억 포인트

`delic` 계열은 "즐거움을 주는 섬세함" 이미지.

- `delicate`: 섬세하고 조심스러운
- `delicacy`: 섬세함 또는 진미
- `delicious`: 감각에 즐거움을 주는 맛
- `delicatessen`: 맛있는 조리 식품을 다루는 곳
- `delectable`: 즐겁게 맛있는

## 비교 단어

### licere

- 뜻: 허락되다, 합법이다
- 구분 포인트: `license`, `illicit`, `licit`은 허락·합법 계열. `delicate`, `delicious`는 즐거움·섬세함 계열.

### fragile

- 뜻: 깨지기 쉬운
- 구분 포인트: `fragile`은 부서짐에 초점. `delicate`는 물리적 약함뿐 아니라 미묘함, 세심함, 고급스러운 섬세함까지 포함.

### subtle

- 뜻: 미묘한, 포착하기 어려운
- 구분 포인트: [[subtil]]은 알아차리기 어려운 미세함. `delicate`는 다루기 조심스러운 섬세함.
