# fabr

- 출발 단어: `fabricate`
- 핵심 어원: Latin `faber` = craftsman, maker; `fabrica` = workshop, structure
- 핵심 이미지: 장인이 재료를 짜 맞추어 만들어 냄
- 관련 문서: [[fac]], [[fibr]]
- 주의: `fabric`은 `fiber`와 철자가 비슷하지만 [[fibr]] 계열이 아니라 `faber/fabrica` 계열.

## 출발 단어 분석

`fabricate` = `fabric` + `-ate`

- `fabric`: 만든 구조물, 짜인 직물
- `-ate`: 만들다, 그렇게 하다
- 원래 이미지: 재료를 짜 맞추어 제작함
- 현대 의미: 만들다, 조립하다; 꾸며내다, 날조하다

## 단어들

### fabricate

- 발음: /ˈfæbrəˌkeɪt/
- 구성: Latin `fabricare` = make, construct
- 뜻: 만들다, 제작하다; 꾸며내다, 날조하다
- 어원적 연결: 실제 물건을 제작하듯 이야기도 인위적으로 만들어 냄
- 사용 맥락: `fabricate evidence` = 증거를 날조하다, `fabricate parts` = 부품을 제작하다

### fabrication

- 발음: /ˌfæbrɪˈkeɪʃən/
- 구성: `fabricate` + `-ion`
- 뜻: 제작, 조립; 날조, 꾸며낸 이야기
- 어원적 연결: 만들어진 결과물. 사실이 아니라 인위적으로 짜 맞춘 말도 포함.
- 사용 맥락: `a complete fabrication` = 완전한 날조

### fabricated

- 발음: /ˈfæbrɪˌkeɪtəd/
- 구성: `fabricate` + `-ed`
- 뜻: 제작된; 조작된, 날조된
- 어원적 연결: 자연스럽게 생긴 것이 아니라 인위적으로 만들어진 상태
- 사용 맥락: `fabricated claims` = 날조된 주장

### fabric

- 발음: /ˈfæbrɪk/
- 구성: Latin `fabrica` = workshop, structure
- 뜻: 직물; 구조, 조직
- 어원적 연결: 실이나 재료를 짜 맞추어 만든 것
- 사용 맥락: `cotton fabric` = 면직물, `the fabric of society` = 사회의 구조

### prefab

- 발음: /ˈpriˈfæb/
- 구성: `pre-` = beforehand + `fabricate`
- 뜻: 조립식 건물; 미리 제작된
- 어원적 연결: 현장에서 만들기 전에 미리 제작함
- 사용 맥락: `a prefab house` = 조립식 주택

### refabricate

- 구성: `re-` = again + `fabricate`
- 뜻: 다시 제작하다, 재가공하다
- 어원적 연결: 이미 만든 것을 다시 짜 맞추어 제작함
- 사용 맥락: 기술·제조 문맥에서 드문 표현

## 기억 포인트

`fabr`는 "짜 맞추어 제작함" 이미지.

- `fabricate`: 제작하다; 사실처럼 꾸며내다
- `fabrication`: 제작물; 날조
- `fabricated`: 인위적으로 만들어진, 조작된
- `fabric`: 짜 맞추어 만든 직물이나 구조
- `prefab`: 미리 제작된 것

## 비교 단어

### fiber

- 뜻: 섬유, 섬유질
- 구분 포인트: [[fibr]]의 Latin `fibra` 계열. `fabric`은 섬유로 만든 결과물일 수 있지만 어원은 `faber/fabrica`.

### forge

- 뜻: 만들다, 위조하다
- 구분 포인트: 금속을 두드려 만드는 이미지에서 "위조하다"로 확장. `fabricate`도 "날조하다"가 있지만, 핵심 이미지는 짜 맞추어 제작함.

### fac

- 뜻: 만들다, 하다
- 구분 포인트: [[fac]]는 일반적인 "만들다/do" 계열. `fabr`는 장인의 제작·구조물 이미지가 더 뚜렷함.
