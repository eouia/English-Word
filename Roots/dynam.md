# dynam

- 출발 단어: `dynamic`
- 핵심 어원: Greek `dynamis` = 힘, 능력, power
- 핵심 이미지: 안에서 움직이게 하는 힘과 능력
- 관련 문서: [[val]], [[crat]]
- 주의: `dynasty`는 지배자의 힘·권력 계열로 의미가 정치 쪽으로 확장됨.

## 출발 단어 분석

`dynamic` ← Greek `dynamikos`

- `dynamis`: power, force, ability
- `dynamikos`: powerful
- 원래 이미지: 힘이 있어 작용하고 움직임
- 현대 의미: 역동적인, 동적인

## 단어들

### dynamic

- 구성: Greek `dynamis` + `-ic`
- 뜻: 역동적인, 동적인
- 어원적 연결: 힘이 작용해 움직임과 변화를 만듦
- 사용 맥락: `a dynamic market` = 역동적인 시장

### dynamics

- 구성: `dynamic` + `-s`
- 뜻: 역학, 동역학, 관계의 작동 방식
- 어원적 연결: 힘이 어떻게 작용하는지 다룸
- 사용 맥락: `group dynamics` = 집단 역학

### dynamism

- 구성: `dynamis` + `-ism`
- 뜻: 활력, 역동성
- 어원적 연결: 내부의 힘이 활발하게 움직이는 성질
- 사용 맥락: `economic dynamism` = 경제적 역동성

### dynamite

- 구성: Greek `dynamis` + `-ite`
- 뜻: 다이너마이트, 폭발적인 것
- 어원적 연결: 강한 힘을 내는 폭발물
- 사용 맥락: `a stick of dynamite` = 다이너마이트 한 막대

### dynamo

- 구성: Greek `dynamis`
- 뜻: 발전기, 활력이 넘치는 사람
- 어원적 연결: 힘을 만들어 내거나 계속 움직이는 원천
- 사용 맥락: `a human dynamo` = 에너지가 넘치는 사람

### dynamometer

- 구성: `dynamo-` = power + [[metr]] = measure
- 뜻: 동력계, 힘 측정기
- 어원적 연결: 힘이나 동력을 재는 장치
- 사용 맥락: 공학 맥락에서 힘·토크 측정 장치를 가리킴

### dynasty

- 구성: Greek `dynastes` = ruler, lord
- 뜻: 왕조, 권력 가문
- 어원적 연결: 힘과 권력을 가진 지배자 집안
- 사용 맥락: `the Ming dynasty` = 명나라 왕조

### dynastic

- 구성: `dynasty` + `-ic`
- 뜻: 왕조의, 세습 권력의
- 어원적 연결: 지배 가문의 권력과 관련됨
- 사용 맥락: `dynastic rule` = 왕조 지배

## 기억 포인트

`dynam` 계열은 "힘과 능력" 이미지.

- `dynamic`: 힘이 있어 움직임
- `dynamics`: 힘의 작용 방식
- `dynamism`: 활력과 역동성
- `dynamite`: 강한 폭발력
- `dynamo`: 힘을 만들어 내는 원천
- `dynasty`: 권력을 가진 집안

## 비교 단어

### val

- 뜻: 힘이 있다, 가치가 있다
- 구분 포인트: [[val]]은 힘과 가치·유효성. `dynam`은 움직임을 만드는 힘과 능력.

### crat

- 뜻: 권력, 지배
- 구분 포인트: [[crat]]은 지배 체제와 권력 구조. `dynam`은 더 넓은 힘과 능력의 이미지.
