# sult

- 출발 단어: `desultory`
- 핵심 어원: Latin `salire` / `saltus` / `sult-` = leap, jump
- 핵심 이미지: 뛰어오르거나 여기저기 뛰어넘음
- 관련 문서: [[gress]], [[cad]], [[jac]]
- 주의: `salt` = 소금은 별도 어원. `consult`는 [[consul]] 계열. `salient`, `desultory`, `assault`, `result`의 `sult/sal`은 "뛰다" 계열.

## 출발 단어 분석

`desultory` ← Latin `desultor` = circus rider who jumps from one horse to another

- `de-`: down, away, from
- `sult`: leap, jump
- `-ory`: ~하는 성질의
- 원래 이미지: 말에서 말로 뛰어 옮겨 다님
- 의미 변화: 여기저기 뛰어다님 → 한곳에 머물지 않음 → 산만한, 두서없는
- 현대 의미: 산만한, 두서없는, 일관성 없는

## 단어들

### desultory

- 발음: /ˈdɛsəˌltɔri/
- 구성: Latin `desultor` = one who jumps down/from one thing to another + `-ory`
- 뜻: 산만한, 두서없는, 일관성 없는
- 어원적 연결: 말에서 말로 뛰어 옮기듯 주제나 행동이 이리저리 건너뜀
- 사용 맥락: `a desultory conversation` = 두서없는 대화

### salient

- 발음: /ˈseɪliənt/
- 구성: Latin `salire` = leap
- 뜻: 두드러진, 눈에 띄는
- 어원적 연결: 밖으로 뛰어나온 것처럼 눈에 확 띔
- 사용 맥락: `the salient point` = 핵심적으로 눈에 띄는 점

### assail

- 발음: /əˈseɪl/
- 구성: Latin `ad-` = toward + `salire` = leap
- 뜻: 공격하다, 몰아치다
- 어원적 연결: 상대에게 뛰어들어 공격함
- 사용 맥락: `be assailed by doubts` = 의심에 시달리다

### assault

- 발음: /əˈsɔlt/
- 구성: Old French `asaut` ← Latin `ad-` + `saltus` = leap
- 뜻: 공격, 폭행; 공격하다
- 어원적 연결: 상대에게 뛰어드는 공격
- 사용 맥락: `an assault on the city` = 도시 공격

### insult

- 발음: n. /ˈɪˌnsʌlt/, v. /ˌɪˈnsʌlt/
- 구성: Latin `in-` = on/upon + `salire` = leap
- 뜻: 모욕하다; 모욕
- 어원적 연결: 남 위로 뛰어올라 짓밟듯 무례하게 대함
- 사용 맥락: `a personal insult` = 개인적 모욕

### exult

- 발음: /ɪˈgzʌlt/
- 구성: Latin `ex-` = out/up + `salire` = leap
- 뜻: 크게 기뻐하다, 환희하다
- 어원적 연결: 기뻐서 뛰어오르는 모습
- 사용 맥락: `exult in victory` = 승리에 환호하다

### result

- 발음: /rɪˈzʌlt/
- 구성: Latin `re-` = back + `salire` = leap
- 뜻: 결과; 결과로 생기다
- 어원적 연결: 어떤 행동 뒤에 다시 튀어나와 나타난 것
- 사용 맥락: `the result of the experiment` = 실험 결과

### resilient

- 발음: /rɪˈzɪljənt/
- 구성: Latin `resilire` ← `re-` = back + `salire` = leap
- 뜻: 회복력 있는, 탄력 있는
- 어원적 연결: 눌리거나 충격을 받아도 다시 튀어 오름
- 사용 맥락: `a resilient community` = 회복력이 강한 공동체

### resilience

- 발음: /rɪˈzɪliəns/
- 구성: `resilient` 계열 + `-ence`
- 뜻: 회복력, 탄력성
- 어원적 연결: 어려움 뒤에 다시 튀어 올라 회복하는 힘
- 사용 맥락: `emotional resilience` = 정서적 회복력

### resiliency

- 발음: /rɪˈzɪljənsi/
- 구성: `resilient` 계열 + `-cy`
- 뜻: 회복력, 탄력성
- 어원적 연결: `resilience`와 거의 같은 뜻. 특히 기술·조직 맥락에서 자주 사용
- 사용 맥락: `system resiliency` = 시스템 회복 탄력성

### somersault

- 발음: /ˈsʌmɚˌsɔlt/
- 구성: Old French `sombresaut`, from `sobre` = over + `saut` = leap
- 뜻: 공중제비
- 어원적 연결: 몸을 넘겨 뛰는 동작
- 사용 맥락: `do a somersault` = 공중제비를 하다

### sally

- 발음: /ˈsæli/
- 구성: Latin `salire` = leap
- 뜻: 돌격, 출격; 재치 있는 말
- 어원적 연결: 밖으로 뛰쳐나가는 움직임. 말에서는 갑자기 튀어나온 재치
- 사용 맥락: `a witty sally` = 재치 있는 한마디

## 기억 포인트

`sult/sal` 계열은 "뛰다" 이미지.

- `desultory`: 이리저리 뛰어넘어 두서없음
- `salient`: 밖으로 뛰어나와 눈에 띔
- `assault`: 상대에게 뛰어드는 공격
- `insult`: 위로 뛰어올라 짓밟듯 모욕함
- `exult`: 기뻐서 뛰어오름
- `result`: 뒤에 튀어나온 결과
- `resilient`: 다시 튀어 올라 회복함

## 비교 단어

### gress

- 뜻: 걷다, 나아가다
- 구분 포인트: [[gress]]는 걸어서 진행함. `sult`는 뛰어오르거나 건너뜀.

### cad

- 뜻: 떨어지다
- 구분 포인트: [[cad]]는 아래로 떨어짐. `sult`는 뛰어오르는 동작과 그 결과.

### jac

- 뜻: 던지다
- 구분 포인트: [[jac]]는 무언가를 밖으로 던짐. `sult`는 몸이나 상태가 튀어 오르거나 건너뜀.

### consult

- 뜻: 상담하다, 조언을 구하다, 참고하다
- 구분 포인트: [[consul]]의 Latin `consulere/consultare` = deliberate, take counsel 계열. 철자에 `sult`가 보이지만 `salire` = leap 계열로 보지 않음.
