# rud

- 출발 단어: `rudimentary`
- 핵심 어원: Latin `rudis` = rough, untrained, raw
- 핵심 이미지: 아직 다듬어지지 않은 거친 초보 상태
- 관련 문서: [[schol]], [[art]]
- 주의: `rude`, `rudimentary`, `erudite`가 모두 `rudis` 계열로 연결됨. "무례함"보다 "다듬어지지 않음"이 출발점.

## 출발 단어 분석

`rudimentary` = `rudiment` + `-ary`

- `rudis`: 거친, 훈련되지 않은, 날것의
- `rudiment`: 기초, 초보 단계
- `-ary`: ~에 관한
- 원래 이미지: 아직 세련되게 다듬어지지 않은 첫 단계
- 현대 의미: 기초적인, 초보적인, 미발달의

## 단어들

### rudimentary

- 구성: `rudiment` + `-ary`
- 뜻: 기초적인, 초보적인, 미발달의
- 어원적 연결: 아직 완전히 다듬어지지 않은 기본 단계
- 사용 맥락: `rudimentary knowledge` = 기초 지식

### rudiment

- 구성: Latin `rudimentum` ← `rudis`
- 뜻: 기초, 기본 원리; 미발달 기관
- 어원적 연결: 훈련과 발전이 시작되는 가장 기초 단계
- 사용 맥락: `the rudiments of grammar` = 문법의 기초

### rude

- 구성: Latin `rudis`
- 뜻: 무례한, 거친, 투박한
- 어원적 연결: 다듬어지지 않고 세련되지 않은 태도나 형태
- 사용 맥락: `a rude remark` = 무례한 말

### rudely

- 구성: `rude` + `-ly`
- 뜻: 무례하게, 거칠게
- 어원적 연결: 다듬어지지 않은 태도로 행동함
- 사용 맥락: `speak rudely` = 무례하게 말하다

### erudite

- 구성: Latin `eruditus` ← `e-` = out + `rudis`
- 뜻: 박식한, 학식 있는
- 어원적 연결: 거칠고 무지한 상태에서 벗어나 잘 교육됨
- 사용 맥락: `an erudite scholar` = 박식한 학자

### erudition

- 구성: Latin `eruditio` ← `erudire`
- 뜻: 박식함, 학식
- 어원적 연결: 무지하고 다듬어지지 않은 상태에서 벗어난 지식
- 사용 맥락: `great erudition` = 대단한 학식

## 기억 포인트

`rud` 계열은 "거칠고 아직 다듬어지지 않음" 이미지.

- `rudimentary`: 기초적인, 미발달의
- `rudiment`: 기초, 기본 원리
- `rude`: 다듬어지지 않은 → 무례한
- `erudite`: 거친 무지에서 벗어난 → 박식한
- `erudition`: 학식

## 비교 단어

### elementary

- 뜻: 기초적인
- 구분 포인트: `elementary`는 기본 요소·초급 단계에 초점. `rudimentary`는 아직 미숙하고 덜 발달한 느낌이 더 강할 수 있음.

### crude

- 뜻: 날것의, 조잡한, 거친
- 구분 포인트: `crude`도 덜 다듬어진 느낌이 있지만 어원은 다름. `rude`와 뜻은 닿지만 같은 어근으로 단정하지 않음.

### scholarly

- 뜻: 학문적인, 학식 있는
- 구분 포인트: [[schol]]은 학문·학교 계열. `erudite`는 "무지한 상태에서 벗어나 교육됨"의 이미지가 강함.
