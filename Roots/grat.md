# grat

- 출발 단어: `agreement`
- 핵심 어원: Latin `gratus` = pleasing, thankful, agreeable
- 핵심 이미지: 마음에 들어 서로 맞거나 고마워함
- 관련 문서: [[pact]], [[plac]]
- 주의: `grade`, `gradual`의 `grad` = step 계열과 다름. `agreement`는 "뜻이 서로 맞음" 이미지.

## 출발 단어 분석

`agreement` ← Old French `agreer` = receive favorably, please

- `a-/ad-`: to, toward
- `gratus`: pleasing, agreeable
- `-ment`: 명사 접미사
- 원래 이미지: 서로 마음에 들어 좋게 받아들임
- 의미 변화: 마음에 듦 → 의견이 맞음 → 합의, 동의
- 현대 의미: 합의, 동의, 계약

## 단어들

### agreement

- 구성: `agree` + `-ment`
- 뜻: 합의, 동의, 계약
- 어원적 연결: 서로 마음이나 뜻이 맞아 받아들인 상태
- 사용 맥락: `reach an agreement` = 합의에 이르다

### agrément

- 구성: French `agrément` ← `agréer` = approve, agree
- 뜻: 아그레망; 외교 사절 파견에 대한 접수국의 사전 동의
- 어원적 연결: 파견될 대표를 접수국이 받아들일 만하다고 승인함
- 사용 맥락: `receive agrément` = 아그레망을 받다
- 주의: 영어에서도 쓰이지만 외교·법률 문맥의 전문어. 일반 합의는 보통 `agreement`.

### persona non grata

- 구성: Latin `persona` = person + `non` = not + `grata` = welcome, pleasing
- 뜻: 환영받지 못하는 사람, 기피 인물; 외교상 기피 인물
- 어원적 연결: 그 사람이 접수국이나 집단에 "마음에 들지 않는/받아들일 수 없는" 사람으로 선언됨
- 사용 맥락: `declare a diplomat persona non grata` = 외교관을 기피 인물로 선언하다
- 주의: 복수형은 Latin식 `personae non gratae`도 가능. 일반 영어에서는 단수 표현을 고정구처럼 쓰는 경우가 많음.

### persona grata

- 구성: Latin `persona` = person + `grata` = welcome, pleasing
- 뜻: 환영받는 사람, 받아들여지는 외교 대표
- 어원적 연결: 받아들일 만한, 호의적으로 승인된 사람
- 사용 맥락: `a persona grata in diplomatic circles` = 외교계에서 환영받는 인물

### agree

- 구성: Old French `agreer` ← Latin `gratus` = pleasing
- 뜻: 동의하다, 의견이 맞다
- 어원적 연결: 어떤 의견이나 조건이 마음에 맞아 받아들여짐
- 사용 맥락: `agree with the proposal` = 제안에 동의하다

### agreeable

- 구성: `agree` + `-able`
- 뜻: 기분 좋은, 동의할 수 있는
- 어원적 연결: 마음에 들어 받아들일 만함
- 사용 맥락: `an agreeable solution` = 받아들일 만한 해결책

### disagreement

- 구성: `dis-` = apart/not + `agreement`
- 뜻: 불일치, 의견 차이, 다툼
- 어원적 연결: 서로 마음이나 뜻이 맞지 않음
- 사용 맥락: `a serious disagreement` = 심각한 의견 차이

### gratitude

- 구성: Latin `gratitudo` ← `gratus` = thankful
- 뜻: 감사, 고마움
- 어원적 연결: 받은 것이 마음에 들어 고마워하는 마음
- 사용 맥락: `express gratitude` = 감사를 표현하다

### grateful

- 구성: `grate`/`grat-` + `-ful`
- 뜻: 고마워하는, 감사하는
- 어원적 연결: 받은 호의가 기쁘고 마음에 들어 고마움
- 사용 맥락: `be grateful for help` = 도움에 감사하다

### gratify

- 구성: Latin `gratificari` = oblige, please
- 뜻: 만족시키다, 기쁘게 하다
- 어원적 연결: 상대가 기뻐하고 마음에 들게 함
- 사용 맥락: `gratify a desire` = 욕구를 만족시키다

### gratification

- 구성: `gratify` + `-ation`
- 뜻: 만족, 충족
- 어원적 연결: 마음에 들어 기쁜 상태
- 사용 맥락: `instant gratification` = 즉각적 만족

### congratulate

- 구성: Latin `con-` = together + `gratulari` = show joy
- 뜻: 축하하다
- 어원적 연결: 함께 기뻐하고 좋은 마음을 표현함
- 사용 맥락: `congratulate the winner` = 승자를 축하하다

### grace

- 구성: Latin `gratia` = favor, thanks, pleasing quality
- 뜻: 은혜, 우아함, 품위
- 어원적 연결: 마음에 드는 호의나 아름다운 성질
- 사용 맥락: `with grace` = 우아하게

### gracious

- 구성: `grace` + `-ous`
- 뜻: 품위 있는, 친절한, 너그러운
- 어원적 연결: 호의를 베풀고 마음 좋게 대하는 성질
- 사용 맥락: `a gracious host` = 친절하고 품위 있는 주인

### gratefulness

- 구성: `grateful` + `-ness`
- 뜻: 고마워함, 감사하는 마음
- 어원적 연결: 받은 호의를 좋게 여기고 감사하는 상태
- 사용 맥락: `a sense of gratefulness` = 감사하는 마음

## 기억 포인트

`grat` 계열은 "마음에 들어 좋음" 이미지.

- `agree`: 뜻이 맞아 받아들임
- `agreement`: 서로 뜻이 맞은 상태
- `agrément`: 외교 사절을 받아들이겠다는 공식 동의
- `persona non grata`: 받아들일 수 없는 사람
- `gratitude`: 받은 것이 마음에 들어 고마움
- `gratify`: 마음에 들게 해 만족시킴
- `congratulate`: 좋은 일에 함께 기뻐함
- `grace`: 호의와 우아함

## 비교 단어

### pact

- 뜻: 약속으로 묶은 합의
- 구분 포인트: [[pact]]는 공식적 약속·협정 느낌. `agreement`는 더 일반적인 동의·합의.

### plac

- 뜻: 기쁘게 하다, 달래다
- 구분 포인트: [[plac]]는 상대를 달래거나 만족시키는 행동. `grat`는 마음에 들어 고마워하거나 뜻이 맞는 상태.

### pac

- 뜻: 평화, 갈등이 멈춘 상태
- 구분 포인트: [[pact]] 문서의 `peace/pacify` 갈래는 갈등이 없는 평화. `agreement`는 의견이 맞아 합의된 상태.
