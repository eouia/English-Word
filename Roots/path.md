# path

- 출발 단어: `empathy`
- 핵심 어원: Greek `pathos` = 느낌, 고통, 겪음
- 핵심 이미지: 마음으로 겪는 감정과 고통
- 관련 문서: [[psych]], [[sent]], [[sym]]
- 주의: `path` = 길은 Greek `pathos`와 별도 어원.

## 출발 단어 분석

`empathy` ← Greek `em-` + `pathos`

- `em-` / `en-`: in
- `pathos`: feeling, suffering
- 원래 이미지: 다른 사람의 감정 안으로 들어감
- 현대 의미: 공감

## 단어들

### empathy

- 구성: `em-` = in + `pathos` = feeling
- 뜻: 공감
- 어원적 연결: 다른 사람의 감정 안으로 들어가 느낌
- 사용 맥락: `show empathy` = 공감을 보이다

### sympathetic

- 구성: [[sym]] = with + `pathos`
- 뜻: 동정적인, 공감하는
- 어원적 연결: 감정을 함께 겪음
- 사용 맥락: `a sympathetic listener` = 공감해 주는 청자

### sympathy

- 구성: [[sym]] = with + `pathos`
- 뜻: 동정, 공감
- 어원적 연결: 고통이나 감정을 함께 느낌
- 사용 맥락: `feel sympathy for victims` = 피해자들에게 동정을 느끼다

### apathy

- 구성: `a-` = without + `pathos`
- 뜻: 무관심, 냉담
- 어원적 연결: 감정이나 관심이 없음
- 사용 맥락: `voter apathy` = 유권자 무관심

### antipathy

- 구성: `anti-` = against + `pathos`
- 뜻: 반감, 혐오감
- 어원적 연결: 어떤 대상에 맞서는 감정
- 사용 맥락: `deep antipathy toward corruption` = 부패에 대한 깊은 반감

### pathetic

- 구성: `pathos` + `-etic`
- 뜻: 불쌍한, 한심한
- 어원적 연결: 원래는 감정을 일으키는 것. 현대에는 불쌍하거나 초라한 느낌이 강함
- 사용 맥락: `a pathetic excuse` = 한심한 변명

### pathology

- 구성: `patho-` = disease, suffering + [[logos]] = study
- 뜻: 병리학, 병적 상태
- 어원적 연결: 고통과 질병의 원리를 설명하는 학문
- 사용 맥락: `clinical pathology` = 임상 병리학

### pathological

- 구성: `pathology` + `-ical`
- 뜻: 병적인, 병리학의
- 어원적 연결: 질병이나 비정상적 상태와 관련됨
- 사용 맥락: `pathological behavior` = 병적인 행동

## 기억 포인트

`path` 계열은 "느낌과 고통" 이미지.

- `empathy`: 감정 안으로 들어감
- `sympathy`: 감정을 함께 느낌
- `apathy`: 감정이 없음
- `antipathy`: 반대 감정, 반감
- `pathetic`: 감정을 자극함 → 불쌍함, 한심함
- `pathology`: 고통과 질병을 다룸
