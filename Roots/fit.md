# fit

- 출발 단어: `fit`
- 핵심 어원: Middle English `fitten` = 알맞다, 맞추다
- 핵심 의미: 크기·상황·목적에 알맞게 맞음
- 관련 문서: [[mod]], [[fac]], Latin `aptus`
- 주의: 발작, 격발의 `fit`은 다른 어원 가능성이 커서 분리.

## 출발 단어 분석

`fit` ← Middle English `fitten`

- 원래 이미지: 어떤 대상이 자리나 목적에 알맞게 들어맞음
- 어원은 완전히 투명하지 않음. 스칸디나비아계 `fitja` = 짜다, 얽다 와 관련될 가능성도 언급됨
- 현대 핵심: 맞다, 맞추다, 적합하다, 건강한

## 단어들

### fit

- 발음: /ˈfɪt/
- 구성: Middle English `fitten`
- 뜻: 맞다, 맞추다, 적합한, 건강한
- 어원적 연결: 대상이 자리·목적·상황에 알맞음
- 사용 맥락: `The key fits the lock.` = 열쇠가 자물쇠에 맞다

### fitting

- 발음: /ˈfɪtɪŋ/
- 구성: `fit` + `-ing`
- 뜻: 알맞은, 적절한, 피팅
- 어원적 연결: 상황에 맞게 들어맞는 성질
- 사용 맥락: `a fitting tribute` = 적절한 헌사

### fitness

- 발음: /ˈfɪtnəs/
- 구성: `fit` + `-ness`
- 뜻: 적합성, 건강 상태, 체력
- 어원적 연결: 목적에 맞는 상태 → 몸이 활동에 적합한 상태
- 사용 맥락: `physical fitness` = 신체 건강

### fitted

- 발음: /ˈfɪtəd/
- 구성: `fit` + `-ed`
- 뜻: 꼭 맞는, 맞춤의
- 어원적 연결: 몸이나 자리에 맞게 조정된 상태
- 사용 맥락: `a fitted jacket` = 몸에 맞는 재킷

### misfit

- 발음: /ˈmɪˌsfɪt/
- 구성: `mis-` = badly, wrongly + `fit`
- 뜻: 부적응자, 맞지 않는 것
- 어원적 연결: 자리나 집단에 잘 맞지 않음
- 사용 맥락: `a social misfit` = 사회적 부적응자

### outfit

- 발음: /ˈaʊˌtfɪt/
- 구성: `out` + `fit`
- 뜻: 복장, 장비 일체
- 어원적 연결: 밖으로 나가거나 일을 시작할 수 있게 맞춰 갖춘 것
- 사용 맥락: `a winter outfit` = 겨울 복장

### fit out

- 구성: `fit` + `out`
- 뜻: 장비를 갖추다, 준비시키다
- 어원적 연결: 목적에 맞게 필요한 것을 갖추어 내보냄
- 사용 맥락: `fit out a ship` = 배에 장비를 갖추다

### retrofit

- 발음: /ˈrɛtroʊˌfɪt/
- 구성: `retro-` = back, later + `fit`
- 뜻: 기존 물건에 새 장비를 장착하다, 개조하다
- 어원적 연결: 나중에 새 목적에 맞도록 다시 맞춤
- 사용 맥락: `retrofit old buildings` = 오래된 건물을 개조하다

### refit

- 발음: /riˈfɪt/
- 구성: `re-` = again + `fit`
- 뜻: 다시 맞추다, 재정비하다
- 어원적 연결: 다시 목적에 맞는 상태로 만듦
- 사용 맥락: `refit the equipment` = 장비를 재정비하다

### fitful

- 발음: /ˈfɪtfəl/
- 구성: 다른 `fit` = 발작, 격발 + `-ful`
- 뜻: 간헐적인, 불규칙한
- 어원적 연결: 갑작스럽게 한 번씩 일어나는 발작 같은 움직임
- 사용 맥락: `fitful sleep` = 뒤척이는 불규칙한 잠

## 기억 포인트

- `fit`의 핵심은 “알맞게 들어맞음”.
- `fitness`는 원래 “목적에 맞는 상태”이고, 여기서 신체 건강·체력 의미가 강해짐.
- `outfit`은 “목적에 맞게 갖춘 것”에서 복장·장비.
- `retrofit`은 “나중에 다시 맞춰 장착함”.

## 비교 단어

### benefit

- [[fac]]의 `benefit`은 Latin `bene` = well + `facere` = do 계열.
- 끝 철자가 `fit`처럼 보여도 같은 어원 아님.

### fit

- 발작, 경련, 갑작스러운 감정 폭발의 `fit`은 별도 어원.
- `a fit of anger`, `a coughing fit` 같은 표현.
- 현대 철자는 같지만 “알맞다”의 `fit`과 문서상 분리해서 기억.

### apt

- `apt`도 알맞은, ~하기 쉬운 뜻.
- 라틴어 `aptus` = fastened, fitted 계열. 자세한 정리: [[apt]].
