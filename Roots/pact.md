# pact

- 출발 단어: `pact`
- 핵심 어원: 라틴어 `pacisci` / `pangere` = 합의하다, 고정하다, 묶다
- 핵심 이미지: 약속으로 서로를 묶음
- 관련 형태: `pact`, `pac`, `pax`, `pactum`, `compact`
- 관련 문서: [[plac]]
- 주의: `pact/compact/impact`는 "박다, 고정하다, 묶다" 갈래가 강하고, `peace/pacify/pacific`은 `pax` = 평화 갈래가 강함. 역사적으로 연결되지만 학습할 때는 두 갈래로 구분.

## 출발 단어 분석

`pact`는 라틴어 `pactum`에서 옴.

- `pacisci`: 합의하다, 계약하다
- `pangere`: 고정하다, 묶다

따라서 `pact`는 "서로를 묶는 합의" → "협정, 조약"의 뜻.

## 단어들

#### pact / impact 갈래

### pact

- 구성: 라틴어 `pactum`
- 뜻: 협정, 약속, 조약
- 어원적 연결: 합의로 서로를 묶음
- 사용 맥락: `a peace pact` = 평화 협정

### compact

- 구성: `com-` + `pact`
- 뜻: 조밀한, 소형의, 협정
- 어원적 연결: 함께 단단히 묶거나 고정함
- 사용 맥락: `a compact car` = 소형차, `a political compact` = 정치적 협약

### impact

- 구성: Latin `impactus` ← `in-` = into + `pangere` = 박다, 고정하다
- 뜻: 충격, 영향
- 어원적 연결: 안으로 세게 박아 넣음. 물리적 충돌에서 심리적·사회적 영향으로 확장
- 사용 맥락: `have an impact on society` = 사회에 영향을 미치다

### impactful

- 구성: `impact` + `-ful`
- 뜻: 영향력이 큰, 강한 인상을 주는
- 어원적 연결: 안으로 박히듯 강한 효과를 남김
- 사용 맥락: `an impactful speech` = 강한 인상을 남긴 연설

### impinge

- 구성: `in-` + `pangere`
- 뜻: 영향을 미치다, 침해하다, 부딪치다
- 어원적 연결: 안으로 박거나 부딪침
- 사용 맥락: `impinge on freedom` = 자유를 침해하다

#### peace / pacify 갈래

### pacify

- 구성: `pac-` = peace + `-fy`
- 뜻: 진정시키다, 평화롭게 하다
- 어원적 연결: 갈등을 평화로운 상태로 만듦
- 사용 맥락: `pacify the crowd` = 군중을 진정시키다

### appease

- 구성: Old French `apaisier` = bring to peace
- 뜻: 달래다, 누그러뜨리다, 비위를 맞추다
- 어원적 연결: `a-` = to + `pais` = peace. 상대를 평화로운 상태로 향하게 함
- 사용 맥락: `appease angry customers` = 화난 고객들을 달래다

### pacific

- 구성: `pac-` + `-ific`
- 뜻: 평화로운
- 어원적 연결: 갈등이 없는 평화로운 상태
- 사용 맥락: `a pacific attitude` = 평화적인 태도

### pacifist

- 구성: `pac-` = peace + `-ist`
- 뜻: 평화주의자
- 어원적 연결: 전쟁보다 평화 상태를 추구하는 사람
- 사용 맥락: `a committed pacifist` = 확고한 평화주의자

### peace

- 구성: Latin `pax / pac-`
- 뜻: 평화
- 어원적 연결: 전쟁이나 갈등이 멈춘 안정된 상태
- 사용 맥락: `keep the peace` = 평화를 유지하다

## 비교 단어

### contract

- 구성: `con-` + `tract`
- 뜻: 계약, 수축하다
- 어원적 연결: [[tract]] 계열. 함께 끌어 묶음
- 구분 포인트: `contract` = 끌어 묶은 계약, `pact` = 약속으로 묶은 합의

### agreement

- 구성: `ad-` + `gratus` 계열
- 뜻: 합의
- 어원적 연결: 뜻이 맞음
- 구분 포인트: `agreement`는 일반적 합의, `pact`는 더 공식적/강한 협정 느낌

## 기억 포인트

`pact` 문서는 두 갈래로 기억.

- `pact`: 서로를 묶는 약속
- `compact`: 함께 단단히 묶임
- `impact`: 안으로 박혀 영향 줌
- `impactful`: 강하게 박히듯 효과를 남김
- `impinge`: 부딪쳐 침해함
- `pacify`: 평화로운 상태로 만듦
- `pacifist`: 평화를 추구하는 사람
- `appease`: 상대를 평화로운 상태로 달램
- `peace`: 갈등이 멈춘 상태
