# graph

- 출발 단어: `graph`
- 핵심 의미: 쓰다, 그리다, 기록하다
- 관련 형태: Greek `graphein` = write, scratch, draw
- 관련 문서: [[dia]], [[para]], [[top]], [[dem]]

## 핵심 이미지

`graph` 계열의 바탕은 "긁어 쓰거나 그려 기록함".

선을 그려 나타내면 `graph`.
시각적으로 그려 보이면 `graphic`.
빛으로 그린 기록은 `photograph`.
삶을 적은 기록은 `biography`.
땅의 모양을 적으면 `geography`, `topography`.
글 옆의 표시에서 문단이 되면 `paragraph`.

## 단어들

### graph

- 구성: Greek `graphein` = write, draw
- 뜻: 그래프, 도표; 그래프로 나타내다
- 연결: 수나 관계를 선과 점으로 그려 기록함
- 사용 맥락: `draw a graph` = 그래프를 그리다

### graphic

- 구성: `graph` + `-ic`
- 뜻: 그림의, 도표의, 생생한
- 연결: 눈에 보이게 그려 보여 줌
- 사용 맥락: `graphic design` = 그래픽 디자인

### graphics

- 구성: `graphic` + `-s`
- 뜻: 그래픽, 시각 자료
- 연결: 그려서 보여 주는 이미지나 도표
- 사용 맥락: `computer graphics` = 컴퓨터 그래픽

### photograph

- 구성: Greek `photo-` = light + `graph` = write, draw
- 뜻: 사진
- 연결: 빛으로 그린 기록
- 사용 맥락: `take a photograph` = 사진을 찍다

### photography

- 구성: `photo-` = light + `graphy` = writing, drawing
- 뜻: 사진술, 사진 촬영
- 연결: 빛으로 이미지를 기록하는 기술
- 사용 맥락: `digital photography` = 디지털 사진

### biography

- 구성: Greek `bio-` = life + `graphy` = writing
- 뜻: 전기, 일대기
- 연결: 한 사람의 삶을 적은 기록
- 사용 맥락: `write a biography` = 전기를 쓰다

### geography

- 구성: Greek `geo-` = earth + `graphy` = writing, description
- 뜻: 지리, 지리학
- 연결: 땅과 지역을 기록하고 설명함
- 사용 맥락: `study geography` = 지리를 공부하다

### topography

- 구성: Greek `topos` = place + `graphia` = writing, description
- 뜻: 지형, 지형 묘사
- 연결: 장소의 모양을 그려 적음
- 사용 맥락: `mountain topography` = 산악 지형

### paragraph

- 구성: Greek `para-` = beside + `graphein` = write
- 뜻: 문단
- 연결: 원래는 글 옆에 표시를 해 새 단락을 알린 것
- 사용 맥락: `write a paragraph` = 문단을 쓰다

### diagram

- 구성: `dia-` = across, through + `graph` = draw
- 뜻: 도표, 도식
- 연결: 선을 가로질러 그려 구조를 보여 줌
- 사용 맥락: `draw a diagram` = 도표를 그리다

### demographic

- 구성: `demo` = people + `graph` = write, record
- 뜻: 인구통계의
- 연결: 사람 집단을 기록하고 분류함
- 사용 맥락: `demographic data` = 인구통계 자료

## 비교 단어

`gram`, `grammar`, `program`도 Greek `gramma` = written mark 계열과 연결됨.
`graph`는 행위나 기술인 "쓰다, 그리다, 기록하다"가 더 잘 보이는 형태.

`scribe`, `script`는 Latin 계열의 "쓰다" 어근.
`graph`는 Greek 계열이라 `photograph`, `biography`, `geography`, `diagram` 같은 학술어와 합성어에 자주 보임.
