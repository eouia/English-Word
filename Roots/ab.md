# ab

- 출발 단어: `absent`
- 핵심 어원: Latin prefix `ab-`, `abs-`, `a-` = away, from, off
- 핵심 이미지: 어떤 기준점에서 떨어짐, 벗어남
- 관련 문서: [[ex]], [[de]], [[se]]
- 주의: 모든 `ab-` 철자가 이 접두사인 것은 아님. `abyss`는 Greek `abyssos` 계열이라 별도 [[abyss]] 문서.

## 출발 단어 분석

`absent` = `ab-/abs-` + Latin `esse`

- `ab-/abs-`: away, from
- `esse`: be
- 원래 이미지: 있어야 할 자리에서 떨어져 있음
- 현대 의미: 부재한, 결석한

## 단어들

### absent

- 구성: `abs-` = away + Latin `esse` = be
- 뜻: 부재한, 결석한
- 어원적 연결: 있어야 할 자리에서 떨어져 있음
- 사용 맥락: `absent from school` = 학교에 결석한
- 관련: [[present#absent|present 문서의 absent]]

### absence

- 구성: `absent` 계열
- 뜻: 부재, 결석, 없음
- 어원적 연결: 있어야 할 곳에 있지 않은 상태
- 사용 맥락: `in the absence of evidence` = 증거가 없는 상황에서

### abnormal

- 구성: `ab-` = away from + `normal`
- 뜻: 비정상적인, 일반적 기준에서 벗어난
- 어원적 연결: 정상 기준에서 떨어져 있음
- 사용 맥락: `abnormal behavior` = 비정상적 행동

### abduct

- 구성: `ab-` = away + Latin `ducere` = lead
- 뜻: 납치하다, 유괴하다
- 어원적 연결: 사람을 원래 자리에서 끌고 감
- 사용 맥락: `abduct a child` = 아이를 유괴하다
- 관련: [[duc#abduct|duc 문서의 abduct]]

### abstain

- 구성: `abs-` = away + Latin `tenere` = hold
- 뜻: 삼가다, 기권하다
- 어원적 연결: 손을 대지 않고 떨어져 있음
- 사용 맥락: `abstain from alcohol` = 술을 삼가다

### abstract

- 구성: `abs-` = away + Latin `trahere` = draw, pull
- 뜻: 추상적인; 요약하다
- 어원적 연결: 구체적 대상에서 끌어내어 따로 생각함
- 사용 맥락: `abstract concepts` = 추상 개념
- 관련: [[tract-pull#abstract|tract-pull 문서의 abstract]]

### abrupt

- 구성: `ab-` = off + Latin `rumpere` = break
- 뜻: 갑작스러운, 퉁명스러운
- 어원적 연결: 흐름이 끊겨 떨어진 듯함
- 사용 맥락: `an abrupt change` = 갑작스러운 변화
- 관련: [[rupt#abrupt|rupt 문서의 abrupt]]

### abuse

- 구성: `ab-` = away, wrongly + Latin `uti` = use
- 뜻: 남용하다, 학대하다
- 어원적 연결: 바른 사용에서 벗어나 잘못 씀
- 사용 맥락: `abuse power` = 권력을 남용하다
- 주의: 사람에게 쓰면 신체적·정서적 학대의 뜻이 강함.

### abhor

- 구성: `ab-` = away + Latin `horrere` = shudder
- 뜻: 혐오하다, 몹시 싫어하다
- 어원적 연결: 몸서리치며 떨어져 나가려 함
- 사용 맥락: `abhor violence` = 폭력을 혐오하다

### abject

- 구성: `ab-` = away, down + Latin `jacere` = throw
- 뜻: 비참한, 굴욕적인
- 어원적 연결: 아래로 내던져져 낮아진 상태
- 사용 맥락: `abject poverty` = 극심한 빈곤

### abrogate

- 구성: `ab-` = away + Latin `rogare` = ask, propose a law
- 뜻: 폐지하다, 철회하다
- 어원적 연결: 법이나 합의를 효력에서 떼어 냄
- 사용 맥락: `abrogate a treaty` = 조약을 폐기하다

### absolve

- 구성: `ab-` = away + Latin `solvere` = loosen
- 뜻: 면제하다, 무죄로 하다
- 어원적 연결: 책임이나 죄에서 풀어 떨어뜨림
- 사용 맥락: `absolve someone of blame` = 책임을 면제하다
- 관련: [[solv#absolve|solv 문서의 absolve]]

## 기억 포인트

`ab-` 계열은 "기준점에서 떨어짐, 벗어남" 이미지.

- `absent`: 자리에서 떨어져 있음
- `abnormal`: 정상 기준에서 벗어남
- `abduct`: 끌고 멀리 데려감
- `abstain`: 손대지 않고 떨어져 있음
- `abstract`: 구체물에서 끌어내 따로 생각함
- `abrupt`: 흐름에서 끊겨 떨어짐
- `abuse`: 바른 사용에서 벗어남

## 비교 단어

### ex

- 뜻: 밖으로, 밖에
- 구분 포인트: [[ex]]는 안에서 밖으로 나가는 이미지. `ab-`는 기준점에서 멀어지거나 떨어지는 이미지.

### de

- 뜻: 아래로, 떨어져, 제거
- 구분 포인트: [[de]]는 낮아짐·제거·반전의 느낌이 강함. `ab-`는 떨어져 벗어남.

### se

- 뜻: 따로, 떨어져
- 구분 포인트: [[se]]는 분리된 상태를 강조. `ab-`는 출발점에서 멀어지는 방향을 강조.
