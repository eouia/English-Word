# phren

- 출발 단어: `schizophrenia`
- 핵심 어원: Greek `phren` = 마음, 정신, 횡격막
- 핵심 이미지: 마음과 정신의 자리로 여겨진 몸속 중심
- 관련 문서: [[psych]], [[ment]], [[som]]
- 주의: `phrenology`는 역사적으로 쓰였지만 현재는 과학으로 인정되지 않는 골상학.

## 출발 단어 분석

`schizophrenia` ← Greek `schizein` + `phren`

- `schizein`: split
- `phren`: mind
- 원래 이미지: 마음이 갈라진 상태
- 현대 의미: 조현병

## 단어들

### schizophrenia

- 구성: Greek `schizein` = split + `phren` = mind
- 뜻: 조현병
- 어원적 연결: 문자 그대로는 "마음의 분열". 현대 의학 의미와 단순히 같지는 않음
- 사용 맥락: `treatment for schizophrenia` = 조현병 치료

### schizophrenic

- 구성: `schizophrenia` + `-ic`
- 뜻: 조현병의, 조현병 환자
- 어원적 연결: 조현병과 관련됨
- 사용 맥락: 의학 맥락에서는 낙인 표현을 피하고 `person with schizophrenia`가 더 조심스러운 표현

### phrenology

- 구성: `phren` = mind + [[logos]] = study
- 뜻: 골상학
- 어원적 연결: 머리뼈 모양으로 정신 특성을 읽으려 한 옛 이론
- 사용 맥락: 현재는 사이비 과학의 예로 언급됨

### phrenic

- 구성: Greek `phren` + `-ic`
- 뜻: 횡격막의
- 어원적 연결: `phren`이 마음뿐 아니라 횡격막도 가리킨 역사적 의미
- 사용 맥락: `phrenic nerve` = 횡격막 신경

## 기억 포인트

`phren` 계열은 "마음과 횡격막" 이미지.

- `schizophrenia`: 마음이 갈라졌다는 고대식 명명
- `phrenology`: 마음을 연구한다는 이름의 옛 골상학
- `phrenic`: 횡격막의

## 비교 단어

### psych

- 뜻: 마음, 정신
- 구분 포인트: [[psych]]는 현대 심리·정신 단어의 중심. `phren`은 단어 수는 적지만 `schizophrenia`에서 학습 가치가 큼.

### ment

- 뜻: 마음, 정신
- 구분 포인트: [[ment]]는 라틴어 마음 계열. `phren`은 그리스어의 오래된 마음·횡격막 계열.
