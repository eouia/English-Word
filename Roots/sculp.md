# sculp

- 출발 단어: `sculpture`
- 핵심 어원: Latin `sculpere`, past participle `sculptus` = 새기다, 조각하다
- 더 깊은 연결: Latin `scalpere` = carve, cut; PIE `*skel-` = cut, split
- 핵심 이미지: 단단한 표면을 깎고 새겨 형태를 만듦
- 관련 문서: [[sect]], [[scind]], [[cid]], [[scrib]]
- 주의: `sculpt`는 영어에서 비교적 늦게 생긴 말. `sculpture/sculptor`가 먼저 들어오고 거기서 동사로 쓰임.

## 출발 단어 분석

`sculpture` = Latin `sculptura` ← `sculptus` ← `sculpere`

- `sculpere`: carve, engrave
- `sculptus`: carved, engraved
- 원래 이미지: 돌·나무·금속을 깎아 형상을 만듦
- 현대 의미: 조각, 조각품, 조각하다

## 단어들

### sculpture

- 구성: Latin `sculptura` ← `sculptus`
- 뜻: 조각, 조각품, 조각하다
- 어원적 연결: 깎고 새겨 만든 형상
- 사용 맥락: `modern sculpture` = 현대 조각

### sculpt

- 구성: back-formation from `sculptor/sculpture`
- 뜻: 조각하다, 형태를 만들다
- 어원적 연결: 깎거나 빚어 형상을 만듦
- 사용 맥락: `sculpt a statue` = 상을 조각하다

### sculpted

- 구성: `sculpt` + `-ed`
- 뜻: 조각된, 선명하게 다듬어진
- 어원적 연결: 깎아 만든 듯한 형태
- 사용 맥락: `sculpted features` = 조각 같은 이목구비

### sculptor

- 구성: Latin `sculptor` = one who carves
- 뜻: 조각가
- 어원적 연결: 깎고 새기는 사람
- 사용 맥락: `a famous sculptor` = 유명한 조각가

### sculptural

- 구성: `sculpture` + `-al`
- 뜻: 조각의, 조각 같은
- 어원적 연결: 조각처럼 입체적이고 다듬어진 형태
- 사용 맥락: `sculptural form` = 조각적 형태

### sculpting

- 구성: `sculpt` + `-ing`
- 뜻: 조각하기, 조형
- 어원적 연결: 재료를 깎거나 빚어 형태를 만드는 행위
- 사용 맥락: `ice sculpting` = 얼음 조각

### scalpel

- 구성: Latin `scalpellum` ← `scalprum` = knife, chisel
- 뜻: 외과용 메스
- 어원적 연결: 작고 날카롭게 자르는 도구
- 사용 맥락: surgery 맥락

### scalp

- 구성: 관련 어원이 불확실하지만 `scalpere`와 전통적으로 비교됨
- 뜻: 두피; 두피를 벗기다
- 어원적 연결: 표면을 벗기거나 자르는 이미지와 연결 가능
- 사용 맥락: `dry scalp` = 건조한 두피

## 기억 포인트

`sculp` 계열은 "깎아 새겨 형태 만들기" 이미지.

- `sculpture`: 깎아 만든 조각
- `sculpt`: 조각하다
- `sculptor`: 조각가
- `sculptural`: 조각 같은
- `scalpel`: 자르는 작은 칼
