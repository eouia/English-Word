# lud

- 출발 단어: `elusive`
- 핵심 어원: 라틴어 `ludere` = 놀다, 장난치다, 속이다
- 핵심 의미: 장난처럼 다루다, 속이며 피하다, 말로 암시하다
- 주요 형태: `lud`, `lus`, `lude`, `lusion`
- 관련 문서: [[ex]], [[in]], [[con]], [[logos]]
- 주의: “놀다”에서 “놀리다/속이다/피하다”로 의미가 비유적으로 확장됨.

## 출발 단어 분석

`elusive` = `e-` + `lus` + `-ive`

- `e-`: out, away
- `ludere`: 놀다, 장난치다
- 원래 이미지: 잡으려 하면 장난치듯 빠져나감
- 현대 의미: 붙잡기 어려운, 이해하기 어려운

## 단어들

### elusive

- 구성: `e-` = out, away + `lus` + `-ive`
- 뜻: 붙잡기 어려운, 규정하기 어려운
- 어원적 연결: 잡히지 않고 장난치듯 빠져나가는 성질
- 사용 맥락: `an elusive answer` = 잡히지 않는 답

### elude

- 구성: `e-` = out, away + `lude`
- 뜻: 피하다, 이해되지 않다
- 어원적 연결: 밖으로 빠져나가며 상대를 헛수고하게 함
- 사용 맥락: `The solution eluded him.` = 그는 해결책을 찾지 못했다

### elusion

- 구성: `elude` + `-ion`
- 뜻: 회피, 빠져나감
- 어원적 연결: 장난치듯 빠져나가는 행위
- 사용 맥락: `tax elusion` = 조세 회피

### illusion

- 구성: `in-` = at, upon + `lusion`
- 뜻: 환상, 착각
- 어원적 연결: 감각이나 마음을 가지고 장난쳐 속이는 모습
- 사용 맥락: `an optical illusion` = 착시

### illusory

- 구성: `illusion` + `-ory`
- 뜻: 환상적인, 실제가 아닌
- 어원적 연결: 장난처럼 속이는 겉모습
- 사용 맥락: `illusory gains` = 허상에 가까운 이익

### delude

- 구성: `de-` = down, away + `lude`
- 뜻: 속이다, 착각하게 하다
- 어원적 연결: 장난처럼 마음을 잘못된 쪽으로 몰아감
- 사용 맥락: `delude oneself` = 스스로를 속이다

### delusion

- 구성: `delude` + `-ion`
- 뜻: 망상, 착각
- 어원적 연결: 속임에 넘어가 잘못 믿는 상태
- 사용 맥락: `a dangerous delusion` = 위험한 망상

### allude

- 구성: `ad-` = to + `lude`
- 뜻: 암시하다, 넌지시 언급하다
- 어원적 연결: 직접 말하지 않고 말장난하듯 살짝 건드림
- 사용 맥락: `allude to a problem` = 문제를 암시하다

### allusion

- 구성: `allude` + `-ion`
- 뜻: 암시, 간접 언급
- 어원적 연결: 직접 말하지 않고 가볍게 건드린 표현
- 사용 맥락: `a literary allusion` = 문학적 암시

### collude

- 구성: `com-` = together + `lude`
- 뜻: 공모하다, 결탁하다
- 어원적 연결: 함께 몰래 장난치듯 일을 꾸밈
- 사용 맥락: `collude with rivals` = 경쟁자들과 결탁하다

### collusion

- 구성: `collude` + `-ion`
- 뜻: 공모, 결탁
- 어원적 연결: 함께 짜고 속이는 행위
- 사용 맥락: `price collusion` = 가격 담합

### prelude

- 구성: `pre-` = before + `lude`
- 뜻: 서곡, 전조
- 어원적 연결: 본격적인 연주나 사건 전에 먼저 하는 놀이/연주
- 사용 맥락: `a prelude to war` = 전쟁의 전조

### interlude

- 구성: `inter-` = between + `lude`
- 뜻: 막간, 중간 삽입곡
- 어원적 연결: 본 흐름 사이에 들어간 짧은 놀이/연주
- 사용 맥락: `a brief interlude` = 짧은 막간

### ludicrous

- 구성: Latin `ludicrus` = sportive, playful
- 뜻: 우스꽝스러운, 터무니없는
- 어원적 연결: 놀이처럼 진지하지 않고 우스운 모습
- 사용 맥락: `a ludicrous idea` = 터무니없는 생각

## 기억 포인트

- `lud`는 “놀다, 장난치다”.
- `elusive`: 장난치듯 빠져나가 잡히지 않음.
- `illusion`: 마음이나 눈을 가지고 노는 착각.
- `allude`: 직접 말하지 않고 살짝 건드림.
- `collude`: 함께 장난치듯 몰래 꾸밈.
- `prelude/interlude`: 연주·극의 앞이나 사이에 들어가는 놀이/연주.

## 비교 단어

### elusive vs allusive

- `elusive`: 잡히지 않고 빠져나가는.
- `allusive`: 암시가 많은, 간접 언급하는.
- 둘 다 `ludere` 계열이지만 접두사가 다름.

### illusion vs delusion

- `illusion`: 감각이나 인상 때문에 생기는 착각.
- `delusion`: 더 강하게 잘못 믿는 상태, 망상.
- 둘 다 “속이는 장난” 이미지.

### play

- `play`는 게르만계 기본어.
- 의미는 같지만 라틴어 `ludere` 계열은 아님.
