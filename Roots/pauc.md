# pauc

- 출발 단어: `paucity`
- 핵심 어원: Latin `paucus` = few, little
- 핵심 이미지: 수나 양이 적음, 충분하지 않음
- 관련 문서: [[spers]]

## 단어들

### paucity

- 구성: Latin `paucitas` < `paucus` = few, little
- 뜻: 부족, 결핍, 적음
- 어원적 연결: 수나 양이 적어서 충분하지 않은 상태
- 사용 맥락: `a paucity of evidence` = 증거 부족

### paucities

- 구성: `paucity`의 복수형
- 뜻: 여러 종류의 부족, 결핍들
- 어원적 연결: 부족한 상태가 여러 영역에 나타남
- 사용 맥락: 격식 있는 문맥에서 여러 결핍을 말할 때 사용

### pauciloquent

- 구성: Latin `pauci` = few + `loqui` = speak
- 뜻: 말수가 적은
- 어원적 연결: 말을 적게 함
- 사용 맥락: 드문 단어이므로 인식용. 시험 핵심어는 `paucity`

### pauciflorous

- 구성: Latin `pauci` = few + `flor` = flower
- 뜻: 꽃이 적은
- 어원적 연결: 꽃의 수가 적음
- 사용 맥락: 식물학 문맥의 전문어이므로 참고용

## 기억 포인트

`pauc` 계열은 "few, little" 이미지.

- `paucity`: 적음, 부족
- `pauciloquent`: 말수가 적은
- `pauciflorous`: 꽃이 적은

시험 어휘로는 `paucity` 하나가 핵심. 나머지는 Latin `pauci-`가 "few"라는 점을 보여 주는 보조 단어.

## 비교 단어

### scarcity

- 뜻: 부족, 희소성
- 구분 포인트: `scarcity`는 자원이나 물자가 부족한 상태를 넓게 말함. `paucity`는 격식 있는 말로 "수나 양이 너무 적음"을 강조.

### dearth

- 뜻: 부족, 결핍
- 구분 포인트: `dearth`는 필요한 것이 충분하지 않은 상태. `paucity`보다 문학적이거나 격식 있는 느낌.

### sparse

- 뜻: 드문드문한, 희박한
- 구분 포인트: [[spers]] 계열. `sparse`는 흩어져 있어 밀도가 낮음. `paucity`는 전체 수량이 적음.
