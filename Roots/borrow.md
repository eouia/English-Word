# borrow

- 출발 단어: `borrow`
- 핵심 어원: Old English `borgian` = lend, borrow, pledge
- 핵심 이미지: 보증이나 약속을 두고 잠시 가져옴
- 관련 문서: 없음
- 주의: `borrow`는 현대 영어에서 더 작은 접두사·어근으로 나누기보다 Old English 기본 동사 계열로 보는 편이 안전함.

## 출발 단어 분석

`borrow` ← Old English `borgian`

- `borgian`: lend, borrow, pledge, be surety
- `borg`: pledge, security
- 원래 이미지: 보증이나 담보를 바탕으로 물건을 주고받음
- 의미 변화: 보증을 두고 빌리거나 빌려줌 → 잠시 가져다 쓰고 돌려줌
- 현대 의미: 빌리다, 차용하다

## 단어들

### borrow

- 발음: /ˈbɑˌroʊ/
- 구성: Old English `borgian` = lend, borrow, pledge
- 뜻: 빌리다, 차용하다
- 어원적 연결: 돌려줄 것을 전제로 잠시 가져옴
- 사용 맥락: `borrow money` = 돈을 빌리다

### borrowed

- 발음: /ˈbɑˌroʊd/
- 구성: `borrow` + `-ed`
- 뜻: 빌린, 차용한
- 어원적 연결: 원래 자기 것이 아니라 잠시 가져온 상태
- 사용 맥락: `borrowed time` = 덤으로 얻은 시간, 예정 밖의 여유

### borrower

- 발음: /ˈbɑroʊɚ/
- 구성: `borrow` + `-er`
- 뜻: 빌리는 사람, 차용자
- 어원적 연결: 돌려줄 의무를 지고 가져가는 사람
- 사용 맥락: `student borrowers` = 학자금 대출자

### borrowing

- 발음: /ˈbɑroʊɪŋ/
- 구성: `borrow` + `-ing`
- 뜻: 차용, 빌림; 차용어
- 어원적 연결: 물건뿐 아니라 말·아이디어도 다른 곳에서 가져옴
- 사용 맥락: `a borrowing from French` = 프랑스어에서 온 차용어

### borrowings

- 발음: /ˈbɑroʊɪŋz/
- 구성: `borrowing`의 복수형
- 뜻: 차입금, 차용어들
- 어원적 연결: 여러 번 빌려 온 것들
- 사용 맥락: 경제 문맥에서는 `government borrowings` = 정부 차입금

## 기억 포인트

`borrow` 계열은 "돌려줄 것을 전제로 잠시 가져옴" 이미지.

- `borrow`: 빌리다
- `borrowed`: 빌린, 차용한
- `borrower`: 빌리는 사람
- `borrowing`: 차용, 차용어
- `borrowings`: 차입금, 차용어들

## 비교 단어

### lend

- 뜻: 빌려주다
- 구분 포인트: `borrow`는 받는 쪽, `lend`는 주는 쪽. `I borrowed money from her` = 내가 그녀에게서 빌림. `She lent me money` = 그녀가 내게 빌려줌.

### loan

- 뜻: 대출, 빌려준 돈; 빌려주다
- 구분 포인트: 명사로 자주 쓰이며 금융·공식 문맥에 강함. `borrow`는 동작 중심의 일반 동사.

### debt

- 뜻: 빚, 채무
- 구분 포인트: 빌린 뒤 갚아야 할 의무나 남은 금액. `borrow`는 빚이 생기는 행위.

### owe

- 뜻: 빚지고 있다, 갚아야 한다
- 구분 포인트: `borrow`는 가져오는 행위, `owe`는 그 결과로 갚을 의무가 남은 상태.
