# verg

- 출발 단어: `diverged`
- 핵심 어원: Latin `vergere` = bend, turn, incline, tend toward
- 핵심 이미지: 어떤 방향으로 기울거나 향함
- 관련 문서: [[vers]], [[clin]], [[tend]]
- 주의: [[vers]]는 Latin `vertere` = turn 계열. `verg`는 Latin `vergere` = incline, tend toward 계열. 의미는 가깝지만 어원은 분리.

## 출발 단어 분석

`diverged` ← Modern Latin `divergere` = go in different directions

- `di-/dis-`: apart, in different directions
- `vergere`: bend, turn, incline, tend toward
- `-ed`: 과거형·과거분사 접미사
- 원래 이미지: 한 지점에서 서로 다른 방향으로 기울어 감
- 의미 변화: 방향이 갈라짐 → 의견·형태·발전 경로가 달라짐
- 현대 의미: 갈라졌다, 달라졌다, 분기했다

## 단어들

### diverged

- 구성: `diverge` + `-ed`
- 뜻: 갈라진, 달라진, 분기한
- 어원적 연결: 공통 지점에서 서로 다른 방향으로 기울어 나간 상태
- 사용 맥락: `their paths diverged` = 그들의 길이 갈라졌다

### diverge

- 구성: Latin `di-/dis-` = apart + `vergere` = incline, tend
- 뜻: 갈라지다, 달라지다, 분기하다
- 어원적 연결: 한 지점에서 서로 다른 방향으로 향함
- 사용 맥락: `opinions diverge` = 의견이 갈라지다

### divergence

- 구성: `diverge` + `-ence`
- 뜻: 차이, 분기, 발산
- 어원적 연결: 방향이나 발전 경로가 갈라지는 상태
- 사용 맥락: `a divergence of views` = 견해 차이

### divergent

- 구성: `diverge` + `-ent`
- 뜻: 갈라지는, 다른 방향의, 상이한
- 어원적 연결: 같은 출발점에서 다른 쪽으로 향함
- 사용 맥락: `divergent opinions` = 서로 다른 의견들

### converge

- 구성: Latin `con-` = together + `vergere` = incline, tend
- 뜻: 모이다, 수렴하다
- 어원적 연결: 여러 방향에서 한 지점으로 함께 기울어 감
- 사용 맥락: `the lines converge` = 선들이 한 점으로 모이다

### convergence

- 구성: `converge` + `-ence`
- 뜻: 수렴, 융합
- 어원적 연결: 여러 흐름이 한 방향·한 지점으로 모임
- 사용 맥락: `technological convergence` = 기술 융합

### convergent

- 구성: `converge` + `-ent`
- 뜻: 수렴하는, 한데 모이는
- 어원적 연결: 서로 다른 방향에서 한 지점으로 향함
- 사용 맥락: `convergent evolution` = 수렴 진화

### verge

- 구성: Latin `vergere` = bend, incline, tend
- 뜻: 가장자리, 직전; ~에 가까워지다
- 어원적 연결: 어떤 경계나 상태 쪽으로 기울어 감
- 사용 맥락: `on the verge of collapse` = 붕괴 직전의

## 기억 포인트

`verg` 계열은 "방향이 기울어 향함" 이미지.

- `diverge`: 서로 다른 방향으로 기울어 갈라짐
- `diverged`: 갈라져 달라진
- `divergence`: 갈라짐, 차이
- `converge`: 한 방향으로 기울어 모임
- `convergence`: 수렴
- `verge`: 어떤 상태의 가장자리로 기울어 감

## 비교 단어

### vers

- 뜻: 돌다, 돌리다, 방향을 바꾸다
- 구분 포인트: [[vers]]는 `vertere` = turn. `verg`는 `vergere` = incline, tend toward.

### clin

- 뜻: 기울다, 기대다
- 구분 포인트: [[clin]]도 기울어짐. `verg`는 어떤 방향이나 지점으로 향해 기울어 감.

### tend

- 뜻: 뻗다, 향하다
- 구분 포인트: [[tend]]는 방향으로 뻗음. `verg`는 경로가 기울거나 수렴·발산하는 느낌.
