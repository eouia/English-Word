# in

- 출발 단어: `include`
- 핵심 접두사: Latin `in-` / `im-` / `il-` / `ir-` = in, into 또는 not
- 핵심 이미지: 안으로 들어감 / 반대로 부정함
- 관련해서 헷갈릴 수 있는 접두사: [[em]] = 안에 넣다, 어떤 상태로 만들다, [[ex]] = 밖으로

## 출발 단어 분석

`in-`은 큰 두 갈래가 있음.

- `in-` = in, into: 안으로, 안에
- `in-` = not: 아니다, 반대

형태도 뒤 소리에 따라 바뀜.

- `im-`: `impossible`, `imperfect`
- `il-`: `illegal`, `illogical`
- `ir-`: `irregular`, `irresponsible`

## 갈래 1: 안으로

### include

- 발음: /ˌɪˈnklud/
- 구성: `in-` = in + [[clud]]
- 뜻: 포함하다
- 어원적 연결: 안에 닫아 넣음
- 사용 맥락: `include examples` = 예시를 포함하다

### import

- 발음: n. /ˈɪˌmpɔrt/, v. /ˌɪˈmpɔrt/
- 구성: `in-` = into + [[port]]
- 뜻: 수입하다
- 어원적 연결: 안으로 실어 들임
- 사용 맥락: `import goods` = 상품을 수입하다

### influence

- 발음: /ˈɪnfluəns/
- 구성: `in-` = into + [[flu]]
- 뜻: 영향, 영향을 미치다
- 어원적 연결: 안으로 흘러 들어와 작용함
- 사용 맥락: `influence public opinion` = 여론에 영향을 미치다

### inspect

- 발음: /ˌɪˈnspɛkt/
- 구성: `in-` = into + [[spect]]
- 뜻: 검사하다
- 어원적 연결: 안쪽까지 들여다봄
- 사용 맥락: `inspect the building` = 건물을 점검하다

### inscribe

- 발음: /ˌɪˈnskraɪb/
- 구성: `in-` = in, on + [[scrib]]
- 뜻: 새기다, 적어 넣다
- 어원적 연결: 안이나 표면에 써 넣음
- 사용 맥락: `inscribe a name` = 이름을 새기다

### induce

- 발음: /ˌɪˈndus/
- 구성: `in-` = into + [[duc]]
- 뜻: 유도하다, 초래하다
- 어원적 연결: 어떤 상태 안으로 이끌어 감
- 사용 맥락: `induce sleep` = 잠을 유도하다

### impose

- 발음: /ˌɪˈmpoʊz/
- 구성: `in-` = on + [[pos]]
- 뜻: 부과하다, 강요하다
- 어원적 연결: 위에 얹어 놓음
- 사용 맥락: `impose a tax` = 세금을 부과하다

### incarnate

- 발음: /ˌɪˈnkɑrnət/
- 구성: `in-` = into + [[carn]]
- 뜻: 육체를 입은, 구현된
- 어원적 연결: 살 안으로 들어가 몸을 얻음
- 사용 맥락: `evil incarnate` = 악의 화신

## 갈래 2: not

### invisible

- 발음: /ˌɪˈnvɪzəbəl/
- 구성: `in-` = not + [[vid]]
- 뜻: 보이지 않는
- 어원적 연결: 볼 수 없음
- 사용 맥락: `invisible barriers` = 보이지 않는 장벽

### incredible

- 발음: /ˌɪˈnkrɛdəbəl/
- 구성: `in-` = not + [[cred]]
- 뜻: 믿기 어려운, 대단한
- 어원적 연결: 믿을 수 없을 만큼 큼
- 사용 맥락: `an incredible achievement` = 놀라운 성취

### intolerant

- 발음: /ˌɪˈntɑlɚənt/
- 구성: `in-` = not + [[tol]]
- 뜻: 관용 없는, 참지 못하는
- 어원적 연결: 받아들이거나 견디지 않음
- 사용 맥락: `intolerant attitudes` = 편협한 태도

### independent

- 발음: /ˌɪndɪˈpɛndənt/
- 구성: `in-` = not + [[pend]]
- 뜻: 독립적인
- 어원적 연결: 다른 것에 매달려 있지 않음
- 사용 맥락: `independent research` = 독립적 연구

### impenetrable

- 발음: /ˌɪˈmpɛnətrəbəl/
- 구성: `im-` = not + [[penetr]]
- 뜻: 뚫고 들어갈 수 없는, 이해하기 어려운
- 어원적 연결: 안으로 들어갈 수 없음
- 사용 맥락: `an impenetrable forest` = 뚫고 들어가기 어려운 숲

### imperfect

- 발음: /ˌɪˈmpɝfɪkt/
- 구성: `im-` = not + [[fac]]
- 뜻: 불완전한
- 어원적 연결: 완전히 만들어지지 않음
- 사용 맥락: `imperfect evidence` = 불완전한 증거

### illegal

- 발음: /ˌɪˈligəl/
- 구성: `il-` = not + [[leg]]
- 뜻: 불법의
- 어원적 연결: 법에 맞지 않음
- 사용 맥락: `illegal activity` = 불법 활동

### illogical

- 발음: /ˌɪˈlɑdʒɪkəl/
- 구성: `il-` = not + [[logos]]
- 뜻: 비논리적인
- 어원적 연결: 논리에 맞지 않음
- 사용 맥락: `an illogical argument` = 비논리적 주장

### irregular

- 발음: /ˌɪˈrɛgjəlɚ/
- 구성: `ir-` = not + Latin `regularis`
- 뜻: 불규칙한
- 어원적 연결: 규칙에 맞지 않음
- 사용 맥락: `irregular patterns` = 불규칙한 패턴

### irresponsible

- 발음: /ˌɪrəˈspɑnsəbəl/
- 구성: `ir-` = not + [[spond]]
- 뜻: 무책임한
- 어원적 연결: 응답하거나 책임질 준비가 없음
- 사용 맥락: `irresponsible behavior` = 무책임한 행동

## 기억 포인트

`in-`은 먼저 문맥으로 갈래를 판단함.

- 안으로: `include`, `import`, `influence`, `inspect`, `inscribe`, `induce`
- 부정: `invisible`, `incredible`, `independent`, `imperfect`, `illegal`, `irregular`

## 비교 단어

### ex

- 뜻: 밖으로
- 구분 포인트: [[ex]]는 밖으로 나감. `import`는 안으로 들여옴, `export`는 밖으로 내보냄

### em

- 뜻: 안에 넣다, 어떤 상태로 만들다
- 구분 포인트: [[em]]은 `en-/em-` 동사화 접두사로 "~하게 만들다"에 자주 씀. `in-`은 위치나 부정 기능이 더 큼
