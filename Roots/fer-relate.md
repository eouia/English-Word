# fer-relate

- 출발 단어: `refer`
- 핵심 어원: Latin `ferre` / `lat-` = 가져가다, 옮기다
- 핵심 이미지: 생각·정보를 다시 가져와 연결함
- 허브 문서: [[fer]]

## 단어들

### refer

- 구성: `re-` = back + `fer`
- 뜻: 언급하다, 참조하다, 맡기다
- 어원적 연결: 어떤 문제나 정보를 뒤로/다른 곳으로 가져가 연결함
- 사용 맥락: `refer to a source` = 출처를 참조하다

### reference

- 구성: `refer` + `-ence`
- 뜻: 언급, 참조, 추천서
- 어원적 연결: 다시 가져가 연결하는 것
- 사용 맥락: `for future reference` = 나중에 참고하기 위해

### relate

- 구성: `re-` = back + Latin `lat-` = carried
- 뜻: 관련시키다, 관련되다, 말하다
- 어원적 연결: 어떤 것을 다시 가져와 다른 것과 연결함. 이야기를 가져와 전함
- 사용 맥락: `relate the theory to practice` = 이론을 실제와 관련시키다

### related

- 구성: `relate` + `-ed`
- 뜻: 관련된, 친척 관계의
- 어원적 연결: 서로 연결되어 있음
- 사용 맥락: `closely related topics` = 밀접히 관련된 주제들

### relation

- 구성: Latin `relatio` < `relatus`
- 뜻: 관계, 관련, 친척
- 어원적 연결: 서로 가져다 놓아 연결된 상태
- 사용 맥락: `the relation between cause and effect` = 원인과 결과의 관계

### relative

- 구성: `relate` 계열 + `-ive`
- 뜻: 상대적인, 친척
- 어원적 연결: 다른 것과의 관계 속에서 정해짐
- 사용 맥락: `relative importance` = 상대적 중요성

### correlate

- 구성: `con-` = together + `relate`
- 뜻: 상관관계를 가지다, 서로 관련시키다
- 어원적 연결: 둘 이상을 함께 연결해 봄
- 사용 맥락: `correlate with income` = 소득과 상관관계가 있다

### correlation

- 구성: `correlate` + `-ion`
- 뜻: 상관관계
- 어원적 연결: 함께 연결되어 움직이는 관계
- 사용 맥락: `a strong correlation` = 강한 상관관계

## 기억 포인트

`fer-relate`는 "가져와 연결함" 갈래.

- `refer`: 다시 가져가 참조함
- `relate`: 가져와 관계를 맺음
- `relative`: 관계 속에서 정해짐
- `correlate`: 함께 연결해 봄
