# vers

- 출발 단어: `convert`
- 핵심 어원: 라틴어 `vertere` / `vers-` = 돌다, 돌리다
- 핵심 이미지: 방향을 바꾸어 돌림
- 관련해서 헷갈릴 수 있는 어근: [[tour]] = 돌다, 둘러가다 계열, [[tract]] = 끌다, 당기다 계열, [[verg]] = 기울어 향하다 계열

## 출발 단어 분석

`convert`는 `con-` + `vertere`.

- `con-`: together, completely
- `vertere`: 돌리다

따라서 `convert`는 "완전히 방향을 돌리다" → "전환하다, 개종시키다"의 뜻.

## 단어들

### convert

- 발음: n. /ˈkɑnvɚt/, v. /kəˈnvɝt/
- 구성: `con-` = completely + `vert`
- 뜻: 전환하다, 개종하다
- 어원적 연결: 방향이나 상태를 완전히 돌림
- 사용 맥락: `convert energy into heat` = 에너지를 열로 전환하다

### conversion

- 발음: /kəˈnvɝʒən/
- 구성: `con-` + `vers` + `-ion`
- 뜻: 전환, 개종
- 어원적 연결: 다른 상태로 방향이 돌아감
- 사용 맥락: `data conversion` = 데이터 변환

### reverse

- 발음: /rɪˈvɝs/
- 구성: `re-` = back + `vers`
- 뜻: 뒤집다, 반대로 하다
- 어원적 연결: 뒤로 돌림
- 사용 맥락: `reverse the decision` = 결정을 뒤집다

### averse

- 발음: /əˈvɝs/
- 구성: Latin `aversus` = turned away
- 뜻: 꺼리는, 싫어하는
- 어원적 연결: 마음이 어떤 대상에서 돌아서 멀어짐
- 사용 맥락: `be averse to risk` = 위험을 꺼리다
- 주의: 보통 `be averse to` 형태. `adverse`는 "불리한, 반대의"로 의미와 철자가 다름.

### aversion

- 발음: /əˈvɝʒən/
- 구성: `averse` + `-ion`
- 뜻: 혐오감, 강한 반감, 꺼림
- 어원적 연결: 마음이 어떤 대상에서 돌아서 멀어진 상태
- 사용 맥락: `an aversion to conflict` = 갈등을 몹시 꺼림
- 주의: 단순한 dislike보다 강한 반감. 보통 `aversion to` 형태.

### adverse

- 발음: /ædˈvɝs/
- 구성: Latin `adversus` = turned toward, opposing
- 뜻: 불리한, 해로운, 반대의
- 어원적 연결: 어떤 방향에 맞서 돌아서 있음
- 사용 맥락: `adverse effects` = 부작용, 해로운 영향
- 주의: 사람의 감정적 꺼림은 `averse`. 조건·상황·효과가 불리하거나 해로우면 `adverse`.

### diverse

- 발음: /daɪˈvɝs/
- 구성: `di-` = apart + `vers`
- 뜻: 다양한
- 어원적 연결: 여러 방향으로 돌아 흩어짐
- 사용 맥락: `diverse cultures` = 다양한 문화

### diversity

- 발음: /dɪˈvɝsɪti/
- 구성: `diverse` + `-ity`
- 뜻: 다양성
- 어원적 연결: 방향이 여러 갈래로 나뉜 상태
- 사용 맥락: `biological diversity` = 생물 다양성

### divert

- 발음: /daɪˈvɝt/
- 구성: `di-` = aside + `vert`
- 뜻: 전환하다, 주의를 돌리다
- 어원적 연결: 옆길로 돌림
- 사용 맥락: `divert attention` = 주의를 돌리다

### divorce

- 발음: /dɪˈvɔrs/
- 구성: Latin `divortium` / `divertere` = turn apart, separate
- 뜻: 이혼, 이혼하다
- 어원적 연결: 부부가 같은 길에서 서로 다른 방향으로 돌아서 갈라짐
- 사용 맥락: `file for divorce` = 이혼을 신청하다
- 주의: 법률·관계 맥락의 "갈라섬". 일반적인 "다양함"은 `diverse`, 주의나 길을 "돌림"은 `divert`.

### advertise

- 발음: /ˈædvɚˌtaɪz/
- 구성: `ad-` = to + `vert`
- 뜻: 광고하다
- 어원적 연결: 사람들의 주의를 어떤 쪽으로 돌림
- 사용 맥락: `advertise a product` = 제품을 광고하다

### advertising

- 발음: /ˈædvɚˌtaɪzɪŋ/
- 구성: `advertise` + `-ing`
- 뜻: 광고 활동, 광고의
- 어원적 연결: 사람들의 주의를 상품이나 메시지 쪽으로 돌리는 활동
- 사용 맥락: `advertising campaigns` = 광고 캠페인

### advertisement

- 발음: /æˈdvɝtəzmənt/
- 구성: `advertise` + `-ment`
- 뜻: 광고
- 어원적 연결: 주의를 돌리게 만드는 것
- 사용 맥락: `online advertisements` = 온라인 광고

### advertent

- 구성: Latin `advertere` = turn toward
- 뜻: 주의를 기울이는, 의식적인
- 어원적 연결: 마음이나 주의를 어떤 대상 쪽으로 돌림
- 사용 맥락: 드문 표현. 보통 `inadvertent`의 반대말로 이해.

### inadvertent

- 발음: /ˌɪnəˈdvɝtənt/
- 구성: `in-` = not + `advertent`
- 뜻: 부주의한, 의도하지 않은
- 어원적 연결: 주의를 그쪽으로 돌리지 못한 상태
- 사용 맥락: `an inadvertent error` = 부주의로 인한 실수

### inadvertently

- 발음: /ˌɪnəˈdvɝtəntli/
- 구성: `inadvertent` + `-ly`
- 뜻: 무심코, 부주의로, 의도치 않게
- 어원적 연결: 주의를 돌리지 못한 채 행동함
- 사용 맥락: `inadvertently delete a file` = 무심코 파일을 삭제하다

### inadvertence

- 발음: /ˌɪnəˈdvɝtəns/
- 구성: `inadvertent` + `-ence`
- 뜻: 부주의, 무심함
- 어원적 연결: 주의가 필요한 곳으로 마음을 돌리지 않은 상태
- 사용 맥락: 문어적 표현. `through inadvertence` = 부주의로

### controversy

- 발음: /ˈkɑntrəˌvɝsi/
- 구성: `contra-` = against + `vers`
- 뜻: 논란, 논쟁
- 어원적 연결: 서로 반대 방향으로 돌아섬
- 사용 맥락: `a public controversy` = 공적 논란

### controversial

- 발음: /ˌkɑntrəˈvɝʃəl/
- 구성: `controversy` + `-al`
- 뜻: 논란이 많은
- 어원적 연결: 의견이 반대 방향으로 갈림
- 사용 맥락: `a controversial issue` = 논란이 많은 쟁점

### uncontroversial

- 발음: /ˌʌˌnkɑntrəˈvɝʃəl/
- 구성: `un-` = not + `controversial`
- 뜻: 논란의 여지가 없는, 이견이 거의 없는
- 어원적 연결: 의견이 반대 방향으로 크게 갈리지 않음
- 사용 맥락: `an uncontroversial proposal` = 논란이 없는 제안

### uncontroversially

- 구성: `uncontroversial` + `-ly`
- 뜻: 논란 없이, 이견 없이
- 어원적 연결: 반대 방향의 논쟁을 일으키지 않는 방식
- 사용 맥락: `be uncontroversially accepted` = 논란 없이 받아들여지다

### version

- 발음: /ˈvɝʒən/
- 구성: `vers` + `-ion`
- 뜻: 판, 버전, 해석
- 어원적 연결: 어떤 방향으로 돌려 표현한 형태
- 사용 맥락: `the latest version` = 최신 버전

### versatile

- 발음: /ˈvɝsətəl/
- 구성: Latin `versatilis` = turning easily
- 뜻: 다재다능한
- 어원적 연결: 여러 방향으로 쉽게 돌릴 수 있음
- 사용 맥락: `a versatile tool` = 다용도 도구

### versus

- 발음: /ˈvɝsəs/
- 구성: Latin `versus` = turned toward
- 뜻: ~대, ~에 맞서
- 어원적 연결: 서로 마주 향해 돌아섬
- 사용 맥락: `nature versus nurture` = 본성 대 양육

## 기억 포인트

`vers` 계열은 "돌다 / 돌리다" 이미지.

- `convert`: 완전히 돌려 바꿈
- `reverse`: 뒤로 돌림
- `averse`: 마음이 돌아서 꺼림
- `aversion`: 마음이 돌아선 반감
- `adverse`: 맞서 돌아선 불리한 조건
- `diverse`: 여러 방향으로 돌아 흩어짐
- `divert`: 옆으로 돌림
- `divorce`: 서로 다른 방향으로 돌아서 갈라짐
- `advertise`: 주의를 그쪽으로 돌림
- `inadvertently`: 주의를 돌리지 못한 채 무심코
- `controversy`: 반대 방향으로 돌아선 논쟁
- `uncontroversial`: 반대 방향의 논쟁이 거의 없음
- `versatile`: 여러 방향으로 잘 돌릴 수 있음

## 비교 단어

### transform

- 뜻: 변형하다
- 구분 포인트: [[form]] 계열로 "모양을 바꿈"에 초점. `convert`는 방향이나 상태를 돌려 전환함

### tour

- 뜻: 돌다, 둘러가다
- 구분 포인트: [[tour]]는 `contour`, `detour`, `tournament`, `turn`처럼 회전·순회·둘레를 도는 이미지. `vers`는 방향을 돌려 전환하는 이미지.

### verg

- 뜻: 기울어 향하다, 수렴하거나 발산하다
- 구분 포인트: [[verg]]는 Latin `vergere` = incline, tend toward. `diverge`, `converge`는 이쪽. `vers`는 Latin `vertere` = turn.
