# vari

- 출발 단어: `various`
- 핵심 어원: 라틴어 `varius` = 다양한, 다른, 얼룩진
- 핵심 이미지: 한 가지가 아니라 여러 모습으로 달라짐
- 관련해서 헷갈릴 수 있는 어근: [[divers]] = 여러 방향으로 흩어짐, 다름 계열, [[alter]] = 다른 하나, 바꾸다 계열

## 출발 단어 분석

`various`는 Latin `varius` + `-ous`.

- `varius`: varied, different
- `-ous`: 형용사형 접미사

따라서 `various`는 "여러 모습으로 다른" → "다양한, 여러 가지의" 뜻.

## 단어들

### various

- 발음: /ˈvɛriəs/
- 구성: `vari` + `-ous`
- 뜻: 다양한, 여러 가지의
- 어원적 연결: 모습이나 종류가 한 가지로 고정되지 않음
- 사용 맥락: `various methods` = 다양한 방법들

### vary

- 발음: /ˈvɛri/
- 구성: Latin `variare`
- 뜻: 달라지다, 다르게 하다
- 어원적 연결: 상태나 모습이 바뀌어 달라짐
- 사용 맥락: `Prices vary by region.` = 가격은 지역에 따라 다름

### variety

- 발음: /vɚˈaɪəti/
- 구성: `vari` + `-ety`
- 뜻: 다양성, 종류
- 어원적 연결: 서로 다른 모습들이 모인 상태
- 사용 맥락: `a variety of reasons` = 여러 가지 이유

### variation

- 발음: /ˌvɛriˈeɪʃən/
- 구성: `vary` + `-ation`
- 뜻: 변화, 변형, 차이
- 어원적 연결: 원래 형태에서 달라진 모습
- 사용 맥락: `regional variation` = 지역적 차이

### variable

- 발음: /ˈvɛriəbəl/
- 구성: `vary` + `-able`
- 뜻: 변할 수 있는, 변수
- 어원적 연결: 값이나 상태가 달라질 수 있음
- 사용 맥락: `variable costs` = 변동 비용, `a variable in an equation` = 방정식의 변수

### invariably

- 발음: /ˌɪˈnvɛriəbli/
- 구성: `in-` = not + `variable` + `-ly`
- 뜻: 항상, 변함없이
- 어원적 연결: 변하지 않고 같은 상태로
- 사용 맥락: `She is invariably polite.` = 그녀는 늘 예의 바름

### variance

- 발음: /ˈvɛriəns/
- 구성: `vary` + `-ance`
- 뜻: 차이, 불일치, 분산
- 어원적 연결: 서로 달라진 정도
- 사용 맥락: `at variance with the facts` = 사실과 어긋나는

## 기억 포인트

`vari` 계열은 "여러 모습으로 달라짐" 이미지.

- `various`: 여러 가지의
- `vary`: 달라지다
- `variety`: 다양한 종류
- `variation`: 변형, 차이
- `variable`: 변할 수 있는
- `invariably`: 변함없이, 항상

## 비교 단어

### diverse

- 뜻: 다양한, 서로 다른
- 구분 포인트: [[divers]]는 여러 방향으로 갈라진 이미지. `various`는 여러 종류가 있다는 느낌이 더 일반적

### alter

- 뜻: 바꾸다
- 구분 포인트: [[alter]]는 다른 상태로 바꿈. `vary`는 조건에 따라 달라지거나 서로 차이가 남
