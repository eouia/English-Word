# lex

- 출발 단어: `lexicon`
- 핵심 어원: Greek `lexis` = 말, 단어, 표현
- 관련 형태: Greek `legein` = 말하다, 고르다
- 핵심 이미지: 한 언어의 단어와 표현 목록
- 관련 문서: [[logos]], [[dict]], [[verb]]
- 주의: Latin `lex` = law 계열과 철자가 같지만, 이 문서는 Greek `lexis` = word 계열.

## 출발 단어 분석

`lexicon` ← Greek `lexikon`

- `lexis`: word, speech
- `lexikon`: wordbook
- 원래 이미지: 단어를 모아 둔 책
- 현대 의미: 어휘 목록, 어휘 체계

## 단어들

### lexicon

- 구성: Greek `lexikon`
- 뜻: 어휘 목록, 사전, 전문 용어 체계
- 어원적 연결: 단어를 모아 둔 목록
- 사용 맥락: `the political lexicon` = 정치 어휘 체계

### lexical

- 구성: `lexicon` + `-al`
- 뜻: 어휘의, 단어의
- 어원적 연결: 문법보다 단어 자체와 관련됨
- 사용 맥락: `lexical meaning` = 어휘적 의미

### lexicography

- 구성: `lexico-` = word + `-graphy` = writing
- 뜻: 사전 편찬
- 어원적 연결: 단어를 기록하고 설명하는 일
- 사용 맥락: `modern lexicography` = 현대 사전 편찬

### lexicographer

- 구성: `lexicography` + `-er`
- 뜻: 사전 편찬자
- 어원적 연결: 단어 목록을 정리하고 설명하는 사람
- 사용 맥락: `a noted lexicographer` = 저명한 사전 편찬자

### lexicology

- 구성: `lexico-` = word + [[logos]] = study
- 뜻: 어휘론
- 어원적 연결: 단어와 어휘 체계를 연구함
- 사용 맥락: 언어학 맥락에서 쓰임

### dyslexia

- 구성: `dys-` = difficult + Greek `lexis` = word
- 뜻: 난독증
- 어원적 연결: 단어를 읽고 처리하는 데 어려움이 있음
- 사용 맥락: `students with dyslexia` = 난독증이 있는 학생들

## 기억 포인트

`lex` 계열은 "단어 목록과 어휘" 이미지.

- `lexicon`: 어휘 목록
- `lexical`: 어휘의
- `lexicography`: 사전 편찬
- `dyslexia`: 단어 읽기 어려움

## 비교 단어

### logos

- 뜻: 말, 설명, 이성
- 구분 포인트: [[logos]]는 말이 이성·논리·학문으로 확장. `lex`는 단어 목록과 어휘 체계에 초점.

### dict

- 뜻: 말하다, 선언하다
- 구분 포인트: [[dict]]는 말하는 행위. `lex`는 말의 재료인 단어 목록.
