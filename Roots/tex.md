# tex

- 출발 단어: `text`
- 핵심 어원: 라틴어 `texere` = 짜다, 엮다
- 핵심 의미: 실·말·상황을 엮어 만든 구조
- 관련 문서: [[liter]], [[con]], [[struct]]
- 주의: `text`는 원래 “짜인 것”이라는 이미지. 글은 말이 엮인 직물처럼 이해.

## 출발 단어 분석

`text` ← Latin `textus` ← `texere`

- `texere`: 짜다, 엮다
- `textus`: 짜인 것, 엮인 구조
- 원래 이미지: 실을 엮어 천을 만들듯 말과 문장을 엮음
- 현대 의미: 글, 본문, 문자 메시지

## 단어들

### text

- 구성: Latin `textus`
- 뜻: 글, 본문, 문자 메시지
- 어원적 연결: 말과 문장이 엮여 만들어진 것
- 사용 맥락: `read the text carefully` = 본문을 주의 깊게 읽다

### textual

- 구성: `text` + `-ual`
- 뜻: 본문의, 텍스트상의
- 어원적 연결: 엮인 글 자체와 관련됨
- 사용 맥락: `textual evidence` = 본문상의 증거

### context

- 구성: `con-` = together + `tex`
- 뜻: 문맥, 상황
- 어원적 연결: 말이나 사건이 함께 엮인 배경
- 사용 맥락: `understand the context` = 문맥을 이해하다

### contextual

- 구성: `context` + `-ual`
- 뜻: 문맥상의, 상황적인
- 어원적 연결: 함께 엮인 주변 조건과 관련됨
- 사용 맥락: `contextual clues` = 문맥 단서

### texture

- 구성: Latin `textura` = 짜임
- 뜻: 질감, 조직, 짜임새
- 어원적 연결: 실이나 요소가 엮여 생긴 표면과 구조
- 사용 맥락: `the texture of fabric` = 천의 질감

### textile

- 구성: Latin `textilis` = 짜서 만든
- 뜻: 직물, 섬유 제품
- 어원적 연결: 실을 엮어 만든 물건
- 사용 맥락: `the textile industry` = 섬유 산업

### subtext

- 구성: `sub-` = under + `text`
- 뜻: 숨은 뜻, 속뜻
- 어원적 연결: 겉으로 드러난 말 아래에 엮인 의미
- 사용 맥락: `the political subtext` = 정치적 속뜻

### pretext

- 구성: `pre-` = before + `tex`
- 뜻: 구실, 핑계
- 어원적 연결: 앞에 짜서 내세운 말
- 사용 맥락: `use safety as a pretext` = 안전을 구실로 삼다

### hypertext

- 구성: `hyper-` = beyond + `text`
- 뜻: 하이퍼텍스트
- 어원적 연결: 링크로 서로 엮여 확장되는 글
- 사용 맥락: `hypertext links` = 하이퍼텍스트 링크

### intertextual

- 구성: `inter-` = between + `textual`
- 뜻: 상호텍스트적인
- 어원적 연결: 여러 텍스트가 서로 엮여 의미를 만듦
- 사용 맥락: `intertextual references` = 상호텍스트적 참조

## 기억 포인트

- `tex`는 “엮다, 짜다”.
- `text`: 말이 엮인 것.
- `context`: 함께 엮인 주변 말과 상황.
- `texture/textile`: 실제 실이나 요소의 짜임.
- `pretext`: 앞에 짜서 내세운 핑계.

## 비교 단어

### liter

- [[liter]]는 글자, 글, 문학 계열.
- `text`는 글자의 존재보다 “말과 문장이 엮인 구조”에 초점.

### weave

- `weave`는 게르만계 기본어로 “짜다, 엮다”.
- 의미 이미지는 매우 가깝지만 `texere` 직접 계열은 아님.

### context vs content

- `context`: 함께 엮인 주변 배경.
- `content`: 안에 담긴 내용. 라틴어 `continere` = 함께 붙잡아 담다 계열.
