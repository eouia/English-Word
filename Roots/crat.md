# crat

- 출발 단어: `democracy`
- 핵심 어원: 그리스어 `kratos` = power, rule
- 핵심 이미지: 권력을 가지고 지배함
- 관련해서 헷갈릴 수 있는 어근: [[dem]] = 사람들, 민중 계열, [[arch]] = 지배자, 시작 계열

## 출발 단어 분석

`-cracy`와 `-crat`는 Greek `kratos`에서 온 형태.

- `-cracy`: 지배 체제
- `-crat`: 지배자, 그 체제를 지지하는 사람

따라서 `democracy`는 "people-rule", `democrat`은 "people-rule을 지지하는 사람"의 구조.

## 단어들

### democracy

- 구성: [[dem]] = people + `-cracy` = rule
- 뜻: 민주주의
- 어원적 연결: 민중이 권력을 가지는 체제
- 사용 맥락: `liberal democracy` = 자유 민주주의

### democrat

- 구성: [[dem]] = people + `-crat` = ruler/supporter
- 뜻: 민주주의자, 민주당원
- 어원적 연결: 민중 지배를 지지하는 사람
- 사용 맥락: `a democrat by conviction` = 신념상 민주주의자

### aristocracy

- 구성: `aristo-` = best, noble + `-cracy`
- 뜻: 귀족정, 귀족 계급
- 어원적 연결: "최고층"이 지배하는 체제
- 사용 맥락: `the old aristocracy` = 옛 귀족 계층

### aristocrat

- 구성: `aristo-` + `-crat`
- 뜻: 귀족
- 어원적 연결: 귀족 지배층의 사람
- 사용 맥락: `a wealthy aristocrat` = 부유한 귀족

### autocracy

- 구성: `auto-` = self + `-cracy`
- 뜻: 독재 정치
- 어원적 연결: 한 사람이 스스로 권력을 가짐
- 사용 맥락: `military autocracy` = 군사 독재

### autocrat

- 구성: `auto-` + `-crat`
- 뜻: 독재자
- 어원적 연결: 스스로 절대 권력을 쥔 사람
- 사용 맥락: `a ruthless autocrat` = 무자비한 독재자

### bureaucracy

- 구성: French `bureau` = office + `-cracy`
- 뜻: 관료제
- 어원적 연결: 사무실/관료 조직이 지배하는 체계
- 사용 맥락: `government bureaucracy` = 정부 관료제

### bureaucrat

- 구성: `bureau` + `-crat`
- 뜻: 관료
- 어원적 연결: 관료제 안에서 권한을 행사하는 사람
- 사용 맥락: `a senior bureaucrat` = 고위 관료

### meritocracy

- 구성: `merit` = worth + `-cracy`
- 뜻: 능력주의
- 어원적 연결: 능력 있는 사람이 권력을 얻는 체제
- 사용 맥락: `academic meritocracy` = 학문적 능력주의

### technocracy

- 구성: `techno-` = skill, technology + `-cracy`
- 뜻: 기술관료제
- 어원적 연결: 전문 기술자들이 지배하는 체제
- 사용 맥락: `a technocracy of experts` = 전문가 기술관료제

### theocracy

- 구성: `theo-` = god + `-cracy`
- 뜻: 신정 정치
- 어원적 연결: 신의 권위로 지배하는 체제
- 사용 맥락: `a theocracy` = 신정국가

## 기억 포인트

`crat` 계열은 "권력 / 지배 체제" 이미지.

- `democracy`: 민중의 지배
- `aristocracy`: 귀족의 지배
- `autocracy`: 한 사람의 지배
- `bureaucracy`: 관료 조직의 지배
- `meritocracy`: 능력 기준의 지배
- `theocracy`: 신권 지배

## 비교 단어

### arch

- 뜻: 지배자, 우두머리, 시작
- 구분 포인트: [[arch]]는 `monarch`, `anarchy`, `hierarchy` 같은 계열. `crat`은 권력이나 지배 체제에 초점

### dem

- 뜻: 사람들, 민중
- 구분 포인트: [[dem]]은 사람들 자체. `crat`은 그 사람들이 권력을 가지는 구조
