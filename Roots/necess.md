# necess

- 출발 단어: `necessitate`
- 핵심 어원: Latin `necesse` = 피할 수 없는, 반드시 필요한
- 분석: `ne-` = not + `cedere` = withdraw, go away, yield
- 핵심 의미: 물러날 수 없음, 피할 수 없음, 반드시 필요함
- 관련 문서: [[cess]], [[quest]], [[ple]]
- 주의: `necessary`는 겉으로 `cess`가 보이지만 현대 영어에서는 `necess-` 덩어리로 기억하는 편이 안전.

## 출발 단어 분석

`necessitate` ← Medieval Latin `necessitare` ← Latin `necessitas`

- `necesse`: unavoidable, indispensable
- 원래 이미지: 뒤로 물러나 피할 수 없음
- 현대 의미: 필요하게 만들다, 불가피하게 하다

## 단어들

### necessitate

- 구성: `necessity` 계열 + `-ate`
- 뜻: 필요하게 만들다, 불가피하게 하다
- 어원적 연결: 어떤 상황이 물러날 수 없는 요구를 만듦
- 사용 맥락: `injuries may necessitate surgery` = 부상이 수술을 필요하게 할 수 있음

### necessitation

- 구성: `necessitate` + `-ion`
- 뜻: 필요하게 만듦, 불가피하게 함
- 어원적 연결: 어떤 일을 피할 수 없게 만드는 행위
- 사용 맥락: formal / philosophical context

### necessary

- 구성: Latin `necessarius` ← `necesse`
- 뜻: 필요한, 필수적인, 불가피한
- 어원적 연결: 피하거나 생략할 수 없음
- 사용 맥락: `necessary information` = 필요한 정보

### necessarily

- 구성: `necessary` + `-ly`
- 뜻: 반드시, 필연적으로
- 어원적 연결: 달리 될 수 없이
- 사용 맥락: `not necessarily true` = 반드시 사실은 아닌

### necessity

- 구성: Latin `necessitas`
- 뜻: 필요, 필수품, 필연성
- 어원적 연결: 피할 수 없는 요구나 조건
- 사용 맥락: `a matter of necessity` = 필요의 문제

### necessities

- 구성: `necessity`의 복수형
- 뜻: 필수품, 생필품
- 어원적 연결: 없어서는 안 되는 것들
- 사용 맥락: `basic necessities` = 기본 생필품

### necessitous

- 구성: `necessity` 계열 + `-ous`
- 뜻: 가난한, 궁핍한; 필요한
- 어원적 연결: 필수품이 부족해 필요에 몰린 상태
- 사용 맥락: formal / literary context

### unnecessary

- 구성: `un-` + `necessary`
- 뜻: 불필요한
- 어원적 연결: 피할 수 없는 요구가 아님
- 사용 맥락: `unnecessary delays` = 불필요한 지연

### necessarian

- 구성: `necessary` 계열 + `-an`
- 뜻: 필연론자
- 어원적 연결: 모든 일이 필연적으로 일어난다고 보는 사람
- 사용 맥락: philosophy context

### necessitarian

- 구성: `necessity` 계열 + `-arian`
- 뜻: 필연론자, 필연론의
- 어원적 연결: 자유보다 필연성을 강조하는 입장
- 사용 맥락: philosophical determinism context

## 기억 포인트

- `necess-`: 물러날 수 없음.
- `necessary`: 피할 수 없이 필요한.
- `necessity`: 피할 수 없는 필요.
- `necessitate`: 필요하게 만들다.
- `necessarily`: 반드시, 필연적으로.

## 비교 단어

### cess

- [[cess]]는 가다, 물러나다, 양보하다.
- `necessary`의 깊은 분석은 `ne-` + `cedere` = 물러날 수 없음.
- 다만 현대 단어 학습에서는 `necess-`를 독립 덩어리로 기억.

### require

- [[quest]]의 `require`는 필요한 것을 요구함.
- `necessitate`는 상황이 어떤 조치를 불가피하게 만듦.

### sufficient

- [[fac]]의 `sufficient`는 충분한, 모자라지 않음.
- `necessary`는 반드시 있어야 함.
- 시험 논리에서는 `necessary condition` = 필요조건, `sufficient condition` = 충분조건.
