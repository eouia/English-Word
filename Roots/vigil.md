# vigil

- 출발 단어: `vigilance`
- 핵심 어원: Latin `vigil` = awake, watchful
- 핵심 이미지: 깨어서 지켜봄
- 관련 문서: 없음
- 주의: `vigilante`는 법적 권한 없이 스스로 단속·응징하는 사람. 단순히 "경계하는 사람"보다 강한 사회적 의미가 있음.

## 출발 단어 분석

`vigilance` = `vigilant` 계열 + `-ance`

- `vigil`: 깨어 있음, 밤샘 감시
- `vigilant`: 경계하는, 방심하지 않는
- `-ance`: 상태나 성질을 나타내는 명사 접미사
- `vigilance`: 경계, 방심하지 않음

Latin `vigil`은 "깨어 있는"의 뜻. 잠들지 않고 지켜보는 이미지에서 "주의 깊은 감시, 경계심"으로 발전.

## 단어들

### vigilance

- 발음: /ˈvɪdʒələns/
- 구성: Latin `vigil` = awake, watchful + `-ance`
- 뜻: 경계, 방심하지 않음
- 어원적 연결: 잠들지 않고 계속 살펴봄
- 사용 맥락: `constant vigilance` = 끊임없는 경계

### vigilant

- 발음: /ˈvɪdʒələnt/
- 구성: Latin `vigilans` = watchful
- 뜻: 경계하는, 방심하지 않는
- 어원적 연결: 깨어서 위험이나 변화를 살핌
- 사용 맥락: `remain vigilant` = 경계를 늦추지 않다

### vigilantly

- 구성: `vigilant` + `-ly`
- 뜻: 경계하여, 주의 깊게
- 어원적 연결: 잠들지 않고 살피는 방식
- 사용 맥락: `monitor vigilantly` = 주의 깊게 감시하다

### vigil

- 발음: /ˈvɪdʒəl/
- 구성: Latin `vigil` = awake
- 뜻: 철야, 밤샘 기도, 밤샘 감시
- 어원적 연결: 밤에도 깨어 지켜보거나 기도함
- 사용 맥락: `hold a candlelight vigil` = 촛불 추모 집회를 열다

### vigilante

- 발음: /ˌvɪdʒəˈlænti/
- 구성: Spanish `vigilante` = watchman, guard ← Latin `vigil`
- 뜻: 자경단원, 사적 제재자
- 어원적 연결: 스스로 감시하고 단속하는 사람
- 사용 맥락: `vigilante justice` = 사적 제재

### hypervigilance

- 구성: `hyper-` = over, excessive + `vigilance`
- 뜻: 과각성, 과도한 경계
- 어원적 연결: 위험을 지나치게 계속 살피는 상태
- 사용 맥락: trauma, anxiety 문맥에서 자주 보임.

## 기억 포인트

`vigil` 계열은 "깨어서 지켜봄" 이미지.

- `vigilance`: 경계, 방심하지 않음
- `vigilant`: 경계하는
- `vigil`: 밤샘 감시나 기도
- `vigilante`: 자경단원, 사적 제재자
- `hypervigilance`: 과도한 경계

## 비교 단어

### alert

- 뜻: 경계하는, 기민한
- 구분 포인트: `alert`는 즉각 반응할 준비가 된 상태. `vigilant`는 계속 깨어 지켜보는 지속적 경계.

### watchful

- 뜻: 주의 깊게 살피는
- 구분 포인트: `watchful`은 일상적인 주시. `vigilant`는 위험을 막기 위한 더 긴장된 경계.

### wary

- 뜻: 조심하는, 경계하는
- 구분 포인트: `wary`는 의심과 조심스러움. `vigilant`는 감시와 주의 유지에 초점.
