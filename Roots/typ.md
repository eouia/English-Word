# typ

- 출발 단어: `type`
- 핵심 어원: Greek `typos` = 찍힌 자국, 본보기, 모양
- 핵심 이미지: 눌러 찍힌 모양에서 종류와 전형으로 확장
- 관련 문서: [[arch]], [[form]], [[morph]], [[gram]]
- 주의: `type`은 "종류"와 "글자 활자"가 모두 같은 `typos` 이미지에서 확장됨.

## 출발 단어 분석

`type` ← Greek `typos`

- `typos`: blow, mark, impression, model
- 원래 이미지: 찍혀 남은 자국이나 본보기
- 현대 의미: 종류, 유형, 활자, 입력하다

## 단어들

### type

- 발음: /ˈtaɪp/
- 구성: Greek `typos`
- 뜻: 종류, 유형, 활자, 입력하다
- 어원적 연결: 찍힌 모양이나 본보기에서 같은 종류를 구분함
- 사용 맥락: `a type of problem` = 문제의 한 유형

### typical

- 발음: /ˈtɪpəkəl/
- 구성: `type` + `-ical`
- 뜻: 전형적인, 대표적인
- 어원적 연결: 어떤 유형의 본보기에 맞음
- 사용 맥락: `a typical example` = 전형적인 예

### typically

- 발음: /ˈtɪpɪkli/
- 구성: `typical` + `-ly`
- 뜻: 보통, 전형적으로
- 어원적 연결: 어떤 유형에서 흔히 보이는 방식
- 사용 맥락: `typically occurs in winter` = 보통 겨울에 발생함

### prototype

- 발음: /ˈproʊtəˌtaɪp/
- 구성: `proto-` = first + `type`
- 뜻: 원형, 시제품
- 어원적 연결: 처음 만든 본보기
- 사용 맥락: `build a prototype` = 시제품을 만들다

### archetype

- 발음: /ˈɑrkɪˌtaɪp/
- 구성: [[arch]] = original, first + `type`
- 뜻: 원형, 전형
- 어원적 연결: 처음 되는 본보기
- 사용 맥락: `a hero archetype` = 영웅 원형

### stereotype

- 발음: /ˈstɛriəˌtaɪp/
- 구성: Greek `stereos` = solid + `type`
- 뜻: 고정관념, 정형화하다
- 어원적 연결: 원래는 단단한 인쇄판. 굳어진 유형이라는 뜻으로 확장
- 사용 맥락: `challenge stereotypes` = 고정관념에 도전하다

### typography

- 발음: /təˈpɑgrəfi/
- 구성: `type` + [[graph]] = writing
- 뜻: 타이포그래피, 활자 배열
- 어원적 연결: 찍힌 글자 모양을 다루는 기술
- 사용 맥락: `digital typography` = 디지털 타이포그래피

### typology

- 발음: /ˌtaɪˈpoʊləˌgɪ/
- 구성: `type` + [[logos]] = study
- 뜻: 유형학, 유형 분류
- 어원적 연결: 종류와 유형을 체계적으로 분류함
- 사용 맥락: `linguistic typology` = 언어 유형론

### typewriter

- 발음: /ˈtaɪˌpraɪtɚ/
- 구성: `type` + `writer`
- 뜻: 타자기
- 어원적 연결: 활자를 찍어 글을 쓰는 기계
- 사용 맥락: `an old typewriter` = 오래된 타자기

## 기억 포인트

`typ` 계열은 "찍힌 모양과 본보기" 이미지.

- `type`: 종류, 활자
- `typical`: 유형에 맞는 전형적 모습
- `prototype`: 처음 본보기
- `archetype`: 원형
- `stereotype`: 굳어진 유형
- `typography`: 활자 모양을 다루는 기술
- `typology`: 유형을 분류하는 학문
