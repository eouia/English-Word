# imag

- 출발 단어: `imaginative`
- 핵심 어원: Latin `imago/imaginis` = image, likeness
- 핵심 이미지: 마음속에 그림이나 닮은 모습을 떠올림
- 관련 문서: [[imit]], [[vid]]
- 주의: `imitate`는 [[imit]]의 "따라 하다" 계열. `image`와 닮음의 이미지가 겹치지만 어근 문서는 분리.

## 출발 단어 분석

`imaginative` = `imagine` + `-ative`

- `imagine`: 마음속에 그림을 만들다
- `-ative`: 성질·능력을 나타내는 접미사
- 원래 이미지: 마음속 이미지를 잘 만들어 냄
- 현대 의미: 상상력이 풍부한, 창의적인

## 단어들

### imaginative

- 발음: /ˌɪˈmædʒənətɪv/
- 구성: `imagine` + `-ative`
- 뜻: 상상력이 풍부한, 창의적인
- 어원적 연결: 마음속에 새로운 이미지를 잘 만들어 냄
- 사용 맥락: `an imaginative solution` = 창의적인 해결책, `an imaginative child` = 상상력이 풍부한 아이

### imagine

- 발음: /ˌɪˈmædʒən/
- 구성: Old French `imaginer` ← Latin `imaginari` < `imago`
- 뜻: 상상하다, 마음속에 그리다
- 어원적 연결: 실제로 보이지 않는 것을 마음속 이미지로 만듦
- 사용 맥락: `imagine the future` = 미래를 상상하다

### imagination

- 발음: /ˌɪˌmædʒəˈneɪʃən/
- 구성: `imagine` + `-ation`
- 뜻: 상상력, 상상
- 어원적 연결: 마음속 이미지를 만들어 내는 능력이나 작용
- 사용 맥락: `use your imagination` = 상상력을 발휘하다

### imaginary

- 발음: /ˌɪˈmædʒəˌnɛri/
- 구성: `imagine` + `-ary`
- 뜻: 상상의, 가상의
- 어원적 연결: 현실에 직접 있는 것이 아니라 마음속 이미지로 존재함
- 사용 맥락: `an imaginary friend` = 상상 속 친구

### image

- 발음: /ˈɪmədʒ/
- 구성: Old French `image` ← Latin `imago`
- 뜻: 이미지, 형상, 모습
- 어원적 연결: 어떤 대상을 닮은 보이는 형태
- 사용 맥락: `public image` = 대중적 이미지, `digital image` = 디지털 이미지

### imagery

- 발음: /ˈɪmədʒri/
- 구성: `image` + `-ery`
- 뜻: 심상, 이미지 표현
- 어원적 연결: 글·예술 속에서 떠오르게 하는 그림 같은 표현
- 사용 맥락: `visual imagery` = 시각적 심상

### imaginable

- 발음: /ˌɪˈmædʒənəbəl/
- 구성: `imagine` + `-able`
- 뜻: 상상할 수 있는
- 어원적 연결: 마음속 그림으로 만들 수 있음
- 사용 맥락: `every imaginable method` = 상상 가능한 모든 방법

### unimaginable

- 발음: /ˌʌnɪˈmædʒɪnəbəl/
- 구성: `un-` = not + `imaginable`
- 뜻: 상상할 수 없는
- 어원적 연결: 마음속 이미지로도 그리기 어려움
- 사용 맥락: `unimaginable pain` = 상상할 수 없는 고통

## 기억 포인트

`imag`는 "마음속 이미지" 계열.

- `image`: 보이는 형상
- `imagine`: 마음속에 그림을 만듦
- `imagination`: 이미지를 만드는 능력
- `imaginative`: 상상력 있게 이미지를 만듦
- `imaginary`: 마음속에만 있는
- `imagery`: 글이나 예술이 떠올리게 하는 이미지

## 비교 단어

### imitate

- 뜻: 모방하다, 흉내 내다
- 구분 포인트: [[imit]] 계열. 원본과 닮게 따라 하는 행동에 초점.

### visual

- 뜻: 시각의, 보이는
- 구분 포인트: [[vid]] 계열. 실제로 보는 행위와 시각 정보에 초점.
