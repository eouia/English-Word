# som

- 출발 단어: `psychosomatic`
- 핵심 어원: Greek `soma` = 몸
- 핵심 이미지: 살아 있는 몸, 신체
- 관련 문서: [[psych]], [[corp]], [[gen]]
- 주의: [[corp]]는 Latin `corpus` = 몸. `som`은 Greek `soma` = 몸.

## 출발 단어 분석

`psychosomatic` ← Greek `psyche` + `soma`

- `psyche`: mind
- `soma`: body
- 원래 이미지: 마음과 몸이 연결됨
- 현대 의미: 심신의, 정신 신체의

## 단어들

### somatic

- 구성: Greek `soma` + `-atic`
- 뜻: 신체의, 체세포의
- 어원적 연결: 마음이나 생식세포가 아니라 몸 자체에 관련됨
- 사용 맥락: `somatic symptoms` = 신체 증상

### psychosomatic

- 구성: [[psych]] = mind + `soma` = body
- 뜻: 심신의, 정신 신체의
- 어원적 연결: 마음 상태가 몸의 증상과 연결됨
- 사용 맥락: `psychosomatic illness` = 심신성 질환

### chromosome

- 구성: Greek `chroma` = color + `soma` = body
- 뜻: 염색체
- 어원적 연결: 염색되는 작은 몸체
- 사용 맥락: `human chromosomes` = 인간 염색체

### somatotype

- 구성: `soma` = body + `type`
- 뜻: 체형
- 어원적 연결: 몸의 유형
- 사용 맥락: 운동·체형 분류 맥락에서 쓰임

### somatization

- 구성: `soma` + `-ization`
- 뜻: 신체화
- 어원적 연결: 심리적 문제가 몸의 증상으로 나타남
- 사용 맥락: `somatization disorder` = 신체화 장애

## 기억 포인트

`som` 계열은 "몸" 이미지.

- `somatic`: 몸의
- `psychosomatic`: 마음과 몸의 연결
- `chromosome`: 염색되는 몸체
- `somatization`: 마음 문제가 몸으로 나타남

## 비교 단어

### corp

- 뜻: 몸, 한 덩어리
- 구분 포인트: [[corp]]는 라틴어 몸 계열. 회사·조직으로도 확장. `som`은 그리스어 몸 계열로 의학·생물학 단어에 자주 보임.

### psych

- 뜻: 마음, 정신
- 구분 포인트: [[psych]]는 마음. `som`은 몸. `psychosomatic`에서 둘이 만남.
