# tabul

- 출발 단어: `table`
- 핵심 어원: 라틴어 `tabula` = 판, 널빤지, 기록판, 표
- 핵심 의미: 평평한 판 위에 놓거나 적은 것
- 관련 문서: [[scrib]], [[stat]]
- 주의: `stable`, `tractable`, `portable`의 `-able`은 접미사. `table` 계열이 아님.

## 출발 단어 분석

`table` ← 라틴어 `tabula`

- 원래 이미지: 평평한 널빤지, 글을 적는 판
- 의미 확장: 판 → 탁자 → 판에 정리한 목록/표
- 현대 핵심: 물건을 올려놓는 판, 정보를 배열한 표

## 단어들

### table

- 발음: /ˈteɪbəl/
- 구성: 라틴어 `tabula`
- 뜻: 탁자, 표
- 어원적 연결: 평평한 판 위에 놓거나 적은 것
- 사용 맥락: `a dining table` = 식탁, `a data table` = 데이터 표

### tablet

- 발음: /ˈtæblət/
- 구성: `table` + `-et`
- 뜻: 작은 판, 알약, 태블릿 기기
- 어원적 연결: 작은 판 모양의 것
- 사용 맥락: `a tablet computer` = 태블릿 컴퓨터

### tabulate

- 발음: /ˈtæbjəˌleɪt/
- 구성: `tabul` + `-ate`
- 뜻: 표로 만들다, 표에 정리하다
- 어원적 연결: 정보를 판 위의 칸처럼 배열함
- 사용 맥락: `tabulate the results` = 결과를 표로 정리하다

### tabulation

- 발음: /ˌtæbjəˈleɪʃən/
- 구성: `tabulate` + `-ion`
- 뜻: 표 작성, 표로 된 정리
- 어원적 연결: 데이터를 표 형태로 배열한 결과
- 사용 맥락: `data tabulation` = 데이터 표 작성

### tabular

- 구성: `tabul` + `-ar`
- 뜻: 표 형식의, 표로 된
- 어원적 연결: 칸을 나누어 판 위에 배열한 형태
- 사용 맥락: `tabular form` = 표 형식

### tableau

- 발음: /təˈbloʊ/
- 구성: 프랑스어 `tableau` ← `table`
- 뜻: 장면, 그림 같은 배열, 표
- 어원적 연결: 판 위에 펼쳐 보인 장면
- 사용 맥락: `a dramatic tableau` = 극적인 장면

### tab

- 발음: /ˈtæb/
- 구성: `tablet` 또는 작은 판 조각 이미지와 관련
- 뜻: 작은 꼬리표, 탭, 계산서
- 어원적 연결: 튀어나온 작은 표식 또는 기록 조각
- 사용 맥락: `open a new tab` = 새 탭을 열다

## 기억 포인트

- `tabul`의 기본 그림은 “평평한 판”.
- `table`은 탁자와 표가 같은 뿌리에서 갈라진 단어.
- `tabulate`, `tabular`는 시험 지문에서 “자료를 표로 정리하다/표 형식의”로 자주 등장.

## 비교 단어

### -able

- `stable`, `tractable`, `portable`의 끝 `-able`은 “~할 수 있는” 접미사.
- `table`과 철자가 겹칠 뿐 어근이 다름.

### tablet

- `tablet`은 같은 계열이지만 현대에서는 알약, 기기 뜻이 강함.
- 원래 “작은 판” 이미지에서 확장.
