# future

- 출발 단어: `future`
- 핵심 의미: 앞으로 있을 것, 아직 오지 않은 시간
- 관련 형태: Latin `futurus` = going to be, about to be
- 관련 문서: [[ante]]

## 핵심 이미지

`future` 계열의 바탕은 "앞으로 있게 될 것".

아직 오지 않았지만 앞으로 있을 시간은 `future`.
미래와 관련된 성질은 `futuristic`.
미래 지향적 예술·사상은 `futurism`.
미래를 연구하거나 강조하는 사람은 `futurist`.

## 단어들

### future

- 발음: /ˈfjutʃɚ/
- 구성: Latin `futurus` = going to be, about to be
- 뜻: 미래, 장래; 미래의
- 연결: 아직 존재하지 않지만 앞으로 있게 될 것
- 사용 맥락: `plan for the future` = 미래를 계획하다

### futurity

- 구성: `future` + `-ity`
- 뜻: 미래성, 장래; 후세
- 연결: 앞으로 있을 시간이나 상태
- 사용 맥락: 문학적·철학적 표현. 일반 시험 어휘에서는 낮은 우선순위.

### futuristic

- 발음: /ˌfjutʃɚˈɪstɪk/
- 구성: `future` + `-istic`
- 뜻: 미래적인, 미래 지향적인
- 연결: 앞으로 올 시대의 기술이나 모습처럼 보임
- 사용 맥락: `a futuristic design` = 미래적인 디자인

### futurism

- 발음: /ˈfjutʃɚˌɪzəm/
- 구성: `future` + `-ism`
- 뜻: 미래주의
- 연결: 과거보다 미래, 속도, 기술, 변화를 강조하는 사상·예술 운동
- 사용 맥락: art, design, technology 문맥.

### futurist

- 발음: /ˈfjutʃɚɪst/
- 구성: `future` + `-ist`
- 뜻: 미래학자, 미래주의자
- 연결: 미래의 변화와 가능성을 연구하거나 주장하는 사람
- 사용 맥락: `a technology futurist` = 기술 미래학자

## 비교 단어

`future`의 Latin `futurus`는 깊게 보면 "be"와 관련된 미래분사 형태.
다만 현대 영어 학습에서는 `future` 자체를 기본 단어로 익히는 편이 더 실용적.

`ante`는 [[ante]] 문서처럼 "앞에, 이전에"라는 시간·위치 이미지.
`future`는 "앞으로 있을 시간"이라 시간 방향은 비슷하지만 어원은 다름.
