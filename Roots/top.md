# top

- 출발 단어: `topic`
- 핵심 의미: 장소, 자리; 논의가 놓이는 자리
- 관련 형태: Greek `topos` = place
- 관련 문서: [[loc]], [[graph]]

## 핵심 이미지

`top` 계열의 바탕은 "무언가가 놓이는 자리".

논의가 놓이는 자리는 `topic`.
그 주제와 관련되면 `topical`.
장소를 그려 적으면 `topography`.
어디에도 없는 장소는 `utopia`.
나쁜 장소로 상상된 세계는 `dystopia`.

## 단어들

### topic

- 발음: /ˈtɑpɪk/
- 구성: Greek `topos` = place
- 뜻: 주제, 화제
- 연결: 말이나 글이 놓이는 자리
- 사용 맥락: `choose a topic` = 주제를 고르다

### topical

- 발음: /ˈtɑpəkəl/
- 구성: `topic` + `-al`
- 뜻: 주제와 관련된; 시사적인; 국소적인
- 연결: 특정 주제나 특정 부위에 놓임
- 사용 맥락: `a topical issue` = 시사적인 쟁점, `topical cream` = 바르는 크림

### topography

- 발음: /təˈpɑgrəfi/
- 구성: Greek `topos` = place + `graphia` = writing, description
- 뜻: 지형, 지형 묘사
- 연결: 장소의 모양을 그리거나 기록함
- 사용 맥락: `mountain topography` = 산악 지형

### topographical

- 구성: `topography` + `-ical`
- 뜻: 지형의, 지형학적인
- 연결: 장소의 표면 형태와 관련됨
- 사용 맥락: `a topographical map` = 지형도

### topology

- 발음: /təˈpɔlədʒi/
- 구성: Greek `topos` = place + `-logy` = study
- 뜻: 위상수학, 배치 구조
- 연결: 위치와 연결 관계를 다루는 학문
- 사용 맥락: mathematics, network 문맥.

### topological

- 발음: /təpəˈlɑdʒɪkəl/
- 구성: `topology` + `-ical`
- 뜻: 위상수학의, 위치 관계의
- 연결: 정확한 거리보다 연결과 배치 관계가 중심
- 사용 맥락: `topological structure` = 위상적 구조

### utopia

- 발음: /juˈtoʊpiə/
- 구성: Greek `ou-` = not + `topos` = place
- 뜻: 유토피아, 이상향
- 연결: 현실에는 없는 완벽한 장소
- 사용 맥락: `a political utopia` = 정치적 이상향

### dystopia

- 구성: Greek `dys-` = bad, difficult + `topos` = place
- 뜻: 디스토피아, 암울한 가상 사회
- 연결: 나쁜 장소, 살기 어려운 사회
- 사용 맥락: `a dystopian novel` = 디스토피아 소설

## 비교 단어

`loc`는 [[loc]] 문서처럼 Latin `locus` = 장소, 자리 계열.
`top`은 Greek `topos` = 장소 계열로, 학술어와 추상어에 자주 보임.

`graph`는 [[graph]] 문서처럼 "쓰다, 그리다, 기록하다" 계열.
`topography`는 `topos` + `graph`가 결합한 말.
