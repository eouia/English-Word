# nat-birth

- 출발 단어: `native`
- 핵심 어원: Latin `nasci` = 태어나다
- 핵심 이미지: 태어난 상태, 타고난 성질, 새로 생겨남
- 허브 문서: [[nat]]

## 단어들

### native

- 구성: Latin `nativus` = born, inborn
- 뜻: 태어난 곳의, 원주민의; 타고난
- 어원적 연결: 어떤 장소나 성질 안에서 태어남
- 사용 맥락: `native language` = 모국어

### nonnative

- 구성: `non-` + `native`
- 뜻: 토착이 아닌, 원어민이 아닌
- 어원적 연결: 그곳에서 태어난 것이 아님
- 사용 맥락: `nonnative speakers` = 비원어민 화자들

### nativity

- 구성: Latin `nativitas` = birth
- 뜻: 탄생, 출생; 예수 탄생
- 어원적 연결: 태어나는 사건
- 사용 맥락: `the Nativity` = 예수 탄생

### natal

- 구성: Latin `natalis` = pertaining to birth
- 뜻: 출생의, 탄생의
- 어원적 연결: 태어남과 관련됨
- 사용 맥락: `natal care` = 출산 관련 돌봄

### prenatal

- 구성: `pre-` = before + `natal`
- 뜻: 출생 전의, 산전의
- 어원적 연결: 태어나기 전
- 사용 맥락: `prenatal care` = 산전 관리

### postnatal

- 구성: `post-` = after + `natal`
- 뜻: 출생 후의, 산후의
- 어원적 연결: 태어난 뒤
- 사용 맥락: `postnatal development` = 출생 후 발달

### innate

- 구성: `in-` = in + Latin `natus` = born
- 뜻: 타고난, 선천적인
- 어원적 연결: 태어날 때 안에 들어 있음
- 사용 맥락: `innate ability` = 타고난 능력

### connate

- 구성: `con-` = together + `nate`
- 뜻: 함께 태어난, 선천적인
- 어원적 연결: 같이 태어나거나 본래 함께 있음
- 사용 맥락: 전문·학술 맥락에서 사용

### nascent

- 구성: Latin `nascentem` = being born
- 뜻: 막 생겨나는, 초기의
- 어원적 연결: 지금 막 태어나고 있는 상태
- 사용 맥락: `a nascent industry` = 막 생겨나는 산업

### renaissance

- 구성: French `renaissance` = rebirth
- 뜻: 르네상스, 부흥, 재탄생
- 어원적 연결: 다시 태어남
- 사용 맥락: `a cultural renaissance` = 문화적 부흥

### nee

- 구성: French `nee` = born
- 뜻: 결혼 전 성이 ~인
- 어원적 연결: 태어날 때의 이름
- 사용 맥락: `Mary Smith, nee Jones` = 결혼 전 성이 Jones인 Mary Smith

## 기억 포인트

`nat-birth`는 "태어남과 타고남" 갈래.

- `native`: 그곳에서 태어난
- `natal`: 출생의
- `innate`: 타고난
- `nascent`: 막 생겨나는
- `renaissance`: 다시 태어남
