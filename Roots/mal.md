# mal

- 출발 단어: `malevolent`
- 핵심 어원: Latin `malus` / `male` = bad, badly
- 핵심 이미지: 나쁘게, 잘못, 해롭게
- 관련 문서: [[vol]], [[bene]]
- 주의: `mal-`은 접두사 성격이 강함. "나쁜 방향"을 더하지만, 뒤 어근의 의미를 함께 봐야 정확함.

## 출발 단어 분석

`malevolent` = `mal-` + `vol` + `-ent`

- `mal-`: bad, badly
- `volens`: wishing, willing
- 원래 이미지: 나쁜 일을 바라거나 해치려는 마음
- 현대 의미: 악의적인, 해치려는

## 단어들

### malevolent

- 구성: `mal-` = bad + Latin `volens` = wishing
- 뜻: 악의적인, 해치려는
- 어원적 연결: 나쁜 일이 일어나기를 바라는 마음
- 사용 맥락: `malevolent intent` = 악의적 의도
- 관련: [[vol]]

### malevolence

- 구성: `mal-` = bad + `vol` = will
- 뜻: 악의, 해치려는 마음
- 어원적 연결: 나쁜 뜻을 품은 마음
- 사용 맥락: `an expression of malevolence` = 악의의 표현
- 관련: [[vol]]

### malicious

- 구성: Latin `malitia` = badness, spite + `-ous`
- 뜻: 악의적인, 해치려는
- 어원적 연결: 나쁜 마음이나 해칠 의도를 드러냄
- 사용 맥락: `malicious rumors` = 악의적인 소문

### malice

- 구성: Latin `malitia` = badness, spite
- 뜻: 악의, 원한
- 어원적 연결: 나쁘게 해치려는 마음
- 사용 맥락: `without malice` = 악의 없이

### malfunction

- 구성: `mal-` = badly + `function`
- 뜻: 오작동하다; 오작동
- 어원적 연결: 기능이 나쁘게 작동함
- 사용 맥락: `equipment malfunction` = 장비 오작동

### malpractice

- 구성: `mal-` = bad + `practice`
- 뜻: 배임, 위법 행위, 의료 과실
- 어원적 연결: 직무나 전문 행위를 나쁘게 수행함
- 사용 맥락: `medical malpractice` = 의료 과실

### malnutrition

- 구성: `mal-` = badly + `nutrition`
- 뜻: 영양실조
- 어원적 연결: 영양 상태가 나쁘거나 잘못됨
- 사용 맥락: `child malnutrition` = 아동 영양실조

## 기억 포인트

`mal-`은 "나쁘게/잘못/해롭게" 방향을 더함.

- `malevolent`: 나쁜 뜻을 품은
- `malicious`: 악의적인
- `malice`: 악의
- `malfunction`: 오작동
- `malpractice`: 위법 행위, 과실
- `malnutrition`: 영양실조

## 비교 단어

### bene

- 뜻: 좋게, 이롭게
- 구분 포인트: [[bene]]는 `mal-`의 반대 방향. `benevolent` = 선의적인, `malevolent` = 악의적인.

### mis

- 뜻: 잘못, 틀리게
- 구분 포인트: `mis-`는 Germanic 계열 접두사로 "wrongly" 느낌. `mal-`은 Latin/French 계열로 공식어·전문어에 많이 남음.
