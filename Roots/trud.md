# trud

- 출발 단어: `unobtrusive`
- 핵심 어원: Latin `trudere` = push, thrust
- 핵심 이미지: 밀고 들어가거나 앞으로 튀어나옴
- 관련 문서: [[subtil]]
- 주의: 표준 철자는 `unobtrusive`. `unobstrusive`처럼 `str`을 넣지 않음.

## 출발 단어 분석

`unobtrusive` = `un-` + `obtrusive`

- `un-`: not
- `obtrusive`: 눈에 거슬리게 튀어나온, 주제넘게 끼어드는
- `unobtrusive`: 눈에 띄지 않는, 튀지 않는, 조심스러운

`obtrusive`는 Latin `obtrudere` = thrust upon, push against에서 온 말. 무엇을 앞에 밀어 넣어 눈에 띄게 만들거나 상대에게 강요하는 이미지.

## 단어들

### unobtrusive

- 발음: /ˌʌnəˈbtrusɪv/
- 구성: `un-` = not + `obtrusive`
- 뜻: 눈에 띄지 않는, 튀지 않는, 조심스러운
- 어원적 연결: 앞으로 밀고 나와 방해하지 않음
- 사용 맥락: `an unobtrusive design` = 튀지 않는 디자인, `unobtrusive background music` = 거슬리지 않는 배경 음악

### unobtrusively

- 구성: `unobtrusive` + `-ly`
- 뜻: 눈에 띄지 않게, 조심스럽게
- 어원적 연결: 자신을 앞으로 밀어 넣지 않는 방식
- 사용 맥락: `work unobtrusively` = 조용히 티 내지 않고 일하다

### unobtrusiveness

- 구성: `unobtrusive` + `-ness`
- 뜻: 눈에 띄지 않음, 조심스러움
- 어원적 연결: 튀어나와 방해하지 않는 성질
- 사용 맥락: `the unobtrusiveness of the interface` = 인터페이스의 눈에 거슬리지 않음

### obtrusive

- 발음: /əˈbtrusɪv/
- 구성: Latin `obtrudere` = thrust upon
- 뜻: 눈에 거슬리는, 주제넘게 끼어드는
- 어원적 연결: 앞으로 밀고 나와 시야나 상황을 방해함
- 사용 맥락: `obtrusive ads` = 눈에 거슬리는 광고

### obtrusively

- 구성: `obtrusive` + `-ly`
- 뜻: 눈에 거슬리게, 주제넘게
- 어원적 연결: 자신을 강하게 밀어 넣는 방식
- 사용 맥락: `speak obtrusively` = 주제넘게 말하다

### obtrude

- 발음: /əˈbtrud/
- 구성: `ob-` = against, toward + Latin `trudere` = push
- 뜻: 불쑥 끼어들다, 강요하다
- 어원적 연결: 상대 앞에 억지로 밀어 넣음
- 사용 맥락: `obtrude on someone's privacy` = 남의 사생활에 불쑥 끼어들다

### intrusive

- 발음: /ˌɪˈntrusɪv/
- 구성: Latin `intrudere` = push in
- 뜻: 침입하는, 간섭하는, 거슬리는
- 어원적 연결: 안으로 밀고 들어감
- 사용 맥락: `intrusive questions` = 사생활을 캐묻는 질문들

### intrusion

- 발음: /ˌɪˈntruʒən/
- 구성: `intrude` + `-ion`
- 뜻: 침입, 간섭, 끼어듦
- 어원적 연결: 안으로 밀고 들어가는 행위
- 사용 맥락: `an intrusion into privacy` = 사생활 침해

## 기억 포인트

`trud` 계열은 "밀고 들어감" 이미지.

- `obtrusive`: 앞에 밀고 나와 눈에 거슬리는
- `unobtrusive`: 앞으로 튀어나오지 않는, 조심스러운
- `obtrude`: 불쑥 끼어들다
- `intrusive`: 안으로 밀고 들어와 거슬리는
- `intrusion`: 침입, 간섭

## 비교 단어

### subtle

- 뜻: 미묘한, 섬세한
- 구분 포인트: [[subtil]]은 미세해서 바로 드러나지 않는 섬세함. `unobtrusive`는 일부러 앞으로 튀어나오지 않아 거슬리지 않는 느낌.

### modest

- 뜻: 겸손한, 소박한
- 구분 포인트: `modest`는 태도나 규모가 과하지 않음. `unobtrusive`는 존재하더라도 시야나 관심을 방해하지 않음.

### discreet

- 뜻: 신중한, 눈에 띄지 않는
- 구분 포인트: `discreet`은 말과 행동을 조심해 문제를 만들지 않음. `unobtrusive`는 물건·디자인·행동이 튀지 않는 느낌.

### intrusive

- 뜻: 침입하는, 간섭하는
- 구분 포인트: `intrusive`는 안으로 밀고 들어와 방해함. `unobtrusive`는 그 반대로 조용히 물러나 있음.
