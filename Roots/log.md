# log

- 출발 단어: `log`
- 핵심 어원: Old Norse `lag` / English `log` = 쓰러진 나무, 통나무
- 핵심 이미지: 통나무 조각으로 속도나 사건을 표시하고 기록함
- 관련해서 헷갈릴 수 있는 어근: [[logos]] = 말, 설명, 논리, 학문 계열

## 출발 단어 분석

영어 `log`는 Greek `logos`와 다른 계열.

- 원래 뜻: 통나무, 나무토막
- 항해에서의 변화: 줄에 매단 나무토막을 물에 던져 배의 속도를 잼
- 의미 변화: 속도 기록 장부 → 항해 기록 → 일반적인 기록

따라서 현대 `log`는 "통나무"와 "기록" 두 뜻이 함께 남아 있음.

## 단어들

### log

- 구성: English `log`
- 뜻: 통나무, 기록, 기록하다
- 어원적 연결: 통나무 조각으로 항해 속도를 재고 기록하던 데서 기록의 뜻이 생김
- 사용 맥락: `keep a log` = 기록을 남기다

### logbook

- 구성: `log` + `book`
- 뜻: 기록부, 항해일지
- 어원적 연결: 항해 속도와 사건을 적는 장부
- 사용 맥락: `a ship's logbook` = 선박 항해일지

### backlog

- 구성: `back` + `log`
- 뜻: 밀린 일, 적체
- 어원적 연결: 뒤에 쌓여 있는 기록이나 처리할 일
- 사용 맥락: `clear the backlog` = 밀린 일을 처리하다

### login

- 구성: `log` + `in`
- 뜻: 로그인하다
- 어원적 연결: 시스템 기록에 들어감
- 사용 맥락: `log in to the account` = 계정에 로그인하다

### logout

- 구성: `log` + `out`
- 뜻: 로그아웃하다
- 어원적 연결: 시스템 기록에서 나감
- 사용 맥락: `log out of the app` = 앱에서 로그아웃하다

### weblog

- 구성: `web` + `log`
- 뜻: 웹로그, 블로그의 원형
- 어원적 연결: 웹에 남긴 기록
- 사용 맥락: `weblog`에서 `blog`가 생김

### blog

- 구성: `web` + `log`의 축약
- 뜻: 블로그
- 어원적 연결: 웹에 남기는 기록
- 사용 맥락: `write a blog post` = 블로그 글을 쓰다

### logger

- 구성: `log` + `-er`
- 뜻: 벌목꾼, 기록 장치
- 어원적 연결: 통나무를 다루는 사람 또는 기록을 남기는 도구
- 사용 맥락: `data logger` = 데이터 기록 장치

## 기억 포인트

`log` 계열은 "통나무 / 기록" 이미지.

- `log`: 통나무 또는 기록
- `logbook`: 기록 장부
- `backlog`: 뒤에 쌓인 미처리 기록
- `login` / `logout`: 시스템 기록에 들어가고 나감
- `blog`: 웹에 남긴 기록

## 비교 단어

### logos

- 뜻: 말, 설명, 이성, 학문
- 구분 포인트: [[logos]]는 Greek `logos` 계열. `logic`, `dialogue`, `biology`는 이쪽. 영어 `log` = 통나무/기록과 별도
