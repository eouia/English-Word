# void

- 출발 단어: `avoid`
- 핵심 어원: Old French `voide` = empty
- 핵심 이미지: 비어 있음, 비워 냄, 효력이 사라짐
- 관련 문서: [[val]]
- 주의: `avoid`는 현대 형태만 보면 `a-` + `void`처럼 보이지만, 역사적으로는 Old French `esvuidier` = empty out, clear away 계열을 거쳐 들어온 말. 현대 영어에서는 "피하다" 뜻으로 굳어짐.

## 출발 단어 분석

`avoid` ← Old French `esvuidier` / `avoider` = empty out, clear away

- `voide`: empty
- 원래 이미지: 어떤 곳을 비우거나 치움
- 의미 변화: 비워 내다 → 자리를 비우다/물러나다 → 피하다
- 현대 의미: 피하다, 방지하다

## 단어들

### avoid

- 구성: Old French `esvuidier` = empty out, clear away
- 뜻: 피하다, 방지하다
- 어원적 연결: 위험하거나 불필요한 것에서 자리를 비워 벗어남
- 사용 맥락: `avoid mistakes` = 실수를 피하다

### avoidance

- 구성: `avoid` + `-ance`
- 뜻: 회피, 방지
- 어원적 연결: 어떤 상황에서 벗어나도록 비켜 감
- 사용 맥락: `tax avoidance` = 조세 회피

### avoidable

- 구성: `avoid` + `-able`
- 뜻: 피할 수 있는, 막을 수 있는
- 어원적 연결: 미리 비켜 가거나 제거할 수 있음
- 사용 맥락: `avoidable errors` = 피할 수 있는 오류

### unavoidable

- 구성: `un-` + `avoidable`
- 뜻: 피할 수 없는, 불가피한
- 어원적 연결: 비켜 갈 길이 없음
- 사용 맥락: `an unavoidable delay` = 피할 수 없는 지연

### void

- 구성: Old French `voide` = empty
- 뜻: 빈, 공허한; 무효의
- 어원적 연결: 내용이 비어 있음 → 법적 효력도 비어 있음
- 사용 맥락: `null and void` = 무효인

### voided

- 구성: `void` + `-ed`
- 뜻: 무효화된, 취소된
- 어원적 연결: 효력을 비워 없앤 상태
- 사용 맥락: `a voided contract` = 무효화된 계약

### voidable

- 구성: `void` + `-able`
- 뜻: 무효로 할 수 있는
- 어원적 연결: 효력을 비워 없앨 수 있음
- 사용 맥락: 법률·계약 문맥에서 자주 씀

### devoid

- 구성: Old French `devoider` = empty out
- 뜻: 전혀 없는, 결여된
- 어원적 연결: 어떤 성질이나 내용이 비어 있음
- 사용 맥락: `devoid of emotion` = 감정이 전혀 없는

## 기억 포인트

`void` 계열은 "비어 있음/비워서 없앰" 이미지.

- `avoid`: 피하다
- `avoidance`: 회피, 방지
- `avoidable`: 피할 수 있는
- `unavoidable`: 피할 수 없는
- `void`: 빈, 무효의
- `devoid`: 전혀 없는

## 비교 단어

### evade

- 뜻: 교묘히 피하다, 회피하다
- 구분 포인트: Latin `e-` = out + `vadere` = go 계열. "빠져나가다" 이미지가 강함. `avoid`는 더 일반적인 "피하다/막다".

### inevitable

- 뜻: 불가피한, 필연적인
- 구분 포인트: [[evit]]의 Latin `evitare` = avoid 계열. `unavoidable`과 의미는 가깝지만 어원은 다름.

### vacant

- 뜻: 빈, 비어 있는
- 구분 포인트: Latin `vacare` = be empty 계열. 빈자리, 공석, 빈 방에 자주 씀. `void`는 물리적 빈 공간뿐 아니라 공허함과 법적 무효까지 확장.

### valid

- 뜻: 유효한, 타당한
- 구분 포인트: [[val]]은 Latin `valere` = be strong, be worth 계열. `void` = 무효의 와 의미상 반대가 될 수 있지만 어원은 다름.
