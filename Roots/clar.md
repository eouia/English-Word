# clar

- 출발 단어: `clarify`
- 핵심 어원: 라틴어 `clarus` = 밝은, 분명한, 유명한
- 핵심 이미지: 밝게 드러나 분명해짐
- 관련해서 헷갈릴 수 있는 어근: [[vid]] = 보다, 보아서 판단하다 계열

## 출발 단어 분석

`clarify`는 `clar` + `-ify`.

- `clarus`: bright, clear
- `-ify`: make

따라서 `clarify`는 "분명하게 만들다" → "명확히 하다, 해명하다"의 뜻.

## 단어들

### clear

- 구성: Old French `cler`, Latin `clarus`
- 뜻: 분명한, 맑은, 치우다
- 어원적 연결: 밝고 흐리지 않아 잘 보임
- 사용 맥락: `a clear explanation` = 분명한 설명

### clarify

- 구성: `clar` + `-ify`
- 뜻: 명확히 하다, 분명히 하다
- 어원적 연결: 흐릿한 것을 밝고 분명하게 만듦
- 사용 맥락: `clarify the meaning` = 의미를 명확히 하다

### clarification

- 구성: `clarify` + `-ation`
- 뜻: 설명, 해명, 명확화
- 어원적 연결: 분명하게 만드는 과정
- 사용 맥락: `ask for clarification` = 설명을 요청하다

### clarity

- 구성: `clar` + `-ity`
- 뜻: 명확성, 맑음
- 어원적 연결: 밝고 분명한 성질
- 사용 맥락: `write with clarity` = 명확하게 쓰다

### declare

- 구성: `de-` = thoroughly + `clar`
- 뜻: 선언하다, 분명히 말하다
- 어원적 연결: 완전히 밝게 드러내 말함
- 사용 맥락: `declare independence` = 독립을 선언하다

### declaration

- 구성: `declare` + `-ation`
- 뜻: 선언, 발표
- 어원적 연결: 분명히 드러내 말한 것
- 사용 맥락: `a declaration of intent` = 의사 표명

### evident

- 구성: Latin `evidens` = clearly seen
- 뜻: 명백한
- 어원적 연결: [[vid]] 계열. 분명히 보인다는 의미라 `clear`와 뜻이 가까움
- 사용 맥락: `It is evident that...` = ...라는 점이 명백함

## 기억 포인트

`clar` 계열은 "밝고 분명함" 이미지.

- `clear`: 밝고 흐리지 않음
- `clarify`: 분명하게 만듦
- `clarity`: 분명한 성질
- `declare`: 분명히 드러내 말함

## 비교 단어

### vid

- 뜻: 보다
- 구분 포인트: [[vid]]는 보아서 판단함. `clar`는 밝고 분명하게 드러난 상태
