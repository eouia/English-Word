# arcan

- 출발 단어: `arcane`
- 핵심 어원: Latin `arcanus` = secret, hidden
- 관련 형태: Latin `arca` = chest, box
- 핵심 이미지: 상자 안에 감춰진 것, 비밀스러운 지식
- 관련 문서: [[cover]]
- 주의: `arcane`은 [[arch]] = 시작·우두머리 계열과 다름. 철자가 비슷해도 `monarch`, `archive`, `architect`와 직접 묶지 않음.

## 출발 단어 분석

`arcane` ← Latin `arcanus`

- `arca`: chest, box
- `arcanus`: hidden, secret
- 원래 이미지: 상자 안에 감춰져 아무나 모르는 것
- 현대 의미: 난해한, 비밀스러운, 소수만 아는

## 단어들

### arcane

- 발음: /ˌɑˈrkeɪn/
- 구성: Latin `arcanus` = secret, hidden
- 뜻: 난해한, 비밀스러운, 소수만 아는
- 어원적 연결: 감춰진 지식처럼 쉽게 접근하기 어려움
- 사용 맥락: `arcane rules` = 난해한 규칙

### arcanum

- 발음: /ˈɑrkənəm/
- 구성: Latin `arcanum` = secret
- 뜻: 비밀, 신비한 지식
- 어원적 연결: 감춰진 비밀 지식
- 사용 맥락: 문학적·신비주의 맥락에서 쓰임

## 기억 포인트

`arcan` 계열은 "감춰진 비밀 지식" 이미지.

- `arcane`: 난해하고 비밀스러운
- `arcanum`: 비밀, 신비한 지식

## 비교 단어

### arch

- 뜻: 시작, 원형, 우두머리
- 구분 포인트: [[arch]]는 Greek `arkhein` 계열. `arcan`은 Latin `arca/arcanus` 계열로 감춰진 것에 초점.

### cover

- 뜻: 덮다, 가리다
- 구분 포인트: [[cover]]는 덮거나 드러내는 일반 이미지. `arcan`은 감춰져 소수만 아는 비밀 지식.
