# voc

- 출발 단어: `revoke`
- 핵심 어원: 라틴어 `vocare` / `vox` = 부르다, 목소리
- 핵심 이미지: 목소리로 부름
- 관련 형태: `voc`, `vok`, `voice`, `vouch`
- 관련해서 헷갈릴 수 있는 어근: [[dict]] = 말하다, 선언하다 계열

## 출발 단어 분석

`revoke`는 `re-` + `vocare`.

- `re-`: back
- `vocare`: 부르다

따라서 `revoke`는 "다시 불러들임" → "취소하다, 철회하다"의 뜻.

## 단어들

### revoke

- 구성: `re-` + `vocare`
- 뜻: 취소하다, 철회하다
- 어원적 연결: 이미 준 권리나 명령을 다시 불러들임
- 사용 맥락: `revoke a license` = 면허를 취소하다

### revocation

- 구성: `revoke` + `-ation`
- 뜻: 취소, 철회
- 어원적 연결: 다시 불러들여 효력을 없애는 행위
- 사용 맥락: `revocation of permission` = 허가 취소

### invoke

- 구성: `in-` + `vocare`
- 뜻: 부르다, 기원하다, 언급하다
- 어원적 연결: 안으로/위해 불러냄
- 사용 맥락: `invoke a rule` = 규칙을 들먹이다, 적용하다

### invocation

- 구성: `invoke` + `-ation`
- 뜻: 기원, 호출, 언급
- 어원적 연결: 누군가나 규칙을 불러냄
- 사용 맥락: `an invocation of tradition` = 전통의 언급

### evoke

- 구성: `e- / ex-` = out + `vocare`
- 뜻: 불러일으키다
- 어원적 연결: 감정이나 기억을 밖으로 불러냄
- 사용 맥락: `evoke memories` = 기억을 불러일으키다

### evocative

- 구성: `evoke` + `-ative`
- 뜻: 환기시키는, 연상시키는
- 어원적 연결: 감정이나 이미지를 불러내는 성질
- 사용 맥락: `evocative music` = 감정을 불러일으키는 음악

### provoke

- 구성: `pro-` = forth + `vocare`
- 뜻: 자극하다, 유발하다
- 어원적 연결: 앞으로 불러내 반응을 일으킴
- 사용 맥락: `provoke anger` = 분노를 유발하다

### provocative

- 구성: `provoke` + `-ative`
- 뜻: 도발적인, 자극적인
- 어원적 연결: 반응을 불러내는 성질
- 사용 맥락: `a provocative question` = 도발적인 질문

### convoke

- 구성: `con-` = together + `vocare`
- 뜻: 소집하다
- 어원적 연결: 사람들을 함께 부름
- 사용 맥락: `convoke a meeting` = 회의를 소집하다

### vocation

- 구성: `vocare` 계열
- 뜻: 직업, 소명
- 어원적 연결: 부름을 받은 일
- 사용 맥락: `a sense of vocation` = 소명의식

### vocational

- 구성: `vocation` + `-al`
- 뜻: 직업의, 직업 교육의
- 어원적 연결: 직업적 부름이나 훈련과 관련
- 사용 맥락: `vocational training` = 직업 훈련

### vocabulary

- 구성: Latin `vocabulum` = 이름, 단어
- 뜻: 어휘
- 어원적 연결: 불러 이름 붙인 말들의 집합
- 사용 맥락: `build vocabulary` = 어휘를 늘리다

### vocal

- 구성: Latin `vox`
- 뜻: 목소리의, 목소리를 내는
- 어원적 연결: 소리나 목소리와 관련
- 사용 맥락: `vocal support` = 공개적인 지지

### vociferous

- 구성: `vox` + `ferre` = carry
- 뜻: 시끄럽게 외치는
- 어원적 연결: 목소리를 크게 실어 냄
- 사용 맥락: `vociferous opposition` = 시끄러운 반대

### advocate

- 구성: `ad-` + `vocare`
- 뜻: 옹호자, 옹호하다
- 어원적 연결: 누군가를 위해 불려 나온 사람
- 사용 맥락: `advocate for reform` = 개혁을 옹호하다

### equivocal

- 구성: [[equ]] = equal + `voc`
- 뜻: 애매한, 모호한
- 어원적 연결: 둘 이상의 목소리나 의미가 같게 들림
- 사용 맥락: `an equivocal answer` = 애매한 대답

### unequivocal

- 구성: `un-` + `equivocal`
- 뜻: 명백한, 분명한
- 어원적 연결: 여러 의미로 갈리지 않는 목소리
- 사용 맥락: `unequivocal evidence` = 명백한 증거

## 기억 포인트

`voc` 계열은 "부름, 목소리" 이미지.

- `revoke`: 다시 불러들여 취소함
- `invoke`: 불러내 적용함
- `evoke`: 감정/기억을 불러냄
- `provoke`: 반응을 앞으로 불러냄
- `convoke`: 함께 부름
- `vocation`: 부름 받은 일
- `vocabulary`: 이름 붙인 말들
- `vocal`: 목소리의
- `advocate`: 위해 불려 나온 사람
- `equivocal`: 여러 목소리로 들려 애매함
