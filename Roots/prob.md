# prob

- 출발 단어: `probability`
- 핵심 어원: Latin `probare` = test, prove, approve
- 핵심 이미지: 시험해 보고 참됨이나 가치를 인정함
- 관련 문서: [[ver]], [[preci]], [[estim]]

## 단어들

### probability

- 발음: /ˌprɑbəˈbɪləˌti/
- 구성: Latin `probabilitas` < `probabilis` = credible, probable
- 뜻: 가능성, 확률
- 어원적 연결: 원래는 시험하거나 검토해 믿을 만하다고 여겨지는 성질
- 사용 맥락: `high probability` = 높은 확률

### probable

- 발음: /ˈprɑbəbəl/
- 구성: Latin `probabilis` = provable, credible, likely
- 뜻: 있을 법한, 그럴듯한
- 어원적 연결: 검토해 보니 믿을 만하거나 가능성이 있음
- 사용 맥락: `a probable cause` = 그럴듯한 원인

### probably

- 발음: /ˈprɑbəˌbli/
- 구성: `probable` + `-ly`
- 뜻: 아마, 아마도
- 어원적 연결: 증거나 판단상 그럴 가능성이 높음
- 사용 맥락: `It will probably rain.` = 아마 비가 올 것이다

### improbable

- 발음: /ˌɪˈmprɑbəbəl/
- 구성: `im-` = not + `probable`
- 뜻: 있을 법하지 않은, 희박한
- 어원적 연결: 검토해 보아도 그럴 가능성이 낮음
- 사용 맥락: `an improbable story` = 믿기 어려운 이야기

### prove

- 발음: /ˈpruv/
- 구성: Old French `prover` < Latin `probare`
- 뜻: 증명하다, 입증하다
- 어원적 연결: 시험해 보고 참임을 보임
- 사용 맥락: `prove a theory` = 이론을 증명하다

### proof

- 발음: /ˈpruf/
- 구성: Old French `prove` 계열
- 뜻: 증거, 증명
- 어원적 연결: 시험과 검증을 통과해 참임을 보이는 것
- 사용 맥락: `proof of identity` = 신원 증명

### approve

- 발음: /əˈpruv/
- 구성: `ad-` = to + Latin `probare`
- 뜻: 승인하다, 좋다고 인정하다
- 어원적 연결: 시험해 보고 가치나 타당성을 인정함
- 사용 맥락: `approve a plan` = 계획을 승인하다

### approval

- 발음: /əˈpruvəl/
- 구성: `approve` + `-al`
- 뜻: 승인, 인정
- 어원적 연결: 타당하다고 인정한 결과
- 사용 맥락: `official approval` = 공식 승인

### probe

- 발음: /ˈproʊb/
- 구성: Latin `probare` 계열
- 뜻: 조사하다, 탐사하다; 탐침
- 어원적 연결: 찔러 보듯 시험하고 조사함
- 사용 맥락: `probe the cause` = 원인을 조사하다

### probation

- 발음: /proʊˈbeɪʃən/
- 구성: Latin `probatio` = testing, proof
- 뜻: 보호 관찰, 수습 기간, 시험 기간
- 어원적 연결: 일정 기간 시험해 보며 판단함
- 사용 맥락: `on probation` = 보호 관찰 중, 수습 중

## 기억 포인트

`prob` 계열은 "시험해 보고 인정함" 이미지.

- `prove`: 증명하다
- `proof`: 증거
- `probable`: 믿을 만한 → 있을 법한
- `probability`: 가능성, 확률
- `approve`: 인정하고 승인하다
- `probe`: 조사하다
- `probation`: 시험 기간

## 비교 단어

### ver

- 뜻: 참, 진짜, 검증
- 구분 포인트: [[ver]]는 참됨 자체와 검증. `prob`는 시험해 보고 참이나 가치를 인정하는 과정.

### preci

- 뜻: 가격, 가치, 귀하게 여김
- 구분 포인트: [[preci]]는 가치를 매기고 귀하게 여기는 계열. `approve`는 시험·검토 후 좋다고 인정함.

### estim

- 뜻: 가치를 매기다, 어림해 판단하다
- 구분 포인트: [[estim]]은 평가·추정. `probability`는 검토 가능한 그럴듯함에서 확률 의미로 확장.
