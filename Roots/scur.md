# scur

- 출발 단어: `obscure`
- 핵심 어원: 라틴어 `obscurus` = dark, dim, unclear
- 핵심 이미지: 어둡게 덮여 잘 보이지 않음
- 관련해서 헷갈릴 수 있는 어근: [[clar]] = 밝은, 분명한 계열, [[cel]] = 숨기다, 감추다 계열

## 출발 단어 분석

`obscure`는 Latin `obscurus`에서 온 단어.

- `ob-`: over, against
- `scurus`: dark, covered의 느낌

따라서 `obscure`는 "어둡게 가려진"의 뜻. 여기서 "잘 알려지지 않은, 이해하기 어려운" 또는 동사로 "가리다, 흐리게 하다"의 뜻.

## 단어들

### obscure

- 발음: /əˈbskjʊr/
- 구성: Latin `obscurus`
- 뜻: 모호한, 잘 알려지지 않은; 가리다, 흐리게 하다
- 어원적 연결: 어둡게 덮여 잘 보이지 않음
- 사용 맥락: `an obscure reference` = 잘 알려지지 않은 언급, `clouds obscure the sun` = 구름이 해를 가리다

### obscurity

- 발음: /əˈbskjʊrəti/
- 구성: `obscure` + `-ity`
- 뜻: 불명확함, 무명, 잘 알려지지 않음
- 어원적 연결: 어둠 속에 있어 잘 드러나지 않는 상태
- 사용 맥락: `live in obscurity` = 무명으로 살다

### obscurantism

- 구성: `obscurant` + `-ism`
- 뜻: 반계몽주의, 지식 보급을 막는 태도
- 어원적 연결: 지식을 밝히지 않고 어둡게 남겨 두려는 태도
- 사용 맥락: intellectual / historical context

### chiaroscuro

- 구성: Italian `chiaro` = clear, light + `oscuro` = dark
- 뜻: 명암법
- 어원적 연결: 밝음과 어둠의 대비
- 사용 맥락: art context

## 기억 포인트

`scur/obscur` 계열은 "어둠, 불명확함" 이미지.

- `obscure`: 어둡게 가려진 → 모호한, 잘 알려지지 않은
- `obscurity`: 어둠 속에 있음 → 불명확함, 무명
- `obscurantism`: 밝히지 않고 어둡게 두려는 태도

## 비교 단어

### clear

- 뜻: 맑은, 분명한
- 구분 포인트: [[clar]] 계열. `clar`는 밝고 분명함, `obscure`는 어둡고 잘 보이지 않음

### conceal

- 뜻: 숨기다, 감추다
- 구분 포인트: [[cel]]은 감추는 동작. `obscure`는 어둡게 하거나 흐리게 해서 잘 보이지 않게 만드는 느낌
