# brev

- 출발 단어: `brief`
- 핵심 어원: Latin `brevis` = 짧은
- 핵심 이미지: 길이나 시간이 짧음
- 관련 문서: [[long]], [[tempor]]

## 출발 단어 분석

`brief` ← Latin `brevis`

- `brevis`: short
- 원래 이미지: 짧게 줄인 문서나 말
- 현대 의미: 짧은, 간단한; 보고서, 사건 개요

## 단어들

### brief

- 발음: /ˈbrif/
- 구성: Latin `brevis` = short
- 뜻: 짧은, 간단한; 요약 보고서
- 어원적 연결: 길게 늘이지 않고 짧게 정리함
- 사용 맥락: `a brief explanation` = 간단한 설명

### briefly

- 발음: /ˈbrifli/
- 구성: `brief` + `-ly`
- 뜻: 간단히, 잠시
- 어원적 연결: 짧은 방식으로 말하거나 짧은 시간 동안 일어남
- 사용 맥락: `briefly discuss the issue` = 그 문제를 간단히 논의하다

### brevity

- 발음: /ˈbrɛvəti/
- 구성: Latin `brevitas` = shortness
- 뜻: 간결함, 짧음
- 어원적 연결: 말이나 글이 짧고 압축되어 있음
- 사용 맥락: `brevity is essential` = 간결함이 필수다

### abbreviate

- 발음: /əˈbriviˌeɪt/
- 구성: `ad-` = to + Latin `brevis` = short
- 뜻: 축약하다, 줄여 쓰다
- 어원적 연결: 긴 표현을 더 짧게 만듦
- 사용 맥락: `abbreviate United Nations to UN` = United Nations를 UN으로 줄이다

### abbreviation

- 발음: /əˌbriviˈeɪʃən/
- 구성: `abbreviate` + `-ion`
- 뜻: 약어, 축약형
- 어원적 연결: 짧게 줄인 말
- 사용 맥락: `NASA is an abbreviation` = NASA는 약어다

### abridge

- 발음: /əˈbrɪdʒ/
- 구성: Old French `abregier` ← Latin `abbreviare`
- 뜻: 줄이다, 축약하다
- 어원적 연결: 긴 글이나 권리를 짧게 줄이거나 제한함
- 사용 맥락: `an abridged version` = 축약판
- 주의: 권리·자유를 `abridge`하면 "제한하다"의 뜻.

### unabridged

- 발음: /ˌʌnəˈbrɪdʒd/
- 구성: `un-` = not + `abridged`
- 뜻: 축약하지 않은, 완전판의
- 어원적 연결: 짧게 줄이지 않은 상태
- 사용 맥락: `an unabridged dictionary` = 완전판 사전

## 기억 포인트

`brev` 계열은 "짧게 줄임" 이미지.

- `brief`: 짧은 설명이나 문서
- `brevity`: 간결함
- `abbreviate`: 짧게 줄여 쓰다
- `abbreviation`: 줄인 말
- `abridge`: 글이나 권리를 줄이다
- `unabridged`: 줄이지 않은 완전판

## 비교 단어

### long

- 뜻: 길다
- 구분 포인트: [[long]]은 길이와 지속. `brev`는 짧음과 축약.

### tempor

- 뜻: 시간, 기간
- 구분 포인트: [[tempor]]는 시간 자체와 시기. `brev`는 시간이나 표현의 짧음.
