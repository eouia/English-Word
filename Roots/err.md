# err

- 출발 단어: `error`
- 핵심 어원: Latin `errare` = 헤매다, 잘못 가다
- 핵심 이미지: 길이나 판단에서 벗어남
- 관련 문서: [[via]], [[fals]]

## 출발 단어 분석

`error` ← Latin `errare`

- `errare`: wander, stray, be mistaken
- 원래 이미지: 길을 벗어나 헤맴
- 현대 의미: 오류, 실수

## 단어들

### error

- 구성: Latin `error` = wandering, mistake
- 뜻: 오류, 실수
- 어원적 연결: 올바른 길이나 판단에서 벗어남
- 사용 맥락: `a calculation error` = 계산 오류

### err

- 구성: Latin `errare`
- 뜻: 잘못하다, 실수하다
- 어원적 연결: 바른 방향에서 벗어남
- 사용 맥락: `to err is human` = 실수는 인간적이다

### erroneous

- 구성: `error` + `-ous`
- 뜻: 잘못된, 오류가 있는
- 어원적 연결: 판단이나 정보가 바른 길에서 벗어남
- 사용 맥락: `an erroneous assumption` = 잘못된 가정

### erratic

- 구성: Latin `erraticus` = wandering
- 뜻: 불규칙한, 종잡을 수 없는
- 어원적 연결: 일정한 길 없이 이리저리 헤맴
- 사용 맥락: `erratic behavior` = 불규칙한 행동

### aberration

- 구성: `ab-` = away + Latin `errare`
- 뜻: 일탈, 이상 현상
- 어원적 연결: 정상 경로에서 벗어난 상태
- 사용 맥락: `a temporary aberration` = 일시적 일탈

## 기억 포인트

`err` 계열은 "잘못된 길로 벗어남" 이미지.

- `error`: 오류
- `err`: 실수하다
- `erroneous`: 잘못된
- `erratic`: 불규칙하게 헤매는
- `aberration`: 정상에서 벗어난 일탈
