# mit

- 출발 단어: `submit`
- 핵심 어원: 라틴어 `mittere` / `miss-` = 보내다, 놓아 보내다
- 핵심 이미지: 어떤 방향으로 보냄
- 관련해서 헷갈릴 수 있는 어근: [[duc]] = 이끌다 계열, [[fer]] = 나르다 계열

## 출발 단어 분석

`submit`은 `sub-` + `mittere`.

- `sub-`: under
- `mittere`: 보내다

따라서 `submit`은 "아래로 보내다" → "제출하다, 복종하다"의 뜻.

## 단어들

### submit

- 구성: `sub-` = under + `mit`
- 뜻: 제출하다, 복종하다
- 어원적 연결: 문서를 아래/앞으로 보내거나, 자신을 권위 아래에 둠
- 사용 맥락: `submit an application` = 지원서를 제출하다

### submission

- 구성: `sub-` + `miss` + `-ion`
- 뜻: 제출, 복종
- 어원적 연결: 아래로 보내진 상태
- 사용 맥락: `online submission` = 온라인 제출

### admit

- 구성: `ad-` = to + `mit`
- 뜻: 인정하다, 입장을 허가하다
- 어원적 연결: 안으로 보내 들임. 사실을 마음 안으로 받아들임
- 사용 맥락: `admit a mistake` = 실수를 인정하다

### admission

- 구성: `ad-` + `miss` + `-ion`
- 뜻: 입장, 입학, 인정
- 어원적 연결: 안으로 들여보냄
- 사용 맥락: `college admission` = 대학 입학

### commit

- 구성: `com-` = together, fully + `mit`
- 뜻: 저지르다, 전념하다, 맡기다
- 어원적 연결: 어떤 일에 자신을 완전히 보내 묶음
- 사용 맥락: `commit to a goal` = 목표에 전념하다

### commitment

- 구성: `commit` + `-ment`
- 뜻: 약속, 전념, 책임
- 어원적 연결: 자신을 어떤 일에 보내 묶은 상태
- 사용 맥락: `a long-term commitment` = 장기적 약속

### permit

- 구성: `per-` = through + `mit`
- 뜻: 허락하다, 허가증
- 어원적 연결: 통과해 지나가도록 보내 줌
- 사용 맥락: `parking permit` = 주차 허가증

### permission

- 구성: `per-` + `miss` + `-ion`
- 뜻: 허락, 허가
- 어원적 연결: 통과하도록 보내 줌
- 사용 맥락: `ask permission` = 허락을 구하다

### emit

- 구성: `e-` = out + `mit`
- 뜻: 내뿜다, 방출하다
- 어원적 연결: 밖으로 보냄
- 사용 맥락: `emit light` = 빛을 내다

### transmit

- 구성: `trans-` = across + `mit`
- 뜻: 전송하다, 전달하다
- 어원적 연결: 가로질러 보냄
- 사용 맥락: `transmit data` = 데이터를 전송하다

### omit

- 구성: `ob-` = away, aside + `mit`
- 뜻: 빠뜨리다, 생략하다
- 어원적 연결: 옆으로 보내 제외함
- 사용 맥락: `omit details` = 세부사항을 생략하다

### dismiss

- 구성: `dis-` = away + `miss`
- 뜻: 해고하다, 묵살하다, 해산시키다
- 어원적 연결: 멀리 보내 버림
- 사용 맥락: `dismiss the claim` = 주장을 묵살하다

### mission

- 구성: `miss` + `-ion`
- 뜻: 임무, 사명
- 어원적 연결: 보냄을 받은 일
- 사용 맥락: `a rescue mission` = 구조 임무

### promise

- 구성: `pro-` = forward + `miss`
- 뜻: 약속하다, 약속
- 어원적 연결: 앞으로 내보낸 말. 미래 행동을 말로 내놓음
- 사용 맥락: `make a promise` = 약속하다

### premise

- 구성: `pre-` = before + `miss`
- 뜻: 전제, 전제로 삼다
- 어원적 연결: 논의 앞에 먼저 놓이거나 보내진 명제
- 사용 맥락: `a false premise` = 잘못된 전제

### surmise

- 구성: Old French `surmettre` = put upon, accuse ← `sur-` + `mettre`
- 뜻: 추측하다, 추정하다; 추측
- 어원적 연결: 어떤 생각이나 혐의를 위에 얹어 놓음 → 확실한 증거 없이 판단을 세움
- 사용 맥락: `surmise the cause` = 원인을 추측하다, `a reasonable surmise` = 그럴듯한 추측

### surmised

- 구성: `surmise` + `-ed`
- 뜻: 추측한, 추정된
- 어원적 연결: 확실히 본 것이 아니라 근거 위에 세워 놓은 판단
- 사용 맥락: `as many scholars have surmised` = 많은 학자들이 추측했듯이

## 기억 포인트

`mit` 계열은 "보내다" 이미지.

- `submit`: 아래로 보내 제출함
- `admit`: 안으로 들여보냄
- `commit`: 자신을 완전히 보내 맡김
- `permit`: 통과하도록 보내 줌
- `emit`: 밖으로 보냄
- `transmit`: 건너 보내 전달함
- `dismiss`: 멀리 보내 버림
- `premise`: 논의 앞에 먼저 놓인 말
- `surmise`: 근거 위에 얹어 놓은 추측
