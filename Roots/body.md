# body

- 출발 단어: `embody`
- 핵심 어원: Old English `bodig` = 몸, 신체
- 핵심 이미지: 추상적인 것이 몸이나 구체적 형태를 가짐
- 관련해서 헷갈릴 수 있는 어근: [[corp]] = 라틴어 `corpus` 계열의 몸, [[carn]] = 살/육체

## 출발 단어 분석

`embody`는 `em-` + `body`.

- `em-`: 안에 넣다, ~으로 만들다
- `body`: 몸, 구체적 형태

따라서 `embody`는 "몸 안에 넣다" → "추상적 생각이나 가치를 구체적으로 드러내다"의 뜻.

## 단어들

### body

- 구성: Old English `bodig`
- 뜻: 몸, 신체, 단체, 본문
- 어원적 연결: 사람의 몸에서 출발해, 하나의 덩어리나 조직 전체로 의미 확대
- 사용 맥락: `the human body` = 인체, `the body of an essay` = 글의 본문

### bodily

- 구성: `body` + `-ly`
- 뜻: 신체의, 몸 전체로
- 어원적 연결: 몸과 직접 관련됨
- 사용 맥락: `bodily harm` = 신체적 상해

### embody

- 구성: `em-` = in, into + `body`
- 뜻: 구현하다, 구체화하다, 체현하다
- 어원적 연결: 생각, 가치, 성질이 몸을 얻어 구체적 모습으로 나타남
- 사용 맥락: `embody democratic values` = 민주적 가치를 구현하다

### embodiment

- 구성: `embody` + `-ment`
- 뜻: 구현, 화신, 전형
- 어원적 연결: 추상적 성질이 몸을 얻어 나타난 존재나 형태
- 사용 맥락: `the embodiment of courage` = 용기의 화신

### disembodied

- 구성: `dis-` = apart, away + `embodied`
- 뜻: 몸에서 분리된, 실체 없는
- 어원적 연결: 몸이나 구체적 형태에서 떨어져 있음
- 사용 맥락: `a disembodied voice` = 실체 없이 들리는 목소리

### anybody

- 구성: `any` + `body`
- 뜻: 누구든지, 아무도
- 어원적 연결: 어떤 몸을 가진 사람이라도
- 사용 맥락: `Is anybody there?` = 거기 누구 있나요?

### somebody

- 구성: `some` + `body`
- 뜻: 누군가, 중요한 사람
- 어원적 연결: 어떤 몸을 가진 한 사람
- 사용 맥락: `Somebody called you.` = 누군가 너에게 전화했다

### nobody

- 구성: `no` + `body`
- 뜻: 아무도 아니다, 보잘것없는 사람
- 어원적 연결: 어떤 사람의 몸도 없음
- 사용 맥락: `Nobody knows.` = 아무도 모른다

## 기억 포인트

`body` 계열은 "몸 / 구체적 형태" 이미지.

- `body`: 몸, 하나의 덩어리
- `bodily`: 몸과 관련된
- `embody`: 추상적인 것을 몸으로 드러냄
- `embodiment`: 추상적 성질이 몸을 얻은 형태
- `disembodied`: 몸에서 분리된

## 비교 단어

### corp

- 구성: Latin `corpus` / `corpor-`
- 뜻: 몸, 신체, 한 덩어리
- 구분 포인트: [[corp]]는 라틴어 계열의 학술적/법적 느낌이 강함. `body`는 영어 고유어 계열로 더 일상적이고 직접적임
