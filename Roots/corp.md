# corp

- 출발 단어: `corporate`
- 핵심 어원: 라틴어 `corpus` / `corpor-` = 몸, 신체, 한 덩어리
- 핵심 이미지: 여러 부분이 모여 하나의 몸을 이룸
- 관련 문서: [[som]]
- 관련해서 헷갈릴 수 있는 단어: `corps`는 발음이 `/kor/`, `corpse`는 발음이 `/korps/`

## 출발 단어 분석

`corporate`는 `corpor-` + `-ate`.

- `corpor-`: 몸, 한 덩어리
- `-ate`: 형용사/동사 형성

따라서 `corporate`는 "하나의 몸을 이룬" → "법인화된, 회사의"의 뜻.

## 단어들

### corporate

- 구성: `corpor-` + `-ate`
- 뜻: 회사의, 법인의, 공동의
- 어원적 연결: 여러 사람이 하나의 법적 몸처럼 묶인 상태
- 사용 맥락: `corporate culture` = 기업 문화

### corporation

- 구성: `corporate` + `-ion`
- 뜻: 법인, 회사
- 어원적 연결: 법적으로 하나의 몸처럼 인정된 조직
- 사용 맥락: `a multinational corporation` = 다국적 기업

### incorporate

- 구성: `in-` = into + `corpor-`
- 뜻: 포함하다, 법인으로 만들다
- 어원적 연결: 어떤 것을 한 몸 안으로 넣음
- 사용 맥락: `incorporate feedback` = 피드백을 반영하다

### incorporated

- 구성: `incorporate` + `-ed`
- 뜻: 법인화된, 포함된
- 어원적 연결: 하나의 법적 몸 안에 들어간 상태
- 사용 맥락: `Apple Inc.` = Apple Incorporated

### corporeal

- 구성: Latin `corporeus`
- 뜻: 육체의, 물질적인
- 어원적 연결: 몸을 가진 것, 신체와 관련된 것
- 사용 맥락: `corporeal existence` = 육체적 존재

### incorporeal

- 구성: `in-` = not + `corporeal`
- 뜻: 비물질적인, 육체가 없는
- 어원적 연결: 몸을 가지지 않은 상태
- 사용 맥락: `incorporeal beings` = 비물질적 존재

### corporal

- 구성: Latin `corporalis`
- 뜻: 신체의, 육체의
- 어원적 연결: 몸에 직접 관련됨
- 사용 맥락: `corporal punishment` = 체벌

### corpse

- 구성: Old French `corps`, Latin `corpus`
- 뜻: 시체
- 어원적 연결: 생명이 빠진 몸
- 사용 맥락: `discover a corpse` = 시체를 발견하다

### corps

- 구성: French `corps`
- 뜻: 부대, 단체
- 어원적 연결: 여러 사람이 하나의 몸처럼 움직이는 조직
- 사용 맥락: `the Marine Corps` = 해병대

### corpus

- 구성: Latin `corpus`
- 뜻: 말뭉치, 자료체, 신체
- 어원적 연결: 여러 텍스트나 자료가 모여 이룬 한 덩어리
- 사용 맥락: `a text corpus` = 텍스트 말뭉치

## 기억 포인트

`corp` 계열은 "몸 / 한 덩어리" 이미지.

- `corporate`: 하나의 몸을 이룬 회사의
- `corporation`: 법적으로 하나의 몸인 조직
- `incorporate`: 한 몸 안에 넣음
- `corporeal`: 몸을 가진, 물질적인
- `incorporeal`: 몸이 없는, 비물질적인
- `corporal`: 신체의
- `corpse`: 죽은 몸
- `corps`: 하나의 몸처럼 움직이는 조직
- `corpus`: 자료가 모인 한 덩어리

## 비교 단어

### corpse / corps

- `corpse`: 시체. 마지막 `pse`를 발음함
- `corps`: 부대, 단체. 프랑스어식으로 `/kor/`처럼 발음
- 구분 포인트: 둘 다 `corpus`에서 왔지만, 현대 영어에서는 발음과 의미가 크게 다름
