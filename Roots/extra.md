# extra

- 출발 단어: `extraneous`
- 핵심 어원: Latin `extra` = outside, beyond
- 핵심 이미지: 경계나 범위의 바깥에 있음
- 관련 문서: [[ex]], [[ord]], [[terr]]
- 주의: `extraordinary`는 `extra-` + `ordinary`이므로 [[ord]]에도 연결됨.

## 출발 단어 분석

`extraneous` = `extra-` + `-neous`

- `extra-`: outside, beyond
- `-neous`: 성질을 나타내는 형용사형 어미
- 원래 이미지: 본래 범위 밖에서 온 것
- 현대 의미: 관련 없는, 불필요한, 외부의

## 단어들

### extraneous

- 구성: Latin `extraneus` ← `extra` = outside
- 뜻: 관련 없는, 불필요한, 외부에서 온
- 어원적 연결: 핵심 범위 밖에 있어 본질과 관계가 약함
- 사용 맥락: `extraneous information` = 불필요한 정보, `extraneous noise` = 외부 잡음

### extra

- 구성: Latin `extra` = outside, beyond
- 뜻: 추가의, 여분의; 추가분
- 어원적 연결: 기본 범위 밖에 더 붙은 것
- 사용 맥락: `extra time` = 추가 시간, `pay extra` = 추가 요금을 내다

### extraordinary

- 구성: `extra-` = beyond + `ordinary`
- 뜻: 비범한, 놀라운, 특별한
- 어원적 연결: 보통 질서나 수준의 바깥에 있음
- 사용 맥락: `an extraordinary achievement` = 비범한 성취
- 관련: [[ord]]

### external

- 구성: Latin `externus` ← `extra`
- 뜻: 외부의, 겉의
- 어원적 연결: 안쪽이 아니라 바깥쪽에 있음
- 사용 맥락: `external pressure` = 외부 압력, `external appearance` = 외관

### exterior

- 구성: Latin `exterior` = outer, outward
- 뜻: 외부, 겉면; 외부의
- 어원적 연결: 더 바깥쪽에 있는 부분
- 사용 맥락: `the building's exterior` = 건물 외부

### extrinsic

- 구성: Latin `extrinsecus` = from outside
- 뜻: 외적인, 비본질적인
- 어원적 연결: 내부 본질에서 나온 것이 아니라 바깥에서 붙은 성질
- 사용 맥락: `extrinsic motivation` = 외적 동기

### extraterrestrial

- 구성: `extra-` = beyond + `terrestrial` = earthly
- 뜻: 지구 밖의, 외계의; 외계 생명체
- 어원적 연결: 지구라는 범위의 바깥에 있음
- 사용 맥락: `extraterrestrial life` = 외계 생명체
- 관련: [[terr]]

### strange

- 구성: Old French `estrange` ← Latin `extraneus`
- 뜻: 이상한, 낯선
- 어원적 연결: 자기 영역 밖에서 온 낯선 것
- 사용 맥락: `a strange sound` = 이상한 소리

### stranger

- 구성: `strange` + `-er`
- 뜻: 낯선 사람, 이방인
- 어원적 연결: 자기 공동체나 익숙한 범위 밖에서 온 사람
- 사용 맥락: `talk to a stranger` = 낯선 사람에게 말하다

## 기억 포인트

`extra`는 "범위 밖" 이미지.

- `extraneous`: 핵심 범위 밖이라 관련 없거나 불필요함
- `extra`: 기본 범위 밖에 더 붙음
- `extraordinary`: 보통 범위 밖의 특별함
- `external`: 내부가 아닌 외부
- `exterior`: 더 바깥쪽 면
- `extrinsic`: 본질 밖에서 붙은 성질
- `extraterrestrial`: 지구 밖
- `strange`: 익숙한 영역 밖에서 와서 낯섦

## 비교 단어

### intrinsic

- 뜻: 본질적인, 내재적인
- 구분 포인트: `extrinsic`의 반대. 내부 본질에 속하는 성질.

### foreign

- 뜻: 외국의, 낯선
- 구분 포인트: 의미는 비슷하지만 Latin `foris` = outside 계열. `strange`는 Latin `extraneus` 계열.
