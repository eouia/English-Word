# sist

- 출발 단어: `exist`
- 추가 기준 단어: `assistant`
- 핵심 어원: 라틴어 `sistere` = 서게 하다, 서다, 놓이다
- 핵심 이미지: 어떤 자리에 서 있음 / 버티고 있음
- 관련해서 헷갈릴 수 있는 어근: [[pos]] = 놓다, 두다 계열, [[stat]] = 서 있는 상태 계열, [[sym]]의 `system` = Greek `histanai`, 함께 세우다 계열

## 출발 단어 분석

`exist`는 `ex-` + `sistere`.

- `ex-`: out
- `sistere`: stand, cause to stand

따라서 `exist`는 원래 "밖으로 서다, 나타나 서 있음" → "존재하다"의 뜻.

## 단어들

### exist

- 구성: [[ex]] = out + `sist`
- 뜻: 존재하다
- 어원적 연결: 밖으로 드러나 서 있음
- 사용 맥락: `The problem still exists.` = 그 문제는 여전히 존재함

### existence

- 구성: `exist` + `-ence`
- 뜻: 존재, 생존
- 어원적 연결: 실제로 서 있는 상태
- 사용 맥락: `human existence` = 인간 존재

### existent

- 구성: `exist` + `-ent`
- 뜻: 존재하는
- 어원적 연결: 실제로 서 있거나 있음
- 사용 맥락: `currently existent forms` = 현재 존재하는 형태들

### coexist

- 구성: [[con]]의 `co-` = together + `exist`
- 뜻: 공존하다
- 어원적 연결: 함께 존재함
- 사용 맥락: `coexist peacefully` = 평화롭게 공존하다

### assist

- 구성: `ad-` = to + `sist`
- 뜻: 돕다, 보조하다
- 어원적 연결: 곁에 서서 도와줌
- 사용 맥락: `assist the team` = 팀을 돕다

### assistant

- 구성: `assist` + `-ant`
- 뜻: 조수, 보조자
- 어원적 연결: `ad-` = to, toward + `sistere` = stand. 곁에 서서 도와주는 사람
- 사용 맥락: `a research assistant` = 연구 조교, `a teaching assistant` = 조교
- 학습 포인트: 현대 영어에서는 `assist` + `-ant`로 보되, 깊은 이미지는 "옆에 서서 받쳐 줌"

### consist

- 구성: [[con]] = together + `sist`
- 뜻: 구성되다, 있다
- 어원적 연결: 여러 요소가 함께 서 있음
- 사용 맥락: `The team consists of five members.` = 그 팀은 다섯 명으로 구성됨

### consistent

- 구성: `consist` + `-ent`
- 뜻: 일관된, 모순 없는
- 어원적 연결: 요소들이 함께 서서 흔들리지 않음
- 사용 맥락: `consistent results` = 일관된 결과

### insist

- 구성: [[in]] = on, upon + `sist`
- 뜻: 주장하다, 고집하다
- 어원적 연결: 어떤 입장 위에 굳게 서 있음
- 사용 맥락: `insist on fairness` = 공정성을 주장하다

### persist

- 구성: `per-` = through, thoroughly + `sist`
- 뜻: 지속되다, 끈질기게 계속하다
- 어원적 연결: 끝까지 버티고 서 있음
- 사용 맥락: `persistent problems` = 지속적인 문제

### persistent

- 구성: `persist` + `-ent`
- 뜻: 지속적인, 끈질긴
- 어원적 연결: 쉽게 사라지지 않고 계속 버티는 성질
- 사용 맥락: `a persistent cough` = 지속되는 기침

### persistence

- 구성: `persistent` + `-ence`
- 뜻: 지속성, 끈기
- 어원적 연결: 끝까지 버티고 계속되는 성질
- 사용 맥락: `persistence pays off` = 끈기는 보답을 받는다

### persistently

- 구성: `persistent` + `-ly`
- 뜻: 지속적으로, 끈질기게
- 어원적 연결: 멈추지 않고 계속 버티는 방식
- 사용 맥락: `persistently ask questions` = 끈질기게 질문하다

### resist

- 구성: `re-` = back, against + `sist`
- 뜻: 저항하다, 견디다
- 어원적 연결: 맞서 서 있음
- 사용 맥락: `resist pressure` = 압력에 저항하다

### resistance

- 구성: `resist` + `-ance`
- 뜻: 저항, 저항력
- 어원적 연결: 맞서 버티는 힘
- 사용 맥락: `antibiotic resistance` = 항생제 내성

### subsist

- 구성: `sub-` = under + `sist`
- 뜻: 근근이 살아가다, 존속하다
- 어원적 연결: 밑에서 버티며 존재함
- 사용 맥락: `subsist on rice` = 쌀로 연명하다

## 기억 포인트

`sist` 계열은 "서 있음 / 버팀" 이미지.

- `exist`: 밖으로 드러나 서 있음
- `coexist`: 함께 서서 존재함
- `assist`: 곁에 서서 도움
- `consist`: 함께 서서 구성됨
- `insist`: 한 입장 위에 굳게 섬
- `persist`: 끝까지 버티고 섬
- `persistent`: 계속 버티는
- `resist`: 맞서 섬

## 비교 단어

### system

- 뜻: 체계, 시스템
- 구분 포인트: [[sym]]의 `system`은 Greek `syn-` + `histanai` 계열. 의미 이미지는 "함께 세워짐"으로 가깝지만, `exist`, `consist`, `resist`의 `sist`는 Latin `sistere` 계열

### stand

- 뜻: 서다, 입장
- 구분 포인트: [[stand]]는 영어 기본어. `sist`는 라틴계 시험 어휘에서 `exist`, `resist`, `persist`처럼 나타남

### stat

- 뜻: 서 있는 상태, 지위, 공식적 위치
- 구분 포인트: [[stat]]은 같은 "서다" 이미지라도 `state`, `status`, `station`, `statute`처럼 상태나 제도적으로 세워 둔 것에 가까움
