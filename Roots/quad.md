# quad

- 출발 단어: `square`
- 핵심 어원: Latin `quattuor` = four, `quadrus` = square
- 핵심 이미지: 넷, 네모, 네 부분
- 관련 어근: [[uni]], [[equ]], [[metr]]

## 출발 단어 분석

`square`는 Old French `esquarre`를 거쳐 Vulgar Latin `exquadra` / `exquadrare`에서 온 말.

이 바탕에는 Latin `quadrus` = square가 있고, 더 깊게는 `quattuor` = four와 연결됨.

따라서 `square`는 "네 쪽이 맞게 세워진 모양" → "정사각형, 제곱, 반듯한"의 뜻.

## 단어들

### square

- 구성: Old French `esquarre` < Vulgar Latin `exquadra`
- 의미: 정사각형, 광장, 제곱하다, 반듯한
- 어원적 연결: 네 변이 맞는 모양
- 사용 맥락: `a square table` = 네모난 탁자

### squared

- 구성: `square` + `-ed`
- 의미: 제곱한, 네모지게 만든
- 어원적 연결: 수를 정사각형 넓이처럼 곱함
- 사용 맥락: `x squared` = x의 제곱

### squarely

- 구성: `square` + `-ly`
- 의미: 정면으로, 똑바로, 공정하게
- 어원적 연결: 반듯하게 맞선 상태
- 사용 맥락: `face the issue squarely` = 문제를 정면으로 마주하다

### quadratic

- 구성: Latin `quadratus` = made square + `-ic`
- 의미: 이차의, 제곱의
- 어원적 연결: 제곱 항과 관련됨
- 사용 맥락: `a quadratic equation` = 이차방정식

### quadrant

- 구성: Latin `quadrans` = fourth part
- 의미: 사분면, 4분의 1 구역
- 어원적 연결: 전체를 네 부분으로 나눈 한 부분
- 사용 맥락: `the first quadrant` = 제1사분면

### quarter

- 구성: Latin `quartarius` = fourth part
- 의미: 4분의 1, 분기, 15분
- 어원적 연결: 넷으로 나눈 한 몫
- 사용 맥락: `a quarter of an hour` = 15분

### quart

- 구성: Latin `quartus` = fourth
- 의미: 쿼트, 4분의 1갤런 단위
- 어원적 연결: 원래 4분의 1 단위
- 사용 맥락: `a quart of milk` = 우유 1쿼트

### quartet

- 구성: Italian `quartetto` < Latin `quartus`
- 의미: 4중주, 4인조
- 어원적 연결: 네 명 또는 네 악기로 이뤄진 묶음
- 사용 맥락: `a string quartet` = 현악 4중주

### quadruple

- 구성: Latin `quadruplus` = fourfold
- 의미: 네 배의, 네 배가 되다
- 어원적 연결: 네 겹
- 사용 맥락: `quadruple the amount` = 양을 네 배로 늘리다

### quadrilateral

- 구성: `quadri-` = four + `lateral` = side
- 의미: 사각형
- 어원적 연결: 네 변을 가진 도형
- 사용 맥락: `a quadrilateral shape` = 사각형 모양

### quadrangle

- 구성: `quadri-` = four + [[angl]] = angle
- 의미: 사각형, 사각 안뜰
- 어원적 연결: 네 각을 가진 도형이나 공간
- 사용 맥락: `a school quadrangle` = 학교의 사각 안뜰

### squad

- 구성: French `escouade` / Italian `squadra` < Vulgar Latin `exquadra`
- 의미: 분대, 팀
- 어원적 연결: 원래 네모난 대형이나 작은 무리 이미지에서 군사 단위로 확장
- 사용 맥락: `a rescue squad` = 구조대

## 기억 포인트

- `square`: 네모, 제곱, 반듯함
- `quadratic`: 제곱, 이차
- `quadrant`: 네 부분 중 한 구역
- `quarter`: 네 부분 중 하나
- `quadruple`: 네 배
- `quadrilateral`: 네 변 도형
- `quadrangle`: 네 각 도형

## 비교 단어

- [[equ]]
  - 핵심: 같음, 균형
  - 구분 포인트: `square`는 네 변과 각이 맞는 도형. `equation`은 양쪽을 같게 맞춘 식

- [[metr]]
  - 핵심: 재다, 척도
  - 구분 포인트: `perimeter`는 둘레를 잰 길이. `square`는 네모난 도형 또는 제곱

- [[uni]]
  - 핵심: 하나
  - 구분 포인트: `uni`는 하나. `quad`는 넷
