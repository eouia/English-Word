# ven

- 출발 표현: `veni, vidi, vici`
- 핵심 어원: 라틴어 `venire` / `vent-` = 오다
- 핵심 이미지: 어떤 곳으로 오거나, 어떤 일이 오듯 발생함
- 관련해서 헷갈릴 수 있는 어근: [[voc]] = 부르다, [[vot]] = 맹세하다

## 출발 표현 분석

`veni, vidi, vici`는 율리우스 카이사르의 말로 알려진 라틴어 표현.

- `veni`: I came = 나는 왔다
- `vidi`: I saw = 나는 보았다
- `vici`: I conquered = 나는 이겼다

여기서 `veni`는 라틴어 `venire` = "오다"의 1인칭 과거형.

영어에서는 주로 `ven-`, `vent-` 형태로 남음.

## 단어들

### convene

- 구성: `con-` = together + `ven`
- 뜻: 모이다, 소집하다
- 어원적 연결: 사람들이 함께 옴
- 사용 맥락: `convene a meeting` = 회의를 소집하다

### convention

- 구성: `con-` + `vent` + `-ion`
- 뜻: 관례, 대회, 협약
- 어원적 연결: 사람들이 함께 와서 정한 방식이나 모임
- 사용 맥락: `social conventions` = 사회적 관습

### covenant

- 구성: Old French `covenant`, from `convenir`
- 뜻: 계약, 서약, 언약
- 어원적 연결: 사람들이 함께 와서 합의한 약속
- 사용 맥락: `a covenant with God` = 신과의 언약
- 철자 주의: `convenant`가 아니라 `covenant`

### convenient

- 구성: Latin `convenire`
- 뜻: 편리한, 알맞은
- 어원적 연결: 여러 조건이 함께 와서 잘 맞음
- 사용 맥락: `a convenient time` = 편리한 시간

### convenience

- 구성: `convenient` 계열 + `-ence`
- 뜻: 편리함, 편의, 편의 시설
- 어원적 연결: 조건들이 함께 맞아 사용하기 쉬운 상태
- 사용 맥락: `for your convenience` = 편의를 위해

### inconvenient

- 구성: `in-` = not + `convenient`
- 뜻: 불편한
- 어원적 연결: 조건이 잘 맞지 않음
- 사용 맥락: `an inconvenient time` = 불편한 시간

### conventional

- 구성: `convention` + `-al`
- 뜻: 관습적인, 전통적인
- 어원적 연결: 함께 정해 온 방식에 따름
- 사용 맥락: `conventional wisdom` = 통념

### intervene

- 구성: `inter-` = between + `ven`
- 뜻: 개입하다, 끼어들다
- 어원적 연결: 사이로 들어옴
- 사용 맥락: `intervene in a dispute` = 분쟁에 개입하다

### intervention

- 구성: `intervene` + `-ion`
- 뜻: 개입, 중재
- 어원적 연결: 사건 사이에 들어와 흐름을 바꿈
- 사용 맥락: `government intervention` = 정부 개입

### event

- 구성: Latin `evenire` = come out, happen
- 뜻: 사건, 행사
- 어원적 연결: 어떤 일이 밖으로 나와 발생함
- 사용 맥락: `a major event` = 중대한 사건

### eventual

- 구성: `event` + `-ual`
- 뜻: 최종적인, 결국의
- 어원적 연결: 시간이 지나 결국 일어나게 되는 것
- 사용 맥락: `the eventual result` = 최종 결과

### prevent

- 구성: `pre-` = before + `vent`
- 뜻: 막다, 예방하다
- 어원적 연결: 원래 "먼저 오다". 어떤 일이 오기 전에 앞질러 막음
- 사용 맥락: `prevent disease` = 질병을 예방하다

### prevention

- 구성: `prevent` + `-ion`
- 뜻: 예방, 방지
- 어원적 연결: 일이 생기기 전에 먼저 막는 행위
- 사용 맥락: `crime prevention` = 범죄 예방

### preventable

- 구성: `prevent` + `-able`
- 뜻: 예방 가능한, 막을 수 있는
- 어원적 연결: 일이 오기 전에 앞질러 막을 수 있음
- 사용 맥락: `preventable disease` = 예방 가능한 질병

### preventability

- 구성: `preventable` + `-ity`
- 뜻: 예방 가능성
- 어원적 연결: 미리 막을 수 있는 성질
- 사용 맥락: 의료·안전 문맥에서 사고나 질병의 예방 가능성을 말할 때 사용

### preventive

- 구성: `prevent` + `-ive`
- 뜻: 예방의, 예방적인
- 어원적 연결: 일이 생기기 전에 막는 성질
- 사용 맥락: `preventive measures` = 예방 조치

### invent

- 구성: `in-` = upon, into + `vent`
- 뜻: 발명하다, 지어내다
- 어원적 연결: 원래 "우연히 찾아오다, 발견하다". 이후 새로운 것을 찾아내고 만들어냄
- 사용 맥락: `invent a device` = 장치를 발명하다

### invention

- 구성: `invent` + `-ion`
- 뜻: 발명, 발명품
- 어원적 연결: 찾아낸 새로운 생각이나 물건
- 사용 맥락: `a useful invention` = 유용한 발명품

### inventory

- 구성: Latin `inventarium`, from `invenire`
- 뜻: 재고, 목록, 재고 목록
- 어원적 연결: 찾아낸 물건들을 적어 둔 목록. 이후 보유 물품 전체나 재고를 가리킴
- 사용 맥락: `check inventory` = 재고를 확인하다

### venue

- 구성: Old French `venue`, from `venir`
- 뜻: 장소, 개최지
- 어원적 연결: 사람들이 오게 되는 장소
- 사용 맥락: `a wedding venue` = 결혼식 장소

### avenue

- 구성: Old French `avenue`, from `avenir`
- 뜻: 대로, 접근 방법
- 어원적 연결: 어떤 곳으로 오거나 접근하는 길
- 사용 맥락: `explore every avenue` = 모든 방법을 모색하다

### advent

- 구성: `ad-` = to + `vent`
- 뜻: 도래, 출현
- 어원적 연결: 어떤 중요한 것이 다가와 나타남
- 사용 맥락: `the advent of the internet` = 인터넷의 등장

### adventure

- 구성: Old French `aventure`, from Latin `advenire`
- 뜻: 모험
- 어원적 연결: 앞으로 다가오는 예상 밖의 일
- 사용 맥락: `go on an adventure` = 모험을 떠나다

### revenue

- 구성: Old French `revenue`, from `revenir`
- 뜻: 수입, 세입
- 어원적 연결: 다시 돌아오는 돈
- 사용 맥락: `tax revenue` = 세수

### souvenir

- 구성: French `souvenir`, from Latin `subvenire`
- 뜻: 기념품
- 어원적 연결: 기억 속으로 떠오르게 하는 것. 원래 "마음에 오다" 이미지
- 사용 맥락: `buy a souvenir` = 기념품을 사다

## 기억 포인트

`ven` 계열은 "오다" 이미지.

- `convene`: 함께 오다
- `covenant`: 함께 와서 맺은 약속
- `intervene`: 사이에 오다
- `event`: 일이 밖으로 나와 발생함
- `prevent`: 먼저 와서 막음
- `preventable`: 먼저 막을 수 있음
- `invent`: 찾아오듯 발견해 만들어냄
- `inventory`: 찾아낸 물건들의 목록
- `venue`: 사람들이 오는 장소
- `advent`: 다가와 나타남
- `revenue`: 다시 돌아오는 돈

## 비교 단어

### venom

- 뜻: 독
- 구분 포인트: `venom`은 라틴어 `venenum` = 독 계열. `venire` = 오다 계열과 직접 묶지 않음

### vend

- 뜻: 팔다
- 구분 포인트: `vend`는 라틴어 `vendere` = 팔다 계열. `venire`와 형태가 비슷하지만 별도로 기억하는 편이 좋음
