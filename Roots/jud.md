# jud

- 출발 단어: `judge`
- 핵심 어원: Latin `iudex` / `iudicium` = 판사, 판단, 재판
- 구성 배경: Latin `ius` = 법, 권리 + `dicere` = 말하다
- 핵심 이미지: 법에 따라 말로 판단을 내림
- 관련 문서: [[court]], [[jur]], [[dict]], [[leg]], [[rect]]
- 주의: [[jur]]는 법·권리·맹세 자체. `jud`는 그 법을 바탕으로 판단하고 판결하는 행위.

## 출발 단어 분석

`judge` ← Old French `juger` ← Latin `iudicare`

- `ius`: 법, 권리
- `dicere`: 말하다, 선언하다
- `iudicare`: 판단하다, 판결하다
- 현대 의미: 판단하다, 판사

## 단어들

### judge

- 발음: /ˈdʒʌdʒ/
- 구성: Latin `iudicare`
- 뜻: 판단하다, 판사
- 어원적 연결: 법에 따라 판단을 말함
- 사용 맥락: `judge the evidence` = 증거를 판단하다

### judgment

- 발음: /ˈdʒʌdʒmənt/
- 구성: `judge` + `-ment`
- 뜻: 판단, 판결, 판단력
- 어원적 연결: 판단이 내려진 결과
- 사용 맥락: `sound judgment` = 건전한 판단력

### judicial

- 발음: /dʒuˈdɪʃəl/
- 구성: Latin `iudicialis`
- 뜻: 사법의, 재판의
- 어원적 연결: 판결과 법원에 관련됨
- 사용 맥락: `judicial review` = 사법 심사

### judiciary

- 발음: /dʒuˈdɪʃiˌɛri/
- 구성: Latin `iudiciarius`
- 뜻: 사법부
- 어원적 연결: 판단과 재판을 담당하는 기관
- 사용 맥락: `an independent judiciary` = 독립된 사법부

### judicious

- 발음: /dʒuˈdɪʃəs/
- 구성: Latin `iudicium` = judgment
- 뜻: 현명한, 판단력 있는
- 어원적 연결: 좋은 판단을 내릴 줄 앎
- 사용 맥락: `a judicious choice` = 현명한 선택

### adjudicate

- 발음: /əˈdʒudɪˌkeɪt/
- 구성: `ad-` = to + `judicate`
- 뜻: 판결하다, 심판하다
- 어원적 연결: 사안에 대해 판단을 내려 줌
- 사용 맥락: `adjudicate a dispute` = 분쟁을 판정하다

### adjudication

- 발음: /əˌdʒudəˈkeɪʃən/
- 구성: `adjudicate` + `-ion`
- 뜻: 판결, 심판
- 어원적 연결: 판단이 공식적으로 내려지는 과정
- 사용 맥락: `legal adjudication` = 법적 판정

### prejudice

- 발음: /ˈprɛdʒədɪs/
- 구성: `pre-` = before + `judicium` = judgment
- 뜻: 편견, 선입견
- 어원적 연결: 충분히 살피기 전에 이미 내린 판단
- 사용 맥락: `racial prejudice` = 인종적 편견

### prejudicial

- 발음: /ˌprɛdʒəˈdɪʃəl/
- 구성: `prejudice` + `-ial`
- 뜻: 해로운, 편견을 주는
- 어원적 연결: 앞선 판단이 해를 끼침
- 사용 맥락: `prejudicial evidence` = 불리한 영향을 주는 증거

## 기억 포인트

`jud` 계열은 "판단과 판결" 이미지.

- `judge`: 판단하다
- `judgment`: 판단 결과
- `judicial`: 재판·사법의
- `judicious`: 판단력 있는
- `prejudice`: 미리 판단한 편견

## 비교 단어

### jur

- 뜻: 법, 권리, 맹세
- 구분 포인트: [[jur]]는 `jury`, `jurisdiction`, `justice`처럼 법적 권리와 맹세의 기반. `jud`는 그 법을 적용해 판단함.
