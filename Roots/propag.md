# propag

- 출발 단어: `propagate`
- 핵심 어원: Latin `propagare` = spread, extend, increase, reproduce
- 핵심 이미지: 뻗어 나가게 함, 번식시킴, 널리 퍼뜨림
- 관련 문서: [[pact]]
- 주의: `propagate`는 `pro-` + `pag`처럼 보이지만 현대 학습에서는 `propag-` 전체를 "퍼뜨리다, 번식시키다"로 묶는 편이 안전함.

## 출발 단어 분석

`propagate` ← Latin `propagare`

- `propagare`: spread, extend, increase, reproduce
- 원래 이미지: 식물이 새 가지나 싹을 뻗어 퍼짐
- 현대 의미: 번식시키다, 전파하다, 퍼뜨리다

## 단어들

### propagate

- 발음: /ˈprɑpəˌgeɪt/
- 구성: Latin `propagare` = spread, reproduce
- 뜻: 번식시키다, 전파하다, 퍼뜨리다
- 어원적 연결: 생명·생각·신호가 뻗어 나가게 함
- 사용 맥락: `propagate plants` = 식물을 번식시키다, `propagate an idea` = 생각을 퍼뜨리다

### propagation

- 발음: /ˌprɑpəˈgeɪʃən/
- 구성: `propagate` + `-ion`
- 뜻: 번식, 전파, 확산
- 어원적 연결: 무언가가 퍼져 나가는 과정
- 사용 맥락: `sound propagation` = 소리의 전파

### propagator

- 구성: `propagate` + `-or`
- 뜻: 전파자, 번식시키는 사람·장치
- 어원적 연결: 어떤 것을 퍼뜨리는 주체
- 사용 맥락: `a propagator of rumors` = 소문을 퍼뜨리는 사람

### propaganda

- 발음: /ˌprɑpəˈgændə/
- 구성: Latin `propaganda` = things to be propagated
- 뜻: 선전, 선전물
- 어원적 연결: 널리 퍼뜨려야 할 생각이나 주장
- 사용 맥락: `political propaganda` = 정치 선전

### propagandist

- 발음: /ˌprɑpəˈgændəst/
- 구성: `propaganda` + `-ist`
- 뜻: 선전가
- 어원적 연결: 특정 주장이나 사상을 퍼뜨리는 사람
- 사용 맥락: `a wartime propagandist` = 전시 선전가

### propagandize

- 발음: /ˌprɑpəˈgæˌndaɪz/
- 구성: `propaganda` + `-ize`
- 뜻: 선전하다, 선전으로 설득하려 하다
- 어원적 연결: 특정 생각을 의도적으로 퍼뜨림
- 사용 맥락: `propagandize the public` = 대중에게 선전하다

## 기억 포인트

`propag` 계열은 "퍼뜨림, 번식, 전파" 이미지.

- `propagate`: 퍼뜨리다, 번식시키다
- `propagation`: 전파, 확산
- `propagator`: 퍼뜨리는 주체
- `propaganda`: 퍼뜨릴 목적으로 만든 주장·선전
- `propagandist`: 선전가

## 비교 단어

### pact

- 뜻: 묶다, 고정하다, 합의하다
- 구분 포인트: [[pact]]는 Latin `pangere`의 "박다, 고정하다, 묶다" 이미지. `propag`는 결과적으로 뻗어 퍼지는 쪽에 초점이 있음.

### spread

- 뜻: 퍼지다, 펼치다
- 구분 포인트: `spread`는 일상 기본어. `propagate`는 생물 번식, 파동·신호 전파, 사상 확산처럼 더 학술적·공식적 맥락에서 자주 씀.
