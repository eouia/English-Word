# miss

- 출발 단어: `mission`
- 핵심 어원: Latin `mittere` = 보내다, 놓아 보내다
- 관련 형태: `mit-`, `miss-`
- 핵심 이미지: 어떤 것을 보내거나 내보냄
- 관련 문서: [[mit]]
- 주의: 이 문서는 [[mit]]의 검색용·형태 보강 문서. 자세한 계열은 [[mit]] 참조.

## 출발 단어 분석

`mission` ← Latin `missio`

- `mittere`: send
- `missio`: sending
- 원래 이미지: 보내진 일, 파견
- 현대 의미: 임무, 사명

## 단어들

### mission

- 구성: Latin `missio`
- 뜻: 임무, 사명, 파견
- 어원적 연결: 어떤 목적을 위해 보내진 일
- 사용 맥락: `complete a mission` = 임무를 완수하다

### admission

- 구성: `ad-` = to + Latin `mittere`
- 뜻: 입장, 입학 허가, 인정
- 어원적 연결: 안으로 들여보냄
- 사용 맥락: `college admission` = 대학 입학 허가

### permission

- 구성: `per-` = through + Latin `mittere`
- 뜻: 허가
- 어원적 연결: 통과해 지나가도록 보내 줌
- 사용 맥락: `ask permission` = 허락을 구하다

### dismissal

- 구성: `dismiss` + `-al`
- 뜻: 해고, 해산, 묵살
- 어원적 연결: 밖으로 보내 버림
- 사용 맥락: `wrongful dismissal` = 부당 해고

### transmission

- 구성: `trans-` = across + Latin `mittere`
- 뜻: 전송, 전달, 전염
- 어원적 연결: 가로질러 보냄
- 사용 맥락: `disease transmission` = 질병 전파

### submission

- 구성: `sub-` = under + Latin `mittere`
- 뜻: 제출, 복종
- 어원적 연결: 아래에 보내어 맡김
- 사용 맥락: `submit an assignment` = 과제를 제출하다

### omission

- 구성: `ob-` = away + Latin `mittere`
- 뜻: 누락, 생략
- 어원적 연결: 보내지 않고 빠뜨림
- 사용 맥락: `an accidental omission` = 우발적 누락

## 기억 포인트

`miss` 계열은 [[mit]]의 과거분사형에서 나온 "보내짐" 이미지.

- `mission`: 보내진 임무
- `admission`: 안으로 들여보냄
- `permission`: 지나가도록 보내 줌
- `dismissal`: 밖으로 보내 버림
- `transmission`: 가로질러 보냄

## 비교 단어

### mit

- 뜻: 보내다, 놓아 보내다
- 구분 포인트: [[mit]]는 기본 동사형. `miss`는 `mission`, `admission`, `permission`처럼 명사형에서 자주 보이는 형태.
