# med

- 출발 단어: `medical`
- 핵심 어원: 라틴어 `mederi` / `medicus` = 치료하다, 의사
- 핵심 이미지: 병을 돌보고 치료함
- 관련 문서: [[iatr]], [[path]]
- 관련해서 헷갈릴 수 있는 어근: [[medi]] = 가운데, 사이, 중간 계열

## 출발 단어 분석

`medical`은 Latin `medicus`에서 온 말.

- `mederi`: 치료하다, 돌보다
- `medicus`: 의사, 치료의
- `medical`: 의학의, 치료와 관련된

따라서 `medical`은 "치료하는 일과 관련된" → "의학의, 의료의" 뜻.

## 단어들

### medical

- 발음: /ˈmɛdəkəl/
- 구성: Latin `medicus` + `-al`
- 뜻: 의학의, 의료의
- 어원적 연결: 치료 행위와 관련됨
- 사용 맥락: `medical treatment` = 의학적 치료

### medicine

- 발음: /ˈmɛdəsən/
- 구성: Latin `medicina`
- 뜻: 의학, 약
- 어원적 연결: 치료하는 기술 또는 치료에 쓰는 물질
- 사용 맥락: `take medicine` = 약을 먹다

### medicinal

- 발음: /məˈdɪsənəl/
- 구성: `medicine` + `-al`
- 뜻: 약효가 있는, 의약의
- 어원적 연결: 치료 효과가 있음
- 사용 맥락: `medicinal herbs` = 약초

### medication

- 발음: /ˌmɛdəˈkeɪʃən/
- 구성: `medicate` + `-ion`
- 뜻: 약물, 약물 치료
- 어원적 연결: 약으로 치료하는 행위나 그 약
- 사용 맥락: `prescribe medication` = 약을 처방하다

### medicate

- 발음: /ˈmɛdɪˌkeɪt/
- 구성: Latin `medicari` = treat medically
- 뜻: 약물로 치료하다
- 어원적 연결: 약이나 의학적 방법으로 다룸
- 사용 맥락: `medicate a patient` = 환자에게 약물 치료를 하다

### medic

- 발음: /ˈmɛdɪk/
- 구성: Latin `medicus`
- 뜻: 의무병, 의료인
- 어원적 연결: 치료하는 사람
- 사용 맥락: `a combat medic` = 전투 의무병

### Medicare

- 발음: /ˈmɛdəˌkɛr/
- 구성: `medi-` + `care`
- 뜻: 미국의 노인·장애인 대상 공적 의료보험
- 어원적 연결: 의료적 돌봄
- 사용 맥락: `Medicare benefits` = 메디케어 혜택

### Medicaid

- 발음: /ˈmɛdəˌkeɪd/
- 구성: `medi-` + `aid`
- 뜻: 미국의 저소득층 대상 공적 의료지원
- 어원적 연결: 의료적 지원
- 사용 맥락: `Medicaid coverage` = 메디케이드 보장

### paramedic

- 발음: /ˌpɛrəˈmɛdɪk/
- 구성: `para-` = beside, alongside + `medic`
- 뜻: 응급구조사
- 어원적 연결: 의사 곁에서 긴급 치료를 돕는 사람
- 사용 맥락: `call the paramedics` = 응급구조사를 부르다

### biomedical

- 발음: /ˌbaɪoʊˈmɛdɪkəl/
- 구성: `bio-` = life + `medical`
- 뜻: 생명의학의
- 어원적 연결: 생명 현상과 의학을 함께 다룸
- 사용 맥락: `biomedical research` = 생명의학 연구

### remedy

- 발음: /ˈrɛmədi/
- 구성: Latin `re-` + `mederi`
- 뜻: 치료법, 해결책, 바로잡다
- 어원적 연결: 다시 건강한 상태로 치료함. 이후 문제 해결책으로 확장
- 사용 맥락: `a remedy for stress` = 스트레스 해소법

### remedial

- 발음: /rɪˈmidiəl/
- 구성: `remedy` + `-al`
- 뜻: 교정의, 보충의, 치료의
- 어원적 연결: 부족하거나 잘못된 상태를 고쳐 줌
- 사용 맥락: `remedial classes` = 보충 수업

## 기억 포인트

`med` 계열은 "치료 / 의료" 이미지.

- `medical`: 치료와 관련됨
- `medicine`: 치료 기술 또는 약
- `medication`: 약물 치료
- `medic`: 치료하는 사람
- `paramedic`: 응급 의료인
- `remedy`: 치료법에서 해결책으로 확장
- `remedial`: 고쳐 주는, 보충하는

## 비교 단어

### medi

- 뜻: 가운데, 사이, 중간
- 구분 포인트: [[medi]]는 Latin `medius` = middle 계열. `mediate`, `medium`, `median`, `immediate`처럼 나타남. `medical`, `medicine`, `remedy`의 `med`는 치료 계열
