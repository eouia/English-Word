# stinct

- 출발 단어: `distinct`
- 핵심 어원: Latin `stinguere` = prick, goad; later press, quench in prefixed forms
- 핵심 이미지: 찌르거나 눌러 표시하고, 갈라 구별하거나 꺼뜨림
- 관련 문서: [[cern]], [[mark]], [[crit]]
- 주의: 이 계열은 의미 변화가 복잡함. `distinct/distinguish`는 "찌르듯 표시해 구별함", `instinct`는 "안에서 찌르는 충동", `extinct/extinguish`는 "눌러 꺼뜨림"으로 갈라 기억.

## 출발 단어 분석

`distinct` ← Latin `distinctus` = separated, marked off

- `dis-`: apart
- `stinguere`: prick, mark, press
- 원래 이미지: 찌르거나 표시를 내어 서로 갈라 놓음
- 의미 변화: 표시해 나눔 → 서로 다른, 분명히 구별되는
- 현대 의미: 뚜렷한, 별개의, 분명한

## 단어들

### distinct

- 발음: /dɪˈstɪŋkt/
- 구성: Latin `dis-` = apart + `stinguere` = prick, mark
- 뜻: 뚜렷한, 별개의, 분명한
- 어원적 연결: 표시를 내어 서로 갈라 구별된 상태
- 사용 맥락: `a distinct difference` = 뚜렷한 차이

### distinction

- 발음: /dɪˈstɪŋkʃən/
- 구성: `distinct` + `-ion`
- 뜻: 구별, 차이; 탁월함
- 어원적 연결: 서로 다르다고 표시해 나누는 것. 남들과 구별될 만큼 뛰어남으로도 확장
- 사용 맥락: `make a distinction` = 구별하다, `graduate with distinction` = 우수한 성적으로 졸업하다

### distinctive

- 발음: /dɪˈstɪŋktɪv/
- 구성: `distinct` + `-ive`
- 뜻: 독특한, 특징적인
- 어원적 연결: 다른 것과 구별되게 만드는 표시나 성질
- 사용 맥락: `a distinctive style` = 독특한 스타일

### distinguish

- 발음: /dɪˈstɪŋgwɪʃ/
- 구성: Latin `distinguere` = separate, mark off, distinguish
- 뜻: 구별하다, 식별하다; 돋보이게 하다
- 어원적 연결: 표시를 내어 서로 다른 것을 알아봄
- 사용 맥락: `distinguish fact from opinion` = 사실과 의견을 구별하다

### distinguished

- 발음: /dɪˈstɪŋgwɪʃt/
- 구성: `distinguish` + `-ed`
- 뜻: 저명한, 뛰어난
- 어원적 연결: 다른 사람과 뚜렷이 구별될 만큼 두드러짐
- 사용 맥락: `a distinguished scholar` = 저명한 학자

### indistinct

- 발음: /ˌɪndɪˈstɪŋkt/
- 구성: `in-` = not + `distinct`
- 뜻: 불분명한, 희미한
- 어원적 연결: 경계나 표시가 또렷하지 않아 구별하기 어려움
- 사용 맥락: `an indistinct outline` = 희미한 윤곽

### instinct

- 발음: /ˈɪnstɪŋkt/
- 구성: Latin `in-` = in + `stinguere` = prick, goad
- 뜻: 본능, 직감
- 어원적 연결: 안에서 찌르듯 밀어붙이는 자연스러운 충동
- 사용 맥락: `survival instinct` = 생존 본능

### instinctive

- 발음: /ˌɪˈnstɪŋktɪv/
- 구성: `instinct` + `-ive`
- 뜻: 본능적인, 직감적인
- 어원적 연결: 의식적 계산보다 안쪽 충동에 의해 움직임
- 사용 맥락: `an instinctive reaction` = 본능적인 반응

### extinct

- 발음: /ɪˈkstɪŋkt/
- 구성: Latin `ex-` = out + `stinguere` = quench, put out
- 뜻: 멸종한, 사라진, 꺼진
- 어원적 연결: 불을 꺼뜨리듯 생명이나 존재가 사라짐
- 사용 맥락: `an extinct species` = 멸종한 종

### extinction

- 발음: /ɪˈkstɪŋkʃən/
- 구성: `extinct` + `-ion`
- 뜻: 멸종, 소멸
- 어원적 연결: 존재의 불이 꺼져 사라지는 과정
- 사용 맥락: `face extinction` = 멸종 위기에 처하다

### extinguish

- 발음: /ɪˈkstɪŋgwɪʃ/
- 구성: Latin `ex-` = out + `stinguere` = quench
- 뜻: 불을 끄다, 없애다
- 어원적 연결: 눌러 꺼뜨려 더 이상 타지 않게 함
- 사용 맥락: `extinguish a fire` = 불을 끄다

## 기억 포인트

`stinct/stinguish` 계열은 "찌르거나 눌러 표시·자극·소멸시킴" 이미지.

- `distinct`: 표시되어 별개로 구별됨
- `distinction`: 구별이나 차이
- `distinguish`: 표시해 구별함
- `instinct`: 안에서 찌르는 본능적 충동
- `extinct`: 꺼져 사라진
- `extinguish`: 꺼뜨림
