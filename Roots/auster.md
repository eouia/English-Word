# auster

- 출발 단어: `austere`
- 핵심 어원: Greek `austeros` = harsh, dry, bitter
- 핵심 이미지: 건조하고 거칠며 꾸밈이 없음
- 관련 문서: 없음
- 주의: `austere`는 현대 형태에서 `aust` + `-ere`처럼 생산적으로 쪼개기보다 Greek `austeros` 전체에서 온 말로 보는 편이 안전함.

## 출발 단어 분석

`austere` ← Greek `austeros` → Latin `austerus` → French/English `austere`

- `austeros`: harsh, dry, bitter
- 원래 이미지: 맛이나 성질이 건조하고 거침
- 의미 변화: 거친 맛 → 엄격한 성격 → 꾸밈과 사치를 줄인 상태
- 현대 의미: 엄격한, 검소한, 장식이 거의 없는

## 단어들

### austere

- 구성: Greek `austeros` = harsh, dry, bitter
- 뜻: 엄격한, 검소한, 꾸밈없는
- 어원적 연결: 단맛이나 부드러움이 빠진 듯 건조하고 거친 성질
- 사용 맥락: `an austere lifestyle` = 검소하고 절제된 생활 방식

### austerity

- 구성: `austere` + `-ity`
- 뜻: 엄격함, 검소함, 긴축
- 어원적 연결: 여유와 장식을 줄이고 필수적인 것만 남김
- 사용 맥락: `austerity measures` = 긴축 조치

### austerely

- 구성: `austere` + `-ly`
- 뜻: 엄격하게, 간소하게, 꾸밈없이
- 어원적 연결: 부드러운 장식 없이 절제된 방식
- 사용 맥락: `austerely furnished` = 간소하게 꾸며진

### austereness

- 구성: `austere` + `-ness`
- 뜻: 엄격함, 검소함, 장식 없음
- 어원적 연결: 거칠고 절제된 성질
- 사용 맥락: `austerity`보다 덜 흔함. 성질 자체를 말할 때 씀.

## 기억 포인트

`auster` 계열은 "건조하고 거친 절제" 이미지.

- `austere`: 엄격한, 검소한, 꾸밈없는
- `austerity`: 검소함, 긴축
- `austerely`: 엄격하게, 간소하게
- `austereness`: 엄격함, 장식 없음

## 비교 단어

### severe

- 뜻: 심한, 엄격한, 가혹한
- 구분 포인트: `severe`는 Latin `severus` = serious, strict 계열. `austere`보다 "심각함/가혹함" 쪽이 강함.

### strict

- 뜻: 엄격한, 엄밀한
- 구분 포인트: 규칙을 단단히 조이는 느낌. `austere`는 생활 방식, 분위기, 디자인이 절제되고 꾸밈없는 느낌까지 포함.

### frugal

- 뜻: 검소한, 절약하는
- 구분 포인트: 돈과 자원을 아끼는 습관. `austere`는 더 차갑고 엄격한 분위기를 줄 수 있음.
