# dign

- 출발 단어: `dignity`
- 핵심 어원: Latin `dignus` = 가치 있는, 알맞은
- 핵심 이미지: 마땅한 가치와 품위
- 관련 문서: [[val]], [[honor]]

## 출발 단어 분석

`dignity` ← Latin `dignitas`

- `dignus`: worthy
- `dignitas`: worth, rank, dignity
- 원래 이미지: 마땅히 존중받을 가치가 있음
- 현대 의미: 존엄, 품위

## 단어들

### dignity

- 구성: Latin `dignitas`
- 뜻: 존엄, 품위
- 어원적 연결: 사람이나 지위가 지닌 마땅한 가치
- 사용 맥락: `human dignity` = 인간의 존엄

### dignify

- 구성: `dign` + `-ify`
- 뜻: 품위 있게 하다, 격을 높이다
- 어원적 연결: 어떤 대상에 가치와 품위를 부여함
- 사용 맥락: `dignify the occasion` = 행사의 품격을 높이다

### dignified

- 구성: `dignify` + `-ed`
- 뜻: 품위 있는, 위엄 있는
- 어원적 연결: 행동이나 태도에 마땅한 품격이 있음
- 사용 맥락: `a dignified response` = 품위 있는 대응

### indignant

- 구성: `in-` = not + Latin `dignus`
- 뜻: 분개한
- 어원적 연결: 마땅하지 않은 대우나 부당함에 화가 남
- 사용 맥락: `indignant at the accusation` = 그 비난에 분개한

### indignation

- 구성: `indignant` + `-ion`
- 뜻: 분개, 의분
- 어원적 연결: 가치에 맞지 않는 부당함에 대한 분노
- 사용 맥락: `public indignation` = 대중의 분노

## 기억 포인트

`dign` 계열은 "마땅한 가치와 품위" 이미지.

- `dignity`: 존엄과 품위
- `dignify`: 품위를 부여함
- `dignified`: 품위 있는
- `indignant`: 부당함에 분개한
- `indignation`: 의분
