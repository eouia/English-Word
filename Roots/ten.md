# ten

- 출발 단어: `maintain`
- 핵심 어원: 라틴어 `tenere` / `tent-` / `tain` = 잡다, 유지하다
- 핵심 이미지: 손에 잡고 놓지 않음
- 관련해서 헷갈릴 수 있는 어근: [[prehend]] = 붙잡다, 이해하다 계열, [[cap]] = 잡다, 받다 계열, [[tend]] = 뻗다, 향하다 계열

## 출발 단어 분석

`maintain`은 Latin `manu tenere`에서 온 말.

- `manu`: by hand
- `tenere`: hold

따라서 `maintain`은 "손으로 붙잡고 유지하다" → "유지하다, 주장하다"의 뜻.

## 단어들

### maintain

- 발음: /meɪˈnteɪn/
- 구성: `manu` = hand + `tain`
- 뜻: 유지하다, 주장하다
- 어원적 연결: 손에 쥐고 계속 보존함
- 사용 맥락: `maintain balance` = 균형을 유지하다

### maintenance

- 발음: /ˈmeɪntənəns/
- 구성: `maintain` + `-ance`
- 뜻: 유지, 보수
- 어원적 연결: 계속 붙잡아 좋은 상태로 둠
- 사용 맥락: `regular maintenance` = 정기 보수

### contain

- 발음: /kəˈnteɪn/
- 구성: `con-` = together + `tain`
- 뜻: 포함하다, 억제하다
- 어원적 연결: 함께 붙잡아 안에 담음
- 사용 맥락: `contain information` = 정보를 포함하다

### container

- 발음: /kəˈnteɪnɚ/
- 구성: `contain` + `-er`
- 뜻: 용기, 컨테이너
- 어원적 연결: 안에 붙잡아 담는 것
- 사용 맥락: `a storage container` = 저장 용기

### retain

- 발음: /rɪˈteɪn/
- 구성: `re-` = back + `tain`
- 뜻: 보유하다, 유지하다
- 어원적 연결: 뒤로 붙잡아 놓지 않음
- 사용 맥락: `retain customers` = 고객을 유지하다

### obtain

- 발음: /əˈbteɪn/
- 구성: `ob-` = toward + `tain`
- 뜻: 얻다, 획득하다
- 어원적 연결: 붙잡아 자기 것으로 얻음
- 사용 맥락: `obtain permission` = 허가를 얻다

### sustain

- 발음: /səˈsteɪn/
- 구성: `sub-` = under + `tain`
- 뜻: 지속하다, 지탱하다
- 어원적 연결: 아래에서 붙잡아 받침
- 사용 맥락: `sustain growth` = 성장을 지속하다

### detain

- 발음: /dɪˈteɪn/
- 구성: `de-` = away, down + `tain`
- 뜻: 붙잡아 두다, 구금하다
- 어원적 연결: 움직이지 못하게 잡아 둠
- 사용 맥락: `detain a suspect` = 용의자를 구금하다

### tenant

- 발음: /ˈtɛnənt/
- 구성: Latin `tenens` = holding
- 뜻: 세입자, 임차인
- 어원적 연결: 땅이나 집을 점유해 가진 사람
- 사용 맥락: `a new tenant` = 새 세입자

### tenure

- 발음: /ˈtɛnjɚ/
- 구성: Latin `tenere`
- 뜻: 재임 기간, 종신 재직권
- 어원적 연결: 직위나 땅을 붙잡고 있는 상태
- 사용 맥락: `academic tenure` = 교수 종신 재직권

### tentative

- 발음: /ˈtɛntətɪv/
- 구성: Latin `tentare` = try, test
- 뜻: 잠정적인, 시험적인
- 어원적 연결: 손으로 만져 보듯 시험함. `tenere`와 가까운 역사적 계열
- 사용 맥락: `a tentative plan` = 잠정적 계획

### tenacious

- 발음: /təˈneɪʃəs/
- 구성: Latin `tenax` = holding fast
- 뜻: 끈질긴, 집요한
- 어원적 연결: 단단히 붙잡고 놓지 않음
- 사용 맥락: `tenacious resistance` = 끈질긴 저항

## 기억 포인트

`ten` 계열은 "붙잡고 유지함" 이미지.

- `maintain`: 손에 쥐고 유지함
- `contain`: 안에 붙잡아 담음
- `retain`: 계속 붙잡아 보유함
- `obtain`: 잡아 얻음
- `sustain`: 아래에서 붙잡아 지탱함
- `detain`: 붙잡아 둠
- `tenacious`: 놓지 않고 끈질김

## 비교 단어

### prehend

- 뜻: 붙잡다, 이해하다
- 구분 포인트: [[prehend]]는 잡는 순간의 이미지. `ten`은 잡고 유지하는 상태가 강함

### cap

- 뜻: 잡다, 받다
- 구분 포인트: [[cap]]은 잡아 받거나 취함. `ten`은 붙잡아 보유함

### tend

- 뜻: 뻗다, 향하다, 팽팽해지다
- 구분 포인트: [[tend]]는 Latin `tendere` 계열. `extend`, `tension`, `attention`은 이쪽. `maintain`, `contain`, `retain`의 `ten`은 Latin `tenere` = 잡다 계열
