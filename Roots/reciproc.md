# reciproc

- 출발 단어: `reciprocate`
- 핵심 어원: Latin `reciprocus` = moving backward and forward
- 핵심 이미지: 오가며 서로 되돌려 줌
- 관련 문서: [[mut]], [[cap]]
- 주의: `receive`, `receptive`의 `recipere`와 철자가 닮았지만, `reciprocate`는 `reciprocus`의 상호 왕복 이미지로 따로 정리.

## 출발 단어 분석

`reciprocate` ← Latin `reciprocus`

- `reciprocus`: backward and forward, alternating
- `-ate`: 동사화 접미사
- 원래 이미지: 앞뒤로 오가며 같은 방향으로 되돌려 줌
- 현대 의미: 보답하다, 화답하다, 상호 교환하다

## 단어들

### reciprocate

- 구성: Latin `reciprocus` + `-ate`
- 뜻: 보답하다, 화답하다, 주고받다
- 어원적 연결: 받은 감정·행동·혜택을 다시 상대에게 돌려줌
- 사용 맥락: `reciprocate kindness` = 친절에 보답하다

### reciprocal

- 구성: Latin `reciprocus` + `-al`
- 뜻: 상호의, 서로 주고받는; 역수의
- 어원적 연결: 양쪽이 서로 오가며 대응함
- 사용 맥락: `a reciprocal relationship` = 상호 관계

### reciprocity

- 구성: `reciprocal` 계열 + `-ity`
- 뜻: 상호성, 호혜, 상호 교환
- 어원적 연결: 한쪽의 행위가 다른 쪽에서 되돌아오는 관계
- 사용 맥락: `the principle of reciprocity` = 상호성의 원칙

### reciprocation

- 구성: `reciprocate` + `-ion`
- 뜻: 보답, 화답, 상호 작용
- 어원적 연결: 받은 것을 다시 되돌려 주는 행위
- 사용 맥락: `without reciprocation` = 보답 없이

### reciprocally

- 구성: `reciprocal` + `-ly`
- 뜻: 상호적으로, 서로
- 어원적 연결: 양쪽이 서로 대응하는 방식
- 사용 맥락: `reciprocally beneficial` = 상호 이익이 되는

## 기억 포인트

`reciproc` 계열은 "오가며 되돌려 줌" 이미지.

- `reciprocate`: 보답하다, 화답하다
- `reciprocal`: 상호의, 서로 대응하는
- `reciprocity`: 상호성, 호혜
- `reciprocation`: 보답, 화답
- `reciprocally`: 상호적으로

## 비교 단어

### mutual

- 뜻: 상호의, 서로의
- 구분 포인트: [[mut]]의 `mutual`은 서로 공유하거나 주고받는 관계. `reciprocal`은 한쪽 행동이 다른 쪽에서 되돌아오는 대응성이 더 강함.

### receptive

- 뜻: 잘 받아들이는, 수용적인
- 구분 포인트: [[cap]]의 `receptive`는 받아들이는 성질. `reciprocate`는 받은 것을 다시 돌려주는 행위.
