# tali

- 출발 단어: `retaliate`
- 핵심 어원: Latin `retaliare` ← `re-` + `talis` = such, of such kind
- 핵심 이미지: 같은 종류로 되갚음
- 관련 문서: [[reciproc]], [[equ]]
- 주의: `retaliate`는 단순한 상호 교환이 아니라 공격·피해에 대한 보복의 느낌이 강함.

## 출발 단어 분석

`retaliate` ← Latin `retaliare`

- `re-`: back
- `talis`: such, such kind
- 원래 이미지: 받은 것과 같은 종류로 되돌려 줌
- 현대 의미: 보복하다, 앙갚음하다

## 단어들

### retaliate

- 구성: `re-` = back + Latin `talis` = such kind
- 뜻: 보복하다, 앙갚음하다
- 어원적 연결: 받은 공격이나 피해를 같은 식으로 되돌려 줌
- 사용 맥락: `retaliate against an attack` = 공격에 보복하다

### retaliation

- 구성: `retaliate` + `-ion`
- 뜻: 보복, 앙갚음
- 어원적 연결: 받은 피해에 대응해 되돌려 주는 행위
- 사용 맥락: `fear of retaliation` = 보복에 대한 두려움

### retaliatory

- 구성: `retaliate` + `-ory`
- 뜻: 보복의, 보복적인
- 어원적 연결: 되갚기 위해 행해지는 성질
- 사용 맥락: `retaliatory measures` = 보복 조치

### retaliative

- 구성: `retaliate` + `-ive`
- 뜻: 보복적인
- 어원적 연결: 받은 공격을 되갚으려는 성질
- 사용 맥락: 드문 단어. 보통 `retaliatory`를 더 많이 씀.

## 기억 포인트

`tali` 계열은 "같은 종류로 되갚음" 이미지.

- `retaliate`: 보복하다
- `retaliation`: 보복
- `retaliatory`: 보복의
- `retaliative`: 보복적인

## 비교 단어

### reciprocate

- 뜻: 보답하다, 화답하다
- 구분 포인트: [[reciproc]]는 받은 것을 되돌려 주는 상호성. `retaliate`는 피해나 공격을 되갚는 보복성.

### equal

- 뜻: 같은, 동등한
- 구분 포인트: [[equ]]는 같음과 균형. `retaliate`는 받은 것과 같은 종류로 되돌려 주는 행동.

### revenge

- 뜻: 복수하다, 복수
- 구분 포인트: `revenge`는 감정적 복수의 느낌이 강함. `retaliate`는 공격에 대한 대응 조치처럼 정치·군사·조직 맥락에서도 자주 씀.
