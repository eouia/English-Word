# cent

- 출발 단어: `century`
- 핵심 어원: Latin `centum` = hundred
- 핵심 의미: 100, 또는 100분의 1
- 관련 문서: [[numer]], [[metr]], [[ped]], [[moon]], [[deci]]
- 주의: `century`의 원래 뜻은 “100년”이 아니라 “100개로 된 집단”. 100년 의미는 뒤에 굳어짐.

## 출발 단어 분석

`century` ← Latin `centuria` = 100개로 된 집단

- `centum`: 100
- `centuria`: 100개 단위, 로마 군대의 100명 단위
- 원래 이미지: 100개를 한 단위로 묶음
- 현대 의미: 100년, 세기

## 단어들

### century

- 발음: /ˈsɛntʃɚi/
- 구성: Latin `centuria` = group of one hundred
- 뜻: 세기, 100년
- 어원적 연결: 100개 단위 → 100년 단위
- 사용 맥락: `the twenty-first century` = 21세기

### cent

- 발음: /ˈsɛnt/
- 구성: Latin `centum` = hundred
- 뜻: 센트, 1달러의 100분의 1
- 어원적 연결: 전체를 100으로 나눈 한 부분
- 사용 맥락: `fifty cents` = 50센트

### percent

- 발음: /pɚˈsɛnt/
- 구성: `per` = by, for each + `cent` = hundred
- 뜻: 퍼센트, 백분율
- 어원적 연결: 100마다 얼마인지 나타냄
- 사용 맥락: `ten percent` = 10퍼센트

### percentage

- 발음: /pɚˈsɛntədʒ/
- 구성: `percent` + `-age`
- 뜻: 비율, 백분율
- 어원적 연결: 100을 기준으로 나타낸 몫
- 사용 맥락: `a high percentage` = 높은 비율

### centennial

- 발음: /sɛˈntɛniəl/
- 구성: `cent` = hundred + Latin `annus` = year
- 뜻: 100년마다의, 100주년
- 어원적 연결: 100년이 된 때
- 사용 맥락: `a centennial celebration` = 100주년 기념식

### bicentennial

- 발음: /ˌbaɪsɛˈntɛniəl/
- 구성: `bi-` = two + `centennial`
- 뜻: 200주년
- 어원적 연결: 100년이 두 번 지난 기념
- 사용 맥락: `the bicentennial of independence` = 독립 200주년

### centenary

- 발음: /ˈsɛntəˌnɛri/
- 구성: Latin `centenarius` = of a hundred
- 뜻: 100주년, 100년의
- 어원적 연결: 100이라는 수와 관련됨
- 사용 맥락: `the centenary of the school` = 학교 100주년

### centenarian

- 발음: /ˌsɛntəˈnɛriən/
- 구성: `centenary` 계열 + `-ian`
- 뜻: 100세 이상인 사람
- 어원적 연결: 100년을 산 사람
- 사용 맥락: `a healthy centenarian` = 건강한 100세 이상 노인

### centimeter

- 발음: /ˈsɛntəˌmitɚ/
- 구성: `centi-` = one hundredth + [[metr]]
- 뜻: 센티미터, 1미터의 100분의 1
- 어원적 연결: 미터를 100으로 나눈 한 단위
- 사용 맥락: `ten centimeters long` = 길이 10센티미터

### centigrade

- 발음: /ˈsɛntəˌgreɪd/
- 구성: `centi-` = hundred + `grade` = step, degree
- 뜻: 섭씨의, 100등분 온도 눈금의
- 어원적 연결: 물의 어는점과 끓는점 사이를 100등분한 눈금
- 사용 맥락: `degrees centigrade` = 섭씨 도

### centipede

- 발음: /ˈsɛntɪˌpid/
- 구성: `centi-` = hundred + [[ped]]
- 뜻: 지네
- 어원적 연결: 발이 100개쯤 되는 듯 많다는 이미지
- 사용 맥락: `a centipede has many legs` = 지네는 다리가 많음

### centurion

- 발음: /sɛˈntʊriən/
- 구성: Latin `centurio` ← `centuria`
- 뜻: 로마 군대의 백인대장
- 어원적 연결: 100명 단위 부대를 이끄는 장교
- 사용 맥락: Roman history 맥락

## 기억 포인트

- `cent`의 기본값은 100.
- 돈·측정에서는 “100분의 1”로 자주 바뀜.
- `century`: 100년.
- `percent`: 100마다 얼마.
- `centimeter`: 1미터의 100분의 1.
- `centurion`: 100명 단위의 지휘관.

## 비교 단어

### numer

- [[numer]]는 수를 세고 숫자로 나타내는 일반 계열.
- `cent`는 특정 숫자 100.

### metr

- [[metr]]는 재다, 측정하다.
- `centimeter`는 `centi-` + `meter`라서 숫자와 측정이 결합된 단어.

### decade

- `decade`는 [[deci]] 문서의 Greek/Latin `deca-/decem` = 10 계열.
- `century`는 100년, `decade`는 10년.
