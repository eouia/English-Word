# onym

- 출발 단어: `synonym`
- 핵심 어원: Greek `onyma` / `onoma` = 이름
- 핵심 이미지: 이름을 붙이고 이름 사이의 관계를 구분함
- 관련 문서: [[nomin]], [[sym]], [[phone]]
- 주의: `synonym`, `antonym`, `homonym`은 단어의 의미·소리 관계를 이름으로 분류하는 말.

## 출발 단어 분석

`synonym` ← Greek `syn-` + `onyma`

- `syn-`: together, same
- `onyma`: name
- 원래 이미지: 같은 이름이나 같은 뜻의 이름
- 현대 의미: 동의어

## 단어들

### synonym

- 발음: /ˈsɪnəˌnɪm/
- 구성: [[sym]] = together, same + `onym` = name
- 뜻: 동의어
- 어원적 연결: 같은 의미를 가리키는 이름
- 사용 맥락: `a synonym for happy` = happy의 동의어

### antonym

- 구성: `anti-` = opposite + `onym`
- 뜻: 반의어
- 어원적 연결: 반대 의미를 가진 이름
- 사용 맥락: `hot and cold are antonyms` = hot과 cold는 반의어

### homonym

- 발음: /ˈhɔmənɪm/
- 구성: `homo-` = same + `onym`
- 뜻: 동음이의어, 동형이의어
- 어원적 연결: 이름이나 형태가 같지만 뜻이 다름
- 사용 맥락: `bank` as a homonym = bank의 동형이의어

### anonymous

- 발음: /əˈnɑnəməs/
- 구성: `an-` = without + `onym`
- 뜻: 익명의
- 어원적 연결: 이름이 없음
- 사용 맥락: `an anonymous source` = 익명의 제보자

### pseudonym

- 발음: /ˈsudəˌnɪm/
- 구성: `pseudo-` = false + `onym`
- 뜻: 가명, 필명
- 어원적 연결: 진짜 이름 대신 쓰는 가짜 이름
- 사용 맥락: `write under a pseudonym` = 필명으로 쓰다

### acronym

- 발음: /ˈækrənɪm/
- 구성: Greek `akros` = tip, end + `onym`
- 뜻: 두문자어
- 어원적 연결: 단어들의 앞머리 글자로 만든 이름
- 사용 맥락: `NASA is an acronym` = NASA는 두문자어

### eponym

- 구성: `epi-` = upon, after + `onym`
- 뜻: 이름의 유래가 된 사람, 그 이름
- 어원적 연결: 누군가의 이름을 따라 붙인 말
- 사용 맥락: `sandwich` as an eponym = sandwich의 이름 유래

## 기억 포인트

`onym` 계열은 "이름" 이미지.

- `synonym`: 같은 의미의 이름
- `antonym`: 반대 의미의 이름
- `anonymous`: 이름이 없음
- `pseudonym`: 가짜 이름
- `acronym`: 앞글자로 만든 이름

## 비교 단어

### nomin

- 뜻: 이름, 이름 붙이다
- 구분 포인트: [[nomin]]은 라틴어 이름 계열. `onym`은 그리스어 이름 계열.

### phone

- 뜻: 소리
- 구분 포인트: [[phone]]은 소리. `homophone`은 소리가 같은 말, `homonym`은 이름·형태가 같은 말.
