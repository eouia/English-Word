# photo

- 출발 단어: `photograph`
- 핵심 어원: Greek `phos` / `photo-` = 빛
- 핵심 이미지: 빛으로 드러내고 기록함
- 관련 문서: [[luc]], [[graph]], [[thes]]

## 단어들

### photograph

- 구성: `photo` = light + [[graph]] = write
- 뜻: 사진
- 어원적 연결: 빛으로 기록한 것
- 사용 맥락: `take a photograph` = 사진을 찍다

### photography

- 구성: `photo` + `graphy`
- 뜻: 사진술
- 어원적 연결: 빛으로 이미지를 기록하는 기술
- 사용 맥락: `digital photography` = 디지털 사진술

### photosynthesis

- 구성: `photo` = light + `synthesis` = putting together
- 뜻: 광합성
- 어원적 연결: 빛을 이용해 물질을 합성함
- 사용 맥락: `plants perform photosynthesis` = 식물이 광합성을 하다

### photogenic

- 구성: `photo` + `-genic`
- 뜻: 사진이 잘 받는
- 어원적 연결: 빛과 사진에서 잘 드러남
- 사용 맥락: `a photogenic landscape` = 사진이 잘 받는 풍경

### photon

- 구성: Greek `phos`
- 뜻: 광자
- 어원적 연결: 빛의 입자
- 사용 맥락: 물리학 맥락에서 쓰임

## 기억 포인트

`photo` 계열은 "빛" 이미지.

- `photograph`: 빛으로 기록함
- `photography`: 빛 기록 기술
- `photosynthesis`: 빛으로 합성함
- `photogenic`: 빛과 사진에 잘 드러남

## 비교 단어

### luc

- 뜻: 빛, 빛나다
- 구분 포인트: [[luc]]는 라틴어 빛 계열. `photo`는 그리스어 빛 계열.
