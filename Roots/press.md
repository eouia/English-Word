# press

- 출발 단어: `pressure`
- 핵심 어원: 라틴어 `premere` / `press-` = 누르다
- 핵심 이미지: 힘을 주어 누름
- 관련해서 헷갈릴 수 있는 어근: [[strain]] = 조이다, 제한하다 계열

## 출발 단어 분석

`pressure`는 Latin `pressura`에서 온 말.

- `premere`: 누르다
- `press-`: 눌린, 누른 형태

따라서 `pressure`는 "누르는 힘" → "압력, 압박"의 뜻.

## 단어들

### pressure

- 구성: `press` + `-ure`
- 뜻: 압력, 압박
- 어원적 연결: 힘으로 누름
- 사용 맥락: `under pressure` = 압박을 받는

### press

- 구성: Latin `pressare`
- 뜻: 누르다, 압박하다, 언론
- 어원적 연결: 종이를 눌러 찍어 내던 인쇄기에서 언론의 뜻도 생김
- 사용 맥락: `press the button` = 버튼을 누르다

### compress

- 구성: `com-` = together + `press`
- 뜻: 압축하다
- 어원적 연결: 함께 눌러 작게 만듦
- 사용 맥락: `compress a file` = 파일을 압축하다

### compression

- 구성: `com-` + `press` + `-ion`
- 뜻: 압축
- 어원적 연결: 눌러 줄임
- 사용 맥락: `data compression` = 데이터 압축

### express

- 구성: `ex-` = out + `press`
- 뜻: 표현하다, 명시적인, 급행의
- 어원적 연결: 눌러 밖으로 짜냄. 생각을 밖으로 드러냄. 길이나 절차를 막힘없이 곧장 빼낸다는 뜻에서 "급행"도 됨
- 사용 맥락: `express an opinion` = 의견을 표현하다

### expression

- 구성: `ex-` + `press` + `-ion`
- 뜻: 표현, 표정
- 어원적 연결: 안의 것을 밖으로 짜내 드러냄
- 사용 맥락: `facial expression` = 얼굴 표정

### expressive

- 구성: `express` + `-ive`
- 뜻: 표현력이 있는, 표정이 풍부한
- 어원적 연결: 안의 감정이나 생각을 밖으로 잘 드러냄
- 사용 맥락: `an expressive face` = 표정이 풍부한 얼굴

### expressly

- 구성: `express` + `-ly`
- 뜻: 명백히, 특별히
- 어원적 연결: 뜻을 밖으로 분명히 드러내어 말함
- 사용 맥락: `expressly forbidden` = 명시적으로 금지된

### expressway

- 구성: `express` + `way`
- 뜻: 고속도로
- 어원적 연결: 목적지까지 빠르게 곧장 가게 만든 길
- 사용 맥락: `take the expressway` = 고속도로를 타다

### impress

- 구성: `in-` = into + `press`
- 뜻: 깊은 인상을 주다
- 어원적 연결: 마음 안에 눌러 찍음
- 사용 맥락: `impress the audience` = 청중에게 깊은 인상을 주다

### impression

- 구성: `in-` + `press` + `-ion`
- 뜻: 인상, 자국
- 어원적 연결: 눌러 찍힌 흔적
- 사용 맥락: `make a strong impression` = 강한 인상을 남기다

### depress

- 구성: `de-` = down + `press`
- 뜻: 우울하게 하다, 낮추다
- 어원적 연결: 아래로 눌러 가라앉힘
- 사용 맥락: `depress prices` = 가격을 낮추다

### depression

- 구성: `de-` + `press` + `-ion`
- 뜻: 우울증, 불황, 움푹 들어간 곳
- 어원적 연결: 아래로 눌려 내려간 상태
- 사용 맥락: `economic depression` = 경제 불황

### suppress

- 구성: `sub-` = under + `press`
- 뜻: 억누르다, 진압하다
- 어원적 연결: 아래로 눌러 드러나지 못하게 함
- 사용 맥락: `suppress evidence` = 증거를 은폐하다

### oppress

- 구성: `ob-` = against + `press`
- 뜻: 억압하다
- 어원적 연결: 맞서 세게 눌러 짓누름
- 사용 맥락: `oppress minorities` = 소수자를 억압하다

### repress

- 구성: `re-` = back + `press`
- 뜻: 억누르다, 참다
- 어원적 연결: 뒤로 눌러 밀어 넣음
- 사용 맥락: `repress emotions` = 감정을 억누르다

### pressurize

- 구성: `pressure` + `-ize`
- 뜻: 압력을 가하다, 압력을 유지하다
- 어원적 연결: 압력 상태로 만듦
- 사용 맥락: `pressurize the cabin` = 객실 기압을 유지하다

## 기억 포인트

`press` 계열은 "누름" 이미지.

- `compress`: 함께 눌러 압축함
- `express`: 밖으로 눌러 짜내 표현함
- `expressly`: 분명히 밖으로 드러내 말함
- `impress`: 안에 눌러 찍어 인상을 남김
- `depress`: 아래로 눌러 가라앉힘
- `suppress`: 아래로 눌러 드러나지 못하게 함
- `oppress`: 맞서 짓누름
- `repress`: 뒤로 눌러 억제함
