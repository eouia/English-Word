# fort

- 출발 단어: `force`
- 핵심 어원: Latin `fortis` = 강한, 힘센
- 관련 형태: Old French `force` = 힘, 강제
- 핵심 이미지: 강한 힘으로 버티거나 밀어붙임
- 관련 문서: [[dynam]], [[val]], [[crat]], [[press]]
- 주의: `fort` = 요새는 "강하게 만든 곳"의 이미지. `force`와 같은 Latin `fortis` 계열.

## 출발 단어 분석

`force` ← Old French `force` ← Latin `fortis`

- `fortis`: strong
- Old French `force`: strength, power, violence
- 원래 이미지: 강한 힘
- 현대 의미: 힘, 무력, 강제하다

## 단어들

### force

- 구성: Old French `force` ← Latin `fortis`
- 뜻: 힘, 무력, 강제하다
- 어원적 연결: 강한 힘으로 밀어붙임
- 사용 맥락: `use force` = 힘을 사용하다

### forceful

- 구성: `force` + `-ful`
- 뜻: 강력한, 단호한
- 어원적 연결: 힘이 강하게 드러남
- 사용 맥락: `a forceful argument` = 강력한 주장

### forcible

- 구성: `force` + `-ible`
- 뜻: 강제적인, 힘으로 하는
- 어원적 연결: 물리적 힘이나 강제로 이루어짐
- 사용 맥락: `forcible removal` = 강제 퇴거

### enforce

- 구성: `en-` = make, put into + `force`
- 뜻: 시행하다, 집행하다, 강제하다
- 어원적 연결: 규칙에 힘을 실어 실제로 따르게 함
- 사용 맥락: `enforce the law` = 법을 집행하다

### enforcement

- 구성: `enforce` + `-ment`
- 뜻: 집행, 시행
- 어원적 연결: 힘을 실어 규칙을 작동하게 하는 행위
- 사용 맥락: `law enforcement` = 법 집행

### reinforce

- 구성: `re-` = again, back + `enforce`
- 뜻: 강화하다, 보강하다
- 어원적 연결: 힘을 다시 더해 튼튼하게 함
- 사용 맥락: `reinforce the structure` = 구조물을 보강하다

### reinforcement

- 구성: `reinforce` + `-ment`
- 뜻: 보강, 강화, 지원 병력
- 어원적 연결: 힘을 더해 버티게 만드는 것
- 사용 맥락: `positive reinforcement` = 긍정적 강화

### fort

- 구성: Latin `fortis`
- 뜻: 요새
- 어원적 연결: 강하게 만든 방어 장소
- 사용 맥락: `build a fort` = 요새를 짓다

### fortress

- 구성: `fort` + `-ress`
- 뜻: 요새, 견고한 장소
- 어원적 연결: 공격에 버틸 만큼 강한 곳
- 사용 맥락: `a mountain fortress` = 산악 요새

### fortify

- 구성: `fortis` = strong + `-fy` = make
- 뜻: 강화하다, 요새화하다
- 어원적 연결: 강하게 만들거나 방어력을 더함
- 사용 맥락: `fortify the walls` = 성벽을 강화하다

### fortitude

- 구성: Latin `fortitudo` = strength
- 뜻: 용기, 인내력
- 어원적 연결: 어려움을 버티는 마음의 힘
- 사용 맥락: `show fortitude` = 인내심과 용기를 보이다

### effort

- 구성: Old French `esforcier` = exert force
- 뜻: 노력
- 어원적 연결: 힘을 밖으로 내어 씀
- 사용 맥락: `make an effort` = 노력하다

## 기억 포인트

`fort` 계열은 "강한 힘" 이미지.

- `force`: 힘
- `enforce`: 힘을 실어 집행함
- `reinforce`: 힘을 더해 보강함
- `fort`: 강한 방어 장소
- `fortify`: 강하게 만듦
- `fortitude`: 버티는 마음의 힘
- `effort`: 힘을 내어 씀

- 뜻: 힘, 능력
- 구분 포인트: [[dynam]]은 움직임을 만드는 힘과 능력. `fort`는 강함, 강제, 방어, 버팀의 힘.

### val

- 뜻: 힘이 있다, 가치가 있다
- 구분 포인트: [[val]]은 힘에서 가치와 유효성으로 확장. `fort`는 강한 힘 자체와 강화 행위.

### press

- 뜻: 누르다, 압박하다
- 구분 포인트: [[press]]는 힘을 가하는 방식이 "누름". `fort`는 강함과 강제의 일반 이미지.
