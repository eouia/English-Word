# charact

- 출발 단어: `characterization`
- 핵심 어원: Greek `kharakter` = engraved mark, distinctive mark
- 핵심 이미지: 새겨진 표시가 어떤 대상을 구별함
- 관련 문서: [[graph]], [[typ]]
- 주의: `character`는 "문자", "성격", "등장인물" 뜻이 모두 있음. 공통 이미지는 구별되게 새겨진 표시나 특징.

## 출발 단어 분석

`characterization` = `character` + `-ize` + `-ation`

- `character`: distinctive mark, nature, person in a story
- `-ize`: ~하게 만들다, ~로 나타내다
- `-ation`: 명사화
- 원래 이미지: 대상의 특징을 표시해 드러냄
- 현대 의미: 묘사, 성격 부여, 특징 규정

## 단어들

### character

- 구성: Greek `kharakter` = engraved mark
- 뜻: 성격, 특징; 등장인물; 문자
- 어원적 연결: 새겨진 표식처럼 어떤 사람·대상을 구별하게 하는 특징
- 사용 맥락: `a complex character` = 복합적인 인물

### characteristic

- 구성: `character` + `-istic`
- 뜻: 특징적인; 특징
- 어원적 연결: 대상을 알아보게 하는 고유한 표시
- 사용 맥락: `a characteristic feature` = 특징적인 요소

### characteristically

- 구성: `characteristic` + `-ally`
- 뜻: 특징적으로, 그 사람답게
- 어원적 연결: 고유한 특징이 드러나는 방식
- 사용 맥락: `characteristically direct` = 그 사람답게 직설적인

### characterize

- 구성: `character` + `-ize`
- 뜻: 특징짓다, 묘사하다
- 어원적 연결: 어떤 대상의 구별되는 표지를 드러냄
- 사용 맥락: `characterize the era` = 그 시대를 특징짓다

### characterization

- 구성: `characterize` + `-ation`
- 뜻: 묘사, 특징 규정; 인물 조형
- 어원적 연결: 인물이나 대상을 특정한 특징으로 드러내는 과정
- 사용 맥락: `the characterization of the hero` = 주인공의 인물 묘사

### characterized

- 구성: `characterize` + `-ed`
- 뜻: 특징지어진, 묘사된
- 어원적 연결: 특정 특징으로 표시된 상태
- 사용 맥락: `a period characterized by change` = 변화로 특징지어지는 시기

### caricature

- 구성: Italian `caricatura` = a loading, exaggeration
- 뜻: 풍자화, 희화화
- 어원적 연결: 엄밀히는 `character`와 다른 계열이지만, 인물의 특징을 과장해 드러낸다는 학습 연결이 있음
- 사용 맥락: `a political caricature` = 정치 풍자화

## 기억 포인트

`charact` 계열은 "구별되게 새겨진 표시와 특징" 이미지.

- `character`: 성격, 특징, 등장인물, 문자
- `characteristic`: 특징적인, 특징
- `characterize`: 특징짓다
- `characterization`: 묘사, 특징 규정
- `characterized`: 특징지어진

## 비교 단어

### graph

- 뜻: 쓰다, 그리다, 기록하다
- 구분 포인트: [[graph]]는 긁어 쓰거나 그리는 행위. `charact`는 새겨진 표시가 대상의 특징이 되는 쪽.

### typ

- 뜻: 찍힌 자국, 유형, 본보기
- 구분 포인트: [[typ]]은 찍힌 모양에서 유형·전형으로 확장. `character`는 구별 표지에서 성격·인물·문자로 확장.

### charisma

- 뜻: 카리스마, 매력적 권위
- 구분 포인트: Greek `kharisma` = gift, favor 계열. 철자와 발음이 비슷해도 `character`와 어원은 다름.
