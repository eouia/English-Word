# cord

- 출발 단어: `record`
- 핵심 어원: 라틴어 `cor`, `cordis` = heart, 마음, 심장
- 핵심 이미지: 마음속 중심 / 마음을 함께함 / 마음에 다시 새김
- 관련해서 헷갈릴 수 있는 어근: [[card]] = Greek `kardia`, 심장 계열, `cord` = 줄, 끈 계열은 별도 어원

## 출발 단어 분석

`record`는 `re-` + `cord`.

- `re-`: again, back
- `cord`: heart, mind

따라서 `record`는 원래 "마음에 다시 떠올리다, 기억하다"의 뜻. 여기서 "기억해 둔 것" → "글로 남긴 기록" → "공식 기록, 녹음, 음반"으로 확장.

## 단어들

### record

- 발음: n. /ˈrɛkɚd/, v. /rɪˈkɔrd/
- 구성: `re-` = again, back + `cord`
- 뜻: 기록하다, 기록, 음반
- 어원적 연결: 마음에 다시 새기거나 떠올림
- 사용 맥락: `record the results` = 결과를 기록하다, `medical records` = 의료 기록

### recording

- 발음: /rəˈkɔrdɪŋ/
- 구성: `record` + `-ing`
- 뜻: 녹음, 녹화, 기록 행위
- 어원적 연결: 소리나 장면을 나중에 다시 떠올릴 수 있게 남김
- 사용 맥락: `a video recording` = 동영상 녹화물

### recorder

- 발음: /rɪˈkɔrdɚ/
- 구성: `record` + `-er`
- 뜻: 기록자, 녹음기, 리코더
- 어원적 연결: 기록하거나 다시 들을 수 있게 남기는 사람/도구
- 사용 맥락: `a voice recorder` = 음성 녹음기

### accord

- 발음: /əˈkɔrd/
- 구성: `ad-` = to + `cord`
- 뜻: 일치, 합의; 주다
- 어원적 연결: 마음이 한쪽으로 맞춰짐
- 사용 맥락: `reach an accord` = 합의에 이르다

### accordance

- 발음: /əˈkɔrdəns/
- 구성: `accord` + `-ance`
- 뜻: 일치, 부합
- 어원적 연결: 마음이나 기준이 맞아 있는 상태
- 사용 맥락: `in accordance with the rules` = 규칙에 따라

### concord

- 발음: /ˈkɑˌnkɔrd/
- 구성: [[con]] = together + `cord`
- 뜻: 화합, 일치
- 어원적 연결: 마음이 함께함
- 사용 맥락: `social concord` = 사회적 화합

### discord

- 발음: /ˈdɪskɔrd/
- 구성: `dis-` = apart + `cord`
- 뜻: 불화, 불일치
- 어원적 연결: 마음이 갈라짐
- 사용 맥락: `political discord` = 정치적 불화

### cordial

- 발음: /ˈkɔrdʒəl/
- 구성: `cord` + `-ial`
- 뜻: 진심 어린, 다정한
- 어원적 연결: 마음에서 우러난 태도
- 사용 맥락: `a cordial welcome` = 진심 어린 환영

### courage

- 발음: /ˈkɝədʒ/
- 구성: Old French `corage` = heart, spirit
- 뜻: 용기
- 어원적 연결: 마음의 힘, 심지가 강함
- 사용 맥락: `show courage` = 용기를 보이다

### encourage

- 발음: /ɛˈnkɝɪdʒ/
- 구성: [[em]]의 `en-` = make, put into + `courage`
- 뜻: 격려하다, 용기를 주다
- 어원적 연결: 마음에 힘을 넣어 줌
- 사용 맥락: `encourage students` = 학생들을 격려하다

### discourage

- 발음: /dɪˈskɝɪdʒ/
- 구성: `dis-` = away, reverse + `courage`
- 뜻: 낙담시키다, 막다
- 어원적 연결: 마음의 힘을 빼앗음
- 사용 맥락: `discourage smoking` = 흡연을 억제하다

## 기억 포인트

`cord` 계열은 "심장"에서 "마음, 기억, 합의, 용기"로 확장.

- `record`: 마음에 다시 새김 → 기록
- `accord`: 마음이 맞음 → 합의
- `concord`: 마음이 함께함 → 화합
- `discord`: 마음이 갈라짐 → 불화
- `cordial`: 마음에서 나옴 → 진심 어린
- `courage`: 마음의 힘 → 용기

## 비교 단어

### cord

- 뜻: 줄, 끈
- 구분 포인트: 영어 `cord` = string, rope는 별도 어원. `record`, `accord`, `cordial`의 `cord` = heart 계열과 섞지 않음

### cardiac

- 뜻: 심장의
- 구분 포인트: Greek `kardia` 계열. 의미는 가깝지만 라틴어 `cor`, `cordis` 계열과 형태가 다름
