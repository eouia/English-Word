# plic-apply

- 출발 단어: `apply`
- 핵심 어원: Latin `applicare` = 접어 붙이다
- 핵심 이미지: 어떤 것을 대상에 접어 붙여 씀
- 허브 문서: [[plic]]

## 단어들

### apply

- 발음: /əˈplaɪ/
- 구성: `ad-` = to + `plic` = fold
- 뜻: 적용하다, 지원하다, 바르다
- 어원적 연결: 어떤 것을 대상에 접어 붙임
- 사용 맥락: `apply the rule` = 규칙을 적용하다

### application

- 발음: /ˌæpləˈkeɪʃən/
- 구성: `apply` + `-ation`
- 뜻: 적용, 신청, 지원서, 앱
- 어원적 연결: 어떤 것을 실제 대상에 붙여 쓰거나, 자신을 기회에 붙임
- 사용 맥락: `submit an application` = 지원서를 제출하다

### applicant

- 발음: /ˈæplɪkənt/
- 구성: `apply` + `-ant`
- 뜻: 지원자, 신청자
- 어원적 연결: 어떤 자리나 기회에 자신을 붙여 내미는 사람
- 사용 맥락: `job applicants` = 구직 지원자들

### applicable

- 발음: /ˈæpləkəbəl/
- 구성: `apply` + `-able`
- 뜻: 적용 가능한, 해당되는
- 어원적 연결: 특정 대상에 붙여 쓸 수 있음
- 사용 맥락: `applicable laws` = 적용 가능한 법

### applied

- 발음: /əˈplaɪd/
- 구성: `apply` + `-ed`
- 뜻: 적용된, 응용의
- 어원적 연결: 이론을 실제 대상에 붙여 씀
- 사용 맥락: `applied science` = 응용 과학

## 기억 포인트

`plic-apply`는 "접어 붙여 적용함" 갈래.

- `apply`: 붙여 쓰다
- `application`: 적용, 지원
- `applicant`: 지원자
- `applicable`: 붙여 쓸 수 있음
