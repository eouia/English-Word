# trans

- 출발 단어: `transform`
- 핵심 의미: 가로질러, 건너, 넘어서
- 관련 형태: Latin `trans` = across, beyond, through
- 관련 문서: [[form]], [[fer]], [[mit]], [[beyond]]

## 핵심 이미지

`trans-`의 바탕은 "한쪽에서 다른 쪽으로 건너감".

형태를 건너 바꾸면 `transform`.
물건이나 권리를 건너 옮기면 `transfer`.
정보를 건너 보내면 `transmit`.
뜻을 다른 언어로 건너 옮기면 `translate`.
상태가 넘어가면 `transition`.

## 단어들

### trans

- 발음: /ˈtrænz/
- 구성: Latin `trans` = across, beyond
- 뜻: 가로질러, 건너, 넘어서
- 연결: 경계나 거리의 한쪽에서 다른 쪽으로 감
- 사용 맥락: Latin 계열 접두사로 자주 쓰임.

### transform

- 발음: /træˈnsfɔrm/
- 구성: `trans-` = across/beyond + `form`
- 뜻: 변형하다, 완전히 바꾸다
- 연결: 기존 형태를 넘어 다른 형태로 바꿈
- 사용 맥락: `transform the system` = 시스템을 바꾸다

### transfer

- 발음: n. /ˈtrænsfɚ/, v. /træˈnsfɝ/
- 구성: `trans-` = across + `fer` = carry
- 뜻: 옮기다, 이전하다, 전송하다
- 연결: 한쪽에서 다른 쪽으로 실어 나름
- 사용 맥락: `transfer money` = 돈을 이체하다

### transmit

- 발음: /træˈnzmɪt/
- 구성: `trans-` = across + `mit` = send
- 뜻: 전송하다, 전달하다
- 연결: 정보를 건너 보내 전달함
- 사용 맥락: `transmit data` = 데이터를 전송하다

### translate

- 발음: /træˈnzleɪt/
- 구성: `trans-` = across + Latin `lat-` = carried
- 뜻: 번역하다
- 연결: 뜻을 한 언어에서 다른 언어로 건너 옮김
- 사용 맥락: `translate Korean into English` = 한국어를 영어로 번역하다

### transition

- 발음: /træˈnzɪʃən/
- 구성: `trans-` = across + Latin `ire/it-` = go
- 뜻: 전환, 이행
- 연결: 한 상태에서 다른 상태로 건너감
- 사용 맥락: `a smooth transition` = 순조로운 전환

### transparent

- 발음: /træˈnspɛrənt/
- 구성: `trans-` = through + Latin `parere` = appear
- 뜻: 투명한, 속이 보이는
- 연결: 빛이나 시야가 통과해 나타남
- 사용 맥락: `transparent glass` = 투명한 유리

### transaction

- 발음: /træˈnzækʃən/
- 구성: `trans-` = across + `act`
- 뜻: 거래, 처리
- 연결: 사람들 사이를 오가며 이루어지는 행위
- 사용 맥락: `a financial transaction` = 금융 거래

## 비교 단어

`beyond`는 [[beyond]] 문서처럼 Germanic 기본어.
`trans-`는 라틴계 접두사로, 단어 안에서 "건너감, 전환"의 방향을 더함.

`super-`는 [[super]] 문서처럼 위에, 넘어서.
`para-`는 [[para]] 문서처럼 옆에, 넘어서, 반대로.
