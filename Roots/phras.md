# phras

- 출발 단어: `phrase`
- 핵심 어원: 그리스어 `phrazein` = 말하다, 선언하다, 가리키다, 표현하다
- 핵심 의미: 생각을 말로 표현한 묶음
- 관련 문서: [[logos]], [[dict]], [[fari]], [[tex]]
- 주의: `phrase`는 “문구”뿐 아니라 “표현 방식”의 느낌도 강함.

## 출발 단어 분석

`phrase` ← Late Latin `phrasis` ← Greek `phrasis`

- `phrazein`: 말하다, 드러내다, 알려 주다
- `phrasis`: 말하기, 표현 방식
- 원래 이미지: 생각이나 뜻을 말로 가리켜 드러냄
- 현대 의미: 구, 문구, 표현

## 단어들

### phrase

- 발음: /ˈfreɪz/
- 구성: Greek `phrasis`
- 뜻: 구, 문구, 표현
- 어원적 연결: 뜻을 말로 표현한 작은 묶음
- 사용 맥락: `a common phrase` = 흔한 표현

### phrasal

- 구성: `phrase` + `-al`
- 뜻: 구의, 구로 된
- 어원적 연결: 단어들이 묶여 하나의 표현 단위를 이룸
- 사용 맥락: `phrasal verb` = 구동사

### phrasing

- 발음: /ˈfreɪzɪŋ/
- 구성: `phrase` + `-ing`
- 뜻: 표현 방식, 말의 구성
- 어원적 연결: 뜻을 어떤 말의 묶음으로 표현하는가
- 사용 맥락: `careful phrasing` = 신중한 표현

### rephrase

- 발음: /riˈfreɪz/
- 구성: `re-` = again + `phrase`
- 뜻: 다시 표현하다, 바꾸어 말하다
- 어원적 연결: 같은 뜻을 다른 말의 묶음으로 다시 드러냄
- 사용 맥락: `rephrase the question` = 질문을 바꾸어 말하다

### paraphrase

- 발음: /ˈpɛrəˌfreɪz/
- 구성: `para-` = beside + `phrase`
- 뜻: 바꾸어 말하다, 의역하다
- 어원적 연결: 원문 곁에 다른 말로 뜻을 풀어 둠
- 사용 맥락: `paraphrase a paragraph` = 문단을 바꾸어 말하다

### phraseology

- 발음: /ˌfreɪziˈɔlɔdʒi/
- 구성: `phrase` + `-ology`
- 뜻: 표현법, 어법, 문구 모음
- 어원적 연결: 어떤 분야나 사람의 말 표현 방식
- 사용 맥락: `legal phraseology` = 법률 표현법

### catchphrase

- 구성: `catch` + `phrase`
- 뜻: 유행어, 대표 문구
- 어원적 연결: 사람들의 귀를 붙잡는 표현
- 사용 맥락: `a campaign catchphrase` = 캠페인 구호

### set phrase

- 구성: `set` + `phrase`
- 뜻: 정해진 표현, 관용구
- 어원적 연결: 고정된 말의 묶음
- 사용 맥락: `a set phrase in English` = 영어의 정해진 표현

### periphrasis

- 구성: `peri-` = around + `phrasis`
- 뜻: 우회적 표현, 돌려 말하기
- 어원적 연결: 직접 말하지 않고 둘러서 표현함
- 사용 맥락: `use periphrasis` = 우회 표현을 쓰다

### antiphrasis

- 구성: `anti-` = opposite + `phrasis`
- 뜻: 반어적 표현
- 어원적 연결: 본뜻과 반대되는 말로 표현함
- 사용 맥락: `antiphrasis in irony` = 아이러니 속 반어 표현

## 기억 포인트

- `phras`는 “뜻을 말로 표현함”.
- `phrase`: 뜻을 담은 말의 묶음.
- `rephrase`: 다시 표현함.
- `paraphrase`: 같은 뜻을 다른 말로 풀어 씀.
- `phraseology`: 특정 분야나 사람의 표현 방식.

## 비교 단어

### logos

- [[logos]]는 말, 설명, 이성, 학문 계열.
- `phras`는 이성 체계보다 “표현 방식/문구”에 초점.

### dict

- [[dict]]는 말하다, 선언하다, 말로 정하다.
- `phrase`는 정해진 말의 묶음이나 표현 방식에 가까움.

### expression

- [[press]]의 `expression`은 “밖으로 눌러 드러냄”.
- `phrase`는 드러난 말의 구체적 형태, 문구.
