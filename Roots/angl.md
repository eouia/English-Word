# angl

- 출발 단어: `angle`
- 핵심 어원: Latin `angulus` = corner, angle
- 핵심 이미지: 꺾인 모서리, 두 선이 만나는 각
- 관련 어근: [[quad]], [[metr]], [[rect]]

## 출발 단어 분석

`angle`은 Latin `angulus` = corner, angle에서 온 말.

핵심은 "꺾인 모서리"이며, 수학에서는 두 선이나 면이 만나 이루는 벌어진 정도를 뜻함.

낚싯바늘을 뜻하던 Old English `angel`도 "굽은 갈고리" 이미지라 의미상 가깝지만, 현대 학습에서는 수학의 `angle`을 중심으로 잡는 편이 좋음.

## 단어들

### angle

- 구성: Latin `angulus`
- 의미: 각, 각도, 관점
- 어원적 연결: 두 선이 꺾여 만나는 모서리
- 사용 맥락: `a 90-degree angle` = 90도 각

### angular

- 구성: `angle` + `-ar`
- 의미: 각진, 모난
- 어원적 연결: 둥글지 않고 각이 드러남
- 사용 맥락: `angular features` = 각진 이목구비

### triangle

- 구성: [[tri]] = three + `angle`
- 의미: 삼각형
- 어원적 연결: 세 각을 가진 도형
- 사용 맥락: `the area of a triangle` = 삼각형의 넓이

### rectangle

- 구성: Latin `rectus` = right, straight + `angle`
- 의미: 직사각형
- 어원적 연결: 바른 각, 즉 직각을 가진 도형
- 사용 맥락: `the perimeter of a rectangle` = 직사각형의 둘레

### quadrangle

- 구성: `quadri-` = four + `angle`
- 의미: 사각형, 네모난 안뜰
- 어원적 연결: 네 각을 가진 도형이나 공간
- 사용 맥락: `a college quadrangle` = 대학의 사각 안뜰

### diagonal

- 구성: Greek [[dia]] = across + `gonia` = angle
- 의미: 대각선의, 대각선
- 어원적 연결: 각에서 각으로 가로질러 감
- 사용 맥락: `a diagonal line` = 대각선

### polygon

- 구성: Greek `poly-` = many + `gonia` = angle
- 의미: 다각형
- 어원적 연결: 많은 각을 가진 도형
- 사용 맥락: `a regular polygon` = 정다각형

### angler

- 구성: Old English `angel` = fishhook + `-er`
- 의미: 낚시꾼
- 어원적 연결: 굽은 갈고리로 낚는 사람
- 사용 맥락: `a skilled angler` = 숙련된 낚시꾼

## 기억 포인트

- `angle`: 꺾인 모서리, 각
- `angular`: 각진
- `triangle`: 세 각
- `rectangle`: 직각을 가진 도형
- `quadrangle`: 네 각
- `diagonal`: 각에서 각으로 가로지름
- `polygon`: 많은 각

## 비교 단어

- [[quad]]
  - 핵심: 넷, 네모
  - 구분 포인트: `quadrangle`은 네 각. `quadrilateral`은 네 변. 둘 다 `quad` = 넷 이미지

- [[metr]]
  - 핵심: 재다, 척도
  - 구분 포인트: `angle`은 도형의 꺾임 자체. `meter/measure`는 그것을 재는 기준이나 행위

- `angel`
  - 핵심: Greek `angelos` = messenger
  - 구분 포인트: 천사 `angel`은 "전달자" 계열. 각 `angle`이나 낚시꾼 `angler`와 어원 다름

- [[rect]]
  - 핵심: Latin `rectus` = straight, right
  - 구분 포인트: `rectangle`은 `rect` = 바른/직각 + `angle` = 각
