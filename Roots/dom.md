# dom

- 출발 단어: `dominance`
- 핵심 어원: Latin `domus` = house; `dominus` = master, lord
- 핵심 이미지: 집의 주인, 자기 영역을 다스리는 권한
- 관련 문서: [[reg]], [[govern]], [[doom]], [[dorm]]
- 주의: `kingdom`, `freedom`, `wisdom`의 `-dom`은 [[doom]] 계열. `dormitory`는 [[dorm]] 계열. 이 문서는 `dominus` = 주인·지배자와 `domus` = 집 중심.

## 출발 단어 분석

`dominance` ← Latin `dominari` = rule, govern

- `domin-`: master, lord
- `-ance`: 상태·성질 명사 접미사
- 원래 이미지: 집의 주인이 자기 집과 사람을 다스림
- 의미 변화: 주인의 권한 → 지배력, 우세함
- 현대 의미: 지배, 우세, 우월성

## 단어들

### dominance

- 발음: /ˈdɑmənəns/
- 구성: `dominant` 계열 + `-ance`
- 뜻: 지배, 우세, 우월성
- 어원적 연결: 주인처럼 위에서 통제하거나 더 강한 위치를 차지함
- 사용 맥락: `market dominance` = 시장 지배력

### dominant

- 발음: /ˈdɑmənənt/
- 구성: Latin `dominans` = ruling
- 뜻: 지배적인, 우세한; 우성의
- 어원적 연결: 다른 것 위에 주인처럼 영향력을 행사함
- 사용 맥락: `a dominant position` = 지배적 위치, `a dominant gene` = 우성 유전자

### dominate

- 발음: /ˈdɑməˌneɪt/
- 구성: Latin `dominari` = rule, govern
- 뜻: 지배하다, 장악하다, 압도하다
- 어원적 연결: 주인처럼 통제권을 가짐
- 사용 맥락: `dominate the market` = 시장을 지배하다

### domination

- 발음: /ˌdɑməˈneɪʃən/
- 구성: `dominate` + `-ion`
- 뜻: 지배, 통제
- 어원적 연결: 한쪽이 주인처럼 힘을 행사하는 상태
- 사용 맥락: `political domination` = 정치적 지배

### dominion

- 발음: /dəˈmɪnjən/
- 구성: Latin `dominium` = ownership, rule
- 뜻: 지배권, 영토, 통치권
- 어원적 연결: 주인이 소유하고 다스리는 영역
- 사용 맥락: `have dominion over a territory` = 영토에 대한 지배권을 갖다

### domain

- 발음: /doʊˈmeɪn/
- 구성: Old French `domaine` ← Latin `dominium` = ownership, property
- 뜻: 영역, 분야, 영토; 도메인
- 어원적 연결: 주인이 소유하거나 통제하는 범위
- 사용 맥락: `the domain of science` = 과학의 영역

### domestic

- 발음: /dəˈmɛstɪk/
- 구성: Latin `domesticus` ← `domus` = house
- 뜻: 가정의, 국내의
- 어원적 연결: 집 안에 속한 것. 국가에서는 나라 안쪽으로 확장
- 사용 맥락: `domestic policy` = 국내 정책, `domestic chores` = 집안일

### domesticate

- 발음: /dəˈmɛstəˌkeɪt/
- 구성: `domestic` + `-ate`
- 뜻: 길들이다, 가축화하다
- 어원적 연결: 야생의 것을 집 안의 관리 영역으로 들임
- 사용 맥락: `domesticate animals` = 동물을 가축화하다

### domicile

- 구성: Latin `domicilium` = dwelling place
- 뜻: 주소지, 거주지
- 어원적 연결: 사람이 사는 집이나 법적 거주지
- 사용 맥락: `legal domicile` = 법적 주소지

### condominium

- 발음: /ˌkɑndəˈmɪniəm/
- 구성: Latin `con-` = together + `dominium` = ownership
- 뜻: 콘도미니엄, 공동 소유 주택
- 어원적 연결: 함께 소유하는 주거 영역
- 사용 맥락: `buy a condominium` = 콘도를 사다

### domineer

- 발음: /ˌdɑməˈnɪr/
- 구성: `domineer` ← `domineer`/`domineer over`, from `domin-`
- 뜻: 거만하게 지배하다, 군림하다
- 어원적 연결: 주인처럼 과도하게 통제하려 함
- 사용 맥락: `domineer over others` = 남에게 군림하다

### Anno Domini

- 구성: Latin `anno` = in the year + `Domini` = of the Lord
- 뜻: 주님의 해에, 서기
- 어원적 연결: `Dominus` = 주님, 주인. `Domini`는 그 속격 형태
- 사용 맥락: `AD 2026` = 서기 2026년
- 주의: 정확한 표기는 `Anno Domini`. 약어 `AD`는 연도 앞에 쓰는 전통이 있음.

## 기억 포인트

`dom/domin` 계열은 "집의 주인과 지배 영역" 이미지.

- `dominance`: 지배력
- `dominant`: 지배적인, 우세한
- `dominate`: 주인처럼 장악함
- `dominion`: 지배권과 영토
- `domain`: 통제·소유 영역
- `domestic`: 집 안의, 국내의
- `domesticate`: 집의 관리 영역으로 들임
- `Anno Domini`: 주님의 해, 서기

## 비교 단어

### reg

- 뜻: 다스리다, 똑바로 이끌다
- 구분 포인트: [[reg]]는 왕·규칙·방향 잡기 이미지. `dom`은 집의 주인과 소유 영역 이미지.

### govern

- 뜻: 조종하다, 통치하다
- 구분 포인트: [[govern]]은 배를 조종하듯 운영·통치함. `dominate`는 위에서 장악하고 우세를 점함.

### pot

- 뜻: 힘, 능력
- 구분 포인트: `pot` 계열은 할 수 있는 힘·잠재력. `dominance`는 실제로 위에서 지배하는 우세함.

### kingdom / -dom

- 뜻: 왕국; 상태·영역을 만드는 접미사
- 구분 포인트: [[doom]]의 Old English `dōm` = judgment, authority, state 계열. Latin `domus/dominus`와 다름.

### dormitory

- 뜻: 기숙사, 공동 침실
- 구분 포인트: [[dorm]]의 Latin `dormire` = sleep 계열. 철자상 `dom`처럼 보여도 `dominance/domain/domestic` 계열이 아님.
