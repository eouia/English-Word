# vapor

- 출발 단어: `evaporate`
- 핵심 어원: 라틴어 `vapor` = steam, vapor, heat
- 핵심 이미지: 액체가 증기로 변해 밖으로 흩어짐
- 관련해서 헷갈릴 수 있는 어근: [[spir]] = 숨쉬다, 숨 계열, [[ex]] = 밖으로 계열 접두사

## 출발 단어 분석

`evaporate`는 `e-` + `vapor` + `-ate`.

- `e-`: out, away
- `vapor`: steam, vapor
- `-ate`: 동사형 접미사

따라서 `evaporate`는 "증기로 되어 밖으로 나가다"의 뜻. 여기서 "증발하다"가 되고, 비유적으로는 "사라지다, 없어지다"의 뜻도 됨.

## 단어들

### evaporate

- 구성: `e-` = out + `vapor` + `-ate`
- 뜻: 증발하다, 사라지다
- 어원적 연결: 액체가 증기로 변해 밖으로 흩어짐
- 사용 맥락: `Water evaporates quickly in heat.` = 물은 더위 속에서 빨리 증발함

### evaporation

- 구성: `evaporate` + `-ion`
- 뜻: 증발
- 어원적 연결: 액체가 증기로 바뀌어 흩어지는 과정
- 사용 맥락: `water loss through evaporation` = 증발로 인한 수분 손실

### evaporative

- 구성: `evaporate` + `-ive`
- 뜻: 증발의, 증발을 이용한
- 어원적 연결: 증기로 바뀌는 작용과 관련됨
- 사용 맥락: `evaporative cooling` = 증발 냉각

### vapor

- 구성: Latin `vapor`
- 뜻: 증기, 수증기
- 어원적 연결: 열로 인해 떠오르는 습한 기체
- 사용 맥락: `water vapor` = 수증기

### vaporize

- 구성: `vapor` + `-ize`
- 뜻: 증발시키다, 기화하다
- 어원적 연결: 어떤 것을 증기 상태로 만듦
- 사용 맥락: `vaporize a liquid` = 액체를 기화시키다

### vaporization

- 구성: `vaporize` + `-ation`
- 뜻: 기화, 증발
- 어원적 연결: 액체나 고체가 증기 상태로 변하는 과정
- 사용 맥락: `the vaporization of fuel` = 연료의 기화

### vapour

- 구성: British spelling of `vapor`
- 뜻: 증기
- 어원적 연결: `vapor`의 영국식 철자
- 사용 맥락: `water vapour` = 수증기

## 기억 포인트

`vapor` 계열은 "증기" 이미지.

- `vapor`: 증기
- `evaporate`: 증기가 되어 밖으로 나감 → 증발하다, 사라지다
- `evaporation`: 증발 과정
- `vaporize`: 증기 상태로 만들다
- `vaporization`: 기화

## 비교 단어

### expire

- 뜻: 만료되다, 숨을 내쉬다
- 구분 포인트: [[spir]] 계열. 공기나 숨이 밖으로 나가는 이미지. `evaporate`는 액체가 증기로 바뀌어 사라지는 이미지

### disappear

- 뜻: 사라지다
- 구분 포인트: 일반 의미는 겹치지만 어원 이미지는 다름. `evaporate`는 실제 또는 비유적으로 "증기처럼 없어짐"의 느낌
