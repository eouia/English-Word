# em

- 출발 단어: `embody`
- 핵심 접두사: `en-` / `em-` = in, into, make
- 핵심 이미지: 안에 넣거나, 어떤 상태로 만들다
- 관련해서 헷갈릴 수 있는 접두사: `ex-` = 밖으로, [[sym]] = 함께

## 출발 단어 분석

`em-`은 보통 `en-`이 `b`, `m`, `p` 앞에서 바뀐 형태.

- `en-`: in, into
- `em-`: `b`, `m`, `p` 앞의 동화형
- 의미: 안에 넣다, ~으로 만들다, ~을 부여하다

따라서 `embody`는 `em-` + `body`로, "몸 안에 넣다" → "구체화하다, 구현하다"의 뜻.

## 주요 형태

- `en-`: `enable`, `enlarge`, `encourage`, `enrich`
- `em-`: `embody`, `empower`, `employ`, `embed`

단, 모든 `em-`이 `en-`의 변형은 아님. `emit`, `emigrate`, `emancipate`는 다른 계열.

## 단어들

### embody

- 발음: /ɪˈmbɑdi/
- 구성: `em-` = into + `body`
- 뜻: 구현하다, 구체화하다, 몸으로 드러내다
- 어원적 연결: 추상적인 것을 몸 안에 넣어 드러냄
- 사용 맥락: `embody democratic values` = 민주적 가치를 구현하다
- 관련 문서: [[body]]

### embodiment

- 발음: /ɛˈmbɑdimənt/
- 구성: `embody` + `-ment`
- 뜻: 구현, 화신, 구체적 표현
- 어원적 연결: 몸을 얻어 드러난 상태
- 사용 맥락: `the embodiment of courage` = 용기의 화신
- 관련 문서: [[body]]

### empower

- 발음: /ɪˈmpaʊɚ/
- 구성: `em-` = make, give + `power`
- 뜻: 권한을 주다, 힘을 실어 주다
- 어원적 연결: 힘을 안에 부여함
- 사용 맥락: `empower employees` = 직원들에게 권한을 부여하다

### empowerment

- 발음: /ɪˈmpaʊɚmənt/
- 구성: `empower` + `-ment`
- 뜻: 권한 부여, 역량 강화
- 어원적 연결: 힘을 부여받은 상태
- 사용 맥락: `community empowerment` = 지역사회 역량 강화

### employ

- 발음: /ɛˈmplɔɪ/
- 구성: Old French `employer` = put to use
- 뜻: 고용하다, 사용하다
- 어원적 연결: 사람이나 수단을 일 안에 넣어 쓰게 함
- 사용 맥락: `employ a strategy` = 전략을 사용하다

### employment

- 발음: /ɛˈmplɔɪmənt/
- 구성: `employ` + `-ment`
- 뜻: 고용, 일자리
- 어원적 연결: 일에 넣어 쓰는 상태
- 사용 맥락: `full-time employment` = 정규직 고용

### embed

- 발음: /ɪˈmbɛd/
- 구성: `em-` = in + `bed`
- 뜻: 박아 넣다, 내장하다
- 어원적 연결: 어떤 것을 안에 깊이 넣음
- 사용 맥락: `embed a video` = 동영상을 삽입하다

### embark

- 발음: /ɛˈmbɑrk/
- 구성: `em-` = into + `bark` = boat
- 뜻: 승선하다, 착수하다
- 어원적 연결: 배 안으로 들어감 → 일을 시작함
- 사용 맥락: `embark on a journey` = 여행을 시작하다

### embrace

- 발음: /ɛˈmbreɪs/
- 구성: `em-` = in + `brace` = arms
- 뜻: 껴안다, 받아들이다
- 어원적 연결: 팔 안에 넣음
- 사용 맥락: `embrace change` = 변화를 받아들이다

### enclose

- 발음: /ɪˈnkloʊz/
- 구성: `en-` = in + [[clud]]
- 뜻: 둘러싸다, 동봉하다
- 어원적 연결: 안에 넣고 닫음
- 사용 맥락: `enclose a document` = 문서를 동봉하다

### enable

- 발음: /ɛˈneɪbəl/
- 구성: `en-` = make + `able`
- 뜻: 가능하게 하다
- 어원적 연결: 할 수 있는 상태로 만듦
- 사용 맥락: `enable access` = 접근을 가능하게 하다

### enlarge

- 발음: /ˌɛˈnlɑrdʒ/
- 구성: `en-` = make + `large`
- 뜻: 크게 하다, 확대하다
- 어원적 연결: 큰 상태로 만듦
- 사용 맥락: `enlarge an image` = 이미지를 확대하다

### encourage

- 발음: /ɛˈnkɝɪdʒ/
- 구성: `en-` = put into + `courage`
- 뜻: 격려하다
- 어원적 연결: 용기를 안에 넣어 줌
- 사용 맥락: `encourage students` = 학생들을 격려하다

### enrich

- 발음: /ɛˈnrɪtʃ/
- 구성: `en-` = make + `rich`
- 뜻: 풍요롭게 하다

## 주의할 단어

### emit

- 발음: /ɪˈmɪt/
- 구성: `e-` = out + [[mit]]
- 뜻: 내뿜다, 방출하다
- 구분 포인트: `em-` = 안에 넣다가 아니라 `ex-` 계열의 `e-` + `mit`

### emigrate

- 발음: /ˈɛməˌgreɪt/
- 구성: `e-` = out + `migrate`
- 뜻: 이민 가다, 나라를 떠나다
- 구분 포인트: 밖으로 이동함. `en-/em-` = 안으로 넣다 계열이 아님

### emancipate

- 발음: /ɪˈmænsəˌpeɪt/
- 구성: `e-` = out + [[manu]] 계열
- 뜻: 해방하다
- 구분 포인트: 손으로 쥔 소유 상태에서 밖으로 풀어 줌. `em-` 접두사 문서에서는 혼동 주의

## 기억 포인트

`em-` / `en-` 계열은 "안에 넣거나 어떤 상태로 만듦".

- `embody`: 몸 안에 넣어 구현함
- `empower`: 힘을 부여함
- `employ`: 일 안에 넣어 사용함
- `embed`: 안에 박아 넣음
- `embrace`: 팔 안에 넣음
- `enable`: 할 수 있게 만듦
- `enlarge`: 크게 만듦
- `encourage`: 용기를 넣어 줌

## 비교 단어

### ex-

- 뜻: 밖으로
- 구분 포인트: `emit`, `emigrate`, `emancipate`는 겉으로 `em-`처럼 보여도 밖으로 나가는 `e-/ex-` 계열

### in-

- 뜻: 안에, 안으로
- 구분 포인트: `en-/em-`은 `in-`과 의미가 가까우며, 특히 동사를 만들어 "~하게 하다"로 자주 씀
