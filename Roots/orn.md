# orn

- 출발 단어: `ornamental`
- 핵심 어원: Latin `ornare` = 꾸미다, 장식하다, 갖추다
- 핵심 이미지: 보기 좋게 꾸미거나 필요한 것을 갖춤
- 관련 문서: 없음

## 단어들

### ornamental

- 구성: `ornament` + `-al`
- 뜻: 장식용의, 장식적인
- 어원적 연결: 실용보다 꾸미는 기능이 중심
- 사용 맥락: `ornamental plants` = 관상용 식물

### ornament

- 구성: Latin `ornamentum` = equipment, decoration
- 뜻: 장식품, 장식
- 어원적 연결: 대상을 보기 좋게 갖추는 것
- 사용 맥락: `Christmas ornaments` = 크리스마스 장식품

### ornamentation

- 구성: `ornament` + `-ation`
- 뜻: 장식, 장식 양식
- 어원적 연결: 장식을 더하는 행위나 결과
- 사용 맥락: 건축, 음악, 디자인에서 꾸밈 요소를 말할 때 사용

### adorn

- 구성: `ad-` = to + Latin `ornare`
- 뜻: 장식하다, 꾸미다
- 어원적 연결: 어떤 대상에 꾸밈을 더함
- 사용 맥락: `flowers adorned the table` = 꽃이 식탁을 장식함

### ornate

- 구성: Latin `ornatus` = decorated
- 뜻: 화려하게 장식된, 지나치게 꾸민
- 어원적 연결: 장식이 많이 붙은 상태
- 사용 맥락: `an ornate ceiling` = 화려하게 장식된 천장

### ornate style

- 구성: `ornate` + `style`
- 뜻: 화려한 문체, 장식적인 양식
- 어원적 연결: 표현이나 형태가 많이 꾸며짐
- 사용 맥락: 글쓰기나 건축 양식이 장황하고 화려할 때 사용

### suborn

- 구성: `sub-` = secretly, under + Latin `ornare` = equip, furnish
- 뜻: 매수하여 위증하게 하다, 사주하다
- 어원적 연결: 몰래 사람을 "준비시켜" 나쁜 일을 하게 함
- 사용 맥락: `suborn a witness` = 증인을 매수해 위증하게 하다

### subornation

- 구성: `suborn` + `-ation`
- 뜻: 매수, 위증 교사
- 어원적 연결: 몰래 사람을 준비시키거나 사주하는 행위
- 사용 맥락: 법률 문맥에서 주로 사용되므로 인식용 어휘

## 기억 포인트

`orn` 계열은 "꾸미다, 갖추다" 이미지.

- `ornamental`: 장식용의
- `ornament`: 장식품
- `adorn`: 꾸미다
- `ornate`: 화려하게 장식된
- `suborn`: 몰래 사주하다

## 비교 단어

### decor

- 뜻: 어울리게 꾸미다, 품위 있게 장식하다
- 구분 포인트: `decor`는 "어울림, 품위, 보기 좋음"에 가까움. `orn`은 장식을 붙이거나 갖추는 행위.

### equip

- 뜻: 필요한 장비를 갖추다
- 구분 포인트: `ornare`에는 원래 "갖추다, 준비시키다" 뜻도 있음. 현대 영어에서는 장비 쪽은 `equip`, 장식 쪽은 `orn`으로 분리해 기억.
