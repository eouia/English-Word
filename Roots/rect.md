# rect

- 출발 단어: `correct`
- 핵심 어원: Latin `rectus` = straight, right
- 핵심 이미지: 곧게 펴짐, 바른 방향
- 관련 어근: [[reg]], [[angl]], [[right]], French `droit`

## 출발 단어 분석

`correct`는 Latin `corrigere`에서 온 말.

- `com-`: together, completely
- `regere`: direct, guide
- `rectus`: straight, right

핵심은 "삐뚤어진 것을 바르게 세움"임.

따라서 `correct`는 "바르게 하다" → "고치다, 정확한"의 뜻.

## 단어들

### correct

- 발음: /kɚˈɛkt/
- 구성: `cor-` = together/completely + `rect`
- 의미: 고치다, 올바른, 정확한
- 어원적 연결: 틀어진 것을 바르게 만듦
- 사용 맥락: `correct an error` = 오류를 고치다

### correction

- 발음: /kɚˈɛkʃən/
- 구성: `correct` + `-ion`
- 의미: 수정, 교정
- 어원적 연결: 바르게 고치는 행위
- 사용 맥락: `make a correction` = 수정하다

### incorrect

- 발음: /ˌɪnkɚˈɛkt/
- 구성: [[in]] = not + `correct`
- 의미: 틀린, 부정확한
- 어원적 연결: 바르지 않음
- 사용 맥락: `an incorrect answer` = 틀린 답

### direct

- 발음: /dɚˈɛkt/
- 구성: `di-` / `dis-` + `rect`
- 의미: 직접의, 지시하다, 향하게 하다
- 어원적 연결: 곧은 방향으로 이끎
- 사용 맥락: `direct the team` = 팀을 지휘하다

### direction

- 발음: /dɚˈɛkʃən/
- 구성: `direct` + `-ion`
- 의미: 방향, 지시
- 어원적 연결: 곧게 향하게 하는 길
- 사용 맥락: `follow directions` = 지시를 따르다

### director

- 발음: /dɚˈɛktɚ/
- 구성: `direct` + `-or`
- 의미: 감독, 관리자, 이사
- 어원적 연결: 방향을 잡아 이끄는 사람
- 사용 맥락: `a film director` = 영화감독

### rector

- 발음: /ˈrɛktɚ/
- 구성: Latin `rector` = ruler, guide
- 의미: 교구 목사, 대학 총장; 지도자
- 어원적 연결: 공동체나 기관의 방향을 바로잡아 이끄는 사람
- 사용 맥락: `the rector of the church` = 그 교회의 교구 목사

### rectory

- 발음: /ˈrɛktɚi/
- 구성: `rector` + `-y`
- 의미: 교구 목사관, 교구 목사의 주거지
- 어원적 연결: `rector`가 사는 집이나 그 직분과 관련된 장소
- 사용 맥락: `live in the rectory` = 목사관에 살다

### rectorate

- 구성: `rector` + `-ate`
- 의미: 총장직, 교구 목사직, 그 임기
- 어원적 연결: 지도자·관리자인 `rector`의 직위
- 사용 맥락: university / church context

### rectify

- 발음: /ˈrɛktəˌfaɪ/
- 구성: Latin `rectus` = right + `facere` = make
- 의미: 바로잡다, 교정하다
- 어원적 연결: 바른 상태로 만듦
- 사용 맥락: `rectify the mistake` = 실수를 바로잡다

### redress

- 발음: /rɪˈdrɛs/
- 구성: `re-` = again/back + Old French `redrecier/redresier`
- 의미: 바로잡다, 시정하다; 보상, 구제
- 역사적 형태: Old French `redrecier/redresier` ← `re-` + `drecier/dresser` ← Vulgar Latin `*directiare` ← Latin `directus`
- 어원적 연결: `directus` = 곧은, 똑바른 계열. 틀어진 상태를 다시 똑바로 세움. Latin `rectus`에서 바로 온 형태는 아니지만, `direct`와 같은 "곧게 바로잡음" 갈래
- 사용 맥락: `redress a grievance` = 불만·피해를 시정하다

### redressal

- 구성: `redress` + `-al`
- 의미: 시정, 구제, 보상
- 어원적 연결: 잘못이나 피해를 다시 바르게 세우는 행위
- 사용 맥락: `grievance redressal` = 고충 처리, 피해 구제

### rectangle

- 발음: /ˈrɛktæŋgəl/
- 구성: `rect` = right/straight + [[angl]] = angle
- 의미: 직사각형
- 어원적 연결: 바른 각, 즉 직각을 가진 도형
- 사용 맥락: `the area of a rectangle` = 직사각형의 넓이

### rectangular

- 발음: /rɛˈktæŋgjəlɚ/
- 구성: `rectangle` + `-ar`
- 의미: 직사각형의
- 어원적 연결: 직각으로 된 네모 모양
- 사용 맥락: `a rectangular table` = 직사각형 탁자

### rectitude

- 발음: /ˈrɛktɪˌtud/
- 구성: Latin `rectitudo`
- 의미: 정직, 올바름
- 어원적 연결: 마음이나 행동이 곧고 바름
- 사용 맥락: `moral rectitude` = 도덕적 올바름

### erect

- 발음: /ɪˈrɛkt/
- 구성: `e-` = out/up + `rect`
- 의미: 똑바로 세우다, 직립한
- 어원적 연결: 위로 곧게 세움
- 사용 맥락: `erect a monument` = 기념비를 세우다

### adroit

- 발음: /əˈdrɔɪt/
- 구성: French `a droit` = according to right
- 의미: 능숙한, 재치 있는, 솜씨 좋은
- 어원적 연결: 바른쪽·바른 방향으로 잘 움직임 → 손재주와 판단이 능숙함
- 사용 맥락: `an adroit negotiator` = 능숙한 협상가

### adroitly

- 발음: /əˈdrɔɪtli/
- 구성: `adroit` + `-ly`
- 의미: 능숙하게, 재치 있게
- 어원적 연결: 바르게 잘 처리하는 방식
- 사용 맥락: `handle the issue adroitly` = 문제를 능숙하게 처리하다

### adroitness

- 구성: `adroit` + `-ness`
- 의미: 능숙함, 재치, 솜씨
- 어원적 연결: 바른 방향으로 잘 다루는 능력
- 사용 맥락: `political adroitness` = 정치적 수완

### maladroit

- 발음: /ˌmæləˈdrɔɪt/
- 구성: `mal-` = bad + `adroit`
- 의미: 서투른, 솜씨 없는, 눈치 없는
- 어원적 연결: 바르게 처리하지 못함
- 사용 맥락: `a maladroit response` = 서투른 대응

## 기억 포인트

- `correct`: 바르게 고침
- `direct`: 곧은 방향으로 이끎
- `rector`: 방향을 바로 이끄는 지도자
- `rectory`: `rector`의 집, 목사관
- `rectify`: 바르게 만듦
- `redress`: 잘못이나 피해를 바로잡음
- `rectangle`: 바른 각
- `rectitude`: 곧은 마음
- `erect`: 곧게 세움
- `adroit`: 바른쪽·바른 방향으로 능숙하게 처리함

## 비교 단어

- [[angl]]
  - 핵심: 각, 모서리
  - 구분 포인트: `rectangle`은 `rect` = 바른/직각 + `angle` = 각

- [[fac]]
  - 핵심: 만들다, 하다
  - 구분 포인트: `rectify`의 `-fy`는 `facere` = make. 전체 뜻은 "바르게 만들다"

- [[duc]]
  - 핵심: 이끌다
  - 구분 포인트: `direct`는 방향을 바로 잡아 지시함. `conduct/produce`의 `duc`는 이끌고 가는 동작이 더 중심

- `redress`
  - 핵심: 다시 바로 세우다, 피해를 시정하다
  - 구분 포인트: `redress`는 Old French `redrecier/redresier`를 거쳐 온 말. 그 바탕의 `dresser/drecier`는 Vulgar Latin `*directiare` ← Latin `directus` 계열이라 `direct`와 가까움. `rectify`처럼 Latin `rectus`를 직접 붙인 형태는 아님

- [[right]]
  - 핵심: 곧고 바른 것, 옳음
  - 구분 포인트: `right`는 Old English `riht` 계열. `rect`는 Latin `rectus` 계열. 깊은 뿌리는 닿지만 영어 학습 문서에서는 분리

### dexterous

- `dexterous`, `dexterity`는 Latin `dexter` = right hand 계열.
- `adroit`와 뜻은 가깝지만, `adroit`는 French `droit` = right/straight 계열.

### deft

- `deft`는 Old English/Germanic 계열.
- 의미는 “솜씨 좋은”으로 비슷하지만 `adroit`와 어원 무관.
