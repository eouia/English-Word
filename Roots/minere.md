# minere

- 출발 단어: `imminent`
- 핵심 어원: Latin `minere` = project, jut out, overhang
- 핵심 이미지: 위나 앞쪽으로 돌출해 다가오거나 두드러짐
- 관련 문서: [[min]], [[mont]]
- 주의: [[min]]의 `minor/minus` = 작다, 줄이다 계열과 다른 어원. 철자가 같아 보여도 섞지 않음.

## 출발 단어 분석

`imminent` = `im-` + `minere` + `-ent`

- `im-`: on, upon, toward
- `minere`: project, overhang
- `-ent`: 현재분사형 형용사 어미
- 원래 이미지: 위로 드리워져 곧 닥칠 듯함
- 현대 의미: 임박한, 곧 일어날

## 단어들

### imminent

- 구성: Latin `imminere` = overhang, impend
- 뜻: 임박한, 곧 닥칠
- 어원적 연결: 무언가가 머리 위로 드리워져 곧 닥칠 듯한 상태
- 사용 맥락: `imminent danger` = 임박한 위험, `an imminent decision` = 곧 내려질 결정

### imminence

- 구성: `imminent` + `-ence`
- 뜻: 임박함, 절박하게 다가옴
- 어원적 연결: 사건이 가까이 드리워진 상태
- 사용 맥락: `the imminence of war` = 전쟁의 임박함

### imminently

- 구성: `imminent` + `-ly`
- 뜻: 곧, 임박하여
- 어원적 연결: 거의 닥쳐온 방식
- 사용 맥락: `imminently possible` = 곧 가능할 수 있는

### eminent

- 구성: Latin `eminere` = stand out, project
- 뜻: 저명한, 뛰어난
- 어원적 연결: 남들 위로 튀어나와 두드러짐
- 사용 맥락: `an eminent scholar` = 저명한 학자

### eminence

- 구성: `eminent` + `-ence`
- 뜻: 저명함, 탁월함; 높은 곳
- 어원적 연결: 높이 솟아 두드러진 상태
- 사용 맥락: `rise to eminence` = 명성을 얻다

### prominent

- 구성: Latin `prominere` = jut forward, stand out
- 뜻: 두드러진, 저명한
- 어원적 연결: 앞쪽으로 튀어나와 눈에 잘 띔
- 사용 맥락: `a prominent feature` = 두드러진 특징, `a prominent leader` = 저명한 지도자

### prominence

- 구성: `prominent` + `-ence`
- 뜻: 두드러짐, 중요성, 명성
- 어원적 연결: 표면이나 무리에서 앞으로 튀어나와 눈에 띄는 상태
- 사용 맥락: `gain prominence` = 두각을 나타내다

### menace

- 구성: Old French `menace` ← Latin `minacia` = threat
- 뜻: 위협, 위협하다
- 어원적 연결: 돌출한 위협이 가까이 다가오는 느낌
- 사용 맥락: `a public menace` = 공공의 위협, `menace society` = 사회를 위협하다

### minatory

- 구성: Latin `minatorius` < `minari` = threaten
- 뜻: 위협적인, 협박하는
- 어원적 연결: 다가와 위협을 드리우는 성질
- 사용 맥락: 문어적 표현. 일반 시험 어휘로는 드묾.

## 기억 포인트

`minere`는 "튀어나와 드리워짐" 이미지.

- `imminent`: 머리 위로 드리워져 곧 닥침
- `eminent`: 위로 두드러져 뛰어남
- `prominent`: 앞으로 튀어나와 눈에 띔
- `menace`: 위협이 돌출해 다가옴
- `minatory`: 위협적인

## 비교 단어

### min

- 뜻: 작다, 줄이다
- 구분 포인트: [[min]]의 `minor`, `minus`, `minimum`과 다른 어원. `imminent`는 "작음"이 아니라 "드리워져 다가옴".

### impending

- 뜻: 임박한
- 구분 포인트: [[pend]] 계열. 위에 매달려 곧 떨어질 듯한 이미지. `imminent`와 의미가 매우 가까움.

### immanent

- 뜻: 내재하는
- 구분 포인트: [[manere]] 계열. 철자·발음이 비슷하지만 `imminent`와 어원이 다름.
