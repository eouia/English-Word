# cant

- 출발 단어: `recant`
- 핵심 어원: Latin `canere` / `cantare` = sing, chant
- 핵심 이미지: 노래하듯 말하거나 외움
- 관련 문서: [[call]]
- 주의: `cant` "위선적 말투, 전문 은어"는 의미 발달이 다소 복잡함. 여기서는 `canere/cantare`의 노래·외침 계열 중심으로 정리.

## 출발 단어 분석

`recant` ← Latin `recantare`

- `re-`: back
- `cantare`: 노래하다, 외다
- 원래 이미지: 말했던 것을 되돌려 말함
- 현대 의미: 주장을 철회하다, 신념을 취소하다

## 단어들

### recant

- 구성: `re-` = back + Latin `cantare`
- 뜻: 철회하다, 취소하다
- 어원적 연결: 이미 말한 주장이나 신념을 되돌려 말함
- 사용 맥락: `recant a confession` = 자백을 철회하다

### recantation

- 구성: `recant` + `-ation`
- 뜻: 철회, 취소 선언
- 어원적 연결: 이전 발언을 되돌리는 공식적 말
- 사용 맥락: `a public recantation` = 공개 철회

### chant

- 구성: Old French `chanter` ← Latin `cantare`
- 뜻: 노래하다, 구호를 외치다; 성가, 구호
- 어원적 연결: 반복적이고 리듬 있게 말하거나 노래함
- 사용 맥락: `chant slogans` = 구호를 외치다

### enchant

- 구성: `en-` = in, into + `chant`
- 뜻: 매혹하다, 황홀하게 하다
- 어원적 연결: 주문이나 노래로 마음을 사로잡음
- 사용 맥락: `be enchanted by the music` = 음악에 매혹되다

### enchanting

- 구성: `enchant` + `-ing`
- 뜻: 매혹적인, 황홀한
- 어원적 연결: 주문처럼 사람을 끌어당기는 성질
- 사용 맥락: `an enchanting story` = 매혹적인 이야기

### incantation

- 구성: `in-` = upon/into + Latin `cantare`
- 뜻: 주문, 주문 외우기
- 어원적 연결: 마법적 힘을 기대하며 노래하듯 외는 말
- 사용 맥락: fantasy, myth, ritual context

### cantata

- 구성: Italian `cantata` ← Latin `cantare`
- 뜻: 칸타타, 성악곡
- 어원적 연결: 노래하도록 만들어진 음악 형식
- 사용 맥락: music context

## 기억 포인트

`cant` 계열은 "노래하듯 말함" 이미지.

- `recant`: 말한 것을 되돌림
- `recantation`: 철회 선언
- `chant`: 노래하거나 구호를 외침
- `enchant`: 주문처럼 매혹함
- `incantation`: 주문 외우기
- `cantata`: 노래하는 음악 형식
