# long

- 출발 단어: `length`
- 핵심 어원: Old English `lang` / `long` = long
- 핵심 이미지: 길게 뻗어 있음
- 관련 어근: [[high]], [[tend]], Latin `durare`

## 출발 단어 분석

`length`는 `long`의 명사형.

Old English `lang` / `long`은 물리적으로 "긴" 상태를 뜻했고, 시간, 글, 거리, 과정이 길게 이어지는 뜻으로 확장됨.

라틴계 `longus`도 같은 뜻의 "긴" 말이라 `longitude`, `prolong` 같은 시험 어휘와 함께 묶어 외우기 좋음.

## 단어들

### long

- 구성: Old English `long`
- 의미: 긴, 오래
- 어원적 연결: 공간이나 시간이 길게 뻗음
- 사용 맥락: `a long road` = 긴 길, `a long time` = 오랜 시간

### length

- 구성: `long` 계열 + 명사형
- 의미: 길이, 기간
- 어원적 연결: 길게 뻗은 정도
- 사용 맥락: `measure the length` = 길이를 재다

### lengthy

- 구성: `length` + `-y`
- 의미: 긴, 장황한
- 어원적 연결: 길이가 길거나 시간이 오래 걸림
- 사용 맥락: `a lengthy discussion` = 긴 토론

### lengthen

- 구성: `length` + `-en`
- 의미: 길게 하다, 길어지다
- 어원적 연결: 길이를 더 늘림
- 사용 맥락: `lengthen the deadline` = 마감 기한을 늘리다

### longitude

- 구성: Latin `longus` = long + `-itude`
- 의미: 경도
- 어원적 연결: 지구를 세로 방향으로 길게 재는 좌표
- 사용 맥락: `latitude and longitude` = 위도와 경도

### longitudinal

- 구성: `longitude` / Latin `longus` + `-al`
- 의미: 세로의, 장기적인
- 어원적 연결: 길이 방향을 따라 이어짐
- 사용 맥락: `a longitudinal study` = 장기 추적 연구

### prolong

- 구성: `pro-` = forward + Latin `longus`
- 의미: 연장하다, 오래 끌다
- 어원적 연결: 앞으로 더 길게 늘림
- 사용 맥락: `prolong the meeting` = 회의를 길게 끌다

### prolonged

- 구성: `prolong` + `-ed`
- 의미: 오래 계속된, 장기적인
- 어원적 연결: 길게 늘어진 상태
- 사용 맥락: `prolonged exposure` = 장기간 노출

### belong

- 구성: Old English `be-` + `longen`
- 의미: 속하다
- 어원적 연결: 현대 형태상 `long`처럼 보이지만 "길다"와 직접 연결해 외우기 어려움
- 사용 맥락: `belong to a group` = 집단에 속하다

## 기억 포인트

- `long`: 길다, 오래
- `length`: 길이, 기간
- `lengthy`: 길고 장황함
- `lengthen`: 길게 늘림
- `longitude`: 지구의 긴 방향 좌표
- `prolong`: 앞으로 더 길게 끌다

## 비교 단어

- [[high]]
  - 핵심: 높음, 위쪽
  - 구분 포인트: `height`는 위로 잰 정도. `length`는 길게 뻗은 정도

- [[metr]]
  - 핵심: 재다, 척도
  - 구분 포인트: `measure the length`에서 `measure`는 재는 행위, `length`는 재어진 긴 정도

- [[tend]]
  - 핵심: 뻗다, 향하다
  - 구분 포인트: `extend`는 뻗어 나가는 동작. `length`는 뻗은 거리나 기간
