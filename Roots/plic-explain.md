# plic-explain

- 출발 단어: `explicit`
- 핵심 어원: Latin `plicare` = 접다
- 핵심 이미지: 접혀 숨은 뜻을 펴서 드러냄
- 허브 문서: [[plic]]

## 단어들

### imply

- 구성: `in-` = in + `ply/plic` = fold
- 뜻: 암시하다, 함축하다
- 어원적 연결: 뜻이 말 안에 접혀 들어 있음
- 사용 맥락: `imply criticism` = 비판을 암시하다

### implication

- 구성: `imply/implicate` + `-ation`
- 뜻: 함의, 영향, 연루
- 어원적 연결: 어떤 뜻이나 사람이 상황 안에 접혀 들어감
- 사용 맥락: `policy implications` = 정책적 함의

### implicit

- 구성: Latin `implicitus` = enfolded
- 뜻: 암묵적인, 내포된
- 어원적 연결: 겉으로 펴 보이지 않고 안에 접혀 있음
- 사용 맥락: `implicit meaning` = 암묵적 의미

### implicate

- 구성: `in-` = in + `plic` = fold
- 뜻: 연루시키다, 관련시키다
- 어원적 연결: 어떤 사건 안으로 접어 넣음
- 사용 맥락: `be implicated in a scandal` = 스캔들에 연루되다

### explicate

- 구성: `ex-` = out + `plic` = fold
- 뜻: 자세히 설명하다, 풀어 밝히다
- 어원적 연결: 접힌 것을 밖으로 펴서 보이게 함
- 사용 맥락: `explicate a poem` = 시를 자세히 해석하다

### explicable

- 구성: `ex-` = out + `plic` = fold + `-able`
- 뜻: 설명할 수 있는, 해명 가능한
- 어원적 연결: 접혀 숨은 뜻을 펴서 드러낼 수 있음
- 사용 맥락: `a fully explicable result` = 충분히 설명 가능한 결과

### inexplicable

- 구성: `in-` = not + `explicable`
- 뜻: 설명할 수 없는, 이해하기 어려운
- 어원적 연결: 접힌 뜻이나 원인을 펴서 드러낼 수 없음
- 사용 맥락: `an inexplicable error` = 설명할 수 없는 오류, `inexplicable behavior` = 이해하기 어려운 행동

### inexplicably

- 구성: `inexplicable` + `-ly`
- 뜻: 설명할 수 없게, 이상하게도
- 어원적 연결: 이유가 펴져 드러나지 않는 방식
- 사용 맥락: `inexplicably absent` = 이유를 알 수 없이 결석한

### explicit

- 구성: Latin `explicitus` = unfolded
- 뜻: 명시적인, 분명한
- 어원적 연결: 접힌 것이 펴져 드러남
- 사용 맥락: `explicit instructions` = 명시적 지시

### exploit

- 구성: Old French `esploit` ← Latin `explicitum` = unfolded, carried out
- 뜻: 이용하다, 착취하다; 위업, 공적
- 어원적 연결: 접힌 것을 펴서 실행함. 실행된 일이나 성과에서 "이용하다"로 확장
- 사용 맥락: `exploit natural resources` = 천연자원을 이용하다

### exploitation

- 구성: `exploit` + `-ation`
- 뜻: 이용, 개발, 착취
- 어원적 연결: 어떤 가능성이나 자원을 펴서 실제 이익으로 사용함
- 사용 맥락: `economic exploitation` = 경제적 착취

### exploitative

- 구성: `exploit` + `-ative`
- 뜻: 착취적인, 부당하게 이용하는
- 어원적 연결: 남이나 자원을 자기 이익을 위해 펴 쓰는 성격
- 사용 맥락: `exploitative labor practices` = 착취적인 노동 관행

## 기억 포인트

`plic-explain`은 "접힌 것을 펴서 드러냄" 갈래.

- `imply`: 뜻이 안에 접혀 있음
- `implicit`: 접혀 있어 드러나지 않음
- `inexplicable`: 펴서 설명할 수 없음
- `explicit`: 펴져 드러남
- `exploit`: 펴서 실행하고 이용함
