# colon

- 출발 단어: `colonialism`
- 핵심 어원: Latin `colonus` = 농부, 정착민
- 핵심 의미: 정착지, 식민지, 이주해 차지함
- 관련 문서: [[in]]
- 주의: `colon` = 대장의 일부는 Greek `kolon` 계열로 이 문서의 `colony`와 어원 다름.

## 출발 단어 분석

`colonialism` = `colony` + `-al` + `-ism`

- `colonus`: 농부, 정착민
- `colonia`: 정착지, 식민지
- `-ism`: 체제, 사상, 관행
- 원래 이미지: 사람들이 옮겨 가 경작하고 정착한 곳
- 의미 확장: 정착지 → 지배 식민지 → 식민 지배 체제

## 단어들

### colony

- 발음: /ˈkɑləni/
- 구성: Latin `colonia`
- 뜻: 식민지, 집단 거주지; 생물 군체
- 어원적 연결: 사람들이 옮겨 가 정착한 곳
- 사용 맥락: `a British colony` = 영국 식민지, `an ant colony` = 개미 군체

### colonial

- 발음: /kəˈloʊniəl/
- 구성: `colony` + `-al`
- 뜻: 식민지의, 식민 시대의
- 어원적 연결: 식민지와 관련됨
- 사용 맥락: `colonial rule` = 식민 통치

### colonialism

- 발음: /kəˈloʊniəˌlɪzəm/
- 구성: `colonial` + `-ism`
- 뜻: 식민주의, 식민 지배 체제
- 어원적 연결: 정착지를 지배하고 이용하는 체제
- 사용 맥락: `European colonialism` = 유럽 식민주의

### colonize

- 발음: /ˈkɑləˌnaɪz/
- 구성: `colony` + `-ize`
- 뜻: 식민지화하다, 정착하다
- 어원적 연결: 새로운 지역에 정착지를 만들거나 지배 체제를 세움
- 사용 맥락: `colonize new territories` = 새 영토를 식민지화하다

### colonization

- 발음: /ˌkɑlənɪˈzeɪʃən/
- 구성: `colonize` + `-ation`
- 뜻: 식민지화, 정착
- 어원적 연결: 정착지를 만들고 지배권을 세우는 과정
- 사용 맥락: `the colonization of the Americas` = 아메리카 식민지화

### colonist

- 발음: /ˈkɑlənɪst/
- 구성: `colony` + `-ist`
- 뜻: 식민지 주민, 정착민
- 어원적 연결: 식민지에 정착한 사람
- 사용 맥락: `early colonists` = 초기 식민지 정착민

### postcolonial

- 구성: `post-` = after + `colonial`
- 뜻: 탈식민의, 식민지 이후의
- 어원적 연결: 식민 지배 이후의 정치·문화 상태
- 사용 맥락: `postcolonial literature` = 탈식민 문학

### decolonize

- 구성: `de-` = reverse/remove + `colonize`
- 뜻: 탈식민화하다
- 어원적 연결: 식민 지배 구조를 제거함
- 사용 맥락: `decolonize education` = 교육을 탈식민화하다

### decolonization

- 구성: `decolonize` + `-ation`
- 뜻: 탈식민화
- 어원적 연결: 식민 통치에서 벗어나는 과정
- 사용 맥락: `postwar decolonization` = 전후 탈식민화

## 기억 포인트

`colon` 계열은 "정착해서 지배권을 세운 곳" 이미지.

- `colony`: 식민지, 군체
- `colonialism`: 식민 지배 체제
- `colonization`: 식민지화 과정
- `postcolonial`: 식민 지배 이후

## 비교 단어

- 관련 비교: `culture`, `cultivate`, `postcolonial`

### culture

- 뜻: 문화, 경작
- 구분 포인트: `culture`, `cultivate`도 Latin `colere` = 경작하다 계열. `colony`는 정착민과 정착지 쪽으로 굳어진 갈래.
