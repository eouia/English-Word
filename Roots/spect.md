# spect

- 출발 단어: `inspect`
- 핵심 어원: 라틴어 `specere` / `spectare` = 보다, 살피다
- 핵심 이미지: 방향을 두고 바라봄
- 관련해서 헷갈릴 수 있는 어근: [[vid]] = 보다, 보아서 판단하다 계열

## 출발 단어 분석

`inspect`는 `in-` + `spect`.

- `in-`: into
- `spectare`: 바라보다, 살피다

따라서 `inspect`는 "안쪽까지 들여다보다" → "검사하다, 점검하다"의 뜻.

## 단어들

### inspect

- 구성: `in-` = into + `spect`
- 뜻: 검사하다, 점검하다
- 어원적 연결: 안쪽까지 자세히 봄
- 사용 맥락: `inspect the building` = 건물을 점검하다

### inspection

- 구성: `in-` + `spect` + `-ion`
- 뜻: 검사, 점검
- 어원적 연결: 안을 들여다보는 행위
- 사용 맥락: `safety inspection` = 안전 점검

### respect

- 구성: `re-` = back + `spect`
- 뜻: 존중하다, 존경
- 어원적 연결: 다시 돌아보아 가치를 인정함
- 사용 맥락: `respect different opinions` = 다른 의견을 존중하다

### despise

- 구성: `de-` = down + Latin `specere` = look
- 뜻: 경멸하다, 업신여기다
- 어원적 연결: 아래로 내려다봄
- 사용 맥락: `despise cruelty` = 잔인함을 경멸하다

### despite

- 구성: Old French `despit` ← Latin `despectus` = looking down on
- 뜻: ~에도 불구하고
- 어원적 연결: 원래는 "멸시, 악의, 반감"의 뜻. `in despite of` = 반감에도 불구하고 → 현대 `despite` = ~에도 불구하고
- 사용 맥락: `despite the rain` = 비에도 불구하고

### spite

- 구성: shortened form of `despite`
- 뜻: 악의, 앙심; 괴롭히다
- 어원적 연결: 멸시나 반감이 남아 있는 마음
- 사용 맥락: `out of spite` = 앙심 때문에, `in spite of the delay` = 지연에도 불구하고

### respective

- 구성: `respect` + `-ive`
- 뜻: 각각의
- 어원적 연결: 각 대상을 따로 돌아보아 구분함
- 사용 맥락: `their respective roles` = 그들 각각의 역할

### suspect

- 구성: `sub-` = under + `spect`
- 뜻: 의심하다, 용의자
- 어원적 연결: 겉 아래를 수상하게 봄
- 사용 맥락: `suspect fraud` = 사기를 의심하다

### suspicion

- 구성: Latin `suspicio`
- 뜻: 의심, 혐의
- 어원적 연결: 아래에 숨은 것을 보는 느낌
- 사용 맥락: `raise suspicion` = 의심을 불러일으키다

### prospect

- 구성: `pro-` = forward + `spect`
- 뜻: 전망, 가능성
- 어원적 연결: 앞을 바라봄
- 사용 맥락: `job prospects` = 취업 전망

### perspective

- 구성: `per-` = through + `spect`
- 뜻: 관점, 시각, 원근법
- 어원적 연결: 꿰뚫어 보는 방식
- 사용 맥락: `from my perspective` = 내 관점에서

### spectator

- 구성: `spectate` + `-or`
- 뜻: 관중, 구경꾼
- 어원적 연결: 바라보는 사람
- 사용 맥락: `sports spectators` = 스포츠 관중

### spectacle

- 구성: Latin `spectaculum`
- 뜻: 구경거리, 장관
- 어원적 연결: 볼 만한 것
- 사용 맥락: `a public spectacle` = 공개적 구경거리

### spectacular

- 구성: `spectacle` + `-ar`
- 뜻: 장관의, 화려한
- 어원적 연결: 눈에 띄게 볼 만함
- 사용 맥락: `a spectacular view` = 장관

### conspicuous

- 구성: Latin `con-` = completely + `spicere` = look, see
- 뜻: 눈에 잘 띄는, 두드러진
- 어원적 연결: 완전히 보일 만큼 드러나 있음
- 사용 맥락: `a conspicuous error` = 눈에 띄는 오류

### inconspicuous

- 구성: `in-` = not + `conspicuous`
- 뜻: 눈에 잘 띄지 않는, 수수한
- 어원적 연결: 시야에 뚜렷하게 드러나 보이지 않음
- 사용 맥락: `an inconspicuous entrance` = 눈에 잘 띄지 않는 입구, `remain inconspicuous` = 눈에 띄지 않게 지내다

### conspicuously

- 구성: `conspicuous` + `-ly`
- 뜻: 눈에 띄게, 두드러지게
- 어원적 연결: 쉽게 보일 만큼 뚜렷한 방식
- 사용 맥락: `conspicuously absent` = 눈에 띄게 빠진

### inconspicuously

- 구성: `inconspicuous` + `-ly`
- 뜻: 눈에 띄지 않게, 조용히
- 어원적 연결: 남의 시선에 뚜렷하게 잡히지 않는 방식
- 사용 맥락: `leave inconspicuously` = 눈에 띄지 않게 떠나다

### conspicuousness

- 구성: `conspicuous` + `-ness`
- 뜻: 눈에 띔, 두드러짐
- 어원적 연결: 시야에 뚜렷하게 들어오는 성질
- 사용 맥락: 문어적 표현

### speculate

- 구성: Latin `speculari` = observe
- 뜻: 추측하다, 투기하다
- 어원적 연결: 멀리 바라보며 가능성을 헤아림
- 사용 맥락: `speculate about the cause` = 원인을 추측하다

### introspect

- 구성: `intro-` = inward + `spect`
- 뜻: 자기 성찰하다
- 어원적 연결: 자기 안쪽을 들여다봄
- 사용 맥락: `introspect about motives` = 동기를 성찰하다

### retrospective

- 구성: `retro-` = backward + `spect`
- 뜻: 회고적인, 회고전
- 어원적 연결: 뒤를 돌아봄
- 사용 맥락: `a retrospective analysis` = 회고적 분석

## 기억 포인트

`spect` 계열은 "방향을 두고 봄" 이미지.

- `inspect`: 안을 봄
- `respect`: 다시 돌아보아 존중함
- `despise`: 아래로 내려다봄 → 경멸하다
- `despite`: 멸시/반감에도 불구하고 → ~에도 불구하고
- `suspect`: 아래를 수상하게 봄
- `prospect`: 앞을 봄
- `perspective`: 통과해 보는 관점
- `conspicuous`: 완전히 보여 눈에 띔
- `inconspicuous`: 뚜렷하게 보이지 않아 눈에 덜 띔
- `retrospective`: 뒤를 돌아봄

## 비교 단어

### speci

- 뜻: 보이는 모양에서 구분되는 종류, 특정성
- 연결 포인트: [[speci]]는 `species`, `specific`, `special`, `specimen`처럼 `specere`에서 나온 "종류/특수성" 갈래.

### vid

- 뜻: 보다, 보아서 판단하다
- 구분 포인트: [[vid]]는 보임과 판단의 일반 이미지. `spect`는 특정 방향으로 바라보거나 살피는 느낌이 강함
