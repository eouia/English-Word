# right

- 출발 단어: `right`
- 핵심 어원: Old English `riht` = just, good, proper, straight
- 핵심 이미지: 곧고 바름, 정당함
- 관련 어근: [[reg]], [[rect]], [[leg]], [[equ]]

## 출발 단어 분석

`right`는 Old English `riht`에서 온 게르만계 기본어.

원래 핵심은 "곧은, 바른, 알맞은, 정당한"이며, 여기서 "정답, 권리, 오른쪽"까지 확장됨.

Latin `rectus`, `regere`와는 같은 깊은 PIE `reg-` 계열의 친족어지만, 영어 안에서는 [[rect]], [[reg]]와 분리해서 정리함.

## 단어들

### right

- 구성: Old English `riht`
- 의미: 옳은, 맞는, 오른쪽, 권리
- 어원적 연결: 곧고 바른 것
- 사용 맥락: `the right answer` = 맞는 답

### rightly

- 구성: `right` + `-ly`
- 의미: 올바르게, 당연히
- 어원적 연결: 바른 방식으로
- 사용 맥락: `rightly criticized` = 정당하게 비판받은

### rightness

- 구성: `right` + `-ness`
- 의미: 옳음, 정당성
- 어원적 연결: 바른 상태
- 사용 맥락: `the rightness of the decision` = 그 결정의 정당성

### righteous

- 구성: Old English `rihtwis` = right-wise
- 의미: 의로운, 독선적인
- 어원적 연결: 바른 방식에 맞음
- 사용 맥락: `righteous anger` = 정당한 분노

### rightful

- 구성: `right` + `-ful`
- 의미: 정당한, 합법적인
- 어원적 연결: 바른 권리에 맞음
- 사용 맥락: `the rightful owner` = 정당한 소유자

### rights

- 구성: `right`의 명사 복수
- 의미: 권리
- 어원적 연결: 정당하게 요구할 수 있는 것
- 사용 맥락: `human rights` = 인권

### upright

- 구성: `up` + `right`
- 의미: 똑바로 선, 정직한
- 어원적 연결: 위로 곧게 서 있음. 성품도 곧음
- 사용 맥락: `an upright citizen` = 정직한 시민

### downright

- 구성: `down` + `right`
- 의미: 완전히, 노골적인
- 어원적 연결: 곧장 아래로, 완전히 바르게/직접적으로
- 사용 맥락: `downright rude` = 노골적으로 무례한

### rightward

- 구성: `right` + `-ward`
- 의미: 오른쪽으로
- 어원적 연결: 오른쪽 방향으로
- 사용 맥락: `move rightward` = 오른쪽으로 움직이다

### aright

- 구성: `a-` + `right`
- 의미: 올바르게
- 어원적 연결: 바른 상태로
- 사용 맥락: `set things aright` = 일을 바로잡다

## 기억 포인트

- `right`: 곧고 바른 것
- `right answer`: 맞는 답
- `rights`: 정당한 권리
- `righteous`: 의로운
- `rightful`: 정당한
- `upright`: 곧게 선, 정직한

## 비교 단어

- [[rect]]
  - 핵심: Latin `rectus` = straight, right
  - 구분 포인트: `right`는 게르만계 기본어. `correct`, `direct`, `rectify`는 라틴계 `rect` 문서에서 정리

- [[leg]]
  - 핵심: 법, 정해진 규칙
  - 구분 포인트: `legal rights`는 법으로 인정된 권리. `right` 자체는 정당함/권리의 기본어

- [[equ]]
  - 핵심: 같음, 공정함
  - 구분 포인트: `equal rights`는 모두에게 같은 권리. `right`는 권리나 옳음 자체
