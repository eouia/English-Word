# iatr

- 출발 단어: `psychiatry`
- 핵심 어원: Greek `iatros` = 의사, 치료자
- 관련 형태: Greek `iatreia` = 치료
- 핵심 이미지: 병을 치료하는 의사와 치료 행위
- 관련 문서: [[psych]], [[path]], [[med]]
- 주의: 전문 의학어가 많으므로 시험·교양 어휘 중심으로 정리. `pediatric`의 `ped-`는 Greek `pais` = child 계열로, [[ped]] = foot과 어원이 다름.

## 출발 단어 분석

`psychiatry` ← Greek `psyche` + `iatreia`

- `psyche`: mind
- `iatreia`: healing
- 원래 이미지: 마음을 치료함
- 현대 의미: 정신의학

## 단어들

### psychiatry

- 발음: /saɪˈkaɪətri/
- 구성: [[psych]] = mind + Greek `iatreia` = healing
- 뜻: 정신의학
- 어원적 연결: 마음의 병을 치료하는 의학 분야
- 사용 맥락: `modern psychiatry` = 현대 정신의학

### psychiatrist

- 발음: /səˈkaɪətrəst/
- 구성: [[psych]] = mind + Greek `iatros` = doctor
- 뜻: 정신과 의사
- 어원적 연결: 마음의 병을 다루는 의사
- 사용 맥락: `consult a psychiatrist` = 정신과 의사와 상담하다

### pediatric

- 발음: /ˌpidiˈætrɪk/
- 구성: Greek `pais` = child + `iatros`
- 뜻: 소아과의
- 어원적 연결: 아이를 치료하는 의학 분야와 관련됨
- 사용 맥락: `pediatric care` = 소아 진료

### pediatrics

- 발음: /ˌpidiˈætrɪks/
- 구성: Greek `pais` = child + `iatros`
- 뜻: 소아과, 소아의학
- 어원적 연결: 아이의 병을 치료하는 의학 분야
- 사용 맥락: `a specialist in pediatrics` = 소아과 전문의

### geriatric

- 발음: /ˌdʒɛriˈætrɪk/
- 구성: Greek `geras` = old age + `iatros`
- 뜻: 노인 의학의, 노인의
- 어원적 연결: 노년기의 건강과 치료에 관련됨
- 사용 맥락: `geriatric medicine` = 노인 의학

### geriatrics

- 발음: /ˌdʒɛriˈætrɪks/
- 구성: Greek `geras` + `iatros`
- 뜻: 노인 의학
- 어원적 연결: 노인의 질병과 치료를 다루는 분야
- 사용 맥락: `a department of geriatrics` = 노인 의학과

### iatrogenic

- 구성: `iatro-` = doctor, treatment + `-genic` = produced
- 뜻: 의인성의, 치료 때문에 생긴
- 어원적 연결: 치료나 의료 행위에서 생겨남
- 사용 맥락: `iatrogenic complications` = 의료 행위로 생긴 합병증

## 기억 포인트

`iatr` 계열은 "의사와 치료" 이미지.

- `psychiatry`: 마음 치료
- `pediatrics`: 아이 치료
- `geriatrics`: 노인 치료
- `iatrogenic`: 치료 때문에 생김

## 비교 단어

### path

- 뜻: 고통, 질병
- 구분 포인트: [[path]]는 겪는 고통과 병적 상태. `iatr`은 그것을 치료하는 의사와 치료 행위.

### psych

- 뜻: 마음, 정신
- 구분 포인트: [[psych]]가 `iatr`과 결합하면 `psychiatry`, `psychiatrist`.
