# luc

- 출발 단어: `lucid`
- 핵심 어원: Latin `lux` / `lucere` = 빛, 빛나다
- 관련 형태: `luc-`, `lum-`, `lumin-`
- 핵심 이미지: 빛이 비쳐 분명해짐
- 관련 문서: [[photo]], [[clar]]

## 단어들

### lucid

- 구성: Latin `lucidus` = bright
- 뜻: 명료한, 또렷한
- 어원적 연결: 빛이 비쳐 분명하게 보임
- 사용 맥락: `a lucid explanation` = 명료한 설명

### elucidate

- 구성: `e-` = out + Latin `lucidus`
- 뜻: 설명하다, 밝히다
- 어원적 연결: 어두운 것을 밝게 드러냄
- 사용 맥락: `elucidate the point` = 요점을 밝히다

### translucent

- 구성: `trans-` = through + Latin `lucere`
- 뜻: 반투명의
- 어원적 연결: 빛이 통과함
- 사용 맥락: `translucent glass` = 반투명 유리

### illuminate

- 구성: `in-` = into + Latin `lumen` = light
- 뜻: 비추다, 밝히다
- 어원적 연결: 빛을 넣어 밝게 함
- 사용 맥락: `illuminate the room` = 방을 밝히다

### luminous

- 구성: Latin `lumen`
- 뜻: 빛나는, 선명한
- 어원적 연결: 빛을 내는 성질
- 사용 맥락: `luminous stars` = 빛나는 별들

### luminary

- 구성: Latin `lumen`
- 뜻: 발광체, 저명인사
- 어원적 연결: 빛을 내는 존재에서 뛰어난 사람으로 확장
- 사용 맥락: `a literary luminary` = 문학계의 거장

## 기억 포인트

`luc/lum` 계열은 "빛과 명료함" 이미지.

- `lucid`: 빛처럼 명료한
- `elucidate`: 밝혀 설명함
- `translucent`: 빛이 통과함
- `illuminate`: 빛을 비춤
- `luminous`: 빛나는
