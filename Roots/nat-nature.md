# nat-nature

- 출발 단어: `nature`
- 핵심 어원: Latin `natura` = birth, natural character
- 핵심 이미지: 태어난 그대로의 성질과 자연의 질서
- 허브 문서: [[nat]]

## 단어들

### nature

- 발음: /ˈneɪtʃɚ/
- 구성: Latin `natura` = birth, natural character
- 뜻: 자연, 본성
- 어원적 연결: 태어날 때부터 가진 성질 → 자연의 질서
- 사용 맥락: `human nature` = 인간 본성

### natural

- 발음: /ˈnætʃɚəl/
- 구성: `nature` + `-al`
- 뜻: 자연의, 자연스러운, 타고난
- 어원적 연결: 태어난 그대로의 성질에 맞음
- 사용 맥락: `a natural response` = 자연스러운 반응

### naturally

- 발음: /ˈnætʃɚəli/
- 구성: `natural` + `-ly`
- 뜻: 자연스럽게, 당연히
- 어원적 연결: 본래 성질에 맞게
- 사용 맥락: `act naturally` = 자연스럽게 행동하다

### supernatural

- 발음: /ˌsupɚˈnætʃɚəl/
- 구성: `super-` = above + `natural`
- 뜻: 초자연적인
- 어원적 연결: 자연의 질서를 넘어선 것
- 사용 맥락: `supernatural powers` = 초자연적 힘

### denature

- 발음: /dɪˈneɪtʃɚ/
- 구성: `de-` = away, remove + `nature`
- 뜻: 변성시키다, 본성을 바꾸다
- 어원적 연결: 원래의 자연적 성질을 없애거나 바꿈
- 사용 맥락: `denature a protein` = 단백질을 변성시키다

### naive

- 발음: /ˌnaɪˈiv/
- 구성: French `naif/naive` ← Latin `nativus`
- 뜻: 순진한, 세상 물정 모르는
- 어원적 연결: 태어난 그대로의 꾸밈없는 상태
- 사용 맥락: `a naive assumption` = 순진한 가정

## 기억 포인트

`nat-nature`는 "태어난 그대로의 성질" 갈래.

- `nature`: 본성, 자연
- `natural`: 본성에 맞는
- `supernatural`: 자연을 넘어선
- `denature`: 본성을 바꿈
- `naive`: 태어난 그대로처럼 순진한
