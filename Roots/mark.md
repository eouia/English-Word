# mark

- 출발 단어: `demarcate`
- 핵심 어원: Germanic `marka` = boundary, sign; Old French/Spanish `marque/marca` = mark, boundary
- 핵심 이미지: 경계를 긋거나 알아볼 표시를 남김
- 관련 문서: [[sign]], [[not]], [[marg]]
- 주의: `market`은 [[merc]] 계열로 따로 봄. `March` = 3월은 Mars 계열. `march` = 행진하다는 별도 어원으로 보는 편이 안전함.

## 출발 단어 분석

`demarcate` ← Spanish/Portuguese `demarcar` = mark out, set boundaries

- `de-`: down, completely, out
- `mark/marc`: boundary, sign
- `-ate`: 동사 접미사
- 원래 이미지: 땅이나 범위를 표시해 경계를 정함
- 의미 변화: 경계선을 그어 정함 → 구분을 명확히 함
- 현대 의미: 경계를 정하다, 구분하다

## 단어들

### demarcate

- 구성: `de-` + `mark/marc` + `-ate`
- 뜻: 경계를 정하다, 구분하다
- 어원적 연결: 어디까지가 한 영역인지 표시를 남겨 선을 그음
- 사용 맥락: `demarcate the boundary` = 경계를 정하다

### demarcation

- 발음: /ˌdimɑˈrkeɪʃən/
- 구성: `demarcate` + `-ion`
- 뜻: 경계 설정, 구분선
- 어원적 연결: 표시로 갈라 놓은 경계
- 사용 맥락: `a line of demarcation` = 경계선

### mark

- 발음: /ˈmɑrk/
- 구성: Germanic `marka` = boundary, sign
- 뜻: 표시, 자국; 표시하다
- 어원적 연결: 알아볼 수 있도록 남긴 흔적이나 경계 표식
- 사용 맥락: `mark the answer` = 답을 표시하다

### march

- 발음: /ˈmɑrtʃ/
- 구성: Old French `marche` = boundary, frontier
- 뜻: 변경, 접경지, 국경 지대
- 어원적 연결: 나라나 영지의 가장자리를 표시하는 경계 지역
- 사용 맥락: `the Welsh Marches` = 웨일스 접경 변경 지대

### marcher

- 발음: /ˈmɑrtʃɚ/
- 구성: `march` = borderland + `-er`
- 뜻: 변경 지대의, 접경 지역의
- 어원적 연결: 중심부가 아니라 경계에 놓인 지역을 가리킴
- 사용 맥락: `a marcher lord` = 변경 지대 영주

### marquess / marquis

- 발음: `marquess` /ˈmɑrkwəs/; `marquis` /ˌmɑˈrki/
- 구성: Old French `marchis` = ruler of a border area
- 뜻: 후작
- 어원적 연결: 원래는 왕국의 변경 지대를 맡은 귀족
- 사용 맥락: `the Marquess of Salisbury` = 솔즈베리 후작

### margrave

- 발음: /ˈmɑˌrgreɪv/
- 구성: German `Markgraf` = count of the march
- 뜻: 변경백
- 어원적 연결: 접경지 `Mark`를 다스리던 백작급 영주
- 사용 맥락: `the Margrave of Brandenburg` = 브란덴부르크 변경백

### marker

- 발음: /ˈmɑrkɚ/
- 구성: `mark` + `-er`
- 뜻: 표시물, 표지; 마커펜
- 어원적 연결: 위치나 의미를 알아보게 하는 표시
- 사용 맥락: `a historical marker` = 역사 표지

### landmark

- 발음: /ˈlæˌndmɑrk/
- 구성: `land` + `mark`
- 뜻: 주요 지형지물, 획기적 사건
- 어원적 연결: 땅에서 위치를 알아보게 하는 표시
- 사용 맥락: `a landmark decision` = 획기적인 판결

### benchmark

- 발음: /ˈbɛˌntʃmɑrk/
- 구성: `bench` + `mark`
- 뜻: 기준점, 벤치마크
- 어원적 연결: 원래는 측량 기준으로 삼는 표식. 이후 성과 비교 기준으로 확장
- 사용 맥락: `set a benchmark` = 기준을 세우다

### trademark

- 발음: /ˈtreɪˌdmɑrk/
- 구성: `trade` + `mark`
- 뜻: 상표; 특징
- 어원적 연결: 상품이나 회사를 알아보게 하는 거래상의 표시
- 사용 맥락: `a registered trademark` = 등록 상표

### hallmark

- 발음: /ˈhɑˌlmɑrk/
- 구성: `hall` + `mark`
- 뜻: 특징, 품질 보증 표시
- 어원적 연결: 원래 금은 제품의 품질을 보증하는 공식 표시
- 사용 맥락: `the hallmark of good writing` = 좋은 글의 특징

### remarkable

- 발음: /rɪˈmɑrkəbəl/
- 구성: `remark` + `-able`
- 뜻: 주목할 만한, 놀라운
- 어원적 연결: 특별히 표시하거나 언급할 만큼 눈에 띔
- 사용 맥락: `a remarkable achievement` = 주목할 만한 성취

### remark

- 발음: /rɪˈmɑrk/
- 구성: `re-` = again/intensively + `mark`
- 뜻: 말하다, 언급하다; 발언
- 어원적 연결: 눈에 띄는 점을 표시하듯 말로 짚음
- 사용 맥락: `make a brief remark` = 짧게 발언하다

## 기억 포인트

`mark` 계열은 "표시를 남겨 알아보게 함" 이미지.

- `mark`: 표시
- `demarcate`: 경계 표시로 구분함
- `march`: 국경의 변경 지대
- `marquess/marquis`: 변경을 맡은 귀족에서 온 후작
- `margrave`: 변경 지대를 다스린 백작
- `landmark`: 위치를 알아보게 하는 표시
- `benchmark`: 비교 기준이 되는 표시
- `trademark`: 상품을 알아보게 하는 표시
- `remarkable`: 표시해 둘 만큼 눈에 띄는

## 비교 단어

### sign

- 뜻: 표시, 신호, 서명
- 구분 포인트: [[sign]]은 신호와 의미 전달 쪽이 강함. `mark`는 눈에 보이는 자국·표식·경계 이미지가 강함.

### not

- 뜻: 표시하다, 적어 두다
- 구분 포인트: [[not]]은 기록·주석·알림 쪽. `mark`는 경계나 식별 표식 쪽.

### marg

- 뜻: 가장자리, 경계
- 구분 포인트: `margin`, `marginal`은 가장자리 공간. `demarcate`는 그 경계를 표시해 정하는 동작.

### March / march

- 뜻: 3월; 행진하다
- 구분 포인트: `March` = 3월은 Latin `Martius` = Mars의 달. `march` = 행진하다는 Old French `marcher` 계열로, 경계 의미의 `march`와는 구분해 두는 편이 안전함.
