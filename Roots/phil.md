# phil

- 출발 단어: `philosophy`
- 핵심 어원: Greek `philos` = loving, dear, friendly
- 핵심 이미지: 사랑함, 좋아함, 가까이 여김
- 관련 문서: [[phob]], [[logos]]
- 주의: `phil`은 "사랑"을 뜻하지만 낭만적 사랑보다 애호·우정·지적 관심에 가까운 경우가 많음.

## 출발 단어 분석

`philosophy` = `philo` + `sophy`

- `philo`: loving
- `sophy`: wisdom
- 원래 이미지: 지혜를 사랑함
- 현대 의미: 철학, 사고 체계

## 단어들

### philosophy

- 구성: Greek `philo` = loving + `sophia` = wisdom
- 뜻: 철학, 사고방식
- 어원적 연결: 지혜를 사랑하고 추구함
- 사용 맥락: `political philosophy` = 정치철학

### philosopher

- 구성: `philosophy` + `-er`
- 뜻: 철학자
- 어원적 연결: 지혜를 사랑하고 탐구하는 사람
- 사용 맥락: `ancient Greek philosophers` = 고대 그리스 철학자들

### philosophical

- 구성: `philosophy` + `-ical`
- 뜻: 철학적인, 사색적인
- 어원적 연결: 지혜와 근본 원리를 따지는 태도
- 사용 맥락: `a philosophical question` = 철학적 질문

### philanthropy

- 구성: Greek `philo` = loving + `anthropos` = human
- 뜻: 박애, 자선 활동
- 어원적 연결: 사람을 사랑함 → 인류애와 공익적 기부
- 사용 맥락: `corporate philanthropy` = 기업 자선 활동

### philanthropist

- 구성: `philanthropy` + `-ist`
- 뜻: 자선가, 박애주의자
- 어원적 연결: 사람을 사랑해 공익을 위해 돕는 사람
- 사용 맥락: `a wealthy philanthropist` = 부유한 자선가

### philology

- 구성: Greek `philo` = loving + `logos` = word, learning
- 뜻: 문헌학, 언어문헌 연구
- 어원적 연결: 말과 글, 학문을 사랑해 연구함
- 사용 맥락: 고전 문헌·언어사 연구에서 쓰임

### Anglophile

- 구성: `Anglo-` = English + `-phile`
- 뜻: 영국·영어권 문화 애호가
- 어원적 연결: 영국적인 것을 좋아하는 사람
- 사용 맥락: `an Anglophile reader` = 영국 문화를 좋아하는 독자

### bibliophile

- 구성: Greek `biblio` = book + `-phile`
- 뜻: 책 애호가
- 어원적 연결: 책을 사랑하는 사람
- 사용 맥락: `a bibliophile's collection` = 책 애호가의 장서

### cinephile

- 구성: `cine` = cinema + `-phile`
- 뜻: 영화 애호가
- 어원적 연결: 영화를 사랑하는 사람
- 사용 맥락: `a serious cinephile` = 진지한 영화 애호가

### audiophile

- 구성: Latin `audio` = hear + Greek `-phile`
- 뜻: 오디오 애호가, 음향 애호가
- 어원적 연결: 좋은 소리와 음향 장비를 좋아하는 사람
- 사용 맥락: `audiophile headphones` = 오디오 애호가용 헤드폰

## 기억 포인트

`phil` 계열은 "사랑함, 좋아함, 가까이 여김" 이미지.

- `philosophy`: 지혜를 사랑함
- `philanthropy`: 사람을 사랑함
- `philology`: 말과 글을 사랑해 연구함
- `bibliophile`: 책을 좋아하는 사람
- `cinephile`: 영화를 좋아하는 사람

## 비교 단어

### phob

- 뜻: 두려움, 혐오
- 구분 포인트: [[phob]]는 피하고 싶은 대상. `phil`은 가까이하고 싶은 대상.

- 뜻: 사랑
- 구분 포인트: `amor`는 라틴어 계열의 사랑. `phil`은 Greek 계열로 애호·우정·학문적 사랑에 자주 쓰임.
