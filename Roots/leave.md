# leave

- 출발 단어: `leave`
- 핵심 의미: 남겨 두다, 떠나다, 허락
- 관련 형태: Old English `laefan` = leave, let remain; Old English `leafe` = permission
- 관련 문서: [[rest]], [[cred]]

## 핵심 이미지

`leave`의 바탕은 "뒤에 남겨 둠".

어떤 장소를 뒤에 남기면 `leave` = 떠나다.
물건이나 사람을 남겨 두면 `leave` = 남기다.
남은 것은 `left`, `leftover`.
무언가를 빼고 남기면 `leave out`.
허락의 뜻인 `leave`는 "허락, 휴가"로 따로 자주 씀.

## 단어들

### leave

- 발음: /ˈliv/
- 구성: Old English `laefan` = leave, let remain
- 뜻: 떠나다, 남기다; 허락, 휴가
- 연결: 어떤 것을 뒤에 남겨 두거나, 어떤 행동을 허락함
- 사용 맥락: `leave home` = 집을 떠나다, `leave a message` = 메시지를 남기다

### leaving

- 발음: /ˈlivɪŋ/
- 구성: `leave` + `-ing`
- 뜻: 떠남, 남김
- 연결: 어떤 장소나 상태를 뒤에 남기고 벗어나는 과정
- 사용 맥락: `leaving school` = 학교를 떠남

### left

- 발음: /ˈlɛft/
- 구성: `leave`의 과거·과거분사
- 뜻: 떠났다, 남겨졌다; 남은
- 연결: 떠난 뒤에 남은 상태
- 사용 맥락: `There is no time left.` = 남은 시간이 없다

### leftover

- 발음: /ˈlɛˌftoʊvɚ/
- 구성: `left` + `over`
- 뜻: 남은 것, 잔여물
- 연결: 쓰거나 먹고 난 뒤 남겨진 것
- 사용 맥락: `leftover food` = 남은 음식

### leave behind

- 구성: `leave` + `behind`
- 뜻: 뒤에 남기다, 두고 가다
- 연결: 떠나면서 어떤 것을 뒤쪽에 남김
- 사용 맥락: `leave memories behind` = 기억을 남기다

### leave out

- 구성: `leave` + `out`
- 뜻: 빠뜨리다, 제외하다
- 연결: 전체 안에 넣지 않고 밖에 남겨 둠
- 사용 맥락: `leave out important details` = 중요한 세부 사항을 빠뜨리다

### leave off

- 구성: `leave` + `off`
- 뜻: 그만두다, 중단하다
- 연결: 하던 일을 떨어뜨려 놓고 멈춤
- 사용 맥락: `leave off work early` = 일을 일찍 그만두다

### leave of absence

- 구성: `leave` = permission + `absence`
- 뜻: 휴직, 휴가, 결석 허가
- 연결: 자리를 비워도 된다는 허락
- 사용 맥락: `take a leave of absence` = 휴직하다

## 비교 단어

`remain`은 [[rest]] 문서처럼 "남아 있다"가 중심.
`leave`는 떠나는 쪽과 남겨 두는 쪽을 함께 가짐.

`believe`, `belief`는 믿음 계열.
철자상 `leave`와 비슷해 보이지만 현대 학습에서는 [[cred]]의 믿다 계열과 비교하는 편이 더 실용적.

`leaf`는 나뭇잎 계열로, `leave`의 과거형 `left`와 섞지 않음.
