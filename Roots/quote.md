# quote

- 출발 단어: `quote`
- 핵심 어원: 라틴어 `quotus` / `quotare` = 몇 번째의, 번호를 매기다, 표시하다
- 핵심 이미지: 표시해서 따로 떼어냄
- 비교: [[quoth]] = 말하다, 선언하다 계열

## 출발 단어 분석

`quote`는 [[quoth]]와 모양은 비슷하지만 다른 어원.

- `quotus`: 몇 번째의, 얼마나 많은
- `quotare`: 번호를 매기다, 표시하다

따라서 `quote`는 원래 "표시하다" → "글에서 표시해 가져오다" → "인용하다"의 뜻.

## 단어들

### quote

- 발음: /ˈkwoʊt/
- 구성: 라틴어 `quotare`
- 뜻: 인용하다, 견적을 내다
- 어원적 연결: 문장이나 가격을 표시해 제시함
- 사용 맥락: `quote a passage` = 구절을 인용하다

### quotation

- 발음: /kwoʊˈteɪʃən/
- 구성: `quote` + `-ation`
- 뜻: 인용, 인용문
- 어원적 연결: 표시해 가져온 말이나 문장
- 사용 맥락: `a direct quotation` = 직접 인용

### unquote

- 발음: /ˈʌˈnkwoʊt/
- 구성: `un-` = reverse/end + `quote`
- 뜻: 인용 끝, 따옴표를 닫다
- 어원적 연결: 인용으로 표시한 부분을 끝냄
- 사용 맥락: `quote, unquote` = 이른바, 따옴표 쳐서 말하자면

### quota

- 발음: /ˈkwoʊtə/
- 구성: 라틴어 `quota pars` = 얼마만큼의 부분
- 뜻: 할당량, 몫
- 어원적 연결: 전체 중 정해진 수량이나 몫
- 사용 맥락: `meet a quota` = 할당량을 채우다

### quotient

- 발음: /ˈkwoʊʃənt/
- 구성: 라틴어 `quotiens` = 몇 번이나
- 뜻: 몫
- 어원적 연결: 나눗셈에서 몇 번 들어가는지의 결과
- 사용 맥락: `the quotient of 12 divided by 3` = 12를 3으로 나눈 몫

### quotidian

- 구성: 라틴어 `quotidie` = 매일
- 뜻: 매일의, 일상적인
- 어원적 연결: `quot` = how many + `dies` = day. 날마다 반복되는 것
- 사용 맥락: `quotidian tasks` = 일상적인 일들

## 비교 단어

### quoth

- 구성: 고대 영어 `cweðan` = 말하다
- 뜻: 말했다
- 어원적 연결: [[quote]] 계열 아님. [[quoth]] 참고
- 구분 포인트: `quoth` = 말했다, `quote` = 인용하다

### bequeath

- 구성: 고대 영어 `be-` + `cweðan`
- 뜻: 유언으로 남기다
- 어원적 연결: [[quote]] 계열 아님. [[quoth]] 계열
- 구분 포인트: `bequeath`는 "말로 남김", `quote`는 "표시해 인용함"

## 기억 포인트

`quote` 계열은 "수량/순서를 표시함" 이미지.

- `quote`: 표시해 인용함
- `quotation`: 인용문
- `unquote`: 인용 표시를 끝냄
- `quota`: 정해진 몫
- `quotient`: 나눗셈의 몫
- `quotidian`: 매일의

비교:

- [[quoth]]: 말하다 계열
- [[quote]]: 표시하다, 수량을 정하다 계열
