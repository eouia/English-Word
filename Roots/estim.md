# estim

- 출발 단어: `estimate`
- 핵심 어원: Latin `aestimare` = value, appraise, reckon
- 핵심 이미지: 값을 매기고 어림하여 판단함
- 관련 어근: [[val]], [[equ]], [[cred]]

## 출발 단어 분석

`estimate`는 Latin `aestimare`에서 온 말.

핵심은 "가치를 매김"에서 출발해, 숫자나 가능성을 "대략 판단함"으로 확장됨.

따라서 `estimate`는 정확한 계산이라기보다 "근거를 가지고 어림함"의 느낌.

## 단어들

### estimate

- 발음: n. /ˈɛstəmət/, v. /ˈɛstəˌmeɪt/
- 구성: Latin `aestimare`
- 의미: 추정하다, 견적을 내다, 평가하다
- 어원적 연결: 값이나 정도를 매겨 판단함
- 사용 맥락: `estimate the cost` = 비용을 추산하다

### estimation

- 발음: /ˌɛstəˈmeɪʃən/
- 구성: `estimate` + `-ion`
- 의미: 추정, 판단, 평가
- 어원적 연결: 값을 매기거나 정도를 판단하는 과정
- 사용 맥락: `in my estimation` = 내 판단으로는

### estimated

- 발음: /ˈɛstəˌmeɪtəd/
- 구성: `estimate` + `-ed`
- 의미: 추정된
- 어원적 연결: 대략 계산해 매긴 수치
- 사용 맥락: `the estimated cost` = 추정 비용

### underestimate

- 발음: /ˈʌndɚˈɛstəˌmeɪt/
- 구성: `under-` + `estimate`
- 의미: 과소평가하다, 낮게 추산하다
- 어원적 연결: 실제보다 낮은 값으로 판단함
- 사용 맥락: `underestimate the difficulty` = 난이도를 과소평가하다

### overestimate

- 발음: /ˌoʊvɚˈɛstəˌmeɪt/
- 구성: `over-` + `estimate`
- 의미: 과대평가하다, 높게 추산하다
- 어원적 연결: 실제보다 높은 값으로 판단함
- 사용 맥락: `overestimate demand` = 수요를 과대평가하다

### esteem

- 발음: /əˈstim/
- 구성: Old French `estimer` < Latin `aestimare`
- 의미: 존중, 존경하다
- 어원적 연결: 어떤 사람이나 가치를 높게 평가함
- 사용 맥락: `hold someone in high esteem` = 누군가를 높이 존중하다

### self-esteem

- 구성: `self` + `esteem`
- 의미: 자존감
- 어원적 연결: 자기 자신에게 매기는 가치감
- 사용 맥락: `low self-esteem` = 낮은 자존감

## 기억 포인트

- `estimate`: 값을 어림해 매김
- `underestimate`: 낮게 매김
- `overestimate`: 높게 매김
- `esteem`: 높게 평가하고 존중함
- `self-esteem`: 자기 가치 평가

## 비교 단어

- [[val]]
  - 핵심: 가치, 힘, 효력
  - 구분 포인트: `val`은 가치 자체나 효력. `estim`은 그 가치를 판단하고 매기는 행위

- [[equ]]
  - 핵심: 같음, 균형
  - 구분 포인트: `equation`은 정확히 맞춰 푸는 느낌. `estimate`는 대략 값을 잡는 느낌

- [[cred]]
  - 핵심: 믿다
  - 구분 포인트: `credible estimate`는 믿을 만한 추정치. `cred`는 신뢰, `estim`은 추산
