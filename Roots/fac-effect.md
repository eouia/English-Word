# fac-effect

- 출발 단어: `effect`
- 핵심 어원: Latin `facere` / `fect-` = 만들다, 하다
- 핵심 이미지: 작용해서 결과나 상태를 만들어 냄
- 허브 문서: [[fac]]

## 단어들

### effect

- 구성: `ex-` = out + `fect`
- 뜻: 효과, 결과
- 어원적 연결: 행위가 밖으로 만들어 낸 결과
- 사용 맥락: `the effect of stress` = 스트레스의 영향

### effective

- 구성: `effect` + `-ive`
- 뜻: 효과적인, 실제로 시행되는
- 어원적 연결: 결과를 만들어 냄
- 사용 맥락: `an effective method` = 효과적인 방법

### ineffective

- 구성: `in-` = not + `effective`
- 뜻: 효과 없는, 비효과적인
- 어원적 연결: 원하는 결과를 만들어 내지 못함
- 사용 맥락: `an ineffective policy` = 효과 없는 정책

### effectual

- 구성: `effect` + `-ual`
- 뜻: 효과 있는, 유효한
- 어원적 연결: 의도한 결과를 실제로 만들어 냄
- 사용 맥락: `effectual action` = 효과 있는 조치

### ineffectual

- 구성: `in-` = not + `effectual`
- 뜻: 효과 없는, 무력한
- 어원적 연결: 결과를 만들어 내는 힘이 없음
- 사용 맥락: `an ineffectual attempt` = 효과 없는 시도, `an ineffectual leader` = 무력한 지도자

### ineffectually

- 구성: `ineffectual` + `-ly`
- 뜻: 효과 없이, 무력하게
- 어원적 연결: 결과를 만들지 못하는 방식
- 사용 맥락: `argue ineffectually` = 별 효과 없이 주장하다

### affect

- 구성: `ad-` = to + `fect`
- 뜻: 영향을 미치다
- 어원적 연결: 어떤 대상에 작용해 상태를 만들어 바꿈
- 사용 맥락: `affect the outcome` = 결과에 영향을 미치다

### affection

- 구성: `ad-` + `fect` + `-ion`
- 뜻: 애정
- 어원적 연결: 마음이 어떤 쪽으로 만들어져 기울어진 상태
- 사용 맥락: `show affection` = 애정을 보이다

### defect

- 구성: `de-` = down, away + `fect`
- 뜻: 결함, 결점, 이탈하다
- 어원적 연결: 제대로 만들어진 상태에서 빠짐
- 사용 맥락: `a product defect` = 제품 결함

### perfect

- 구성: `per-` = through, completely + `fect`
- 뜻: 완전한, 완벽한
- 어원적 연결: 끝까지 완전히 만들어진 상태
- 사용 맥락: `a perfect example` = 완벽한 예

## 기억 포인트

`fac-effect`는 "작용해 결과를 만듦" 갈래.

- `effect`: 결과
- `effective`: 결과를 내는
- `ineffectual`: 결과를 내지 못하는
- `affect`: 영향을 주어 바꿈
- `defect`: 만들어진 상태에서 빠짐
- `perfect`: 완전히 만들어짐
