# trust

- 출발 단어: `trust`
- 핵심 어원: Old Norse / Germanic 계열 `traust` = 믿을 만함, 도움, 확신
- 핵심 이미지: 단단히 믿고 기대어 맡김
- 관련해서 헷갈릴 수 있는 어근: [[cred]] = 라틴어 `credere`, 믿다 / 맡기다 계열, [[fid]] = 라틴어 `fides`, 믿음 / 충실함 계열

## 출발 단어 분석

`trust`는 라틴어 어근이 아니라 게르만계 기본어.

- Old Norse `traust`: 도움, 보호, 확신
- Germanic 계열의 `true`, `truth`, `troth`와 의미상 가까움

따라서 `trust`의 핵심은 "믿을 수 있을 만큼 단단함" → "신뢰, 믿고 맡김".

## 단어들

### trust

- 발음: /ˈtrʌst/
- 구성: Germanic `trust`
- 뜻: 신뢰, 믿다, 맡기다
- 어원적 연결: 믿을 만하다고 보고 기대어 맡김
- 사용 맥락: `trust your judgment` = 네 판단을 믿다

### trusted

- 발음: /ˈtrʌstɪd/
- 구성: `trust` + `-ed`
- 뜻: 신뢰받는, 믿을 수 있는
- 어원적 연결: 이미 믿음의 대상이 된 상태
- 사용 맥락: `a trusted adviser` = 신뢰받는 조언자

### trustworthy

- 발음: /ˈtrʌˌstwɝði/
- 구성: `trust` + `worthy`
- 뜻: 신뢰할 만한
- 어원적 연결: 믿음을 받을 가치가 있음
- 사용 맥락: `a trustworthy source` = 믿을 만한 출처

### entrust

- 발음: /ɛˈntrʌst/
- 구성: `en-` = into, make + `trust`
- 뜻: 맡기다, 위임하다
- 어원적 연결: 신뢰 안에 넣음. 믿고 책임을 넘김
- 사용 맥락: `entrust a task to someone` = 누군가에게 일을 맡기다

### distrust

- 발음: /dɪˈstrʌst/
- 구성: `dis-` = opposite, away + `trust`
- 뜻: 불신하다, 의심
- 어원적 연결: 믿음을 거둬 둠
- 사용 맥락: `distrust the report` = 그 보고서를 불신하다

### mistrust

- 발음: /mɪˈstrʌst/
- 구성: `mis-` = wrong, bad + `trust`
- 뜻: 불신하다, 의심하다
- 어원적 연결: 제대로 믿지 못함
- 사용 맥락: `mistrust his motives` = 그의 동기를 의심하다

### true

- 발음: /ˈtru/
- 구성: Germanic `true`
- 뜻: 진짜의, 참된, 충실한
- 어원적 연결: 원래 "믿을 만한, 충실한" 이미지
- 사용 맥락: `a true friend` = 진정한 친구

### truth

- 발음: /ˈtruθ/
- 구성: `true` + `-th`
- 뜻: 진실, 사실
- 어원적 연결: 믿을 수 있는 참된 상태
- 사용 맥락: `tell the truth` = 진실을 말하다

### troth

- 발음: /ˈtroʊθ/
- 구성: Germanic `troth`
- 뜻: 약속, 충실함
- 어원적 연결: 믿고 지키는 충실함. 현대 영어에서는 주로 오래된 표현
- 사용 맥락: `pledge one's troth` = 약혼을 맹세하다

## 기억 포인트

`trust` 계열은 "단단히 믿고 맡김" 이미지.

- `trust`: 믿고 맡김
- `trustworthy`: 믿을 가치가 있음
- `entrust`: 믿고 넘겨 맡김
- `distrust`, `mistrust`: 믿지 못함
- `true`, `truth`: 믿을 수 있을 만큼 참됨

## 비교 단어

### cred

- 뜻: 믿다, 믿고 맡기다
- 구분 포인트: [[cred]]는 라틴계 시험 어휘에서 "신용, 믿을 만함, 공식 인정"으로 많이 나타남. `trust`는 일상적인 신뢰와 충실함의 기본어

### fid

- 뜻: 믿음, 충실함, 신의
- 구분 포인트: [[fid]]는 `faith`, `fidelity`, `confidence`처럼 믿음과 충실함의 라틴계 어휘. `trust`는 일상 기본어.
