# struct

- 출발 단어: `obstruct`
- 핵심 어원: 라틴어 `struere` / `struct-` = 쌓다, 세우다, 배열하다
- 핵심 이미지: 재료나 생각을 쌓아 세움
- 관련해서 헷갈릴 수 있는 어근: [[scrib]] = 쓰다, [[tract]] = 끌다

## 출발 단어 분석

`obstruct`는 `ob-` + `struct`.

- `ob-`: against, in front of
- `struct`: 쌓다, 세우다

따라서 `obstruct`는 "앞에 쌓아 세우다" → "막다, 방해하다"의 뜻.

## 단어들

### obstruct

- 발음: /əˈbstrʌkt/
- 구성: `ob-` = in front of, against + `struct`
- 뜻: 막다, 방해하다
- 어원적 연결: 앞에 무언가를 쌓아 길이나 진행을 막음
- 사용 맥락: `obstruct traffic` = 교통을 방해하다

### obstruction

- 발음: /əˈbstrʌkʃən/
- 구성: `obstruct` + `-ion`
- 뜻: 방해, 장애물, 막힘
- 어원적 연결: 앞에 쌓여 막는 것
- 사용 맥락: `an obstruction in the road` = 도로 위 장애물

### construct

- 발음: n. /ˈkɑnstrəkt/, v. /kəˈnstrʌkt/
- 구성: `con-` = together + `struct`
- 뜻: 건설하다, 구성하다
- 어원적 연결: 여러 재료를 함께 쌓아 세움
- 사용 맥락: `construct a bridge` = 다리를 건설하다

### construction

- 발음: /kəˈnstrʌkʃən/
- 구성: `construct` + `-ion`
- 뜻: 건설, 구조, 해석
- 어원적 연결: 재료나 의미를 쌓아 만든 것
- 사용 맥락: `under construction` = 공사 중

### structure

- 발음: /ˈstrʌktʃɚ/
- 구성: Latin `structura`
- 뜻: 구조, 체계
- 어원적 연결: 쌓아 세워진 배열
- 사용 맥락: `sentence structure` = 문장 구조

### structural

- 발음: /ˈstrʌktʃɚəl/
- 구성: `structure` + `-al`
- 뜻: 구조적인
- 어원적 연결: 전체를 지탱하는 짜임과 관련
- 사용 맥락: `structural problems` = 구조적 문제

### destruct

- 발음: /dɪˈstrʌkt/
- 구성: `de-` = down, away + `struct`
- 뜻: 파괴하다
- 어원적 연결: 쌓아 세운 것을 무너뜨림
- 사용 맥락: 주로 `self-destruct` 형태로 사용

### destruction

- 발음: /dɪˈstrʌkʃən/
- 구성: `destruct` + `-ion`
- 뜻: 파괴, 파멸
- 어원적 연결: 세워진 구조를 무너뜨림
- 사용 맥락: `environmental destruction` = 환경 파괴

### destructive

- 발음: /dɪˈstrʌktɪv/
- 구성: `destruct` + `-ive`
- 뜻: 파괴적인
- 어원적 연결: 구조를 무너뜨리는 성질
- 사용 맥락: `destructive behavior` = 파괴적인 행동

### instruct

- 발음: /ˌɪˈnstrʌkt/
- 구성: `in-` = in + `struct`
- 뜻: 가르치다, 지시하다
- 어원적 연결: 지식이나 절차를 마음속에 정리해 세워 줌
- 사용 맥락: `instruct students` = 학생들을 가르치다

### instruction

- 발음: /ˌɪˈnstrʌkʃən/
- 구성: `instruct` + `-ion`
- 뜻: 지시, 설명, 교육
- 어원적 연결: 따라야 할 절차를 차례로 세워 줌
- 사용 맥락: `follow the instructions` = 지시 사항을 따르다

### instructor

- 발음: /ˌɪˈnstrʌktɚ/
- 구성: `instruct` + `-or`
- 뜻: 강사, 교관
- 어원적 연결: 지식이나 절차를 세워 주는 사람
- 사용 맥락: `a driving instructor` = 운전 강사

### instrument

- 발음: /ˈɪnstrəmənt/
- 구성: Latin `instrumentum`
- 뜻: 도구, 악기, 수단
- 어원적 연결: 어떤 일을 수행하도록 마련된 도구. `struere` 계열과 연결
- 사용 맥락: `a musical instrument` = 악기

### instrumental

- 발음: /ˌɪnstrəˈmɛntəl/
- 구성: `instrument` + `-al`
- 뜻: 중요한 역할을 하는, 도구의, 기악의
- 어원적 연결: 어떤 일을 이루는 수단으로 작용함
- 사용 맥락: `be instrumental in the success` = 성공에 중요한 역할을 하다

### infrastructure

- 발음: /ˌɪnfrəˈstrʌktʃɚ/
- 구성: `infra-` = below + `structure`
- 뜻: 기반 시설
- 어원적 연결: 아래에서 사회나 조직을 받치는 구조
- 사용 맥락: `public infrastructure` = 공공 기반 시설

## 기억 포인트

`struct` 계열은 "쌓아 세움" 이미지.

- `obstruct`: 앞에 쌓아 막음
- `construct`: 함께 쌓아 만듦
- `structure`: 쌓아 세운 짜임
- `destruction`: 세운 것을 무너뜨림
- `instruct`: 지식이나 절차를 세워 줌
- `instrument`: 일을 하도록 마련된 도구
- `infrastructure`: 아래에서 받치는 구조

## 비교 단어

### destroy

- 뜻: 파괴하다
- 구분 포인트: `destroy`도 라틴어 `destruere` 계열. 현대 형태는 `struct`가 보이지 않지만 `destruction`, `destructive`와 연결됨

### obstacle

- 뜻: 장애물
- 구분 포인트: `obstruct`와 뜻은 비슷하지만, `obstacle`은 "앞에 서 있음" 계열로 별도 정리 가능
