# present

- 출발 단어: `represent`
- 핵심 어원: 라틴어 `praeesse` / `praesent-` = 앞에 있다, 가까이 있다
- 핵심 이미지: 눈앞에 있음. 앞에 놓아 보이게 함
- 관련해서 헷갈릴 수 있는 어근: [[sent]] = 느끼다, 생각하다, 판단하다 계열

## 출발 단어 분석

`represent`는 라틴어 `repraesentare`에서 옴.

- `re-`: 다시, 앞에 다시
- `present`: 앞에 있게 하다, 나타내다

따라서 `represent`는 "대상을 다시 앞에 세워 보이게 함" → "나타내다, 상징하다, 대표하다"의 뜻.

## 단어들

### present

- 발음: n./adj. /ˈprɛzənt/, v. /prɪˈzɛnt/
- 구성: `prae-` = before + `esse` = be
- 뜻: 현재의, 참석한, 선물, 제시하다
- 어원적 연결: 눈앞에 있는 상태. 시간으로는 "지금", 장소로는 "참석", 행위로는 "앞에 내놓음"
- 사용 맥락: `be present at the meeting` = 회의에 참석하다

### presence

- 발음: /ˈprɛzəns/
- 구성: `present` + `-ence`
- 뜻: 있음, 참석, 존재감
- 어원적 연결: 눈앞이나 그 자리에 있는 상태
- 사용 맥락: `her presence in the room` = 방 안의 그녀의 존재감

### presentation

- 발음: /ˌprɛzəˈnteɪʃən/
- 구성: `present` + `-ation`
- 뜻: 발표, 제시, 증정
- 어원적 연결: 정보나 물건을 사람들 앞에 내놓음
- 사용 맥락: `give a presentation` = 발표하다

### presenter

- 발음: /ˈprɛzəntɚ/
- 구성: `present` + `-er`
- 뜻: 발표자, 진행자
- 어원적 연결: 내용을 사람들 앞에 내놓는 사람
- 사용 맥락: `a TV presenter` = TV 진행자

### represent

- 발음: /ˌrɛprɪˈzɛnt/
- 구성: `re-` + `present`
- 뜻: 나타내다, 상징하다, 대표하다
- 어원적 연결: 직접 없는 대상을 앞에 다시 세워 보이게 함
- 사용 맥락: `This symbol represents peace.` = 이 상징은 평화를 나타냄

### representation

- 발음: /ˌrɛprəzɛˈnteɪʃən/
- 구성: `represent` + `-ation`
- 뜻: 표현, 묘사, 대표, 대의
- 어원적 연결: 어떤 대상이나 집단을 대신 앞에 세워 보여 주는 행위
- 사용 맥락: `political representation` = 정치적 대표

### representative

- 발음: /ˌrɛprəˈzɛntətɪv/
- 구성: `represent` + `-ative`
- 뜻: 대표자, 대표하는, 전형적인
- 어원적 연결: 사람이나 집단을 대신 앞에 서는 사람 또는 사례
- 사용 맥락: `a sales representative` = 영업 담당자

### representational

- 발음: /ˌrɛprəzəˈnteɪʃənəl/
- 구성: `representation` + `-al`
- 뜻: 대상을 알아볼 수 있게 표현하는
- 어원적 연결: 실제 대상을 앞에 다시 보여 주는 성격
- 사용 맥락: `representational art` = 구상 미술

### misrepresent

- 발음: /mɪˌsrɛprəˈzɛnt/
- 구성: `mis-` = wrongly + `represent`
- 뜻: 잘못 나타내다, 왜곡하다
- 어원적 연결: 대상을 앞에 다시 세워 보이되 잘못된 모습으로 보이게 함
- 사용 맥락: `misrepresent the facts` = 사실을 왜곡하다

### misrepresentation

- 발음: /mɪˌsrɛprɪzɛˈnteɪʃən/
- 구성: `misrepresent` + `-ation`
- 뜻: 허위 진술, 왜곡, 오표현
- 어원적 연결: 사실이나 대상을 잘못 앞에 세워 보여 준 결과
- 사용 맥락: `a serious misrepresentation` = 중대한 허위 진술

### absent

- 발음: /ˈæbsənt/
- 구성: `ab-` = away + `esse` = be
- 뜻: 결석한, 없는
- 어원적 연결: 그 앞이나 그 자리에 있지 않음
- 사용 맥락: `be absent from school` = 학교에 결석하다

### absence

- 발음: /ˈæbsəns/
- 구성: `absent` + `-ence`
- 뜻: 부재, 결석, 없음
- 어원적 연결: 있어야 할 자리에 없는 상태
- 사용 맥락: `in the absence of evidence` = 증거가 없는 상태에서

## 기억 포인트

`present` 계열은 "앞에 있음 / 앞에 내놓음" 이미지.

- `present`: 눈앞에 있음, 앞에 내놓음
- `presence`: 그 자리에 있음
- `presentation`: 앞에 내놓아 보여 주는 일
- `represent`: 없는 대상을 다시 앞에 세워 보여 줌
- `misrepresent`: 잘못 앞에 세워 보여 줌
- `representative`: 대신 앞에 서는 사람
- `absent`: 앞에 있지 않음
