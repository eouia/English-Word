# ped

- 출발 단어: `expedition`
- 핵심 어원: 라틴어 `pes` / `ped-` = 발
- 핵심 이미지: 발로 가거나, 발의 움직임을 막고 풀어 줌
- 관련해서 헷갈릴 수 있는 어근: [[cess]] = 가다, 물러나다 계열, [[plore]] = 탐사하다 계열

## 출발 단어 분석

`expedition`은 Latin `expeditio`에서 온 말.

- `ex-`: out
- `pes` / `ped-`: foot
- `expedire`: 발을 묶은 것을 풀다, 방해를 제거하다

따라서 `expedition`은 원래 "발을 풀어 앞으로 가게 함" → "신속한 처리" → "목적을 가진 여행, 원정, 탐사"의 뜻.

## 단어들

### expedition

- 구성: `ex-` = out + `ped`
- 뜻: 원정, 탐험, 신속한 처리
- 어원적 연결: 발의 방해를 풀고 목적지로 나아감
- 사용 맥락: `a scientific expedition` = 과학 탐사

### expedient

- 구성: `ex-` = out + `ped`
- 뜻: 편리한, 방편적인, 임시방편
- 어원적 연결: 앞으로 나아가게 해 주는 수단
- 사용 맥락: `a politically expedient choice` = 정치적으로 편리한 선택

### expedite

- 구성: `ex-` = out + `ped`
- 뜻: 신속히 처리하다, 촉진하다
- 어원적 연결: 발을 묶은 것을 풀어 빨리 나아가게 함
- 사용 맥락: `expedite the process` = 절차를 신속히 처리하다

### expeditious

- 구성: `expedite` + `-ious`
- 뜻: 신속한, 효율적인
- 어원적 연결: 막힘 없이 빨리 진행됨
- 사용 맥락: `an expeditious response` = 신속한 대응

### impede

- 구성: `in-` = in + `ped`
- 뜻: 방해하다, 지연시키다
- 어원적 연결: 발을 묶어 움직이지 못하게 함
- 사용 맥락: `impede progress` = 진전을 방해하다

### impediment

- 구성: `impede` + `-ment`
- 뜻: 장애물, 방해 요인
- 어원적 연결: 발을 묶는 것
- 사용 맥락: `a speech impediment` = 언어 장애

### pedestrian

- 구성: `pedestr-` = on foot
- 뜻: 보행자, 평범한
- 어원적 연결: 발로 걷는 사람. "걸어 다니는 수준"에서 평범하다는 뜻도 생김
- 사용 맥락: `pedestrian safety` = 보행자 안전

### pedal

- 구성: `ped` + `-al`
- 뜻: 페달, 페달을 밟다
- 어원적 연결: 발로 밟는 장치
- 사용 맥락: `press the brake pedal` = 브레이크 페달을 밟다

### pedestal

- 구성: Latin `pedestal` = foot of a stall
- 뜻: 받침대, 대좌
- 어원적 연결: 구조물의 발 역할을 하는 받침
- 사용 맥락: `put someone on a pedestal` = 누군가를 떠받들다

### centipede

- 구성: [[cent]] = hundred + `ped`
- 뜻: 지네
- 어원적 연결: 발이 많다는 이미지
- 사용 맥락: `a centipede has many legs` = 지네는 다리가 많다

### tripod

- 구성: Greek [[tri]] = three + `pod`
- 뜻: 삼각대
- 어원적 연결: 세 발을 가진 받침
- 사용 맥락: `set up a tripod` = 삼각대를 설치하다

### podium

- 구성: Greek `podion` = little foot, base
- 뜻: 연단, 시상대
- 어원적 연결: 발을 올려 서는 받침
- 사용 맥락: `stand on the podium` = 시상대에 서다

## 기억 포인트

`ped` 계열은 "발 / 발의 움직임" 이미지.

- `expedition`: 발의 방해를 풀고 나아가는 여행
- `expedite`: 막힘을 풀어 신속히 진행함
- `impede`: 발을 묶어 방해함
- `pedestrian`: 발로 걷는 사람
- `pedal`: 발로 밟는 장치
- `pedestal`: 발처럼 받치는 기반

## 비교 단어

### plore

- 뜻: 탐사하다, 호소하다
- 구분 포인트: [[plore]]의 `explore`는 낯선 곳을 살핌. `expedition`은 발을 풀어 목적지로 가는 조직적 여행

### cess

- 뜻: 가다, 물러나다
- 구분 포인트: [[cess]]는 움직임 자체. `ped`는 발이나 발의 움직임을 막고 푸는 이미지
