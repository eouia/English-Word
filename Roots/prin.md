# prin

- 출발 단어: `principal`
- 핵심 어원: Latin `princeps`, `principium` = first, chief, beginning
- 핵심 이미지: 첫째 자리, 우두머리, 처음에 놓인 기준
- 관련 문서: [[prim]]
- 주의: `principal`과 `principle`은 발음이 같아 헷갈림. `principal`은 사람·금액·중요성, `principle`은 원칙·원리.

## 출발 단어 분석

`principal` ← Latin `principalis`

- `princeps`: first, chief
- `principalis`: first, chief, principal
- 원래 이미지: 첫째 위치에 있는 사람이나 것
- 현대 의미: 주요한, 교장, 원금

## 단어들

### principal

- 발음: /ˈprɪnsəpəl/
- 구성: Latin `princeps` = first, chief + `-al`
- 뜻: 주요한, 교장, 원금
- 어원적 연결: 첫째 위치에 있는 사람·것 → 가장 중요한 것
- 사용 맥락: `the principal reason` = 주된 이유, `school principal` = 교장, `principal amount` = 원금

### principle

- 발음: /ˈprɪnsəpəl/
- 구성: Latin `principium` = beginning, foundation
- 뜻: 원칙, 원리
- 어원적 연결: 처음에 놓이는 근본 기준
- 사용 맥락: `a basic principle` = 기본 원칙

### principled

- 발음: /ˈprɪnsəpəld/
- 구성: `principle` + `-ed`
- 뜻: 원칙 있는, 신념에 따른
- 어원적 연결: 행동의 시작점이 되는 기준을 가짐
- 사용 맥락: `a principled decision` = 원칙 있는 결정

### principality

- 발음: /ˌprɪnsɪˈpælɪti/
- 구성: Latin `princeps` 계열 + `-ality`
- 뜻: 공국, 군주의 지배 지역
- 어원적 연결: 첫째 지위의 통치자가 다스리는 영역
- 사용 맥락: `the Principality of Monaco` = 모나코 공국

### prince

- 발음: /ˈprɪns/
- 구성: Latin `princeps` = first, chief
- 뜻: 왕자, 군주
- 어원적 연결: 첫째 사람, 우두머리
- 사용 맥락: `the crown prince` = 왕세자

### princess

- 발음: /ˈprɪnsɛs/
- 구성: `prince` + `-ess`
- 뜻: 공주, 왕자비
- 어원적 연결: 왕족의 높은 지위와 관련됨
- 사용 맥락: `a royal princess` = 왕실 공주

### princely

- 발음: /ˈprɪnsli/
- 구성: `prince` + `-ly`
- 뜻: 왕자다운, 아주 큰
- 어원적 연결: 왕자나 군주처럼 크고 품위 있음
- 사용 맥락: `a princely sum` = 거액

## 기억 포인트

`prin` 계열은 "첫째 자리, 우두머리, 처음 기준" 이미지.

- `principal`: 첫째 위치의 → 주요한, 교장, 원금
- `principle`: 처음에 놓인 기준 → 원칙
- `prince`: 첫째 사람, 우두머리 → 왕자·군주
- `principled`: 원칙을 가진

## 비교 단어

### prim

- 뜻: 첫째, 가장 앞선, 가장 중요한
- 구분 포인트: [[prim]]은 Latin `primus` = first 계열. `prin`은 `princeps/principium`에서 온 "첫째 위치의 사람·기준" 쪽이 더 강함.
