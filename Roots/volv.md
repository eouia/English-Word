# volv

- 출발 단어: `evolve`
- 핵심 어원: Latin `volvere` / `volut-` = roll, turn, wind
- 핵심 이미지: 말린 것이 굴러가거나 펼쳐짐
- 관련 어근: [[vers]], [[tend]], Latin `voluntas`

## 출발 단어 분석

`evolve`는 Latin `evolvere`에서 온 말.

- `e- / ex-`: out
- `volvere`: roll, turn, unroll

따라서 `evolve`는 원래 "말린 것을 밖으로 펼침" → "점차 전개되다, 진화하다"의 뜻.

## 단어들

### evolve

- 구성: [[ex]] = out + `volvere`
- 의미: 진화하다, 발전하다
- 어원적 연결: 말린 것이 밖으로 펼쳐지듯 변화가 전개됨
- 사용 맥락: `species evolve` = 종이 진화하다

### evolution

- 구성: `e-` + `volut-` + `-ion`
- 의미: 진화, 발전
- 어원적 연결: 안에 말려 있던 것이 펼쳐져 나오는 과정
- 사용 맥락: `the evolution of language` = 언어의 진화

### evolutionary

- 구성: `evolution` + `-ary`
- 의미: 진화의, 점진적 발전의
- 어원적 연결: 전개와 변화 과정에 관련됨
- 사용 맥락: `evolutionary theory` = 진화 이론

### revolve

- 구성: `re-` = again/back + `volvere`
- 의미: 회전하다, 중심으로 돌다
- 어원적 연결: 다시 돌고 돌아감
- 사용 맥락: `planets revolve around the sun` = 행성이 태양 주위를 돈다

### revolution

- 구성: `re-` + `volut-` + `-ion`
- 의미: 혁명, 회전
- 어원적 연결: 한 바퀴 돌아 기존 질서가 뒤집힘
- 사용 맥락: `the Industrial Revolution` = 산업혁명

### revolutionary

- 구성: `revolution` + `-ary`
- 의미: 혁명적인
- 어원적 연결: 질서를 크게 뒤집어 돌려 놓음
- 사용 맥락: `a revolutionary idea` = 혁명적인 생각

### involve

- 구성: [[in]] = in + `volvere`
- 의미: 포함하다, 관련시키다
- 어원적 연결: 안으로 말아 넣음
- 사용 맥락: `involve several factors` = 여러 요인을 포함하다

### involvement

- 구성: `involve` + `-ment`
- 의미: 관여, 관련
- 어원적 연결: 어떤 일 안에 말려 들어감
- 사용 맥락: `student involvement` = 학생 참여

### convoluted

- 구성: `con-` = together + `volut-`
- 의미: 복잡하게 얽힌, 꼬인
- 어원적 연결: 함께 돌돌 말려 얽힘
- 사용 맥락: `a convoluted argument` = 복잡하게 꼬인 주장

### volume

- 구성: Latin `volumen` = roll, scroll
- 의미: 책 한 권, 부피, 음량
- 어원적 연결: 원래 말아 둔 두루마리. 이후 책, 한 덩어리의 양으로 확장
- 사용 맥락: `turn up the volume` = 음량을 높이다

### devolve

- 구성: `de-` = down + `volvere`
- 의미: 이양되다, 퇴화하다
- 어원적 연결: 아래로 굴러 내려가 맡겨지거나 나빠짐
- 사용 맥락: `devolve power to local governments` = 권한을 지방 정부에 이양하다

## 기억 포인트

- `evolve`: 밖으로 펼쳐져 발전함
- `evolution`: 전개와 진화
- `revolve`: 다시 돌고 돎
- `revolution`: 한 바퀴 돌아 질서가 뒤집힘
- `involve`: 안으로 말아 넣어 관련시킴
- `convoluted`: 함께 말려 복잡하게 꼬임
- `volume`: 말린 두루마리, 한 덩어리의 양
