# melan

- 출발 단어: `melanin`
- 핵심 어원: Greek `melas`, `melan-` = 검은, 어두운
- 핵심 이미지: 검은색, 어두운 색소, 어둡게 변한 상태
- 관련 문서: [[alb]], [[chrom]]
- 주의: `melan` 계열은 생물·의학 어휘가 많음. 일반 학습에서는 `melanin`, `melancholy`, `melanoma` 정도가 우선.

## 출발 단어 분석

`melanin` ← Greek `melan-` + 생물학 명사형

- `melan-`: black, dark
- 원래 이미지: 피부·머리카락·눈에 어두운 색을 내는 색소
- 현대 의미: 멜라닌, 어두운 색소

## 단어들

### melanin

- 발음: /ˈmɛlənən/
- 구성: Greek `melan-` = black, dark + `-in`
- 뜻: 멜라닌, 어두운 색소
- 어원적 연결: 피부·머리카락·눈의 어두운 색을 만드는 색소
- 사용 맥락: `melanin production` = 멜라닌 생성

### melanoma

- 발음: /ˌmɛləˈnoʊmə/
- 구성: Greek `melan-` = black, dark + `-oma` = tumor
- 뜻: 흑색종
- 어원적 연결: 어두운 색소 세포와 관련된 종양
- 사용 맥락: 건강·의학 기사에서 자주 보임

### melanocyte

- 구성: `melan-` = black, dark + `cyte` = cell
- 뜻: 멜라닌 세포
- 어원적 연결: 멜라닌을 만드는 세포
- 사용 맥락: 생물·의학 맥락에서 쓰임

### melanism

- 구성: `melan-` = black, dark + `-ism`
- 뜻: 흑색증, 검은 색소가 많은 상태
- 어원적 연결: 몸빛이 보통보다 검게 나타나는 현상
- 사용 맥락: 동물의 색 변이를 설명할 때 쓰임

### melanistic

- 구성: `melanism` + `-ic`
- 뜻: 흑색증의, 검은 색소가 많은
- 어원적 연결: 어두운 색소가 두드러지는 성질
- 사용 맥락: `a melanistic leopard` = 검은 표범

### melancholy

- 발음: /ˈmɛləˌnkɑli/
- 구성: Greek `melas` = black + `chole` = bile
- 뜻: 우울, 우울한 기분
- 어원적 연결: 고대 의학에서 "검은 담즙"이 우울한 기질을 만든다고 본 데서 출발
- 사용 맥락: 문학적·일상적 표현으로 `a melancholy mood`

### melancholic

- 발음: /ˌmɛləˈnkɑlɪk/
- 구성: `melancholy` + `-ic`
- 뜻: 우울한, 우울증적인
- 어원적 연결: 검은 담즙이라는 오래된 의학 관념에서 우울한 성향으로 확장
- 사용 맥락: `a melancholic song` = 우울한 노래

## 기억 포인트

`melan` 계열은 "검은색, 어두운 색" 이미지.

- `melanin`: 어두운 색소
- `melanoma`: 어두운 색소 세포 관련 종양
- `melanocyte`: 멜라닌을 만드는 세포
- `melancholy`: 검은 담즙 → 우울

## 비교 단어

### alb

- 뜻: 흰색
- 구분 포인트: [[alb]]는 흰색·흰 바탕. `melan`은 검은색·어두운 색소.

### chrom

- 뜻: 색
- 구분 포인트: [[chrom]]은 색 전체. `melan`은 그중 검은색·어두운 색.
