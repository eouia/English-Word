# penetr

- 출발 단어: `penetrate`
- 핵심 어원: 라틴어 `penetrare` / `penitus` = 안으로 들어가다, 깊숙이
- 핵심 이미지: 겉을 뚫고 안쪽 깊숙이 들어감
- 관련해서 헷갈릴 수 있는 어근: [[pend]] = 매달다, 지불하다 계열, [[press]] = 누르다 계열

## 출발 단어 분석

`penetrate`는 Latin `penetrare`에서 온 말.

- `penitus`: within, inward, deep inside
- `penetrare`: 안으로 들어가다, 뚫고 들어가다

따라서 `penetrate`는 "안쪽 깊숙이 들어가다" → "관통하다, 침투하다, 이해하다"의 뜻.

## 단어들

### penetrate

- 발음: /ˈpɛnəˌtreɪt/
- 구성: Latin `penetrare`
- 뜻: 관통하다, 침투하다, 꿰뚫어 이해하다
- 어원적 연결: 표면을 지나 안쪽 깊이 들어감
- 사용 맥락: `penetrate the market` = 시장에 침투하다

### penetration

- 발음: /ˌpɛnəˈtreɪʃən/
- 구성: `penetrate` + `-ion`
- 뜻: 침투, 관통, 이해력
- 어원적 연결: 안으로 뚫고 들어가는 행위나 정도
- 사용 맥락: `market penetration` = 시장 침투율

### penetrative

- 구성: `penetrate` + `-ive`
- 뜻: 관통하는, 침투성의
- 어원적 연결: 안쪽으로 깊이 들어가는 성질
- 사용 맥락: `penetrative force` = 관통력

### impenetrable

- 발음: /ˌɪˈmpɛnətrəbəl/
- 구성: `in-` = not + `penetrable`
- 뜻: 뚫고 들어갈 수 없는, 이해하기 어려운
- 어원적 연결: 안으로 들어갈 수 없음
- 사용 맥락: `an impenetrable forest` = 뚫고 들어가기 어려운 숲

### penetrating

- 발음: /ˈpɛnəˌtreɪtɪŋ/
- 구성: `penetrate` + `-ing`
- 뜻: 날카로운, 꿰뚫는, 통찰력 있는
- 어원적 연결: 겉을 넘어 깊이 들어감
- 사용 맥락: `a penetrating question` = 핵심을 찌르는 질문

## 기억 포인트

`penetr` 계열은 "안쪽 깊이 들어감" 이미지.

- `penetrate`: 겉을 뚫고 들어감
- `penetration`: 침투나 관통
- `impenetrable`: 뚫고 들어갈 수 없음
- `penetrating`: 깊이 찌르는, 통찰력 있는

## 비교 단어

### pend

- 뜻: 매달다, 무게를 달다, 지불하다
- 구분 포인트: [[pend]]의 `pen-`처럼 보이지만 관련 없음. `suspend`, `spend`, `perpendicular`는 `pendere` 계열

### press

- 뜻: 누르다
- 구분 포인트: [[press]]는 밖에서 힘을 주어 누름. `penetrate`는 표면을 지나 안쪽으로 들어감

### penal / penitent

- 뜻: 형벌의 / 뉘우치는
- 구분 포인트: `penalty`, `penitent`, `repent`는 벌이나 뉘우침 쪽 계열. `penetrate`의 `penetrare`와는 별도
