# imper

- 출발 단어: `empire`
- 핵심 어원: Latin `imperare` = 명령하다, 지배하다
- 중간 형태: Latin `imperium` = 명령권, 통치권, 지배 영역
- 핵심 의미: 명령하고 다스리는 권위
- 관련 문서: [[in]], [[crat]], [[govern]], [[arch]]
- 주의: `imper-`는 형태상 `in-` + `parare`에서 왔지만, 현대 학습에서는 “명령권·제국” 계열로 따로 기억.

## 출발 단어 분석

`empire` ← Old French `empire` ← Latin `imperium`

- `imperium`: 명령권, 통치권, 지배권
- `imperare`: 명령하다, 지배하다
- 원래 이미지: 권위를 가지고 명령을 내림
- 현대 의미: 제국, 거대한 지배 체계

## 단어들

### empire

- 구성: Latin `imperium` = command, authority
- 뜻: 제국, 대규모 지배권
- 어원적 연결: 한 권력이 명령권을 행사하는 영역
- 사용 맥락: `the Roman Empire` = 로마 제국

### emperor

- 구성: Latin `imperator` = commander
- 뜻: 황제
- 어원적 연결: 원래는 명령을 내리는 지휘관, 뒤에 제국의 군주
- 사용 맥락: `the emperor of Japan` = 일본 천황

### empress

- 구성: `emperor` 계열 여성형
- 뜻: 여황제, 황후
- 어원적 연결: 제국의 최고 통치권과 연결됨
- 사용 맥락: `Empress Dowager` = 황태후

### imperial

- 구성: Latin `imperialis` ← `imperium`
- 뜻: 제국의, 황제의, 제국주의적인
- 어원적 연결: 명령권과 통치권을 가진 제국에 속함
- 사용 맥락: `imperial power` = 제국 권력

### imperialism

- 구성: `imperial` + `-ism`
- 뜻: 제국주의
- 어원적 연결: 한 나라가 다른 지역에 통치권과 영향력을 행사함
- 사용 맥락: `European imperialism` = 유럽 제국주의

### imperialist

- 구성: `imperial` + `-ist`
- 뜻: 제국주의자, 제국주의의
- 어원적 연결: 제국적 지배 확장을 지지하거나 실행하는 사람·정책
- 사용 맥락: `imperialist policy` = 제국주의 정책

### imperative

- 구성: Latin `imperativus` ← `imperare`
- 뜻: 필수적인, 명령적인; 명령법
- 어원적 연결: 명령처럼 반드시 해야 하는 것
- 사용 맥락: `an imperative task` = 반드시 해야 할 일

### imperious

- 구성: Latin `imperiosus` ← `imperium`
- 뜻: 거만하게 명령하는, 고압적인
- 어원적 연결: 명령권을 가진 사람처럼 행동함
- 사용 맥락: `an imperious tone` = 고압적인 말투

### imperiousness

- 구성: `imperious` + `-ness`
- 뜻: 고압적 태도, 거만함
- 어원적 연결: 남에게 명령하려는 태도
- 사용 맥락: `his imperiousness annoyed them` = 그의 고압적 태도가 그들을 짜증나게 함

### imperium

- 구성: Latin `imperium`
- 뜻: 통치권, 지배권, 제국적 권한
- 어원적 연결: 로마식 명령권·군사 지휘권 이미지
- 사용 맥락: `Roman imperium` = 로마의 통치권

### imperator

- 구성: Latin `imperator` = commander
- 뜻: 최고 지휘관, 황제 칭호
- 어원적 연결: 명령을 내리는 사람
- 사용 맥락: 역사·로마사 맥락에서 사용

## 기억 포인트

- `imper-`의 핵심은 “명령권”.
- `empire`: 명령권이 미치는 영역.
- `emperor`: 명령권을 가진 최고 통치자.
- `imperative`: 명령처럼 반드시 해야 하는 것.
- `imperious`: 황제처럼 명령하는 태도.

## 비교 단어

- [[govern]]은 방향을 잡아 조종하고 운영함.
- `imper-`는 명령권과 지배 권위가 더 강함.

### crat

- [[crat]]는 권력, 지배 체제.
- `imper-`는 제국·황제·명령권 쪽의 역사적 색이 강함.

### in

- [[in]]의 `im-` = 안으로 / 아니다와 형태가 겹침.
- `imperare`는 역사적으로 `in-` + `parare`에서 왔지만, 현대 단어 `empire`, `imperial`, `imperative`는 별도 계열로 묶는 편이 효율적.
