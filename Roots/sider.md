# sider

- 출발 단어: `consider`
- 핵심 어원: Latin `sidus` / `sideris` = 별, 별자리, 천체
- 핵심 의미: 별을 살피다, 징조를 보다 → 깊이 생각하다 / 바라다
- 관련 문서: [[con]], [[spect]], [[vid]]
- 주의: `asteroid`, `astronomy`, `disaster`의 `astr-/aster-`는 별을 뜻하지만 다른 어근. `sider-`와 같은 문서로 섞지 않음.

## 출발 단어 분석

`consider` ← Latin `considerare` = 자세히 보다, 관찰하다

- `con-`: 함께, 충분히, 집중적으로
- `sidus/sideris`: 별, 별자리
- 전통적 설명: 별의 움직임을 살펴 징조를 읽음
- 현대 의미: 깊이 생각하다, 고려하다

## 단어들

### consider

- 구성: `con-` + `sider` = star
- 뜻: 고려하다, 숙고하다
- 어원적 연결: 별을 주의 깊게 살피듯 신중히 들여다봄
- 사용 맥락: `consider the evidence` = 증거를 검토하다

### consideration

- 구성: `consider` + `-ation`
- 뜻: 고려, 숙고; 배려
- 어원적 연결: 어떤 문제를 깊이 살피는 행위
- 사용 맥락: `take into consideration` = 고려에 넣다

### considerable

- 구성: `consider` + `-able`
- 뜻: 상당한, 꽤 많은
- 어원적 연결: 고려할 만할 정도로 큼
- 사용 맥락: `a considerable amount` = 상당한 양

### considerably

- 구성: `considerable` + `-ly`
- 뜻: 상당히
- 어원적 연결: 고려할 만큼 눈에 띄게
- 사용 맥락: `improve considerably` = 상당히 개선되다

### considerate

- 구성: `consider` + `-ate`
- 뜻: 사려 깊은, 배려심 있는
- 어원적 연결: 남의 상황을 살펴 생각함
- 사용 맥락: `a considerate person` = 사려 깊은 사람

### reconsider

- 구성: `re-` = again + `consider`
- 뜻: 재고하다, 다시 생각하다
- 어원적 연결: 다시 한 번 주의 깊게 살핌
- 사용 맥락: `reconsider the decision` = 결정을 재고하다

### desire

- 구성: Latin `desiderare` 계열
- 뜻: 바라다, 욕망하다; 욕구
- 어원적 연결: 전통적으로 “별이 가져올 것을 기다림”이라는 설명과 연결됨
- 사용 맥락: `a desire for freedom` = 자유에 대한 열망

### desirable

- 구성: `desire` + `-able`
- 뜻: 바람직한, 갖고 싶은
- 어원적 연결: 바랄 만한 것
- 사용 맥락: `a desirable outcome` = 바람직한 결과

### undesirable

- 구성: `un-` + `desirable`
- 뜻: 바람직하지 않은
- 어원적 연결: 바라거나 원할 만하지 않음
- 사용 맥락: `undesirable effects` = 바람직하지 않은 효과

### desiderate

- 구성: Latin `desiderare`
- 뜻: 간절히 바라다, 필요로 하다
- 어원적 연결: `desire`의 더 학술적·격식 있는 형태
- 사용 맥락: 드문 격식어

### desideratum

- 구성: Latin `desideratum` = thing desired
- 뜻: 필요한 것, 바라는 것
- 어원적 연결: 바람의 대상
- 사용 맥락: `a key desideratum` = 핵심 요구 사항

### sidereal

- 구성: Latin `sidereus` = starry
- 뜻: 별의, 항성의
- 어원적 연결: 별과 직접 관련됨
- 사용 맥락: `sidereal time` = 항성시

### sideral

- 구성: `sidereal`의 변형
- 뜻: 별의, 항성의
- 어원적 연결: `sidereal`과 같은 계열
- 사용 맥락: 드문 변형 철자

## 기억 포인트

- `consider`: 별을 살피듯 신중히 봄 → 고려하다.
- `considerate`: 남의 처지를 고려하는 → 사려 깊은.
- `considerable`: 고려할 만큼 큰 → 상당한.
- `desire`: 별이 줄 것을 기다림 → 바라다.
- `sidereal`: 별과 직접 관련된.

## 비교 단어

### spect

- [[spect]]는 보다, 살피다의 일반 어근.
- `consider`는 별을 관찰한다는 특수한 역사 이미지에서 “숙고하다”로 이동.

### vid

- [[vid]]는 보다, 알아보다, 판단하다.
- `consider`는 눈으로 보는 것보다 마음으로 신중히 검토하는 쪽.

### astr / aster

- `asteroid`, `astronomy`, `astral`은 Greek `aster/astron` = star 계열.
- `disaster`도 이쪽 별 어근에서 온 단어로 보는 것이 보통.
- `sider`와 의미는 가깝지만 어원이 다르므로 별도 문서 후보.
