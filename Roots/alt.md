# alt

- 출발 단어: `enhance`
- 핵심 어원: Latin `altus` = high, tall; deep
- 핵심 이미지: 높이 올림, 높은 위치, 자라서 커짐
- 관련 문서: [[high]], [[cresc]], [[mag]]
- 주의: [[high]]는 게르만계 기본어. `enhance`, `altitude`, `exalt`는 라틴계 `altus`로 보는 편이 좋음.

## 출발 단어 분석

`enhance` = Anglo-French `enhauncer/enhaucer` ← Vulgar Latin `*inaltiare` ← Latin `in-` + `altus`

- `in-`: into, make
- `altus`: high, tall
- 원래 이미지: 더 높게 만들다
- 현대 의미: 향상시키다, 강화하다, 돋보이게 하다

## 단어들

### enhance

- 구성: `en-` + Latin `altus` 계열
- 뜻: 향상시키다, 강화하다, 돋보이게 하다
- 어원적 연결: 가치나 정도를 더 높임
- 사용 맥락: `enhance performance` = 성능을 향상시키다

### enhanced

- 구성: `enhance` + `-ed`
- 뜻: 향상된, 강화된
- 어원적 연결: 이전보다 더 높아진 상태
- 사용 맥락: `enhanced security` = 강화된 보안

### enhancement

- 구성: `enhance` + `-ment`
- 뜻: 향상, 강화, 개선
- 어원적 연결: 높아지게 만드는 일이나 결과
- 사용 맥락: `image enhancement` = 이미지 개선

### altitude

- 구성: Latin `altitudo` ← `altus`
- 뜻: 고도, 높이
- 어원적 연결: 높은 정도
- 사용 맥락: `high altitude` = 높은 고도

### altimeter

- 구성: `alti-` = high + [[metr]] = measure
- 뜻: 고도계
- 어원적 연결: 높이를 재는 장치
- 사용 맥락: aviation 맥락

### alto

- 구성: Italian `alto` ← Latin `altus`
- 뜻: 알토, 낮은 여성 음역 또는 높은 남성 음역
- 어원적 연결: 원래 "높은" 음역
- 사용 맥락: `alto voice` = 알토 음성

### exalt

- 구성: Latin `exaltare` = raise high
- 뜻: 높이다, 찬양하다
- 어원적 연결: 위로 들어 올려 높임
- 사용 맥락: `exalt freedom` = 자유를 드높이다

### exalted

- 구성: `exalt` + `-ed`
- 뜻: 고귀한, 높은 지위의, 고양된
- 어원적 연결: 높은 자리나 평가에 올려진 상태
- 사용 맥락: `an exalted position` = 높은 지위

### haughty

- 구성: Old French `haut` = high ← Latin `altus`
- 뜻: 거만한, 오만한
- 어원적 연결: 자신을 높게 둠
- 사용 맥락: `a haughty attitude` = 거만한 태도

### hauteur

- 구성: French `hauteur` ← `haut` = high
- 뜻: 거만함, 오만한 태도
- 어원적 연결: 높은 데서 내려다보는 태도
- 사용 맥락: 문어적 표현

### adult

- 구성: Latin `adultus` ← `adolescere` = grow up
- 뜻: 성인
- 어원적 연결: 자라서 큰 사람
- 사용 맥락: `adult education` = 성인 교육

### adolescent

- 구성: Latin `adolescere` = grow up
- 뜻: 청소년
- 어원적 연결: 아직 자라고 있는 사람
- 사용 맥락: `adolescent behavior` = 청소년 행동

### alumnus

- 구성: Latin `alumnus` = foster son, pupil
- 뜻: 남성 졸업생, 동문
- 어원적 연결: 길러진 사람, 교육받은 사람
- 사용 맥락: `a university alumnus` = 대학 남성 동문

### alumni

- 구성: plural of `alumnus`
- 뜻: 졸업생들, 동문들
- 어원적 연결: 교육기관이 길러낸 사람들
- 사용 맥락: `alumni network` = 동문 네트워크

## 기억 포인트

`alt` 계열은 "높아짐/자람" 이미지.

- `enhance`: 더 높여 좋게 만듦
- `altitude`: 높이
- `altimeter`: 높이를 재는 장치
- `exalt`: 높이 들어 올려 찬양
- `haughty`: 스스로 높은 듯한 태도
- `adult`: 자라서 큰 사람
- `adolescent`: 자라고 있는 사람

## 비교 단어

### high

- 뜻: 높다, 높은
- 구분 포인트: [[high]]는 영어 기본어. `alt`는 라틴계 고급어휘에서 `altitude`, `exalt`, `enhance`처럼 나타남.

### cresc

- 뜻: 자라다, 커지다
- 구분 포인트: [[cresc]]는 성장 과정 자체. `alt`는 자라 높아진 상태나 높이는 행위에 가까움.
