# evit

- 출발 단어: `inevitable`
- 핵심 어원: Latin `evitare` = avoid, shun
- 핵심 이미지: 길에서 비켜나 피함
- 관련 문서: [[void]], [[vad]]
- 주의: `avoid`와 뜻은 가깝지만 직접 어원은 다름. `avoid`는 [[void]]에서 따로 정리.

## 출발 단어 분석

`inevitable` = `in-` + `evitable`

- `in-`: not
- `evitable`: avoidable, 피할 수 있는
- 원래 이미지: 비켜나 피할 수 없음
- 현대 의미: 불가피한, 필연적인

## 단어들

### inevitable

- 구성: Latin `inevitabilis` ← `in-` = not + `evitabilis` = avoidable
- 뜻: 불가피한, 필연적인
- 어원적 연결: 길에서 비켜나 피할 수 없는 일
- 사용 맥락: `an inevitable result` = 필연적인 결과, `inevitable change` = 불가피한 변화

### inevitably

- 구성: `inevitable` + `-ly`
- 뜻: 불가피하게, 필연적으로
- 어원적 연결: 피할 수 없는 방식으로 일어남
- 사용 맥락: `prices will inevitably rise` = 가격은 필연적으로 오를 것이다

### evitable

- 구성: Latin `evitabilis` = avoidable
- 뜻: 피할 수 있는
- 어원적 연결: 길에서 비켜나 피할 수 있음
- 사용 맥락: 드문 단어. 보통 `avoidable`이나 `preventable`을 더 많이 씀.

### inevitability

- 구성: `inevitable` + `-ity`
- 뜻: 불가피성, 필연성
- 어원적 연결: 피할 수 없는 성질이나 상태
- 사용 맥락: `the inevitability of change` = 변화의 불가피성

## 기억 포인트

`evit`는 "비켜나 피함" 이미지.

- `evitable`: 피할 수 있는
- `inevitable`: 피할 수 없는
- `inevitably`: 피할 수 없이, 필연적으로
- `inevitability`: 피할 수 없음, 필연성

## 비교 단어

### avoid

- 뜻: 피하다
- 구분 포인트: [[void]] 계열. 현대 의미는 매우 가깝지만 `avoid`는 역사적으로 "비우다, 치우다" 이미지에서 옴.

### evade

- 뜻: 빠져나가다, 회피하다
- 구분 포인트: Latin `e-` + `vadere` = go 계열. 움직여 빠져나가는 이미지.

### unavoidable

- 뜻: 피할 수 없는
- 연결 포인트: 의미상 `inevitable`과 매우 가까움. `unavoidable`은 [[void]]의 `avoid` 계열.
