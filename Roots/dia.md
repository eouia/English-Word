# dia

- 출발 단어: `diagnosis`
- 핵심 어원: Greek `dia-` = 통하여, 가로질러, 사이에, 따로
- 변형: 모음 앞에서 `di-`로 줄어 보일 수 있음
- 핵심 이미지: 사이를 지나가며 구분하거나, 전체를 관통함
- 관련 문서: [[di]], [[logos]], [[metr]], [[angl]], [[lys]]
- 주의: [[di]] = two와 구분. `dioxide`, `dilemma`, `dichotomy`는 보통 "둘" 계열.

## 출발 단어 분석

`diagnosis` = Greek `dia-` + `gignoskein`

- `dia-`: between, through, apart
- `gignoskein`: 알다
- 원래 이미지: 증상들 사이를 가르며 알아냄
- 현대 의미: 진단, 식별

## 단어들

### diagnosis

- 발음: /ˌdaɪəˈgnoʊsəs/
- 구성: `dia-` = through, apart + Greek `gnosis` = knowledge
- 뜻: 진단, 식별
- 어원적 연결: 여러 가능성 사이를 구분해 알아냄
- 사용 맥락: `make a diagnosis` = 진단을 내리다

### diagnose

- 발음: /ˌdaɪəˈgnoʊs/
- 구성: back-formation from `diagnosis`
- 뜻: 진단하다
- 어원적 연결: 증상 사이를 구분해 병을 알아냄
- 사용 맥락: `diagnose a disease` = 병을 진단하다

### diagnostic

- 발음: /ˌdaɪəˈgnɑstɪk/
- 구성: `diagnosis` 계열 + `-ic`
- 뜻: 진단의, 진단에 쓰이는
- 어원적 연결: 구분해 알아내는 데 관련됨
- 사용 맥락: `diagnostic tests` = 진단 검사

### dialogue

- 발음: /ˈdaɪəˌlɔg/
- 구성: `dia-` = through, between + [[logos]] = speech
- 뜻: 대화, 문답
- 어원적 연결: 말이 서로 사이를 지나감
- 사용 맥락: `open dialogue` = 열린 대화

### diameter

- 발음: /daɪˈæmətɚ/
- 구성: `dia-` = across + [[metr]] = measure
- 뜻: 지름
- 어원적 연결: 원을 가로질러 잰 길이
- 사용 맥락: `the diameter of a circle` = 원의 지름

### diagonal

- 발음: /daɪˈægənəl/
- 구성: `dia-` = across + Greek `gonia` = angle
- 뜻: 대각선의, 대각선
- 어원적 연결: 각에서 각으로 가로지름
- 사용 맥락: `a diagonal line` = 대각선

### diagram

- 발음: /ˈdaɪəˌgræm/
- 구성: `dia-` = across, through + Greek `graphein` = write, draw
- 뜻: 도표, 도식
- 어원적 연결: 선을 가로질러 그어 표시한 그림
- 사용 맥락: `draw a diagram` = 도표를 그리다

### dialect

- 발음: /ˈdaɪəˌlɛkt/
- 구성: Greek `dialektos` ← `dia-` + `legein` = speak
- 뜻: 방언
- 어원적 연결: 사람들 사이에서 오가는 말 방식
- 사용 맥락: `a regional dialect` = 지역 방언

### dialysis

- 발음: /daɪˈæləsəs/
- 구성: `dia-` = through, apart + [[lys]] = loosen
- 뜻: 투석, 분리
- 어원적 연결: 막을 통과해 분리하고 풀어 냄
- 사용 맥락: `kidney dialysis` = 신장 투석

### diaspora

- 발음: /daɪˈæspɚə/
- 구성: `dia-` = across + Greek `speirein` = scatter
- 뜻: 디아스포라, 흩어진 공동체
- 어원적 연결: 여러 곳으로 흩어져 퍼짐
- 사용 맥락: `the Korean diaspora` = 한인 디아스포라

### diaphanous

- 발음: /daɪˈæfənɪs/
- 구성: `dia-` = through + Greek `phainein` = show
- 뜻: 비치는, 투명한, 섬세한
- 어원적 연결: 빛이 통과해 보임
- 사용 맥락: `diaphanous fabric` = 비치는 천

### diarrhea

- 발음: /ˌdaɪɚˈiə/
- 구성: `dia-` = through + Greek `rhein` = flow
- 뜻: 설사
- 어원적 연결: 몸을 통과해 흘러나감
- 사용 맥락: 의학·일상 어휘

### diaphragm

- 발음: /ˈdaɪəˌfræm/
- 구성: `dia-` = across + Greek `phragma` = fence, partition
- 뜻: 횡격막, 조리개
- 어원적 연결: 가로질러 막아 나누는 막
- 사용 맥락: `the diaphragm muscle` = 횡격막

## 기억 포인트

`dia-`는 "사이를 지나감/가로지름/구분함" 이미지.

- `diagnosis`: 사이를 구분해 알아냄
- `dialogue`: 말이 사이를 오감
- `diameter`: 원을 가로질러 잼
- `diagonal`: 각을 가로지름
- `dialysis`: 막을 통과해 분리함
- `diaphanous`: 빛이 통과해 보임

## 비교 단어

### di

- 뜻: 둘, 두 번
- 구분 포인트: [[di]]는 Greek "two". `dioxide`, `dilemma`, `dichotomy`처럼 두 개·두 갈래 이미지.

### dis

- 뜻: 떨어져, 반대로, 부정
- 구분 포인트: Latin `dis-` 계열. `dia-`와 철자가 겹칠 수 있지만 역사 경로와 의미가 다름.
