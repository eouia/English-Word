# dem

- 출발 단어: `democracy`
- 핵심 어원: 그리스어 `demos` = people, district
- 핵심 이미지: 사람들, 민중, 지역 공동체
- 관련해서 헷갈릴 수 있는 어근: [[crat]] = 지배, 권력 계열, [[popul]] = 사람들, 대중 계열

## 출발 단어 분석

`democracy`는 `demo-` + `-cracy`.

- `demos`: people
- `kratos`: rule, power

따라서 `democracy`는 "민중의 지배" → "민주주의"의 뜻.

## 단어들

### democracy

- 구성: `demo` = people + [[crat]] = rule
- 뜻: 민주주의, 민주 정치
- 어원적 연결: 사람들이 권력을 가지는 정치 체제
- 사용 맥락: `representative democracy` = 대의 민주주의

### democratic

- 구성: `democracy` + `-ic`
- 뜻: 민주적인, 민주주의의
- 어원적 연결: 민중의 참여나 지배와 관련됨
- 사용 맥락: `democratic values` = 민주적 가치

### democrat

- 구성: `demo` + `crat`
- 뜻: 민주주의자, 민주당원
- 어원적 연결: 민중 지배를 지지하는 사람
- 사용 맥락: `a committed democrat` = 확고한 민주주의자

### democratize

- 구성: `democrat` + `-ize`
- 뜻: 민주화하다, 대중화하다
- 어원적 연결: 권한이나 접근을 사람들에게 넓힘
- 사용 맥락: `democratize education` = 교육을 대중화하다

### demographic

- 구성: `demo` = people + `graph` = write, record
- 뜻: 인구통계의, 특정 인구 집단
- 어원적 연결: 사람들을 기록하고 분류함
- 사용 맥락: `demographic changes` = 인구통계학적 변화

### demographics

- 구성: `demographic` + `-s`
- 뜻: 인구 통계, 인구 집단 특성
- 어원적 연결: 사람들의 나이, 지역, 소득 등을 기록한 정보
- 사용 맥락: `customer demographics` = 고객 인구통계

### epidemic

- 구성: `epi-` = upon, among + `demo`
- 뜻: 유행병, 급속한 확산
- 어원적 연결: 사람들 사이에 퍼짐
- 사용 맥락: `a flu epidemic` = 독감 유행

### pandemic

- 구성: `pan-` = all + `demo`
- 뜻: 전염병의 세계적 대유행
- 어원적 연결: 모든 사람들 사이에 퍼짐
- 사용 맥락: `a global pandemic` = 세계적 대유행

### endemic

- 구성: `en-` = in + `demo`
- 뜻: 풍토병의, 특정 지역에 고유한
- 어원적 연결: 한 지역 사람들 안에 자리 잡음
- 사용 맥락: `endemic disease` = 풍토병

## 기억 포인트

`dem` 계열은 "사람들 / 민중 / 지역 집단" 이미지.

- `democracy`: 사람들이 지배함
- `democratic`: 민중 참여와 관련됨
- `demographic`: 사람들을 기록하고 분류함
- `epidemic`: 사람들 사이에 퍼짐
- `pandemic`: 모든 사람들 사이에 퍼짐
- `endemic`: 특정 지역 사람들 안에 자리 잡음

## 비교 단어

### crat

- 뜻: 지배, 권력
- 구분 포인트: [[crat]]는 누가 권력을 가지는가에 초점. `dem`은 권력의 주체인 사람들에 초점

### popul

- 뜻: 사람들, 대중
- 구분 포인트: [[popul]]은 Latin `populus` 계열. `popular`, `population`, `public` 등으로 나타남
