# clin

- 출발 단어: `decline`
- 핵심 어원: Latin `clinare` = 기울다, 기대다
- 관련 형태: Greek `klinein` = 기울이다, 눕히다
- 핵심 이미지: 한쪽으로 기울거나 방향을 틀어 기대는 움직임
- 관련 문서: [[tend]], [[vers]], [[stat]]
- 주의: `incline/decline/recline`은 Latin `clinare` 갈래. `clinic/clinical`은 Greek `kline` = bed 갈래. 둘 다 "기울다, 눕다" 이미지와 관련되지만 문서 안에서 갈래를 구분.

## 출발 단어 분석

`decline` ← Latin `declinare`

- `de-`: down, away
- `clinare`: bend, lean
- 원래 이미지: 아래나 옆으로 기울어짐
- 현대 의미: 감소하다, 거절하다, 쇠퇴

## 단어들

#### Latin `clinare` 갈래

### incline

- 구성: `in-` = toward + Latin `clinare`
- 뜻: 기울다, 마음이 향하다
- 어원적 연결: 몸이나 마음이 어떤 쪽으로 기울어짐
- 사용 맥락: `be inclined to agree` = 동의하는 쪽으로 기울다

### inclination

- 구성: `incline` + `-ation`
- 뜻: 경향, 성향, 기울기
- 어원적 연결: 한쪽으로 기울어진 방향
- 사용 맥락: `a natural inclination` = 타고난 성향

### decline

- 구성: `de-` = down, away + Latin `clinare`
- 뜻: 감소하다, 쇠퇴하다, 거절하다
- 어원적 연결: 아래로 기울거나 물러남
- 사용 맥락: `sales decline` = 판매가 감소하다

### recline

- 구성: `re-` = back + Latin `clinare`
- 뜻: 기대다, 눕다
- 어원적 연결: 뒤로 기울어 기대거나 눕음
- 사용 맥락: `recline in a chair` = 의자에 기대다

### proclivity

- 구성: Latin `proclivis` = sloping forward
- 뜻: 성향, 경향
- 어원적 연결: 어떤 쪽으로 앞으로 기울어진 성질
- 사용 맥락: `a proclivity for risk` = 위험을 좋아하는 성향

#### Greek `kline` 갈래

### clinic

- 구성: Greek `kline` = bed
- 뜻: 병원, 진료소
- 어원적 연결: 병상에 누운 환자를 보는 곳
- 사용 맥락: `a health clinic` = 보건 진료소

### clinical

- 구성: `clinic` + `-al`
- 뜻: 임상의, 냉정한
- 어원적 연결: 병상 곁의 실제 진료와 관련됨
- 사용 맥락: `clinical trial` = 임상 시험

### clinician

- 구성: `clinical` + `-ian`
- 뜻: 임상의
- 어원적 연결: 실제 환자 진료를 하는 사람
- 사용 맥락: `an experienced clinician` = 경험 많은 임상의

## 기억 포인트

`clin` 문서는 두 갈래로 기억.

- `incline`: 쪽으로 기울다
- `decline`: 아래로 기울다, 줄다
- `recline`: 뒤로 기대다
- `proclivity`: 한쪽으로 기우는 성향
- `clinic`: 병상 곁의 진료. Greek `kline` 갈래

## 비교 단어

### tend

- 뜻: 뻗다, 향하다
- 구분 포인트: [[tend]]는 방향을 향해 뻗음. `clin`은 한쪽으로 기울어짐.

### vers

- 뜻: 돌다, 방향을 바꾸다
- 구분 포인트: [[vers]]는 방향 전환. `clin`은 기울어진 방향.
