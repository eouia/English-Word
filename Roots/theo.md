# theo

- 출발 단어: `theology`
- 핵심 의미: 신, 신적인 존재
- 관련 형태: Greek `theos` = god
- 관련 문서: [[logos]], [[enth]], [[mono]], [[crat]], [[theor]]

## 핵심 이미지

`theo` 계열의 바탕은 "신".

신에 대한 학문은 `theology`.
신의 존재를 믿는 관점은 `theism`.
신이 하나라는 믿음은 `monotheism`.
신이 여럿이라는 믿음은 `polytheism`.
신권이 다스리는 체제는 `theocracy`.

## 단어들

### theology

- 구성: Greek `theos` = god + `logos` = study, discourse
- 뜻: 신학
- 연결: 신과 종교적 믿음을 연구하는 학문
- 사용 맥락: `study theology` = 신학을 공부하다

### theological

- 구성: `theology` + `-ical`
- 뜻: 신학의, 신학적인
- 연결: 신과 종교 교리에 관한 논의
- 사용 맥락: `a theological debate` = 신학적 논쟁

### theologian

- 구성: `theology` + `-ian`
- 뜻: 신학자
- 연결: 신학을 연구하는 사람
- 사용 맥락: `a Christian theologian` = 기독교 신학자

### theism

- 구성: `theo` = god + `-ism`
- 뜻: 유신론
- 연결: 신의 존재를 믿는 관점
- 사용 맥락: philosophy / religion 문맥.

### theist

- 구성: `theism` + `-ist`
- 뜻: 유신론자
- 연결: 신의 존재를 믿는 사람
- 사용 맥락: `theist and atheist views` = 유신론과 무신론 관점

### atheist

- 구성: `a-` = not + `theist`
- 뜻: 무신론자
- 연결: 신의 존재를 믿지 않는 사람
- 사용 맥락: `an atheist argument` = 무신론적 주장

### atheism

- 구성: `a-` = not + `theism`
- 뜻: 무신론
- 연결: 신의 존재를 부정하거나 믿지 않는 입장
- 사용 맥락: philosophy / religion 문맥.

### monotheism

- 구성: `mono-` = one + `theism`
- 뜻: 일신론
- 연결: 신은 하나라는 믿음
- 사용 맥락: `Judaism, Christianity, and Islam are monotheistic religions.`

### polytheism

- 구성: `poly-` = many + `theism`
- 뜻: 다신론
- 연결: 여러 신의 존재를 믿음
- 사용 맥락: ancient religion 문맥에서 자주 보임.

### theocracy

- 구성: `theo` = god + `-cracy` = rule
- 뜻: 신정 정치, 신정국가
- 연결: 신이나 종교 권위가 다스리는 체제
- 사용 맥락: `a theocracy` = 신정국가

## 비교 단어

`theory`, `theater`, `theorem`은 [[theor]] 계열.
Greek `theoria/theorein` = 바라보다, 관찰하다에서 온 말로, `theos` = 신과 구분.

`logos`는 [[logos]] 문서처럼 말, 설명, 학문 계열.
`theology`는 `theo` + `logos`라 "신에 관한 학문"으로 이해하면 좋음.
