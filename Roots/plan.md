# plan

- 출발 단어: `explain`
- 핵심 의미: 평평한, 펼쳐 보이다, 분명하게 하다
- 관련 형태: Latin `planus` = flat, level, clear
- 관련 문서: [[ex]], [[clar]]

## 핵심 이미지

`plan` 계열의 바탕은 "울퉁불퉁하거나 접힌 것을 평평하게 펼침".

평평하면 `plain`, `plane`.
생각을 펼쳐 보이면 `explain`.
펼쳐 놓은 도면에서 앞으로 할 일을 정하면 `plan`.
평면과 관련되면 `planar`.

## 단어들

### explain

- 발음: /ɪˈkspleɪn/
- 구성: Latin `ex-` = out + `planus` = flat, clear
- 뜻: 설명하다
- 연결: 접혀 있거나 복잡한 것을 밖으로 펼쳐 평평하고 분명하게 보여 줌
- 사용 맥락: `explain the rule` = 규칙을 설명하다

### explanation

- 발음: /ˌɛkspləˈneɪʃən/
- 구성: `explain` + `-ation`
- 뜻: 설명, 해명
- 연결: 복잡한 내용을 펼쳐 이해할 수 있게 만든 것
- 사용 맥락: `a clear explanation` = 분명한 설명

### explanatory

- 발음: /ɪˈksplænəˌtɔri/
- 구성: `explain` + `-atory`
- 뜻: 설명하는, 설명을 위한
- 연결: 이해되도록 펼쳐 보여 주는 기능
- 사용 맥락: `explanatory notes` = 설명 주석

### plain

- 발음: /ˈpleɪn/
- 구성: Latin `planus` = flat, clear
- 뜻: 평평한, 분명한, 수수한; 평야
- 연결: 굴곡이나 장식이 없어 눈에 바로 드러남
- 사용 맥락: `plain language` = 쉬운 말, `a plain white shirt` = 무늬 없는 흰 셔츠

### plane

- 발음: /ˈpleɪn/
- 구성: Latin `planum` = flat surface
- 뜻: 평면; 비행기
- 연결: 원래는 평평한 면. 비행기는 `airplane`에서 온 별도 현대 확장.
- 사용 맥락: `a horizontal plane` = 수평면

### plan

- 발음: /ˈplæn/
- 구성: French `plan` = ground plan, map, from Latin `planus`
- 뜻: 계획, 도면; 계획하다
- 연결: 평면에 펼쳐 놓은 도면에서 앞으로 할 일을 정함
- 사용 맥락: `make a plan` = 계획을 세우다

### planner

- 발음: /ˈplænɚ/
- 구성: `plan` + `-er`
- 뜻: 계획자, 플래너
- 연결: 일을 미리 펼쳐 놓고 정리하는 사람이나 도구
- 사용 맥락: `an urban planner` = 도시 계획가

### planar

- 발음: /ˈpleɪnɚ/
- 구성: `plane` + `-ar`
- 뜻: 평면의
- 연결: 평평한 면과 관련됨
- 사용 맥락: geometry, design, engineering 문맥.

### plateau

- 발음: /plæˈtoʊ/
- 구성: Old French `plateau`, from `plat` = flat
- 뜻: 고원; 정체기
- 연결: 높은 곳에 있는 평평한 땅에서, 변화가 멈춘 평평한 상태로 확장
- 사용 맥락: `reach a plateau` = 정체기에 이르다

## 비교 단어

`clarify`, `clear`는 [[clar]] 계열로 "밝게 하다, 분명하게 하다"가 중심.
`explain`은 접힌 것을 펼쳐 평평하게 보이게 한다는 이미지가 강함.

`planet`은 Greek `planetes` = wanderer 계열.
철자는 비슷하지만 `planus` = flat 계열의 `plan/plane/plain/explain`과 구분.
