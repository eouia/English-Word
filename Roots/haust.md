# haust

- 출발 단어: `exhaustive`
- 핵심 어원: Latin `haurire` = draw up, drain, drink
- 핵심 이미지: 물을 퍼내듯 끝까지 끌어내어 비움
- 관련 문서: [[ex]]
- 주의: 현대 영어에서 `haust` 단독 형태는 거의 쓰이지 않음. 주로 `exhaust` 계열로 학습.

## 출발 단어 분석

`exhaustive` = `exhaust` + `-ive`

- `exhaust`: 완전히 끌어내다, 다 써 버리다
- `-ive`: 성질을 나타내는 접미사
- 원래 이미지: 남김없이 퍼내어 비우는 성질
- 현대 의미: 철저한, 빠짐없는

## 단어들

### exhaustive

- 발음: /ɪˈgzɔstɪv/
- 구성: `exhaust` + `-ive`
- 뜻: 철저한, 빠짐없는, 포괄적인
- 어원적 연결: 가능한 항목을 끝까지 다 끌어내어 남기지 않음
- 사용 맥락: `an exhaustive search` = 빠짐없는 철저한 조사

### exhaust

- 발음: /ɪˈgzɔst/
- 구성: Latin `exhaurire` ← `ex-` = out + `haurire` = draw up, drain
- 뜻: 다 써 버리다, 소진시키다; 배기가스
- 어원적 연결: 안에 든 것을 밖으로 끝까지 퍼내어 비움
- 사용 맥락: `exhaust resources` = 자원을 소진하다, `car exhaust` = 자동차 배기가스

### exhausted

- 발음: /ɪˈgzɔstɪd/
- 구성: `exhaust` + `-ed`
- 뜻: 지친, 기진맥진한; 다 써 버린
- 어원적 연결: 힘이나 자원이 모두 빠져나가 비워진 상태
- 사용 맥락: `feel exhausted` = 완전히 지치다

### exhaustion

- 발음: /ɪˈgzɔstʃən/
- 구성: `exhaust` + `-ion`
- 뜻: 탈진, 고갈, 소진
- 어원적 연결: 힘·자원·가능성이 끝까지 빠져나간 상태
- 사용 맥락: `heat exhaustion` = 열탈진, `resource exhaustion` = 자원 고갈

### inexhaustible

- 발음: /ˌɪnɪˈgzɔstəbəl/
- 구성: `in-` = not + `exhaust` + `-ible`
- 뜻: 고갈되지 않는, 무궁무진한
- 어원적 연결: 아무리 퍼내도 끝까지 비워지지 않음
- 사용 맥락: `inexhaustible energy` = 지치지 않는 에너지

### exhaustively

- 발음: /ɪˈgzɑstɪvli/
- 구성: `exhaustive` + `-ly`
- 뜻: 철저하게, 빠짐없이
- 어원적 연결: 가능한 것을 남김없이 다 끌어내는 방식
- 사용 맥락: `study the issue exhaustively` = 그 문제를 철저히 연구하다

## 기억 포인트

`haust`는 "퍼내어 비움" 이미지.

- `exhaust`: 다 퍼내어 소진함
- `exhausted`: 힘이 다 빠져나가 지침
- `exhaustion`: 탈진, 고갈
- `exhaustive`: 남김없이 다 다룸
- `inexhaustible`: 다 퍼내도 고갈되지 않음

## 비교 단어

### drain

- 뜻: 물을 빼다, 고갈시키다
- 구분 포인트: 의미는 비슷하지만 Germanic 계열 일반 동사. `exhaust`는 Latin `haurire` 계열.

### deplete

- 뜻: 고갈시키다, 크게 줄이다
- 구분 포인트: [[pl]]의 Latin `plere` = fill 계열. "채워진 것을 비워 부족하게 만듦" 이미지.
