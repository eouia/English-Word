# quer

- 출발 단어: `quarrel`
- 핵심 어원: Latin `queri` = complain, lament
- 핵심 이미지: 불평을 말로 내놓고 다툼으로 번짐
- 관련 문서: [[quest]], [[quad]]
- 주의: `quarrel` "말다툼"은 `queri` 계열. `quarrel` "석궁 화살"은 네모난 촉에서 온 [[quad]] 계열.

## 출발 단어 분석

`quarrel` ← Old French `querele` ← Latin `querela`

- `queri`: 불평하다, 하소연하다
- `querela`: 불평, 고소, 다툼의 원인
- 원래 이미지: 불만을 말로 제기함
- 현대 의미: 말다툼, 다툼; 다투다

## 단어들

### quarrel

- 발음: /ˈkwɔrəl/
- 구성: Latin `querela` = complaint
- 뜻: 말다툼, 다툼; 다투다
- 어원적 연결: 불평이나 불만을 말로 제기하다가 충돌함
- 사용 맥락: `have a quarrel with a friend` = 친구와 말다툼하다

### quarrelsome

- 발음: /ˈkwɔrəlsəm/
- 구성: `quarrel` + `-some`
- 뜻: 다투기 좋아하는, 시비를 잘 거는
- 어원적 연결: 불평과 다툼을 자주 일으키는 성향
- 사용 맥락: `a quarrelsome neighbor` = 시비를 잘 거는 이웃

### quarreling

- 발음: /ˈkwɑrəlɪŋ/
- 구성: `quarrel` + `-ing`
- 뜻: 말다툼, 다투는 중인
- 어원적 연결: 불평이나 주장이 부딪쳐 말싸움이 이어짐
- 사용 맥락: `stop quarreling` = 말다툼을 그만두다

### querulous

- 발음: /ˈkwɛrələs/
- 구성: Latin `querulus` ← `queri`
- 뜻: 불평 많은, 투덜거리는
- 어원적 연결: 불만을 계속 말로 드러냄
- 사용 맥락: `a querulous tone` = 불평 섞인 말투

### querulously

- 구성: `querulous` + `-ly`
- 뜻: 불평스럽게, 투덜거리며
- 어원적 연결: 불만을 계속 드러내는 방식
- 사용 맥락: `speak querulously` = 투덜거리듯 말하다

### querulousness

- 구성: `querulous` + `-ness`
- 뜻: 불평 많음, 투덜거림
- 어원적 연결: 작은 일에도 불만을 말로 드러내는 성질
- 사용 맥락: 문어적 표현. 성격이나 말투를 묘사할 때 사용.

## 기억 포인트

`quer` 계열은 "불평이 말다툼으로 번짐" 이미지.

- `quarrel`: 말다툼
- `quarrelsome`: 시비를 잘 거는
- `quarreling`: 말다툼
- `querulous`: 불평 많은
- `querulously`: 불평스럽게
- `querulousness`: 불평 많은 성질

## 비교 단어

### quest

- 뜻: 찾기, 탐구; 요청
- 구분 포인트: [[quest]]는 Latin `quaerere` = 찾다, 묻다 계열. `quarrel`과 철자가 닮았지만 "불평하다"의 `queri`와 다름.

### quad

- 뜻: 넷, 네모
- 구분 포인트: [[quad]] 계열의 `quarrel`은 네모난 석궁 화살 뜻. 현대 시험 어휘에서는 보통 `quarrel` = 말다툼으로 익히면 충분함.
