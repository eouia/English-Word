# satur

- 출발 단어: `saturate`
- 핵심 어원: Latin `satur` = full, sated
- 핵심 이미지: 충분히 차서 더 들어갈 여지가 없음
- 관련 문서: [[ple]]
- 주의: `satisfy`와 깊은 의미권은 가깝지만, 여기서는 `saturate/saturation`의 포화 이미지 중심으로 정리.

## 출발 단어 분석

`saturate` ← Latin `saturare` ← `satur`

- `satur`: 가득 찬, 충분히 먹은
- `saturare`: 가득 채우다, 흠뻑 적시다
- 원래 이미지: 더 이상 받을 수 없을 만큼 채움
- 현대 의미: 포화시키다, 흠뻑 적시다, 가득 채우다

## 단어들

### saturate

- 구성: Latin `saturare` ← `satur`
- 뜻: 포화시키다, 흠뻑 적시다, 가득 채우다
- 어원적 연결: 더 들어갈 여지가 없을 만큼 충분히 채움
- 사용 맥락: `saturate the market` = 시장을 포화시키다

### saturated

- 구성: `saturate` + `-ed`
- 뜻: 포화된, 흠뻑 젖은
- 어원적 연결: 이미 가득 차서 더 받아들이기 어려운 상태
- 사용 맥락: `saturated soil` = 물이 포화된 흙, `saturated fat` = 포화 지방

### saturation

- 구성: `saturate` + `-ion`
- 뜻: 포화, 침투, 채도
- 어원적 연결: 충분히 차서 더 이상 늘어나기 어려운 상태
- 사용 맥락: `oxygen saturation` = 산소 포화도, `color saturation` = 색 채도

### unsaturated

- 구성: `un-` = not + `saturated`
- 뜻: 불포화의, 포화되지 않은
- 어원적 연결: 아직 더 채워질 수 있음
- 사용 맥락: `unsaturated fat` = 불포화 지방

### satiety

- 구성: Latin `satietas` ← `satis/satur` 계열
- 뜻: 포만감, 충분히 만족한 상태
- 어원적 연결: 충분히 차서 더 원하지 않는 상태
- 사용 맥락: `a feeling of satiety` = 포만감

### insatiable

- 구성: `in-` = not + Latin `satiare` = satisfy
- 뜻: 만족할 줄 모르는, 탐욕스러운
- 어원적 연결: 아무리 채워도 충분하다고 느끼지 못함
- 사용 맥락: `insatiable curiosity` = 끝없는 호기심

## 기억 포인트

`satur` 계열은 "충분히 차서 포화됨" 이미지.

- `saturate`: 가득 채우다, 포화시키다
- `saturated`: 포화된, 흠뻑 젖은
- `saturation`: 포화, 채도
- `unsaturated`: 아직 포화되지 않은
- `satiety`: 포만감
- `insatiable`: 채워도 만족하지 못하는

## 비교 단어

### complete

- 뜻: 완성하다, 완전한
- 구분 포인트: [[ple]]는 빠진 부분을 채워 완성함. `satur`는 더 이상 들어갈 수 없을 만큼 가득 참.

### satisfy

- 뜻: 만족시키다, 충족시키다
- 구분 포인트: `satisfy`는 충분함을 채워 요구를 충족함. `saturate`는 물질·시장·색·비율이 포화되는 느낌이 강함.
