# fals

- 출발 단어: `false`
- 핵심 어원: Latin `fallere` = 속이다, 실망시키다
- 관련 형태: `fals-`, `fall-`
- 핵심 이미지: 진실처럼 보이지만 속이거나 틀림
- 관련 문서: [[ver]], [[cred]]
- 주의: `fall` = 떨어지다는 게르만계로, 이 문서의 Latin `fallere`와 다름.

## 출발 단어 분석

`false` ← Latin `falsus`

- `fallere`: deceive
- `falsus`: deceptive, mistaken
- 원래 이미지: 속이는 것
- 현대 의미: 거짓의, 틀린

## 단어들

### false

- 발음: /ˈfɔls/
- 구성: Latin `falsus`
- 뜻: 거짓의, 틀린
- 어원적 연결: 진실이 아니거나 속이는 것
- 사용 맥락: `false information` = 거짓 정보

### falsify

- 발음: /ˈfɔlsəˌfaɪ/
- 구성: `false` + `-fy`
- 뜻: 위조하다, 반증하다
- 어원적 연결: 거짓으로 만들거나 거짓임을 보임
- 사용 맥락: `falsify records` = 기록을 위조하다

### falsification

- 발음: /ˌfælsəfəˈkeɪʃən/
- 구성: `falsify` + `-ication`
- 뜻: 위조, 반증
- 어원적 연결: 거짓으로 만들거나 거짓임을 밝힘
- 사용 맥락: `data falsification` = 자료 조작

### fallacy

- 발음: /ˈfæləsi/
- 구성: Latin `fallacia` = deception
- 뜻: 오류, 궤변
- 어원적 연결: 그럴듯하지만 속이는 논리
- 사용 맥락: `a logical fallacy` = 논리적 오류

### fallacious

- 발음: /fəˈleɪʃəs/
- 구성: `fallacy` + `-ous`
- 뜻: 오류가 있는, 그릇된
- 어원적 연결: 속이는 논리나 잘못된 판단을 포함함
- 사용 맥락: `a fallacious argument` = 오류가 있는 주장

### infallible

- 발음: /ˌɪˈnfæləbəl/
- 구성: `in-` = not + Latin `fallere`
- 뜻: 절대 틀리지 않는
- 어원적 연결: 속거나 틀리지 않음
- 사용 맥락: `an infallible method` = 틀림없는 방법

## 기억 포인트

`fals/fall` 계열은 "속임과 오류" 이미지.

- `false`: 거짓의
- `falsify`: 거짓으로 만들다
- `fallacy`: 속이는 논리
- `fallacious`: 오류가 있는
- `infallible`: 틀리지 않는

## 비교 단어

### cred

- 뜻: 믿다
- 구분 포인트: [[cred]]는 믿음. `fals`는 믿음을 깨는 거짓과 오류.
