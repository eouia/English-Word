# hind

- 출발 단어: `hinder`
- 핵심 어원: Old English `hindan` / `hinder` = behind, at the back
- 핵심 이미지: 뒤쪽에 남겨 두어 앞으로 나아가지 못하게 함
- 관련 문서: [[hamper]], [[ped]], [[fore]]
- 주의: 동물 `hind` = 암사슴은 별도 어원으로 보는 것이 안전함. 여기서는 "뒤쪽, 뒤처짐" 계열만 다룸.

## 출발 단어 분석

`hinder`는 Old English `hindrian` = injure, keep back 계열.

- 핵심 형태: `hind` = behind, back
- 원래 이미지: 뒤쪽으로 밀어 두거나 앞으로 못 가게 함
- 현대 의미: 방해하다, 저해하다

## 단어들

### hinder

- 발음: /ˈhɪndɚ/
- 구성: Old English `hindrian` = keep back, injure
- 뜻: 방해하다, 저해하다, 지연시키다
- 어원적 연결: 앞으로 나아가야 할 것을 뒤쪽에 붙잡아 둠
- 사용 맥락: `hinder progress` = 진전을 방해하다, `hinder development` = 발전을 저해하다

### hindrance

- 발음: /ˈhɪndrəns/
- 구성: `hinder` + `-ance`
- 뜻: 방해, 장애물
- 어원적 연결: 앞으로 나아가는 것을 뒤로 붙잡는 것
- 사용 맥락: `a serious hindrance to growth` = 성장의 심각한 장애물

### hindering

- 발음: /ˈhɪndɚɪŋ/
- 구성: `hinder` + `-ing`
- 뜻: 방해하는, 저해하는
- 어원적 연결: 진행을 계속 뒤로 붙잡는 작용
- 사용 맥락: `hindering innovation` = 혁신을 저해하는

### hindered

- 발음: /ˈhɪndɚd/
- 구성: `hinder` + `-ed`
- 뜻: 방해받은, 저해된
- 어원적 연결: 앞으로 나아가는 과정이 붙잡힌 상태
- 사용 맥락: `growth was hindered by poor policy` = 나쁜 정책 때문에 성장이 저해되었다

### behind

- 발음: /bɪˈhaɪnd/
- 구성: `be-` + Old English `hindan` = from behind
- 뜻: 뒤에, 뒤처져
- 어원적 연결: 앞쪽이 아니라 뒤쪽에 위치함
- 사용 맥락: `fall behind schedule` = 일정에서 뒤처지다

### hindsight

- 발음: /ˈhaɪˌndsaɪt/
- 구성: `hind` = behind + `sight`
- 뜻: 뒤늦은 깨달음, 사후 판단
- 어원적 연결: 일이 지난 뒤, 뒤쪽에서 돌아보는 시야
- 사용 맥락: `in hindsight` = 돌이켜 보면

### hindmost

- 구성: `hind` + `-most`
- 뜻: 가장 뒤의, 맨 뒤쪽의
- 어원적 연결: 위치상 가장 뒤에 있음
- 사용 맥락: 문어적 표현. `the hindmost part` = 맨 뒤쪽 부분

## 기억 포인트

`hind`는 "뒤쪽에 붙잡아 둠" 이미지.

- `hinder`: 앞으로 못 가게 방해함
- `hindrance`: 앞으로 가는 길의 방해물
- `behind`: 뒤에 있음
- `hindsight`: 지나간 뒤에 보는 시야
- `hindmost`: 맨 뒤쪽

## 비교 단어

### hamper

- 뜻: 방해하다, 제약하다
- 구분 포인트: [[hamper]]는 묶거나 제약해 진행을 어렵게 하는 느낌. `hinder`는 앞으로 나아감을 뒤에서 붙잡는 일반적 방해.

### impede

- 뜻: 방해하다, 지연시키다
- 구분 포인트: [[ped]]의 Latin `pes/pedis` = foot 계열. 발을 막아 진행을 늦추는 이미지.

### obstruct

- 뜻: 막다, 방해하다
- 구분 포인트: [[struct]] 계열. 앞에 장애물을 세워 물리적·제도적으로 막는 이미지.
