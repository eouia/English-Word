# aud

- 출발 단어: `audience`
- 핵심 어원: Latin `audire` = 듣다
- 핵심 이미지: 소리를 듣고 받아들임
- 관련 문서: [[phone]], [[voc]], [[dict]]
- 주의: `audit`은 원래 "계산을 들으며 확인함"에서 회계 감사의 뜻으로 확장.

## 출발 단어 분석

`audience` ← Latin `audientia`

- `audire`: hear
- `audientia`: hearing, listening
- 원래 이미지: 듣는 사람들
- 현대 의미: 청중, 관객

## 단어들

### audience

- 구성: Latin `audire` + `-ence`
- 뜻: 청중, 관객
- 어원적 연결: 듣는 사람들의 모임
- 사용 맥락: `a large audience` = 많은 청중

### audible

- 구성: `audire` + `-ible`
- 뜻: 들을 수 있는
- 어원적 연결: 귀로 들을 수 있음
- 사용 맥락: `barely audible` = 겨우 들리는

### audio

- 구성: Latin `audire`
- 뜻: 음향, 오디오
- 어원적 연결: 듣는 소리와 관련됨
- 사용 맥락: `audio quality` = 음질

### audition

- 구성: Latin `auditio` = hearing
- 뜻: 오디션, 청각
- 어원적 연결: 원래는 들어 보는 행위. 현대에는 실력을 들어 보고 평가함
- 사용 맥락: `hold an audition` = 오디션을 열다

### auditorium

- 구성: Latin `auditorium`
- 뜻: 강당, 객석
- 어원적 연결: 사람들이 듣기 위해 모이는 장소
- 사용 맥락: `school auditorium` = 학교 강당

### auditory

- 구성: `audire` + `-ory`
- 뜻: 청각의
- 어원적 연결: 듣는 감각과 관련됨
- 사용 맥락: `auditory processing` = 청각 처리

### audit

- 구성: Latin `audire`
- 뜻: 감사하다, 회계 감사
- 어원적 연결: 장부 계산을 소리 내어 읽는 것을 듣고 확인함
- 사용 맥락: `conduct an audit` = 감사를 실시하다

### auditor

- 구성: `audit` + `-or`
- 뜻: 감사관, 청강생
- 어원적 연결: 듣고 확인하는 사람
- 사용 맥락: `an external auditor` = 외부 감사인

## 기억 포인트

`aud` 계열은 "듣다" 이미지.

- `audience`: 듣는 사람들
- `audible`: 들을 수 있음
- `audio`: 들리는 소리
- `audition`: 들어 보고 평가함
- `audit`: 듣고 확인함

- 구분 포인트: [[phone]]은 소리 자체. `aud`는 소리를 듣는 행위.
