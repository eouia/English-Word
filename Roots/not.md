# not

- 출발 단어: `annotation`
- 핵심 의미: 표시하다, 알아보게 적어 두다
- 관련 형태: Latin `nota` = mark, sign, note; `notare` = mark, note
- 관련 문서: [[sign]], [[indic]]

## 핵심 이미지

`not` 계열의 바탕은 "눈에 띄게 표시를 남김".

짧게 적어 둔 표시는 `note`.
덧붙여 적은 설명은 `annotation`.
기호 체계로 적는 방식은 `notation`.
표시해서 알게 하면 `notice`, `notify`.
어떤 뜻을 표시해 나타내면 `denote`.
알아차려 마음속 생각으로 잡으면 `notion`, `notional`.

## 단어들

### annotation

- 구성: Latin `ad-` = to + `notare` = mark, note + `-ion`
- 뜻: 주석, 주해, 짧은 설명
- 연결: 본문 옆에 덧붙여 표시하고 적어 둔 설명
- 사용 맥락: `add annotations to a text` = 글에 주석을 달다

### annotate

- 구성: `ad-` + `notare`
- 뜻: 주석을 달다, 설명을 덧붙이다
- 연결: 읽는 사람이 알아보도록 표시와 설명을 붙임
- 사용 맥락: `annotate an article` = 기사에 주석을 달다

### note

- 구성: Latin `nota` = mark, sign
- 뜻: 메모, 주석, 음표; 적어 두다, 주목하다
- 연결: 기억하거나 알아보게 남긴 표시
- 사용 맥락: `take notes` = 필기하다

### notable

- 구성: `note` + `-able`
- 뜻: 주목할 만한, 두드러진
- 연결: 표시해 둘 만큼 눈에 띔
- 사용 맥락: `a notable achievement` = 주목할 만한 성취

### notice

- 구성: Latin `notitia` = being known, information
- 뜻: 알아차림, 통지, 공고; 알아차리다
- 연결: 표시가 눈에 들어와 알게 됨
- 사용 맥락: `short notice` = 촉박한 통보

### noticeable

- 구성: `notice` + `-able`
- 뜻: 눈에 띄는, 알아차릴 수 있는
- 연결: 표시나 차이가 충분히 드러남
- 사용 맥락: `a noticeable difference` = 눈에 띄는 차이

### notion

- 구성: Latin `notio` = idea, conception, becoming known
- 뜻: 개념, 생각, 관념
- 연결: 알아차린 것을 마음속 표시처럼 잡아 둔 생각
- 사용 맥락: `a basic notion of fairness` = 공정성에 대한 기본 개념

### notional

- 구성: `notion` + `-al`
- 뜻: 개념상의, 관념적인, 명목상의
- 연결: 실제 물체나 실행보다 마음속 개념으로 잡힌 것
- 사용 맥락: `a notional value` = 명목 가치

### notify

- 구성: Latin `notificare` = make known
- 뜻: 알리다, 통지하다
- 연결: 상대가 알도록 표시하거나 소식을 전함
- 사용 맥락: `notify customers of a change` = 고객에게 변경 사항을 알리다

### notification

- 구성: `notify` + `-ication`
- 뜻: 알림, 통지
- 연결: 알게 해 주는 표시나 메시지
- 사용 맥락: `turn off notifications` = 알림을 끄다

### notation

- 구성: `note` + `-ation`
- 뜻: 표기법, 기호 체계
- 연결: 정보를 표시로 적는 방식
- 사용 맥락: `musical notation` = 악보 표기법

### denote

- 구성: Latin `de-` = down/out + `notare` = mark
- 뜻: 나타내다, 표시하다, 뜻하다
- 연결: 표시가 특정 의미를 가리킴
- 사용 맥락: `X denotes an unknown value.` = X는 미지의 값을 나타낸다

### connotation

- 구성: `con-` = together + `notare` + `-ion`
- 뜻: 함축, 부가적 의미
- 연결: 기본 표시와 함께 따라붙는 느낌이나 의미
- 사용 맥락: `negative connotations` = 부정적 함축

## 비교 단어

`sign`은 [[sign]] 계열로 "표시 자체, 신호, 서명" 이미지가 강함.
`not`은 그 표시를 "알아보게 적어 두다, 주석으로 남기다" 쪽에 가까움.

`index`는 [[indic]] 계열로 "가리켜 알려 주는 표시" 중심.
`annotation`은 본문에 덧붙은 설명 표시라는 점에서 구분.
