# scrib

- 출발 단어: `ascribe`
- 핵심 어원: 라틴어 `scribere` / `script-` = 쓰다
- 핵심 이미지: 글로 적어 붙이거나, 적어서 정함
- 관련해서 헷갈릴 수 있는 어근: [[trib]]의 `attribute` = 원인이나 성질을 배정함

## 출발 단어 분석

`ascribe`는 `ad-` + `scribere`.

- `ad-`: to, toward
- `scribere`: 쓰다

따라서 `ascribe`는 "어떤 대상 쪽에 써 붙이다" → "원인, 공로, 성질을 ~의 탓/덕으로 돌리다"의 뜻.

## 단어들

### ascribe

- 구성: `ad-` = to + `scribe`
- 뜻: ~의 탓으로 돌리다, ~에 속한다고 보다
- 어원적 연결: 원인이나 성질을 어떤 대상에게 써 붙임
- 사용 맥락: `ascribe success to luck` = 성공을 운 덕분으로 돌리다

### ascription

- 구성: `ascribe` + `-ion`
- 뜻: 귀속, 돌림, 부여
- 어원적 연결: 어떤 원인이나 성질을 대상에게 붙여 적는 행위
- 사용 맥락: `the ascription of blame` = 책임의 귀속

### describe

- 구성: `de-` = down, fully + `scribe`
- 뜻: 묘사하다, 설명하다
- 어원적 연결: 어떤 대상을 글로 써 내려가며 드러냄
- 사용 맥락: `describe the scene` = 장면을 묘사하다

### description

- 구성: `describe` + `-ion`
- 뜻: 묘사, 설명
- 어원적 연결: 글이나 말로 적어 드러낸 내용
- 사용 맥락: `a detailed description` = 자세한 설명

### prescribe

- 구성: `pre-` = before + `scribe`
- 뜻: 처방하다, 규정하다
- 어원적 연결: 미리 써서 정해 줌
- 사용 맥락: `prescribe medicine` = 약을 처방하다

### prescription

- 구성: `prescribe` + `-ion`
- 뜻: 처방전, 처방, 규정
- 어원적 연결: 미리 적어 내려준 지시
- 사용 맥락: `a doctor's prescription` = 의사의 처방전

### inscribe

- 구성: `in-` = in, on + `scribe`
- 뜻: 새기다, 적다
- 어원적 연결: 표면 안이나 위에 글을 씀
- 사용 맥락: `inscribe a name on a ring` = 반지에 이름을 새기다

### inscription

- 구성: `inscribe` + `-ion`
- 뜻: 새겨진 글, 비문
- 어원적 연결: 표면에 적히거나 새겨진 글
- 사용 맥락: `an ancient inscription` = 고대 비문

### subscribe

- 구성: `sub-` = under + `scribe`
- 뜻: 구독하다, 동의하다
- 어원적 연결: 아래에 이름을 써서 동의하거나 가입함
- 사용 맥락: `subscribe to a magazine` = 잡지를 구독하다

### subscription

- 구성: `subscribe` + `-ion`
- 뜻: 구독, 가입
- 어원적 연결: 이름을 써 넣고 계속 받는 계약
- 사용 맥락: `a monthly subscription` = 월간 구독

### transcribe

- 구성: `trans-` = across + `scribe`
- 뜻: 베껴 쓰다, 필사하다, 전사하다
- 어원적 연결: 한 곳의 말을 다른 형태로 옮겨 적음
- 사용 맥락: `transcribe an interview` = 인터뷰를 전사하다

### transcript

- 구성: `trans-` + `script`
- 뜻: 성적표, 기록문, 전사본
- 어원적 연결: 옮겨 적은 공식 기록
- 사용 맥락: `an academic transcript` = 성적 증명서

### manuscript

- 구성: Latin `manu` = by hand + `script`
- 뜻: 원고, 필사본
- 어원적 연결: 손으로 쓴 글
- 사용 맥락: `submit a manuscript` = 원고를 제출하다

### script

- 구성: Latin `scriptum`
- 뜻: 대본, 문자, 글씨체
- 어원적 연결: 쓰인 것
- 사용 맥락: `a movie script` = 영화 대본

### scripture

- 구성: Latin `scriptura`
- 뜻: 성서, 경전
- 어원적 연결: 권위 있게 쓰인 글
- 사용 맥락: `read scripture` = 성서를 읽다

### scribe

- 구성: Latin `scriba`
- 뜻: 필경사, 서기
- 어원적 연결: 글을 쓰는 사람
- 사용 맥락: `a medieval scribe` = 중세 필경사

## 기억 포인트

`scrib` 계열은 "쓰다" 이미지.

- `ascribe`: 대상에게 써 붙임
- `describe`: 써 내려가며 설명함
- `prescribe`: 미리 써서 지시함
- `inscribe`: 표면에 써 넣음
- `subscribe`: 아래에 이름을 씀
- `transcribe`: 옮겨 씀
- `manuscript`: 손으로 쓴 글
- `script`: 쓰인 것

## 비교 단어

### attribute

- 구성: `ad-` + `tribute`
- 뜻: ~의 탓으로 돌리다, 속성
- 구분 포인트: [[trib]]의 `attribute`는 원인이나 성질을 어떤 대상에게 "배정함". `ascribe`는 그 대상에게 "써 붙임" 이미지

### proscribe

- 구성: `pro-` = forth, publicly + `scribe`
- 뜻: 금지하다
- 구분 포인트: 공개적으로 써 붙여 금지하거나 추방 목록에 올리는 이미지. `prescribe` = 미리 써서 정함과 구분
