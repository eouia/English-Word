# mag-master

- 상위 문서: [[mag]]
- 핵심 어원: Latin `magister` = 우두머리, 스승, 관리자
- 핵심 이미지: 더 큰 권위와 지식을 가진 사람
- 관련 갈래: [[mag-major]], [[mag-magn]]

## 단어들

### magistrate

- 발음: /ˈmædʒəˌstreɪt/
- 구성: Latin `magistratus` ← `magister`
- 뜻: 치안판사, 행정관
- 어원적 연결: 큰 권한을 가진 공적 관리자
- 사용 맥락: `appear before a magistrate` = 치안판사 앞에 출석하다

### magistracy

- 구성: `magistrate` 계열 + `-cy`
- 뜻: 치안판사직, 행정관직, 사법 행정기관
- 어원적 연결: 공적 권위를 가진 직위나 집단
- 사용 맥락: legal / administrative context

### magisterial

- 발음: /ˌmædʒɪˈstiriəl/
- 구성: Latin `magister` + `-ial`
- 뜻: 권위 있는, 위엄 있는; 치안판사의
- 어원적 연결: 스승·관리자처럼 위에서 판단하고 지시함
- 사용 맥락: `a magisterial study` = 권위 있는 연구

### magistral

- 구성: Latin `magistralis`
- 뜻: 스승의, 권위 있는, 주된
- 어원적 연결: `magister`의 지위나 가르침과 관련됨
- 사용 맥락: 드문 격식어

### magister

- 구성: Latin `magister`
- 뜻: 스승, 주인, 관리자
- 어원적 연결: 더 큰 권위와 지식을 가진 사람
- 사용 맥락: Latin / academic context

### master

- 발음: /ˈmæstɚ/
- 구성: Old English `maegester` ← Latin `magister`
- 뜻: 주인, 대가, 석사; 숙달하다
- 어원적 연결: 더 큰 권위나 기술을 가진 사람
- 사용 맥락: `master a language` = 언어를 숙달하다

### masterpiece

- 발음: /ˈmæstɚˌpis/
- 구성: `master` + [[piece]]
- 뜻: 걸작, 대표작
- 어원적 연결: 대가의 솜씨가 드러난 작품
- 사용 맥락: `a literary masterpiece` = 문학적 걸작

### maestro

- 발음: /ˈmaɪstroʊ/
- 구성: Italian `maestro` ← Latin `magister`
- 뜻: 거장, 지휘자
- 어원적 연결: 음악이나 예술에서 스승·대가의 지위
- 사용 맥락: `a musical maestro` = 음악 거장

### mister

- 발음: /ˈmɪstɚ/
- 구성: 약화된 `master`
- 뜻: Mr., 남성 경칭
- 어원적 연결: 원래 `master`의 존칭적 형태가 약해짐
- 사용 맥락: `Mister Smith` = 스미스 씨

## 기억 포인트

`mag-master`는 "더 높은 사람" 이미지.

- `magister`: 스승, 관리자
- `magistrate`: 공식 권한을 가진 관리자
- `master`: 주인, 대가
- `maestro`: 예술의 거장
