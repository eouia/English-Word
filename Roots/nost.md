# nost

- 출발 단어: `nostalgia`
- 핵심 의미: 돌아가고 싶은 마음, 귀향의 아픔
- 관련 형태: Greek `nostos` = return home, homecoming; `algos` = pain

## 핵심 이미지

`nostalgia`의 바탕은 "집으로 돌아가고 싶은 아픔".

Greek `nostos`는 집으로 돌아감.
Greek `algos`는 고통.
둘이 합쳐져 `nostalgia` = 돌아가고 싶은 마음에서 오는 아픔.
현대 영어에서는 고향뿐 아니라 과거의 시간과 분위기를 그리워하는 감정으로 넓어짐.

## 단어들

### nostalgia

- 구성: Greek `nostos` = return home + `algos` = pain
- 뜻: 향수, 과거에 대한 그리움
- 연결: 돌아갈 수 없는 집이나 과거를 그리워하며 느끼는 아픔
- 사용 맥락: `feel nostalgia for childhood` = 어린 시절에 대한 향수를 느끼다

### nostalgic

- 구성: `nostalgia` + `-ic`
- 뜻: 향수를 불러일으키는, 그리워하는
- 연결: 과거나 고향을 떠올리게 하여 그리움을 일으킴
- 사용 맥락: `a nostalgic song` = 향수를 불러일으키는 노래

### nostalgically

- 구성: `nostalgic` + `-ally`
- 뜻: 향수에 젖어, 그리운 듯이
- 연결: 과거를 그리워하는 방식으로
- 사용 맥락: `look back nostalgically` = 향수에 젖어 돌아보다

### nostos

- 구성: Greek `nostos` = homecoming
- 뜻: 귀향, 집으로 돌아감
- 연결: `nostalgia`의 앞부분. 현대 일반 영어 단어라기보다 문학·어원 설명용.
- 사용 맥락: Greek epic 문맥에서 보임.

## 비교 단어

`algos` = pain은 `analgesic`, `neuralgia`, `algophobia` 같은 의학·학술어에 보임.
다만 이 vault에서는 아직 `alg` 문서를 만들 만큼 핵심 단어가 많지 않으므로, 우선 `nostalgia` 안에서만 설명.

`homesick`은 Germanic 계열의 쉬운 표현.
`nostalgia`는 더 문학적이고, 집뿐 아니라 과거 전체에 대한 그리움까지 포함.
