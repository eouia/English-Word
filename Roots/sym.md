# sym

- 출발 단어: `symbol`
- 핵심 접두사: Greek `syn-` / `sym-` = together, with
- 핵심 이미지: 함께, 같이, 한데 모임
- 관련 문서: [[path]], [[aesthes]]
- 관련해서 헷갈릴 수 있는 어근: [[uni]] = 하나로 묶임 계열, [[con]] = 함께 계열

## 출발 단어 분석

`sym-`은 Greek `syn-`의 동화형.

- 기본형: `syn-` = together, with
- `b`, `m`, `p` 앞: `sym-`
- `l` 앞: `syl-`
- 일부 형태: `sy-`

따라서 `symbol`은 원래 "함께 던져 맞춘 표시" → "상징"의 뜻.

## 주요 형태

- `syn-`: `synthesis`, `syntax`, `synonym`
- `sym-`: `symbol`, `sympathy`, `symphony`
- `syl-`: `syllable`
- `sy-`: `system`

## 단어들

### symbol

- 구성: `sym-` = together + Greek `ballein` = throw
- 뜻: 상징, 기호
- 어원적 연결: 둘을 함께 던져 맞춘 표시. 어떤 의미를 대신 나타내는 것
- 사용 맥락: `a symbol of peace` = 평화의 상징

### symbolic

- 구성: `symbol` + `-ic`
- 뜻: 상징적인
- 어원적 연결: 어떤 의미를 함께 담아 나타냄
- 사용 맥락: `symbolic meaning` = 상징적 의미

### sympathy

- 구성: `sym-` = with + `pathos` = feeling, suffering
- 뜻: 동정, 공감
- 어원적 연결: 감정을 함께 겪음
- 사용 맥락: `feel sympathy for victims` = 피해자들에게 동정을 느끼다

### sympathetic

- 구성: `sympathy` + `-etic`
- 뜻: 동정적인, 공감하는
- 어원적 연결: 감정을 함께함
- 사용 맥락: `a sympathetic listener` = 공감해 주는 청자

### symphony

- 구성: `sym-` = together + `phone` = sound
- 뜻: 교향곡
- 어원적 연결: 여러 소리가 함께 울림
- 사용 맥락: `Beethoven's Fifth Symphony` = 베토벤 교향곡 5번

### symptom

- 구성: `sym-` = together + Greek `piptein` = fall
- 뜻: 증상, 징후
- 어원적 연결: 병과 함께 나타나는 것
- 사용 맥락: `early symptoms` = 초기 증상

### synthesis

- 구성: `syn-` = together + `thesis` = placing
- 뜻: 종합, 합성
- 어원적 연결: 여러 요소를 함께 놓아 하나로 만듦
- 사용 맥락: `protein synthesis` = 단백질 합성

### synthetic

- 구성: `synthesis` + `-etic`
- 뜻: 합성의, 인조의
- 어원적 연결: 여러 요소를 함께 놓아 만든 것
- 사용 맥락: `synthetic materials` = 합성 소재

### syntax

- 구성: `syn-` = together + `taxis` = arrangement
- 뜻: 구문, 문장 구조
- 어원적 연결: 단어들을 함께 배열하는 규칙
- 사용 맥락: `English syntax` = 영어 구문론

### synonym

- 구성: `syn-` = together + `onym` = name
- 뜻: 동의어
- 어원적 연결: 같은 뜻을 함께 나누는 이름
- 사용 맥락: `a synonym for happy` = happy의 동의어

### synchronize

- 구성: `syn-` = together + `chronos` = time
- 뜻: 동시에 맞추다, 동기화하다
- 어원적 연결: 시간을 함께 맞춤
- 사용 맥락: `synchronize schedules` = 일정을 맞추다

### synergy

- 구성: `syn-` = together + `ergon` = work
- 뜻: 시너지, 상승효과
- 어원적 연결: 함께 일해 더 큰 효과를 냄
- 사용 맥락: `team synergy` = 팀 시너지

### syllabus

- 구성: Greek `syllabe` = taking together
- 뜻: 강의계획서, 요강
- 어원적 연결: 함께 모아 잡은 내용 목록
- 사용 맥락: `course syllabus` = 강의계획서

### syllable

- 구성: `syl-` = together + Greek `labein` = take
- 뜻: 음절
- 어원적 연결: 소리들이 함께 잡혀 한 단위가 됨
- 사용 맥락: `three syllables` = 세 음절

### system

- 구성: `sy-` = together + Greek `histanai` = stand
- 뜻: 체계, 시스템
- 어원적 연결: 함께 세워진 구조
- 사용 맥락: `a legal system` = 법 체계

### systematic

- 구성: `system` + `-atic`
- 뜻: 체계적인, 조직적인
- 어원적 연결: 전체 구조에 맞추어 질서 있게 세움
- 사용 맥락: `a systematic approach` = 체계적인 접근

### systemic

- 구성: `system` + `-ic`
- 뜻: 체계 전체의, 전신의
- 어원적 연결: 한 부분이 아니라 전체 구조에 걸쳐 있음
- 사용 맥락: `systemic problems` = 구조적 문제들

### ecosystem

- 구성: `eco-` = house, environment + `system`
- 뜻: 생태계
- 어원적 연결: 생물과 환경이 함께 세워진 상호 연결 구조
- 사용 맥락: `a forest ecosystem` = 숲 생태계

### subsystem

- 구성: `sub-` = under + `system`
- 뜻: 하위 체계
- 어원적 연결: 큰 체계 아래에 들어 있는 작은 구조
- 사용 맥락: `a computer subsystem` = 컴퓨터 하위 시스템

## 기억 포인트

`sym-` / `syn-` 계열은 "함께" 이미지.

- `symbol`: 의미를 함께 맞춘 표시
- `sympathy`: 감정을 함께 느낌
- `symphony`: 소리가 함께 울림
- `synthesis`: 요소를 함께 놓아 종합함
- `syntax`: 단어를 함께 배열함
- `synchronize`: 시간을 함께 맞춤
- `system`: 함께 세워진 구조
- `systemic`: 전체 체계에 퍼져 있음

## 비교 단어

### con-

- 뜻: 함께
- 구분 포인트: [[con]]은 Latin계 접두사. `compose`, `concur`, `connect`처럼 라틴계 단어에 많음. `sym-/syn-`은 Greek계 단어에 많음

### uni

- 뜻: 하나
- 구분 포인트: [[uni]]는 "하나" 자체. `sym-/syn-`은 여러 요소가 함께 있는 상태
