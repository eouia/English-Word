# mono

- 출발 단어: `monologue`
- 핵심 어원: Greek `monos` = 하나의, 홀로인, 유일한
- 핵심 이미지: 하나만 있음, 혼자 있음, 하나로 제한됨
- 관련 문서: [[uni]], [[singul]], [[solus]], [[poly]], [[logos]], [[bi]], [[di]], [[tri]]
- 주의: [[uni]]는 Latin `unus` 계열. `mono-`는 Greek `monos` 계열.

## 출발 단어 분석

`monologue` = Greek `mono-` + `logos`

- `mono-`: 하나의, 혼자의
- `logos`: 말, 설명
- 원래 이미지: 한 사람이 하는 말
- 현대 의미: 독백, 일인 발화

## 단어들

### monologue

- 구성: `mono-` = one + [[logos]] = speech
- 뜻: 독백, 장광설
- 어원적 연결: 한 사람이 이어 가는 말
- 사용 맥락: `an inner monologue` = 내면의 독백

### monopoly

- 구성: `mono-` = one + Greek `polein` = sell
- 뜻: 독점
- 어원적 연결: 하나의 판매자나 세력이 시장을 차지함
- 사용 맥락: `a market monopoly` = 시장 독점

### monopolize

- 구성: `monopoly` + `-ize`
- 뜻: 독점하다
- 어원적 연결: 하나의 주체가 전부 차지함
- 사용 맥락: `monopolize attention` = 관심을 독차지하다

### monotheism

- 구성: `mono-` = one + Greek `theos` = god
- 뜻: 일신론
- 어원적 연결: 하나의 신만 인정함
- 사용 맥락: `monotheistic religions` = 일신교

### monochrome

- 구성: `mono-` = one + Greek `chroma` = color
- 뜻: 단색의, 흑백의
- 어원적 연결: 하나의 색으로 이루어짐
- 사용 맥락: `a monochrome image` = 단색 이미지

### monogamy

- 구성: `mono-` = one + Greek `gamos` = marriage
- 뜻: 일부일처제
- 어원적 연결: 한 배우자와의 결혼
- 사용 맥락: `monogamy and polygamy` = 일부일처제와 다혼제

### monolith

- 구성: `mono-` = one + Greek `lithos` = stone
- 뜻: 하나의 큰 돌, 단일 거대 조직
- 어원적 연결: 한 덩어리의 돌
- 사용 맥락: `a corporate monolith` = 거대 단일 기업 조직

### monolingual

- 구성: `mono-` = one + Latin `lingua` = language
- 뜻: 한 언어만 쓰는
- 어원적 연결: 하나의 언어에 속함
- 사용 맥락: `a monolingual dictionary` = 단일 언어 사전

### monorail

- 구성: `mono-` = one + `rail`
- 뜻: 모노레일
- 어원적 연결: 하나의 레일 위를 달림
- 사용 맥락: `take the monorail` = 모노레일을 타다

## 기억 포인트

`mono-`는 "하나/홀로" 이미지.

- `monologue`: 한 사람의 말
- `monopoly`: 한 주체의 독점
- `monotheism`: 하나의 신
- `monochrome`: 하나의 색
- `monolith`: 한 덩어리 돌

## 비교 단어

### uni

- 뜻: 하나
- 구분 포인트: [[uni]]는 Latin `unus` 계열. `unity`, `unit`, `uniform`처럼 라틴계 또는 프랑스어 경로 단어에 많이 보임.

### singul / solus

- 뜻: 하나씩 따로 / 홀로
- 구분 포인트: [[singul]]은 `single`, `singular`처럼 개별 단위. [[solus]]는 `solo`, `sole`, `solitary`처럼 혼자 있음.

### poly

- 뜻: 많음, 여러 개
- 구분 포인트: [[poly]]는 `polyglot`, `polygon`, `polytheism`처럼 여러 개를 나타내는 Greek 접두사. `mono-`의 반대 축.
