# prehend

- 출발 단어: `apprehend`
- 핵심 어원: 라틴어 `prehendere` = 붙잡다, 움켜쥐다
- 핵심 이미지: 손이나 마음으로 붙잡음
- 비교: [[seize]] = 갑자기 붙잡아 장악함

## 출발 단어 분석

`apprehend`는 `ad-` + `prehendere`.

- `ad-`: toward
- `prehendere`: 붙잡다, 움켜쥐다

따라서 `apprehend`는 "붙잡다" → "체포하다" 또는 "마음으로 붙잡다" → "이해하다"의 뜻.

## 단어들

### apprehend

- 발음: /ˌæprɪˈhɛnd/
- 구성: `ad-` + `prehendere`
- 뜻: 체포하다, 이해하다
- 어원적 연결: 몸으로 붙잡거나 마음으로 붙잡음
- 사용 맥락: `apprehend a suspect` = 용의자를 체포하다

### apprehension

- 발음: /ˌæprɪˈhɛnʃən/
- 구성: `apprehend` + `-ion`
- 뜻: 이해, 불안, 체포
- 어원적 연결: 마음으로 붙잡음. 또는 두려운 가능성이 마음을 붙잡음
- 사용 맥락: `a sense of apprehension` = 불안감

### apprehensive

- 발음: /ˌæprɪˈhɛnsɪv/
- 구성: `apprehension` + `-ive`
- 뜻: 걱정하는, 불안한
- 어원적 연결: 걱정거리가 마음을 붙잡은 상태
- 사용 맥락: `feel apprehensive` = 불안해하다

### comprehend

- 발음: /ˌkɑmpriˈhɛnd/
- 구성: `com-` + `prehendere`
- 뜻: 이해하다, 포함하다
- 어원적 연결: 함께 붙잡아 전체를 파악함
- 사용 맥락: `comprehend the problem` = 문제를 이해하다

### comprehension

- 발음: /ˌkɑmpriˈhɛnʃən/
- 구성: `comprehend` + `-ion`
- 뜻: 이해, 독해력
- 어원적 연결: 전체를 붙잡아 파악하는 능력
- 사용 맥락: `reading comprehension` = 독해력

### comprehensive

- 발음: /ˌkɑmpriˈhɛnsɪv/
- 구성: `comprehend` + `-ive`
- 뜻: 포괄적인
- 어원적 연결: 여러 요소를 함께 붙잡아 포함함
- 사용 맥락: `a comprehensive plan` = 종합 계획

### comprise

- 발음: /kəˈmpraɪz/
- 구성: Old French `compris`, from `comprendre` = include, contain
- 뜻: 구성하다, 포함하다
- 어원적 연결: 여러 부분을 함께 붙잡아 하나의 전체 안에 둠
- 사용 맥락: `The committee comprises five members.` = 위원회는 다섯 명으로 구성됨

### comprised

- 발음: /kəˈmpraɪzd/
- 구성: `comprise` + `-ed`
- 뜻: 구성된, 포함된
- 어원적 연결: 여러 부분이 함께 붙잡혀 전체를 이룬 상태
- 사용 맥락: `a team comprised of experts` = 전문가들로 구성된 팀

### comprising

- 발음: /kəˈmpraɪzɪŋ/
- 구성: `comprise` + `-ing`
- 뜻: 구성하는, 포함하는
- 어원적 연결: 전체 안에 여러 요소를 함께 포함함
- 사용 맥락: `a package comprising several tools` = 여러 도구로 구성된 패키지

### prehensile

- 구성: `prehendere` + `-ile`
- 뜻: 붙잡을 수 있는
- 어원적 연결: 손, 꼬리 등이 물체를 잡을 수 있음
- 사용 맥락: `a prehensile tail` = 물체를 잡을 수 있는 꼬리

### apprentice

- 발음: /əˈprɛntəs/
- 구성: Old French `apprendre` = 배우다, 붙잡다
- 뜻: 견습생
- 어원적 연결: 기술을 마음으로 붙잡아 배우는 사람
- 사용 맥락: `an apprentice carpenter` = 목수 견습생

### reprehensible

- 발음: /ˌrɛprɪˈhɛnsəbəl/
- 구성: `re-` + `prehendere` + `-ible`
- 뜻: 비난받을 만한
- 어원적 연결: 붙잡아 되돌려 책망할 만함
- 사용 맥락: `reprehensible behavior` = 비난받을 만한 행동

### reprisal

- 발음: /riˈpraɪzəl/
- 구성: Old French `reprendre` = 되찾다, 붙잡다
- 뜻: 보복
- 어원적 연결: 당한 것을 되잡아 갚음
- 사용 맥락: `fear reprisals` = 보복을 두려워하다

### surprise

- 발음: /sɚˈpraɪz/
- 구성: `sur-` + `prise`
- 뜻: 놀라게 하다, 기습
- 어원적 연결: 위에서 갑자기 붙잡음
- 사용 맥락: `a surprise attack` = 기습 공격

### enterprise

- 발음: /ˈɛntɚˌpraɪz/
- 구성: Old French `entreprendre` = 착수하다
- 뜻: 기업, 사업, 진취성
- 어원적 연결: 어떤 일을 붙잡고 시작함
- 사용 맥락: `a business enterprise` = 기업

### entrepreneur

- 발음: /ˌɑntrəprəˈnɝ/
- 구성: French `entreprendre` = 착수하다
- 뜻: 기업가
- 어원적 연결: 일을 붙잡고 시작하는 사람
- 사용 맥락: `a successful entrepreneur` = 성공한 기업가

## 비교 단어

### seize

- 구성: Old French `saisir`
- 뜻: 붙잡다, 장악하다
- 어원적 연결: [[seize]] 계열. 갑자기 붙잡아 자기 통제 아래 둠
- 구분 포인트: `seize` = 강하게 장악함, `apprehend` = 붙잡음/이해함/체포함

### grasp

- 구성: Germanic 계열
- 뜻: 꽉 잡다, 이해하다
- 어원적 연결: `prehend`와 의미는 비슷하지만 어원 다름
- 구분 포인트: `grasp`는 일상어, `prehend` 계열은 라틴어 기반 고급어

## 기억 포인트

`prehend` 계열은 "붙잡음" 이미지.

- `apprehend`: 용의자나 의미를 붙잡음
- `apprehension`: 마음을 붙잡은 이해/불안
- `comprehend`: 함께 붙잡아 이해함
- `comprehensive`: 함께 붙잡아 포괄함
- `comprise`: 함께 붙잡아 구성하거나 포함함
- `prehensile`: 붙잡을 수 있음
- `apprentice`: 기술을 붙잡아 배우는 사람
- `reprehensible`: 붙잡아 책망할 만함
- `surprise`: 갑자기 붙잡음
- `enterprise`: 일을 붙잡고 시작함
