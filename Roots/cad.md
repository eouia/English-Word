# cad

- 출발 단어: `accidental`
- 핵심 어원: Latin `cadere` = fall, happen
- 관련 형태: `cad-`, `cid-`, `cas-`
- 핵심 이미지: 떨어지다, 우연히 일어나다, 어떤 상태로 빠지다
- 관련 문서: [[cid]]
- 주의: 이 문서의 `cid`는 "떨어지다" 계열. [[cid]] = 자르다, 베다 계열과 철자가 같아도 어원이 다름.

## 출발 단어 분석

`accidental` = `accident` + `-al`

- `accident` ← Latin `accidere`
- `ad-`: to, toward
- `cadere`: fall, happen
- 원래 이미지: 어떤 일 쪽으로 떨어져 닥침
- 현대 의미: 우연한, 뜻하지 않은, 사고의

## 단어들

### accident

- 구성: Latin `ad-` = to + `cadere` = fall, happen
- 뜻: 사고, 우연한 일
- 어원적 연결: 의도하지 않은 일이 닥쳐옴
- 사용 맥락: `a car accident` = 교통사고, `by accident` = 우연히

### accidental

- 구성: `accident` + `-al`
- 뜻: 우연한, 뜻밖의, 사고의
- 어원적 연결: 계획한 것이 아니라 우연히 일어남
- 사용 맥락: `an accidental discovery` = 우연한 발견

### accidentally

- 구성: `accidental` + `-ly`
- 뜻: 우연히, 실수로
- 어원적 연결: 의도하지 않고 일이 일어남
- 사용 맥락: `accidentally delete a file` = 실수로 파일을 삭제하다

### incident

- 구성: Latin `in-` = into, upon + `cadere` = fall
- 뜻: 사건, 일어난 일
- 어원적 연결: 어떤 상황 안으로 떨어져 발생한 일
- 사용 맥락: `a serious incident` = 심각한 사건

### case

- 구성: Latin `casus` = fall, event
- 뜻: 경우, 사건, 사례; 소송 사건
- 어원적 연결: 실제로 떨어져 닥친 일, 발생한 사건
- 사용 맥락: `in this case` = 이 경우에는
- 주의: 물건을 담는 `case` = 상자, 케이스는 보통 Latin `capsa` = box 계열로 별도 어원.

### casual

- 구성: Latin `casualis` = by chance
- 뜻: 우연한, 격식 없는
- 어원적 연결: 계획보다 우연히 생긴 일에서 출발. 이후 엄격하지 않은 태도·복장으로 확장
- 사용 맥락: `casual clothes` = 평상복

### casually

- 구성: `casual` + `-ly`
- 뜻: 우연히; 격식 없이, 대수롭지 않게
- 어원적 연결: 정해진 형식이나 계획에 얽매이지 않는 방식
- 사용 맥락: `speak casually` = 격식 없이 말하다

### incidental

- 구성: `incident` + `-al`
- 뜻: 부수적인, 우연히 따르는
- 어원적 연결: 중심 사건에 딸려 일어남
- 사용 맥락: `incidental expenses` = 부대 비용

### coincidence

- 구성: `co-` = together + `incidence`
- 뜻: 우연의 일치
- 어원적 연결: 여러 일이 함께 떨어져 맞아떨어짐
- 사용 맥락: `a strange coincidence` = 이상한 우연의 일치

### coincide

- 구성: `co-` = together + `incidere` = fall upon
- 뜻: 동시에 일어나다, 일치하다
- 어원적 연결: 두 일이 같은 지점에 함께 떨어짐
- 사용 맥락: `The dates coincide.` = 날짜가 겹친다

### decadent

- 구성: `de-` = down + `cadere` = fall
- 뜻: 퇴폐적인, 쇠퇴한
- 어원적 연결: 아래로 떨어져 쇠퇴함
- 사용 맥락: `a decadent society` = 퇴폐적인 사회

### decay

- 구성: Old French `decair` ← Latin `de-` + `cadere`
- 뜻: 썩다, 쇠퇴하다
- 어원적 연결: 아래로 떨어져 무너지고 상함
- 사용 맥락: `tooth decay` = 충치

### cascade

- 구성: Italian `cascata` ← Latin `cadere`
- 뜻: 폭포처럼 쏟아지다, 연쇄적으로 이어지다
- 어원적 연결: 물이 떨어져 내려감
- 사용 맥락: `a cascade of events` = 연쇄적인 사건

### cadaver

- 구성: Latin `cadaver`, 전통적으로 `cadere`와 연결됨
- 뜻: 시체, 해부용 사체
- 어원적 연결: 쓰러져 죽은 몸이라는 이미지
- 사용 맥락: 의학·해부학 맥락

## 기억 포인트

`cad` 계열은 "떨어지다, 닥치다, 일어나다" 이미지.

- `accident`: 우연히 닥친 일
- `accidental`: 우연한
- `incident`: 발생한 사건
- `case`: 발생한 사건, 경우
- `casual`: 우연한, 격식 없는
- `coincide`: 함께 맞아떨어짐
- `decay`: 아래로 떨어져 쇠퇴함
- `cascade`: 떨어져 쏟아짐

## 비교 단어

### cid

- 뜻: 자르다, 잘라 경계를 내다
- 구분 포인트: [[cid]]는 Latin `caedere` = cut 계열. `cad`는 Latin `cadere` = fall 계열.

### caus

- 뜻: 원인, 이유, 책임 사유
- 구분 포인트: [[caus]]는 Latin `causa` = cause, reason 계열. `case`는 보통 Latin `casus` = fall, event 계열이라 `cause`와 어원은 구분.
