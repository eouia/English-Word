# derm

- 출발 단어: `dermatology`
- 핵심 어원: Greek `derma` / `dermatos` = 피부, 벗겨진 가죽
- 깊은 어근: PIE `*der-` = 찢다, 벗기다, 껍질을 까다
- 핵심 이미지: 몸을 덮는 피부와 가죽
- 관련 문서: [[logos]], [[epi]], [[hypo]], [[tax]]
- 주의: 전문 의학어는 너무 깊게 확장하지 않고, 시험·교양 어휘 중심으로 정리.

## 출발 단어 분석

`dermatology` = Greek `dermat-` + `-logy`

- `dermat-`: 피부의
- `logos/-logy`: 학문, 연구
- 원래 이미지: 피부를 연구하는 학문
- 현대 의미: 피부과학, 피부과

## 단어들

### dermatology

- 발음: /ˌdɝməˈtɑlədʒi/
- 구성: `dermat` = skin + [[logos]] = study
- 뜻: 피부과학, 피부과
- 어원적 연결: 피부와 피부 질환을 연구하는 분야
- 사용 맥락: `a dermatology clinic` = 피부과 병원

### dermatologist

- 발음: /ˌdɝməˈtɑlədʒɪst/
- 구성: `dermatology` + `-ist`
- 뜻: 피부과 의사
- 어원적 연결: 피부를 전문적으로 연구하고 치료하는 사람
- 사용 맥락: `see a dermatologist` = 피부과 의사를 만나다

### dermatological

- 발음: /ˌdɝmətəˈlɑdʒɪkəl/
- 구성: `dermatology` + `-ical`
- 뜻: 피부과의, 피부학의
- 어원적 연결: 피부 연구나 치료와 관련됨
- 사용 맥락: `dermatological treatment` = 피부과 치료

### dermatitis

- 발음: /ˌdɝməˈtaɪtɪs/
- 구성: `dermat` = skin + `-itis` = inflammation
- 뜻: 피부염
- 어원적 연결: 피부에 생긴 염증
- 사용 맥락: `contact dermatitis` = 접촉성 피부염

### dermis

- 발음: /ˈdɝməs/
- 구성: Greek `derma`
- 뜻: 진피
- 어원적 연결: 피부의 안쪽 층
- 사용 맥락: `the dermis lies below the epidermis` = 진피는 표피 아래에 있다

### epidermis

- 발음: /ˌɛpəˈdɝməs/
- 구성: `epi-` = upon + `dermis`
- 뜻: 표피
- 어원적 연결: 피부 위쪽 바깥층
- 사용 맥락: `the outer epidermis` = 바깥 표피

### epidermal

- 발음: /ˌɛpəˈdɝməl/
- 구성: `epidermis` + `-al`
- 뜻: 표피의
- 어원적 연결: 피부의 바깥층에 관련됨
- 사용 맥락: `epidermal cells` = 표피 세포

### hypodermic

- 발음: /ˌhaɪpəˈdɝmɪk/
- 구성: `hypo-` = under + `derm`
- 뜻: 피하의, 피하 주사의
- 어원적 연결: 피부 아래로 들어감
- 사용 맥락: `a hypodermic needle` = 피하 주사 바늘

### pachyderm

- 구성: Greek `pachys` = thick + `derm`
- 뜻: 두꺼운 피부를 가진 동물
- 어원적 연결: 두꺼운 피부
- 사용 맥락: `elephants are pachyderms` = 코끼리는 후피동물로 불렸다

### taxidermy

- 발음: /ˈtæksəˌdɝmi/
- 구성: Greek `taxis` = arrangement + `derma` = skin
- 뜻: 박제술
- 어원적 연결: 벗긴 가죽을 배열해 동물 모습을 보존함
- 사용 맥락: `a taxidermy specimen` = 박제 표본

### scleroderma

- 발음: /ˌsklɪrəˈdɝmə/
- 구성: Greek `skleros` = hard + `derma` = skin
- 뜻: 경피증
- 어원적 연결: 피부가 단단해지는 질환
- 사용 맥락: 의학 용어로 주로 사용

### echinoderm

- 구성: Greek `echinos` = hedgehog, sea urchin + `derm`
- 뜻: 극피동물
- 어원적 연결: 가시 같은 피부를 가진 동물
- 사용 맥락: `starfish are echinoderms` = 불가사리는 극피동물이다

## 기억 포인트

`derm` 계열은 "피부/가죽" 이미지.

- `dermatology`: 피부를 연구하는 학문
- `dermatologist`: 피부 전문의
- `dermatitis`: 피부 염증
- `epidermis`: 피부 위층
- `hypodermic`: 피부 아래
- `taxidermy`: 가죽을 배열해 박제함

## 비교 단어

### skin

- 뜻: 피부
- 구분 포인트: Germanic 기본어. `derm/dermat`는 Greek 학술·의학 어휘에서 주로 보임.

### cutaneous

- 뜻: 피부의
- 구분 포인트: Latin `cutis` = skin 계열. 의학어에서 `subcutaneous`처럼 따로 자주 보이는 별도 후보.
