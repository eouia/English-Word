# leg

- 출발 단어: `legal`
- 핵심 어원: 라틴어 `lex` / `leg-` = 법, 정해진 것
- 핵심 이미지: 사회가 "정해 놓은 규칙"
- 관련해서 헷갈릴 수 있는 어근: `leg` = 읽다/고르다 계열은 별도 문서로 구분할 수 있음

## 출발 단어 분석

`legal`은 현대 학습용으로 보면 `leg` + `-al`로 이해할 수 있다.

- `leg`: 법, 규칙과 관련된 어원
- `-al`: "~와 관련된", "~의 성질을 가진"이라는 형용사 접미사

따라서 `legal`은 "법과 관련된" → "합법적인"의 뜻.

## 단어들

### legal

- 구성: `leg` + `-al`
- 뜻: 합법적인, 법률상의
- 어원적 연결: 법에 맞는 상태를 가리킴
- 사용 맥락: `legal advice` = 법률 조언, `legal rights` = 법적 권리

### illegal

- 구성: `il-` + `legal`
- 뜻: 불법의
- 어원적 연결: `il-` = 부정 접두사. "법에 맞지 않는"의 뜻
- 사용 맥락: `illegal activity` = 불법 행위

### legality

- 구성: `legal` + `-ity`
- 뜻: 합법성
- 어원적 연결: 어떤 행동이나 상태가 법적으로 허용되는지 나타냄
- 사용 맥락: `question the legality of the decision` = 그 결정의 합법성에 의문을 제기하다

### legislation

- 구성: 라틴어 `legis` = 법 + `lation` 계열
- 뜻: 법률 제정, 법률
- 어원적 연결: 법을 만들거나 제정하는 과정과 관련
- 사용 맥락: `environmental legislation` = 환경 관련 법률

### legislature

- 구성: `legis` = 법 + `-ture`
- 뜻: 입법부
- 어원적 연결: 법을 만드는 기관
- 사용 맥락: `state legislature` = 주 의회

### legitimate

- 구성: 라틴어 `legitimus` = 법에 맞는
- 뜻: 합법적인, 정당한, 타당한
- 어원적 연결: 원래 "법에 의해 인정된"의 의미. 지금은 더 넓게 "정당한"의 뜻
- 사용 맥락: `a legitimate concern` = 타당한 우려

### legitimize

- 구성: `legitimate` + `-ize`
- 뜻: 합법화하다, 정당화하다
- 어원적 연결: 어떤 행동이나 제도를 법적 또는 사회적으로 인정받게 만듦
- 사용 맥락: `legitimize the practice` = 그 관행을 정당화하다

### privilege

- 구성: 라틴어 `privus` = 개인의 + `lex/leg-` = 법
- 뜻: 특권
- 어원적 연결: 원래 "개인에게만 적용되는 특별한 법"의 뜻. 이후 특정 사람이나 집단에게 주어지는 특별한 권리
- 사용 맥락: `special privilege` = 특별한 특권

### loyal

- 구성: 고대 프랑스어 `loial`, 라틴어 `legalis`와 연결
- 뜻: 충성스러운
- 어원적 연결: 원래 "법에 맞는, 신의 있는"의 의미. 사람이나 집단에 대한 충성으로 확장
- 사용 맥락: `loyal customers` = 충성 고객

## 기억 포인트

`leg` 계열은 "법으로 정해진 기준" 이미지.

- `legal`: 법에 맞는
- `illegal`: 법에 맞지 않는
- `legislation`: 법을 만드는 일
- `legitimate`: 법적/사회적으로 정당한
- `privilege`: 특정 사람에게만 주어진 특별한 법적 권리
- `loyal`: 원래는 법과 신의를 지키는 태도에서 출발
