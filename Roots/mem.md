# mem

- 출발 단어: `memory`
- 핵심 어원: Latin `memoria` = 기억
- 동사형: Latin `memorare` = 기억하다, 언급하다
- 핵심 이미지: 마음에 남겨 다시 떠올림
- 관련 문서: [[cogn]], [[ment]], [[narr]]
- 주의: `member`는 Latin `membrum` = 몸의 일부 계열. `memory`와 어원 분리.

## 출발 단어 분석

`memory` ← Old French `memoire` ← Latin `memoria`

- `memor`: 마음에 두는, 기억하는
- `memoria`: 기억
- 원래 이미지: 마음속에 보존된 것
- 현대 의미: 기억, 기억력, 추억, 저장 장치

## 단어들

### memory

- 발음: /ˈmɛmɚi/
- 구성: Latin `memoria`
- 뜻: 기억, 기억력, 추억
- 어원적 연결: 마음속에 남아 다시 떠올릴 수 있는 것
- 사용 맥락: `childhood memories` = 어린 시절의 기억

### memorable

- 발음: /ˈmɛmɚəbəl/
- 구성: `memory` 계열 + `-able`
- 뜻: 기억할 만한, 인상적인
- 어원적 연결: 마음에 남을 만큼 두드러짐
- 사용 맥락: `a memorable event` = 기억에 남는 사건

### memorize

- 발음: /ˈmɛmɚˌaɪz/
- 구성: `memory` + `-ize`
- 뜻: 암기하다
- 어원적 연결: 마음속에 기억으로 넣음
- 사용 맥락: `memorize vocabulary` = 어휘를 암기하다

### memorial

- 발음: /məˈmɔriəl/
- 구성: Latin `memorialis`
- 뜻: 기념물, 추모의
- 어원적 연결: 기억하게 만들기 위해 세운 것
- 사용 맥락: `a war memorial` = 전쟁 기념비

### commemorate

- 발음: /kəˈmɛmɚˌeɪt/
- 구성: `com-` = together + `memorare` = remember
- 뜻: 기념하다
- 어원적 연결: 함께 기억 속에 되살림
- 사용 맥락: `commemorate an anniversary` = 기념일을 기리다

### remembrance

- 발음: /riˈmɛmbrəns/
- 구성: `remember` + `-ance`
- 뜻: 기억, 추모
- 어원적 연결: 다시 마음에 떠올리는 행위
- 사용 맥락: `a day of remembrance` = 추모의 날

### remember

- 발음: /rɪˈmɛmbɚ/
- 구성: Old French `remembrer` ← Latin `rememorari`
- 뜻: 기억하다
- 어원적 연결: 다시 마음속에 떠올림
- 사용 맥락: `remember the date` = 날짜를 기억하다

### memoir

- 발음: /ˈmɛˌmwɑr/
- 구성: French `memoire` = memory, written account
- 뜻: 회고록
- 어원적 연결: 기억을 글로 정리한 것
- 사용 맥락: `write a memoir` = 회고록을 쓰다

### memo

- 발음: /ˈmɛˌmoʊ/
- 구성: 줄임말, `memorandum`에서 옴
- 뜻: 메모, 사내 문서
- 어원적 연결: 기억하거나 알리기 위해 적은 것
- 사용 맥락: `send a memo` = 메모를 보내다

### memorandum

- 발음: /ˌmɛmɚˈændəm/
- 구성: Latin `memorandum` = thing to be remembered
- 뜻: 메모, 각서
- 어원적 연결: 기억되어야 할 것
- 사용 맥락: `a memorandum of understanding` = 양해각서

## 기억 포인트

`mem` 계열은 "마음에 남겨 다시 떠올림" 이미지.

- `memory`: 기억
- `memorize`: 기억 속에 넣음
- `memorial`: 기억하게 만드는 것
- `commemorate`: 함께 기억함
- `memoir`: 기억을 글로 씀

## 비교 단어

### member

- 뜻: 구성원, 신체의 일부
- 구분 포인트: Latin `membrum` 계열. `memory`의 `memoria`와 분리.
