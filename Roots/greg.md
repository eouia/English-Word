# greg

- 출발 단어: `segregation`
- 핵심 어원: Latin `grex`, `gregis` = flock, herd
- 핵심 이미지: 무리, 떼, 집단
- 관련 문서: [[sect]]
- 주의: `egregious`는 원래 "무리 밖으로 두드러진" 긍정적 의미도 있었지만, 현대 영어에서는 주로 "터무니없이 나쁜" 뜻.
- 주의: `segregate`는 `segment`와 형태·의미가 비슷해 보이지만, `segment`는 [[sect]] 계열의 "잘라낸 부분" 이미지.

## 출발 단어 분석

`segregation` = `se-` + `greg` + `-ation`

- `se-`: apart, away
- `greg`: flock, group
- `-ation`: 과정·상태 명사
- 원래 이미지: 무리에서 따로 떼어 놓음
- 현대 의미: 분리, 특히 제도적 인종 분리

## 단어들

### segregation

- 구성: `se-` = apart + Latin `grex/gregis` = flock
- 뜻: 분리, 격리; 인종 분리
- 어원적 연결: 한 집단을 무리에서 따로 떼어 둠
- 사용 맥락: `racial segregation` = 인종 분리
- 주의: 역사·사회 맥락에서는 단순 분리보다 제도적 차별 구조를 가리키는 경우가 많음.

### segregate

- 구성: `se-` + `greg`
- 뜻: 분리하다, 격리하다
- 어원적 연결: 집단에서 따로 떼어 놓음
- 사용 맥락: `segregate students by race` = 학생을 인종별로 분리하다

### segregated

- 구성: `segregate` + `-ed`
- 뜻: 분리된, 격리된
- 어원적 연결: 한 무리에서 따로 떼어진 상태
- 사용 맥락: `segregated schools` = 분리 학교

### desegregate

- 구성: `de-` = reverse + `segregate`
- 뜻: 인종 분리를 철폐하다, 통합하다
- 어원적 연결: 따로 떼어 둔 상태를 되돌림
- 사용 맥락: `desegregate public schools` = 공립학교 인종 분리를 철폐하다

### desegregation

- 구성: `desegregate` + `-ion`
- 뜻: 인종 분리 철폐, 통합
- 어원적 연결: 분리된 집단을 다시 같은 공간·제도 안으로 돌려놓는 과정
- 사용 맥락: `school desegregation` = 학교 인종 분리 철폐

### aggregate

- 구성: `ad-` = to + `greg`
- 뜻: 모으다; 총합, 집합체
- 어원적 연결: 여러 개체를 한 무리로 모음
- 사용 맥락: `aggregate data` = 데이터를 집계하다

### aggregation

- 구성: `aggregate` + `-ion`
- 뜻: 집합, 집계
- 어원적 연결: 흩어진 것을 한 무리로 모으는 과정
- 사용 맥락: `data aggregation` = 데이터 집계

### congregate

- 구성: `con-` = together + `greg`
- 뜻: 모이다, 집합하다
- 어원적 연결: 사람들이 한 무리로 함께 모임
- 사용 맥락: `people congregate in the square` = 사람들이 광장에 모이다

### congregation

- 구성: `congregate` + `-ion`
- 뜻: 신도 회중, 모인 사람들
- 어원적 연결: 예배나 공동체 활동을 위해 함께 모인 무리
- 사용 맥락: `a local congregation` = 지역 신도 공동체

### gregarious

- 구성: Latin `gregarius` = belonging to a flock
- 뜻: 사교적인, 무리 지어 사는
- 어원적 연결: 무리 속에 있기를 좋아함
- 사용 맥락: `a gregarious person` = 사교적인 사람

### egregious

- 구성: `e-/ex-` = out + `greg`
- 뜻: 터무니없는, 지독한
- 어원적 연결: 무리 밖으로 두드러질 만큼 눈에 띔
- 사용 맥락: `an egregious error` = 터무니없는 실수
- 주의: 현대 영어에서는 거의 부정적 의미. "뛰어난"이라는 옛 의미는 시험 독해에서 역사적 설명으로만 접할 가능성이 있음.

## 기억 포인트

`greg` 계열은 "무리와 집단" 이미지.

- `aggregate`: 한 무리로 모음
- `congregate`: 함께 모임
- `congregation`: 모인 신도·회중
- `segregate`: 무리에서 따로 떼어냄
- `segregation`: 제도적 분리
- `gregarious`: 무리 속에 있기를 좋아함
- `egregious`: 무리 밖으로 두드러질 만큼 심함

`segregate`와 `segment`의 차이:

- `segregate`: `se-` + `greg`, 무리에서 따로 떼어 놓음
- `segment`: Latin `segmentum` ← `secare`, 전체를 잘라낸 한 조각

## 비교 단어

- `collect`: 흩어진 것을 모으는 일반 동작. `greg`는 무리·떼라는 집단 이미지가 강함.
- `separate`: 일반적으로 나누다. `segregate`는 사회·제도적 분리 맥락에서 더 무거움.
