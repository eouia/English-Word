# lev

- 출발 단어: `relevant`
- 핵심 의미: 들어 올리다, 가볍게 하다
- 관련 형태: Latin `levare` = raise, lift, lighten; `levis` = light
- 관련 문서: [[alt]], [[high]]

## 핵심 이미지

`lev` 계열의 바탕은 "가볍게 들어 올림".

높이 들어 올리면 `elevate`.
부담을 덜어 가볍게 하면 `relieve`, `alleviate`.
들어 올리는 도구는 `lever`.
문제에서 중요하게 들어 올려지는 것은 `relevant`.
무게감 없이 가벼운 태도는 `levity`.

## 단어들

### relevant

- 발음: /ˈrɛləvənt/
- 구성: Latin `re-` = again/up + `levare` = raise
- 뜻: 관련 있는, 적절한
- 연결: 논의에서 중요하게 들어 올려져 고려될 만함
- 사용 맥락: `relevant information` = 관련 정보

### relevance

- 발음: /ˈrɛləvəns/
- 구성: `relevant` + `-ce`
- 뜻: 관련성, 적절성
- 연결: 어떤 문제와 관계되어 중요하게 다뤄질 만한 정도
- 사용 맥락: `the relevance of the evidence` = 그 증거의 관련성

### irrelevant

- 발음: /ˌɪˈrɛləvənt/
- 구성: `ir-` = not + `relevant`
- 뜻: 관련 없는, 부적절한
- 연결: 논의에서 들어 올려 고려할 필요가 없음
- 사용 맥락: `irrelevant details` = 관련 없는 세부 사항

### elevate

- 발음: /ˈɛləˌveɪt/
- 구성: `e-` = out/up + `levare` = lift
- 뜻: 들어 올리다, 승진시키다, 높이다
- 연결: 더 높은 위치나 지위로 올림
- 사용 맥락: `elevate standards` = 기준을 높이다

### elevation

- 발음: /ˌɛləˈveɪʃən/
- 구성: `elevate` + `-ion`
- 뜻: 고도, 상승, 승진
- 연결: 높이 올라간 상태나 과정
- 사용 맥락: `high elevation` = 높은 고도

### relieve

- 발음: /rɪˈliv/
- 구성: `re-` = back/again + `levare` = lighten
- 뜻: 덜어 주다, 완화하다, 구제하다
- 연결: 부담이나 고통을 가볍게 해 줌
- 사용 맥락: `relieve stress` = 스트레스를 덜다

### relief

- 발음: /rɪˈlif/
- 구성: `relieve` 계열
- 뜻: 안도, 완화, 구호; 부조
- 연결: 무거운 부담이 덜어져 가벼워짐
- 사용 맥락: `a sense of relief` = 안도감

### alleviate

- 발음: /əˈliviˌeɪt/
- 구성: Latin `ad-` = to + `levare` = lighten
- 뜻: 완화하다, 덜어 주다
- 연결: 고통이나 문제의 무게를 가볍게 함
- 사용 맥락: `alleviate poverty` = 빈곤을 완화하다

### lever

- 발음: /ˈlɛvɚ/
- 구성: Old French `levier`, from `lever` = raise
- 뜻: 지렛대
- 연결: 무거운 것을 들어 올리는 도구
- 사용 맥락: `pull a lever` = 레버를 당기다

### leverage

- 발음: /ˈlɛvɚɪdʒ/
- 구성: `lever` + `-age`
- 뜻: 지렛대 작용; 영향력, 활용
- 연결: 작은 힘으로 큰 것을 들어 올리는 효과에서 영향력으로 확장
- 사용 맥락: `use data as leverage` = 데이터를 영향력으로 활용하다

### levity

- 발음: /ˈlɛvɪti/
- 구성: Latin `levitas` = lightness
- 뜻: 경솔함, 가벼움
- 연결: 무게감이나 진지함이 부족한 상태
- 사용 맥락: `a moment of levity` = 잠깐의 가벼운 분위기

## 비교 단어

`alt`는 [[alt]] 문서처럼 높음, 높이다 계열.
`lev`는 "가볍게 하거나 들어 올림"이 중심이라 `relieve`, `alleviate`, `lever`까지 연결됨.

`high`는 [[high]] 문서처럼 독립 형용사 "높음"이 중심.
`elevate`는 어떤 것을 위로 들어 올리는 동작이 강함.
