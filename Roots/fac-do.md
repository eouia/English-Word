# fac-do

- 출발 단어: `efficient`
- 핵심 어원: Latin `facere` / `fic-` / `facilis` = 하다, 하기 쉬운
- 핵심 이미지: 일을 해내거나 쉽게 만듦
- 허브 문서: [[fac]]

## 단어들

### efficient

- 발음: /ɪˈfɪʃənt/
- 구성: `ex-` = out + `fic`
- 뜻: 효율적인
- 어원적 연결: 결과를 잘 만들어 냄
- 사용 맥락: `an efficient system` = 효율적인 시스템

### sufficient

- 발음: /səˈfɪʃənt/
- 구성: `sub-` = under, up to + `fic`
- 뜻: 충분한
- 어원적 연결: 필요한 정도까지 만들어 채움
- 사용 맥락: `sufficient evidence` = 충분한 증거

### proficient

- 발음: /prɑˈfɪʃənt/
- 구성: `pro-` = forward + `fic`
- 뜻: 능숙한, 숙달된
- 어원적 연결: 앞으로 일을 잘 해 나갈 수 있는 상태
- 사용 맥락: `proficient in English` = 영어에 능숙한

### deficient

- 발음: /dɪˈfɪʃənt/
- 구성: `de-` = down, away + `fic`
- 뜻: 부족한, 결핍된
- 어원적 연결: 만들어져 있어야 할 것이 빠짐
- 사용 맥락: `deficient in iron` = 철분이 부족한

### difficult

- 발음: /ˈdɪfəkəlt/
- 구성: `dis-` = apart, away + `facilis` = easy to do
- 뜻: 어려운
- 어원적 연결: 하기가 쉽지 않음
- 사용 맥락: `a difficult task` = 어려운 과제

### facilitate

- 발음: /fəˈsɪləˌteɪt/
- 구성: `facilis` = easy to do + `-ate`
- 뜻: 촉진하다, 쉽게 하다
- 어원적 연결: 하기가 쉽게 만듦
- 사용 맥락: `facilitate discussion` = 토론을 촉진하다

### feasible

- 발음: /ˈfizəbəl/
- 구성: Old French `faisable` = able to be done
- 뜻: 실행 가능한, 실현 가능한
- 어원적 연결: 할 수 있는 것
- 사용 맥락: `a feasible plan` = 실행 가능한 계획

### office

- 발음: /ˈɔfɪs/
- 구성: Latin `officium` = service, duty
- 뜻: 사무실, 직무, 관직
- 어원적 연결: 맡아서 해야 하는 일 → 그 일을 하는 자리나 장소
- 사용 맥락: `public office` = 공직

### official

- 발음: /əˈfɪʃəl/
- 구성: `office` + `-ial`
- 뜻: 공식적인, 공무상의, 공무원
- 어원적 연결: 직무나 관직에 속한 것
- 사용 맥락: `an official statement` = 공식 성명

### officer

- 발음: /ˈɔfəsɚ/
- 구성: `office` + `-er`
- 뜻: 장교, 경찰관, 임원, 공무원
- 어원적 연결: 특정 직무나 권한을 맡은 사람
- 사용 맥락: `a police officer` = 경찰관, `a company officer` = 회사 임원

### officiate

- 발음: /əˈfɪʃiˌeɪt/
- 구성: Latin `officium` 계열 + `-ate`
- 뜻: 주례하다, 의식을 집행하다, 심판을 맡다
- 어원적 연결: 공식 직무를 수행함
- 사용 맥락: `officiate at a wedding` = 결혼식 주례를 맡다

## 기억 포인트

`fac-do`는 "일을 해냄 / 쉽게 함" 갈래.

- `efficient`: 결과를 잘 냄
- `sufficient`: 충분히 채움
- `proficient`: 일을 잘 해냄
- `deficient`: 부족함
- `facilitate`: 쉽게 만듦
- `feasible`: 할 수 있음
- `office`: 맡은 일, 직무의 자리
- `official`: 직무·관직에 속한 공식적 성격
