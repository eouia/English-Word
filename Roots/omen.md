# omen

- 출발 단어: `omen`
- 핵심 어원: Latin `omen` = 징조, 예언적 신호
- 변형: `omen`, `omin`
- 핵심 의미: 미래의 좋고 나쁜 일을 예고하는 신호
- 관련 문서: [[sider]], [[spect]]
- 주의: Latin `omen` 자체의 더 깊은 어원은 불확실. `auspicious`, `augury`, `portent`는 의미는 가깝지만 다른 어원.

## 출발 단어 분석

`omen` ← Latin `omen`

- 뜻: 좋은 일이나 나쁜 일을 예고한다고 여겨지는 신호
- 원래 이미지: 우연한 사건을 미래의 표시로 읽음
- 현대 의미: 징조, 전조

## 단어들

### omen

- 구성: Latin `omen`
- 뜻: 징조, 전조
- 어원적 연결: 미래 일을 예고한다고 여겨지는 신호
- 사용 맥락: `a bad omen` = 나쁜 징조

### ominous

- 구성: Latin `ominosus` ← `omen`
- 뜻: 불길한, 흉조의
- 어원적 연결: 나쁜 일이 올 것 같은 징조를 띰
- 사용 맥락: `ominous clouds` = 불길한 구름

### ominously

- 구성: `ominous` + `-ly`
- 뜻: 불길하게
- 어원적 연결: 나쁜 징조처럼 느껴지는 방식
- 사용 맥락: `the sky darkened ominously` = 하늘이 불길하게 어두워졌다

### ominousness

- 구성: `ominous` + `-ness`
- 뜻: 불길함
- 어원적 연결: 흉조처럼 느껴지는 성질
- 사용 맥락: literary context

### abominate

- 구성: `ab-` = away from + `omin` = omen
- 뜻: 혐오하다, 몹시 싫어하다
- 어원적 연결: 나쁜 징조를 피하듯 멀리함
- 사용 맥락: `abominate cruelty` = 잔인함을 혐오하다

### abomination

- 구성: `abominate` + `-ion`
- 뜻: 혐오스러운 것, 가증스러운 것
- 어원적 연결: 피해야 할 불길한 것으로 여겨짐
- 사용 맥락: `an abomination to society` = 사회에 혐오스러운 것

### abominable

- 구성: `abominate` + `-able`
- 뜻: 혐오스러운, 끔찍한
- 어원적 연결: 나쁜 징조처럼 꺼리고 피할 만함
- 사용 맥락: `abominable behavior` = 혐오스러운 행동

### abominably

- 구성: `abominable` + `-ly`
- 뜻: 끔찍하게, 형편없이
- 어원적 연결: 혐오할 만큼 나쁘게
- 사용 맥락: `abominably cold weather` = 끔찍하게 추운 날씨

## 기억 포인트

- `omen`: 미래를 예고하는 신호.
- `ominous`: 나쁜 징조가 있는 듯한.
- `abominate`: 나쁜 징조를 피하듯 혐오하다.
- `abominable`: 혐오스러운, 끔찍한.

## 비교 단어

### auspicious

- `auspicious`는 Latin `auspex/auspicium` = 새의 비행을 관찰해 점침.
- 뜻은 “길조의, 상서로운”이지만 `omen`과 어원은 다름.

### augury

- `augury`는 로마의 점술·예언 관찰과 관련된 다른 계열.
- 현대 의미는 `omen`과 가까운 “전조, 징조”.

### portent

- `portent`는 Latin `portendere` = 앞에 보여 주다 계열로 설명됨.
- `omen`보다 크고 불길한 예언적 신호 느낌이 강함.

### sider

- [[sider]]의 `consider`, `desire`는 별을 살피는 전통적 이미지와 연결됨.
- `omen`은 별뿐 아니라 어떤 사건이든 징조로 읽는 일반 단어.
