# ord

- 출발 단어: `subordinate`
- 핵심 어원: 라틴어 `ordo` / `ordin-` = 줄, 순서, 등급, 배열
- 핵심 이미지: 정해진 순서나 등급에 배치함
- 비교: [[sed]]의 `subside`, `subsidy`와 다름

## 출발 단어 분석

`subordinate`는 `sub-` + `ordinare`.

- `sub-`: 아래
- `ordinare`: 순서대로 배열하다, 질서 있게 하다

따라서 `subordinate`는 "아래 등급에 배치된" → "하위의, 부하"의 뜻.

## 단어들

### subordinate

- 발음: /səˈbɔrdəˌneɪt/
- 구성: `sub-` + `ordinare`
- 뜻: 하위의, 부하, 종속시키다
- 어원적 연결: 어떤 질서나 등급 아래에 놓임
- 사용 맥락: `a subordinate role` = 하위 역할

### subordination

- 발음: /səˌbɔrdəˈneɪʃən/
- 구성: `subordinate` + `-ion`
- 뜻: 종속, 하위 배치
- 어원적 연결: 더 낮은 등급에 배치된 상태
- 사용 맥락: `subordination of personal interests` = 개인적 이익의 종속

### order

- 발음: /ˈɔrdɚ/
- 구성: 라틴어 `ordo`
- 뜻: 순서, 질서, 명령, 주문
- 어원적 연결: 줄이나 등급으로 배열된 상태
- 사용 맥락: `in order` = 순서대로, 질서 있게

### ordinary

- 발음: /ˈɔrdəˌnɛri/
- 구성: `order` 계열
- 뜻: 보통의, 평범한
- 어원적 연결: 정해진 질서 안에 있는 것
- 사용 맥락: `ordinary people` = 평범한 사람들

### extraordinary

- 발음: /ˌɛkstrəˈɔrdəˌnɛri/
- 구성: `extra-` + `ordinary`
- 뜻: 비범한, 특별한
- 어원적 연결: 보통 질서 밖에 있음
- 사용 맥락: `extraordinary talent` = 비범한 재능

### ordinance

- 발음: /ˈɔrdənəns/
- 구성: `ordinare` 계열
- 뜻: 조례, 법령
- 어원적 연결: 사회 질서를 위해 정해 둔 규칙
- 사용 맥락: `city ordinance` = 시 조례

### ordain

- 발음: /ɔˈrdeɪn/
- 구성: 라틴어 `ordinare`
- 뜻: 임명하다, 정하다
- 어원적 연결: 특정 질서나 직위 안에 배치함
- 사용 맥락: `be ordained as a priest` = 사제로 임명되다

### coordinate

- 발음: /koʊˈɔrdənət/
- 구성: `co-` + `ordinate`
- 뜻: 조정하다, 동등한, 좌표
- 어원적 연결: 함께 순서나 위치를 맞춤
- 사용 맥락: `coordinate schedules` = 일정을 조율하다

### coordination

- 발음: /koʊˌɔrdəˈneɪʃən/
- 구성: `coordinate` + `-ion`
- 뜻: 조정, 협응
- 어원적 연결: 여러 요소의 순서나 움직임을 맞춤
- 사용 맥락: `team coordination` = 팀 협업 조정

### disorder

- 발음: /dɪˈsɔrdɚ/
- 구성: `dis-` + `order`
- 뜻: 무질서, 장애
- 어원적 연결: 질서가 깨진 상태
- 사용 맥락: `sleep disorder` = 수면 장애

### inordinate

- 발음: /ˌɪˈnɔrdənɪt/
- 구성: `in-` + `ordinate`
- 뜻: 과도한, 지나친
- 어원적 연결: 질서나 한계를 벗어남
- 사용 맥락: `inordinate delay` = 지나친 지연

## 기억 포인트

`ord` 계열은 "순서와 등급" 이미지.

- `order`: 순서, 질서
- `ordinary`: 질서 안의 보통 것
- `extraordinary`: 보통 질서 밖의 특별한 것
- `subordinate`: 아래 등급에 놓인 것
- `coordinate`: 함께 순서를 맞춤
- `ordinance`: 질서를 위해 정한 규칙
- `ordain`: 직위나 질서 안에 임명함
- `disorder`: 질서가 깨짐
- `inordinate`: 질서나 한계를 벗어남
