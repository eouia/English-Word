# proach

- 출발 단어: `approach`
- 핵심 어원: Old French `aprochier` = come nearer
- 깊은 어원: Latin `prope` = near
- 핵심 이미지: 가까이 다가감, 가까이 오게 함
- 관련 문서: [[proxim]]
- 주의: `approach`는 의미상 [[proxim]]과 가깝지만, 학습용으로는 Old French `aprochier` 계열의 `proach` 형태로 따로 정리.

## 출발 단어 분석

`approach` ← Old French `aprochier`

- `a-`: to
- `prochier`: come near
- Latin `prope`: near
- 원래 이미지: 가까운 쪽으로 다가감
- 현대 의미: 접근하다, 접근법, 다가옴

## 단어들

### approach

- 발음: /əˈproʊtʃ/
- 구성: Old French `aprochier` = come nearer
- 뜻: 접근하다, 다가가다, 접근법
- 어원적 연결: 대상에 가까이 감. 생각에서는 문제를 다루는 방식으로 확장
- 사용 맥락: `approach the problem` = 문제에 접근하다

### approachable

- 발음: /əˈproʊtʃəbəl/
- 구성: `approach` + `-able`
- 뜻: 접근하기 쉬운, 친근한
- 어원적 연결: 가까이 다가갈 수 있음
- 사용 맥락: `an approachable teacher` = 다가가기 쉬운 선생님

### reproach

- 발음: /riˈproʊtʃ/
- 구성: Old French `reprochier` = bring near, blame
- 뜻: 비난하다, 책망하다
- 어원적 연결: 잘못을 상대 앞에 가까이 가져와 들이댐
- 사용 맥락: `reproach someone for lying` = 거짓말을 한 사람을 책망하다

### irreproachable

- 구성: `in-` = not + `reproachable`
- 뜻: 나무랄 데 없는
- 어원적 연결: 비난을 가까이 가져다 붙일 수 없음
- 사용 맥락: `irreproachable conduct` = 나무랄 데 없는 행동

### encroach

- 발음: /ɪˈnkroʊtʃ/
- 구성: Old French `encrochier`, 의미상 침범해 가까이 들어감
- 뜻: 침범하다, 잠식하다
- 어원적 연결: 남의 영역 안쪽으로 가까이 들어와 차지함
- 사용 맥락: `encroach on privacy` = 사생활을 침해하다

### encroachment

- 발음: /ɛˈnkroʊtʃmənt/
- 구성: `encroach` + `-ment`
- 뜻: 침범, 잠식
- 어원적 연결: 경계 안으로 들어와 차지하는 행위
- 사용 맥락: `urban encroachment` = 도시의 잠식

## 기억 포인트

`proach` 계열은 "가까이 다가감" 이미지.

- `approach`: 가까이 감, 접근법
- `approachable`: 다가가기 쉬운
- `reproach`: 잘못을 앞에 들이대며 책망함
- `irreproachable`: 책망할 수 없는
- `encroach`: 남의 영역에 다가와 침범함

## 비교 단어

### proxim

- 뜻: 가까움, 거의 맞음
- 구분 포인트: [[proxim]]은 Latin `proximus` = nearest 계열. `proach`는 가까이 다가가는 동작과 접근법 이미지가 강함.

### access

- 뜻: 접근, 접근권
- 구분 포인트: `access`는 [[cess]] = 가다 계열로 "들어갈 수 있음"이 중심. `approach`는 가까이 다가가는 방식까지 포함.
