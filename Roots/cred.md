# cred

- 출발 단어: `credit`
- 핵심 어원: 라틴어 `credere` = 믿다, 맡기다
- 핵심 이미지: 믿고 맡김
- 관련해서 헷갈릴 수 있는 어근: [[fid]] = 믿음, 충실함 / [[spond]] = 약속하다, 보증하다 계열

## 출발 단어 분석

`credit`은 Latin `creditum`에서 온 말.

- `credere`: 믿다, 맡기다
- `creditum`: 맡겨진 것, 빌려준 것

따라서 `credit`은 "믿고 맡긴 것" → "신용, 공로, 학점, 입금"의 뜻.

## 단어들

### credit

- 발음: /ˈkrɛdət/
- 구성: Latin `creditum`
- 뜻: 신용, 공로, 학점, 입금하다
- 어원적 연결: 믿고 맡겨 인정한 것
- 사용 맥락: `give credit for the work` = 그 일의 공로를 인정하다

### credible

- 발음: /ˈkrɛdəbəl/
- 구성: `cred` + `-ible`
- 뜻: 믿을 만한
- 어원적 연결: 믿을 수 있음
- 사용 맥락: `credible evidence` = 믿을 만한 증거

### credibility

- 발음: /ˌkrɛdəˈbɪlɪti/
- 구성: `credible` + `-ity`
- 뜻: 신뢰성
- 어원적 연결: 믿을 수 있는 성질
- 사용 맥락: `lose credibility` = 신뢰를 잃다

### incredible

- 발음: /ˌɪˈnkrɛdəbəl/
- 구성: `in-` = not + `credible`
- 뜻: 믿기 어려운, 대단한
- 어원적 연결: 믿기 어려울 만큼 큼
- 사용 맥락: `an incredible achievement` = 놀라운 성취

### credulous

- 발음: /ˈkrɛdʒələs/
- 구성: `cred` + `-ulous`
- 뜻: 잘 믿는, 잘 속는
- 어원적 연결: 너무 쉽게 믿음
- 사용 맥락: `credulous consumers` = 잘 속는 소비자들

### incredulous

- 발음: /ˌɪˈnkrɛdʒələs/
- 구성: `in-` = not + `credulous`
- 뜻: 믿지 못하는, 의심하는
- 어원적 연결: 쉽게 믿지 않음
- 사용 맥락: `an incredulous look` = 못 믿겠다는 표정

### creed

- 발음: /ˈkrid/
- 구성: Latin `credo` = I believe
- 뜻: 신조, 신념
- 어원적 연결: 믿는 바를 문장으로 굳힌 것
- 사용 맥락: `a political creed` = 정치적 신조

### credentials

- 발음: /krəˈdɛnʃəlz/
- 구성: Medieval Latin `credentialis`
- 뜻: 자격, 신임장
- 어원적 연결: 믿고 맡길 수 있음을 보여 주는 증명
- 사용 맥락: `academic credentials` = 학문적 자격

### discredit

- 발음: /dɪˈskrɛdət/
- 구성: `dis-` = apart, away + `credit`
- 뜻: 신뢰를 떨어뜨리다, 불신하게 만들다
- 어원적 연결: 믿음을 떼어 냄
- 사용 맥락: `discredit a theory` = 이론의 신뢰를 떨어뜨리다

### accredit

- 발음: /əˌkrɛdət/
- 구성: `ad-` = to + `credit`
- 뜻: 인가하다, 공로로 돌리다
- 어원적 연결: 믿음을 공식적으로 부여함
- 사용 맥락: `an accredited school` = 공인된 학교

## 기억 포인트

`cred` 계열은 "믿음" 이미지.

- `credit`: 믿고 맡긴 신용
- `credible`: 믿을 만함
- `incredible`: 믿기 어려움
- `credulous`: 너무 잘 믿음
- `creed`: 믿는 바
- `credentials`: 믿게 만드는 자격

## 비교 단어

### trust

- 뜻: 믿다, 신뢰
- 구분 포인트: [[trust]]는 일반적 신뢰와 충실함의 기본어. `cred`는 라틴계 어휘에서 "믿을 만함, 신용, 공식 인정"으로 많이 나타남

### fid

- 뜻: 믿음, 충실함, 신의
- 구분 포인트: [[fid]]는 `fidelity`, `faithful`, `confidence`처럼 믿음을 지키거나 믿고 기대는 상태에 가까움. `cred`는 믿을 만하다고 판단하고 신용을 부여하는 쪽.
