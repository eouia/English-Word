# fruit

- 출발 단어: `fruitless`
- 핵심 어원: Latin `fructus` = fruit, enjoyment, profit
- 핵심 이미지: 노력이나 성장 끝에 맺히는 열매와 성과
- 관련 문서: [[use]], [[auster]]
- 주의: `frugal`은 Latin `frux/frugis` = fruit, produce 계열로 가깝지만, 현대 의미는 "절약하는" 쪽으로 굳어짐.

## 출발 단어 분석

`fruitless` = `fruit` + `-less`

- `fruit`: 열매, 성과
- `-less`: without
- 원래 이미지: 열매가 맺히지 않음
- 현대 의미: 성과 없는, 헛된

## 단어들

### fruitless

- 구성: `fruit` + `-less`
- 뜻: 성과 없는, 헛된, 무익한
- 어원적 연결: 노력했지만 열매나 결과가 맺히지 않음
- 사용 맥락: `a fruitless search` = 성과 없는 수색, `fruitless efforts` = 헛된 노력

### fruit

- 구성: Old French `fruit` ← Latin `fructus`
- 뜻: 열매, 과일; 성과
- 어원적 연결: 식물이 맺는 열매에서 노력의 결과로 의미 확장
- 사용 맥락: `bear fruit` = 결실을 맺다, `the fruit of hard work` = 노력의 성과

### fruitful

- 구성: `fruit` + `-ful`
- 뜻: 생산적인, 유익한, 결실 있는
- 어원적 연결: 열매를 많이 맺듯 결과와 성과가 풍부함
- 사용 맥락: `a fruitful discussion` = 유익한 논의

### fruition

- 구성: Latin `frui` = enjoy + `-tion`
- 뜻: 성취, 실현, 결실
- 어원적 연결: 노력의 열매를 누리는 상태
- 사용 맥락: `bring a plan to fruition` = 계획을 실현하다

### fructify

- 구성: Latin `fructificare` = make fruitful
- 뜻: 열매를 맺다, 결실을 맺게 하다
- 어원적 연결: 실제 또는 비유적 열매를 만들어 냄
- 사용 맥락: 문어적 표현. 현대 일반 시험 어휘로는 드묾.

### fructose

- 구성: Latin `fructus` + chemical suffix `-ose`
- 뜻: 과당
- 어원적 연결: 과일에 들어 있는 당
- 사용 맥락: 영양·화학 맥락

## 기억 포인트

`fruit`는 "열매와 성과" 이미지.

- `fruit`: 열매, 성과
- `fruitful`: 열매가 많아 유익함
- `fruitless`: 열매가 없어 헛됨
- `fruition`: 열매를 누리는 성취
- `fructify`: 열매를 맺게 함

## 비교 단어

### frugal

- 뜻: 절약하는, 검소한
- 구분 포인트: Latin `frux/frugis` = fruit, produce 계열. 원래 "생산물·가치 있는 것"과 관련되지만, 현대 의미는 자원을 아껴 쓰는 태도.

### useless

- 뜻: 쓸모없는
- 구분 포인트: [[use]] 계열. `fruitless`는 노력의 "결과가 없음"에 초점.

### fertile

- 뜻: 비옥한, 생산적인
- 구분 포인트: 열매를 맺을 수 있는 생산 능력에 초점. `fruitful`은 실제로 성과가 많이 나오는 상태.
