# tranquil

- 출발 단어: `tranquil`
- 핵심 어원: Latin `tranquillus` = calm, quiet, still
- 핵심 이미지: 흔들림 없이 잔잔하고 평온함
- 관련 문서: [[plac]]
- 철자 변형: British English에서는 `tranquillity`, `tranquillise`처럼 `l`을 두 번 쓰는 형태도 있음.

## 출발 단어 분석

`tranquil` ← Latin `tranquillus`

- `tranquillus`: calm, quiet, still
- 현대 의미: 고요한, 평온한, 차분한

원래 이미지는 바람이나 물결이 가라앉은 듯 조용하고 안정된 상태. 마음, 장소, 분위기 모두에 쓸 수 있음.

## 단어들

### tranquil

- 구성: Latin `tranquillus` = calm, quiet, still
- 뜻: 고요한, 평온한, 차분한
- 어원적 연결: 흔들림이나 소란이 사라져 잔잔함
- 사용 맥락: `a tranquil lake` = 잔잔한 호수, `a tranquil mind` = 평온한 마음

### tranquility

- 구성: `tranquil` + `-ity`
- 뜻: 평온, 고요, 안정
- 어원적 연결: 소란이나 불안이 없는 상태
- 사용 맥락: `a sense of tranquility` = 평온한 느낌

### tranquillity

- 구성: `tranquility`의 British spelling
- 뜻: 평온, 고요
- 어원적 연결: `tranquility`와 같음
- 사용 맥락: British English에서 자주 보임.

### tranquilize

- 구성: `tranquil` + `-ize`
- 뜻: 진정시키다, 안정제를 투여하다
- 어원적 연결: 불안하거나 흥분한 상태를 평온하게 만듦
- 사용 맥락: `tranquilize an animal` = 동물을 진정시키다

### tranquilizer

- 구성: `tranquilize` + `-er`
- 뜻: 진정제, 안정제
- 어원적 연결: 마음이나 몸의 흥분을 가라앉히는 것
- 사용 맥락: `take a tranquilizer` = 진정제를 복용하다

### tranquilly

- 구성: `tranquil` + `-ly`
- 뜻: 평온하게, 고요히
- 어원적 연결: 흔들림 없이 차분한 방식
- 사용 맥락: `sleep tranquilly` = 평온히 잠들다

## 기억 포인트

`tranquil` 계열은 "잔잔하고 평온함" 이미지.

- `tranquil`: 고요한, 평온한
- `tranquility`: 평온, 고요
- `tranquilize`: 진정시키다
- `tranquilizer`: 진정제
- `tranquilly`: 평온하게

## 비교 단어

### calm

- 뜻: 차분한, 진정시키다
- 구분 포인트: `calm`은 가장 일반적인 차분함. `tranquil`은 문어적이고, 장소나 마음이 깊이 고요한 느낌.

### placate

- 뜻: 달래다, 진정시키다
- 구분 포인트: [[plac]]의 `placate`는 화난 사람을 만족시켜 누그러뜨림. `tranquilize`는 흥분·불안을 가라앉혀 평온하게 함.

### serene

- 뜻: 평온한, 맑고 고요한
- 구분 포인트: `serene`은 품위 있고 맑은 평온함. `tranquil`은 소란이 사라진 잔잔한 안정감.

### quiet

- 뜻: 조용한
- 구분 포인트: `quiet`는 소리가 적음에 초점. `tranquil`은 소리뿐 아니라 마음과 분위기의 평온함까지 포함.
