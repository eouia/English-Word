# fid

- 출발 단어: `fidelity`
- 핵심 어원: Latin `fides` = 믿음, 신뢰, 충실함
- 형용사형: Latin `fidelis` = 믿을 만한, 충실한
- 핵심 이미지: 믿음을 지키는 충실함
- 관련 문서: [[cred]], [[trust]], [[vot]], [[spond]]
- 주의: [[cred]]는 "믿다, 신용하다, 인정하다" 쪽. `fid`는 "믿음, 충실함, 신의" 쪽.

## 출발 단어 분석

`fidelity` ← Latin `fidelitas`

- `fides`: 믿음, 신의
- `fidelis`: 충실한, 믿을 만한
- `fidelitas`: 충실함, 신의
- 현대 의미: 충실함, 신의, 정확한 재현도

`fidelity`는 "믿음을 지킴"에서 출발해, 사람 사이의 충실함뿐 아니라 소리·이미지·자료를 원본에 충실하게 재현하는 정도까지 확장.

## 단어들

### fidelity

- 구성: Latin `fidelitas`
- 뜻: 충실함, 신의, 정확도
- 어원적 연결: 믿음을 지키는 성질
- 사용 맥락: `high fidelity audio` = 원음에 충실한 오디오

### infidelity

- 구성: `in-` = not + `fidelity`
- 뜻: 불충실, 부정, 배신
- 어원적 연결: 믿음이나 약속을 지키지 않음
- 사용 맥락: `marital infidelity` = 결혼 관계에서의 부정

### faithful

- 구성: `faith` + `-ful`
- 뜻: 충실한, 신실한, 정확한
- 어원적 연결: 믿음을 가득 지니고 지킴
- 사용 맥락: `a faithful friend` = 충실한 친구

### faithfully

- 구성: `faithful` + `-ly`
- 뜻: 충실하게, 정확하게
- 어원적 연결: 믿음과 약속에 맞게 행동함
- 사용 맥락: `follow the instructions faithfully` = 지시를 충실히 따르다

### faith

- 구성: Old French `feid/foi` ← Latin `fides`
- 뜻: 믿음, 신앙, 신뢰
- 어원적 연결: 믿고 의지하는 마음
- 사용 맥락: `have faith in someone` = 누군가를 믿다

### bona fide

- 구성: Latin `bona fide` = in good faith
- 뜻: 진짜의, 선의의
- 어원적 연결: 좋은 믿음으로, 진심으로
- 사용 맥락: `a bona fide expert` = 진짜 전문가

### affidavit

- 구성: Medieval Latin `affidare` = 맹세하여 보증하다
- 뜻: 진술서, 선서 진술서
- 어원적 연결: 믿음과 맹세로 보증한 문서
- 사용 맥락: `sign an affidavit` = 선서 진술서에 서명하다

### fiduciary

- 구성: Latin `fiducia` = trust
- 뜻: 신탁의, 수탁자의, 신뢰 관계의
- 어원적 연결: 믿고 맡긴 재산이나 책임을 다루는 관계
- 사용 맥락: `fiduciary duty` = 신탁상 의무

### confident

- 구성: `con-` = with + Latin `fidere` = trust
- 뜻: 확신하는, 자신 있는
- 어원적 연결: 함께 믿고 의지함 → 확신을 가짐
- 사용 맥락: `feel confident` = 자신감을 느끼다

### confidence

- 구성: `confident` + `-ence`
- 뜻: 자신감, 신뢰, 확신
- 어원적 연결: 믿고 기대는 마음
- 사용 맥락: `gain confidence` = 자신감을 얻다

### confidential

- 구성: `confidence` + `-al`
- 뜻: 비밀의, 기밀의
- 어원적 연결: 믿고 맡겨 말한 것
- 사용 맥락: `confidential information` = 기밀 정보

### confide

- 구성: `con-` = with + `fidere` = trust
- 뜻: 털어놓다, 믿고 맡기다
- 어원적 연결: 믿고 마음속 말을 맡김
- 사용 맥락: `confide in a friend` = 친구에게 속마음을 털어놓다

### diffident

- 구성: `dis-` = away + `fidere` = trust
- 뜻: 자신 없는, 소심한
- 어원적 연결: 믿음이 떨어져 확신하지 못함
- 사용 맥락: `a diffident manner` = 자신 없어 보이는 태도

### perfidy

- 구성: Latin `perfidia` = faithlessness
- 뜻: 배신, 불충
- 어원적 연결: 믿음을 저버림
- 사용 맥락: `an act of perfidy` = 배신 행위

## 기억 포인트

`fid` 계열은 "믿음을 지킴" 이미지.

- `fidelity`: 믿음에 충실함
- `faithful`: 믿음을 지키는
- `bona fide`: 좋은 믿음으로 한, 진짜의
- `confide`: 믿고 털어놓음
- `confidence`: 믿고 기대는 확신
- `diffident`: 믿음이 부족해 자신 없음

## 비교 단어

### cred

- 뜻: 믿다, 신용하다, 인정하다
- 구분 포인트: [[cred]]는 "믿을 만하다고 판단하고 신용을 부여함"에 가까움. `fid`는 "믿음 자체와 그 믿음을 지키는 충실함"에 가까움.

### trust

- 뜻: 믿다, 신뢰하다
- 구분 포인트: [[trust]]는 게르만계 기본어. 일상적 신뢰와 참됨의 중심어.
