# hand

- 출발 단어: `offhand`
- 핵심 어원: Old English `hand` = 손
- 핵심 이미지: 손에 잡고 다룸 / 즉석에서 손에서 나옴
- 관련해서 헷갈릴 수 있는 어근: [[manu]] = 라틴어 손 계열, [[tract-handle]] = 다루다, 처리하다 계열

## 출발 단어 분석

`offhand`는 `off` + `hand`.

- `off`: 떨어져, 즉석의 느낌
- `hand`: 손

따라서 `offhand`는 원래 "손에서 바로 나온" 느낌. 충분히 생각하거나 준비하지 않고 바로 말하거나 행동함 → "즉석의, 무심한, 퉁명스러운"의 뜻.

## 단어들

### offhand

- 구성: `off` + `hand`
- 뜻: 즉석의, 준비 없는; 무심한, 퉁명스러운
- 어원적 연결: 손에서 바로 나와 깊이 생각하지 않은 태도
- 사용 맥락: `an offhand remark` = 무심코 한 말

### hand

- 구성: Old English `hand`
- 뜻: 손; 건네주다
- 어원적 연결: 잡고 다루거나 넘겨주는 신체 부위
- 사용 맥락: `hand in a report` = 보고서를 제출하다

### handy

- 구성: `hand` + `-y`
- 뜻: 편리한, 손재주 있는
- 어원적 연결: 손에 가까이 있거나 손으로 잘 다룰 수 있음
- 사용 맥락: `a handy tool` = 편리한 도구

### handle

- 구성: `hand` + `-le`
- 뜻: 손잡이; 다루다, 처리하다
- 어원적 연결: 손으로 잡거나 손에 쥐고 다룸
- 사용 맥락: `handle a problem` = 문제를 처리하다

### handful

- 구성: `hand` + `ful`
- 뜻: 한 줌; 다루기 힘든 사람/일
- 어원적 연결: 손에 가득 찰 만큼의 양. 비유적으로 손에 버거운 대상
- 사용 맥락: `a handful of people` = 소수의 사람들

## 기억 포인트

`hand` 계열은 "손에 잡고 다룸" 이미지.

- `offhand`: 손에서 바로 나온 → 즉석의, 무심한
- `handy`: 손 가까이에 있어 편리한
- `handle`: 손으로 잡고 다루다
- `handful`: 손에 가득한 양, 다루기 벅찬 것

## 비교 단어

### manual

- 뜻: 손의, 수동의
- 구분 포인트: [[manu]]는 라틴어 `manus` 계열. `hand`는 영어 기본어 계열

### tractable

- 뜻: 다루기 쉬운
- 구분 포인트: [[tract-handle]]은 끌고 다루는 라틴계 이미지. `handle`은 손으로 직접 잡고 처리하는 영어 기본어 이미지
