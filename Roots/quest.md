# quest

- 출발 단어: `quest`, `request`
- 핵심 어원: 라틴어 `quaerere` / `quaesit-` = 찾다, 묻다, 구하다
- 핵심 이미지: 답이나 대상을 찾아감
- 비교: [[quoth]] = 말하다, 선언하다 계열
- 비교: [[quer]] = 불평하다, 말다툼하다 계열

## 출발 단어 분석

`quest`, `request`는 [[quoth]] 계열의 `bequest`와 어원 다름.

- `quaerere`: 찾다, 묻다, 구하다
- `quaesitus`: 찾아진, 구해진

따라서 `quest`는 "찾아가는 일" → "탐색, 추구"의 뜻.  
`request`는 "다시/강하게 구함" → "요청"의 뜻.

## 단어들

### quest

- 구성: 라틴어 `quaerere` 계열
- 뜻: 탐색, 추구
- 어원적 연결: 무언가를 찾아 나서는 과정
- 사용 맥락: `a quest for knowledge` = 지식 탐구

### request

- 구성: `re-` + `quest`
- 뜻: 요청, 요청하다
- 어원적 연결: 무언가를 구하거나 청함
- 사용 맥락: `request information` = 정보를 요청하다

### question

- 구성: 라틴어 `quaestio`
- 뜻: 질문, 문제
- 어원적 연결: 답을 찾기 위해 묻는 것
- 사용 맥락: `answer a question` = 질문에 답하다

### questionable

- 구성: `question` + `-able`
- 뜻: 의심스러운, 논란의 여지가 있는
- 어원적 연결: 질문을 받을 수 있음 → 확실하지 않아 의심받음
- 사용 맥락: `a questionable decision` = 의심스러운 결정

### query

- 구성: 라틴어 `quaere` = 찾아라, 물어라
- 뜻: 질문, 문의
- 어원적 연결: 정보를 찾기 위한 질문
- 사용 맥락: `a search query` = 검색어, 검색 질문

### inquire

- 구성: `in-` + `quaerere`
- 뜻: 묻다, 조사하다
- 어원적 연결: 안으로 파고들며 찾음
- 사용 맥락: `inquire about the schedule` = 일정에 대해 문의하다

### inquiry

- 구성: `inquire` 계열 명사
- 뜻: 문의, 조사
- 어원적 연결: 답이나 사실을 찾는 과정
- 사용 맥락: `a public inquiry` = 공개 조사

### require

- 구성: `re-` + `quaerere`
- 뜻: 요구하다, 필요로 하다
- 어원적 연결: 필요한 것을 구함
- 사용 맥락: `require approval` = 승인을 요구하다

### acquire

- 구성: `ad-` + `quaerere`
- 뜻: 얻다, 습득하다
- 어원적 연결: 찾아서 자기 쪽으로 가져옴
- 사용 맥락: `acquire skills` = 기술을 습득하다

### exquisite

- 구성: 라틴어 `exquirere` = 철저히 찾아내다
- 뜻: 정교한, 매우 아름다운
- 어원적 연결: 세밀하게 골라낸 듯한 품질
- 사용 맥락: `exquisite detail` = 정교한 세부 묘사

## 비교 단어

### bequest

- 구성: 고대 영어 `be-` + `cweðan` 계열
- 뜻: 유증, 유산으로 남긴 것
- 어원적 연결: [[quest]] 계열 아님. [[quoth]] 계열
- 구분 포인트: `quest` = 찾음, `bequest` = 유언으로 남긴 것

### bequeath

- 구성: 고대 영어 `be-` + `cweðan`
- 뜻: 유언으로 남기다
- 어원적 연결: [[quest]] 계열 아님. [[quoth]] 계열
- 구분 포인트: `request` = 요청하다, `bequeath` = 유언으로 남기다

## 기억 포인트

`quest` 계열은 "찾고 묻고 구함" 이미지.

- `quest`: 찾아감
- `request`: 요청함
- `question`: 답을 찾기 위해 물음
- `questionable`: 질문받을 만큼 의심스러움
- `query`: 문의함
- `inquire`: 조사하며 물음
- `require`: 필요한 것을 요구함
- `acquire`: 찾아 얻음
- `exquisite`: 세밀히 골라낸 듯 정교함

비교:

- [[quoth]]: 말하다, 선언하다 계열
- `bequest`: [[quest]]처럼 보이지만 [[quoth]] 계열
- [[quer]]: 불평하다, 말다툼하다 계열. `query`와 `querulous`는 철자가 비슷하지만 어원과 의미 이미지가 다름
