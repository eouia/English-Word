# cand

- 출발 단어: `candid`
- 핵심 어원: Latin `candidus` / `candere` = white, bright, shine
- 핵심 이미지: 희고 밝아 숨김이 없음
- 관련 문서: [[alb]], [[luc]], [[clar]]
- 주의: `candid`의 "솔직한" 뜻은 원래 흰색·밝음 이미지에서 온 비유. 현대 형태에서 `can` + `did`처럼 나누지 않음.

## 출발 단어 분석

`candid` ← Latin `candidus`

- `candere`: to shine, be white
- `candidus`: white, bright, pure
- 원래 이미지: 흰 옷처럼 깨끗하고 밝게 드러남
- 의미 변화: 흰/밝은 → 순수한, 숨김없는 → 솔직한
- 현대 의미: 솔직한, 숨김없는; 자연스러운 사진의

## 단어들

### candid

- 발음: /ˈkændəd/
- 구성: Latin `candidus` = white, bright, pure
- 뜻: 솔직한, 숨김없는; 자연스러운
- 어원적 연결: 감추지 않고 밝게 드러난 상태
- 사용 맥락: `a candid answer` = 솔직한 대답

### candidly

- 발음: /ˈkændɪdli/
- 구성: `candid` + `-ly`
- 뜻: 솔직히, 숨김없이
- 어원적 연결: 어둡게 감추지 않고 밝게 말함
- 사용 맥락: `speak candidly` = 솔직히 말하다

### candor

- 발음: /ˈkændɚ/
- 구성: Latin `candor` = whiteness, brightness, openness
- 뜻: 솔직함, 공정함
- 어원적 연결: 흰색처럼 숨김없이 드러난 태도
- 사용 맥락: `with candor` = 솔직하게

### candour

- 구성: `candor`의 British spelling
- 뜻: 솔직함
- 어원적 연결: `candor`와 같음
- 사용 맥락: `political candour` = 정치적 솔직함

### candidate

- 발음: /ˈkændədeɪt/
- 구성: Latin `candidatus` = white-robed
- 뜻: 후보자
- 어원적 연결: 고대 로마 공직 후보자가 흰 토가를 입은 데서 유래
- 사용 맥락: `a presidential candidate` = 대통령 후보

### candidacy

- 발음: /ˈkændɪdəsi/
- 구성: `candidate` + `-acy`
- 뜻: 입후보, 후보 자격
- 어원적 연결: 후보자로 나선 상태
- 사용 맥락: `announce one's candidacy` = 출마를 선언하다

### candle

- 발음: /ˈkændəl/
- 구성: Latin `candela` = candle, light
- 뜻: 양초
- 어원적 연결: 빛나고 밝히는 물건
- 사용 맥락: `light a candle` = 양초에 불을 붙이다

### chandelier

- 발음: /ʃændəˈlɪr/
- 구성: French `chandelier` ← `chandelle` = candle
- 뜻: 샹들리에, 장식용 천장등
- 어원적 연결: 원래 촛불을 여러 개 꽂아 매단 등기구
- 사용 맥락: `a crystal chandelier` = 크리스털 샹들리에

### candelabra

- 발음: /ˌkændəˈlɑbrə/
- 구성: Latin `candelabrum` = candlestick
- 뜻: 가지가 여러 개인 촛대
- 어원적 연결: 여러 촛불을 받치는 장식용 촛대
- 사용 맥락: `silver candelabra` = 은제 다지 촛대

### incandescent

- 발음: /ˌɪnkəˈndɛsənt/
- 구성: `in-` + Latin `candescere` = become white-hot
- 뜻: 백열의, 강렬히 빛나는
- 어원적 연결: 뜨거워져 희게 빛남
- 사용 맥락: `an incandescent bulb` = 백열전구

## 기억 포인트

`cand` 계열은 "희고 밝게 드러남" 이미지.

- `candid`: 솔직한, 숨김없는
- `candor`: 솔직함
- `candidate`: 흰 옷을 입은 후보자
- `candle`: 빛을 내는 양초
- `chandelier`: 촛불을 매달던 장식용 등
- `candelabra`: 여러 촛불을 받치는 촛대
- `incandescent`: 백열의, 강렬히 빛나는

## 비교 단어

### alb

- 뜻: 흰색, 흰 바탕
- 구분 포인트: [[alb]]는 흰색 자체나 흰 바탕. `cand`는 흰색에서 밝음·순수함·솔직함으로 확장.

### luc

- 뜻: 빛, 빛나다
- 구분 포인트: [[luc]]는 빛이 비쳐 명료해지는 이미지. `cand`는 희고 밝은 상태와 그 비유적 솔직함.

### frank

- 뜻: 솔직한
- 구분 포인트: `frank`는 더 직접적이고 거리낌 없는 말투. `candid`는 숨김없이 밝히는 공식적·정중한 느낌이 강함.
