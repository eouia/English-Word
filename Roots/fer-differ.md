# fer-differ

- 출발 단어: `differ`
- 핵심 어원: Latin `ferre` = 나르다, 가져가다
- 핵심 이미지: 서로 다른 방향으로 가져가 벌어짐
- 허브 문서: [[fer]]

## 단어들

### differ

- 발음: /ˈdɪfɚ/
- 구성: `dis-` = apart + `fer`
- 뜻: 다르다, 의견이 갈리다
- 어원적 연결: 서로 다른 방향으로 나뉘어 감
- 사용 맥락: `opinions differ` = 의견이 다르다

### different

- 발음: /ˈdɪfɚənt/
- 구성: `differ` + `-ent`
- 뜻: 다른, 차이가 나는
- 어원적 연결: 서로 떨어져 나뉘어 간 상태
- 사용 맥락: `a different approach` = 다른 접근법

### difference

- 발음: /ˈdɪfɚəns/
- 구성: `differ` + `-ence`
- 뜻: 차이, 다름
- 어원적 연결: 둘 이상이 서로 벌어져 생긴 차이
- 사용 맥락: `make a difference` = 변화를 만들다, 영향을 주다

### indifferent

- 발음: /ˌɪˈndɪfrənt/
- 구성: `in-` = not + `different`
- 뜻: 무관심한, 대수롭지 않게 여기는
- 어원적 연결: 차이를 두지 않음 → 특별히 신경 쓰지 않음
- 사용 맥락: `indifferent to criticism` = 비판에 무관심한

### defer

- 발음: /dɪˈfɝ/
- 구성: `de-` = away, down + `fer`
- 뜻: 연기하다, 따르다
- 어원적 연결: 일을 다른 때로 옮김. 판단을 상대에게 넘김
- 사용 맥락: `defer a decision` = 결정을 미루다

### prefer

- 발음: /prəˈfɝ/
- 구성: `pre-` = before + `fer`
- 뜻: 더 좋아하다, 선호하다
- 어원적 연결: 어떤 것을 앞에 가져다 놓음
- 사용 맥락: `prefer tea to coffee` = 커피보다 차를 선호하다

### preference

- 발음: /ˈprɛfɚəns/
- 구성: `prefer` + `-ence`
- 뜻: 선호, 우선권
- 어원적 연결: 앞에 가져다 놓는 선택
- 사용 맥락: `personal preference` = 개인적 선호

## 기억 포인트

`fer-differ`는 "방향이 갈라짐 / 우선순위가 달라짐" 갈래.

- `differ`: 갈라져 다름
- `different`: 다른
- `indifferent`: 차이를 두지 않아 무관심함
- `defer`: 다른 때나 사람에게 넘김
- `prefer`: 앞에 가져다 놓아 선호함
