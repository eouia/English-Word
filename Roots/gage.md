# gage

- 출발 단어: `disengage`
- 핵심 어원: Old French `gage` = pledge, security; Germanic `wadja-` = pledge
- 핵심 이미지: 약속·담보로 묶고, 그 묶임을 풀거나 걸어 둠
- 관련 문서: [[mort]], [[trust]], [[borrow]]
- 주의: `gauge` = 재다, 측정 기준은 별도 어원으로 보는 편이 안전함. `engage`, `wage`, `wager`, `mortgage`의 `gage/wage`는 약속·담보 계열.

## 출발 단어 분석

`disengage` ← French `désengager` = release from a pledge or engagement

- `dis-`: apart, away, undo
- `en-`: in, into
- `gage`: pledge, security
- 원래 이미지: 담보·약속 안에 묶인 상태에서 풀어 냄
- 의미 변화: 약속이나 구속에서 풀다 → 얽힘·접촉·관여에서 떼어 내다
- 현대 의미: 풀다, 떼어 내다; 관여를 끊다, 빠져나오다

## 단어들

### disengage

- 발음: /dɪsɪˈngeɪdʒ/
- 구성: `dis-` = undo/away + `engage`
- 뜻: 풀다, 분리하다; 관여를 끊다
- 어원적 연결: 약속이나 얽힘에 묶인 상태에서 빠져나오게 함
- 사용 맥락: `disengage from the debate` = 논쟁에서 빠져나오다

### disengagement

- 발음: /dɪsɪˈngeɪdʒmənt/
- 구성: `disengage` + `-ment`
- 뜻: 이탈, 분리, 관여 중단
- 어원적 연결: 묶임이나 접촉에서 벗어나는 과정
- 사용 맥락: `military disengagement` = 군사적 철수·교전 중지

### engage

- 발음: /ɛˈngeɪdʒ/
- 구성: Old French `engager` = pledge, bind by pledge
- 뜻: 관여하다, 참여시키다; 약혼하다; 끌어들이다
- 어원적 연결: 약속이나 담보로 묶음. 이후 관심·일·관계에 묶여 들어감
- 사용 맥락: `engage in discussion` = 토론에 참여하다

### engagement

- 발음: /ɛˈngeɪdʒmənt/
- 구성: `engage` + `-ment`
- 뜻: 약혼, 약속, 참여, 교전
- 어원적 연결: 약속이나 관계로 묶인 상태
- 사용 맥락: `an engagement ring` = 약혼반지, `community engagement` = 지역사회 참여

### engaged

- 발음: /ɛˈngeɪdʒd/
- 구성: `engage` + `-ed`
- 뜻: 약혼한, 관여한, 바쁜
- 어원적 연결: 약속·관계·일에 묶여 있는 상태
- 사용 맥락: `be engaged to someone` = 누군가와 약혼해 있다

### gage

- 발음: /ˈgeɪdʒ/
- 구성: Old French `gage` = pledge, security
- 뜻: 담보, 보증물; 담보로 맡기다
- 어원적 연결: 약속 이행을 보장하기 위해 걸어 둔 것
- 사용 맥락: archaic/formal word. `throw down the gage` = 결투를 신청하다

### wage

- 발음: /ˈweɪdʒ/
- 구성: Anglo-French/Old North French `wage` = pledge, pay, reward
- 뜻: 임금; 벌이다
- 어원적 연결: 원래는 약속된 보상·지급. `wage war`는 큰일에 무엇을 걸고 벌임
- 사용 맥락: `monthly wages` = 월급, `wage a campaign` = 캠페인을 벌이다

### wager

- 발음: /ˈweɪdʒɚ/
- 구성: `wage` + `-er`
- 뜻: 내기, 걸다
- 어원적 연결: 돈이나 물건을 담보처럼 걸고 결과를 기다림
- 사용 맥락: `make a wager` = 내기를 하다

### mortgage

- 발음: /ˈmɔrgədʒ/
- 구성: Old French `mort gage` = dead pledge
- 뜻: 주택 담보 대출, 저당
- 어원적 연결: 빚을 갚으면 담보가 "죽고", 갚지 못하면 권리가 넘어가는 약속
- 사용 맥락: `pay off a mortgage` = 주택 대출을 갚다
- 관련: [[mort]]

### wed

- 발음: /ˈwɛd/
- 구성: Old English `weddian` = pledge, covenant
- 뜻: 결혼하다
- 어원적 연결: 원래 약속·서약으로 관계를 묶음
- 사용 맥락: `be wed in June` = 6월에 결혼하다

## 기억 포인트

`gage/wage` 계열은 "약속이나 담보로 묶음" 이미지.

- `engage`: 약속·관계·일에 묶여 들어감
- `engagement`: 약속이나 관여 상태
- `disengage`: 묶임에서 풀려남
- `gage`: 담보, 보증물
- `wager`: 내기로 걸어 둠
- `mortgage`: 죽은 담보 약속
- `wed`: 서약으로 결혼함

## 비교 단어

### mort

- 뜻: 죽음
- 구분 포인트: [[mort]]의 `mortgage`는 `mort` + `gage`. `mort`는 "죽은", `gage`는 "담보".

### trust

- 뜻: 믿고 맡김
- 구분 포인트: [[trust]]는 신뢰. `gage`는 신뢰를 보장하기 위해 실제 약속이나 담보를 걸어 둠.

### gauge

- 뜻: 재다, 측정 기준
- 구분 포인트: `gauge`는 측정 기준·계량 계열. `engage/disengage`의 `gage` = pledge와 구분.
