# port

- 출발 단어: `transport`
- 핵심 어원: 라틴어 `portare` = 나르다, 운반하다
- 핵심 이미지: 짐을 들고 옮김
- 관련해서 헷갈릴 수 있는 어근: [[fer]] = 나르다, 지니다 계열, [[mit]] = 보내다 계열, [[gest]] = 나르다, 처리하다 계열

## 출발 단어 분석

`transport`는 `trans-` + `portare`.

- `trans-`: across
- `portare`: 나르다, 운반하다

따라서 `transport`는 "건너 운반하다" → "수송하다, 이동시키다"의 뜻.

## 단어들

### transport

- 발음: n. /ˈtrænspɔrt/, v. /træˈnspɔrt/
- 구성: `trans-` = across + `port`
- 뜻: 수송하다, 운송, 교통
- 어원적 연결: 한쪽에서 다른 쪽으로 실어 나름
- 사용 맥락: `public transport` = 대중교통

### transportation

- 발음: /ˌtrænspɚˈteɪʃən/
- 구성: `transport` + `-ation`
- 뜻: 수송, 교통
- 어원적 연결: 사람이나 물건을 옮겨 나르는 체계
- 사용 맥락: `transportation costs` = 운송비

### import

- 발음: n. /ˈɪˌmpɔrt/, v. /ˌɪˈmpɔrt/
- 구성: `in-` = into + `port`
- 뜻: 수입하다, 수입품
- 어원적 연결: 안으로 실어 들임
- 사용 맥락: `import goods` = 상품을 수입하다

### export

- 발음: n. /ˈɛkspɔrt/, v. /ɪˈkspɔrt/
- 구성: `ex-` = out + `port`
- 뜻: 수출하다, 수출품
- 어원적 연결: 밖으로 실어 보냄
- 사용 맥락: `export cars` = 자동차를 수출하다

### portable

- 발음: /ˈpɔrtəbəl/
- 구성: `port` + `-able`
- 뜻: 휴대용의, 들고 다닐 수 있는
- 어원적 연결: 옮겨 나를 수 있음
- 사용 맥락: `a portable charger` = 휴대용 충전기

### porter

- 발음: /ˈpɔrtɚ/
- 구성: `port` + `-er`
- 뜻: 짐꾼, 운반인
- 어원적 연결: 짐을 나르는 사람
- 사용 맥락: `a hotel porter` = 호텔 짐꾼

### portfolio

- 발음: /pɔˈrtfoʊliˌoʊ/
- 구성: Italian `portafoglio` = carry leaves
- 뜻: 포트폴리오, 자산 구성
- 어원적 연결: 종이나 자료를 들고 다니는 묶음
- 사용 맥락: `investment portfolio` = 투자 포트폴리오

### report

- 발음: /riˈpɔrt/
- 구성: `re-` = back + `port`
- 뜻: 보고하다, 보고서
- 어원적 연결: 정보를 다시 가져와 전달함
- 사용 맥락: `submit a report` = 보고서를 제출하다

### support

- 발음: /səˈpɔrt/
- 구성: `sub-` = under + `port`
- 뜻: 지지하다, 지원하다
- 어원적 연결: 아래에서 받쳐 나름
- 사용 맥락: `support the proposal` = 제안을 지지하다

### deport

- 발음: /dɪˈpɔrt/
- 구성: `de-` = away + `port`
- 뜻: 추방하다
- 어원적 연결: 밖으로 실어 보내 버림
- 사용 맥락: `deport illegal residents` = 불법 체류자를 추방하다

### comport

- 발음: /kəˈmpɔrt/
- 구성: `com-` = together + `port`
- 뜻: 처신하다, 어울리다
- 어원적 연결: 자신을 어떤 방식으로 가지고 다님
- 사용 맥락: `comport oneself with dignity` = 품위 있게 처신하다

### important

- 발음: /ˌɪˈmpɔrtənt/
- 구성: `in-` = into + `port`
- 뜻: 중요한
- 어원적 연결: 무게를 안으로 가져 지님. 영향이나 의미를 실어 나르는 느낌
- 사용 맥락: `an important issue` = 중요한 문제

### importance

- 발음: /ˌɪˈmpɔrtəns/
- 구성: `important` + `-ance`
- 뜻: 중요성
- 어원적 연결: 어떤 일이나 대상이 지닌 무게와 영향
- 사용 맥락: `the importance of education` = 교육의 중요성

### importantly

- 발음: /ˌɪˈmpɔrtəntli/
- 구성: `important` + `-ly`
- 뜻: 중요하게, 중요한 점은
- 어원적 연결: 무게 있는 의미를 가지고
- 사용 맥락: `More importantly, ...` = 더 중요한 것은 ...

### unimportant

- 발음: /ənɪˈmpɔrtənt/
- 구성: `un-` = not + `important`
- 뜻: 중요하지 않은, 사소한
- 어원적 연결: 의미나 영향의 무게가 크지 않음
- 사용 맥락: `an unimportant detail` = 중요하지 않은 세부 사항

### unimportance

- 구성: `un-` = not + `importance`
- 뜻: 중요하지 않음, 사소함
- 어원적 연결: 무게나 영향이 거의 없는 상태
- 사용 맥락: 문어적 표현. 일상에서는 `lack of importance`가 더 자연스러움

## 기억 포인트

`port` 계열은 "들고 옮김" 이미지.

- `transport`: 건너 운반함
- `import`: 안으로 들여옴
- `export`: 밖으로 내보냄
- `portable`: 들고 다닐 수 있음
- `report`: 정보를 다시 가져와 전함
- `support`: 아래에서 받쳐 줌
- `important`: 의미의 무게를 지님
- `unimportant`: 의미의 무게가 크지 않음

## 비교 단어

### fer

- 뜻: 나르다, 지니다
- 구분 포인트: [[fer]]는 추상적 "가져가다/지니다"까지 넓음. `port`는 짐을 실제로 들고 옮기는 이미지가 더 선명함

### mit

- 뜻: 보내다
- 구분 포인트: [[mit]]는 목적지로 보내는 행위. `port`는 운반해 옮기는 행위
