# nutri

- 출발 단어: `nutrition`
- 핵심 어원: Latin `nutrire` = 먹여 기르다, 양육하다
- 핵심 이미지: 먹이고 길러 성장하게 함
- 관련 문서: [[mal]], [[vit]]

## 출발 단어 분석

`nutrition` ← Latin `nutrire`

- `nutrire`: nourish, feed
- 원래 이미지: 먹여서 생명을 유지하고 자라게 함
- 현대 의미: 영양, 영양 공급

## 단어들

### nutrition

- 발음: /nuˈtrɪʃən/
- 구성: Latin `nutrire` + `-ion`
- 뜻: 영양, 영양 섭취
- 어원적 연결: 몸을 먹여 기르는 과정
- 사용 맥락: `proper nutrition` = 적절한 영양

### nutrient

- 발음: /ˈnutriənt/
- 구성: Latin `nutriens` = nourishing
- 뜻: 영양소
- 어원적 연결: 몸을 기르는 물질
- 사용 맥락: `essential nutrients` = 필수 영양소

### nutritious

- 발음: /nuˈtrɪʃəs/
- 구성: `nutrition` + `-ous`
- 뜻: 영양가 있는
- 어원적 연결: 몸을 잘 기를 수 있음
- 사용 맥락: `nutritious food` = 영양가 있는 음식

### malnutrition

- 발음: /ˌmælnuˈtrɪʃən/
- 구성: [[mal]] = bad + `nutrition`
- 뜻: 영양실조
- 어원적 연결: 영양 공급이 나쁘거나 부족함
- 사용 맥락: `child malnutrition` = 아동 영양실조

### nourish

- 발음: /ˈnɝɪʃ/
- 구성: Old French `nourrir` ← Latin `nutrire`
- 뜻: 영양을 공급하다, 기르다
- 어원적 연결: 먹여서 생명과 성장을 유지함
- 사용 맥락: `nourish the body` = 몸에 영양을 공급하다

### nurture

- 발음: /ˈnɝtʃɚ/
- 구성: Old French `nourriture` ← Latin `nutrire`
- 뜻: 양육하다, 키우다; 양육
- 어원적 연결: 먹이고 돌보아 성장하게 함
- 사용 맥락: `nurture talent` = 재능을 기르다

## 기억 포인트

`nutri` 계열은 "먹여 기름" 이미지.

- `nutrition`: 영양 공급
- `nutrient`: 영양소
- `nutritious`: 영양가 있는
- `malnutrition`: 나쁜 영양 상태
- `nourish`: 영양을 주다
- `nurture`: 돌보아 키우다
