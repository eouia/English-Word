# narr

- 출발 단어: `narrative`
- 핵심 어원: 라틴어 `narrare` = 말하다, 이야기하다, 설명하다
- 핵심 의미: 사건을 순서 있게 말해 알게 함
- 관련 문서: [[fari]], [[dict]], [[logos]], [[cogn]]
- 주의: `narrare`는 더 깊게 `gnarus` = 알고 있는, 익숙한 과 연결. “알게 만들다” 이미지.

## 출발 단어 분석

`narrative` = `narrat-` + `-ive`

- `narrare`: 이야기하다, 설명하다
- `-ive`: ~하는 성질의
- 원래 이미지: 사건을 말로 풀어 상대가 알게 함
- 현대 의미: 이야기, 서사, 서술 방식

## 단어들

### narrative

- 발음: /ˈnærətɪv/
- 구성: `narrat-` + `-ive`
- 뜻: 이야기, 서사, 서술의
- 어원적 연결: 사건을 순서 있게 말해 알게 하는 방식
- 사용 맥락: `a historical narrative` = 역사적 서사

### narrate

- 발음: /ˈnɛˌreɪt/
- 구성: `narr` + `-ate`
- 뜻: 이야기하다, 서술하다
- 어원적 연결: 사건을 말로 풀어 전달함
- 사용 맥락: `narrate a story` = 이야기를 서술하다

### narration

- 발음: /nɛˈreɪʃən/
- 구성: `narrate` + `-ion`
- 뜻: 서술, 내레이션
- 어원적 연결: 이야기를 말로 풀어 가는 행위
- 사용 맥락: `first-person narration` = 1인칭 서술

### narrator

- 발음: /ˈnɛreɪtɚ/
- 구성: `narrate` + `-or`
- 뜻: 서술자, 해설자
- 어원적 연결: 사건을 말해 주는 사람
- 사용 맥락: `an unreliable narrator` = 믿기 어려운 서술자

### narrative structure

- 구성: `narrative` + `structure`
- 뜻: 서사 구조
- 어원적 연결: 사건을 어떤 순서와 틀로 말하는가
- 사용 맥락: `analyze the narrative structure` = 서사 구조를 분석하다

### narratology

- 구성: `narrat-` + Greek `-logy` = study
- 뜻: 서사학
- 어원적 연결: 이야기의 구조와 방식에 대한 학문
- 사용 맥락: `modern narratology` = 현대 서사학

## 기억 포인트

- `narr`는 “이야기해서 알게 함”.
- `narrative`는 단순한 `story`보다 “사건을 배열하고 해석하는 서술 방식” 느낌이 강함.
- 시험 지문에서는 역사, 문학, 미디어, 사회 담론의 “서사/프레임” 뜻으로 자주 등장.

## 비교 단어

### story

- `story`는 라틴어 `historia` 계열.
- 의미는 가깝지만 `narr` 계열은 아님.

### novel

- `novel`은 라틴어 `novus` = new 계열.
- 이야기 장르지만 어원은 `narr`와 다름.

### cogn

- [[cogn]]은 “알다, 인식하다” 계열.
- `narr`는 더 깊은 층에서 “알고 있음/알게 함”과 연결되지만, 영어 학습에서는 이야기 계열로 별도 관리.
