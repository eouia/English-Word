# sect

- 출발 단어: `section`
- 핵심 어원: Latin `secare`, past participle `sectus` = 자르다
- 핵심 이미지: 잘라 나누기, 가로질러 끊기
- 관련 문서: [[cid]], [[scind]], [[circ]], [[angl]]
- 주의: [[cid]]는 Latin `caedere` = 자르다, 죽이다 계열. `sect`는 Latin `secare` = 자르다 계열.

## 출발 단어 분석

`section` = Latin `sectio` ← `secare`

- `secare`: cut, cut off
- `sectio`: 자름, 잘라낸 부분, 나눈 구역
- 현대 의미: 부분, 구역, 절, 단면

## 단어들

### section

- 발음: /ˈsɛkʃən/
- 구성: Latin `sectio` = cutting, division
- 뜻: 부분, 구역, 절, 단면
- 어원적 연결: 잘라 나눈 한 부분
- 사용 맥락: `a section of the report` = 보고서의 한 절

### sector

- 발음: /ˈsɛktɚ/
- 구성: Latin `sector` = cutter, section of a circle
- 뜻: 부문, 분야, 부채꼴 영역
- 어원적 연결: 원이나 사회를 잘라 나눈 구역
- 사용 맥락: `the private sector` = 민간 부문

### segment

- 발음: /ˈsɛgmənt/
- 구성: Latin `segmentum` ← `secare`
- 뜻: 부분, 조각, 구간
- 어원적 연결: 잘라낸 한 조각
- 사용 맥락: `a market segment` = 시장 세분 구역

### sectional

- 발음: /ˈsɛkʃənəl/
- 구성: `section` + `-al`
- 뜻: 부분적인, 지역적인, 단면의
- 어원적 연결: 전체를 나눈 한 부분에 속함
- 사용 맥락: `sectional interests` = 지역·부문 이익

### bisect

- 구성: `bi-` = two + `sect`
- 뜻: 이등분하다
- 어원적 연결: 둘로 자름
- 사용 맥락: `bisect an angle` = 각을 이등분하다

### dissect

- 발음: /daɪˈsɛkt/
- 구성: `dis-` = apart + `sect`
- 뜻: 해부하다, 세밀히 분석하다
- 어원적 연결: 잘라 따로 나누어 살핌
- 사용 맥락: `dissect an argument` = 논증을 세밀히 분석하다

### dissection

- 발음: /daɪˈsɛkʃən/
- 구성: `dissect` + `-ion`
- 뜻: 해부, 세밀한 분석
- 어원적 연결: 잘라 나누는 행위
- 사용 맥락: `a careful dissection of the text` = 글의 세밀한 분석

### insect

- 발음: /ˈɪˌnsɛkt/
- 구성: Latin `insectum` ← `in-` = into + `secare`
- 뜻: 곤충
- 어원적 연결: 몸이 마디로 잘려 나뉜 듯 보임
- 사용 맥락: `insect repellent` = 방충제

### intersection

- 발음: /ˌɪntɚˈsɛkʃən/
- 구성: `inter-` = between + `section`
- 뜻: 교차점, 교집합
- 어원적 연결: 선들이 서로 가로질러 자르며 만나는 곳
- 사용 맥락: `a busy intersection` = 붐비는 교차로

### intersect

- 발음: /ˌɪntɚˈsɛkt/
- 구성: `inter-` = between + `sect`
- 뜻: 교차하다, 가로지르다
- 어원적 연결: 서로 사이를 지나며 자름
- 사용 맥락: `two lines intersect` = 두 선이 교차함

### secant

- 구성: Latin `secans`, present participle of `secare`
- 뜻: 할선
- 어원적 연결: 원을 잘라 지나가는 선
- 사용 맥락: 기하·삼각함수 맥락

### transect

- 발음: /ˈtræˌnsɛkt/
- 구성: `trans-` = across + `sect`
- 뜻: 가로지르다, 횡단 절단하다
- 어원적 연결: 가로질러 자름
- 사용 맥락: 생태 조사·해부 맥락

### resect

- 구성: `re-` = back + `sect`
- 뜻: 절제하다, 잘라내다
- 어원적 연결: 몸의 일부를 뒤로 잘라 제거함
- 사용 맥락: 의학 맥락

### sect

- 발음: /ˈsɛkt/
- 구성: 전통적으로 Latin `secta`와 연결
- 뜻: 종파, 분파
- 어원적 연결: 큰 집단에서 갈라져 나온 무리로 이해 가능. 다만 `sequi` = follow 계열과의 역사적 혼동이 있음
- 사용 맥락: `a religious sect` = 종교 분파

## 기억 포인트

`sect` 계열은 "잘라 나누기" 이미지.

- `section`: 잘라 나눈 부분
- `sector`: 잘라 나눈 부문
- `segment`: 잘라낸 조각
- `bisect`: 둘로 자름
- `dissect`: 갈라 자르고 분석함
- `insect`: 몸이 마디로 나뉜 생물
- `intersection`: 선들이 서로 자르며 만나는 곳
