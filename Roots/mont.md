# mont

- 출발 단어: `paramount`
- 핵심 어원: Latin `mons`, `montem` = mountain
- 핵심 이미지: 산처럼 솟아오름, 위에 있음
- 관련 문서: [[alt]], [[super]], [[ultra]]

## 단어들

### paramount

- 발음: /ˈpɛrəˌmaʊnt/
- 구성: Anglo-French `paramont` = above, superior < Old French `par` + `amont`
- 뜻: 최고의, 가장 중요한, 우선적인
- 어원적 연결: `amont`는 Latin `ad montem` = to the mountain, upward에서 옴. "위에 있음"에서 최고 지위나 중요성으로 확장
- 사용 맥락: `of paramount importance` = 가장 중요한

### paramountcy

- 구성: `paramount` + `-cy`
- 뜻: 최고 지위, 우월성
- 어원적 연결: 다른 것들보다 위에 있는 상태
- 사용 맥락: 정치, 법, 역사 문맥에서 최고 권위나 우월성을 말할 때 사용

### mount

- 발음: /ˈmaʊnt/
- 구성: Old French `monter` < Latin `mons`, `montem`
- 뜻: 오르다, 올라타다; 설치하다
- 어원적 연결: 산이나 높은 곳으로 올라감
- 사용 맥락: `mount the stairs` = 계단을 오르다

### mountain

- 발음: /ˈmaʊntən/
- 구성: Old French `montaigne` < Latin `montanea`
- 뜻: 산
- 어원적 연결: 높이 솟은 지형
- 사용 맥락: `climb a mountain` = 산을 오르다

### amount

- 발음: /əˈmaʊnt/
- 구성: Old French `amonter` = go up, rise, amount to
- 뜻: 총액, 양; ~에 달하다
- 어원적 연결: 수나 가치가 위로 올라가 어떤 총량에 이름
- 사용 맥락: `the total amount` = 총액

### surmount

- 발음: /sɚˈmaʊnt/
- 구성: `sur-` = over, above + `mount`
- 뜻: 극복하다, 넘다
- 어원적 연결: 산이나 장애물 위로 올라 넘어감
- 사용 맥락: `surmount obstacles` = 장애물을 극복하다

### paramount importance

- 구성: `paramount` + `importance`
- 뜻: 가장 중요한 것, 최우선 중요성
- 어원적 연결: 중요도의 꼭대기에 있음
- 사용 맥락: `safety is of paramount importance` = 안전이 가장 중요하다

## 기억 포인트

`mont` 계열은 "산, 위로 오름, 꼭대기" 이미지.

- `paramount`: 가장 위에 있는 → 최고의, 가장 중요한
- `mount`: 오르다
- `mountain`: 산
- `amount`: 위로 올라 총량에 이르다
- `surmount`: 위로 넘어 극복하다

## 비교 단어

### para

- 뜻: Greek `para-` = 옆에, 곁에, 넘어서, 반대로
- 구분 포인트: `paramount`의 `para-`처럼 보이는 부분은 기존 [[para]]의 Greek 접두사가 아님. Anglo-French `paramont` 전체로 기억.

### alt

- 뜻: 높다, 높이다
- 구분 포인트: [[alt]]는 Latin `altus` = high 계열. `mont`는 산과 오름 이미지.

### over

- 뜻: 위로, 넘어서
- 구분 포인트: `over`는 영어 접두사. `paramount`, `surmount`는 French/Latin 계열의 "위, 산" 이미지.
