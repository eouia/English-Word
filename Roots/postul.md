# postul

- 출발 단어: `postulate`
- 핵심 어원: Latin `postulare` = ask, demand, claim, require
- 핵심 이미지: 어떤 것을 요구하거나, 증명 없이 받아들이라고 제시함
- 관련 문서: [[pet]], [[pos]]

## 단어들

### postulate

- 발음: /ˈpɑstʃəˌleɪt/
- 구성: Latin `postulare` = ask, demand
- 뜻: 가정하다, 전제로 삼다; 공준, 가정
- 어원적 연결: 논의나 증명을 시작하기 위해 받아들일 것을 요구함
- 사용 맥락: `postulate a cause` = 원인을 가정하다

### postulated

- 발음: /ˈpɑstʃəˌleɪtɪd/
- 구성: `postulate` + `-ed`
- 뜻: 가정된, 전제로 세워진
- 어원적 연결: 논의의 출발점으로 요구되어 놓인 상태
- 사용 맥락: `a postulated mechanism` = 가정된 메커니즘

### postulating

- 구성: `postulate` + `-ing`
- 뜻: 가정하는 것, 전제로 삼는 것
- 어원적 연결: 어떤 명제를 받아들이라고 제시하는 행위
- 사용 맥락: `postulating hidden variables` = 숨은 변수를 가정함

### postulation

- 구성: `postulate` + `-ion`
- 뜻: 가정, 전제 설정
- 어원적 연결: 증명 전에 받아들일 것을 요구하는 행위
- 사용 맥락: 학술적 문맥에서 전제 설정을 말할 때 사용

### postulant

- 구성: Latin `postulans` = one who asks
- 뜻: 지원자, 청원자; 수도회 입회 지원자
- 어원적 연결: 받아들여 달라고 요청하는 사람
- 사용 맥락: 종교·격식 문맥에서 주로 사용

### expostulate

- 구성: `ex-` = out, from + `postulare` = demand
- 뜻: 항의하다, 따지다
- 어원적 연결: 이유나 설명을 강하게 요구함
- 사용 맥락: `expostulate with a friend` = 친구에게 따져 말하다

### expostulation

- 구성: `expostulate` + `-ion`
- 뜻: 항의, 이의 제기
- 어원적 연결: 밖으로 꺼내 강하게 요구하고 따지는 행위
- 사용 맥락: 격식 있는 문체에서 항의나 반박을 말할 때 사용

## 기억 포인트

`postul` 계열은 "요구하다, 전제로 받아들이게 하다" 이미지.

- `postulate`: 전제로 삼다, 공준
- `postulation`: 가정, 전제 설정
- `postulant`: 받아들여 달라고 요청하는 사람
- `expostulate`: 따지며 항의하다

## 비교 단어

### posit

- 뜻: 놓다, 전제로 두다
- 구분 포인트: [[pos]] 계열. `posit`은 "놓다" 이미지가 강하고, `postulate`는 "받아들이라고 요구하다" 이미지가 강함.

### petition

- 뜻: 청원하다, 청원
- 구분 포인트: [[pet]] 계열. 둘 다 요구의 느낌이 있지만, `petition`은 공식 요청, `postulate`는 논리·학문에서 전제로 요구하는 명제.
