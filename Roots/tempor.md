# tempor

- 출발 단어: `temporary`
- 핵심 어원: Latin `tempus` / `temporis` = 시간
- 핵심 이미지: 시간 속에 놓인 것, 일정 기간만 지속됨
- 관련 문서: [[recent]], [[moon]], [[tend]]
- 주의: `temper` = 조절하다, 누그러뜨리다는 Latin `temperare` 계열. `tempor`와 섞지 않음.

## 출발 단어 분석

`temporary` ← Latin `temporarius`

- `tempus`: 시간
- `temporarius`: 시간에 속한, 잠시의
- 원래 이미지: 영구적이지 않고 일정 시간에만 해당함
- 현대 의미: 일시적인, 임시의

## 단어들

### temporary

- 구성: Latin `temporarius`
- 뜻: 일시적인, 임시의
- 어원적 연결: 일정 시간 동안만 지속됨
- 사용 맥락: `a temporary solution` = 임시 해결책

### temporarily

- 구성: `temporary` + `-ly`
- 뜻: 일시적으로
- 어원적 연결: 제한된 시간 동안만
- 사용 맥락: `temporarily closed` = 임시 휴업

### temporal

- 구성: Latin `temporalis`
- 뜻: 시간의, 세속의
- 어원적 연결: 영원한 것이 아니라 시간 속에 속함
- 사용 맥락: `temporal order` = 시간적 순서

### contemporary

- 구성: `con-` = together + `tempor` = time
- 뜻: 동시대의, 현대의
- 어원적 연결: 같은 시간에 함께 존재함
- 사용 맥락: `contemporary art` = 현대 미술

### contemporaneous

- 구성: `con-` + `tempor` + `-aneous`
- 뜻: 동시에 일어난, 같은 시대의
- 어원적 연결: 같은 시간대에 함께 발생함
- 사용 맥락: `contemporaneous records` = 동시대 기록

### tempo

- 구성: Italian `tempo` ← Latin `tempus`
- 뜻: 박자, 속도
- 어원적 연결: 음악에서 시간의 흐름과 빠르기
- 사용 맥락: `a fast tempo` = 빠른 박자

### extemporaneous

- 구성: Latin `ex tempore` = out of the moment
- 뜻: 즉석의, 즉흥적인
- 어원적 연결: 그 순간의 시간에서 바로 나옴
- 사용 맥락: `an extemporaneous speech` = 즉흥 연설

### extempore

- 구성: Latin `ex tempore`
- 뜻: 즉석에서, 즉흥적으로
- 어원적 연결: 미리 준비한 것이 아니라 그 시간에 맞춰 나옴
- 사용 맥락: `speak extempore` = 즉석에서 말하다

## 기억 포인트

`tempor` 계열은 "시간" 이미지.

- `temporary`: 시간 제한이 있음
- `temporal`: 시간에 속함
- `contemporary`: 같은 시간대에 있음
- `tempo`: 시간의 빠르기

## 비교 단어

### recent

- 뜻: 최근의, 오래되지 않은
- 구분 포인트: [[recent]]는 "갓 생겨 오래되지 않음". `tempor`는 시간 자체나 시간 속 위치.

### temper

- 뜻: 조절하다, 누그러뜨리다, 기질
- 구분 포인트: Latin `temperare` = 알맞게 섞다, 조절하다 계열. `tempus`와 직접 같은 문서로 묶지 않음.
