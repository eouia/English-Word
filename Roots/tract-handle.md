# tract-handle

- 상위 문서: [[tract]]
- 핵심 어원: 라틴어 `trahere` / `tractare` = 끌다, 다루다
- 핵심 이미지: 끌고 다루다, 문제나 사람을 처리함
- 관련 갈래: [[tract-pull]], [[tract-trace]]

## 단어들

### intractable

- 구성: `in-` + `tractable`
- 뜻: 다루기 힘든, 고집 센
- 어원적 연결: 끌고 다루기 어려움
- 사용 맥락: `an intractable problem` = 다루기 어려운 문제

### tractable

- 구성: `tract` + `-able`
- 뜻: 다루기 쉬운, 유순한
- 어원적 연결: 원하는 방향으로 끌고 갈 수 있음
- 사용 맥락: `a tractable child` = 다루기 쉬운 아이

### treat

- 구성: Old French `traitier`, Latin `tractare`
- 뜻: 다루다, 대우하다, 치료하다, 한턱내다
- 어원적 연결: 원래 "끌고 다루다". 이후 문제를 다루고, 사람을 대하고, 병을 다룸
- 사용 맥락: `treat a patient` = 환자를 치료하다

### treatment

- 구성: `treat` + `-ment`
- 뜻: 대우, 치료, 처리
- 어원적 연결: 사람, 병, 문제를 다루는 방식
- 사용 맥락: `medical treatment` = 의학적 치료

### treaty

- 구성: Old French `traitie`, Latin `tractatus`
- 뜻: 조약, 협정
- 어원적 연결: 당사자들이 문제를 다루고 협상한 결과
- 사용 맥락: `sign a treaty` = 조약에 서명하다

### treatise

- 구성: Old French `traitis`, Latin `tractatus`
- 뜻: 논문, 논고
- 어원적 연결: 한 주제를 자세히 다룬 글
- 사용 맥락: `a philosophical treatise` = 철학 논고
