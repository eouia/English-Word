# fin

- 출발 단어: `final`
- 핵심 어원: Latin `finis` = 끝, 경계, 한계
- 핵심 이미지: 끝을 그어 범위를 정함
- 관련 문서: [[term]], [[recent]], [[cid]]
- 주의: `fine`은 "끝/완성/벌금" 쪽 의미가 섞여 발전했지만, 현대 뜻이 넓어 항목별로 조심해서 보기.

## 출발 단어 분석

`final` ← Latin `finalis` ← `finis`

- `finis`: 끝, 경계, 한계
- `finalis`: 끝에 속한
- 원래 이미지: 더 나아가지 않는 마지막 경계
- 현대 의미: 마지막의, 최종의

## 단어들

### final

- 구성: Latin `finalis`
- 뜻: 마지막의, 최종의
- 어원적 연결: 끝에 속함
- 사용 맥락: `the final decision` = 최종 결정

### finally

- 구성: `final` + `-ly`
- 뜻: 마침내, 최종적으로
- 어원적 연결: 끝에 이르러
- 사용 맥락: `finally arrive` = 마침내 도착하다

### finish

- 구성: Old French `finiss-` ← Latin `finire`
- 뜻: 끝내다, 마치다
- 어원적 연결: 끝의 경계에 이르게 함
- 사용 맥락: `finish the project` = 프로젝트를 끝내다

### finite

- 구성: Latin `finitus` ← `finire`
- 뜻: 유한한, 한정된
- 어원적 연결: 끝과 경계가 있음
- 사용 맥락: `finite resources` = 유한한 자원

### infinite

- 구성: `in-` = not + `finite`
- 뜻: 무한한
- 어원적 연결: 끝이나 한계가 없음
- 사용 맥락: `infinite possibilities` = 무한한 가능성

### define

- 구성: `de-` = completely + `finire` = set bounds
- 뜻: 정의하다, 경계를 정하다
- 어원적 연결: 의미의 경계를 분명히 그음
- 사용 맥락: `define a concept` = 개념을 정의하다

### definition

- 구성: `define` + `-ition`
- 뜻: 정의
- 어원적 연결: 말의 의미 경계를 정한 것
- 사용 맥락: `a clear definition` = 명확한 정의

### definite

- 구성: Latin `definitus`
- 뜻: 분명한, 확정된
- 어원적 연결: 경계가 명확히 정해짐
- 사용 맥락: `a definite answer` = 분명한 답

### indefinite

- 구성: `in-` = not + `definite`
- 뜻: 불명확한, 무기한의
- 어원적 연결: 경계나 끝이 정해지지 않음
- 사용 맥락: `an indefinite period` = 무기한

### confine

- 구성: `con-` = together + `finis` = boundary
- 뜻: 제한하다, 가두다
- 어원적 연결: 경계 안에 함께 묶어 둠
- 사용 맥락: `confine the discussion` = 논의를 제한하다

### confinement

- 구성: `confine` + `-ment`
- 뜻: 감금, 제한
- 어원적 연결: 경계 안에 갇힌 상태
- 사용 맥락: `solitary confinement` = 독방 감금

### refine

- 구성: `re-` + `fine`
- 뜻: 정제하다, 다듬다
- 어원적 연결: 끝까지 세밀하게 다듬어 완성도를 높임
- 사용 맥락: `refine a theory` = 이론을 다듬다

## 기억 포인트

`fin` 계열은 "끝과 경계" 이미지.

- `final`: 끝에 있는
- `finish`: 끝내다
- `finite`: 끝이 있음
- `define`: 의미의 경계를 정함
- `confine`: 경계 안에 가둠

## 비교 단어

### term

- 뜻: 끝점, 경계표, 기간
- 구분 포인트: [[term]]은 `terminal`, `terminate`, `determine`처럼 경계표지나 기간의 끝점 느낌. `fin`은 끝·한계를 그어 범위를 정하는 넓은 이미지.
