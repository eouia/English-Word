# patr

- 출발 단어: `paternal`
- 핵심 어원: Latin `pater` = 아버지
- 관련 형태: `patr-`, `pater-`
- 핵심 이미지: 아버지와 아버지 쪽 계통
- 관련 문서: [[matr]], [[frater]], [[nat]]

## 단어들

### paternal

- 발음: /pəˈtɝnəl/
- 구성: Latin `pater` + `-nal`
- 뜻: 아버지의, 부계의
- 어원적 연결: 아버지 쪽에 속함
- 사용 맥락: `paternal advice` = 아버지 같은 조언

### paternity

- 발음: /pəˈtɝnɪti/
- 구성: `pater` + `-ity`
- 뜻: 아버지임, 친자 관계
- 어원적 연결: 아버지의 신분이나 관계
- 사용 맥락: `paternity test` = 친자 확인 검사

### patriarch

- 발음: /ˈpeɪtriˌɑrk/
- 구성: `patri-` = father + Greek `archein` = rule
- 뜻: 가장, 족장
- 어원적 연결: 아버지처럼 집단을 지배하는 사람
- 사용 맥락: `family patriarch` = 집안의 가장

### patriarchy

- 발음: /ˈpeɪtriˌɑrki/
- 구성: `patriarch` + `-y`
- 뜻: 가부장제
- 어원적 연결: 아버지·남성 중심의 지배 구조
- 사용 맥락: `challenge patriarchy` = 가부장제에 도전하다

### patriot

- 발음: /ˈpeɪtriət/
- 구성: Greek `patrios` = of one's fathers
- 뜻: 애국자
- 어원적 연결: 조상의 땅, 아버지들의 나라를 사랑함
- 사용 맥락: `a devoted patriot` = 헌신적인 애국자

### expatriate

- 발음: /ɛˈkspeɪtriˌeɪt/
- 구성: `ex-` = out + Latin `patria` = fatherland
- 뜻: 국외 거주자, 국외로 이주하다
- 어원적 연결: 조국 밖으로 나감
- 사용 맥락: `an expatriate community` = 해외 거주자 공동체

## 기억 포인트

`patr` 계열은 "아버지와 조국" 이미지.

- `paternal`: 아버지의
- `paternity`: 아버지임
- `patriarch`: 가장
- `patriot`: 조국을 사랑하는 사람
- `expatriate`: 조국 밖에 사는 사람
