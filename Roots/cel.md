# cel

- 출발 단어: `conceal`
- 핵심 어원: Latin `celare` = hide, conceal
- 핵심 이미지: 보이지 않게 숨김
- 관련 문서: [[cover]], [[vel]], [[arcan]]
- 주의: 이 문서의 `cel`은 숨김 계열. `cell` = 작은 방, 세포는 별도 `cella` 계열로 구분하는 편이 안전함.

## 출발 단어 분석

`conceal` ← Old French `conceler` ← Latin `concelare`

- `con-`: completely, together
- `celare`: hide
- 원래 이미지: 완전히 숨겨 보이지 않게 함
- 현대 의미: 숨기다, 감추다

## 단어들

### conceal

- 구성: Latin `con-` = completely + `celare` = hide
- 뜻: 숨기다, 감추다
- 어원적 연결: 어떤 사실이나 물건을 보이지 않게 숨김
- 사용 맥락: `conceal the truth` = 진실을 숨기다

### concealed

- 구성: `conceal` + `-ed`
- 뜻: 숨겨진, 감춰진
- 어원적 연결: 밖으로 드러나지 않게 숨겨진 상태
- 사용 맥락: `a concealed weapon` = 숨겨진 무기

### concealing

- 구성: `conceal` + `-ing`
- 뜻: 숨기는, 감추는
- 어원적 연결: 보이지 않게 만드는 과정
- 사용 맥락: `concealing evidence` = 증거를 은폐함

### concealment

- 구성: `conceal` + `-ment`
- 뜻: 은폐, 숨김
- 어원적 연결: 숨겨 보이지 않게 하는 행위나 상태
- 사용 맥락: `deliberate concealment` = 고의적 은폐

### occult

- 구성: Latin `ob-` = over, against + `celare` = hide
- 뜻: 신비주의의, 비밀스러운; 숨기다
- 어원적 연결: 가려져 숨겨진 지식이나 현상
- 사용 맥락: `occult practices` = 신비주의 관행

### occultation

- 구성: `occult` + `-ation`
- 뜻: 엄폐, 가림; 천체가 다른 천체를 가림
- 어원적 연결: 한 물체가 다른 물체를 숨겨 보이지 않게 함
- 사용 맥락: `lunar occultation` = 달에 의한 엄폐

## 기억 포인트

`cel` 계열은 "숨겨 보이지 않게 함" 이미지.

- `conceal`: 숨기다
- `concealed`: 숨겨진
- `concealing`: 숨기는
- `concealment`: 은폐
- `occult`: 숨겨진, 신비주의의
- `occultation`: 엄폐, 가림

## 비교 단어

### cover

- 뜻: 덮다, 가리다
- 구분 포인트: [[cover]]는 위에서 덮는 이미지. `conceal`은 숨겨 보이지 않게 하는 동작 자체에 초점.

### vel

- 뜻: 베일, 베일로 가림
- 구분 포인트: [[vel]]은 베일을 씌우거나 걷어 드러내는 이미지. `conceal`은 꼭 덮개가 없어도 감추는 행위.

### arcan

- 뜻: 비밀스러운, 감춰진 지식
- 구분 포인트: [[arcan]]은 상자 안에 감춰진 비밀 지식. `cel`은 일반적인 숨김·은폐 동작.
