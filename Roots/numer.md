# numer

- 출발 단어: `number`
- 핵심 어원: 라틴어 `numerus` = 수, 숫자, 수량
- 핵심 의미: 세어진 수, 수량, 숫자로 표시함
- 관련 문서: [[sum]], [[metr]], [[ord]]
- 주의: `numb` = 감각이 없는. 어원이 달라 `number`와 분리.

## 출발 단어 분석

`number` ← 라틴어 `numerus`

- 원래 이미지: 세어서 얻은 수
- 의미 확장: 숫자 → 수량 → 번호를 붙이다
- 현대 핵심: 숫자, 수량, 번호

## 단어들

### number

- 구성: 라틴어 `numerus`
- 뜻: 숫자, 수, 번호
- 어원적 연결: 세어서 얻은 값
- 사용 맥락: `a phone number` = 전화번호

### numeral

- 구성: `numer` + `-al`
- 뜻: 숫자, 숫자를 나타내는 기호
- 어원적 연결: 수를 표시하는 글자나 기호
- 사용 맥락: `Roman numerals` = 로마 숫자

### numeric

- 구성: `numer` + `-ic`
- 뜻: 숫자의, 수치의
- 어원적 연결: 숫자로 표현되는 성질
- 사용 맥락: `numeric data` = 숫자 데이터

### numerical

- 구성: `numeric` + `-al`
- 뜻: 수치상의, 숫자로 된
- 어원적 연결: 수로 나타낸 정보
- 사용 맥락: `numerical analysis` = 수치 분석

### numerous

- 구성: `numer` + `-ous`
- 뜻: 많은, 다수의
- 어원적 연결: 셀 수가 많음
- 사용 맥락: `numerous examples` = 많은 예

### innumerable

- 구성: `in-` = not + `numerable`
- 뜻: 셀 수 없이 많은
- 어원적 연결: 숫자로 다 세기 어려울 만큼 많음
- 사용 맥락: `innumerable stars` = 셀 수 없이 많은 별

### enumerate

- 구성: `e-` = out + `numer` + `-ate`
- 뜻: 열거하다, 하나씩 세다
- 어원적 연결: 항목을 밖으로 꺼내어 번호처럼 세움
- 사용 맥락: `enumerate the reasons` = 이유를 하나씩 열거하다

### enumeration

- 구성: `enumerate` + `-ion`
- 뜻: 열거, 목록화
- 어원적 연결: 하나씩 세어 나열한 결과
- 사용 맥락: `a detailed enumeration` = 자세한 열거

### numerator

- 구성: `numer` + `-ator`
- 뜻: 분자
- 어원적 연결: 분수에서 위쪽에 적힌 수
- 사용 맥락: `the numerator of a fraction` = 분수의 분자

### denominator

- 구성: `de-` + `nominare` = 이름 붙이다
- 뜻: 분모
- 어원적 연결: `numer` 계열은 아니지만 `numerator`와 짝으로 학습
- 사용 맥락: `common denominator` = 공통분모

## 기억 포인트

- `numer`는 “세어진 수”가 핵심.
- `number`는 숫자와 번호 둘 다 가능.
- `numerous`는 “수가 많음”, `innumerable`은 “세어 번호 붙일 수 없을 만큼 많음”.
- `enumerate`는 시험 지문에서 “하나씩 열거하다”로 자주 등장.

## 비교 단어

### numb

- `numb`는 감각이 없는, 마비된.
- 고대 영어 계열로 `number`와 다른 어원.

### sum

- [[sum]]은 “합계, 최고점, 요약” 계열.
- 수와 관련되지만 `numerus` 계열은 아님.
