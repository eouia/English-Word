# sum

- 출발 단어: `sum`
- 핵심 어원: Latin `summa` = total, highest point
- 핵심 이미지: 위에 모인 전체, 합계, 핵심 요약
- 관련 어근: [[capit]], [[high]], [[sumere]]

## 출발 단어 분석

`sum`은 Old French `summe/somme`를 거쳐 Latin `summa`에서 온 말.

`summa`는 원래 "가장 높은 것, 맨 위"의 뜻에서 "전체, 총액, 핵심"으로 확장됨.

따라서 `sum`은 "맨 위에 모아 올린 전체" → "합계"의 뜻.

### sum

- 구성: Latin `summa`
- 의미: 합계, 총액
- 어원적 연결: 여러 값을 하나로 모은 전체
- 사용 맥락: `the sum of the numbers` = 숫자들의 합

### summary

- 구성: Latin `summarium` < `summa`
- 의미: 요약, 개요
- 어원적 연결: 전체 내용을 핵심만 모아 놓음
- 사용 맥락: `write a summary` = 요약문을 쓰다

### summarize

- 구성: `summary` + `-ize`
- 의미: 요약하다
- 어원적 연결: 긴 내용을 핵심 합계처럼 모음
- 사용 맥락: `summarize the article` = 기사를 요약하다

### summation

- 구성: `sum` + `-ation`
- 의미: 합산, 요약
- 어원적 연결: 여러 항목을 더해 하나로 모음
- 사용 맥락: `mathematical summation` = 수학적 합산

### summit

- 구성: Old French `sommet` < Latin `summum` = highest
- 의미: 정상, 정상 회담
- 어원적 연결: 가장 높은 지점
- 사용 맥락: `reach the summit` = 정상에 도달하다

### consummate

- 구성: `con-` = completely + Latin `summa`
- 의미: 완성하다, 완전한
- 어원적 연결: 모든 것을 합쳐 끝까지 완성함
- 사용 맥락: `a consummate professional` = 완벽한 전문가

## 기억 포인트

- `sum`: 합계
- `summary`: 핵심을 모은 요약
- `summit`: 가장 높은 지점
- `consummate`: 완성된, 완전한

## 비교 단어

- [[capit]]
  - 핵심: 머리, 꼭대기
  - 구분 포인트: `summit`은 가장 높은 지점. `capital`은 머리, 중심, 으뜸의 이미지

- [[high]]
  - 핵심: 높음
  - 구분 포인트: `high`는 일반적인 높음. `summit`은 꼭대기 지점

- [[sumere]]
  - 핵심: 취하다, 집어 들다
  - 구분 포인트: `summa`는 합계/최고점. `sumere`는 `assume`, `consume`, `resume`처럼 취하는 동작
