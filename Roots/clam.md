# clam

- 출발 단어: `proclaim`
- 핵심 어원: Latin `clamare` = cry out, shout, call
- 핵심 이미지: 큰소리로 외쳐 알림
- 관련 문서: [[voc]], [[call]]

## 단어들

### proclaim

- 발음: /proʊˈkleɪm/
- 구성: `pro-` = forward, forth + Latin `clamare` = cry out
- 뜻: 선언하다, 공표하다
- 어원적 연결: 앞으로 크게 외쳐 널리 알림
- 사용 맥락: `proclaim independence` = 독립을 선언하다

### proclaimed

- 발음: /proʊˈkleɪmd/
- 구성: `proclaim` + `-ed`
- 뜻: 선언된, 공표된
- 어원적 연결: 공개적으로 외쳐 알려진 상태
- 사용 맥락: `a proclaimed state of emergency` = 선포된 비상사태

### proclamation

- 발음: /ˌprɑkləˈmeɪʃən/
- 구성: `pro-` + `clam` + `-ation`
- 뜻: 선언, 선포, 포고문
- 어원적 연결: 공개적으로 외쳐 알리는 행위나 문서
- 사용 맥락: `a presidential proclamation` = 대통령 포고문

### claim

- 발음: /ˈkleɪm/
- 구성: Old French `clamer` < Latin `clamare`
- 뜻: 주장하다, 요구하다; 주장, 청구
- 어원적 연결: 권리나 사실을 큰소리로 내세움
- 사용 맥락: `claim responsibility` = 책임을 주장하다

### claimant

- 발음: /ˈkleɪmənt/
- 구성: `claim` + `-ant`
- 뜻: 청구인, 권리 주장자
- 어원적 연결: 자기 권리를 주장하는 사람
- 사용 맥락: `insurance claimant` = 보험 청구인

### acclaim

- 발음: /əˈkleɪm/
- 구성: `ad-` = to + `clamare`
- 뜻: 환호하다, 찬사를 보내다; 찬사
- 어원적 연결: 누군가를 향해 외쳐 칭찬함
- 사용 맥락: `critical acclaim` = 비평가들의 찬사

### exclaim

- 발음: /ɪˈkskleɪm/
- 구성: `ex-` = out + `clamare`
- 뜻: 외치다, 탄성을 지르다
- 어원적 연결: 소리가 밖으로 터져 나옴
- 사용 맥락: `exclaim in surprise` = 놀라서 외치다

### exclamation

- 발음: /ˌɛkskləˈmeɪʃən/
- 구성: `exclaim` + `-ation`
- 뜻: 외침, 감탄
- 어원적 연결: 밖으로 터져 나온 말
- 사용 맥락: `an exclamation mark` = 느낌표

### clamor

- 발음: /ˈklæmɚ/
- 구성: Latin `clamor` = shout, loud cry
- 뜻: 아우성, 소란; 강하게 요구하다
- 어원적 연결: 여럿이 큰소리로 외침
- 사용 맥락: `public clamor for reform` = 개혁을 요구하는 대중의 아우성

### declaim

- 발음: /dɪˈkleɪm/
- 구성: `de-` + `clamare`
- 뜻: 열변을 토하다, 웅변조로 말하다
- 어원적 연결: 큰소리로 공식적으로 말함
- 사용 맥락: `declaim against injustice` = 부정의에 대해 열변을 토하다

## 기억 포인트

`clam` 계열은 "외치다, 큰소리로 알리다" 이미지.

- `proclaim`: 앞으로 외쳐 공표함
- `claim`: 권리나 사실을 외쳐 주장함
- `acclaim`: 향해 외쳐 찬사 보냄
- `exclaim`: 밖으로 외침
- `clamor`: 큰소리의 소란

## 비교 단어

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 voice/call의 넓은 계열. `clam`은 큰소리로 외침에 초점.

### call

- 뜻: 부르다, 소집하다
- 구분 포인트: [[call]]은 영어 기본어. `clam`은 라틴계 시험 어휘에서 보임.

### laud

- 뜻: 칭찬하다, 찬양하다
- 구분 포인트: `acclaim`은 외쳐 찬사를 보내는 행위. `laud`는 칭찬 자체에 초점.
