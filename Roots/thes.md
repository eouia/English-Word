# thes

- 출발 단어: `hypothetical`
- 핵심 의미: 놓다, 세워 두다, 전제로 놓다
- 관련 형태: Greek `tithenai` = put, place; `thesis` = a placing, proposition
- 관련 문서: [[sym]], [[logos]]

## 핵심 이미지

`thes` 계열의 바탕은 "생각이나 대상을 어떤 자리에 놓음".

주장으로 놓은 것은 `thesis`.
그 아래에 임시로 놓은 전제는 `hypothesis`.
그 전제에 근거한 것은 `hypothetical`.
서로 반대로 놓인 주장은 `antithesis`.
여러 요소를 함께 놓으면 `synthesis`.

## 단어들

### hypothetical

- 구성: `hypo-` = under + `thesis` = placing, proposition + `-ical`
- 뜻: 가상의, 가정의, 가설적인
- 연결: 실제로 확정한 것이 아니라 아래에 임시 전제로 놓은 상태
- 사용 맥락: `a hypothetical question` = 가정적인 질문

### hypothesis

- 구성: Greek `hypo-` = under + `thesis` = placing
- 뜻: 가설, 가정
- 연결: 논의나 실험을 위해 밑에 깔아 두는 전제
- 사용 맥락: `test a hypothesis` = 가설을 검증하다

### hypothesize

- 구성: `hypothesis` + `-ize`
- 뜻: 가설을 세우다
- 연결: 설명을 위해 임시 전제를 놓음
- 사용 맥락: `Scientists hypothesize that...` = 과학자들은 ...라고 가설을 세운다

### thesis

- 구성: Greek `thesis` = placing, proposition
- 뜻: 논제, 학위 논문, 주장
- 연결: 토론이나 연구의 중심에 놓인 주장
- 사용 맥락: `write a thesis` = 논문을 쓰다

### antithesis

- 구성: `anti-` = against + `thesis`
- 뜻: 정반대, 대조되는 것
- 연결: 어떤 주장 맞은편에 놓인 반대 주장
- 사용 맥락: `the antithesis of freedom` = 자유의 정반대

### synthesis

- 구성: `syn-` = together + `thesis`
- 뜻: 종합, 합성
- 연결: 여러 요소를 함께 놓아 하나로 만듦
- 사용 맥락: `a synthesis of ideas` = 생각들의 종합

### synthetic

- 구성: `synthesis` + `-etic`
- 뜻: 합성의, 인조의
- 연결: 여러 요소를 함께 놓아 만들어 낸 것
- 사용 맥락: `synthetic fabric` = 합성 섬유

### parenthesis

- 구성: `para-` = beside + `en-` = in + `thesis`
- 뜻: 괄호, 삽입구
- 연결: 본문 옆이나 안에 덧붙여 놓은 말
- 사용 맥락: `put the phrase in parentheses` = 그 구를 괄호 안에 넣다

## 비교 단어

`theme`도 Greek `tithenai`와 연결되는 말로 볼 수 있지만, 현대 학습에서는 "주제, 테마"로 따로 익히는 편이 좋음.

`synthesis`는 [[sym]] 문서에도 있음.
그쪽은 `syn-` = 함께 접두사 중심, 이 문서는 `thesis` = 놓음 중심.
