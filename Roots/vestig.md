# vestig

- 출발 단어: `investigate`
- 핵심 어원: 라틴어 `vestigium` = footprint, track, trace
- 핵심 이미지: 발자국 / 흔적을 따라가며 밝힘
- 관련해서 헷갈릴 수 있는 어근: [[tract-trace]] = 끌고 지나간 선, 자국 계열, [[quest]] = 찾다, 묻다 계열

## 출발 단어 분석

`investigate`는 `in-` + `vestig` + `-ate`.

- `in-`: into
- `vestig`: footprint, trace
- `-ate`: 동사형 접미사

따라서 `investigate`는 원래 "흔적 안으로 들어가 따라가다"의 뜻. 여기서 "단서를 추적하다" → "조사하다, 수사하다"의 뜻.

## 단어들

### investigate

- 발음: /ˌɪˈnvɛstəˌgeɪt/
- 구성: `in-` = into + `vestig` + `-ate`
- 뜻: 조사하다, 수사하다
- 어원적 연결: 흔적을 따라 들어가 사실을 밝힘
- 사용 맥락: `investigate the cause` = 원인을 조사하다

### investigation

- 발음: /ˌɪˌnvɛstəˈgeɪʃən/
- 구성: `investigate` + `-ion`
- 뜻: 조사, 수사
- 어원적 연결: 단서와 흔적을 따라 사실을 밝히는 과정
- 사용 맥락: `a criminal investigation` = 범죄 수사

### investigator

- 발음: /ˌɪˈnvɛstəˌgeɪtɚ/
- 구성: `investigate` + `-or`
- 뜻: 조사관, 수사관
- 어원적 연결: 흔적을 추적하는 사람
- 사용 맥락: `a private investigator` = 사설 탐정

### investigative

- 발음: /ˌɪˈnvɛstəˌgeɪtɪv/
- 구성: `investigate` + `-ive`
- 뜻: 조사의, 탐사 보도의
- 어원적 연결: 숨은 사실을 추적해 밝히는 성격
- 사용 맥락: `investigative journalism` = 탐사 보도

### vestige

- 발음: /ˈvɛstɪdʒ/
- 구성: Latin `vestigium`
- 뜻: 흔적, 자취
- 어원적 연결: 남아 있는 발자국이나 흔적
- 사용 맥락: `a vestige of the past` = 과거의 흔적

### vestigial

- 발음: /vəˈstɪdʒiəl/
- 구성: `vestige` + `-ial`
- 뜻: 흔적만 남은, 퇴화한
- 어원적 연결: 예전 기능의 자취만 남음
- 사용 맥락: `vestigial organs` = 흔적 기관

## 기억 포인트

`vestig` 계열은 "발자국, 흔적" 이미지.

- `vestige`: 남은 흔적
- `vestigial`: 흔적만 남은
- `investigate`: 흔적을 따라 들어감 → 조사하다
- `investigation`: 흔적 추적의 과정 → 조사, 수사

## 비교 단어

### trace

- 뜻: 흔적, 추적하다
- 구분 포인트: [[tract-trace]] 계열. 의미는 가깝지만 `investigate`, `vestige`는 Latin `vestigium` = footprint 계열

### inquire

- 뜻: 묻다, 조사하다
- 구분 포인트: [[quest]] 계열. 질문을 통해 찾는 이미지가 강함. `investigate`는 단서와 흔적을 따라가는 이미지
