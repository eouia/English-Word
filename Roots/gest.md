# gest

- 출발 단어: `suggest`
- 핵심 어원: 라틴어 `gerere` / `gest-` = 나르다, 가져가다, 수행하다
- 핵심 이미지: 어떤 것을 가져와 드러내거나 처리함
- 관련해서 헷갈릴 수 있는 어근: `jest` = 농담 계열과 구분 필요

## 출발 단어 분석

`suggest`는 라틴어 `suggestus`에서 옴.

- `sub-`: under, from below
- `gest`: 가져오다, 나르다

따라서 `suggest`는 "아래에서 가져와 올리다" → "생각을 슬쩍 내놓다, 제안하다"의 뜻.

## 단어들

### suggest

- 구성: `sub-` = under + `gest`
- 뜻: 제안하다, 암시하다
- 어원적 연결: 생각이나 가능성을 아래에서 살짝 가져와 내놓음
- 사용 맥락: `suggest a solution` = 해결책을 제안하다

### suggestion

- 구성: `suggest` + `-ion`
- 뜻: 제안, 암시
- 어원적 연결: 마음속에 가져와진 생각
- 사용 맥락: `make a suggestion` = 제안하다

### suggestive

- 구성: `suggest` + `-ive`
- 뜻: 암시하는, 연상시키는
- 어원적 연결: 어떤 생각을 마음속으로 가져오게 함
- 사용 맥락: `suggestive evidence` = 암시적인 증거

### digest

- 구성: `di-` = apart + `gest`
- 뜻: 소화하다, 이해하다, 요약
- 어원적 연결: 여러 부분으로 나누어 처리함. 음식은 소화하고, 정보는 이해하기 쉽게 처리함
- 사용 맥락: `digest information` = 정보를 소화하다

### digestion

- 구성: `digest` + `-ion`
- 뜻: 소화, 이해
- 어원적 연결: 음식이나 정보를 처리해 받아들이는 과정
- 사용 맥락: `aid digestion` = 소화를 돕다

### gesture

- 구성: Latin `gestura`
- 뜻: 몸짓, 제스처
- 어원적 연결: 몸을 움직여 뜻을 가져다 보임
- 사용 맥락: `make a gesture` = 몸짓을 하다

### gestation

- 구성: Latin `gestare` = carry
- 뜻: 임신 기간, 발달 기간
- 어원적 연결: 태아나 생각을 안에 지니고 나르는 기간
- 사용 맥락: `a long gestation period` = 긴 임신 기간

### congest

- 구성: `con-` = together + `gest`
- 뜻: 혼잡하게 하다, 막히다
- 어원적 연결: 많은 것이 함께 운반되어 몰려 있음
- 사용 맥락: `congested traffic` = 정체된 교통

### congestion

- 구성: `congest` + `-ion`
- 뜻: 혼잡, 충혈, 막힘
- 어원적 연결: 많은 것이 한곳에 몰려 움직이지 못함
- 사용 맥락: `nasal congestion` = 코막힘

### register

- 구성: Latin `regesta` = recorded things
- 뜻: 등록하다, 기록부, 음역
- 어원적 연결: 가져온 일을 기록으로 올려 둠. 역사적으로 `re-` + `gerere` 계열
- 사용 맥락: `register for a course` = 강좌에 등록하다

### registry

- 구성: `register` 계열
- 뜻: 등록부, 등기소
- 어원적 연결: 기록으로 올려 둔 목록
- 사용 맥락: `land registry` = 토지 등기소

### exaggerate

- 구성: Latin `exaggerare`, 관련 어원은 `agger` = pile
- 뜻: 과장하다
- 어원적 연결: `gest` 계열 아님. 형태가 비슷해도 `suggest`와 직접 묶지 않음
- 사용 맥락: `exaggerate the problem` = 문제를 과장하다

## 기억 포인트

`gest` 계열은 "가져와 처리하거나 드러냄" 이미지.

- `suggest`: 생각을 가져와 내놓음
- `digest`: 나누어 처리해 소화함
- `gesture`: 뜻을 몸짓으로 드러냄
- `gestation`: 안에 지니고 나르는 기간
- `congest`: 한곳에 몰려 막힘
- `register`: 기록으로 올려 둠

## 비교 단어

### jest

- 뜻: 농담
- 구분 포인트: `gesture`와 역사적으로 닿는 부분이 있지만, 현대 시험 어휘 학습에서는 `jest` = 농담으로 따로 기억하는 편이 좋음

### exaggerate

- 뜻: 과장하다
- 구분 포인트: `suggest`와 `-gest` 모양이 비슷하지만, `exaggerate`는 `gest` = 나르다 계열로 보지 않음
