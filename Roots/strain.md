# strain

- 출발 단어: `restrain`
- 핵심 어원: 라틴어 `stringere` = 세게 묶다, 당기다, 조이다
- 핵심 이미지: 팽팽하게 조여 묶음
- 관련 형태: `strain`, `strict`, `string`

## 출발 단어 분석

`restrain`은 `re-` + `stringere` 계열.

- `re-`: back, again
- `stringere`: 세게 묶다, 조이다

따라서 `restrain`은 "뒤로 잡아 묶음" → "제지하다, 억제하다"의 뜻.

## 단어들

### restrain

- 발음: /riˈstreɪn/
- 구성: `re-` + `stringere`
- 뜻: 제지하다, 억제하다
- 어원적 연결: 뒤로 잡아 묶어 움직이지 못하게 함
- 사용 맥락: `restrain emotions` = 감정을 억제하다

### restraint

- 발음: /rɪˈstreɪnt/
- 구성: `restrain` + `-t`
- 뜻: 억제, 제지, restraint 장치
- 어원적 연결: 움직임이나 행동을 묶어 제한함
- 사용 맥락: `show restraint` = 자제력을 보이다

### constrain

- 발음: /kəˈnstreɪn/
- 구성: `con-` + `stringere`
- 뜻: 제한하다, 강제하다
- 어원적 연결: 함께 꽉 묶어 선택지를 좁힘
- 사용 맥락: `constrain growth` = 성장을 제한하다

### constraint

- 발음: /kəˈnstreɪnt/
- 구성: `constrain` + `-t`
- 뜻: 제약, 제한 조건
- 어원적 연결: 행동이나 가능성을 묶는 조건
- 사용 맥락: `budget constraints` = 예산 제약

### restrict

- 발음: /riˈstrɪkt/
- 구성: `re-` + `strict`
- 뜻: 제한하다
- 어원적 연결: 좁게 조여 범위를 줄임
- 사용 맥락: `restrict access` = 접근을 제한하다

### restriction

- 발음: /riˈstrɪkʃən/
- 구성: `restrict` + `-ion`
- 뜻: 제한, 규제
- 어원적 연결: 허용 범위를 조여 둔 규칙
- 사용 맥락: `travel restrictions` = 여행 제한

### strict

- 발음: /ˈstrɪkt/
- 구성: 라틴어 `strictus` = 팽팽하게 조인
- 뜻: 엄격한
- 어원적 연결: 규칙이나 기준을 느슨하게 두지 않음
- 사용 맥락: `strict rules` = 엄격한 규칙

### stringent

- 발음: /ˈstrɪndʒənt/
- 구성: 라틴어 `stringere` 계열
- 뜻: 엄격한, 긴박한
- 어원적 연결: 강하게 조여 여유가 없음
- 사용 맥락: `stringent standards` = 엄격한 기준

### strain

- 발음: /ˈstreɪn/
- 구성: 프랑스어를 거친 `stringere` 계열
- 뜻: 긴장, 부담, 무리하게 쓰다
- 어원적 연결: 팽팽하게 당겨져 부담이 걸림
- 사용 맥락: `financial strain` = 재정적 부담

### stress

- 발음: /ˈstrɛs/
- 구성: `distress`에서 짧아진 형태
- 뜻: 스트레스, 압박
- 어원적 연결: 조이고 누르는 압박감
- 사용 맥락: `reduce stress` = 스트레스를 줄이다

### distress

- 발음: /dɪˈstrɛs/
- 구성: Old French `destresse` ← Latin `distringere` = pull apart, stretch, hinder
- 뜻: 고통, 괴로움, 곤란; 괴롭히다
- 어원적 연결: 사방으로 조이고 당겨 압박하는 상태
- 사용 맥락: `emotional distress` = 정서적 고통

### distressed

- 발음: /dɪˈstrɛst/
- 구성: `distress` + `-ed`
- 뜻: 괴로워하는, 곤란한; 낡아 보이게 처리한
- 어원적 연결: 압박과 고통에 조여 있는 상태. 물건에는 일부러 닳고 손상된 듯 보이게 만든 상태
- 사용 맥락: `look distressed` = 괴로워 보이다, `distressed jeans` = 빈티지 가공 청바지

### district

- 발음: /ˈdɪstrɪkt/
- 구성: 라틴어 `districtus` = 묶인, 제한된
- 뜻: 구역, 지구
- 어원적 연결: 행정적으로 묶어 제한한 영역
- 사용 맥락: `a school district` = 학군

## 기억 포인트

`strain` 계열은 "팽팽하게 조임" 이미지.

- `restrain`: 뒤로 묶어 제지함
- `constrain`: 함께 묶어 제한함
- `restrict`: 범위를 좁게 조임
- `strict`: 느슨함 없이 엄격함
- `stringent`: 강하게 조여 엄격함
- `strain`: 당겨져 부담이 걸림
- `stress`: 압박감
- `distress`: 심한 압박과 고통
- `distressed`: 압박을 받아 괴로운 상태
- `district`: 묶어 나눈 구역
