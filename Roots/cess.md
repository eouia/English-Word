# cess

- 출발 단어: `process`, `success`, `access`
- 핵심 어원: 라틴어 `cedere` / `cess-` = 가다, 물러나다, 양보하다
- 핵심 이미지: 어떤 방향으로 움직임
- 비교: [[sess]] = 앉다 계열

## 출발 단어 분석

`cess`는 `cede`의 변형으로 보면 좋음.

- `cede`: 가다, 물러나다, 양보하다
- `cess`: `cedere`의 과거분사형/파생 형태

따라서 `process`, `access`, `success`의 `cess`는 [[sess]]와 다름.  
핵심은 "앉음"이 아니라 "이동, 진행, 물러남".

## 단어들

### cede

- 발음: /ˈsid/
- 구성: 라틴어 `cedere`
- 뜻: 양도하다, 넘겨주다
- 어원적 연결: 뒤로 물러나며 권리나 땅을 넘김
- 사용 맥락: `cede territory` = 영토를 양도하다

### proceed

- 발음: /prəˈsid/
- 구성: `pro-` = forward + `cede`
- 뜻: 진행하다, 계속하다
- 어원적 연결: 앞으로 나아감
- 사용 맥락: `proceed with the plan` = 계획을 진행하다

### procedure

- 발음: /prəˈsidʒɚ/
- 구성: `proceed` 계열 명사
- 뜻: 절차
- 어원적 연결: 일이 앞으로 나아가는 정해진 방식
- 사용 맥락: `follow the procedure` = 절차를 따르다

### process

- 발음: /ˈprɑˌsɛs/
- 구성: `pro-` + `cess`
- 뜻: 과정, 처리하다
- 어원적 연결: 앞으로 진행되는 단계들의 흐름
- 사용 맥락: `the learning process` = 학습 과정

### access

- 발음: /ˈæˌksɛs/
- 구성: `ad-` = toward + `cess`
- 뜻: 접근, 접근 권한
- 어원적 연결: 어떤 곳을 향해 감
- 사용 맥락: `access to information` = 정보 접근

### accessible

- 발음: /æˈksɛsəbəl/
- 구성: `access` + `-ible`
- 뜻: 접근 가능한, 이해하기 쉬운
- 어원적 연결: 다가갈 수 있는 상태
- 사용 맥락: `accessible language` = 이해하기 쉬운 언어

### excess

- 발음: /ˈɛˌksɛs/
- 구성: `ex-` = out, beyond + `cess`
- 뜻: 초과, 과잉
- 어원적 연결: 한계를 넘어 나아감
- 사용 맥락: `excess weight` = 초과 체중

### exceed

- 발음: /ɪˈksid/
- 구성: `ex-` + `ceed`
- 뜻: 초과하다
- 어원적 연결: 기준을 넘어감
- 사용 맥락: `exceed expectations` = 기대를 넘어서다

### success

- 발음: /səˈksɛs/
- 구성: `sub-` = under, after + `cess`
- 뜻: 성공
- 어원적 연결: 뒤따라 와서 좋은 결과에 이름
- 사용 맥락: `achieve success` = 성공을 이루다

### succeed

- 발음: /səˈksid/
- 구성: `sub-` + `ceed`
- 뜻: 성공하다, 뒤를 잇다
- 어원적 연결: 뒤따라감. 이후 "목표에 도달함" 또는 "후임이 됨"
- 사용 맥락: `succeed in business` = 사업에 성공하다

### succession

- 발음: /səˈksɛʃən/
- 구성: `success` 계열
- 뜻: 연속, 계승
- 어원적 연결: 하나가 다른 하나의 뒤를 이어 감
- 사용 맥락: `a succession of events` = 연속된 사건들

### predecessor

- 발음: /ˈprɛdəˌsɛsɚ/
- 구성: `pre-` = before + `de` + `cess`
- 뜻: 전임자, 이전 것
- 어원적 연결: 앞서 가 있던 사람이나 것
- 사용 맥락: `his predecessor` = 그의 전임자

### precede

- 발음: /prɪˈsid/
- 구성: `pre-` + `cede`
- 뜻: 앞서다
- 어원적 연결: 먼저 감
- 사용 맥락: `a chapter that precedes the conclusion` = 결론 앞에 오는 장

### recede

- 발음: /rɪˈsid/
- 구성: `re-` = back + `cede`
- 뜻: 물러나다, 약해지다
- 어원적 연결: 뒤로 감
- 사용 맥락: `receding hairline` = 뒤로 물러난 헤어라인

### recess

- 발음: /rɪˈsɛs/
- 구성: `re-` + `cess`
- 뜻: 휴식, 휴회, 움푹 들어간 곳
- 어원적 연결: 뒤로 물러나 쉬는 시간 또는 들어간 공간
- 사용 맥락: `during recess` = 쉬는 시간에

### recession

- 발음: /rɪˈsɛʃən/
- 구성: `recess` + `-ion`
- 뜻: 경기 침체, 후퇴
- 어원적 연결: 경제 활동이 뒤로 물러남
- 사용 맥락: `economic recession` = 경기 침체

### concede

- 발음: /kəˈnsid/
- 구성: `con-` + `cede`
- 뜻: 인정하다, 양보하다
- 어원적 연결: 함께 물러나며 받아들임
- 사용 맥락: `concede defeat` = 패배를 인정하다

### concession

- 발음: /kəˈnsɛʃən/
- 구성: `concede` 계열 명사
- 뜻: 양보, 인정, concession 매장
- 어원적 연결: 권리나 입장을 내어줌
- 사용 맥락: `make a concession` = 양보하다

### secede

- 발음: /sɪˈsid/
- 구성: `se-` = apart + `cede`
- 뜻: 탈퇴하다, 분리 독립하다
- 어원적 연결: 따로 떨어져 나감
- 사용 맥락: `secede from the union` = 연방에서 탈퇴하다

## 헷갈리는 형태

- [[cess]]: 가다, 물러나다, 양보하다
- [[sess]]: 앉다, 자리 잡다

예:

- `process`: 앞으로 감
- `session`: 함께 앉음
- `access`: 향해 감
- `assess`: 옆에 앉아 판단함
- `success`: 뒤따라 좋은 결과에 이름
- `obsession`: 마음 앞에 앉아 사로잡음

## 기억 포인트

`cess` 계열은 "움직임" 이미지.

- `process`: 앞으로 진행
- `access`: 향해 접근
- `excess`: 넘어감
- `success`: 뒤따라 목표에 이름
- `precede`: 앞서 감
- `recede`: 뒤로 물러남
- `recess`: 뒤로 물러난 시간/공간
- `concede`: 물러나 인정함
- `secede`: 따로 떨어져 나감
