# fac

- 출발 단어: `effect`
- 핵심 어원: Latin `facere` / `fact-` / `fect-` / `fic-` = 만들다, 하다
- 핵심 이미지: 어떤 상태나 결과를 만들어 냄
- 하위 문서: [[fac-effect]], [[fac-make]], [[fac-do]]
- 관련 문서: [[duc]], [[manu]], [[art]], [[fig]]

## 출발 단어 분석

`effect`는 `ex-` + `facere`.

- `ex-`: out
- `facere`: 만들다, 하다
- 원래 이미지: 밖으로 만들어 낸 것
- 현대 의미: 결과, 효과

## 갈래

### [[fac-effect]]

- 핵심 이미지: 작용해서 결과나 상태를 만들어 냄
- 대표 단어: `effect`, `effective`, `affect`, `affection`, `defect`, `perfect`

### [[fac-make]]

- 핵심 이미지: 실제로 만들거나 만들어진 것
- 대표 단어: `factory`, `factor`, `fact`, `manufacture`, `artificial`, `feature`

### [[fac-do]]

- 핵심 이미지: 할 수 있음, 충분히 해냄, 쉽게 만듦
- 대표 단어: `efficient`, `sufficient`, `proficient`, `deficient`, `difficult`, `facilitate`, `feasible`, `office`

## 기억 포인트

`fac` 계열은 "만들다 / 하다".

- `effect`: 만들어진 결과
- `affect`: 작용해 상태를 바꿈
- `factory`: 만드는 장소
- `factor`: 결과를 만드는 요인
- `efficient`: 결과를 잘 만들어 냄
- `feasible`: 할 수 있음

## 비교 단어

### produce

- 뜻: 생산하다, 만들어 내다
- 구분 포인트: [[duc]] 계열. 앞으로 이끌어 내어 생산함. `fac`는 만들거나 행해 결과를 냄.

### fig

- 뜻: 빚어 만든 모양, 꾸며 낸 것
- 구분 포인트: [[fig]]의 `fiction`, `figure`, `feign`은 Latin `fingere/figura` 계열. `fac`의 `facere`와 어원 다름.
