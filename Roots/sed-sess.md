# sed-sess

- 상위 문서: [[sed]]
- 핵심 어원: 라틴어 `sedere` / `sessus` = 앉다, 앉아 있음
- 핵심 이미지: 앉아 차지함 / 둘러앉아 포위함 / 옆에 앉아 판단함
- 관련 갈래: [[sed-sit]], [[sed-sedere]]

## 단어들

### siege

- 구성: 고대 프랑스어 `siege` = 자리, 왕좌
- 뜻: 포위 공격
- 어원적 연결: 군대가 성 앞에 눌러앉아 포위함
- 사용 맥락: `lay siege to a city` = 도시를 포위하다

### besiege

- 구성: `be-` + `siege`
- 뜻: 포위하다, 에워싸다
- 어원적 연결: 적의 장소 주변에 자리 잡고 둘러쌈
- 사용 맥락: `besiege a fortress` = 요새를 포위하다

### obsess

- 구성: `ob-` = against, before + `sedere`
- 뜻: 사로잡다, 집착하게 하다
- 어원적 연결: 원래 "앞에 앉아 포위함". 이후 생각이 마음을 포위하듯 사로잡음
- 사용 맥락: `be obsessed with success` = 성공에 집착하다

### obsession

- 구성: `obsess` + `-ion`
- 뜻: 집착, 강박
- 어원적 연결: 마음이 어떤 생각에 포위된 상태
- 사용 맥락: `an obsession with perfection` = 완벽에 대한 집착

### possess

- 구성: 라틴어 `possidere` = 차지하고 앉다
- 뜻: 소유하다, 지니다
- 어원적 연결: 어떤 자리에 앉아 차지함
- 비교: [[seize]] = 갑자기 붙잡아 장악함
- 사용 맥락: `possess rare talent` = 드문 재능을 지니다

### possession

- 구성: `possess` + `-ion`
- 뜻: 소유, 소유물
- 어원적 연결: 차지하고 있는 상태 또는 물건
- 사용 맥락: `personal possessions` = 개인 소유물

### assess

- 구성: `ad-` + `sedere`
- 뜻: 평가하다
- 어원적 연결: 옆에 앉아 판단함
- 사용 맥락: `assess the risk` = 위험을 평가하다

### assessment

- 구성: `assess` + `-ment`
- 뜻: 평가, 사정
- 어원적 연결: 옆에 앉아 판단한 결과
- 사용 맥락: `risk assessment` = 위험 평가

### assessor

- 구성: `assess` + `-or`
- 뜻: 평가자, 감정인
- 어원적 연결: 옆에 앉아 판단하는 사람
- 사용 맥락: `a tax assessor` = 세무 감정인

### reassess

- 구성: `re-` = again + `assess`
- 뜻: 재평가하다
- 어원적 연결: 다시 앉아 판단하듯 새로 평가함
- 사용 맥락: `reassess the situation` = 상황을 재평가하다

### subsidy

- 구성: 라틴어 `subsidium` = 지원군, 도움
- 뜻: 보조금
- 어원적 연결: 뒤쪽에 앉아 대기하는 지원 병력에서 "도움, 지원"으로 확장
- 사용 맥락: `government subsidies` = 정부 보조금

### subsidiary

- 구성: 라틴어 `subsidium` 계열
- 뜻: 보조의, 자회사
- 어원적 연결: 뒤쪽에서 대기하며 지원하는 것. 이후 주된 회사 아래에서 보조 역할을 하는 회사
- 사용 맥락: `a subsidiary company` = 자회사

### subordinate

`subordinate`는 [[sed]] 계열 아님. [[ord]] 참고.

- 구성: `sub-` + `ordinare`
- 뜻: 하위의, 부하
- 구분 포인트: `subsidy/subsidiary` = 뒤에서 지원, `subordinate` = 아래 등급에 배치

### insidious

- 구성: 라틴어 `insidiae` = 매복
- 뜻: 교활한, 은밀히 해로운
- 어원적 연결: 안쪽에 숨어 앉아 기다림
- 사용 맥락: `an insidious disease` = 은밀히 진행되는 질병

## 기억 포인트

`sed-sess`는 앉은 자리가 힘이나 압박을 만듦.

- `siege`: 성 앞에 눌러앉아 포위함
- `obsess`: 생각이 마음을 포위함
- `possess`: 차지하고 앉음
- `assess`: 옆에 앉아 판단함
- `subsidy`: 뒤에 대기하는 지원
- `insidious`: 숨어 앉아 해로움
