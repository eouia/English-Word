# liter

- 출발 단어: `literary`
- 핵심 어원: Latin `littera` / `litera` = letter, writing
- 핵심 이미지: 글자, 글, 문자로 된 것
- 관련 어근: [[scrib]], [[dict]], [[logos]]

## 출발 단어 분석

`literary`는 Latin `littera` = letter에서 온 말.

`littera`는 "글자"에서 출발해 "글, 문서, 문학"으로 확장됨.

따라서 `literary`는 "글로 된 것, 문학과 관련된 것"의 뜻.

## 단어들

### literary

- 발음: /ˈlɪtɚˌɛri/
- 구성: `literature` 계열 + `-ary`
- 의미: 문학의, 문어적인
- 어원적 연결: 글과 문학에 관련됨
- 사용 맥락: `literary criticism` = 문학 비평

### literature

- 발음: /ˈlɪtɚətʃɚ/
- 구성: Latin `litteratura` = writing, learning
- 의미: 문학, 문헌
- 어원적 연결: 글로 쓰인 작품이나 지식
- 사용 맥락: `English literature` = 영문학

### literal

- 발음: /ˈlɪtɚəl/
- 구성: Latin `litteralis` = of letters
- 의미: 문자 그대로의
- 어원적 연결: 글자에 적힌 그대로
- 사용 맥락: `a literal meaning` = 문자 그대로의 의미

### literally

- 발음: /ˈlɪtɚəli/
- 구성: `literal` + `-ly`
- 의미: 문자 그대로, 실제로
- 어원적 연결: 글자 그대로 해석함
- 사용 맥락: `literally true` = 말 그대로 사실인

### literacy

- 발음: /ˈlɪtɚəsi/
- 구성: `literate` + `-cy`
- 의미: 읽고 쓰는 능력, 문해력
- 어원적 연결: 글자를 다룰 수 있음
- 사용 맥락: `digital literacy` = 디지털 문해력

### literate

- 발음: /ˈlɪtɚət/
- 구성: Latin `litteratus` = educated, lettered
- 의미: 글을 읽고 쓸 줄 아는, 교양 있는
- 어원적 연결: 글자를 아는 상태
- 사용 맥락: `a literate society` = 문해력이 있는 사회

### illiterate

- 발음: /ˌɪˈlɪtɚət/
- 구성: [[in]] = not + `literate`
- 의미: 문맹의, 글을 모르는
- 어원적 연결: 글자를 다루지 못함
- 사용 맥락: `functionally illiterate` = 기능적 문맹의

### letter

- 발음: /ˈlɛtɚ/
- 구성: Old French `letre` < Latin `littera`
- 의미: 글자, 편지
- 어원적 연결: 글자 하나 또는 글자로 쓴 메시지
- 사용 맥락: `write a letter` = 편지를 쓰다

### lettering

- 발음: /ˈlɛtɚɪŋ/
- 구성: `letter` + `-ing`
- 의미: 글자 쓰기, 글자 디자인
- 어원적 연결: 글자를 쓰거나 꾸밈
- 사용 맥락: `hand lettering` = 손글씨 레터링

### alliteration

- 발음: /əˈlɪtɚˌeɪʃən/
- 구성: `ad-` = to + `littera` + `-ation`
- 의미: 두운, 같은 첫소리 반복
- 어원적 연결: 글자나 소리의 반복
- 사용 맥락: `use alliteration` = 두운을 사용하다

## 기억 포인트

- `letter`: 글자, 편지
- `literal`: 글자 그대로
- `literature`: 글로 된 작품
- `literary`: 문학의
- `literate`: 글을 아는
- `illiterate`: 글을 모르는
- `alliteration`: 같은 글자/소리의 반복
