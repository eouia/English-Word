# cern

- 출발 단어: `concern`
- 핵심 어원: Latin `cernere` = separate, sift, distinguish, decide
- 핵심 이미지: 걸러 구별하고 판단함
- 관련 문서: [[cid]], [[jud]]
- 주의: `concern`의 현대 뜻 "걱정"은 "나와 관련되어 마음에 걸림"에서 확장. 원래는 구별·관련 판단의 이미지.

## 출발 단어 분석

`concern` ← Latin `concernere`

- `con-`: together, with
- `cernere`: separate, sift, distinguish
- 원래 이미지: 여러 요소를 함께 걸러 살피고 관계를 따짐
- 의미 변화: 구별해 판단함 → 어떤 일과 관련됨 → 마음에 걸림, 걱정됨
- 현대 의미: 관련되다, 걱정시키다; 관심사, 우려

## 단어들

### concern

- 구성: Latin `con-` = together + `cernere` = distinguish, decide
- 뜻: 관련되다, 걱정시키다; 관심사, 우려
- 어원적 연결: 어떤 일이 나와 관련 있다고 판단되어 마음에 걸림
- 사용 맥락: `a matter of concern` = 우려되는 문제

### concerned

- 구성: `concern` + `-ed`
- 뜻: 걱정하는, 관련된
- 어원적 연결: 어떤 일과 관련되어 마음이 붙잡힌 상태
- 사용 맥락: `concerned parents` = 걱정하는 부모들

### concerning

- 구성: `concern` + `-ing`
- 뜻: ~에 관한; 걱정스러운
- 어원적 연결: 어떤 대상과 관련됨
- 사용 맥락: `rules concerning safety` = 안전에 관한 규칙

### discern

- 구성: Latin `dis-` = apart + `cernere` = distinguish
- 뜻: 알아차리다, 식별하다
- 어원적 연결: 서로 다른 것을 갈라 구별해 봄
- 사용 맥락: `discern a pattern` = 패턴을 알아차리다

### discernment

- 구성: `discern` + `-ment`
- 뜻: 분별력, 식별력
- 어원적 연결: 차이를 잘 가려 판단하는 능력
- 사용 맥락: `show discernment` = 분별력을 보이다

### certain

- 구성: Latin `certus` = decided, settled
- 뜻: 확실한, 특정한
- 어원적 연결: 구별하고 판단이 내려져 확정됨
- 사용 맥락: `a certain amount` = 일정한 양

### ascertain

- 구성: Old French `acertener` = make certain
- 뜻: 확인하다, 알아내다
- 어원적 연결: 불확실한 것을 판단해 확실하게 만듦
- 사용 맥락: `ascertain the cause` = 원인을 확인하다

### secret

- 구성: Latin `se-` = apart + `cernere` = separate
- 뜻: 비밀, 비밀의
- 어원적 연결: 따로 갈라 숨겨 둔 것
- 사용 맥락: `keep a secret` = 비밀을 지키다

### discrete

- 구성: Latin `discretus` = separated
- 뜻: 분리된, 별개의
- 어원적 연결: 서로 갈라져 구별됨
- 사용 맥락: `discrete categories` = 별개의 범주

### discreet

- 구성: Latin `discretus` = discerning, prudent
- 뜻: 신중한, 분별 있는
- 어원적 연결: 잘 구별해 판단하므로 조심스럽게 행동함
- 사용 맥락: `be discreet about the matter` = 그 일을 신중히 다루다

### discretion

- 구성: Latin `discretio` = separation, discernment
- 뜻: 신중함, 분별력; 재량, 결정권
- 어원적 연결: 상황을 잘 구별해 판단하는 능력. 그래서 스스로 판단해 결정할 수 있는 재량으로 확장
- 사용 맥락: `use your discretion` = 재량껏 판단하다

### discretionary

- 구성: `discretion` + `-ary`
- 뜻: 재량의, 임의의
- 어원적 연결: 정해진 규칙만이 아니라 판단자의 분별과 선택에 맡겨짐
- 사용 맥락: `discretionary spending` = 재량 지출

### decree

- 구성: Latin `decernere` = decide, determine
- 뜻: 법령, 명령; 포고하다
- 어원적 연결: 구별해 판단한 뒤 공식적으로 결정함
- 사용 맥락: `issue a decree` = 법령을 공포하다

## 기억 포인트

`cern` 계열은 "가려 구별하고 판단함" 이미지.

- `concern`: 관련되어 마음에 걸림
- `discern`: 구별해 알아차림
- `discernment`: 분별력
- `certain`: 판단이 확정된
- `ascertain`: 확실히 알아냄
- `secret`: 따로 갈라 숨긴 것
- `discrete`: 별개의
- `discreet`: 분별 있어 신중한
- `discretion`: 분별력과 재량
- `discretionary`: 재량에 맡겨진

## 비교 단어

### cid

- 뜻: 자르다, 잘라 경계를 내다
- 구분 포인트: [[cid]]는 물리적·비유적 절단 이미지. `cern`은 걸러 구별하고 판단하는 이미지.

### judge

- 뜻: 판단하다, 재판하다
- 구분 포인트: 판단 행위 자체는 가깝지만, `cern`은 먼저 차이를 가려 구별하는 느낌이 강함.

### crisis

- 뜻: 위기, 중대한 고비
- 구분 포인트: Greek `krinein` = separate, judge 계열. 의미 이미지는 비슷하지만 Latin `cernere`와는 별도 Greek 계열로 보는 편이 안전함.
