# qual

- 출발 단어: `quality`
- 핵심 어원: Latin `qualis` = of what kind, 어떤 종류의
- 핵심 이미지: 어떤 성질인가, 어떤 기준을 충족하는가
- 관련 문서: [[quant]], [[equ]]
- 주의: `quality`는 성질과 질. `quantity`는 양과 수량이라 [[quant]]에서 따로 정리.

## 출발 단어 분석

`quality` ← Old French `qualite` ← Latin `qualitas` ← `qualis`

- `qualis`: 어떤 종류의, 어떤 성질의
- `-ity`: 상태나 성질을 나타내는 접미사
- 원래 이미지: 어떤 종류의 성질을 가진 상태
- 현대 의미: 질, 품질, 특성

## 단어들

### quality

- 발음: /ˈkwɑləti/
- 구성: Latin `qualitas` ← `qualis`
- 뜻: 질, 품질, 특성
- 어원적 연결: 어떤 종류의 성질을 가지는지 나타냄
- 사용 맥락: `high-quality materials` = 고품질 재료

### qualitative

- 발음: /ˈkwɑləˌteɪtɪv/
- 구성: `quality` + `-ative`
- 뜻: 질적인, 성질에 관한
- 어원적 연결: 숫자보다 성질과 종류를 따짐
- 사용 맥락: `qualitative research` = 질적 연구

### qualify

- 발음: /ˈkwɑləˌfaɪ/
- 구성: Latin `qualis` + `-ify`
- 뜻: 자격을 주다, 자격을 얻다, 제한하다
- 어원적 연결: 어떤 기준의 성질을 갖추게 하거나 조건을 붙임
- 사용 맥락: `qualify for the exam` = 시험 응시 자격을 얻다

### qualified

- 발음: /ˈkwɑləˌfaɪd/
- 구성: `qualify` + `-ed`
- 뜻: 자격 있는, 적임의; 제한적인
- 어원적 연결: 요구되는 성질이나 기준을 갖춘 상태
- 사용 맥락: `a qualified teacher` = 자격 있는 교사

### qualification

- 발음: /ˌkwɑləfəˈkeɪʃən/
- 구성: `qualify` + `-ication`
- 뜻: 자격, 조건, 제한
- 어원적 연결: 어떤 기준을 충족함 또는 조건을 붙임
- 사용 맥락: `academic qualifications` = 학력 자격

### disqualify

- 발음: /dɪˈskwɑləˌfaɪ/
- 구성: `dis-` = remove, apart + `qualify`
- 뜻: 자격을 박탈하다, 실격시키다
- 어원적 연결: 기준을 충족하는 성질이나 자격을 제거함
- 사용 맥락: `be disqualified from the race` = 경기에서 실격되다

## 기억 포인트

`qual` 계열은 "어떤 성질인가" 이미지.

- `quality`: 질, 특성
- `qualitative`: 성질로 따지는
- `qualify`: 기준을 갖추게 하거나 조건을 붙임
- `qualified`: 자격을 갖춘
- `qualification`: 자격, 조건
- `disqualify`: 자격을 없앰

## 비교 단어

### quantity

- 뜻: 양, 수량
- 구분 포인트: [[quant]]는 "얼마나 많은가"에 초점. `qual`은 "어떤 성질인가"에 초점.

### equal

- 뜻: 같은, 동등한
- 구분 포인트: [[equ]] 계열. `equality`는 `quality`와 철자가 닮았지만 핵심은 "성질"이 아니라 "같음".
- 예: `equality before the law` = 법 앞의 평등
