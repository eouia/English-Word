# argu

- 출발 단어: `argue`
- 핵심 어원: 라틴어 `arguere` = 분명히 보이게 하다, 증명하다, 주장하다
- 핵심 의미: 이유를 들어 밝히고 증명함
- 관련 문서: [[clar]], [[dict]], [[logos]], [[val]]
- 주의: 더 깊게는 PIE `arg-` = 빛나다, 희다 와 연결. 영어 학습에서는 “밝히다 → 증명하다/논증하다”로 기억.

## 출발 단어 분석

`argue` ← Old French `arguer` ← Latin `arguere`

- 원래 이미지: 어두운 것을 밝게 드러냄
- 의미 확장: 분명히 보이게 함 → 증명함 → 이유를 들어 주장함 → 말다툼함
- 현대 핵심: 이유와 근거를 들어 말함

## 단어들

### argue

- 구성: Latin `arguere`
- 뜻: 주장하다, 논증하다, 말다툼하다
- 어원적 연결: 근거를 들어 어떤 점을 분명히 드러냄
- 사용 맥락: `argue for reform` = 개혁을 주장하다

### argument

- 구성: `argue` + `-ment`
- 뜻: 논증, 주장, 말다툼
- 어원적 연결: 어떤 결론을 밝히기 위해 제시한 이유들의 묶음
- 사용 맥락: `a strong argument` = 강한 논거

### argumentative

- 구성: `argument` + `-ative`
- 뜻: 논쟁적인, 따지기 좋아하는
- 어원적 연결: 주장을 세우고 맞서는 성향
- 사용 맥락: `an argumentative tone` = 논쟁적인 말투

### arguable

- 구성: `argue` + `-able`
- 뜻: 논할 여지가 있는, 주장 가능한
- 어원적 연결: 이유를 들어 따져 볼 수 있음
- 사용 맥락: `an arguable point` = 논의할 만한 지점

### arguably

- 구성: `arguable` + `-ly`
- 뜻: 주장하건대, 아마도
- 어원적 연결: 논거를 들어 그렇게 말할 수 있음
- 사용 맥락: `arguably the best option` = 아마도 가장 좋은 선택지

### counterargument

- 구성: `counter-` = against + `argument`
- 뜻: 반론, 반대 논거
- 어원적 연결: 앞선 주장에 맞서 제시한 이유
- 사용 맥락: `present a counterargument` = 반론을 제시하다

### arguer

- 구성: `argue` + `-er`
- 뜻: 논증하는 사람, 논쟁하는 사람
- 어원적 연결: 이유를 들어 주장하는 사람
- 사용 맥락: `a skilled arguer` = 논증에 능한 사람

### arguendo

- 구성: Latin `arguendo` = arguing
- 뜻: 논의를 위해서, 가정상
- 어원적 연결: 법률·논증에서 “일단 그렇게 주장한다고 치고”
- 사용 맥락: `assuming arguendo` = 논의를 위해 가정하면

## 기억 포인트

- `argu`는 “밝혀서 증명함”.
- `argue`는 꼭 싸우다는 뜻만이 아니라, 학술 글에서 “근거를 들어 주장하다”가 핵심.
- `argument`는 감정적 말다툼보다 “논거의 구조”로 자주 나옴.
- `arguably`는 “논증 가능하게 말하자면” → “아마도, 주장하건대”.

## 비교 단어

### argent

- `argent`는 은, 은빛의 뜻.
- 화학 기호 `Ag`는 라틴어 `argentum` = 은에서 옴.
- `Argentina`도 “은빛의, 은의 땅” 이미지와 연결.
- 더 깊은 PIE `arg-` = 빛나다, 희다 와 연결되지만, 현대 영어 학습에서는 `argue`와 따로 기억.

### clarify

- [[clar]]는 밝고 분명하게 하다.
- `argue`도 “밝히다” 이미지가 있지만, 실제 쓰임은 이유·논거를 통한 주장 쪽.
