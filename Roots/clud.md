# clud

- 출발 단어: `include`
- 핵심 어원: Latin `claudere` = 닫다, 막다, 가두다
- 변형: `clud`, `clus`, `claus`, `clos`
- 핵심 의미: 닫아 넣거나, 닫아 막거나, 닫아 끝냄
- 관련 문서: [[in]], [[ex]], [[con]], [[struct]]
- 주의: 합성어에서는 Latin `-cludere`, 과거분사형에서는 `-clusus`가 자주 보임.

## 출발 단어 분석

`include` = `in-` + Latin `claudere`

- `in-`: 안으로
- `claudere`: 닫다
- 원래 이미지: 안에 넣고 문을 닫음
- 현대 의미: 포함하다

## 단어들

### include

- 발음: /ˌɪˈnklud/
- 구성: `in-` = in + `clud` = close
- 뜻: 포함하다
- 어원적 연결: 안에 넣고 닫아 함께 둠
- 사용 맥락: `include examples` = 예시를 포함하다

### inclusion

- 발음: /ˌɪˈnkluʒən/
- 구성: `in-` + `clus` + `-ion`
- 뜻: 포함, 포용
- 어원적 연결: 안에 넣어 함께 둔 상태
- 사용 맥락: `social inclusion` = 사회적 포용

### inclusive

- 발음: /ˌɪˈnklusɪv/
- 구성: `in-` + `clus` + `-ive`
- 뜻: 포괄적인, 포함하는
- 어원적 연결: 닫힌 범위 안에 여러 대상을 넣음
- 사용 맥락: `inclusive education` = 통합 교육

### exclude

- 발음: /ɪˈksklud/
- 구성: `ex-` = out + `clud` = close
- 뜻: 제외하다, 배제하다
- 어원적 연결: 밖에 두고 문을 닫음
- 사용 맥락: `exclude irrelevant data` = 관련 없는 자료를 제외하다

### exclusion

- 발음: /ɪˈkskluʒən/
- 구성: `ex-` + `clus` + `-ion`
- 뜻: 제외, 배제
- 어원적 연결: 닫힌 범위 밖으로 밀어냄
- 사용 맥락: `exclusion from the group` = 집단에서의 배제

### exclusive

- 발음: /ɪˈksklusɪv/
- 구성: `ex-` + `clus` + `-ive`
- 뜻: 독점적인, 배타적인, 고급 전용의
- 어원적 연결: 다른 것을 밖에 닫아 두고 일부만 안에 둠
- 사용 맥락: `exclusive access` = 독점 접근권

### conclude

- 발음: /kəˈnklud/
- 구성: `con-` = together, completely + `clud` = close
- 뜻: 결론짓다, 끝내다
- 어원적 연결: 논의나 일을 닫아 끝맺음
- 사용 맥락: `conclude the meeting` = 회의를 마무리하다

### conclusion

- 발음: /kəˈnkluʒən/
- 구성: `con-` + `clus` + `-ion`
- 뜻: 결론, 마무리
- 어원적 연결: 생각의 문을 닫아 끝낸 지점
- 사용 맥락: `draw a conclusion` = 결론을 내리다

### conclusive

- 발음: /kəˈnklusɪv/
- 구성: `con-` + `clus` + `-ive`
- 뜻: 결정적인, 확정적인
- 어원적 연결: 더 논의할 여지를 닫아 버릴 만큼 결정적임
- 사용 맥락: `conclusive evidence` = 결정적 증거

### preclude

- 발음: /prɪˈklud/
- 구성: `pre-` = before + `clud` = close
- 뜻: 못 하게 막다, 불가능하게 하다
- 어원적 연결: 앞에서 미리 닫아 길을 막음
- 사용 맥락: `preclude further discussion` = 추가 논의를 막다

### precluded

- 발음: /prɪˈkludɪd/
- 구성: `preclude` + `-ed`
- 뜻: 배제된, 막힌, 불가능해진
- 어원적 연결: 미리 길이 닫혀 더 진행할 수 없는 상태
- 사용 맥락: `precluded by law` = 법으로 금지된

### preclusion

- 발음: /prɪˈkluʒən/
- 구성: `pre-` + `clus` + `-ion`
- 뜻: 배제, 차단, 금지
- 어원적 연결: 미리 닫아 어떤 가능성을 막는 행위나 상태
- 사용 맥락: `issue preclusion` = 쟁점 배제 원칙

### preclusive

- 구성: `preclusion` + `-ive`
- 뜻: 배제하는, 차단하는
- 어원적 연결: 가능성을 미리 닫아 버리는 성질
- 사용 맥락: 법률·격식 문맥에서 주로 사용

### seclude

- 발음: /səˈklud/
- 구성: `se-` = apart + `clud` = close
- 뜻: 격리하다, 은둔시키다
- 어원적 연결: 따로 떼어 닫아 둠
- 사용 맥락: `seclude oneself` = 은둔하다

### secluded

- 발음: /sɪˈkludɪd/
- 구성: `seclude` + `-ed`
- 뜻: 외딴, 한적한
- 어원적 연결: 다른 곳과 떨어져 닫힌 듯한 장소
- 사용 맥락: `a secluded village` = 외딴 마을

### recluse

- 발음: /rɪˈklus/
- 구성: Latin `recludere` = shut away
- 뜻: 은둔자
- 어원적 연결: 사회로부터 닫혀 떨어져 사는 사람
- 사용 맥락: `live as a recluse` = 은둔자로 살다

### occlude

- 구성: `ob-` = against + `clud` = close
- 뜻: 막다, 폐색하다
- 어원적 연결: 통로나 흐름을 닫아 막음
- 사용 맥락: `occlude an artery` = 동맥을 막다

### occlusion

- 발음: /əˈkluʒən/
- 구성: `ob-` + `clus` + `-ion`
- 뜻: 막힘, 폐색; 치아 맞물림
- 어원적 연결: 닫혀서 통로가 막힌 상태
- 사용 맥락: `vascular occlusion` = 혈관 폐색

### disclose

- 발음: /dɪˈskloʊz/
- 구성: `dis-` = apart, away + `close`
- 뜻: 드러내다, 공개하다
- 어원적 연결: 닫힌 것을 열어 감춰진 내용을 드러냄
- 사용 맥락: `disclose information` = 정보를 공개하다

### disclosure

- 발음: /dɪˈskloʊʒɚ/
- 구성: `disclose` + `-ure`
- 뜻: 공개, 폭로
- 어원적 연결: 닫힌 정보를 열어 보임
- 사용 맥락: `financial disclosure` = 재무 정보 공개

### close

- 발음: adj./adv. /ˈkloʊs/, v. /ˈkloʊz/
- 구성: Old French `clos` ← Latin `clausus`
- 뜻: 닫다; 가까운
- 어원적 연결: 닫힌 상태에서 “틈이 없음” → 가까움
- 사용 맥락: `close the door`, `a close friend`

### closure

- 발음: /ˈkloʊʒɚ/
- 구성: `close` + `-ure`
- 뜻: 폐쇄, 종결, 심리적 매듭
- 어원적 연결: 열려 있던 것을 닫아 끝냄
- 사용 맥락: `seek closure` = 심리적 매듭을 찾다

### clause

- 발음: /ˈklɔz/
- 구성: Latin `clausula` = closing, ending
- 뜻: 절, 조항
- 어원적 연결: 문장이나 법률 문서 안에서 닫힌 한 단위
- 사용 맥락: `a relative clause` = 관계절

## 기억 포인트

- `include`: 안에 닫아 넣음.
- `exclude`: 밖에 닫아 둠.
- `conclude`: 완전히 닫아 끝냄.
- `preclude`: 미리 닫아 막음.
- `preclusion`: 미리 닫아 배제함.
- `seclude`: 따로 닫아 격리함.
- `disclose`: 닫힌 것을 열어 드러냄.

## 비교 단어

### obstruct

- [[struct]]의 `obstruct`는 길 위에 쌓아 막음.
- `occlude`, `preclude`는 문이나 통로를 닫는 이미지.

### include vs involve

- `include`는 범위 안에 넣음.
- `involve`는 말려들게 하거나 관련시키는 느낌.

### clause vs sentence

- `clause`는 문장 안의 닫힌 문법 단위.
- `sentence`는 판단·의견이 말로 정리된 완결 문장.
