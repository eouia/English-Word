# dict

- 출발 단어: `dictionary`
- 핵심 어원: 라틴어 `dicere` / `dict-` = 말하다, 선언하다
- 핵심 이미지: 말로 정해 밝힘
- 관련해서 헷갈릴 수 있는 어근: [[voc]] = 부르다, 목소리 계열, [[logos]] = 말, 설명, 이성 계열, [[lect]] = 읽다, 낭독하다 계열

## 출발 단어 분석

`dictionary`는 Latin `dictio`에서 온 말.

- `dicere`: 말하다
- `dictio`: 말, 표현, 말하는 방식
- `dictionary`: 말들의 모음, 사전

따라서 `dict`는 "말하다"에서 "말로 규정하다, 선언하다, 예측하다"까지 확장됨.

## 단어들

### dictionary

- 발음: /ˈdɪkʃəˌnɛri/
- 구성: `dict` + `-ion` + `-ary`
- 뜻: 사전
- 어원적 연결: 말과 표현을 모아 둔 책
- 사용 맥락: `look up a word in a dictionary` = 사전에서 단어를 찾다

### dictate

- 발음: /dɪˈkteɪt/
- 구성: `dict` + `-ate`
- 뜻: 받아쓰게 하다, 명령하다
- 어원적 연결: 말로 불러 주거나 지시함
- 사용 맥락: `dictate a letter` = 편지를 받아쓰게 하다

### dictation

- 발음: /dɪˈkteɪʃən/
- 구성: `dictate` + `-ion`
- 뜻: 받아쓰기, 구술
- 어원적 연결: 말로 불러 주는 행위
- 사용 맥락: `English dictation` = 영어 받아쓰기

### dictator

- 발음: /dɪˈkteɪtɚ/
- 구성: `dictate` + `-or`
- 뜻: 독재자
- 어원적 연결: 말로 명령을 내려 지배하는 사람
- 사용 맥락: `a military dictator` = 군사 독재자

### predict

- 발음: /prɪˈdɪkt/
- 구성: `pre-` = before + `dict`
- 뜻: 예측하다
- 어원적 연결: 미리 말함
- 사용 맥락: `predict the result` = 결과를 예측하다

### prediction

- 발음: /priˈdɪkʃən/
- 구성: `predict` + `-ion`
- 뜻: 예측
- 어원적 연결: 미리 말한 판단
- 사용 맥락: `make a prediction` = 예측하다

### predictable

- 발음: /prɪˈdɪktəbəl/
- 구성: `predict` + `-able`
- 뜻: 예측 가능한, 뻔한
- 어원적 연결: 미리 말할 수 있을 만큼 패턴이 보임
- 사용 맥락: `a predictable outcome` = 예측 가능한 결과

### unpredictable

- 발음: /ˌʌnprɪˈdɪktəbəl/
- 구성: `un-` = not + `predictable`
- 뜻: 예측할 수 없는, 종잡을 수 없는
- 어원적 연결: 미리 말할 수 있을 만큼 흐름이 안정되어 있지 않음
- 사용 맥락: `unpredictable weather` = 예측하기 어려운 날씨, `unpredictable behavior` = 종잡을 수 없는 행동

### unpredictability

- 발음: /ˌʌnprəˌdɪktəˈbɪlɪti/
- 구성: `unpredictable` + `-ity`
- 뜻: 예측 불가능성
- 어원적 연결: 미리 말하기 어려운 성질
- 사용 맥락: `market unpredictability` = 시장의 예측 불가능성

### contradict

- 발음: /ˌkɑntrəˈdɪkt/
- 구성: `contra-` = against + `dict`
- 뜻: 반박하다, 모순되다
- 어원적 연결: 반대쪽에서 말함
- 사용 맥락: `contradict the evidence` = 증거와 모순되다

### condition

- 발음: /kəˈndɪʃən/
- 구성: Latin `condicio` / `condicere` = agreement, stipulation
- 뜻: 조건, 상태; 조건화하다
- 어원적 연결: 함께 말해 정한 사항 → 어떤 일이 성립하는 조건 → 놓인 상태
- 사용 맥락: `under certain conditions` = 특정 조건 아래에서

### conditional

- 발음: /kəˈndɪʃənəl/
- 구성: `condition` + `-al`
- 뜻: 조건부의, 조건적인
- 어원적 연결: 정해진 조건에 따라 달라짐
- 사용 맥락: `conditional approval` = 조건부 승인

### conditionally

- 발음: /kəˈndɪʃənəli/
- 구성: `conditional` + `-ly`
- 뜻: 조건부로
- 어원적 연결: 어떤 조건이 충족될 때만
- 사용 맥락: `conditionally accepted` = 조건부로 받아들여진

### conditioning

- 발음: /kəˈndɪʃənɪŋ/
- 구성: `condition` + `-ing`
- 뜻: 조건화, 훈련; 조절
- 어원적 연결: 특정 상태나 반응이 생기도록 조건을 만듦
- 사용 맥락: `classical conditioning` = 고전적 조건화

### verdict

- 발음: /ˈvɝdɪkt/
- 구성: Latin `verus` = true + `dict`
- 뜻: 평결, 판정
- 어원적 연결: 진실을 말한 판단
- 사용 맥락: `reach a verdict` = 평결에 이르다

### addict

- 발음: n. /ˈæˌdɪkt/, v. /əˈdɪkt/
- 구성: `ad-` = to + `dict`
- 뜻: 중독자, 중독되게 하다
- 어원적 연결: 원래 "말로 넘겨 맡기다, 바치다"에서 특정 대상에 매인 상태로 변화
- 사용 맥락: `addicted to games` = 게임에 중독된

## 기억 포인트

`dict` 계열은 "말로 밝힘 / 정함" 이미지.

- `dictate`: 말로 지시함
- `predict`: 미리 말함
- `unpredictable`: 미리 말할 수 없음
- `contradict`: 반대로 말함
- `condition`: 함께 말해 정한 조건
- `verdict`: 진실을 말한 판정
- `dictionary`: 말들을 모은 책

## 비교 단어

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 목소리로 부름. `dict`는 말로 내용이나 판단을 밝힘

### logos

- 뜻: 말, 설명, 이성
- 구분 포인트: [[logos]]는 말의 내용과 논리. `dict`는 말하는 행위와 선언에 가까움

### index

- 뜻: 가리켜 알려 주는 표시, 색인, 지표
- 연결 포인트: [[indic]]의 `indicate`, `indicator`, `indict`는 `dicare/dictare`와 연결되지만, 말보다 "가리킴/표시" 이미지가 강함

### fore

- 뜻: 앞에, 미리
- 구분 포인트: [[fore]]는 영어 접두사로 시간상 앞섬. `predict`는 Latin 계열로 "미리 말함"에 초점.
