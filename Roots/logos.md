# logos

- 출발 단어: `logic`
- 핵심 어원: 그리스어 `logos` = 말, 설명, 이성, 논리
- 핵심 이미지: 말로 설명하고 이성으로 따짐
- 관련해서 헷갈릴 수 있는 어근: [[dia]], [[mono]], [[di]], [[psych]], [[path]], [[phren]], [[log]] = 통나무, 기록 계열, [[voc]] = 부르다, 목소리 계열, [[scrib]] = 쓰다 계열

## 출발 단어 분석

`logic`은 Greek `logike`에서 온 말.

- `logos`: word, speech, reason
- `logike`: reasoning

따라서 `logic`은 "말과 이성으로 따지는 방식" → "논리"의 뜻.

## 단어들

### logic

- 구성: Greek `logos`
- 뜻: 논리, 논리학
- 어원적 연결: 이성적으로 설명하고 따지는 체계
- 사용 맥락: `basic logic` = 기본 논리

### logical

- 구성: `logic` + `-al`
- 뜻: 논리적인, 타당한
- 어원적 연결: 말과 이성이 맞게 이어짐
- 사용 맥락: `a logical conclusion` = 논리적 결론

### logically

- 구성: `logical` + `-ly`
- 뜻: 논리적으로
- 어원적 연결: 말과 이유가 타당하게 이어지는 방식
- 사용 맥락: `think logically` = 논리적으로 생각하다

### illogical

- 구성: `in-` = not + `logical`
- 뜻: 비논리적인
- 어원적 연결: 말과 이유가 이성적으로 이어지지 않음
- 사용 맥락: `an illogical claim` = 비논리적인 주장

### logistics

- 구성: Greek `logistikos` = skilled in calculation
- 뜻: 물류, 병참, 운영 계획
- 어원적 연결: 계산하고 따져 배치하는 일
- 사용 맥락: `event logistics` = 행사 운영/물류

### dialogue

- 구성: [[dia]] = through, between + `log`
- 뜻: 대화, 문답
- 어원적 연결: 말이 서로 사이를 지나감
- 사용 맥락: `open dialogue` = 열린 대화

### monologue

- 구성: [[mono]] = one + `log`
- 뜻: 독백
- 어원적 연결: 한 사람이 하는 말
- 사용 맥락: `an inner monologue` = 내면의 독백

### prologue

- 구성: `pro-` = before + `log`
- 뜻: 서문, 머리말
- 어원적 연결: 본문 앞에 놓인 말
- 사용 맥락: `the prologue of a novel` = 소설의 서문

### epilogue

- 구성: `epi-` = after, upon + `log`
- 뜻: 맺음말, 후일담
- 어원적 연결: 이야기 뒤에 덧붙는 말
- 사용 맥락: `a brief epilogue` = 짧은 맺음말

### apology

- 구성: `apo-` = away, from + `log`
- 뜻: 사과, 변명, 해명
- 어원적 연결: 원래는 자신을 말로 변호함. 현대 영어에서는 주로 사과
- 사용 맥락: `offer an apology` = 사과하다

### analogy

- 구성: `ana-` = up, according to + `log`
- 뜻: 유추, 비유
- 어원적 연결: 말과 이성으로 비례 관계를 따져 봄
- 사용 맥락: `draw an analogy` = 유추하다

### analogous

- 구성: `analogy` 계열 + `-ous`
- 뜻: 유사한, 비슷하게 대응되는
- 어원적 연결: 두 대상 사이의 비례 관계나 구조가 말과 이성으로 맞아떨어짐
- 사용 맥락: `analogous situations` = 유사한 상황들

### analogue / analog

- 구성: Greek `analogos` = proportionate
- 뜻: 유사물, 대응물, 아날로그의
- 어원적 연결: 어떤 것과 비례적으로 대응되는 것
- 사용 맥락: `a modern analogue` = 현대의 대응물, `analog signal` = 아날로그 신호

### catalogue

- 구성: `kata-` = down, fully + `log`
- 뜻: 목록, 목록으로 만들다
- 어원적 연결: 항목을 말로 하나씩 내려 적은 목록
- 사용 맥락: `a product catalogue` = 상품 목록

### biology

- 구성: `bio-` = life + `-logy`
- 뜻: 생물학
- 어원적 연결: 생명에 대한 학문적 설명
- 사용 맥락: `study biology` = 생물학을 공부하다

### psychology

- 구성: `psycho-` = mind + `-logy`
- 뜻: 심리학
- 어원적 연결: 마음에 대한 학문적 설명
- 사용 맥락: `clinical psychology` = 임상심리학

### sociology

- 구성: `socio-` = society + `-logy`
- 뜻: 사회학
- 어원적 연결: 사회에 대한 학문적 설명
- 사용 맥락: `urban sociology` = 도시사회학

### ideology

- 구성: `idea` + `-logy`
- 뜻: 이념, 관념 체계
- 어원적 연결: 생각들을 말과 논리로 엮은 체계
- 사용 맥락: `political ideology` = 정치 이념

### technology

- 구성: `techno-` = art, skill + `-logy`
- 뜻: 기술, 과학기술
- 어원적 연결: 기술에 대한 체계적 지식
- 사용 맥락: `modern technology` = 현대 기술

### etymology

- 구성: `etymo-` = true sense + `-logy`
- 뜻: 어원학
- 어원적 연결: 말의 참된 의미를 따지는 학문
- 사용 맥락: `English etymology` = 영어 어원

## 기억 포인트

`logos` 계열은 "말 / 설명 / 이성 / 학문" 이미지.

- `logic`: 이성적으로 따짐
- `dialogue`: 말이 사이를 오감
- `prologue`: 앞에 놓인 말
- `epilogue`: 뒤에 붙은 말
- `apology`: 말로 해명하거나 사과함
- `analogy`: 비례 관계를 따져 비유함
- `analogous`: 구조가 대응되어 유사함
- `-logy`: 어떤 분야에 대한 학문적 설명

## 비교 단어

### log

- 뜻: 통나무, 항해/컴퓨터 기록
- 구분 포인트: [[log]] = 통나무/기록은 Greek `logos`와 다른 계열. `logic`, `dialogue`, `biology`의 `logos`와 직접 섞지 않음

### voc

- 뜻: 부르다, 목소리
- 구분 포인트: [[voc]]는 소리 내어 부름. `logos`는 말의 내용, 설명, 이성적 구조에 가까움
