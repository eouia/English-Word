# stanc

- 출발 단어: `instance`
- 핵심 어원: Latin `stare`, present participle `stans/stant-` = 서다
- 핵심 이미지: 어떤 자리에 서 있음, 곁에 서서 압박함, 서 있는 경우
- 관련 문서: [[stat]], [[stand]], [[sist]], [[circ]]
- 주의: [[stat]]은 `state/status/statute`처럼 세워진 상태·제도 중심. `stanc`는 `instance/instant/stance/substance`처럼 `stans/stantia` 형태 중심.

## 출발 단어 분석

`instance` = Old French `instance` ← Latin `instantia` ← `instare`

- `in-`: in, upon
- `stare`: stand
- `instare`: 곁에 서다, 다가서다, 압박하다
- `instantia`: 가까이 서 있음, 긴급함, 요구; 나중에 "경우, 예"
- 현대 의미: 사례, 경우, 예

## 단어들

### instance

- 발음: /ˈɪnstəns/
- 구성: Latin `instantia` ← `instare`
- 뜻: 사례, 경우, 예
- 어원적 연결: 앞에 서서 제시되는 구체적 경우
- 사용 맥락: `for instance` = 예를 들어

### instant

- 발음: /ˈɪnstənt/
- 구성: Latin `instans` = standing near, urgent
- 뜻: 즉각적인, 순간
- 어원적 연결: 바로 곁에 서 있는 것 → 눈앞에 닥친 것
- 사용 맥락: `an instant reply` = 즉각적인 답

### instantly

- 발음: /ˈɪnstəntli/
- 구성: `instant` + `-ly`
- 뜻: 즉시, 곧바로
- 어원적 연결: 바로 눈앞에 닥친 순간에
- 사용 맥락: `respond instantly` = 즉시 응답하다

### instantiate

- 발음: /ˌɪˈnstænʃiˌeɪt/
- 구성: `instance` + `-ate`
- 뜻: 예를 들다; 인스턴스를 만들다
- 어원적 연결: 추상적 개념을 구체적 사례로 세워 보임
- 사용 맥락: programming `instantiate a class` = 클래스를 인스턴스화하다

### instantiation

- 발음: /ˌɪˌnstænʃiˈeɪʃən/
- 구성: `instantiate` + `-ion`
- 뜻: 예시화, 인스턴스화
- 어원적 연결: 구체적 사례로 서게 만드는 과정
- 사용 맥락: philosophy / programming 맥락

### stance

- 발음: /ˈstæns/
- 구성: Latin `stantia` = standing place
- 뜻: 자세, 입장
- 어원적 연결: 어디에 어떻게 서 있는가
- 사용 맥락: `a political stance` = 정치적 입장

### distance

- 발음: /ˈdɪstəns/
- 구성: Latin `dis-` = apart + `stare`
- 뜻: 거리, 간격
- 어원적 연결: 떨어져 서 있음
- 사용 맥락: `social distance` = 사회적 거리

### distant

- 발음: /ˈdɪstənt/
- 구성: `distance` 계열
- 뜻: 먼, 거리를 둔
- 어원적 연결: 떨어져 서 있는 상태
- 사용 맥락: `a distant memory` = 먼 기억

### constant

- 발음: /ˈkɑnstənt/
- 구성: Latin `con-` = together, firmly + `stare`
- 뜻: 일정한, 끊임없는
- 어원적 연결: 단단히 함께 서 있어 변하지 않음
- 사용 맥락: `constant pressure` = 지속적인 압박

### constancy

- 발음: /ˈkɑnstənsi/
- 구성: `constant` + `-cy`
- 뜻: 지속성, 변함없음, 충실함
- 어원적 연결: 흔들리지 않고 계속 서 있음
- 사용 맥락: `constancy of purpose` = 목적의 일관성

### substance

- 발음: /ˈsʌbstəns/
- 구성: Latin `sub-` = under + `stare`
- 뜻: 물질, 본질, 내용
- 어원적 연결: 아래에 서서 받치는 것 → 실제 바탕
- 사용 맥락: `a dangerous substance` = 위험 물질

### substantial

- 발음: /səˈbstænʃəl/
- 구성: `substance` + `-al`
- 뜻: 상당한, 실질적인
- 어원적 연결: 실체와 바탕이 충분히 있음
- 사용 맥락: `substantial evidence` = 상당한 증거

### circumstance

- 발음: /ˈsɝkəˌmstæns/
- 구성: Latin `circum` = around + `stare`
- 뜻: 상황, 정황
- 어원적 연결: 어떤 일 주변에 서 있는 조건들
- 사용 맥락: `under the circumstances` = 그런 상황에서

## 기억 포인트

`stanc/stant` 계열은 "어떤 자리에 서 있음" 이미지.

- `instance`: 앞에 세워 든 사례
- `instant`: 바로 곁에 선 순간
- `stance`: 서 있는 자세와 입장
- `distance`: 떨어져 서 있음
- `constant`: 단단히 서서 변하지 않음
- `substance`: 아래에서 받치는 실체
- `circumstance`: 주변에 서 있는 조건
