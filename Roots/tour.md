# tour

- 출발 단어: `contour`
- 핵심 어원: Latin `tornare` = 선반으로 돌려 깎다, 둥글게 만들다
- 더 이른 형태: Greek `tornos` = 선반, 원을 그리는 도구
- 핵심 이미지: 빙 돌거나 둘러가며 형태를 만듦
- 관련 문서: [[vers]], [[volv]], [[circ]]
- 주의: [[vers]]도 "돌다" 계열이지만 Latin `vertere` = 방향을 돌리다. `tour`는 `tornare/tornos`의 회전·원형·순회 이미지.

## 출발 단어 분석

`contour` ← French `contour` ← Italian / Medieval Latin `contornare`

- `con-`: 함께, 완전히, 둘레로
- `tornare`: 돌리다, 둥글게 만들다
- 원래 이미지: 둘레를 돌아 선을 그림
- 현대 의미: 윤곽, 등고선, 윤곽을 그리다

## 단어들

### contour

- 발음: /ˈkɑˌntʊr/
- 구성: `con-` + `tour` = turn around
- 뜻: 윤곽, 등고선, 윤곽을 그리다
- 어원적 연결: 바깥 둘레를 돌아가며 그린 선
- 사용 맥락: `contour lines` = 등고선

### contouring

- 구성: `contour` + `-ing`
- 뜻: 윤곽 만들기, 얼굴 윤곽 화장
- 어원적 연결: 둘레와 곡선을 따라 형태를 살림
- 사용 맥락: `facial contouring` = 얼굴 윤곽 보정

### tour

- 발음: /ˈtʊr/
- 구성: Old French `tour` ← `torner`
- 뜻: 여행, 순회, 둘러보기
- 어원적 연결: 한 바퀴 돌며 방문함
- 사용 맥락: `a city tour` = 도시 관광

### tourist

- 발음: /ˈtʊrəst/
- 구성: `tour` + `-ist`
- 뜻: 관광객
- 어원적 연결: 여러 곳을 돌며 방문하는 사람
- 사용 맥락: `foreign tourists` = 외국인 관광객

### tourism

- 발음: /ˈtʊˌrɪzəm/
- 구성: `tour` + `-ism`
- 뜻: 관광업, 관광
- 어원적 연결: 둘러보는 여행 활동과 산업
- 사용 맥락: `medical tourism` = 의료 관광

### touring

- 발음: /ˈtʊrɪŋ/
- 구성: `tour` + `-ing`
- 뜻: 순회하는, 여행하는
- 어원적 연결: 여러 곳을 돌며 이동함
- 사용 맥락: `a touring musician` = 순회공연 음악가

### detour

- 발음: /dɪˈtʊr/
- 구성: `de-` = away + `tour`
- 뜻: 우회로, 우회하다
- 어원적 연결: 곧장 가지 않고 돌아감
- 사용 맥락: `take a detour` = 우회하다

### tournament

- 발음: /ˈtʊrnəmənt/
- 구성: Old French `tornoiement` ← `tornoier` = joust, turn around
- 뜻: 토너먼트, 경기 대회
- 어원적 연결: 말을 돌리며 겨루는 중세 경기에서 확장
- 사용 맥락: `a tennis tournament` = 테니스 대회

### turn

- 발음: /ˈtɝn/
- 구성: Old French `torner` / Old English `turnian`
- 뜻: 돌다, 돌리다, 차례
- 어원적 연결: 축이나 방향을 돌림
- 사용 맥락: `turn the wheel` = 바퀴를 돌리다

### return

- 발음: /rɪˈtɝn/
- 구성: `re-` = back + `turn`
- 뜻: 돌아오다, 돌려주다
- 어원적 연결: 다시 방향을 돌려 옴
- 사용 맥락: `return home` = 집으로 돌아오다

### attorney

- 발음: /əˈtɝni/
- 구성: Old French `atorner` = assign, appoint
- 뜻: 변호사, 대리인
- 어원적 연결: 누군가에게 일을 돌려 맡긴 사람
- 사용 맥락: `defense attorney` = 변호인

### attorn

- 구성: Old French `atorner`
- 뜻: 법적으로 권리나 의무를 이전하다
- 어원적 연결: 권한을 다른 쪽으로 돌려 넘김
- 사용 맥락: 법률 용어로 제한적 사용

## 기억 포인트

`tour` 계열은 "돌다, 둘러가다" 이미지.

- `contour`: 둘레를 돌아 그은 윤곽
- `tour`: 한 바퀴 돌며 둘러보기
- `detour`: 돌아가는 길
- `tournament`: 말을 돌리며 겨루던 경기
- `turn`: 돌다, 방향을 바꾸다
- `return`: 다시 돌아오다

## 비교 단어

### vers

- 뜻: 방향을 돌리다
- 구분 포인트: [[vers]]는 Latin `vertere` = turn. `convert`, `reverse`, `diverse`처럼 방향 전환 중심.

### volv

- 뜻: 굴리다, 말다
- 구분 포인트: [[volv]]는 `roll` 이미지. `tour`는 회전·순회·둘레를 도는 이미지.
