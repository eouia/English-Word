# anomal

- 출발 단어: `anomaly`
- 핵심 어원: Greek `anomalos` = uneven, irregular
- 구성 배경: `an-` = not + `homalos` = even, smooth
- 핵심 이미지: 고르지 않음, 규칙에서 벗어남, 예외적 현상
- 관련 문서: 없음
- 주의: `anomaly`는 단순히 "이상한 것"보다, 기대한 규칙·패턴에서 벗어난 관찰값이나 현상이라는 느낌이 강함.

## 출발 단어 분석

`anomaly` ← Greek `anomalia`

- `an-`: not
- `homalos`: even, smooth
- `anomalos`: uneven, irregular
- 원래 이미지: 평평하거나 고르지 않은 상태
- 현대 의미: 이례적인 것, 변칙, 이상 현상

## 단어들

### anomaly

- 발음: /əˈnɑməli/
- 구성: Greek `anomalos` = uneven, irregular + `-y`
- 뜻: 이례적인 것, 변칙, 이상 현상
- 어원적 연결: 정상적 패턴에서 고르지 않게 벗어난 것
- 사용 맥락: `a statistical anomaly` = 통계적 이상값

### anomalous

- 발음: /əˈnɑmələs/
- 구성: `anomaly` 계열 + `-ous`
- 뜻: 이례적인, 변칙적인
- 어원적 연결: 규칙이나 기대와 맞지 않게 고르지 않음
- 사용 맥락: `an anomalous result` = 이례적인 결과

### anomalously

- 구성: `anomalous` + `-ly`
- 뜻: 이례적으로, 변칙적으로
- 어원적 연결: 보통 패턴과 다르게
- 사용 맥락: `anomalously high temperatures` = 이례적으로 높은 기온

## 기억 포인트

`anomal` 계열은 "규칙에서 벗어난 고르지 않음" 이미지.

- `anomaly`: 패턴에서 벗어난 이상 현상
- `anomalous`: 이례적인, 변칙적인
- `anomalously`: 이례적으로

## 비교 단어

### normal

- 뜻: 정상적인, 기준에 맞는
- 구분 포인트: `normal`은 기준·규칙에 맞음. `anomaly`는 그 기준에서 벗어난 예외.

### abnormal

- 뜻: 비정상적인
- 구분 포인트: `abnormal`은 일반적인 "정상이 아님". `anomalous`는 관찰된 패턴이나 규칙에서 벗어난다는 분석적 느낌이 강함.

### homogeneous

- 뜻: 균질한, 같은 종류로 이루어진
- 구분 포인트: `homogeneous`의 `homo-` = same. `anomaly`의 `homalos` = even, smooth와 닮았지만 학습용으로는 별도 처리.
