# cipher

- 출발 단어: `indecipherable`
- 핵심 어원: Arabic `sifr` = zero, empty
- 핵심 이미지: 빈 자리의 숫자에서 읽기 어려운 암호로 확장
- 관련 문서: [[numer]], [[crypt]]
- 주의: `cipher`는 숫자·0에서 암호 의미로 확장. 일반적인 "수" 계열 [[numer]]와는 분리.

## 출발 단어 분석

`indecipherable` = `in-` + `decipher` + `-able`

- `in-`: not
- `decipher`: 암호를 풀어 읽다
- `-able`: 할 수 있는
- 원래 이미지: 암호를 풀어 읽을 수 없음
- 현대 의미: 판독할 수 없는, 해독 불가능한

## 단어들

### indecipherable

- 구성: `in-` = not + `decipherable`
- 뜻: 판독할 수 없는, 해독 불가능한
- 어원적 연결: 암호처럼 되어 있어 풀어 읽을 수 없음
- 사용 맥락: `indecipherable handwriting` = 알아볼 수 없는 필체, `an indecipherable code` = 해독 불가능한 암호

### decipher

- 구성: `de-` = undo, remove + `cipher`
- 뜻: 해독하다, 판독하다
- 어원적 연결: 암호나 읽기 어려운 기호를 풀어 의미를 드러냄
- 사용 맥락: `decipher an ancient text` = 고대 문헌을 판독하다

### decipherable

- 구성: `decipher` + `-able`
- 뜻: 해독할 수 있는, 판독 가능한
- 어원적 연결: 암호나 기호를 풀어 읽을 수 있음
- 사용 맥락: `barely decipherable notes` = 간신히 알아볼 수 있는 메모

### decipherment

- 구성: `decipher` + `-ment`
- 뜻: 해독, 판독
- 어원적 연결: 암호나 미지의 문자를 풀어 읽는 과정
- 사용 맥락: `the decipherment of hieroglyphs` = 상형문자의 해독

### cipher

- 구성: Old French `cifre` ← Arabic `sifr` = zero, empty
- 뜻: 암호, 암호 체계; 0; 하찮은 사람
- 어원적 연결: 0이라는 낯선 기호에서 숫자·기호 체계, 나아가 비밀 기호 체계로 확장
- 사용 맥락: `a secret cipher` = 비밀 암호, `write in cipher` = 암호로 쓰다

### encipher

- 구성: `en-` = put into + `cipher`
- 뜻: 암호화하다
- 어원적 연결: 메시지를 암호 체계 안에 넣음
- 사용 맥락: `encipher a message` = 메시지를 암호화하다

### zero

- 구성: Italian `zero` ← Medieval Latin `zephirum` ← Arabic `sifr`
- 뜻: 0, 영
- 어원적 연결: `cipher`와 같은 Arabic `sifr`에서 온 이중어
- 사용 맥락: `zero percent` = 0퍼센트, `below zero` = 영하

## 기억 포인트

`cipher`는 "0에서 암호로 간 기호" 이미지.

- `cipher`: 암호, 0
- `decipher`: 암호를 풀어 읽음
- `indecipherable`: 풀어 읽을 수 없음
- `encipher`: 암호로 만듦
- `zero`: 같은 `sifr`에서 온 0

## 비교 단어

### numer

- 뜻: 수, 숫자
- 구분 포인트: [[numer]]는 Latin `numerus` = number 계열. `cipher`는 Arabic `sifr`에서 온 별도 계열.

### crypt

- 뜻: 숨기다, 비밀의
- 구분 포인트: Greek `kryptos` = hidden 계열. 암호의 "숨김" 의미는 가깝지만 `cipher`의 직접 어원은 숫자 0.

### code

- 뜻: 암호, 규칙 체계
- 구분 포인트: Latin `codex` = book, tablet 계열. 현대 의미는 암호와 겹치지만 어원은 별도.
