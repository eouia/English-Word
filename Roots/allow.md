# allow

- 출발 단어: `allow`
- 핵심 어원: Anglo-French `alouer/allouer`에서 두 계열이 섞임
- 계열 1: Latin `allaudare` = 칭찬하다, 승인하다
- 계열 2: Medieval Latin `allocare` = 배정하다, 할당하다
- 핵심 이미지: 인정하여 허락함, 몫을 배정함
- 관련 문서: [[loc]], [[laud]], [[mit]]
- 주의: `allow`는 한 어근으로만 정리하기 어려운 혼합 어원 단어.

## 출발 단어 분석

`allow`는 Middle English `allowen/alowen` ← Anglo-French `alouer/allouer`.

- `allaudare`: 칭찬하다, 좋다고 인정하다
- `allocare`: 어떤 몫이나 자리를 배정하다
- 두 프랑스어 형태와 의미가 영어에서 섞이며 `approve`, `admit`, `permit`, `grant` 의미로 확장

현대 핵심 의미는 "허락하다, 인정하다, 가능하게 하다".

## 단어들

### allow

- 구성: Anglo-French `alouer/allouer`
- 뜻: 허락하다, 인정하다, 가능하게 하다
- 어원적 연결: 좋다고 인정하거나, 몫·권리를 배정해 줌
- 사용 맥락: `allow access` = 접근을 허용하다

### allowed

- 구성: `allow` + `-ed`
- 뜻: 허용된, 허락된
- 어원적 연결: 인정되어 가능해진 상태
- 사용 맥락: `not allowed` = 허용되지 않음

### allowing

- 구성: `allow` + `-ing`
- 뜻: 허락하는, 가능하게 하는; 고려하면
- 어원적 연결: 어떤 조건이나 여지를 인정함
- 사용 맥락: `allowing for delays` = 지연을 감안하면

### allowable

- 구성: `allow` + `-able`
- 뜻: 허용 가능한, 인정될 수 있는
- 어원적 연결: 규칙 안에서 인정될 수 있음
- 사용 맥락: `allowable expenses` = 허용 비용

### allowance

- 구성: `allow` + `-ance`
- 뜻: 용돈, 수당, 허용량, 참작
- 어원적 연결: 배정된 몫 또는 인정된 여지
- 사용 맥락: `a monthly allowance` = 월 용돈

### disallow

- 구성: `dis-` = reverse, not + `allow`
- 뜻: 허용하지 않다, 인정하지 않다
- 어원적 연결: 인정·허락을 거부함
- 사용 맥락: `disallow a claim` = 청구를 인정하지 않다

### disallowance

- 구성: `disallow` + `-ance`
- 뜻: 불허, 부인, 인정 거부
- 어원적 연결: 허용이나 공제를 인정하지 않음
- 사용 맥락: 세무·법률 맥락

## 기억 포인트

`allow`는 두 이미지가 합쳐진 단어.

- 승인 계열: 좋다고 인정함 → 허락함
- 배정 계열: 몫이나 권리를 배정함 → 허용량, 수당
- `allowable`: 규칙상 인정 가능
- `allowance`: 배정된 몫 또는 참작 여지
- `disallow`: 인정하지 않음

## 비교 단어

### loc

- 뜻: 장소, 자리, 배정
- 연결 포인트: [[loc]]의 `allocate`와 `allowance`의 "배정된 몫" 의미는 Medieval Latin `allocare` 쪽에서 만남.

### laud

- 뜻: 칭찬하다
- 연결 포인트: `allow`의 "승인하다" 의미에는 Latin `allaudare` = praise 계열이 섞임.

### permit

- 뜻: 허가하다
- 구분 포인트: [[mit]]의 `permit`은 "통과하도록 보내 줌" 이미지. `allow`는 인정·배정의 혼합 이미지.
