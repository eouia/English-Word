# duc

- 출발 단어: `conduct`
- 핵심 어원: 라틴어 `ducere` / `duct-` = 이끌다, 끌고 가다
- 핵심 이미지: 방향을 잡아 이끌어 감
- 관련해서 헷갈릴 수 있는 어근: [[tract]] = 끌다, 당기다 계열

## 출발 단어 분석

`conduct`는 `con-` + `duct`.

- `con-`: together
- `ducere`: 이끌다

따라서 `conduct`는 "함께 이끌어 감" → "수행하다, 지휘하다, 행동"의 뜻.

## 단어들

### conduct

- 발음: n. /ˈkɑndəkt/, v. /kɑˈndʌkt/
- 구성: `con-` = together + `duct`
- 뜻: 수행하다, 지휘하다, 행동
- 어원적 연결: 사람이나 일을 함께 이끌어 감
- 사용 맥락: `conduct research` = 연구를 수행하다

### produce

- 발음: n. /ˈproʊdus/, v. /prəˈdus/
- 구성: `pro-` = forward + `duce`
- 뜻: 생산하다, 만들어 내다
- 어원적 연결: 앞으로 이끌어 내어 나타나게 함
- 사용 맥락: `produce evidence` = 증거를 제시하다

### product

- 발음: /ˈprɑdəkt/
- 구성: Latin `productum` ← `pro-` = forward + `ducere` = lead
- 뜻: 생산물, 제품, 결과
- 어원적 연결: 앞으로 이끌어 내어 만들어진 것
- 사용 맥락: `a new product` = 신제품

### production

- 발음: /prəˈdʌkʃən/
- 구성: `produce` + `-ion`
- 뜻: 생산, 제작, 생산물
- 어원적 연결: 앞으로 이끌어 내어 만들어 내는 과정
- 사용 맥락: `mass production` = 대량 생산

### productive

- 발음: /prəˈdʌktɪv/
- 구성: `produce` + `-ive`
- 뜻: 생산적인, 산출력이 있는
- 어원적 연결: 결과물을 잘 이끌어 내는 성질
- 사용 맥락: `a productive meeting` = 생산적인 회의

### productivity

- 발음: /ˌproʊdəˈktɪvəti/
- 구성: `productive` + `-ity`
- 뜻: 생산성
- 어원적 연결: 결과물을 이끌어 내는 능력의 정도
- 사용 맥락: `increase productivity` = 생산성을 높이다

### reduce

- 발음: /rəˈdus/
- 구성: `re-` = back + `duce`
- 뜻: 줄이다, 감소시키다
- 어원적 연결: 뒤로 이끌어 낮은 상태로 되돌림
- 사용 맥락: `reduce costs` = 비용을 줄이다

### introduce

- 발음: /ˌɪntrəˈdus/
- 구성: `intro-` = inward + `duce`
- 뜻: 소개하다, 도입하다
- 어원적 연결: 안으로 이끌어 들임
- 사용 맥락: `introduce a new policy` = 새 정책을 도입하다

### induce

- 발음: /ˌɪˈndus/
- 구성: `in-` = into + `duce`
- 뜻: 유도하다, 초래하다
- 어원적 연결: 어떤 상태 안으로 이끌어 감
- 사용 맥락: `induce sleep` = 잠을 유도하다

### deduce

- 발음: /dɪˈdus/
- 구성: `de-` = down, from + `duce`
- 뜻: 추론하다, 연역하다
- 어원적 연결: 전제에서 결론을 이끌어 냄
- 사용 맥락: `deduce the answer` = 답을 추론하다

### abduct

- 발음: /æˈbdʌkt/
- 구성: `ab-` = away + `duct`
- 뜻: 납치하다, 유괴하다
- 어원적 연결: 멀리 끌고 감
- 사용 맥락: `abduct a child` = 아이를 납치하다

### duct

- 발음: /ˈdʌkt/
- 구성: Latin `ductus`
- 뜻: 관, 도관
- 어원적 연결: 공기나 액체를 이끌어 흐르게 하는 통로
- 사용 맥락: `air duct` = 공기 통로

### aqueduct

- 발음: /ˈækwəˌdʌkt/
- 구성: `aqua` = water + `duct`
- 뜻: 수로, 송수로
- 어원적 연결: 물을 이끌어 가는 구조물
- 사용 맥락: `Roman aqueducts` = 로마의 수로

### educate

- 발음: /ˈɛdʒəˌkeɪt/
- 구성: Latin `educare` / `educere`
- 뜻: 교육하다
- 어원적 연결: 사람의 능력을 밖으로 이끌어 기름. 현대 어원 설명에서는 `ducere`와 연결해 기억하기 좋음
- 사용 맥락: `educate students` = 학생들을 교육하다

### duke

- 발음: /ˈduk/
- 구성: Latin `dux` = leader
- 뜻: 공작
- 어원적 연결: 군대나 사람을 이끄는 지도자
- 사용 맥락: `the Duke of Wellington` = 웰링턴 공작

### seduce

- 발음: /sɪˈdus/
- 구성: `se-` = apart, away + `duce`
- 뜻: 유혹하다, 꾀다
- 어원적 연결: 바른 길에서 다른 쪽으로 이끌어 감
- 사용 맥락: `seduce voters with promises` = 약속으로 유권자를 유혹하다

## 기억 포인트

`duc` 계열은 "이끌다" 이미지.

- `conduct`: 함께 이끌어 수행함
- `produce`: 앞으로 이끌어 내놓음
- `product`: 이끌어 내 만들어진 결과물
- `productive`: 결과물을 잘 이끌어 냄
- `reduce`: 뒤로 이끌어 줄임
- `introduce`: 안으로 이끌어 들임
- `deduce`: 근거에서 결론을 이끌어 냄
- `seduce`: 다른 길로 이끌어 유혹함

## 비교 단어

### tract

- 뜻: 끌다, 당기다
- 구분 포인트: [[tract]]는 힘으로 잡아 끄는 느낌. `duc`는 방향을 잡아 앞에서 이끄는 느낌
