# exempl

- 출발 단어: `example`
- 핵심 어원: Latin `exemplum` = sample, pattern, example
- 핵심 이미지: 전체에서 밖으로 꺼내 보이는 표본
- 관련 어근: [[ex]], [[sumere]], [[cap]]

## 출발 단어 분석

`example`은 Latin `exemplum`에서 온 말.

`exemplum`은 Latin `eximere` = take out, remove와 연결됨.

- `ex-`: out
- `emere`: take, obtain

따라서 `example`은 "밖으로 꺼내 보인 것" → "예, 본보기"의 뜻.

## 단어들

### example

- 발음: /ɪˈgzæmpəl/
- 구성: Latin `exemplum`
- 의미: 예, 사례, 본보기
- 어원적 연결: 전체에서 하나를 꺼내 보여 주는 표본
- 사용 맥락: `give an example` = 예를 들다

### sample

- 발음: /ˈsæmpəl/
- 구성: Old French `essample`, `example`의 줄어든 형태
- 의미: 표본, 견본, 샘플
- 어원적 연결: 전체에서 꺼낸 작은 일부
- 사용 맥락: `a blood sample` = 혈액 샘플

### exemplar

- 발음: /ɪˈgzɛmplɑr/
- 구성: Latin `exemplar` = model, pattern
- 의미: 모범, 전형
- 어원적 연결: 따라 할 만한 본보기
- 사용 맥락: `an exemplar of courage` = 용기의 전형

### exemplary

- 발음: /ɪˈgzɛmplɚi/
- 구성: `exemplar` 계열 + `-y`
- 의미: 모범적인, 본보기로 삼을 만한
- 어원적 연결: 다른 사람에게 보여 줄 만한 표본
- 사용 맥락: `exemplary behavior` = 모범적인 행동

### exemplify

- 발음: /ɪˈgzɛmpləˌfaɪ/
- 구성: `example/exempl-` + `-ify`
- 의미: 예시하다, 전형적으로 보여 주다
- 어원적 연결: 어떤 성질을 예로 꺼내 보여 줌
- 사용 맥락: `exemplify the principle` = 그 원칙을 예시하다

### exemplification

- 구성: `exemplify` + `-ication`
- 의미: 예시, 실례
- 어원적 연결: 예를 들어 보여 주는 행위
- 사용 맥락: `an exemplification of the rule` = 그 규칙의 예시

### exempt

- 발음: /ɪˈgzɛmpt/
- 구성: Latin `exemptus` < `eximere`
- 의미: 면제된, 제외하다
- 어원적 연결: 전체 의무나 규칙에서 밖으로 꺼내 뺌
- 사용 맥락: `be exempt from tax` = 세금이 면제되다

### exemption

- 발음: /ɪˈgzɛmpʃən/
- 구성: `exempt` + `-ion`
- 의미: 면제, 제외
- 어원적 연결: 의무에서 빼내 주는 행위
- 사용 맥락: `tax exemption` = 세금 면제

## 기억 포인트

- `example`: 꺼내 보여 주는 예
- `sample`: 전체에서 꺼낸 표본
- `exemplar`: 따라 할 본보기
- `exemplary`: 본보기로 삼을 만한
- `exemplify`: 예로 보여 줌
- `exempt`: 규칙 밖으로 빼냄

## 비교 단어

- [[sumere]]
  - 핵심: 취하다, 집어 들다
  - 구분 포인트: `sumere`는 `sub-` + `emere` 계열. `example/exempt`도 `emere` = take와 연결되지만, `ex-`로 "밖으로 꺼냄"이 핵심

- [[cap]]
  - 핵심: 잡다, 취하다
  - 구분 포인트: `capere`는 잡거나 받음. `exemplum`은 꺼내어 보여 주는 표본

- [[part]]
  - 핵심: 부분, 몫
  - 구분 포인트: `sample`은 전체에서 뽑은 표본. `part`는 전체를 이루는 부분 일반
