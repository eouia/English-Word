# capit

- 출발 단어: `capital`
- 핵심 어원: 라틴어 `caput` / `capit-` = 머리, 꼭대기, 우두머리
- 핵심 이미지: 몸의 머리처럼 가장 위나 중심에 있는 것
- 관련해서 헷갈릴 수 있는 어근: [[cap]] = 잡다, 받다 계열

## 출발 단어 분석

`capital`은 Latin `capitalis`에서 온 말.

- `caput`: 머리
- `capitalis`: 머리에 관한, 주요한

따라서 `capital`은 "머리에 해당하는 것" → "중요한, 수도, 자본, 대문자"의 뜻.

## 단어들

### capital

- 구성: `capit` + `-al`
- 뜻: 수도, 자본, 대문자, 중요한
- 어원적 연결: 국가나 조직의 머리 역할을 하는 것
- 사용 맥락: `the capital city` = 수도

### capitalize

- 구성: `capital` + `-ize`
- 뜻: 대문자로 쓰다, 자본화하다, 이용하다
- 어원적 연결: 글자의 머리급 형태로 만들거나, 자본으로 바꿈
- 사용 맥락: `capitalize on an opportunity` = 기회를 활용하다

### captain

- 구성: Old French `capitaine` < Latin `capitaneus`
- 뜻: 선장, 주장, 대장
- 어원적 연결: 무리의 머리 역할을 하는 사람
- 사용 맥락: `team captain` = 팀 주장

### chapter

- 구성: Latin `capitulum` = little head
- 뜻: 장, 챕터
- 어원적 연결: 글의 작은 머리. 내용을 나누는 제목 단위
- 사용 맥락: `the first chapter` = 첫 장

### chief

- 구성: Old French `chef` < Latin `caput`
- 뜻: 우두머리, 주요한
- 어원적 연결: 집단의 머리
- 사용 맥락: `chief executive officer` = 최고경영자

### achieve

- 구성: Old French `achever` < `chef` = head, end
- 뜻: 성취하다, 달성하다
- 어원적 연결: 일의 머리끝, 끝점에 도달함. 현대 형태에서는 직접 분해하기 어려움
- 사용 맥락: `achieve a goal` = 목표를 달성하다

### mischief

- 구성: Old French `meschef` = misfortune
- 뜻: 장난, 해악
- 어원적 연결: 나쁜 머리/나쁜 끝이라는 역사적 이미지. 현대 의미는 "해가 되는 장난"
- 사용 맥락: `get into mischief` = 장난을 치다

### cap

- 구성: Old English `caeppe` / Latin `cappa`
- 뜻: 모자, 덮개, 한도
- 어원적 연결: 머리를 덮는 것. `capital`의 `caput`와 의미상 가깝지만, 어원 경로는 완전히 단정하지 않음
- 사용 맥락: `wear a cap` = 모자를 쓰다

### capsize

- 구성: `cap` + `size`
- 뜻: 뒤집히다, 전복되다
- 어원적 연결: 머리나 윗부분이 아래로 가며 뒤집힘
- 사용 맥락: `The boat capsized.` = 배가 전복됐다

### capstone

- 구성: `cap` + `stone`
- 뜻: 꼭대기 돌, 완성 단계
- 어원적 연결: 구조물 맨 위를 덮는 돌
- 사용 맥락: `a capstone project` = 종합 완성 프로젝트

## 기억 포인트

`capit` 계열은 "머리 / 꼭대기 / 우두머리" 이미지.

- `capital`: 국가나 글자, 돈의 머리급 요소
- `captain`: 무리의 머리
- `chapter`: 글의 작은 머리
- `chief`: 집단의 우두머리
- `achieve`: 끝점까지 도달함
- `cap`: 머리를 덮는 것

## 비교 단어

### cap

- 뜻: 잡다, 받다
- 구분 포인트: [[cap]]은 Latin `capere` = 잡다 계열. `capit`은 Latin `caput` = 머리 계열. 형태가 비슷해도 시험 단어에서는 갈라 외우는 편이 좋음

### head

- 뜻: 머리, 우두머리
- 구분 포인트: [[head]]는 게르만계 기본어. `capit`은 라틴계 고급어휘에서 `capital`, `captain`, `chapter`처럼 나타남
